## 1. 本次目标

基于 `衢州杭氧1#42000数据.csv` 中的 AI705（粗氩氩含量）时序数据，先完成一版“氮塞/疑似氮塞候选区间”初筛，为后续挑选真实案例、做 Agent 诊断分析提供事件清单。

本次不采用固定绝对阈值（如 AI705 < 97），而采用：

```text
1 min 重采样 → AI705 平滑 → 自适应基准线 → 相对基准的下凹识别 → 候选事件合并 → 前后 30 min 对比 → 初步分级
```

## 8. 后续真实案例分析建议

建议先挑 6 个窗口做人工案例库：

| 类型 | 建议数量 | 用途 |
|---|---:|---|
| 强疑似氮塞 | 2~3 个 | 验证系统能不能识别典型下凹并组织证据 |
| 轻微疑似氮塞 | 1~2 个 | 验证系统能不能识别 99.5~98 附近的小幅下凹 |
| 疑似但不是 | 2~3 个 | 验证系统能不能排除负荷扰动、仪表异常、普通波动 |

每个案例建议补充：

```text
1. AI705 原始曲线与自适应基准；
2. 开始时间、谷底时间、结束时间；
3. 下凹幅度、持续时间、恢复比例；
4. AI701、FI702、FIC701、FIC101/FIQ101 的前后对比；
5. 当班操作记录、报警记录、是否校表或仪表维护；
6. 专家标注：确认氮塞 / 轻微氮塞 / 排除氮塞 / 继续观察。
```

## 9. 当前结论

这批数据中，AI705 并非只出现一次大幅下降，而是存在多次“下凹—回升”候选区间。按当前初筛规则，已有足够样本构建：

- 正样本：高风险 AI705 下凹 + 多测点联动；
- 边界样本：轻微 AI705 下凹；
- 反例样本：AI705 有下凹但联动证据不足或存在负荷扰动。

因此，下一步可以正式进入“真实案例标注 + Agent 诊断验证”阶段。
