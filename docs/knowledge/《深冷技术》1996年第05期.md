# （5）关于变压吸附(PSA)制氧装置

为取得  $95\%$  纯度氧，笔者对采用变压吸附(PSA)装置的辩议：

如前所述，国内外PSA制氧装置的发展，对于中小型钢铁企业上马制氧机以富氧喷煤，被认为有其巨大优势，这是因为喷煤本无需用纯氧。且如果喷煤富氧按  $3\% \sim 5\%$  考虑，毕竟是少量，适合于PSA  $1500\mathrm{m}^3 /\mathrm{h}$  以下的规模。

大体上说，PSA装置有以下优点：[1]

a. 制氧站投资少。PSA制氧设备的投资只有相同能力深冷设备的  $50\% \sim 60\%$  。  
b. 制氧能耗较低。电耗  $0.42 \sim 0.54 \mathrm{~kWh} / \mathrm{m}^{3} \mathrm{O}_{2}$ , 并有较好的经济效益, 如1500  $\mathrm{m}^{3} / \mathrm{h}$  的PSA装置用于  $500 \mathrm{~m}^{3}$  高炉富氧, 富氧率可达  $2.3\%$  。富氧后煤比提高  $50 \mathrm{~kg} / \mathrm{t}$  铁左右, 年经济效益可增加  $200 \sim 300$  万元, 与

建立同规模的深冷空分设备相比，可节省约1000万元的投资，节省近400万元/年的运行费用。

但是变压吸附亦有许多不利方面：

① 变压吸附的产率无法满足冶金行业的需求，即便对喷煤富氧亦只能对中小型企业。有远见的考虑是不能将来被这一“钉子”制时，不如将用深冷设备一次到位。  
② 毕竟能耗抵不过大型深冷制氧（0.4 $\mathrm{kWh / m^3O_2}$ ），且变压吸附交换频繁，常有故障需停车维修，在运转率上远不如深冷设备稳定。如果分子筛吸附剂每10年更换一次，设备投资实在亦是很昂贵的。提取率  $< 50\%$  ，氧纯度只能达到  $93\%$  左右。  
③ 综合利用差。一是不能提取氮气[9]，而氩是炼钢轧钢急待发展的新技术；二是不能同时制造氮气，而深冷装置能同时制造氮

气，可抵消PSA的经济优势，因为氮气是炼铁炼钢不可缺少的；三是更谈不上稀有气体的综合开发，正是物以稀为贵，开发稀有气体不仅有很大的经济性，更主要的是促使许多新技术、新科研项目上马。

④ 不具生产的灵活性，不能与其他氧气系统共配合。在停车期间因不能提取液氧积存，设有备用手段。即使在有深冷制氧机存在的条件下，在管理上兼顾两地，为其备用亦不方便。

虽然，从钢铁厂全局来说可以有以上的宏观认识，但必须说对于高炉富氧鼓风及电炉炼钢等不需纯氧，单炉用氧量较小的生产工业，PSA装置有其巨大的优越性：成本低、占地小、随时开停方便、操作简单及全盘程控等，尤其是发展为VPSA富氧装置（称为真空变压吸附，吸附压力在  $0.6\mathrm{MPa}$  以下，解吸压力在  $-0.48\mathrm{MPa}$  以下），能耗降低及可靠性方面都有很大进展。例如，据Praxair介绍其VPSA装置提取率已超过  $50\%$  ，装置可靠性已达  $98\%$  ，切换阀门已可达100万次以上。因此，不应该囿于一种观点，能耗及新技术就可能使VPSA上升为第一位考虑，而深冷装置及全用深冷装置的方案就可能让位，特别是在  $6000\mathrm{m}^3 /\mathrm{hO}_2$  以下的装置。所谓无氮、氩，VPSA与深冷装置不协调的缺点就可能另谋筹措，不在话下了。

# （6）关于  $\mathrm{PSA} + \mathrm{ASU}$  制氧装置

近来出现的另一种新制氧机，是  $\mathrm{PSA}+$  ASU装置[16]，亦是可以出纯氧或  $95\%$  氧的，但对此种装置因尚无投产运行的实例可资了解，尚应存一些必要的疑虑，不宜轻举。

例如说这种制氧机似乎可以改造旧制氧机，扩大不止一倍的产量。但是，旧制氧机大部分是可逆板翅式换热器流程，要改造就要新加  $\mathrm{CO}_{2}$  、  $\mathrm{H}_{2} \mathrm{O}$  的分子筛吸附器及富氧的分子筛吸附器两组，本身就有个有无空间放得下的问题。其次精馏塔进料是  $80 \% \mathrm{O}_{2}$  的富

氧空气而不是  $21\% \mathrm{O}_2$  的空气，需要拆除许多塔板，降低塔高，改造冷箱；还带来原可逆板翅试换热器的不合用，因为返流气体氧、氮比改换原通道排列显然不适应；长短是否合宜及膨胀量的变化都可能引起。再加上另加液氮液氧泵、氨预冷系统而要放弃原有的氮、氧压缩机，特别是因扩大生产还要另加空压机并联。而空压机并联由于排出气流不可避免的对抵，会减少  $3\% \sim 5\%$  甚至更多的空压机出力，还不要说原有厂房是否留有加空压机的空间。这样的改造对原有空分设备几乎是体无完肤的再造，最后带来的是制氧成本的升高。

随着喷煤的富氧要求，旧制氧机走这条路能否成功，最好能由UEI公司在中国先作一个试点。至于说PSA + ASU装置更宜一次新建，亦应存在一些思考。且不说波兰分二期上马完善是何原因，不惜这样二次改造。在我国空分设备是适应任务需要一次建成投产的，谈不上分两次的必要。至于说到其采用液氧泵、液氮泵压缩氧、氮，却正是冷箱内压缩新技术的开发，是空分冷箱经多年运行日趋要普遍采用的[23]，谈不上这是PSA + ASU的专有技术。PSA + ASU装置真正落实的只能是单位制氧电耗  $0.69\mathrm{kWh} / \mathrm{m}^3\mathrm{O}_2$  的高指标。正像目前用填料塔精馏生产纯氩一样，亦可用于PSA + ASU而无需再加含氢的镍合金作触媒以省去氢气站，岂不更好！采用含氢镍合金触媒根本不能作为节约优点来比较，倒是加氨预冷却是与现分子筛流程中努力取消氟里昂预冷水机组背道而驰。氨预冷是当年“3350”高低压空分设备时的累赘，早已被淘汰。现又被在PSA + ASU中恢复，不能不说是倒退。

说到  $\mathrm{PSA} + \mathrm{ASU}$  可以减少中压贮罐及液氧贮槽，更不符合生产实际。这两者设置的大小决定于炼钢生产及其他生产的需要。中压贮罐的设置，是一旦当空分故障能保证一

炉钢吹出的最低需要，贮存1小时生产的氧恐怕是各厂选用的最低值，以便尚有平衡吹停氧的高低峰差，从而降低氧压机后压力以节能。若贮罐大，压力升不上很高时就又开始吹氧。所以，球罐的大小是要在固定投资加运行费用综合找出最佳值来确定的。对于空分设备故障及较长时间的停车以及短暂时间的高峰用量的补充，则是需由液氧蒸发来保证的。液氧贮存最好能在两天以上，冷箱中液氧断无这种可能。相反，依靠冷箱液氧泵加压经主热交换器送出氧的压力，随炼钢吹氧的供和停而波动。这种压力起伏因多炉炼钢的无规律性，对主热交换器温度场的影响不可估计。不稳定的氧(氮)气流对要求稳定的精馏系统是不利的。所以，这种装置对炼钢生产并不一定相宜，却对平稳用氧的炼铁倒是合适。无论如何，这种装置应在进一步了解后才宜决策。其实，现在只用其PSA部分已经可以达到  $80\% \mathrm{O}_2$  的氧气，即可送炼铁吹氧，何需再去ASU中转一趟呢？！

（7）用  $99.6\%$  的纯氧给高炉喷煤供氧的可行性

由于塔板减少所带来的压力降低是较少的一部分，因此用现有或选用双高纯空分设备降低纯度至  $95\%$  或更低操作，是可取之路，既能达到降低能耗的大部效果，又能为统一安排供送提供物质基础[24]。由此，对  $99.6\%$ $\mathrm{O}_2$  的纯氧供高炉喷煤的可行性，亦应进一步与炼铁工作者共同认识。

用  $90\% \sim 95\%$  氧纯度生产，有以下几点不利：

① 不能积存  $99.6\%$  纯液氧，不利于生产的备用与调剂。  
$②$ $90\% \sim 95\%$  氧的压送，除有效氧外，其中氮气需额外能耗，已如前述。  
③ 不能高提取率的提氩，更不要说提其他稀有气体，而炼钢对氩的需要，稀有气体的价值。总之其综合经济效益要比低纯度供

氧大得多。

$④$  特别重要的一点是喷煤新技术——氧煤枪及其安全技术。例如瑞典、德国、日本以及我国都在开发，可达到  $3\%$  富氧、煤比  $140\sim 270\mathrm{kg / t}$  铁。起初的同轴旋流式氧煤枪是煤粉走在中间，氧气走在外环，这样煤粉射流有可能被速度较大的氧气射流所包围而与热风隔开，加热速度受到影响。随后我国又开发了外环走煤、里环走氧的新型氧煤枪[1]。此氧煤枪直接插入高炉炉前风口内 $400\sim 800\mathrm{mm}$  深处。图10、图11可以看出，所供的氧气应是纯氧而不带，氮气最好。技术的发展又回过来要求提供纯氧。这从根本上为高纯氧用于喷煤带来强生命力。

⑤ 熔融还原炼铁亦是决定采用纯氧有利。因此将来钢铁厂内最大制氧机是双高纯制氧机，炼钢亦同，应统一为双高纯。  
⑥ 特别重要的是，如果喷煤采用低纯氧，即不能与炼钢高纯氧互补，如果炼铁或炼钢制氧机一旦故障，就不能借助自动调节手段达到互补，保持平稳供氧，以及共用液

氧系统。

⑦所谓纯氧生产氮气少的问题，现已清楚是指氧纯度低，就可由下塔直接多抽取压力氮，数量当然与纯度有反比倾向。现在已经改为充分利用熵值，宁可先作节流制冷，再由上塔顶可不受氧纯度限制抽取更多氮气再压缩，这就不成为问题了，因为氮压缩机总要设立，并且要供钢厂更多氮气。

以下情况亦是值得参考的：

a. 德士古煤制气用氧亦是可用  $95\%$  或更低的纯度，但为首钢做的技术方案与施工设计都是采用  $99.6\%$  纯氧，主要原因就是带

入的  $5\%$  氮不起好作用及考虑到在钢厂中的互相辅助统筹。只有依靠原有的液氧系统，供氧才有保障。

# 3. 喷煤富氧究竟富到多少为合适

喷煤富氧的富化程度将决定制氧机供氧的多少或者说新上制氧机组的大小，这是一个关键问题。但经过国内外的实践，以下结果是被认识到的：

（1）每增加  $1\%$  富氧量，煤比可增加13 $\sim 18\mathrm{kg / t}$  铁[1][24][28]，高炉可增产  $2.8\% \sim$ $3 \%$  ，煤焦置换比为0.8。

（2）喷煤量的目标是  $150\sim 200\mathrm{kg / t}$  铁，乃至  $250~\mathrm{kg / t}$  铁，已属最大值。这些都可在富氧  $3\% \sim 5\%$  以下做到。表1中已列出1000 $\mathbf{m}^3$  高炉，富氧  $2 \%$  和  $4 \%$  时，年产生铁可达68和172万吨，最大需氧量约需2763及5350 $\mathrm{m}^3 /\mathrm{h}$  （见上期第2页）。

（3）根据这样用氧规模，在很长阶段制

氧机不应考虑富氧至  $30\% \mathrm{O}_2$  以上的预留。试验表明： $31\%$  氧的喷吹，渣铁变稠，风压急剧升高，发生周期性顽固悬料，增产率随浓度升高下降许多[28]。为了进一步表明富氧  $3\% \sim 5\%$  已经足够，请见表9与图13。

4. 富氧氧气是从高炉鼓风机入口供入还是由机后压入好

由于氧气出冷箱略有正压由入口吸入，是完全可以的。日立在水岛钢铁厂的47000 $\mathrm{m^3 / h}$  高炉专用制氧机上采用一个混合装置[8]，即用一个适当的喷嘴，并选取一个吹氧点距风机入口至少  $30~\mathrm{mm}$  ，可以获得氧含量完全均匀的分布。加氧管系向风管垂直进入，插入至风管中心线(或与风管垂直线  $10\sim 15^{\circ}$  夹角)，插入管上布满小孔，并使之和风管中空气流动方向成直角。在鼓风机后实际测定氧含量分布稳定在  $\pm 1\%$  范围内。高炉鼓风机不需进行大的改装，但安全设施有：

$①$  空气过滤器的灭火器。  
② 风机及风管防蚀的氮封（当风机停车时）。  
③ 当富氧比及氧含量不正常时的紧急切断氧气装置。  
④ 各种流量计出现故障时停止供氧。  
⑤ 由于高炉故障，风量下降时停止供应富氧。  
⑥ 当风机脉震时停止供应富氧。  
⑦ 鼓风机的轴封材料，改用  $10\% \mathrm{Ni}$  、 $90\% \mathrm{Cu}$  的CNPI材料，同时将轴封几何形状改变为一种低热阻的形状，使当转子与迷宫密封发生接触的情况下发生的热量最小且散热较快。  
⑧ 将风机及阀门、管道所有涂油全部清除。

显然这些措施还是较麻烦的，我们国内大都没有做到或没有注意到，加上入口供氧存在氧量损失  $10\% \sim 15\%$  [2]。同时吸入氧气要占去一部分鼓风机出力。吸入富氧似乎令人却步，当然好处是省去氧压机。

对于机后加入氧气，大家最可惜的是随着炼钢用氧的加压，常是把高压氧并入高炉风机后低压，失去许多压缩功。其实情况并

不如此。首先不管是专用空分或者专拨一台空分供高炉氧，结果都可用一台氧压机专门为高炉压送。其机后压力是稳定的与高炉风机后空气压力相平衡而略高，不会再是高压供氧，这样可以避免占用高炉鼓风机能力。其次一般配置都是一台氧压机对一套空分装置，不设备用。而氧压机一旦有故障，确实要消耗较长时间检修，这台高炉供氧氧压机实际是一台备用机，届时升至高压转供或兼供炼钢使用。这样就是说设这台对应一套空分的氧压机是不可少的[29]，再者加压后输入高炉可使管道直径大为减小，特别是氧煤枪的使用更需如此。加压输氧为将来普遍采用氧煤枪技术一次到位，亦显得十分必要。用氧煤枪的安全设施及调节控制已经研究成功，属高炉喷煤技术范畴不在这里赘述[1]。

因此压力供氧其实是很妥适的，并且还可借压氧系统与炼钢供氧系统及氧气贮罐、液氧系统统一安排[28]。

# 5. 喷煤用制氧机设置位置讨论

（1）高炉专用制氧机当然应该放在炼铁厂附近，专用目的即在于此。但如果用双高纯制氧机生产纯氧或  $95\% \sim 90\%$  氧供高炉，如前所述，自是应与炼钢用制氧机放在一起，这样做的优点可以归结如下：

① 可以用一套球罐系统及液氧冷备用系统，随时作为炼钢炼铁的备用及补偿高峰用氧，并且完全由计算机自动控制，安全及时，确保钢铁用氧稳定及可靠调节[21、29、30]。管网系统亦最为简捷、灵活，节省基建投资，占地最少。同时，这种备用亦只是不正常时的调节手段，偶一为之（高峰调节大，有利于钢的生产，总效益可更好），谈不上能耗的加大。  
② 管理一体化、科学化。生产、管理、操作及维修人员都是一套班子，协调统一，胸怀全局。而且辅助系统，如供水及循环冷却，供电配电及其备用机组都是最省、最稳

定，不是炼钢炼铁的附属车间，而是独立的化工企业，专业发展，自成一体，职责分明，有利节能及专业技术钻研。在备件及原材料消耗品的准备上，都可备一共用（放在一起容易采用型号相同机械），特别是大零部件，如轴承、转子等，节约，便于管理。

③ 因此，固定资产投资及运行管理费用、能耗的总经济效益最省，最出水平。为此，现使用双高纯装置降低至  $95\% \sim 90\%$  氧纯度生产，虽有小失，但总收益最大。

④ 对回收稀有气体（低纯度亦可略提一些粗稀有气体，供集中提纯）、生产混配气、特气，可以专业化，才能高技术发展；对给城市液体供气，长距离供气才能兼顾，很好开展。这些都因有雄厚的力量作后盾，达到主任务明确，副生产活跃，大有利于国计民生及支持其他行业。

⑤ 有了这样的认识，才能在制氧中心留出足够发展的预留地，才不致仅因没有空地，有碍发展，造成分处另建的遗憾，达不到上述收益。存在决定上述意识，意识反过来影响行动。

所以只要有些观点，不再把炼铁看作炼钢生产剩余氧气的缓冲用户，而是钢铁并重，确保炼铁基本恒定和连续供氧（在确保一定数量基础上还可以吃一些炼钢富余氧），有利于炼铁炉况顺行，成为炼铁技术的基本条件。同样当作主要任务，则这种合中有分、分中有合的辩证方式，从钢铁厂总体来说，才是最好的生产和供氧策略。

（2）但是亦有一些情况是应该考虑炼铁用制氧机单独设在炼铁厂附近的。如果是涉及到利用高炉燃气或燃烧废气热量进而发电的流程，甚至为此还可能因为把制氧机改为中压流程系统[0、10]。但即使这样亦还要设法在这种特殊制氧机管系上接通与钢铁厂制氧中心的联通管，才能共享液氧冷备用，以保证炼铁的可靠稳定用氧。

# 四、制氧机供氧管系的基本模式

上述“三”讨论的观念，体现在设计上首先是体现在供氧系统图上（见图17）。以下是集中了上述所有观点设想的雏议之二。显然，各厂结合本身情况可作因地制宜的变更，但不离其宗的是这种既共同生产又分别供氧的基本模式[20、21、29、30]。流程说明：

（1）此系统是按炼钢炼铁的制氧集中生产，分别供氧的合中有分原则规划设计的。炼铁方面如果采用大制氧机配两台氧压机，亦可分出一台供炼铁，将其中另一台供炼钢。图17阀A即表示多余氧转供其他，还表示当炼铁因故骤然断氧，除炼铁可放散外，亦可因管内压力骤升而开启在不及调整时，阀B亦会放散。

# （2）炼钢系统完全靠自调

关于炼钢供氧系统及液氧贮运系统的设计配置，作者论文[21][30]中已有详细的论述，这里仅扼要说明其运行作用。

① 即当球罐储满压力将超时，一般不是安全阀跳开而是转向打开氧压机前阀C低压放散，并由于流量变化，使空压机微关可调叶片减少空气量。炼铁系统在炼铁用量减少时亦有这样的调整功能。  
$②$  而当炼钢系统用氧量突升时，亦通过使用量及压力的降低至低限而调整可调叶片

③ 只有当炼钢生产规律化后，才能根据规律制定炼钢的调整软件跟踪调整。例如以每小时(约炼两炉钢)为一调整单元时间，炼钢不规律生产，则制氧只有定量生产，造成放散量增大。  
④ 减少低压放散氧率是非常重要的，可以将之转移“放散”至炼铁供氧管系。方式是这样的：阀C旁可并列一组阀C'，接通炼铁氧气低压管。可在阀C未开放之前，先使C'开启，微开将氧气放至炼铁管线内供用，以减少放散率。当需要大放散时，阀C才开。更可以在氧压机机后联通管上，加阀M，当压力过高时，直接自调将本该放散的量放入炼铁制氧机后压力氧管内，既减少放散又收回压力能，M阀和C'阀两者取一即可。

（3）当炼铁制氧机故障停车，则：

① 管内压力突然低至下限，则D阀切断，E阀开启同时动作，即将炼钢系统压出

的氧转供炼铁，而球罐储氧维持炼钢吹炼；与此同时，因管线压力低下，信号使液氧泵自动启动，液氧泵因泵前阀门永远开启着，泵后密封回路始终流通着，即泵永远处在冷状态备用，所以可以马上自动运转。液氧汽化器，如果与液氩汽化器合在一个加热器壳内，则液氧汽化当即进行。蒸汽阀门只是因热负荷加大而开大。即使是单独存在，因为器内常温水存在并温度骤降，亦可立即促使蒸汽阀打开进行汽化。  
② 当这些程序在一分多钟内完成后，E、F阀同时供氧，使压力上升至上限，则反过来关闭E阀，连锁开启D阀。而F阀因维持压力开度仍正常液氧汽化供氧，直至炼铁制氧机恢复供氧，又因再达压力上限而停泵，F阀关闭，完成保证故障供氧任务。  
（4）炼钢系统如果发生故障，球罐压力降至下限，则同样液氧泵自动启动汽化通过G阀补氧。球罐有约1小时储量，完全来得及保证炼钢稳定用量。

（5）即使炼铁制氧机因厂内地域所限，与炼钢制氧机不放在一处，几条联络管线(A、E、F、M阀线)亦可设置。当然生产  $90\% \sim 95\%$  氧，只能靠炼钢系统补偿保生产，炼铁要补炼钢，只能提高氧纯度才能进行，而把纯度调上去，却是需要时间的。只有纯度一致时，遥控打开E阀，氧压机亦会自动升压与炼钢管系平衡，但这样炼铁供氧将自动关闭，避免管道压力过高。反过来，通过E阀反补给炼钢，直到液氧系统发挥作用。

（6）H、I、J阀组系炼钢压力供氧自动调节系统[30]。吹氧时，H阀全开，I、J补充阀用球罐存氧补充高峰缺氧，氧压机后为1.6MPa，停吹时H阀关闭，氧气补入球罐，提高罐压。所以在正常生产时，炼钢压力波动管线与炼铁定压供氧管线是互相隔断的。只有在非常情况时才会自动顺序操作，可以完全由计算机自动控制，亦可用气动仪表自动控制，非人力所能及时做到的。

另外，炼铁空分可选用炼铁专用的制氧机，那怕是分设两地，亦可采用这一管线基本模式。可按  $90\% \sim 95\%$  生产，亦可  $99.6\%$  纯氧生产。管线仍可体现“分合结合一网二系”的供氧，各厂可因地制宜。

# 五、结语

1. 喷煤富氧以富氧率  $3\% \sim 5\%$  、纯度  $90\% \sim 95\%$  为宜，较之纯氧节能较多。在论证与计算之后，提出以双高纯制氧机降低纯度至  $90\% \sim 95\%$  操作运行，是比较达变的。搞炼铁专用制氧机，从全局及经济效益比较后，弊多利少，不如双高纯塔生产低纯灵活。  
2. 制氧机应放在一起，形成钢铁厂制、供氧中心，既有利运行安全，又方便管理，是最节能的。炼铁  $0.5 \sim 0.6 \mathrm{MPa}$  供氧专线，比供鼓风机前更有利于将来新技术的运用（如果吸入状态供氧，亦要将总干管建设成  $1.0 \mathrm{MPa}^{\prime}$ ）压力管线，以应付将来预见不到的变化，这样对管材要求所增不大。

3. 必须对纯氧供炼铁有认识上的准备。因氧煤枪的运用，乃至 COREX 法熔融还原炼铁等发展方向，都会为要求纯氧单独供炼铁，铺平道路。从整体设计来说，以压力纯氧供炼铁亦是很节能的。得出这样的认识来自本文否定之否定的分析，亦是尊重三十多年来富氧炼铁、顶吹炼钢的实践。  
4. 提供一钢铁厂制、供氧配置及管系基本模式，以供各厂因地制宜。

蒸馏(精馏，下同)塔分为板式塔和填料塔。板式塔有泡罩塔、筛板塔、浮阀塔及其变种。填料塔又有规整填料塔和散堆填料塔。一般来说，板式塔的结构较简单、适应性强、宜于放大、造价便宜，但是其效率低、压降高、持液量大；规整填料塔效率高、压降低、持液量小，但成本高。然而在目前运转的塔器设备中，仍以板式塔为主，开发应用的则以现代填料塔为主导。美国著名学者Fair教授认为，最近十多年蒸馏装置的头号新闻就是汽-液接触的填料塔逐步取代板式塔，规整填料作为具有低压降、高传质效率的技术而成为佼佼者。我们认为是符合现代填料塔分离技术进展实际情况的。

# 一、规整填料及其在空分设备中的应用

理想的规整填料有以下优点：（1）压降非常小。填料内汽相在液相膜外成对流传质，不存在塔板清液层的阻力，筛孔阻力小，正常情况下，规整填料塔的阻力通常只有相应筛板塔阻力的  $1 / 5\sim 1 / 6$  ，甚至更小。每块理论筛板塔的阻力通常在  $35\mathrm{mmH_2O}$  左右，而

规整填料可使每块理论塔板的压力降降低到  $3\sim 5\mathrm{mmH_2O}$ ；(2)节能；(3)可减少塔径；(4)液体滞留量少，启动和操作变动较快；(5)分离效率高；(6)操作弹性大，适应性强；(7)产品提取率高。

基于规整填料的上述优点，所以在炼油、化工等领域中早已广泛应用。近十年来已被迅速推广到空分行业中应用。在众多型号规格的规整填料中，要数瑞士苏尔寿公司研制的麦勒派克（Mellapak）规整填料在空分设备中应用得最多。从1984—1993年，苏尔寿公司为德国、美国、日本、意大利、比利时、希腊、荷兰、加拿大等国家的空分设备制造商提供了68台上塔、粗氩塔、精氩塔等的规整填料。其中上塔20台，塔径  $0.85 \sim 4.8\mathrm{m}$ ；粗氩塔25台，塔径  $0.16 \sim 2.85\mathrm{m}$ 。德国林德公司(Linde)、法国液化空气公司(L'AIR)、美国空气制品公司(APCI)和美国实用气体公司(Praxair)等四大公司都从1984—1985年开始研究，把规整填料应用于大型空分设备的上塔和粗氩塔上，1986—1987年用于工

业成套装置，技术上很快达到成熟。Linde公司1990年将此技术应用于  $66000\mathrm{m}^3 /\mathrm{h}$  空分设备，1987—1992年出售的空分设备中填料塔占  $37\%$  （共16套）；APCI公司已有20多套填料塔在世界各地运行和安装，其中包括数套容量为  $50000\mathrm{m}^3 /\mathrm{h}$  和  $63000\mathrm{m}^3 /\mathrm{h}$  空分设备，提供给上海宝钢的  $72000\mathrm{m}^3 /\mathrm{h}$  带全精馏制氩的空分设备正在建设中，它的气氧工况的产氩量为  $2300\mathrm{m}^3 /\mathrm{h}$ ，变工况最低产氩量是 $1200\mathrm{m}^3 /\mathrm{h}$ ，纯度均为  $99.999\%$  Ar  $(\mathrm{O}_2 < 2$  ppm，  $\mathbf{N}_2 < 5$  ppm);L’AIR公司已在40000 $\mathrm{m}^3 /\mathrm{h}$  和  $72200\mathrm{m}^3 /\mathrm{h}$  空分设备上应用；Praexair公司于1988年和1991年分别在35000 $\mathrm{m}^3 /\mathrm{h}$  和  $46600\mathrm{m}^3 /\mathrm{h}$  空分设备上应用成功。

进入90年代后，国际知名的深冷设备公司在上塔和氩塔中以规整填料塔取代了筛板塔，仅对下塔仍保留采用筛板塔。但近来下塔改用规整填料塔也有逐步增加的趋势，例如美国Praxair公司为印度Jinda钢厂建造的当今世界最大的日产2700t氧气的空分设备，其上塔、下塔、氩塔均采用填料塔。

生产气态产品的大型空分设备的上塔采用规整填料后，其压力降只有筛板塔的十分之一，使空气进塔压力降低0.07MPa，因而

可使原料空气的压缩能耗减少  $6.5\%$  左右。规整填料塔的负荷范围大约可在  $30\% \sim 110\%$  之间，而筛板塔的负荷范围只能在  $70\% \sim 100\%$  之间，因此填料塔的设计灵活性要高得多，能使空分设备在很低的部分负荷状态下运转。由于填料塔的液体滞留量很小，因此采用填料塔的空分设备更有利于变负荷工况的生产，对氧气和氮气需求量的变化能迅速作出反应，当需要时，工况转换的时间也可尽量缩短。例如Praxair公司提供的采用填料塔的  $15000\mathrm{m}^3 /\mathrm{h}$  空分设备，从工况I完全转换成工况Ⅱ所需的时间仅为15分钟，而相应的筛板塔则需几小时甚至于十几小时。转换时间的长短当然也与空分设备的自动化控制水平和操作熟练程度有关，但填料塔的固有特性是起了关键性和决定性作用的。

# 三、低温空气分离全精馏制氩工艺

1. 低温空气分离全精馏制氩工艺和设备产生的理论基础和条件

从事低温空气分离设备研究的不少科技人员，为尽力排除常规制氩工艺中的上述缺点，近二十多年来积极探索新材料、新工艺，终于研制成功适用于低温空气分离的低压降而分离效率又很高的规整填料。借助于现代计算手段的不断完善，特别是氧一氩分离所需的理论塔板数的正确获得和高效填料塔的设计成功，再加上分子筛纯化空气（吸附清除  $\mathrm{H}_2\mathrm{O}$  和  $\mathrm{CO}_{2}$  ）工艺早已取代了存有系统压力交变的切换式流程，使精馏系统的压力较为稳定等有利条件，德国林德公司首先在1990年发明了低温空气分离全精馏制氩新工艺流程及设备。

筛孔塔板的最小压力降是受到限制的，即必须维持气泡穿过液相这一个稳定的两相区。为实现这个稳定性所需的最小压降（每块理论筛孔塔板约为  $35\mathrm{mmH_2O}$ ），因为粗氩塔顶部的氩气要被与相应上塔部位压力下蒸发的富氧液空所冷凝，因此，粗氩塔顶部可能达到的最低温度是一定的（不能太低，冷凝器需要保持一定的传热温差），也就是说粗氩塔顶部的压力也是一定的，这个能使粗氩塔有效工作的总压降仅能设置  $40\sim 70$  块理论筛孔塔板数。很显然，采用阻力如此之大的有限筛孔塔板是不可能将蒸发温度靠得很近的氧一氩彻底分离的。然而，利用规整填料可使每块理论填料塔板的压降降低到  $5\sim 15\mathrm{mmH_2O}$ 。在低温空分设备的精馏塔中利用规整填料将产生两种重要结果，它们均可以提高有较高纯度的粗氩的提取率。规整填料用于上塔可以降低整个精馏系统的操作压力，较低的精馏压力可增大各组分的相对挥发度，这一点对上塔提馏段的氩氧分离是最有意义的。

Thorogood 等发现：使用规整填料的精

馏塔系统，较之在可比条件下使用筛孔塔板时的精馏系统，氩的提取率有明显的提高。在同样的理论塔板数情况下，用填料塔可使氩的提取率增大  $5\% \sim 12\%$  ；规整填料用于粗氩塔，可使粗氩塔设置更多的理论塔板数，而且粗氩塔的总压降仍可控制在同样许可的有效范围内，这一措施同样也提高了氩的提取率。

当填料粗氩塔允许设置足够多的理论塔板数之后，就可直接通过粗氩塔生产出含氧量极低的粗氩，这是空分全精馏制氩工艺得以实现的根本所在。图3是相关文献刊登的显示氩中氧浓度与粗氩塔理论塔板数之间的关系曲线。值得提出的是为计算制作这个图表所进行的模拟设计中，上塔总板数、下塔总板数和其他相关参数均保持不变。这个图中各点的氩的总回收率也不变，均为  $94\%$  。

2. 低温空分全精馏制氩流程工艺和设

备简介

欲获得  $\leqslant 2\mathrm{ppmO_2}$  的粗氩，粗氩塔必须有相当多的理论塔板数，方能取代当今用得最多的常规粗氩加氢除氧纯化工艺，这样的粗氩塔势必很高，给塔和精馏塔冷箱的结构设计带来困难。

从分馏塔上塔下部的适当部位抽取气相氯馏分进入粗氯塔I和粗氯塔Ⅱ，冷凝蒸发器装在粗氯塔Ⅱ的顶部，以液空过冷器后引来的液空作冷源，绝大部分的粗氨被冷凝作为塔内回流液，塔内上升气体在具有很大比表面积的规整填料的液膜外呈对流传质。回流入粗氯塔Ⅱ底部的液态粗氯只能靠增压泵增压后输送到粗氯塔I段顶部作为该塔段精馏所必须的回流液。粗氯塔I底部的液体靠位差和自重回入精馏塔上塔的相应部位。正因为塔体很高而且中间还有一台低温液体输送泵，这样一个从氯馏分（含氧  $90\%$  ）通过填料塔精馏，最终在粗氯塔Ⅱ的顶部获得含氧  $< 2 \mathrm{ppm}$  （含氧量减少了约45万倍）的粗氯气

的相平衡建立过程的时间是相当长的，这就是为什么全精馏粗氩塔的启动时间要比以往常规制氩粗氩塔（粗氩含氧  $2.5\%$  ）的启动时间要长得多的根本原因之所在。带有足够余压的无氧粗氩由粗氩塔顶部直接导入精氩塔的中上部，并在精氩塔内进行氩一氮分离，最终在精氩塔的塔釜获得高纯度产品液氩。

塔顶冷凝器以过冷液氮作冷源，塔釜蒸发器通常以下塔的压力氮气作热源，用来蒸发液氩，以满足精氩塔下部精馏所必须的上升气体。总之，全精馏制氩和常规制氩所采用的精氩系统是一样的，所不同的是：

（1）前者放空的余气中不含氢气，因此热质交换更彻底，操作更方便、更安全、更稳定。  
（2）前者塔釜排出的产品氩气纯度更高，质量更好。原因是：

① 前者能连续精馏，粗氩塔不仅清除了氧组分，而且同时也清除了烃类和其它重质杂质。  
② 清除了氮组分以及其它轻质杂质，更不存在氢气。  
③ 由于整个流程工艺过程都封闭在低温冷箱内完成，无外界杂质（如粉尘、水蒸气和其它污染物等）进入，所以产品气体的露点更低，纯度更好。

# 六、结论

1. 空分设备是耗能大户，因此节能是空分发展的中心之一，分馏塔上塔和氩塔应用具有压降非常小而分离效率又非常高的规整填料是世界潮流，势在必行，发展很快。  
2. 填料塔设备的一次投资费用虽然高于筛板塔，但其运行费用（节能和操作维护）比筛板塔要低得多，加上产品提取率提高所得的经济效益，估计运行半年即可回收所增加的设备投资。  
3. 填料塔由于持液量小, 无论在负荷调节范围和负荷调节速度上, 都比筛板塔更为

灵活和快捷，有利于变工况生产及操作。

4. 具有相对压力稳定性的分子筛纯化空气的空分流程的运行推广、低压降而分离效率又很高、价格适宜的空分专用规整填料的试制成功，以及氧一氩分离需用正确理论塔板数的获得并赋之于诸塔器设计的成功等等，是低温空分全精馏制氩流程得以实现的基本先决条件。  
5. 低温空分全精馏制氩流程，是目前技术最先进、运行最可靠、经济效益最高的新型低温空分制氩流程，值得推广，它特别适用于钢铁企业。

# 对实际的空分精馏塔直接进行模拟计算

宋海华

张巍坚

（天津大学化学工程研究所，300072） （杭州制氧机集团有限公司技开处，310004)

摘要 三维非平衡混合池模型，是一种对实际精馏过程直接进行模拟计算的方法。它可以真实而灵活地描述工业精馏塔内存在的液体和气体的各种不规则的流动与混合现象及两相间非理想的传质过程，能够给出精馏塔内两相的组成、温度与流率的三维分布。利用此模型对三套实际的工业空分精馏塔进行的模拟计算获得了很好的效果，证明这是一个有广阔发展前景的新的空分精馏计算方法。

主题词：空气分离 工业精馏塔 精馏模拟 三维非平衡混合池模型

# 一、前言

对空分精馏塔准确地进行模拟计算对于制氧生产设备的设计与操作至关重要。对于一个多元精馏过程进行模拟计算首先要在一定的假设条件下建立一个模型精馏塔，并根据一系列守衡原理和物理、化学定律写出描述精馏过程的数学模型，然后在某些约束条件下进行求解，从而得到气液两相的组成、温度和流率等参数在精馏塔内的分布，以及影响这些参数分布的因素。根据模拟计算的结果，还可以通过改进设计或者改进操作等措施，来进一步达到提高精馏分离能力和降低能量消耗的目的。由于建立模型精馏塔时采用的假设条件不同，大致可以将模拟精馏过程的数学模型分为三类：

# 1. 平衡级模型

平衡级模型，基于两个主要的假设：（1）“全混级”假设：设塔板上的液体和板间的气体完全混合，具有均匀的温度与组成；（2）“理论级”假设：设离开每块塔板的液体和气体混合物之间处于相平衡。满足这两个基本条件的精馏塔板称为平衡级或理论塔板，其所构成的精馏塔称为理论塔。

一般可以利用各个组分的相平衡（E）和

物料衡算(M)方程式，以及焓衡算(H)和组分摩尔分数加和(S)方程式，描述每个平衡级上发生的精馏分离过程。将理论精馏塔内各个平衡级(包括塔顶冷凝器和塔底再沸器)的MESH方程组合在一起即为平衡级精馏模型的数学形式，这是一个非线性方程组，可以采用许多不同的方法求解，如逐板计算法、三对角线矩阵法、牛顿一拉夫森法、松弛法等[1]。利用平衡级模型进行设计计算的基础是理论塔板，能够给出完成一定精馏分离任务所需要的平衡级数或理论塔板数  $N_{T}$ ，而不是实际塔板数  $N_{A}$ ，所以还需要对整个精馏塔或对每一精馏塔段估计出一个总包效率  $E_{T}$  的数值，再将理论塔板数换算为实际塔板数

$$
N _ {A} = N _ {T} / E _ {T} \tag {1}
$$

由于总包效率  $E_{T}$  概括了实际精馏过程中包含的所有的非理想性质，影响因素十分复杂，根本无法准确地得知它的数值，只能凭经验粗略地估计。所以由平衡级精馏模拟计算获得的结果往往与实际生产的情况相差较远，很难满足日益发展的工业生产的要求。

# 2. 非平衡级模型

非平衡级精馏模型，取消了“理论级”假

设，认为模型精馏塔中的每块塔板都是非平衡塔板，即由于气液相间传质过程的非理想性，使得离开一块塔板的气体和液体之间实际上并不处于相平衡。为了计算离开塔板的气相混合物的实际组成，需要考虑表示非平衡传质特点的传质动力学(R)方程[2]，通过求解MERSH方程组，能够得到各个非平衡级上的气液两相的组成、温度和流率。

但是非平衡级模型仍然保留了“全混级”的假设，仅仅对于小直径的塔板这个假设才是近似正确的。而现在大量的试验研究结果已经证明，精馏塔内气体和液体的不规则流动与不均匀混合，对精馏过程有着不可忽略的影响，所以非平衡级模型只适用于小型精馏塔，而不能准确地模拟实际大型工业塔中进行的精馏过程。

# 3. 三维非平衡混合池模型

三维非平衡混合池模型[3]，完全取消了平衡级精馏模型中的“理论级”与“全混级”假设，在充分考虑塔板上气液两相流体的不规则流动与不均匀混合，以及气液两相间的非理想传质的基础上建立的模型，能够真实地模拟工业塔中进行的精馏分离过程。实际应用已经证明，三维非平衡混合池模型，是一个具有很好的发展前景的新的精馏模拟计算方法[4]，下面作详细论述。

# 二、三维非平衡混合池模型

三维非平衡混合池模型的基础是实际的大型精馏塔，它不仅可以反映塔内气液两相间传质过程的非理想性，而且还可以模拟大型塔板上存在着的各种复杂的流动和混合现象，以及塔板间存在的雾沫夹带和漏液等，从而能够逼真地模拟实际的精馏过程。

# 1. 精馏塔内液体和气体的流动与混合

设一个精馏塔有N层塔板，并设每块塔板上的鼓泡液体在竖直方向上完全混合，而在水平方向上则划分成一定数目的二维分布的液体混合池，每个混合池内的液体完全混

合，具有均匀的温度与组成。因为塔板上液体的流动状况十分复杂，所以划分的混合池数目越多，越能够更真实地反映实际情况。但若混合池的数目过多，会大大增加模拟计算的工作量。

对于制氧设备中经常使用的直径  $2\mathrm{m}$  左右的双溢流塔板，将其划分成20个正方形的混合池已能满足计算精度的要求，板间的气体也划分为同样数目的混合池。由于对称性只需对四分之一块塔板进行计算，它包括J $= 5$  个混合池，如图1所示，其中第6号混合池为降液管。

# 2. 气液相间的非平衡传质

设精馏物系为C元混合物，每个液相混合池的鼓泡液体中气液两相间实际进行的传质过程与理想的平衡传质过程之间的偏差一般用点效率  $E_{OGji}(i = 1,\dots ,c;j = 1,\dots ,J)$  表示。利用点效率可以写出离开液相混合池的气相混合物组成与混合池内液相混合物组成间的关系式，即为R方程。

# 3. 模型的数学描述

对于每层塔板上的每个混合池，除了可以写出表示不规则流动一混合特点的F方程和表示非平衡传质特点的R方程之外，还可以写出各组分的相平衡(E)方程与物料衡算(M)方程，以及组分摩尔分数加和(S)方程与焓衡算(H)方程。利用这样的MERFSH方程组可以完整地描述一个混合池内实际发生的精馏分离过程，将全塔各层塔板上所有混合池的MERFSH方程组结合在一起，便为三维非平衡混合池模型的数学形式。

# 4. 数学模型求解

MERFSH方程组中包含大量的变量与参数，大致可以将它们分为三类：

（1）由计算任务指定的工艺条件：包括精馏塔或并馏塔的结构参数，进料的流率，温度，压力与组成，以及精馏物系的有关物性参数，如分子量、临界性质、偏心因子、分子体积等。

（2）可以由适当的公式或经验关联式计

算的参数，主要有：

① 流体力学参数：设精馏塔或并馏塔内各塔板的压力为线性分布，并根据塔器设计中普遍应用的公式计算塔板上鼓泡液体的高度与密度、降液管内液面高度等流体力学参数。  
② 相平衡与热力学参数：利用Peng-Robinson状态方程，通过泡点法计算各混合池内的气一液相平衡常数及气液相混合物的密度与焓。  
③ 描述塔板上液相的不规则流动与非均匀混合特性的参数：大直径圆形塔板上液体的流动与混合状况十分复杂，本文作者曾经在直径  $1.2\mathrm{m}$  和  $2.4\mathrm{m}$  的实验塔中进行冷模实验[6]，得到了一系列经验关联式，可以用来计算塔板上各混合池内纵横两个方向的液体流动速率  $u_{zj}$  和  $u_{wj}(j = 1,2,\dots ,J)$ ，液相的涡流扩散系数  $D_{E}$  和涡流扩散循环流率  $Q_{j}$ ，以及气相的涡流扩散系数  $D_G$  和涡流扩散循环流率  $\phi_j(j = 1,2,\dots ,J + 1)$ 。  
④ 精馏点效率：本文采用伪二组元方法由AIChe模型[6]预测各个液相混合池内各组元的点效率  $E_{OGji}(i = 1,2,3;j = 1,2,\dots ,J)$  。  
（3）需要先设定初值然后在求解过程中由迭代计算逐步逼近其准确数值的变量：MERFSH方程组是一个很大的非线性方程组，不能直接求解，所以要首先设定各层塔板上的液体(或气体)的流率以及各混合池温度的初值将MERF方程组线性化，此线性方程组的系数矩阵为  $-2\mathrm{C}(\mathrm{NJ} + 1)\times 2\mathrm{C}(\mathrm{NJ}$ $+1$  )阶的非对称结构的稀疏方阵。本文采取分块矩阵逐步松弛迭代法求解，得到全塔各混合池内液相(或气相)混合物的组成。然后利用S方程作泡点计算对原先设定的混合池温度进行迭代校正，利用H方程作热量衡算对原先设定的各层塔板上的液体(或气体)流率进行迭代校正，直至迭代校正的偏差小于

允许误差为止。最后能够得到各层实际塔板上气液相两相浓度、温度和流率的二维分布，也就是能够给出整个实际精馏塔内各类精馏过程变量的三维分布状况。

# 四、结论

本文提出的三维非平衡混合池模型，能够灵活、逼真地直接模拟实际的工业精馏过程，利用这个模型对三套空分精馏塔的模拟计算结果，与生产现场的数据十分接近，证明了它的准确性。同时模拟计算的结果，还可以给出气液两相混合物的组成、温度及流率等参数，在塔内各层实际塔板上的分布状况，不但可以使我们更深刻地理解精馏塔内实际发生的精馏过程，而且还可以使我们能够在更可靠的基础上对精馏塔的设计与操作进行改进，从而达到最大限度地提高生产效率与降低能量消耗的目的。

# 二、低纯度空分设备的特点

# 1. 能耗低

由于与常规空分设备相比，低纯度空分设备工艺流程更为简单，取消了纯氮副塔，上下塔板数大为减少，分别为26块和50块，这大大降低了阻力，使得空压机出口排压得以降低，从而达到降低能耗的目的，性能考核时电耗值为  $0.3959\mathrm{kWh / m^3O_2}$  ，比设计值  $0.435\mathrm{kWh / m^3O_2}$  低  $9.02\%$  。

# 2. 纯度不够稳定，压力波动大

引起这个问题的原因是氧纯度越低，气液相氧纯度差越大，见图1。

# 四、环境特点

由于空分设备在有色冶炼行业的应用才刚起步，许多环境条件与其他行业不一样，具体表现在：

# 1. 氧压力、纯度的要求

在我厂，氧主要应用在以下三个方面：

# （1）用于闪速冶炼

即在闪速冶炼时，氧与压缩空气进行合理配比，提高吹炼空气含氧量。目前，铜精矿处理量从原先的  $39\mathrm{t / h}$  上升到80t/h以上，烟气中  $\mathrm{SO}_2$  浓度从  $8\%$  上升到  $11\%$  ，富氧技改工程，使我厂的铜产量从用氧前的7.5万吨升至今年的11万吨，效益十分明显。对氧的要求首先是纯度必须稳定，即控制在95  $\pm 0.5\%$  （这比日本的  $\pm 1\%$  范围还要小）；另外压力波动要小，须控制在  $22\mathrm{kPa}$  左右，最小不能低于  $18\mathrm{kPa}$  ，这对直接从上塔抽取氧产品而没有任何缓冲系统来说，要承受工况、管网波动的影响，有一定难度。

# （2）闪速炉炉前烧铜口时用

闪速炉炉前烧铜口用氧，压力须在1.2MPa以上，富氧投产前系用氧气瓶吹，既不

# 五、技改措施

# 1. 空分工况的调整

根据具体情况，我们主要做了以下工作：

（1）将运行纯度由  $90\%$  提高到  $95\%$  ，以减少主冷液氧和产品氧的纯度差，从而降低液氧纯度波动带来的影响。  
（2）调整TIC1～4四组氧通道和TDIC $40\equiv 1$ 、TDIC  $40\equiv 2$  空气进板式分配阀，确保氧通道的每一大组都有一个通道阀门全开，保证流动阻力最小；两大组空气不致偏流，这样经过调整的正返流气量，能使每组板式负荷相当，冷端温差控制在  $2.8^{\circ}C$  以内。  
（3）全关进膨胀机前旁通，调整中部温度在  $-102^{\circ}C$  左右，使板式冷段负荷不致过大。控制进塔空气中  $\mathrm{CO}_{2}$  含量在  $5\mathrm{ppm}$  以下，同时提高机前温度，实现高温高焓降，这时膨胀后过热度为  $44\mathrm{K}$ ，虽有些大，但因有较大的上塔设计余量作保证，所以不影响精馏工况。  
（4）调节液氮进上塔阀HCV1，气氛出上塔阀HCV2、HCV12、HCV13，将出上塔氮纯度由  $97\%$  提到  $99.9\%$  ，提高了氧提取率。

（5）在保证板式自清除的前提下，逐步将切换时间从3分钟延长到  $6\sim 8$  分钟，减少切换损失以及由于切换对工况的影响，稳定了氧纯度，提高了氧产量。

通过上述手段，  $6500\mathrm{m}^3 /\mathrm{h}$  空分设备的产量提高了  $200\mathrm{m}^3 /\mathrm{h}$  ，提取率由  $89\%$  提高到 $93\%$  。纯度波动范围，由  $95\pm 1\%$  时都不易控制，到现稳定在  $95\pm 0.5\%$  ，也成为一件比较容易做到的事。

# 2. 设备潜力的挖掘

我们编制了一套电算程序，从流程计算和精馏计算两个方面对空分进行分析。我们用实测数据对程序进行校核，发现效果较好，误差在  $2\%$  以内。

用这套程序，我们模拟了  $35000\sim 40000$ $\mathrm{m^3 / h}$  的不同加工空气量情况下，夏季和冬季的运行工况。发现靠增加空气量来提高氧产量的途径是可行的。1992年1月，设备被调整至最佳状况，进口导叶开满，将空气出各级中间冷却器温度尽可能降低，采取了一系列提高气量的手段后，运行结果表明生产 $7800\sim 7900\mathrm{m^3 / h}$  的氧是完全可能的。

随后我们与杭氧联系，委托其重新设计制造符合三元流理论的新转子，1993年年修

投入使用，效果很好，产量增幅在  $300\mathrm{m}^3 /\mathrm{h}$  左右。该转子的设计获杭氧科技进步一等奖，在我厂也因增产明显获得了科技进步三等奖。

# 节流前温度对氧气纯度的影响

张庆强

（郑州瑞卡福医用氧有限公司，郑州高新技术开发区木兰里，450001）

我公司是河南省卫生厅指定的医用氧生产厂家，设备采用杭氧生产的KZON-140/500型空分设备，采用液氧泵内压缩流程，自1996年1月运转以来，整体状况良好。

但分馏塔运行周期末，高压空气节流阀往往发生堵塞现象，造成节流前温度  $\left(\mathrm{TI}_{3}\right)$  急剧下降，虽然经过刮霜，但  $\mathrm{TI}_{3}$  往往难以回到原来的温度，且每经过一次高压空气节流阀  $\left(\mathrm{V}_{1}\right)$  堵塞现象，氧气的纯度都会急剧下降，一般从  $99.7\%$  下降到  $98\%$  左右；如果不及时调节，则纯度达不到规定要求。

开始，我们采用继续关小  $\mathbf{V}_2$ 、 $\mathbf{V}_3$  阀（液空、液氮节流阀）的方法调节纯度，几乎没有效果。后来，我们干脆停下液氧泵，重新进行调纯。这样，虽然最后将纯度调好了，但浪费了时间和能源。

1996年8月21日，我中班接班时，交班人告诉我刚刚发生了一次  $\mathbf{V}_{\uparrow}$  阀堵塞现象，氧气纯度下降到  $99.1\%$  ，当时  $\mathrm{TI}_3$  为-  $175^{\circ}C$  左右。开始我想通过关小氧气放空阀的方法来提高纯度，但效果不明显；后来又采用关小  $\mathbf{V}_{3}$  （液氮节流阀）的方法，仍无效果。

经过分析，我想既然纯度下降是由节-1阀  $(\mathbf{V}_1)$  堵塞、  $\mathrm{TI}_3$  降低引起，那么纯度降低

肯定应与节流前温度  $\mathrm{TI}_3$  有关。从理论上分析：  $\mathrm{TI}_3$  越低，则下塔液空温度也越低，上塔液氧温度也相应降低，则液空、液氧中的氮分子不易蒸发，液氧纯度就降低，氧气纯度也相应降低。

这样一分析，我马上作了试验：将  $\mathbf{V}_{1}$  阀开大1/4周，只见  $\mathrm{TI}_3$  迅速回升，由原来的 $-174^{\circ}C$  回升到-  $154^{\circ}C$  左右保持不变。10分钟后，化验氧气纯度为  $99.5\%$  ，已达到要求值；20分钟后，再化验，氧气纯度达  $99.8\%$  。实践证明了上述理论分析的正确性。

我接着想，是否在一定范围内  $\mathrm{TI}_3$  越高越好呢？仅仅从哲学上讲，答案应该是否定的，事物都有其两重性，受节流前温度影响的不仅仅是纯度这一个因素。照原来的思路分析下去， $\mathrm{TI}_3$  如果升高，则成品液氧的温度也升高，那么液氧就易蒸发和汽化。汽化率高到一定程度，势必影响液氧的产量。所以， $\mathrm{TI}_3$  并非越高越好。

总之，在正常运转工况下，节流前温度以控制在当时压力下相对应的液化温度为宜。

希望兄弟单位在实践中继续验证或提出不同意见。

（1996年9月29日收稿）

# 溴化锂冷水机组应用于空分设备的尝试

孙全海

（南京扬子石化公司烯烃厂空分车间，南京市大厂区，210048）

溴化锂冷水机组属吸收式制冷机类，一般用作中央空调主设备。1995年6月中旬，南京扬子石化公司烯烃厂B套  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备的氟利昂冷冻机(苏尔寿产品)压缩机电机烧坏，一时无法修复。幸好几年前，曾对预冷系统水流路及分子筛纯化器进行过改造，使冷冻机在停车时仍然能维持空分设备运行。其时加工空气量约为设计量的  $70\%$  空气出空冷塔温度为  $28\sim 30^{\circ}C$  ，出分子筛空气中  $\mathrm{CO}_{2}$  含量在周期末时为  $5\mathrm{ppm}$  左右。

“扬子”两套  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备生产的氧气大部分作为乙二醇装置的原料气，氧气产量减少  $3000\mathrm{m}^3 /\mathrm{h}$  ，意味着乙二醇产量相应减少。为避免这种不利状况长时间持续，厂、车间领导决定购置一台由常州能源设备总厂生产的溴化锂冷水机组，并于1995年7月底投入运行，使得B套空分设备恢复了满负荷生产状态。

以水为制冷剂、溴化锂水溶液为吸收剂的溴化锂冷水机组的制冷原理，是利用溴化锂浓溶液在常温下具有强烈吸水性的特性来吸收水蒸气而本身被稀释，稀溶液高温下蒸发出水分而浓缩，然后浓溶液降温恢复吸水能力；水蒸气常温下被溴化锂浓溶液吸收，高温下从溶液中蒸发出来，然后冷却并冷凝成冷剂水。冷剂水低压下蒸发而吸收冷冻水的热量，从而完成制冷循环。本机组采用0.6MPa蒸汽为热源，无压缩机，耗电部分主要是几个小功率的循环泵，一般只能得到  $5\%$  以上（通常为  $7\%$  )的冷冻水。

机组投运后的主要问题，是真空度难维

持。水只有在绝对压力为  $874\mathrm{Pa}$  的低压下才能于  $5^{\circ}C$  沸腾，不凝性气体的渗入将破坏吸收过程，从而使冷冻水温度升高。溴化锂冷水机组的隔膜阀结构简单，由于隔膜将阀杆与阀体隔开，密封性好；但阀门开关次数多了或用力过猛，隔膜易产生裂纹，从而空气渗入。需要经常排除不凝性气体或启动真空泵抽真空，严重时只得临时将溴冷机停下更换损坏的阀门。

在机组运行进入1995年10月份后，经常出现吸收器液位突然下降，而高压发生器液位突然上升，跟着冷冻水温大幅度上升的现象。笔者经仔细观察与分析，认为这种现象是浓溶液在回吸收器管路中结晶所引起的。溶液是7月份调配的，由于当时冷却水温及环境气温较高，故浓溶液在回吸收器的管路中流动时温度较高，此时即使溶液浓度较大也不容易结晶。而到了10月份后，水温和气温显著下降，回吸收器管路的浓溶液温度也明显下降，此时如果溶液浓度较大，就会部分结晶。

结晶体阻塞吸收泵前过滤器，使泵前管路阻力增大，至吸收泵严重汽蚀而不能正常工作时，吸收器液位就会突然下降，而高压发生器及低压发生器液位升高。低压发生器上虽设置有自动溶晶溢流管，但结晶体溶解毕竟需要一定的时间（2～3分钟）。而在此期间往往由于高压发生器液位过高，部分溶液已经从蒸汽管路进入了冷剂水中而使冷剂水中混有溴化锂，受污染的冷剂水其沸点升高，饱和蒸汽压下降，冷剂水不容易蒸发，

冷冻水温也就升高了。此时需要将冷剂水作旁通处理，然后重新调整，冷冻水温才能恢复正常。经笔者建议，在往溶液中加4桶（约80升）水后，这种现象就再也没有出现过。

总的来说，本溴冷机在运行稳定可靠性上，尚比不上原先的氟利昂冷冻机，这当然与自控系统一直没投用，及操作员熟练性有关，但隔膜阀易损坏也是个重要因素。1996年3月初，原冷冻机电机修复投用，溴冷机便停下作为备用冷冻机。

笔者认为，虽然氟利昂作为蒸汽压缩式制冷机的优良制冷剂已使用多年，但由于其对大气臭氧层的破坏，国际公约正逐步削减其使用量，在合适的代用品一时难以找到的情况下，近来在空调领域发展很快的溴化锂吸收式制冷机，如能克服自身的一些缺点，则有可能在未来的空分领域中，占据一席之地。

（1996年5月初稿，9月改稿）

# 一、技术性能对比、测定分析

NR-28型空分设备主要设计指标为：加工空气量  $28000\mathrm{m}^3 /\mathrm{h}$  ，氧气产量  $5000\mathrm{m}^3 /\mathrm{h}$  氮气产量  $5500\mathrm{m}^3 /\mathrm{h}$  ，氧气纯度  $99.6\%$  ，氮气纯度  $99.9\%$  ，上塔压力  $0.04\sim 0.05\mathrm{MPa}$  下塔压力  $0.40\sim 0.48\mathrm{MPa}$  ，板式冷端温度 $-168\sim -172^{\circ}C$  ，板式中部温度  $-110\sim$ $-130^{\circ}C_{\circ}$

运行至1995年9月时，针对上述参数我们又进行了测定：

加工空气量  $22000\mathrm{m}^3 /\mathrm{h}$  ，氧气产量3200 $\mathrm{m^3 / h}$  ，氮气产量  $3700\mathrm{m^3 / h}$  ，氧气纯度 $97.0\%$  ，氮气纯度  $99.9\%$  ，上塔压力0.047MPa，下塔压力0.44MPa，板式冷端温度 $-180^{\circ}C$  ，板式中部温度  $-150^{\circ}C_{\circ}$

可见，该装置存在严重缺陷，已经威胁到我厂合成氨系统的正常生产。针对这一现状，我们对  $5000\mathrm{m}^3 /\mathrm{h}$  空分设备进行了大量

的工艺测定和数据分析及物料衡算。

# 1. 板式系统测定

1994年10月21日、22日测定，测点AE-12（污氮出液化器、进板式冷端前)与  $\mathbf{B}_6,\mathbf{B}_7$  产 $\mathrm{B_8,B_9}$  （止逆阀箱V1、V2、V3、V4后测点），其结果基本无明显变化，从而断定止逆阀箱没有泄漏。但第一组板式是否泄漏并不明显，后经1995年7月～9月连续测定，得出结论：第一组板式无泄漏（该组4台板式，均系我厂提供工艺参数，杭氧制作，并于1989年更新）。

但是从所测数据看出，第二组板式有明显泄漏，且呈有一定的规律，即R4通道走空气、R3走污氮时，从R4中漏入污氮通道的空气量较大。当R3通道走空气、R4通道走污氮时，从R3中漏入污氮通道的空气量较少，证明R4的泄漏较R3严重。

在1995年的停车检修中，经检查发现第二组4台板式换热器表面均出现不同程度的鼓包，尤其R4的两台板式最为严重，这与当初的测定结果完全吻合。

# 2. 过冷器系统的测定

尽管  $\mathrm{NR} = 28$  型空分设备出气能力，已经严重不足，但从出分馏塔的纯氮气来看，并

没有变化。故证明液空过冷器和纯氮液化器并没有泄漏。而液氮过冷器若出现泄漏，则只能是高压侧的纯液氮漏入低压侧的污氮之中。为此，我们针对液氮过冷器，自1994年11月起进行了连续性的几次测定。

经测定与计算，证明液氮过冷器也有相当严重的泄漏量，在1995年10月进行的检修中也证明了这一点，在总计137根  $\phi 10\times 1\mathrm{mm}$  的蛇形肋管中，共发现4根漏管，与工艺测定的结果完全一致。

# 3. 液化器系统的检测

与纯氮液化器相同，出板式的氧气与从主冷上部抽出的氧气对此分析可以看出，氧液化器也不存在泄漏。而污氮液化器的工况，可从AE-11（污氮出过冷器纯度），AE-12(污氮出液化器纯度)清楚的看到，该液化器也不存在泄漏。

# 4. 问题分析

该空分设备为什么会出现严重的出力不足呢？从以上的测定结果可以看出：由于板式换热器的泄漏，正流空气的一部分在尚未进入精馏系统时，已漏入返流的污氮通道中而被放空，致使进入精馏系统的空气量大大减少，进而影响整个精馏工况，导致氧、氮产量降低。再加上液氮过冷器严重泄漏，使作为上塔洗涤用的液氮量大大降低，从而直接影响到上塔精馏系统的正常运行。

这样，进一步降低了氧、氮产量，也波及到了氧纯度的下降。同时由于污氮过冷器的泄漏，进而导致污氮液化器的工况变差，致使板式冷端温度过低和部分空气提前液化。又因板式泄漏，加上冷端温度过低，致使板式中部温度下降至  $-150^{\circ}\mathrm{C}$ ，而中部温度过冷，反过来又影响到板式冷端温度。这样，就造成了板式系统的恶性循环，最终也导致了空分设备的出力不足。

# 二、主要改造项目

1. 空压机供油系统

原供油系统中设有油压槽，它是为防止突然停电断油造成转子轴瓦缺油被烧而设置的。但多年来，由于油压槽空气密封不好而泄漏，且经常充油满罐，加上油压槽又安置在一楼，根本起不到应有的作用。为此在厂房近10米高的空中靠墙安装了一台同体积的高位油箱，油路和供给空压机的油路出入口并联，从而代替了原有的压力油箱，也轻松地解决了一旦停电，便有足够的润滑油供给靠惯性运转的空压机。

# 2. 空气吸入口呢绒袋过滤器

NR-28型空分设备的原配空压机吸入口为呢绒袋过滤器除尘系统，由于空压机长期运转，致使靠近吸入口的呢绒袋向吸入口伸展，呢绒袋松驰，虽经紧固，但运行一段时间后仍发生偏离，直接影响到反吹风环的上下移动。由于呢绒袋阻力的增加，只好从人孔进去，人为地用木棒敲打，以减少阻力。这样，就无法保证空压机的安全正常运行。为此，采用了在呢绒袋内加紧固钢丝或铜筋的办法，得到了良好的效果。

# 3. 板式换热器

NR-28型空分设备内置两组共8台板翅式换热器(简称板式)，1989年由于第一组板式出现泄漏现象，随之在杭氧订制了4台板式，并在当年对第一组板式全部进行了更新。第二组则利用第一组中较好的板式进行了调整更换。虽新旧板式几何尺寸稍有差异（新板式冷端较原旧板式冷端长  $400\mathrm{mm}$ ），但由于均无泄漏现象，故开车后整套装置运行基本正常。随着我厂二期扩建工程的逐步投产，加上第二组板式(旧的)换热器和液氮过冷器的泄漏， $5000\mathrm{m}^3/\mathrm{h}$  空分设备的缺陷随之也接踵而来。1994年我厂决定又从杭氧订做了4台板式，几何尺寸与1989年更新的完全相同。在兰州化学工业公司化肥厂等兄弟单位的配合下，于1995年10月完成了改造。从而也彻底改变了由于板式几何尺寸不一，中部温度

不易控制的现状。目前该装置已安全运行近一年，尚未有任何问题暴露出来。

# 4. 空压机出口处补加空气量

由于空压机的长期运行，加上设备老化，冷却效果差，接触式迷宫密封所存在的缺陷，使空压机输气量随之下降，使空分设备的正常运行受到严重影响。为此，我厂采用了给空压机出口补加空气的办法来解决。即用2TY型透平压缩机(原设计为压氧气用)压空气补入原配空压机出口，从而来提高进分馏塔的加工空气量。在1995年下半年的并网运行中，由于空气量增加了近  $5000\mathrm{m}^3 /\mathrm{h}$  ，才得以使  $5000\mathrm{m}^3 /\mathrm{h}$  空分设备维持正常生产。

但2TY型压缩机补气能力毕竟有限，目前在提高加工空气量方面，我厂已着手准备进行第二次大的改造。即新增一台DH-63型空气透平压缩机，并将  $5000\mathrm{m}^3 /\mathrm{h}$  空分设备

原配空压机的迷宫密封系统进行改造，这样一方面可解决空压机的备机问题，保证空分设备正常的加工空气量，同时也为空分设备安全长周期运行提供有力的保障。

# 三、改造后的运行效果

从改造后的运行效果来看，由于空压机设置了高位油箱，再未出现原压力油槽漏油污染和浪费的事故，同时也基本杜绝了一旦断电造成压缩机缺油事故的发生。仅此一项，所节约的润滑油和减少设备检修频次带来的经济效益，就相当可观。若加上设备改造和缺陷的消除所带来的经济效益，就更加明显。

1995年10月，随着2TY型压缩机补气工作的实施，使空分设备的各项经济技术指标均得到提高，并达到或部分超过了原设计能力，经济效益明显。

# DH-01型微量氧分析仪常见故障分析及防范

杨滨华

（哈尔滨制氧机厂计测处，黑龙江省哈尔滨市南岗区哈西大街109号，150080）

DH-01型微量氧分析仪是基于对氧敏感的  $\mathrm{Ag - Pb}$  碱性原电池的工作原理，对空分产品的氮气、氩气中的微量氧进行连续自动分析的仪器。但该分析仪对应用技术要求较高，当操作失误时会影响分析结果的准确度。以下结合我厂的实践，分析常见故障产生的原因，并提出为防止这些故障发生应采取的措施。

# 一、空气渗透产生的原因及防范措施

通常，在分析仪器的工作环境中，含有 $21\%$  的氧，而待分析气样中的氧浓度仅为几个ppm。由于两者间的浓度差较大，当所采用的取样管防渗透能力较差时，外界氧分子将通过取样管壁渗透到待分析的气样中，从而导致分析结果增高。

例如：在测量瓶装纯氮气中的微量氧含量时，当采用  $\phi 9\times 1.5\mathrm{mm}$  、  $\mathbf{L} = 1.5\mathrm{m}$  的乳胶管取样，分析仪指示的氧含量为22ppm；当改用  $\phi 6\times 1\mathrm{mm}$  、  $\mathbf{L} = 1.5\mathrm{m}$  的紫铜管取样，分析仪指示的氧含量为8ppm；当采用比色分析，3次分析结果的平均氧含量为6ppm。

该实验结果表明： $\mathbf{L} = 1.5 \mathrm{~m}$  的乳胶管渗透了  $14 \mathrm{ppm}$  的氧。对气体中微量氧的分析，渗透是客观存在的一个重要问题。要解决这一问题，首先取样管应采用防渗透性能良好的金属管。另外分析仪器的安装位置距取样点越近越好，这样可缩短取样管的长度，减少误差来源，保证分析结果的准确。

# 二、浮子流量计中浮子下移产生的原因及防范措施

在分析仪器使用过程中，当流程气样很

稳定时，浮子流量计中的浮子不稳且不断下移。通过分析，产生这种现象的原因，是气样经氢氧化钾溶液洗涤后粘度较大，在通过干燥器时带走了部分硅胶粉末。当气体流经针形阀时，这些粉末便聚集在针形阀门的阀口处，使阀口变小，气阻变大，流量逐渐变小，所以浮子就慢慢下移。

为防止该现象出现，可在向干燥器加硅胶之前，先将硅胶过筛分离出粉末，然后再装入干燥器。此外在干燥器出口处装入脱脂棉或脱脂纱布进行过滤，可避免上述故障的发生。

# 三、氢氧电解池中电极上黑色沉积物产生的原因及防范措施

在分析仪器使用过程中，有时会出现电极(尤其是负极)逐渐变黑，并有黑色沉积物沉积到电极上或脱落到溶液中的现象。该现象的出现，会影响氢氧电解池的正常工作。产生该现象的原因，是在配制氢氧化钾溶液时，氢氧化钾或蒸馏水不纯，导致所配成的氢氧化钾溶液含有杂质，而这些杂质在电解过程中会沉积到电极表面形成黑色沉积物。电极上沉积物的形成，会影响电极的导电性能，导致氢氧电解池的电解电流不稳定，必须经常校对，从而给正常分析带来麻烦。

为防止该现象发生，应在配制氢氧化钾溶液时，采用高纯度（GR或AR）的氢氧化钾和二次蒸馏水或离子交换水。还应对配制好的氢氧化钾溶液进行预处理，即先用备用电极电解氢氧化钾溶液，待溶液中杂质沉积再将溶液过滤后装入电解池内。

# 四、溶液倒流产生的原因及防范措施

分析仪器停机后，发现洗涤瓶及液封瓶中氢氧化钾液体的液位发生变化，氢氧化钾溶液倒流至电磁阀及燃烧室内。当电磁阀受到氢氧化钾溶液腐蚀，以及钯触媒被氢氧化钾溶液浸池时，分析仪器便无法正常工作。发生该现象的原因是停机后，传送器内部温度变化，导致体系内残留气体压力发生变化，体系内压力不平衡致使体系内溶液倒流。

防止上述现象发生的措施，是先停机、后停气。即在停止分析前，先将分析仪的电源关闭，使传送器内加热停止，被分析气体仍通过传送器。待传送器内温度降至室温，

体系内温度及压力平衡后再停气。

最好有一备用气源（钢瓶装的高纯氮或氩气），当空分设备紧急停车时，可用备用气源向分析仪供气，然后关闭分析仪电源，待传送器内温度降至室温后，再关闭备用气源。最好封闭仪器的出口，因空分设备紧急停车也常是造成分析仪中溶液倒灌的原因。

以上论述了我厂在使用  $\mathrm{DH} \equiv 01$  型微量氧分析仪过程中遇到的故障，以及为防止这些故障所采取的措施。实践证明，只要正确地使用、维护该分析仪，就能够准确检测空分产品氮气或氩气中微量氧的含量。

# 防止氧气瓶燃烧爆炸充装法

杨光军

（深圳中宏工业气体有限公司，广东深圳蛇口工业区淞湖路，518067）

摘要 简述了氧气瓶燃烧爆炸的原因与防范措施：用便携式可燃气体检测仪对余压气瓶逐只检测后充装，对无余压气瓶经采用技术措施后亦进行可燃气体检测，结果发现隐患瓶比例大大上升，保障了气瓶的安全充装，效果极佳。

主题词：氧气瓶 燃烧爆炸 原因 可燃气体检测 安全技术

# 一、问题与原因

由于氧气瓶燃烧爆炸造成人员伤亡，厂房严重破坏，给气瓶充装和使用单位造成严重损失，此类恶性事故无论在国外还是国内都已非鲜为人知。且此类事故在国内有上升的趋势，在行业内造成很大的影响。

对于在我国国内以及国外氧气瓶着火爆炸事故大致可分为三种情况：1.在充装过程中燃烧爆炸；2.在贮运、使用过程中燃烧爆炸；3.在充装过程中燃烧着火。

这三种情况第1、2种情况一旦发生均多为恶性事故，机毁人亡，摧毁厂房，损失严重。能够引起氧气瓶爆炸的主要原因，是瓶内混有可燃气体，且可燃气体浓度达到或超过爆炸下限，不论是否受到外界能量的激发，在充装使用操作中极容易发生燃烧爆炸事故。从国内外有关严重恶性爆炸事故的报道来看，钢瓶内混有氢气造成燃烧爆炸的事故较多，损失最大而且多发生在充装现场。

对于第3种情况，这种情况发生的频率较恶性爆炸多，但造成的伤亡和损失较轻，主要原因是瓶内虽混有可燃气体但量较少，且大部分是由于瓶阀内含有少量油脂、油棉纱、橡胶垫等少量可燃或易燃质造成。

对于能够引起氧气瓶燃烧、爆炸的机

理，大家都已非常明白，但能够产生满足燃烧爆炸条件的根本原因有下列几个方面：

1. 虽然在安全管理上注重了气瓶检验、充装，严格按《气瓶安全监察规程》管理气瓶，但在实际运作中还存在很大的差距和不足。  
2. 行业内竞争激烈，气瓶管理、充装存在“重效益轻安全”的倾向，造成气瓶检验、充装管理不严，出现混装现象。  
3. 行业外使用单位和使用者对气瓶安全使用管理不善，欠缺气瓶安全使用常识，违章、无证操作现象严重，留下隐患。  
4. 现有的管理、操作条件难以杜绝和排除事故隐患。

由于以上诸多原因对混有可燃气体或可燃物质的氧气瓶缺少有效的管理控制和安全措施，将其拒之“门”外，发生燃烧，爆炸事故就很难杜绝和避免了。

# 二、措施与结果

如何在氧气瓶充装过程中避免和杜绝燃烧、爆炸事故发生，多年来在我公司一直是管理部十分注意的一项日常管理工作。为保证安全充装，首先我们严格、认真地执行了《气瓶安全监察规程》，加强气瓶管理和充装前的检查；收集国内外气瓶安全事故报道，组织员工学习，吸取经验教训，提高员工安

全警惕和自觉性。但仍出现个别氧气瓶在充装结束时或结束后发生着火事故，以致烧伤或烧毁瓶阀或工具。对员工安全造成威胁，在充装操作时心中没底，存在不安全感——瓶内余气是否安全？不知道。

在氧气瓶的充装过程中，能否保证安全充装、安全贮运和使用，只凭简单的行政或形式管理是不够的，既然已明了引起氧气瓶燃烧爆炸的机理原因，能否判断和查找出每一只回充装站的氧气空瓶，其余气是否混有可燃气体，或瓶内是否错充过可燃气体，若能逐一查找并剔出含有可燃气体的隐患瓶，这样才能完全避免和杜绝燃烧爆炸恶性事故的发生。

为保证安全充装，我们首先采取国外安全充装技术要求，对每只氧气瓶在正式充装前象充装高纯气体一样，放空余气、抽真空后再进行充装氧气。但这样做对混有可燃气体的钢瓶，在放空时仍避免不了由摩擦而引起的燃烧爆炸事故。为要对每个空氧气瓶内余气到底是否安全探个明白——瓶内余气是否有可燃气体，寻找一个有效的检查手段和措施，于是我们购置了一种便携式可燃气体检测仪。其检测气体测试范围、响应时间、报警方式及重量均较理想。于1995年4月我们正式将该可燃气体检测仪投入到氧气瓶安全充装程序上。对每只氧气瓶在未连接到充装台之前进行逐瓶检测，效果极佳。该报警仪对诸如氢、乙炔、一氧化碳、烷、醇、苯、酮等可燃气体均能声、光报警，报警设定值可在爆炸下限内任意设定。

在实际使用中，自1995年4月至1996年4月我们一直坚持做到充装前检查一可燃气体检测仪检测一放空一抽真空一充装等这种安全充装程序，逐瓶检测，发现瓶内含有可燃气体的隐患瓶约占四万分之一。对发现的隐患瓶做好记录，立即剔出、放空、灌水处理后再投入充装和使用。在我们发现含有可

燃气体的隐患瓶中，都是具有余压的氧气空瓶，而对无余压的氧气空瓶，可燃气体检测仪就无法检测，那么这些瓶就不能一定保证安全充装和使用。

1996年5月份，我们采取技术措施对这些无余压空瓶也能够进行可燃气体检测，结果发现含有可燃气体的隐患瓶的比例由四万分之一上升到五千分之一，令我们非常震惊，因此对我们充装操作工来讲，在未做这项可燃气体检测之前每充装五千只氧气瓶就有可能遇到一次最小是着火的事故隐患（对是否一定发生着火或爆炸事故，还要取决于瓶内可燃气体的浓度、充装速度、瓶阀阀芯的松紧程度及瓶阀开关速度等因素，能否引起着火、燃烧或爆炸）。从我们目前发现的这些事故隐患来看，瓶内的可燃气体主要是乙炔，乙炔主要是由使用者在操作中不慎迫使乙炔返流到氧气瓶内，特别是余压越低者，含有可燃气体——乙炔的瓶越多，比例越大。

# 三、结束语

上述这种充装方法在我们公司已列入氧气充装程序——安全操作规程中去，形成了制度，操作工深感若不认真执行这种充装程序便有不安全感，因为隐患就在身边。执行这种充装程序对稳定员工生产情绪和热情起到积极作用，同时也提高了产品质量，最主要的是保证了充装贮运和使用安全。

对于这种充装方法能  $100\%$  的对氧气瓶进行检测，能否  $100\%$  的保证安全，杜绝和避免氧气瓶发生燃烧爆炸呢？对于这一问题还有待我们进一步研究、解决，因为对于不含有可燃气体，而只含有油脂、油棉纱、橡胶等，非正常填充的可燃物质，检测仪是测不出来的，对这些隐患排除还需我们在充装前的检查中认真检查，努力工作。

以上所谈，若有不妥，请同行批评指正。

# 主题词：重有色金属 富氧 冶炼 应用 空分设备 变压吸附制氧 选型

富氧应用于重有色金属冶炼始于50年代。由于重有色金属生产方式中火法冶金占生产总量一半以上，所以具有非常重要的地位。如铜产量的  $80\%$  、铅产量的  $100\%$  、锌产量的  $40\%$  、镍产量的  $90\%$  的、锡产量的  $100\%$  ，均来自火法冶金。火法冶金具有悠久的历史，并具有以下特点：在操作温度为  $800\sim 1600^{\circ}C$  时，化学反应速度快，能用温度的变化来改变反应的平衡。有色金属硫化物矿中的硫和铁氧化时可释放大量的热，从而代替矿物燃料，即所谓自热熔炼。火法冶金的物料中含有高浓度的金属富集，熔融金属与熔渣不互熔。冶金渣在自然界中产生大量含尘及含腐蚀性气体的烟气，操作条件差。

重有色金属火法冶炼过程中，必须使用多种冶金设备，计有闪速炉、转炉、烟化炉、焙烧炉、鼓风炉、熔炼炉、旋涡炉、回转窑、竖罐及各种反射炉等，在不同的重有色冶炼厂不同的工艺流程中它们各有其用途。例如用沸腾焙烧炉进行铅锌精矿、硫铁矿、锡精矿的焙烧获取焙砂。用鼓风炉冶炼铜、铅锌和镍的精矿以制取粗金属或冰铜。用转炉进

行冰铜或低冰镍的处理以取得粗铜或高冰镍。闪速炉用于处理铜、镍等硫精矿进行脱硫，并产出高品位的冰铜。反射炉熔炼主要用于铜冶炼及锡冶炼。

由于火法冶金具有单位生产能力高及以上各项特点，所以今后可能继续保持这种主导地位。但是由于当今世界上能源的紧缺以及价格的上涨，加上精矿的价格提高，环境保护政策标准越来越严，促使现代火法冶金技术大大减少必须净化处理的烟气量及漏烟量，尽可能利用矿料自热进行熔炼，尽可能全面地回收余热，降低总能耗。同时满足越来越严格的环境保护标准要求，进一步提高经济效益。为此研究发展了多种火法冶金工艺流程及技术措施。其中一个重要的措施，就是最大限度地使用工业纯氧或富氧以代替空气进行熔炼。由于富氧燃烧提高了炉温，强化了熔炼过程，同时少了鼓入的氮气，就降低了能耗，减少了烟气量，并提高了烟气中  $\mathrm{SO}_2$  的浓度，可多生产硫酸。关于这方面实验数据及生产上实测纪录，国内外杂志都有报道。

近30年来，火法冶金方面研究出的新方法，主要有熔池富氧熔炼、基夫赛特法、Q.S.L.法、诺兰达法、三菱法等。有一个共同的方面，就是鼓入富氧或工业纯氧。下面摘录有关国内外重有色冶炼厂方面的报道，便可以说明这一些。这些企业应用富氧之后，均取得了较好的综合经济效益。如日本古河矿业公司足尾冶炼厂1978年开始在闪速炉熔炼中使用富氧，一般富氧浓度达  $40\% \sim 42\%$  ，生产能力提高  $30\%$  左右，在实际生产中富氧浓度为  $35\% \sim 90\%$  。表1为足尾冶炼厂富氧冶炼与空气冶炼的比较资料。

对于在重有色火法冶金中采用富氧和工业纯氧的经济评价，资料上时有介绍，但较为粗略。在设计院为具体工程所作可行性研究报告中，分析较为具体。由于重有色冶金行业各个工厂情况各不相同，要根据具体的工艺条件、生产情况、运行工况才可进行计算，其工作量很大。我们可以从以下几方面来分析这个问题：采用富氧熔炼后生产能力均有所提高（如闪速炉熔炼提高了  $30\%$ ），熔炼总能耗下降，排烟量及电耗下降，烟气中  $\mathrm{SO}_2$  浓度提高后硫酸产量增加。以上各项总的收入与制氧成本之差就构成了应用富氧之后所增加的额外经济效益，一般情况下所取得的经济效益是可观的。但我们仍要看到，在我国重有色冶金工厂应用富氧或工业纯氧还是不够的，主要是由于制氧装置设备费用高。但随着矿料价格不断上涨，环境保护标

准越来越严格，促使重有色冶金企业去寻求可以提高生产能力、降低燃料消耗、提高矿料中金属回收率以及减轻环境污染，更加适应品位越来越低的矿料的火法冶金的炉型和工艺。火法冶金应用富氧，才成为提高经济效益减少环境污染的重要措施之一。

在80年代，重有色冶炼企业所设计的制氧站均为深冷精馏装置，其投资有时占到总投资的十分之一左右，企业筹措资金困难，因此难以建成投入使用。90年代，由于国内空分行业研制成功变压吸附制氧装置（PSA和VPSA），从而为现代火法冶金应用富氧在我国的发展创造了极为有利的前景。设备费为同型号深冷精馏制氧装置的  $70\%$  ，土建费用也大为降低。由于变压吸附的塔体在露天布置，所以厂房高度降低了不少，当然造价也就降低了。更重要的是大为降低了制氧成本，变压吸附装置电耗  $0.45\mathrm{kWh / m^3O_2}$  而深冷精馏装置电耗  $0.75\mathrm{kWh / m^3O_2}$  （中小型)，这意味着在制氧成本上前者每立方米氧气便宜0.15元。

重有色冶炼火法冶金主要应用浓度  $24\% \sim 90\%$  的富氧，所用制氧装置产量在3000 $\mathrm{m^3 / h}$  以下居多，正好在变压吸附装置的最佳参数范围，此时氧浓度可达  $93\%$  。如果用量

再大可以选用多套，当然是否合理必须经技术经济方案比较后定。加之操作简单，设备启停方便，所以我们在进行制氧装置选型时应优先选用变压吸附制氧装置。当然对于用氧量太大且纯度要求在  $95\%$  以上时，可以选用深冷精馏制氧装置或变压吸附与深冷精馏联合装置（后者国外还在研究之中）。

重有色冶炼企业环境条件差，这是选用变压吸附制氧装置应予以注意的。火法冶金烟气中含尘及  $\mathrm{SO}_2$  浓度较高，特别是事故排放时更是如此。查实测资料，有个冶炼厂  $\mathrm{SO}_2$  最大落地浓度可达  $14.64\mathrm{mg / m^3}$  。曾经有用户反映分子筛在  $\mathrm{SO}_2$  的影响下，使用仅一年便需更换。分子筛对于抗  $\mathrm{SO}_2$  的能力还需制造厂确定，或者在设计变压吸附装置时，有预处理措施。

# 制氧工问题解答（四则）

1. 问：我厂有一套  $150 \mathrm{~m}^{3} / \mathrm{h}$  制氧机，配套  $3.0 \mathrm{MPa}$  无润滑空压机和两台透平膨胀机，加温后重新开车，发现节-4阀难关，液氯液面逐渐下降，产量随而降低。请问原因何在？应如何处理？

答：经查分馏塔，氧氮出口管有“冒汗”现象，透平膨胀机过桥有结霜现象。由于配套空压机最高压力为  $3.0\mathrm{MPa}$ ，已经无法提高。这种塔对气候特别敏感，防漏即成为是首要的，因此建议把珠光砂内气体引出进行分析，结果漏出气体尚无大碍；若发现是液体少量外泄，就得停车维修了。冷量不平衡是液面下跌的主要原因。

根据操作记录，各项主要指标如压力、温度、热端温差等都较正常，从节一4阀难关及  $\mathbf{T}_{3}$  为  $-152^{\circ}C$  ，可以断定下塔过热了。为“拉牢”这个塔的工况，工人们确实动了脑筋，液面只能让它少跌或拉牢，但却无济于事。从总体来说，违反了操作原则，即分馏塔的调节，必须在液面稳定下进行，其变动是有一定限度的。目前这种操作方法，原则上说是错误的，最终是把塔开“僵”了。

这只塔的产冷量后备不足，在高温季节表现尤为明显，塔的压力已达到2.8MPa，产冷潜力耗光了，为此必须在维修上下功夫，春秋冬季时节，就要注意防漏，同时加强操作培训，使冷量损失  $(q_{2} + q_{3})$  最少，这就是操作秘诀所在。

这种塔的病症，我们称为“气候病”，采取适当防暑降温措施，也是一种治标方法。

2. 问：节流阀和膨胀机的区别，前者是等焓节流，后者为等熵膨胀，除具有节流作

用外还有膨胀对外做功。请问膨胀机工作时，如果把驱动风机（或驱动电机）拿掉，使气体对外不做功，那么膨胀机是不是变成和节流阀一样，有同样的功效呢？（天津学员问）

答：节流阀和膨胀机是工业上常用的两种获得低温的方法和设备，同样是气体绝热膨胀使温度降低，区别在于节流膨胀的温降及单位制冷量，都小于膨胀机绝热等熵膨胀的温度降及单位制冷量。从气体的  $T - S$  （温一熵)图可以看得很清楚。

# 3.问：怎样判断冷凝蒸发器漏气？

答：冷凝蒸发器是上、下塔的纽带，使下塔上升的蒸汽冷凝为下塔提供回流液，使液氧蒸发为上塔提供蒸汽。冷凝蒸发器一旦发生漏气，会直接影响上、下塔的蒸馏工况和产品纯度。冷凝蒸发器的漏气，分内漏和外漏两种；内漏是指氮气漏入液氧侧，外漏指液氧或液氮漏入冷箱绝热层。

内漏的显著特征是氧气纯度下降，达不到规定要求。微小的漏气，氧气可以达到标准，但产量下降；漏气严重时，不仅氧气纯度达不到要求，中压压力下降，低压上升，液氮纯度破坏，氮气纯度下降。漏气部发生在液氧浸到的管中，分析液氧和气氧纯度，查氧氮  $T = P = x = y$  图，汽液一般能达到平衡，而漏气部位发生在汽相，汽液是不平衡的。

外漏的特征是分馏塔外壳结霜，冷损增加，高压压力偏高。微量外漏不易被察觉；漏气严重时，液氧液面下降甚至液氧不能积累，珠光砂从冷箱中吹出。

发生冷凝蒸发器漏气时，应停车加温吹除，然后关闭分馏塔所有阀门，对中压、低压进行保压试验，当中压升高到工作压力，

低压压力明显上升，中压压力明显下降，冷凝蒸发器漏气的可能性较大。此时必须注意检查中压、低压加温阀和液氮、液空节流阀的密封性，确保不漏后，在中压外部不漏的情况下，中压压力下降，低压升高，则可认为中压漏低压。但需排除液空过冷器漏气的可能性，将切断液空过冷器通路，如老150 $\mathrm{m^3 / h}$  空分塔，可把通-1阀、节-3阀关闭，排除该系统的压力，再对中压系统试压，便可得出冷凝蒸发器是否漏的结论。

若中压压力保压2小时，压力不下降，低压压力又无上升，则可认为冷凝蒸发器不存在内漏。

若是低压系统能保压，停压4小时不下降，中压系统外部又找不到漏气处，中压压力保不住压时，则可认为中压系统漏入冷箱绝热层。

若是低压系统在氧、氮、馏分出口管和安全阀加盲板下，外部又找不到漏，则低压系统冷箱内漏气。

经上述分析，塔内存在漏气，应扒出珠光砂，进行全面试压确定漏气部位。若冷凝蒸发器内漏，应拆下上塔，向冷凝蒸发器内注满水，对中压系统升压，即可确定列管、封头、接管部位的漏气。确定冷凝蒸发列管漏气时，应将冷凝蒸发器卸下取出列管芯子，用专用工具试压，找出具体漏气的管子，两端用铜锥体和焊锡堵死，经试压不漏后再装复。冷凝蒸发器修理时的列管堵死根数，最多不得超过总数的  $10\%$  。冷凝蒸发器的修理，应由制造厂的专业人员承担。

# 4. 问：冷凝蒸发器漏气判断失误的原因何在？

答：某厂的一套高纯氮设备连续运转多年，纯度、产量都正常。停车加温吹除后，再次开车，经三、四班调纯氮气纯度达不到要求，停车4小时后试压，当中压压力升到0.2

MPa，低压压力迅速上升到0.06 MPa，经分析是冷凝蒸发器漏气。于是一方面扒砂、搭脚手架，另一方面向制造厂求援。

我处接到冷凝蒸发漏气的信息，本着为用户服务、急用户所急的态度，立即组织人员，带着修理工具赴现场。经试压，认为冷凝蒸发器不存在内漏。

为什么用户试压时，中压仅  $0.2\mathrm{MPa}$  而低压会升到  $0.06\mathrm{MPa}$  呢？询问操作工后，原来用户试压时是在临时停车后4小时，塔内尚有液氧液面  $300~\mathrm{cm}$  四氯化碳柱，当热的高压空气进塔后，使冷凝蒸发器的液氧激剧气化，低压压力迅速上升。

那末为什么氮气纯度调整不好呢？影响氮气纯度下降的因素很多；有氧、氮流量的影响，制冷量的影响，液氮纯度的影响，馏分抽取量的影响，空气量多少的影响等等。其中该厂的空压机各级吹除阀漏气严重，调整时又缺少对液空、液氮的分析，又缺少液空液面计，节-2、第-4阀开度凭操作工的经验开关。据笔者的分析，主要是空气量偏少、下塔纯度未调好的可能较大。

该塔经试压，塔内存在四处较严重的漏气。经焊补后，装回珠光砂，关严吹除阀减少漏气。再次开车后，氮气纯度达到要求。

由中国工业气体工业协会组织召开的“全国气体行业合资企业工作研讨会”于1996年5月28日～31日在山东省青岛市召开。化工部行业指导司合资企业乡镇企业管理处杨开武处长、青岛市化工总公司的有关领导及清华大学经济管理学院李卓参加了会议并讲了话。气体行业的11个合资企业和未合资企业的中方经理以及有关领导20余人参加了此次会议。中国工业气体工业协会秘书长孙国民主持了会议。此次会议得到了青岛氧气厂和青岛鲁海液化空气公司的支持。

随着我国经济的发展，改革开放的深入，世界各大气体公司纷纷进入中国市场，近几年我国工业气体行业也掀起了合资热潮，气体行业的一些骨干

企业先后与这些世界著名的气体公司联姻，组成了各种形式的气体合资企业，引进了资金、设备、技术和管理，牵制带动了气体行业的发展。经过几年的运行实践，各个企业在合资合作过程中创造了很多成功的经验，也积累了不少现实的教训。对这几年气体行业合资合作情况进行认真的回顾与反思就显得非常重要。我们这次研讨，目的是使气体行业的合资企业办得更好，国有资产和中方利益得到更好的维护，让那些即将合资的企业吸取已经合资企业的经验和教训，为即将成立的合资企业创造更加有利的条件。

会上与会代表各自介绍了本企业的情况，进行了热烈的讨论和认真的回顾与反思，并且进行了深

入的探讨，总结了气体行业的合资经验教训，使感性认识有了初步升华，达到了预期的目的，取得了圆满的成功。

研讨会归纳了以下十六点体会和建议，供广大工业气体行业和有关政府部门参考。

1. 在谈合资之前，慎重选择合资伙伴。选择那些知名度高，信誉好，技术先进，条件对我们有利的外国公司作为我们合作的伙伴。  
2. 选择好合资的方式。坚持大型企业双方对等投资，中小企业灵活掌握投资的原则。合资企业模式最好采用母子公司的模式（如天津华北氧气厂合资模式），以便母公司对子公司的有效控制，而不能采用母体厂与合资厂成为兄弟厂的模式。合资企业的名称应尽量体现合资双方的企业的含义（如上海宝钢普莱克斯实用气体有限公司），中方最好不用地名代替其企业的含义。  
3. 在合资前和合资合同中充分考虑好母体厂不合资部分的出路，解决好合资厂与母体厂的关系。合资后，不能使母体厂处于无产品、无场地、无生产条件的境地。  
4. 在合资企业中坚持党的的领导，使党团活动正常进行并得到加强。  
5. 合资中合理评估中方无形资产和非经营性资产的价值，不能无偿投入或象征性做价投入。  
6. 合理评估中方固定资产。充分考虑国有资产在形成时所特有的物价环境与近几年物价上涨对资产升值的影响。  
7. 坚持双方共同投资，共同选择设备的原则。不能由一方单独做价投入或选择设备，以保证投入设备的先进性和价格的合理性。  
8. 合资企业的董事长和总经理最好采用中外方轮换担任的方式，这样可避免合资企业的经营管理权长期有一方掌握的弊端。  
9. 合资企业的管理模式应采取把中方优良传统和国外的先进管理方法结合起来，实行中外并蓄，扬长避短，先立后破，大胆创新的办法。  
10. 合资企业中坚持中外方员工（包括总经理）同工同酬。外方员工（含总经理）工资不足部分用双方对等提取某项费用的方法解决，对等提取的资金外方作为补充工资发给在合资企业中的员工（含总经理），中方作为中方职工的福利基金使用。  
11. 在合资企业经营过程中，正确处理好与外

方的关系。把眼前利益和长远利益结合起来，经营管理上真诚合作，利益上当仁不让。既全力维护中方利益和职工合法权益，又注意外方投资者的利益。坚持公平原则，重大问题必须经双方协商一致后执行。

12. 工业气体行业多数合资企业运行以后，均已获利，外商为了拓展前景看好的中国工业气体市场，提出增资方案，中方一般应采用如下方法：一要采取措施保证中方股份不下降；二是选择用利润分成分期分批再投入的方式；三是双方商定，外商先行投入，中方保留几年的股份购买权，几年后资金筹措完毕购回股权；四是中小企业中方无力再投入的情况下，灵活掌握，允许外方再投入增加股份。  
13. 合资企业的中方主管部门克服重引进轻管理的思想，依照合同实施对合资企业的管理，避免外方拥有  $50\%$  的股份，而实施  $100\%$  的管理体制。  
14. 建议国有资产所有者对合资企业的经营者建立起有效的制约机制。董事会是合资企业的最高决策机构。董事长一般也有中方领导人兼任，但有时起不到应有作用。原因主要是没有将中方董事长、董事对国有资产的管理职能在制度上明确下来。一般情况下，中方担任的董事长因兼职过多，精力和时间都不可能放在某一个合资企业，这样就对合资企业情况了解不够，造成中方管理不到位。建议国有企业管理者对中方经营者建立有效约束机制，使其真正行使产权管理职能，使经营者对合资企业的国有资产负其责任。  
15. 对合资企业的财务核算制定有效的管理方法，合资双方共同对财务进行审计而不是某一单方审计，以保证利润核算的真实性、可靠性，保证国有资产收益的实现。  
16. 建议政府主管部门改变工业气体合资过程中中方各自为政的局面。

因为跨国气体公司来华投资是有计划、有目标、有步骤、有长远的战略意图的，他们将中国市场作为一个整体通盘考虑本公司的投资和发展问题，而我方却是各自为战，各种利益主体独立行事，自相竞争，降低了同国外投资者的讨价还价能力。而跨国气体公司则利用其在技术开发、全球战略、资金雄厚等方面的优势与我国功能单一的单个企业谈判、合资、竞争，在战略上肯定是要取胜的。

# 氧压机使用自动疏水器效果好

杨玉雪

（张家口煤矿机械厂动力分厂，河北张家口市工业路6号，075025）

空分设备制出的氧气，用氧压机灌入钢瓶或充入管道，因氧压机气缸是用蒸馏水润滑的，水分会带进气缸，其进入多少与操作条件有关。虽然氧气经氧压机三级压缩后，要经过气水分离器清除水分，但制氧操作是三班作业，早中班操作工能按规程每半小时排放积水一次，到夜班由于人的精力等各种因素影响，往往不能按规程排放积水，致使管路有水压入钢瓶，长时间就会使钢瓶遭到锈蚀，并严重影响焊接质量。尤其是我们北方，冬天寒冷，管路经常冻结，影响生产。

根据以上情况，我们于1993年8月安装了浙江玉环动力机厂(注)生产的“SY-170氧压机自动疏水器”，并自制联接管和支架，设置在氧压机气水分离器的下边，其进水口接通气水分离器出口。由于有均压管的作用，保持疏水器中压力与气水分离器中压力相同，因而沉降到气水分离器底部的液体(水)，经过滤器过滤后，能靠自重顺利地流入疏水器内，而后在疏水器内靠浮筒的升沉作用，疏水阀自动启闭，依靠气体的压力及时向外排放积水，使氧气不带水分。

浙江玉环生产的  $\mathbf{SY} = 170$  自动疏水器，

与氧压机排气出口的气水分离器配套使用。它能随压力升降，及时自动地将气水分离器中的水排掉，达到排水阻气的目的。同时可防止氧气夹带未及时排除的水分进入充氧钢瓶，并消除手动排水时所发出的噪声。因此，氧压机自动疏水器，是制氧厂站安全生产的益友。

从我厂安全使用两年多的情况来看，SY-170自动疏水器确实发挥了它的特性，效果良好，能及时排放气水分离器中的积水，无噪声污染，管路和钢瓶内无积水，冬季未再发现冰冻管路的情况。维护也很方便，只要定期从过滤器反向吹除即可。同时减轻了操作工的劳动强度，代替了手工操作，我们将在其余的氧压机上推广应用。

（1996年3月7日收稿）

# XMC-300 手动操作器

# 空分控制仪表的新品种

数字显示，CPU芯片，外形尺寸小，节约投资

![](images/d0389b29fd4a2346b1eceb11eabbfc8402ba49780fe2b81d7ef87d91710ac4f4.jpg)

XMC-300手动操作器是采用INTEL公司MCS-51系列单片机开发的，是一种可与计算机系统(DCS)或常规调节器配套使用的辅助单元类仪表。

XMC-300手动操作器具有自动、手动显示；上、下限报警设定；二路  $4\sim 20\mathrm{mA}$  电流隔离输出（一路至调节阀，另一路反馈至中控室DCS系统或调节器）；手动/自动切换及手动/自动状态开关量输出。在自动状态下，手动跟踪自动，可实现自动至手动无扰动切换。

在手动状态下，若中控室DCS（或调节器）根据手动/自动状态信号切换至跟踪手动输出状态，使DCS（或调节器）的输出跟踪手动操作器的输出值，则可实现DCS（或调节器）与手操器之间双向无扰动切换。在手动状态下，用户按“增”或“减”键，即可实现对输出阀位值的控制。是DCS（或调节器）理想的配套手操器。

XMC-300手动操作器，已成功地在攀钢 $10000\mathrm{m}^3 /\mathrm{h}$  和吉化电石厂两套  $6000\mathrm{m}^3 /\mathrm{h}$  及上海金山石化  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备上投用。
