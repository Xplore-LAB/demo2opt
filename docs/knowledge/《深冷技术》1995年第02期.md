# 生产管理者的好助手——透平膨胀机智能转速表

换等项操作(具体时间由用户根据工艺要求自己设定)。还具有以上各项操作的执行监督功能,能够自动判断每次操作是否有效。并且配有类似飞机上的“黑匣子”,能够把每天各班进行的有效排污、切换次数,各班发生的最高转速,超速时间(精确到秒),累计运行时间,以及开停机时刻,都自动贮存在“黑匣子”中,供有关人员通过按钮随时查询,从而保证了各项操作的可靠执行,提高了空分设备运行的完好率,是管理者的好助手。

# 90年代低温及其应用进展

陈国邦（译编）

（浙江大学制冷与低温工程研究所，杭州市玉古路20号，310027）

摘要 主要讲述了国际上90年代低温制冷机的发展，包括微微型制冷机、混合气体工质、高效蓄冷器填料、高可靠性制冷机、吸附制冷机等；以及介绍了低温超导电性的发展情况，包括高温超导体的挑战，LTS应用、HTS的应用；最后简介了其它的低温应用情况。图6表1。

# 主题词：低温制冷机 混合气体 工质 寄冷器填料 吸附制冷机 低温超导

低温学(Cryogenics)与低温工程(Cryogenic Engineering)在许多方面是互不分割的，但人们往往将低温学看成是物理、化学及工程技术在极低温区的延伸。鉴于物质在极低温下发生的不平常现象，以及用来处理这些特殊现象的专门技术本身的特点，应该将低温学与低温工程分为不同的研究领域。本文只包括低温学及低温工程发展的部分内容，限于讨论低温制冷机、超导系统及有关技术在高科技工业中的应用。

# 一、低温制冷机的发展

采用高压气体节流产生制冷效应是一种最古老也是效率最低的制冷方法。但是，采用新的制造工艺、多级膨胀、复合循环以及混合气体制冷剂，可使节流装置的性能得到明显改善，以至JET制冷在许多条件下能与通常具有更高效率的制冷循环竞争。JET制冷器的优点有结构简单、尺寸小、重量轻、没有机械噪声和振动。

微微型制冷机

利特尔教授(Little)提出了一种制造J-T

制冷器的新方法。他采用光刻技术在玻璃底板上制成微小的换热器通道、膨胀毛细管和贮液器，然后将玻璃底板与盖板熔焊在一起，构成密闭的制冷器。这种微型制冷器已有各种规格的产品。一只在80K具有250mW制冷量的制冷器，换热器通道宽200μm，深30μm。通道误差±2μm，粘结强度能承受15~40MPa的压力。外型尺寸为7.5cm×1.4cm×0.2cm。

光刻法允许应用更复杂的膨胀循环和多级节流，采用双压J-T循环可获得更低温度或在相同的输入功率下具有较高的制冷系数(COP)。一般采用液氮预冷的二级机可获20K，三级机可达10K低温。多级制冷器对浸蚀工艺和尺寸控制有更高的要求，以保证为获得所需的制冷量要求的流量与各流道的容量相平衡。

混合气体工质

采用氮-烃气体混合物替代单一气体作工质可使微微型制冷器获得较高的COP。该项技术是阿尔夫耶夫（Alfeev）等在20年前

提出的专利。可是由于气体混合物中烃类气具有可燃性，阻碍了该技术的进一步研究和发展，直至发现加入极少量哈隆  $\mathrm{(CF_3Br)}$  使之变为不可燃物质为止。现已有一系列氮-烃气体混合物能安全地用作J-T制冷剂，在液氮温度下，其制冷效率达到卡诺值的  $50\%$  （不包括压缩机损失）。

该项技术的优点，可考察图1所示的由 $27\%$  甲烷、  $50\%$  乙烷、  $13\%$  丙烷及  $10\%$  丁烷（体积比)构成的混合烃的制冷性能，加以说明。采用这种混合制冷剂，由初始条件4.5MPa、300K节流到1大气压，理论上可获得200K出口温度。而在相同条件下采用纯氮气节流，则只能冷却到  $291~\mathrm{K}$  。也就是说，混合制冷剂可将制冷温差增大11倍。或者采用 $3\sim 5\mathrm{MPa}$  的低压混合气的制冷效应，可与 $12\sim 15\mathrm{MPa}$  的氮气相比拟。工作压力的降低，激励了开发长寿命的封闭循环J-T制冷器。显然，需要大量的实验工作去获得不同的气体混合物的性质，探明各种不同制冷任务所需的最佳组分，了解在不同温位能形成的不同液相的效应。

高效的蓄冷器填料

吉福特-麦克马洪循环(G-M)制冷机是用得最广泛的装置。具有一级膨胀的G-M机可达到  $65\sim 80\mathrm{K}$  温度，二级机可达  $10\sim 15\mathrm{K}$  。二级机的预冷依靠应用各种蓄冷填料的蓄冷器来完成，蓄冷填料应具有高的热容量和热导率。铅丸是广泛应用的填料，然而，当温度低于  $10\sim 12\mathrm{K}$  时，其热容量变得无效。长期以来，没有一种合适的填料能使G-M机的低温级(第二级或第三级)获得4K温度。于是，为获得4.2K，以便使超导核磁共振成象(MRI)磁体杜瓦中蒸发的氦气再液化，不得不在二级G-M制冷机上附加一个采用紧凑式换热器的J-T回路。

过去几年，桥本巍洲(Hashimoto)等通过实验推出一系列在低温下会呈现磁相变的重稀土化合物，在低温下这种蓄冷材料会发生物理相变，其比热容就可能增大。图2和图3示出部分实验结果。图2示出  $\mathrm{Er}_{1 - x}\mathrm{Dy}_x-$ $\mathrm{Ni_2}$  、ErNi、  $\mathrm{Er}_3\mathrm{Ni}$  和  $\mathrm{Gd}_{0.5}\mathrm{Er}_{0.5}\mathrm{Rh}$  在低于20K的比热容。图3是  $\mathrm{Er}_{0.9}\mathrm{Yb}_{0.1}\mathrm{Ni}$  和ErNi的比热容与Pb(铅)的比较。1993年，该研究小组研究了  $\mathrm{ErNi}_{1 - x}\mathrm{Co}_x$  系的比热容，实验指出，在7K时这些新材料的比热容是  $\mathrm{Er}_3\mathrm{Ni}^{\ddagger}$  的2倍。采用这些特殊材料，可使G-M机的第二级制冷量比过去采用单一铅丸有了明显增大。Nagao等报道了用于MRI杜瓦再冷凝的三级G-M制冷机，采用  $\mathrm{Er}_{1.5}\mathrm{Mo}_{1.5}$  Ru作为第三级蓄冷器材料，制冷量在4.2K时大于  $150~\mathrm{mW}$ ，使MRI超导杜瓦中蒸发的氦气再液化成为可能。初步的研究结果表明，这种装置的体积比传统的4K制冷机小，可靠性增大，总运转成本降低(图4）。

吸附制冷机

这种制冷方法利用气体通过表面的物理吸附或内部的化学吸收得以压缩，然后通过J-T膨胀阀获得制冷量。在吸附式压缩机中，低压气体在各种温度下被吸附在固体上。当气-固系统被加热时，温度将升高到  $100\sim$ $300^{\circ}C$  ，解吸的气体被加压到高压，为J-T制冷循环提供产冷量。

因为新近已采用由若干吸附单元组成的吸附压缩机，故对吸附剂周期性的加热和冷却所需的大功率要求，得以明显的下降。即利用正在冷却的单元中放出的热量来预热待加热的单元。

# 二、超导电性的发展

超导电性是指当直流电流动时，电阻消失了的物理状态。在1911年奥尼斯(Onnes)发现超导电性后的许多年中，只有液氮可用作超导器件的冷却剂。即使如此，仍有几种低温超导体(LTS)的商业应用获得发展。1987年陶瓷高温超导体(HTS)的发现，可采用便宜得多的液氮作为冷却剂，引发了应用HTS的巨大兴趣。其实，直到现在才开始寻找应用HTS的方法。

场。但是这种方法不能制造符合商业要求的长线。

最广泛采用的制造 HTS 导线的方法是用 BSCCO 陶瓷粉末装入管内烧结法。首先将 HTS 细粉末装入空心银管内，然后用机械法拉长到形成一股线烧结而成。这种线在  $4\mathrm{K}$  下具有很高的磁场，但当温度高于 20 到  $30\mathrm{K}$  时性能明显下降。因此现在面临的挑战是充分增高每股超导线的临界电流密度，以便能采用液氮作为冷却剂。

在磁体工作时，沿着超导线将会产生磁流。通过超导线的电流和垂直于电流的磁场一起产生的罗伦兹力，力图将磁流线推入超导体，磁流线的运动引起磁体发热和能量的消散，造成严重的磁损。因此，应使磁流线不运动或“钉扎”在当地。HTS材料出现这类问题要比LTS更多。至今在诸多HTS中，仅有TBCOO族的1223相在磁流密度方面，在77K具有获得可用磁场强度的潜力。此外，用管内填粉法烧制HTS材料的方法也需要改进，因为陶瓷粉末与银质管壁会产生反应。

由于 HTS 陶瓷的脆性，不能绕制成超导磁体。一种解决方法是先将填粉管绕制成磁体，然后进行热处理制成产品。另一种方法是制成许多细丝，使之在粉末热处理后仍有足够的柔性。开发能够盘成线圈的极细 HTS 带的研究工作已取得进展。

# LTS应用

低温超导体都是采用液氦冷却的，应用最广泛的材料是NbTi线，临界温度为9.6K。这种软线的实用磁场一般不超过8T。最近，将许多  $\mathrm{Nb}_{3}\mathrm{Sn}$  细丝编制在一起的超导线能在高达20T的磁场下工作，这种超导装置可以应用在10K附近工作的二级低温制冷机来冷却，可免去对液氮或超流氮的需要。

LTS的主要市场目前多在医学方面，最大的应用就是核磁共振成象(MRI)。通过在大到2T的磁场中绘制氢原子的密度而将人

体器官成象。更高的磁强能使  $\mathbf{Na}^{23}$  和  $\mathbf{P}^{41}$  共振，可用来研究人体代谢的动态过程。为了保证足够的图象清晰度，MRI应有极好的磁场均匀性，在被检区的磁场均匀度应高于 $10~\mathrm{ppm}$  。另外，采用小型的超导量子装置(SQUID)传感器，可使医学专家们探测出人脑或心脏的磁场波动。

长期以来，一直在开发大型超导磁能贮存(SMES)系统，用以将低峰周期中多余的电能贮存在超导线圈中。虽然SMES的贮能效率高达  $90\%$  ，但其时要求所贮能量高于100MWh，而且作用于导体上的磁力要求将SMES系统埋入地下，这会增加投资。

小型的 SMES 装置, 可在短周期内放出兆瓦电力, 目前被用户用来消除电压不足或其它瞬时电力干扰。现在的 SMES 装置均用 LTS 材料, 如能采用 HTS 材料则会显著降低成本。

# HTS的应用

用 HTS 制成的液氮深度传感器 已有商品。近来还应用 HTS 材料作为低温恒温器的电流引线，将电力从环境温度供给处于液氮中的超导器件。其优点是消除了 HTS 引线超导部分的焦耳热，使导入液氮的热漏减少，因而可延长工作时间。

磁轴承是HTS材料实用的代表。其原理是用一个圆柱形永久磁体悬浮在HTS磁体上面，永久磁体则绕着其轴线转动构成了良好的轴承，其摩擦系数可低于  $10^{-8}$  。低摩擦轴承的潜力可以用作飞轮能贮系统（Fly-wei1 Energy Storage Systems）的轴承，目前已研制出演示装置。

用 HTS 材料制作的故障电流限制器 (FCL) 是一种防止反常大电流、保护生活设施的保险器。在这种装置中，超导 FCL 流动着正常电流而没有电阻，一旦反常电流超过了超电体的临界电流值，它就变成有电阻的正常态，这时便将电流分流到交流回路。

设计时因顾及投资等种种因素，根本没有考虑进行富氧鼓风这一环节。即使象宝山钢铁总厂全部引进的1号高炉，其设计的最高富氧鼓风能力亦仅  $3\%$  。该炉运行至今的近10年历史中，送风中富氧率几乎从未达到过此水平。而国内其他企业中的高炉，仅是利用满足炼钢需要后的剩余制氧能力以减少氧放散，来达到制氧工序节能、高炉降低生产成本双重目的，但这属于权宜之计。

尽管炼铁工业者早已熟知富氧鼓风具有强化高炉冶炼和在某一范围内可降低焦比和生产成本等一系列优越性，但全焦冶炼条件下，富氧鼓风口循环区火焰温度过高、炉腹煤气量变少，由此而造成的上凉下热，使高炉难以驾驭而不能多用。以往仅在有限的小范围内进行过一些短暂的富氧冶炼试验，其鼓风富氧率最高也不超过  $6\% \sim 7\%$  ，且无法维持长期的稳定操作。

长期稳定(年平均)喷吹  $200\mathrm{kg} / \mathrm{t}$  铁或稍多些的煤粉或粒煤，目前已没有技术上的障碍。其尚未实现的主要原因之一，是煤制粉能力暂时未跟上，同时高炉操作上的适应也需有一过程。煤焦置换比，则是经常令人关心的问题。

（2）风口氧浓度  $30\% \sim 40\%$  ——中富氧

中富氧时煤比可达  $250\sim 300\mathrm{kg / t}$  铁。为了确保煤粉的燃烧，并考虑到含氧  $30\%$  以上高温空气会对送风系统设备（包括热风炉，例如炉算子）产生不良影响，氧气宜从氧煤枪处进入风口循环区。至于氧煤枪对提高煤粉点火时氧碳比及改善煤粉燃烧的作用已有众多研究，不再赘述。

当富氧率超过  $10\%$  ，尤其是接近  $20\%$  时，会引起另一问题，即单位生铁需风量将有所下降(参见表1)。虽然高炉利用系数的增加会抵消部分的风量过剩，但毕竟出现了鼓风机能力裕余的可能。为了维持风机的正常运转，不致发生喘振，有可能迫使部分鼓风放散而浪费了能量，故对此负面效应还需加以关注。

（3）入高炉氧浓度  $40\% \sim 90\% \sim 99.5\%$  ——高富氧或全氧（无氮鼓风）

对应的喷煤量在  $350\mathrm{kg / t}$  铁以上。国内外已提出了若干种全氧或高富氧大喷煤量高炉方案(表3）[4]。日本钢管公司在  $3.9\mathrm{m}^3$  小高炉上进行的全氧鼓风冶炼试验，证实了从风口鼓入常温工业纯氧、炉身中部喷入预热煤气的可行性。该炉在喷煤量达  $363\mathrm{kg / t}$  铁之前，炉尘量和炉尘中含碳量基本不变，炉况顺行[6]。

2. 制氧机出口的氧气浓度

由于用深度冷冻空气分离法生产含氧 $60\% \sim 90\%$  的氧气折算成纯氧时，较之生产工业纯氧，能(电)耗和成本皆明显下降，故有识之士早已提出不用工业纯氧作为高炉富氧鼓风的氧源，国外已有付诸实践的例子。迄今此技术未被广泛应用的原因是炼铁高炉用氧量较少，但大喷吹时代已即将来临，未雨绸缪，此期时矣。

笔者曾向下述各方求教，归纳如下：

(1) 炼铁工作者的想法

绝大多数人认为，与转炉炼钢不同，氧气纯度对铁水质量关系不大，关键是需要连续稳定地供氧。而氧气单耗剧增后，对生铁成本的影响已不能忽视，因此廉价氧会倍受欢迎，含氧  $90\% \sim 95\%$  的氧气正符合高炉富氧大喷煤的需求，更不用说用氧量倍增的各种煤基熔融还原炼铁技术了。

(2）能源管理者的想法

对于设有能源中心的大型钢铁企业，其管理者希望无论炼铁或炼钢全部应用生产高纯氧的制氧机组。他们认为：一是便于统一调度和管理，若运行中某一机组发生短期故障，其余机组可立即支援以防事态扩大；二是可降低维修费用，因备品备件通用性强，维护方便；三是生产高纯氧非但可获得钢水处理时不可缺少的副产品氩气外，还能通过分离气体的深加工提取多种经济价值很高的稀有气体，有助于降低制氧机的运营费用。

# （3）制氧机生产者的想法

为了提供廉价的氧气，空分行业也曾开发了若干有别于传统深度冷冻空气分离的节能氧气生产法，如利用分子筛吸附原理的变压吸附法和膜分离法等。但迄今为止，这些方法的产率无法满足冶金行业的需求。展望到炼铁用氧将有质的飞跃，制氧机生产者期待与炼铁业合作开发专门用于炼铁配套的高炉和熔融还原型制氧机，且自成系列化。

# 四、结语

随着高炉喷煤量的逐渐增长，当煤比超过焦比时，必然伴陪着氧气需求量的飞跃，其数量将大于转炉炼钢的耗氧量，故未来联合企业中的氧气最大用户非炼铁莫属。

由于工艺过程的不同，炼铁与炼钢工序对氧供应上的数量和质量要求有所差别，炼铁需要长期稳定的低压（  $< 0.3\mathrm{MPa}$  ，加入鼓风)或中压（  $< 1.0\mathrm{MPa}$  、用于氧煤枪）氧气，希望价格低廉，但纯度可不必太高。因此有必要开发专门用于炼铁生产的制氧机。采用中压空气分离循环法生产含氧  $90\% \sim 95\%$  的中纯度氧气，是一种较适宜的节能工艺，可

望降低氧气的生产成本；若再与利用中温余热或剩余高炉作为热源，对空气产品氮气加压，使其成为膨胀透平的工质进行发电则更为理想。

借鉴高炉喷煤制粉工序的磨煤装备从最初采用球磨机转为目前流行应用中速磨的历史经验中可知，炼铁工作者对制氧工艺的发展有所了解，进而与空分行业人员合作，已是时不待我了。

# 前言

天津华北氧气厂有一套  $700\mathrm{kg / h}$  液氧液氮设备，配用三台活塞膨胀机制取冷量。由于活塞膨胀机具有摩擦部件和往复运动的特点，因此故障率高、维修费用大、工作不稳定、噪声大。经过几年使用以后，特别是随着市场对液体的需求量日渐增多，该厂液体产品供不应求，问题便尖锐化了。因为活塞膨胀机停车检修影响生产，更换易损件每年要花十几万元。因此该厂迫切希望杭州制氧机厂能为之研制相应的透平膨胀机以取代活塞膨胀机。

透平膨胀机系高速旋转运动，又没有互相摩擦的零部件，因而克服了上述活塞膨胀机的缺点。同时由于近年来透平膨胀机技术的进步，可以用增压风机制动，使进入膨胀机的气体先在增压机里进行预压缩，使活塞压缩机的排气压力降低，这样便可降低整个装置的能耗。但由于受该厂投资及场地等因素的限制，要求透平膨胀机设计成单级膨胀，而效率不能低于原来活塞膨胀机。在高压力、高焓降、低容积流量之下，透平膨胀机在设计制造过程中就要克服很多技术上的困难。

# 一、技术上的困难及其对策

# 1. 工质状态参数及热力参数的处理

在低温及高压下，工质状态已大大偏离了理想气体状态方程，而一般的热力图线在此区域又查不到满意的结果。因此我们采用热力性质表，再利用高次曲线二维插值编制计算机程序，这样便可获得较为准确的压缩因子，然后根据公式计算热力参数。实践证明，这种方法可以得到满意的结果。

# 2. 高压受压零件的处理

膨胀机蜗壳、增压机蜗壳、增压机气体冷却器、管道、波纹管(共有4种波纹管)等均属高压受压元件。属压力容器的则按照压力容器标准进行设计和制造，其他的也要设法核算和试压，外购件则按特殊订货处理。

值得一提的是膨胀机蜗壳，由于压力

高，又是低温工作，不能采用以往的铸铝合金结构。我们采用不锈钢铸件，但是由于奥氏体不锈钢在降温过程中会发生向马氏体的金相转变，晶粒长大，因而会严重破坏精密加工件的尺寸精度和几何精度。所以选择材料时不应采用其转变温度在常温与蜗壳工作温度之间的材料。我们经估算，并经多种材料多次低温试验，选用铸造分厂自己配方的不锈钢铸造蜗壳，其转变温度低于  $-196^{\circ}C$

# 3. 关于转子轴向力和止推轴承

由于压力高，同时增压机进口压力也很高（约  $2.8\mathrm{MPa}$  绝），因而转子轴向力很大，止推轴承难以承受。我们采取了以下措施：

# （1）提高轴向力计算准确度

一般推荐叶轮轴向力计算公式推导过程中，假定转盘与壳体间隙中气体重度不变且等于外圆处重度。实际上因离心力的作用，气体压力是由外圆到内圆逐渐降低的（一般假定呈抛物线分布）；同时由于摩擦功的加入，气体温度也有由外向内升高的趋势，两种因素都促使气体重度变小。而如果假定间隙中温度不变且等于外圆处温度的话，误差就会小些。因此，我们采用这种方法对轴向力公式进行重新推导和积分。

# (2）采用轴向力平衡手段

在增压轮轮背某一直径处增加一迷宫密封并将增压机蜗壳中气体压力引入密封内径侧，这样便提高了整个轮背的压力水平而起到平衡轴向力的作用。至于密封处的直径则要进行详细的计算才能确定。

# （3）适当提高止推轴承的承载能力

在设计止推轴承时，适当调整原有轴承的结构尺寸及止推面斜度，以提高止推轴承的承载能力，同时又要兼顾到轴承功耗不要太高。

# 4. 关于工作转速和临界转速

根据我厂生产的系列档次，确定膨胀机叶轮直径为  $\phi 130\mathrm{mm}$  之后，工作转速便和膨

胀机的性能密切相关，同时也和临界转速及轴承功耗相关联。从膨胀机性能出发，转速应设计在速比  $\mathbf{u}_1 / c_0$  的最佳值上；从临界转速出发，转速(包括最大允许值)应低于临界转速并有合理的安全裕度；从轴承功耗出发则转速越低越好。综合三者因素并考虑如叶轮强度、轴承发热等因素后，取一个适当的工作转速使速比  $\mathbf{u}_1 / c_0$  比较接近最佳值。

对于临界转速的计算，有一个很重要的因素是要对转子上每个零件的重量和重心位置进行精确计算，而这些零件特别是叶轮的几何形状又十分复杂。为此，我们专门编制了计算程序来进行计算。对于由直线和圆弧组成的回转体，我们的计算程序是采用直接积分的准确方法，而不是以往采用的梯形分段的近似方法。

# 5. 提高机组效率的手段

与常规参数的膨胀机相比，尽管对本机的效率要求并不算高，但在本机高压力、高膨胀比(29.6)和小容积流量的特殊情况下，要达到所要求的性能指标是很困难的。而且是首次开发这种膨胀机，缺乏经验和资料，因此，在设计工作中，必须想尽一切办法来提高机组性能。

# （1）速比  $\mathbf{u}_1 / \mathbf{c}_0$  不要偏离最佳值太远

在临界转速允许的前提下，综合考虑轴承耗功、发热等因素，尽量使速比  $\mathbf{u}_1 / \mathbf{c}_0$  不要偏最佳值太远。

# （2）喷嘴的设计

由于膨胀比太大，喷嘴出口气流马赫数已达1.49，依靠斜切口来进行超音速膨胀已不能适应。因此我们将喷嘴叶型设计成喉部截面后带扩张段的曲线型喷嘴，使效率尽可能提高。喷嘴叶片可以转动以适应产量调节的需要。同时为了防止气流经过叶片顶部间隙短路而造成损失，采用了喷嘴盖板的弹性压紧机构，使得喷嘴叶片既可以转动，又不形成间隙。采用这种方法的关键是要对喷嘴

盖板两边的气体作用力进行详细的计算，以选择合理的密封直径。

# (3）叶轮流道的设计

膨胀机叶轮叶片采用高次抛物线成型。为了获得性能较好的流道形状，我们对叶轮流道内从进口到出口进行了气体流速变化的计算。将流道长度分成12段，共计算13个速度值以判断其速度变化是否合理（不得有减速流动）。调整叶片形状和反复计算子午面成型，以获得较为理想的流道设计。

（4）增压机叶轮采用效率较好的后弯式叶轮

# (5) 尽量减少轴承耗功

尽量减少轴承耗功以提高膨胀功转换为压缩功的转换效率。如适当减小径向轴承宽度，因为对于高速轻载转子来说，一般径向轴承的承载能力是比较富裕的。

以上列举了本机设计研制过程中所碰到的技术上的主要困难及其所采取的措施。当然技术上的困难并不只这些。比如由于整个机组气体压力的提高就会带来许多困难，如受压零件的强度和刚性问题、高压气体的内漏和外漏防止问题，以及受压弹性元件(波纹管)的外购问题(所需4种波纹管按常规国内无法提供，经与制造厂反复商讨才得以按特殊设计特殊订货解决)等，这里不详述了。

作。液体氩在塔底生成再送入贮罐。含氩低于  $0.5\%$  的富氮蒸汽作为过热产品，出塔再补入氮循环系统。该塔回流是靠塔顶直接进入液氮，而再沸是靠中压氮气冷凝来实现的。

燃料气是下列三种气体的混合物：从甲烷塔底部来的甲烷、从第一洗涤塔来的富氢气体和从提氩塔再沸器来的放空气。这种燃料气用来进行分子筛干燥系统的再生或送往合成氨装置的燃料气系统。

氙产品分离与液化所需要的冷量是由一个开式氮循环系统提供的。进入系统的是从提氙塔来的过热氮气，而一些过剩的氮气则排入大气。氮循环系统包含一个六级离心压缩机、一个带增压压缩机制动的膨胀机和相应的换热器。在甲烷塔再沸器中，氮气大约在  $3500\mathrm{kPa}$  压力下被冷凝，而在提氙塔再沸器中，氮气大约在  $1200\mathrm{kPa}$  压力下被冷凝。膨胀机则在这个高压和中压两个压力等级之间运行。这个过程也能产生液氮，并在运行中积累下来。它可用于停车时的清洗和解冻。

这就进一步改善了装置的经济性。

# 控制系统

单元控制在很早就被公认为装置的一个重要特征。这有以下几个原因：

1. 装置要求在高回收率情况下获得高纯度氩气，这就需要对两个主塔的双组分精馏过程进行相当严格的工艺规范控制。  
2. 制冷系统的物流是一个复杂的工艺过程，在这个过程中任何工艺参数的变化，都将引起整个系统变化，甚至影响全过程的完成。  
3. 分子筛干燥系统在每8小时再生切换时，原料气流量的逐步变化是要结合温度变化进行预测的。

精馏塔的双组分控制对控制系统的设计提出了一个要求，即甲烷塔的过热气中，甲烷要减低到0.1ppmv以下，而塔底液体中氩要减低到  $0.5\%$  以下。过热气中，甲烷纯度是个重要参数，它的任何波动都将影响最终的氩产品质量。回流量要适合于维持蒸汽流出量和甲烷引出时液位的平衡。一个在线甲烷分析仪(ppm级)用来监测送到缓冲罐去氩产品的甲烷含量。氩回收塔则是较为常规的控制。

在设计阶段已经考虑将动态模拟方法试用于精馏控制方案的模型(DCS)。其目标是要构思出各种方案，用来解决任何可能出现的问题。然而两个可以选择的办法都要保证严格控制净化流程及回收流程的许可流量。如果出现什么重要的控制问题，应用已经安装的DCS系统就能很容易地重新构思解决办法。另一个方法是采用有效的人工控制。必须指出：应用动态模拟方法能够对操作人员的培训带来巨大的收益，并能在操作过程中提高回收率或降低不合格品数量，从而增加产量。在装置建设时，这对操作者是有好处的，但对设计者来说，动态模拟的好处是不太明显的。

# 装置特点

本装置的设计中，重要机器的任何失灵都已考虑了氢回收装置的恢复问题，以使对合成氨装置的影响减少到最少程度。

本装置包含两个带回流的板翅式冷凝器（即精馏塔）。这是能在一个塔器中有效地进行多级分离和换热，并具有较高效率的设备，它代表了这个领域里目前工艺技术的最高水平。它能在没有外加分离器或泵的情况下，可以使液体从冷凝器中冷凝再回流入塔。

制冷所需的能量不是由原料气的高压能量提供的，而是由一个前面已经提到的开式氮循环系统来提供。这个系统用一个标准型压缩机，它附带的缓冲器，可以保护系统内气体不被油、水或空气污染。

设置的一个空气组分分析室，包含有一套选择性特殊分析仪器。这套分析仪器用于过程监测(不控制)、工艺跳车(按产品质量规范)，以及装车产品质量控制和记录。

这些仪器的测量范围是：

*原料气中甲烷 0~20%  
*甲烷塔塔底甲烷 75~100%  
*甲烷塔过热器甲烷 0~100 ppmv  
*氨成品中氮含量 0~200 ppmv  
*氨成品中氧含量 0 ~10 ppmv  
*氩成品中水分露点 -100°C

*全系统色谱监控记录，带有8个位置的自动取样，监测液氩中不纯物(ppmv)。

装置控制用分离的三段式控制器在提氩装置控制室进行操作。也可以用合成氨装置控制室的DCS间接控制。这个系统以标准的PC机组成，可以进行所有的趋势记录、报警记录以及控制回路的运转及预测。

# 装置试运行

这套装置是Costain公司设计、制造并安装的第一套用合成氨弛放气提取氩气的装置。初期的一些困难问题已被预想到了，剩下的这些毕竟是次要也是容易解决的事情。

复杂的控制方案使原料气变化及其带来的不稳定因素，都不影响装置在设计的最小能耗及最大氩回收率指标下运行。相当程度上是控制系统稳定了装置的各种变化。

分子筛干燥的切换系统的改善使提氩装置有了一个较稳定的原料气气源。在改进之前，每8小时进行切换时，波动将持续1.5小时。这是由于在切换之前经过容器加压的气体流量大幅度减少，以及由于床层冷却不完全而温度升高的缘故。

精馏塔再沸器及冷凝器控制系统的综合反馈检查引出一些有细微差别，但可以改善控制器运行的控制方案，这就大大改善了系统的稳定性。只有在原料气条件变化较大之时，才有必要用人工来调整控制器。这就基本上保证了在整个时间内使能耗保持在一个最小值。

三个在线分析仪用来监测氩产品中的甲烷、氮和氧等杂质。如果产品不合格，氩就转入缓冲器再进行重新加工。另外有一个色谱仪用于氩产品纯度监测。在分析仪和色谱仪两者之间，数据的一致性很难达到，这就造成产品质量数据的不确定性。为此安装三个附加的分析仪，用来核对氩产品纯度。

AMC工厂的主要业务是生产化肥，而氨只是一个副产品。在试运行之前，曾担心提氨装置会导致合成氨装置的故障，从而造成氨产量损失。然而在试运行期间合成氨装置一直保持在线运行，尽管在最初阶段提氨装置不稳定而曾经造成燃料气波动。

# 其它

供给合成氨装置的天然气含有  $0.1\%$  的氮。它在供给提氙装置时已经浓缩到  $1.5\%$  。AMC工厂不知道在其装置中使用的色谱仪要用氦来作为载气。氦挥发后随着氢回到合成氨系统中去，它不影响氩回收的工艺过程。

但它增加了弛放气的流量，并且不可避免地要影响装置的氢回收工艺过程。

由于气回收装置的设置，氢回收率已从 $90\%$  提高到  $94\%$  。然而这不能使合成氨装置在操作中再进一步有效地节约能耗。氢纯度提高到  $92\%$  ，这意味着氮回收率将会略有下降。气回收装置出来的氮气将排入大气，因此把氮返回合成氨循环，减少进入合成氨装置的空气量，将进一步回收能量。

在氙产品说明书中明确提出，在通常情况下要求排除潜在的污染物。原料气中氮、氢、水和甲烷能用吸收或吸附法加以清除，然而不能使低沸点的氧和氩分离。由于原料气中含氩量为  $5\%$  ，而氙产品中氧含量最大不超过1ppm。这就意味着原料气中氧含量要低于0.05ppm。这个问题用氨吸收和循环压缩的反复操作可以成功地加以解决。

# 装置布局

氩生产的经济性，正如工艺过程中的相对强化以及资金的适度投入一样，是很重要的。然而有许多其它要考虑的因素，它们对总体方案是同等重要的。

1. 氩回收装置的布局尽量靠近合成氨装置的铁路系统。因为它的装料管线是真空绝热套管，这是一项主要费用。  
2. 当几套合成氨装置合用一套提氩装置时，由于形成了经济规模使工艺过程更为经济。然而，由于存在一个大量液体专用户的长距离运输问题，所以这种经济性就不是永远现实的。  
3. 这种装置方案存在一个肥料行情的敏感问题。肥料行情可能引起合成氨装置减产甚至停车。在减产时提氯装置能量节约是很少的。没有一个合成氨装置操作者单单为了生产氯而去维持合成氨装置的运行。所以这种方案的可行性同时是与肥料价格紧密相连的。  
4. 这种装置需要布局在合成氨装置操

工作室与氩产品销售点之间。同时，谁拥有这套装置？谁操作并维修这套装置？驰放气供给及燃料气生产有多少变化？如果合成氨装置出现一些被动而导致氩产量损失谁来处理？所有这些问题，都需要加以明确。

# 现有装置的匹配

在合成氨装置中能否追加一套氩回收装置，仅取决于合成氨装置的型式，而不管它是否已经设置了氢回收装置。

氩分离和液化所需的能量设计值，随着弛放气压力增加而减少，随着氮/氩比值减少而减少，也随着甲烷含量增加而减少[3]。

合成氨装置的型式是重要的。“标准”型设计放出高压弛放气，可以获得氩回收率达 $96\%$ ，其典型设计能耗为  $1100 \mathrm{~kWh} / \mathrm{t}^{**}$ 。然而，“布朗净化”(Braun Purifier)工艺生产两种弛放气：一股是低压引入氩系统，这种氩回收是不经济的；另一股是高压富氩气体引入氩系统，可使设计的氩回收能耗在900  $\mathrm{kWh} / \mathrm{t}^{**}$ 。然而，这种氩回收量仅是进入装置的空气中氩总量的 $50\%$ 。这意味着这两套装置中两种弛放气回收需要找出一个适当的运行负荷。

如果合成氨装置已经有了氢回收装置，它可能是深冷或膜分离工艺[4]，从这些装置出来的弛放气是不同的，需要制定不同的工艺方法。

用膜分离法氢回收装置出来的燃料气是高压的，从而降低了设计能耗及氩回收装置的成本。

用深冷法氢回收装置出来的燃料气是低压的，但它是富氩气体，因此其结果是设计能耗略有增加。这种燃料气从现有冷箱出来能用作干燥和氨解吸。这样就不需要用干燥的原料气。对于AMC工厂有几种合适的选择，即将氩回收“包藏”在现有的氢回收冷箱中，或者在新的氩回收冷箱中考虑氢回收。

# 进一步提高效益

这种原始型的气回收装置，可以用增加设备的方法，来降低产品装载过程中的气回收损失，以及在低负荷时，利用已经安装的设备能力。

排放物再冷凝

从真空绝热氩贮罐中蒸发的氩损失是很小的。但是由于产品的价值较高，可以把它回收在一个氮冷凝器中。但当氩产品灌装到铁路或公路槽车中去的时候，氩损失是可观的。尤其是槽车到达温暖的地方而需要降温时，或者槽车被污染而需要清洗时，或者贮罐与装卸点之间距离很大时，这种氩损失是很大的。

在这种情况下，附加一个氮冷凝冷却器能够用来回收氩的蒸发损失。如果回收气体能净化，可再进入氩贮罐，否则就要重新进行加工。

液氮

装置产生的液氮蒸汽是被送入真空绝热贮罐的。提氩装置满负荷生产时，液氮产量是很小的。而当提氩装置降负荷时，循环压缩机所富裕的设备容量可以有效地用来生产额外的液氮。这种液氮作为停车过程中装置解除以及开车或运行过程中的净化之用。

再处理设备

真空绝热贮罐能作为一个缓冲器，可储存液体供提氩装置进行再加工。该贮罐的设置，是考虑到工艺过程被打乱而出现不合格品，以及将铁路槽车中回收的氩重新加工。

氮

从提氯装置出来的过量氮大多是排放于大气之中。如果合成氨装置的空压机是整个工艺的“瓶颈”环节，那么可以用一个较小型的压缩机将氮液回入合成循环的收入口[6]。这将导致成本的略微增加，但也将使氨产量增加。这就足以抵销其增支因素。

# 开发前景

在成功地开发第一套气回收装置以后，Costain公司希望能扩大已有的业绩，并完善其附加装置。然而氩的市场情况是复杂的，并且受到制氧及钢铁工业状况的极大制约。建设这种从合成氨弛放气回收氩的新装置在世界范围内还是有机会的。至少已经有一个装置将在欧洲配合独立承包商进行筹建了，可能进一步在美国和印度筹建[7]。气回收是一个最困难的深冷装置设计课题，每一个方案都是一个挑战，但都已被Costain公司成功地解决了。

对主冷凝蒸发器设计主要有两个要求：

1. 满足热负荷传递要求

主冷凝蒸发器的设计，是根据总体流程计算和精馏计算结果，给出热负荷Q和液氧与气氮的温差  $\Delta t$  ，来计算其传热面积的，不同的空分设备容量要配置与之相适应的主冷凝蒸发器。

在现代大中型空分设备中，为了便于制造和组装，主冷凝蒸发器的设计，大都是配置一个或数个板翅式换热器单元，而每个单

元高度一般为  $1.8\mathrm{m}$  。设计规定，运转中操作液氧液面比板式单元高出  $200\mathrm{mm}$ ，以实现主冷凝发器全浸操作。

2. 满足安全运转的要求

国内外实践证明：不仅  $\mathbf{C}_2\mathbf{H}_2$  存在且在一定条件下引起主冷凝蒸发器爆炸外，其它的碳氢化合物  $\mathrm{C_mH_n}$  的浓缩，在一定的条件也会引起爆炸。因此主冷凝蒸发器液氧内的碳氢化合物含量要控制在一定范围。《深冷技术》1991年第2期，提供了生产氧氮的普通型空分设备液氧中的碳氢化合物测定值如表1，提氮型空分设备中碳氢化合物(除  $\mathrm{C}_2\mathrm{H}_2$  外)的含量如表2。

# 三、初步看法

通过以上的计算和分析，笔者有以下两点初步看法：

1. 为了使空分设备能符合设计工况下正常运转和实现安全运转，必须正确地控制主冷凝蒸发器液氧液位高度，并符合设计规定的要求。  
2. 盲目随意提高 主冷凝蒸发器 液氧液位高度，不仅会造成空分设备的误操作和不正常状态，而且将使能耗增加，因此正确控制主冷凝蒸发器液氧液位，可以节约电耗，而不要盲目随意提高。

# 三套  $1000 \mathrm{~m}^{3} / \mathrm{h}$  空分设备投运及并机介绍

袁加宇

（江苏省张家港润忠钢铁有限公司  $6000\mathrm{m}^3 /\mathrm{h}$  氧站筹建处，张家港市锦丰镇，215625）

摘要 本文主要描述了杭氧制造的三套  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备，在工程设计、安装施工、单机试车、联动试车以及三套设备并机连接后，出现的问题和经验。并详述了操作过程和应该注意的要点，可供有关厂家参考。

主题词：中型空气分离设备 工程设计 安装 裸冷 试车 操作 运行工况

江苏张家港沙钢集团公司作为全国短流程法生产钢铁的典型单位，于1988年引进75吨超高功率电炉之际，配套了杭氧生产的 $1000\mathrm{m}^3 /\mathrm{h}$  空分设备三套。第一套  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备于1990年9月建成投产；为缓和生产用氧的供需矛盾，又在1991年建成第二套“1000"空分设备；随着生产的不断发展，在1994年1月又上了第三套“1000”。

这三套  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备，是杭氧引进联邦德国林德公司技术后的改进产品。其中，第三套“1000”空分设备所配的板翅式换热器是杭氧真空钎接的第一家用户。三套设备自投运以来，虽出现过一些问题，但工况比较稳定，氧气产量在  $1150\sim 1200\mathrm{m}^3 /\mathrm{h}$  ，氩气产量在  $15\mathrm{m}^3 /\mathrm{h}$  以上。现将三套空分设备安装调试中的有关情况作一小结。

# 一、基础

空分塔及透平压缩机基础比较重要，根据抗氧提供的资料，我们又参考《深冷技术》有关空分基础的载文，进行施工。在浇注珠光砂混凝土时，实行计量过磅，严格配比，确保基础的抗压强度及绝热效果。两个大基础在本区域沙土的特定条件下，利用扩大下层面积来提高受压能力。而空压机基础浇灌后，经60吨重量作沉降试验，下沉数值达到满意的效果。

消音坑基础吸取了同行业一起爆炸事故的教训，在消音坑底部增加了一根可排放积水的管道，来观察基础中是否有积水与渗漏现象。同时，可排除因仪表故障，使水冷塔水倒回消音坑的进水故障。

# 二、安装设计

三套  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备的安装设计，我们作了如下改进：

1. 透平空压机供油装置中的油箱加温，建议杭氧采用电加热加温。压力油罐内充气腔，设计成在开车前用仪表空气充气，也可用气瓶充气，并装有止回阀门，以防止油罐内压力油倒回充气管。  
2. 供三套空分设备的仪表气管道，在出过滤器后，选用了  $\phi 30 \times 2\mathrm{mm}$  紫铜管，避免了管道杂质对减压过滤器所产生的堵塞，而造成部分仪表阀门不动作。  
3. 水泵选用了  $485 \mathrm{~m}^{3} / \mathrm{h}$  流量的离心泵四台，并自制水过滤器四台，三用一备，供三套空分设备同时生产用冷却水。  
4. 自制液体排放贮液桶，采用了透平空压机压缩后的热空气去贮液桶汽化液体，省去了长距离的蒸汽管；同时，用热空气吹除阀门上的结霜，减少了操作的不便。  
5. 设计了三套空分设备之间的连接。  
（1）仪表空气。三套空分设备可用仪表

空压机供气，又可用任何一套空分设备的中部环流气经板翅式换热器复热后供气，也可用下塔抽取的压力氮气经复热后供气。三套空分设备既可连接使用，也可单独使用。

（2）压力氮气的连接使用。为考虑空分设备中的液空吸附器、液氧吸附器以及膨胀机的加温，我们将三套空分设备下塔抽出的压力氮气连接起来使用。通过加热炉，进入各加温系统管道，将运行中的一套空分设备的压力氮气对另一套未运行或检修的空分设备提前进行加温，或对空分塔进行气密性试验及查漏。  
（3）成品氮气的连接。本公司氮气用量比较小，考虑到设备的大加温，均采用从第一阶段去除水分的气体。经复热后，进入塔内各部位进行自加温，时间36小时。实现了氮气连接之后，用运行中的某一套空分设备氮气(流量  $1200\mathrm{m}^3 /\mathrm{h}$  ，压力  $18\sim 20\mathrm{kPa})$  ，对另一套空分设备进行全系统的自加温。实践证明，氮气加温范围广，加温彻底，经氮气加温的分馏塔，不但省去了36小时的启动自加温时间，效果也比第一阶段去除水分的气体加温更佳，分馏塔可运行18个月，上、下塔及板翅式换热器也没有发现明显的阻力增大现象。  
（4）空气管道的连接。主要是考虑到空压机故障、空分塔的检修等因素，尽可能地避免不必要的分馏塔大加温，调换空压机轮流供气，维持塔内冷量平衡。  
（5）氩系统的连接。包括三套空分设备中的五台氩压机，三台  $5\mathrm{m}^3 /\mathrm{h}$  膜压机，两台20 $\mathrm{m^3 / h}$  膜压机。氩系统连接成功后，不受机械故障、停机的限制，可灵活地调节机器台数。

# 三、安装过程

三套“1000”空分设备的安装，都由江苏省江都市安装公司承包。整个安装采取了立体作业，双方人员密切配合，在安装周期4.5～5.5个月内便调试出氧。安装中，双方人

员始终坚持高标准、严要求，针对“高速、深冷、禁油、密封、防漏”的特点，分别采取严格的措施。

# 1.透平空压机的安装

安装中，保证了精确的轴承间隙及较高的水平度与同心度，反复找准达6、7次。油箱经反复搓洗，用面粉团沾洗砂粒。对油管反复敲打，酸洗三次。用滤油机，对N32*防锈汽轮机油进行过滤，再经过滤器，打入油箱。清洗40小时后，再用滤油机过滤，继续清洗30小时，确认各进口无杂质后，进入试车。压缩机和电机的底脚螺栓用  $600^{\#}$  水泥浇灌，基础垫铁都是通过磨床磨平后才予使用，以减少振动。

# 2. 分馏塔内焊接

由专职人员进行施工，关键部位实施两把焊枪双面对焊，采用二次焊接的方法，并用X光探伤拍片，在低温下收缩后，氩弧焊缝无一漏气。

# 3. 清洗脱脂

禁油部位做到了绝对禁止沾染油污，严格用清洗剂清洗，并经检查后投用。

# 4. 手动、气动阀门调校

由安装人员对手动阀门、气动阀门逐只进行试压、验收，对自动阀门进行研磨、调校。研磨后的阀门泄漏率，都在资料的允许范围内。

# 5. 容器试压

空分设备配套各容器，按制造厂规定，在保证期内可不经试压直接安装。但应我方要求，安装前仍对各容器进行试压。气密性试验及负压显示表明，容器不同程度地有些问题，后由杭氧技术人员赴现场加以解决。

# 6. 空分塔安装

# 主要采取了以下措施：

测温电阻在第二套空分设备安装时改成了双支电阻。安装前，进行低温液氮测试，电控、仪控系统在安装后，对空分设备的控制实现了全面调校。报警、联锁显示动作灵敏，并作启动模拟动作，使其具备投运条件。

# 四、裸冷与试生产

裸冷不管在冬天或夏天，很难达到裸冷温度的最低标准。但裸冷是对管道、法兰、阀门的一次冷态考验。三套空分设备在裸冷中未发现漏气，其间，我们对法兰螺母作了一次扳紧。

第一套空分设备的启动，由仪表空压机供气，将吸附器、膨胀机、可逆式板翅换热器，下塔、上塔分步进行加温，出口温度高于常温，缩短了自加温时间，仅用了4小时。

第二套、第三套空分设备实施了成品氮气的连接。不用去除水分的空气吹除，而直接用氮气吹除约10小时左右。

三套空分设备都能一次调试成功，启动出氧时间一般在35小时左右，最快可在24小时内出氧。通过连接之后的开车，每次大加温、调试，可节约资金5万元。

# 五、操作体会

可逆切换式板式流程  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备，其仪表气一定要干净，不能带有水分，以保证切换阀门的正常启闭。

# 1. 启动阶段

第一阶段，通过水分冻结区，操作中考虑板式温度与膨胀机的前后温度关系。充分发挥两台膨胀机的最大制冷能力，合理利用板式旁通阀。当板式温度达  $-60^{\circ}\mathrm{C}$  时，不必

急于转环流。当板式温度达  $-80^{\circ}\mathrm{C}$  时，转入第二阶段，用下塔的气体经控制阀去环流，可缩短第一阶段的时间。

第二阶段，冷却设备及各通道。由于此阶段的温度较高，操作速度可快一些，板式温度不会有较大的回升。关键是要掌握下塔顶部抽出的气量去膨胀机和上塔，以加速玉冷凝器的气体循环及冷却。由于及时调节了各阀门，使一种冷量平衡进入另一种冷量平衡，从而达到最快的降温速度。

第三阶段，集中冷却板翅式换热器，飞越渡过二氧化碳冻结区。本阶段转换的阀门较多，时间较短，操作要细心。当阀门转换后，机后温度已达  $-145^{\circ}C$  ，就可飞越渡过二氧化碳冻结区。当机后温度达-  $185^{\circ}C$  时，转入第四阶段。

第四阶段，冷却同第二阶段相同，但必须缓慢进行。在控制板式温度的情况下，尽快地转为气体大循环，同时，应防止膨胀机后产生液体。

# 2. 操作要点

（1）合理掌握三路气体进入膨胀机的调节，维持最大产冷量。  
（2）掌握液空调节阀、污液氮调节阀，纯液氮调节阀的开关度。  
（3）灵活应用氧、氮及污氮气排出阀、及时提高上塔压力，减少上、下塔压差。  
（4）严格控制好各点温度。

在液体积累时，最好不作过大的调节。待液氧面升至  $2000\mathrm{mm}$  以上时，才逐步调节，以缩短开车时间。调节时，逐步观察上、下塔阻力的变化，判断塔板上的精馏工况变化。膨胀空气要在膨胀机减量后送上塔，以不破坏上塔最小回流比为原则。最后投入液氧自循环时必须缓慢，以防液面下降过多。

# 3. 正常工况下的调节

空分设备的出氧量多少，与设备的性能和操作管理有关。出氧量的多少，主要与以

下三点有关：

一是膨胀机的膨胀空气，能否全部送至上塔，参加精馏（关键是膨胀机的产冷量，减少产冷量，能达到冷量平衡，膨胀机的气体可全部入塔精馏）。

二是氮-1阀能否全开，取决于上、下塔的精馏工况。

三是努力提高污氮气纯度，可使1000 $\mathrm{m^3 / h}$  的空分设备，出氧量提高到  $1150\sim 1200$ $\mathrm{m^3 / h}$  左右。

$1000\mathrm{m}^3 /\mathrm{h}$  空分设备，在操作精氩塔时，因氩气气量小  $(15\mathrm{m}^{3} / \mathrm{h})$  ，其操作较麻烦。关

键在于精氩塔产生液体前，不能操之过急。塔内必须稳定，以免出现液体后产生液泛。我公司第一、二套空分设备所配氩塔直径为 $\phi 100\mathrm{mm}$ ，第三台氩塔为  $\phi 120\mathrm{mm}$  （杭氧根据运转实际情况而改），氩气产量可达  $18\sim 20$ $\mathrm{m^3 / h}$  。精氩塔产生液泛时比较难控制，一般只能加温重新启动精馏塔。当然，氩系统的配合也极为重要，以保证气体压力的稳定。

总之，杭氧产的这三套  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备，在张家港运行良好，受到有关考察单位的高度评价。

# 一、设备情况

我厂氮氧站的2-1.67/150型（苏州地区机械厂造)和3Z3-1.67/150型(四川空气设备厂造)氧压机，都是70年代的产品。因使用时间较长，气缸的磨损十分严重。

首先，气缸的上沿在活塞行程以外，而下部又长年磨损，内径增大。所以，在气缸口处形成了一圈凸起的边缘，此凸缘处内径较小。在一、二级新活塞环装入时，往往要将活塞环搭口锯掉一截，才能装入。而在气缸下部活塞环胀开，搭口错开处距离加大，搭口的根部容易断裂。由于三级活塞为皮碗式，伸缩性较小，在装入时，就容易将新活塞损坏。我厂曾使用手提式磨模机，用以打磨此凸沿，效果不很理想。

其次，气缸由于长年磨损，都形成了两头小、中间大的“鼓形”。如，3Z3-1.67/150型氧压机的二级气缸上凸沿处尺寸为  $\phi 118$  mm，底部  $\phi 118.92\mathrm{mm}$ ，而中部最大处的尺寸为  $\phi 120.70\mathrm{mm}$ ，与所要求的二级缸圆柱度小于  $0.035\mathrm{mm}$  相差甚远。另外，气缸内壁粗糙度也大大地增加，纵向的纹路清晰可见。圆柱度和粗糙度的增加，使活塞的磨损大大地加快。

由于以上两种原因，活塞环经常损坏。这样，不仅使设备维修工作量加大；而且，有时由于氧压机充气量严重不足，大量的氧气、氮气放空，造成巨大的浪费。活塞环的频繁更换，也使产品的成本增加。严重地制约了我厂的生产。

# 一、DW型低温胶的一般性能

DW型低温胶粘剂是在低温状态下使用的胶粘剂，由于工作环境的要求（要承受较强的冷冲击、较大的温差和应力等），必然与常规胶粘剂有一定的区别。DW型胶有四种型号：DW-1型，其主体成分为聚醚聚氨酯预聚体；DW-2型，为E-53型环氧改性聚氨酯；DW-3型，为四氢呋喃聚醚环氧树脂；DW-4型，为环氧改性聚氨酯树脂。

# 二、DW型低温胶粘剂在修复  $150\mathrm{m}^3 /\mathrm{h}$  空分设备分馏塔冷凝蒸发器外筒体大面积泄漏上的应用

# 1. 冷凝蒸发器外筒大面积泄漏情况

广东省佛山市铸钢厂  $150\mathrm{m}^3 /\mathrm{h}$  空分设备140/600型分馏塔冷凝蒸发器的外形尺寸为  $\phi 650\times 1140\mathrm{mm}$  ，上、下管板间有  $\phi 8\times$  0.5、长  $600~\mathrm{mm}$  呈六角形排列的铜管1783根，除列管采用T3铜管外，其它均采用H62制成。铜管两端外表，分别与上、下管板间用 $3\mathrm{mm}$  厚的锡铅焊进行密封。

经裸冷试车以后，发现冷凝蒸发器外筒体发生大面积泄漏，其大面积泄漏的裂纹示意图见图4所示。

![](images/1a5f98d7fb225a1df7afd57dd4c7ffc998e85d45e0dc4584c6c05b9ebc015202.jpg)

开始，该厂自行用303焊料（其中含银 $45\%$ ）进行修补，反而使焊后筒壁的裂纹迅速增加和扩大。焊后进行  $0.04\mathrm{MPa}$  表压的气密性试验，发现冷凝蒸发器筒体上部的主裂纹总长度约为  $700\sim 800\mathrm{mm}$ ；在筒体沿抱箍中心上下约  $150\mathrm{mm}$  范围内的整个圆周面有多处细微裂纹。从实际使用效果来讲，该冷凝蒸发器实际上已处于报废。

另则，若采用银焊的修漏工艺，对整个冷凝蒸发器及以后的制氧工况也存在着极大

的潜伏危险。因为银的熔点较锡要高得多，若冷凝蒸发器壳体采用银焊，由于外筒壳壁厚仅为  $1.5\mathrm{mm}$ ，银焊熔点一般在  $720\sim 830$ $\mathbf{^\circ C}$ ，因而焊接时的高温很容易透过  $1.5\mathrm{mm}$  的铜壁而传导给壳体之内的冷凝管，导致冷凝管锡焊因受热熔化而串漏。因而象这类薄壁冷凝蒸发器外筒体大面积泄漏，采用银焊的修复工艺，是不适宜的。

# 2. 选用胶种，确定粘接工艺

针对以上冷凝蒸发器大面积泄漏的特点，修复冷凝蒸发器外筒体泄漏的低温胶不仅要满足  $-183\sim -196^{\circ}C$  深冷状态下运行的工艺条件，还必须满足频繁开车及停车加温时温度交变的考验。其次，低温胶本身要求有较好的流动性和固化性。

用  $\phi 5\mathrm{mm}$  玻璃棒测定，在连续注胶的情况下，DW-3型胶的流速为  $200~\mathrm{mm / s}$ ，适用的温度范围为  $-269\sim 60^{\circ}C$ ，流动性好，适用于膨胀系数不同材料之间的粘接，在室温条件下可以预固化；DW-4型胶起始粘接力高，固化快，韧性和弹性特别优良。

针对以上特性，选用流动性好的DW-3型胶和固化快的DW-4型胶，分别作为通道灌注和表面铜板衬贴之用，是较合适的。笔者应佛山市铸钢厂相邀，前往该厂修补，运用我厂在3200及  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备上应用低温胶的经验，制订修补的技术方案。现将修补情况介绍如下。

在修补上，我们根据泄漏的部位以及泄漏的程度的大小分别进行处理。但对大部分泄漏部位，基本上采用分块粘堵的方法。先对泄漏部分表面进行清除，再对主要裂纹用手锉进行扩口处理，再用医疗注射器向裂纹小通道内灌注DW-3型胶，使裂纹周围都能够均匀地注满胶液。依据分块粘堵的原则，用  $1\mathrm{mmH62}$  板材剪成泄漏部位所需面积、形状基本上相同的贴板，并用手锤在铜板四周打成约  $1\mathrm{mm}$  的翻边。将玻璃纤维布剪

成比铜板四周大出约  $3\mathrm{cm}$  的方块，用分析纯酒精或丙酮洗涤，并让溶剂挥发；再用DW-3型胶浸渍，在泄漏部位粘贴三层，将翻边铜板内层涂复DW-4型胶，粘压在玻璃纤维布上，用铁丝将铜板与冷凝蒸发器紧固在一起。佛山市铸钢厂I、Ⅲ号冷凝蒸发器补漏位置示意图见图5、图6。

DW-4型胶在室温下7天才能固化，在 $60\%$  的温度下2小时即可固化。而DW-3型胶的固化条件更为苛刻，必须控制升温曲线至  $60\sim 70^{\circ}C$  ，才能最终固化，因而加温、保温成为粘补工艺中的主要技术环节之一。当然，从固化技术要求上讲，最好采用整体加温、保温，但是对  $\phi 650\times 1140\mathrm{mm}$  的分馏塔采用整体加温、保温显然是相当困难的。

根据生产现场安全条件，我们因陋就简，接入几块  $220\mathrm{V}$  、1000W的电加热板，作为固化的加温设备，布置在补漏部位的周围及中心位置上；然后再均匀地布置4个测温点，并作好测温记录，以变压调节器来控制电加热板的温度升降速率及恒温时间，使胶层达到良好的固化状态。

佛山铸钢厂140/600型分馏塔冷凝蒸发器外筒体大面积泄漏修复以后，氧气产量为 $150\mathrm{m}^3 /\mathrm{h}$  ，氧气纯度保持在  $99.5\%$  以上，达到原设计水平。证明采用DW型低温胶修补 $150\mathrm{m}^3 /\mathrm{h}$  空分设备的冷凝蒸发器外筒体大面积泄漏是相当成功的，且修理质量很高。

# 三、经济效益及结语

主冷凝蒸发器外筒体应用DW型低温胶粘补工艺进行修补，只用了5天时间、7000元左右。如果更换冷凝蒸发器，整个过程大约需要两个月，约需耗资10万元左右。可见用低温胶粘补的经济效益十分明显。

薄壁壳体的裂纹焊接修复技术难度较大，尤为低温容器。采用覆盖铜板的玻璃纤维布填料及DW型低温胶粘贴方法，可以在生产现场完成大面积补漏，不但简单可行，费用低，而且可满足较快恢复生产的要求，实用性好，具有推广应用的价值。

# 一、引出问题

我厂在生产浮法玻璃中，所用的三套KZON-160/550-I型高纯氮设备，是原江苏吴县制氧机厂1986年8月首批制造的，平时备一开二。自1989年6月31日投运到1993年7月6日第一次停车冷修，一直连续运转。

以上三套设备总的来讲情况尚好，但除设备本身在制造中存在一些质量问题外，在设计中也有许多不足。四年来，不仅给设备的安全运行带来障碍，而且给维修和操作工作也带来诸多麻烦。1991年7月因  $3^{*}$  预冷机出故障，且又不能互换，而导致  $3^{*}$  塔被迫停运。鉴于此，在厂领导的大力支持下，自1991年8月就开始对三套空分设备的系统改造进行探讨，并随后分阶段实施（即分备用状态立即实施的技改项目和冷修期间实施的技改项目两种）。

# 二、空压机的改型

空压机选型和运转的好坏，是关系到空分设备能否正常运行的大问题。我厂原配套三台5L-18/40型空压机，是单纯将5L-16/40型空压机的电机转速由428 r/min 提高到

500 r/min 的改型产品。改型后，空压机在短时间内或间断运行还可以，一旦连续运行，故障便频频出现。其原因是转速增加  $16.8\%$  （产气量也相应增加  $16.8\%$  )后，各级冷却器的散热面积并非相应增加。于是，各级气缸的排气温度增加很快，导致冷却水出口温度常年都在  $45\sim 50^{\circ}C$  。仅1991年度，二级冷却器芯子就更换了十多个，价值15万元。

1992年我们在安徽蚌埠玻璃设计院的帮助下，决定将原空压机取消，改用沈阳气体压缩机厂产的5L-16/50型空压机。该机自1993年8月运转至今，运行平稳，噪声小，振动小，事故率下降  $50\%$  以上。技术参数对比见下表。

原设计中，三台空压机的二级气缸平衡管，通过一个  $\phi 76\mathrm{mm}$  钢管集中送到室外集油池。这样，运行中平衡管内出来的部分油水，就在二级压力(1.4MPa)的作用下，返回到备用机的二级气缸内，造成不必要的空压机部件和原料空气污染。同时，也可能因返回油水过多而造成备用机的击缸现象。为克服相互影响，我们现改为由每台空压机的平衡管网各自送入室外集油池。

# 3. 供水管网的改进

原三台空压机的进水管，只有一个DN120、PN16的闸阀集中控制水源，一旦某台空压机供水管路出现故障时，就无法停水检修。为了克服这种相互影响，就在总阀不动的情况下，在进每台空压机的支管路上各加一只DN80、PN16的闸阀来各自单独控制空压机的供水。

# 四、油水吸附过滤器的改造

1991年10月，在检修纯化器时，发现吸附筒进气口处的少量分子筛变成了黑色，这就是“油中毒”现象。长期使用下来，除了造成分子筛大量“中毒”失效外，当大量的油污进入塔内后，会更危险。查其原因，一是排污不彻底，二是油吸附过滤器选型偏小。为此我们先从操作入手，将排污次数由每小时1次改为半小时1次。半年之后，再次检修纯化器，情况有所改进，但还不彻底。

于是，在1993年7月冷修期间，将原外形尺寸  $\phi 210\times 2200$  的油吸附过滤器改用  $\phi 420$ $\times 2200$  的油吸附过滤器，吸附能力提高一倍以上。1994年4月再次检修纯化器时，没发

现分子筛变色的现象。

# 五、纯化器一预冷机系统的并网串联

原设计中，三套纯化器一预冷机系统之间各自为独立设备，单独使用而不能互换。一旦配套部机发生故障，就需停整套设备或启动备用机。于是我们将三套纯化器一预冷机系统之间增设管网联络，互为备用。一旦分馏塔以外的任何单体设备出现故障，都可随时切换到备用机上。

# 六、蒸发器的技术改造

蒸发器内的50多根蛇形铜管和两端板的连接部位是用低温焊锡(HISnPb 58-2)焊接的。铜管内介质为F12,压力不大于  $0.3\mathrm{MPa}$ ; 铜管外介质为空气,压力为  $1.8\sim 3.7\mathrm{MPa}$ ,即管内外压差有  $3.4\mathrm{MPa}$  之大。加上低温焊锡的强度较低,故在长期运行中,介质空气就穿透焊锡层而进入F12中,致使预冷机失去正常降温能力。从1990年6月至1991年10月,如此串气现象竟发生15次之多。后来,我们先后将三台蒸发器由低温锡焊改为银焊(HIHgCu 52-36),使用两年来,未发生过一次串气现象。

# 七、纯化器和预冷机组的隔热改造

1993年7月冷修期间，我们将纯化器和预冷机组进行了热保温和冷保温（具体是利用岩棉、玻璃布、塑料薄膜等按标准要求进行操作的）。通过近10个月的使用，节能效果可观。保温前每活化一次分子筛需  $4\sim 5$  小时，现缩为  $3\sim 4$  小时。每台加热炉功率为 $54\mathrm{kW}$  ，这样，每天可节约  $108\mathrm{kW}\cdot \mathrm{h}$  ，一年节约  $40\mathrm{kKW}\cdot \mathrm{h}$  ，一年就收回全部投资。

# 八、结论

通过对三套KZON-160/550-I型高纯氮设备的系统改造，不仅改善了设备的互换性，提高了设备的利用率，而且使设备的事故率降低  $50\%$  以上。虽然新选空压机的设计

排气量，比原空压机少  $2\mathrm{m}^3/\mathrm{min}$ ，但由于选型合理，质量提高，分馏塔产量却比以前提高  $10\%$ 。同时，其它技术参数也处在有史以来的最佳状态。

# 无热再生空气净化装置的改进

陈明

（浩良河化肥厂空分车间，黑龙江省伊春市南岔区，153103）

摘要 介绍了该厂两台  $2\mathrm{Z}-6/8$  型仪表空压机配套的无热再生装置，其仪表空气出干燥器后露点太高。当露点在  $-20^{\circ}\mathrm{C}$  左右时，便使室外空气管路内出现“挂冰冻堵”现象。后来，去掉限流孔板，用节流阀单独控制逆流反吹净化空气量，使露点降至  $-40^{\circ}\mathrm{C}$  左右，增加了再生气量。图1。

# 主题词 仪表空气压缩机 无热再生装置 空气 露点 节流阀 调节

# 一、问题情况

在化肥生产中，仪表空气是化工操作气动仪表、阀门的动力来源。我厂1990年上二期工程时，我们空分车间又安装使用了两台2Z-6/8型仪表空压机，配套的无热再生装置是常州能源设备总厂产品。该装置在产品标牌上，既无使用工艺条件，又无仪表空气出干燥器后所应达到的露点。在冬季，当仪表空气露点只约  $-20^{\circ}\mathrm{C}(0.9\mathrm{g} / \mathrm{m}^{3})$  时，便使室外空气管路内出现“挂冰冻堵”现象，造成仪表空气压力过低而引发气动仪表、阀门失灵，连续三年影响我厂化工生产的正常运行。

# 二、原因分析

针对仪表空气出干燥器后露点高的问题，我们最初认为是干燥剂选型不对。1993年按生产厂家的意见，将5A和13X分子筛

改用  $\mathrm{Al}_{2}\mathrm{O}_{3}$  干燥剂后，但效果仍很小。

这之后，我们开始在工艺、设备上查找分析，认识到空气经过吸附剂层时，被吸附的水分与下列因素有关：

1. 空气的压力、温度和含湿量；  
2. 空气的流动速度及吸附剂层的高度；  
3. 吸附剂再生的完善程度。

从实际操作工艺指标和设备结构来看，我们排除了前两种因素，确认问题的主要原因是干燥器自身再生能力差。

此无热再生空气净化装置，是利用短循环周期吸附剂表面吸附理论，加压吸附，减压再生和低流量转换的吸附热原理。其再生过程是瞬间卸压，吸附剂析出水分，并继续以部分净化气体经节流阀和限流孔板，对干燥剂层自上而下的返吹再生(附图)。节流阀

流通孔径  $\phi 5\mathrm{mm}$  ，限流孔板孔径  $\phi 2\mathrm{mm}$  ，这种双重节流虽然能大大降低再生气体的流动速度，有利于再生效果，但它却影响了再生气量的通过，使干燥剂在有限的循环再生时间内得不到彻底再生，仪表空气经干燥器后的露点也就降低不大。

![](images/8b8cb36d8130793f67ce4f47656e85feab47db32908e1d8bb7dae1e204b571cd.jpg)  
元热再生工艺流程图

# 三、解决措施

通过分析，我们计算了再生干燥器所需要的再生气量，总结出逆流反吹的净化空气量，通常为空气量加压吸附的绝压倒数。根据压缩机的性能和我厂仪表气源的要求，末

级压力一般为  $0.6\mathrm{MPa}$  ，所以再生气体的量约为  $50\mathrm{m}^3 /\mathrm{h}$  （即  $360 / 7$  )左右。为此，我们对再生流路进行了两种改进：

第一，去掉限流孔板，只用节流阀调节逆流反吹净化空气量；

第二，取消节流阀，把限流孔板孔径由 $\phi 2\mathrm{mm}$  增大到  $\phi 4\mathrm{mm}$

以上两种改进均达到增加再生气体量的目的。经冬、夏两季大气不同湿度检验，仪表空气露点均在  $-40^{\circ}\mathrm{C}$  左右，满足了工艺要求，再未出现冻堵仪表管路的事故，保证了化工生产的稳定运行。

从两种改进比较来看，用节流阀控制逆流反吹净化空气量与用扩大孔径的限流孔板相比，更能根据空气吸气湿度自由调节再生气体量。在冬季，空气湿度小，节流阀开得小些；夏季，空气湿度大，节流阀开得稍大些，以合理的使用再生气体量。

(1994年7月收稿)

朱景富

（辽宁省抚顺矿务局氮气厂，抚顺市露天区新屯街，113003）

我厂现使用邯郸制氧机厂产  $300\mathrm{m}^3 /\mathrm{h}$  制氧机两套。因生产医用氧，1993年底，从江西制氧机厂购进两台PLK-25.83/14.2-5.35型无油透平膨胀机，替下了原配套的有油透平膨胀机。运行两个多月后，由于膨胀机空气过滤器堵塞，加之入膨胀机的空气、轴承气和密封气不干净，造成膨胀机卡机。清洗轴承、密封器管路后（在膨胀机前加过滤器），重新开机，开机至液体出现时又发生卡机。几次修复试车，都发生卡机现象，至此全线停机。

# 一、故障现象

1994年5月24日15时开机，25日早4时没有出现液体，但此时各部温度都已低于正常温度。操作工人便逐步关小节-1阀至1/5转来提高高压压力，增大高中压差。早5时多，氧、氮气管路出汗并结霜。调整后，逐步开大节-1阀，不到两小时，液面升至1000mm水柱以上。此时开始关小通-14阀，使膨胀后冷量送入下塔；同时关小节-2、节-3阀，此时液面继续上升，甲膨胀机转速不稳，并出现波动，停开甲机。

由于液面继续上升，乙机只开至2万 $\mathrm{r / min}$  ，关通-14阀、节-2、节-3阀，此时液面还呈上升趋势，继续运行，便出现卡机现象。马上开动甲机(停约10多分钟)，当甲机转速升到  $3\sim 4$  万  $\mathrm{r / min}$  时，甲机也出现卡机现象。当时，膨胀前温度为  $-145\sim -147^{\circ}C$  膨胀后温度为  $-180^{\circ}C$  ，主换热器各温度也等于正常温度。

# 二、原因分析

# 1. 冷量利用不充分，冷损过大

因节-1阀开度过小，塔内返流冷量得不到充分利用，便从主换热器排出；而入塔空气在主换热器中得不到充分冷却，使下塔冷量不足，无液体聚集。而入塔空气大部分经膨胀机制冷后，经通-14阀与塔内返流液体汇合，再经主换热器排到塔外，使氧、氮管路“出汗”并结霜。塔内冷损过大，长时间无液体出现。

# 2. 开大节-1阀后，下塔冷量过盈

开大节-1阀，且关小通-14阀后，由于膨胀后温度和主换热器各部温度过低，入塔空气被急速降温，下塔冷量过盛，不到两小时，液面就升至  $1000\mathrm{mm}$  水柱以上。

# 3. 膨胀机前空气温度过低, 空气被部分液化

因下塔液体出现较晚，通-14阀的关闭时间相应过晚，膨胀机产生的冷量经通-14阀和主换热器排出。主换热器各部温度低于正常温度，使主换热器后、膨胀机前的空气温度过低。膨胀前空气已有部分被液化，一部分液体随着气体通过膨胀机。膨胀机工作轮，受到液体的冲击产生波动，转速不稳。

# 4. 液化空气急速冲向膨胀机工作轮，严重摩擦轴承

甲机出现波动时，轴承可能有轻微擦伤。停机10分钟后重新启动，聚集在机前管路中的液化空气急速冲向膨胀机工作轮，轴承被严重摩擦，甲机卡机。

# 三、措施

重新开启乙机后，开大节-1阀，将膨胀前温度控制在  $-130\sim -135^{\circ}C$  ，膨胀后温度控制在  $-155\sim -160^{\circ}C$  ，逐步提高转速至 $56000\sim 60000r / min$  ，膨胀机运转平稳。停机修复轴承后启动，运行正常。

四、结论

由于操作不当，膨胀机前后温度控制过低，膨胀前空气呈气液态进入膨胀机，使机芯运转失去平衡，造成卡机。因此，在使用无油膨胀机之前，一定要详细掌握有关技术要求，各项技术指标要严格控制在技术要求范围内。

（1994年10月）

# 空压机液击产生的原因及工艺改进

莫家梁

（三明化工厂电石厂，福建省三明市，365001）

摘要 本文论述了  $300\mathrm{m}^3 /\mathrm{h}$  空分设备所配套的空压机产生液击事故的原因，并从工艺上提出了预防液击产生的措施，以及今后开、停车应注意的事项。图2。

主题词：小型空气分离设备 空气压缩机 液击 分析 工艺改进

# 一、事故情况

1993年11月1日8:00, 我厂  $300\mathrm{m}^3/\mathrm{h}$  空分设备  $1^{\#}$  机组准备开车, 操作工已做好开车前准备工作。当电工按下开车电钮时, 发现有水从一级排气管道法兰处喷出, 并发出一声异常的响声, 电工马上紧急停车, 钳工拆开一级排气管道, 大量的水即从管道涌出, 并发现水中有十几个气阀弹簧。拆下一级进排气阀, 发现气阀阀片被打碎、打弯, 气阀阀座也出现很多裂缝, 一级气缸积水严重, 一级活塞杆断裂。拆下端盖, 发现端盖有裂缝, 活塞体被打碎。拧开一级冷却器最下部的旋塞, 有大量的水排出。拆下二、三级进排气阀, 也发现气阀有不同程度的损伤。整个现状“惨不忍睹”。它是空分车间最大的一次设备事故, 直接损失和间接损失达十万元。

# 二、事故的原因分析

显然，这么严重的设备损害，是由于空压机产生液击所造成的。液击是由于空压机的

一级冷却器内漏，造成冷却水“跑”入气缸，被压缩而产生的。我们知道，水是不可压缩的，在开车的瞬间，活塞对水产生压缩，压缩水给活塞一个强大的反作用力，活塞杆受到强大的挤压应力，超过了应力极限而将活塞杆压断，活塞体也被打碎。同时，气缸端盖也由于受到压缩水强大的挤压应力而出现裂缝，气阀阀片不同程度地被打弯、打碎，阀座出现裂缝。

为什么会造成这样的事故呢？我们先来分析一下冷却器的结构。一级冷却器为管壳式换热器，冷却器芯子由236根  $\phi 20\times 2\mathrm{mm}$ $20^{\#}$  无缝钢管与上、下管板焊接而成，中间接有圆缺形折流板和定距管，冷却管按正三角形排列，管外走冷却水，管内走空气，并由上、下管箱气隔板将一级隔为4程，二、三级隔为6程，进排气管口均在下管箱上。

为什么一级冷却器使用还不到一年就漏得这么厉害呢（二、三级冷却器也出现过内

漏，只是没有造成事故）？笔者认为，这跟换热管的材质和冷却水的水质有着密切的关系。从产生腐蚀的情况来看，材质是不理想的。同时，冷却水使用的是工业循环水，硬度高并含有一定的酸性，容易在换热管上结疤、结垢，对钢管产生腐蚀，而影响换热管的寿命。同时，我们发现一级冷却器管内的工作压力为  $0.176\mathrm{MPa}$ ，而管间水的工作压力为  $0.242\mathrm{MPa}$ ，即管外的工作压力比管内高。也就是说，无论是正常开车还是停车，冷却水都易从管外进入管内。如果停车的话，水就会从冷却器倒流到气缸，开车时就会产生液击的事故。

# 三、工艺改进

1. 从上面的分析得知，换热管的材质不好和冷却水的水质不好是造成冷却器内漏的原因。因此，我们将一级冷却器换热管全部更换为优质无缝钢管，以期获得较长的寿命。而对冷却水，建议使用软水，严格控制冷却水的  $\mathsf{pH}$  值。  
2. 我们注意到冷却器的下部有一个用法兰盲死的水压试验排出口（冷却器做水压试验时的排水口），最下部有一个旋塞（用来排污用的），如果在这两个地方各装一个阀

门，停车时将管内、管外的水排尽，则可以有效地防止冷却水进入气缸而产生液击。同时对二、三级冷却器也进行这样的改造（见图1、图2）。

1. 开车前，将阀1、阀2打开，待开车后，将阀1、阀2关闭。

# 氧压机二级盘管和三级气缸爆炸事故剖析

周福亮

（国营四川新都电子动力厂，新都县电子路，610500）

# 一、事故经过

1994年12月21日，维修工将新买的二级冷却盘管更换到3Z3-1.67/150型氧压机上，试车正常，操作工开始充氧灌瓶，约4小时后停车。1994年12月22日凌晨1:25操作工重新启动氧压机，发现一、二、三级排气压力均升不起，停车后检查，发现三级活塞环完全损坏，气缸被拉毛。在22日白天召开的事故分析会上，根据气缸内部、活塞杆均未发现积炭、甚至无熏黑的种种痕迹，判断是长时间缺润滑软化水，气缸内形成高温将聚四氟乙烯活塞环熔化所致。新购冷却盘管是从成都某生资部买来的，买回后未用四氯化碳清洗。厂长责成维修部门加紧抢修，并要求用四氯化碳清洗盘管。

12月23日维修人员用四氯化碳将盘管清洗了三遍，然后装配到氧压机上，其它各零部件也都认真的用四氯化碳清洗。12月24日凌晨1:30，抢修完，试车正常，充了两排氧气瓶后，当第三排氧气瓶充至  $13\mathrm{MPa}$  且充氧工将该排氧气瓶倒换至另一排时，突然听到一声猛烈的爆炸声，这时操作工观察充氧台的氧气压力为  $7\mathrm{MPa}$  。

此次事故，二级冷却紫铜盘管爆裂三处，三级进排气阀压盖及三级安全阀因爆炸力被“打飞”，在水箱面对盘管爆裂的方向三处凸出并形成裂口，气水分离瓶抱箍被挣断，可见威力之猛，幸未伤人。

1994年12月22日和24日连着发生的两起

事故，使人们吃惊。如果说，未用四氯化碳清洗盘管，出了第一次事故，那么，第二次洗了三遍反而出了更大的事故，原因何在？从事制氧工作多年的老同志也有些惶惑，并感叹用四氯化碳清洗了的反而发生更严重的事故，还从来没遇到过。

笔者参加过杭州制氧研究所举办的小型制氧机学习班，从理论、经验、实际情况以及综合分析群众的意见，总结出以下的事故原因、教训及防范措施，供大家参考、探讨。

# 二、事故原因

1. 所用四氯化碳是清洗过分馏塔的，微量油在四氯化碳中，肉眼是观察不到的。将该四氯化碳倒一些在手上，迅速自然蒸发后有微粘的感觉。  
2. 以往本厂所购的盘管是四川空分设备厂产的，而这次恰恰是从成都某生资部购置的，其产品来源有疑点，根据爆炸后的铜管某些碎片结有一层牢固的垢物，可证明铜管内有不明杂质存在。  
3. 该盘管被四氯化碳清洗三遍后, 反而将原先牢固粘附在管壁上的杂质疏松了。  
4. 该盘管虽经清洗，但洗后未用高压氮气吹除、吹干。  
5. 第一次事故后，三级进出口阀门未作处理就原封不动的装上了，第一次事故活塞环损坏后的某些微粒杂质，可能存在于三级气缸及管路中。  
6. 倒换充氧台阀门过快，在降压的过程

中，氧气流速突然增快产生静电火花。

# 三、防范措施

针对事故发生的原因，我们制定了以下防范措施：

1. 今后凡是接触氧气的设备、管路必须清洗和吹除（新购置的也不例外）。  
2.凡是通过氧气的设备、管路试运行时，必须先用氮气吹除，氧压机必须先压氮气。

3. 采购零备件要向对方说清楚工艺要求，并要求对方出具生产厂家产品合格证明及技术依据，以防假冒伪劣产品。  
4. 充氧倒换阀门的过程要缓慢一些，切忌突然开大阀门。

笔者认为只要认真落实以上防范措施，就能杜绝类似事故的发生。在这以后的我厂的运行事实也证明了这一点。

（1995年2月7日）

# 高炉全氧炼铁

高炉全氧炼铁工艺，采用氧气代替热风，喷煤量可达  $300\sim 350\mathrm{kg / t}$  铁，焦比可达  $150\mathrm{kg / t}$  铁。德国、加拿大、日本已提出不同流程，日本钢管公司已在  $3.9\mathrm{m}^3$  试验炉上进行试验。高炉全氧炼铁，其优点，生产率可大幅度提高(可提高  $30\% \sim 200\%$  )，高炉煤气成分变化，由于氧气代替空气鼓风，可大幅度提高间接还原度；可提供高热值煤气(煤气中无氮气)。

日本人认为下个世纪的炼铁工艺是： $50\%$  的铁水由氧气高炉生产， $20\%$  的铁水由熔融还原法生产。

主题词：透平膨胀机 叶轮和导流环 损坏 原因 分析

# 一、事故经过

我厂1*空分设备，是大庆石油化工总厂30万吨乙烯工程配套装置，是80年代初引进德国的大型空分设备，于1986年投产。ETG240NS型膨胀机共两台（位号  $\mathrm{T}_{1} / \mathrm{T}_{2}$  一开一备），自投产以来，一直运行平稳。1994年12月3日，因电厂  $4.0\mathrm{MPa}$  蒸汽压力，下降到 $2.4\mathrm{MPa}$ ，造成空压机(蒸汽透平驱动)停车。接着，使各配套部机相继停止运行，  $\mathbf{T}_2$  膨胀机也是在这次事故中停下来的。这时，发现其叶轮和导流环损坏。

为供分析原因参考，现将该进口膨胀机主要技术参数列于下面：

空气流量 7300  $\mathrm{m}^3 /\mathrm{h}$

入口压力 0.51 MPa

出口压力 0.14 MPa

间隙压力 0.27 MPa

入口温度 -120℃

出口温度 -160°C

膨胀机转速 17035 r/min

发电机转速 3000 r/min

发电机功率 185 kW

# 二、事故分析

因电厂蒸汽压力由  $4.0\mathrm{MPa}$  下降到2.4

MPa，造成空分设备所配空压机停车。而蒸汽压力恢复正常，空压机开车后，出现  $\mathbf{T}_2$  膨胀机盘不动车，只好把  $\mathbf{T}_1$  备用膨胀机开起来。针对  $\mathbf{T}_2$  膨胀机盘不动车，首先对润滑系统进行检查，发现油压为  $0.25\mathrm{MPa}$ ，在工艺设计值内，表明辅助油泵正常。但  $\mathbf{T}_2$  膨胀机启动不起来，作为机械故障初步认为：一是断油造成轴瓦烧毁，二是叶轮被冒脱落卡死。遂决定解体检查，对这两方面进行验证。结果，轴瓦没有损坏，叶轮被冒没有脱落，但叶轮与导流环研死损坏。下面是  $\mathbf{T}_2$  膨胀机叶轮导流环研死损坏示意图。

根据叶轮与导流环损坏的现象，我们又

进一步进行可能性分析，发现  $\mathbf{T}_2$  膨胀机开车时超速轴端振动过大，叶轮叶片与导流环发生磨损。经对膨胀机运行记录检查，此膨胀机是此次事故中停下来的。另一方面，系统中可能进去了比较硬的东西，把光滑的导流环表面刮起毛刺，接着叶轮叶片与导流环上的毛刺进行磨损，然后把磨掉的铝合金受进口导叶的作用形成一个环状物聚集于膨胀机的入口周围处(图中4)。但根据导流环和叶轮被磨掉的铝合金情况， $\mathbf{T}_2$  膨胀机运行时间需超过5分钟，如果进去硬东西，高转速叶轮叶片肯定是被打破，而从被磨掉的铝合金聚膨胀机入口处来看，也是解释不通的。膨胀机运行时入口压力高于出口压力，磨掉的铝合金，应该随着气流从膨胀机出口流出去，因此进去硬东西的分析是不成立。

那么， $\mathbf{T}_{2}$  膨胀机叶轮和导流环损坏是什么原因呢？针对被磨掉的铝合金跑到膨胀机入口这种现象，使我们对问题分析转向发电机的自动控制系统方面。膨胀机工作时带动发电机，发电机控制膨胀机转速，反过来当供给电力时可以作为电动机。

当  $\mathbf{T}_{2}$  膨胀机出入口阀关闭时，膨胀机是在运行的，这时膨胀机变成了压缩机，使出入口改变了， $\mathbf{T}_{2}$  膨胀机蜗壳和管路里的空气被压缩，气体温度迅速升高，叶轮过热及摩擦热，使叶轮叶片迅速热胀，致使叶轮与导流环之间没有间隙（实际间隙  $0.5\mathrm{mm}$ ），从而发生磨损而损坏。

膨胀机变成压缩机是发电机引起的，膨胀机停机时出入口阀关闭，操作说明要求在

1分钟内停下来，各自动系统的控制开关回到备用状态。从图中看，被磨掉的铝合金聚集膨胀机入口，使叶轮和导环损坏。这时发电机变成电动机驱动膨胀机，自动控制系统出了故障，膨胀机出入口阀关闭，转速下降，发电机没有脱闸（膨胀机达到一定转速时发电机合闸和外网供电系统并网）。外网供电系统继续供发电机电力，从而发电机驱动膨胀机运转，直至叶轮与导流环卡死，发电机（电动机）过流保护系统动作停机，这就是造成膨胀机叶轮和导流环损坏的原因。

# 三、事后思考

1. ETG240NS型德国进口膨胀机叶轮和导流环的损坏，原因是膨胀机自控元件失灵造成的。当膨胀机出入口阀关闭时，膨胀机应迅速停下来，接着发电机与外网供电系统应立即脱闸。但自动控制系统元件失灵，致使发电机没有与外网供电系统断开，发电机继续驱动膨胀机运转，造成膨胀机叶轮和导流环磨损卡死事故。  
2. 膨胀机叶轮和导流环的损坏事故，是与操作分不开的。操作规程规定，膨胀机停机时，操作人员要及时到现场检查电器及自动控制系统开关闭合情况，膨胀机是否处于停车状态。  
3. 这起事故, 使我们对自动控制的机械设备有了深刻认识。当机械部件损坏时, 要把自控系统联系起来分析。同时我们进一步认识到, 要重视自动控制系统的检修, 要定期检查自控系统元件, 保证自控系统灵活使用。

# 制氧工问题解答（四则）

1. 问：请讲解 T-S、h-T、h-S 图查法要点？（甘肃白银读者问）

答：气体热力学参数主要有压力（P，绝压）、温度(T，K)、焓(h，千卡/公斤分子)、熵(S，千卡/公斤  $\mathrm{^\circ C}$  )。为查找和使用方便，将它绘成T-S、h-T、h-S三个图。T-S图的纵座标是T，横座标是S。h-T图、h-S图只不过座标绘法不同，仅是座标的转变而已。T-S图有空气、氧、氮、氩等几种，所以首先不要查错图表。因h-T、h-S图是从T-S图转变来的，下面介绍T-S图查法要点。

图中，向左斜线代表等压线（P），向右斜线代表等焓线（h），水平线代表等温线（T），垂直线代表等熵线（S）。从复杂的线簇中理出这四条线，绘在一张小图上，就容易看懂了。下面举例讲解查T-S图方法。

![](images/de715425141b4cfae1fb75fc67b9b628d9d7fbba69c68bce51cc6bb97d5b17c9.jpg)  
T-S 图查法示意图

例如：已知空气  $\mathbf{P} = 300$  绝压， $\mathbf{T} = 300$  K

求：h、S查法。

解： $\mathbf{P} = 300$ 、 $\mathbf{T} = 300$  两根等压与等温线交点  $\mathbf{a}$  是空气的工况，穿过  $\mathbf{a}$  沿等焓线向右就可查得  $\mathrm{h} = 2960$ （千卡/公斤分子），沿垂直线向下在横座标即可查得  $\mathbf{S} = 18$ （千卡/公

2. 问：我厂有一台FON-50/100型分馏塔，近两个月来，液空液面不指示，经检查各接头良好，无漏气现象，液面计畅通，上下阀气量正常，换用液氧液面计也是不指示（但液氧液面是正常指示的），我想是否是膨胀机的毛病？

3. 问:  $150 \mathrm{~m}^{3} / \mathrm{h}$  空分设备启动时温度正常, 而到运转正常时  $\mathbf{T}_{1}$  温度偏低,  $\mathbf{T}_{3}$  温度偏高。应如何解决? (福建惠安读者问)

答：首先谈  $\mathbf{T}_{1}$  和  $\mathbf{T}_{3}$  温度的关系。  $\mathbf{T}_{1}$  是膨胀机进口的空气温度，  $\mathbf{T}_{3}$  是高压空气节流阀前的空气温度。  $\mathbf{T}_{1}$  最低达  $-146\%$  。而  $\mathbf{T}_{3}$  是  $-144^{\circ}C$  。  $\mathbf{T}_{1}$  最高达  $-135^{\circ}C$  ，显然温度计指示不正确。因为  $\mathbf{T}_{1}$  温度代表热交换器I后的空气温度，高压空气经热交换器Ⅱ进一步经返流气体冷却，温度只会降低，绝对不会升高，这只能说明温度计需校验。

再谈温度指示不正确的情况下，如何操作？对  $150\mathrm{m}^3 /\mathrm{h}$  空分设备来说，启动时  $\mathbf{T}_2$  温度要控制，当  $\mathbf{T}_{3}$  温度达  $-155^{\circ}C$  时，应以  $\mathbf{T}_3$  温度为准，只要将  $\mathbf{T}_3$  温度控制在  $-155\sim -165^{\circ}C$  ，不必考虑  $\mathbf{T}_2$  温度。  $\mathbf{T}_1$  温度是为控制  $\mathbf{T}_2$  、  $\mathbf{T}_3$  温度作参考的，因  $\mathbf{T}_1$  温度变化比 $\mathbf{T}_2,\mathbf{T}_3$  温度快，如  $\mathbf{T}_1$  温度上升了，则意味  $\mathbf{T}_3$  温度即将升高，就要提前采取措施，以保持 $\mathbf{T}_3$  温度的稳定。

$\mathbf{T}_{3}$  温度控制是有阶段性的，即从打开节-1阀到降压结束这段时间应严格控制。当降压结束转入正常生产时，不必考虑  $\mathrm{T}_{1}, \mathrm{T}_{2}, \mathrm{T}_{3}$  温度的高低，只要根据液氧液面的变化来操作。换言之，液氧液面稳定是中心环节。若液氧液面上升，在凸轮全开的情况下，只要开大节-1阀，不必考虑温度；当液面下降时，关小节-1阀来维持。在制取部分液氧或液氮时，应关小节-1阀和凸轮来提高液氧液面。

从上的分析来看，贵厂空分设备在正常工作时，只要液氧液面稳定，产量、纯度正常，不必人为去控制  $\mathbf{T}_{1},\mathbf{T}_{2},\mathbf{T}_{3}$  温度。

（杭氧公司质监部 叶必楠）

# 氧吧——时尚的氧气疗法

1994年4月14日《杭州日报》报道杭州市上城区人民医院，增加一项特殊服务——吸氧消除疲劳的“氧吧”。据了解，该医院“氧吧”主要面向病人，如气管炎、心脏病人及孕妇等，效果相当不错。笔者一次去采访时看到一位怀孕反应激烈的年轻女士在吸氧，自说吸后缓解了不少，同时也是为了下一代。另一次去采访时看到一位六七个月的孕妇，为保胎与下一代健康、聪明在吸氧，并讲了其姐姐怀孕，因跌了一跤快要导致流产，后采取吸氧静养保胎，不但保住，而且出生的小孩身体很好。因此该女士从其姐姐的实例，很相信吸氧的疗效，也到医院每天吸氧。据该院医生讲，吸氧后病人自我感觉都很好；当身心倍感疲劳时，吸氧可消除疲劳，焕发精神。这对超常脑、体劳动者，如企业家、知识分子、驾驶员、运动员，以及孕妇、脑功能不全者、手术后康复者，吸氧都有益处。该医院“氧吧”，吸氧1小时收费10元。鉴于医院经济条件，该“氧吧”设施还较简陋，但价格便宜。他们还在考虑，为满足病人要求，在经济条件许可之下，将配置轻质气瓶，上门服务，送“氧吧”到家庭。

随着环境要求、消费水平的提高，杭州又出现了设施较好的洁静幽雅的保健氧吧——“卡依波氧吧”，已于1994年11月18日开始营业。

# 缺氧窒息的危害

空气中的氧通过肺胞进入血液，靠心脏跳动再输送到人的全身。人体内需要氧气量最多的部位是脑细胞，当供氧不充分充足时，就会出现缺氧窒息症状。缺氧的症状除脸色苍白或潮红、脉搏和呼吸次数增加，呼吸困难、眩晕、头痛等外，严重时还会出现意识不清、抽筋、呼吸停止、心脏停搏等症状。脑细胞一旦中断氧气供给，就会引起不可逆变化，严重会逃留下记忆丧失、痴呆，以至变成“植物人”。

# 西昌卫星发射人员误吸氮气险遭不测

担任卫星总装工作的郑德兴、缪世强和李树嵩三位同志，在东方红三号卫星发射进入倒计时前5小时，因误吸氮气而中毒倒地。

1994年11月29日22时左右，在西昌卫星发射中心  $15\mathrm{m}$  高的发射塔七层平台上，试验队员准备进入火箭第三级腹部内舱排除故障，更换仪器。按操作规程，进舱前应戴上面部氧气罩，并作试戴性的深吸气。当试吸气两至三口时，他们凭经验感受不对，郑、缪两位队员当即晕倒在高空平台上。发射阵地立即查明是地面操作手将纯氮气误经压缩空气输送管道，由地下输送到塔架七层平台的氧气面罩内，导致试验人员吸入纯氮气，造成脑缺氧窒息。李树嵩吸入的是弥漫在空气中的氮气，中毒反应较轻。

这次如果不象郑、缪这样的老工程技术人员按规范试戴面罩作试吸气，而是戴上面罩直接进入直径  $3.35\mathrm{m}$  的弹舱内，后果将不堪想象，致险人员也会进去一个牺牲一个（氮气会致人窒息死亡，而且中毒很快很厉害——编者）。正是由于郑、缪两位老技术人员规范化的职业动作，及早制止了操作错误，减少了国家的损失，保障了卫星发射全盘计划的执行。且这三位同志经短暂治疗、休养后，均很快康复，但教训深刻。
