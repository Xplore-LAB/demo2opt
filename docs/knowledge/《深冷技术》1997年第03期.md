# 抚顺某厂“6000”空分设备爆炸 死亡4人

1997年5月16日上午9时05分，辽宁抚顺某化工厂引进的  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备，发生一起极为严重的爆炸事故，死亡4人，受伤30多人。爆炸中心为主冷凝蒸发器，威力巨大，呈粉碎性爆炸，铁块飞出五六百米远处，致人伤亡（死亡4人中的1人，就是被飞出400米以外的一块大铁板击中头部致死的抚顺石油二厂的一名车工），方圆1500米内房屋的门窗玻璃遭到严重震坏。

事故原因初步分析为碳氢化合物致爆，有关部门正组织人员作深入的调查分析。

# 空分设备通过方案设计评审

湘潭钢铁公司  $15000\mathrm{m}^3 /\mathrm{h}$  空分设备和上钢五厂  $12000\mathrm{m}^3 /\mathrm{h}$  空分设备，分别于1997年4月9日和11日在杭州制氧机集团公司通过方案设计评审。

这两套空分设备均采用全低压常温分子筛吸附净化带增压透平膨胀机的工艺流程，在许多关键性的部机上采用了当今世界空分设备最先进的技术，如上塔、粗氩塔采用高效规整填料；主冷采用新型高效换热单元；制氩采用全精馏无氢制氩；仪控系统采用DCS集散系统。整个装置流程先进，适应变负荷能力强，提取率高。

评审组成员及与会的有关专家，对两套空分设备的工艺流程设置、技术经济指标进行了认真讨论和分析，并对其流程的可行性和合理性作了充分肯定。在流程中因用户需要，应用了一些独特的新技

术，在评审会上进行了重点讨论。对湘潭钢铁公司的  $15000\mathrm{m}^3 /\mathrm{h}$  空分设备，因空气预冷系统不配冷冻机后引起的流程设置变动问题和参数选定问题，进行了详细的分析比较。专家们一致认为，这是一次流程简化、方便操作的新尝试，是完全可行的。对上钢五厂  $12000\mathrm{m}^3 /\mathrm{h}$  空分设备中首次配置高纯氧塔及空冷系统中应用填料塔的技术问题，讨论认为技术是可行的，但要尽快完整其计算方法。

# 几种空分制氩流程简介和探讨

王靖昭* 周智勇 何子田

（杭州制氧机集团有限公司技术开发处，杭州东新路90号，310004）

摘要 详细介绍了粗氩经吸附制取纯氩、粗氩经加氢除氧精馏除氮制取纯氩和填料塔全精馏制氩三种制氩流程的流程特点、优缺点及目前各流程的使用情况，并提出了今后制氩流程的发展趋势。图6表1。

主题词：空气分离设备 粗氩 纯氩 吸附 加氢除氧 全精馏 填料塔 流程

随着冶金、化工及电子等工业的不断发展，空分用户对空分设备的需求不只局限于氧、氮，用户还需大量的氩气，甚至其它各种稀有气体；而且用户对氩的提取率的要求也在日趋上升。国产设备为了适应这一发展需要，在制氩方面作了许多改进，尤其是80年代后期，随着分子筛增压流程的开发成功，氩系统无论在提取率还是在自控方面都有了更进一步的发展。

目前，国内空分制氩设备的流程一般分为三种：粗氩经吸附制取纯氩、粗氩经加氢除氧精馏除氮制取纯氩和最新发展起来的填料塔全精馏制氩。

# 一、粗氩经吸附制取纯氩

图1为吸附制氩的流程图。氩馏分从上塔馏分抽口抽出（组分一般为  $90.8\% \mathrm{O}_2$  、 $9.14\% \mathrm{Ar}$  、 $0.06\% \mathrm{N}_2$ ），进入粗氩塔进行氩一氧分离，粗氩塔的冷源为来自下塔并经过冷后的富氧液空。为了保证吸附制氩的可靠性及稳定性，一般粗氩塔出口粗氩的组分取氩  $>98\%$  、氮  $< 0.5\%$  、氧  $< 1.5\%$  ，纯度

大于  $98\%$  Ar的粗氩气经管道直接送入吸附器。考虑到需吸附氧氮和本身的再生问题，吸附器设置两组并切换使用，每组两只，一只吸氮，另一只吸氧。由于分子筛对氧氮吸附的选择性，一般吸氮的这一只内装5A分子筛，吸氧的这一只内装4A分子筛。根据吸附剂的吸附机理，粗氩先进入5A分子筛进行吸氮，再进入4A分子筛进行吸氧，吸

吸附制氩的优点，在于工艺流程简单、操作方便、成本低。但是氩的纯度只能达到 $99.99\% \mathrm{Ar}$ ，要想制取纯度大于  $99.999\% \mathrm{Ar}$  的高纯氩气就很不容易。

吸附制氩的关键在于吸附器本身的设计和制造，由于吸附器需要在低温(90K)下工作，又要在高温（423K）下再生，故对吸附器的结构设计较难确定。目前国内的流程一般均采用液空作为冷源，其结构有夹套式、列管式和复合式(前两者的组合)。对于氩提取量较小的设备，采用同样的气体流速，则吸附器的本体直径较小，这样不论采用何种结构方式均可达到比较理想的冷却效果。但对于提氩量较大的设备来讲，其设计直径势必较大，这样无论采用何种冷却结构，要做到均匀冷却是相当困难的。

因此，由于这些局限性，使得低温吸附制氩的发展受到一定的制约。目前国内吸附制氩设备的提氩量尚未超过  $10\mathrm{m}^3 /\mathrm{h}$  ，所配的空分设备一般也限于  $500~\mathrm{m}^3 /\mathrm{h}$  以下（国内主要在  $150~\mathrm{m}^3 /\mathrm{h}$  空分设备上采用过）。由于吸附制氩设备的成本较低，所以系统所采用的控制及分析的装置也比较原始，一般都采用手动控制及手动分析。

# 二、粗氩加氢除氧精馏除氮制取纯氩

这是目前国内空分设备制取氩的主要流程，自80年代引进德国林德公司设计技术以后，该制氩流程设备得到了不断的扩充和开发，无论是设备的结构还是流程的组织都有了较大的革新。加氢除氧制氩流程共分三部分：粗氩提取系统、加氢除氧系统和纯氩提取系统。从目前国内的设备来看，粗氩提取系统的流程基本未变，还是利用粗氩塔制取纯度为  $96\%$  Ar 的粗氩；纯氩系统一般采用气态进料和液态进料两种流程，加氢除氧系统的原理基本一致，但也有一定的区别。

# 1. 气态进料流程

图2为气态进料流程示意图。氩馏分从上塔抽出，其组分为  $90.8\% \mathrm{O}_2$  、  $9.14\% \mathrm{Ar}$  、  $0.06\% \mathrm{N}_2$  ，进入粗氩塔底部；利用液空做冷源，在粗氩塔内发生精馏，精馏的结果是在粗氩塔的顶部得到气体组分为  $96\% \mathrm{Ar}$  、  $2.5\% \mathrm{O}_2$  、  $1.5\% \mathrm{N}_2$  的粗氩气。此粗氩气经空分主换热器复热后进入加氢除氧系统，加压到  $0.2\mathrm{MPa}$  ，与氢气(纯度大于  $99.5\% \mathrm{H}_2$  的电解氢气)混合，经阻火器和纯化器后除去氧组分；再经空气冷却器和水冷却器分凝部分水蒸气，由汽液分离器分离水分；然后进入干燥器除去微量水分。除去氧组分的氩气称为工艺氩，干燥器出口的工艺氩组分一般为  $96\% \mathrm{Ar} + 2.5\% \mathrm{H}_2 + 1.5\% \mathrm{N}_2$  ，含氧约为 $2\mathrm{ppm}_{\circ}$

将工艺氩送入冷箱，经氩换热器冷却到饱和温度点附近（一般过热  $4^{\circ} \mathrm{C}$ ），再送入精氩塔中部（氩换热器的冷态物流为精氩塔蒸发的氮气和余气）。工艺氩进入精氩塔中部沿塔上升，经精氩冷凝器后，高沸点的氩组分被冷凝，而低沸点的氮和氢组分则随余气排出。精氩塔的冷源为液氮，压力约0.035 MPa，温度约79.5K。另外精氩塔还设有蒸发器，冷凝下来的液氩经蒸发器后被蒸发。这样在精氩塔中不断冷凝、蒸发、精馏，结果在精氩塔底部获得纯度大于  $99.99\%$  Ar的高纯液氩，如果需要也可改产气氩。精氩蒸发器的热源为下塔来的压力氮气。

气态进料流程的特点是工艺简单，操作简便，产品纯度可达到  $99.999\% \mathrm{Ar}$  。但是此种流程的过量氢气无法回收，而且塔顶的余气排放量也较大（一般约为粗氩量的  $7\%$  ）。所以氩的提取率相对来讲不是太高。下面就系统中的主要设备结构加以介绍。

# (2）粗氙塔体

目前国内空分设备的粗氩塔，一般采用环流塔板、对流塔板或导向塔板。

环流塔板采用同向流，其塔板效率比较高。但由于环流塔板有一个中心筒而使塔径变得很大，而且其结构较复杂，对于小设备尚可，对于大的提氩设备就不太适用。

对流塔板采用对向流，其塔板效率相对于环流塔板要低一些，但它没有中心筒，而且结构简单，这样就大大减少了塔的直径。目前，国内大的空分提氩系统大多采用此种塔板。

导向塔板是以对流塔板为基础发展而成的。它的主要特点是使塔板的液层流动有一定的方向性和流动性，加速传质传热，从而提高塔板效率，降低阻力。目前，杭氧集团 $6000\mathrm{m}^3 /\mathrm{h}$  及  $10000\mathrm{m}^3 /\mathrm{h}$  以上空分设备的粗氩塔，均采用此种塔板。

导向塔板是在原筛板塔板的表面冲压许多斜坡口而成，使液体的流动性更好，其安装时的方向性相当重要，绝对不能装反，否则塔板效率会更低。

# (3）精氩塔

精氩塔由冷凝器、蒸发器及塔体组成，过去精氩塔的塔体均为环流塔板，现在基本上采用对流塔板，对于特别小的精氩塔还可采用高效填料塔。

精氩冷凝器及蒸发器，目前国内均采用板翅式结构，但引进的  $30000\mathrm{m^3 / h}$  大型空分设备，其结构仍采用列管和绕管式换热器。精氩冷凝器在设计时必须考虑不产生液泛的最小截面，否则精氩塔的工况将不稳定。

精氩塔的主要功能是氩一氮分离，由于氩和氮的沸点相差较远，相对来讲分离较容易些。

# （4）纯化系统

为了顺利除去粗氩中的氧组分及其生成物水分，氩纯化系统设置一系列大小不一的设备：阻火器是为了防止纯化器中氢氧反应产生的火焰倒流而设置的，其内部充填的是热容量较大的拉西铜和拉西铝；纯化器内充填的是催化剂钯触媒，其床层高度一般为600 $\sim 800\mathrm{mm}$  ，以保证气体充分接触，达到除尽氧组分的效果；三只冷却器的作用是冷却工艺氩并分凝少量水分；水分离器分凝工艺氩中的水分并将其排出系统；氩干燥器采用4A分子筛，对工艺氩中微量水进行吸附，4A分子筛不仅对水有吸附性，而且对氧组分也有一定的吸附能力。

加氢除氧系统根据工艺要求必须要有一定量的氢气，除尽一个体积的氧气需两个体积的氢气相配，而且氢氧反应后还需一定量的过量氢气，以便达到最佳的除氧效果。一般情况下，工艺氩中的过量氢约为  $2.5\%$  。所以采用气态进料流程，当提氩量较大时，氢气耗量较大，而且过量氢气全部随着余气排放到大气中，又不能回收，显得不很经济。但在提氩量较小的情况下，这种流程还是可行的。目前，国内空分提氩量小于  $100\mathrm{m}^3 /\mathrm{h}$  的制氩设备均采用这种流程。

# 2. 液态进针流程

图4为液态进料流程。氩馏分经粗氩塔后获得组分为  $96\% \mathrm{Ar} + 2.5\% \mathrm{O}_2 + 1.5\% \mathrm{N}_2$  的粗氩，再送入纯化系统，与精氩塔来的混和气（组分为  $48\% \mathrm{Ar} + 48\% \mathrm{H}_2 + 4\% \mathrm{N}_2$ ）混合后进入氩压缩机，压缩到  $0.32\mathrm{MPa}$ ，再与氢气混合。然后经纯化器除去氧组分，经干燥器吸附水分，再送入冷箱，经氩换热器冷却到饱和状态（一般过热  $4^{\circ}\mathrm{C}$ ）后，先导入精氩塔蒸发器，在蒸发器内工艺中的高沸点组分被冷凝，而低沸点的组分成为不凝气由蒸发器上部排出。不凝气通常被称之为混合气，这股气体的量一般约为粗氩量的  $5\%$ ，排出后经管道减压后送到压缩机前，与粗氩混合，并进行压缩，从而回收了大部分的过量氢气。

在精氩塔蒸发器中被冷凝的液态工艺氙经调节阀减压到  $0.045\mathrm{MPa}$  后，送入精氩塔

这一流程的氩压缩机为了安全，有时也采用水环式压缩机。采用水环式压缩机时，压缩机后还需设置一个水分离器，加氢的位置将移到压缩机进口，这样可使氢气与粗氩混合得更加均匀，并使其除氧效果更好。

液态进料流程的优点是能回收大量的过量氢气，从而减少用户的氢气耗量。另外，液态进料流程由于在精氩蒸发器中将高低沸点的组分先粗略地分离了一次，使得塔内的氩一氮分离更容易、更彻底、更完善，并可获得更高纯度的氩气。

这种流程在气量很大的情况下也可采用无油润滑螺杆式压缩机。目前，国内一般在提氩量小于  $190\mathrm{m}^3 /\mathrm{h}$  时，用无油润滑活塞式压缩机；提氩量大于  $190\mathrm{m}^3 /\mathrm{h}$  时，用水环式压缩机；提氩量大于  $600\mathrm{m}^3 /\mathrm{h}$  时，采用无油润滑螺杆式压缩机。

# 3. 粗氩自增压流程

图5为粗氩自增压流程示意图。该流程的特点是：在粗氩塔的冷凝器提取粗液氩，然后利用粗液氩的液氩柱进行增压，在冷箱内或冷箱外气化，从而达到增压的目的。一般增压后的压力可达到  $0.15\mathrm{MPa}$ ，当然这主要与粗氩塔的安装位置有关。

由于采用了自增压流程，从而省掉了加氢除氧系统中的压缩机机组，这样使整个系统的运转比较稳定。这种流程的缺点是：①过量氢不能回收，主要是一般情况下其增压后的压力无法达到回收过量氢所需的压力；②由于增压后的压力较低，则饱和工艺氩气体中的含水量较大，这样氩干燥器的负荷就要大大增加，造成吸附器的体积也增大。

除氧制氩这一传统的、复杂的、而又不十分安全的流程，并使国内空分发展步入一个新的历史时代。

采用全精馏制氯有如下优点：

① 可提高氩的提取率。  
② 运行安全，可靠。  
③ 无需氢气和繁杂的氩纯化系统。  
$④$  操作简便，负荷调节范围广。  
⑤ 可减少用户的占地面积（不需再建氩净化间和氢站）。  
⑥ 此种流程最适合于变工况生产。

采用全精馏制氯必须解决好如下问题：

① 采用这种流程，必须采用新型的高效规整填料塔，而且要低压降、高分离效率的填料，其分布器的设计更是填料塔设计的关键。目前国内制造填料的厂家不少，但真正已用于空分的并不多。  
② 采用此种流程，必须要有一台运行稳定可靠的低温液体泵，如果此泵不可靠那么粗氩塔就不能工作，采用此种流程也会变得毫无意义。

摘要 从理论上分析了大型空分设备中切换板翅式换热器与其对应的分、汇流总管的联接型式：Z型、π型及变径Z型布置对流量的影响，并作出各布置型式的压力分布曲线，提出板式等径配管π型布置为最佳方案。但限于实际情况，在设备改造中采用变径Z型，实际运行证实效果显著。图3表1。

主题词：大型空气分离设备 切换 板翅式换热器 总管 流型 布置 流量

大型空分设备中的板翅式换热器与其对应的分、汇流总管的联接型式，通常有Z型和π型两种，某厂从日本神钢引进的26000 $\mathrm{m^3 / h}$  空分设备就采用了Z型布置，该设备为污氮膨胀制冷，切换式换热器冻结流程。换热器冷箱由15个切换板翅式换热器（简称板式）组成，分三大组并联工作，每组由5个板式组成。在任一组中，因与板翅式换热器联接的等径分、汇流总管流体流向配置不当，引起板式中流量的分配不均，恶化了板式的换热效果。

# 四、三种布置型式的比较

假设板式各自通道气阻值、相互间排列距离、支管结构尺寸等相同，分流总管I、汇流总管Ⅱ上的压力分布对各股流体的流量分配起着一定的作用，亦即板式两端的压差直接影响到各股流的流量分配。因此，等径 $\pi$  型与变径Z型布置在流量分配上较均匀一

些，而等径Z型则逊色许多。

当然，不管采用何种型式，A～E5个板式流量的不均匀性总是存在的。从总体上讲，变径Z型与等径π型在分配均匀性的程度上好一些。在5个支管通流截面和与分流总管I通流截面之比值A小于1时，流量分配的不均匀性程度的降低就可以明显体现出来，亦即，随着A的减少，等径π型比等径Z型更易达到分配的均匀性。

对于等径  $\pi$  型与变径Z型，虽然总管线上的阻力损失与板式上的相比极其微小，但等径  $\pi$  型比变径Z型的管线阻力要小，因此配管在布置上应首先考虑  $\pi$  型结构。

# 五、处理情况

$26000\mathrm{m}^3 /\mathrm{h}$  空分设备未改造前，冷、热流体在板式中的分配不均、偏流，导致换热恶化现象，原先一直使用调节阀来处理。就板式中换热的主导流体空气、污氮气而言，靠15个空气调节阀和15个污氮调节阀强制分配流体，来促使在15个板式换热器内流量均匀分配，其它流体分配亦依靠调节阀来控制，结果，最终使得管网流阻增大，带来后果是空压机能耗增加，排气量下降。

改造工作牵涉到的环节较多，特别是板式冷箱现场条件限制，五通阀下部的配管密集，空间局限性很大，所以我们采用了变径Z型布置，而没有采用等径  $\pi$  型。

选用变径Z型后，分流总管I按原尺寸进行，汇流总管Ⅱ因流体密度有所变化，比原尺寸扩大了一些。空气、污氮气等主要流体配管均作了更改，依靠合理设计的配管来分配流量。

运行结果，所有的调节阀基本上处于全开位置，开启程度具有一致性，且仅作微小

的流量调节，消除了原调节阀操作的麻烦。

# 六、结束语

由于管子加工上的因素，笔者认为，一般新设计或现场条件允许的场合，还是采用 $\pi$  型布置为优。

需要说明的是，板式本身的阻力是决定整个流体的流量分配的主要因素，应使各板式的流阻具有等同性来控制偏流的影响。截面比值A的影响，比总管流阻对流量分配的影响要大。

# 绝热珠光砂烘干过程的参数计算

杨阳\*

（江西制氧机厂，江西九江妙智，320008）

摘要 真空粉末绝热贮槽一般采用珠光砂作绝热材料，珠光砂的含湿量是影响贮槽绝热效果的重要参数。本文探讨烘干过程中各环节状态参数的计算，烘干负荷、热负荷及干空气需求量；状态参数间的关系和控制要点。并以算例示明。参1。

# 主题词：烘干 含湿量 珠光砂 计算

真空粉末绝热贮槽中所采用的绝热材料目前多用珠光砂。为了保证珠光砂的绝热性能，对其容重、导热系数、含水率等均有指标要求。而含水率一项指标尤为重要，因含水率不仅直接影响导热系数值，更严重的是增大放气率，影响绝热夹层的真空度，破坏绝热夹层的绝热效果。因此，在真空粉末绝热贮槽的生产过程中，应严格控制珠光砂的含水率，通常采用烘干措施来保证其含水率指标。

烘干的一般过程是，空气经加热器加热后送入专用烘干筒，吹过珠光砂带走其中的水分。烘干的效果与烘干过程中空气流经各环节的状态参数密切相关。本文探讨烘干过程各环节状态参数的计算及其控制要点。

# 一、烘干过程的定性分析

烘干装置是利用未饱和空气吹过被烘干物，吸收其中水分的设备。为提高空气的吸湿能力，一般先将空气加热，使其相对湿度下降。加热后的空气经过烘干筒吹过珠光砂后，空气的相对湿度上升、含湿量增大，从而把珠光砂中的水分带走。空气在经过加热器的过程中，除吸入热量使温度上升外，与

外界无介质交换，其组成成分完全没有发生变化，因而含湿量d也保持不变。空气吹过珠光砂的过程可近似地认为是绝热过程，故该过程中空气的焓值I近乎不变。

烘干过程的控制要点是空气进烘干筒前的温度、含湿量以及吹过珠光砂以后出口处的温度。以下通过各环节状态参数的定量分析加以说明。

# 二、烘干过程的状态参数计算

# 1. 空气的初态参数

设初始相对湿度为  $\varphi_{1}$  ，初始温度为  $t_1$  （℃），初始压力为大气压  $p_0(\mathrm{MPa})$  ，饱和水蒸气所对应的饱和蒸汽压为  $\mathbf{p}_1(\mathbf{M}\mathbf{P}\mathbf{a})$  。则此状态下空气的含湿量  $d_{1}$  为：

$$
d _ {1} = 6 2 2 \varphi_ {1} p _ {1} / \left(p _ {0} - \varphi_ {1} p _ {1}\right) (\mathrm {g} / \mathrm {k g} \text {干 空 气}) \tag {1}
$$

含湿量为  $d$  的湿空气的焓  $I$  是千克干空气的焓和0.001d千克蒸汽焓的总和：

$$
I = i \text {气} + 0. 0 0 1 d i \text {汽} \quad (\mathrm {k c a l / k g} \text {于 空 气}) \tag {2}
$$

当温差不大时，可将空气的定压比热  $C_p$  当作定值，则：

$$
i _ {\text {气}} = C _ {p} \bullet t \quad (\mathrm {k c a l / k g}) \tag {3}
$$

水蒸气的焓有足够准确的半经验公式：

$$
i _ {\text {汽}} = 5 9 5 + 0. 4 7 t \quad (\mathrm {k c a l} / \mathrm {k g}) \tag {4}
$$

因此，得：

$$
\begin{array}{l} I = C _ {p} \cdot t + 0. 0 0 1 d (5 9 5 + 0. 4 7 t) \\ \left(\mathrm {k c a l} / \mathrm {k g} \text {干 空 气}\right) \tag {5} \\ \end{array}
$$

于是，有：

$$
\begin{array}{l} I _ {1} = C _ {P} \cdot t _ {1} + 0. 0 0 1 d _ {1} (5 9 5 + 0. 4 7 t _ {1}) \\ \left(\mathrm {k c a l} / \mathrm {k g} \text {干 空 气}\right) \tag {6} \\ \end{array}
$$

# 2. 加热后空气的状态

空气通过加热器未与外界发生任何介质交换，空气的组分没有变化，故含湿量不发生变化，即有：

$$
d _ {2} = d _ {1} \quad (\mathrm {g / k g} \text {干 空 气}) \tag {7}
$$

设控制加热器后空气温度为  $t_2$  ，压力仍近似地认为是  $\pmb{p}_0$  ，据饱和水蒸气表可查得此状态下空气中所含水蒸气之对应的饱和蒸汽压为  $\pmb{p}_2$  ，则可计算空气的相对湿度下降到 $\varphi_{2}$

$$
\varphi_ {2} = d _ {2} p _ {0} / \left[ p _ {2} \left(d _ {2} + 6 2 2\right) \right] \tag {8}
$$

此状态空气的焓为：

$$
\begin{array}{l} I _ {2} = C _ {\rho} \cdot t _ {2} + 0. 0 0 1 d _ {2} (5 9 5 + 0. 4 7 t _ {2}) \\ \left(\mathrm {k c a l} / \mathrm {k g} \text {干 空 气}\right) \tag {9} \\ \end{array}
$$

# 3. 烘干筒出口空气状态

热空气经过烘干筒吸收珠光砂中的水分后，使热空气相对湿度和含湿量均增加，温度下降。该过程可近似视为绝热过程，因而焓未变化：

$$
I _ {3} = I _ {2} \quad (\mathrm {k c a l / k g} \text {干 空 气}) \tag {10}
$$

设测得出口空气温度为  $t_3$  ，由式(5)和式（10）可得：

$$
\begin{array}{l} d _ {3} = \left(I _ {3} - C _ {\rho} \cdot t _ {3}\right) \times 1 0 0 0 / (5 9 5 + 0. 4 7 t _ {3}) \\ (\mathrm {g} / \mathrm {k g} \text {干 空 气}) \tag {11} \\ \end{array}
$$

空气中水蒸气压可查得为  $p_3$  ，则：

$$
\varphi_ {3} = d _ {3} \cdot p _ {0} / [ p _ {3} \cdot (d _ {3} + 6 2 2) ] \tag {12}
$$

# 三、烘干负荷、热负荷及干空气需求量

设一次烘干珠光砂质量为  $M(g)$  ，烘干前后的含水率分别为  $\eta_{x}$  和  $\eta_{y}$  ，不计珠光砂烘干前后总质量的变化，即视  $M$  为恒值，则烘

干过程中空气应带走的水分质量为：

$$
\Delta m = M \left(\eta_ {x} - \eta_ {y}\right) \quad (g) \tag {13}
$$

空气含湿量的增量即为每千克干空气所带走的水分：

$$
\Delta d = d _ {9} - d _ {1} \quad (g / k g \text {干 空 气}) \tag {14}
$$

因此，烘干负荷为  $\Delta m$  ，所需干空气质量为：

$$
m _ {g} = \Delta m / \Delta d \quad (\mathrm {k g}) \tag {15}
$$

加热器的热负荷为：

$$
Q = m _ {g} \left(I _ {2} - I _ {1}\right) / k \quad (\mathrm {k c a l}) \tag {16}
$$

式中  $k$  ——加热器的热效率， $k < 1$

# 四、状态参数间的关系和控制要点

终态参数  $t_3$  随着时间的变化而变化。开始时，因珠光砂中含水率较高，空气中带出的水分较多，致使温差较大。临近结束时， $t_3$  升高。理论上讲，当珠光砂初烘干至含水率为零时，应有  $t_3 = t_2$ ，这在实际上是不可能达到的。实际情况是最终出口空气含湿量为  $d_3$ 。

将式(7)和式(9)代入式(11)得：

$$

# 液氨温区二级脉管制冷机的试验研究

余建平* 陈国邦 郑建耀 邱利民 黄志秀

（浙江大学制冷与低温研究所，杭州玉泉玉古路20号，310027）

摘要 本文提出了脉管制冷机的一种新的调相机构——双小孔结构，分析了双小孔的工作机理。根据这一相移器的原理，自行设计了一台二级脉管制冷机，经多次试验，该制冷机的一、二级分别达到了46K和3.0K的最低制冷温度。图3表2。

主题词：脉管制冷机 双小孔 制冷机

# 前言

低温制冷机在低温电子器件和超导量子干涉仪(SQUID)、资源卫星、低温泵等方面有着广泛的应用。当代高新技术的发展，对小型低温制冷机的可靠性和寿命等的要求越来越高，现有可供应用的回热式制冷机如Stirling、G-M、VM、SV等，由于存在低温区运动的排出器而造成磨损、振动、污染等，已很难适应长期无维修运行的要求。这样，脉管制冷机脱颖而出，由于它没有低温运动部件而受到世人的关注，近年来成为小型低温制冷机的研究热点。

脉管制冷机是1963年由美国的Gifford和Longsworth发明的，它利用压力周期性变化的气流在一根封闭管中作绝热膨胀而实现制冷的，当时只获得了  $124\mathrm{K}$  的制冷温度。直到二十年之后，Mikulin和朱绍伟等[2,3]提出了小孔气库、双向进气等被称作调相机构(或叫相移器)的结构以后，脉管制冷机性能得到了大大提高，单级脉管制冷机的最低

制冷温度不断下降。目前单级脉管的无负荷制冷温度已达到  $23.5\mathrm{K}$ ，其性能已接近其它型式的回热式制冷机，但是，要在液氮温区获得应用，还需研究多级脉管制冷机。

目前世界上有多个实验室正在从事多级脉管制冷机的研究工作，以求获得更低的制冷温度。1988年，北京中科院低温中心周远等[4]采用双级双压缩机的二级脉管制冷机获得了31K的低温。1989年，Tward等[6]采用线性马达驱动压缩机的二级脉管制冷机获得的最低温度为26K。1993年，日本东芝公司大谷安见等[8]研制的二级脉管制冷机达到了19.1K的低温；同年，日本大学的高金林和松原洋一等[7]采用三级脉管制冷机首次在脉管上实现了氮的液化，最低温度达2.9K。1995年日本的Tanaka等人[8]采用二级脉管制冷机达到了15K的低温；美国NIST的Bradley等[9]报道了用三级脉管制冷机获得10.5K的无负荷制冷温度。1996年，北京低温中心的巨永林等[10]研制了由旋转阀配气

的多路旁通二级脉管制冷机，获得了11.5K最低制冷温度。目前有关低温脉管的工作特性尚不够清楚，其效率还有待进一步提高。

为此，我们从分析脉管制冷机的机理出发，提出了一种新颖的调相机构，即双小孔结构。并根据这一相移器的原理，自行设计了一台二级脉管制冷机。经多次试验，该制冷机的第一级和第二级的最低制冷温度分别达到  $46\mathrm{K}$  和  $3.0\mathrm{K}$ ，在国内首次利用两级脉管达到了液氮温度。

# 一、双小孔的作用机理

实验发现，双向进气型脉管制冷机引入的第二进气阀使脉管壁面近热端处出现了一个远大于室温的温度峰值，且这一峰值随系统压力的增大而增加[11,12]。产生这一现象的原因可能是：双向进气型二级脉管制冷机中回热器的阻力较大，使得低温级脉管中的压比降低；同时由于低温级脉管较长，压缩过程产生的热量难以全部被气体活塞传递到脉管热端。如图1所示，双向进气型脉管制冷机中脉管近热端气体向室温耗散的动力仅依赖于小孔两端的压差  $\Delta P = P_{H} - P_{m}(P_{H}$  和 $P_{m}$  分别指脉管和气库中的压力）。

在我们提出的双小孔型结构中，在第二级气库与压缩机进气管之间连接一第二小孔阀，打开第二小孔并维持一个合适的开度，气流便在第二级脉管与压缩机入口间形成一条通路，这样脉管近热端气体向室温耗散的动力变为  $\Delta P^{\prime} = P_{H} - P_{L}$  （  $P_L$  为压缩机进气压力）。在压缩过程中，第二小孔增强了第二级气库对压缩气体的抽吸作用，有更多压缩过程中产生的热量被气体活塞传递到脉管热端。因此，第二小孔增强了脉管制冷机的调相能力，可以提高脉管制冷机的制冷性能。

# 四、结论

1. 提出了适用于多级脉管制冷机的新颖调相机构——双小孔结构。第二小孔增强当级小孔的调相功能，强化低温级脉管的泵热能力。  
2. 根据这一相移器的原理, 自行设计了一台二级脉管制冷机, 经多次试验, 该制冷机的一、二级分别达到了  $46 \mathrm{~K}$  和  $3.0 \mathrm{~K}$  的最低制冷温度。  
3. 证明了双小孔结构能明显改善二级脉管和回热器上的温度分布，改善多级脉管制冷机的制冷性能。

张远求\*

（开封空分设备厂设计处，开封公园路，475002）

摘要 介绍了应用在  $1500\mathrm{m}^3 /\mathrm{h}$  高纯氮设备上的R100系统的系统配置，并详尽阐述了该系统所具有的用户监视画面、实时控制、报表打印等主要功能，最后分析了该系统的适用范围。图1。

# 主题词：R100系统 DCS系统 功能 效果

R100系统是美国Honeywell公司基于S9000基础上的小型DCS系统。

该系统是建立在9000系列控制器、620逻辑控制器和PC486监控站平台之上的开放式系统，它采用标准的以太网通讯，并在windows3.1环境之下为用户提供有效而方便的监控手段。在S9000系统的基础上，它增加了顺序控制图这一组态工具，增强了绘图、显示趋势、报警和报表等多种监控功能，使组态与使用操作更为方便。R150系统是R100系统的升级版本，它使R100系统的组态软件进一步强化和优化，可在windows3.1的平台上显示中文，同时增加了一些控制功能。本文介绍的是R100系统在  $1500\mathrm{m}^3 /\mathrm{h}$  高纯氮设备控制上的应用。

# 一、系统配置

# 1. 常规配置

测温元件选用PT100铂热电阻。压力、液位、阻力选用ROSEMOUNT公司的1151变送器和上海FOXBORO公司的840系列变送器。分析仪表选用ROSEMOUNT公司的分析仪，  $4\sim 20\mathrm{mA}$  输出信号进入R100系统进行显示，联锁控制。转速测量选用上海转速仪表厂的磁电传感器，其频率信号经频率电流转换器转换成  $4\sim 20\mathrm{mA}$  信号后进入

R100系统进行联锁控制。另外，再配以常规执行机构和就地显示仪表。

# 2. R100系统配置

整个系统以中心控制室集中监视、集中操作为主，就地显示为辅，信号的采集和控制在控制器进行处理，做到了危险部分分散。系统拓扑结构如图。

该系统选用了2台SE9119控制器，1台SE9025扩展箱，5块热电阻输入模块，5块通用模拟量输入模块，9块模拟量输出模块，5块开关量输入模块，9块开关量输出模块。PC监控站选用研华工控机IPC486/338M内存，260MM硬盘，  $3^{\prime \prime}$  、  $5^{\prime \prime}$  软驱各1个，配EPSONLQ-1600K打印机和鼠标器。监控软件选用SE9721标准型，以

# 二、系统主要功能

# 1. 用户监视画面

为了形象地监视、分析和操作整个空分设备，共生成空压机系统、水冷系统、纯化系统、膨胀机系统、精馏塔系统以及贮槽系统等动态流程画面。各模拟采样参数均以数字形式精确显示，并配以液位的上下波动、阀门的开关动作、管道气体的流动等形象地显示出来。开关量的采样参数，可以电机的转动、图形符号的颜色变化、闪烁等方式动态显示。对于需控制的参数，均设置了开窗口功能，利用鼠标器可以方便地打开、关闭子窗口。在子窗口中，可以利用鼠标器或键盘方便地调整控制参数（PID调节具有自整定功能）、手动/自动切换，以及遥控操作。为了方便用户的操作，还设置了一些特殊功能键，实现了一些电器的点动操作和画面的快速切换。

另外，为了方便开车调试、事故分析，并从尊重用户习惯出发，还设置了报警显示画面、实时趋势和历史趋势画面及棒图分组画面。

# 2. 实时控制

本套设备除工况稳定时，无须频繁操作的少数参数之外，20个主要的参数实现了自动调节和自动联锁控制。下面就几个重要的控制回路作简单介绍。

# （1）主冷液面控制

主冷液面调节是为了保证设备的精馏工况而设置的，主冷液面的波动（正常  $1700\mathrm{mm}$ ）将直接影响产品的纯度和产量。本套设备设计了两种工况：工况I氮气产量  $1500\mathrm{m}^3/\mathrm{h}$ ，由废气进膨胀机的旁通量来调节冷量，控制主冷液面的稳定；工况Ⅱ氮气产量  $1370\mathrm{m}^3/\mathrm{h}$ 、液氮产量  $130\mathrm{m}^3/\mathrm{h}$ ，由抽取氮的薄膜阀

开度控制主冷液面的稳定。针对两种不同工况的控制方式，本人在连续控制图的设计中，做了一些优化工作。在动态流程画面上，用户只需用鼠标器方便地选中光字排“工况Ⅰ”或“工况Ⅱ”，两种工况即可方便地转换，控制算法也自动切换到目前选中的工况上。

# （2）膨胀机的转速控制

膨胀机的转速控制是维持塔内冷量平衡的一个重要参数，它是由多变量实现自动调节和自动保护联锁的一个复杂控制回路。本人在连续控制图的设计中，用进口压力、喷嘴开度以及制动风机实现转速调节的同时，还设置了油压、油泵电机、冷却水、轴承温度等参数自动联锁停车保护功能。另外，还设置了点动快速停车功能，保证了膨胀机安全、稳定、可靠的运行。

# （3）分子筛切换系统的控制

在空分设备中，分子筛吸附器的切换系统是整套设备控制的心脏部分，它的工作好坏直接影响整套设备的运行。系统由13个二位式切换阀和两台电加热器组成。该切换系统的控制是通过顺序控制图来实现的，通过时间顺序触发输出开关量状态，开关量信号直接控制五通电磁阀（为了保证可靠性，电磁阀选用德国HERION公司的产品），通过五通电磁阀的得失电来控制进入切换阀气缸的气信号，从而达到控制球阀切换的目的。在编制该顺序控制图时，巧妙的编程技巧使该程序具备如下特点：

① 一旦系统出现故障或断电，切换程序停止运行，阀门状态保持现状。  
② 自动跟踪阀门状态，实现多点投入。  
③ 加温、冷吹时间可以在线任意修改。  
④ 在自动切换状态下，可以手动操作阀门、电加热器，也可将切换程序停止，手动操作。  
⑤ 每当进入切换状态，报警提醒操作人员注意，通过流程画面观察执行机构的好

坏，一旦输入与反馈不一致就引起报警。以上这些优点保证了切换系统安全、稳定、可靠、长期运行。

# 3. 报表打印

两台操作站配置了两台打印机，其中一台专门负责随时打印报警、操作记录，便于事故分析，提高生产管理；另一台负责定时或随时打印历史趋势、运行日志等报表。运行日志报表可以利用EXCEL报表软件生成各种自由格式报表。

# 4. 操作

操作人员只需根据流程画面上的提示信息和窗口信息，利用鼠标器逐层选中，即可方便地进行诸如手动/自动切换、遥控操作、流程画面之间的切换、报表的打印等操作。除非想用一些特殊功能键来实现快速操作，否则所有操作不必借助键盘。保证了操作的单一性、方便性和可靠性，便于用户掌握。

# 三、使用效果与分析

本套制氧设备已经在上海耀华一皮尔金

顿玻璃股份有限公司连续运行一年多，其升级版本R150系统在胜利油田也已开车近一年，通过跟踪，运行效果良好。

应该说明的是，在开发使用R100系统的过程中，本人也发现R100系统存在的一些问题。首先，其电源模件、回路处理器和逻辑处理器不具备冗余，给用户造成一些疑虑，这一点似乎它的高可靠性可以弥补它的不足。但它不具备单点下装功能（程序只能整体下装），给现场开车调试带来了困挠。特别是改变控制策略时（这正是DCS的优点），程序的在线重新下装所引起的问题不容忽视。因此，在大中型空分设备测控点较多的情况下，选用R100系统应慎重。勿容置疑的是，R100系统的集成化、小型化、开放性体现了90年代DCS的特点。由于它投资少，性能/价格比高，配置灵活，功能强大，丰富的汉化软件，因此特别适合单高空分设备以及小型空分设备选用。

# 主题词：空气分离设备改造 预冷装置 氨氟利昂R22 低温水比较建议

分子筛型空分设备与蓄冷器型、切换式换热器型空分设备相比，分子筛型空分设备具有流程简单、操作方便、更加安全可靠、可以提取较多的纯产品等优点，所以一些空分设备用户相继把蓄冷器型、切换式换热器型空分设备改造成分子筛型空分设备。在分子筛型空分设备中，为保证分子筛吸附器的吸附效果，需要把进分子筛吸附器的空气冷却到  $8\%$  左右，而原空分设备的预冷系统不能满足这一要求，必须增加一套空气低温预冷装置。

本制冷工程部在空分设备变型改造设计中，本着“技术先进、节省投资”的原则，根据用户的具体情况，采用了四种空气低温预冷装置，均取得了满意的效果。这四种空气低温预冷装置是根据冷却空气的介质和换热设备不同来划分的。

# 主题词：离心式压缩机 安装 体会

DA350-61型离心式空气压缩机组，是 $3200\mathrm{m}^3 /\mathrm{h}$  制氧机的配套部机。包括  $2500\mathrm{kW}$  电动机一台、增速器一台、  $8600\mathrm{r / min}$  空气压缩机一台，我们在安装中，学习了兄弟单位的经验，参阅了一些资料，安装工作比较谨慎，做到了一次试车成功。下面就安装中的一些体会，写出来供大家参考。

# 一、设备基础的处理

为了便于今后检修时重新找正，设备基础不搞二次灌浆。预埋垫铁(钢板)在第一次灌浆时同时填上，然后找一下水平即可，这样预埋垫铁比较牢固。斜铁垫放在预埋垫铁上，斜铁底面在与预埋铁的接触面要进行研磨。设备的高低，用斜铁来调整，最后用电焊点焊。设备的四周则采用四个顶紧螺钉（增速机和电动机也是这样）固定，左右移动通过四个顶紧螺钉即可。这样，设备的高低及左右移动都是活的，找正方便，找正速度大大提高，平时检修时不用打二次灌浆，我们称这样的基础为“活基础”。

# 二、增速器的找正和齿轮的啮合

增速器在机组中占有重要的位置，它的找正及齿轮的啮合，直接影响到机组的运转。因此，安装中应先将增速器定位，然后空压机和电动机都以增速器为基准找正。增速器研瓦时，边研瓦、边研齿轮啮合及两轴的轴

距和轴的水平度（水平度要结合齿轮的啮合情况来定）。我们认为，只要两轴的中心距研好，齿轮的啮合研好，就是轴本身不水平，也没有什么大问题。

在研齿轮啮合的同时，进行轴瓦刮研。这时，用印色油检查轴颈与轴瓦接触情况，根据接触要求刮研。底瓦的接触角我们采用 $60^{\circ}$ ，齿轮的啮合检查，我们先用印色油，最后压铅丝。据我们的经验，只要涂色检查好了，齿轮的啮合基本无问题。我们所遇到情况是：齿轮啮合好了，但轴就不能水平。我们是这样处理的：低速接头高  $0.04\mathrm{mm}$ ，高速接头高  $0.04\mathrm{mm}$ ，两轴本身都不水平。

# 三、联轴器的找正

联轴器有两个，一个是电机一增速器；一个是风机一增速器。这两个联轴器的找正都很重要。

机组最后一次找正，一定注意要在吊上转子，把好瓦盖，扣上大盖，管道安装完毕的情况下找正。否则，找正完后，又扣大盖又接管道，数据就会变了。

# 四、轴瓦的研磨

轴承的瓦间隙，影响到轴承的温度。大了会振动，小了会发热，因此要仔细处理。我们力争运转后不超过  $60^{\circ}\mathrm{C}$  （当然这个温度与油温有关系，为了降油温，我们在油箱内

加上三组蛇形管，油温保持在  $40^{\circ}\mathrm{C}$  左右），首先加工两根轴：一根粗轴，直径为原来的轴径加上侧间隙；另一根细轴，直径为原来的轴径。先用粗轴研。用粗轴研时，下接触面 $60^{\circ}$  不动，研好后，实际上侧间隙也就出来了；然后再用细轴轻研即可。下接触面目前都向  $30^{\circ}$  发展，我们认为这样接触面少，轴承的进油量大，冷却好，温度低，能更好地形成油膜将轴托起，所以不一定于拘泥于  $60^{\circ}$  的接触面，可以再小点。

为了降低轴承温度，增加进油量也是一个好办法。但是，由于轴瓦的现有槽沟，不能将进入轴瓦的油尽快流出去，油循环量不大。因此，我们采取了这样一个办法：将原

轴瓦的槽沟的两侧各开一条小沟，使进入轴瓦的油尽快流出去，加快油的循环，轴瓦得到了冷却，轴承温度就降下来(见图)。在此之前，

# 五、油系统的清洗

空压机组的油系统就像人身的血液循环系统一样重要，许多烧瓦事故，基本上都是由于油管内有异物或断油而造成的。由于油系统管道的口径小、弯头多、长度大，难以清理，容易被忽视或马马虎虎的应付过去。我们认为，油中的杂质，一来自管道本身，二是油过滤器漏网。因此我们对管道进行认真的酸洗，中和后，将铁丝穿进管道，来回拉动除锈，然后立即冲洗干净，再用干燥氮气将管道一一吹干，接着灌进10\*机械油，两头扎起来放好。这一切都是连续的，放久了就会锈蚀。试车前，平日定时地将油泵启动，用油冲刷管壁，并多次清洗过滤器。同时对油箱和增速器箱也仔细地用煤油清洗干净，在空压机试车前再换上合格的22#透平油。

主题词：空气分离设备 氩系统 串通 合并 分析 效果

我厂三套切换式流程的KDON=1000/1000型空分设备，是杭氧80年代中期产品，均配置氩提取系统。单套空分精氩产量为  $12\sim 15\mathrm{m}^3 /\mathrm{h}$  ，其中  $1^{*}$  空分氩塔工况不稳定，一直未工作；  $2^{*},3^{*}$  正常。由于精氩的市场需求较不稳定，有时需求量少，只得停产，高峰期则供不应求，造成两台精氩系统经常开开停停，浪费较严重。为了解决这一矛盾，我们根据现有条件，将两套粗氩气合并进入一套氩纯化系统，取得良好效果。

# 一、合并调氩的可行性

1. 精氩压缩机的压缩量较大，可满足两台空分设备提供的粗氩压缩量。  
2. 按设计， $2^{\#}$  精氩塔塔板直径为  $\phi 100\mathrm{mm}$ ，而  $3^{\#}$  精氩塔的塔板直径，在原设计基础上增为  $\phi 120\mathrm{min}$ ，因此正常工况下， $3^{\#}$  塔塔板尚有  $44\%$  的潜力未利用。  
3. 本厂精氩压缩机配备充足，且互相连通。  
4. 三套氙提取系统设备互相串通，为合并调氙提供了有利条件。

通过分析，我们认为已基本具备合并调氮条件，于1994年8月9日进行了两台合并调氮的试验。

# 二、合并调氯过程

$3^{*}$  氩系统处于正常工作状态，缓缓打开

$2^{*}$  、  $3^{*}$  粗氩连通阀门，  $2^{*}$  粗氩导入  $3^{*}$  粗氩平衡器。此时关闭粗氩回流阀，氩压机排气压力升高，开大工艺氩进精氩塔阀门HC-704，以稳定压力，同时开大加氢调节阀，控制微量氧的超标报警，控制  $2^{*}$  粗氩量  $10\sim 15\mathrm{m}^3 /\mathrm{h},$ $3^{*}$  粗氩量  $15\sim 18\mathrm{m}^3 /\mathrm{h}_{\circ}$

精氩塔内因工艺氩量增多，热负荷增大，工况有所波动。加大余气放空量，增大放空氮气量，调节塔内回流比，压力、阻力均有所上升，过一段时间，精氩塔工况趋于稳定。而两台粗氩塔由于各抽取量减少，纯度变好，粗氩中氧含量明显降低。当日顺利出精氩，且平均精氩量在  $25\mathrm{m}^3 /\mathrm{h}$  左右，与单纯一台机工作相比，产量明显提高。

在1994年8月9日试验基础上，又分别于8月12、13、26日进行多次试验，通过努力，取得成功。班产由原来的20瓶上升到  $35\sim 40$  瓶。合并前后主要参数的比较见表1。

# 三、应注意的几个问题

1. 由于两台空分设备的粗氩共同使用，故对两台的粗氩纯度都应十分重视。当一台粗氩纯度变坏时，应迅速切断该台机粗氩送出阀，并打开放空阀，以保证输入的粗氩质量，且保证继续以单机生产。  
2. 合并连通前，稍开大加氢调节阀及HCl-704阀，保证工艺氩纯度和压力稳定。

# 二、上塔进料路线改变的依据

上塔进料改变是否可行，主要取决于能否满足工艺需要，即上塔液空进料管线能否

达到原进料量和上塔液空进料口的改变会否影响上塔的正常精馏分离。因液空自下塔底部经过冷器E2过冷后分为两路：一小部分液空通过调-10阀后进入C3塔的冷凝器中作为粗氩塔的冷源，在冷凝器的底部通过通-3阀(DN50)返回到上塔的54层，防止碳氢化合物的聚集，换热后的气体(DN200)也返回上塔的54层，增加参与精馏的的原料量，以提高氧氮的提取率；大部分液空通过调-1阀作为上塔的主要原料，进入61层。如果调-1阀不通，所有液空按原工艺路线经调-10后进入C3后，再通过通-3阀返回到上塔，液空的最大流量取决于自C3塔至C2塔之间的DN50管线流通能力。根据设计参数，在实际工况压力等参数下，DN50的管线流通能力无法满足工艺液空进料量，因此该路线满足不了精馏需要。因C3塔顶部设有液空气化返回线DN200，通过强度核算我们认为该管线完全可以分流一部分液空原料，使进C3的液空分两路进入上塔做原料，此时C3塔停止分离粗氩(原粗氩全部放空)，仅起到

一个缓冲容器的作用。

原液空工艺流程：

C1-E2-调-1-C2  
L-10-C3-通-3-C2

改变后液空工艺流程：

C1-E2-调-10-C3-通-3-C2 C2

下塔液空液位的显示和调节，仍用原调-1阀的调节器来控制调-10阀。

通过查找上塔的结构图发现，原液空进料口位于上塔的第61层，由C3返回的上塔的进料口为第54层，较原进料口降低7层。由于进料口的降低，提馏段塔板数减少，不利于氧气提取，同时在上塔的塔板数一定的情况下，精馏段的塔板数相应增加，对氮气提纯有利。由于空分塔的设计通常有一定的设计余量，根据多年的操作经验，我们可通过调节精馏工况来达到工艺要求。

# 三、运行效果和经济效益

通过以上改变原料路线后，二空分装置下塔液位逐渐达到正常水平，通过工艺技术人员和操作员的摸索调试，装置精馏工况稳定，产品纯度合格，其中氧气产量可达5400 $\mathrm{m^3 / h}$ ，氮气产量达到  $12000\mathrm{m^3 / h}$ 。该参数没有达到设计产量，其中一个重要原因是空压机的出力不足，压缩量仅为  $34000\mathrm{m^3 / h}$ ，较设计值低  $4000\mathrm{m^3 / h}$ 。

液空进料改变后，“二空分”根据总厂调度室指令备用运行，在“一空分”正常情况下产品全部放空。自1996年5月份一直维持至今累积运行近6个月，在改变原料路线运行期间，一空分装置发生晃电、4.0MPa蒸汽压力下降和主换热器反吹等事故5次，由于二空分装置的及时供气，保证了用气装置的平稳运行。

# 贮气囊的“湿润”现象及其对氧气纯度的影响

李朝元*

（山东省黄岛发电厂氧气厂，山东青岛市黄岛区，266500）

摘要 用充水方法对贮气囊检漏，其表面的“湿润”现象证明，贮气囊的“微孔”是造成氧气纯度降低的唯一原因。建议用户选择贮气囊时，最好用充水法检验贮气囊的质量，或干脆选购橡胶气囊，以确保产品氧纯度。

主题词：氧气 贮气囊 温润 充水法 纯度

我厂于1994年11月购进邯郸制氧机厂生产的KZO-50型空分设备一套，随机配有江苏省江都市某厂生产的  $50\mathrm{m}^3$  贮气囊两只，安放在氧压机上方，用钢管尼龙网架支撑。该机组于1994年12月30日建成投产。

# 一、用户反映产品氯纯度低

投产初期，由于缺少经验，值班人员只化验送氧纯度，均为  $99.75\% \sim 99.8\%$  。产品投放市场后，不断接到用户“氧气不纯”的反映。当时我们怀疑是新氧气瓶内空气尚未置换彻底的缘故，未加以重视。数月后，用户反映不断增加，此时才从氧气瓶中取样化验，化验结果为  $97\%$  左右。之后，我们对系统管路、仪表、阀门及氧压机一二三级活塞杆密封处一一查漏，消除可能的缺陷，但成效不大，最高纯度仅达国家的二类标准  $99.2\%$  且数量极少。

然后又用肥皂水对贮气囊检漏，先检查出入口、接头，未发现漏点；又用同样方法对贮气囊进行  $100\%$  的全面检漏，仍查不到漏点。最后，请求制造厂家派员前来处理，均

未得到满意的结果，纯度一直徘徊在  $97\% \sim$ $99.2\%$  之间，面临着失去整个市场的危险。

我们果断停止销售，集中精力采取“分段查漏”的办法，即从缓冲桶、氧压机一二三级压力表阀及其出入口、贮气囊出入口、充氧台总阀与分阀等各段取样，分析化验。

经过四个昼夜千余次的化验分析，结果发现两只贮气囊中一只贮气囊的氧气纯度偏低。将该贮气囊解下，单独使用另一只贮气囊，分析氧气纯度普遍达到  $99.6\%$  ，最高的达到  $99.7\%$  。

# 二、产品氧纯度低的原因及其发现

为了搞清原因，我们又将两只贮气囊解下，作充水试验，充水量约占贮气囊容积的2/3，然后缓慢滚动。结果，纯度低的那只贮气囊虽然无一处泄漏，但其表面却有“湿润”现象，说明有少量水渗透。另一只贮气囊既无泄漏，也无“湿润”现象。于是我们得出结论：贮气囊“湿润”，说明贮气囊有“微孔”，“微孔”使水渗透，它是造成氧气纯度降低的唯一原因。

# 三、理论分析

事后，我们对“湿润”的贮气囊进行剪样分析，其材料结构为双面涂胶的帆布，布厚约  $1\mathrm{mm}$  。由于帆布经纬线交错欠紧密，涂胶较薄，故存有“微孔”。

空气进入贮气囊，必须具备两个条件：

1. 空气的分子半径，必须小于水的分子半径；  
2. 袁外空气和囊内氧气能够互相扩散，且空气能够克服囊内不大于  $10 \mathrm{~cmH}_{2} \mathrm{O}$  （气囊使用说明书规定的最大充气压力为  $10 \mathrm{~cmH}_{2} \mathrm{O}$ ）的氧气压力进入贮气囊，才能使氧气纯度降低。

查阅有关资料，找到了满足以上两个条件的理论根据。

① 从《实用化工手册》[1]中我们查到与之有关元素的分子半径即范德华半径。氧分子为2.8、氮分子为3.0、水分子为3.9，可见，水的分子半径最大，只要水能通过，空气主要是占其中  $68\%$  的氮气便可以通过。满足第一个条件。  
② 查《化工辞典》，知“扩散是由于微粒（分子、原子等）的运动而产生的物质迁移现象，可由物质在气、液或固相的同一相或不同相间进行，主要是由于浓度差，也可由于温度差和湍流运动等，微粒从浓度较大的区域向较小的区域迁移，直到相间浓度达到平衡为止”；“扩散系数是表物质扩散能力的物理量，可以理解为沿扩散方向在单位时间内

物质的浓度降为一单位时的传递量，与压强的关系极小”。[2]满足第二个条件。

也就是说：囊壁在具备空气分子主要是占  $68\%$  的氮气分子和氧气分子可以通过的条件下，尽管囊内氧气具有一定压力（  $\leqslant 10\mathrm{cm}$ $\mathrm{H}_2\mathrm{O})$  ，而由于分子的扩散作用，空气相一定会逆压向囊内的氧气相扩散，造成氧气浓度降低。如果有足够的时间，两相浓度最终将趋于一致。

另外，“物质平衡理论”即任何一种物质总是要努力恢复和保持原有的平衡状态的趋势，也同样可以做为第二条件的理论依据。

# 四、建议

我们又购置了一只贮气囊以取代“湿润”贮气囊。使用前，作了同样的充水试验，发现局部仍有“湿润”现象，面积约  $2\mathrm{m}^2$  ，使用后纯度也有所降低。故建议用户在选择贮气囊时最好用充水的办法检验一下贮气囊质量，或者干脆选购橡胶气囊。贮气囊制造厂也要注意所进原料的质量，制造出高密封性、无“湿润”现象的贮气囊，以确保氧气纯度，给用户提供合格的氧气产品，创出更高的经济效益。

# 4DE150-4A型迷宫式活塞氧压机维修

邓韬

(广州市粤港气体工业有限公司设备部, 广州市白鹤洞芳村大道 1 号广钢厂内, 510381)

摘要 4DE150-4A型迷宫式活塞氧压机在调试及使用过程中出现二级排压偏高，停机检查，发现二级压力偏高是由于三、四级活塞与气缸内壁间隙太大，氧气通过间隙泄漏入三、四级活塞下的空腔。后经维修，将缸体与机体对中好，迷宫式活塞氧压机运转情况转入正常。图1表1。

# 主题词：迷宫式 氧气 活塞 压缩机 摩擦 间隙偏差 维修

4DE150-4A型迷宫式活塞压缩机(LA-BYRINTH—PISTONCOMPRESSOR)和普通活塞压缩机不同，它没有活塞环和导向环(见图)。在迷宫式活塞压缩机的气缸内壁和活塞表面上有十分细密的环状槽，这样，在气缸内壁和活塞表面之间就形成了迷宫密封。当活塞上行时，被压缩的高压氧气沿活塞四周向下泄漏，但由于受到迷宫的节流作用，气体受到阻滞。这样，气缸内壁和活塞表面之间的迷宫密封实际上起到了活塞环的作用，但无机械摩擦。

![](images/ef7238140fe2bdac052513875f1ec8e4302bbb0f09a405e3127b51c96a60c811.jpg)  
迷宫式活塞压缩机原理图

采用这种形式的密封，只有当活塞表面与气缸内壁之间的间隙小且均匀，密封效果才会很好。如果由于活塞与缸体的对中不好或其它原因，造成各方向的磨损不均，使某个或某些方向的间隙增大，就会加剧泄漏。因此，活塞与缸体的对中是十分重要的。

# 一、迷宫式活塞氩压机简介

为满足灌充瓶装氧气的需要，我公司从

瑞士SULZER公司购入一台4DE150-4A型迷宫式活塞氧压机。该机处理气量为500 $\mathrm{m}^3 /\mathrm{h}$  ，排压为  $17\mathrm{MPa}(\mathbf{G})$  ，级数为4级(其中一、二级为双作用式，三、四级为单作用式)。据悉，该机是我国唯一的一台用于充瓶的迷宫式活塞压缩机。

该机于1995年10月4日～12月24日进行安装调试，其间有SULZER专家进行现场指导。用氧跑合后，三、四级活塞与气缸的间隙四个方向偏差较大。我方现场工作人员曾多次就此事向SULZER专家提出质疑，但SULZER专家认为间隙正常。

# 二、故障现象及原因判断

当该机交付使用约一个月后，中间压力显著升高，尤其是二级排压偏高太多，不得不停机处理。我们先怀疑二级排压的偏高，是由于三级吸入阀的泄漏引起。经检查，并未发现泄漏情况，换上备件后，开机情况依旧，证明三级吸入阀确未发生泄漏。

该机三、四级活塞下部有一个连通的空腔，并有一管道与二级出口管道相连，故我们判断中间压力(尤其是二级排压)偏高，是由于三、四级活塞与气缸内壁磨损造成间隙增大，从而使高压气体泄漏入空腔，进而使二级排压增高。

由于该机仍处于保修期，我们不便作过多的维修工作。于是，我们将该机存在的问题及我们的判断电传给上海SULZER工程机械制造有限公司，由SULZER公司派专家前来维修。

# 三、检查情况

打开三、四级缸盖，测间隙，发现间隙较以前增大很多。吊出三、四级活塞及活塞杆，发现活塞磨损严重，基本证实我们的判断。测活塞与活塞杆的径向跳动，径向跳动值均在  $0.01\mathrm{mm}$  以下，说明活塞杆的垂直度很好。检查各级阀门，发现无问题。进一步证实我们的判断。

# 四、引起间隙偏差的原因及维修

为了便于进一步查找引起间隙偏差的原因，我们采取一种较稳妥的方法，即确保整个活塞安装过程无偏差，尽量使安装过程无偏差积累，从而将引起间隙偏差的原因在安装的最后一个环节表现出来。查找步骤：

1. 换上新的活塞。  
2. 测活塞杆与活塞的径向跳动，发现径向跳动值均在  $0.01\mathrm{mm}$  左右。拧紧活塞头防松螺母后，再测径向跳动，发现径向跳动值亦均在  $0.01\mathrm{mm}$  左右。可见，活塞与活塞杆的垂直度很好。  
3. 连接活塞杆与十字头。为确保活塞杆安装的垂直度，我们在活塞杆与十字头的连接上狠下功夫。反复找平连接用锲形块的各个面，清洁活塞杆与十字头的接触面。因为活塞杆较长，此处极微小的偏差将引起顶部活塞与气缸内壁间隙的很大偏差。  
4. 测活塞与气缸内壁间的间隙，发现间隙情况很不理想。

由于前面的安装精度很高，因此，我们

断定引起间隙偏差的主要原因是缸体与机体对中不好。为此：

（1）移动缸体，反复对中，直至成功。  
（2）用氮气跑合十分钟后，取下缸盖，检查间隙和观察活塞表面磨损情况，发现情况良好。三级活塞小部分密封齿上有局部磨痕，下部光面无痕迹；四级活塞四周有均匀且微量的磨痕。  
（3）用氮气跑合两小时后，取下缸盖，检查间隙和观察活塞表面磨损情况，发现情况良好。三级活塞表面除有一片无磨痕区域外，其余表面有细微均匀的磨痕；四级活塞表面有比较均匀的磨痕，其中一小部分磨损稍严重一些。SULZER专家认为情况良好。

用氧气试充瓶，记录有关参数，中间压力基本正常（见表），数据的重复性很好。1996年3月13日交付我方便用。

梅正清 邓怀飙

# 主题词：分馏塔 热交换器 集合器 管束 断裂

我厂1978年安装投产的  $150\mathrm{m}^3 /\mathrm{h}$  空分设备，由于受市场的制约，一直采用间断生产的方式制氧，这样就增加了KFS=860Ⅱ分馏塔冷热交替的频率，加上制造中焊剂的腐蚀作用，导致热交换器中的集合器管束根部发生断裂，造成严重泄漏，冷损较大，塔内保温材料和塔外壳都结了冰，产量下降  $25\%$  ，难以维持生产。

怎样堵漏和改造，是一个棘手的问题。从1984年第一次发现泄漏至今，消漏堵漏工作曾经进行过多次，方法均为火焰补焊和浇锡。但由于泄漏部位是在管束与集合器花板的连接处（见图1），管束由69根直径  $\phi 10\mathrm{mm}$  和1根直径  $\phi 32\mathrm{mm}$  的紫铜管组成，且集合器距上下热交换器隔层出口最短的一组管束只有  $200\mathrm{mm}$ ，补焊只能在原位置上对外部的管子进行，芯部管子焊不到；用浇铸锡的方法，光是清洗除垢工作就无法达到要求，因此几次补焊和浇锡均告失败。集合器泄漏已到了难以维持生产的境地，若购新换热器，不仅需几十万元资金，且更换周期很长。

《手册》的建立只是以文字形式把制度汇编成册，公布于众，而关键的问题是管理人员和车间职工在实际工作中，如何去严格执行，按章操作不违规。而车间管理者要不断宣传、不断检查，强化岗位教育，从难从严管理，使各项规章制度深入人心，得到职工的理解认可，并自觉地按章执行。

检”，车间专职设备技术人员对装置所有设备一周进行一次全面仔细专业检查，发现问题及时反映，制订措施和方案，联系有关单位进行处理。“月考核”，车间每月对设备运行管理状况进行全面考核，结合“日查”、“周检”的情况，对设备管理、维护和操作的有关人员进行考核，并与当月奖金挂钩，奖惩兑现。

# 三、工艺技术管理

1. 岗位操作工每小时一次工艺技术指标检查，发现问题及时调整和处理。车间专职工艺技术管理人员，每周一次对全装置关键工艺技术指标进行检查，发现问题及时提出及时处理。如空压机油压、油位；冷却水温度、压力，是否按时吹除；纯化器再生、活化、切换是否按时，再生进出口温度是否正常；分馏塔、液空、液氧、液位各节流阀开度，以及各点温度；膨胀机、氮压机、氧压机冷却水温度、压力等，都是直接关系安全生产的关键部位和指标，都必须严格检查，同时每月对车间工艺纪律执行情况、产品合格率、工艺指标控制率、供气平稳率等进行考核。

# 四、气瓶充装管理

气瓶作为移动式反复充装的压力容器，是小空分单位安全生产管理中的一处重要环节，是引起较多伤亡爆炸事故之所在。究其原因，主要是气瓶充装管理不严，充装前未严格检查。因此，严格执行劳动部“89瓶规”，强化气瓶充装前检验，严把预检、复检关，坚持气瓶“八不充”原则，是杜绝气瓶事故发生的关键。坚持气瓶的定期检验制度：盛装易燃易爆气体的气瓶，每三年检验一次；盛装惰性气体的气瓶，每五年检验一次；盛装腐蚀性气体的气瓶，每两年检验一次。同时加强气体充装工的培训和取证工作，加强操作时的操作行为管理，杜绝超压充装。

总之，通过以上一系列的管理，严格执行各项规章制度、操作规程，坚持以人为本，牢固树立“安全第一”的思想，在实际工作中力求做到预防为主，就能做好安全生产管理工作。

# 一、事故经过

1993年5月15日至17日，该机组配电系统发生4次低压电断电事故，每次约1小时后生产才恢复正常。低压电断电，使氨压机停车，导致进分子筛吸附器的空气温度由正常值  $5\sim 10^{\circ}\mathrm{C}$  升至  $40^{\circ}\mathrm{C}$  以上；此时空压机未停，但由于遥控放散阀已坏，而操作工以为时间不长，没有关系，未进行手动放散，使 $40^{\circ}\mathrm{C}$  的空气照常向分子筛吸附器吹送。至17日晚8:00左右，发现两组主换热器的中抽温度开始拉开距离，虽经操作人员努力调整，至18日0:00两组的中抽温度分别是-  $170^{\circ}\mathrm{C}$  、 $-50^{\circ}\mathrm{C}$ ，已无法控制，机组被迫停车。

# 一起氢气充装过程爆燃事故分析

万景隆

（化工部光明化工研究设计院，大连市481信箱，110031）

# 一、事故经过

1997年3月4日中午12时5分，大连化工部光明化工研究设计院高纯氢充装岗位发生爆燃，在百米之外听到巨大爆鸣声，火焰直冲厂房顶棚。由于处理及时，没有造成人员伤亡和更大的财产损失。

# 二、事故原因分析

现场勘察，气瓶完好无损，保持正常充装位置，手轮被火焰烧出一个  $5\mathrm{mm}$  宽的豁口，充装卡具完好，一侧有火焰烧烤痕迹。

检查瓶阀，发现瓶阀开启3/4圈，充装卡具没卡紧。

在现场调查的基础上，经分析造成此次爆燃事故的原因是：

1. 充装卡具没卡紧，漏气。  
2. 待充气瓶处于负压状态。

爆燃发生在切换瞬间，此时待充装气瓶是经过抽空处理的，处于负压状态。当开启瓶阀瞬间，不仅氢气在压力差推动下向气瓶内流动，外界的空气也在负压的抽吸下，通过卡具泄漏点向气瓶里流动，此时，氢气和空气混合，形成爆鸣气，在高速气流摩擦产生的热量和静电的作用下发生氢气爆鸣，并

引燃氢气。

# 三、经验和教训

1. 充装卡具一定要卡紧，不能漏气。如卡具卡紧不漏气，即使气瓶处于负压状态，空气也不能进入，就不会形成氢气燃烧爆炸的条件。  
2. 灭火使用了一个干粉灭火车、四个1211灭火器，也没把火熄灭。切断气源后，火自动熄灭。氢气的猝熄距离小，仅是丙烷的1/3。这起事故，给我们一个很深刻的认识，就是氢气火焰没有颜色，火焰又远离漏气点，在火灾现场只听到漏气声，看不到火焰在什么地方，给救火带来困难。所以，扑灭氢火焰的最佳方法是切断气源。  
3. 氢气最低点燃能量是  $0.045 \times 10^{-3}$  焦耳，是甲烷的  $1 / 10$  。一个人穿绝缘鞋站在地板上，获得的最大静电能量大约为  $15 \times 10^{-3}$  焦耳，是氢气最低点火能的333倍，可见人体上的静电能量，点燃氢气是绰绰有余的，对此，以前我们并没有足够的认识。近几年来穿化纤工作服操作已相当普遍，通过这次事故应引起高度重视。

# * 问答（应知应会）*

# 制氧工问题解答（两则）

1. 问：我厂  $50\mathrm{m}^3 /\mathrm{h}$  制氯机生产相对稳定，开车率达  $90\%$  以上。请问是否可以在空压机进口增加一台风机来提高氯气产量？在分馏塔设计保险系数不知的情况下，气量要增加多大？（答四川某厂读者问）

答：分馏塔的保险系数即后备系数有多少，一般设计时按  $20\%$  进行，不论热交换器、主冷凝器或上下塔都是如此。如果设备制造和维护正常，通过实验考核。  $50\mathrm{m}^3 /\mathrm{h}$  制氧机产量可以达到  $80\mathrm{m}^3 /\mathrm{h}$  ，即实际的后备超过  $20\%$  。应当说明，设计中  $20\%$  余量是给制造缺陷补偿用的，例如热交换器漏、管子堵掉几根、分馏塔板筛孔堵塞等等。所以在挖潜之前，对自己的设备究竟有多少潜力，自己心中必须要有一个底。一般判断，你们这台  $50\mathrm{m}^3 /\mathrm{h}$  分馏塔，产量增加到  $60\sim 70\mathrm{m}^3 /\mathrm{h}$  是可行的。

你们这台设备的薄弱环节是配套机组，首先是空压机，原设计的活塞环的线速度太高，估计是80年代产品。有的厂家把水冷却器加大了，从室内搬至室外，以降低温度；干燥器须加高，否则开车周期会缩短（相对于设计工况而言，但当前的操作压力为2.6MPa，换一种高效分子筛，不加高也可以）；管道布置欠妥当，空压机出口压力经干燥（纯化)器到进分馏塔的压力差有  $0.2\sim 0.3$  MPa，阻力损失大，应改进管道布置，尽力少弯头。

在决定挖潜之前，要对现有设备进行一次维修，派人到有关厂家调查有关配套机组的使用情况，一并考虑检修，最后拍板。

杭氧厂在60年代对  $50\mathrm{m}^3 /\mathrm{h}$  制氧机做过试验，产量可以增加一倍，不过到  $110\mathrm{m}^3 /\mathrm{h}$

后下塔压力增加到  $0.75\mathrm{MPa}$  、上塔  $0.07\mathrm{MPa}$  以上，精馏效果有所下降，主冷面积明显不够，其他配套机组都失去平衡。经过这次设计上的“革命”，我们学到了如何提高自行设计和配套的能力，并提高了职工素质。所以“不可能成功”的推断不完全正确。只要全面考虑，勇于实践，加风机增加气量，增产氧气是可能的。

2. 问：纯化器的保险系数有哪些影响因素？应该如何确定？

答：分馏塔的开车周期与纯化器设计及操作工况好坏大有关系，它是空压机和分馏塔上下工序之间的纽节；倘若加以忽视，开车周期就会缩短。

先说说空压机末端冷却器出口温度对周期的影响。温度增高，则空气中饱和含水量将成倍增加，纯化器要吸收的水分（即负荷）增加，保险系数就降低，设备的工作周期便缩短。

进纯化器的空气压力对周期也有影响，设计时是按理论最低压力计算的，操作压力高于最低设计压力，则空气中携带的饱和含水量相对减少，纯化器的负荷下降，保险系数就增加，也有利于延长设备的工作周期。

再谈分馏塔工况好坏对纯化器工作及周期的关系。中压流程的高压压力与膨胀机工作好坏(即产冷量高低)密切相关，膨胀机单位制冷量低，则高压压力升高，进膨胀机气量增加，对纯化器工作有利；进分馏塔的温度过高，则换热器的保险系数大大减少，对工作周期更加不利。

据上所述，高压压力提高可延长工作周

期，但是耗电量随之增加，这是人们所不希望的。如何把操作压力降低、进塔温度（即纯化器工作温度）降低，是操作工所要掌握的两个大问题，也就是说纯化器的工作压力和工作温度，应当控制在纯化器的设计规范上。压力高低要由分馏塔与膨胀机的操作好坏而定；工作温度的大小，由空压机的好坏特别是冷却器的冷却效果而定。

纯化器的工作压力和温度，根据制氧机总体设计的最低压力和平均温度来计算，然后加上  $20\%$  的保险系数。80年代的分子筛制造质量不稳定，为使设备安全运转，规定的保险系数要低于上述数据，或者把设备的工作周期定得短一些。有些人提出为什么有的制氧机周期可达一年以上，而有的制氧机说

明书规定工作周期只有两个月，就是因为互相影响的因素很多，设计人员很难将精确的数据相告。

纯化器只能承受每小时吸收多少公斤的水分(和  $\mathrm{CO}_{2}$  )，如若给它的负荷超过了能力许可，只好把多余水分转嫁给分馏塔了，其后果是周期大大缩短，责任不能由纯化器承担，道理很简单，它的能力后备被空压机或分馏塔吃丢了。

所以要延长空分设备的周期，就必须综合考虑空压机、纯化器及分馏系统。用现代话来说，是一个系统工程，是深层处的结构改进问题，有其规律性的技术内涵，值得我们学习研究。

# 工业气体氮、氧、氢应用三例

# 一、充氮轮胎

美国空气制品公司开发了一种新的轮胎充气系统，它充入轮胎的不是空气，而是氮气，从而提高了轮胎的性能和安全性，降低了成本费用，并且减少了对环境的污染。

充氮轮胎适用于民用飞机与军用飞机、宇航飞船运输车、赛车、挖土机以及采矿设备等。

氮气在轮胎橡胶中向外扩散的速度，比空气低 $30\% \sim 40\%$  ，因此能够在较长的时间里保持适当的轮胎压力，能在路上保持正确的轨迹，从而减少了轮胎的磨损。

如果充入潮湿带油的压缩空气，会导致轮胎内部的氧化和轮子的腐蚀。由于氮气是不含水分的洁净气体，因此能大大减少轮胎的氧化和轮子的腐蚀，延长轮胎的使用寿命。

如今，对于公路上行驶的车辆来说，如果使轮胎的寿命延长  $5\%$  ，就意味着每年少废弃1000万个轮胎，这对于环境保护来说无疑起着一种积极的作用。

充气适当的轮胎行驶起来比充气不足的轮胎快捷，因而能使燃料消耗降低  $10\%$  ，同时也减少了对空气的污染。据美国能源部报道，由于行驶车辆轮胎气压不足，每天要损失200万加仑的汽油。如果采用充氮轮胎，就可以大大减少这项损失。

# 二、氧、氮、氢在玻璃生产中的应用

瑞典 AGA 公司和英国 Lahti 玻璃厂签订了一项为期 12 年的合作协议，价值约为 700 万芬兰马克。这项协议旨在帮助芬兰发展平板玻璃工业。

在玻璃熔炼炉中加入氧气，不仅能提高产量，还能降低成本费用，减少对空气的污染。平板玻璃的生产采用浮法玻璃生产工艺，即让液态玻璃流到熔锡之上，并用氮气和氢气作为保护气体。

AGA公司负责为生产平板玻璃提供所需要的各种气体。

纯液氮，用自备槽车运往深圳大亚湾核电站。目前粤港公司日产氧气24万  $\mathrm{m^3}$  ，计划投资更多的液化装置，积极发展珠江三角洲气体市场。目标是建立华南最大的工业气体公司，为此正在筹建“广东气体集团公司”。粤港公司营运颇好，1996年税后利润达2500万元，人均创利8万元。

# 二、港氯的开拓应用与专业服务

香港氧气公司有个服务宗旨（港氧也称为“目标”），叫做——“以气体为本，再发展配套设备，向气体用户提供‘一条龙’专业服务”。这个宗旨，港氧不仅用于香港本地，也用于“南中国部”。

就在这个方针目标(宗旨)下，港氧不断开拓服务领域，向外扩大经销业务，既有气体，也有设备。下面仅将收集到的一些信息作一简介：

1994年2月，港氧装备一辆全新车辆，载重达1400公斤，车上装一台容量为731公升的小型液氮槽，为小型液氮罐用户提供快捷的注氮服务。

1994年5月，港氧获得为期18个月的合约，为29家医院进行整个医疗气体支管系统的勘察、评估及提高等级的工作。

●港氧的液氮，除供香港电子、煤气、食品行业外，香港新机场工程是最大用户（混凝土施工作冷冻剂用），下一个大用户是香港新地铁线路施工工程。

●港氧的“液氮混凝土冷冻技术”，已被香港赤沥角新机场工程建造商会采用，应用于建造新机场连接马湾及北大屿山的汲水门大桥。这座大桥(也叫青马大桥)全长1.4公里，是世界最长的吊桥之一，桥中两条混凝土柱要支持吊桥的重量，结构质量至关重要，港氧提供“液氮混凝土冷冻技术”，以确保混凝土的拉力强度达到预期的标准。为此港氧提供给建造商的液氮贮罐及液氮喷枪，安装于施工现场，1994年初开始使用液氮，一直供应到1995年底工程竣工时为止。浇注混凝土用的液氮，视气温而定，估计一天最高可达50公吨液氮。液氮用于大型混凝土浇注行业，可以获得高强度的混凝土。因施工时如遇气温较高，会加速混凝土老化，降低强度，最后会崩裂。传统的方法是用冷水或冰作冷冻剂，但冷冻效率较差，制成的混凝土质量

也就差。而用液氮作冷冻剂，温度可以低到 $-196^{\circ}C$  ，液氮吸热后迅速汽化成气体溢出，不会停留在混凝土内，使混凝土质量控制较为容易。

●港氧“液氮土壤冷冻技术”。港氧对香港观塘丽港城对开污水站地下混凝土井（深30米），在三日内灌注进120多公吨液氮，通过30多米钢管不停地把液氮灌进护土门附近的泥土，如期形成一堵厚1.5米以上、温度为 $-30^{\circ}C$  的冰拱，将“饱和土壤”凝结成一道坚固的“冷墙”，防止了泥土涌入井内，使整个庞大的排污计划得以顺利进行，成为香港首次应用液氮冻土技术于基建工程上。

●港氧已获得一些工程项目的供气合约，最大一项是香港赤沥角新机场工程，由港氧提供各种工业气体及焊接设备，所提供的工业气体除氧气、乙炔气外，还有二氧化碳、氩气和“美易烧”（即  $\mathrm{CO}_{2} + \mathrm{Ar}$ ）。由于赤沥角工程施工多在岛上，港氧就提供定期的航班，用驳船运送气瓶到马湾、深水角、东涌和赤沥角。

$\bullet$  港氧还将氮气用于大型工程车辆（如赤沥角新机场施工的巨型挖土机和运土车）轮胎（直径大到3米)充气，以减缓轮胎老化和金属内轮氧化。

1993年，港氧还向社会推出一种利用干冰颗粒代替传统的喷砂除锈工艺。这项技术是利用液氮汽化所产生的高速气流，通过专用设备夹带干冰颗粒，从喷枪内高速喷出，直接打到工件上，除锈效果良好。其优点是不会污染环境，不会产生火花；缺点是投资成本较喷砂为高。

1992年，港氧提出用氮气及二氧化碳混合气包装中秋月饼，以延长月饼贮藏期及抑止霉菌生长。经市场推广，反应良好，并进行了实质性试验。

$\bullet$  港氧尝试推广一种游泳池酸碱度控制技术。有时游泳池池水会使人双眼感到刺痛，

睁不开眼，这是因为池水在消毒时，酸碱度控制不理想。最适合的游泳池池水酸碱度为 $7.2\sim 7.6$  。为此，港氧气体发展部尝试推广一种池水酸碱度控制技术，其原理是将二氧化碳溶入池水中，产生缓冲溶液，从而把酸碱度维持在  $7\sim 8$  度之间，这样就可舒舒服服睁开双眼游泳了。

$\bullet$  港氧气体发展部在1994年9月引入一种全新的冷冻运输方法——雪袋。在运送食物前，利用二氧化碳的独有特性，把液态二氧化碳减压成  $-78^{\circ}C$  的干冰雪花，然后直接将雪花射向一个预先安装在冷冻货柜内的尼龙透气雪袋；在喷射雪花的同时，货柜内的暖空气会被排出车外。在运输途中，冷冻的二氧化碳气体会从尼龙透气雪袋慢慢渗出，使货柜内的食物能够长时间保持在低温环境里。雪袋简单可靠，成本甚低，大小及形状可随各类冷冻货柜车随意设计。港氧气体发展部已与专责运送麦当劳冻肉的运输公司联系上，应用此项新的冷冻技术。

●港氧用“发光波”制造“人造月亮”。为增加晚上户外活动的气氛，又作照明，港氧特从法国引进“发光波”，并于1994年11月在将军澳厂房区试升。“发光波”，实是一只直径达1.7米的大气球，气球内除氦气外，还有两个共2000W的电灯泡，在8米高空，尤如“人造月亮”，能照亮1000平方米的地面。

●港氧试验研究不停产修理液体管道——冻管技术。修理或更换大型装置管道或阀门时，通常要临时关闭部分设备，采用冷冻套管技术就可以不停产检修。这个喉管冷冻技术，就是采用套管或容器把管道套住，然后注入冷冻剂，如液氮，管道内液体即被冻住而形成“实心塞”，将需修理的部分作局部隔离，便可进行修理了。套管（容器）为上下模件，配以喉管外的大小，造一个不渗漏的外套空间。但密封材料较难，往往液氮会渗漏出来。港氧气体发展部经过多次试验研

完，终于找到一种合适的密封材料，再加上容器设计方面的改进，用钢管代替以往的木板及更小巧的固定方法，经实践再未出现液氮渗漏情况，工程也迅速完成。

●港氧的液氮和氮气用于：医疗上用于保存组织及器官，或简单的外科手术上；食品急冻及包装，公认为最经济、效率最高及符合卫生的食品保鲜方法；为液化石油气运输船转注汽化氮气，以减低易燃石油气着火的机会；加入氮气，使铜片在退火过程中保持表面光亮无氧化；用氮气充灌飞机轮胎；降落时能平稳无弹性；炎夏暑假，用液氮制造飘雪乐园的仿真雪；再就是液氮用于混凝土浇注，以控制其温度；用于深冷法维修管道，可不停产检修；……。

1995年12月，港氧医疗快速服务车正式投入服务，范围包括香港、九龙、新界。这使港氧“服务为先”的目标更跨进了一步。此车由一辆客货车改装而成，主要装备有小型制氧机、G尺码和小型氧气瓶及附带的配件，如氧气表、氧气面罩及导管等。车上还安装一部手提电话，可直接与公司和客户联系，提供紧急服务给顾客。“医疗快速服务车”主要服务对象为一般家居用氧客户和紧急需要氧气的病人，同时也到医院进行宣传。

●港氧代理各国名牌的焊接器材，为此研制各种焊接用特殊混合气。

至1995年底，港氧拥有55辆不同种类、不同大小的运输车辆，担负运送各式各样的气体给不同的客户，主要将气体运往大型客户及作往来将军澳厂与大埔厂之用。在运送液氮、液氧、液氩及二氧化碳方面，有不同的槽车及罐车，最大的容量为27000公升，小的也有7000公升。最小巧为车上置的600升的小型液氮罐，为小客户提供液氮送货服务。随着液氮需求的不断增加，1992年港氧以150多万元从法国购入一辆大型液氮槽车，全长9.9米，载液氮25620公升，相当于

20.7公吨，总重量为32公吨。该槽车采用多层真空绝热，装有高准确度流量计以测度液体输送量。

●港氧小型液氮转注。港氧设于大埔厂房的小型液氮转注装置，是香港液态及气态氨的一个重要来源，液氮由美国加州洛杉矶BOC Gases、America的工业城的3400加仑液罐送到香港的(需约30天)，再转注到大埔厂小型液氮贮罐，大都是250升或300升，注满后即运送给客户（因每天有  $0.7\%$  的蒸发损失）。

# 三、港氧的产品质量政策与生产安全意识

优质产品和满意服务，加精益求精，是港氧的长远目标。满足顾客的要求，是港氧的一贯宗旨。执行董事傅立德先生制定的品质政策，其中一项说：“本公司的宗旨，乃是向客户提供最优质的产品和有关的服务，并确保在符合一切标准规格外，同时使客户感到绝对满意。”因此，港氧员工尽忠职守，敬业乐业；诚恳服务，顾客至上，成为港氧的

一贯作风。

1992年11月，港氧荣获ISO9000资格。继ISO9000之后，港氧又进入另一行动，即“全面品质管理”（TQM）。它是以“顾客为先”的一套管理方法，通过每位员工积极参与改进自己所负责的工作程序，不断改进，提供给顾客一贯优质的产品和客户服务。1994年3月，港氧“客户服务及品质局”宣告成立，负责统筹、监察和审核客户服务及品质改善计划的工作。

港氧的品质系统，由文件管制，用特别纸张印制港氧的“品质手册”、“运作指示”和“工作指引”，做到有规可循，有章可依。

港氧还非常注意、重视安全生产，经常成功地推行法国母公司法液空的“零意外月”活动，做到无工伤事故发生，并把这种精神与行动在日常中持续下去。1994年7月12日，港氧全体员工获得“一年无工伤”佳绩，坑口厂更成功地跨越“一千日无失时工伤”的大关，向三年无工伤迈进。1994年8月，港氧加入法国气体集团“零”意外俱乐部，其加入条件必须是法国气体集团之属下公司，并要具备一年无失时工伤的记录。1994年11月，英国氧气公司颁发一项“完美安全奖”给港氧，以表扬港氧在太平洋区得到最佳员工安全记录。

“零意外”，已成为港氧的永久目标。年初，港氧提出“全年零意外”目标。将母公司的“安全巡视及安全核查”成功地引入公司，持之以恒地实行。

港氧执行董事傅立德先生曾指出：“要把安全工作的基础打得更巩固，提高安全意识是公司重要的长期方针，不只是为了记录，更主要是为了保障大家的安全和健康。”

秉承港氧原则“运作均须符合严格的职业健康和工业安全准则，并关注环境保护。”1995年港氧正式成立了一个名为“特殊气体紧急事故应变系统”，此系统为专门安全及有

效处理一切有关带毒性、腐蚀性、致癌及自燃气体的紧急事故，其成员共7人。1995年10月  $3\sim 5$  日，在大埔厂举办了一个特殊气体事故处理训练，特从澳洲及美国请来三位世界知名的专家负责教导。

# 1996年度全球气体贸易概况

1996年，全球各工业气体公司都采用了各自不同的贸易途经，在世界各国积极进行气体贸易，取得了厚实丰果。1996年全球的气体贸易总额约为280亿美元，280亿美元的全球工业气体市场就象一只圆的大饼，被全球8大工业气体公司瓜分蚕食。每家气体公司都有各自的优势，因而走向国际气体贸易市场的道路和方法也各不相同。下面就全球8大工业气体公司1996年度的气体贸易情况作一概述。

# 一、法国液化空气公司（L'Air）

法国液化空气公司1996年在全球的280亿美元气体贸易总额中约占  $18\%$  份额，位居第一。

由于法国液化空气公司在亚洲和美国不断努力寻找合作伙伴，提高了生产率，致使其1996年上半年

年的净利润比1995年同期增长了  $1.7\%$  。1996年下半年，法国液化空气公司与丹麦Haldor Topsoe A/S公司就合作生产制氢设备、一氧化碳及二氧化碳设备问题达成协议。根据双方合作条款，法国液化空气公司提供生产设备及分离纯化产品的技术，丹麦Haldor Topsoe A/S公司提供用于设备中的催化剂及有关技术，并在世界各地共同销售上述设备和气体。同时，法国液化空气公司还可利用丹麦Haldor Topsoe A/S公司的有关技术，接受世界各国关于制氢设备、一氧化碳和二氧化碳设备的工程合同。两公司的全面合作，将会大大提高法国液化空气公司的全球性效益。

# 二、英国BOC集团（BOC）

1996年，BOC在全球280亿美元气体贸易总额中约占  $15.6\%$  份额，位居第二。

1996年第三季度，BOC集团生产利润比1995年同期的生产利润增长了  $11\%$  ，而1995年第三季度的生产利润比1994年同期的生产利润仅增长  $7\%$  。为了在2000年前占有更多的国际气体市场，取得更高的生产利润，BOC继续努力向全球气体市场挺进。为夺取南美工业气体市场，提高其在南美的知名度，1996年BOC在南美投资建造一台  $10000\mathrm{m}^3 /\mathrm{h}$  制氢设备；在进入白俄罗斯工业气体市场的同时，1996年BOC与印度签订了一项1290吨/天的制氧制氮设备合同。根据BOC目前现存股票，通过预托证券公司的商业电视节目，引起了华尔街的密切注视。截至1996年底，BOC已有  $5\%$  以上的股份在美国，约有  $20\%$  以上的BOC人员在美国从事气体生产和贸易，在美国的生产销售额达10多亿英镑。

# 三、美国普莱克斯公司（Praxair）

1996年，美国普莱克斯公司在全球280亿美元气体贸易总额中约占  $14.4\%$  份额，继法国液化空气公司和英国BOC集团之后，位居第三。

1996年1月，美国普莱克斯公司兼并丁原为全球九大工业气体公司之一的美国液碳公司后（美国液碳公司原为美国CBI公司的子公司，1995年12月被Praxair兼并），引起了华尔街对这一问题的注意。Praxair公司1996年除为CBI还清债务、放弃部分CBI原签订的合同和合理使用及解决CBI人员外，它还能保持稳定的收入。Praxair1996年第二季度的销售额就比1995年同期销售额增长了  $39\%$  。

在近几年"时间里，Praxair公司气体销售额 $50\%$  以上不是在美国市场上获得，而是在国际市场上成交。因此，Praxair公司在全球的知名度随其气体贸易向全球的推进而变得越来越高。为其项目（设备设计和制造)实施规划、财政报告制度和其它的各项活动，更有效地推向全球，Praxair公司于1992年开始实行其《加速质量实施》纲要。项目执行规划是Praxair公司实施各项全球工作的重点，因此，Praxair公司的美国、巴西、意大利和中国都设有项目执行规划小组，以协调纽约总部的订购、计划和人员调度等方面的工作。

从1992年至1996年，Praxair公司的气体贸易已从全球的15个国家扩展到45个国家，贸易几乎翻了一番。由于兼并美国液碳公司，大大地提高了

Praxair公司二氧化碳在全球的销售量，并保持了二氧化碳在全球的主导地位，巩固了其非低温制气法在全球的领先地位。

# 四、美国空气制品公司（APCI）

1996年，美国空气制品公司在全球的280亿美元气体贸易总额中约占  $8.4\%$  份额，位居第四。

美国空气制品公司很早就进入国际气体贸易市场，自1957年进入英国气体市场后，至今已发展到亚洲、非洲和北美30个国家的气体市场，北美是它最大的气体市场，也是最具有发展潜力的地区。美国空气制品公司之所以向欧洲、亚洲和拉丁美洲国家扩充自己的气体市场，其原因在于那些国家在21世纪中是全球绝大多数国家中具有最有发展前途的国家。因此，美国空气制品公司通过在那些国家建立起自己的拳头产品和市场，以发展和扩充自己的势力范围，利用自己的技术、产品和售后服务，来获取气体用户的信任。

电子工业和化学制品加工业是美国空气制品公司的两个最大市场。90年代初，美国空气制品公司开始增加它的投资项目，使其在世界各地建立的气体市场能同时得到发展。三年前，为了提高在工业气体行业中的地位，该公司在全球同时推行了气体贸易和设备贸易一体化策略。决策层采取了许多措施，以确保其全球气体贸易和设备贸易一体化的实现。1996年，美国空气制品公司已完成了以市场为核心的改组工作，重建了一支多功能市场队伍，在全球经营电子工业和化学制品加工业方面的气体贸易，协调和管理不同地方的贸易、技术和财政等方面的工作。此外，1996年该公司还加强了工程设计和生产制造的力量。

# 五、瑞典AGA公司

1996年，瑞典AGA公司在全球280亿美元气体贸易总额中约占  $6.8\%$  份额，位居第五。

近年来，瑞典AGA公司通过把产品打入美国市场，逐步扩大其在全球的气体贸易。1996年，瑞典AGA公司投资3000万芬兰马克，为芬兰Raularuuki公司建造了一台  $1000\mathrm{m}^3 /\mathrm{h}$  制氢设备，以提高和改变天然气的质量，保护环境。1998年中期，AGA公司还将为芬兰西南的Fundia公司投资9000万芬兰马克，建造一台制氧设备，以取代Fundia钢铁厂的旧制氧设备。为了加强巴西的气体贸易，满足巴西饮料和食品业发展的需求，AGA公司

1996年在巴西的Rio de Janeiro地区建造一台价值930万美元的3100吨/年二氧化碳生产设备。1997年，AGA公司还将与巴西Peixoto de Castro集团合作生产氢气，以供该集团化工和石化业的需求。

# 六、德国林德公司(Linde)

1996年，林德公司在全球280亿美元气体贸易总额中约占  $5.5\%$  份额，位居第六。

林德公司全球气体市场首要目标是中国，自70年代末80年代初进入中国气体市场后，至1996年已占领了中国许多气体市场。1995年，林德公司向澳大利亚进军，1996年扩大了在澳的气体市场。

# 七、德国MG集团

1996年，德国MG集团在全球的280亿美元气体贸易总额中约占  $5.4\%$  份额，位在第七。

MG集团于1992年开始其全球性的气体贸易规划，至1996年，MG在全球已有30多家子公司，并在美国和欧洲已建有几套空分设备。MG全球性气体贸易市场的重点是美国，亚太地区的韩国、印尼、新加坡、印度及中国的几家合资公司，还有中东地区的阿拉伯联合酋长国的迪拜国。MG在欧洲的芬

兰、保加利亚、匈牙利、意大利、英国和南斯拉夫都设有分公司，其目的是想通过这些分公司扩大全球气体市场的占有率和发展新的气体市场。

在近几年的时间里，MG的气体设备和气体销售额都一直在稳步上升。目前，MG在全球的31个国家和地区已有80多家合资公司。在气体分离方法方面，MG采用低温和非低温两种方法来生产各种气体，并能用膜分离法生产氮气。由于MG集团于1996年与意大利的Geogas公司和匈牙利的MGMOL GAS公司合资控股，因此大大加强了MG在欧洲气体市场的二氧化碳销售地位。

# 八、日本酸素公司(NSC)

1996年，日本酸素公司在全球的280亿美元气体贸易总额中约占  $4.0\%$  份额，位居第八。

上述全球八大气体公司，1996年在全球280亿美元气体贸易总额中共占有  $78.1\%$  份额。俄罗斯、中国、印度和瑞士等国的气体公司在280亿美元中占有  $21.9\%$  。
