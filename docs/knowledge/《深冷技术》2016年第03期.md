2. 2
高效低阻力精馏塔
精馏塔是空分设备最主要的核心部机之一，以
填料与分布器为代表的塔内件的性能优劣直接决定
了空分设备的分离效率。杭氧通过高效精馏塔塔内
件的集成创新技术，实现了超低温条件下氧、氮介
质的多相连续精馏，使精馏效率大幅提升，达到了
国际先进水平。
( 1)
通过开发填料的光边结构等特殊结构形
态，提高了规整填料的整体通量; 特殊的表面处理
提高了液膜润湿率，从而保证填料传质性能的充分
发挥。
( 2)
杭氧通过开发新型分布器结构，开发出
悬槽式带垂直布液板的槽式液体分布器。其具有分
布流量大、流量变负荷能力强、抗堵塞及分布性能
稳定等优点，使常规的点分配形式变成线性分配形
式，充分保证了分布器的水平度而不受填料床层变
化的影响，有效达到液体充分混合和均匀分布的效
果，不仅提高了塔内件的操作弹性，更能发挥填料
性能。
同时开发了能适应多点进料分布、液体分布点
阵变化、避免气体区域性集中和偏流放大效应的精
馏塔内件，使空分设备的整体性能得到了大幅提高。

富氧燃煤( 氧气/二氧化碳)
发电技术能直

富氧燃煤空分流程特点
富氧燃煤对氧气的要求和其他行业对氧气的要

需求量大、氧气纯度不高( ≥95% ) ，不需要氮
气、氩气及液体产品。

4. 1
空气过滤和压缩( 分为高压力部分和低压力
部分)
高压力部分: 空气首先进入自洁式空气过滤
器，被除去灰尘和其他颗粒杂质，然后进入高压力
空压机，经过多级压缩后进入空冷塔，压缩机级间
的热量被中间冷却器中的冷却水带走。
低压力部分: 同高压力部分。
4. 2
空气冷却和吸附( 分为高压力部分和低压力
部分)
高压力部分: 空气在进入分子筛吸附器前在空
冷塔中冷却，以尽可能降低空气温度、减小空气中
的水含量，从而降低分子筛吸附器的工作负荷，并
对空气进行洗涤。进入空冷塔上部的冷冻水，首先
在水冷塔中利用干燥的出精馏塔的污氮气进行冷
却，然后再进入空冷塔上部。
分子筛纯化系统由2 只分子筛吸附器和蒸汽加
热器组成，分子筛吸附器吸附空气中的水分、二氧
化碳和一些碳氢化合物，2 只分子筛吸附器一只工
作、另一只再生。
低压力部分: 同高压力部分。
4. 3
空气精馏
净化后高压力部分的加工空气分为两部分: 一
小部分被抽出作为仪表空气; 另一部分进入主换热
器，被返流气体冷却。冷却后的空气分成两股: 一
股进入高压力下塔; 另一股进入液氧蒸发器加热液
氧使液氧汽化，被冷凝成液体后一部分进入高压力
下塔，另一部分进入低压力下塔。空气经下塔初步
精馏后，在下塔底部获得液空，在顶部获得液氮。
净化后低压力部分的加工空气分为两部分: 一
部分进入主换热器，被返流气体冷却，冷却后的空
气进入低压力下塔; 另一部分相当于膨胀量的空气
进入膨胀机增压端增压，经增压后冷却器冷却后进
入主换热器，被冷却到一定温度后从主换热器中部
抽出进入膨胀机，膨胀后的大部分空气进入上塔。

大气中的有害污染物，包括碳氢化合物、氧化
亚氮、二氧化碳、有机气溶胶等，对于空分设备是
一种危害。行业内，由于大气污染物引发的事故教
训非常惨痛和深刻，如中国的抚顺事故、马来西亚
的宾特鲁事故等。因此对于大气有害污染物，应当
采取必要的风险控制及防范措施。特别是在项目建
设初期，对于它们的危害性应当进行仔细评估，研
究分析它们对设备布置的影响，以避免影响空分设
备的运行可靠性与安全性。大气有害污染物包括可
燃杂质如乙炔、丙烷等，腐蚀杂质如二氧化硫、氨
等，以及堵塞杂质如二氧化碳、氧化亚氮等。下面
主要分析讨论二氧化碳对空分设备的危害与应对
措施。

二氧化碳的性质与危害
大气中的主要成分是78. 08% 的氮、20. 95%
的氧、0. 93% 的氩以及0. 036% 的二氧化碳，还包
括其他微量元素，如氖、氦、氪、氙等。大气中二
氧化碳的含量随工业化进程在世界范围内逐渐增
加。200 多年之前大气中二氧化碳的含量约为280 ×
－6，而目前世界气象组织全球大气监测网的多个
监测站测得大气中二氧化碳的含量均已超过了400 ×
－6这一阈值。
二氧化碳常温下是一种无色、无味气体，密度
比空气大，能溶于水，与水反应生成碳酸，不支持
燃烧，固态二氧化碳俗称为干冰。二氧化碳被认为
是人类活动排放的最主要的温室气体，是造成气候
变暖的罪魁祸首。纯二氧化碳的温度—压力平衡曲
线如图1 所示。

空分工厂布置考虑
空分工厂主要包含空压机系统、空气预冷系
统、分子筛纯化系统、空分冷箱和液体贮槽区等几
个组成部分，各组成部分间由内管廊连接。
空分工厂布置的目标是在满足工艺要求的前提
下安全、可靠、经济。设计布置时至少应当考虑以
下几个方面。
( 1)
工艺要求: 满足流程要求，考虑设备尺
寸大小等。
( 2)
客户需求:
包括需要的装置产量大小，
物流方向如动力电源、蒸汽及冷却水来源，相关产
品交接点，今后发展需求，构建筑物如机器厂房、
办公室、控制室、变电站和维修车间等要求。
( 3)
厂区条件: 应考虑场地条件，包括厂区
大小、气象条件及周围环境。
( 4)
厂址选择: 需要考虑周围地理环境，选
择最佳场地，空分工厂应当在污染源的全年最小频
率风向的下风侧。
( 5)
绿化要求及布置优化: 绿化覆盖率，厂
区环境美化，方案比较与选择。
( 6)
设计规范: 符合规范要求，考虑各个最
小空间及间距要求，包括安全间距、分离间距、安
装间距、维修间距及操作间距等。安全间距是针对
危险源与人身最小安全距离而言，而分离间距是指
危险源与设备的最小距离。下面主要讨论安全
间距。
安全间距是指相关人身风险可以得到控制的最
小的危害源与人身的距离。应当考虑防火防爆、污
染物以及噪声的影响。对于噪声，应当满足国家标
准《工业企业厂界环境噪声排放标准》
( GB
12348—2008) ［4］对厂界的要求，使噪声不超过规
定要求。针对防火防爆，应遵循国家有关强制条
文，如标准GB 16912—2008 以及《建筑设计防火
规范》( GB 50016—2014)
［5］的规定，包括氧气设
施与可燃储罐间距，消防要求如通道宽度大于4 m
等要求。考虑污染物的影响，应当进行必要的定量
风险分析，选择最佳空气吸入口。
二氧化碳扩散分析
现在主要讨论如何进行必要的定量风险分析，
考虑二氧化碳污染物的影响，选择最佳空气吸
入口。
首先对于拟建空分设备的周围场地应进行调
研，了解已有或今后可能有的各个排放点的情况，
调查周围场地气象条件，然后进行必要的污染物定
量扩散计算。污染物扩散是指其进入大气变成云团
随时间和空间变化的情况，与许多因素有关，如排
放物的成分、排放量及出口流速、温度和压力、排
放塔的直径或截面积、高度及坐标位置。当然与场
地气象条件密切相关，如风速、风向( 主导风向
或风玫瑰图) 、最高/最低环境温度及相对湿度、
表面粗糙系数及大气稳定度等。
当大气环境温度高时，对排放扩散会造成不利
影响，风速会影响排放气体与空气的混合速率，低
风速通常对排放扩散不利。主频风向在总图布置时
应予以考虑，但排放设计时应考虑任意风向。
在大气污染物扩散规律的研究分析中，大气稳
定度是非常重要的参数，它是指气团受任意方向扰

结束语
空分设备的布置需要考虑周围环境的影响，特
别是大气污染物如二氧化碳等的影响。项目建设前

分子筛具有均匀的微孔孔道结构，小于孔径的
分子能够被吸进孔道内，大于孔径的分子则被挡在
孔道外，根据分子的大小，把各种组分分离，目前
习惯称之为沸石或沸石分子筛。沸石分子筛的早期
研究源于19 世纪，20 世纪五六十年代开始工业
化。近年来，沸石分子筛的基础科学研究和工业应
用迅速发展，促使其在气体吸附分离、催化、材料
等科学领域广泛使用。下面具体介绍沸石分子筛的
研究状况及在空分设备中的应用。
分子筛的研究状况
［1-3］
1. 1
沸石分子筛的概念
沸石分子筛，通常被定义为具有被很多大的离
子和结晶水分子占据孔道骨架结构的铝硅酸盐化合
物，具有微孔孔道结构( 孔道尺寸范围在2 nm 以
下) 。经典的化学式为:
［M2 (Ⅰ) ，M (Ⅱ) ］O·
Al2O3·nSiO2 ·mH2O，M2 (Ⅰ)
和M (Ⅱ)
为可以

+ ∞，m 为
结晶水的摩尔数。
沸石分子筛在过去的几十年中发展迅速，通过
改变分子筛的硅铝比，开发了一系列低硅( Si/Al =

1. 2
分子筛的合成
通常分子筛的合成途径为水热或溶剂热合成
法。水热或溶剂热合成是指在一定温度和压强的条
件下，利用溶剂中的特定化学反应进行合成，其体
系往往处于非平衡状态。目前，水热合成是沸石分
子筛等大量微孔化合物最好的合成方法，提高了反
应物的溶解度和反应活性，使初级凝胶重排及溶
解，提高成核及晶化速度。水热合成的影响因素很
多，主要包括: 反应物的组成及物源种类、硅铝
比、溶液的pH 值、陈化条件、晶化温度与时间等
诸多因素。
1. 3
分子筛的孔道控制
由于分子筛的择形效应，大孔化成为许多科研
人员的研究方向。目前的研究成果表明，分子筛的
孔道包括6、8、9、10、12、14、18 和20 元环。
沸石分子筛的孔径变化范围为0. 3
1. 2 nm，孔道
变化不连续，如从6 元环到8 元环，而且还要受到
阳离子交换度、温度效应等影响。同时，科学家们
利用分子筛本身的结构缺陷( 如晶格的错位堆
积) 、孔道内的离子接枝等作为控制分子筛孔道的
手段，近年来提出的修饰技术主要是分子筛的同晶
交换、分子筛内部结构的修饰以及外表面修饰。
1. 4
分子筛吸附剂主要性能指标
分子筛吸附为物理吸附，吸附是由于分子间的
吸附力的存在而产生的，吸附力是分子间的作用
力，即范德华力，它与气体分子、吸附剂分子的自
身性质有关。
分子筛吸附剂是对气体组分的选择性吸附分
离。吸附性能的主要指标是吸附容量。吸附容量分
为静态和动态两种。静态吸附容量，是指在一定温
度和被吸附组分浓度一定的情况下，每单位重量或
单位体积的吸附剂达到平衡时所能吸附物质的最大
量，即吸附剂的吸附容量能达到的最大值( 平衡
值) 。动态吸附容量是吸附剂到达“转效点”时的
吸附值( 吸附器内单位吸附剂的平均吸附容量) 。
动态吸附容量是吸附床层设计的重要依据，主要通
过试验数据而来。动态吸附容量的大小主要受以下
几种因素的影响: 吸附质分压、气体流速、进气的
相对湿度、吸附过程的温度、再生程度和吸附剂床
层设计等。
分子筛在空分设备中的应用
2. 1
深冷空分分子筛纯化系统
［4-6］
在深冷空分设备中，分子筛纯化系统的作用就
是清除空气中所含的水分、乙炔、二氧化碳、烯烃
等杂质，防止这些杂质随空气进入低温换热器、透
平膨胀机或精馏塔等设备造成冻堵，从而保证空分
设备长期安全运行。
分子筛纯化系统完成吸附杂质、净化空气的功
能是通过吸附剂来实现的。一般利用活性氧化铝对
空气中饱和水吸附容量大的特点，初步吸附水分，
再利用分子筛进行深度吸附，使处理后的空气干燥
到需要的露点。目前，也有在活性氧化铝的下层铺
垫惰性氧化铝。惰性氧化铝具有耐高温高压、化学
性能稳定、耐腐蚀等优点，能支撑和保护吸附床层
中的活性氧化铝及分子筛，增加气流均布等，但惰
性氧化铝本身不会吸附。
空气通常在较低的温度( 5
20 ℃左右)
下
进入分子筛吸附器。分子筛对水分、乙炔和二氧化
碳等极性或不饱和分子有很强的亲和力，其吸附性
能顺序是: 水分＞乙炔＞二氧化碳。而出吸附剂床
层的空气中杂质的顺序为:
甲烷( CH4 ) 、乙烷
( C2H6)
＞乙烯( C2H4 ) 、丙烷( C3H8 ) 、二氧化
碳＞乙炔( C2H2 )
＞丙烯( C3H6 )
＞丁烷
( C4H10)
＞丁烯( C4H8) 。因此在分子筛吸附器工
作周期内控制出口空气中二氧化碳的含量，就能达
到去除乙炔、丙烯、丁烷和丁烯等目的。
空气中对分子筛有害的杂质有: 二氧化硫、氮
氧化物( NO、NO2 ) 、氯化氢、氯、硫化氢和氨
等。在上述有害杂质中，氯化氢、氨最易被水洗
涤，其次二氧化硫、三氧化硫和二氧化氮也能被水
洗涤。但空气中所有在水中的溶解度比二氧化硫低
的杂质，都不能被洗涤，如硫化氢、氯、二氧化
碳、一氧化碳、一氧化氮和碳氢化合物等，但能够
被分子筛吸附。其吸附的强弱顺序为: 水＞氨＞硫
化氢＞二氧化硫＞二氧化碳。空气中杂质诸如二氧
化硫、氯化氢、氯和一氧化氮被分子筛吸附的同时
会与吸附的水分子发生反应生成酸性物质，进而与
分子筛发生反应，从而降低分子筛的吸附效率，并
且缩短分子筛的使用寿命。
在分子筛吸附器的设计中，除选用性能好的吸

200 ℃，也有应用

2. 2
变压吸附空分设备
变压吸附空分系统应用于分离空气，主要制取
富氧、富氮气体。变压吸附的机理是: 利用沸石分
子筛对氮气的吸附亲和力高于对氧气的特性来吸附
氮气和制取氧气; 利用氧气在碳分子筛微孔系统狭
窄空隙中的扩散速度大于氮气的扩散速度，在远离
平衡条件下分离氧气和制取氮气。通常变压吸附法
在常温下进行，工艺上有加压吸附、常压解吸; 常
压吸附、真空解吸

2. 3
分子筛的改性
对分子筛吸附剂的改性目前主要集中在调节分
子筛的硅铝比以及阳离子改性
［9，10］。
沸石分子筛中的非骨架阳离子是对吸附起主要
作用的因素，调节分子筛硅铝比可以调节其吸附性
能，即调节沸石分子筛上的离子交换位的数量，即
可交换离子容量。这是因为沸石骨架中硅氧四面体
是电中性，铝氧四面体的数量决定着可交换离子的
容量。所以调节硅铝比可以改变阳离子交换的容
量，增加极性电位从而改善吸附剂的性能。这一应
用主要集中在深冷空分领域内的分子筛吸附剂改
进，以增加对极性分子二氧化碳等的吸附性能。

制动电机工作原理
鼠笼式异步电机和鼠笼式异步发电机都是由定
子和转子两大部分组成，结构基本相同。但鼠笼式
异步电机是将电能转化为机械能，鼠笼式异步发电
机则是将机械能转化为电能。
当制动电机接入电网后，定子三相绕组中通入
对称三相电流，会在电机的内部空间产生一个旋转
磁场，磁场的转速n1 为60f1 /p ( f1 为外加电源电
压的频率，p 为磁极对数，n1 为同步转速) 。电机
转子的转速也就是电机的转速，用n 表示。这个转
速由膨胀机通过传动装置带动电机转子产生，并且
让转子转速大于电机内旋转磁场的转速，这时就会
切割磁感线产生感应电流，制动电机就成为发电机
向电网输出电能。这个过程中，制动电机会先处于
电动状态，带动膨胀机快速达到电机额定转速。然
后膨胀机继续提速，电机转速超过磁场转速，切割
磁感线发电，制动电机工作在发电状态。

电机并网
在设备开启前，需要确定制动电机的转向是否
与膨胀机转向一致。只有电机处于运行状态时的旋
转方向与膨胀机运行时的旋转方向一致，才可以将
发电机与膨胀机连接在一起。
电机的额定转速为3000 r/min，膨胀机的设计
转速为15000 r/min，因此传动装置选择1 /5 的减
速箱。
膨胀机启动时，发电并网控制保护柜断路器处
于分闸状态，制动电机此时未与电网连接，电机内
部空间不存在旋转磁场。在膨胀机启动后，膨胀机
拖动制动电机加速旋转，尽可能接近同步转速时并
网。开封空分通过DCS 系统检测膨胀机转速。当
DCS 系统测速单元检测到膨胀机转速达到额定转速
14950 r/min 时，即制动电机侧转速达到2990 r/min
时，DCS 系统发出合闸命令至发电并网控制保护
柜，发电并网控制保护柜断路器合闸，制动电机并
网，此时制动电机定子绕组的电势和频率取决于电
网电源的电压和频率。
并网后，制动电机定子加入三相对称电流，在
电机内部空间产生旋转磁场。由于此时膨胀机还没
达到额定转速，电机转速n 小于n1，制动电机处
于电动机状态下工作。因此在并网后的短暂时间
内，制动电机吸收电网电能，拖动膨胀机加速旋
转。当膨胀机转速达到15000 r/min、电机转速达
到3000 r/min 时，转子导体与旋转磁场旋转速度和
方向一致，没有相对运动，导体中不产生感生电动
势，所以，这个状态下制动电机既不发电，也不从
电网吸收电能。
随着膨胀机进口喷嘴逐渐打开，进液量不断加
大，膨胀机转速随之升高，电机转速n 大于n1，
制动电机的转速高于旋转磁场的转速，切割磁力线
产生感生电动势、感生电流，将机械能转化为电
能，电机发电。同时产生一个与转向相反的力，阻
碍转子加速旋转，转速越高，这个阻碍的力就越
大，直到膨胀机的驱动转矩与制动电机产生的反向
制动的电磁转矩相平衡时，膨胀机转速将不再加速
上升而逐渐趋于稳定，此时膨胀机便在某一转速下
稳定运行，这个转速在15000 r/min 附近并且略大
于15000 r/min，并网过程结束。

3. 2
电机断网
膨胀机正常停车或故障停车时，工艺先发出停
运膨胀机信号。停运膨胀机，经过5 秒左右时间，
制动电机发出的电功率降至额定功率的20% 左右
时，DCS 系统发出停车命令至发电并网控制保护
柜，发电并网控制保护柜断路器分闸，制动电机脱
离电网，励磁消失，制动电机不再发电。
膨胀机带制动电机的作用除了发电，将膨胀机
的动能转化为电能送入电网以外，更主要的作用是
用来平衡膨胀机膨胀功的驱动力矩，使膨胀机稳定
运行。所以除非发生短路事故，发电并网控制保护
柜断路器可以直接跳闸，使发电机脱离电网，与此
同时发送一个停机信号给DCS 系统，停运膨胀机。
其他电气故障都要将信号发送至DCS 系统，实施
完停运膨胀机的相应操作，然后由DCS 系统发信
号给发电并网控制保护柜，发电并网控制保护柜断
路器分闸，制动电机与电网脱离。
3. 3
保护措施
当膨胀机由于某种原因导致并网时间过长或并
网失败，制动电机没有起到制动效果，膨胀机转速
快速升高直至发生飞车事故。或者由于某种原因进
液量过小，致使膨胀机转速降至额定转速以下，制
动电机转速低于3000 r/min，制动电机工作在电动
机状态。而此时如果仍处于并网状态，虽然对电机
并不能造成危害，但是由于电机拖动膨胀机高速旋
转，膨胀机进液口进液量不足，很可能使膨胀机损
坏，破坏工况，所以不允许长时间使用电机拖动膨
胀机运行。
开封空分选择使用逆功率继电器来感知交变电
流的方向。它是一种避免发电机变为电机运行状态
的一种保护继电器，当检测到反方向电流时会启动
逆功率继电器时，发出跳闸信号，同时将此信号发
送至DCS 系统，使膨胀机停运，以免制动电机断
网后膨胀机飞车。而且使用逆功率继电器检测的逆

随着国民经济的快速发展和低温技术的普及，
液氮、液氧、液氢、液氩和液化天然气等低温液体
的应用日益广泛，各行业对贮存和输送低温液体的
低温容器的需求不断增长，真空粉末绝热贮槽作为
低温容器的一大类，亦得到了前所未有的发展
［1］。
奥氏体不锈钢属于面心立方晶体，不存在脆性
转变温度，具有良好的低温性能，是一种应用广泛
的低温容器用钢。传统的低温容器设计方法是基于
弹性设计准则，只利用了材料的弹性承载能力。为
了充分发挥奥氏体不锈钢材料的塑性承载潜能，国
外提出了奥氏体不锈钢应变强化技术，并将这种技
·
·

术引入低温容器的设计、制造过程中。研究结果表
明，采用应变强化技术，使材料发生一部分塑性变
形，在大幅提高奥氏体不锈钢屈服强度的同时，对
材料的其他力学性能均不造成大的影响。采用材料
应变强化后新的屈服强度设计的容器，其壁厚通常
可以减小30%
50% ，有利于节省材料，降低制
造成本及运输中的能耗，经济效益显著。
［2，3］

钢材料( 即屈服强度与抗拉强度严格等同于标准
规定值的材料) ，为保证材料新的屈服强度σk 值，
圆筒环向最大应变会达到8. 7% ，而实际圆筒的最

5%
［4］。澳大利亚AS 1210

［5］，

［6］。此外，从目前国内采用应变
强化技术的经验来看，304 不锈钢制低温容器应变
强化后的应变一般不超过6% 。对于大型真空粉末

槽，应在保证应变强化充分的前提下，尽可能地减
小应变，从而使套装过程能顺利完成。

2. 3
强化试验过程的控制
在强化试验过程中，泵的选用以及升压速率的
控制至关重要。若泵的流速不够，对于如此大的容
器，会造成升压速率过慢甚至无法升压的情况; 若
泵的流速过大，则会使容器升压过快，不易控制其
最高强化压力，会使容器处于更高的强化压力下进
行应变强化，这对控制容器应变不利。
在强化试验中，杭氧低温容器有限公司合理选
择了泵的大小，保证在全速运行下，升压速率不超
过0. 1 MPa /min，同时通过高精度的压力变送器连
接电脑控制系统对压力进行控制，保证误差为
± 0. 03 MPa。在强化试验的各个阶段，用不同的加
载速率进行加载( 如图1 所示) ，在接近强化压力
时采用间断启泵的办法缓慢升压，从而保证在压力
达到强化压力时能及时停泵，防止压力过高加大容
器的变形量。比起传统的人工操作，采用控制系统
进行升压和保压更为准确和方便。

节，选择时应考虑物料性质、操作条件、塔设备性
能，以及塔设备的制造、安装、运转和维修等。
板式塔和填料塔是气、液传质最常用的两种设
备，板式塔的压力降比填料塔大。对于板式塔，大
塔效率比小塔高; 对于填料塔，塔径1500 mm 以
下效率较高。板式塔直径大时一般比填料塔造价
低，而直径小于800 mm 时填料塔造价更低。根据
这个项目甲烷精馏塔的液量和气量，塔径较小，适
合用填料塔。
填料又分规整填料和散堆填料。规整填料比散
堆填料比表面积大，压降小，效率更高，检修方
便。而散堆填料更适用于液量大，气、液相密度差
小的体系。由于这个项目甲烷精馏塔内的气、液相
密度差小，选用比表面积相对较小的散堆填料更有
利于气、液相的分离。但是散堆填料更适合现场安
装，塔设备在厂内安装后卧式运输时容易造成填料
不均匀，因此考虑选用规整填料。

寸的选择应兼顾塔设备成本、制造难易及运输等问

的测温、测振、测速仪表采取良好的密封措施，可
采用O 形圈、密封胶等密封方式。杭氧曾尝试用
环氧树脂对轴承温度线进行密封，试验时实现承压
1 MPa 短时间内不会泄漏，但担心环氧树脂长期工
作在油雾介质中难于保证密封持续性而最终放弃使
用。油箱及各仪表管线上的阀门密封性也要好，以
尽可能减小氢气泄漏到大气中的泄漏率。由于该氢
气膨胀机的工艺介质压力并不高，对于法兰密封面
来讲，采用聚四氟乙烯做密封垫片就可以了，当然
也可以采用聚四氟乙烯缠绕垫或石墨缠绕垫。
氢气膨胀机的流程设置及机组置换
氢气膨胀机轴封结构采用迷宫密封，为不使密
封气污染工艺管道内的介质，轴封密封气采用常温
工艺气( 氢气) 。为避免外界大气透过各处密封面
进入机组内造成爆炸危险，油箱内气体宜保持一定
的正压。而且一定的正压可使迷宫密封前后压比减
小，从而减少密封气体泄漏到油箱内的气量。油箱
气体排往火炬的管路上宜设置一个稳压阀，以稳定
油箱内气体压力。稳压阀压力设定值一般不宜高于
工艺管道内介质的压力，以免在密封气压力不够的
情况下油箱内气体倒流入工艺管道内而污染工艺介
质。为增强设备的可调节性及持续运行保障性，对
于一些调节阀宜设立带旁通阀的阀组。油箱内有一
定的气体压力时，油箱供油仍需保证轴承前后油压
压差在合适的范围内，以保证机组轴承正常工作。
机组在开机前必须进行气体置换。先用氮气置
换机组内部及工艺管道、仪表管道内空气，为保证
工艺管道内介质不被氮气污染，再用工艺气置换氮
气。机组流程及结构设计必须考虑不留较大置换死
角，对于一些气体不易流通的较大空腔，可通过设
置阀门将气体顺利排出，从而实现氮气能较彻底置
换空气，且有利于缩短置换所需时间。置换操作可
采用先充气，待置换腔内压力升高到一定程度后再
放气的方法。这种反复充放气的方法易于将各个角
落内的气体排出，缩短置换所需时间，提高置换彻
底性。在整个置换过程中，应保证置换腔内为正
压，否则外界空气即有进入的可能，造成置换失
效。氮气置换空气后，机组内部气体介质氧气含量
宜达到《氢气使用安全技术规程》
( GB 4962—
2008)
规定的氧气体积分数≤0. 5% 的要求。
氢气膨胀机的气密性检测
在制造过程中，宜对氢气膨胀机各部机的壳体
进行气密性检测，整套机组在组装完成后及开车前
均宜进行气密性检测，以保证整套机组的气密性，
减小工作过程中气体的泄漏率。用户现场一般用中
性发泡剂或肥皂水检测，达到无气泡产生，并用氢
设备泄漏检测仪对周围空气中的氢含量进行安全检
测，达到标准GB 4962—2008 规定的保证空气中最
高氢气含量不超过1% ( 体积分数)
的要求。对于
安装在室内的机组，建筑物宜采取强制通风措施。

调试过程中遇到的问题及处理措施
两台氢气膨胀机组，一用一备，在一台投入使
用的过程中，另一台处于停机备用状态。在现场调
试时发现处于备用状态的氢气膨胀机膨胀端蜗壳内
气压比工艺进气管内介质压力还要高，而油箱内气
压也比设计工况高。初步分析原因可能是留在封闭
蜗壳内的原有低温介质在停机后慢慢升温到环境温
度，在升温过程中气体内能增加而不断升压，造成
蜗壳内气压不断升高。由于密封气压力会自动跟踪
膨胀端蜗壳内压力，蜗壳内气压升高的同时密封气
压力也会升高，造成密封气压力过高，从而增加泄
漏入油箱的气体流量，导致油箱排火炬稳压阀排气
不畅而使油箱升压。这时宜及时开启相应阀门将气
体排出到火炬管网，以免产生安全隐患。作为安全
措施，每次停机后宜通过阀门将膨胀端蜗壳内残留
的低温介质排出到火炬。
在采取措施降低蜗壳及油箱内压力不久，蜗壳
内压力再次上升到较高的压力。怀疑有低温工艺气
通过膨胀机出口管道上的阀门泄漏入蜗壳内。在将
膨胀端出口管路上阀门打开后，蜗壳内气压升高现
象消失。
氢气是一种易泄漏物质，管道上阀门密封性要
好，根据标准GB 50177—2005 的12. 0. 3 条款要
求，氢气管道上阀门最好采用截止阀或球阀，以免
因气体泄漏产生不良影响。

原因分析
分子筛纯化系统改造更换时，按照用户前期要
求，2 只吸附器的基础利旧，吸附器外形尺寸与管

系统内管道整体更换，配管尽可能保持原有走向。
由于现场原有2 只吸附器的基础定位相距较近，改
造后的吸附器位置也无法变更，使配管空间受限，
对于原有配管的优化设计难以实现。设计人员在布
置吸附器空气出口管道时，综合考虑阀门操作方便

2. 2
高分子化合物的形成原因
经过对垢样中酸不溶物的生物降解和灼烧量分
析，判断不溶物为有机化合物。同时对循环使用的
缓蚀阻垢剂与杀菌灭藻剂在使用浓度下彼此相容性
进行试验，发现有黏性物质吸附在玻璃容器表面。
通过试验进一步确认不溶物为高分子化合物。导致
这种结果的原因是: 循环水系统同时使用的缓蚀阻
垢剂与杀菌灭藻剂配伍性不好，两种药剂相互作用
而产生的高分子缔结物黏附在换热器表面。

防范控制措施
防范控制水垢的方法有化学方法和物理方法。
国内已使用的物理方法有磁化处理及静电处理。目
前物理方法多应用在单台设备或小型循环水系统
中，其技术尚待掌握。对于大、中型循环冷却水系
统来说，采用化学方法处理较成熟、经济和有效。
因为空分系统最常见的水垢是碳酸钙，所以各种方
法的重点是解决碳酸钙沉积问题，以下是常用的化
学方法:
( 1)
除去部分成垢离子。碳酸钙水垢的成垢
离子是钙及碳酸根离子( 由碳酸氢根离子转化而
来) 。在补充水中钙离子及碱度较高时，可在预处
理工序中除掉部分硬度和碱度，即采用软化补充
水。软化水的成垢离子含量较低，有利于提高循环
水的浓度倍数。常用的软化方法有离子交换法及石
灰软化法两种。
·
·

( 2)
加酸或二氧化碳气，降低循环水的pH
值，稳定碳酸氢盐。
( 3)
在循环水中加入少量阻垢剂，破坏碳酸
钙等成垢盐类的结晶生长，防止水垢沉积。
( 4)
注意药剂的相容性。循环水系统运行时，
必须先搞清所投加的缓蚀阻垢剂和杀菌灭藻剂的性
能，特别要注意两种药剂间的相溶性，以避免因药
剂不相溶而影响药剂活性，甚至产生高分子化合物
及其他物质，并沉淀在填料及换热器表面。
( 5)
控制有机磷分解。在使用有机磷做缓蚀
阻垢剂时，尽量不采用含有活性氯的杀生剂，防止
有机磷水解造成磷酸根离子富集生成磷酸钙，并沉
积在换热器表面。
( 6)
做好循环水系统的日常运行管理，定时、
定期进行水质监测，及时做好监测报告; 做好垢样
采集分析工作，建立完善的水处理技术基础资料
库; 加强药剂的质量控制，发现异常应及时解决，
避免工艺介质对循环水质的影响。

厂选用的端子来选择与之配套的接线鼻子，且标明
型号、规格和品牌，不配套情况就很难避免。

防范经验
2. 1
设计阶段把好端子选型关，做好设备本质
安全
从宝钢制氧厂一、二、三期工程的实践来看，
一期电气、仪表设备因接线端子松动故障几乎为
零，而二、三期因接线端子松动引发跳车的故障相
对较多。一个非常重要的原因是一期设备从日本神
户制钢所成套引进，电气设备主要为东芝和日立公
司的产品，日本产品有一个显著的特点是动力和控
制回路全部选用圆形( O 形)
端子，接线非常可
靠。而二、三期设备多为欧美或国产设备，控制回
路多为欧式管型( 针形)
端子。这种端子相对东
芝或日立公司的圆形( O 形)
端子尺寸要小得多，
端子间距要小得多，且一旦螺丝松动，管型( 针
形)
鼻子便很容易从端子中滑出来，松动引发的
故障就高得多。为此，防范经验是在新建工程或改
造项目的设备采购订货技术协议上写明“所有端
子选用圆形( O 形)
端子” 的技术要求，如在设
计阶段把好端子选型关，做好端子本质安全，可大
大减小端子松动故障率。
2. 2
施工环节严把接线质量关，减少施工隐患
施工环节也很重要，一方面要关注压线钳是否
符合规范要求，在施工现场诸如压线钳规格与接线
鼻子不匹配，工具陈旧压不出丝扣、干脆用老虎钳
替代等现象不少见; 另一方面，要关注施工人员是
否会准确使用压线工具，并掌握压线要领，在施工
现场诸如不会根据接线鼻子准确选用压线工具或压
线部位不准确等情况也不少见，很容易留下虚接、
松动隐患; 再者，要关注所用的接线鼻子是否与端
子相匹配，接线鼻子必须与端子相匹配，特别是管
型( 针形)
鼻子，要注意其长短，选用过长鼻子
接线后，其裸露部位较易发生短路或触电危险，选
用过短鼻子易留下局部接触的隐患。以上三种施工
现象的防范办法只能靠加大检查和过程跟踪力度来
发现和纠正。
2. 3
设备受电前或检修完成前，务必逐根拉拔端
子接线，确保接线紧固、可靠
设备受电前或检修完成前的检查非常关键，业
主不要怕麻烦，务必亲自通过拉拔的方法来逐根排
查是否存在接线端子松动、接线虚接现象。一旦发
现松动，立刻整改。通过此方法可消除80% 以上
的接线端子松动或虚接隐患。
