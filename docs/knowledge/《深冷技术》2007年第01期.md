容易飞扬,
会侵入五官,

或窒息死亡。近年来, 全国空分行业因冷箱检修扒
砂而造成的人员伤亡、财产损失事故不断增加。为
提高扒砂安全系数,

珠光砂简介
珠光砂又名膨胀珍珠岩,
是由玻璃质熔岩经破
碎及高温燃烧而成的一种内部为蜂窝状结构、呈白
色圆珠松散颗粒状的材料;
其主要化学成分:
SiO2
68% ~ 76%,
Al2O3 11% ~ 16% , Na2O 4% ~ 5% ;
无毒、无味、吸音、不燃烧且不腐蚀。由于它具有
极低的导热性、可靠的化学稳定性、良好的充填流
动性和极低的堆积密度, 而被广泛应用于工业、农
业、环保、化工、饮食和建筑等各个行业, 尤其在
深冷工程、空分设备及一些综合工程中具有其他绝
热材料所不可替代的卓越性能。
冷箱保护措施
为了保证空分设备及冷箱的安全, 以下保护措
施可供参考。
( 1) 增设冷箱内压力指示仪,
控制冷箱内压力
在200~ 500Pa。
( 2) 在空分设备临时停车后,
冷箱内处于低温
状态,
外界湿空气容易进入珠光砂内。仪表气源若
为干燥氮气, 可采用仪表气源充当冷箱密封气( 正
常运行中禁止用氧含量高的空气) 。
( 3) 冷箱顶部需设有冷箱安全呼吸阀( 或呼吸
筒) 。
( 4) 建议将冷箱密封气分层通入直至顶部。
#
#

( 5) 可分别于冷箱底部、中部设置冷箱内气体
取样阀, 当冷箱内压力过高或冷箱塔皮结霜较重,
或者怀疑冷箱内设备泄漏时,
可分别从取样阀取
样,
通过分析气样氧含量来分析和判断故障。
( 6) 一般空分设备不设冷箱内珠光砂加温系
统,
建议增设加温系统。
( 7) 若发现冷箱内设备出现泄漏现象, 应及时

措施。
( 8) 如果条件允许, 可以对冷箱内部容器、关
键阀门或低温液体管道进行单独隔离保冷,
便于故
障的快速处理,
而不用对整个冷箱进行扒砂处理或
损坏冷箱, 即只进行局部的扒砂处理。

置24 小时,

低温行业珠光砂使用时的安全问题
低堆积密度的安全问题
低堆积密度珠光砂在保温效果上有一定的优
势,
由于单位体积质量轻, 热导率更低, 低温保冷
效果好,
可降低设计用量, 减小冷箱体积。低堆积
密度珠光砂可以降低因堆积高度所产生压强对冷箱
内部管道的影响。但由于质量小,
在扒装低堆积密
度珠光砂时, 更容易飘散, 损耗量会增加,
并引起
一般环境污染问题。此外由于珠光砂质量较小,
对
人体呼吸系统和视觉系统会形成伤害。长期直接接
触会对肺部造成慢性损伤。如果人体陷入珠光砂
堆, 会对鼻腔、肺部形成严重伤害,
抢救不及时,
将导致呼吸困难甚至窒息死亡。因而在生产、装卸
和扒填珠光砂时人员必须穿戴防护用品,
如口罩、
眼罩和用大块透明纱巾包头, 如有专用防护用具,
安全性更佳。
强吸水性的安全问题
由于珠光砂具有强吸水性,
珠光砂一旦受潮,
隔热效果降低,
空分设备冷损加大, 低温液体产品
产量降低, 严重的会影响到精馏塔的正常精馏,
造
成工况不稳定甚至无法正常生产。通常情况下,
容
易受潮的部位大多集中在冷箱顶部, 主要是因为冷
箱顶部设计有呼吸阀, 潮湿空气容易侵入。加上冷
箱顶部仓盖有时密封不严,
会造成潮气侵入。
正常情况下,
由于空分设备冷箱都进行充氮保
护, 潮气不容易进入冷箱。但如果空分设备跳车或
在停车情况下, 对于那些使用自身生产的氮气作为
密封气源的设备,
停车后充氮气源中断,
潮空气很
容易侵入冷箱。地处南方的空分设备更容易遇到此
类问题, 且侵入量更多。潮气一旦侵入,
顶部被冷
透的珠光砂会迅速使水蒸气凝结, 并导致顶部珠光
砂板结, 时间越长, 板结量越大, 在低温冷却作用
下, 板结硬度增加。如果遇空分设备大修扒砂, 会
难以正常流淌, 至扒砂后期仍会大量篷聚并积存于
顶部,
造成清理困难。在清除过程中, 破碎后的坚
硬的板结碎块往往会坠落并砸坏冷箱下部设备, 尤
其是较细的仪表管路等设备。因此,
空分设备处于
冷态时, 必须保证密封干燥气源不中断。最好配置
冷箱密封测压装置, 对之进行监控。一方面,
可监
测密封氮气供应情况,
另一方面, 可同时监控冷箱
内有无设备、管路泄漏。
小粒度的安全问题
通常情况下,
大多数人认为珠光砂只要保温效
果好,
粒度大小无关紧要。但事实上粒度过大, 缝
隙大,
冷箱充氮后更容易将冷量带出, 反而影响保
温隔热, 因而冷箱大小一旦固定, 保温效果势必降
低, 只能以增大冷箱体积弥补不足。粒度过小, 珠
光砂透气性差,
冷箱充氮的作用范围会减小。同
时, 粒度小更容易板结。此外粒度小的珠光砂, 单
体接触面积增加,
流动摩擦阻力大,
流动性差。空
分设备如进行冷箱内检修扒砂,
珠光砂不能均匀地
放出。如果冷箱中上部分珠光砂受潮, 珠光砂会篷
集在中上部, 不能正常流出, 往往下部珠光砂放完
掏空,
而上部珠光砂仍在淤积篷聚。一旦淤积珠光
砂松动, 会迅速坠落并从下部冷箱扒砂口涌出, 形
成喷砂, 对扒砂人员造成人身伤害;
同时下坠珠光
砂会对冷箱内管路形成强力冲击, 损坏设备。
此外,
最值得注意的是, 低粒度珠光砂由于透
气性差, 一旦空分设备发生低温液体泄漏, 低温液
体积存并滞留在珠光砂的时间更长,
空分设备排液
加热后,
汽化气体会不易散发,
造成冷箱压力升
高, 会形成顶部喷砂。若加热不彻底就开始扒砂,
外部空气会迅速与珠光砂中低温液体接触换热。珠
光砂中液体迅速汽化,
造成冷箱内压急剧升高, 会
使冷箱鼓胀变形,
发生严重喷砂事故, 甚至爆炸。

强吸附性能的安全问题
珠光砂的多孔结构决定了珠光砂具有很强的吸
附性能。对于水而言, 它具有强吸水性。对于低温
液体的吸附, 其能力也不差。笔者曾在一次喷砂事
故现场发现, 喷砂事故发生数小时后,
扒开散落在
地上的珠光砂,
仍不时噼啪作响。被吸附在珠光砂
内的低温液体,
遇外界空气后, 迅速在珠光砂颗粒
内部汽化膨胀,
将珠光砂颗粒胀裂, 并发出噼啪响
声。喷砂事故发生数小时后化验堆积珠光砂上方的
空气氧含量, 其氧含量仍在26%。
因而如果发生冷箱内低温液体泄漏, 针对珠光
砂的低温吸附特性,
一定要在冷箱设备加热回温后
方能扒砂。加热过程中必须打开顶部密封仓盖。不
能为抢检修进度,
一边加热,
一边扒砂。加热初
期,
要控制加热气量, 后期可加大加热气量。判别
加热是否彻底,
除观察冷箱各点温度是否回升并接
近常温外, 还需要在切断加热气源和停充冷箱基础
氮气后,
观察基础温度是否明显下降。如果事先明
确发现冷箱内设备存在泄漏,
且能初步判断出冷箱
内部泄漏点基本位置, 最好在加热结束后进行静置
判断。若漏点在冷箱下部, 建议在停充氮气并关小
加热气保压4 小时后( 防止珠光砂从漏点侵入管
路) , 再观察冷箱各点和基础温度保持情况。若漏
点在中上部, 建议在停充氮气和关小加热气8 小时
后,
再观察冷箱各点和基础温度保持情况,
若降温
不明显可基本认为加热彻底,
可安排扒砂。

恶性喷砂事故机理
发生恶性喷砂事故, 放散速度过快是一个重要
的因素。很多情况下,
空分设备扒砂检修时,
无论
施工方和业主都希望尽快放尽珠光砂,
刚开始放
砂, 就打开底部所有仓门或放散孔进行放砂,
由于
珠光砂自重较轻,
流动性较好,
放砂速度过快, 很
容易使外界空气从底部进入仓内,
犹如酒瓶倒酒,
需要不断回气一样。空气从底部涌入, 在常温状态
不会造成危险, 如果珠光砂仍为冷态, 危险性将大
大增强。空气进入后先收缩后膨胀,
会使出砂速度
更快,
如果珠光砂内有吸附的液体,
会在冷箱内形
成强对流换热, 最先接触空气的被吸附液体会快速
汽化, 冷箱内压急速上升,
加剧底部珠光砂的喷
涌。如果压力无法及时外泄, 下部冷箱在强大内压
作用下将会鼓胀变形。压力一旦从底部人孔泄出
后, 冷箱下部会形成空区,
使中上部珠光砂整体下
落, 顶部空气吸入, 形成强对流换热。在冷箱上部
吸入的空气先收缩后膨胀,
吸入的空气使上部珠光
砂内吸附的液体汽化,
汽化后上部冷箱压力升高,
一旦不能及时外泄, 就会使冷箱中上部鼓胀变形,
尤其是在冷箱顶部仓盖未完全打开情况下,
更危
险。这种强对流换热引起的作用过程, 即吸气) 对

空分设备喷砂事故预防措施
结合多起事故处理经验,
笔者认为需关注下述
几个方面, 才能更有效避免喷砂事故发生。
( 1) 选用珠光砂要严格, 除堆积密度、热导率
适宜外,
对粒度要提出要求,
尤其是冷箱较高情况
下,
粒度要适宜。同时建议使用憎水珠光砂。
( 2) 要采取措施防止珠光砂受潮, 雨天不能装
填珠光砂。
( 3) 空分设备冷箱处于冷态情况下,
不能停止
干燥的密封氮气源。
( 4) 空分设备加热不彻底,
不能扒砂。加热彻
底与否,
要在停止加热及密封气源后,
以设备各测
点温度和基础温度回升到常温与否为准。
( 5) 空分设备一旦出现冷箱内液体泄漏,
要及
时排空塔釜及容器内液体。
( 6) 空分设备出现低温液体泄漏, 要及时打开
冷箱顶部所有仓盖,
作为压力释放口,
防止憋压危
险。尽管这样可能会从顶部喷砂,
现场会引起范围
更广的环境问题, 但相对于设备及人身安全, 仍是
必要的安全措施。
( 7) 空分设备开始扒砂前, 要切断充氮气源,
以防加热不彻底,
使珠光砂内残余吸附液体在珠光
砂流动过程中汽化过快, 加快珠光砂喷涌速度。
( 8) 如果冷箱内存在管路泄漏, 为防止珠光砂
从漏点侵入, 对漏点位应采取隔离式管路切断, 并
通加热气源, 小气量低压保压。
( 9) 排放珠光砂时, 原则上要从翻板式放砂口
放砂。
( 10) 空分设备扒砂初期, 不可同时打开两个
或两个以上大型人孔放砂,
以防空气大量涌入, 在
加热不彻底情况下, 珠光砂内残余吸附低温液体导
致冷箱内压升高,
形成恶性喷砂事故。
( 11) 扒砂人员进入现场前要进行严格安全教
育、安全交底, 要进行逃生应急指导培训。
( 12) 扒砂人员必须穿戴好防护用品,
尤其对
口鼻及眼部进行必要的防护。
( 13) 扒砂前要事先清点入场人数,
交待逃生
路线,
逃生后应迅速在安全地点集合, 清点人数。
( 14) 扒砂时应先放砂, 后清砂,
放砂时清砂
人员应远离放砂孔, 放砂结束后, 清砂人员才能进
场清理珠光砂。
( 15) 一旦发生喷砂事故, 应及时启动抢险预
案, 烟雾散后应迅速赶到堆砂现场,
清点人数, 并
观察珠光砂堆有无异常翻动,
一旦发现人员被困,
要集中所有人力、物力进行抢救。
关于低温行业珠光砂使用安全的体会
( 1) 对于空分设备冷箱充氮压力的监测,
在冷
箱设计时应给予高度重视。往往大家只注重主冷
箱, 但随着内压缩流程的发展,
液氧、液氩返流和
板翅式换热器隔箱内压力监控也是很必要的。
( 2) 冷箱内基础温度的测点应多点平衡布置,
因为这是检测冷箱内微量低温液体泄漏的重要手
段, 所以一定要布置。
( 3) 重视空分设备放砂设计,
应向机械化、工
程化发展,
利用抽吸式机械从上到下逐层放砂, 这
样更安全、更环保。利用人工放砂,
不安全因素太
多, 易造成事故。
( 4) 人工翻板式放砂孔,
设计时应全部采用不
锈钢材质,
以防止时间过长锈死无法使用。放砂孔
应在三面冷箱均衡布置,
放砂孔孔径应大于
400mm, 否则珠光砂不易放出。¬

压充氧软管燃爆事故为使用单位违章操作所致。

因,
是使用了不合格酒精清洗新型充氧车内部禁油
器件及高压充氧软管。经长沙市某化工研究院化验
送交的酒精样品溶液中,
酒精含量为85% ,
水含
量为8%,
可燃物杂质含量为7% 。使用这样的溶

供应管路和增压管路之间设置的安全阀, 是为
了保证氧压机增压过程超压时将氧气自动回放到供
应管路而设置的。只有当增压管路与供应管路的压

在此次升压试漏工作中,
氧气从第一组氧气瓶

阀9 关闭。由于

造成增压管路升压

压差不足于使安全阀开启,
导致增压

经以上分析, 可以得出造成本次事故的主要原
因:
修理人员在大修和排除故障过程中, 更换充氧
车液气分离器体与盖之间的密封圈时,
错误地将本
应安装紫铜材质的垫片( 充氧车技术说明书已有规
定) 改装了橡胶材质的/ O0 形圈; 在装配时又未
将橡胶圈完全装正( 也可能是密封槽边缘有毛刺等
缺陷) , 造成压偏( 或压断) 。在修理厂对充氧车进
行密封试验检查时,
增压管路压力增压至15MPa,
工作正常。而经修理工现场修理后进行密封试验
时, 当管路系统氧气压力严重超压时, 橡胶圈开裂
漏气。高压、高温的氧气从橡胶圈裂纹处高速冲
出, 摩擦产生的火花或静电将橡胶圈和密封胶点
燃, 在纯氧的条件下急剧燃烧,
使液气分离器盖在
高温下瞬间熔化,
熔液被大量高压氧气吹出,
形成
缺口; 分离器上部挡水金属丝网等机件也被熔化,
在旋转气流作用下附着于容器内壁。燃烧着的气流
从熔化的缺口处高速喷出。

( 1) 增强法规意识, 严格按章维修。在装备使
用、维修中, 要严格按照技术规程实施,
不能任意
改变技术要求。高压、高温的氧气,
必须用紫铜垫
或聚四氟乙烯材质的密封圈, 不能随意更换。密封
圈在安装时, 也不必敷设或涂抹其他材料。
( 2) 改进安全阀的设置方式,
确保安全性能。
充氧车在供应管路与增压管路之间设置的安全阀,
其出气端应改成与大气连通。在安全阀起跳后, 可
将气体排出, 避免供应管路氧气压力较高时安全阀
不能开启现象的出现。¬
