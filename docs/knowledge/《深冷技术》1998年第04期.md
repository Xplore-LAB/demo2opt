（杭州制氧机集团有限公司透平压缩机厂，杭州市东新路90号，310004）

摘要 比较详细地介绍了淮南化工总厂  $28000\mathrm{m}^3 /\mathrm{h}$  空分设备配套增压透平膨胀机组的设计方法、特点以及出厂试车运行情况。试车结果各参数正常，法方专家表示满意。图4表2。

关键词：大型空气分离设备 增压透平膨胀机 设计 试车

# 1 概述

1996年10月，杭氧液空有限公司与安徽省淮南化工总厂签署了  $28000\mathrm{m}^3 /\mathrm{h}$  空分设备供货合同。随即，杭氧液空与杭氧就该项目中的两台增压透平膨胀机供货进行多次商谈。杭氧液空参照ATLAS COPCO的报价技术要求，制定了“低温增压透平膨胀机专项技术条件”。对膨胀机组的流程组织、性能保证、重要零部件制造、工厂试车及现场性能考核、膨胀机转子振动、测试仪表要求，以及接线箱的设计和防喘振系统等，都作了严格要求，许多内容要求达到国际通用标准，如美国API标准等。

该膨胀机组供货合同于1997年1月7日正式签署生效。合同规定增压空气过滤器需采用管道式锥形过滤器，气体冷却器采用不锈钢换热管，出厂前需进行氦检漏。因此，这两台增压透平膨胀机组是杭氧迄今技术性能要求最高、制造难度最大、膨胀气量（体积流量）最大的膨胀机产品。

# 3 设计方法及特点

针对杭氧液空提出的原始设计参数，我们进行了大量的性能计算和结构设计，包括膨胀机性能计算、增压机性能计算、叶轮强度计算、转子振动计算及轴向推力计算、转动喷嘴叶片设计计算、膨胀机、增压机蜗壳计算，及气、电、管路接线箱设计，经多方案比较，最终确定结构的几何参数。

# 3.1 膨胀机设计计算

为了准确计算膨胀机中各计算站的流体参数，考虑到介质温度在  $-105\sim -170^{\circ}C$  范围内低温下工作，必须对计算程序中每个计算站进行修正。为使杭氧的膨胀机性能有进一步的提高，使之接近或达到国际先进水平，早在1992年杭氧就引进美国北方研究工程公司（NREC）透平设计加工软件。该软件采用三元流设计，程序中的许多经验数据来源于试验结果或权威部门（如NASA报告等），具有较高的可靠性。膨胀机性能计算采用RIT-DAP程序，工作轮的三元流分析计算采用COMIG程序。其中工作轮造型采用SPECIG模块，该模块中必须输入三方面内容：

第一，子午面流道选择；第二，叶片厚度分布规律；第三，叶片角度分布规律。

流场分析采用BANIG模块，该模块采用流线曲率法经迭代计算确定流场分布。最终叶片定型是通过比较一系列设计的叶片表面气流速度分布，综合评价而定。COMIG计算过程为：

# 3.2 增压机设计

增压机设计是增压透平膨胀机设计的重要部分，目前杭氧在增压机设计中均采用三元叶轮，可用NREC压缩机设计软件PREDIG与COMIG或采用成熟的模型级设计。

由于我公司有性能优良的来自 DEMAG 公司  $\phi 300\mathrm{mm}$  的模型级叶轮参数，与本膨胀

机增压轮尺寸接近，因此最终确定采用模化法设计。

模型级叶轮参数如下：

叶轮直径  $\phi 300\mathrm{mm}$

叶片数 17片

进口叶片角  $\beta_{1} = 32^{\circ}$

出口叶片角  $\beta_{2} = 50^{\circ}$

出口叶片高度  $b_{2} = 12.5\mathrm{mm}$

增压轮直径  $\phi 264\mathrm{mm}$

等压线喘振线流量比  $\mathrm{Q_{\mathrm{surg}} / Q_{des} = 0.622}$

# 3.4 喷嘴叶片设计

喷嘴叶片是膨胀机的重要气动元件，以往对这种膨胀机是采用小叶片结构。由于该叶片结构复杂，叶片数较多（39片），摩擦损失比较大。针对本项目的要求，为提高膨胀机效率，我们采用较大叶片，叶片数减小，叶片结构大为简化，叶型更加合理。

# 3.5 防喘振设计

杭氧液空对增压机的防喘振系统要求甚严，必须同时采用压差控制与流量控制以消除增压机稳定运行区域以外运行的危险。压差控制采用回流阀，流量测量元件采用文丘里管。防喘振保护线的设置在理论计算的基础上，根据现场实际测定喘振点后，简化成线性方程，输入计算机进行控制（见图2）。

# 3.6 接线箱设计

杭氧液空要求膨胀机组自带气、电、管路接线箱，将所有密封气、压力油箱液面及电信号如转速、温度等由测量就地点接到接线箱中，且所有管子均采用不锈钢。膨胀机组现场安装时所有接线均由接线箱统一对外，这种设计可大大方便现场工程设计与安装，这种结构在国外厂家已普遍采用，值得推广。

关键词：大型空气分离设备 分馏塔 爆炸 恢复工程 设计 措施 考核

# 1 概况

抚顺乙烯化工有限公司于90年代初期从国外引进一套  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备，与环氧乙烷、乙二醇工程配套。1997年5月16日，该套  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备分馏塔发生重大爆炸事故（《深冷技术》1997年第4期已有介绍）。

开封空分集团有限公司得知消息后，立即派人赶赴抚顺乙烯化工有限公司。快速的反应，诚恳的态度、可行的方案，使抚顺乙烯化工有限公司领导感受到了开封空分集团有限公司的一片诚意。他们展开了紧张细致的市场调研，在同国内、外几家公司进行谈判接触后，经过技术方案、价格、交货期等多方面的综合比较，最终于1997年6月13日与开封空分集团有限公司签订了FON-6000/6000型分馏塔设备的订货合同。合同要求乙方对该项工程的设计、制造、安装、调试实行总承包，且工期不超过五个月。

# 2 原空分设备分馏塔的特点

（1）上分馏塔下段采用填料结构型式，阻力小，设计空气压缩机排气压力较低（实际值一般在  $0.5\mathrm{MPa}$  左右）。  
(2）产品纯度高。氧气纯度  $99.8\% \mathrm{O}_2$  ，氮气纯度  $< 10\times 10^{-6}\dot{\mathrm{O}}_2$  。  
(3）生产部分液体产品。

# 3 新分馏塔的设计、制造

新分馏塔的设计完全是针对性的，即流程组织是在原来  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备的基础上进行，一切要按原设计条件。原上塔下段采用填料结构型式，加工空气压力比较低，影响到主冷的传热温差和主冷的结构；其次氧气、氮气纯度要求较高，精馏塔性能至关重要；另外，要满足液体工况，高效率的膨胀机是关键。甲方大气条件较差，含碳氢化合物较高，安全防爆更要加以充分考虑。

针对以上问题，我们采取了以下设计措施：①在主冷凝蒸发器上面加辅冷凝蒸发器；②精馏塔采用新型环流塔板；③采用新型增压空气透平膨胀机；④安全防爆措施：

主冷板式单元采用新型防爆结构，减少碳氢化合物在其中的聚集；设计  $1\%$  液氧排放；增加液氧吸附器。

热虹吸蒸发器热源采用膨胀空气，一方面减少了膨胀空气进上塔的过热度，改善上塔精馏工况；另一方面也加大了液氧的吸附循环量，对主冷安全更加有利。

下面从保持原空分流程的特点，介绍一下开空公司配供的新分馏塔在设计上所采取的措施：

# 3.1 氧气纯度高

用户要求氧气纯度保证大于  $99.8\% \mathrm{O}_2$ ，这个纯度在国内空分设备中还不太多。要保证  $99.8\%$  以上纯度的前提下，满足产量的要求，就对上精馏塔提出更高的要求。在上精馏塔中，采用了我公司独有的新技术——高效环流结构筛板塔，可使精馏塔的精馏效率提高  $5\% \sim 8\%$ ，同时对各馏分的抽口位置进行计算机模拟优化设计，从设计上保证了能满足纯度口的要求。

# 3.2 氧提取率高

由于原空压机排气量已经给定，只有 $29500\mathrm{m}^3 /\mathrm{h}$  ，提取率已达到  $95\%$  以上。我们新配的空分塔，要达到这一要求，我们除了在设计上采用高效环流结构筛板塔以外，还采取了改善上塔精馏工况这一措施。具体做法是利用液氧对膨胀后的空气进行冷却，以减少膨胀空气进上塔的过热度。经过冷却的膨胀后空气温度降低到  $-177\%$  ，从而改善了上塔的精馏工况，使产品氧气的提取率得以提高。

# 3.3 空压机排压较低

由于原装置中上塔下段采用的是填料结构，致使其空压机排压只有  $510\mathrm{kPa}$  ，而传统的筛板结构上塔，其空压机排压一般都在 $550\mathrm{kPa}$  左右。要使  $510\mathrm{kPa}$  的排压满足工艺要求，我们在设计上采用了以下几个措施：其一是改进了冷凝蒸发器的设计，减少冷凝

蒸发器冷凝侧和蒸发侧的换热温差，由原传统的  $1.5\sim 1.6\mathrm{K}$  减少到现在的  $1.0\sim 1.1\mathrm{K}$  ，这一新技术的采用就使空压机排压降低约 $20\mathrm{kPa}$  ；其次是改进了主换热器各流路的管道设计，并且对冷箱内管道配置进行计算机辅助优化设计，以尽可能使管道布置更合理，减少流动阻力。

# 3.4 空气中杂质含量高

抚顺乙烯公司大气杂质如下： $\mathrm{CO}_{2} 400 \times 10^{-6}$ ，总烃  $12.2 \times 10^{-6}$ ；乙烯  $1.1 \mathrm{mg} / \mathrm{m}^{3}$ ；甲烷  $10 \times 10^{-6}$ ；乙炔  $2.4 \times 10^{-6}$ 。这种恶劣的大气条件给空分设备长期安全运转带来了很大的潜在危险。为了消除这个危险，我们在设计上采取了以下几点：其一是对分子筛吸附剂的数量进行调整，每台吸附器的填充量增加  $15\%$ ；其二是采用我公司独有的防爆式结构主冷，这种结构从根本上解决了碳氢化合物在主冷板式单元中的积聚，从而避免了由于碳氢化合物的积聚而引起的主冷爆炸；其三是采用了液氧吸附器，液氧通过该吸附器，液氧中部分碳氢化合物可以被吸附净化，使主冷液氧的碳氢化合物的含量减少，减少了爆炸的可能性；其四是增加了液氧排放，在不影响主冷工作及产品产量纯度的前提下，从主冷中排放一定比例的液氧至氧气产品中，也可以对主冷中碳氢化合物浓度得以稀释。以上四条措施的采用，保证了空分设备的安全运转。

# 3.5 采用计算机辅助设计

由于本装置的交货周期很短，从合同生效到装置运转只有5个月时间。设计周期要在50天内全部完成，主要得益于计算机辅助设计。我们利用计算机进行工艺流程的计算及组织优化、分馏塔内管道设计、应力分析等等，不仅从时间上满足了要求，而且也提高了设计的精确度和可靠性。

摘要 本文主要介绍了两种变压吸附新工艺，即从变换气中脱碳同时制取纯二氧化碳产品的工艺以及从富含  $\mathrm{H}_{2}$  和CO的混合气中同时制取纯氢和纯一氧化碳产品的工艺。图5表2。

关键词：变压吸附技术 纯  $\mathbf{H}_2$  纯CO同时制取变换气中脱碳 制纯  $\mathrm{CO}_{2}$  新工艺

# 1 前言

变压吸附技术（Pressure Swing Adsorption，简称PSA）是一项新兴的气体分离净化技术，自Skarstrom等人发明了PSA技术以来，在过去的几十年中，PSA技术以其能耗低、流程简单、装置自动化程度高、操作方便等优点赢得了人们不断的关注，取得了重大进展。在我国该项技术的研究起步于杭州制氧机研究所“人背马驼”的PSA制氧机，进入70年代，由化工部西南化工研究设计院率先对PSA制氢进行研究。1981年我国第一套PSA提氢工业装置在上海嘉定县化肥厂问世，在随后的十几年中我国PSA技术在化工、冶金、电子、医药、食品等行业得到了迅速的推广和应用。目前建成和在建的PSA工业装置有400余套，分别用于以下领域：

（1）从各种含氢混合气中分离提纯氢气，氢气纯度  $95\% \sim 99.999\%$  
(2）从合成氨变换气中脱除  $\mathrm{CO}_{2}$  ，净化气中  $\mathrm{CO}_{2}$  可脱除到  $0.1\% \sim 1.2\%$

(5）空气分离制富氧，氧含量为  $30\% \sim 93\%$  
(6）空气分离制富氮，氮含量为  $98\%$ $\sim 99.9\%$

# 2 变压吸附的基本原理

变压吸附的基本原理是利用吸附剂对不同气体的吸附容量随压力的不同而有差异的特性，在吸附剂选择吸附的条件下，加压吸附混合物中的杂质（或产品）组分，减压解吸这些杂质（或产品）组分而使吸附剂得到再生，以达到连续制取产品的目的。由于对产品的不同需求，可在吸附剂的气相或固相获得，从而构成了变压吸附的不同工艺步

骤。图1为变压吸附工艺原理图。

图1 变压吸附工艺原理图

气体吸附分离成功与否很大程度上依赖吸附剂的性能。变压吸附工艺中常用的吸附剂有沸石分子筛、活性炭、硅胶、活性氧化铝、碳分子筛等，对不同的分离对象选择不同的吸附剂。PSA工艺通常由吸附、降压（即顺放、逆放、冲洗、置换、抽空等）、升压等基本步骤组成。一般采用两塔或多塔流程来实现工艺过程的连续性。操作周期根据气体组成、压力、流量和产品要求来确定，一般为3~30分钟。工艺流程及自动控制调节系统是实现变压吸附最佳工艺的重要因素。变压吸附工艺操作压力根据产品种类、原料气组成、吸附剂性能、工艺特点及前后工序的情况确定，一般在  $0.05\sim 3.00\mathrm{MPa}$  范围内。

# 3.1 意义

变换气脱碳是合成氨原料气净化的一个重要工序，我国大多数小氮肥企业目前采用的是碳化工艺，即把合成的氨几乎全部返回碳化工序用于脱除变换气中的  $\mathrm{CO}_{2}$ ，同时得到产品碳酸氢铵。这些企业大都以碳铵为主要产品，经济效益受碳铵销路的制约。由于碳铵市场波动大，而碳铵生产成本又高，因此很多小氮肥企业都处于微利和亏损的边

缘。因工艺流程限制，这些合成氨厂不能有更多的商品液氨投放市场。PSA脱碳技术成功地用于合成氨厂，部分或全部替代小氨厂原有的碳化工艺，增加商品液氨产量，改变了这类工厂产品结构，现已投入运行的各种规模的PSA脱碳装置有60多套，为工厂带来了良好的经济效益。

（2）对空分设备进行模块化设计，供用户有多种选择。

（4）分馏塔需在厂内总装，且便于公路、铁路、水路的运输。

# 2 流程简介及技术指标

该空分设备采用分子筛增压、吸附制氩流程（见图1）。

空气经过过滤器（AF），进入无油螺杆压缩机（SC）压缩，经空气预冷机组（RU）预冷后进入自动切换的分子筛纯化器（MS）中吸附去除空气中的  $\mathrm{CO}_{2}$  、  $\mathrm{H}_2\mathrm{O}$  等杂质后，大部分进入主换热器（E1）经返流气换热后入下塔（C1)，另一部分空气进入气体轴承增压透平膨胀机的增压机（BB)，增压后进入主换热器，然后从主换热器中部抽出去膨胀机（ET)，膨胀产冷后进入上塔（C2）参加精馏。

入下塔的空气经分馏后，得到的富氧液空及液氮经过冷器（E2）过冷，通过V2、V3阀节流后入上塔进一步分馏。在上塔底部及顶部得到合格的氧气和氮气，氧气经主换热器复热后送出冷箱。上塔抽出的氮气、污氮气经过冷器、主换热器复热后氮气作为产品输出，污氮气去纯化器作再生气。

配氩塔产氩时，从上塔抽取氩馏分进入粗氩塔（C3），在粗氩塔顶部得到合格的粗氩，进入两组切换使用的氩吸附器，吸附去除少量的氮和氧，即可得到合格的精氩气。

该空分设备可生产适量的高纯液氮、液氧和高纯压力氮。

# 4 膨胀空气入上塔对上塔的影响

在设计过程中，我们将小型空分设备膨胀空气对上塔的影响，及小型空分设备带氩后冷损的问题，作为设计重点来解决。后一问题在下一点中将阐明。

在大型空分设备中，若是只生产气体或带少量液体产品，根据拉赫曼原理，经膨胀后的空气，可以利用上塔的精馏潜力，全部进入上塔参加精馏。这部分空气一般占空气总量的  $20\% \sim 25\%$  ，这不仅提高了氧的提取率，对氩馏分的影响也不大。

但在小型低压空分流程中，单位加工空气所需冷量要大的多，尤其是在带氩的情况下，这样所需膨胀量相应就要增大，这部分膨胀空气量超出了上塔所能利用的精馏潜力。为保证上塔的精馏工况和氩馏分的稳定，这势必要将部分膨胀空气旁通入污氮通道，造成氧的提取率下降，单位制氧能耗上升，这就迫使我们在设计中解决这个问题，最大限度的利用该部分膨胀空气。我们采取了以下措施：

（1）开发气体轴承增压透平膨胀机。因气体轴承耗功小，在小气量膨胀的情况下，比油轴承回收的膨胀功要大的多，增压明显。这就大大提高了单位膨胀空气的焓降，增加了产冷量，相对减少了膨胀量。  
（2）严格保证冷箱内低温管道与箱壁的间距，减少跑冷损失。

（3）对主换热器的设计提出了较高的要求，保证复热不足冷损减小到最小程度。  
（4）通过电算软件，对膨胀空气进上塔的多种方案进行对比，确定一个最佳值。

通过以上措施，空分设备在实际运行中，达到了较理想的状态，在不带氩的情况下，膨胀空气可全部进入上塔参加精馏。在带氩的情况下，进上塔的膨胀空气占参加精馏空气总量的  $17\% \sim 20\%$  ，与设计较为相符。超出该值，氩馏分中氮含量增加，粗氩塔工况不稳定，有较大的波动。

# 5 吸附制氩的实现

由于吸附制氩的特殊性，要实现吸附制氩，必须要有稳定的粗氩（ $\geqslant 98\% \mathrm{Ar}$ ），这我们在氩馏分抽出位置、结构以及粗氩塔的设计上已作考虑；另外还必须要有足够的冷量，在冷却吸附筒时以供调节，并保持吸附筒的低温环境（ $-180^{\circ} \mathrm{C}$ ）。

在这里将重点探讨吸附制氩冷损问题，它包含了氩塔的绝热冷损、冷却吸附器冷损及产品排放的冷损。后两项可根据分子筛量、产量等定量地计算出来，绝热冷损这一项是我们这次设计中需解决的问题，这直接关系到氩塔能否正常运行。

在大型空分设备中，采用加氢除氧制氩，绝热冷损的计算方法，是在不带氩的基础上增加一定的系数（一般为  $5\%$  ）来确定，在无氢制氩时，该系数还要大。

在小型空分设备中，单位空气的绝热冷损要大得多，这套  $350\mathrm{m}^3 /\mathrm{h}$  空分设备所配氩塔，有单独的冷箱，体积与主冷箱相近。我们在氩塔绝热冷损分配上，经过多种方法的计算比较确定，该部分绝热冷损占空分绝热冷损的  $40\%$  ，氩塔所需的冷量占空分总冷损的  $50\%$  左右。这就要求透平膨胀机有可靠的调节手段，保证在带氩时提供足够的冷量。在以后空分设备的运行中得到了证明，这样的冷损分配与实际相符，保证了氩系统的稳定运行。

现以成都  $350\mathrm{m}^3 /\mathrm{h}$  （带氩）空分设备为例，在冷却吸附器的过程中，提高膨胀机的转速，加大膨胀气量，增加产冷量。膨胀气量进塔控制在参加精馏空气总量的约  $18\%$  ，对氩馏分影响较小，粗氩纯度仍保证在  $99\%$

Ar以上，产量  $\sim 8\mathrm{m}^3 /\mathrm{h}$  。一组吸附筒完成冷却，送入粗氩，经吸附后得到的精氩气含氧 $\leqslant 1\times 10^{-6}$  ，该组吸附筒经过60多小时工作，含氧仍  $<  1\times 10^{-6}$  。同时另一组吸附筒外筒温度已冷却到-  $180\%$  ，并有一定的积液，这就有条件切换，保证氩的连续生产。

在制取氩气的同时，氧气、氮气质量都有提高，主塔工况也很稳定。氧气产量在 $350\mathrm{m}^3 /\mathrm{h}$  左右，纯度  $99.6\% \sim 99.7\% \mathrm{O}_2$  ，氮气超出了设计指标。若用户自备的空压机排气量达到设计要求（设计  $2250\mathrm{m}^3 /\mathrm{h}$  ，实际 $2050\mathrm{m}^3 /\mathrm{h})$  ，氧气产量将会大大提高。膨胀机是在满负荷的  $85\%$  左右运行，这给空分运行留有了一定的冷量调节余量，对吸附制氙需要的冷量调节是很有必要的。

# 1 高沸点组分（甲烷、氮、氩）在分馏塔中的分布规律

要从空分设备中采用低温精馏方法制取纯度为  $99.995\% \mathrm{O}_2$  以上高纯度液氧（杂质总含量为  $50\mathrm{ppm}$ ）（ppm即  $10^{-6}$ ，下同），主要须解决两个问题：一是把氩和氮从氧中分离；二是防止氮、氙和碳氢化合物（主要是甲烷）等杂质在液氧中浓缩。而后者是制取

高纯度液氧产品的难点和关键之处。

液氧中的甲烷、氮、氩主要随加工空气带入。大气中含有  $1\sim 2\mathrm{ppm}$  甲烷、1ppm 氮、 $0.08\mathrm{ppm}$  氙。空冷塔冷却水中的部分甲烷（地下水甲烷含量较高）也随加工空气一起带入分馏塔。而分子筛吸附器中的  $13\mathrm{X}$  分子筛对其吸附清除效果不大。甲烷、氮、氙属高沸点组分，在常压下，甲烷的沸点为 $111.7\mathrm{K}$  、氮为  $119.79\mathrm{K}$  、氙为  $165.02\mathrm{K}$  ，它们都能强烈溶解于液氧中。这些组分随空气进入分馏塔后首先溶解于下塔液空中，并随液空进入到上塔，最终聚集在主冷液氧中。甲烷的浓度浓缩至30倍左右，氮、氙浓缩至40倍左右。因此，正常工况下主冷液氧中甲烷浓度可达  $50\sim 100\mathrm{ppm}$  ，而氮为 $\sim 40\mathrm{ppm}$  ，氙为  $\sim 3.2\mathrm{ppm}$  。

# 2.1 双塔制取高纯度液氧方法

图1 双塔制取高纯液氧工艺流程

1-下塔 2-主冷凝蒸发器 3-上塔 4-氧塔I 5-氧塔Ⅱ 6-氧液化器

从上塔低部抽取含  $99.6\% \mathrm{O}_2$  、  $0.04\% \mathrm{Ar}$  的气氧到氧塔I下部，用液氮来冷却气氧，用下塔空气来蒸发液氧。从氧塔I顶部取出气氧导入氧塔Ⅱ，氧塔I底部液氧回流至主冷。氧塔I作用是脱除气氧中的甲烷、氮、氙等高沸点组分。经过分离，气氧中的甲烷由最初的  $30\sim 50\mathrm{ppm}$  降至  $1\sim 2\mathrm{ppm}$  ，而氮、

氙更低。脱除高沸点组分后的气氧在氧塔Ⅱ中再进行氧氩分离，在塔底得到  $99.995\% \mathrm{O}_2$  的高纯气氧，再经过液化器液化成液氧，从底部取出后送至液氧贮槽。

由于双塔流程比较复杂，需两只氧塔和一只液化器，相应配置的阀门、管路、控制系统也很多，给操作带来不便，这就要求我们研制出更加简易的流程。

# 2.2 提氩空分用单塔制取高纯度液氧的新设计

从上塔高沸点组分分布规律来看，制取高纯液氧的原料应该从离主冷远些而氧纯度又不能太低的位置抽取较为合适。提氩空分为单塔流程的实现创造了条件。我们研究了国外一些先进流程，研制出用单塔制取高纯度液氧的新流程，如图2（设备正在安装中）。

图2 单塔制取高纯液氧工艺流程  
![](images/ce2240e149310cb230f649b26b5a62639218b05f7defee1d1881fdc470d70c96.jpg)  
1—下塔 2—主冷凝蒸发器 3—上塔 4—粗氩塔 I 段 5—高纯氧塔

由于上塔氩馏分中甲烷、氮、氩含量极少，它在粗氩回流液中含量也很少。在粗氩塔下部设置6块洗涤塔板后，可以把氩馏分中少量甲烷、氮、氩洗涤下来，随液氧回流到上塔，而从粗氩塔下部第6块塔板上抽取富氧液体来制取高纯液氧，其甲烷、氮、氩含量更少。

正是由于从粗氩塔抽取液氧而不是用上塔底部甲烷、氮、氙含量为几十个ppm的气氧来制取高纯液氧，可以省去高沸点组分分离塔，直接用一只氧一氩氮分离塔从氧中除去氩、氮。

在高纯氧塔设计中，产品液氧在塔的下部第3块塔板处抽取，这个作用如同粗氩塔洗涤塔板，可以对不希望有的微量甲烷、氮、氩起阻挡作用。塔底相当于5倍的液体量回流至主冷，这样可以防止甲烷、氮、氩在高纯氧塔底部液氧中缓慢积聚。由于直接

抽取液体，不设液化器，使流程进一步简化。

在高纯氧塔设计中还应注意塔的高度和安装位置。粗氩塔下部的富氧液体流到高纯氧塔顶部以及高纯氧塔底部的液氧回流至主冷都是靠液体位差来实现的。因此，在设计时，高纯氧塔顶部与粗氩塔底部、高纯氧塔底部与主冷之间要有一定的高度差。在保证氧与氩、氮分离所需塔板前提下，高纯氧塔高度也应严格控制，使得这一流程能顺利实现。

# 3 两种流程比较

可以看出，在提氩空分设备中采用单塔制取高纯液氧比用双塔流程优势明显。设备减少，流程大大简化，操作趋于简单，启动时间也缩短。相应的投资节省，深受用户青睐。如果操作得当，高纯氧塔底部液氧中甲烷、氮、氩不会超量，这就意味着液氧视实际情况可以少回主冷，产品液氧可以多抽取一些。这给增产提供了非常有利条件。

# 大型空气分离系统：低温法、PSA法及薄膜法

# 1 序言

本文讨论了空气分离，特别是大型空气分离系统，包括低温法、PSA法及薄膜法。由于每种过程都有其不同的个性特点，因此，这三种方法可以达到的最大容量是各不相同的。例如，对低温装置来说，已经建成的单台制氧装置的最大容量为  $3000\mathrm{t / d}$  （吨/天），而最大的PSA装置为  $300\mathrm{t / d}$ ，薄膜装置为  $580\mathrm{t / d}$  氮。因为最大的低温装置主要是用来供应气体产品的，因此，装置所生产的液体量还不包括在内。

这些方法的应用受到经济因素、使用过程所要求的纯度和压力的影响和限制。要求的生产量很大，直至  $3000\sim 4000\mathrm{t / d}$  氧这样的装置，毫无疑问只能使用低温装置。当产量低于  $200\mathrm{t / d}$ ，要求的氧气纯度只是中等纯度，则PSA法是适合的。当要求纯度在 $97\% \sim 99\%$  ，产量在  $600\mathrm{t / d}$  以下的氮气时，可以考虑采用薄膜法。

这三种方法中的每一种都能生产氮气，因为氮气在一些市场具有重要性。但要同时生产氧、氮和氩，就只能使用低温装置。

# 2 市场

氧的应用很多，如煤的气化，不带发电装置的煤气化如合成气的生产，带发电装置的煤气化如集成气化联合循环发电法，即所谓IGCC过程。另外如天然气重整生产甲醇，FCCU过程中催化剂的再生，由乙烯氧化生产乙烯的氧化物，用氧气提高硫的回收率（SURE过程）等等。

氮气可以单独生产，也可以作为空分装

置的副产品。由于氮气的惰性，它可以用于烃的加工处理过程（惰性保护），也可以作为原料气与氢气在催化剂的作用下合成生产氨。

只生产氮气的空分装置，称为氮气发生器（制氮机），适用于不需要氧但对氮的需要量很大的场合。例如为提高原油的采收率（EOR），需要将氮在很高的压力下注入油田，以便能从油井中得到更多的原油（强化采油）。

氮的另外的用途如用于抑止化学过程或精制过程，以便从潜在的可燃材料中除去大气中的氧。

目前，对氧需求量最大的是钢铁企业和化工过程。今后，则可能是综合煤气化联合循环发电（IGCC）过程。

钢铁企业需要大量的高纯氧（ $99.5\%$  以上），因为钢的纯度及质量要求氧中的含氮量达到很低的程度。用于钢铁企业的大多数空分装置都同时生产氩，一部分产品氩用于钢的最终精练，另一部分产品氩则向市场出售或出口。为了生产氩，也要求氧气有高的纯度。一些有再循环物流的化工过程也要求高纯氧，因为低的惰性气体含量（主要是氩），可以使这股再循环量减少，相应的设备也可以小一些。

另一些过程也可以使用低纯度氧（如 $95\%$ ），其优点是这种氧的能耗较低。

![](images/c7f5b4aa8f152518efcbb8a92d82659162c6a9f1a97c05241be84c49ac804e07.jpg)  
图3 中间闪蒸详图

这种循环特别适用于为集成气化联合循环发电法即IGCC循环提供氧气。在这种情况下，空气压力要比通常的低压循环中的空气压力高，氧、氮两种产品都有所需要的压力。

高压塔的典型压力是15bar，中间闪蒸

塔的压力是10bar，低压塔的压力是5bar。在图3所示的过程中，闪蒸分离器（闪蒸塔）生产贫氧液体回流液和因蒸发以后得到的更富的富氧液体，这两种情况都可以改进低压塔的操作条件。这意味着循环的操作压力可以提高，而无需效率较低的设备，如再循环系统来保持其回收率。使用闪蒸分离器后，与常规的低压循环相比，2000t/d氧的制氧装置的能耗可以减少2MW。

液体泵可以替代外压缩使产品在用户需要的压力下输出。在这种循环中，液体（液氧或液氮）在进入主换热器以前被泵压缩到所需要的压力，在主换热器中与原料空气进行换热而蒸发（汽化）。这种循环的优点是不需要产品气压缩机，或者可以减少压缩机所需要的级数，因为对于氧压机来说，需要特别的设计、特殊的材料等等。

如图3中使用液体泵和闪蒸分离器（闪蒸塔）的装置与低压循环相比，将具有更好的经济性。当产品压力为  $3\sim 20\mathrm{bar}$  及大于40bar时，使用液体泵是特别有利的。

# 3 氮气发生器（制氮机）

氮气发生器（制氮机）被设计成只生产氮气。图4是带再循环气的单塔氮发生器的流程示意图。供入空气中的氧在废气中排出。与带副产品的吨氧装置相比，其流程图是相当简单的。空气在主换热器中被冷却，送入塔内，在塔内被分离成富氧液体及氮。塔顶的冷凝蒸发器被节流后的富氧液体冷却，使部分氮气冷凝作为塔的回流液。蒸发后的富氧液体在膨胀机中膨胀，膨胀机与一台冷压缩机连成一体，使部分蒸汽压缩后再回到塔底，以增大塔的再沸腾量和回流液，从而改善提高氮的提取率。单塔氮发生器的氮回收（提取）率约为入口空气量的  $40\%$  ，而带再循环的类似的发生器的氮回收率可以达到入口空气量的  $60\% \sim 65\%$  。对特大规模的装置则最好采用双塔系统，因为这种装

置的氮回收率可以超过  $70\%$  ，与产量较小的装置相比，它还能降低投资和功耗。

# 4 低温空气分离装置的最新发展

低温空气分离装置已经取得了一些发展，这些进展包括：净化的改进；规整填料用于精馏塔；计算机控制系统在负荷变化时在保证产品纯度的前提下，最大限度地改变产品的产量；取消CFC制冷装置；采用高效透平原料空压机及带增压压缩机的磁轴承透平膨胀机，以最大限度地回收能量等。

# 5 变压吸附

变压吸附系统可用于生产氧气和氮气两种产品，但是生产氧气和氮气的方法是不同的，要选择不同的吸附剂和吸附流程。变压吸附制氮的流程示意图见图6。空气经压缩、冷却后进入两个吸附床中的一个，吸附剂采用碳分子筛（CMS），氧先于氮被碳分子筛吸附，氮气则通过床层后被收集起来作为产品输出。

![](images/273af6ad898e2e285f355734691e5f7f6f10f8e629d4e498d791abc2c8fd3112.jpg)  
图6 PSA制氮装置示意图

经过相当短的时间后（一个周期只有几分钟），碳分子筛会被氧气所饱和，这时床层必须进行解吸，向大气放空可以除去原料空气中大部分被吸附的氧气、水及二氧化碳。与此同时，另一个吸附器则被充压，并和第一个吸附床那样生产出氮气产品。在第一个吸附床再生结束后，则再次投入生产氮气，如此不断循环。

用这种方法生产出的氮气纯度为  $97\% \sim 99\%$  以上。再附加一个专门的净化系统，

可以使氮气纯度达到  $99.995\%$  。

图7是一个真空变压吸附（VPSA）制氧装置的流程示意图，这是一种现在得到广泛应用的典型的变压吸附制氧装置。除了使用不同的吸附剂，即大部分使用分子筛，或者分子筛与活性氧化铝（铝胶）的双层床结构之外，其它均和上面所说的变压吸附（PSA）制氮装置相类似。在这儿，氮气优先被吸附床吸附。另外，解吸不是向大气放空，而是使用真空泵来除去在一个循环结束后被吸附剂吸附的大部分剩余氮气。

![](images/82313d94264bef6f9a974623c2025ee80bae8ee3a57ac3f192b4249421d9645d.jpg)  
图7VPSA制氧装置示意图

用PSA或VPSA法来生产氧气的主要优点是氧气的成本低，这是由于吸附剂，即分子筛的改进所实现的。早期的单位功耗为 $1.0\mathrm{kWh / m^3O_2}$ ；大约在1986年改用13X分子筛后，单位功耗为  $0.5\mathrm{kWh / m^3O_2}$ ；而现在使用了改进型的分子筛后，单位功耗可以达到 $0.3\mathrm{kWh / m^3O_2}$ 。由于在工艺设计上的不断改进，使变压吸附装置的投资也逐渐有所降低。

由PSA/VPSA得到的氧气纯度为  $90\% \sim 95\%$  ，最高可达  $95.5\%$  ，其余组分主要是氩。提高纯度要增加功耗，对PSA装置来说，氧气纯度从  $90\%$  提高至  $95\%$  ，功耗要增加  $5\%$  以上；对VPSA装置来说，功耗则要增加  $15\% \sim 20\%$  。

最大的VPSA制氧装置是  $300\mathrm{t / d}$  ，是由日本三菱公司生产的。现在看来，变压吸附制氧装置（PSA与VPSA）与只生产氧气单一产品的低温装置的分界线大概是  $200\mathrm{t / d}$  。当然，如果还需要其它的空气分离产品，则宁

可选用低温空分装置。因为变压吸附装置不可能同时生产氮气和氧气、氩气及液体产品。

PSA和VPSA变压吸附制氧装置可用于污水处理（造纸污水、纸浆处理等）及富氧燃烧过程（玻璃窑炉）等。

# 6 薄膜分离

在薄膜分离系统中，压缩空气先过滤除去颗粒状物质，而后通过薄膜。图8是这个系统的示意图。只有氮气缓慢地透过薄膜，而另外一些气体如水蒸气、二氧化碳和氧气则很快的穿透过薄膜，其穿透过薄膜的速度要比氮气快5倍左右，这个系数称为分离因子。

![](images/70163e0bca4220e21f6c497e8b4b241fee571e73732f7d74c283f1c182458b8a.jpg)  
图8 薄膜系统示意图

可以期望，薄膜材料的进一步改进将提供更高的分离因子，从而使薄膜法生产氮气的经济性得到进一步的改善。

薄膜被制造成管状，很多的薄膜管子组成一个压力壳。原料空气通常是进入管内，从壳侧回收透渗过来的气体。薄膜本身是一种半渗透性材料，支承在没有选择性的薄膜护套上。聚砜（Polysulphone）薄膜或者乙酸纤维（Acetate）薄膜对氧气的渗透性是氮气

的  $4\sim 5$  倍。单根管的直径可以达到  $0.2\mathrm{mm}$  或更小，这些管子密封在管板上组成管簇。

在容量大的情况下，重要的是这些管子必须有均匀的直径和均匀的、最大的渗透性，不均匀性会使装置的尺寸更大。因此，对大容量、较高纯度的生产装置来说，要非常小心的考虑这些因素。由于这些原因，薄膜装置的尺寸有一个上限，在更大的容量时，必须将几个装置并联起来，因此就增大了投资。原料空气中氮的回收率取决于薄膜的渗透性和所要求的产品纯度。例如，当氮气纯度为  $90\%$  时，回收率可达入口空气的 $70\%$  ，而当氮气纯度为  $99\%$  时，回收率将降低一半。

到现在为止，已见报道的最大的薄膜装置的生产能力是  $580\mathrm{t / d}$  氮。薄膜装置是紧凑的，也容易实现自动化，但是氮气的纯度要低一些，为  $90\% \sim 99\%$  。

今后，在进一步发展后，可以期望薄膜装置的尺寸将进一步增大。在过去的十年里，薄膜材料的成本已降低至约  $15\% \sim 25\%$  ，渗透性也得到了显著的提高，在这两个方面，可以期望今后还能得到进一步的改进和完善。

# 1 前言

洛阳轴承集团公司氮气站工程，1992年设计安装，1993年4月试车成功，运行四年来，氮气产量和纯度达到设计要求，工况稳定良好。

氮基保护热处理在我公司的应用，实现了轴承零件的无氧化光亮淬火（脱贫碳层深 $\leqslant 0.03\mathrm{mm}$ ），而且还可以有效地实现渗碳、碳氮共渗等多种工艺，提高了轴承的综合质量，效果明显，效益显著。下面将自行设计的氮气站作一介绍。

# 2 空分设备的选型

根据公司的现状和发展规划，我们主要考虑以下几个问题。

# 2.1 设计总容量

洛轴公司热处理厂共有九个车间，改造后多采用氮基气氛辊底炉、铸链炉、滚筒炉等热处理生产线，氮气耗量约为  $10\sim 40\mathrm{m}^3 /\mathrm{h}$  台，根据氮气供应平衡表，全厂氮气最大需求为  $840\mathrm{m}^3 /\mathrm{h}$  ，因而我们选择设计总容量为 $1000\mathrm{m}^3 /\mathrm{h}$  。

# 2.2 供气参数

热处理工艺要求氮气纯度  $\geqslant 99.995\%$  ，入炉压力  $\geqslant 8000\mathrm{Pa}$  ，露点  $-60^{\circ}\mathrm{C} \sim -50^{\circ}\mathrm{C}$  。考虑到管网损失等因素，我们确定供气参数

为：高纯氮气纯度  $\geqslant 99.9995\%$  ，压力  $\geqslant$ $0.03\mathrm{MPa}$  ，露点  $-65^{\circ}C$  。

# 2.3 设备选择

经综合比较，我们选择了两套苏州制氧机厂制造的KZON-160/500-Ⅱ型空分设备。该设备采用中压带膨胀机的流程，工艺线路及设备配置如下：空气过滤器  $\rightarrow$  空压机(5L-16/50型）  $\rightarrow$  预冷器（UF-960/45型）  $\rightarrow$  高效油吸附过滤器  $\rightarrow$  纯化器（HXK-1080/40-Ⅱ型）  $\rightarrow$  膨胀机（PLK-  $8.33\times 2 / 20 - 6$  型）  $\rightarrow$  分馏塔（FON-160/500型）  $\rightarrow$  产品。流程特点如下：

2.3.1 氮气产量  $500\mathrm{m}^3 /\mathrm{h}$  ，纯度  $99.9995\%$  压力  $0.03\mathrm{MPa}$  ，露点  $-65\%$  ，满足了洛轴公司工艺要求。  
2.3.2 氧气产量  $160\mathrm{m}^3 /\mathrm{h}$  ，纯度  $99.5\%$  ，压力  $0.03\mathrm{MPa}$  ，露点  $-65^{\circ}C$  。之所以选择“双高”设备，是考虑公司有一定的外购氧量，这样就可以节约外购费用，同时也可外销创造效益。再者运行费用分摊在两个产品上可降低成本。  
2.3.3 液氮产量  $25\mathrm{L} / \mathrm{h}$ ，为以后液氮贮运创造条件。  
2.3.4 空压机选用有油润滑空压机，辅以各级冷却器及油水分离器清除油水，并在预

冷器后设高效油吸附过滤器捕捉油雾，可避免油水使纯化器分子筛失效。

2.3.5 氟利昂预冷器的作用使空气温度降至  $5\%$  ，析出大量水分，减轻了纯化器的负荷，有利于分子筛对水分、二氧化碳、乙炔等碳氢化合物的共吸附，延长了纯化器的切换周期，从而保证了分馏塔的稳定运行。  
2.3.6 纯化器采用铝胶作为前级吸附干燥，除去较多的水分，再用13X条形分子筛作为后级深度干燥及共吸附，能更好地发挥分子筛纯化优势。同时，在管路上设计了分子筛再生通道，可以在纯化器内直接对新分子筛进行活化，无需借助烘箱，节省人力、物力，提高了效率。

2.3.7 采用透平膨胀机，制冷效率高，积液时间短，结构较活塞式膨胀机简单，操作维护方便。  
2.3.8 采用南京第三分析仪器厂生产的Z0-102A型氧化锆氧分析仪在线监测产品纯度及运行工况，性能稳定，灵敏度高，量程宽 $(0.1\times 10^{-6}\sim 100\% \mathrm{O}_2)$  ，使用非常方便。

# 4 设计中考虑的几个问题

# 4.1 备用问题

氮气是一种需要连续供给的产品气，因此在设计时必须考虑备用问题。我们主要从两方面考虑：一是设备能力的备用，二是产品氮气贮量的备用。

从设备来看，全厂氮气最大需求量为 $840\mathrm{m}^3 /\mathrm{h}$  ，而设计容量为  $1000\mathrm{m}^3 /\mathrm{h}$  ，有一定的余额，即便一套设备发生故障，另一套仍可供全厂绝大部分炉子使用。另外，考虑到空压机故障较多，单独准备了一台备用空压机，作为两套设备的备用。此外，公司原有一套  $145\mathrm{m}^3 /\mathrm{h}$  的碳分子筛变压吸附制氮机作为整个系统的备用。

产品氮气的贮存采用气态中压贮存。据调查，一般压缩机（包括无油润滑压缩机）都不适合压送高纯氮气，所以我们选用北京通用机器厂生产的1G3Z-20/B-I型膜压泵，保证在加压过程中不降低纯度，并设计了3个  $80\mathrm{m}^3$  、压力为  $1.3\mathrm{MPa}$  的贮气罐。这样，总贮气能力约为  $3000\mathrm{m}^3$  ，需要供氮时通过调压阀组向高纯管网送气，可供主要热处理炉使用1~2个班次。

随着企业的发展，外围用氮量的增加，我们考虑上液态高纯氮贮运，充分利用设备的产氮能力。

# 4.2 高纯氮气管网输送问题

高纯氮气的输送极易受到湿气、空气和尘埃颗粒等的污染，为保证管网输送后的使用要求，往往需要在使用点增设终端纯化器。为使高纯气体能够通过终端纯化器和高效过滤器，一般采用中压输送，典型的输送系统为：

分馏塔  $0.03\mathrm{MPa}\rightarrow$  稳压罐  $\rightarrow$  无油润滑压缩机  $\rightarrow 0.5\sim 0.8\mathrm{MPa}\rightarrow$  中压氮气纯化装置  $\rightarrow$  管网  $\rightarrow$  终端氮气纯化装置  $\rightarrow$  减压阀  $\rightarrow$ $0.03\mathrm{MPa}\rightarrow$  高效过滤器  $\rightarrow$  使用点（热处理炉）4.2.1 改进前的问题

如果采用这样的高纯氮气输送系统，我

们将遇到以下问题：

4.2.1.1 设备投资增加。由于采用无油润滑压缩机输送高纯氮气，引起高纯氮气纯度较多下降，必须经过网前及终端二级纯化，减压后方能使用。既浪费了空分设备本身的能力，又使投资增加。  
4.2.1.2 基建规模扩大。在面积受限的情况下更为突出。  
4.2.1.3 运行维护费用增加。特别是压缩机和纯化器电耗过大。  
4.2.1.4 使管理变得复杂化。

# 4.2.2 改进的措施

经综合考虑，我们决定采用高纯氮气直接输送使用，并采取以下措施保证输送质量：

4.2.2.1 适当加大氮气输送管径，减小沿程损失。  
4.2.2.2 尽量减少支管、盲管、U形弯管等，以避免吹扫时造成盲路和短路，不易彻底清除杂质。  
4.2.2.3 采用优质无缝钢管，先用长纤织物擦洗内壁，再用酸洗方法除去内壁粘附的各种氧化物、尘埃和油脂等杂质。  
4.2.2.4 管道连接采用焊接并尽量减少和合理选用附件，如阀门、接头、垫片、仪表等，避免任何轻微的泄漏，否则即使管网内氮气流维持正压，外部空气亦会反扩散渗入管内污染高纯氮气，造成纯度下降。  
4.2.2.5 使用前用高纯氮气长时间吹洗，排除管道内杂质气体和管内壁吸附的水汽，直至各使用点参数达标。  
4.2.2.6 在线监测各使用点氮气纯度、压力、露点，保证管网长期稳定高质运行。

高纯氮气低压输送方案，经我们实际使用4年来，情况良好。最远输送距离  $1100\mathrm{m}$  输送过程纯度下降  $5\sim 8\times 10^{-6}$  ，露点最多升高  $14^{\circ}C$  ，最远点压力仍有  $12000\mathrm{Pa}$  。这样，使用点参数达到设计要求。

# 4.3 空压机房降噪减振问题

空压机房的噪声主要由空气动力噪声、机械噪声和电机噪声三者迭加而成，约90 $\sim 130\mathrm{dB}$  。根据卫生部、国家劳动总局标准规定，每个工作日连续工作  $8\mathrm{h}$  ，允许的噪声值为85dB，最大值为90dB。为保障站内工人健康及周围环境不受污染，在设计上采取了以下措施减振降噪：

4.3.1 空压机与其它设备隔离，中间以隔音门连通，室内采用吸声墙壁，消除机房的混响声。吸声墙壁的具体作法是采用超细玻璃棉为吸声材料，容重  $15\mathrm{kg} / \mathrm{m}^3$  ，厚度 $0.05\mathrm{m}$  ，木龙骨框架，吸声材料填于其内，金属拉网罩面层。

4.3.2 空气压缩机基础作隔振沟，以阻止振动传输，产生噪声。隔振沟环空压机基础，深  $1.5\mathrm{m}$ ，宽  $0.1\mathrm{m}$ ，内填黄砂，上口沥青麻丝封死。

4.4.2 将 GC-24 型充氧台一字排列改为平行分布于 T 形防爆墙两侧（如图 1）。这样两组气瓶充装工位分开，切换阀门及各种仪表装于 T 形墙外侧，操作安全方便，充装与搬运气瓶互不交叉，符合充装安全作业程序。

4.4.3 充装夹具改为防错装接头，以防气体混装。

4.4.4 对厂方提供的承压  $14.7\mathrm{MPa}$  压力的  $\phi 25\times 3.5\mathrm{mm}$  紫铜管进行了强度校核计算，发现安全裕量不足，我们改为  $\phi 28\times 5\mathrm{mm}$  黄铜管，从而增加了强度，保证了安全。

4.5.1 纯化器后高压空气露点和  $\mathrm{CO}_{2}$  含量的测定，采用的仪器是上海唐山仪表厂生产

的DWS-ⅡB型微量水分析仪和南京分析仪器厂生产的DD-10型微量  $\mathrm{CO}_{2}$  分析仪。由此，我们可以了解纯化器的工作情况，分子筛吸附效果如何，切换周期是否合适等等，能有效地避免水分、  $\mathrm{CO}_{2}$  带入分馏塔内造成冻结堵塞，这对稳定生产、增产降耗有很好的指导作用。

4.5.2 氮、氧产品纯度的监测，采用南京第三分析仪器厂生产的Z0-102A型氧化锆氧分析仪。  
4.5.3 液氧、液空中油分和碳氢化合物含量的测定，采用上海第三分析仪器厂生产的722型光栅分光光度仪。

# 1 管式  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备改造的必要性

天津石化公司原有两套70年代生产的石头盘管蓄冷器切换式流程的  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备，自1975年投产运转以来，刚投产时氧氮产量、纯度能达到设计指标，但是后来由于塔内泄漏，冷损增加，为了补充冷损，被迫增大了膨胀量，至停车改造前，氧氮产量下降到只有设计指标的  $50\%$  多。

该套空分设备与目前先进的空分设备水平相比，存在许多问题：

# 1.1 成套工艺流程落后

原空分设备的流程为石头盘管蓄冷器的切换式流程，而目前先进的空分设备均采用分子筛吸附式流程。由于流程的落后性，氧氮产量比例一般为  $1:1.1$  ，而吸附式流程氧氮产量比例高达  $1:2\sim 2.5$  ，原氧氮产量比例值偏低，不能满足工艺生产的要求。

同时，切换式流程在操作上的复杂性，给用户生产上带来了许多麻烦和困难。

# 1.2 技术经济指标落后

该套空分设备在设计时，其提取率仅达到  $\rho_{0_2} = 70\% \sim 75\%$  ，实际运转后证明还难达到。而目前先进的空分设备设计的提取率高达  $\rho_{0_2} = 90\% \sim 98\%$  ，两者相差甚远。

另外，单位氧气电耗量大，根据实践统

计高达  $\sim 0.80\mathrm{kWh} / \mathrm{m}^3\mathrm{O}_2$  以上。

# 1.3 成套配套落后，分馏塔结构落后

原成套配套中透平膨胀机两台，流量分别为  $2200$  、  $2500\mathrm{m}^3 /\mathrm{h}$  各一台，其结构均为固定式，不可调。为了调节产冷匹配冷损的变化，只好被迫采用能量损失较大的机前节流的办法。

分馏塔上的低温阀门在低温状态下不灵活，操作困难。

另外，管子穿过冷箱板时，有一道密封面在冷箱内，当该道密封面发生泄漏时无法检修，轻则影响技术经济指标，重则不能出氧氮产品，必须停车扒珠光砂处理。

# 1.4 用户迫切要求扩容生产

由于氧氮产量的下降，不能满足生产的需要，另外原氮气纯度为  $100 \times 10^{-6} \mathrm{O}_2$  ，也不能满足工艺生产的要求。因此扩容生产，不仅要求氮气纯度为  $10 \times 10^{-6} \mathrm{O}_2$  ，而且氮气量要从  $1000 \mathrm{~m}^3 / \mathrm{h}$  提高到  $2000 \mathrm{~m}^3 / \mathrm{h}$  ，且已为燃眉之急。

# 变工况生产提高液氮产量的探索与实践

泉体民

（新疆独山子石化总厂乙烯厂空分车间，新疆独山子，833600)

摘要针对独山子乙烯厂用氮特点，空分设备进行内部挖潜，进行了变工况生产的探索与实践，减少了氧气、氮气放空，使液氮产量从原设计  $70\mathrm{m}^3 /\mathrm{h}$  （折气态）增加到  $320\mathrm{m}^3 /\mathrm{h}$  （折气态），提高了装置在事故状态下的应变能力。图1表2。

关键词：中型空气分离设备 变工况操作 液氮 产量 提高

新疆独山子石化总厂乙烯厂空分车间配有A、B两套杭氧厂产KDON-4500/9000型空分设备，另外还配有川空厂产  $300\mathrm{m}^3 /\mathrm{h}$  液氧、  $400\mathrm{m}^3 /\mathrm{h}$  液氮贮存汽化系统各一套。自1994年投产以来，围绕安全、稳定、长周期生产和节能降耗，提高经济效益，我们对B套空分设备进行了变工况生产、提高液氮产量的探索与实践，取得了一定的效果，为乙烯厂的生产提供了可靠的保证条件。现介绍如下。

# 1 问题的提出

空分设备自1994年10月到1997年10月已运行三年，在此期间乙烯厂高峰用氮 $8000\sim 9000\mathrm{m}^3 /\mathrm{h}$  ，正常  $6500\sim 8000\mathrm{m}^3 /\mathrm{h}$  。一方面，非高峰用氮期间，空分设备时常出现大量富余氮气放空且时间较长，造成能源浪费。另一方面，由于污闪晃电等因素，往往造成全厂包括空分设备的停工。此时全厂开车过程中的用氮，完全依赖大量液氮汽化外送以保全厂。而处理这类事故期间，直接受到空分设备生产液氮能力的限制。按设计，空分设备正常生产时，产液氮  $70\mathrm{m}^3 /\mathrm{h}$  （折气态)，允许事故频次为2次/年，满罐间隔

为155天。所以，装置在事故状态下的应变能力很差。另外，随着液氮在各领域的广泛运用，其需求量也在逐年递增。所以，在资金投入少或不投入的前提下，既满足氮气正常生产又多产液氮，成为当务之急。

# 2 变工况生产的可能性

由于我厂空分设备以生产气态产品为主，设计指标氧气产量为  $4500\mathrm{m}^3 /\mathrm{h}$  ，氮气产量为  $9000\mathrm{m}^3 /\mathrm{h}$  ，液氧  $50\mathrm{m}^3 /\mathrm{h}$  （折气态），液氮  $70\mathrm{m}^3 /\mathrm{h}$  （折气态）。如果在维持设计工况的基础上进一步增大液氮的取出，则十分困难。

那么，用于以生产气态产品为主的空分设备，是否有能力在原液氮产量的基础上进一步增加液氮产量呢？我们针对以往运行参数进行分析，如果减少氧气、氮气产量，将减少的气体作为膨胀空气以增加冷量转移到液氮产品上，重新建立物料平衡及冷量平衡。通过粗算，我们得知，如果改变原设计生产工况进行变工况生产，该空分设备应有  $320\mathrm{m}^3 /\mathrm{h}$  （折气态）左右的最大液氮产量可以获得。

空分设备的冷量主要是靠膨胀机制冷来

提供，根据空分设备的冷量平衡，我们知道：在装置正常运转中，膨胀机的制冷量 $Q_{e}$  与压缩机的等温节流效应  $\Delta H$  之和，应等于跑冷损失  $\delta_{2}$  、复热不足损失  $\delta_{3}$  及抽取液态产品的冷损失  $Q_{0}$  之和。即  $\Delta H + Q_{e} = \delta_{2} + \delta_{3} + Q_{0}$  。由于  $\Delta H$  受空分设备的空气压力和温度的限制，可调性很小。而膨胀机的制冷量  $Q_{e} = V_{t} \times h_{o} \times \eta$  ，在整个装置中占总制冷量的  $80\% \sim 90\%$  ，要提高液氮产量（即补偿抽取液态产品所带走的冷量损失  $Q_{0}$  ），只有通过变工况运行，加大膨胀机制冷量  $Q_{e}$  来实现。

# 3 提高液氮产量的要素分析及操作

# 3.1 膨胀量的影响及调整

膨胀空气量增加，制冷量增大，液体产率增加。我厂空分设备在设计工况下运行时，维持设备冷量平衡的关系已确定，进上塔参加精馏的膨胀空气最大量为  $3000\mathrm{m}^3 /\mathrm{h}$  也已受到限制。所以，当膨胀量在  $3000\mathrm{m}^3 /\mathrm{h}$  的基础上变化时，对上塔的精馏就会产生影响。

增大膨胀量的重要途径就是开另一台备用膨胀机，但随着膨胀量的不断增加，污氮含氧开始偏高，氧提取率下降，氧、氮气的纯度也随着下降，不利于产品气体的提取。如果采取变工况生产，在产品氧、氮质量控制不超标的前提下，按系统管网上实际需求外送氧、氮气，即可控制产品纯度，又可减少放空造成的损失。

应当指出，膨胀量过分增大，反而导致液体产率下降，精馏工况恶化。这是由于：进上塔的膨胀空气量过大，导致塔内汽液平衡失调，复热不足冷损增大。

# 3.2 膨胀前后温度对膨胀量的影响及控制

随着膨胀量的增加，液体产率先增加，在达到一最大值后又下降。过分增加最佳膨胀空气量，会使中抽温度下降，复热不足，冷损增大，因而导致液体产率降低。选取较

低的膨胀前温度和相应的最佳膨胀空气量，能获得较高的液体产率。

然而，膨胀前温度并非选取得越低越好，通过作图法确定膨胀机前最低温度示意图（图1）中可看出：当膨胀前空气温度低于  $-143^{\circ}C$  时，膨胀后气体温度下降到所在压力下的液化温度，膨胀后有液体产生，这样将破坏整个精馏工况。同时，膨胀机出口压力表、转速表指示抖动，膨胀机运转声音异常。严重时膨胀机内带液，会使膨胀机叶轮和导流器造成损坏。另一方面，膨胀空气是否液化取决于膨胀后的空气压力和温度，控制膨胀空气的温度高于膨胀后压力下的空气液化温度，即膨胀空气过热度，就可避免空气在膨胀机内液化。

根据上述分析，我们开备用膨胀机，将设计规定的  $2300\sim 3000\mathrm{m}^3 /\mathrm{h}$  入上塔膨胀空气量，逐步提高至  $3800\mathrm{m}^3 /\mathrm{h}$  左右，其余膨胀量旁通，膨胀机的调整以控制膨胀前温度 $\mathrm{T}_{1}$  不低于-  $143\%$  为前提，随着  $\mathbf{T}_1$  的逐渐降低控制好最佳膨胀空气量  $\mathrm{V_T}$  ，以获取最大的膨胀机制冷量，从而提高液体产率。

# 3.3 跑冷损失  $\delta_{2}$  的影响因素及控制

跑冷损失与环境条件、绝热条件以及管道阀门的密封性等因素有关。装置在运行

中，保冷箱内绝热材料受潮、阀门密封性不好造成漏气、漏液、主冷液氧的排放等均可造成  $\delta_{2}$  的增大，造成富余冷量的损失，不利于液体产品的提取。

空分设备在日常运行中，应时常检查更换受潮的绝热材料珠光砂，同时保冷箱内所通氮气应有微正压，避免珠光砂受潮。一般空分设备正常运转后由绝热材料阀门泄漏等致使  $\delta_{2}$  增大的因素已基本不变，只要控制主冷液氧的排放即可控制跑冷损失  $\delta_{2}$  的增大。

# 3.4 复热不足冷损失  $\delta_{3}$  的影响及控制

$\delta_{3}$  是随主换热器热端温差的增大而增大。空分设备在运行中，随着膨胀量的不断增大，膨胀前温度降低，热端温差增大，焓降减低，引起复热不足、冷损增大，污氮含氧增高，精馏分离困难，氧、氮纯度就会降低。

设计工况下运行热端温差一般为  $2^{\circ}\mathrm{C}$  但在变工况条件下，由于膨胀量的增大，氧、氮放空进一步减少，必然造成污氮含氧增高，热端温差增大，实际操作中只需控制好富余膨胀量的旁通及上塔压力，即可控制好热端温差。另外，热端温差增大至一定范围，氧、氮纯度才逐渐超出设计指标。通过分析和实际操作中的探索，热端温差只要控制在  $4^{\circ}\mathrm{C}$  以内，氧、氮质量就完全可以保证。控制好热端温差，复热不足冷损失  $\delta_{3}$  即可控制不再进一步增大。

# 3.5 气态产品取出量的影响及控制

气态产品氧、氮气如果仍按设计工况取出时，放空造成的冷损失增大。液态产品的提取就会受到很大的影响。

空分设备运行中，仅考虑单一产品（氧气或氮气）大幅度的减少，不仅对液体产品的提取有影响，而且气液比失调对整个精馏工况影响很大。变工况下，氧、氮气只有按比例同时减小到一定程度，用多余空气增加膨胀量，以增加冷量来补充液氮带走的部分

冷量损失。同时，又不影响变工况生产时的精馏工况。

需要提出的是，由于分子筛再生周期的切换，造成进塔气流及出塔氧、氮气的波动，对精馏工况不利，在实际操作调整中必须保留少部分氧、氮气的放空，以维持由于切换过程中造成的气流波动对精馏所产生的影响。

# 4 变工况操作方法

为使变工况生产调整既不影响外送氧、氮气体，又能最大限度的提取液态产品，整个变工况过程中，我们分三个阶段，使液氮产量大幅度提高。操作方法简述如下：

（1）调整纯化器切换阀，在分子筛再生一个周期后的切换时间不能过快，以最大限度地避免切换损失及切换时压力波动过大。否则，对变工况调整不利，形成氧、氮纯度波动。  
（2）开备用膨胀机，调整膨胀机转速，控制好膨胀机前温度不低于  $-143^{\circ}\mathrm{C}$ ，调整进上塔参加精馏的膨胀空气量为  $3700\sim 3800\mathrm{m}^3 /\mathrm{h}$ ，富余膨胀空气量由V5阀控制旁通量。  
（3）关闭液氧取出阀V7，减少液氧取出带走的冷量损失。  
（4）提高冷水机组负荷，控制低温水出口温度在  $3\sim 7\%$  为最佳，从而使空冷塔气体出口温度进一步降低为  $8\sim 13\%$  
（5）略关V51阀，略开液氮调节阀V3，全开液氮取出阀V8，加大液氮取出。  
(6）保持  $24000\sim 25000\mathrm{m}^3 /\mathrm{h}$  原有加工空气量不变，依次将氧气取出量从  $4500\mathrm{m}^3 /\mathrm{h}$  降为  $3300\sim 3600\mathrm{m}^3 /\mathrm{h}$  ，氮气取出量从 $9000\mathrm{m}^3 /\mathrm{h}$  降为  $6600\sim 7200\mathrm{m}^3 /\mathrm{h}$  ，提高上塔压力至  $0.056\sim 0.058\mathrm{MPa}$  。控制下塔阻力不低于  $11.3\mathrm{kPa}$  ，上塔阻力不低于  $17.5\mathrm{kPa}$

# 2 调整粗氩塔工况的理由

分析以上流程，我们认识到：粗氩塔作为上塔的一个辅塔，两者之间必然存在着相互影响、相互制约的关系。主塔良好的精馏工况是粗氩塔稳定运行的基础，反过来，对粗氩塔运行工况进行适当的调整，必将影响到上塔精馏工况。理由是：

（1）粗氩塔精馏物料氩馏分是从上塔中下部（第38块塔板）上升蒸汽中抽取的，随着粗氩塔运行工况的调整，氩馏分抽取量发生变化，上塔中下部的上升蒸汽量必然相应变化。如果氩馏分抽取量增加，上塔中下部上升蒸汽量相应减少。  
（2）粗氩塔底部得到的富氧氙馏分全部回流上塔。由于粗氩塔完成了氧一氩（氮）

的分离，液态富氧氙馏分中氧含量增加，其回流主塔，必然有利于氧的提取。从理论上讲，富氧氙馏分回流量越多，越有利于上塔对氧组分的精馏。

（3）粗氩塔冷源来自下塔底部的富氧液空，在其与粗氩换热的同时，自身由液态转变为气态，返回上塔中部。对上塔来讲，这部分富氧液空由下流液体变成上升蒸汽，减少了提馏段液空回流量，增加了精馏段上升蒸汽量。上升蒸汽量的增加，降低了精馏段回流比。这对于高纯氮的提取是不利的，但有利于提高提馏段回流液中氧组分含量。  
（4）从本质上讲，粗氩塔是在不增加上塔塔板阻力的前提下，完成了对上塔部分物料的精馏，相当于增加了上塔的精馏能力。

# 3 调整粗氩塔工况的步骤

基于以上分析，为了优化上塔精馏操作，提高氧气产量，我们作了如下调整：

（1）提高粗氩塔负荷，即适当提高粗氩产量，相应增加氩馏分抽取量和富氧氩馏分回流量，最大限度地发挥粗氩塔精馏能力。由于粗氩塔设计余量较大，粗氩产量可以由设计的  $140\mathrm{m}^3 /\mathrm{h}$  提高到  $180\mathrm{m}^3 /\mathrm{h}$  以上，相应的，氩馏分抽取量也由设计的  $4940\mathrm{m}^3 /\mathrm{h}$  提高到  $6350\mathrm{m}^3 /\mathrm{h}$  以上。  
（2）为了满足粗氩塔提量后的冷量需求，适当开大液空进粗氩塔冷凝器调节阀PDCV701，增加粗氩塔冷凝蒸发器中液空进量，保持粗氩塔冷量平衡，使液空汽化量由原来的  $7000\mathrm{m}^3 /\mathrm{h}$  上升到  $9000\mathrm{m}^3 /\mathrm{h}$  以上。  
（3）考虑到主塔精馏段由于液空汽化量的增加而导致回流比的下降，为了不影响氮气产量和纯度，适当开大HC1阀和HC2阀。在保证液氮纯度合格的前提下，尽可能多地向上塔导入回流液。

# 用富氧液空冻管更换重油管道阀门

钱启良

(秦皇岛市浮法玻璃二厂，河北省秦皇岛市先锋路，066000)

浮法玻璃生产中，锡槽内需用氮气作保护气。而空分设备底部的富氧液空被定期排掉，造成浪费。我们从医院用液氮冷冻血管受到启发，大胆提出用富氧液空做冷冻液，给不能停产的重油管线实行局部冷冻，即对重油管线上已坏的阀门前后两个局部位置浇冷冻液体，使近  $100\%$  的重油冷却凝固，再更换其中已坏的阀门。具体措施如下：

1. 重油与水不同，它升温、降温的过程比水慢，所以重油管线冷冻所用的时间要长。被浇注冷冻液的重油管道表面在停止浇注后，确实能长时间的出现白霜，才说明管内的油已经凝固，这时才可拆换已坏的阀门。在换阀门时，一定要连续浇注冷冻液于管道的表面，否则，还可能在没换好阀门时，就有重油流出。  
2. 冷冻是在管线的局部位置上进行的。这个位置很短，使重油冷冻后的缩、涨力可

沿管子长度方向消失，不会造成管子破裂。

3. 在正常供油情况下，我们选择油罐内油位较低时施行冷冻更换阀门的工作。这时，流速低，流量小。  
4. 我们在管道局部浇冷冻液的位置上加绕吸水、保水材料，使富氧液空冷冻液在其上有足够的停留与作用时间，节约冷冻液，提高冷冻效果。  
5. -192℃的富氧液空浇在约  $100\%$  的重油管线表面，对人体安全构成严重威胁，要采取严格的防冻、防烫措施。

# 5L-16/50型空压机的改进使用介绍

殷继承

（太原化工厂空分车间，山西省太原市义井，030021）

自70年代以来，太原化工厂一直使用5L-16/50型空压机作为  $150\mathrm{m}^3 /\mathrm{h}$  空分设备部机。进入90年代后，随着该机器的老化，我们先后进行了如下改进。

# 1 采用聚四氟乙烯活塞环，取消注油器

5L-15/50型空压机原配注油器装置的使用，是基于金属活塞环的特性。由于注油器中  $19^{\#}$  机油大量带入加工空气中，压缩机气道中机油在高温中不断炭化，使机体散热受阻，气量大幅下降。同时，清理积炭费时费工，雾状机油带入分子筛中，给净化工作带来困难。结合兄弟单位的经验，我们用聚四氟乙烯活塞环取代金属活塞环。值得一提的是，聚四氟乙烯活塞环必须选用填加铜粉的，否则，由于该环耐磨性差，气量将得不到保证。尤其是在机器启动阶段，压力达 $5.0\mathrm{MPa}$ ，温度近  $180\%$  ，曾发生过24小时内两套无铜粉聚四氟活塞环损坏的故障。而加入铜粉后，由于耐磨性大为提高，平均寿命达3.5个月。同时，我们取消了注油器，这样，空气中含油量降低后，既降低了积炭速度，又改善了净化空气质量。

# 2 取消空压机进口加压装置

为了增加加工空气量，我厂原在  $2^{\#}5\mathrm{L}-16/50$  型空压机入口管线中增设了增压装置。该装置设计气量达  $3000\mathrm{m}^3/\mathrm{h}$  以上，然而风压较小，气量也没有大的改善。同时，噪声很大，给操作人员判断运行设备故障带来不便。通过对太原市大气压力和压缩行程容积的测算，得出的结论是理论上无需增压，随之彻底清理空气过滤器和入口处管线，重新审定了一级阀门的弹簧K值，调

整一级活塞间隙总量，并由原  $11\mathrm{mm}$  减至8 $\sim 6.5\mathrm{mm}$  。取消增压装置后，目前气量完全满足工艺要求。

# 3 更换冷却器芯子

冷却器是空压机的重要组成部分。冷却效果的优劣，直接影响压缩机能耗。同时，温度过高，会使空气中的油、水得不到有效分离，还会使活塞环寿命变短，油的分解会使设备中积炭增加，水消耗猛增，净化负荷增大。为节约开支，1986年我们选用外厂生产的钢质一级冷却器芯子，与沈阳产原件相比，性能相差甚远。使用前者，一级温度为  $60\sim 80^{\circ}\mathrm{C}$  （操作规程要求  $\leqslant 40^{\circ}\mathrm{C}$  。而1992年2月更换为沈阳产紫铜冷却器芯子，一级温度便降至  $40^{\circ}\mathrm{C}$ ，用水量明显下降。

在检修二级冷却器过程中，发现其由于结构特殊，挡流板紧固部分松动。在气流的冲击下高频振动，多次使挡流板断裂，紫铜管频繁被擦伤，最终漏气。被损坏的散热管几乎全部为外侧边缘散热管。经研究发现，挡流板如果在使用前加固焊接，就可以杜绝此类故障的发生，而不是仅依赖支撑架上的铜管来固定挡流板。

# 4 设置自动排油、水装置

为将空压机的油、水及时排放，我们和仪表技术人员联合攻关，于1997年初，研制成功一套以一个1800秒和一个60秒时间继电器为核心的控制机构，以一个FDF-16型电磁阀为执行机构的空压机二级自动排油水装置。经半年的使用，效果很好。后来由于该阀的耐压有限，内部构造不适合于频繁开启、闭合，价格较高，而改用2ZD-25型电磁阀。

其优点是安装简单，只需前置一相应球阀，使用寿命长，价格低。继而推广至三级油水分离及分子筛前置油水分离器，效果令人满意。具体程序：每隔0～1800秒（可调）排油、水一次，而每次排油、水，间隔0～60秒（可调），同时多个电磁阀只需一套时间控制装置，继电器工作电压220V，可输出220V和380V两种电压，电流可达10A。这样，自动、定时、适度地排油、水，满足了工艺要求。

# 5 采用新励磁柜

空压机配套的TDK118-24/14型电机，励磁柜是重要附件，由于老化，其硅整流部分，故障频繁，检修任务繁重。我们在1995年、1997年先后更新，采用西安电力电子技术研究所制造的BL1A-300A/50型励磁柜，它采用集成块电路，使故障判断及检修简单化。

通过以上改进，空压机及空分设备保持了良好的运行状态，创造了较好的经济效益。

（1998年1月21日收稿）

# “14000”空分设备故障分析与处理

# 袁安邦

（湘潭钢铁公司动力厂空分车间，湖南省湘潭市，411101）

摘要 介绍了国产首套KDON-14000/14000型空分设备在投产、操作过程中，发生常温管道冻裂事故和管道堵塞故障的情况，后采取补焊、保温、改进操作等方法，解决了问题。图1。

# 1 故障情况

# 1.1 常温管道冻裂故障

1995年12月10日和1997年2月26日， $14000\mathrm{m}^3 /\mathrm{h}$  空分设备分别因检修DH90-1型空压机而停运。停运后，岗位人员发现空气进冷箱导气阀前，保温管道外部保温层结霜严重，并大面积带湿，进一步检查，发现该管已经冻裂。

# 1.2 管道堵塞故障

1.2.1 1997年2月28日， $14000\mathrm{m}^3/\mathrm{h}$  空分设备配套DH90-1型空压机突发事故检修；1997年3月28日重新启动后，随即对冷箱进行全面加温吹扫，发现由下塔至氩换热器E4压力氮（提供冷箱保护气和氩干燥器再生气）通道冰堵。  
1.2.2 1993年8月，在氩系统调试过程中，由于仪控系统未能正常投运，造成了精氩调试中一定的盲目性，导致了余气管路和混合气管道先后产生冰堵。  
1.2.3 氩纯化系统在操作过程中，经常发生水环式压缩机入口温度、压力下降，粗氩“吃”不进量，最终导致机前管线冰堵，寒冷季节更是如此。  
1.2.4 1994年5月， $14000\mathrm{m}^3/\mathrm{h}$  空分设备停机检修后系统进行全面加温吹扫。投粗氩塔

时发现粗氩塔冷凝器液空回流阀V704管线严重堵塞，导致冷态下该处对应温度T1703只有  $-110\%$  （正常时  $-183\%$  ），冷凝器换热效果下降，严重影响了粗氩塔的工况。

# 2 原因分析

# 2.1 常温管道冻裂故障

这两次停机均系短期停机，空压机停机时，进冷箱导气阀自动关闭，冷箱系统静置。此时，下塔液体液位有可能短时间超过空气进料口。由于下塔尚有一定余压，空气液封管起不到应有的作用，而氩换热器空气阀V711、V771又没有关，塔内压力将大量低温汽液经E4空气通道压至V101阀前空气管线，喷向常温碳钢管壁而产生应力，引起冻裂（见图1）。

图1 常温管道冻裂处示意图

# 2.2 管道堵塞故障

2.2.1 此次故障是因为“14000”空分设备长期停机，在排掉塔内低温液体后，冷箱处于封闭状态（上塔压力调节阀仍投自动）。但由于压力氮出冷箱阀V770遥控不好，仍有一定开度，相当于该压力氮管道与大气相通，昼夜气温变化导致湿气被吸入仍处冷态的氩换热器压力氮通道，引起冰堵。

2.2.2 调试过程中，精氩塔冷凝器上的液氮（调节阀开度偏大，塔内产生负压（此时压力显示仍为正压，说明显示有误），致使外界湿空气通过余气管路倒流入冷箱，余气通道发生冰堵。混合气管道冰堵，也有类似的原因（当混合气放空时）。另一个原因，就是当水环式压缩机停车时，纯化系统中的带压补充水倒入混合气管道（此时精氩塔已为常压），导致该管道冻结。

2.2.3 水环式压缩机是由纯化系统的回流阀 V2109 来控制粗氩量的，出冷箱的粗氩温度设计为  $11\%$  ，进入水环式压缩机前，要与混合气、氢气、补充水及回流气（含游离水）混合。由于粗氩为干燥气体，与水混合后，温度下降。若温度过低，则会导致水凝固成冰而堵塞入口管道。出冷箱的粗氩管道没有保温，寒冷季节时，粗氩温度更低，所以更易发生冰堵，导致氩纯化系统不能正常工作。

2.2.4 在加温吹扫前，液空回流阀V704及管道是畅通的。吹扫后反而堵塞，说明不太可能是冰堵（因系统已呈热态），那么分子筛粉末脏堵的可能性很大。进一步检查发现，纯化器的分子筛粉化严重，仪表气过滤器中存有大量分子筛粉末，而且在上塔液氧、下塔液空中亦发现有分子筛粉末，进一步证明了这一推论的正确性。

# 3 事后处理

冷箱导气阀前管道冻裂后，我们进行了补焊处理，并在操作上吸取教训，规定空压

机停车、导气阀关闭前，先关闭氩换热器空气通道V771、V711阀，并加以确认。停车时，全开液空、污液氮、液氮节流阀，将下塔液体尽量打至上塔，同时使下塔尽快卸压。同时我们也建议设计制造单位考虑在导气阀后，再引出进E4的正流空气管线。

空分设备长期停机且冷箱处于热态时，一定要封闭冷箱系统各阀并逐个加以确认。精馏系统尤其是氩系统投运操作阶段，要防止负压的产生。因为氩换热器的余气、混合气通道数很少，若有水分进入，极易产生冰堵，且解冻起来相当困难。空分设备停机，冷箱处于冷态时，则要防止精馏塔超压。余气、混合气通道冻堵后，一般是采取加温吹扫的方法加以解冻。

水环式压缩机机前管线冰堵，主要是因为粗氩管道没有保温，这是安装时的疏漏，后来管道保温后，我们还加装了伴热带，以确保机前温度不过低而产生冰堵。

对于水环式压缩机的补充水，我们改造成在水环机入口阀后加补充水，并且规定开机加补充水前，一定要先关闭水环式压缩机进出口阀，以确保补充水不倒流至氩换热器的混合气通道而产生冰堵。

水环式压缩机机前管道冰堵，可以采取停机静置粗氩放空的方法进行解冻。但也可以不停机处理，方法是打开压缩机热气旁通，和用热气回流加热机前管道。该方法，在实际操作中取得了良好效果。

V704及其管道脏堵，给粗氩塔的正常投运带来相当难度。若采取常规的吹扫方法，则很难吹通；而若对冷箱作扒砂处理，则生产受到严重影响。为此，我们采取改进操作，加大冷凝器液空工作液面至  $1800\mathrm{mm}$  （原设计为  $600\mathrm{mm}$  ），使该板式为全浸式操作，从而保证了粗氩塔的正常工况。

（1997年11月改稿）

# 一起带检验环氧气瓶爆炸事故分析

李国威 田农

（武汉市汉钢氧气厂，武汉市汉阳区龙灯堤224号，430050）

# 1 事故经过

1996年9月23日10：40分左右，充装台正在充装的30瓶氧气压力已达  $13.5\mathrm{MPa}$  一操作工到高压总阀处操作高压阀，另一操作工进充装间去关氧气瓶阀。此时，高压阀处及东组充装台的压力为  $13.8\mathrm{MPa}$ ，氧压机三级出口压力为  $14.0\mathrm{MPa}$  。正当班长关完一个瓶阀时，靠其右手第三个氧气瓶突然发生爆炸，巨大的气浪将充装间的两人掀倒在地。随后氧压机紧急停车并打开放空阀放空。事故造成轻伤两人，门窗玻璃震碎。

# 2 现场调查情况

该瓶的防护链条被扯断，附近的4、5个气瓶被震得东倒西歪。爆炸气瓶瓶体碎成三块，全飞向了充装间西边。瓶身二分之一以上部分为最大块，中下部一块次之，瓶底一块最小，它们分别距爆炸点  $8.1\mathrm{m}$  、  $11.5\mathrm{m}$  和  $19.3\mathrm{m}$  。在东边还找到了瓶阀和检验环。

根据对气瓶碎片检查发现：该瓶属上海制造（上01），为1978年7月出厂，编号863076，重量  $58\mathrm{kg}$  ，容积41.2L。从检验环上查明：检验单位编号  $\times \times \times$  （不便公开），检验有效期为1994年～1997年5月，可充装 $15\mathrm{MPa}$  压力。据肉眼观察：该瓶碎片外表符合“八不充装”规定，但内壁腐蚀严重，坑凹不平，多处壁厚超薄，无燃烧痕迹。

# 3 事故原因分析

此瓶外观符合“八不充装”的规定，充装在检验有效期规定时间内且在规定的压力范围内。在此瓶爆炸之前，我厂已安全充装161万多瓶氧气。为了找到爆炸原因，1996年9月25日我厂将爆炸气瓶的全部碎片送

到武钢锅炉压力容器检验站、武钢钢研所，进行了超声测厚宏观照相，扫描电镜微组织分析，电子探针和能谱仪锈层分析、化学成分分析、爆破压力计算。

结果表明：氧气瓶材质为  $40\mathrm{Mn}$ ，各项化学元素的成分均在范围之内；气瓶组织无明显异常，硫化物和氧化物正常，断口形态呈过载断裂；宏观检测气瓶内壁氧化腐蚀严重，蚀坑最大深度为  $2.8\mathrm{mm}$ ，开口直径  $3\sim 4\mathrm{mm}$ ；超声测厚表明：钢瓶剩余壁厚范围为  $3.0\sim 5.9\mathrm{mm}$ ，钢瓶最薄处用游标卡尺测出的厚为  $1.2\mathrm{mm}$ ，金相测厚为  $1.28\mathrm{mm}$ ；经爆破压力计算其爆破压力为  $10.7\mathrm{MPa}$ ，小于充装压力  $13.5\mathrm{MPa}$ 。故此我们认为爆炸的主要原因是钢瓶内壁腐蚀严重，导致钢瓶壁厚不均匀减薄，使钢瓶承载能力下降，不能满足气瓶的充装压力而引起爆炸。

# 4 事故的教训与措施

4.1 严格执行气瓶充装的安全操作规程。由于操作工按操作规程挂了防护链条以致操作工没有受到其它气瓶倒地或旋转而造成的伤害，否则后果不堪设想。  
4.2 坚持先关防爆墙外高压总阀然后再进充装间关瓶阀的规程。部分操作工为了抢时间，边关高压总阀边进去关瓶阀，是十分危险的，应予杜绝。  
4.3 对使用期接近20年的气瓶应加大力度，重点检查钢瓶重量，重量减轻的应作测厚、内壁腐蚀检验。  
4.4 提高我厂钢质无缝气瓶检验站气瓶的自检率，掌握在我厂充装气瓶的状况。

(1998年1月改稿)

# 制氧工问题解答（三则）

1. 问：我厂有一台  $50\mathrm{m}^3 /\mathrm{h}$  制氧机，纯化器（原为干燥器）已改用13X分子筛。近来发现纯化器堵，经多次再生，重新开车后只开三天纯化器又堵，进塔高压空气压力逐渐下降，车子开不下去了，我厂近邻(20m左右）有三个硫酸厂，请问是否是分子筛被毒化引起的？（杭州  $\times x$  制氧站问）

答：我认为问题提法不正确，应是热交换器被堵，故障原因在纯化器才对。既然纯化器是新购的，13X分子筛装进后没有多少时间就发现热交换器堵，问题逐渐严重以至开不下去。按理说，纯化器和分馏塔只开三天就不行，是不可能的。那么是否因为近邻有硫酸厂引起分子筛中毒呢？我的判断也是否定的。因酸气（ $\mathrm{SO}_2$ ）会被分子筛吸附，况且该厂也非新建。中毒最大可能是压缩机的润滑油，但也不是三五天短期形成的，这些方面都不可能。

经过进一步了解才发现该分子筛属转手杂牌货，颜色有点反黄，由此断定分子筛是伪劣的不合格品，还可能伴随有水分进入吸附器，故必须更换分子筛，同时检查压缩机末端冷却器有否渗漏（后证实，更换了分子筛之后，问题便解决了）。

记得南京南化公司化肥厂有两台6000 $\mathrm{m}^3 /\mathrm{h}$  制氧机，由于购买的是非空分专用分子筛，也出现过高压空气压力逐渐下降，后经德国专家发现，全部更换为上海UOP13X-APG分子筛，问题才得以解决。

至于末端冷却器泄漏，过去碰到过，因停车卸压后可能有冷却水倒流进冷却器管中，再开车时水随空气进入吸附器，故不可

忽视。

制氧机是以空气为原料的空气分离设备，不论大、中、小制氧流程，都少不了纯化器和分馏塔。空气家族中除氧、氮之外，还有水及二氧化碳等杂质。这两种组分量虽少，但危险甚大。两种组分进入分馏塔，就会以雪花和干冰形态把管道、阀门堵塞，制氧机就无法正常运转了。因此空气在进入分馏塔之前就得把妨碍分馏塔正常工作的水及二氧化碳彻底干净地消灭掉，所用的手段就是采用装有13X分子筛的纯化器。对制氧机而言，如果没有强有力的净化系统，纯化器中分子筛失灵，或性能不合格，制氧机就无法有序地连续运转。以往我们大多重视空压机、分馏塔的操作和管理，把纯化器当做辅机，不予重视，购用不合格的分子筛，就是一例。这种观点，必须引起氧气站全体操作人员和管理人员的严肃思考。

2. 问：我厂一台  $50\mathrm{m}^3 /\mathrm{h}$  制氧机，1976年产品，间断生产。最近氧液面从  $10\mathrm{cm}$  下降至  $4\mathrm{cm}$ ，换热器热端出“汗”，遂停车抓漏、洗塔，再开车情况仍无改变。请问是否冷量不足引起的？（温岭  $\times x$  制氧站问）

答：经过了解，该塔正常工况是  $\mathrm{P_1 / P_2 / }$ $\mathrm{P}_3$  为  $2.4 / 0.47 / 0.055\mathrm{MPa}$  ，  $\mathrm{T_l / T_2 / T_3}$  为-130/-157/-174℃；节-1/节-2/节-3为 $\frac{1.5}{12} /\frac{9}{12} /\frac{6}{12}$  周；液氧面  $5\sim 10\mathrm{cm}$  ，氧纯度 $99.2\%$  ，氧产量偏少。最近发现换热器热端出“汗”，经停车抓漏，找到氧气热端漏点。经焊补、检修，接着又洗塔再开车，其工况

$\mathrm{P_1 / P_2 / P_3}$  为  $5 / 0.55 / 0.06\mathrm{MPa}$  ，上塔压力还在上升，膨胀机温度基本正常  $[\mathrm{P}_{\text{进}} / \mathrm{P}_{\text{出}}$  为 $(2.8\sim 3)\mathrm{/0.6MPa}]$  ，氧纯度及氧产量都小，热交换器热端管子仍有出“汗”。

这只塔系间断生产，平时就已带病工作，所称“正常工况”，其实不正常，有以下几点可以看出：（1） $\mathrm{T}_{1} = -130^{\circ}\mathrm{C}$ ，热端温差必大；（2） $\mathrm{T}_{3} = -174^{\circ}\mathrm{C}$ ，说明冷端过冷了，温度也不对，所以  $\mathrm{P}_{1} = 2.4\mathrm{MPa}$  是个假像；（3）节-1阀为1.5/12周，关得小了些。因此，初步可以得出结论为，间断生产的启动操作不当。

由于冷端过冷，高压偏低，冷量不足，以致液氧面逐渐下降，所以车开不下去。

再看“停车抓漏洗塔”后的工况吧，我认为可能是加温中反而把设备搞坏了，为什么呢？（1） $\mathrm{P}_{1} = 5\mathrm{MPa}$ ，液面只能保持  $3 \sim 4\mathrm{cm}$ ；（2）膨胀机  $\mathrm{P}_{\text{进}} = 3\mathrm{MPa}$ ，制冷量还不足以保持冷量平衡；（3）上塔压力  $\mathrm{P}_{3} = 0.06\mathrm{MPa}$ ，还在继续上升中，这个局面是主冷漏的象征（后来在检查主冷时果然发现主冷下管板漏了）。

总之，这只塔的毛病是操作不当引起的，洗塔加温后引起主冷漏，须大修才能解决问题。由此可见，制氧车间操作工是何等重要！必须深入学习和培训，才能上岗操作；且在操作中要继续提高理论水平和技术水平，否则会把设备开坏，损失不菲。这只塔的遭遇，就是一个典型例子。

# 3. 问：下塔液空节流后的液体与流体纯度（含氧量）是怎样变化的？

答：下塔（含压力精馏塔）的液空纯度是由下塔的回流液及进入下塔的加工空气中的含湿量混合起来决定的。液空经节流后剩余的液体含氧量会增浓，并同时产生一部分

饱和蒸汽，其含氧量会减少，进上塔提馏段的回流液减少，对上塔的精馏工况有一定影响。无论是经过过冷还是未经过冷的下塔液空，在一定压力下的焓值（看作氧一氮二元混合物），可由液体的含氧量（或含氮量）及压力查得（见《深冷简报》1970年第3期附图20)。液空过冷后的焓值可沿等组分线往下移动某个过冷度查得。

大家知道，液空节流后剩余液体纯度的变化是与液空的过冷度、液空节流前后的压力及液空节流前的纯度有关。不管节流后的液体及气体纯度怎样变化，但节流后的汽、液总焓值及总组分是不变的。其值可由剩余的液体及节流所产生的气体两部分叠加而成。只要上述三个因素给定，液空节流后的液体纯度就可以确定，同时节流后的汽化率也可确定。其变化规律大致是液空节流前的过冷度越大、节流前后的压比越小及节流前的液空含氧量越低，则节流后液体的含氧量越低；反之则相反。具体的变化情况，根据节流前后的物料、组分及焓值平衡式可推得液空节流后的汽化率

$$
\alpha_ {R} = \frac {\mathrm {i} _ {R} - \mathrm {i} ^ {\prime}}{\mathrm {i} ^ {\prime \prime} - \mathrm {i} ^ {\prime}} = \frac {\mathrm {x} ^ {\prime} - \mathrm {x} _ {R}}{\mathrm {x} ^ {\prime} - \mathrm {y} ^ {\prime \prime}}
$$

剩余液体含氧量  $\mathbf{x}^{\prime} = \frac{\mathbf{x}_{\mathrm{R}} - \alpha_{\mathrm{R}}\mathbf{y}^{\prime\prime}}{(1 - \alpha_{\mathrm{R}})}$

式中  $\mathrm{i}_{\mathrm{R}}$  、  $\mathbf{x}_{\mathbb{R}}$  ——分别为节流前液空焓值及含氧量

$\mathbf{i}^{\prime}$  、  $\mathbf{x}^{\prime}$  及  $\mathbf{i}^{\prime \prime}$  、  $\mathbf{y}^{\prime \prime}$  ——分别为节流后的液体及气体的焓值及含氧量

下面举例说明：

（1）以  $150\mathrm{m}^3 /\mathrm{h}$  空分设备为例：液空节流前的过冷度分别为  $3\%$  及  $0\%$  ；工作压力 $0.48\mathrm{MPa}$  （表）；含氧量  $38\%$  ，节流后为饱和状态，压力为  $0.038\mathrm{MPa}$  （表），用试凑法查上述附图20，并计算得节流后的汽化率分别为  $12.5\%$  及  $17.3\%$  ；节流后的液体及

气体含氧量分别为  $41\%$  与  $16.5\%$  及  $42.5\%$  与  $17\%$  。

（2）举  $50\mathrm{m}^3 /\mathrm{h}$  （医用氧）空分设备为例：液空的过冷度分别为  $4\%$  及  $0\%$  ；工作压力  $1.3\mathrm{MPa}$  （表）；含氧量  $20.95\%$  （空气全冷凝），节流后为饱和状态，压力为 $0.4\mathrm{MPa}$  （表），用同上方法求得节流后的汽化率分别为  $15.4\%$  及  $19.6\%$  ；节流后的液体及汽体的含氧量分别为  $23\%$  与  $16.5\%$  及

23.5%与10.5%。

至于对液空在各种压力及过冷度下节流后含氧量的变化情况，同样可用上述方法求得。一般工艺流程对节流后的汽、液组分测定是没有设置测点的，如果一定要分析的话，应添加设备使汽液分离后再测定，分析仪表仍采用铜氨溶液吸收法手工分析。
