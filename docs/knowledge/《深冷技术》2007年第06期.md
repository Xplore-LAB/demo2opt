箱内珠光砂。在扒珠光砂的过程中, 存在着一定的
危险, 即容易发生砂爆事故,
导致人员伤亡。每年
都有大量的空分设备进行冷箱内设备检修,
由于对
扒砂过程的危害性认识不够,
往往导致了不必要的
事故发生。笔者就扒砂过程中, 砂爆产生的原因和
应该注意的安全事项做一介绍, 以减少砂爆事故的
发生。
砂爆事故产生的几点原因
冷箱内污氮气量不足, 易造成局部氧量集中
空分设备运行过程中, 由于内容物为低压氮或
液氮的设备和管道表面温度低于常压空气中氧气的
凝结点,
所以冷箱内会出现氧富集现象。一般情况
下,
正常操作过程中要求向冷箱充入一定流量的污
氮气, 如果污氮气不充足, 气流分布不均可能造成
局部的氧量集中。
冷箱内珠光砂局部升温不到位造成富氧液体
积聚
目前,
采用分子筛净化流程的空分设备大部分
取消了外置加温用电加热器, 加温过程相对需要较
长时间,
一般为3~ 7 天,
这个加温过程可能会受
生产周期的限制被缩短。因此,
在冷箱壁与内部设
备之间中间区域的珠光砂没有被加热到常温,
可能
仍有富氧液体积聚。在扒珠光砂过程中这部分富氧
液体被迅速复热汽化,
氧浓度瞬时超标。
冷箱内设备漏液
打开冷箱人孔时,
泄漏在冷箱空间的低温液体
大量汽化,
冷箱内珠光砂大量喷泄出来。当需要扒
砂检修时, 检修人员往往会先打开冷箱下部人孔,
此时珠光砂就顺势而出, 外界热空气就必然会进入
冷箱,
使冷箱内空气产生对流,
加上珠光砂的流动
#
#

摩擦,
导致留存在珠光砂内的低温液体急剧汽化,
冷箱内压力迅速升高, 聚集的能量从人孔释放,
发
生砂爆事故。
珠光砂受潮结块
由于冷箱充气不足或一段时间没充、充装珠光
砂过程中下雨等原因,
造成珠光砂潮湿含水并结
块, 致使珠光砂流动性差。当底部珠光砂被扒出
后,
上部的珠光砂不是缓慢下落,
而是突然大面积
坍塌, 使其从冷箱底部的人孔处喷涌而出。这是由
于冷箱上部结冰严重, 珠光砂板结, 使冷箱不能与
大气相通, 而且上部管道有一定的支撑作用, 一部
分珠光砂不能正常下流,
当扒砂到一定量的时候在
冷箱的中部形成无砂的负压区。在自重和大气压的
作用下,
当上部管道支撑不住上部的珠光砂时,
管
道上部的珠光砂就会迅速下落并压缩负压区的气
体,
气体再压缩下部的珠光砂, 从而造成大量的珠
光砂从冷箱内喷出。
精馏塔内残留液体
精馏塔内有液体残留, 而有液体残留段的两端
阀门关闭, 液体蒸发后压力不断升高,
超过设备材
质或焊缝强度,
造成物理爆炸, 从而使得冷箱内部
压力升高, 珠光砂从人孔喷涌而出。如在某些换热
器通道堵漏过程中没有将相邻通道打通,
冷态时,
液体泄漏过封闭一侧, 复热过程中, 汽化的液体没
有足够的时间返回,
或者温度升高后泄漏的点被热
胀冷缩作用所闭合,
造成换热器爆裂,
同时引起珠
光砂喷出冷箱。
冷箱内氧含量超标
若冷箱中泄漏液体为液氧, 由于珠光砂的吸附
作用, 使得珠光砂中含有液氧, 复热不彻底。而且
在停车加温时,
一般不再具有正常运行时的污氮气
可供使用,
冷箱内充的是空气,
氧浓度下降得很
慢。有时在刚扒砂时,
化验显示氧含量已经合格,
等上面的珠光砂落下后,
氧含量又再升高,
若分析
化验不及时, 同样存在危险。
扒砂前的准备工作
( 1) 扒砂前将顶部人孔全部打开, 冷箱密封气
保持开度, 塔内开始加温复热。此时泄漏的液体被
塔器和冷箱密封气加温后,
变成蒸汽从人孔泄放
掉,
冷箱内设备加热到常温后关闭所有气源, 包括
冷箱密封气,
分析化验冷箱内氧含量低于21% 后
再开始实施扒砂。
( 2) 做好扒砂前的各项准备工作,
冷箱周围设
置警戒线及在紧急情况下的安全撤离通道。最好在
人孔前方设置一个扇形的围堰,
万一喷砂时给珠光
砂一个流动导向;
而人员则在侧面装砂,
以避免人
员被埋在喷出的砂中。无关人员不得进入现场, 每
个人都要有监护对象和被监护对象。一旦发生事故
或正常收工, 要清点人员,
保证安全。
( 3) 注意各设备各部分压力,
避免有两端被封
死的管道或容器存在。
扒砂过程中的注意事项
( 1) 扒砂作业人员一定要穿戴好防护用品。扒
砂时如果劳保用品穿戴不规范,
衣服密封不好有可
能引起皮肤过敏;
如果不戴口罩, 珠光砂被大量吸
入呼吸系统, 容易引起肺部疾病; 万一珠光砂进入
眼睛,
千万不要用手揉, 因为珠光砂很涩, 不易出
来, 应用水冲, 如果不行,
尽快去医院处理。
( 2) 实施扒砂时,
事先用做好的钢管顶住人
孔, 然后拆卸人孔螺丝, 但螺丝不能全拿掉,
应在
上角上留一个, 但必须松开。此时人站在二楼, 周
围所有人员必须撤离。站在二楼的人员,
将顶住的
钢管移开,
并用另一根绳索将人孔向侧面拉开旋
转。此时珠光砂就会从人孔周围排放出来, 但决不
会失控。逐渐增大人孔开度, 控制排砂速度。到后
期, 人孔可以再开大以至全开。
( 3) 时刻观察冷箱内珠光砂的高度是否一直都
在下降, 如持续一段时间没有下降,
而底部仍继续
在流出珠光砂, 就要用木槌在冷箱四周敲击以震动
冷箱壁, 防止结块的珠光砂突然坠落, 砸伤设备管
道和发生砂爆事故。
( 4) 扒砂过程中,
作业人员应密切注意安全,
有的空分设备冷箱喷砂和砂爆事故是在扒砂进行了
一段时间后发生的。
( 5) 冷箱内珠光砂存量少, 无法靠自然流动流
淌到冷箱外面时,
可用木制扒子将珠光砂扒到塔外
再装袋。禁止用铁制工具,
以免损坏管道电缆或仪
表测量点。可派人员到冷箱上用木棍或木槌敲击冷
箱壁,
以震落塔壁内、槽钢上残留的珠光砂。
( 6) 扒砂过程中,
时刻检测珠光砂上部冷箱内
空气的氧含量,
当氧含量高时( 不得超过27%) ,
停止扒砂,
等到氧含量降低后,
再继续扒砂。
设计上应采取的防范措施
( 1) 高度重视空分设备冷箱充氮压力的检测。
#
#

随着内压缩流程的发展,
不仅要检测主冷箱内的充
氮压力,
而且液氧、液氩返流和板翅式换热器隔箱
内压力的监控也很重要。
( 2) 冷箱内的检测点应多点平衡布置, 用来检
测是否泄漏。
( 3) 空分设备的排砂设计,
应向机械化、工程
化发展,
有条件的可利用抽吸式机械从上到下逐层
放砂, 这样更安全、更环保。利用人孔放砂, 不安
全因素太多, 容易发生事故。
( 4) 人孔翻板式放砂孔, 设计时应全部采用不
锈钢材质,
以防止时间过长翻板锈死而无法使用。
同时, 放砂孔应在冷箱三面均衡布置,
放砂孔孔径
应大于400mm,
否则珠光砂不易流淌出来。
结束语
在空分设备事故中,
除主冷爆炸事故外, 最有
可能和最危险的是在扒珠光砂时发生的喷砂和砂爆
事故。但主冷爆炸问题大家比较重视, 并已采取了
相关预防措施, 而类似这样的珠光砂砂爆事故往往
容易疏忽,
因此事故发生率高。砂爆事故不仅危及
扒砂人员的人身安全,
而且会损坏冷箱内的管道,
甚至精馏塔等冷箱内设备的垂直度。故在扒砂作业
时, 一定要高度重视,
以防止事故发生。¬

四级蜗壳结构原因
由于扩压器与蜗壳的接触只有一颗销钉固定,
在该水平方向没有其他有效的约束条件,
导致扩压

煤制油分煤炭直接液化和间接液化制油两大类, 直接
法煤耗低, 3t 煤生产1t 油, 但直接法反应条件苛刻, 对设
备材质要求高; 间接法约5t 煤制1t 油, 具有技术密集型和
资本密集型的特点。世界上目前只有南非拥有一套年产800
万t 油品的间接煤制油工厂。其他发达国家都有成熟技术,
煤业集团与南非萨索公司在陕西和宁夏合作建设两座煤炭
间接液化工厂, 首期建设规模300 万t/ a, 总投资分别为300
亿元左右, 实现了煤制油建设示范项目。神华集团在鄂尔
多斯建设一套世界上独有的大型煤直接液化制油项目,
建
设规模500 万t/ a, 其中一期320 万t/ a, 建成投产后, 每年
用煤量970 万t, 生产各种油品320 万t, 其中汽油50 万t、
柴油215 万t、液化气31 万t 和混合二甲苯24 万t。
中国争取15 年时间生产5000 万t 的煤制油品, 相当于
100 万桶石油; 再通过生物燃料生产2000 万t 替代能源, 从
而达到7000 万t 的替代规模。如果加上中国自产2 亿t 石
油, 就是217 亿t, 2020 年需求预测41 5 亿t 石油,
这样把
中国的石油对外依存度控制在50% 左右。
煤制油技术上的不成熟、巨大投资以及对环境、水资
源可能造成的影响成为煤制油项目上马的主要风险。煤制
油项目要尊重科学规律和经济规律, 先行试点不可一哄而
上。神华煤直接液化每3~ 4t 煤转化为1t 油, 需消耗8~ 9t
水, 还要消耗大量电能, 需建设15 万kW 的电厂。间接液
化耗水是直接液化的115 倍。煤制油项目生产1t 油要排放7
~ 8t 二氧化碳。但是从客观上看对空气污染的程度, 电厂
比煤液化严重得多; 耗水方面, 化肥项目和煤制甲醇项目
也都高于煤制油。神华现在建设的工厂将达到零排放,
废
渣、废水循环利用。中国的煤制油产业能否发展,
最终要
看市场, 包括油价、煤价和水价的市场供求状况。
