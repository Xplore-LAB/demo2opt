近年来，从制氧站到用户，氧气调节阀组事故
频发，有的氧气调节阀燃爆，有的氧气过滤器烧
毁，有的阻火管熔断，更有甚者，整个氧气调节阀
组燃爆，设备严重损坏，影响生产，损失巨大。氧
气调节阀组已成为燃爆事故频发的危险源，应该引
起关注。
氧气调节阀组燃爆事故原因分析
1. 1
氧气管道和调节阀组未吹扫干净、施工质
量差
# 转炉
投产不久，炉顶氧气调节阀组突然发出一声巨响，
发生重大燃爆事故。氧气调节阀烧毁，氧气过滤器
烧出一个大洞，部分氧气管道熔断，高温带压熔渣
还将旁边的钢平台烧穿，并将邻近的其他介质管道
烧损，现场一片狼藉，被迫停产，幸未伤人。经检
查分析，确认事故的主要原因是: 由于客观条件所
限和对安全重视不够，没有将新建的氧气管道和调
节阀组吹扫干净。在烧坏的氧气过滤器内还存有未
烧尽的铁锈、焊渣颗粒，这些异物在纯氧氛围中带
压高速流动时撞击、摩擦，将阀门、管道、过滤器
引燃。施工质量差也是重要原因。不少焊缝未焊
透，现场制作的弯头、三通、异径管等管道附件不
规范，内部有许多突出部分，在氧气高速流动中这
都是容易引起燃爆事故的“撞击场合”。最终，这
套氧气调节阀组完全报废。
氧气调压站的氧气调节阀组发生燃爆。原因也是氧
气管道焊接施工后，焊渣、灰尘、杂物等未吹扫干
净，部分聚集在氧气过滤器内，没有定期清扫，这
些异物不断与高压、高速的氧气摩擦、旋转、撞
击，最终发生燃爆事故。
1. 2
氧气快速切断阀选材或操作不当
中板钢厂250 t 转炉氧气调压室发生燃爆事故，氧
气调节阀及其前后的快速切断阀烧毁，设备损坏严
重，影响炼钢生产。此类事故曾发生多起，究其原
因主要有二: 一是引进的氧气快速切断阀( 蝶阀)
阀体上的密封圈采用了可燃的橡胶材质，在高速、
高压纯氧流中很容易被引燃，这是主要诱因。二是
氧气调节阀前后都设有氧气快速切断阀，需要时能
快速切断氧源，保证安全。但不能快开，若有时快
开，在约1. 5 MPa 压差下氧气高速流动、摩擦、撞
击，极易发生燃爆事故。
1. 3
氧气调节阀组及相关管路选材不当、氧压
偏高
司氧气厂一个氧气调压站突然发出一声巨响，发生
燃爆，有火焰红光，持续了2
3 分钟，断氧后火
焰熄灭。燃爆时氧气压力为2. 5 MPa ( G，下同) ，
正常生产，无任何操作。氧气管道管径DN250 mm，
氧气调节阀口径DN200 mm ( 实际有效通径为
DN150 mm) ，调节阀为从国外引进的全不锈钢截
止阀。调节阀中部阀芯、阀座全部烧毁，入口部分
挂在入口法兰上，出口熔毁，部分落地，出口法兰
及长约500 mm 的氧气管道熔化后落地; 金属熔渣
四溅，喷满氧气调压室内部; 氧气调节阀入口的氧
气过滤器严重变形，但未燃烧。造成大范围断氧，
严重影响炼钢、炼铁生产。
经多次分析研究，认为事故原因主要是: ①引
进的氧气调节阀结构不合理，阀芯与阀座严重偏
磨，成为引燃激发能源。②氧气调节阀材质不合
理，阀芯和阀座均为不锈钢。按照国家安全标准，
这种摩擦、“撞击场合” 应采用不能燃烧的铜合
金。后来制造厂改为采用强度更高且不燃烧的蒙乃
尔( MONEL)
合金。③氧气球罐内无防锈涂料，
罐区氧气管道为碳钢管，可能有锈渣、异物随氧气
流移动到氧气调压站。④氧气过滤器虽未燃爆，但
内外材质为可被引燃的不锈钢，国家安全标准规定
其内件应为铜及铜合金，滤网宜优先选用镍铜合
金，其次为铜合金，必须整改。⑤发生事故时氧气
压力偏高，危险性更大。
氧气调节阀组危险性分析
现从燃爆三要素( 可燃物、氧化剂、激发能
源)
基础理论，结合氧气调节阀组的事故案例对
其进行危险性分析。
2. 1
可燃物
可燃物一般指气体燃料、液体燃料、固体燃料
和一切可以燃烧的物质。氧气调节阀组的阀门、过
滤器、管道及管件等脱脂不净，含有油污; 阀门密
封件、法兰垫片采用可燃材料( 如橡胶等) ; 管道
施工中遗留有破布、手套等易燃物品。这些都是非
常危险的可燃物。
·
·

现对氧气调节阀组的阀门、过滤器、管道及管
件的材质重点探讨一下。碳钢或不锈钢正常环境下
不可燃，但用这样的材料做成氧气调节阀组，其内
部流动着高压纯氧时，一旦混有激发能源就会被点
燃，成为可燃材料。这有理论依据，也被大量事故
证明。碳钢含碳，属可燃性材料，大量的铁素体在
纯氧中可燃，而且燃烧时放热量大，温升很快。不
锈钢中铬含量17%
19% 、镍含量9%
12% ，而
铬含量达到10%
13% 就不生锈，避免了一些激
发能源的产生，比碳钢安全; 但其绝大部分组分仍
然是铁素体，在高压纯氧中可被点燃; 且因其导热
性不好，仅为碳钢的三分之一，一经点燃，烧得更
快。只有铜及铜合金、镍及镍基合金等属不扩散性
燃烧材料。氧气调节阀组材质的选择，应该根据安
全第一、技术经济合理的原则进行。
2. 2
氧化剂
氧气本身就是一种强氧化剂。纯度愈高，压力
愈高，危险性愈大。
氧气纯度越高，物质燃点越低。钢铁在空气中
不能被点燃，而在纯氧中可被点燃。
氧气压力越高，物质燃点越低。当氧气压力为
3 MPa 时，物质燃点比常压时下降约100 ℃。
氧气调节阀组的设备及管道内充满高压、高速
纯氧流，这是其功能之必需，不可避免。
2. 3
激发能源
2. 3. 1
典型的“撞击场合”
欧洲工业气体协会( EIGA)
通过大量科学试
验与工程实践经验总结制定了《氧气管道系统》
标准，得到世界同行认可。该标准将氧气管道、阀
门、管件等科学地分为“撞击场合”和“非撞击
场合”。
使氧气流动方向突然改变或产生旋涡，从而引
起氧气中夹带的颗粒及异物对管壁的撞击，这样的
位置称为“撞击场合”; 否则称“非撞击场合”。
“撞击场合” 容易发生激发能源，引起燃烧与爆
炸，是危险场合，安全控制要求更加严格。氧气调
节阀组的调节阀、入口阀、出口阀、旁通阀和过滤
器等都是典型的“撞击场合”，其中的氧气调节阀
和氧气过滤器，更易产生激发能源，尤为危险。
2. 3. 2
氧气调节阀动作频繁、氧气流速高
为了稳定用户工况，只能保持氧气压力或流量
相对稳定，氧气调节阀需频繁动作，产生激发能
源。当调节阀结构不合理，有偏磨现象时更易引发
燃爆事故。
当氧气流量保持不变时，氧气流速与氧气压力
成反比，与管道或阀门有效通径的平方成反比。对
氧气调节阀而言，为维持氧气流量不变，因减压使
氧气流速增大，又因有效通径缩小1
2 个等级，
流速更大大增大。比如: 氧气管道DN250 mm，氧
气压力2 MPa，输送氧量为20000 m
3 /h ( 5. 6 m
3 /s)
时，氧气流速计算值为5. 7 m/s。当通过有效通径
为DN150 mm 的氧气调节阀减压至1 MPa 后，调节
阀出口氧气流速计算值为31. 7 m/s，流速增大
5. 56 倍，并超过奥氏体不锈钢在1
3 MPa 氧气压
力下所允许的最高流速( 25 m/s) ，非常危险，若
有铁锈、焊渣等异物撞击，极易燃爆。
2. 3. 3

阀件间的摩擦和静电感应

3. 3
清洁的施工是安全的关键
( 1)
施工、安装单位应具备相应的资质。施
工、安装完毕应做好安全、质量检查和验收交接。
( 2)
氧气调节阀组的设备管道施工安装，除
严格遵循压力管道施工有关国家标准外，还有其特
殊的安全规定。如管道与阀门等设备严格除锈脱
脂、焊接探伤、吹扫验收等，若这些环节把关不
严，容易引发燃爆事故。除锈可用喷砂、酸洗等方
法，接触氧气的表面应彻底去除毛刺、焊瘤、黏
砂、铁锈和其他可燃物，保持内壁光滑清洁。脱脂
可用无机非可燃清洗剂、二氯乙烷、三氯乙烯等溶
剂，并应用紫外线检查法、樟脑检查法或溶剂分析
法进行检查，直到合格为止。在施工安装过程中要
采取措施，防止油脂、可燃物、铁锈、焊渣和砂土
等再度污染。管道施工安装完毕，在系统调节阀、
入口阀、出口阀、旁通阀和过滤器尚未安装的情况
下，用无油干燥空气或氮气将管道系统吹扫干净，
直至管道系统内无铁锈、尘埃及其他杂物为止。吹
扫速度不小于20 m/s，且不低于氧气管道设计的最
大流速。吹扫完毕再安装阀门、过滤器等设备。
( 3)
氧气管道系统严格焊接探伤、进行压力
及泄漏性试验是保证施工安装质量和安全的基本要
求与重要环节。这方面由于执行较好，造成的事故
不多，此处不赘述。
3. 4
规范的运行是安全的保障
规范的运行包括氧气调节阀组的操作、维修与
管理。
( 1)
氧气调节阀组中的入口阀、出口阀、旁
通阀等手动调节阀门的开启应缓慢进行，并通过长
阀杆在防护墙外操作，以保护操作者。禁止非调节
氧气阀门作调节阀门使用。严禁用普通油脂润滑阀
杆丝扣，而要用不会燃烧的合格硅油、硅脂或二硫
化钼。
( 2)

速开启，以避免大量氧气在大压差下高速流动引发
燃爆事故。
( 3)
氧气调节阀是典型的“撞击场合”，且动
作频繁、流速高，必须稳妥地遥控操作，一般采用
气动阀门，以保证安全。
( 4)
氧气管道或阀门着火时，应立即切断气
源，以防事故扩大。
( 5)
氧气过滤器应定期清扫，清除铁锈与异
物，保护氧气调节阀，防止发生燃爆事故。若不定
期清扫，氧气过滤器不仅不保安，反而会成为安全
隐患。其定期清扫周期由使用单位根据实际清扫情
况、清洁程度而定。氧气管道系统在施工、检修
后，要送氧前必须检查并清扫氧气过滤器，确认内
部清洁无异物后方能送氧，以保安全。
( 6)
氧气管道及氧气调节阀组要经常检查并
维护。碳钢主干管宜五年进行一次吹扫，以防止铁
锈聚集，引发火灾; 宜五年进行一次管壁测厚，主
要测定弯头及氧气调节阀后的管道，以确保这些易
冲刷、磨损部位的管壁处于安全厚度。每年校验一
次安全接地装置。
( 7)
要经常检查并校验氧气调节阀组的仪控
系统，以保证其安全、准确、灵敏、可靠。

论

在西北地区，汽轮机排汽、凝汽系统多为空冷

外界环境温度较低，则空压机入口气体温度较
低，为防止空压机入口导叶冻结，需要在导叶阀处

在空分设备大型化的过程中，设计人员在设计
主冷单元时发现，为了维持主冷的高度不变( 主
冷液位太高，能耗也会相应增加) ，就必须增大主
冷的换热截面，主塔及冷箱尺寸也要相应增大，这
样会大幅增加一次投资成本。为了解决这种矛盾，
设计人员在主冷单元的创新设计上百花齐放，出现
了很多种结构布置形式，如: 辅冷、多主冷环绕主
塔布置、多层主冷、上塔地面布置+ 液氧泵等，其
中多层主冷以技术成熟、结构简单、投资小等优点
成为近年来主冷结构的主流。

设计时的控制措施
双层主冷在拥有多个优点的同时，也有一个突
出的弱点。在临时停车时，上层主冷的液体会通过
连接上、下层主冷之间的溢流孔流到下层主冷。由
于没有控制手段，上层液体一直会流入下层，使下
层主冷液位不断升高，最后与上层主冷连为一体。
因此，在重新启动空分设备时，由于下层主冷液氧
液位太高，导致下层主冷板式传热温差太小，无法
正常工作，必须先把液氧大量排放后才能正常启
动，这样操作既不方便，又使冷损较大。
近些年，市场上出现了一种新型双层主冷结构
( 如图1 所示) ，旨在解决这个问题。
当空分设备临时停车时，如果上层壳体内液氧
较多( 即液位高于浮球至密封锥面的高度) ，溢流
孔将关闭，上层壳体内的液氧只能通过降液管注入
下层壳体中。当上层壳体内的液氧高度降至与降液
管上端高度相同时，液氧将停止流入下层壳体，从
而防止下层壳体内流入过多液氧，妨碍空分设备重
新启动。如果上层壳体内积液较少( 即液位低于
浮球至密封锥面的高度) ，溢流孔常开，从而使上

工艺操作优化措施
3. 1
液氧泵开车前的优化措施
液氧泵开车前除进行正常的电气仪表检查外，
液氧泵的预冷非常重要。预冷的目的是确保液氧泵
的叶轮、轴和密封圈等部件的温度在开车前达到运
行时的温度，即把常温下的设备冷却到工作状态的
温度条件，开车前操作包括加温和预冷两个阶段。
( 1)
预冷前先加温。缓慢打开液氧泵出口加
温阀或进口加温阀对液氧泵加温，当进、出口吹除
阀外露点在－65 ℃以下、泄漏气温度高于0 ℃时，
关闭加温阀。
( 2)
预冷液氧泵前现场确认盘车正常。现场
打开泵前管道排气阀、出口排气阀和泵体排气阀，
在控制室缓慢打开回流阀，使液氧泵进口阀的开度
从0 增大至5% 。当泵前吹除阀有液体吹出时将其
关闭，在控制室再把液氧泵进口阀的开度从5% 增
大至10% 。预冷期间进口阀门要逐渐开大，前期
2 小时应以气冷为主，气量要由小逐渐变大。后期
以液体预冷为主，预冷期间泵体排气阀必须始终保
持微开并处于结霜状态，在控制室根据现场排气情
况调节进口阀的开度。高压液氧泵大修后的预冷时
间为4 小时，短期停车后的预冷时间为2 小时。预
冷合格的标准是泵体和出口排气阀连续出液。整个
预冷期间要注意液氧泵的进口压力，如果压力过
高，说明液氧泵内部仍有液体汽化现象，预冷不完
全，应缓慢开大泵后吹除阀和泵体排气阀。
3. 2

前
言
气相色谱分析广泛地应用在石油化工、医药卫
生、环境监测、生物化学等领域。该分析方法是以
气体作为流动相将被测样品带入分离系统和检测系
统，载气就是气相色谱分析仪器的血液，其纯度是
影响气相色谱分析仪分析灵敏度的一个重要因素。
随着气相色谱分析仪的发展，氦离子检测器和
氩离子检测器的最低检测浓度均已达到10
－9 级
( ppb 级)
的标准，具有极高的灵敏度。由于现在
的气体生产厂家生产的高纯气体标准通常为
99. 999% ( 5N) ，用户只能选取5N 的气体作为仪
器使用的载气，使得仪器的高灵敏度无法发挥，导
致测量准确性降低。
利用纯度为99. 9999% ( 6N)
甚至更高级别的
气体作为仪器用的载气，存在使用量大、采购不便
和价格高的问题。可见，为高灵敏度的气相色谱仪
配置一套载气纯化器，提高载气的纯度来提高仪器
灵敏度和准确性非常必要，所以使用气体纯化器对
5N 级别的载气进行纯化是一种很方便且成本较低
的可行性方法。
武汉钢铁集团氧气有限责任公司( 以下简称:
武钢氧气公司)
试验验证了使用纯化器前后气相
色谱分析仪器的稳定性、灵敏度的区别和使用高纯
度载气对测量准确性的影响。

对配置载气纯化器的理论分析
色谱分析法基本上使用了外标法。比如需要测
定含有单一杂质的某样品气，假设仪器用的载气中
该杂质的含量为a ( 即行业中常说的本底) ，单组
分标准气的该组分含量为X，得到相应的电信号值
为H; 所测定的样品气中的实际杂质含量为x，得

载气中含有的该杂质和需要测定样品气( 或者标

中该杂质含量的合量。所以按照外标法，

高峰电工况流程简介
原料空气自吸入口吸入，经自洁式空气过滤器
除去灰尘及其他机械杂质。过滤后的空气进入离心
空压机，被压缩到约0. 47 MPa 后进入空冷塔冷却。
冷却水为经水冷塔冷却后的低温水。空气自下向上
穿过空冷塔，在冷却的同时，又得到清洗。
经空冷塔冷却后的空气进入切换使用的分子筛
吸附器，空气中的二氧化碳、碳氢化合物和水分被
吸附。分子筛吸附器为两只切换使用，其中一只工
作时，另一只再生，定时自动切换。
净化空气分成两股: 一股直接进入主换热器，
被返流气体冷却至饱和温度后直接进下塔。一股进
入空气增压机TC2 增压到约0. 73 MPa，增压后空
气经冷却至常温后又分为两股，一股进入低温膨胀
机增压端B1，经增压、冷却后进入主换热器，再
从主换热器中部抽出进入膨胀机膨胀端ET1，膨胀
后进上塔; 一股直接进入主换热器冷却至饱和温度
后，进入液氧自增压器参与换热，出液氧自增压器
后节流进入下塔。
空气经下塔初步精馏后，获得液空、纯液氮和
污液氮，并经过冷器过冷后节流进入上塔。经上塔
进一步精馏后，在上塔底部获得液氧，一部分液氧
进入液氧自增压器，汽化后进入主换热器复热出冷
箱，进入氧气管网。在液氧自增压器底部抽取部分
液氧送入贮槽。
从上塔上部引出污氮气经过冷器、主换热器复
热出冷箱后分成两部分: 一部分进入分子筛纯化系
统的电加热器加热，作为分子筛再生气体，其余污
氮气去水冷塔。
从上塔中部抽取一定量的氩馏分送入粗氩塔，
粗氩塔在结构上分为二段，粗氩Ⅱ塔底部的回流液
体经循环粗氩泵加压后进入粗氩Ⅰ塔顶部作为回流
液; 氩馏分经粗氩塔精馏得到粗液氩，粗液氩进入
精氩塔中部，经精氩塔精馏后在塔底部得到精液
氩，精液氩直接进入贮槽。

低谷电工况流程简介
空气的吸入、冷却、纯化、精馏过程跟高峰电

2. 7 MPa。增压空气冷却至常温后分为两股: 一股
进入高温膨胀机增压端B2，继续增压、冷却过程，

利用峰谷电运行的空分设备，可以节省可观的
电费支出，从空分设备运行成本预算和空分设备建
设一次性投资预算角度考虑，选用能够实现峰谷电
运行的空分设备是有运行成本优势的。但也应当注
意，机器的频繁开、停对设备的要求和损伤会加
大，对电网的影响也比较大。

以确保高压空气出主换热器的温度。

体膨胀机出现故障时采用旁路维持主要产品( 特

加压节能

膨胀机增压端增压产品氮气或者正流空

有再沸器与无再沸器
( 1)
有再沸器: 再沸器的作用是将塔底液体

( 2)
无再沸器: 产品提取率相对较低。

目前，国内空分设备生产厂家不仅在空分设备
的生产能力上直追世界先进水平，而且在先进性上

耗是一个重要指标。实现空分设备的先进指标，除
采取先进的工艺流程外，先进的控制策略，如变负
荷控制就是一个重要的手段。
众所周知，空分设备制氧是一种连续的、不间
断的生产工艺，而后续用氧量却处于波动状态，自
动变负荷控制功能可实现空分设备能够根据用户后

空分设备实现自动变负荷运行时，工艺参数的
·

A + K 平衡流量计是美国马歇尔航空飞行中心( NASA Marshall Space Flight Center)
针对航天飞机的主发动机液氧测
量而设计发明的一种新型差压式流量传感器。采用使流体流过经平衡优化计算而分布的多孔节流件后达到平衡节流，故也
称为平衡流量计( BFM，Balanced Flow Meter) 。其特点是当流体穿过圆盘的函数孔时，流体将被平衡调整，涡流被最小化。
空分设备常用低温液体流量计测量原理:
( 1)
差压式流量计: 当充满管道的流体流经
管道内的节流件时，流束将在节流件处形成局部收
缩。此时流速增大，静压降低，在节流件前后产生
差压，流量愈大，差压愈大，因此可根据差压来衡
量流量的大小。这种测量方式以连续流动性方程
( 质量守恒定律)
和伯努利方程( 能量守恒定律)
为基础。
( 2)
涡街流量计:
涡街流量计( Vortex Flow
Meter)
是基于卡门涡街原理制成的一种流体振荡
性流量计，即在流动的流体中放置一个非流线型的
对称形状的物体( 涡街流量传感器，称之为旋涡
发生体) ，就会在其下游两侧产生两列有规律的旋
涡，即卡门旋涡，其旋涡的频率正比于流过液体的
速度。涡街流量计由涡街流量传感器和流量显示仪
表两部分构成。
( 3)
科里奥利质量流量计: 利用科里奥利效
应测量通过元件移动的质量数量，其实质是测量流
量通过U 形管而在垂直于流动方向上引起的振动，
流体通过U 形管时伴随有引起旋流的振动，旋流
角度越大，流量越大，产生与质量成正比的科里奥
利原理。是一种直接式质量流量计。
案例分析及使用说明
2. 1
案例分析
在实际测量低温液体流量时，使用过文丘里
管、A + K 平衡流量计和涡街流量计等测量仪表，
从现场使用情况来看，结果都不是很稳定，下面用
几个案例加以说明。
案例一: 在某空分设备中采用文丘里管来测量
低温液体流量，工艺要求安装方式为垂直，且流体
方向为自上而下。而在实际使用时，流量计却没有
指示，在现场条件许可前提下，对管路方向进行了
整改，文丘里管的安装方向也由垂直改为水平，流
量计正常工作。
·
·

案例二: 在另一套空分设备中选用A + K 平衡
式流量计来测量低温液体流量，水平安装，引压管
却采用了向下走向，实际测量时流量波动较大。在
对变送器增加阻尼后，情况稍有好转。
从以上的使用情况可以看出，低温液体不但由
于其特殊性，当流经节流装置时，由于节流作用，
处于临界点的液体会发生相变作用，有部分液体会
产生汽化; 而且对低温液体流量计的安装要求也特
别高，稍有不慎，就将严重影响测量的结果。在案
例二中，如果引压管改为向上走向，情况就会有所
改观。
2. 2
使用说明
在选用检测仪表来测量低温液体流量时，不但
要充分考虑流体的特性、安装条件如管道的布置方
向、流体走向、直管段长度等必要的条件外，对于
节流件的结构及加工工艺也要进行深一步的考虑，
除此以外还要考虑液体的过冷度及低温冷箱这一特
定的条件。
低温液体的生产是在冷箱内这一特定环境下完
成的，冷箱内环境温度一般在－40 ℃以下，容器
和管道内介质的保温是依靠冷箱内填充珠光砂材料
来实现，冷箱相当于一个密闭容器，冷箱内仪表的
选用除按常规条件进行选型外，还要考虑测量元件
安装时与管道容器之间的密封性和仪表元件的耐低
温条件。
2. 2. 1
差压式流量计
在利用差压式原理进行低温液体流量测量时，
检测元件首先要考虑选用整体式，与工艺管道之间
必须采用焊接式的连接方式，以防泄漏。仪表引压
管安装时在保证管内介质全部均匀汽化的前提下要
尽量短，并采用斜向上走向，从取压点到冷箱壁的
坡度可以考虑在1∶10 的范围以上，引至冷箱壁后
可采用先垂直向上后水平再垂直向下的交叉走向，
引至冷箱壁根部阀，总之要确保引压管内液体充分
汽化; 同时F + 、F －取压仪表管( 如图1 所示)
间必须绝对地平行排列，根部阀位置也不能低于水
平线下，只有这样才能有效减小测量误差，保证测
量精度。
2. 2. 2

堵塞原因
一般情况下，精氩冷凝器发生堵塞有三个
原因。
3. 1
固体杂质堵塞板式通道
固体杂质，如加工制造时产生的铝屑，或者安
装焊接时形成的焊渣脱落等，因为安装完成后冷箱
内部容器和管道吹扫不彻底，在空分设备运行时堵
塞精氩冷凝器的板式通道。这种情况多发生在空分
设备第一次开车时。
3. 2
氩固化堵塞板式通道
因为出粗氩冷凝器进精氩塔的粗氩气中氩含量
≥99. 6% ，工作压力约为0. 024 MPa，其对应的凝
固温度约为84 K
［1］，而精氩冷凝器一般采用出过
冷器并节流的液氮作为冷源，液氮蒸发压力约为
0. 05 MPa，其饱和温度约为81 K，这两股流体换热
后很容易出现氩固化并堵塞精氩冷凝器板式通道的
现象。这种情况多发生在空分设备正常运行时。
3. 3
冰或者干冰堵塞板式通道
这种情况多发生在空分设备停车保冷一段时间
后再启动时。原因是保冷操作时精氩冷凝器内部出
现负压，而且与大气相通管路上的阀门关闭不严
实，导致大气被倒吸入精氩冷凝器内部。
基于这次堵塞故障发生在刚刚投运精氩塔时且
之前余气放空流量一直有指示值，可以得出造成精
氩冷凝器堵塞的原因为: 液氮导入精氩冷凝器过
快，进入精氩塔的粗氩气未充分精馏甚至可能没有
精馏就上升进入精氩冷凝器与液氮换热，以致被冻
结成固体，完全堵塞了板式单元出口。
处理方法
考虑到氩在一个大气压下的沸点约为87 K
［1］，
与凝固点仅相差3K，且故障发生的时间较短，氩
固化的量应该较少，在分析制氩系统的工艺流程后
认为对精氩冷凝器板式单元进行局部加温即可消除
堵塞，无需排放精氩蒸发器底部液体后对整个精氩
塔组合进行加温。这样既可以节省时间，又可以减
少冷量浪费。
首先关闭FV01605A 阀和LV01606 阀，在冷箱
根部阀处脱开精氩塔顶部压力测点PIAS01604，然
后从就地分析点AE01651 引入一股调节阀用的仪
表气对板式单元进行反吹，待精氩塔顶部回流液温
度回升到80 K 以上后停止加温，最后恢复仪表测
点，重新投运精氩塔，工况恢复正常。整个处理过
程耗时约半小时。

动力资源，空分设备运行的安全性，事关全厂生产

为确保三套空分设备安全运行，对氧气燃爆的
相关事故进行了收集、整理、分析、研究，认识到
发生概率最大的氧气燃爆事故主要有两种: 一是主
冷爆炸事故; 二是氧气管线燃爆事故。标准《深
度冷冻法生产氧气及相关气体安全技术规程》( GB
16912—2008)
的实施，抑制了部分氧气燃爆事故
的发生，但是没有得到完全控制，其根本原因是部
分单位对标准理解不全面，执行不严格。现对主冷
爆炸、氧气管线燃爆等事故发生的原因和预防措施
进行浅显分析，与同行们共勉。

1. 1

心脏部件，主冷中氧气纯度较高( 正常情况下氧

1. 1. 1
空分设备选址不合理
标准GB 16912—2008 中4. 2. 1a)
规定:
“氧
气生产场所应选择在环境清洁地区，并布置在有害
气体及固体尘埃散发源的全年最小频率风向的下风
侧，应考虑周围企业扩建时可能对本厂安全带来的
影响。”虽然标准有明确规定，但是个别单位因企
业扩产导致空间不足，或设计时没有严格执行国家
标准，导致空分设备周围的空气质量差，大量的乙
炔、甲烷、乙烷和乙烯等碳氢化合物被吸入空压
机，进而进入精馏塔，最终引发主冷爆炸事故。
1. 1. 2
无精确的分析仪器
液氧中乙炔的危险性最大，被公认为是主冷爆
炸的原因。乙炔含量的传统分析方法是乙炔铜目视
比色法。随着安全压力的不断加大，该分析方法已
经无法满足工艺要求，原因有三: 一是从液氧取样
到蒸发、吸收、比色，至少需要4 小时，分析结果
严重滞后，无法及时监控主冷状况; 二是目视比色
法本身需要配置比色色阶，系统误差大、操作复
杂，而且药品质量、纯度问题无法科学标定; 三是
乙炔铜目视比色分析法无法检测到主冷液氧中的甲
烷、乙烷、乙烯等碳氢化合物含量。
目前国内能够独立生产碳氢化合物含量分析色
谱仪的单位不多，而且仪器的精度和重现性无法精
确到0. 01 × 10
－6; 国外进口的碳氢化合物含量分
析色谱仪虽然精度高，但是故障率较大。况且，如
果只有一台碳氢化合物含量分析色谱仪，一旦故障
检修周期较长，无法满足至少八小时监测一次的要
求。以上仪器问题的存在也加大了空分设备主冷爆
炸的危险性。
1. 1. 3
大气环境质量的控制难度加大
随着化工行业的不断扩充，空气质量发生了很
大变化，钢厂或化工厂放空气体中的甲烷、乙烷、
乙烯和乙炔等碳氢化合物含量不断增大，总烃含量
一旦超标，主冷极易发生爆炸事故。
1. 2
氧气管线燃爆主要原因
氧气管线燃爆的原因是多方面的，绝对不是单
一的。换言之，只要氧气系统存在安全隐患，早晚
会发生燃爆事故。
1. 2. 1
管道和阀门未安全脱脂
润滑油在氧气中的着火温度较低，为250
300 ℃，是管道的起火源之一。若管壁附着一层7
8 μm 厚的油膜，在1. 6 MPa 压力下，氧气中的铁
锈与管壁摩擦会发生强烈燃烧;
当油膜厚度在
10 μm时就会发生爆炸。油脂存在是燃爆事故发生
的主要原因。
1. 2. 2
管道内大量铁锈结存
管道吹扫不干净，管内锈蚀严重或由于气流与
管壁长期摩擦产生铁锈粉，特别是曾使用湿氧的老
管线，管内存在大量的脱落鳞屑，随干氧与管内壁
发生干摩擦，温度会高达1000 ℃以上。而这些铁
锈粉与不完全氧化的氧化铁和四氧化三铁，当其粒
度小时( 10
100 目) ，燃点很低( 300
400 ℃) ，
不但极易与氧继续发生氧化产生热量，而且会产生
静电( 电压甚至高达几千伏特) 。特别是刚打开阀
门的一瞬间，这些杂质会产生旋涡现象，在阀前
后、弯头、T 形连接管处，导致局部温度急剧上
升，引起管道起火; 或长期冲击管壁，使局部管道
壁厚减薄，引起管道破裂而燃烧。
·
·

1. 2. 3
阀门和管道材质的选择不合理以及管道布
置不合理
氧气管道的燃爆事故绝大部分是在阀门开启、
关闭时发生的，所以阀门选择的合理性颇为重要。
国内好多厂家在选择阀门的材料和结构时不慎重，
甚至选用铸铁闸阀，而在闸阀前后又不设置均压旁
通阀，只是选用一般的无缝碳素钢管。有些氧气管
路有U 形或倒U 形弯，使铁锈等杂物易积聚。有
些氧气管道弯管曲率半径选择过小，有些氧气管道
与高压电线、重油管线布置很近……这些都大大提
高了氧气管道发生燃爆事故的概率。
1. 2. 4
管理不严，职责不清，缺乏送氧操作规程
好多钢厂的氧气厂只负责本界区的氧气管道，
下游的氧气管道无人管理。有的没有定期进行管壁
厚度测量; 有的送氧操作没有规程可循; 有的在送
氧时打开排放阀，这样虽能保证送氧压力不大于
1. 6 MPa，但由于压差大造成的氧气流速过快也会
引起燃烧爆炸。

氧气燃爆事故的防范措施
2. 1
主冷爆炸的防范措施
2. 1. 1
选择合适的空分设备位置
空分设备必须放置在一年四季主导风向的下风
侧，且尽量远离二氧化碳、乙炔、甲烷、乙烷和乙
烯等物质的排放口，避免碳氢化合物在主冷中积聚
而导致爆炸事故的发生。
2. 1. 2
配置灵敏度高、重现性好的碳氢化合物含
量分析仪器
分析数据是工艺操作的眼睛，分析数据是否准
确直接影响到空分设备运行的安全性。因此，选择
分析仪器时必须选择精度高、重现性好、基线稳的
高质量碳氢化合物含量分析色谱仪，最好有两种仪
器的分析数据进行比对，避免一种仪器故障时数据
缺失，造成事故发生。
2. 1. 3
选择安全性更高的内压缩流程
空分流程目前主要分为外压缩流程和内压缩流
程，外压缩流程中，正常情况下只有1% 的安全液
氧排放量; 而内压缩流程中液氧直接经液氧泵加压
后从主冷排出，不存在碳氢化合物浓缩问题。因
此，碳氢化合物总量在内压缩流程中远远低于外压
缩流程，大大提高了空分设备安全性。
2. 1. 4
加大外压缩流程空分设备的液氧排放量
外压缩流程设计的安全液氧排放量至少为
1% ，但是该数据是液氧排放量的最低限，无论气
氧产量如何，安全液氧排放量必须保证。如果分析
发现主冷中碳氢化合物含量偏高，必须加大液氧排
放量，确保主冷安全。
2. 2
氧气管线燃爆的防范措施
2. 2. 1
管道和阀门必须彻底脱脂和吹扫
氧气管道和阀门必须彻底脱脂，脱脂要求和质
量检验按相关标准来执行和验收。
·
·

焊接管道时一定要彻底清除管道内的焊渣，焊
缝必须探伤。
管道送气前必须进行气压试验，彻底吹扫，吹
扫气流速不低于20 m/s，直至管内无铁锈、无尘土
为止。如果管线长，应进行分段吹扫。试压、吹扫
气体必须用干燥无油的氮气或空气。
2. 2. 2
管路设计和材料选择必须合理
管路设计时要采取防雷、防火和防铁锈措施，
防止因雷击或管内铁锈积累而燃烧。如果不得不设
计倒U 形管补偿或L 形管补偿，而又没有采取防
止铁锈的其他有效措施时，必须设置排放铁锈管。
2. 2. 3
按标准要求选择阀门形式及前后管材
不锈钢即使燃烧也是局部的，如果毁坏也仅是
内部压力之故，所以氧气管道必须选择不锈钢管
材。安装旁通阀的目的，是为了减小阀前后的压
差; 而阀的安装位置还必须考虑周围环境的安全条
件。如果有干净气源可用气动遥控操作来控制阀
门，也可以用电动阀，但电气设备应考虑防爆。
若氧气管路没有设置送氧阀的旁通均匀阀，送
氧时必须在阀后充干燥无油的氮气或空气，当阀后
压力比阀前压力低0. 3 MPa 时，再打开送氧阀。
2. 2. 4
加强氧气管路管理
加强氧气管路管理，绘制管路图，编排管道、
阀门号码，建立管路和阀门的档案和管理制度。
结束语
主冷爆炸、氧气管线燃爆事故固然可怕，但是
也无需谈虎色变，事故是可防可控的，只要严格执
行氧气生产和充装的相关操作规程，注重细节管
理，加大安全投入，空分设备的安全运行肯定能得

事故原因排查
( 1)
排除氧气管道内进入石子、纸屑、木屑
的可能性，而与氧气接触的所有密封件或可能与氧
气接触的部件均不采用橡胶类制品，所以不可能由
以上物质引燃。
( 2)
排除油脂类物质进入管道的可能性。从
氧气出塔至氧压机之后的输送管道与外界没有完全
隔离。制氧设备与氧气接触部分禁油，可能与进塔
气体接触的冷冻水也无油。如果是油脂类进入输氧
管路，首先之前的精馏部分就会发生事故。故非油
脂类物质所致。
( 3)
氧气在管道内流速过高，因摩擦产生大
量火花。碳钢管道或铸钢阀门产生的铁锈或铸件上
脱落的颗粒与管道摩擦产生火花导致燃烧，当氧气
输送管道上的截止阀开启时，若阀前后压差很大，
氧气流速瞬间可达200 m/s，一般碳钢就会燃烧起
火引起爆炸。从氧气出塔管道到氧压机至球罐区域
的管阀都采用不锈钢管及硅黄铜阀门，氧气流速也
不可能达到200 m/s 以上，更不会有铁锈及铁质颗
粒在过滤器内引燃，且当时也没有操作送气作业。
( 4)
氧压机气阀的阀片断裂脱落后进入管道，
随氧气流移至过滤器内，因不能通过过滤网，产生
摩擦发热引燃了过滤器。检查6 台氧压机的气阀，
均没有发现有阀片损坏的现象，可以排除这个

( 5)
氧气管道对接法兰等处泄漏，外来火源
引燃爆炸。经确认事故发生时现场并无检修动火作
业，制氧为封闭式管理，也无闲杂人员出入。这点
也可以排除。
事故原因分析
为了查明氧气过滤器烧毁原因，拆除烧毁的过
滤器，发现过滤器两端连接管道上的法兰与管道的
焊接口上焊缝异常，在管内的焊缝连接处挂着一圈
焊瘤。再用强光检查管内其他焊接口，发现了同样
的问题，这些管道明显没有按照氧气管道施工规范
的要求———氩弧焊单面焊接双面成型的工艺进行规
范施工，使管内留有大量黏连在焊缝上的焊瘤，这
些焊瘤在管道施工完吹扫中不一定会完全脱落，在
停、送氧气的过程中，脱落并随气流移动到氧气过
滤器的不锈钢过滤网前，由于大颗粒的焊瘤不能通
过网孔，留在过滤网前的焊瘤颗粒被气流吹至过滤
器的死角内，随着气体的流动及用户用气量的波动
而不能稳流的氧气通过过滤网时产生旋流，焊瘤颗
粒在旋流的气体作用下与过滤器壁发生高速摩擦，
当温度达到钢管燃点时发生了燃爆事故。
而且，发生事故的过滤器外壳体、内部构件以
及过滤网全部采用不锈钢材料，而标准《深度冷
冻法生产氧气及相关气体安全技术规程》
( GB
16912—2008)
的规定: 氧气过滤器内件应选用铜
及铜合金，滤网宜优先选用镍铜合金材质。如采用
镍铜合金为构件及滤网，当过滤器达到燃烧条件着
火后，镍铜合金材质与火焰发生化学反应也会起到
阻止燃烧的目的。过滤器制造厂家未按国家标准选
择过滤器材质，也是燃爆事故的原因之一。
解决方案
( 1)
为防止通过过滤网的小颗粒焊瘤进入后
面的氧气管路，进而给氧气输送埋下安全隐患，打
开氧气球罐的排污阀进行吹扫，在确保无杂质后结
束吹扫工作; 为了确保用户处氧气管道的安全，停
止向用户供氧，用氧压机压缩氮气对管道进行彻底
吹扫，确认管道干净后恢复送氧。
( 2)
施工存在问题的管道，属于安装并投运
3 /h 空分设备配套氧压机至球
罐的管道。停用这批管道，重新安装管道。制定详
细的施工方案，由有资质的管道施工单位严格按照
氧气管道施工规范作业，施工时抽样检查，根据设
计要求对固定口及转动口焊接的焊缝比例进行拍片
检查，完全合格后投入使用。
( 3)
根据标准GB 16912—2008 的规定，改造
氧气过滤器。氧气过滤器内件选用铜及铜合金，过
