# 现代液化天然气工厂建设的投资分析

朱刚顾安忠

（上海交通大学低温与制冷工程系，上海市华山路1954号，200030）

关键词：液化天然气 液化天然气工厂 设计原则 投资分析

# 1 前言

天然气是一种优质能源，其主要成分是甲烷，具有燃烧热值高、污染少的特点。近30年以来，由于石油危机的冲击和环境污染问题日趋严重，世界能源结构在逐渐变化，天然气正在成为最受欢迎的能源之一。1992年世界能源一次消费中，天然气已占 $23\%$ ，石油和煤分别占 $42\%$ 和 $27\%$ ，核能和其它占 $8\%$ 。据美国IGT的研究报告，到2010年天然气将超过煤炭，成为继石油之后的世界第二大能源。

天然气的产地往往远离能源消耗区，特别是海上天然气的开发必须解决运输问题。与气态天然气相比，液化后的天然气只有原体积的1/625，所以采用LNG的形式对天然气的储存、运输及合理运用都具有十分明显的优越性。自1980年以来，世界LNG贸易量以平均  $8\%$  的速度增长，占世界天然气总贸易量的  $25\%$  。1996年世界LNG贸易量已

达  $9.2 \times 10^{10} \mathrm{~m}^{3}$ ，据预测到2000年LNG贸易量可达  $12.2 \times 10^{10} \mathrm{~m}^{3}$ 。

LNG 工业链包括气田建设（投资占总投资的  $10\% \sim 20\%$ ）、天然气液化（ $25\% \sim 35\%$ ）、运输（ $15\% \sim 25\%$ ）、LNG 接收终端建设（ $5\% \sim 15\%$ ）、天然气管网建设与发电（ $25\% \sim 35\%$ ）等环节，投资庞大，大大增加了 LNG 的成本。特别是 80 年代以来随着原油价格的稳定和天然气主要消费市场的成熟，液化天然气市场面临管道天然气、石油以及其它燃料的竞争。这对液化天然气工厂的建设提出了新的要求，即在保证工厂的产量、安全性和可靠性的基础上尽可能地降低投资成本，从而使产品的价格更有市场竞争力。降低 LNG 工厂的投资成本已成为国际天然气会议的热门话题和 LNG 工厂建设的重要指标。目前正在建设中的阿曼、卡塔尔、特立尼达和尼日利亚 LNG 工厂都采取措施降低了投资成本，单位产品的投资比

10年前降低了  $10\% \sim 15\%$  。

我国有丰富的天然气资源，是世界天然气大国之一。仅就30个主要盆地及地区的天然气四种成因的预测，远景资源达33.36 $\times 10^{12}\mathrm{m}^3$  。发展我国的LNG工业，可以有效利用天然气资源，改善我国以煤炭为主的能源结构，减少环境污染，解决能源供应日趋紧张的局面。目前，我国正在建设和设计中的LNG项目有新疆LNG工程、东海天然气调峰工程、成都LNG工程和渤海LNG工程。深入了解现代LNG工厂的投资情况对指导我国LNG项目的建设有一定借鉴价值。本文针对国外LNG工厂投资的分布情况以及降低投资的措施作一简要介绍。

# 2 LNG工厂建设投资的趋势

首先我们回顾一下自60年代以来LNG工厂的投资情况。图1给出了世界上正在运行或建造中的17个重要LNG项目的单位产品投资情况（美元/吨每年）。图中的投资成本以工厂投产的年份为基准并考虑了通货膨胀的因素。

![](images/bab0086248e2506c821b207fe75f1949b1e5c330109ca7de1e411e89e5afe58a.jpg)  
图1LNG项目的投资成本

国1从一个侧面反映了LNG工业的发展情况。最早的LNG工厂的天然气液化循环采用经典逐级制冷循环，工程设计借鉴了空分、提氦等工艺流程的经验。液化天然气产量一般较小，年产量100万吨左右，工厂的投资较少。但由于该液化流程的管道及控制复杂，维护不便，70年代后很少采用。70年代早期利比亚、阿尔及利亚等地建造的LNG工厂采用混合制冷剂液化天然气循

环，减少了设备投资，液化天然气产量也有了较大提高。但由于此类工厂的设计和操作缺乏经验，几个LNG工厂的产量未达到设计要求，投资也超过了预算。

70年代后期和80年代初期，APCI发展了丙烷预冷的混合制冷剂液化流程，并在大型的LNG工厂得到了广泛应用。此期间全球的高通货膨胀率大大增加了设备投资，促使LNG工厂建设采用有经验的设计、采购、建造承包商EPC有偿投标的形式，设计原则也趋向保守以保证产品供应的可靠性。80年代中后期APCI的丙烷预冷的混合制冷剂液化技术有了进一步发展，促进了LNG工厂生产线规模的扩大。在此期间，LNG工厂的设计采用了主传动轴技术，燃气透平开始成为较佳的制冷压缩机驱动机形式，空气冷却方式证明可以替代传统的海水冷却方式。3 现代LNG工厂的设计原则

# 3.1 生产规模的经济性

LNG工厂的总产量与单线生产规模的增加都可以减少LNG工厂的投资。当前正在建设中的阿曼LNG工厂的单线生产规模已达每年  $3.3\times 10^{6}$  吨，大大超过了80年代 $2.0\sim 2.5\times 10^{6}$  吨的水平。现代LNG工厂单线生产规模的扩大主要应归功于大型单轴燃气透平的应用。早期的LNG工程采用较小的双轴燃气透平作为压缩机驱动机，例如额定功率28MW的GE-5双轴燃气透平，限制了最大生产线规模为  $2.7\times 10^{6}$  吨。而现在采用额定功率38.5MW的GE-6燃气透平驱动丙烷制冷压缩机，额定功率80MW的GE-7燃气透平驱动混合制冷剂压缩机，可使单线生产规模达到  $3.3\times 10^{6}$  吨/年。如果全部采用GE-7燃气透平作为压缩机驱动机，单线生产规模可望达到  $4\times 10^{6}$  吨/年。

由于LNG工厂的投资大约仅  $5\%$  与总产量有关，在生产线数目相同的情况下随着生产线规模的扩大，投资费用的增加并不显著，所以增加单线生产规模可以收到最大的规模效益。Merlin协会的报告指出，对于同样拥有两条生产线的LNG工厂，年产量  $8 \times 10^{6}$  吨的工厂总投资仅比年产量  $6.6 \times 10^{6}$  吨的工厂多  $10\% \sim 15\%$  ，但是单位产品的投资要低  $15\%$  ，生产线的运行费用也低。这是由于在生产线的数目相同的情况下，操作人员的工资是固定的，与生产规模无关，其它诸如技术咨询以及管理费用也基本不变。另外在总产量一定时，单线生产规模的扩大可以减少生产线的数目，不仅缩短了工厂建设周期，还节省了大量设计、采购和建设承包商的费用。

# 3.2 减少设计的余量

现代LNG工厂的设计是在吸取了30多年来LNG工厂的设计、运行经验的基础上逐渐发展起来的。它基于生产者和购买者之间达成的这样一个共识，即完成契约的义务，保证LNG供应的安全与可靠。LNG工业发展的早期，由于技术不成熟，设计原则较为保守，设计余量取的较大，增加了投资成本。现在随着LNG技术的成熟与人们管理运行经验日趋丰富，已经能够保证长期稳定地供应LNG产品，工厂设计的风险系数可以适当减小。确定设计余量一般主要考虑以下因素：（1）设备的老化情况；（2）原料气参数以及运行中流程参数的变化；（3）设计误差以及设备尺寸的限制。一般而言，设计余量取为总设计产量的  $8\% \sim 12\%$  。

# 3.3 竞争机制和EPC承包商

70年代的LNG工厂建设采用3到4个投标者总承包的方式。工程开工后，由于承包商数量有限，经验不足，另一方面缺乏国际石油公司的技术指导，再加上国际性通货膨胀的影响，总投资大大超过了预算，而且往往不能按时投产。

现代的LNG建设项目中，设计、采购和建设（EPC）承包策略发挥着重要作用，直接影响到LNG项目的建设周期、设计工作的费用以及整个工程的质量。业主与承包商共同努力完成LNG工厂的基础设计方案和具体实施规划，通常负责工程前后期设计任务的承包商获得EPC合同，继续进行施工阶段的工作。这样在工程设计阶段就可预定LNG工厂的关键设备，缩短了工程周期。随着LNG工厂设计与建造技术的成熟，其技术规范也越来越详细、明确，合格的EPC承包商的数目大为增加。在满足LNG工厂技术要求的前提下，不同的EPC承包商针对具体的工程设计展开竞争，可以节省数百万美元的EPC费用。

# 4 影响LNG工厂建设投资的因素及节省投资的措施

# 4.1 工厂的选址与布局

LNG工厂的厂址对总投资影响很大，理想的厂址应满足以下条件：

- 地质条件稳定，地势较为平坦。  
- 接近气源且有充足的水源供应。  
- 原有的基础设施较好，交通方便，船坞可满足  $300 \sim 500$  吨货物的装卸需要。

通常很难找到符合所有以上条件的地方，承包商必须对多处可能的厂址进行综合评价，列出工程预算，选取最佳厂址。

工厂的布局直接影响到工厂建设的周期和费用。紧凑的布局使管道较为密集，减少了直管段的长度和支撑的数目，同时也增加了管接头等配件的数目。但考虑到管接头等配件的价格比直管高，过多的接头会增加管道投资，同时狭窄的管道空间不利于维护。因此承包商必须对工厂的整体布局进行优化，在保证安全可靠的前提下，寻求最低的成本。具体节资措施有：（1）优先布置管线，减少管接头和阀等配件的数目；（2）设备尽可能靠近管道支架，以避免不必要的结构支承；（3）使用预制的混凝土结构支架。

# 4.2 生产线的数量和产量

一般说来，出于生产可靠性的考虑，LNG工厂配置不只一条生产线。选择和配置生产线数量和产量的根据是上游天然气的供应能力，下游LNG船的运送能力和LNG工厂的经济性。由于总产量一定的情况下生产线规模的扩大可以节省约  $15\%$  的设备投资，近年来LNG工厂的单线生产能力不断增加，如图2所示。

但是，这并不意味着所有情况下生产线的规模越大越好。LNG工厂的生产规模要受到天然气资源的储量和LNG市场需求情况的制约。

早期LNG供应合同一般是20年，现在已长达25年。对于一个年产量  $8 \times 10^{6}$  吨的天然气项目来说，在合同期间所需的原料气多达  $3 \times 10^{12} \mathrm{~m}^{3}$ ，如此规模的天然气气田在全球范围内也是有限的。同时，生产规模的扩大也带来了产品销售的问题。自1985年以来，LNG工厂单条生产线的产量已增加了约  $75\%$  ，随之而来总产量的增加已大大超过了区域性LNG市场的需求。LNG生产者不得不在全球范围内寻找买主，激烈的市场竞争会直接影响工厂效益。因此LNG工厂的建设必须因地制宜，综合考虑就近的天然气资源储量和LNG市场需求的情况。

随着LNG工厂设计和管理经验日趋丰富，设备的长期可靠运行已成为现实；同时LNG市场的供货渠道也不断增加，这都减少了配置多条生产线保证产品供应的重要性。在天然气储量和LNG市场需求不足以满足多条生产线要求的情况下，可以考虑只配置一条生产线。例如特立尼达的LNG项目根据市场的需求情况就只配置了一条年产量  $3 \times 10^{6}$  吨的生产线。如果预计LNG销售市场前景看好时，LNG工厂的建设可以采用分批投资的方式，即先建设一条生产规模较小的生产线满足最初的市场需求，以后条件许可时再扩建生产线的规模。

# 4.3 工艺流程技术

LNG工厂中最主要的工艺流程是原料气预处理流程和液化流程。

# 4.3.1 原料气预处理流程技术

原料气中  $\mathrm{CO}_{2}$  和  $\mathrm{H}_{2} \mathrm{~S}$  的含量对预处理设备的投资影响很大，很高的酸气含量将使预

处理设备庞大复杂并增加了加热和制冷的负荷，因此选择最优的预处理流程将大大降低预处理设备的投资。

目前，胺法、Sulfinol法和Benfield法是最有效的酸气脱除方法。最终LNG产品要求  $\mathrm{H}_2\mathrm{S}$  的含量低于  $4\times 10^{-6}$ ，总的含硫量低于  $25\times 10^{-6}$  。当  $\mathrm{CO}_{2}$  含量超过  $15\%$  时，可以采用低温分离法或膜分离法。当分析膜分离法的经济性时，不仅要考虑到膜的费用，还要考虑到进入膜分离装置前原料气的预处理费用，据评估两者大致相等。当原料气中硫的含量较高时，要求预处理设备能够有效地去除其中的硫成分。如果通常的酸气处理设备不能完全去除硫醇，就要附加一套分子筛吸附设备去除剩余的硫醇。Sulfinol作为溶剂能有效地降低原料气中硫醇的含量，适用于处理高含硫量的原料气。

# 4.3.2 液化流程

液化流程设备的投资占LNG工厂总投资的  $32\%$  ，直接影响到LNG工厂建设及其运行的经济性。就丙烷预冷混合制冷剂液化流程的设备投资情况而言，压缩机/驱动机、换热器与分馏塔的投资所占的份额分别为 $58\%$  、  $36\%$  和  $6\%$  ，其它液化流程的投资与其类似。由此可见，压缩机/驱动机是投资的重点，是衡量工厂经济性的主要因素。

设计LNG工厂时，生产线的规模往往以适应现有的压缩机/驱动机能力而选定，即要求发挥选定的压缩机、驱动机的最大工作能力并尽量减少每套机组内机器的数量。所以，采用大型的燃气透平作为制冷压缩机的驱动机可以扩大生产线的规模，降低单位产品投资，获得最大的规模效益。表2列出了目前应用于LNG工业的GE系列的燃气透平数据。

（1）有效运行时间——指工厂每年正常运行的天数，即一年中扣除计划内设备的维护时间与计划外停工时间。燃气透平驱动的LNG工厂计划内设备维护时间主要是由燃气透平的维护时间决定，计划外的停工时间则是由设备故障引起的。  
(2）易维护性——主要与换热器类型和绝热系统有关。  
(3) 运行可靠性——依赖于液化流程与机械设备的稳定性，还与控制回路的复杂程度有关。

特立尼达LNG工厂采用了菲利浦石油公司的级联式液化技术，打破了70年代以来丙烷预冷混合制冷剂液化循环一统天下的局面。GAZ DE FRANCE开发了CII液化流程，可节省  $20\%$  的设备投资。BHP也推出了新的液化流程，并称投资要优于目前的液化技术。

一般而言，液化流程的简化在节省投资的同时往往会造成热力学效率的下降。但是，精简的液化流程减少了设备投资费用和建设周期，适用于小型的LNG项目（年产量小于  $2 \times 10^{6}$  吨）。而且由于其结构紧凑的特点，可作为海上流动LNG工厂的液化流程，有效回收一些远海小规模的天然气资源，效益显著。

# 西南化工研究设计院PSA研究所已推广503套PSA装置

在1999年4月13~16日·北京中国首届国际工业气体技术、设备与应用展览会上，变压吸附参展单位之多，成为非低温空气分离的一条特别风景线，令参观的行业人士关注。其中，作为我国变压吸附的龙头科研单位、软硬件一体化的西南化工研究设计院变压吸附分离工程研究所，以其较强的参展阵势、领先的分离技术、超群的成果业绩，在展览会上独领风骚。下面根据在展览会上获得的样本及资料，将西南院PSA所作一介绍。

西南化工研究设计院，始建于1958年，现有职工1600余人，全院设有16个研究、开发、设计所室，7个生产及辅助工厂。西南院从70年代初在国内率先进行变压吸附气体分离技术(PSA)的研究工作。1981年在国内首次开发成功从合成氨弛放气中回收纯氢装置，实现工业化应用。1992年专门成立变压吸附气体分离工程研究所，现有工程技术人员200余人，目前已成为世界上PSA技术应用开发领域最广、推广成套装置数量最多的专业化研究所。

# 空分装置中的氧化亚氮

Dr. ulrike wenning, Hollriegelskreuth

# 1 序

对空分装置来说，氧化亚氮（也叫一氧化二氮，分子式为  $\mathrm{N}_2\mathrm{O}$  ）是一个相当新的问题，因为在过去从未对此有足够的关注。氧化亚氮有时会使空分装置出问题，如可使管道或热交换器产生堵塞，或者在精馏塔中累积起来并使产品受到污染。

另一方面，现在都要求有较高纯度的产品，杂质的含量通常要求达到ppb级（即 $10^{-9}$ ，下同——编者）。半导体工业尤其需要超高纯的气体。

因此，氧化亚氮以及它在空分装置中的行为正在变成一个越来越受到重视并且使人感兴趣的研究领域。本文试图研究的问题是氧化亚氮在空分装置中的流动轨迹。

# 2 大气中的氧化亚氮

现在，氧化亚氮（ $\mathrm{N}_2\mathrm{O}$ ）在大气中的浓度约为  $310\mathrm{ppb}$ （ $0.6\mathrm{mg/m}^3$ ），比  $\mathrm{NO}$  和  $\mathrm{NO}_2$  的浓度要高一到二个数量级。这个数值还在不断地增加，每年约增大  $0.2\%$  到  $0.3\%$  或  $0.7\mathrm{ppb}$ 。在1960年，氧化亚氮的浓度是  $292\mathrm{ppb}$ ，1980年增加到  $302\mathrm{ppb}$ 。由于氧化亚氮是相当惰性的，因此它在大气中的寿命可达  $130\sim 170$  年。这就引起了一系列的环境问题：氧化亚氮会使平流层升高，使臭氧层毁坏；另一方面，在对流层，它可以使温室效应增强  $4\% \sim 5\%$ 。

# 3 氧化亚氮的来源

三分之一的  $\mathrm{N}_2\mathrm{O}$  是人为生成的，但多数是天然生成的。土壤微生物尤其是在热带土壤中以及在海洋中的氮化和脱氮活动生成

了大气中的大部分氧化亚氮。类似的反应也使废水处理装置周围的氧化亚氮浓度升高。硝酸盐肥料被微生物分解后也会使农场周围的氧化亚氮浓度升高。

矿物燃料、生物体以及废弃物的燃烧也会生成氧化亚氮。

在一些工业过程中，例如在生产臭氧和聚酰胺时， $\mathrm{N}_2\mathrm{O}$  是一种副产品。

减少废气中的氮氧化物（ $\mathrm{NO}_{\mathrm{x}}$ ）是现在的普遍要求。然而，在使用SCR或SNCR过程以还原  $\mathrm{NO}_{\mathrm{x}}$  时， $\mathrm{N}_2\mathrm{O}$  可能成为二次生成物，但其生成的程度和条件总的来说还没有完全搞清楚。

此外，由于在汽车上广泛用催化转换器以改善空气的质量，也将导致  $\mathrm{N}_2\mathrm{O}$  浓度的增大。另一方面，当废气中存在  $\mathrm{NO}_{\mathrm{x}}$  、 $\mathrm{H}_2\mathrm{O}$  、 $\mathrm{SO}_2$  及  $\mathrm{O}_2$  的情况下，也发现了  $\mathrm{N}_2\mathrm{O}$  。

人类形成的一些发酵源也会使大气中的氧化亚氮浓度逐年增加。由于氧化亚氮在化学上是相当惰性的，因此现在氧化亚氮还没有达到极限。

在氧化亚氮生成源的附近，空气中氧化亚氮的浓度将远远超过其平均值，已经发现可以达到几个ppm（3个ppm以上）（ppm即 $10^{-6}$ ，下同——编者）。

氧化亚氮在化学上是相当惰性的，它既不会造成腐蚀，也不会引起爆炸。

# 5 对空分装置的影响

氧化亚氮在空分装置的不同部位都会带来一些问题。在极端情况时，会使空分装置停机以除去累积的  $\mathrm{N}_2\mathrm{O}$  。图4给出了空分装置的典型流程图。空气经过滤器、压缩机、冷却器、水分离器后进入装置。 $\mathrm{N}_2\mathrm{O}$  在水中的溶解度很小，因此，不可能在这一阶段随着水的分离而被大量除去。

# 6 分子筛

在空分装置中，分子筛用来吸附空气中的  $\mathrm{CO}_{2}$  及  $\mathrm{H}_{2} \mathrm{O}$  。为了彻底去除  $\mathrm{CO}_{2}$  及  $\mathrm{H}_{2} \mathrm{O}$ ，对吸附循环进行了最佳化。

分子筛，例如5A分子筛，对氧化亚氮的吸附能力要小于对二氧化碳的吸附能力，因此，在  $\mathrm{CO}_{2}$  穿透以前，  $\mathbf{N}_2\mathbf{O}$  早已穿透。此外，在共吸附的情况下，分子筛对  $\mathbf{N}_2\mathbf{O}$  吸附能力的下降要比对  $\mathrm{CO}_{2}$  吸附能力的下降明显得多。在有  $\mathrm{CO}_{2}$  存在的情况下，  $\mathbf{N}_2\mathbf{O}$  的穿透曲线要比  $\mathbf{N}_2\mathbf{O}$  单组分吸附时的穿透曲线更陡斜。图5给出了一个例子，说明在不同的  $\mathrm{CO}_{2}$  浓度下，5A分子筛对  $\mathbf{N}_2\mathbf{O}$  的吸附情况，并指出了共吸附的影响。尽管给定的数值是不现实的，但我们可以清楚地看到： $\mathrm{CO}_{2}$  已经把被分子筛微孔吸附的  $\mathbf{N}_2\mathbf{O}$  又置换了出来，且这种效果随着  $\mathrm{CO}_{2}$  浓度的增大而增强。由于这个原因，在吸附器出口处的  $\mathbf{N}_2\mathbf{O}$  浓度，甚至可以超过其在入口处的浓度。这就意味着分子筛不仅没有吸附掉  $\mathbf{N}_2\mathbf{O}$ ，反而把以前吸附在分子筛中的  $\mathbf{N}_2\mathbf{O}$  又释放了出来。

一般来说，大气中  $\mathrm{CO}_{2}$  的浓度是  $350\mathrm{ppm}$ ，而氧化亚氮的浓度仅为  $0.31\mathrm{ppm}$ 。

在这种情况下，吸附剂对  $\mathrm{N}_2\mathrm{O}$  的吸附能力是非常之小的。

如果上面提到的  $\mathrm{N}_2\mathrm{O}$  生成源在周围环境中出现，则  $\mathrm{N}_2\mathrm{O}$  在大气中的含量就会增大，甚至可以达到几个ppm。随着这个浓度的不同和吸附器工作条件的变化，  $\mathrm{N}_2\mathrm{O}$  可以在吸附器切换前或者在吸附器只达到半个工作周期时就穿透出来。使用一个对  $\mathrm{N}_2\mathrm{O}$  适当的吸附循环时间或者在吸附器中装入比吸附  $\mathrm{CO}_{2}$  所需的分子筛更多的吸附剂，对吸附  $\mathrm{N}_2\mathrm{O}$  是有帮助的，但是会使能耗明显增大。对吸附  $\mathrm{N}_2\mathrm{O}$  更适宜的吸附材料的研究在将来是很必要的。

由于分子筛吸附器对加工空气中的  $\mathrm{N}_2\mathrm{O}$  不能彻底除去，  $\mathbf{N}_2\mathbf{O}$  将进入冷箱。

# 7 主换热器

在主换热器里，空气被冷却到接近液化温度。在这里可能引起的问题是：氧化亚氮首先冷凝后生成的固体沉积物可以使空气流路阻塞。

图6给出了  $\mathrm{N}_2\mathrm{O}$  的部分升华曲线（该升华曲线是根据一个热力循环估算得到的）。例如，当总的压力为6bar，氧化亚氮的浓度为1ppm时，根据道尔顿定律，氧化亚氮的分压是  $6\times 10^{-6}\mathrm{bar}$  （假定处于理想状态）。在图的纵坐标上找到  $6\times 10^{-6}\mathrm{bar}$ ，作水平线与升华曲线相交，从横坐标得到的温度是 $113\mathrm{K}$ 。固体  $\mathrm{N}_2\mathrm{O}$  的沉积也从这儿开始。空气的液化温度（冷凝开始温度）约为  $100\mathrm{K}$ 。

![](images/1bf6a4bf9c46f2e969dbde3ad352b33beb21f415d7f5ae26c73aec11dd3c659a.jpg)

在所规定的条件下， $\mathrm{N}_2\mathrm{O}$  可以在空气的冷凝温度以上作为固体沉积出来。

当浓度较低时，分压也相应降低，空气的液化将首先发生。在这种情况下，只有当 $\mathbf{N}_2\mathbf{O}$  在液体空气中的浓度超过其溶解度极限时，才会发生主换热器空气通道的阻塞。

# 8 精馏塔

众所周知，在精馏塔里，空气被主要分离成氮气和氧气。这里，氮是易挥发组分。

在进入精馏塔的加工空气中，凡是挥发度比氧大的所有组分，例如氮、氢、氖和一氧化碳，都会与氮气一道离开精馏塔。而挥发度比氧小的所有组分，例如氪、氙、烃和氧化亚氮，则集中在氧产品中。一般情况下，氧产品由  $99.5\%$  的氧和  $0.5\%$  的氩，痕量杂质：例如甲烷10ppm、其它烃0.5ppm、氪5ppm、氙0.4ppm等组成。在常规的空分装置中，氧化亚氮在液氧中含量的典型数据为0.1到11ppm，其平均值为1.4ppm。

但是，氧化亚氮的浓度常常会超出上面给定的常规数值。当主要生产氧气并且液氧排放不充分时，发现在主冷中氧化亚氮的浓度已累积到  $80\sim 160\mathrm{ppm}$  。如果超过在实际情况下  $\mathbf{N}_2\mathbf{O}$  在液氧中的溶解度极限时（例如当压力为1.3bar对应温度为93K时，  $\mathbf{N}_2\mathbf{O}$  在液氧中的溶解度极限是  $50\sim 100\mathrm{ppm}$  )，就会发生堵塞等问题。这样的浓度是容易实现的，在氧气部分蒸发时尤其是这样。

# 9 稀有气体的生产

氧化亚氮不可能被分子筛完全除去，它将进入提取氮、氩的精馏塔内，在氮、氩浓缩时，氧化亚氮也同时浓缩。

离开富集氮、氩精馏塔后的气体，还要被进一步加工：

氮、氩浓缩物中的主要组分是氧。为了安全的原因，甲烷的含量必须加以控制。由于存在爆炸的危险，甲烷的浓度不允许超过 $0.5\%$  。在稀有气体增浓时，这个危险值是

很容易达到的，因此，必须在接触式反应器中将甲烷通过催化燃烧掉。在这一阶段，气体中各组分的平均浓度约为：氧  $98\%$  、氮 $1\%$  、甲烷最高达  $0.5\%$  、氙  $0.09\%$  、氧化亚氮  $100\sim 150\mathrm{ppm}$  以及痕量氙和其它杂质。催化剂一般由贵金属组成，例如钯，以 $\mathrm{Al_2O_3}$  作为承载物。除去甲烷的催化反应的温度在  $250^{\circ}C$  到  $600^{\circ}C$  之间，反应生成的水和二氧化碳在分子筛吸附器中被除去。

这一阶段也可以用来除去氧化亚氮。

氧化亚氮本身是不燃烧的，但是它可以帮助燃烧。在大气压力下，氧化亚氮约在  $600\sim 650^{\circ}C$  开始热分解，热分解后的产物是氧和氮。这个反应是一个放热反应，是不可逆的。在有合适的催化剂存在的情况下，氧化亚氮可以在低得多的温度下实现分解。上面提到的在接触式反应器中所用的催化剂在某种程度上也可以用来除去氧化亚氮。

已经用含钯  $0.5\%$  、以  $\mathrm{Al}_{2}\mathrm{O}_{3}$  作为载体的催化剂进行过试验。试验中氧化亚氮的含量以及温度都是变化的。对甲烷造成的影响也作了试验。试验气体由空气或氧气及氧化亚氮组成，在几种情况下，当空塔速度为 $2000\mathrm{h}^{-1}$  时，甲烷都穿透过催化剂，而氧化亚氮的分解则从  $180^{\circ}\mathrm{C}$  开始，在  $400\sim 600^{\circ}\mathrm{C}$  时完成。

作为比较，当氧气中含  $0.5\%$  的甲烷时，在同样的催化剂作用下，其燃烧温度是 $170\sim 340\%$

图7给出了氧化亚氮的入口含量为 $50\mathrm{ppm}$  和  $250\mathrm{ppm}$  两种情况下氧化亚氮的转化率（消耗掉的  $\mathrm{N}_2\mathrm{O}/$  入口处的  $\mathrm{N}_2\mathrm{O}$  ）与温度的关系。转化曲线呈典型的“S型”。从图7可见，转化率与入口处的浓度有密切的关系：当入口处氧化亚氮的浓度增大时，“S曲线”向温度高的方向移动。例如，当进口处氧化亚氮的浓度为  $250\mathrm{ppm}$  时，完全分解的温度要求高于  $500^{\circ}C$  ；而当进口处氧化亚

氮的浓度为  $50\mathrm{ppm}$  时，完全分解的温度约为 $400^{\circ}C$  。当存在  $0.5\%$  的甲烷时，  $\mathbf{N}_2\mathbf{O}$  的转化率将降低。

![](images/7d420f19b69a756eeb086d4215412fb83e39cfd65151f49616987d0ea11756ca.jpg)  
图7 两个不同进口浓度时  $\mathsf{N}_2\mathsf{O}$  的转化曲线

如上所述，氧化亚氮在通常使用的催化剂的作用下，可以在接触式反应器中除去一部分，但是当入口浓度较高时，仍然会有没有反应的氧化亚氮剩余物。即使进一步提高温度，也不可能将氧化亚氮完全除去，因为当温度超过  $600^{\circ}\mathrm{C}$  后，催化剂将遭到破坏。

这时，必须选择另外的空塔速度。

如氧化亚氮的转化不完全，则分子筛不仅要吸附甲烷燃烧的产物水及二氧化碳，同时还必须设计得能吸附掉少量剩余的  $\mathrm{N}_2\mathrm{O}$ 。

# 10 结论

现在，对氧化亚氮的考虑对于空分装置来说已经变得比过去更加重要了。主要原因是现在对产品的纯度要求更高，而大气中氧化亚氮的浓度又在不断增加。

将来，还要做很多工作来使更多的氧化亚氮得到转化，当然，更好的办法是用合适的吸附器将加工空气中的氧化亚氮吸附掉，并且要继续寻找更合适的催化剂。另一方面，为了有更好的实验基础，还必须有更多的物理一化学性质数据。

# 1 冷凝蒸发器的典型结构

国内外大型空分冷凝蒸发器可归纳为五种典型结构。

# 2.1 与上、下塔联接整体的冷凝蒸发器

将多个冷凝蒸发器单元在塔内按单层星形排列，或者在塔外加辅助冷凝蒸发器。如镇海石化引进的林德公司  $28000\mathrm{m}^3 /\mathrm{h}$  空分设备和宝钢引进的美国空气制品与化学品公司 $30000\mathrm{m}^3 /\mathrm{h}$  空分设备的冷凝蒸发器。这种布

置型式是最常规的一种，安装方便，管道布置简单，也较节省场地。

# 2.2 单独布置的冷凝蒸发器

将多个冷凝蒸发器单元并联置于圆筒形容器内，用管道和上下塔连接起来，为一单独布置的冷凝蒸发器。

鞍钢氧气厂1978年从日本酸素引进的 $35000\mathrm{m}^3 /\mathrm{h}$  空分设备，其精馏塔为上下塔并列，冷凝蒸发器单独布置于下塔上部，为并列两台（直径  $\times$  长度  $\times$  壁厚）  $\Phi 3500\mathrm{mm}\times$ $7300\mathrm{mm}\times 14\mathrm{mm}$  卧式圆筒形容器，每个容器内并联装有五台单元尺寸为  $900\mathrm{mm}\times$ $1066\mathrm{mm}\times 2700\mathrm{mm}$  的卧式板翅式换热器组，板翅式换热器由上下敞开的氧沸腾通道和封闭的气氮冷凝通道构成。

上海宝钢1986年投产的日本神钢 $30000\mathrm{m}^3 /\mathrm{h}$  空分设备精馏塔，也是由上塔、下塔和冷凝蒸发器三个单独部件组成，并按上塔、冷凝蒸发器、下塔顺序依次布置在同一轴线上，与传统的双级精馏塔结构型式相似，冷凝蒸发器为一台  $\Phi 4000\mathrm{mm}\times 9400\mathrm{mm}$ $\times 14\mathrm{mm}$  卧式圆筒形容器，筒内并联布置两列，每列有七台尺寸为  $1803\mathrm{mm}\times 900\mathrm{mm}\times$

1157.6mm的冷凝蒸发器单元，通道按常规布置，图1为该冷凝蒸发器结构示意图。

# 2.3 多层或双层冷凝蒸发器

将多个板翅式换热器（主冷单元）按上、下两层配置的冷凝蒸发器布置在上、下塔之间，这种结构特别适用于双级精馏。不仅可以满足在断面内有尽可能大的换热面积，并且在负荷变化范围较大时液面较能保持稳定，因而能保持在最佳工况下操作。

首钢氧气厂于1986年从德国林德公司引进带五种稀有气体全提取的  $30000\mathrm{m}^3 /\mathrm{h}$  空分设备，采用了立式双层冷凝蒸发器，如图2所示。冷凝蒸发器为  $\Phi 3850\mathrm{mm}\times 7900\mathrm{mm}\times$ $14\mathrm{mm}$  立式圆筒形容器，分两层布置，板式单元上层三台，下层四台，主冷单元尺寸 $1500\mathrm{mm}\times 1050\mathrm{mm}\times 1190\mathrm{mm}$  。双层主冷中为防止液氧中碳氢化合物在冷凝器上层局部滞留和浓缩，两层间隔板稍有倾斜，并在隔板最低部位的弓形部分开设有直径为  $\Phi 12\mathrm{mm}$  的洗涤用小孔。依靠小孔洗涤作用使单位时间有  $0.1\% \sim 1\%$  液体量直接流到下层，并在下层将液氧量的  $1 \%$  引出塔外，使之在专门装置中汽化，这种结构可避免碳氢化合物积聚而引起主冷爆炸的危险。

# 2.4 高热流管壳式冷凝蒸发器

美国联合碳化物公司从70年代起，在大型空分设备上采用高效传热管壳式冷凝蒸发器。这种冷凝蒸发器在管子内表面沸腾侧

![](images/04d0adaa5a83c8c2ea6f6e083f199213aef490a78a42ee56194b588280d29268.jpg)

有一表面多孔薄层，管子外表面冷凝侧为纵向凹槽。由于两侧相变换热都得到强化，提高了总的传热系数，减少了传热温差，降低

了设备能耗。同时因液体通过多孔表面循环速率高，能有效防止乙炔和碳氢化合物局部浓缩和积聚产生的爆炸危险，有利于设备安全运行，并成功应用于  $35000\mathrm{m}^3 /\mathrm{h}$  和  $58000\mathrm{m}^3$  /h大型空分设备上。

杭氧研究所对高热流管进行试验研究，并成功应用于  $150\mathrm{m}^3 /\mathrm{h}$  空分设备，传热温差由2.5K减小至1.2K左右，传热系数由 $600\mathrm{W / m^2*K}$  提高到  $1500\mathrm{W / m^2*K}$  ，充分显示强化传热性能，并申请了国家实用新型专利。

# 2.5 膜式冷凝蒸发器

膜式冷凝蒸发器，主冷放在液氧面之上，通过液氧泵把液氧压入主冷，与气氮一样自上而下流过主冷的相邻通道，并进行换热，使液氧蒸发，气氮冷凝。因为液氧在主冷通道内是成膜状向下流并蒸发，故称之为膜式冷凝蒸发器。由于冷凝蒸发器没有液氧柱建立起的压力差，故氧沸点不变，从而使主冷两侧的氧、氮温差始终一致。渭河化肥厂1992年引进的法液空  $40000\mathrm{m}^3 /\mathrm{h}$  空分设备为此种结构型式。

# 3 狭缝微膜热虹吸双层冷凝蒸发器

近年来狭缝强化传热理论正在逐渐受到重视。狭缝微膜热虹吸沸腾，它具有显著的强化传热效果。液体在沸腾时产生的汽泡与加热壁面之间存在着一层极薄的液体层，称之为微液膜层。C.M.Voutsikos和R.L.Judd用激光干涉法测得汽泡下的微膜厚度为微米数量级。微膜蒸发机理认为，沸腾传热主要是通过加热壁面上这一微层液膜以蒸发、对流和辐射等形式将热量传给主冷流体，其中最主要的传递方式是微层液膜的导热，传热的主要热阻即来自于这层薄液膜的导热热阻。

采用狭缝通过可以有效地削弱微膜导热热阻，显著强化传热效果。这是因为：第一，随着通道内间隙的减小，汽泡被狭缝通道两壁面挤压成为扁平汽泡，从而使微层液

膜厚度进一步减小；第二，通道内流体受热后，通道内外的流体产生密度差，在热虹吸作用下形成通道从下至上的自然循环流动（称之为热虹吸流动）。由于低温工质流体粘性系数小，流动阻力小，因而热虹吸现象十分明显，加之狭缝通道尺寸小所以通道内流速较高。可见采用狭缝通道可以减薄微层液膜厚度，显著提高传热系数，达到强化传热的目的。

杭氧集团公司与西安交通大学合作，对狭缝微膜热虹吸沸腾进行了试验研究，结果表明与常规板翅式冷凝蒸发器相比，传热系数提高近一倍，温差减少1/3。研究成果已应用于  $150\mathrm{m}^3 /\mathrm{h}$  空分设备，且运行良好。

杭氧集团公司将狭缝微膜热虹吸沸腾研究成果和美国专利（US4606745）所介绍的新型冷凝蒸发器相结合，开发了狭缝微膜双层冷凝蒸发器。这种结构型式冷凝蒸发器，壳体为单层，内部板式单元呈星形布置，将液氧蒸发腔分隔成两层，每隔一定间距开有补液槽，由溢流装置调节液面，氧氮通道按2:1排列，氧侧采用高度低节距小的平直翅片。氮气冷凝腔按常规设计。由于它结构特殊，既减小了液氧柱静压，缩小了主冷的传热温差；也由于液氧蒸发液膜减薄，放热系数得到显著提高，而且可满足在塔截面内传热面积的布置，具有结构新颖、传热温差小、能耗低的优点，已在  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备应用，必将推动我国空分技术的发展。

# 4 结论与分析

大型空分设备冷凝蒸发器结构型式，通过上述分析可以得出以下几点：

（1）单独布置冷凝蒸发器，降低了沸腾通道内静液柱对传热温差的影响和减小冷凝通道内冷凝液膜的热阻，提高了换热表面的传热效率。计算表明，当沸腾通道和冷凝通道都采用当量直径  $\mathrm{d_e} = 2.75\mathrm{mm}$  的翅片，单元高度由  $1800\mathrm{mm}$  降到  $900\mathrm{mm}$  时，生产率提

高  $8\% \sim 10\%$  。但此种结构形式由于冷凝器与上下塔分开，虽不受截面限制，但增加了设备投资费和管路配置，经济性较差。

（2）林德公司采用的双层冷凝器结构，使有限断面布置有尽可能大的换热面积，使冷凝蒸发器直径与上下塔直径大致相同，并对变负荷操作有利于获得较好的操作稳定性和可靠性，方便运输，是一种较实用的结构。其缺点是结构复杂，制造安装较困难。  
（3）沸腾侧为多孔表面、冷凝侧为纵槽的高效管壳式冷凝蒸发器，因它两侧换热系数都得到了强化，大大改善了氧、氮之间的热传导，总传热系数达到  $9000\sim 11000\mathrm{W / m^2}$  ·K，比普通管子提高了  $12\sim 16$  倍，是一种较先进的冷凝蒸发器，但其造价贵。  
（4）膜式冷凝蒸发器，由于主冷内取消了液氧柱，可以使主冷温差降至约  $0.8\%$  与浴式主冷相比减少了  $0.5\sim 0.7\%$  ，下塔顶部压力减少约  $0.2\sim 0.3\mathrm{kPa}$  ，空压机出口空气压力也相应降低，即空压机能耗可降低约

$2\% \sim 2.8\%$  ，此种结构型式的冷凝蒸发器是目前世界上较先进的结构。

（5）狭缝微膜热虹吸双层冷凝蒸发器对沸腾换热具有显著的强化作用，它的传热系数达到  $900\mathrm{W} / \mathrm{m}^2\cdot \mathrm{K}$  ，比常规板翅式冷凝蒸发器提高了1.5倍左右，传热温差下降  $40\%$ $\sim 50\%$  ，且结构简单，制造方便，是我国自行开发成功的新型冷凝蒸发器，它已在新一代大型空分设备中得到应用，将对我国空分技术发展有着重大意义。

# 3 内罐筒体的结构设计及探伤

内罐筒体的安装、施工，在整个低温罐的安装施工中占据很重要的地位，主要体现在焊丝(焊条)的消耗量、组装和检验的工作量。目前，内筒体一般采用0Cr18Ni9等高合金钢材料。纵、环向焊接接头形式采用对接焊，壁厚的选取范围大多在  $5\sim 8\mathrm{mm}$  之间，并作开坡口处理。坡口形式、尺寸，根据GB985—88和GB986—88标准或者视具体情况确定。开坡口的目的主要是为了确保焊接接头的质量，同时也要考虑其经济性。焊接接头处是

否开坡口处理或坡口型式的选择，主要取决于板厚、焊接方法和工艺过程。板厚是焊接方法选择应考虑的重要因素，好的焊接方法可使接头缺陷少，性能好，并且焊丝(焊条)能耗量少。

目前，随着安装施工水平、焊接设备性能等的提高，在保证焊接接头质量性能的前提下，就大型珠光砂堆积绝热低温液体贮罐而言，其内筒体的板厚较薄，焊接方法和工艺过程亦已成熟，故可以作不开坡口处理，可节省焊丝（焊条）用量约  $12\%$  ，见图3示意  $(\delta = 8\mathrm{mm})$  。在经济上，可以降低成本，提高效率，现在一些实力较强、技术水平较高的施工单位，即使设计单位在结构设计上开了坡口。但在施工中，根据其自有成熟的焊接方法和工艺过程，对接头已不作开坡口处理。即使在等强度设计中，筒体各带环板从底部至上部其壁厚是慢慢变薄的，带环板之间若采用中性轴布置则相互之间错边量不会超过  $1\mathrm{mm}$ $(\delta_{1} - \delta_{2}\leqslant 3\mathrm{mm}$  ，不考虑压缩环影响带环板），环向接头亦不作开坡口处理。

内筒体纵、环向焊接接头质量的好坏，大多采用X射线探伤来确定，且其检验的工程量是最大的。虽然API—620附录Q中指出内筒壁设计位置处的最大薄膜应力大于材料抗拉强度约0.1倍时，其纵焊缝采用  $100\%$  X射线探伤，而剩余的纵、环焊缝采用  $25\%$  X射线局部探伤。但是针对低温液体的特点，我公司及其它设计单位，对内筒体从质量和安全角度考虑，均采用  $100\%$  全部X射线探伤。这一点和国外空分公司比如APCI的要求相同，同时APCI公司探伤点的最小长度尺寸也有明确的要求，比API—620中规定的尺寸要长。

内筒体与底板边缘板之间的角向焊接接头其要求是非常严格的，需用真空检漏方式检查其焊接质量。同时，API—620标准中对该处的焊接接头结构设计形式亦作了具体的要求。在一些结构设计中采用了图4(A)形式，对焊缝修弧处理。很明显，其优点在于焊趾a、b，焊肉本身构成了一个圆弧过渡区，减少焊缝的应力峰值。事实上，焊趾a部位不管接头中的焊肉打弧与否，均会出现应力集中现象，是由于要达到变形一致，弯矩作用对其影响最大，该处的最大应力是径向应力，而非环向应力，径向应力是避免不了的。再加上壁板很薄，经打弧修整后使得焊肉变薄，会削弱该焊接接头的强度。

因此，笔者认为，该处的焊接接头的设计可直接采用不修弧的图4(B)形式的角向平直接头，然后对焊肉表面和焊趾处进行适当修整，以保证其平滑过渡避免尺寸变化上的突变，尽量减少应力集中，并具有足够的焊肉保证强度需要，这样来得方便实用。

# 4 底板边缘的焊接结构

内罐底板边缘板之间的焊接结构设计，有的采用无垫板的对接焊接头形式，有的则采用带垫板的单面对接焊形式。对于带垫板的单面对接焊形式，由于垫板不被去除掉，很明显在与其相交的底部角焊缝处产生应力集中现象，特别是在焊趾处。从设计的本意上讲，该处的垫板不应该保留，但由于施工条件的限制，往往对不带垫板的对接焊亦作了放置垫板处理，这实际上是施工单位对施工方案考虑不周的结果，对于有水平的施工单位，增加一道操作支架搭设的工序就可克服。如

果说，在施工或设施中一定要增加垫板的话，则垫板的厚度不超过  $3\mathrm{mm}$ ，以在焊接时不击穿垫板为前提越薄越好。

对外罐而言，如设计中有底板边缘板的话，则边缘板之间的对接焊形式可以作带垫板处理，影响不大。

# 5 外罐筒体的结构设计

外罐在整个大型低温液体贮罐中所起的作用主要用于防止松散的保温材料珠光砂的散落及内外罐夹层中充以干燥的密封氮气，使之保持正压，防止外界湿空气进入保温层而导致保温层失效，影响蒸发率。一般容量在 $200\sim 2000\mathrm{m}^3$  的低温液体贮罐其外罐筒体壁厚一般在  $5\sim 8\mathrm{mm}$  之间。外筒体各段环带板的厚度基本一样或等强度设计各壁厚环带板，其材料一般均使用碳钢。考虑到内罐充装的介质是低温液体且外罐底部容易产生腐蚀，笔者认为在最底下一段环带板和边缘板（如采用的话）材料选用耐低温钢或不锈钢为宜。这是因为：低温贮罐是放置在高台式水

泥基础平台上的，如果万一出现内罐的泄漏，可暂作安全护墙作用余留一定的时间处理突发事故。这样不致于伤及基础周围的工作人员及设备，可提高安全性。如大型低温石油气贮罐等在工程设计中均设有防液堤这样的安全措施。

上述的几处结构设计，在目前运行的大型低温液体贮罐中均使用过，其中一些建议有待于实际使用中验证。笔者从设计、现场技术服务的工作实践中小结于上，供同行参考，并望批评、指正。

# 带氧气增压器的空分流程

# 陈明敏 江楚标

（中国空分设备公司，浙江杭州市艮山门外东新路152号，310004）

在常规的增压膨胀机制冷、分子筛吸附净化的空分流程的基础上，加上一台利用主冷液氧液面高度的氧气增压器（即辅助冷凝器），使出冷箱的氧气压力达到  $1.7\sim 2.0$  bar  $(1\mathrm{bar} = 10^{5}\mathrm{Pa}$  ，下同)的空分流程，就是本文要讨论的内容。

设置氧气增压器的目的，是使用氧压力不高的用户能直接使用出空分设备的氧气，或者是降低氧气再加压时的投资和压缩功耗。80年代末、90年代初进口的首钢 $30000\mathrm{m}^3 /\mathrm{h}$  空分设备（林德公司生产）、武钢 $30000\mathrm{m}^3 /\mathrm{h}$  空分设备（2套，林德公司生产），近年进口的宝钢  $72000\mathrm{m}^3 /\mathrm{h}$  空分设备（APCI生产）和马钢  $35000\mathrm{m}^3 /\mathrm{h}$  空分设备（法液空生产）等，都采用了这种流程。

压力相应提高。液氧柱高度  $\mathrm{H}$  由主冷的液面高度  $\mathbf{H}_{1}$  及主冷液氧抽口到氧气增压器液氧进口之间的垂直液柱高度  $\mathbf{H}_{2}$  两部分组成。 $6000\mathrm{m}^3 /\mathrm{h}$  以上的大型空分设备，其主冷与氧气增压器安装位置的高度差，可使出冷箱的氧气压力提高到2bar以上。然而，利用氧气增压器所能达到的氧气压力，不仅与这个高度差有关，还必须保证氧气增压器中液氧沸腾与空气冷凝之间有必要的温差。一般来说，是这个温差制约了所能达到的氧气压力。

采用规整填料上塔后，由于上塔阻力减小，使空透排压相应降低，达到了节能的目的。但是由于空气排压降低，又要保证氧气增压器一定的温差，氧气出冷箱的压力就要降低一些。

可以看到，这种流程挖掘了空分设备的又一个潜力，即液体的位能。恰当地利用液体的位能，还可以改善某些部机的工作条件，起到改善工况、甚至简化流程、减少部机的作用。

带氧气增压器的空分设备，增加的设备和投资不多，但取得的效益很可观。当氧气终压均为31bar，不同的出空分设备冷箱氧气压力（即氧压机吸入压力）时，氧压机轴功率的对比见表1(德国德马格公司提供)。

应该指出：带氧气增压器的空分流程因为有一股约为氧产量1.3倍的加工空气变成液体进入精馏塔，使入下塔的空气量减少，因而下塔顶的上升蒸汽量减少，冷凝蒸发器的热负荷减小，从主冷送入上塔顶的液氮量也相应减少，因而会影响上塔的精馏工况，使氧、氩提取率相应降低，分离功耗增大。两者相抵，当出冷箱的氧气压力在1.75bar时，实际总功耗的降低是有限的。

出空分设备的氧气压力对节省氧透轴功率和降低投资有重要的影响，因此，在温差和位差许可的情况下，应尽可能提高氧气增压

器中的液氧沸腾压力，以提高出空分设备的氧气压力。已如上述，限止出冷箱氧气压力的主要因素是氧气增压器中液氧沸腾与空气冷凝之间的温差。因此，如果氧气增压器采用传热温差小的膜式蒸发器，出冷箱的氧气压力就能提高，氧压机的功耗及分离加压缩的总功耗就能节省得多一些。如果出冷箱的氧气压力能达到2.0bar左右，氧透还可以减少一段，氧透的造价可进一步降低。对采用填料上塔的空分设备来说，这样的组织才能得到较多的实际好处。

带氧气增压器的空分流程与常规流程的主要区别是精馏系统的组织，其它方面基本一样。在常规流程中，入塔空气基本为饱和状态或少量带液，而在带氧气增压器的空分流程中，有一股约为氧产量1.3倍的液体空气，这就使两种流程中上、下塔的精馏组织大为不同。为使带氧气增压器的空分流程有高的氩提取率，这股液体空气一般不能全部送入下塔，而且，送入下塔的那部分液体空气也不能送入塔底，而是要送到某块适当的塔板上。如果全部送入下塔，则要从下塔适当部位抽出一股数量合理、组分合适的液体送往上塔。因此，合理的精馏组织是该流程的关键所在。

# 1 概述

自90年代初开始，控制系统和计算机技术在空分过程控制中逐渐得到广泛应用。

国际上一些知名的系统制造商在空分行业展开了激烈的角逐。其中取得了较突出业绩的产品主要有：横河的  $\mu \mathrm{XL}$  系统和Honey-well的TDC-3000系统（包括MicroTDC-3000、TPS）以及S9000系列（包括R100、R150、SCAN3000）。

近几年，PLC技术飞速发展，并逐步向以往由DCS主导的过程控制领域渗透。开封空分集团有限公司在其纯氮设备及小型双高产品中使用PLC系统进行控制，成果显著，得到了较好的经济效益和社会效益，现在正致力于把PLC控制应用于  $6000\mathrm{m}^3 /\mathrm{h}$  等级空分设备上。本文介绍的是GEFanuc的90-30热备冗余控制系统。开封空分集团有限公司为九江石化总厂提供的一套KDON-300/2000型空分设备就是采用的这种系统。

空气分离设备是用低温精馏的方法，分离提取空气中的氧、氮、氩等气体的成套装

置。主要由空气过滤系统、空气压缩系统、预冷系统、纯化系统、分馏塔系统以及产品气体压缩、液体产品贮存系统等。

# 2 PLC系统方案统计

# 2.1 系统简介

系列90-30PLC是GE Fanuc系列90可编程控制器家庭中的一员。其HSB30热备冗余系统（Hot Standby Redundant System）具有低成本、高性能、配置灵活等特点，能方便取代中型自动化应用系统场合。

# 2.2 系统硬件配置

操作站采用PC机，最小配置P13332M内存，1G硬盘，要求带CD-ROM和以太网卡，声卡可选。操作系统采用WindowsNT或Windows95。两台打印机，一台作为报表打印机，一台作为报警打印。

PLC双机架采用5槽机架，CPU采用34X或35X，GBC是Genius总线控制模块（GeniusBusController)，可以采用单总线控制，也可如图1采用双总线控制。每台机架还应配置一块以太网通讯模块，以便与上位的操作站通讯。

![](images/f8526a6d650f6419897ac10b748660edba909c975bd0a60fd92a294b5e290d85.jpg)  
图1 系列90-30双总线HSB系统

GE Fanuc 的 I/O 硬件除机架式 I/O 外还有两种配置更灵活的产品——Genius I/O 和 Field Control。九江石化空分采用的是 Field Control，这种 I/O 安装在通用的接线底座上，每块底座可以插两块 I/O，每一个总线接口单元 (Bus Interface Unit) 可以支持 4 个底座。BIU 是 I/O 与 CPU 之间通讯的接口。

HSB系统采用两级通讯，上层操作站与PLC间采用标准TCP/IP协议的以太网通讯，通讯电缆采用同轴电缆或光缆。下层CPU与I/O之间为Genius总线通讯，是由连接在GBC与BIU之间的单屏蔽双绞线来实现的，传输速率153.6K波特，传送距离长达7500英尺。

# 2.3 软件功能

HSB系统需有三套软件，它们分别是CIMPLICITY监控软件、LM90-30梯形图编程软件和HSB热备冗余软件。

CIMPLICITY是操作运行软件，支持量程转换、流程画面、历史趋势、实时趋势、报警管理、数据库等功能。运行环境为Windows95或WindowsNT。

LM90编程软件对PLC进行梯形图组态，支持输入/输出转换、数学运算、PID功能、计时器、计数器、顺序逻辑控制等功能。LM90对PLC组态是通过串行通讯口，其运行环境

为DOS3.0或更高。

整个控制系统的双机架、双Genius总线热备冗余是通过HSB热备软件实现的。该软件作用于LM90软件，功能相当于一个冗余模块，其本身对编程和过程控制不起作用。

# 2.4 控制方案

根据空分工艺过程将整个流程分为几个系统单元，即空压机系统、预冷系统、纯化系统、分馏塔系统，以及产品压缩和液体贮存系统。但笔者建议  $6000\mathrm{m}^3 /\mathrm{h}$  以下空分只建立一个工程(Project)，各系统单元的流程画面、趋势图、报警管理等分别设置，采用状态和命令来控制各系统单元设备的启动、停止，以及一些多变量联锁。另外，充分利用系统的编程优势，对一些重要回路进行优化控制。

# 3 系统功能

# 3.1 数据监视

空分系统所涉及到的温度、压力、液位、阻力、流量等参数的测量，HSB系统都可以直接完成，而无需测量转换。

# 3.2 回路控制

手动遥控的模拟量输出可以直接连接到输出模块。

LM90支持两种PID算法，通过CIMPLICITY和LM90编程，可以使调节回路具有软手动和硬手动两种手动调节方法。由于自动和手动是控制同一个寄存器的输出值，因此，切换是无扰动的。

对于空压机防喘振系统的输出限幅，预冷系统液位的前馈一反馈调节系统，都可以通过HSB方便的组态功能予以实现。

# 3.3 报警和联锁

每当有报警或事故联锁，系统发出声光报警讯号，并自动登录于报警管理器，同时报警打印机打印报警信息。

系统的逻辑联锁控制由LM90梯形逻辑编程实现。

例如本装置中的两台透平膨胀机是一种高速旋转机械设备，在正常运转时的转速达 $30000\mathrm{r / min}$  。按工艺要求当膨胀机转速、或透平端轴承温度、或风机端轴承温度超过上限值(H)、或润滑油压力低于下限值(L)，系统均发出报警；如果膨胀机转速、或透平端轴承温度、或风机端轴承温度超过上上限值（HH）、或润滑油压力低于下下限值(LL)，则自动联锁停膨胀机。

另外，根据实际操作的需要，操作员亦可在操作站的显示器上进行手动操作停膨胀机或将联锁旁路使之不起作用，见图2透平膨胀机逻辑联锁控制图。

# 3.4 显示和操作功能

为了使操作人员清楚了解生产工艺和设备运行情况，有效地控制装置运行，系统设置了流程画面。流程画面能动态显示运行工况参数，显示压缩机、膨胀机、加热器等设备以及各种泵、阀等的状态。如纯化系统分子筛吸附器工作/再生状态，各单元重要控制点的历史趋势画面等，从而让操作人员清楚地了解该单元流程的运行状态，帮助操作人员判断处理问题。操作功能方面，根据生产单元状态，操作人员通过鼠标可以对系统下达相应的命令，改变设备、阀门的状态，以至于手动停车。PID回路控制，通过鼠标在流程图画面上调出PID回路控制面板画面，然后再用鼠标进行手动/自动切换、给定值设定、手操输出、参数整定等操作。操作提示功能，在某些工况，由一定的条件触发会有相应的操作提示；当鼠标操作将对当前工况产生大的影响时，画面弹出“确认”对话框，提出操作员

确认。

# 3.5 数据库和报表

系统具有开放的数据库。

通过VB语言编程，能够做到自动报表，定时打印当班日报。

# 3.6 自诊断功能

CPU具有故障自诊断功能，电源、CPU、BIU等模块对不同故障都有相应指示灯状态的提示。CPU状态(在线/离线)有内部标志位(可读)。另外，LM90软件提供PLC故障表和I/O故障表，帮助编程人员确定系统的故障。

# 3.7 动态监视

CIMPLICITY 支持动态监视功能, 上位机能通过以太网与 PLC 寄存器之间交换信息 (Read/Write), 这样就为组态提供了一个极为方便的工具。

# 4 安全性及在线组态

由于HSB系统采用双CPU热备冗余双总线通讯，这就充分保证了控制器和现场总线的可靠性。个别I/O模块或通道的故障不会引起全面停车，并且I/O模块可以带电插拨，在线组态。

CIMPLICITY软件支持动态编辑功能，在工程运行时，可以对点进行动态修改、增加或删除。

由现场的控制功能实际上是由PLC实现的，因此只要处于正常工况上位PC机死机或因在线组态的原因停止工程的运行将不影响工艺流程，这时PLC将保持工程停止前的状态。

但我们不提倡在软件注册后对组态进行大范围修改。

# 5 与其它系统通讯

空分流程设计中经常配套其它厂商产品特别是进口的压缩机，这就需要与其所配套的PLC或单板机进行通讯。系列90-30提供了通讯控制模块，支持RS-232或RS-485串

行通讯协议。九江石化总厂空分设备配置的英格索兰空压机，其单电机与GEPLC实现双向通讯，使操作人员在主控室和现场都能方便操作。

邢建芳

（杭州制氧机集团有限公司设计院，杭州市东新路90号，310004）

# 1 引言

早在80年代初，杭氧根据用户要求，陆续改造过内压缩医用氧空分设备；90年代，开发了系列产品KDON-140Y/550型内压缩医用氧空分设备；1995年底根据市场需要，开发生产了KZO-60型内压缩纯氧空分设备，现已投入市场。1996年2月杭氧与中国空分设备公司签定一套出口埃及的空分设备——KDON-150Y/20Y型全低压内压缩流程纯氧氮设备，该产品经短短八个月时间，就完成了设计制造。该产品最终于1997年底在用户现场成功地投入了运行。整套设备现运行稳定，仪表投表率  $100\%$  ，全面地实现合同要求，深得用户满意。

# 2 流程简介

本装置为低压、分子筛净化、带内压缩空分流程。原料空气经空气过滤器之后，进入往复式空气压缩机(SC1001)，将空气逐级压缩、冷却至所需要的压力，经预冷机组(RU1101)进行进一步冷却，并在水分离器(WS1203)中分离出冷凝水，然后进入分子筛纯化器(MS1201或MS1202)，将空气中的水

分、二氧化碳及乙炔等杂质除去。洁净的空气分两路，大部分空气经过滤器进入增压透平膨胀机(ET)的增压侧(B)，回收膨胀机的部分输出功，增压后的空气经水冷却器，再进入预冷机组(RU1102)进一步冷却，然后进入冷箱内，在主热交换器(E1)中被冷却至低温抽出，再经增压透平膨胀机膨胀制冷，膨胀后的低温、低压空气返回入主热交换器与正流空气直接进行热交换，复热至常温，出冷箱进入往复式空气压缩机吸入口，完成一次制冷循环。

从纯化器出来的另一部分洁净空气再分两路，一路进入冷箱内主热交换器(E1)被返流低温气体冷却并部分液化，带液的湿空气经节流阀(V1)节流进入下塔(C1)；另一路进入冷箱内高压液氧蒸发器(E2)，被返流的高压氧气冷却，再经液空空气过冷器(E3)上段冷却，经节流阀(V2)节流后与出(V1)阀的湿空气混合后进入下塔(C1)，在下塔内被分离为纯液氮和富氧液空。液空经过冷器(E3)过冷后，经节流阀(V3)节流进入上塔中部；液氮从下塔顶部抽出经过冷器(E4)过冷后，

大部分经节流阀(V4)进入上塔顶部，小部分经节流阀(V5)节流进入平衡器(PV)，然后作为液氮产品，流入液氮贮槽(SV1801)；液氧从冷凝蒸发器(K)底部抽出，经过冷器(E4)过冷之后，部分经液氧泵(OP1)或液氧泵(OP1901)压缩至  $15\mathrm{MPa}$ ，再经高压液氧蒸发

器(E2)汽化，复热至常温出冷箱，由灌充器直接充瓶供用户；部分作为液氧产品，流入液氧贮槽(SV1701)。污氮气从上塔上部吸出，经过冷器(E3)/(E4)和主热交换器(E1)再次复热至常温，作为纯化器的再生气（参见图1）。

# 杭氧新型  $150\mathrm{m}^3 /\mathrm{h}$  空分设备的特殊操作

程开荣

（362100，福建省惠安县城关中兴街大巷二支巷8号）

摘要 介绍杭氧新“150”空分设备，先降压送氧、后收阀调纯的特殊操作方法。同时对新“150”的间断制氧操作，进行了初步的探索和总结。

关键词：新型小型空分设备 先降压送氧 后收阀调纯 间断制氧

由杭州制氧机集团公司设计制造的新型 $150\mathrm{m}^3 /\mathrm{h}$  空分设备（产品代号11081B，以下称新“150”），具有操作压力低、氧气产量高（平均最高产量可达到  $190\mathrm{m}^3 /\mathrm{h}$  以上）、工作稳定、节能显著等优点。由于这套空分设备的设计具有一定的先进性和特殊性，因此在操作和使用上也有不少潜力可挖。根据本人1994年在福建石狮  $\times x$  制氧厂安装、调试的情况来看，感受最深的有以下两个方面：（1）可进行“先降压送氧、后收阀调纯”的特殊操作；（2）新“150”同样适合间断制氧。现介绍如下。

# 1 可进行“先降压送氧、后收阀调纯”的特殊操作

新“150”由于配备了两台高效气体轴承透平膨胀机(以下称透平机)，并采用铝制板翅式换热器与高效多孔管冷凝蒸发器，大大提高了气体等熵膨胀效率和换热效率。因此，空分塔的冷却阶段和积累液体阶段比较快，开机不到5小时，液氧液面就可达到 $400\mathrm{mm}$  四氯化碳柱。但由于新型冷凝蒸发器温差设计只有  $1.3^{\circ}C$  ，故收阀调纯阶段非常困难，几乎每收一次阀都要等十几分钟之后液氧液面才能恢复，而且收阀的幅度还不能太大。

鉴于此种情况，本人试图利用这套设备气量多（收阀时气量还有放空）、冷量足、换

热效果好的特点，采取先降压后再进行收阀调纯，谁知获得了意外的收获。当把高压降到  $1.5\mathrm{MPa}$  时，氧气纯度已达到  $99.8\%$  。此时，节-2阀（液空节流阀）和节-3阀（液氮节流阀）几乎一点未动，馏分阀全关，氧气流量  $80\mathrm{m}^3 /\mathrm{h}$  ，下塔压力约  $0.48\mathrm{MPa}$  ，液氧液面  $350\sim 400\mathrm{mm}$  ，一台透平机运行。将计就计开始送氧，并把氧气流量开到  $120\mathrm{m}^3 /\mathrm{h}$  然后开始收阀。结果不仅收阀容易得多，而且氧气纯度也能保持在  $99.2\%$  以上。

为什么新“150”在下塔还没有调纯之前就能送氧呢？根据本人的理解是：新“150”气量多，冷量足，冷损少，热交换器和冷凝蒸发器换热效果特别好。因此，在液氧液面保持  $350\sim 400\mathrm{mm}$  的情况下，经不断加气量和降压，下塔的压力就会逐步升至 $0.47\sim 0.48\mathrm{MPa}$ ，为冷凝蒸发器的工作创造了条件。同时也为下塔和上塔的精馏提供了回流液，精馏便开始形成。我们可以粗略地测算一下，若上塔压力保持  $0.048\mathrm{MPa}$ ，液氧纯度  $99.5\%$  ，气氮冷凝温度  $-177.65^{\circ}C$ ，冷凝蒸发器的温差为  $1.45^{\circ}C$  左右。由于节-3阀和节-2阀呈全开状态，下塔液氮纯度偏高而液空纯度偏低，但冷凝蒸发器温差偏大则有利于上塔难挥发组分的挥发，从而保证了液氧纯度。因此，虽然下塔未调纯却可以先送氧。现将主要操作方法介绍如下：

当液氧液面积累  $350\mathrm{mm}$  以上时，就可以开始加气量。其操作要领是：

（1）当高压低于  $2.0\mathrm{MPa}$  时，适当关小空压机末级高压空气放空阀，保持高压压力于  $2.0\mathrm{MPa}$ ，并保持平稳。  
（2）当透平机转速低于  $107000r / \min$  时，适当开大膨胀机的进气阀，使两台透平机都保持在最高转速允许的情况下高效率地运行，并保持平稳。  
（3）当液氧液面上升时，应及时、适当地开大节-1阀，以增加入塔空气量，稳定液氧液面。

反复进行以上操作步骤，直至高压空气放空阀全关。

（4）高压放空阀全关后，视液氧液面上升情况，继续开大节-1阀进行降压。  
（5）高压降低后，透平机转速必然随之下降，应继续开大膨胀机的进气阀，保持透平机的最高转速。当透平机的进口压力等于热交换器的高压压力时，开大进气阀不但不能保持两台透平机的转速，反而会使转速下降。此时应逐步关小其中一台透平机的进气阀，直至全关，以保证另一台透平机的高转速运行（注意不能两台平行运行）。

在不断加气量和降压的同时，下塔压力也逐步升高，精馏工况也随着下塔压力的升高而逐步建立。当高压降到  $1.5\mathrm{MPa}$ ，下塔压力升高到  $0.47\sim 0.48\mathrm{MPa}$  时，氧气纯度已达到要求，可以停止降压并开始送氧。

停止降压后，液氧液面肯定会升高。而液氧液面的升高又意味着下塔压力的下降；而下塔压力下降又会影响精馏工况。因此，这个阶段的操作要领是：用收阀（节-2阀和节-3阀）的频率和幅度来控制液氧液面的上升和下塔压力的下降。以稳定精馏工况，保证送氧纯度，直至节-2阀和节-3阀关到正常位置。当然，在送氧过程中，还要注意氧气纯度的分析，若发现氧气纯度下

降，应适当关小氧气流量，等纯度恢复后再开大流量。总之，只要耐心调整，细心操作，这种操作方法是可行的。它并不比“先收阀、后降压”的操作难，而且还可以提前几个小时送氧  $100\sim 120\mathrm{m}^3 /\mathrm{h}$

# 2 新“150”同样适合于间断制氧

据杭氧介绍，新“150”是专门为连续制氧的小厂提高节能而设计的，因此设计的最高启动压力为  $2\mathrm{MPa}$  。对于启动频繁的间断制氧来说不一定能适用。但经过本人一段时间的实际操作后认为，新“150”同样适合于间断制氧。而且比旧“150”更加适合于间断制氧。其理由是：间断后的恢复制氧操作，是在分馏塔处于低温状态下启动的，而透平膨胀机在低温下工作效率特别高，补充冷量非常快，这对于缩短间断后恢复制氧的时间很有利。至于高压启动压力低的问题，这对于原本就不需要太高进气压力的透平机来说，有  $2\mathrm{MPa}$  的高压已经足够了。

本人在石狮  $\times x$  制氧厂试机期间，每月充装氧气量只有  $4000\sim 5000$  瓶，无可奈何之下，只好安排三天开两次机。因此，间断时间比较长，一般都在20小时以上，其中间断时间最长达40多个小时。虽然间断时间这么长，但恢复制氧的时间却很快。一般都在  $30\sim 40$  分钟就可送氧  $100\sim 120\mathrm{m}^3 /\mathrm{h}$  ；60\~70分钟后可送氧  $150\sim 170\mathrm{m}^3 /\mathrm{h}$  ；90分钟后可送氧  $190\mathrm{m}^3 /\mathrm{h}$  以上。平均每瓶电耗7.5度左右（包括生产用电，全厂生活用电及线路、变压器损耗等总平均在内）。如果每月的充瓶量能更高一些，平均电耗还会降低。由此可见，新“150”完全可以适合于间断制氧的需要。下面就如何实行新“150”间断制氧操作方面，作一简单的介绍。

# 2.1 设备的安装和检查（在试机前进行）

（1）设备的安装，特别是工艺管道的布置和经常需要调整的阀门，要适应操作和间断制氧的需要。如空压机末级放空阀和总送

气阀，应与分馏塔氧、氮、馏分阀放在一起。其中空压机的末级放空阀由于调节特别频繁，且调节有效圈数又要求多一些，因此应配用闸阀，便于操作和调节。

（2）纯化器、透平膨胀机的进出口阀、切换阀和放空阀，分馏塔的高压进口阀，都不能有外漏内泄的现象。而且分馏塔高压进口阀和透平膨胀机进出口阀，要能保压24小时压力基本不下降才行。  
（3）分馏塔保温材料要填实、填满。试机后几天内要经常检查和补充保温材料。  
（4）分馏塔塔体的垂直度应按要求检查并校正好。  
（5）纯化器分子筛活化、冷吹、再生、冷却、使用，和加温的风量、压力、温度、时间均要按随机说明书上规定的程序和要求严格执行，并掌握好过渡使用期的时间。安装后第一次活化的冷却介质如果是空气，应按冷却3小时扣1小时的比例从使用周期中扣除。  
（6）透平膨胀机、预冷器机组，在试机前最好请生产厂家进一步检查和调整好。  
（7）空压机冷却水温度不能太低，一般控制在  $30\%$  为宜。  
（8）分馏塔进行彻底加温和吹除。

# 2.2 做好停机前提压和停机后保压操作

停机前的提压和停机后的保压，对于间断制氧来说是非常重要的一环。特别是间断时间比较长的情况下，更不可忽视。具体操作方法和步骤，本人在《深冷技术》1982年第2期“谈间断制氧中分馏塔的提压和保压”一文中已经详细介绍过。新“150”的操作基本上相同，这里就不再重复。需要强调的是，如果设备在间断期间无人值班，低压可以不保压。但要关紧馏分阀、送氧阀、氧放空阀、送氮阀，微开氮放空阀。

新“150”分馏塔保温效果非常好。只需按要求进行“提压”和“保压”，一般停机24小时左右，液氧液面均在  $400\mathrm{mm}$  以上，热交换器压力也在  $1.5\mathrm{MPa}$  以上。

# 2.3 恢复制氧的操作

（1）全开氮放空阀；开氧放空阀约正常流量  $60\mathrm{m}^3 /\mathrm{h}$  的开度。  
（2）开节-2阀10圈，节-3阀8圈。送气入塔前先开节-1阀45度左右回收冷量，并开动预冷机组。  
（3）送气入塔后，先启动一台透平膨胀机。待高压上升至  $2\mathrm{MPa}$  后，先后反复进行以下操作步骤：①用膨胀机进气阀，将透平机的转速逐步加到  $107000\mathrm{r / min}$ ，并稳定之；②用节-1阀寻找温-3下降最快的合适开度；③用空压机末级放空阀，控制高压  $2\mathrm{MPa}$ ，并保持之；④如果液氧液面下降速度太快，应逐步开大节-2阀和节-3阀；⑤如果设备间断时间很长（根据经验在48小时以上者），可再启动一台透平膨胀机进行补充。但不得影响另一台的高速运行。  
（4）用氧放空阀调整氧气流量  $60\sim$ $80\mathrm{m}^3 /\mathrm{h}$  ，氮放空阀如果不影响再生纯化器气量，一般都开在最大位置。让低压尽可能低些有利温度下降。

由于透平膨胀机在低温下高效率的产冷，加上新“150”独特的高换热效率，开机十几分钟，温-2、温-3就达到要求，并且液氧液面以每分钟  $1\sim 2$  格的速度开始回升， $30\sim 40$  分钟液氧液面就可达到  $300\sim 350\mathrm{mm}$  并且氧气纯度也达到要求。可以送氧并将氧气流量开大到  $100\sim 120\mathrm{m}^3 /\mathrm{h}$  。紧接着按前面所说的：“先降压、后收阀”的操作方法，继续调整至全塔正常。

（1999年9月14日收稿）

# 空分设备改造方案的选择及可行性分析

赵林祥 王好民

关键词：空气分离设备 改造方案 效益分析

# 1 概述

我国国产全低压空分设备的发展已有30多年的历史，前20年是在计划经济条件下设计制造的，属于切换流程的空分设备，主要有  $1000\mathrm{m}^3 /\mathrm{h}$  、  $1500\mathrm{m}^3 /\mathrm{h}$  、  $3200\mathrm{m}^3 /\mathrm{h}$  、 $6000\mathrm{m}^3 /\mathrm{h}$  等几种定型规格。后10年是在市场经济条件下设计制造的，属于新型的分子筛预净化流程的空分设备，产品产量规格是不定型的。至今我国共有国产及进口的切换流程空分设备数百套之多，目前有  $80\%$  仍在运行之中。

对于切换流程，不论采用蓄冷器或者切换板式型的自清除流程，由于受到当时的设计和制造水平的限制，以及操作方法、维修手段的差异，使这些已经运行多年的设备出现了这样或那样的问题。表现出效率降低，出力越来越差。造成产品产量减少及纯度降低、维修量及费用加大、能耗增加、运行周期缩短等不良后果。各项技术经济指标与技术进步后的现行设计制造的分子筛预净化流程无法相比。

在经济发展的今天，各个用户根据自身利益及生产发展的需要，对空分设备的产品产量及质量的要求不断地提高，但又受到资

金及其它方面的制约，难以增添新装置，迫切希望将现有的空分设备进行技术改造。

近年来，开空集团公司为用户成功地改造过几十套空分设备。涉及提高双高产品产量及纯度；提高单项氧或氮的产量及纯度；不带氩改为带氩；单高带第二种产品等等。

# 2 各系统技术改造方案

主要有预冷系统的改造，分馏塔系统的改造，另加一个分子筛纯化系统，其它系统为单机部分。由于用户的诸多条件不同，原有设备状况不同，其利用的范围及改造方案也不同，达到的目标亦有区别。这就有个技术改造方案的选择问题。

# 2.1 空气过滤器更新

以前切换流程配套的空压机前过滤器有很多种形式，如自动卷绕式等。用户自己也在不断改变过滤方式，实际上都不理想，存在很多问题。为了保证空压机良好的运行状态，提高过滤效率，现在一般都改进用袋式自动反吹式过滤器，可保证过滤效率达 $98\%$  以上。

# 2.2 空压机改造

由于过去设计制造的空压机是通用性的，故出现不同地区因大气条件和水质条件

不同，空压机出力不一样的情况。在考虑针对性改造设计时，如需加大气量，就应加大加宽转子叶轮；如调换隔板，可改变密封间隙，减少或消除内耗；如需改变转速，就应对原增速箱进行改造；如原中间冷却器换热效果不好，可改变水质条件，清理内部结垢，更换面积较大的芯子，另外采用新密封形式，杜绝内旁通现象。通过改造使空压机达到或超过设计铭牌的排气量及排气压力。

# 2.3 预冷系统改造

无论是哪种预冷系统的流程改造，其目的是空冷塔的出口气温达到要求值，即出预冷系统进入分子筛吸附器的空气温度达到5 $\sim 12^{\circ}C$ 。

# 2.3.1 保证空冷塔出口气温

因以前的各种切换流程对预冷系统的空冷塔出口气温只要求  $27\%$  左右，故对空冷塔的换热效果不太严格。要想降低出口气温，满足分子筛吸附的要求，对空冷塔必须改进，其方法如下：①采取增加塔板数的方法；②采取散装新型填料（共轭环）的方法；③采取规整填料的方法；④采用填料加筛板组合的方法；⑤采取原塔利用，增设第二空冷塔的方法；⑥采取原塔利用，增设塔后换热器的方法。

# 2.3.2 利用水冷塔降低水温

基于分子筛流程的特点，出冷箱的污氮呈干燥状态，因此可充分利用污氮（和部分多余纯氮）的吸湿性，造成水冷塔的负温差，直接在水冷塔中将水的温度降低到要求值。高效率的水冷塔在氧氮产量比为1:1的空分设备中，可使冷水机组的制冷量减少 $50\%$  以上。对不用氮气的钢铁厂家，可利用的氮气和污氮量大，甚至取消冷水机组。对于有条件的化工化肥厂家，如氮气使用量大，用氨换热器可取代水冷塔。用液氨作冷源，与常温水换热，即可达到冷冻水的水温要求。在有深井水或其他冷水的条件下，可

省去冷水机组，充分利用水冷塔的能力，使之达到冷冻水温要求。

# 2.4 设置纯化系统

由切换式流程改造成分子筛预净化流程必须设置纯化系统。其再生用加热器有电加热和蒸汽加热两种。

通过运行对比可知，切换时间由原切换流程的3~4分钟，改为分子筛吸附时间为4~8小时，时间的延长不仅减少了空气的切换损失，而且减少了自动阀、切换阀等易出的故障。保证了整个装置较长的运转周期及精馏工况的稳定性。同时由于  $\mathrm{H}_2\mathrm{O}$  、  $\mathrm{CO}_{2}$  、  $\mathrm{C}_2\mathrm{H}_2$  及其它碳氢化合物已被分子筛吸附清除，避免了切换板式或蓄冷器、液空吸附器、液氧吸附器等因清除效果不佳而将石头粉末、硅胶粉末、二氧化碳、乙炔等杂质带入精馏塔的可能性，保证了冷凝蒸发器及装置的安全运行。

# 2.5 分馏塔系统的改造

因原精馏设备的设计不同，虽说改造的目的一样，但改进的方式及内容有一定的差别，改进后的流程也不同。

# 2.5.1 换热设备的利用

对原蓄冷器流程改造，首先，将蓄冷器去除，增设主换热器；其次，是如何利用原过冷器、液化器，必须认真分析计算。通常采用合并、改用及新增相结合的方法，以保证新流程所需的流通面积、换热面积、液化方式、阻力损失等各方面要求。

对原切换板式流程改造时，如何利用切换板式换热器是改造的关键。理论和实践证明，用切换换热器改作为分子筛预净化流程的主换热器是完全可行的，并且可以配置增压透平膨胀机。以“3200”改造空分运行为例，热端温差可达  $2^{\circ}C$  左右。

# 2.5.2 精馏塔

上塔是精馏的主要设备，可根据改造流程性能指标的要求确定更换新塔或部分改

造。其方案有如下几种：①采用规整填料型上塔。②采用规整填料加筛板的复合塔。③采用高效强化精馏筛板塔。④上塔中部或上部改造段。

在以前的改造设计时，上述方案我公司都采用过。对下塔一般要经过认真检查后才能进行利用。

# 2.5.3 液氧循环流路

改造后的液氧循环流路是主冷底部  $\rightarrow$  液氧吸附器  $\rightarrow$  热虹吸蒸发器，然后返回主冷上部。其热源是膨胀空气，被液氧冷却后，消除了一定的过热度的膨胀空气直接进入上塔参加精馏。同时由于膨胀空气温度较高，与以前氮气作热源的热虹吸相比，热量传递增加了几倍，加大了液氧循环量。而液氧循环经吸附器时，将主冷中因分子筛常温难于吸附的某些微量碳氢化合物再清除，增强了主冷的安全性。

改造设计当然要考虑原主冷是否出现过问题，原主冷配置形式是否理想且符合新流程的要求等等。若更换主冷单元，一般采用新型的防爆结构板束，避免  $\mathsf{CnHm}$  在主冷中的积聚，消除主冷爆炸的危险。

对于分子筛预净化流程的空分设备，在流程设计时，也有不设液氧吸附器的。

# 2.6 采用增压透平膨胀机

空分设备中  $90\%$  以上的冷量是靠膨胀机的等熵膨胀得到的，而正常情况下，膨胀后的空气全部送入上塔参加精馏，因此膨胀空气流量的大小直接影响着上塔的精馏工况及提取率。如何在保证制冷量的同时又尽可能地减少进上塔的膨胀空气量，是提高运行经济指标的要点。

采用增压透平膨胀机之后，可使膨胀机的输出功转化为膨胀空气进膨胀机的势能，提高了膨胀前的压力和温度，增大了焓降，将原来占加工空气量  $20\%$  以上的膨胀量降低到占加工空气量的  $10\%$  左右，即可保证

装置的冷量平衡。改善了上塔的精馏工况，提高了产品的提取率。

如果原流程（切换板式或蓄冷器流程）中的膨胀机组运行较好，虽不增压，在考虑满足生产用气的情况下，仍可利用，这对缩短改造安装周期是有利的，并且可降低改造投资。

# 2.7 无氢制氩

生产氧、氮两种产品的空分设备，配上制氩设备，只是设备的一次性投资，运行成本略有增加。鉴于氩气市场看好，越来越多的用户希望增设制氩装置。加氢除氧制氩工艺已经落伍，原因在于氢气站及氩净化系统的操作及维护投入人力太多，已不适应用户的要求。无氢制氩工艺技术已趋于成熟，目前的成套及改造设备中广泛采用的是用规整填料三塔（粗氩塔分为两塔，精氩塔一个塔）精馏制氩。与加氢制氩相比成本投资基本相等。

最近开空集团公司又推出二塔精馏制氩，省去了精氩塔，使工艺更加简单，操作更加方便。

一般，“3200”空分设备改造配氙塔可得精氩  $70\sim 120\mathrm{m}^3 /\mathrm{h}$  ；“6000”空分设备改造配氙塔可得精氩  $200\mathrm{m}^3 /\mathrm{h}_{\circ}$

# 2.8 总体的变化

作为改造，就会对冷箱总体作相应的改造变动，尽量利用原有的冷箱壳体，梯子平台。由于设备变动，一般都需要加高冷箱，并对冷箱内部管路作相应更改。

根据改造的不同内容，确定改造设备的布置方式。有的因装置改造为带氩产品，故不论是无氢制氩或加氢制氩，若原冷箱内部无法增装氩塔，则需增加一个单独氩冷箱，与主冷箱靠近。有的因流程变化，引起产品气出冷箱位置的变动，就要对原装置外部管路、与其它装置联接管路、与厂区之间的各工程管路作较大的修改，当然相应的阀门、

设备及厂房等亦可能要改动。

随着流程的变动，仪控、电控配置内容及控制方法，依程度不同而相应变化。

# 压缩机增设变频器降低电耗提高效益

陈政忠 胡永添

（福建纺织化纤集团有限公司动力厂技术科，福建省永安，366016）

我厂两台3L-10/8型空气压缩机作为氮气压缩，一般为一开一备状态。电机功率为 $75\mathrm{kW}$ ，最高工作压力为  $0.8\mathrm{MPa}$ ，额定电流为150A，设有两个事故氮贮罐，正常时两罐充满压力后停止使用。用压缩机直接压缩常用氮，以  $0.2\sim 0.25\mathrm{MPa}$  压力送用户，每小时供氮量在  $400\sim 500\mathrm{m}^3$ ，电流在  $60\sim 80\mathrm{A}$ 。所以该机组基本上处于低负荷状态下运转，也就是形成大马拉小车状态，极大地浪费电能。为提高效益，我们采用了加装变频器进行节能改造工作，收到可观的经济效益，在此小结，以供同行参考。

我们采用日本富士公司5000G9S系列产品，型号为FRN75G9S-4CF，它的额定电流为150A，控制电机容量  $75\mathrm{kW}$ ，功能较为完善，并具有多种保护：有过流、过压、欠压、接地、熔断器断路、变频器过载、电机过载。散热板过热、变频器过热等保护。变频器跳闸后，显示故障名称及代号，给事后检查带来很大的方便。

变频器的主要作用是氮气压缩机负载较小时，通过面板控制降低输出电流频率，使电机轴输出转速减小，经变频器内部V/F电路控制，让轴输出转矩不变，可以在低转速下满足设备运行转矩的要求。在运行中，让减荷阀始终保持在全开位置，保证在低转速下，使输出流量能满足工艺要求。这种工作方式改变以往在低负荷时电机仍运转在额定转速的状况，达到降低电耗，延长机械、电机的使用寿命。

1998年9月下旬我们进行变频器安装调试，10月4日投入运行。加装变频器后，很方便地对输出电流频率进行灵活控制，在

输出压力在  $0.20\sim 0.30\mathrm{MPa}$  时调节变频器，降低输出频率，降低电机转速。根据现场运行，输出频率最低可调到  $25\mathrm{Hz}$ ，设备仍能正常工作，变频器在正常流量和压力下可保持输出频率在  $30\sim 34\mathrm{Hz}$  之间，效果显著。

加装变频器后，设备开机前减荷阀仍处于关闭状态，变频器频率预设定在  $35\mathrm{Hz}$  电机启动后，电流频率从零  $\mathrm{Hz}$  启动经过30秒钟后达到  $30\mathrm{Hz}$  。设备运转正常后，开启减荷阀至全开位置，使输出压力达到指标值 $0.2\sim 0.3\mathrm{MPa}$  。若用气量偏少时，供气压力偏高，利用变频器调低频率，降低电机转速，以减少排气量，使压缩机排气量与供气量平衡，使供气压力降低到需要值；同理，如在供气中发现压力下降到下限值时，应及时调高频率，增加排气量，使供气压力上升到需要值。以前供气压力调节用调节减荷阀开度的方法，因设备振动大，减荷阀在运行中常常会自动改变位置，使供气压力波动。

空压机增设变频器的效果：

（1）启动电流很小，不超过56A；而未装变频器前启动电流可达到750A以上。

（2）启动时，把启动变阻器的手柄放在“运行”位置，切断转子油浸变阻器，短接转子回路，采用直接启动，可减少故障率。

(3) 功率因数从 0.68 提高到 0.99。

（4）最为可观的是，电流比改造前减少 $30\sim 40\mathrm{A}$  ，年节电达  $180623.7\mathrm{kWh}$  （一年按11个月计），节约电费(0.56元  $/\mathrm{kW}\cdot \mathrm{h}$  计）101149.27元，不到一年即可收回62000元的投资。所以这种改造技术值得推广使用。

（1999年8月17日收稿）

# 一次成功的节流阀扩孔改造

方明

（安庆石化总厂化肥厂原料车间，安徽安庆市蔡山路68号，246002）

我厂第三套KZON-150/550Ⅳ型空分设备是杭氧产品，于1996年12月安装，1997年4月10日正式投产，经过20个月的实际运行，于1999年1月18日停车，针对运行中存在的问题进行检修，对原料空气节流阀VI进行扩孔改造，取得良好效果。

# 1 存在问题

该塔工况稳定，产品产量、质量均能达到设计要求，且操作压力低。但最大问题是液氧液面上涨较快，表现为分馏系统冷量过剩，即使VI阀全开，每个班(8小时)至少要排1~2次液氧(夏季)，冬天排放液氧的次数更加频繁。由于频繁的排液操作，使得V301（液氧排放阀）出现内漏，关不死，既不安全，又造成冷源浪费。

针对这种情况，在操作过程中我们也采取过措施，如节流透平膨胀机，使膨胀机产冷量下降，虽减少了排液氧的次数，但空压机末级工作压力提高了，达  $1.5\sim 1.6\mathrm{MPa}$  增加了空分设备的能耗。

# 2 改造情况

1999年1月18日，我厂请杭州宏达新技术开发公司的师傅协助对该塔进行停车检修，将冷箱内的珠光砂卸尽，除正常的消缺处理外，顺便对该塔的VI阀进行了处理。将该阀进出口管焊锡熔化后从塔内取出，抽出VI阀的阀芯，并将该阀送至机修车间对通道进行扩孔处理，由  $\Phi 8\mathrm{mm}^{\text{※}}$  扩至  $\Phi 10\mathrm{mm}$ ，后复位。

# 3 改造后效果

该塔检修改造前，尽管VI阀全开，空压机末级工作压力夏季为  $1.4\sim 1.5\mathrm{MPa}$  ，冬季为  $1.3\sim 1.4\mathrm{MPa}$  ，若膨胀机节流，空压机末级压力达  $1.5\sim 1.6\mathrm{MPa}$  。VI阀扩孔改造后，VI阀开度的变化完全能控制液氧液面的高低，不再需要通过排放液氧来调节分馏塔的冷置平衡，使分馏塔能安全稳定运行。同时空压机末级的工作压力也降低了，变为现在的  $0.9\sim 1.0\mathrm{MPa}$  ，压力下降了  $0.3\sim 0.4\mathrm{MPa}$  节能效果显著。

（1999年7月20日收稿）

# 两次空分操作事故的处理和分析

周淦 根

（铜陵有色金属公司第一冶炼厂，安徽省铜陵市金山路 244021）

摘要 简述  $1500\mathrm{m}^3 /\mathrm{h}$  空分设备纯化系统反向进水及主换热器冰堵事故、分子筛粉末进入分馏塔事故，介绍两次事故的处理经过，分析事故发生的原因和内在联系；总结了两次事故的经验教训。图2。

# 关键词：空分设备 故障 反向进水 冰堵 分子筛粉末

我厂KDON-1500/1500型制氧机，1998年3月5日发生  $2^{\#}$  分子筛吸附器反向进水及主换热器冰堵事故，9月10日发生分子筛粉末进入分馏系统的事故。两次事故都属罕见，现分析介绍如下，供同行借鉴。

# 1 分子筛吸附器反向进水及主换热器冰堵

# 1.1 反向进水的经过

1998年3月5日全厂计划进行小修，KDON-1500/1500型制氧机正常短时停车，主要清洗DA200-61型空气压缩机油站的滤油器滤网（已使用15个月）。小修任务于16：00时完成，16：15启动主油泵，16：25启动循环水泵，准备启动空压机，但主油泵启动后，轴位移油压和润滑油压力达不到要求，进而调整油压。双向调整进行了30分钟，17：10启动空压机，将空压机各工艺参数调至正常后，准备向空气冷却塔导气时，17：40发现污氮进水冷却塔压力达 $36\mathrm{kPa}$ （正常工况  $18\sim 24\mathrm{kPa}$ ），迅速将水冷却塔排水，切断预冷常温泵进口管与低温泵进口管串通阀（此DN50阀为投产后加装）。同时切断水冷却塔喷淋进水。

# 1.2 判断、检查处理经过

依据分馏塔停车状况，污氮进水冷却塔

压力达  $36\mathrm{kPa}$ ，首先判断水位达  $6.6 \sim 7m$ ，已超过污氮再生气管道，同时也超过了吸附器高度。两台电加热器已被水泡满，分子筛吸附器可能进水。

水是由加装DN50阀（停车未关，循环水泵启动后未检查）流至低温泵进口管再进水冷却塔的，而水冷塔溢流筒DN50排水管排不及（当时水压  $0.38\mathrm{MPa}$  ，时间达75分钟），水冷却塔水位不断升高，水就进入至污氮总管。

首先检查两台分子筛吸附器，打开吸附器底部排放DN25阀，  $1^{\#}$  吸附器无水，  $2^{\#}$  吸附器排出约  $12\sim 15\mathrm{kg}$  水，证实  $2^{\#}$  吸附器分子筛已被水由上而下浇淋，系再生气进口TG206阀没有关严所致。紧接着拆开两台电加热器底部出口法兰，放出大量清水。

鉴于检查出的状况，我们处理的重点放在两点：一是如何恢复电加热器使用；二是如何将  $2^{n}$  吸附器分子筛再生恢复吸附性能。

# 1.2.1 电加热器进水处理步骤

① 排除电加热器及管道内的存水。

② 手动接通  $1^{\#}$  吸附器用干燥气体对电加热器吹归，流量控制在  $2000\mathrm{m}^3 /\mathrm{h}$  左右，并检查出口气体干燥度，吹扫时间2小时。

$③$  手动控制短时间吹扫相应管道。

④解除电加热器联锁，进行单台单组送电试验。经两个多小时的处理，两台电加热器分4次送电。均无接地现象，说明能投入使用。

# 1.2.2  $2^{\#}$  吸附器分子筛再生处理

# 2 分子筛粉末进分馏塔

# 2.1 中修后空分开车失败

这种状况，使我们意识到问题的严重性，全面检查压力、流量、阻力、液面的测量数据是否准确，同时采取停氧，减少取出量降为约  $500\mathrm{m}^3 /\mathrm{h}$  ，停一台膨胀机调整三只节流阀的开度来调节塔内工况，唯一呈现的现象是主换热器各温差波动较大，在2～ $10\%$  间波动，至10日17：00时仍无好转的迹象，怀疑污氮出口测温有误，更换铂热电阻后，空气与污氮温差达  $13.7\%$  ，随即打开空气过滤器吹除阀排出大量分子筛粉末，打开进主换热器前吹除阀吹出分子筛粉末。断定分子筛粉末进入分馏塔，必须停车清除才能恢复正常运行。

# 2.2 分子筛粉末状况及清除方案

查清分子筛粉末在各部位的程度。整套机组停车后，已不能进行升温，立即打开两台吸附器装填孔，发现滤网M12四只螺母已脱落，筒体架空，滤网多处刮破，分子筛粉末积存很多，颗粒直径缩小；将空气过滤器内芯抽出，底部存有大量分子筛颗粒和粉末，滤芯内筒存有许多分子筛粉末，断开保冷箱外部空气管、污氮管、氮气管、氧气管检查，发现都有分子筛粉末附着在管壁，氮气管相对较轻一些，氧气管极微，随后打开

排放阀，同样有较多的分子筛粉末。

根据检查出的情况和主换热器温差来分析，确断整个空分塔装置都带入了分子筛粉末，且主换热器空气通道附着严重。根据掌握的状况并结合我厂富氧炼铜的主工艺，以及厂生产任务等情况，确定了两步方案：

# 2.2.1 第一步方案

① 更换分子筛；② 修复滤筒及滤芯；③ 用吸附后气体对主换热器空气通道正、反流向吹除；④ 正向吹除污氮、氮气气路；⑤ 对冷箱内所有管道进行吹除，吹除重点放在主换热器，到吹出气体清洁后，再冷开车。

# 2.2.2 第二步方案

如吹扫后开车不成功，即请专业队伍对分馏塔分割成单台进行逐一清洗，再整体联接进行干燥，检验合格后再开车。

# 3 两次事故的内在联系及教训

# 3.1 反向进水后预兆现象

回顾1998年3月5日至9月10日这段时间的种种迹象，综合分析，9月事故是3

月反向进水事故后果的延期暴露，也是必然结果；此间有多次预兆现象出现警示，未能引起警觉。

（1）在3月下旬更换TG206阀时，发现有极细浅褐色粉末。  
（2）5月8日主冷液位（正压管） $\Phi 14\mathrm{mm}\times 2\mathrm{mm}$  管堵塞，改用底部分析管，至6月12日又堵塞；曾用  $0.5\mathrm{MPa}$  瓶氧反冲没有疏通。  
（3）在6月厂小修停车后，再启动时，排放水冷却塔底部存水时，排出乳白色象淡淡的石灰水。  
（4）9月5日拆卸TG117阀修理，内壁附着一层分子筛粉末，管道下部约有  $0.5\mathrm{mm}$  厚度。

造成分子筛产生大量粉末，是被淋水后，虽没有软化，但其强度降低，不耐冲击，且在4月至9月间，因切换阀故障和人为手动切换放空对吸附器产生十余次冲击，尤其是9月4日停空压机时，放空阀打不开，机后压力降不了，便采取经纯化手动切换放空，冲出大量的分子筛粉末。

# 3.2 9月事故与3月事故的内在联系

（1）3月5日事故处理后虽开车成功，实质上  $2^{*}$  吸附器已开始产生粉末，6日又带水进主换热器，分子筛粉末随湿空气进入主换热器，在有一定的湿度条件下，分子筛粉末附着力大幅度提高（作过分子筛粉末调湿涂抹试验），加速了粉末在主换热器的附着，也为后来的吹扫增加了难度。  
(2)  $1^{*}$  吸附器在3月5日没有排出水，但不能说没有进水，只是TG205阀相对严些，进水量很少，都被分子筛所吸收，从扒出的  $1^{*}$  分子筛状况与  $2^{*}$  分子筛比较，粉末少些，颗粒稍大点，但与新分子筛相比，色泽已变，直径也缩小，同样说明被水所淋。  
（3）最关键是空气过滤器滤料不实，气流短路，为分子筛粉末的带入形成了通道。

(4) 切换阀打不开（或关不严）、泄压阀打不开、充压阀打不开等故障，加之多次人为手动切换泄压放空，造成吸附器内滤筒脱位，滤网破损，加速粉末的产生及带出。  
（5）前述预兆现象的出现，足以说明分子筛粉末进入分馏塔，但未能引起足够的重视，当时全厂任务极重，存在往后拖的思想，没有制定处理方案，9月上旬开车失败就成为必然之果。

# 3.3 教训

现在回顾分析这两次极为少见的事故，得出如下教训：

（1）在设备安装期，象空气过滤器关键设备，一定要解体检查内在质量。例如空气过滤器滤料严实，就能将分子筛粉末挡在前面，带不进主换热器和分馏塔，压力降增大，能及早暴露，后果也大大减轻。  
(2) 分子筛只要被水泡或淋后，应果断更换，以免损害扩大。  
(3) 分子筛吸附器后任何一处，发现其粉末（那怕很轻微），都应果断的停车处理，不能拖延。  
（4）对上下分馏塔残留的粉末，利用厂小修（或临时停车）时，让其液体淋集塔釜，然后排放（下塔底液空全排尽，液氧静止时排放一部分），不断地将分子筛粉末带出塔外。  
（5）通过9月事故处理，对分子筛粉末有较深的感性认识。分子筛粉末不溶于低温液体，在  $-80^{\circ}\mathrm{C}$  以下有膨胀性，在对主冷液

面（正压）管及分析管吹除时，约  $-50\%$  至 $-60\%$  时分子筛粉末吹出。在干燥度很高的状态下附着力较小，湿度越大，附着力增加，可用  $80\sim 90\%$  热水清洗，效果理想。事故发生后，进行过小型的清洗试验。

（6）对生产工艺氧的设备，清除分子筛粉末采取吹扫的手段是能够达到要求的；对生产高纯度的空分设备清除分子筛粉末，应采取清洗的方法，才能恢复性能。  
（7）分子筛吸附纯化的空分设备，切换阀的工作性能、状态和严密性极为重要，加强维护避免故障发生。当故障发生时应暂停吸附器工作，杜绝人为手动切换，防止气流对分子筛的冲击。如条件允许应投入一定资金对旧切换阀和控制部分进行更新换代。  
(8) 9月事故处理后，认真进行反思，借鉴同行有益事例，结合本套装置的特点，我们增补了12条操作技术要求，其中针对纯化系统有5条，半年多实施效果较好。

两次罕见的空分事故，事后回顾总结整理出来，以求与同行们探讨，并望能予以批评指教。

# $300\mathrm{m}^3 /\mathrm{h}$  制氧机产质量下降的分析与解决

# 单永山 满德忠 刘凤武

（辽宁渤海造船厂动力分厂技术室，辽宁葫芦岛，125004）

# 1 问题的提出

我厂氧气站  $300\mathrm{m}^3 /\mathrm{h}$  制氧机于1992年7月投入运行，但近年来分馏塔出现了漏冷，各部温度、压力不稳定，而且运行工况逐渐恶化，操作十分困难，氧气产量和质量也随之下降。产量不到  $150\mathrm{m}^3 /\mathrm{h}$  ，氧纯度也经常达不到  $99.2\%$  ，严重影响一线生产。

从外观看，分馏塔壳外大面积结冰，最厚处达到  $200\mathrm{mm}$ ，塔壳正面与东面的结合处还向外冒白色冷气，这些都是分馏塔内管路、部件漏冷所反映出的现象。经过我们查阅资料，现场调查分析认为发生这种情况的主要原因在于珍珠岩不再起保温作用、塔内管路焊口裂缝、法兰连接问题等。

# 2 现状调查

首先我们扒掉塔内结冻、潮湿、不再起保护作用的珍珠岩粉。在塔内设备修好后再更换干燥的珍珠岩粉，以使塔内冷量不损失。扒出塔内珍珠岩粉后，按检漏压力要求（上塔为  $0.05\mathrm{MPa}$ ，下塔为  $0.6\mathrm{MPa}$ ），对分馏塔内设备及管路焊口进行检查，发现下列问题：

（1）氧馏换热器上部氧气层漏氧气；(2）氧馏换热器氧气出口管漏氧气；（3）氧馏换热器下部馏分气进口管焊口漏馏分；(4）氧馏换热器上部馏分气出口连接的膨胀节裂开漏馏分气体；（5）冷凝蒸发器液氧出口管漏液氧；（6）液氧排出阀进口管焊口漏液氧；（7）膨胀机至下塔进口管漏液空。

# 3 制定对策

对于各漏点，我们没有采用已往直接在漏处增补焊锡的焊法，而采用新的方法：

（1）尽量把能拆下的管段用气焊火烤

开。如果另一端用法兰连接，就拧下螺栓，拿到塔外，重新清除焊口污物，挂好焊锡底层，再组焊。

（2）对于确实难拆出的管段或部件，采用烤开原焊口，去除原焊料，使用螺丝刀把管子或部件翻开，使焊口宽度为  $3\sim 5\mathrm{mm}$  深度为  $15\sim 20\mathrm{mm}$ ，用小型手电钻（  $\phi 6\mathrm{mm}$  ）夹上自制小圆形钢丝刷清原焊口污物，然后再进行焊接。

以上两种焊接方法，经试车和运行证明锡焊口十分牢固，并全部一次焊接成功，也不必担心再从修焊过的地方跑冷。

# 4 实施方法

# 4.1 解决法兰漏点

$300\mathrm{m}^3 /\mathrm{h}$  制氧机生产厂家为了制造的方便，多处应焊接的地方改为法兰连接。如分馏塔内  $\phi 800\mathrm{mm}$  的冷凝蒸发器与上塔、下塔的连接应焊接处现改为法兰连接。

法兰连接方式使生产厂家制造组装方便，但制氧设备在长期制氧过程中，各部件因经过多次由  $30\sim -196\%$  的热胀冷缩，法兰垫片易老化，螺栓易被拉长，使法兰连接口容易漏冷。

我们针对  $300\mathrm{m}^3 /\mathrm{h}$  制氧机分馏塔设备制造时法兰连接多的特殊情况，在查找漏点时对不易查出漏点的法兰螺栓全部进行了紧固，部分法兰垫片也进行了更新。特别是冷凝蒸发器与上塔、下塔法兰在分馏塔内是水平连接，采用常规方法，用毛刷蘸肥皂水检查，很难刷到结合处，不易发现漏点。而采用医用注射器(去掉针头)向法兰结合缝内注射肥皂水，这样易检查，易观察，好鉴定。

# 4.2 对阀门进行检修更换

对设备运行期间需经常操作的关键性阀门如：高压空气节流阀、液空节流阀、液氮节液阀，两台膨胀机进出口阀进行了检查、研究，首先更换了阀门法兰连接垫片，然后解决了阀门密封性差、传动件失控、法兰接口漏冷致使操作工不易操作等问题。

# 4.3 运行中查漏

分馏塔全部检修完毕，已往都是启动设备后先不开透平膨胀机，只进行常温找漏点。这种方法如果膨胀机进出口管法兰连接不好，运行中也容易产生漏冷。漏冷在运行中无法解决，只能停机处理，或带病运行，这就影响氧气的产量和质量。所以我们采用了裸冷检漏方法，即制氧机所有设备全部正常启动（包括膨胀机）。结果发现膨胀机进口管法兰连接处漏气、跑冷，及时修理管与换垫，最后设备漏冷问题全部解决。

# 4.4 修研和试压

为了防止制氧工在操作过程中万一失误造成分馏塔内部管路及设备超压、焊接锡口开焊、法兰垫片损坏、紧固法兰的螺栓被拉长发生漏冷等现象，我们对上塔安全阀、下塔安全阀按工艺要求进行了修研和打压校对，进一步保证了分馏塔安全和正常运行。

# 5 结论

$300\mathrm{m}^3 /\mathrm{h}$  制氧机自1998年5月12日检修好后投运至今一切正常，再也见不到分馏塔下部四周结冰和从塔缝冒白色冷气现象，分馏塔内漏冷问题得了根治。在制氧工人的精心操作下，产氧量达到了设计要求，每小时产量为  $300\mathrm{m}^3$  ，纯度为  $99.5\%$  。

$300\mathrm{m}^3 /\mathrm{h}$  制氧机经全面检修后正常运行，每天为工厂多生产的氧气就可以节约购液氧资金5000元以上，正常运行一年就可以节约150万元以上，同时全厂生产用氧气得到了进一步保证。

# 一氧化二氮的积聚和膜式冷凝蒸发器

陈智浩

（莱芜钢铁总厂动力部，山东莱芜市钢城区，271104）

摘要 介绍一氧化二氮（ $\mathrm{N}_2\mathrm{O}$ ）的特性及其在膜式冷凝蒸发器中造成的危害（堵塞设备引发爆炸），结合莱钢  $12000\mathrm{m}^3 /\mathrm{h}$  空分设备提出一些看法，主要危险仍是  $\mathrm{C}_{\mathrm{n}}\mathrm{H}_{\mathrm{m}}$  的浓缩； $\mathrm{N}_2\mathrm{O}$  在空分装置中积聚是现实的；宜用浴式主冷。图2参3

关键词：一氧化二氮  $(\mathsf{N}_2\mathsf{O})$  膜式冷凝蒸发器 危害 清除

在1999年几套大型空分设备爆炸事故发生后，引起世界上几家大的空分设备制造公司的高度重视。林德公司特别对一氧化二氮在空分设备中积聚引发膜式冷凝蒸发器爆炸的可能性做了不少工作。

过去，我们都十分重视碳氢化合物，特别是乙炔在液氧中易造成冷凝蒸发器的爆炸，分别利用分子筛吸附器、液空或液氧吸附器来清除碳氢化合物。但对一氧化二氮的清除缺乏重视。

一氧化二氮（ $\mathrm{N}_2\mathrm{O}$ ），俗称笑气。目前在大气中的含量约为310ppb（ $0.6\mathrm{mg} / \mathrm{m}^3$ ）。它的临界压力为72.7bar，临界温度为309.7K。三相点是182.3K、88bar。在水中溶解度很小。在液氧的温度、压力下，基本呈固体，其固体密度为 $320\mathrm{kg} / \mathrm{m}^3$ （78.16K）。它在液氧中的溶解度随压力和温度的升高而增大。在液氧蒸发温度下， $\mathrm{N}_2\mathrm{O}$ 固体的溶解度曲线如图1。

由图1可见，  $\mathbf{N}_2\mathbf{O}$  固体在液氧中的溶解度随蒸发温度升高而增大，但总的溶解度不

大。如液氧蒸发温度为  $95\mathrm{K}$ ， $\mathrm{N}_2\mathrm{O}$  固体在其中的摩尔浓度约为  $8 \times 10^{-5}$ ，也就是说，每吨液氧中最多可溶解110克  $\mathrm{N}_2\mathrm{O}$  固体，其多余部分将呈固体状态在液氧中悬浮。

![](images/4b3347b5f96c899ed01928d1624d64bc8e7a7456290c73b22eededa1aec0e45f.jpg)  
图1  $\mathbf{N}_2\mathbf{O}$  固体在液氧中的溶解度

我们对液氧和产品气氧分别作了  $\mathrm{N}_2\mathrm{O}$  含量的测量分析，液氧中有明显的  $\mathrm{N}_2\mathrm{O}$  。鉴于分析仪的精度为  $10^{-8}$ ，产品气氧未测出  $\mathrm{N}_2\mathrm{O}$  含量。因此可以认为进入冷箱的  $\mathrm{N}_2\mathrm{O}$  基本都积聚在液氧中。就加工空气量为  $61000\mathrm{m}^3 /\mathrm{h}$  的  $12000\mathrm{m}^3 /\mathrm{h}$  空分设备而言，如  $10\%$  的  $\mathrm{N}_2\mathrm{O}$  穿过分子筛吸附器进入装置，

每昼夜也有  $87.8\mathrm{g}$ $\mathbf{N}_2\mathbf{O}$  在液氧中聚积下来。如果没有液氧排放，经过一年运行，将有 $32\mathrm{kg}$ $\mathbf{N}_2\mathbf{O}$  积累在空分装置中，其中大部分 $\mathbf{N}_2\mathbf{O}$  不能溶解在液氧中，造成了大量  $\mathbf{N}_2\mathbf{O}$  固体的积累。固体  $\mathbf{N}_2\mathbf{O}$  除了造成装置内局部通道的堵塞外，还将对膜式冷凝蒸发器造成更大的危害。

![](images/fdef5c09f55344094a17cfc6a102691945949031fafc2ce34650bf75826254d5.jpg)  
图2膜式冷凝蒸发器液氧通道中  $\mathsf{N}_2\mathsf{O}$  析出

在膜式冷凝蒸发器板翅式通道中，沿壁面向下流动的液氧迅速蒸发。由于种种原因，每个氧通道上的液氧膜并不是很均匀的。在液氧膜很薄的通道处， $\mathrm{N}_2\mathrm{O}$  会附着在壁面上，并累积起来，长期运行就可能堵塞该液氧通道（见图2，示意），使该通道液氧不能流动。通道上部液氧不断蒸发、补充、再蒸发，使该通道中碳氢化合物浓度不断提高。众所周知，浓聚的碳氢化合物在沸腾的液氧中引爆的危险性是很大的。

我厂林德  $12000\mathrm{m}^3 /\mathrm{h}$  空分装置（概况见《深冷技术》1998年第5期）的主冷和辅冷都是膜式冷凝蒸发器。在正常工况下，进主冷液氧设计流量为  $66460\mathrm{m}^3 /\mathrm{h}$  ，进辅冷液氧流量为  $30010\mathrm{m}^3 /\mathrm{h}$  ，除辅冷蒸发  $12000\mathrm{m}^3 /\mathrm{h}$  气氧作为产品外，其余气态氧或液态氧都返回低压塔(即上塔)底部，其中  $26580\mathrm{m}^3 /\mathrm{h}$  气氧作为低压塔上升蒸汽参与精馏。也就是说，经过冷凝蒸发器的液氧约  $40\%$  被蒸发， $60\%$  仅仅穿过通道，未被蒸发。这部分未蒸发的液氧，对通道壁面有冲刷作用，可以减

少  $\mathrm{N}_2\mathrm{O}$  的析出及其在壁面上的附着。因此，林德公司曾建议我厂增大液氧量。但液氧量增大就提高了板式上部液面，造成液氧蒸发压力提高，减少了板式的热负荷，影响产量或上塔液气比。进一步提高液面，可能堵塞液氧进口和气氧出口管道，工况更加恶化。

以  $12000\mathrm{m}^3 /\mathrm{h}$  空分主冷为例，板式顶端到液氧进口管中心线仅  $885\mathrm{mm}$  ，到气氧出口管管缘也仅  $1560\mathrm{mm}$  。因此，设计操作液面高度为  $522\sim 700\mathrm{mm}$  。我们实际操作在 $650\mathrm{mm}$  左右，工况很好。我们试验加大循环液氧量，将液面提高到  $730\mathrm{mm}$  以上，就发现冷凝蒸发器热负荷下降，工况有恶化趋势，只能恢复原工况运行。

综上所述，保持冷凝蒸发器有足够的循环液氧量，是十分重要的。否则，在膜式主冷或辅冷长达  $6\mathrm{m}$  的换热通道中，下部就可能出现干涸区或接近干涸，使氧通道迅速被析出的  $\mathrm{N}_2\mathrm{O}$  固体堵塞，造成碳氢化合物在液氧中浓聚，引发爆炸。但过量的液氧，会使膜式冷凝蒸发器液氧蒸发压力提高，降低了热负荷，丧失或削弱了膜式冷凝蒸发器在空分装置中应用的优势。

据林德公司专家介绍，活性氧化铝和13X分子筛的双层床吸附器，可以部分吸附空气中的  $\mathrm{N}_2\mathrm{O}$  ，但仍有  $10\% \sim 15\%$  的  $\mathrm{N}_2\mathrm{O}$  可穿透分子筛吸附层进入冷箱，浓聚在液氧中。如采用内压缩流程，大量的液氧夹带 $\mathrm{N}_2\mathrm{O}$  离开空分，则可保证安全。对非内压缩流程，如我厂  $12000\mathrm{m}^3 /\mathrm{h}$  机组，虽然实际生产近  $300\mathrm{m}^3 /\mathrm{h}$  液氧，冷凝蒸发器中液氧循环倍率达2.5，但仍会有固态  $\mathrm{N}_2\mathrm{O}$  析出。在林德公司派专人分析的情况下，停机用循环液氧泵对主、辅冷用液氧进行冲洗。在冲洗开始时，液氧中  $\mathrm{N}_2\mathrm{O}$  含量为  $1.5\times 10^{-6}$  ，2小时后就达  $18\times 10^{-6}$  ，7小时后到  $32\times 10^{-6}$  ，然后不再上升。可见，我厂主、辅冷中也有

固态  $\mathrm{N}_2\mathrm{O}$  随冲洗液氧带出来。因此林德专家提出，该套空分装置仍具有爆炸危险，建议对其进行改造，增加液氧循环吸附器，使部分循环液氧经过吸附，其吸附量与进装置量平衡，以避免  $\mathrm{N}_2\mathrm{O}$  浓聚。在改造完毕前，每三个月停车用液氧冲洗主、辅冷。我们认为原设计不可靠，林德公司应负责改造，但林德公司认为双方合同已期满。由于双方意见不一致，改造工作仍未进行。

大气中微量  $\mathbf{N}_2\mathbf{O}$  对空分装置长期安全运行的危害，是过去未受重视的问题。现各大公司开始重视这个问题，并提出了一些见解，值得我们在空分设计、运行中借鉴。概括起来，有以下几点看法：

（1）空分装置中爆炸的主要危险仍是碳氢化合物的浓聚。应进一步加强对它的控制和监督。但仍有难以监测的部位，可能在局部浓聚，引发爆炸。  
(2）大气中微量  $\mathbf{N}_2\mathbf{O}$  在空分装置中聚积是现实的，它可能造成设备的堵塞。在膜

式冷凝蒸发器氧侧的堵塞可能引发板式的爆炸。这种情况是很难监测的，只能在系统解冻过程中测到。

（3）空分装置最好仍采用传统的浴式冷凝蒸发器，因为它的循环倍率高，杂质聚积可能性低，且有长期安全运行的实践。对于膜式冷凝蒸发器，在内压缩流程或大量生产液氧的设备中，还是可以安全运行的；否则，必须加装液氧循环吸附器，以清除  $\mathrm{N}_2\mathrm{O}$  和碳氢化合物。

# 徐州二氧化碳气瓶爆炸事故分析

（徐州气体厂技术开发科，江苏徐州市北郊霸王山前，221007）

# 1 事故概况

1999年5月13日上午10点，徐州铸造总厂收到徐州市金陵气体供应站（个体性质）送来的15只充满的二氧化碳气瓶（由江苏华扬液碳有限责任公司泰兴二氧化碳厂充装），直接卸在厂区露天仓库内。下午13点20分，其中一只气瓶发生爆炸，瞬间产生的冲击波将15只气瓶全部推倒，其中一只气瓶向南射出52米，碰撞在机修车间的铁门上，受阻落地后瓶肩螺纹损坏变形，气瓶泄压后旋转进入车间，冲坏地坪，打翻工作台。一只气瓶向西约  $15^{\circ}$  角射出43米，撞在两墙上落地，瓶头损坏，瓶身有凹点。另外三只气瓶，一只被气浪冲倒，瓶头歪斜、瓶力受损，保险断裂泄压；另一只气瓶向西南方向约  $10^{\circ}$  角射出11.3米，瓶阀折断，泄压；一只气瓶向西南方向约  $30^{\circ}$  角射出15米，翻越1.2米短墙落地。此时现场一片白雾，飞砂走石。气浪将附近2米砖墙推倒。距爆炸点10米处的二层楼机房门窗玻璃全部震碎。紧靠爆炸点4米处有平房一间，工人在房内午休，所幸的是平房外停放一辆3吨铲车，缓解了爆炸所产生的冲击波，使平房没有倒塌。爆炸点现场，也因午休未有工人上班（该厂13:30上班），避免了人员伤亡。当时爆炸声大而沉闷，数里可闻其声。

# 2 爆炸事故分析

# 2.1 爆炸瓶壁厚检测

爆炸气瓶残体落在距爆炸点西南方向5.6米处块石结构墙根下，瓶体撕裂展开。

首先对爆炸钢瓶从轴向、径向进行全方

位密集点测厚。检测表明，壁厚最小 $(4.9\mathrm{mm})$  处位于钢瓶腰部，即瓶口向下 $500\mathrm{mm}$  处的断口边缘。最小壁厚处在同一横截面上，断口边缘处的壁厚由于爆炸瞬间的拉伸作用而明显减薄。检测最小壁厚  $4.9\mathrm{mm}$  平均壁厚  $6.2\mathrm{mm}$  ，减薄率  $[1 - (4.9 / 6.2)]\times$ $100\% = 21\%$  。

# 2.2 断口形态分析

① 破裂的断口呈撕裂状，断口呈暗灰色纤维状，断口不齐平，且与主应力方向呈 $45^{\circ}$ 夹角。

② 爆炸瓶体没有产生碎片。

③气瓶内、外无明显腐蚀。

④ 实际测厚均大于钢瓶标准壁厚。

# 2.3 化学成分分析

经光谱分析表明，爆炸气瓶的材质基本符合《规程》要求。

# 3 对爆炸性质的认定

爆炸钢瓶的编号：02265；水压测验：TP18.8；工作压力：WP12.5；重量：W45.1；容积：41.3；壁厚：S4.5；钢瓶制造厂代号：JP（北京天海工业有限公司）；制造年月：1997年2月；制造检验标记：（检）。钢瓶标注字样，清晰、明显、完整。经查询确定该瓶材质为碳锰钢，钢号37Mn2A。

从以上情况及有关数据分析，爆炸时气瓶没有产生碎片，瓶体也没有应力集中的地方，爆炸的原因很明显是压力较高、超压造成的，其根据如下：

（1）原始钢印很清楚的标明，该瓶工作压力WP12.5，我国1979年4月25日公布第

二版《气瓶安全监察规程》即去消了设计压力  $12.5\mathrm{MPa}$  规格的二氧化碳气瓶，因此该气瓶的流通使用就埋下了事故的隐患。

（2）原国家劳动总局1979年5月8日下达（79）劳锅字31号文《关于加强二氧化碳气瓶安全管理的通知》，重申自1980年1月1日起原设计压力为  $12.5\mathrm{MPa}$  的二氧化碳气瓶规格去消，制造厂不得销售，充装单位也不准再以压力降为  $12.5\mathrm{MPa}$  充二氧化碳。从这次事故可见，《通知》精神没有引起足够的重视。  
（3）按上述文件精神和《气瓶安全监察规程》规定，此类气瓶不能再流通使用，更不得按公称工作压力为  $15\mathrm{MPa}$  二氧化碳气瓶规定的充装系数  $0.6\mathrm{kg / h}$  进行充装，否则将严重超压。这次气瓶爆炸事故显然严重违反了《规定》要求，爆炸就不可避免。

# 4 问题和教训

（1）既然法规和标准已经明确规定，压力为  $12.5\mathrm{MPa}$  的气瓶已不准生产销售，但从该瓶生产日期：“1997年2月”看，仍在市场上销售流通，估计此批气瓶不止一只。要加强检查力度，从制造、销售、充装、使用层层把关，彻底杜绝不合格的气瓶继续流通。

（2）加强法规、标准的学习和宣传，有关单位除学习必要的标准外，还要掌握领会新的规定和要求。这次爆炸的责任单位、销售与使用单位，均缺乏对法规标准的学习，对新的规定要求一片空白，意识淡薄，增加了事故的概率。  
（3）严格执行气瓶充装前的检验工作，是保证安全生产、防止事故发生的重要环节。这次爆炸事故反映出一些充装单位基础工作薄弱，质保体系运转不正常，气瓶超期服役，大循环、大流通使用的状况很普遍。  
（4）近年来气体供应站大量增加，有不少是个体户，其经销人员缺乏一定的培训和必要的安全知识教育，部分气体供应站条件简陋，气瓶混存，管理混乱。在这次事故调查中，因充装站无记录，无法确定气瓶的来源及产权单位，事故调查未能划上圆满的句号。  
（5）临夏季节，气温偏高，充装和使用单位一定要按照《气瓶安全监督规程》第九章的要求，在运输、储存和使用中加强安全管理，防止曝晒，妥善保管，才能避免事故的发生。这次瓶爆，与临夏5月露天曝晒3个多小时，膨胀升压有关。

（1999年6月29日收稿）

# 制氧工问题解答（四则）

1：问：请问空分设备增加鼓风机有增产降耗的作用吗？

答：这个问题值得一谈，因为增加鼓风机还有争论。最近我看到两份资料，一是美国福特公司在汽车系列（为9座、12座、17座位的面包车）中装上了涡轮机，登广告说“装涡轮机的增压发动机赋予小汽车更强劲的马力”（见《南方周末》报）。另一份是60年代杭氧研究所在一次学术会议资料上一篇关于增加鼓风机（即涡轮机）可以增加产量的报道，时隔三十余载，为什么这一技术在我国得不到推广应用？概括地说，是时代背景和措施的关系之故，撇开前者不说，仅就措施的经济性谈一谈。

（1）增产——增加空气量会提高氧产量或氮产量是众所周知的（1999年还有单位询问纯氮产量会不会提高），当时鼓风机排压为  $0.02 \sim 0.03\mathrm{MPa}$  （不论叶氏或罗氏鼓风机），按原空压机气量所要求的鼓风机排压都在  $0.02\mathrm{MPa}$  左右，市场上有鼓风机者气量或多或少不能符合要求，因此提高的产量意义不大，且鼓风机往往要大量放空。因为活塞式压缩机属容积式压缩，我们所需的风机排量  $\mathbf{V} = (1.2 \sim 1.3) \times$  （空压机排量）再加  $5\%$  的调节余量，排压为  $0.035\mathrm{MPa}$  最为理想。风机与制氧是两个行业，市上购买到的是系列产品，不能为制氧机专门设计，制氧机厂也不感兴趣，一搁就是  $20 \sim 30$  年。前两年笔者在回答读者问的短文中说过“ $150\mathrm{m}^3/\mathrm{h}$  制氧机加鼓风机之后，产量可以增到  $200\mathrm{m}^3/\mathrm{h}$ ”，就引起了不少用户的兴趣，来信询问这种鼓风机是哪里买的。笔者想这种专用鼓风机自从增压透平膨胀机的流程问世之后，现在有办法可自行设计了，而且可

购到国外设计软件，可以自己制造了。

（2）降耗——活塞式压缩机的机前增设鼓风机，等于原压缩机中增加一级压缩，从设计热力学原理来说，多级压缩机就是为了节省单位能耗而施行的，所以降耗也是理所当然。问题在于究竟能降低多少？由于鼓风机排压只有  $0.02\sim 0.03\mathrm{MPa}$  （G），老实说能耗方面油水不大。要说节能，在于它有调节功能，可以利用昼夜供电差价的好外，这个差价可以给用户得到很大回报。不可小看，这称之为“调峰型设计”。

（3）适用范围广——据了解，目前中小型制氧机面临的市场情况各不相同，以下几种类型可以适用，范围广阔：①市场萎缩采取间断生产的单位；②市场看好的单位；③原压缩机气量不足，如压缩机老是皮带打滑，重庆有一家说风量标  $860\mathrm{m}^3 /\mathrm{h}$  ，实际达不到  $860\mathrm{m}^3 /\mathrm{h}$  ；④需要调负荷的单位。所以应用范围广，只要是活塞式空压机， $3200\mathrm{m}^3 /\mathrm{h}$  以下的空分设备都可以考虑采用，增加鼓风机来提高纯氧纯氮的生产量。

2问：KDON-400Y/400Y型低温设备投产一年多，每次启动膨胀机后15分钟到半小时，当机后温度到达  $-60^{\circ}\mathrm{C}$  上下时，出现机后背压升高，需停机再生，请问原因在哪里？（西昌卫星发射中心氮氧站问）

答：本流程中有两台空压机和增压透平膨胀机，构成独立的制冷环路，每台压缩机都可以从大气中吸气，也可以从平衡罐吸气，由两只蝶阀控制。

平衡罐属关键部件之一，围绕平衡罐有五只蝶阀，操作一不小心，可能有少量含湿空气从大气中经蝶阀漏入压缩机，经换热

器，膨胀机降低了温度即以雪花状态再次导入换热器，把换热器翅片部分堵塞形成阻力，背压升高，直至堵死。导致湿空气吸入，原因是平衡罐和管道被抽真空，加上蝶阀有微漏造成的。为此，必须要求在动态中保持平衡罐处于正压（如果没有指示就要加一只表），问题便可迎刃而解。再说开车十多次只出现五次，这也可以说明是操作不细心，平衡罐出现负压引起的。

# 3问：活塞式低温液体泵在安全使用上有哪些要求？

答：活塞式低温液体泵广泛应用于液体汽化站，使用中应注意下列要求：

（1）凡是与氧接触的零件、管道、阀门等必须经严格的去油。  
（2）低温液体泵传动部门的油面应经常检查，减速箱的最低油面不得低于大齿轮齿顶，脱离油面不得低于测油杆上最低位置。  
（3）低温液体泵装修内部零件时，工作环境应无灰尘，所用的工具、工作服必须干净，清洗好的零件要用干氮吹燥，防止受潮和灰尘污染。所有管口用白布封住，否则易发生活塞、汽缸拉毛。  
（4）低温液体泵在未预冷彻底前不得启动，不得在无流动介质的情况下运转。  
（5）低温液体泵工作时的最高工作压力不得超过允许最高工作压力。  
(6) 低温液体泵因故障需拆开时，应先用热空气或氮气加热至常温后方可解体。  
（7）低温低温液体泵安装时宜比贮槽液面低  $2\sim 3m$  ，进液管离泵越近越好，以于减少汽化。进口压力不得大于0.2MPa，防止进口波纹管变形和爆破。  
（8）低温液体泵的进出管道中前后有截止阀的状态下，在两阀间各装上一只放空阀、安全阀，防止误操作而超压。  
（8）低温液体泵停止工作后应将立即关

闭贮槽送液阀，排净管道和泵内的液体，防止因温度升高造成液体汽化而超压。

# 4. 问：低温液体汽化器在使用中应注意哪些安全问题？

答：液氧、液氮、液氩等低温液体汽化器广泛应用于液体汽化站，直接供氧或充瓶。为了保证汽化器安全运行，应设置安全控制点，现将有关注意事项介绍如下：

（1）液化气体出汽化器的低温控制联锁点。气体出口温度控制在  $5\sim 30^{\circ}\mathrm{C}$  ，当出口温度低于或等于  $0^{\circ}\mathrm{C}$  时，会自动切断液体泵，终止液体进入汽化器。不带液体泵的设备，温度低于  $0^{\circ}\mathrm{C}$  时能声光报警。  
(2）汽化器水温控制联锁点。汽化器水温应在  $40\sim 60^{\circ}\mathrm{C}$  ，当水温低于或等于  $30^{\circ}\mathrm{C}$  时会自动切断液体泵，终止低温液体进入汽化器。  
（3）液化气体出口压力控制联锁点。根据用户的不同要求，设置合理的气体出口压力设定值，当出口气体压力高于设定值时，会发出声光报警，若继续升高会自动切断液体泵，终止液体进入汽化器。  
（4）液体管中两头设有截止阀的部位装有安全阀、放空阀，防止误操作时的安全。  
（5）汽化器配套的压力表、安全阀，应定期校验。  
（6）用水浴加热的汽化器使用前必须先将水槽的水充满，并加热到  $40\sim 60^{\circ}C$  才能输入液体。而停止汽化器前，先切断输液阀，然后再切断加热电源。汽化器工作中，应经常注意水位，及时补充水量。  
（7）工作中由于流量的改变，会影响汽化后的温度，所以要及时调整水温。  
（8）若发生水温下降  $30\%$  ，应检查电热管是否损坏，必要时应减少输出流量，确保汽化后的气温。汽化器至充装管道发现结冰或结霜时，应停止充装。

# 美开始研制可大幅降低制氧成本的分离膜

在美国能源部联邦能源技术中心的资助下，空气产品与化学品公司的一个研究组1998年10月开始实施一项为期3年、耗资2500万美元的研究计划，目标是将由空气制氧的生产成本削减  $1 / 3$  。这一空气分离技术的基础是采用一类“混合传导”陶瓷膜（  $\mathrm{A}_2\mathrm{B}_2\mathrm{O}_5$  ），A和B位是合理取代的阳离子。膜结构中有排列有序的空隙，从而使膜对氧有  $100\%$  的选择渗透性。

在此技术中，空气流过加热到  $900^{\circ}\mathrm{C}$  的膜表面。氧分子被吸附到热激发的膜上，然后被电离。离子化的氧渗透到膜的另一侧（因存在电位梯度），重新变成  $100\%$  纯的氧气流。释出的电子透过膜回到进气侧重新进行电离。这一科研计划分三步实施：首先试验  $0.1\mathrm{t} / \mathrm{d}$  实验室小试规模的膜组件；然后在模拟工业条件下试验两套  $1\mathrm{t} / \mathrm{d}$  和  $5\mathrm{t} / \mathrm{d}$  规模的装置，以取得设计、工程制造、成本和放大技术的数据；最后将建设  $25\sim 50\mathrm{t} / \mathrm{d}$  规模的“原理验证”设备。
