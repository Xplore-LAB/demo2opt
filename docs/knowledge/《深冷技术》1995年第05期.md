# 一、前言

天然气是一种重要能源，它的燃烧热值高，对环境的污染少，被称为“清洁能源”。随着世界能源需求的增长和天然气技术的进步，天然气在世界能源（石油、煤炭、天然气、核电和水电）结构中的比例明显增长（见表1）。有可能成为下一个时期的主要能源。

天然气的主要组分是甲烷，其沸点为 $-162^{\circ}C$  ，不能在常温下加压液化。因而一般只能铺设管线，以气态形式将天然气从气井输送到遥远的使用地点。传统的高压输送法存在投资高、风险大等缺点，而且不可能用于远洋输送。利用低温技术将天然气液化，以利于贮存、运输和应用，是一项在国外广

泛应用的先进技术。

液化天然气(LNG)的体积只有同量气态体积的1/625，因而将其液化之后，一是有利于降低贮存和运输成本；二是可以提高单位体积的燃值(达到相当于汽油的水平)，用于发电和交通运输工具(汽车、船舶和飞机)的燃料。此外，液化天然气还可用于城市调峰装置，减轻城市高峰能源的紧张状态。国际天然气市场对LNG的需求正在不断上升。我国如能实现液化天然气出口，在地理上处于非常有利的地位，其经济效益将十分明显。

国外天然气  $90\%$  左右是当作能源作用的，约占能源结构的  $20\%$  。我国是天然气资源大国，但长期以来，勘探、开发和利用的水平远低于世界平均水平。我国天然气约有 $40\%$  作为化工原料，其余作为燃料，其值仅占总能源消耗的  $2\%$  （表2），到现在为止，已开发的天然气放空烧掉的仍占相当比例，这是很不合理的。

# 二、国外LNG技术水平及发展趋势

根据国际能源机构的调查报告，鉴于石油资源日趋枯竭，而天然气的应用方兴未艾，天然气的地位将不断上升，加速取代石油。1992年底，全球探明天然气储量为124万亿立方米，可供开采58年，是21世纪能源供给最可靠的保证。而且，世界上还有大量的天然气储量有待进一步探明。

前苏联天然气储量为45.3万亿立方米，居世界第一位，其天然气储量的  $60\%$  集中在俄罗斯。美国天然气储量为4.7万亿立方米，约为前苏联的1/10，居世界第5位，然而其天然气产量长年居世界前列。堪称世界耗能大户的美国，正在增加天然气在能耗中的比重。过去5年，美国对天然气的需求量增加了  $13.5\%$  ，天然气在能源结构中已达  $23.9\%$  预计到2000年可达  $24.5\%$  ，2010年达  $25.5\%$  。

欧洲国家消耗天然气持续增长，预计年消耗量平均增长  $3.1\%$  。金属、食品、纺织和

木材加工等工业，已大规模以天然气取代石油和煤为主要燃料。

亚太地区天然气消费量在能源结构中的比重，将从1989年的  $7.1\%$  增至2000年的 $20\%$  ，2020年预计可达  $25\%$  。2010年，亚洲发展中国家(地区)的天然气消耗在能源结构中的比例，估计将从目前的  $24\%$  增至  $37\%$  东盟尤为突出，泰国1993年天然气消耗增长 $18\%$  ，马来西亚电力工业计划在1997年使天然气在燃料中的比重增至  $65\%$  。预计今后20年里亚太地区用于液化天然气项目的投资总额将达600亿美元。

中东向亚太地区（日本、韩国、台湾）的LNG出口，将从1992年的250万吨增长到2010年的2630万吨，中东LNG出口的增长，反映了全世界对天然气需求量的增加。

当今，国际天然气销售量的  $30\%$  是以液化天然气的形式进行的。近20年来世界上的LNG产量一直以年平均  $23\%$  的速度递增，从而改变了整个气体工业的结构。多年来LNG设备的投资已超过500亿美元，年利润在100亿美元以上。象印尼、泰国、马来西亚等国以天然气作为支柱工业，使经济发生腾飞。

液化天然气非常有利于远途运输，它主要应用于工业和民用燃料，城市天然气调峰、LNG燃料汽车、中小型移动式柴油一天然气发电机组、船舶、飞机等。从经济角度来看，发电厂使用天然气发电和燃煤发电成本持平。从环保角度来看，天然气燃烧排放的二氧化碳比煤少  $40\%$  ，比石油少  $25\%$  ，因而可节省电厂的环保投资。

实践证明，采用LNG作发动机燃料，发动机仅需作适当变动，运行又安全可靠，而且噪声低，污染小，特别是在排放法规日益严格的当今，以LNG为燃料的汽车，排放气明显改善，尤其是致癌的碳烟微粒。在美国，大约有3万辆车是使用天然气作为燃料的。美国天然气协会预计，到2010年，可能会增至1200多万辆天然气汽车，从而每年可以取代60万桶进口石油。美国福特汽车公司和美国天然气研究所耗资5000万美元研制的天然气汽车在美国和加拿大进行了三年行车试验。美国休斯顿城市运输局计划在  $5\sim 6$  年时间内，在城市干线上利用1600辆用LNG驱动的公共汽车。意大利菲亚特公司和斯纳姆公司合作研制的5辆天然气公共汽车在拉文纳市已运行了一年，现在在试制天然气轿车。德国柏林市最近开始使用由天然气驱动的大型公共汽车，建立了天然气充气站。据调查，目前全世界大约有80万辆天然气车辆在试运行。

随着80年代天然气价格的降低和涡轮机的改进，燃气涡轮机在绝大部分混合循环发电厂得到应用。这种电厂把燃气涡轮机产生的剩余热量用于带动蒸汽涡轮机，从而提高了发电效率。混合循环发电厂的效率，80年代末达到了  $40\%$  以上，较之常规采用的兰金循环蒸汽涡轮发电机，这种技术把矿物燃料的化学能转化成电力的效率，一直停留在  $33\%$  。可见，采用天然气的联合循环发电，突破了低效率发电技术的障碍。

以天然气作为涡轮机和发电机动力还有助于提高热电的混合使用，发电过程中产生的废热，可供工厂生产和居民取暖使用。小型“混合”型发电厂，如今在丹麦和北欧地区已属常见。

同常规的燃油和燃煤发电厂相比，燃气发电厂在环保方面占有很大优势，它不排放硫，有害微粒的排放微不足道，一氧化氮的排放量降低  $90\%$  ，二氧化碳的排放量降低 $60\%$  。成本低和公害小这两大特点，使天然气自90年代初在英美两国新建电厂的市场占据主导地位，南亚、远东和拉丁美洲也为天然气的使用展现出广阔的市场。

燃气发电技术的发展可能促使公用部门以每千瓦仅300美元的价格，把数百个老化的燃煤电厂改造成燃气混合循环发电厂。据美国通用电气公司预测，到2005年，全世界燃气电厂的发电能力可能达到40万兆瓦。奥地利、埃及、日本、尼日利亚等国建造的燃气电厂已投入运营。

天然气液化装置按用途可分为基本负荷型与高峰负荷型两种。基本负荷型工厂系指生产供当地使用或外运的大型液化装置。高峰负荷型工厂系指为调节高峰负荷或补充冬季燃料供应的天然气液化装置，通常将低峰负荷时过剩的天然气液化贮存，在高峰时再汽化使用。经过几十年的实践，这些装置的工艺和制造都已达到很高的水平[1]。目前的设计改进，集中于考虑经济性、适应性和灵活性，采用节能工艺流程，提高设备模块化程度，附设气体冷量回收装置，采用高效信息管理系统，增加装置的可靠性和工作寿命，应用新型结构材料和绝热材料等。如英国Castain Petrocarbon公司为澳大利亚制造的一套分离高碳组分的集装式高效回收装置，采用“Petroflux”新流程，获得了比传统的膨胀机流程节能  $60\%$  的效果[6]。在液化天然气过程中放走的不凝性气体中，往往含

有较高浓度的氦气，如果加以提取，可大幅度降低成本。最近报道，波兰Olonova的一个天然气精练工厂装置了氦回收系统，使天然气分离工艺达到新的水平[8][10]。

液化天然气贮罐的设计和制造技术发展较快，地面LNG贮罐的单体容积已从60年代的2万立方米左右增大到目前的24万立方米或更大[2]。近年建造地下LNG贮库的技术发展迅速，例如，日本已经开始建造地下LNG贮库的有：东邦煤气、西部煤气、静冈煤气和仙台煤气等公司。地下贮库的特点有：LNG贮存在地表以下，安全性高；不必在地面建设防液堤等；在地上只露出半圆形顶盖，环境较协调。东京煤气公司1993年在袖浦工厂建成14万立方米的LNG地下贮库，总工程费用为160亿日元。日本自1969年开始从阿拉斯加进口LNG，1990年扩大到6个国家，进口量为3600万吨。预计在2000年将进口4300万吨，到2010年可达5400万吨。法国Botas公司为土耳其承建LNG出口码头，其工程包括3个8.5万立方米的LNG贮罐及汽化系统、带停泊设施的长300米栈桥（可停泊12.5万立方米的LNG船）、24英寸输送管道及配套设施，总投资为2.178亿美元。

由于LNG贮罐的容积达几千立方米以至20多万立方米，其内容器从室温冷却到低温  $(-162^{\circ}C)$  或从低温复温到室温的热胀冷缩会产生较大的热应力，要求有严格的工艺和技术来保证。近年国外发展的薄膜贮槽设计和制造法，被称为无应力贮槽，这使大型LNG船用贮槽的制造技术达到了新的水平。

天然气的另一个用途是作为化工原料，其中最大产品为合成氨和甲醇。天然气经低温分离所得的丙烷、丁烷及其它轻烃组分，可用于生产工业乙烯、丙烯、芳烃等重要化工产品。

此外，LNG汽化时的冷量利用或回收技术，在国外也受到广泛重视，每  $100\mathrm{kg}$  LNG

在常压下汽化时可提供30万千卡的冷量，具有重要的应用价值，由此可开展多项综合利用工程。例如，回收能量专用于从空气中分离氧、氮、氩，从炼气中分离乙烯，以及用于冷藏、空调、食品冷冻、干燥、低温粉碎、海水淡化、冷却大容量电缆等。

在天然气井中气源压力较高的场合，采用膨胀机循环，可以获得高效率和低成本。这时可直接利用高压原料气通过透平膨胀机制冷。膨胀机循环具有操作维护要求低、可靠性高、气源利用率高和对气源组分的变化不敏感等优点。然而，当在低压气井采用膨胀机循环时，需用燃气轮机来驱动压缩机，燃气机要求提供大约  $16\%$  的原料气，就是说只有  $84\%$  的原料气用于液化。

在混合制冷剂循环中，天然气的预冷与液化所需的制冷量是由一个在大约3.0 MPa下工作的混合制冷剂循环系统提供的。典型

# 四、我国的天然气工业现状及对策

我国有丰富的天然气资源，是世界天然气大国之一。已探明的天然气储量近1万亿立方米，据预测，我国的天然气储量可能高达33.36万亿立方米。我国1990年的天然气产量为145亿立方米，并逐年增加。我国最大的天然气生产基地四川省，1993年天然气产量达67.96亿立方米，比上年增长  $4.5\%$  ，创历史纪录。近年在陕北发现中国陆上迄今最大的连片气田，含气面积达2340多平方公里，陕甘宁盆田贮量1200亿立方米。中国海洋石油总公司计划到1997年生产天然气40亿立方米，本世纪末探明储量4000亿立方米，形成年产200亿立方米的生产能力。总之，前景美好，发展的时机已经到来。

但是，我国在天然气利用方面目前仅停留在液化石油气(LPG)生产及近距离管道输送上，至今还没有一座LNG工厂，LNG技术处于十分落后的状态。近年刚刚起步的我国液化天然气研究和开发工作，基本上处于无规则、不协调的分散局面[1~4]，由于人力、财力和物力的限制，步履维难。对LNG工业链（包括天然气的预处理、液化、贮存、运输和应用等环节的技术与装备）缺乏基础研究，对国外LNG技术缺少系统了解，以至对天然气的液化、贮运及应用技术的研究和发展十分缓慢。由于缺乏对LNG工业的理论和技术研究，对流程的组织与设备的配套，处于相当低的水平。一些重要的关键设备，如全带液工作的透平膨胀机以及大型薄膜贮罐的材料与工艺，尚未进行过研究试验。至于综合利用和自动化管理的水平，则与国外相差更远。所有这些，都还远不能适应我国大规模开发利用天然气的要求。

我国天然气资源丰富，能源紧缺，环保问题愈来愈受到重视，这就决定了LNG工业在

我国具有广阔的应用前景。至今已探明的天然气田遍布我国东南西北和近海，让这些丰富的资源早日为我国的工业和民用服务，例如用于城市燃气、发电、化工原料以至出口，这是时代的责任。

目前在东北、陕西、四川、新疆有不少县级所有的小型气田，急需首先开发，以满足农村能源的迫切需要。但由于这些小气田较分散、地形复杂，难以布设管道，宜采用LNG技术，以利于运输和利用。在我国开发小型LNG工厂的技术潜力巨大，我国有数十年制造成套空分装置的经验，有充分的设备配套能力和技术设计力量。主要的困难在于一次性投资大，一般一座10万  $\mathrm{m}^3 /\mathrm{d}$  以上气体处理量的LNG工厂需数千万元以上，需要国家支持。国家应有相应的政策以鼓励LNG的开发和应用。这样，国家支持，地方出土地和原料气，再加上广泛集资、引进外资，完全可以促使小型LNG工业的发展，并为发展大型LNG工业积累经验。

由于国外LNG技术的高度发展，各种大型LNG工厂的机器设备及仪表均能配套，因此，在开发大型LNG工厂时，引进国外先进设备和技术，进行消化吸收；或引进外资，共同投资，是一条重要的发展途径。显然，利用国外资金和已成熟的技术，可加速我国LNG工业的开发和发展，也是加速我国LNG工业现代化的必由之路。

实际上，国外产品价格太高（见本文前一节），难以成套引进。因此，引进关键设备技术是一种可行的方法，例如引进两相透平膨胀机。因为攻关与技术改造相结合是完全可行的，应将目前国内制造空分装置及低温贮罐的有关工厂与高校、科研单位的力量组织起来，共同开发LNG技术，使我国的能源供给水平走上新台阶。

除了天然气的液化和贮运之外，在应用方面，特别是利用天然气和LNG作发动机

燃料技术、天然气混合循环发电技术，以及液化天然气的综合利用技术等的研究开发，也是不容忽视的问题。

# 液空蒸发作冷源时其参数的确定

阎振贵

（杭州制氧机集团公司板式换热器厂，杭州东新路90号，310004）

缩要 本文对单塔以液空为冷源时蒸发冷凝的设计参数——液空节流后的组分、液空蒸发时的组分、节流后的液空蒸发时组分、循环倍率进行了理论探讨，提出了自己的观点，以达到正确地计算传热温差或蒸发压力的目的。并针对该观点进行了讨论。图1。

# 主题词：精馏液空蒸发组分节流循环倍率

单级精馏废气膨胀制取纯氮的流程中，其冷凝蒸发器的冷源及粗氩塔冷凝器的冷源，通常都采用液空蒸发来获得。由于液空节流后其组分要发生变化，在蒸发过程中汽液二相的组分相差又较大，而此组分差又和蒸发过程的温差、几何尺寸、液面高度等因素有关。因此在设计计算其传热温差时，如忽略了上述因素，往往会造成很大的计算误差。而在纯氮设备中，此温差决定了蒸发压力即膨胀机的机前压力，而此压力又影响了膨胀机的产冷量。故从某种意义上讲，传热温差的计算误差会影响流程的成败。由此可知，如何正确地确定液空的组分，从而正确地计算传热温差或蒸发压力，显得非常重要。本文针对这一问题进行了理论探讨。

# 一、液空节流后的组分确定

用液态空气作冷源的冷凝蒸发器，一般都需把中压的液空节流到一个适当的压力，利用节流后剩余的液体空气的蒸发潜热作冷源，把高温的另一侧工质冷却液化下来。因此节流后的液空组分及蒸发压力是首先要确定的。

令液空节流前的流量为  $R$  ，压力为  $P_{R}$

组分为  $x_{R}, (y_{R})$  焓值为  $i_{R}$  （设置液空过冷器的流程为过冷后的焓值），节流后的状态参数如图所示。

![](images/9e055165f257e1c6eae9da65f838714e91bd5b2f6429b604a1287498a1e076e6.jpg)  
图节流后的状态参数

物料平衡：

$$
R = V _ {1} ^ {\prime \prime} \div V _ {1} ^ {\prime} \tag {1}
$$

组分平衡：

$$
R \cdot x _ {R} = V _ {1} ^ {\prime \prime} \cdot y _ {1} + V _ {1} ^ {\prime} \cdot x _ {1} \tag {2}
$$

能量平衡（设冷损等于零）：

$$
R \cdot i _ {R} = V _ {1} ^ {\prime \prime} \cdot i _ {1} ^ {\prime \prime} + V _ {1} ^ {\prime} \cdot i _ {1} ^ {\prime} \tag {3}
$$

由(1)得  $V_{1}^{\prime} = R - V_{1}^{\prime \prime}$  (4)

(4)代入(2)得  $R \cdot x_{R} = V_{1}'' \cdot y_{1} + (R - V_{1}'')x_{1}$

化简得  $V_{1}^{\prime \prime}(y_{1} - x_{1}) = R(x_{R} - x_{1})$

$$
\therefore \quad \frac {V ^ {\prime \prime}}{R} = \frac {x _ {R} - x _ {1}}{y _ {1} - x _ {1}} \tag {5}
$$

(4)代入(3)得  $R\cdot \pmb {i}_R = V_1^{\prime \prime}\cdot \pmb {i}_1^{\prime \prime} + (R - V_1^{\prime \prime})\pmb {i}_1^{\prime}$

化简得  $V_{1}^{\prime \prime}(\pmb{i}_{1}^{\prime \prime} - \pmb{i}_{1}^{\prime}) = R(\pmb{i}_{R} - \pmb{i}_{1}^{\prime})$

$$
\therefore \quad \frac {V _ {1} ^ {\prime \prime}}{R} = \frac {i _ {R} - i _ {1} ^ {\prime}}{i _ {1} ^ {\prime \prime} - i _ {1} ^ {\prime}} \tag {6}
$$

$$
(5) = (6) \quad V _ {1} ^ {\prime \prime} = \frac {x _ {R} - x _ {1}}{y _ {1} - x _ {1}} = \frac {\mathbf {i} _ {R} - \mathbf {i} _ {1} ^ {\prime}}{\mathbf {i} _ {1} ^ {\prime \prime} - \mathbf {i} _ {1}}
$$

通常把节流时产生的汽化量  $V_{1}''$  和液空量  $R$  的比值称作节流汽化率(或节流汽化系数)，用  $\alpha$  表示。

则  $\alpha = V_{1}^{\prime \prime} / R$  (7)

$$
\alpha = \frac {x _ {R} - x _ {1}}{y _ {1} - x _ {1}} = \frac {\dot {i} _ {R} - \dot {i} _ {1} ^ {\prime}}{\dot {i} _ {1} ^ {\prime \prime} - \dot {i} _ {1} ^ {\prime}} \tag {8}
$$

由(8)式可见：节流汽化系数与液空节流前的组分  $x_{R}$  、焓值  $\pmb{i}_{R}$  和节流后的状态 $(T_{1},P_{1})$  有关。

现有一种求取液空节流汽化系数  $\alpha$  的方法。它可以在已知节流前的参数  $(i_R, x_R, P_R)$  的情况下，求得任何节流后的汽化系数及其他参数。现举例说明。

例1，求  $x_{R} = 35\% (\mathrm{O}_{2} + \mathrm{Ar}), P_{R} = 6$  ata饱和状态的液空节流到  $P_{1} = 1.5$  ata时其节流汽化系数及其他参数。

利用此法得到节流后的状态，  $\alpha = 0.165$  即饱和液空的节流汽化系数。

例2，求  $x_{R} = 35\% (\mathrm{O}_{2} + \mathrm{Ar})$ ， $P_{R} = 6$  ata；过冷8K的液空节流到  $P_{1} = 1.5$  ata时的节流汽化系数。

求解得节流后的状态，  $\alpha = 0.088$

这种方法可以避免目前采用的选取  $\alpha$  数值的方法而造成的误差。这个误差直接影响了液相组分的精度，从而影响了传热温差的计算值。利用这种方法同样可以计算纯液氮节流汽化系数或污液氮节流汽化系数。

利用这种方法可以求取某一种组分的液空其节流汽化系数等于零时的最小的过冷度值。如上述例题中的液空，要使  $\alpha = 0$  ，则过冷度  $\Delta t \geqslant 16.3\mathrm{K}$  。同样利用这一方法可以反求欲使汽化系数等于某一个值的过冷度。该方法以后介绍。

液空的节流汽化系数确定以后，其节流后产生的气相、液相数量及组分则完全可以计算了。

# 二、液空蒸发时的组分平衡

节流后的液空（图中  $V_{1}^{\prime}, x_{1}$ ）在蒸发时和其他物流的关系如下：

物料平衡：  $R = V_{5} + V_{6}$  （9）

$$
V _ {5} = V _ {1} ^ {\prime \prime} + V _ {3} \tag {10}
$$

组分平衡：  $R\cdot x_{R} = V_{5}\cdot y_{5} + V_{6}\cdot x_{6}$  (11)

$$
V _ {5} \cdot y _ {5} = V _ {1} ^ {\prime \prime} \cdot y _ {1} + V _ {3} \cdot y _ {3} \tag {12}
$$

1. 液空回流量  $V_{0} = 0$  时的工况

在单级精馏废气膨胀制取纯氮的流程中，冷凝蒸发器中的液空没有回流的可能，因此其值为零。此时的关系式简化为：

$$
R = V _ {5} \tag {13}
$$

$$
\mathcal {Y} _ {5} = R \cdot x _ {R} / V _ {5} = R \cdot x _ {R} / R = x _ {R} \tag {14}
$$

$$
V _ {3} = V _ {5} - V _ {1} ^ {\prime \prime} = R - \alpha \cdot R = R (1 - \alpha) \tag {15}
$$

$$
\begin{array}{l} V _ {3} \cdot y _ {3} = V _ {6} \cdot y _ {6} - V _ {1} ^ {\prime \prime} \cdot y _ {1} \\ = R \cdot x _ {R} - \alpha R y _ {1} = R \left(x _ {R} - \alpha y _ {1}\right) \\ \end{array}
$$

$$
y _ {3} = \frac {R \left(x _ {R} - a y _ {1}\right)}{R (1 - a)} = \frac {x _ {R} - a y _ {1}}{1 - a} \tag {16}
$$

由此可见：在无液空回流的情况下，液空蒸发量  $V_{3}$  及组分  $\pmb{y}_{3}$  ，在液空节流时的前后参数已知的情况下，是一个定值。它仅仅和节流汽化系数有关。而液空汽化的总量  $V_{0}$  及组分  $\pmb{y}_{6}$  ，是和液空节流前的量、组分相等的。当液空过冷到一定程度后，节流汽化系数  $\alpha = 0$  ，此时可得：  $V_{1}^{\prime \prime} = 0; y_{3} = y_{6} = x_{R}; V_{3} = V_{6} = R$  。使蒸发量  $V_{3}$  达最高值  $R$  ，蒸发时产生的气体组分  $\pmb{y}_{3}$  达最小值  $x_{R}$  。而  $\pmb{y}_{3}$  对应的相平衡组分  $\pmb{x}_{3}$  达最低值，即蒸发温度达

最低值，从而使换热温差达最高值。在换热面一定时，蒸发压力可达最高值。

因此在单塔纯氮设备中，提高液空的过冷度，是提高膨胀机进口压力的一个有效措施。同时还可以看出，当  $V_{6} = 0$  时， $y_{3} = (x_{R} - ay_{1}) / (1 - \alpha)$ ，在  $\alpha$  确定后， $y_{3}$  为一定值，其对应的  $x_{3}$  也为一定值。从而可以推算其循环倍率  $\pmb{n}$  与蒸发温度无关。

如何设计和操作，使上述诸定值实现，是装置稳定运行的关键所在。

# 2. 液空有回流时的工况

由式(9)(10)(11)(12)可得

$$
V _ {3} = R (1 - \alpha) - V _ {6} \tag {17}
$$

$$
y _ {3} = \frac {R \left(x _ {R} - \alpha y _ {1}\right) - V _ {6} \cdot x _ {6}}{R (1 - \alpha) - V _ {6}} \tag {18}
$$

$$
V _ {5} = R - V _ {6} \tag {19}
$$

$$
y _ {8} = \frac {R \cdot x _ {R} - V _ {6} \cdot x _ {6}}{R - V _ {6}} \tag {20}
$$

由这些方程可以看出：在有液空回流的蒸发器中，液空蒸发量  $V_{3}$  及其组分  $\pmb{y}_{3}$ ，不仅和节流汽化系数  $\alpha$  有关，而且和液空回流的量  $V_{6}$  及组分  $x_{0}$  密切相关。即这种工况下，蒸发温度和液空的排放量密切相关。而  $y_{3}$  和  $x_{0}$  的关系式为：

$$
n \cdot x _ {6} = y _ {3} + (n - 1) x _ {3} \tag {21}
$$

式中  $x_{3}$  —— $y_{3}$  对应的平衡液相组分

$n$  ——循环倍率  $[n =$  蒸发器入口流量/被蒸发的量  $(V_{3})]$

由式(21)可见：液空蒸发时的组分  $y_{3}$ ，除了与节流汽化系数  $\alpha$ ，液空回流量  $V_{6}$  及其组分  $x_{6}$  有关外，还与蒸发时的流动情况  $n$  有关。而蒸发器的总气量  $V_{6}$  及其组分  $y_{5}$ ，仅和液空的回流量  $V_{6}$  及其组分  $x_{6}$  有关，与其他参数无关。而决定蒸发器工况参数的是  $V_{3}$  及  $y_{3}$ 。因此必须确定液空蒸发时的组分  $y_{3}$  和此组分对应的液空浓度  $x_{3}$ 。

# 三、节流后的液空蒸发时组分的确定

下塔的液空经节流后，部分汽化。未汽

化的液空量为  $V_{1}^{\prime}$  。由物料平衡可得

$$
V _ {1} ^ {\prime} = V _ {3} + V _ {6} = R (1 - \alpha) \tag {22}
$$

组分平衡：  $x_{1}\cdot V_{1}^{\prime} = V_{3}\cdot y_{3} + V_{6}\cdot x_{6}$  (23)

而液空回流量  $V_{0}$  是为了蒸发时的安全而采取的措施。不同流程中，其值相差较大。如在用分子筛纯化空气的空分流程中，此值一般取液空量的  $2\%$  左右；在用切换式换热器冻结纯化空气的流程中，此值一般取液空量的  $10\%$  左右。排放时的组分  $x_{0}$ ，则是和进入蒸发器的液空的组分相同的，要受很多因素影响，现分析如下。

首先，为了蒸发器的安全，不仅要排放部分液空  $V_{0}$  ，而且要保证蒸发过程中在换热器的蒸发表面上不发生干蒸发的现象。一般，为了蒸发  $V_{3} \mathrm{~m}^{3} / \mathrm{h}$  的液空，要有  $n$  倍蒸汽量的液空量  $n \cdot V_{3}$  进入蒸发器蒸发，从而使在蒸发器出口处，有  $(n - 1) \cdot V_{3}$  的液空量回流到蒸发的“大池”中。此  $n$  称做循环倍率。循环液空量  $(n - 1) V_{3}$  和节流后未汽化的液空量  $V_{1}$  混合即为组分  $x_{8}$  的液空，符合方程(21)的关系。

当流程确定后，可以取一个  $V_{6}$  值。  $V_{6}$  已知后  $V_{3}$  可以确定了。剩下两个方程有三个未知数  $y_{3},n,x_{6}$  。要解此方程可以从分析方程(21)开始。由(21)可以看出：当  $\pmb{n}$  足够大时，因  $y_{3}$  是一个小于1的数，所以  $x_{3}\approx$ $x_{6};$  而  $x_{3},y_{3}$  为相平衡关系。因此当  $x_{3}\approx x_{0}$  时，可以求得一个  $y_{3}$  值及  $\pmb{n}$  值。然后以  $\pmb{n}$  为基础求出几何参数及  $\Delta H$  等，从而再修正  $x_{6}$  及  $y_{3}$  值及其他相关参数。

# 四、循环倍率的确定

目前在设计计算中，循环倍率  $n$  值一般是选取一个值，如林德公司推荐在粗氩塔冷凝器中取  $n = 7$  。但如何保证此值的实现，在实际运行中有没有达到，都是未知数。实际上这种做法是盲目的。在单塔的冷凝蒸发器中，循环倍率是多少，取多少为好，也是未知数。因此必须对循环倍率  $n$  和与其有关

的因素加以分析。现以板翅式换热器作冷凝蒸发器时的情况为例进行分析。

$$
\text {热 负 荷} Q = r \cdot V _ {3}, \mathrm {k c a l / h} \tag {24}
$$

式中  $r$  ——液空蒸发潜热，  $\mathrm{kcal / kg}$

$$
\text {面} \quad Q = K \cdot A _ {m} \cdot \Delta T _ {m}, \mathrm {k c a l / h} \tag {25}
$$

# 五、几个可讨论的问题

1. 液空操作液面的高低，对整个换热器的影响

当液空回流量  $V_{\theta}$  不变时， $V_{3}, V_{5}$  为定值，此时  $y_{3}, y_{5}$  的值取决于回流液空的组分  $x_{6}$ ，其关系即方程(21)。而方程(21)取决于循环倍率  $n$  的数值。当操作液面  $\Delta H$  增加时，意味着  $\Delta P$  增加，而此时  $V_{3}$  已定即  $G_{0}$  一定，阻力增加即由  $G$  的增加引起的，而  $G = G_{0} / i, i = 1 / (n - 1)$ ，这表明阻力增加的原因为  $i$  下降和  $n$  增加。从物理意义上看也很容易理解， $\Delta P$  增大由  $n$  增大引起。反之亦然。而  $n$  增大后，由(21)可见， $x_{6}$  向  $x_{3}$  接近，而  $x_{3}$  大于  $x_{6}$  （蒸发后氧浓度增加），则  $n$  上升后  $x_{6}$  上升，液体蒸发温度上升，温差减少，热负荷减少。结论：液面上升，热负荷减少。

反之则不一定如此。液面降到一定程度后  $n$  值下降。此极限值即设计计算的  $\Delta H$  值。在此时  $n = (x_{3} - y_{3}) / (x_{3} - x_{8})$  ，在蒸发压力确定后， $x_{3} - y_{3} = \mathrm{const}$  。因此  $n$  值和  $x_{8}$  成正向发展， $n$  值增加，意味着  $x_{8}$  增加。使  $x_{8}$  更接近  $x_{3}$ ，即  $x_{8}$  含氧量增加，蒸发温度上升，温差减小。而当  $n$  值下降时， $x_{8}$  下降，温差加大。仅从此角度出发即可得出： $n$  值越小越好，即当  $n = 1$  时（干蒸发）， $x_{0} = y_{3}$ ，达最小值，此时蒸发温差最大。要实现  $n = 1$ ，只能  $\Delta P = 0$ 、 $\Delta H = 0$ ，即在没有液面高度时，要实现全换热表面的浸润蒸发才能实现。这种结构的蒸发器目前已进入商品阶段。此时为了安全起见，可以采用其他方法，如加大液空的回流量等措施。加大  $V_{6}$  后可使  $y_{3}$  下降，加大温差。

2. 提高液空的过冷度，降低节流汽化系数，提高蒸发压力，并保持换热正常

在没有液空回流的情况下，由方程(13)、(14)、(15)、(16)可以看出  $V_{6}, y_{5}, V_{3}, y_{3}$  都为定值，其对应的  $x_{3}$  亦为定值，这意味着蒸发温度是个定值。而在有液空回流时， $y^{3} = f(x_{0}, V_{0})$  是变值。由此可知：在无液空回

流时，循环倍率和蒸发温度无关。在这种情况下，循环倍率是唯一的安全措施，因此可以加大循环量，即提高操作液面。进入蒸发器的液空组分，当  $n$  加大后，远离节流后的液空组分  $x_{1}$ ； $n$  减小后，接近  $x_{1}$ ；当  $n = 1$  时， $x_{m} = x_{1}$ 。无论  $n$  值变化多大，除了  $\Delta H$  对蒸发压力有影响外，其余没有影响。

因此，要提高蒸发压力，并能维持换热正常，唯一的措施即提高液空的过冷度，使节流汽化系数下降。

3. 加大液空回流量，也是常用的调节手段之一

由方程(18)可以推算出：当  $V_{0}$  加大后， $y_{3}$  减小，对应的蒸发温度下降，故  $\Delta T_{m}$  加大，但回流的液空组分  $x_{0}$  减小。在同等条件下，加大了上塔下部氧氩分馏段的负荷，这样，会影响氧气的纯度。如果氧气纯度偏高，此时只需加大液空回流量  $V_{0}$ ，则氩塔负荷增加，粗氩产量自然上升，或纯度自然提高，这也是经常采用的调节手段之一。

# 4. 等组分蒸发

由上分析可知：单级精馏废气膨胀的纯氮设备中，其冷凝蒸发器的液空蒸发，虽然 $V_{6} = R,y_{5} = x_{R}$  ，但决非等组分蒸发。实实在在的蒸发组分  $y_{3}$  不等于  $\pmb{x}_{\pmb{R}}$  。同样，它也不是相平衡蒸发，因进入蒸发器的液空组分  $x_{m}$  和  $y_{3}$  没关系，它与循环倍率  $\pmb{n}$  成反比。而蒸发的组分  $y_{3}$  仅仅取决于节流汽化系数  $\alpha$  ，见方程(16)。从总体上讲：在这种冷凝蒸发器中，不管循环倍率如何变化，其出蒸发器的总气相组分  $y_{6}$  和量  $V_{6}$  ，等于液空节流前的组分  $x_{R}$  及量  $R$  。从这一层意义上讲，可以称为等组分蒸发过程。

（1）因为主换热器只起低温产品气体和环境温度原料气热交换作用，故其冷损和压力降损失可做到尽可能低。  
(2) 因为所需交换的热量可以减少到最小, 所以换热器的换热面积也做到最小。

(3) 强化空分, 以减少设备尺寸和费用。

（4）氮气能被直接利用，而水需经过处理后才能使用。

这种型式的综合主要是与分享制冷系统联系在一起。最初综合例子是空气分离与液化过程相综合。早期的空分装置一般都生产液氧产品，且最普遍使用的流程(图7)能有效地把分离和液化过程综合起来。这种流程需要把原料空气压缩到高压(至200巴)，随后除去空气中的杂质进入换热器，冷却到  $200\sim 240\mathrm{K}$ ，部分空气送去膨胀机膨胀，空气压力降至高压精馏塔压力（6巴），进入高压塔，其余的空气继续被返流废氮冷却，然后通过节流阀闪蒸进入高压塔。

空分装置需要制冷，以平衡热量的渗入、换热器的热端损失和所有液态产品所需的冷量。这些冷量通常由部分空气从高压塔压力膨胀到低压塔压力中得到，膨胀后空气可在低压塔中处理。如果不需要氩气，氧气纯度不必过高，就可考虑利用膨胀后的那部分气体，获得较高的氧提取率。在这种情况下，能够获得比通常所需更多的冷量，以很低的成本（电耗和投资费用），就能满足空分所需冷量。

# 四、与非深冷流程综合

虽然，当初研究空分与在环境温度以上的过程没有多大的联系，但事实上，用氧气和生产氧气过程的结合是很有综合价值、并值得考虑的领域。首先，空气分离是一个消耗电能较大的过程，而用氧气通常是高放热过程。这些热量能被用于产生可观的蒸汽，然后在蒸汽透平中膨胀，以提供部分或全部空分所需能量。在用氧过程中可能有一个或

几个废热源可被利用，这些过程还产生能放出大量热量的煤气，这大部分燃料能通过燃气透平发电。蒸汽或燃气透平的直接与空分装置的空压机连在一起，这样，电动机和发电机的费用和损耗可被消除。这也许是最简单的能源综合形式。

从技术的观点看，更有趣的是把空分过程综合进整个流程中的各种可能性，借此提高氧气生产和氧气的使用过程的效率。例如可共用某种设备，如冷却水系统、制冷系统和空气压缩机。这有两种基本方法，象前面提到过和废热利用的综合和通过透平膨胀机发电。共用空压机、空分和燃气透平综合的基本例子是联合循环煤气化发电站。

（2）燃气透平空压机比传统空分装置所需压力高得多，所以运行负荷被强化，冷箱内大部分设备的尺寸可减小。  
（3）空分压力比通常高，意味着在循环中没有低压力，所以大大减少压力降损失。原料气和产品气体之间的压比是输入过程中能量多少的尺度。如果原料空气压力增至11巴，压比从典型低压流程压比接近6，降低至约3.5。因此，象一个简单氧化反应器一样，介质压力过程有较显著的高效率和在广泛的环境过程中能构化出有价值的组合。

被描述与过程的其余部分相互综合的流程还具有另一个优点，就是燃气透平空压机可在没有中间冷却的情况下正常运行，所以其电耗要低于常规空分装置。从透平放出空气所含有的热量，在整个过程中得到有效地利用是很重要的。这些热量也许可有效地用于预热从空分装置生产出来的氧气和氮气。

（1）氮/合成气热交换器取代废热锅炉，因此热量的品质没有任何程度的降低，而有废热锅炉的情况，热交换是以显热与潜热型式进行的，热量的品质将有所降低。

（3）简单的氮气循环发电系统取代蒸汽循环设备的发电系统，可减少许多设备资金。例如，不需要蒸汽锅炉、冷凝液排出泵和锅炉原料水的气液分离器等与蒸汽循环系统有关的设备。

# 五、结论

在不同行业中有大量工艺过程，已开始或至少已在考虑用氧气取代空气，因此将会有很多低成本和高能效的机会。

氧气取代空气在工业增长中扮演重要的角色。在氧化反应、燃烧和微生物处理中使用氧气，可得到以下的益处：

（1）提高设备的生产能力和收得率。  
(2）强化生产过程和减少资金成本。  
（3）减少污染和更有效地废物处理。  
（4）减少能源消耗和降低运行成本。

综合化工艺提供了一种创新的设计方法，事实证明它带来的进一步好处有：

(1) 减少资金费用（用于修理和扩产项目）。  
(2）降低能耗和减少对公用设施要求。  
(3) 运行最佳化。  
（4）有较大的生产弹性。  
(5) 增加装置能力（消除瓶颈效应）。  
(6) 减少对环境的影响。

显然，用氧的好处配以综合化工艺的优点，两者结合将会得到更大的收益。结论是：综合化工艺具有比我们迄今所认识的更加广阔的前景。

膨胀机的压降为  $0.21\sim 0.35\mathrm{MPa}$  ，冷箱的热端温差只有  $2,2^{\circ}C$  。因此，实际上膨胀机X-1的制冷量是很少的，主要靠循环冷却使温度降到  $-175\sim -185^{\circ}C$

甲烷化炉后串接分子筛干燥器及冷箱。分子筛干燥器除去来自甲烷化工序的工艺气中水、  $\mathrm{CO}_{2}$  、  $\mathrm{NH}_{4}$  ，而在冷箱中除去随二段炉过量空气所带入的多余氮，同时亦除去来自上游工序中的  $100\%$ $\mathrm{CH}_4,60\%$  Ar及  $50\%$  CO，并将冷箱出口合成氨原料气中  $\mathbf{H}_2:\mathbf{N}_2$  控制在3:1，因而冷箱实际上是布朗型合成氨装置的核心部分。

# 二、采用深冷净化系统的布朗型合成氨装置的特点

1. 减少一段转化炉的热负荷, 使一段转化炉的出口温度降低

由于采用深冷净化系统，允许在二段转化炉引入过量空气，而产生大量的热，这样就相应减少了一段转化炉的热负荷，使一段转化炉出口温度降到  $700^{\circ}C$  以下，比传统设计的  $815\sim 900^{\circ}C$  要低得多。

2.二段转化炉的出口温度较传统设计值低  $100\%$  多

由于采用深冷净化系统，允许合成氨原料气中的甲烷含量较传统合成氨工艺的甲烷含量高（布朗型二段转化炉出口甲烷含量为  $1.65\%$  ，而传统合成氨工艺仅为  $0.34\%$  ），从而使二段转化炉的出口温度较传统设计值低  $100^{\circ}\mathrm{C}$  之多。

3. 提高了合成氨的净化值

由于采用深冷净化系统，合成氨原料气中的全部甲烷、过量的氮和绝大部分氩被有

6. 采用深冷净化系统，使工艺操作更为灵活、方便、稳定

这一特点主要表现在深冷净化系统的采用及其在整个流程中所处的位置上。C.F.布朗型合成氨工艺将深冷净化系统设计在合成压缩机之前、干燥器之后，这样就将整个合成氨厂有机地分成既相互联系又相互独立的两个部分，其中前半部分主要产品为蒸汽，后半部分主要产品为合成氨，工艺操作就更为灵活、方便、稳定。如果需要增加蒸汽产量，可在二段转化炉加大空气流量，而不影响合成部分的氨产量，因为所有过量的氮在冷箱中均被分离掉，并且合成补充气的氢氮比仍然保持3:1。这对合成回路受空气流量影响的其它合成氨工艺来说是无法办到的。

另外，若二段转化炉出口甲烷含量过高，也不会影响合成回路的弛放气量。因为所有甲烷在深冷净化系统均被除掉，并且合成回路的惰性气体含量保持不变。这点其它

工艺是很难办到的。

7. 采用深冷净化系统使布朗型合成氨厂的能耗大大降低

采用深冷净化系统的布朗型合成氨厂，吨氨能耗降低到29.31GJ/tNH<sub>3</sub>以下，同时减少了设备投资，真正达到了投资少、能耗低、效益高的目的。

# 三、结论及建议

1. C.F.布朗合成氨工艺是代表当今世界先进水平的低能耗合成氨工艺，其设计思想是通过深冷净化冷箱的设置优化生产工艺操作，使合成氨工艺变得更加合理，因而是今后国内外合成氨工业的发展方向。

2. 目前，国内辽宁锦西天然气化工总厂和四川涪陵化工总厂已采用C.F.布朗型合成氨工艺，基本上都处于试生产阶段。其中的深冷净化装置经过生产运行的检验，基本上达到了设计要求，具有运行平稳，操作方便的优点。但亦存在着因膨胀机油闸系统波动导致膨胀机停车，影响冷箱运行等问题，这些需要在以后的生产中加以解决。

3. 国内深冷技术研究及设计、制造单位，应密切注意深冷净化装置在布朗型合成氨工艺中的应用及技术发展动态，根据国情有计划地做好技术开发工作，摆脱布朗型合成氨工艺中深冷净化装置需从国外整套引进的落后局面，开拓深冷技术在化工装置中的新应用领域，这样可使深冷技术从主要从事空气分离中解脱出来，扩大市场范围。

# $150\mathrm{m}^3 /\mathrm{h}$  空分设备内压缩流程改造的理论与实践

陈锡顺

（中国空分设备公司技术发展部，杭州市东新路152号，310004）

摘要 文章结合实际，从冷量消耗、冷平衡、单位能耗等方面入手，分析了150 $\mathrm{m}^3 /\mathrm{h}$  空分设备中压正流膨胀外压缩流程改为内压缩流程的可行性；并提出了内压缩流程今后的改进方向。此改造在哈尔滨汽轮机厂氧气分厂实施后，两年多的运行证明，空分设备运转状况良好，经济效益显著。图2表1。

主题词：小型空气分离设备 中压 膨胀 外压缩 内压缩 液氧泵

70年代初期，德国林德公司第二代小型空分设备的主要特点是采用中压返流膨胀的内压缩流程，并安装在集装箱内，给用户带来很多方便。我国原有的  $150\mathrm{m}^3 /\mathrm{h}$  空分设备已生产近两千套，分布在全国各地，少数出口东南亚和非洲。这些空分设备均采用中压正流膨胀外压缩流程，即产品氧气在分馏塔外部经氧压机压缩后充瓶。由于氧压机一般采用蒸馏水润滑，故高压氧气中含有水分，使钢瓶内积水，内壁锈蚀，产生异味。

而空分设备的内压缩流程，即从冷凝蒸发器中抽取液氧，经液氧泵压缩，热交换器内汽化直接充瓶。这既克服了上述缺点，又提高了瓶氧质量，能同时满足国家《医用氧及航空呼吸用氧》及一类I级《工业用氧》标准。

这种内压缩流程，在国外如美国、原西德、日本、前苏联已被广泛使用。它具有如下特点：第一，氧气纯度不受影响，一般在 $99.7\% \sim 99.8\%$  以上，尤其对切割厚钢板更有效；第二，液氧泵体积小、重量轻、机器零件少、占地少，且只需一级压缩，耗功少；第三，安全性好，从冷凝蒸发器中连续抽取

液氧，碳氢化合物不会在冷凝蒸发器中积聚，液氧泵在低温下压缩液氧，不会燃烧、爆炸；第四，液氧泵不设岗位，可以减少操作人员；第五，提高了设备利用率，解决了“大马拉小车”的问题，且能耗减少。

该设备1993年已在哈尔滨汽轮机厂氧气分厂改造成功，氧气产量达  $130\sim 140\mathrm{m}^3 /\mathrm{h}$  。至今已运转一年多，情况良好。经济效益明显提高，对同类设备的改造具有现实意义。

# 三、内压缩流程讨论

1. 冷量消耗增加

内压缩流程与外压缩流程最大的不同之处，是设备消耗冷量增加，这是由热力学上不可逆性决定的。内压缩流程的冷量损失主要有四方面：第一，消耗在液氧泵压缩功的

热当量上，是以装置跑冷损失的形式加于液氧中，使高压液氧的温度升高  $5\sim 10^{\circ}\mathrm{C}$ ；第二，高压热交换器的热端冷量不完全回收，使冷损失增加，原热交换器热端温差为  $5^{\circ}\mathrm{C}$ ，新热交换器的热端温差上升为  $8\sim 10^{\circ}\mathrm{C}$ ；第三，高压氧气出冷箱，由于压力提高，焓值下降，使装置等温节流效应降低，这部分损失在一个充瓶周期内会有所变化，这可以用平均压力值计算损失，一般取2/3的压力值为计算依据；第四，为液氧泵的跑冷与泄漏损失。

为了尽量减少上述损失，采取了如下措施：第一，减少液氧泵的跑冷及泄漏损失，使液氧泵的小冷箱尽量密封，不使潮气进泵箱，另外使活塞杆的密封及泵的连结接头不漏液；第二，保证进泵的液氧过冷度，使液氧尽量不汽化，以提高泵的指示效率，减少泵的功热损失；第三，尽可能保证热交换器的制造质量，管子要盘得紧，包皮要贴得牢，尽量使气流不短路；第四，中压空气进热交换器的分配要合适，不使热端温差扩大；第五，高压氧要随充瓶压力而变化，不要一直保持高压操作，以减少等温节流效应的损失。

# 2. 解决冷平衡的措施

中压正流膨胀流程，增加冷量的措施：第一，增加膨胀空气量，使下热交换器正返流最小温差缩小到最小，约  $4\%$  ，不出现零温差，甚至负温差；第二，提高空气操作压力，以增加膨胀机进出口焓降及增加装置等温节流效应，按照设计要求膨胀量约占加工空气量的  $50\% \sim 60\%$  ，空气操作压力约为4MPa(G)，而实际运转结果表明，空气操作压力为  $3.2\sim 3.3\mathrm{MPa}(\mathrm{G})$  ，空气没预冷，装置冷量已满足要求，主冷液面还可上涨，冷量很富裕；第三，提高膨胀机的等熵效率，把活塞式膨胀机改为透平膨胀机，使等熵效率达  $70\%$  以上，可使空气操作压力继续下降，对空压机操作更有利。

# 3. 单位能耗是否会增加

按照资料推荐[1]，内压缩流程比外压缩流程单位氧的能耗有所增加。但实际上，对本改造设备而言并非如此。因为  $150\mathrm{m}^3 /\mathrm{h}$  空分设备原配套机组空压机电机功率  $250\mathrm{kW}$  （个别  $200\mathrm{kW}$  ，启动时空气操作压力高，需 $4\sim 5\mathrm{MPa}(\mathbf{G})$  ，而正常工作时仅为  $2\sim 2.2$ $\mathrm{MPa(G)}$  ，改为透平膨胀机后操作压力下降到  $1,6\sim 1,8\mathrm{MPa}(\mathbf{G})$  。这样，在正常工作时电机的效率很低，无形中增加了功率消耗。另外对氧压机而言，2-1.67/150型电机功率为  $28\mathrm{kW}$  （也有  $30\mathrm{kW}$  )，大都配两台；又由于用水润滑，机器零件易损，使气量下降，也增加了耗功。

改为内压缩流程后氧压机取消了，空压机的电机效率提高。经计算，单位氧的电耗反而会下降。由于空分车间无单独的功率表，故这部分功率消耗未测定。但从装机容量来说，内压缩流程的功率会大大下降。液氧泵的电机功率仅  $3\mathrm{kW}$  （几乎可以忽略不计）。哈汽厂空分车间的实际运转证明：因取消氧压机，每年的消耗可节省几万元。加上操作管理上的改善，医用氧经济效益提高，此流程用户比较欢迎。

# 四、今后的改进方向

通过上述实践，为使内压缩流程更加完善，提出今后内压缩流程的改进方向：

1. 为了更好地提高膨胀机的等熵效率，减少维修工作量，建议将活塞式膨胀机改为透平膨胀机[4MPa(G)]，以进一步降低空气操作压力，节省电耗。  
2. 全国其他地区可先预冷空气，使纯化器在一定的温度下（ $10\sim 20^{\circ}C$ ）工作，使分子筛的工作周期和分馏塔的运转周期延长。  
3. 液氧泵的排出阀是泵的关键部件，目前朝下安装不利于排气，而且检修时由于离地较近拆装不方便，建议改为朝上安装。另外液氧泵的汽液分离器的自动阀不灵活，问

气量较大，往往会使泵停止工作。为减少泵的冷损，建议取消汽液分离器，回气管路上加装一只通过阀，以便泵检修。

4. 对于同时要制取高纯氮的用户，仅需对上、下塔作适当改进，增加一定量塔板，调整部分抽口即可。

# 五、结论

实践证明，对原  $150\mathrm{m}^3 /\mathrm{h}$  空分设备进行内压缩流程改造，在技术上是可行的，采用高压热交换器替换中压热交换器方案在国内尚属首次，技术先进，操作方便，生产的产品可以满足双高(氧、氮)的要求，达到国外同类产品水平，对国内同类产品的技术改

造，具有现实指导意义。

# 主题词：空分透平压缩机 在线监测 系统 故障

随着现代化大生产的发展，制氧行业有了很大的变化，加上机械、仪控、电控、装备本身技术复杂程度的提高，机械故障及事故对空分设备的影响就明显增加了。尤其是空气透平压缩机，其运行状况直接关系到空分设备的正常生产。因此，提高对空气透平压缩机的监测能力就成了广大用户的迫切要求。在线监测系统就是在这种情况下推出的一种能快速、准确地分析运行中空气透平压缩机故障隐患的监测仪器，利用它可以发现设备缺陷，并及时进行维修，减少机械故障对制氧机正常生产的影响。

在线监测系统是计算机化的新一代多功能设备状态监测诊断系统，是针对目前国内微机十分普及而进口故障诊断设备十分昂贵且不便于操作推出的。该系统将软件和硬件有机地结合起来，集在线监测仪、信号分析仪、故障诊断仪、数据采集器、信号记录仪、动平衡仪和档案管理系统为一体，是设备振动测试、状态监测、故障分析的强有力工具。该系统一般由传感器、变换器、预处理器、A/D转换器、微机、打印机等组成，其联接方式见图1。

# 一、在线监测系统的工作原理及主要技术指标

在线监测系统的工作原理，是把通过传感器从现场测试得来的模拟电压信号先进行信号放大和抗混滤波，剔除干扰，提高信号的信噪比；然后，通过A/D转换器将模拟信换号转成离散数据信号，送入微机，在专用软件支持下进行分析处理；最后将分析结果显示或打印出来。也可贮存在微机内，随时调出进行分析、比较，以确定运行机械的“健康”状况。

# 二、在线监测系统的实际应用

我厂1989年由四川空分设备厂配套的ITY-115/5.3型空气透平压缩机，从安装运行至今共发生大小事故10多起。其中最严重的是1992年6月，由于突然停电，引起空气透平压缩机排气管上单向阀失灵，操作人员又没有及时处理，使空气透平压缩机转子反转，损坏了轴瓦。因为没有数据连续记录及频谱分析，所以恢复送电后，没及时发现隐患，启动后造成了严重后果，影响了正常生产。为杜绝此类事故的再次发生，我们与北京科技大学合作，在此空气透平压缩机上安装了在线监测系统(见图2)。

# 延长  $50\mathrm{m}^3 /\mathrm{h}$  空分设备运行周期的措施

章学文

[华瑞制药有限公司(退休), 江苏无锡马上, 214092]

朱毅

（江苏扬子江船厂，江苏江阴，214431）

摘要 简要分析了影响  $50\mathrm{m}^3 /\mathrm{h}$  空分设备运行周期的因素；介绍将  $50\mathrm{m}^3 /\mathrm{h}$  空分设备的运行周期延长至1年以上的措施，添置空气预冷机组及高效除油过滤器，使加工空气温度长年保持在  $4^{\circ}C$  左右；并对关键设备进行了技术改进，保证了设备长期运行的可靠性。

# 主题词：小型空气分离设备 运行 周期 技改 效益

$50\mathrm{m}^3 /\mathrm{n}$  空分设备在全国有相当大的数量，能耗普遍较高，运行周期满1年的单位很少。

若将全国  $50\mathrm{m}^3 /\mathrm{h}$  空分设备的运行周期普遍提高至1年以上，那么年经济效益是很可观的。我们也与同行一样，为提高运行周期进行了一些工作，在此与同行共同商讨，共同提高。

# 一、影响因素

影响  $50\mathrm{m}^3 /\mathrm{h}$  空分设备运行周期长短的主要因素，各单位情况可能不尽相同，但大多由两个主要原因所决定。

1. 单机设备故障频繁，迫使停运  
空气压缩机、纯化器、膨胀机、分馏塔等故障，迫使停运。  
2. 分馏塔进入水分及二氧化碳, 堵塞节流阀、管道、塔板孔, 使塔无法运行而迫使停机。

# 二、改进措施

对各单机如空压机、纯化器组、膨胀机、分馏塔、氧压机等进行技术状况分析，找出

影响连续运行1年的主、次因素。凡设备结构及设计制造上存在的缺陷，进行研究、设计改进，逐一予以克服。使设备运行的可靠性、安全性、稳定性、经济性有较大的提高。

1. 改进 L2-5.55/40型空气压缩机的密封与冷却

对活塞杆填料密封圈、刮油圈进行改进，减少活塞杆的带油状况，使曲轴箱油耗降低，从而提高压缩空气的清洁度。

每年清洗一次水腔及冷却器，使冷却系统有良好的换热效率，提高冷却效果。

2. 增加UF-380/40型空气预冷机组

增加空气预冷器，从工艺技术上考虑有两个目的：其一，降低压缩空气出末级油水分离器后的温度，使空气由  $30\sim 35^{\circ}C$  （含湿量  $30.1\sim 39.3g / m^3)$  冷却至  $3\pm 2^{\circ}C$  （含湿量 $5.2\sim 6.8g / m^3)$ ，加工空气的干燥度提高约6倍，并使压缩空气中游离油蒸气在预冷器内得到冷凝并成液滴析出。由于压缩空气干燥度的提高，含油量的降低，使纯化器内分子筛的吸附负荷大为减轻，提高了分子筛对工

氧化碳及碳氢化合物的吸附容量，对设备长时间安全运行极为重要。

其二，降低压缩空气进入分子筛吸附筒前的温度，使压缩空气进入分子筛吸附筒的温度始终保持在  $4^{\circ}C$  左右。由常温下动静吸附特性决定，分子筛最佳共吸附温度在  $4^{\circ}C$  ，高于或低于此温度，分子筛的吸附功能均有所降低，温度越高分子筛的吸附功能越差。所以对预冷机组温度调整控制极为重要。

# 3.增设高效除油过滤器

我们在江苏扬子江船厂的  $50\mathrm{m}^3 /\mathrm{h}$  空分设备的纯化器油水分离器后，再增加一只西安压缩机厂净化设备分厂出品的GYL-5/55型高效除油过滤器，进一步捕捉经预冷器及纯化器油水分离器出来的空气中的机油粒子，从而最大限度降低高压空气中的含油量，这对分子筛免受油中毒、提高分子筛的使用寿命是一个有力的保证；同时有效地保证了分馏塔运行的安全性、稳定性，使分馏塔长期运行有了可靠的保证。

纯化器吸附筒未经任何改动，使用13X型分子筛，加热温度入口为  $240\sim 280^{\circ}C$  ，出口为  $110\sim 120^{\circ}C$  ，加热2.5小时，冷吹5.5小时。每筒使用仍为8小时。加热及冷吹时，氮气量一定要足，冷吹务必要彻底。

实施以上改进措施后，分馏塔连续运行一般在  $300^{\circ}$  天以上，分馏塔管道、阀门、液面计没有任何堵塞现象，工况稳定。

4. 对PZK-5/40型活塞式膨胀机进行技术改进

活塞式膨胀机，进、排气阀门及阀座结构为线密封硬对硬结构，阀座阀芯杆两者易损坏，不耐用，密封面容易失效而造成泄漏，因而使膨胀机的产冷效率逐步降低。一般此种结构的阀往往无法连续运行一年，而且阀

门修理难度大。

为此，我们对进排气阀门结构进行了多次改进，改成硬对软平面型密封结构和硬对软仍为线接触密封结构两种。两种结构均可利用原阀改制，一旦改制后阀座基本上可以长期使用不必更换。阀芯杆也可使用两至三年，修理极为方便。平面密封结构阀只需更换一片纯四氟阀片，另加一根开口销即可。线接触结构阀只需更换尼龙6座圈一只，维修配件费用大为降低，而且密封可靠，运行噪声低，产冷量高，在相同冷损下，运行高压压力较低。

此改革经过多年的运行，表明是很成功的。为  $50\mathrm{m}^3 /\mathrm{h}$  空分设备长周期运行奠定了基础，同时减轻了维修膨胀机的工作量。

进排气阀密封函内的填料，原结构是实硬结构，无弹性。由于聚四氟乙烯的冷热收缩与膨胀系数大，因而阀杆密封处压紧即卡，略松即漏，形成结霜结冰。为此我们对密封圈环的结构进行改进，改成有弹性的密封圈，并改用不同配方。制成后装上，使用效果很好，无卡杆及泄漏现象，中间体内清洁度高，且经久耐用，一套密封圈填料可以使用1年以上。

5. 保证分馏塔无任何泄漏, 不带病运行

对设备全系统进行改造的同时，维护保养好分馏塔的技术状况是至关重要的。在投运前应彻底查漏，节流阀应具有良好的气密性，若发现分馏塔有泄漏现象，应及时补焊修理。从而做到分馏塔无任何泄漏，不带病运行。

$50\mathrm{m}^3 /\mathrm{h}$  空分设备在全系统上，经过优化流程组合，加强单机维修后，我们两单位空分设备的运行周期均达到1年。

朱大余 孙爱俊

（江苏省盐城市钢铁厂空分分厂，江苏盐城市，224002）

盐城钢铁厂空分分厂1*机组空压机(2D8-17/45-I型)，前几年运行工况较好，稳产高产，分馏塔的氧产量一直保持在  $170\mathrm{m}^3 /\mathrm{h}$  以上（冬季）。刚开始空压机三级缸压力一直稳定在  $1.1\sim 1.25\mathrm{MPa}$  （正常工作状态），可随着使用时间的延长，三级压力逐渐下降，最低时只有  $0.85\mathrm{MPa}$  。加工空气量严重不足，分馏塔的工况很难调节；同时还经常造成一级缸超压（一、三级共用一只平衡腔）。紧急停车后，拆换三级活塞环时，发现四根活塞环中，前二道活塞环搭口处已磨光，弹力环也被弹出活塞座，直接与气缸壁磨擦，后两道弹力环已在活塞环座左右两边撞击出二道凹痕，而且弹力环由宽变窄磨损严重。经测量，发现三级缸体缸套底部已磨出坡度，椭圆度锥度严重，大大超过规定使用范围。

一般情况下，三级缸更换活塞环后，能使用2个月，而此时使用不是1个月，最短的时间只用了不到7天，便需更换新环。频频停车换环，造成总厂电炉用氧紧张。同时更换新环最短时间也得2小时以上。迫于缸套磨损严重，我分厂又购置了一只三级缸体，以便作轮换使用。可使用不到一年，此情况又重复出现。在2D8-17/45-I型空压机的众多用户中，也遇到过这种麻烦。

通过认真分析与仔细观察，我们认为造成三级缸活塞环磨损较快的原因，有以下几点：（1）由于三级缸进气温度过高（ $45^{\circ}\mathrm{C}$  左右），水分含量过大，2D8-17/45-I型空压机

是无油润滑设备，进气温度过高。冷却水温度过低时，带压的空气易产生露点，使冷凝水进入气缸。这样就会破坏活塞环（填充聚四氟乙烯）材料的自润滑性，加剧了活塞环的磨损程度。（2)三级缸活塞环磨损过快，在某种程度上取决于二级冷却效果不佳，导致进入三级缸气温过高，该级气体相对含湿量增大。(3)操作工没有定时排放水分或排放不彻底。（4)三级缸进口处下面有一只气水分离器，由于长年在潮湿空气环境下工作，气水分离器内壁腐蚀剥锈，铁屑被吸入气缸，加快了气缸内壁的磨损；同时又由于铁屑的剥落，堵塞了气水分离器吹除管道，加剧了锈水与铁屑的积聚。（5)从使用的情况来看，活塞环本身的质量也存在问题。

由于这五种原因，导致了活塞环的快速磨损，同时也使得三级缸内壁磨损严重，恶性循环后使得空压机加工空气量逐渐减少，造成分馏塔工况难以维持，最后迫使停车。

# 高效除油过滤器使用小结

王元松

（徐州气体厂设备动力科，221007）

我厂1994年3月购西安压缩机厂净化设备分厂一台GYL-16/50型高效除油过滤器，在安装和使用过程中碰到了一些具体问题。为了在制氧工艺上更好地使用高效除油过滤器，现将碰到的问题及改进方法作一小结。

# 一、问题的提出

我厂有两套  $150\mathrm{m}^3 /\mathrm{h}$  制氧机组，均为70年代产品。在运转过程中我们虽然制定了较严格的放油水操作规程，但执行中往往不是那么严格，原分离瓶的结构亦不能彻底清除空气中的油水，这样导致分子筛负荷增大，水分、二氧化碳带入分馏塔，破坏运转工况，使运转周期缩短。严重时分馏塔运转周期才20天左右，造成产量下降，消耗增加，影响了销售。根据以上情况，我们参考有关资料，决定在工艺上增加一台高效除油过滤器，以解决油水分离问题。

# 二、安装和使用

根据我们制氧设备的实际情况，空压机三级油水分离瓶  $(\phi 300\times 2100\mathrm{mm})$  经常处于振动状态，并在进口接管处发生多起裂焊现象，虽经补焊加固但裂焊现象仍时有发生。我们决定将该分离瓶拆除，把纯化器油水分离瓶  $(\phi 400\times 1000\mathrm{mm})$  移到原空压机三级分离瓶处。这样由于该分离瓶较短，不但解决了振动，也不要再购新分离瓶，节省了资金，工艺上也不减少分离瓶的个数，心理比较平衡，工艺也满足要求。我们按照随机说明书的要求，于5月30日安装好高效除油过滤器，并投入运行。

6月2日操作工反映高效除油过滤器下

阀不通，维修工拆检发现堵塞，投通并进行吹除后开车运行。6月8日白班分馏塔与空压机压差增大，热交换器堵塞，被迫停车。从安装高效除油器到停止运行，共计10天。

# 三、分析与改进

问题出现后，我们召开了事故分析会，认为分馏塔堵塞的原因有如下几点：

1. 经拆检发现高效除油器下阀又不通，经追究分析，6月7日中、夜班已经发生高效除油器下阀不通，操作工没有发现，造成高效除油过滤器超负荷运行，气温湿度又大，油水分离后不能及时排出。

![](images/05a10874d59437d9ca62b7fdb4a139da4706868b3b4010c3821783ddd41c795e.jpg)  
高效除油过滤器改进示图

1. 高效除油过滤器 2. 定时放油水指针牌  
3.PN220/DN12吹除阀4.玻璃视镜  
5.  $\phi 21\times 3\mathrm{mm}$  紫铜管

2. 造成高效除油器下阀不通的原因有：

（1）焊接管道中有焊渣，新管道中有未彻底清除的铁锈。  
（2）高效除油器下阀通径设计较小  $(\phi 4$  mm)，有异物很容易造成孔径堵塞，油水不能及时排出(安装运行10天堵塞两次)。  
（3）排放油水管道是封闭的（我们将其管线接到室外废油池内），操作工观察不到油水排放情况，通与不通很容易造成错觉。  
（4）操作人员责任心不强，没有按规定排放。

针对分析情况我们在操作和工艺上进行

了如下改迹：

1. 将高效除油器下阀改为PN220/DN12的吹除阀，使其通径由原  $\phi 4\mathrm{mm}$  扩大为 $\phi 12\mathrm{mm}$ ，便于油水、杂物的排放(见图)。  
2. 增加玻璃视镜，使操作工能直观地看到油水排放情况，防止错觉（见图）。  
3. 增加油水排放指针盘, 将排放油水的时间用指针在盘上表示, 便于监督。

通过以上改进至今运转良好，高效除油器下阀没有再堵现象。

# 精氩塔废气管堵塞故障的处理

陈国霖 程日康

（韵关钢铁集团公司氧气厂，广东韵关市，512123）

摘要 文章探讨了精氩塔废气管被堵塞的机理，介绍了故障的处理方法及防范措施，最后提出了精氩塔的操作及工艺改进设想。图1。

主题词：精氩塔 废气管 堵塞 处理 改进

# 一、前宫

我厂拥有4套空分设备，其中2套可生产纯氩，1套生产粗氩，氩气生产时间已有10多年，空分设备操作经验丰富。1994年底发现  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备精氩塔的废气管被堵塞，无法用升高塔内压力来排除，须经6小时乃至更长时间的加热才能吹通。我们一直认为是液体积存在废气管U形管处造成堵塞。笔者经过多次的探索及实际操作之后，发现：造成废气管堵塞的根本原因不单是液体，而是由固态氩与液态氩形成的胶状混合物所致。此物有很强的粘性力及较差的导热性能。本文就整个探索过程、特别是形成机理方面进行论述，供同行们参考。

# 二、故障现象及处理过程

1994年12月24日14时左右，粗氩压缩机出口压力升高，造成工艺氩缺氢，精氩塔被污染。根据当时废气排出量难以控制、精氩塔压力上升至  $90\mathrm{kPa}$  等主要工况，就调整V704、V753、V752等阀门，进行处理。18时化验精氩纯度为  $2.2\mathrm{ppmO_2}$ ，恢复正常生产。25日正常出氩，但废气排出量有减少和波动的趋势。26日夜班1时至4时，废气时通时不通，4时18分开始全不通，无法排气，精氩塔压力超量程  $100\mathrm{kPa}$ ，阻力下降为0.9kPa（正常值为  $14.5\mathrm{kPa}$ ），无法建立精馏工

况。27日4时30分关闭返流氮，用工艺氩复温 氩换热器，从101K升至243K都无济于事。 8时30分对V753阀进行解体检查，正常。11时用空气加热精氩塔系统，耗用6小时，塔 内温度达291K才把废气管吹通，于当日中 班恢复生产。

在整个故障及处理过程中，有三点值得注意：

1. 从24日白班开始不正常，至26日夜班全部堵塞不通气，经历了30多小时。  
2. 塔内压力高达约  $0.12 \mathrm{MPa}$ , 仍无法吹通。换算成液氩柱, 则为  $8.0 \mathrm{~m}$  液位差。  
3. 加热时间共用16小时，说明堵塞体换热性能差，仅为导热而已。

# 三、故障判断及处理方法

在堵塞形成过程中，伴随有以下参数：液氮蒸发温度为  $-190^{\circ}\mathrm{C}$ ，即  $83\mathrm{K}$ ；废气氙组分曾高达  $90\%$ ，后恢复至  $15\%$  左右；废气管（ $\phi 45$ ）从精氙塔至氩换热器的管路中有U形管，高度为  $0.5\mathrm{m}$ 。

综合以上情况，首先认为堵塞物不是液体。因为U形管即使全部灌满液氩，压力也仅有0.014MPa，而当时塔内压力高达0.12MPa。液氩有强烈的沸腾现象及对流性能，换热效果好，全部蒸发成气体，也无需用10多小时。液体流动性好，在U形管中先在最

低处积累，全部堵塞也无需多长时间。

查阅有关资料，氙的三相点参数：温度83.8K，压力0.068MPa；沸点87.291K，压力0.1MPa；导热系数0.062kJ/m·h·℃。

据此可确定：堵塞物是液氩和固氩的胶状混合物。原因有以下三点。

1. 固体与液体同时形成，在管道中四处分布，要形成大强度堵塞需较长时间。  
2. 胶状物与管道有很强的粘性力，用升高压力是无法排走的。  
3. 胶状物的换热仅靠热传导, 而氩的导热系数小, 需长时间加温才能气化。

在堵塞形成之后，只有全开V756阀，用热空气直接加热废气管，增大温差  $\Delta T$  ，提高换热效果才行。精氩塔系统如图所示。

# 四、堵塞形成机理探讨

在正常精馏工况下，因废气中的氮、氢组分占绝大部分（氮  $40\%$  、氢  $40\%$  ），氩组分

较少，且液氮温度高于氩熔点，故不会产生固态氩。

这次精氩塔被污染，氩组分大量增加，V704阀因液位计LI704不准确，阀门调整不灵，蒸发压力过小，使蒸发温度低于  $85\mathrm{K}$  废气排气量小，塔内压力不断升高，超过 $0.068\mathrm{MPa}$  ，创造了产生固态氩的充分条件。

精馏过程中，随着气体的上升，温度不断下降，换热器顶部(精氩塔顶部)的废气温度最低，与液氮温度相近。当温度下降到三相点温度时，就产生固态氩微小颗粒与液态氩微小液滴，并不断“长大”。由于废气的向外流动，于是废气夹带着小颗粒、小液滴，通过U形管，并进入换热器。在流动的过程中，与管道不断碰撞，使部分颗粒、液滴落在管壁上，部分在U形管堆积，部分进入换热器。

部分颗粒由气体重新携带，积聚到U形管，一部分颗粒因换热器的传热，使氩熔化而流回到U形管底部。随着时间的推移，液、固越积越多，气体的紊流使两者充分混合，最终产生胶状混合物，使管道堵塞。

# 五、精氩塔操作要点

1. 调整液氮阀V704及V752,控制液氮工作压力在  $0.23\sim 0.24\mathrm{MPa}$  之间，蒸发温度在85K以上，即高于-  $188^{\circ}C$  。  
2. 控制精氩塔压力PI702低于0.07MPa，保持废气参数不超过氩三相点：83.8K、0.068MPa。  
3. 调整 V753 废气排放阀，使流量低于  $7\mathrm{m}^3/\mathrm{h}$ ，降低废气氙组分。  
4. 在精氩塔有以下情况时，应该关小V753或全关，以防止氩气凝固在U形管或换热器：①精氩塔污染；②出现液悬现象；③精馏阻力过小。

# 液化天然气：新的洁净能源、电站的无公害燃料

液化天然气(LNG)是“洁净能源”，是高热值的无公害燃料。在发达国家除作民用燃料使用外，作为发电厂燃料使用越来越多，而且从环境保护出发，用天然气取代煤燃烧发电——联合循环发电，将是必然趋势。

液化天然气的沸点为  $-162^{\circ} \mathrm{C}$ , 与中纯甲烷差不多。天然气在液化时, 经过脱尘、脱硫、脱水和脱  $\mathrm{CO}_{2}$  处理, 腐蚀性小, 燃烧后基本不会产生  $\mathrm{CO}_{2}$ , 氮的氧化物也比产生相等热量的其他燃料少一半, 所产生的  $\mathrm{CO}_{2}$  是石油的  $70\%$  、煤炭的  $60\%$  。

液化后的天然气，体积缩小（约1/600），便于输送和储存，因此海上输送时，均以LNG形态输送。海上运输时，LNG船不能采用半加压/半冷冻方式，只能采用常压深低温方式。因为如果把LNG船的设计温度提高到  $-100^{\circ}\mathrm{C}$ ，其液化压力仍需要2.5MPa，对于LNG船的设计是不实际的。而且船体还要采用双重船壳结构，并要有惰性气体注入装置，以防船碰撞和触礁时保护贮罐免遭破坏；一旦发生燃烧也使贮罐保持一定的安全距离，在紧急时便于壳间注入或加热，以防船体过于冷却；注入惰性气体，是为了安全保护。

鉴于液化天然气为常压低温  $(-162^{\circ}\mathrm{C})$  储存，贮槽通常为地面式和地下式两种，而且贮罐容积都做得很大，小的也在6万  $\mathfrak{m}^3$  以上，大的到13万  $\mathfrak{m}^3$  、23.75万  $\mathfrak{m}^3$  ，甚至还有更大的；平面大到一个足球

场那么大。而且贮槽都是双层壳体，内槽通常用耐低温的9Ni钢、铝合金和不锈钢制造。

当今，能源面临着对环境造成污染后果的严重挑战，迫使您作出抉择。现在，在发电方面，解决这个问题的办法，就是利用联合循环发电站燃烧天然气发电，取代燃煤的火电站。

联合循环发电，是燃气轮机排出来的余热，不直接排放，而是通过锅炉产生蒸汽，再带动蒸汽轮机发电。据报道，燃气轮机每发电2兆瓦，蒸汽轮机就可以发电1兆瓦，使能源充分利用，发电效率可超过  $52\%$  。在国外大型联合循环发电中，燃烧采用富氧助燃，这就要用大型制氧机。

日益增长的经济，需要越来越多的能源。我国“九五”规划，也考虑天然气发电。中国海洋石油总公司已在规划液化天然气贮运、储存这个大课题。

# FL-50/200型分馏塔塔板

# 严重堵塞的处理

袁树利 赵建民

（重庆市西南铝加工厂动力分厂氮氧站，重庆市九龙坡区西彭镇，631326）

# 一、故障现象

我站FL=50/200型分馏塔是邯郸制氧机厂1973年的产品（简称1#塔），于1975年6月投产运行，设备运转一直很稳定。分馏塔每隔两年用四氯化碳和热水全面清洗一次，确保了设备的正常运行。

1994年7月，我们对  $1^{\#}$  塔进行清洗，1995年1月16日，对  $1^{\#}$  塔进行加温吹除后启动制氧，待氧气合格后开始制氮。但无论怎样调节，氮气纯度始终达不到要求（含氧  $\leqslant 300$  ppm)，分析各气、液纯度，液氮  $80\sim 90\%$  气氮  $95\%$  、液空  $34\sim 40\%$  。

当时认为可能是清洗加温吹除不彻底所致。即停车对1#塔进行二次清洗，加温吹除后开启制氧，情况同前。再次清洗分馏塔，加温吹除后开设备，液氮和气氮纯度还是达不到要求，被迫停车。

# 二、故障分析及处理

根据三次开车不成功的情况，大家通过分析一致认为液氮、气氮纯度不合格的因素是下塔精馏效果不好，并经综合分析后认为最终原因为下塔塔板严重堵塞。

我站有两套空分设备，曾同时进行清洗，清洗后对  $2^{\#}$  设备及时进行加温吹除并启动制

氧，此设备生产出合格的氧氮产品。而未及时对1*塔进行加温吹除，导致洗塔时的少量四氯化碳和水分滞留在塔内，发生化学反应，腐蚀塔板并使之堵塞。

最后决定将  $1^{\#}$  塔进行解体处理。结果发现下塔果然有  $1 / 3$  板孔被堵，上塔也有部分塔板被堵，冷凝蒸发器列管内壁也不干净。

拆下后我们用热水浸煮以除污垢，但效果不佳。后用盐酸清洗，先将盐酸倒在塔板上，然后迅速加冷清水冲洗，直到冲洗干净（最好用  $5\%$  小苏打浸2分钟，再用自来水冲洗干净——编者）。冲洗过程中最先流出的是浓黄水，后慢慢变干净水。这样塔板孔基本恢复了原状，经过光目测试认为安好，同时也对上塔和冷凝蒸发器进行了处理。清洗后将设备复原进行了试压试漏，并及时对1*塔进行了彻底的加温吹除。

# 三、处理结果

对  $1^{\#}$  塔进行加温吹除后于1995年4月16日启动制氧，顺利生产出合格的氧（纯度  $\geqslant 99.2\%$  ）、氮  $(\leqslant 300\mathrm{ppmO_2})$  产品。经一个月的观察分析，各操作参数比较稳定，设备恢复正常。

（1995年5月16日）

# 液空过冷器代替液化器积液

李晓明 张卫星

（山西省晋城市钢铁厂制氧车间，048000）

我车间在一次  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备开车过程中，曾发生一起通往液化器管道阀门（即蝶-8)被卡死，使液化器不能工作的故障。如果按常规处理，停机扒搭，修理阀门后再开车，这样将停产5天，影响炼钢生产，经济损失很大。为此，我们采取了用液空过冷器代替液化器的办法进行积液处理，结果取得了成功，整个开车用了58小时就完成了启动任务。

# 一、液化器的作用及不能投用的原因

1. 液化器的主要作用有以下几方面：第一，启动积液阶段，液化空气积累液体；第二，正常运转阶段，在可逆式换热器和精馏塔之间进行冷量分配，使污氮或膨胀空气出液化器温度保持在  $-175^{\circ} \mathrm{C}$ ，保证冷端温差。  
2. 开车前对设备进行过全面加温, 完毕之后关闭阀门, 即蝶-8阀全关, 在启动第一至第四阶段均未动过此阀。当转入积液阶段开蝶-8阀时, 才发现蝶-8阀不能开启, 膨胀后空气不能进入液化器, 使液化器不能起到它在启动过程中所起的作用。

蝶-8阀不能开启的原因，我们初步分析为：①加温不彻底，设备中的水分冻结；②加温后关闭阀门时关得太紧，在低温下，被卡死，不能开启。检修后证明是属于后者原因。以此为教训，凡属于低温阀门在关闭时不能过紧，以阀门不漏为准，开启后还要有一定余量。

# 二、积液过程中采取的措施

1. 液空过冷器代替液化器进行积液

我厂的  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备，液空过冷器污氮通道28层、传热面积  $53.3\mathrm{m}^2$  ，液化器污氮通道26层、传热面积  $49\mathrm{m}^2$  。两者比较，液空过冷器比液化器的传热面积大  $7\%$  从传热面积上看，是可以代替液化器的。

在正常运行中，液空过冷器是把返流污氮从  $-190^{\circ}\mathrm{C}$  复热至  $-177^{\circ}\mathrm{C}$ ，使液空进一步过冷以减少汽化率。现在设想，在开车过程中，把膨胀后的空气  $(-188\sim -190^{\circ}\mathrm{C})$  经蝶-2阀，进入上塔，经返流污氮通道，在液空过冷器中与正流空气  $(-172^{\circ}\mathrm{C})$  进行热量交换，就可以使正流空气液化产生液体。

根据以上分析，液空过冷器可以代替液化器使用，控制好液空进上塔调节阀（调-1阀)的开度，就可以直接对主冷进行积液。

2. 尽管如此，还要注意以下几个问题

第一，调-1的开度很重要，调-1阀不能用自动调节，要根据历次开车的经验，手动开一定量，如果开度太大，经调-1阀的气量太多，就可能造成上塔及膨胀机出口压力升高或超压；如果开度太小，液化后的液体不能全部流入上塔，过冷器不能发挥全部效率。

第二，由于液化器不能投用，本应该液化器承担的膨胀后和精馏塔富余冷量平衡工作前移到了板翅式换热器，使板式冷端温差加大。由于我厂的空分设备在冷端已增加一组板式(1989年安装时改造项目)，冷端为两组。所以此次开车，虽然冷端温差大，但未造成热端温差加大和自清除不好等后果。

（1995年5月）

先来看看当天的操作记录，12:20开  $90^{\#}$  阀，升压；12:35全开  $90^{\#}$  阀；12:38启动膨胀机达满负荷。

按操作规程， $90^{\#}$  阀的开启速度约30分钟。从以上记录可以看出， $90^{\#}$  阀开启速度过快。那么， $90^{\#}$  阀膨胀机的开启速度过快怎么能够导致主换热器堵塞呢？

我们知道，空气中的水分、二氧化碳、乙炔等杂质是由13X分子筛吸附脱除的，而13X对三者的吸附顺序为水分  $>$  乙炔  $>$  二氧化碳，即对二氧化碳的吸附速度最慢。正常操作时，分子筛能够将以上的杂质完全脱除掉。但是，当  $90^{\#}$  阀、膨胀机的开启速度过快时，空气进分子筛的流速突然增大，空气出喷淋冷却塔后携带的水分量增多，使分子筛水分吸附饱和层上升。这样就占据了二氧化碳的吸附饱和层和吸附层，吸附达到了极

点，二氧化碳无法被完全吸附下来，进入主换热器后，就冻结在中抽空气与冷端空气的管道内。

# 二、处理方法

空分设备大加热，将主换热器中冻结的杂质吹除。

# 三、经验教训

这次操作事故，明显地说明了我们操作技术不过关，对关键阀门的操作缺乏经验，在开车过程中急于求成。从中我们深深体会到严格执行操作规程的重要，建议兄弟单位， $90^{\#}$  阀门在开车期间的开启时间应在30分钟左右，膨胀机喷嘴的开大要缓慢，进塔空气流量的上升应基本匀速。在以后的开车过程中，我们由于把握住了  $90^{\#}$  阀、膨胀机的开启速度，空分设备未发生类似事故。

# 二、事故调查

事故发生后，先将伤员送往医院救治，然后采取措施保护现场，进行事故调查。

# 1. 现场查看情况

（1） $1^{\#}$  罐外壁及地面留有大量炭黑。

（2）发生事故的阀门排气腔及其法兰烧毁，阀盖上一只螺栓及阀芯尼龙垫片熔掉，压垫片之螺母熔掉一半。

（3）开罐检查：直对进气口方向的内壁上有一直径  $80\mathrm{mm}$  的熔渣团，渣层厚  $0.5\mathrm{mm}$ ；在其左侧约  $60\mathrm{mm}$  处，有一  $20 \times 60\mathrm{mm}$ 、深度为  $2\mathrm{mm}$  的熔坑；连接进出气阀的三通体内满是熔渣，并有一个很深的熔坑。

# 2. 事故调查情况

（1）检查抽四氯化碳的“油抽子”和盛装四氯化碳的盆子，都不干净，有油污。用手

指在所用四氯化碳中捻动，略有滑腻感。据制氧站反映，抽四氯化碳的“油抽子”曾用来抽油，盆子曾装过酒精漆片，四氯化碳也曾经用来洗过塔。

(2）对此四氯化碳作含油量检测：取一盆四氯化碳，待其自然挥发，20小时后才挥发完，盆底留下大量残油。经称量、计算得知，每升四氯化碳中含油90克。《脱脂工程施工及验收规范》（HGJ202—82)中规定：“用于脱脂的有机溶剂含油量不应大于50毫克/升”。“脱脂后的溶剂油脂含量不得超过350毫克/升”。按前者规定，已超标1800倍，按后者规定亦超标250多倍。该四氯化碳根本不能再用作脱脂剂。  
（3）测量发生爆炸阀门的阀杆向上移动了  $1\mathrm{mm}$ ，阀杆螺距为  $1.5\mathrm{mm}$ ，计算阀门手轮旋转了240度，说明开阀过快过猛。

# 三、事故原因分析

根据上述调查结果，很明显，这是一起氧气遇油而导致的化学爆炸事故。分析事故起因及其过程如下：

因为没有经验，操作者对四氯化碳的滑腻感未引起足够重视，在清洗过程中反而使阀件沾染了油脂。阀门回装后对进气端法兰检漏时，氧与油脂就已开始了氧化反应，但由于该阀处于关闭状态，未受到高速氧气流的冲击摩擦，氧化反应还不剧烈，温升缓慢。并且，从送氧检漏开始到事故发生为止，时间很短，还不到1分钟，温度尚未达到爆炸危险极限。当打开此阀对其排气端法兰检漏的瞬间，油脂受到高速氧气流的猛烈冲击摩擦，氧化反应立即加剧，温度陡增；再者，由于开阀速度过快过猛，剧烈摩擦引发火花。在上面两个作用下即引发燃爆，看到冒白烟，操作者虽已产生关阀的念头，但根本来不及动作就已发生爆炸。

爆炸时产生  $3000^{\circ} \mathrm{C}$  以上的高温气流，熔

化了铜质阀体排气腔和钢质三通体内壁，夹带高温熔渣喷入球罐。细散的熔渣遇到罐体冷却较快，就贴敷在内壁上。大块熔渣不易冷却，就在内壁上继续燃烧，形成熔坑。所幸1*罐是空的，其余各罐阀门又都是关闭的，进气管道内氧气容量也仅为  $0.3\mathrm{m}^3$  。否则，事故后果难以想象。

# 四、几点教训

通过这起事故，除了暴露出制氧站对脱脂剂管理不善的问题，也对管道输氧中的安全问题敲响了警钟。在此，谈几点教训：

1. 洁净的脱脂剂与已用过的脱脂剂，应分开保管，与氧接触的零件必须用洁净的脱脂剂清洗，已用过的脱脂剂只能改作它用。若零件上油脂多，需多遍清洗，每遍都应换用洁净的脱脂剂，必要时还应作专门检验。  
2. 抽取脱脂剂与抽油的器具以及盛装容器，必须专管专用，禁止混用，以避免互相污染。  
3. 开启氧阀，要掌握速度缓慢，严禁过猛过快。但氧压高时，截止阀的启闭很困难，通常要借助于自制工具“F”杠杆，操作力度难以控制，阀门关紧时的感觉不清晰，易造成操作过猛过快或损坏阀芯密封垫片。因此，贮氧压力的高低，应以手能够转动阀门手轮来确定为宜。  
4. 几个贮氧罐并联使用时严禁倒空罐（即有压罐内氧气倒往无压罐），以防止一旦发生事故而导致事故范围扩大。  
5. 贮氧罐及其氧阀，必须有可靠的防静电接地装置，以防止锈粉撞击摩擦产生静电引发事故。接地电阻宜小于  $10\Omega$  ，且每年应复测一次。  
6. 操作者应按规定正确穿戴好劳保用品，衣物宜为棉织品，严禁穿化纤制品上岗操作。

(1995年2月)

# 空压机地脚螺栓断裂事故分析及处理

陈宜正

[温州化工总厂（温州制氧二厂），温州市郊双屿，325007]

# 一、事故情况及原因

1993年底，我厂制氧车间KFS-860-Ⅱ型空分设备运行过程中，空压机(5L-16/50型)同步电机侧突然串出一尺多高的火花，并伴有电机油漆烧焦过热气味，立即放空停车。经检查，空压机注油器侧地脚螺栓(M36)丝扣处断裂，螺帽仅旋上一个丝扣，已无法开车；同步励磁电机(TDK118/24-14)转子与定子之间有相擦痕迹。

该  $150\mathrm{m}^3 /\mathrm{h}$  空分设备于70年代中期投产，空压机系沈阳气体压缩机厂产品。10个月前，该处地脚螺栓曾经断裂，当时仅作临时处理，用电焊焊住。近段时间因震动较大，准备停车后设备检修时处理。以前，空压机振动都被视为正常现象，未引起重视，仅用扳手把4只螺母拧紧。这次事故，打乱了正常的生产安排，造成一定的损失。

经初步检查分析，认为此次事故的原因为：空压机长期连续运转，且上次临时处理不完善，使地脚螺栓压帽与机体结合不紧密，受力不均匀，并使其中一地脚螺栓丝扣处应力集中，后疲劳断裂；进一步引起转子与线圈发生摩擦。后又发现4只地脚螺栓两侧斜垫铁过少，垫得不实；基础表面浸油、松软。

# 二、采取的措施

从空压机和基础的受力情况分析，空压机存在往复惯性力、回转惯性力以及倾覆力矩，这些都是外力，在运行中通过机体和地脚螺栓传到基础上。因此为了防止振动，除应有足够大的基础外，机体连接的地脚螺栓

要有一定的预紧力。

由此可见，应使斜垫铁把地脚螺栓两侧垫实，并扳紧螺母，做到牢固坚实。否则，空压机传到基础的惯性力及阻力矩等不仅会使震动加大，而且会把地脚螺栓震断。

因此，采取以下措施：

1. 对基础表面及断裂处一角进行重新处理，由于时间紧，分解拆装比较费时，整体吊出。

2. 拉出定子重新校对电机导轨水平度，调整线圈间隙，并修补定子、转子相擦部位。

5. 试车后，观察地脚螺栓及螺帽，发现螺帽虽扳紧，但由于空压机工作时不可避免受上冲力影响，基础与机体间还有极微小的缝隙，如不消除，还有隐患。因此又采取以下措施：利用极薄的铜片，做成“U”型，利用空压机运转时的上下震动，把“U”型薄铜片慢慢塞进基础与机脚之间。彻底消除缝隙，使地脚螺栓压帽与机体及基础紧密结合。  
6. 加强设备运行时巡回检查记录制度，以利检修及安全生产。

通过以上处理后，空压机运转情况基本正常，再没出现松动现象。

1. 问： $50 \mathrm{~m}^{3} / \mathrm{h}$  空分设备，液化时间长达  $18 \sim 24$  小时，节-1阀开度只能开  $30^{\circ}$  左右，稍开大  $\mathbf{T}_{3}$ ，温度就回升到  $-127^{\circ} \mathrm{C}$  左右，是何原因？

答：根据来信中说，从启动膨胀机到膨胀后温度  $\mathbf{T}_{2}$  降到  $-130^{\circ}C$  需3.5小时，打开节-1阀，  $\mathbf{T}_{3}$  温度降到  $-145^{\circ}C$  左右，并维持 $5\sim 7$  小时，无法降低，  $\mathbf{T}_{2}$  却在  $-163^{\circ}C$  热端温差  $4\sim 5^{\circ}C_{\circ}$

从上述的情况来分析，其原因可能为：

（1）膨胀机效率降低。影响膨胀机效率降低的可能原因有：进气阀门漏气或排气阀漏，造成部分气在膨胀机内节流；膨胀机活塞环磨损，造成漏气；或者长期运转，气缸磨损，间隙增加；活塞与气缸头之间余隙过大；进、排阀门顶杠间隙过小等因素。请根据机器的说明书规定作检查。检测膨胀机的效率根据进、排气压力和进、排气温度，在空气T-S图中找到实际焓降和理论焓降，然后计算出效率，一般活塞式膨胀机效率大于0.65。  
（2）加工空气量偏少。该机出分馏塔的氧气流量、氮气流量、馏分流量三者加起来，若低于  $320\mathrm{m}^3 /\mathrm{h}$  ，则可能认为压缩机或纯化器及其管道有外漏存在。这时应检查：压缩机的转速、阀门、活塞环、气缸、空气过滤器、冷却器等是否有故障；管道连接法兰，吹除阀、纯化器切换阀门是否漏气。并消除之。  
（3）检查分馏塔外筒壳是否结霜、冒汗。若有则应对分馏塔进行加温，扒掉珠光砂，进行试压，有泄漏处应补焊。  
（4）检查珠光砂是否受潮，包括膨胀机

过桥管的绝热材料是否受潮。若手捏紧后，珠光砂结块，应将珠光砂烘干或更换。

2. 全低压切换式空分流程中，空气旁通污氮节流阀的作用如何？在分段冷却法启动空分设备的一～四个阶段中，空气旁通污氮节流阀如何操作？

答：在全低压切换流程中，出板式的空气总管上，总有一个阀旁通污氮通道，全称叫“空气旁通污氮节流阀”。这个阀在启动阶段有其很重要的用途。但是，有些单位在使用这个阀门时，在开车的第一阶段，刚开始就全开这个阀；有的在冷端温度下降到  $-20^{\circ}\mathrm{C}$  时才使用这个阀；有的在第一阶段干脆就不用这个阀。那么，这个阀到底在什么时候使用？开度又如何呢？

要回答这个问题，不妨让我们先看看这个阀有何作用。我认为空气旁通气氮节流阀有三个作用：

一、开大这个阀，使冷端出口空气旁通到污氮通道，实际上就是给板式增加了一股全环流。显而易见，它能起到缩小冷端温差，改善板式自清除的作用。

二、开大这个阀，使空气节流到污氮通道，它可以收回部分冷量。这就是压缩机的节流效应。  
三、开大这个阀，能使板式冷量下移扩大冻结区，其副作用是扩大了热端温差，增加了冷损。

根据这个阀所起的作用，再结合分段冷却法，将各个阶段的特点结合起来，一齐来研究，就会得到使用这个阀的最佳方案。

我认为：分段冷却法的第一阶段，也就是通过水分冻结区，在这个阶段，流路最短，通过塔内的设备最少，但必须通过膨胀机。因为在这一阶段中水分是不会清除干净的，所以水分有可能冻结在膨胀机的进口过滤器上，或冻结在膨胀机的导流器中，致使膨胀机不能很好地运转。如果冻结严重，堵塞了膨胀机，就须停车加温处理，这无疑会延长整个启动时间。所以在这个阶段，最危险的是膨胀机的堵塞问题。只有缩短这一阶段的运行时间，迅速通过水分冻结区，才能有效地防止水分过多地冻结在膨胀机内或过滤器中，使膨胀机避免发生堵塞。因此能否缩短这一阶段的运行时间，是这一阶段操作的关键。

从上面谈到的这个阀的三个作用看，开大这个阀尽管有许多好处，但是在这个阶段使用这个阀，只能延长而不能缩短这一阶段的运行时间。因为它会使冷量下移，把冷量转移到中部，这样冷端的温降，显然要大大减慢。为此，我认为，在启动的第一阶段最好不要使用空气旁通污氮阀。若要使用，最好在冷端温度下降到  $-45\%$  以下时用。因为这时，空气虽含水分，但是含量甚微，这时就不会发生堵塞膨胀机的故障了。

有人提出：在这个阶段不用空气旁通污氮阀，或后期使用这个阀，会不会对板式的自清除有影响？会不会影响到整个空分装置的运行周期呢？

改善板式自清除的办法，不只是增加环流这一种，还有其它办法。如缩短切换时间等。只要操作得当，把温差控制在自清除允许范围之内，是不会影响设备的运行周期的。

在设备启动的第二阶段，也就是用干燥的空气冷却塔内设备的阶段。这一阶段空气中不含水分，二氧化碳也析不出来。所以这一阶段不会发生膨胀机堵塞的问题。只要控制冷端温度不回升到  $-60^{\circ}\mathrm{C}$  以上，就可以开大或全开空气旁通污氮阀，同时加大环流，其好处是：

一、能充分发挥，旁通节流阀的作用（三点作用），尤其能改善板式自清除。  
二、把冻结区向下移，扩大了冻结区的范围，使水分更不易带到塔内其它设备。  
三、把冷量积蓄在板式的冷端和中部，为迅速通过二氧化碳冻结区创造了更好的条件。  
四、开大这个阀和增大环流量，能延长第二阶段的运行时间。这一阶段的运行时间愈长，愈能延长膨胀机在高温高焓降区工作的时间，对缩短整个启动时间有利。

在第三阶段，也就是通过二氧化碳冻结区阶段。当膨胀机的出口温度达  $-135^{\circ}C$  时，二氧化碳就达饱和，开始析出，但是这时二氧化碳析出的量还很少。当温度达  $-150^{\circ}C$  时，二氧化碳过饱和，就大量析出。我们在 $-135\sim -150^{\circ}C$  温度范围内，利用二氧化碳析出量不多的特点，可以开大空气旁通污氮阀和加大环流量，使冷量集中到板式，当膨胀机出口温度下降到  $-150^{\circ}C$  时，全关这个阀和全关环流，使冷端温度迅速下降到  $-165^{\circ}C$  以下，强制通过二氧化碳冻结区。这就是所谓的跳跃式操作法。跳跃式操作法巧妙地使用环流和全环流，迅速通过二氧化碳冻结区，使膨胀机不在这一阶段被冻结堵塞。这确实是一个好的操作方法。

第四阶段，再次冷却设备、积液、调纯

出氧。在这一阶段，随着塔内设备温度的下降和空气的液化，空气的富余量也就不多了，而且为了在积液时，不致产生板式过冷和膨胀机带液问题，在保证自清除的前提下，有必要减少环流和全环流，甚至关闭全环流。使板式中部温度保持在一个较高的水平，为积液创造一个好的条件。

综上所述，对空气旁通污氮节流阀的用法归纳如下：

第一阶段和第四阶段，少用或不用旁通节流阀。第二阶段开大或全开，充分发挥其作用。第三阶段，前用后关。

我们按上述用法进行过实验。在第一阶段不开旁通节流阀，用50分钟就渡过了水分冻结区，而且膨胀机未被堵塞。如果开始就打开旁通节流阀，一般要花  $1.5 \sim 2$  小时，而且膨胀机易发生堵塞问题。用跳跃式操作法，我们只用了0.5小时就渡过了二氧化碳冻结区。按一般操作法，需45分钟到1小时的时间。而且前一种不发生膨胀机冻结，后一种易发生膨胀机冻结堵塞问题。

（江苏江都海外工程公司 蔡春郎）

(063103 河北省唐山市东矿区利康制药厂）

3. 答：如何解决  $150 \mathrm{~m}^{3} / \mathrm{h}$  空分设备所配  $2 \mathrm{Z} 2 - 3 / 165 - \mathrm{I}$  型氧压机高压密封器的漏气问题？

答：杭氧生产的立式二列2Z2-3/165-I氧压机为五级压缩，每小时压氧量为180 $\mathrm{m^3 / h}$  （折合吸入状态)，一套  $150~\mathrm{m^3 / h}$  空分设备只需一台氧压机。无水润滑，提高了氧气的纯度，降低了露点。进、排气管道全部采用1Cr18Ni9Ti奥氏体不锈钢，使该机运行更安全可靠。但由于该氧压机结构较新、转速较高，在平时使用中存在一些问题，尤其是高压列密封器严重漏气问题，影响了压缩机效率和安全。如何解决这个问题呢？

此氧压机密封器分低压列和高压列各一

组，低压列需密封0.7MPa左右的压力，结构较简单，由7盒密封圈和靠近气缸前端的1只阻流环组成。而高压列需密封6.2MPa左右的压力，相对结构也比较复杂，由12盒密封圈组成，在靠近气缸前端有6盒为径向无磨损补偿的阻流环，有效地发挥节流作用。经节流后的气体，由引管接入四级进气管道；后6盒密封圈则与低压列完全相同，具有径向磨损能自己补偿的结构，将微量泄漏气体引入一级进气管道内。

从其结构特点分析可知，安装、操作不当极易引起高压密封器漏气。正确安装保证安全运行，必须注意以下几个问题：

一、各密封盒结合面之间要用较细的研磨砂配磨，一般采用  $\mathbf{W}_{5}$  研磨粉，并注意密封盒结合面的光洁度，不得有横向刻痕、砂眼等缺陷。并用着色法检验结合面，配磨后不要任意调换各密封盒位置，要做到“按图配磨，按配磨秩序组装”。

二、各密封盒和密封圈要严格清洗和去油、脱脂，并用干燥空气吹干、吹净，组装时要注意密封盒结合面上清洁度，不允许沾有机械杂质。氧压机吹扫须彻底，以防止吹扫不净。若管道内的机械杂质遗留在高压密封器上部，会破坏活塞杆与密封圈的跑合面。

三、安装时尤其要注意接四级进气的回气管和接一级进气的回气管的准确安装，如果互相调换将造成严重漏气。

四、氧压机运转时要注意活塞杆表面带油现象，活塞杆表面带油较严重时需及时处理刮油器。密封圈也要及时调换，以保证氧压机安全运行。

注意好以上几点就可以解决该氧压机的高压密封器漏气问题，从而使该氧压机的优点能充分体现出来，为  $150\mathrm{m}^3 /\mathrm{h}$  空分设备更新换代奠定基础。

# 中华人民共和国机械行业标准

# 《铝制板翅式换热器技术条件》等9项标准

# 内容简介

# 1. JB/T7261—94铝制板翅式换热器技术条件

本标准经机械工业部审批为机械行业标准。标准规定了气体分离与液化设备、石油化工设备及机械动力装置等用盐浴钎焊法或真空钎焊法制造的设计压力不大于  $6.3\mathrm{MPa}$ ，设计温度为  $-270\sim 150^{\circ}\mathrm{C}$  的铝制板翅式换热器的技术要求、试验方法和检验规则及标志、包装、运输和贮存。

本标准适用于铝制板翅式换热器。采用其它钎接工艺制造的换热器亦可参照执行。

# 2. JB/T1035—94铜制空气分离设备制造技术规范

本标准经机械工业部审批为机械行业标准。标准规定了空气分离设备中采用铜或铜合金材料制造的筛板式精馏塔、直管式冷凝蒸发器、横流多层管式换热器以及上述相类似的铜及铜合金设备的制造技术要求、检验规则和试验方法及附录A产品试板的制作与试验。

本标准适用于空气分离设备中采用铜或铜合金材料制造的容器和设备。

# 3. JB/T2549—94铝制空气分离设备制造技术规范

本标准经机械工业部审批为机械行业标准。其内容包括：主题内容与适用范围、引用标准、铝制设备制造技术要求、检验项目和试验方法。

本标准适用于铝制空气分离设备的筛板式精馏塔、统管式热交换器、列管式热交换器、吸附器、自动阀箱及管道等。

# 4. JB/T7266—94空气分离设备铜焊射线照相和质量分级

本标准经机械工业部审批为机械行业标准。其内容包括：主题内容与适用范围、引用标准、从事射线照相检验人员要求、检验焊缝的表面要求、射线照相技术要求、照相质量要求、照相底片的观察、焊缝质量分级及射线照相的防护。

本标准适用于空气分离设备中铜制容器和铜制管道的对接焊缝，其它母材为铜基合金的熔化焊对接接头也可参照使用。

# 5.JB/T7262—94铝制板翅式换热器型

# 号编制方法

本标准经机械工业部审批为机械行业标准。其内容包括：主题内容与适用范围、引用标准、型号编制方法和示例。

本标准适用于铝制板翅式换热器的型号编制。

# 6. JB/T7263—94空气分离设备用13X分子筛验收技术条件

本标准经机械工业部审批为机械行业标准。其内容包括：主题内容与适用范围、引用标准、技术要求、试验方法和检验规则、包装、标志、运输和贮存。

本标准适用于空气分离设备用以吸附气体中的水分、乙炔、二氧化碳杂质的13X分子筛。

# 7. JB/T 7284-94回热式制冷机技术条件

本标准经机械工业部审批为机械行业标准。其内容包括：主题内容与适用范围、引用标准、技术要求、试验方法和检验规则、包装、标志、运输和贮存及附录A制冷量的测定(补充件)。

本标准适用于立式单级回热式制冷机。

# 8. JB/T50016-94往复活塞中压氧气压缩机产品质量分等

本标准经机械工业部审批为机械行业标准。标准规定了往复活塞中压氧气压缩机产品质量等级、试验和检验方法、抽样评定方法。

本标准规定了产品主要性能指标的质量等级考核项目。质量等级分为合格品、一等品两个等级。

本标准适用于往复活塞中压氧气压缩机产品质量分等。

# 9. JB/T50017—94铝制板翅式换热器产品质量分等

本标准经机械工业部审批为机械行业标准。标准规定了铝制板翅式换热器产品的质量等级，试验方法和抽样评定办法。

产品根据其质量水平和使用价值分为合格品、一等品、优等品三个质量等级，规定了10项产品性能指标的考核项目。

本标准适于铝制板翅式换热的产品质量分等。
