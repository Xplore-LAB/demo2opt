# 温州氧化铝厂活性氧化铝替代进口获得成功

武钢(集团)公司氧气厂两台  $30000\mathrm{m}^3 /\mathrm{h}$  空分设备(E、F台)，系从联邦德国公司引进，先后于1993年10月投产。其设备的两个空气纯化器为卧式圆柱双层床结构，内充入  $13\mathrm{X}$  分子筛和活性氧化铝两层。底层装活性氧化铝，床层高为  $200~\mathrm{mm}$  ，装入活性氧化铝约7000公斤，用于吸附加工空气中的大部分水分；上层装  $13\mathrm{X}$  分子筛，床层高  $650~\mathrm{mm}$  ，装入  $13\mathrm{X}$  分子筛约21000公斤，用于吸附剩余水分、二氧化碳和大量碳氢化合物。分子筛和活性氧化铝层间用20目的不锈钢丝网隔开。分子筛和活性氧化铝的更换或补充可从两只纯化器圆柱形筒体两头的人孔进行。运行时，两台纯化器交替切换使用（自动切换程序控制），一只使用，另一只加热或冷吹。每只使用的时间约2小时，加热时间约35分钟，冷吹时间约1小时5分钟，切换时间约20分钟，一个周期约为4小时。

原来，两只纯化器装入的活性氧化铝和13X分子筛是随设备引进的，其价格十分昂贵。

由于停电事故，造成计算机失灵误动作，导致E台空分设备的纯化器进水，虽然处理后能正常生产，但吸附水分和二氧化碳的效率有所下降。1996年10月，打开人孔进行检查，发现分子筛和活性氧化铝有部分粉尘和变色(较深)现象。为了确保长期运行，决定更换全部的分子筛和活性氧化铝。

如果用进口德国的活性氧化铝和13X分子筛，其价格十分昂贵，而且到货周期长，远水解不了近渴。于是，决定选用国产活性氧化铝和分子筛替代进口。为了解决国产替代活性氧化铝的厂家和来源，我们首先根据林德公司提供的技术标准，与国内活性氧化铝生产厂家提供的技术标准进行比较，向中国空分设备公司和杭州制氧机研究所进行技术咨询，同时对国内的部分活性氧化铝生产厂家的生产工艺和产品质量以及主要用户进行考察和咨询，

最后选用浙江温州氧化铝厂生产的空分用活性氧化铝替代进口。其主要技术指标如下：

① 型号 WHA-101空分用活性氧化铝  
② 静态吸附容量  $\geqslant 16\% (\mathbf{R} \cdot \mathbf{H} = 60\%$  时）  
③ 粒度  $\phi 3 \sim 5 \mathrm{~mm}$  球型  
④ 强度  $\geqslant 80\mathrm{N}$  （点压/个球）  
⑤ 耗耗  $< 0.4\%$  
⑥ 堆比重  $\geqslant 0.7\mathrm{g / cm}^3$  
⑦ 孔容积  $\geqslant 0.38\mathrm{cm}^3 /\mathrm{g}$  
⑧ 比表面积  $\geqslant 300\mathrm{m}^2 /\mathrm{g}$

为了便利与进口活性氧化铝和分子筛的性能进行比较，决定E台空分设备的A纯化器仍然使用进口分子筛和活性氧化铝；B纯化器选用温州氧化铝厂生产的空分用活性氧化铝和上海浦江分子筛厂生产的13X分子筛。活性氧化铝装入量约7000公斤，13X分子筛装入量约21000公斤。

1996年11月6日，  $30000\mathrm{m}^3 /\mathrm{h}$  空分设备（E台）检修更换分子筛和活性氧化铝完毕，投入运行。新、旧两台纯化器经过两个周期活化，其出口的二氧化碳和水分含量均小于  $1\mathrm{ppm}$  。1997年1月15日，又对E台空分设备的纯化器进行了最大空气处理量的试验运行，累计量从25.7万  $\mathrm{m}^3$  （正常值）增加到35万 $\mathrm{m}^3$  ，使用时间从2小时增加到2.5小时，其出口空气中的二氧化碳和水分含量均符合标准。实践证明： $30000\mathrm{m}^3 /\mathrm{h}$  空分设备E台的A、B两只纯化器的使用效果相当，均能满足空分生产条件要求。温州氧化铝厂生产的空分用活性氧化铝技术指标，与进口的相近，价格适宜，交货及时，售后服务好，安全可以替代进口氧化铝，可以在大型空分设备上推广使用。上海浦江分子筛厂的13X分子筛也可以替代进口分子筛，在大中型空分设备上推广使用。

# 一、前言

80年代末，随着我国国民经济的快速增长，对外开放的继续深入，工业气体的需求量越来越大。近年来国外气体公司看中了我国的广大市场，以合资、独资、三资形式在国内兴办气体企业，他们以各自的技术优势及资金和管理优势，投入资金，引进技术和设备。为了进一步占领气体市场，实行区域割据，各企业都需要生产大量液体，以便于贮存及远程运输，这给生产液体空分设备及气体液化设备的厂家带来机遇。为了尽快适应市场的需求，杭氧率先着手进行液体空分设备的开发和研究，并获得成功，目前在国内外已有了一定的用户和一些新的订单。

液体空分设备产品，具有便利贮存、供应方便、减少放散、保证质量、提高输送效率等优点，将会在空分设备中占更重要的地位。

近几年来，随着透平机械的不断发展，出现了效率较高的中高压透平压缩机，同时又开发了带增压的透平膨胀机和高中压板翅式换热器，这对提高整机效率，改善传热起着重要作用。另外，作为预冷冷源的冷冻机

的性能也不断得到改善，效率不断提高，所有这些技术的发展，为现代中大型液体空分设备的开发奠定了坚实的基础。

液体空分设备的设计，不仅要考虑有较高的液化系数和较低的能耗，同时还要操作方便，降低设备投资。总之，应根据装置大小的不同和用户的具体要求，来确定最佳流程和配套形式。

这里将对几种流程形式作一介绍。

# 三、液化循环分析

上述图1～图6的流程配有液化制冷循环，可统称为膨胀机液化循环流程。膨胀机性能参数对整体流程有很大影响，作为整个装置的产冷设备，膨胀机本身性能的改进，等熵效率的提高，将会提高整体流程的性能。但是在我们流程设计中，必须通过调整流程的参数，最大程度地发挥膨胀机的效果和作用，也就是更大地提高膨胀机的制冷量。

我们知道，膨胀机的制冷量主要取决于膨胀量和焓降，膨胀量占循环总量的比例M是一个十分重要的参数。显然，若  $\mathbf{M} = 1$  则节流量为零，那么就意味着液化系数为零，不能制取液体。而膨胀机的焓降，决定于膨胀前温度。膨胀前温度越高，则焓降就越大，这就是高温高焓降效应。在我们所取的压力范围，同样存在这样的效应。因此，选取合适的膨胀量及膨胀前温度，以使膨胀机制冷量达到最大，这样就能最大程度发挥膨胀机的作用。

设计液体设备另一个重要因素，是换热器的换热工况，也就是换热器的内部温度分布和温差。对于图1、图2流程，由于其循环压力比较低，空气在冷凝过程中存在等温液化过程，为了满足进下塔需要的液体量，使得冷端的温差比较大。

对于图3～图6流程，由于循环压力比较高，高于其临界压力，比热变化剧烈，尤其是正返流间的压差较大，两股气流间的比热相差比较多，正返流气体换热时其温度分布比较复杂，两者间的温差变化较大。换热器温度工况，可用热量温度图(G△h--T图)表示。

换热器正常工况是指在换热器内高压气体热量能传递给返流低压气体，而且在换热器任一截面上高压空气(热气体)温度总高于低压空气(冷气体)温度，其温差在一定范围内，如图7a曲线A热气体，和曲线B（冷气体）所示。如果选取的参数不当，可能使曲线A与曲线B相交如图7b，图上影线部分出现所谓“负温差”。若液化参数选择不当，在设计中“负温差”就可能出现。换热器的内部温差及温度分布，也反映了整体流程的性能，其温差的大小直接反映了不可逆损失程度的大小。

制冷机的设置与否对液化循环性能有较大的影响，从热力学第二定律知道，制取一定的冷量所消耗的功与其品质有关，也就是与制冷温度有关。制冷温度越高，所消耗的功越少，因此在低温设备中，所需的冷量尽量在较高的温度下制取，高温下的冷损用高温下的冷量加以补偿，这样就比较经济。在液化循环中，其本身所采用的制冷循环，其制冷温度是比较低的，所消耗的功也大。而目前常用的冷冻机其制冷效率比较高，所消耗的功比较少，因此从这个角度上来说，在液化循环中，能引入冷冻机的冷量总是能使整体制冷效率提高。

此外，原料空压机、循环压缩机、分子筛纯化器、精馏塔等的性能也是影响液体设备整体性能的重要因素，这里不再作分析。

总之，一套液体空分设备性能的好坏，取决于流程选择是否合适，流程参数选择是

否合理，能耗指标是否已达到最小，及设备占地面积小，投资省，操作维护方便等，以此来进行综合衡量。

另外，常规空分设备流程加液化设备也是一种好的搭配，用来生产液体，尤其是对已有常规空分设备的用户更加适合。

# 一、液氯内压缩流程简介

（浙江大学材料系，杭州玉古路20号，310027）

摘要 奥氏体不锈钢的低温力学性能优良，很适合于制造低温贮槽和装置，空分行业中用于贮存液氮、液氧、液氩等的真空粉末绝热低温液体贮槽就是由奥氏体不锈钢制成。本文详细介绍了这类贮槽的焊接技术。图2表1。

# 主题词：奥氏体不锈钢 真空粉末绝热 低温液体贮槽 焊接

气体常常以液态形式予以贮存和运输，氧气的液化温度为  $-183^{\circ}C$  ，氩气的液化温度为-  $186^{\circ}C$  ，氮气的液化温度为  $-196^{\circ}C$  。常见的液氧、液氩、液氮的低温贮槽为真空粉末绝热式贮槽，空分行业中有许多大大小小的这类贮槽。其中，固定式真空粉末绝热低温液体贮槽的容积，通常多在  $5\sim 100\mathrm{m}^3$  之间。这类贮槽由内旦容器、外壳、管路及蒸发系统等组成，内旦容器多为奥氏体不锈钢制成，外壳多为16MnR、16Mn、Q235-A. F～Q255-A.F等制成，管路系统多为奥氏体不锈钢管制成。

根据相应标准，真空粉末绝热低温液体贮槽的制造、检验及验收要求较高，内旦容器以及汽液管道系统，必须彻底清除固体杂质，应无焊渣、杂物；处于真空系统的零、部件均需进行清洁处理，表面不得有油污、铁屑、焊渣、氧化皮和其它杂物；对接焊缝应作射线探伤，无法进行射线探伤的对接焊缝，如外壳最后复合的环焊缝，必须进行超声波探伤，等等。因此，正确选用焊接方法，制定相应的焊接工艺，以满足相应标准的要求，是十分重要的。

# 一、贮槽结构特点及焊接要求

根据设计参数的不同，内旦容器通常属I类压力容器或Ⅱ类压力容器。由于工作在

低温环境下，故主要的问题是容器的低温韧性问题。国内过去习惯用1Cr18Ni9Ti作为制作内旦容器的主体材料，现在已逐渐越来越多地选用0Cr19Ni9、0Cr18Ni11Ti等低碳奥氏体不锈钢。奥氏体不锈钢的低温韧性是很好的，故相应标准规定“奥氏体高合金钢制低温容器在设计温度高于或等于-196℃时，免作冲击试验”。[1]

内旦容器主要由封头、筒体、管接头组成，无人孔，通常以气体代替液体作耐压试验，故此所有对接环缝、纵缝均作  $100\%$  的射线探伤，管接头与封头或筒体之间的角焊缝作着色探伤。对接焊缝的焊缝系数可取1.0，除容器最后复合的对接环焊缝之外的所有对接环缝、纵缝，可采用双面焊工艺保证全焊透，容器最后复合的环焊缝由于不能采用双面焊工艺，故此只能采用单面焊双面成形工艺保证全焊透。管接头与封头或筒体之间的角焊缝一般也要求全焊透，也可采用双面焊工艺来保证全焊透。

真空粉末绝热低温液体贮槽外壳主要也是由封头、筒体、管接头等组成，无人孔，管接头与封头或筒体之间的角焊缝通常进行着色检查，外壳容器对接纵缝作射线探伤，对接环缝除最后复合缝外也作射线探伤，最后复合环缝进行着色或超声波探伤。同内旦

容器一样，除最合复合环缝之外的所有对接纵缝、环缝均采用双面焊工艺来保证焊接质量，容器最后复合对接环焊缝则采用单面焊双面成形工艺。

产品设计参数不同，内旦、外壳壁厚也不同，通常内旦壁厚  $4\sim 30~\mathrm{mm}$  ，外壳壁厚 $6\sim 12\mathrm{mm}_{\circ}$

# 二、奥氏体不锈钢的焊接

# 1. 奥氏体不锈钢的焊接性分析

一般来讲，奥氏体不锈钢的焊接性良好，但在不同的应用场合，使用性能要求不同，焊接所要注意的问题也不同。对于低温贮槽，最主要的问题是焊缝热裂纹、焊接接头塑性及低温韧性问题。

对于低温贮槽用奥氏体不锈钢，提高焊缝抗裂性的最有效手段是选择相应的焊接填充材料(焊丝或焊条)，使焊缝成为  $\gamma$  （奥氏体）加δ（铁素体)双相组织，但也必须控制 $\delta$  相数量，以保证焊缝韧性。这一点将在下文细述。

焊接奥氏体不锈钢时，应对焊接线能量予以控制。单相奥氏体不锈钢易形成方向性强的柱状晶的焊缝组织，易导致有害杂质的偏析。控制焊接线能量，避免接头出现过热组织，一方面是保证接头的韧性，另一方面是保证接头的塑性。奥氏体钢导热系数小而线膨胀系数大，为减小变形或减小焊接应力，应选用焊接线能量集中的方法，快速焊。选用较细的焊丝或焊条，对接头的力学性能大有好处。

# 2. 影响焊缝低温力学性能的主要因素

低温条件下工作的奥氏体不锈钢关键在韧性指标，即最主要的是防止发生低温脆性破坏。奥氏体不锈钢焊接接头熔敷焊缝金属的低温韧性与许多因素有关。

1）低铁素体含量 众所周知，奥氏体不锈钢焊缝中的铁素体组织，可有效抑制热裂纹，  $4\% \sim 8\%$  的铁素体组织抑制裂纹效果

最佳。但当铁素体组织超过  $3\% \sim 4\%$  时，会降低焊缝的低温冲击韧性。铁素体含量尽可能地少或焊缝为单相的奥氏体组织，低温韧性就高。兼顾到抑制热裂纹和提高焊缝低温韧性，较佳的铁素体含量为  $2\%$  左右[2]。另外，铁素体组织的分布和形态对焊缝低温韧性也有影响，但在实际焊接生产中要控制铁素体的形态和分布是非常困难的。

2）低碳 焊缝金属含碳量低，则低温韧性高。从这点看，为保证焊缝金属的韧性，在焊接材料选择时，应充分考虑其含碳量，使焊缝金属含碳量小于或等于  $0.03\%$  为最佳选择。  
3）低氮 氮可提高不锈钢焊缝金属的抗拉强度和屈服强度，降低其低温韧性。研究表明，焊接低碳或超低碳奥氏体不锈钢时，焊缝含氮量控制在  $0.05\%$  以内，可得到较高的低温韧性。  
4）高镍 研究发现，在一定的允许范围内，较高的含镍量可提高焊缝金属的韧性。当含镍量由  $10\%$  提高到  $20\%$  时，焊缝金属的韧性显著提高。不过高镍不锈钢填充金属，至今还未完全商品化。  
5）碱性焊条 手工电弧焊时，碱性焊条比氧化钛钙型焊条，具有更高的焊缝低温韧性，所以低温容器(装置)手工电弧焊时应选用碱性焊条。  
6）低的夹杂含量 对低碳不锈钢的研究表明，焊缝中低的夹杂含量可提高焊缝金属的低温韧性。焊接工艺对焊缝的夹杂含量有很大影响。例如，生产中发现，熔化极气体保护焊比埋弧焊，更容易满足  $-196^{\circ}C$  以下温度的低温冲击试验要求。这是因为焊缝中的夹杂物通常是一些氧化物，保护气氛的低氧化性有助于使氧化物夹杂减少到最低值。

# 三、贮槽的焊接技术

对于奥氏体不锈钢(1Cr18Ni9Ti、0Cr-19Ni9、00Cr19Ni11等)和16MnR一类的低

合金结构钢来说，可选用的焊接方法很多，从质量、成本、效率和生产条件几方面考虑，目前低温贮槽的焊接生产中主要选用埋弧自动焊(SAW)、手工电弧焊(SMAW)、手工钨极氩弧焊(TIG)、半自动熔化极富氩混合气体保护焊（MAG）等焊接方法，各种焊接方法有各自的特点，其应用范围也不尽相同。

# 1. 组焊顺序

贮槽有立式、卧式之分，其容积有大有小，制定焊接工艺时首先必须考虑产品的组焊顺序，特别是对于容积较大的贮槽，更应重视其组焊顺序。

内旦容器只有一条最后复合的对接环缝，封头、筒体制成后，焊接管接头与封头或筒体之间的角焊缝，并焊接筒体一封头或筒体一筒体对接环焊缝，将内旦组焊成两半截容器，所有焊缝检验合格并清理容器内部后封闭最后复合对接环缝。再对该焊缝进行双壁单投影X射线照相法探伤，若发现超标缺陷则进行返修，直至合格。外壳同内旦相似，封头、筒体等制成后进行组焊，组焊到何种程度，视产品结构而定。一般的较小容积的贮槽，只留一条对接环缝，待内旦容器套装完毕再施焊；较大容积的贮槽，如卧式 $100\mathrm{m}^3$  贮槽，为保证内旦套装，需留两条或4条对接环缝在内旦套入外壳，固定之后再施焊。

SMAW 通常适合板厚  $10 \mathrm{~mm}$  以下的对接焊, 板厚  $10 \mathrm{~mm}$  以上时用 SAW 方法, 效率高、经济、焊接质量稳定且易于控制, 板越厚, 优点越显著。焊接内且容器时, 无论是 SAW 还是 SMAW, 都要严格控制焊接线能量, 选择相应较细的焊条、焊丝, 避免接头出现过热组织, 以保证接头的韧性和塑性。生产实践证明, 内且容器纵缝焊接试板弯曲试验不合格, 往往与焊接线能量过大有直接关系。

目前所用富氯三元混合气为  $\mathrm{Ar} + \mathrm{O}_2 + \mathrm{CO}_2$  。 $\mathrm{O}_2$  和  $\mathrm{CO}_2$  的加入使保护气的氧化性增加，为保证内旦焊缝的低温韧性，必须限制

# 粗氩冷凝器工作原理分析和操作方法探讨

周智勇*

（杭州制氧机集团有限公司技开处，杭州市东新路90号，310004）

摘要 本文阐述了粗氩冷凝器的工作原理，从循环倍率的角度分析了粗氩冷凝器的液空液位、液空回流量、下塔液空含氧量等因素对粗氩冷凝器传热温差的影响。最后介绍了调整粗氩冷凝器工况的操作方法。图1。

主题词：粗氩冷凝器 循环倍率 液空液位 液空回流量 温差 压力 操作

粗氩冷凝器是氩提取设备的关键部机之一，对它的操作是否恰当，将直接影响到粗氩的产量和质量。严重时，甚至会影响到整个空分装置精馏工况的稳定性。

在实际操作中，经常会发生这样的现象，为了提高粗氩冷凝器的热负荷，也即提高粗氩塔的阻力，采用提高液空液位的方法。这种方法在粗氩冷凝器液空液位较低时是有效的，但一旦升高到一定的高度后，再继续提高液空液位，不但不会使粗氩塔阻力提高，反而会使其下降。为什么会发生这种现象呢？其实，只要我们分析一下粗氩冷凝器的工作原理和结构特点，就不难理解这种现象的发生。

# 一、粗氩冷凝器工作原理

粗氩冷凝器与主冷凝蒸发器比较，在结构上有相似之处。只是粗氩冷凝器蒸发侧的介质是液空，冷凝侧的介质是粗氩。粗氩获得冷量被冷凝，同时液空被蒸发。在蒸发侧的液空，是以一定的循环倍率在其通道内流动，即在通道内有大量液空在循环流动，加热汽化的只占小部分。如图所示。一般情况下，液空循环量与汽化量之比在  $7\sim 10$  倍。循环倍率越大，传热强度就越大，粗氩冷凝

![](images/6ce899d7052c0ae38f2daae91ea5d68f52587f52bcdc287b4ebd29bea2efc583.jpg)  
图1 粗氩冷凝器示意图

器的热负荷就越大，反映在粗氩塔上，就会使粗氩塔阻力升高。在冷凝器结构一定的情况下，循环倍率主要受粗氩冷凝器的温差和液空液位两个因素的影响。

# 二、粗氩冷凝器温差对循环倍率的影响

液空在蒸发通道内受冷凝侧粗氩气的加热，液空得到热量，温度升高并部分汽化，使其比重变小。这样，通道内的压力小于通道进口处的压力，使液空在通道内向上流动。所以粗氩冷凝器的温差是产生循环倍率的前提，没有温差，液空就不会自动循环。而粗

氩冷凝器的温差，主要由液空的组分、蒸发侧的压力决定的，当然还受液空液位的影响，这将在后面讨论。

粗氩冷凝器蒸发侧的液空组分在  $60\% \sim 70\% \mathrm{O}_2$  的范围内，在一定的压力下，不同的含氧量就会有不同的蒸发温度。影响液空的含氧量主要有两个因素：一个是下塔液空的含氧量，下塔液空含氧量越高，蒸发侧的液空含氧量也就越高，使冷凝器的蒸发温度升高，平均温差减小；另一个是液空回流量的大小，在产品设计时一般已考虑了取液空总量的  $2\% \sim 10\%$  作为回流液空量。一般情况下，这个百分比每增加一个百分点，会使液空中的含氧量减小  $0.3\% \sim 0.4\%$  个百分点，相应地使蒸发侧的温度降低，从而达到扩大冷凝器温差的目的。

液空蒸发侧压力对冷凝器蒸发温度的影响，一般情况下，蒸发侧的压力越低，越利于提高冷凝器的温差。但蒸发侧的压力不是随意可以降低的，它是受到粗氩冷凝器液空蒸汽回到上塔相应截面的压力所约束的，也即是一一对应的。然而在有限范围内适当调高蒸发侧压力倒是可以的，那就是有的用户要求粗氩塔设计时应考虑较大幅度减负荷运行工况时才适当提高液空蒸发侧的压力，以减小冷凝器的温差，达到减负荷的目的。在这种情况下，就必须在液空蒸汽返回上塔的管线上加只控制阀，才得以实现。

# 三、液空液位对循环倍率的影响

为了使液空在板翅式换热器液空侧通道内以一定的倍率循环流动，在保证与冷凝器冷凝侧有一定温差的前提下，还必须保持有一定的液空液位。由于这个液位的存在，使蒸发侧底部有一定的压力  $\mathbf{P}_2$  ，以克服液空在板式翅片内流动时的阻力。一般情况下，随着液空液位的提高，液位底部的压力  $\mathbf{P}_2$  也会逐渐提高，液空流动加快使循环倍率提高，达到强化冷凝器传热的目的，有利于粗

氩冷凝器热负荷的提高。但当液位提高到一定高度以后，由于  $\mathbf{P}_2$  压力的提高，液空底部的饱和温度升高，使蒸发侧的平均温度升高，缩小了粗氩冷凝器的平均温差。这样，不但没有使粗氩冷凝器的热负荷提高，反而使其下降，严重时还会使粗氩塔的工况受到破坏。

在了解了粗氩冷凝器的工作原理和液空液位、回流液空量、下塔液空含氧量等因素的影响后，在我们的实际操作中，就比较容易灵活掌握调整粗氩冷凝器和粗氩塔的工况。

首先必须保证粗氩冷凝器蒸发侧与冷凝侧有一定的传热温差。可采取以下措施：

1. 使上塔处于相对稳定的工况，尽可能使粗氩气中存在较少量的氮气，一般控制在  $1.5\% \mathrm{N}_2$  以下。  
2. 调整下塔工况，下塔液空中的含氧量控制在  $35.5\% \sim 37.5\% \mathrm{O}_2$  范围内。  
3. 通过开大或关小粗氩冷凝器的液空恒流阀，来调节液空蒸发侧液空的含氧量。

其次，在调试时，可进行调整液空液位高度的试验。即在保证其它工况正常的情况下，使液空液位从低位到高位缓慢提高，开始时随着液位的提高粗氩塔阻力也会同步提高，但到一定的高度以后，若继续提高液位时会使粗氩塔阻力下降，说明在这种情况下，你若想继续提高粗氩塔的阻力，那么只能采用前面提到的第  $1\sim 3$  种方法来调节。了解了这个转折点以后，就可以使我们避免错误的操作。

另一个障碍是LNG的价格竞争，它必须便宜到足以与其它能源相竞争，而天然气生产者在足够高的价格下去谋取利润，因此要补偿这两种价格之差价，则必须使整个LNG链的费用保持足够低的水平。

# LNG链

LNG链主要由以下环节组成：

1. 从气源到液化装置的天然气生产与集输的成套装置。  
2. 液化装置，该装置装在出口国沿海地区。还包括下列设施：  
$\bullet$  原料气纯化装置，除去腐蚀性的、隐性的或可能在液化过程中会固化的物质。  
生产LNG的装置。  
贮槽与槽船装载设备。  
公用事业（热量排出装置，电、空气、氮、软水等）。  
$\bullet$  通用设施（行政建筑、车间、仓库、实

验室、住宅区、飞机场等）。

3. 甲烷槽船，其数量根据运输距离而定。  
4. 装在进口国沿海地区的终端再气化装置。该装置贮存从甲烷船上卸下的LNG，并经汽化再输送到用户。

透平压缩机

液化装置的生产容量常常受制冷剂透平压缩机(简称透平，下同)性能的限制，但其容量还是随着压缩机性能与其驱动透平的改进而提高。对第一套液化装置  $0.43\mathrm{M}\mathrm{t} / \mathrm{y}$  、23MW，到最近的大于  $3\mathrm{M}\mathrm{t} / \mathrm{y}$  、110MW的装置均如此。

第一套液化装置装了蒸汽透平，那时蒸汽透平只是作为驱动机，足够可靠，容易操作，单位容量足够高，转速可以选择，并能直接驱动制冷压缩机。

然而气体透平已取得可观的进展，它们的单位容量已有相当的提高，现在它们的效率已超过  $30\%$  ，并变得更可靠。事实上，从80年代起，气体透平用于驱动制冷压缩机的效果还是比蒸汽透平好。

两轴透平已开始使用，得益于它们可变速与低的启动力，它们更适用于驱动压缩机。但其动力输出作“工业”透平仅限于28MW，作喷射透平能到34MW。为了较低的成本，减少回转机械，单轴透平是适宜的，因为它们的单机容量常可达到100MW。这是当前液化项目的趋势，特别是在马来西亚、尼日利亚、卡塔尔以及阿曼。

低温热交换器

用于天然气液化装置上的热交换器，必须符合以下非常特殊的操作条件：

$\bullet$  一股多股冷凝液逆流而蒸发为汽液混合物。

$\bullet$  进出口流之间有大温差  $(>80^{\circ}\mathrm{C})$  。

$\bullet$  不同循环流之间有小温差  $(< 5^{\circ}C)$  。

低温（低到  $-160^{\circ}\mathrm{C})$  。

换热量非常大(>70MW)。

高流量率  $(>1000\mathrm{T / h})$  。

有两种型式的换热器可用于该装置，即绕管式换热器与铝制板翅式换热器。板翅式比绕管式换热器具有吸引力，特别是它的紧

凑性。在给定的换热面积下，它减少了安装面积，同时成本也比较低，因此除大型天然气液化装置之外，几乎在所有的低温工程应用中，板翅式换热器已替代了绕管式换热器。但尽管采用真空钎接，质量有了改善，板翅式换热器与绕管式换热器相比，还是有所不足。即他们的尺寸受钎接设备的限制，这就意味着许多只板翅式换热器，必须平行地布置，可以想象要达到很多只换热器之间足够均匀的分配是多么的困难。此外，板翅式换热器与绕管式换热器相比，对污染与热冲击更敏感。LNG建造者注重可靠性与操作简单，因此更偏袒具有较大耐用性的绕管式换热器，而不顾其较高的价格。

LNG 贮存

虽然LNG是危险最少的碳氢化合物之一，而且在LNG装置上很少出事故，也不会导致更大的事故。然而，其能量积蓄在LNG贮槽中，可能突然释放而造成重大的事故。

因此，安全是一个长期优先考虑的问题。LNG贮存装置在设计、制造、操作中对贮槽来说要避免任何差错，能使导致迅速出现事故的因素处于控制之下。例如，在贮槽LNG液面低时就不能开启，所有进出口管道全部要在贮槽顶部，所有泵都要浸在LNG中。

安全，这个永恒关心的事，已有不同的工艺技术予以发展，能在室温下建造，在操作温度即LNG温度  $(-160^{\circ}\mathrm{C})$  下工作，而确保无事故的危险。这些工艺技术分为两大类即自支承贮槽与薄膜贮槽。

自支承贮槽

这种工艺技术专门用于地上贮存。其内筒一般由含镍  $9\%$  的钢制成，这不仅适用于低温(充分低的转变温度)的一种相对便宜的钢材，而且是因它能抗裂。

在第一台设备中，其外筒采用普通钢，它只是作为防止水蒸气进入绝热层的障壁。

现在外筒通常使用预应力混凝土，它能抗击外来的最坏事故，包括炸弹、飞机坠毁等。此外，为提高安全性，“双保险”的概念已经使用，它可允许内筒可能的破裂而导致LNG在内外筒之间的积聚。因此，内外筒必须抗冷，并承受水压。

薄膜贮槽

这项工艺技术，是在绕性薄膜层中装上LNG，通常此膜由薄壁不锈钢制成，这种薄壁不锈钢压成浮波以抵抗材料的热收缩。水力静压应力通过冷绝缘作用到外壳上，因此必须有较高的抗压缩性能。

某些公司不愿采用此项技术，声言虽然薄膜式贮槽与半支承式贮槽的需要是不同的，但薄膜式贮槽不能满足“双保险”的需要。因此地上贮存不常采用薄膜贮槽，而地下贮槽采用很多，特别是在日本。

LNG 贮槽的操作

如果不同性质的LNG连续装入贮槽，那么LNG贮存在再汽化终端站的贮槽中，形成不同密度的层次。在一定的条件下，这种分层状态能发生突然的不稳定，形成层次的翻动，导致大量的LNG的突然蒸发。这种现象称之为“滚动”（rollover），是应该避免的，因为它会造成危险事件。为预防这种现象，发展了模拟程序，可预先测量以及时采取措施，向槽内上部或下部灌充液体，或重新混合不同的装载液体。

槽船

现在，槽船技术的发展，已集中在他们的通用设计与船装系统。

通用设计

象液化装置一样，改进目标是降低投资与运行费用，特别是降低蒸发损失与燃料消耗。下面的实践已应用于现代的槽船上：

改进绝热降低蒸发损失，从每天总装载量的  $0.25\%$  降到  $0.15\%$  。

$\bullet$  减少装载贮槽的数量，从5个减少到

4个，这样有较小的制造成本，同时有助于蒸发量的减少，但这意味着水下船身的形状更加短而粗。

$\bullet$  降低燃料消耗有许多研究，用柴油机替代蒸汽透平，但至今还没有实践。

降低槽船的数量，提高运输能力而满足一定量的需要，也能降低投资费用，然而大部分槽船的容量仍保持在120000到130000 $\mathrm{m}^3$  ，尽管对  $200000\mathrm{m}^3$  进行了许多研究，但最大的一条船不大于  $137500\mathrm{m}^3$  。

船装系统

装载LNG贮槽是基于陆地上贮槽的相同原理，薄膜式贮槽与半支承式贮槽各约占市场的一半。

“薄膜式”技术普遍使用由Gaz. Transport公司发展的技术，采用高镍不锈钢薄膜，而大部分使用的半支承贮槽技术，包括最大的槽船（ $137500\mathrm{m}^3$ ）是采用了Moss-Rosenbry的技术。它是球形  $9\% \mathrm{Ni}$  钢槽，以筒体裙座支撑在赤道平行线上，这样可吸收槽处于低温而船体处于常温而产生的不同热收缩。棱柱形半支承新型铝贮槽，由IHHI(Ishlkawajima Harlma Hcavy Industries)发展，已第一次用于  $87500\mathrm{m}^3$  的槽船上，最近已替代老槽船而服务于阿拉斯加与日本之间。这条线还有80年代Conch公司发展的铝棱柱形槽船。

随着改革开放的深入，国有企业也在逐步受市场经济的制约，在生产方式上也必将由过去的粗放管理，转变为精细管理；由过分强调“安全第一”，转变为既要保证安全又要考虑成本效益。这样，在公司内部各厂家节约用氮气的情况下，使得我厂开一套装置就基本上满足了外供氮气的需要，这为我厂全年利税任务的完成打下了坚实的基础。但是，公司领导根据以往的经验，担心一套空分装置的运行一旦发生故障，备用空分装置又不能很快的出产氮气，而中断供氮将给生

产全局带来很大影响。

为此，我们根据现有空分装置的实际情况，提出了空分装置长期“冷备车”方案，这样可以大大缩短备车的产氮气时间，得到了公司领导的认可。下面把我们的做法，作一介绍。

# 一、“冷备车”包括三个方面的内容

1. 待停空分装置停车后，不进行装置的升温，装置保持低温状态。  
2. 将运行空分装置的污氮气总管排放阀连接至“冷备车”空分装置的污氮气总管排

放阀。

3. 开车时将液氧(或液氮)罐内液氧(或液氮)压回至“冷备车”空分装置的主冷内。

# 二、在实施“冷备车”方案时注意的问题

在实施“内容1”时要提醒注意的是：

1. 停车前 注意分析一次主冷液氧内总碳含量，特别是乙炔、乙烯含量，超过规定值时一定要将液氧排放掉，以确保空分装置的安全。  
2. 装置停车后要自始至终保持系统处于正压状态，全关系统阀门，以防外部的湿空气进入系统内。装置保持正压的方法可用将氮气取出阀稍开，以便与运行空分装置的常压氮气球罐相连（装置停车初期采用此法比较好）；另一种方法，就是将两套空分装置的污氮气连通管阀稍开（一般在系统升至 $-80^{\circ}\mathrm{C}$  以上时采用此法）。  
3. 保冷箱珠光砂内要充密封氮气或污氮气。

在实施“内容2”时要提醒注意的是：

1. 注意运行空分装置冷量的平衡。  
2. 注意调节运行空分装置主换热器的热端温差，使出装置的氧气、氮气、污氮气与进装置的空气温度差在  $2 \sim 3^{\circ} \mathrm{C}$ 。  
3. 保持运行空分装置的主冷液面，有下降趋势时要关小污氮气连通阀，或适当加大膨胀量。  
4. 要注意“冷备车”空分装置系统温度的合理分布，以使得主冷、上下塔、主换热器冷端温度最低，最大限度地回收冷量，具体调节回路有：

氧气回路：上塔中下部  $\rightarrow$  主换热器冷端 $\rightarrow$  氧气取出阀。

氮气回路：上塔上部  $\rightarrow$  液空过冷器  $\rightarrow$  主换热器冷端  $\rightarrow$  氮气取出阀。

空气回路：液氮节流阀  $\rightarrow$  液氮过冷器  $\rightarrow$  主冷  $\rightarrow$  下塔  $\rightarrow$  主换热器  $\rightarrow$  分子筛过滤器  $\rightarrow$  排放阀。

污氮气回路：污液氮节流阀  $\rightarrow$  污液氮过冷器  $\rightarrow$  下塔  $\rightarrow$  主换热器  $\rightarrow$  污氮放空阀。

在实施“内容3”时要提醒注意的是：

1. 液氧罐内要定期做总碳分析，特别是乙烯与乙炔的分析（我厂指标要求，当主冷液氧内总碳分析大于  $100\mathrm{ppm}$ ，液氧不再排入罐内）。如返液氮则不存在此问题。  
2. 冷备车装置开车后，当主冷达到一定的低温（最好是已开始积累液体），才可考虑将液氧罐内的液氧返至主冷内，否则将会造成大量液氧蒸发，使总碳等有害杂质浓缩，给装置带来潜在危险。如返液氮则不存在此问题。  
3. 返液氧(或液氮)时要注意液氧(或液氮)罐的压力调节，液氧（或液氮）罐增压阀要开度适当，返液时一定要有人监护，防止超压。

通过这次“冷备车”的实施，我们认为“冷备车”有如下几点好处：

1. “冷备车”状态下开车，比常温状态开车要提前约40小时出产品，减少了一次装置大加温约24小时，这样仅空压机一项就可节约电费  $68 \times 2500 \times 0.5 = 85000$  元。  
2. 使空分装置的备车更具有实际意义，当运行空分装置工况出现异常时，可在短时间内开启“冷备车”装置，避免因空分装置的临时检修而影响下游装置的生产。  
3. “冷备车”状态的装置设备长期处于稳定的低温状态，对设备的长期运行有好处。

4. 大大减少了操作、检修工作量，避免了因此而出现的种种问题。

# 四、结束语

空分装置改造为分子筛流程后，分子筛对乙烯、乙炔有明显的吸附作用，带入装置的二氧化碳与水的含量，也远小于全板式（或蓄冷器）流程主换热器冷端带入系统内的含量(见表1)，这为空分装置的“冷备车”提供了先决条件。

我们认为，对于非分子筛流程的空分装置，不宜采取“冷备车”。这是因为对于正在运行的全板式(或蓄冷器)流程的空分装置，取出污氮气补给“冷备车”装置，势必影响自身的自清除能力，影响装置的正常运行；而对于“冷备车”的全板式(或蓄冷器)流程的空分装置，由于自身存在一定的水分和二氧化碳深冷霜，这样在“冷备车”期间很容易引起溶化或升华而富聚于某处，造成管道、阀门、换热器通道的堵塞，严重时涨裂设备，最终导致“冷备车”的失败。

徐铁燕

（鞍钢设计研究院燃气室，辽宁省鞍山市，114021）

摘要 介绍了鞍钢引进的美国产  $35000\mathrm{m}^3 /\mathrm{h}$  空分设备，用在水冷塔凉水的氮气和产品氮气一样，都是高纯氮气。本文介绍了拟减少入水冷塔  $20000\mathrm{m}^3 /\mathrm{h}$  氮气作为产品氮气。理论分析和实践操作表明，该方法是可行的。图2。

# 主题词：大型空气分离设备 水冷却塔 产品氮气 抽取 分析 实践

鞍钢氧气厂现有六台大型制氧机：两台  $35000\mathrm{m}^3 /\mathrm{h}$  制氧机、四台  $10000\mathrm{m}^3 /\mathrm{h}$  制氧机。每台  $35000\mathrm{m}^3 /\mathrm{h}$  制氧机产氧量35000  $\mathrm{m}^3 /\mathrm{h}$  ，产纯氮量  $25000\mathrm{m}^3 /\mathrm{h}$  ；每台10000  $\mathrm{m}^3 /\mathrm{h}$  制氧机产氧量  $10000\mathrm{m}^3 /\mathrm{h}$  ，产纯氮量 $10000\mathrm{m}^3 /\mathrm{h}$  。按鞍钢氧气用量组织生产，开两台  $35000\mathrm{m}^3 /\mathrm{h}$  制氧机和两台  $10000\mathrm{m}^3 /\mathrm{h}$  制氧机，共生产氧气  $90000\mathrm{m}^3 /\mathrm{h}$  ，纯氮气70000  $\mathrm{m}^3 /\mathrm{h}$  （均为标态，下同）。

生产的氮气经加压后，主要用在转炉气封、煤气回收、转炉顶底复合吹炼、高炉炉顶密封、冷轧罩式炉保护气体。随着生产的发展，新技术的应用，如：炼铁老煤粉车间改造、化工干熄焦工程、一炼钢平炉改转炉、转炉用氮气喷渣补炉等工程的投产和新技术的应用，氮气用量不断增加，达到氮气“产销平衡”，甚至出现供不应求的情况。因此开辟纯氮气新气源或回收污氮，增加氮气产量，是当前生产急需解决的问题。

增加氮气产量有多种途径，多开制氧机增产氮气，每多开一台  $10000\mathrm{m}^3 /\mathrm{h}$  制氧机，可多产纯氮气  $10000\mathrm{m}^3 /\mathrm{h}_{\circ}$  虽然不用投资，但氧气供大于求，造成氧气放散，生产运行费用高，增加氮气成本；新建制氮机组，投

资太大，亦不可取。因此增产氮气最为经济的措施，是在现有的设备上挖潜。

$10000\mathrm{m}^3 /\mathrm{h}$  和  $1^{\#}35000\mathrm{m}^{3} / \mathrm{h}$  制氧机，都是切换板翅式流程，产污氮。因氮气中含水，纯度低，使用受到限制，如果回收，新建的加压机和管道要和现有的纯氮气供应设施分开，形成单独一套系统，供给高炉气封等用户。这种方案是可行的，在首钢已实施，回收  $10000\mathrm{m}^3 /\mathrm{h}$  污氮气，用于高炉炉顶气封。因污氮含水，冬季有时阀门冻结，造成氮气供应中断。从美国空气产品和化学制品公司引进的  $2^{\#}35000\mathrm{m}^3 /\mathrm{h}$  制氧机是分子筛净化空气流程，该机组用在水冷塔凉水的氮气和产品氮气一样，是高纯氮气，水冷塔用氮气84096 $\mathrm{m}^3 /\mathrm{h}$  ，另有产品氮气  $25000\mathrm{m}^3 /\mathrm{h}$  ，纯度均为 $99.999\%$  ，拟减少入水冷塔氮气  $20000\mathrm{m}^3 /\mathrm{h}$  作为产品氮气。由于入水冷塔氮气量减少，为保证工艺用水量与水温要求参数，在水冷塔水泵后增加冷水机组，使水温仍保持  $8\%$  。或以新水代替现水冷塔使用的循环水，试算在水冷塔可抽出多少氮气作为产品氮气。

# 六、结论

1. 通过计算可以证明，在水冷塔出水泵后，加  $550 \mathrm{~kW}$  冷水机组，在水冷塔仍用循环水时，可增产氮气  $20000 \mathrm{~m}^{3} / \mathrm{h}$  。该措施投资少，见效快。  
2. 在水冷塔用新水代替循环水，需要实验测出出塔氮气温度及湿含量，同时用新水后能否会出现水冷塔和空冷塔结垢、堵塞等问题，有待解决。  
3. 理论分析及操作实践表明, 当减少入水冷塔氮气量  $20000 \mathrm{~m}^{3} / \mathrm{h}$  时, 水冷塔操作能满足该工况的变化。本文所讨论的两个措施是能够实施的。

# KL-15型制氧机加温操作的优化

钟家宝

（解放军87505部队四站教研室，成都市太平寺，610043）

摘要 介绍了杭氧产KL-15型制氧机在加温时机的选取、系统加温方法的掌握、三种流程加温方法上的区别及加温过程热-1阀、热-2阀配合使用的情况。

主题词：移动式制氧机 加温 空分器 吸附器 液氧泵 制冷机流程 方法改进 优化

杭州制氧机厂(简称杭氧)生产的KL-15型制氧机，整机原理系统性强、结构复杂。1988年后，该机又经杭氧、解放军后勤学院先后对局部进行过改造，因而在流程、电气原理和操作方法上较以往有所变化。在使用该机时，有的单位仍根据改装前的经验进行操作，加之人员成分新等情况，忽视了已经局部改进这个因素，导致工作中常常发生因加温不当提前冻结现象的发生，影响始动期及正常制氧(氮)工况。笔者就加温操作使用中的一些问题谈几点意见。

# 一、加温时机

制氧站对制氧车全面加温时，在排放尽塔内和贮液器内液体条件下，一般应选择在一个制氧周期结束后的  $1\sim 2$  天为宜。因为此时，装置已自然复热，此时为加温最佳的时机。

应该提出的是，停放一个月以上的制氧机，一般情况下开机前应重新加温，以避免停放期因系统不密封侵入湿空气而在工作中发生冻结的不良现象。

# 二、系统加温

# 1. 空分器的加温

空分器的加温应在制氧结束后，装置自然升温后进行，如加温过早，由于装置内部的温度非常低，一方面会使加温时间延长，

耗工耗能；另一方面低温状态下的装置因骤然受热，产生热应力而遭到破坏。如加温过迟，潮湿的装置容易生锈、腐蚀，缩短使用寿命，同时也影响正常制氧工作。

# 2. 制冷机部分的加温

制冷机测试产量或检修后，应单独加温制冷机的蓄冷器清洗后，应立即用常温吹尽后再加温干燥，以避免因脱脂后的潮湿蓄冷器与空气长期接触，使铜丝绒填料被腐蚀。加温后，电炉出口温度不超过  $50^{\circ}\mathrm{C}$  ，装上冷凝头后，加温应在用氢气置换数次后进行，并开水泵，保持系统有冷却水循环。

# 3. 分子筛吸附器的加温

该加温可放在下次制氧前，提前时间一般掌握在  $2\sim 5$  天，在阀门确实关闭的情况下，自然冷却为宜。在分子筛吸附器的加温过程中，应充分利用电炉热量（尽可能避免电炉自动切断电源）。从一开始就将进口温度控制在  $280\sim 300^{\circ}C$  （用抽真空阀控制加热空气的流量，使电炉温度升高，加热空气量增加；反之，减小）。

# 4. 停放时间不长的制氧机可直接启动

加温后停放时间在一个月左右的制氧机，如各阀门完好，关闭完全，没有湿空气进入，下次可直接开机，进入始动期操作。

# 5. 液氧泵的加温

液氧泵的加温是为充氮封存作准备，充氮封存的目的是为了在短期停放期间不拆液氧泵，避免缸套和柱塞生锈。液氧泵的加温在空分器加温结束前1小时（有吹-6阀）或1.5小时(无吹-6阀)进行。方法是：

（1）开充氧台阀、吹-6阀，拧松液氧泵气封填料螺母。  
（2）开热-3阀1至1.5转，1分钟后关闭通-6，关小吹-2阀到微开状态。在关闭通-6阀和关小吹-2阀过程中要注意观察贮液器的压力，防止超压。  
（3）打开通-9阀，通-10阀短暂开启后关闭。  
（4）用热-3阀调整贮液器压力至0.05MPa。由于空分器正在加温，贮液器的压力可用通-4阀和通-5阀配合调整。

加温结束关闭各阀，拧紧气封填料螺母。

长期停放的制氧机，液氧泵应拆下单独油封。

值得提出的是，每次制氧结束后，吹除液体，待装置自然复热后，立即对设备进行加温，不提倡用压缩空气对低温空分装置吹除到常温，然后停放到下次开机前再加温的习惯方法。

# 三、三种流程加温方法上的区别

1. 原流程。  
2. 1988年以后杭氧出厂和少量由该厂改造的制氧机流程：通-3阀位置不变，经通-3阀的正流空气从冷量回收到热交换器排出的环流空气(不经气泡泵加热体)汇合后，再经通-11阀调节，加热气泡泵，然后进入蒸发器冷凝侧通道。  
3. 后勤学院气体技术教研室改进的流程：改变通-3阀的控制位置，从可逆式换热

器排出的热空气用来加热气泡泵，然后接通-3阀调节，正流空气直接进入蒸发器冷凝侧通道。流程的改变使热-1阀的控制位置也略有改变，各流程经热-1阀的加热空气流路分别如下。

# 四、加温过程热-1阀、热-2阀配合使用

根据流程，经热-1阀的加热空气用于加温空气精馏系统，经热-2阀的加热空气用于加温可逆式换热器。加温过程中，热-1阀和热-2阀控制好坏，对加温影响效果很大。

1. 由于经热-1阀控制的加热气体部分或全部经过可逆式换热器的环流通道，因此

当可逆式换热器温度较低时，这股加热气体会在可逆式换热器中冷却，影响对精馏系统的加温。可见，要使经热-1阀的气体进入冷凝蒸发器之前不使温度降得过多，首先要升高可逆式换热器的温度。在加温开始时，应开大热-2阀和两只旁通阀。随着旁通阀出口气体温度的升高，逐渐关小，但要有一定的开度，以保持有一定的气体排出，不使温度下降。如果从旁通阀排出的气体温度上升过快，要检查通-7阀是否漏气或没有关闭，其现象是旁通阀上部管道发烫。

2. 返流管道上的单向阀及自动阀内两只通正流空气的单向阀是加温中的死区，可用关热-2阀，开大旁通阀，提高塔内压力的办法，使热-1阀控制的部分加热空气由塔内经通-1阀和通-2阀最后从旁通阀排出，达到加温吹除返流管道上单向阀的目的；用关旁通阀和热-1阀的办法，使热-2阀控制的加热空气经自动阀箱内返流单向阀后，再经过正流单向阀直入塔内，达到加温吹除自动阀箱内通正流空气单向阀目的。

# 2. 珠光砂进塔原因分析

$1^{\#}$  制氧机系统加温氮气、空分塔保冷箱珠光砂加温氮气、空分塔保冷箱密封氮气均

采用低压氮气，它来源于低压氮气系统，与氮压机进口氮气同属于一个氮气系统，并连接在同一根总管上。制氧机停机加温期间，因氮气供应量的变化曾使氮压机过吸，造成低压氮气系统的吸气总管出现负压。此时，系统加温氮气、空分塔保冷箱珠光砂加温氮气和密封氮气管道处于工作状态，加温氮气两根DN80管道孔状开口，直接和珠光砂接触，这样大量的珠光砂通过两根DN80加温氮气管道的氮气出口被吸入低压氮气总管。当氮气系统工况变化，低压氮气系统压力上升成为正压时，大量的珠光砂随系统加温氮气就进入了空分系统。

# 3. 珠光砂进塔对空分系统的影响

主冷氧区间的大量珠光砂聚积在换热器翅片上，造成主冷传热恶化，直接影响上、下塔工况。上塔液氧的蒸发量大为减少，上塔精馏工况被破坏，氧氮产量和纯度大幅度下降。下塔的氮气冷凝量大大下降，下塔压力高，原料空气量减少，空压机超压放空。

液空过冷器中的珠光砂堵塞液空通道，大大减少了液空和氮气的有效流通面积。

# 4. 珠光砂进塔故障的处理

上塔珠光砂采用吸尘器进行清扫处理，主冷采用高压水枪反复冲洗，直到干净为止，然后用仪表空气将残留水分吹出塔外。

液空过冷器由于严重堵塞，将其锯下，吊出塔外处理。具体方法是，在氧气通道和液空通道上分别焊上管接头，用高压水进行反冲洗，然后再用仪表空气吹除残留水分。

对塔内各容器、管道均用压缩空气进行反复吹扫，并打开有关人孔确定吹扫效果，确保将空分系统内的珠光砂全部吹除干净。

# 5. 避免珠光砂再次进塔的措施

改造加温系统管道，在空分塔保冷箱加温氮气和密封氮气总管上加装两台空气过滤器，有效的将珠光砂排除在低压氮气系统管道之外。

# 三、蓄冷器系统的检查、处理

1.  $4^{\#}$  蓄冷器中抽流量低的原因

经检查，发现因  $4^{\#}$  抽气中心筒的位置移动，使蓄冷器中的鹅卵石漏进抽气中心筒，堵塞中抽通道，致使中抽流量一直偏低。

2. 对蓄冷器的处理及改造

① 根据鹅卵石漏入抽气中心筒的情况，将中心筒通过一个不锈钢篦子与中抽管道进行有效固定，杜绝这一问题的再次发生。  
② 根据4台蓄冷器阻力大、换热效果差的情况，在每台蓄冷器的底部，各加一根

DN80的不锈钢吹除管，并加装吹除阀，然后每台蓄冷器进行冲洗，清除蓄冷器的积灰和鹅卵石粉末，改善换热效果。

# 四、检修结果

通过检修和改造，1#“3200”制氧机于1997年4月14日开车，4月17日出氧，氧产量达到  $2700\mathrm{m}^3 /\mathrm{h}$  ，纯度  $99.2\% \mathrm{O}_2$  左右，将原准备报废的设备，可以作为  $14000\mathrm{m}^3 /\mathrm{h}$  和 $6000\mathrm{m}^3 /\mathrm{h}$  制氧机的检修备机继续运行，创造了巨大的经济效益。

# 三、加热方案编制

1. 由于氩热交换器是一个六股流的换热器，要想彻底加热，必须停止所有冷气流进入氩热交换器，其中四股冷气流分别是：粗氩、混合气、精氩塔余气和氮气，两股热气流是：工艺氩和空气。  
2. 由于是氩热交换器冻堵，因此不准备加热精氩塔。为防止加热空气在精氩塔内液化，应关闭液氮进精氩塔冷凝器阀V705、V706和工艺液氩进精氩塔中部阀V707，并排放精氩塔中的液态工艺氩和精液氩，精氩塔液态工艺氩侧同时加热。  
3. 氙净化必须停车, 粗氩在进氙热交换器前通过V772放空, 放空量适当控制, 并严密注视加热期间粗氩塔的工况变化。  
4. 氩净化停车后，处理氩压机机前过滤器，更换2#干燥器温度电阻。注意2#干燥器温度更换前必须确保冷吹结束。  
5. 氩热交换器加热时，可适当打开空气进氩热交换器阀，该部分空气可不进下塔而直接放空，以加快加热速度。加热结束标志是：加热气体出氩热交换器温度为  $2^{\circ} \mathrm{C}$  以上并保持1小时。  
6. 氩干燥器加热后, 精氩塔启动前先严格进行置换, 尤其是混合气通道, 以免加热空气与含氢量约为  $50\%$  的混合气形成爆鸣气体, 在精氩塔内发生爆炸。

# 四、加热操作方案

1. 停氩净化系统，停精氩塔，出氩热交换器工艺氩温度调节TIC702置手动。关闭液氩进贮槽阀门V2200和V2201，关闭贮槽内

气体回精氩塔阀门V2226和V2227（注意防止贮罐超压），打开工艺液氩吹除阀V764，排放精氩塔底部工艺液氩，打开精液氩吹除阀V767，排放精液氩。

2. 打开粗氩放空阀V772，让粗氩直接在进氩热交换器之前放空，把粗氩流量自调FIC2201置手动，并关闭氩净化间粗氩放空阀V2102，以便能够较为准确地判断V727处粗氩的放空量，同时密切注意粗氩塔的工况变化，必要时可对粗氩放空量作适当调整。为了防止主冷液位的大幅度变化，可对膨胀量进行适当调整。  
3. 确认氩干燥器电加热器已经断电后，关闭压力氮气出氩热交换器阀V770，切断冷箱密封用氮气和氩干燥器用氮气。  
4. 为防止加热空气在精氩塔内液化，关闭液氮进精氩塔冷凝器阀V705和V706，关闭工艺液氩进精氩塔中部阀V707。为防止精氩塔出现负压把湿空气吸入精氩塔，应关闭余气放空阀V751。  
5. 打开工艺氩进氙热交换器前吹除阀V756，打开工艺液氩吹除阀V764，打开出氙热交换器工艺氩管道加熱阀V761，对精氩塔工艺液氩侧和氩热交换器工艺氩通道进行加热。  
6. 为了使得加热能够快速彻底，空气进氙热交换器阀V771也应打开，稍开空气放空阀V763，关闭空气进下塔阀V711。  
7. 清理氩压机前过滤器，更换  $2^{*}$  氩干燥器温度电阻。  
8. 加热结束后（实际加热时间共7小时），关闭工艺氩管道加熱阀V761，关闭工艺氩管道吹除阀V756，关闭工艺液氩吹除阀V764。  
9. 打开压力氮气出冷箱阀V770，打开粗氩放空阀V2202，并把流量自调投自动，关闭粗氩放空阀V772，用空气进氙热交换器阀V771，调节氙热交换器温度工况到正常。

10. 启动氩压机，加氢，待工艺氩合格且工艺氩过量氢含量为  $2\%$  左右时，送入精氩塔进行置换（因为V707泄漏），打开混合气进氩压机阀，并对混合气管道内空气进行置换，打开工艺液氩进精氩塔阀V707，打开余气放空阀V751，对精氩塔内空气进行置换。  
11. 置换完成后，V705、V706投自动，手动打开V707，精氩塔正式启动。  
12. 当精液氩液位达到  $1.3\mathrm{m}$  时，V707投自动，V708投自动，打开液氩进液氩贮槽前吹除阀V2216，对真空管道内液氩进行置换，待AE705处分析精液氩合格后，关闭液氩进液氩贮槽前吹除阀V2216，打开液氩进液氩贮槽阀V2200(或V2201)，打开气体回精氩塔阀V2226(或V2227)。

# 五、经验总结

1. 氩干燥器  $2^{\#}$  温度电阻损坏时，应立即停车进行更换，以保证氩干燥器的正常运行。由于1997年5月5日15:22工艺氩微量水满表时， $2^{\#}$  干燥器运行刚刚进行了2个小时左右（运行周期应为9小时），因此这次故障很可能是  $2^{\#}$  干燥器在上一周期加热时，加热器控制有问题。  
2. 发现微量水满表后，应先停精氩塔运行，而不应先怀疑仪表指示问题。  
3. 制氩系统产量提高后，干燥器的负荷加重，为了确保干燥器正常工作，冷冻水换热器必须确保换热正常，以减少工艺氩中的含水量。

# 一、一起少见的水冷塔引起的板翅式换热器进水事故

# 1. 事故经过

1990年10月，我公司2号  $1500\mathrm{m}^3 /\mathrm{h}$  制氧机在冷开车第二阶段，因空压机供电系统故障而停车，约2小时后再启动时发现切换板翅式换热器（以下简称“板式”）堵塞，打开四组板式热端的吹除阀  $\mathrm{PF - 157\sim PF - 160}$  ，放出大量的水，后经过多种应急处理无效，迫不得已重新停车对板式进行局部加温吹除。同时，在加温过程中发现板式  $1^{\#}$  、  $4^{\#}$  通道堵塞比  $2^{\#}$  、  $3^{\#}$  通道严重。

# 2. 分析处理

这次事故前板式工况良好，空冷塔水位指示调节也正常，而且在再开时刚启动就发现故障，氮水预冷系统尚未启动，查水泵进出口阀也关闭，空冷塔也无液位，进一步检查发现上水管网至水冷塔的供水阀TG-409未关，由此找到了板式进水的原因：水是由水冷塔通过污氮管在停车时倒入板式的，分析如下（附示意图）：

（1）事故前由于水冷塔水位自调系统不好，操作人员未用薄膜调节阀JL-402，而用手动阀TG-409从上水管网向水冷塔供水，然后再由冷水泵WP2将水送入空冷塔。  
（2）由于这套空分设备只在夏天投用水冷塔，操作人员对水冷塔认识重视不够，因此停车时未关TG-409阀，也未关闭切换系统污氮阀（从板式堵塞程度看停车时  $1^{\#}, 4^{\#}$  通道污氮阀QD-105和QD-108处于开启状

# 3. 防范措施

遇到类似情况停车时，注意关闭TG-409、JL-402及有关污氮阀门，并检查PF-

$157\sim \mathrm{PF - }160$  ，采取上述措施后至今未发生过类似事故。

# 二、水处理系统加药引起纯化器进水

我公司冷却水采自京杭大运河，水质较差，远不能满足制氧冷却用水的要求。因此我们对  $3200\mathrm{m}^3 /\mathrm{h}$  制氧机用冷却水加药净化处理，1994年9月我们向水中投加南京曙光化工一厂产WS-2杀菌灭藻剂后，冷却水产生大量泡沫，出空冷塔空气带水，不得已只好加大补充水量并排放部分循环水，以降低药的浓度，减少泡沫。大约10小时后故障现象消失。1996年6月我们又向水中尝试投加江阴市桐岐联合化工厂产LH-1(A）杀菌灭藻剂，冷却水同样产生大量泡沫，并引发带水故障。

经过摸索，我们认为有两种方法可有效地防止进水事故：其一是将原来一天中一次加入的剂量分多次投加；其二在加杀菌灭藻剂的同时向水中投加“消泡剂”，能阻止泡沫的形成。

# 三、穿流筛板空冷塔引起的进水事故

我厂  $3200\mathrm{m}^3 /\mathrm{h}$  制氧机采用穿流筛板空冷塔(其设计参数见《深冷技术》1994年第6期)，自投产以来，我们发生过十多起因该空冷塔引起的带水事故。

1. 空冷塔顶部冷冻水水分配器注水孔堵塞

1994年7月投产后，至12月出现带水故障，初期在纯化系统切换充气均压阀门打开时，发现出空冷塔的空气带水，分子筛纯化器阻力上升，待切换结束流量稳定后带水量会逐渐减少直至消失，至下一次切换时重复上述故障。

因为该制氧机使用穿流筛板空冷塔，我们判断故障原因是筛孔部分堵塞使空气和水的流通截面积变小，空气流速加快。当切换充气阀打开时，入塔空气流量增大，超过了筛板塔的设计流速而导致空气带水。

为了证实以上的猜测，我们进行了减负荷试验：在故障时，将加工空气量由17500 $\mathrm{m}^3 /\mathrm{h}$  减至  $16500\mathrm{m}^3 /\mathrm{h}$ ，结果故障现象消失。由于我公司炼钢生产不允许制氧停产，我们就进行减负荷运行，以后情况日趋严重，减负荷后又再次出现带水故障，只得再减负荷。至1995年6月份，空气量已减至  $15000\mathrm{m}^3 /\mathrm{h}$ ，再减少将产生漏液影响精馏工况，而且氧产量也将过低，于是停车检查。

打开空冷塔顶人孔后检查，出乎意料的是塔板上直径为  $15\mathrm{mm}$  的筛孔并无堵塞，再查顶部水分离层不锈钢丝网也无损。由于当时我们对冷冻水水分配器工作原理不清楚，也无水分配器图纸，未能排除故障，只好先恢复生产，并请开封空分设备厂协助解决。

在制造厂的帮助下，通过对照布水器图纸，终于找到了带水原因：冷冻水水分配器上116个直径  $6.2\mathrm{mm}$  的注水孔全部堵塞，因此导致冷冻水回水液槽中水位满溢至升气管口后，大部分从升气管内往下流，一小部分就被空气带入纯化器。由于我们在检查时小孔全部堵塞，不知道有这么多注水孔，而误认为升气管还兼作水的溢流管用，就走了弯路。问题查清后，我们将注水孔疏通并针对我公司冷却水质差的问题，把小孔直径扩大至  $7.0\mathrm{mm}$ ，并定期进行疏通清洗，解决了问题（据了解，云南磷肥工业公司  $3200\mathrm{m}^3/\mathrm{h}$  制氧机发生带水故障后，将小孔直径扩至7.5mm，同样取得了满意的效果）。

# 2. 常温冷却水水分配器注水孔堵塞

1997年3月，我厂  $3200\mathrm{m}^3 /\mathrm{h}$  制氧机又出现带水故障，这次故障又有新的特点：带水故障同样出现在纯化系统切换过程中，但不是发生在充气均压时，而是滞后十多分钟，至两个纯化器并联甚至是已切换过来一个纯化器开始泄压时才发生带水。另外，带水之前空冷塔换热效果很快恶化，出空冷塔的空气温度会在几分钟内逐步上升  $4\sim 5\%$  。当

切换结束稳定后故障同样会逐渐消失。

以上现象我们判断故障是由于空冷塔常温冷却水水分配器注水孔堵塞，使冷却水往下流动造成困难，冷却水逐渐在空冷塔上段14块塔板上积聚起来，造成筛板空冷塔液泛导致带水，液泛使空冷塔换热工况恶化使入塔空气温度上升，而冷却水在14块塔板上逐块积聚过程，恰是造成带水比以前滞后十多分钟的原因。另外，由于常温水布水器的注水孔直径为  $11\mathrm{mm}$ ，原来我们认为不易堵塞，因此一年多来在清洗冷冻水水分配器时未同时打开人孔清洗常温冷却水水分配器，很有可能已堵塞。

根据以上推测，我们停机检查，发现常

温水水分配器小孔，果然已被铁锈斑堵塞约 $50\%$  ，疏通清洗后开车，故障排除。

# 3. 存在问题与改进措施

（1）改善冷却水质，避免淤泥、水藻堵塞注水孔。自我们对  $3200\mathrm{m}^3 /\mathrm{h}$  制氧机冷却水进行净化处理后，大大减少了清洗水分配器的频率。  
（2）1997年7月初，我们将空冷塔上下水分配器材质由碳钢改为不锈钢，并适当降低了水分配器上的水位高度，在开车后我们将加工空气量增加至  $18500\mathrm{m}^3 /\mathrm{h}$  也不带水，从而使  $3200\mathrm{m}^3 /\mathrm{h}$  空分设备的氧气产量增加至 $3400\mathrm{m}^3 /\mathrm{h}$  ，效果明显。

（1997年5月初稿，7月16日收改稿）

# 快速测量气体产品中ppm级氧浓度

# 崔熙钟 张丙新

（北京瑞泰科技开发公司空分仪器配套部，北京朝阳门南大街14号3207室，100020）

根据国家气体产品质量标准的规定，出厂气体产品必须进行严格抽检，对于氮、氩、氢、氦中氧的测定，往往由于操作不当，仪器性能欠佳，操作手续繁琐，难以测出满意结果，使厂家和用户都为之颇费力。

EC90型测氧仪，可在短时间内迅速完成气体产品中ppm级氧的测量。

EC90型测氧仪是一种电化学式燃料电池测氧仪，符合国标GB6285—86（气体中微量氧的测定—电化学法）。对于ppm级氧的检测，电池使用寿命3年以上，电池在能耗尽时换上新池，即可使用。该仪器在美国和欧洲已成为“有效测量系统”中标准方法所采用的仪器。

# 一、检测原理

检测器包括阴极、电解液和一个带有限制氧具有一定渗透率的干阴极，在阴极上氧被还原成氢氧根离子，而其在金属阳极上被氧化。

在阴极上反应：

$$
\mathrm {O} _ {2} + 2 \mathrm {H} _ {2} \mathrm {O} + 4 \mathrm {e} \longrightarrow 4 \mathrm {O H} ^ {-}
$$

在阳极上反应：

$$
2 \mathrm {P b} + 4 \mathrm {O H} \longrightarrow 2 \mathrm {P b} (\mathrm {O H}) _ {2} + 4 \mathrm {c}
$$

总化学反应可表示为：

$$
2 \mathrm {P b} + \mathrm {O} _ {2} + 2 \mathrm {H} _ {2} \mathrm {O} \longrightarrow 2 \mathrm {P b} (\mathrm {O H}) _ {2}
$$

反应结果使外电路产生电流，且产生的电流和气体中氧含量成正比，这种燃料电池测氧仪，适用于检测范围  $0\sim 20\mathrm{ppm}$ $\mathbf{O}_2$  和 $0\sim 200\mathrm{ppmO_2}$  ，通过电子电路线性化可测到  $30\% \mathrm{O}_2$

仪器可间歇式使用，也可在线连续使用，

并配置  $4\sim 20\mathrm{mA}$  和  $0\sim 100\mathrm{ml}$  输出、高低报警和远传监测等，是一种较理想的气体中微量氧分析仪。

# 二、测量

# 1. 仪器启动及校正

检查仪器待用状态，气体输入输出阀关闭，放置量程  $0\sim 20$  ppm档，接通电源显示值开始下降，仪器正常时显示值为  $0.5\sim 5$  ppm，稳定几分钟，显示值若不为零，调整电子零点至到显示为0.00。检查各量程档零点显示均为0.00（电子零点，1～2月检查调整一次）。零点调整好以后，以标准气体校正，调节前面板校正旋钮，到显示数值与标气数值相同。

# 2. 样气准备及测量

连接气体减压阀和气路及压力调节器，迅速打开一关闭气瓶阀数次（2～3次）吹扫系统后，气瓶阀放置在开的位置，调整压力在  $0.04 \sim 0.10 \mathrm{MPa}$ ，小流量通气。另一种方法是采用抽空，充气置换方式排掉气路连接系统中死角处残存的氧。

测量：迅速将样气管路接入EC90测氧仪气体入口处，扭紧螺帽，先打开气体输出阀，再打开气体输入阀，调整流量  $50\sim 200~\mathrm{ml}/$  min，数字显示由高值逐渐下降，调量程档 $0\sim 20\mathrm{ppm}$  或  $0\sim 10\mathrm{ppm}$ ，待数字显示稳定，即为测量值。

# 天然气：进入兴旺发展，将成为21世纪的重要能源

据《参考消息》1997年7月1日报道，由于世人开始认识到天然气在环境和商业方面的优势，世界天然气产业正在进入一个健康发展的兴旺时期。

1996年，为满足对天然气——特别是作为联合循环燃气轮机发电厂燃料的天然气——的迅速增加的需求，全球天然气生产增长幅度超过  $35\%$ 。

基础设施一体化和新气田开发是地中海地区的主要课题。意大利国家碳化氢公司等正设法开发北非和埃及的天然气资源。在其他地区天然气业出现了投资热。在南美正在铺设天然气管道，以便把边远的气田同快速发展的电力和工业市场连结起来。

经济增长也是亚洲天然气需求不断增长的主要原因。一些新的液化天然气出口项目，瞄准印度、巴基斯坦、中国和泰国等。

1997年6月10日，世界天然气会议在丹麦首都哥本哈根开幕，来自60个国家的4500名专家探讨天然气行业的市场机会、发展趋势和技术开发等问题。

石油快用完了，未来的能源是天然气。全世界天然气消费到2030年将差不多翻一番，增至41000亿立方米。全世界的天然气总储藏量估计约404万亿立方米，有146万亿立方米天然气是肯定可以开采的。巨大的天然气储藏量，可以满足今后70年的天然气供应。以后天然气将夺走强大的竞争对手石油的顾客，到2010年，仅仅在德国就有二分之一的住宅用天然气取暖。北美和北亚仍将是最重要的天然气市场，将来这些地区的天然气消费量也将约占世界天然气消费量的  $60\%$  。

（顾福民摘编）

# 膜法富氧技术及在工业上的应用进展

沈光林

（中国科学院大连化学物理研究所，大连市110信箱膜中心，116023）

摘要 膜法富氧局部增氧的助燃技术能用于所有燃料和大部分窑炉，而且社会效益和经济效益均显著。如用于玻璃池炉，不但能显著节能和增产，而且玻璃质量提高，池炉寿命延长，烟尘排放达标等，一般  $3\sim 11$  个月就能收回全部投资。此外，膜法富氧用于医疗保健比其它氧源体积小、操作简单、安全和经济。

主题词：膜法富氧局部增氧助燃技术 窑炉 节能 环保 医疗保健

# 一、引言

膜法富氧技术，系指利用空气中各组分透过高分子膜的渗透速率不同，在压力差驱动下，将空气中的氧气富集来获得富氧空气的技术。和深冷法、PSA法相比，膜法具有设备简单、操作方便和安全、起动快、规模可小可中、不污染环境、投资少、节能效果显著、用途广等优点，近年来已引起国内外关注，是一项热门的重要科研及开发项目[1]。它在制备富氧方面的应用正在迅速增长，并正在取代其他高成本且操作不方便的分离技术[2]。工业发达国家称之为“资源的创造性技术”[3]。据文献[4]报道：1982年世界气体分离膜销售量为三百万美元，而1992年猛增到一亿零五百万美元，平均年增长率为  $42.7\%$  。中科院大连化物所自1986年起就一直从事国家“七五”和“八五”重点科技攻关项目——卷式富氧膜、组件、装置及其过程开发和应用的研究。研制成功的“LTV-PS富氧膜，  $\phi 100\times 1000\mathrm{mm}$  卷式组件及装置I型”，1988年获中国科学院科技进步一等奖；1990年“用于玻璃窑炉的高分子膜富氧装置及燃烧技术”的成果，又通过了中国科学院和北京市人民政府组织的联合鉴定[5]，还被确认为国家

“八五”新技术重点推广项目；膜法富氧助燃装置还被评为1991年度国家级新产品，并多次获国际博览会和展览会金奖。

本文主要介绍膜法富氧用于各种窑炉助燃的局部增氧助燃技术及医疗保健等方面的应用进展，着重研究了富氧助燃的节能机理及在玻璃窑炉等方面的典型应用。

# 二、膜法富氧用于助燃的进展

# 1. 膜法富氧用于助燃的节能机理[5-7]

# （1）提高火焰温度

因氮气量减少，空气量及烟气量均显著减少，故火焰温度随着燃烧空气中氧气比例的增加而显著提高，但富氧浓度不宜过高，国内外研究均表明，富氧浓度在  $28\%$  左右时为最佳，而这也正是膜法富氧的最佳浓度范围，因为氧浓度较高时，火焰温度增加较少，而制氧投资等猛增。

# （2）加快燃烧速度，促进燃烧完全

燃料在空气中和在纯氧中的燃烧速度相差甚大，如氢气在纯氧中的燃烧速度是在空气中的4.2倍，天然气则达到10.7倍左右，故用富氧助燃，不仅提高燃烧强度，加快燃烧速度，获得较好的热传导；同时，温度提高了，将有利于燃烧反应完全。

# （3）降低燃料的燃点温度

燃料的燃点温度不是常数，如CO在空气中为  $609\%$  ，在纯氧中仅  $388\%$  ，所以用富氧助燃能提高火焰强度、增加释放热量等。

# （4）减少燃烧后的排气量

用普通空气助燃，约五分之四的氮气不但不参于助燃，还要带走大量的热量。如用富氧助燃，氮气量要减少，故燃烧后的排气量亦减少，从而能提高燃烧效率等。

# （5）增加热量利用率

富氧助燃，对热量的利用率会有所提高，如用普通空气助燃，当加热温度为  $1300^{\circ} \mathrm{C}$  时，其可利用的热量为  $42\%$  ，而用  $26\%$  的富氧空气助燃时可利用热量为  $56\%$  ，增加  $33\%$  ，且氧浓度越高，加热温度越高，所增加的比例越大，因此节能效果就越好。

# （6）降低空气过剩系数

富氧助燃，可适当降低空气的过剩系数，这样燃料消耗就相应减少，从而节约能源。

2. 富氧助燃用于玻璃池炉节能[8]、[8、13]

玻璃工业是耗能大户，国内外都投入了大量的人力物力进行有关节能研究，膜法富氧技术的研究成功，给玻璃工业带来了巨大的经济效益和社会效益。

# （1）膜法富氧在国外玻璃池炉上的应用

首先是日本，由于能源紧张，先后有近20家公司推出膜法富氧装置。日本曾在气、油、煤为燃料的不同场合进行了富氧应用试验，得出如下结论：用  $23\%$  的富氧助燃，可节能  $10\% \sim 25\%$  ；用  $25\%$  的富氧助燃，可节能  $20\% \sim 40\%$  ；用  $27\%$  的富氧助燃，则节能高达  $30\% \sim 50\%$  等。联邦德国在一座马蹄型蓄热炉上用  $27\%$  的富氧试验，使熔化率增加 $56.2\%$  ，能耗则下降  $20\%$  ，而熔化温度提高了  $100^{\circ}C$  。美国一玻璃厂仅用  $23\%$  的富氧助燃，产量增加  $12.3\%$  ，节能约  $9 \%$  ，且成品率提高  $3 \% \sim 10 \%$  ，灰泡数量下降  $40 \%$  ，炉龄延长了  $5\sim 6$  个月。此外，前苏联、英国、

法国、捷克等均有富氧应用的报道。

# （2）膜法富氧用于国内玻璃池炉

① 局部增氧的助燃技术：需要说明的是，国外用的均为整体增氧，所需富氧量非常大，故投资亦很高。我们和北京玻璃集团合作，创造性地开发出了“局部增氧”的助燃技术用于玻璃池炉，所用富氧空气量仅占二次风量的  $1\% \sim 3\%$  ，而原来进风量和引风量均要降低四分之一到二分之一，综合效益十分明显。主要是使富氧通过富氧耐热喷嘴加在最关键部位，从而确保燃料在最需要的地方充分燃烧，即使喷入窑炉内的火焰达到梯度燃烧，使火焰下部由于富氧助燃作用而使温度升高。火焰横断面的温度，呈梯度分布，即火焰的下部是最高温，中部是完全燃烧层，火焰的上部温度比中部低，但却也是完全燃烧层。

局部增氧的助燃技术，主要通过富氧耐热喷嘴的设计（带扩散角的圆形耐热钢管，直径根据炉型和富氧量来定，扩散角一般为15～50度）、安装（一般位于燃料喷嘴下5～40cm处）、富氧流速的调整（一般为60～130m/s)，火焰喷射的高温点，在窑炉内的最佳位置上；还要适当逐步减小燃料量，致使火焰上部温度降低，在降低燃料流量的同时要适当减少二次风进入量；应反复调节“燃风比”（燃料量与进风量之比），这个参数直接关系到富氧燃烧节能的效益。所有参数在调试过程中找出规律之后基本不变，以确保综合效益最佳。一般3～11个月就能收回全部投资。

# 三、膜法富氧用于医疗保健[13-17]

在慢性肺部疾病的康复中，慢性呼吸道疾病致低氧血症，目前主张长期低流量氧疗（氧疗时间每日15小时，氧浓度  $22\% \sim 35\%$ ），恰好膜法富氧的浓度在此范围内，而且我们研制的膜法富氧医疗保健器经大连市第三人民医院、大连医学院附属二院、大连市体委、青海省高原医学研究所等应用表明，比其它氧源疗效还好，具有可行性且更经济、更安全、更方便。

此外，军事医学科学院等研究表明，定期吸氧不仅能清洁呼吸系统，改善内脏功能和视力，使大脑和各组织器官充分恢复疲劳，提高工作能力和效率，而且能增强肌体免疫能力、驻颜美容、预防肿瘤、延年益寿等。

# 四、结束语

膜法富氧的应用非常广泛，可用于有色金属的高温熔炼、玻璃、水泥、陶瓷、砖窑、电瓷和石灰的熔炼，各种工业锅炉、窑炉的燃烧和焚烧，石油精炼、化学过程氧化和部分氧化反应，发酵和生化反应、水产养殖以及医疗保健、氧吧、运动员增氧锻炼、粉尘/高温/有毒等场合吸氧和高原缺氧地区的供氧等。对于助燃，均可采用局部增氧的助燃技术，目前我们正向发电锅炉和冶炼炉推广，初步试验令人鼓舞，相信不久的将来膜法富氧的应用会越来越广，取得更大的社会效益和经济效益。

1997年8月21日12时50分，南宁某化工集团有限公司空分工段充氧台，发生了一起充氧台汇流排放空阀后联接管烧断和一根充氧橡胶软管爆断燃烧的事故。

# 一、事故发生的经过

据有关人员介绍，那天是两人操作充氧，一人看压力，一人巡迴检查等候关瓶阀。当一组压力达  $14\mathrm{MPa}$  、关控制阀倒向另一组（每组5瓶）时，等候关瓶阀者从里面往中间关3瓶完后等待开汇流排放空阀，看压力者出来关剩下的两个瓶阀，关到最后一瓶时，该瓶阀烫手(戴皮手套)，冒黑烟，看到汇流排放空阀后联接管起火烧断，与此同时联接冒烟瓶的充氧橡胶软管爆断燃烧，操作人员迅速离开现场。

# 二、现场调查情况

事故后进入现场看到防护墙上留下一大片黑烟迹。拆开冒烟的QF-2瓶阀检查，阀内有黑烟迹，气塞密封材料不见，阀杆密封圈熔化粘在阀杆上。放空阀后  $\phi 15\mathrm{mm}$  的铁管燃烧时阀接嘴处喷射出一小孔洞，放空操作者右手小指烧伤。充氧橡胶软管在瓶夹端爆

断燃烧，直至瓶内无氧为止。汇流排无过火现象。

# 三、事故原因分析

操作者协作不好，最后一瓶未关好而放空阀已打开，致使  $6\mathrm{m}^3$  、14MPa的高流速氧气冲击放空阀后铁管内的氧化物静电引起燃烧。与此同时，高流速的氧气与橡胶软管内层橡胶激烈摩擦引起燃烧爆炸。

# 四、整改措施

1. 事实已证明充氧橡胶软管不宜充氧，想办法购买以聚四氟乙烯材料生产的充氧软管或高压金属软管。  
2. 放空阀后的联接管改铜质并加粗。  
3. 放空阀前装一压力表，放空时采用开关开的操作方法，检验气瓶是否全部关闭。  
4. 在原有空瓶检验要求的基础上，发现空瓶存气较多者要测定其压力值，对压力较高的氧气瓶要作特殊操作。  
5. 充氧中途不得换插空瓶充氧。  
6. 以本次事故为例，对工段职工进行一次安全教育。

（1997年9月3日写稿，10月收稿）

# 制氧工问题解答（三则）

1. 问：FDN  $50 / 120 - 1$  型设备是单塔，当氧工况时，不能产氮。转工况后，又只能产氮、不能产氯。在调试中，转氨工况12小时后，各液面、温度、压力、阻力、液空纯度都达到设计要求，但氮气产量只有  $80\%$  ，且液氮、氮气纯度只有  $300 \mathrm{ppmO_2}$  （设计要求3 $\sim 5 \mathrm{ppmO_2}$ ）。请问是什么原因？怎样调节（附流程示意图）？

![](images/b5ca353110ec82aad55a805cfbc5482731bd3bd64c7196917d11e53fef1fe2d5.jpg)

答：经分析认为，液氮纯度不好，有可能是V-1阀(开氮工况时应是全关)在低温下微漏，污染液氮，液氮纯度不好又直接影响氮气纯度。我们试着把V-1阀后的V-306吹除阀用软管引到氧化锆分析仪。化验结果低于AE-3分析点的纯度（约  $400\mathrm{ppmO_2}$  ，证明V-1阀有偏漏。微开V-306阀放空，液氮同时放空，AE-3分析点的纯度马上回升。10多分钟就达到设计指标  $3\mathrm{ppmO_2}$  。关闭液氮放空阀，液氮液面上升到设计要求，纯度始终是  $3\mathrm{ppmO_2}$  ，证明判断准确。照常规氮气纯

度也应跟着提高，在AE-1分析点化验氮气纯度还是300多  $\mathsf{ppmO}_2$  。

经进一步分析工艺流程图，氮气从塔内出来分两股：一股上升到冷凝蒸发器被液空冷凝作为塔内回流液，和液氮产品进入液氮灌；另一股产品氮气经V-6阀（在V-5阀前、V-7阀后也是纯氮气)进液化器，经主热交换器常温出塔(0.69MPa)。主热交换器、液化器、液空过冷器，在总装前已经测试，漏的可能性不大。V-6阀是常开的，最后问题集中在V-5、V-7两阀上。在开氮工况时，V-5、V-7阀是关闭的，不允许有泄漏。在V-5阀后，是从上主冷回来的富氧空气通过V-3阀进入废气管路，又一股富氧空气经V-11阀进入膨胀机制冷。在V-7阀前，是经液氧过冷器、液空过冷器回来的膨胀后气体。

在氧工况时，V-6阀关闭、V-7阀打开，冷却氮气通道后放空。氮气纯度不好，有可能是V-5阀或V-7阀有微漏污染氮气，造成氮气纯度上不去。我们试着把V-5阀微开，分析仪上所显示的纯度马上就见好了，5分钟后纯度便达到  $3\mathrm{ppmO_2}$  。经稳定24小时后，设备进入正常状态。

事后再分析后认为，V-5阀前到氮气管路是股死气，阀后压力虽然低于氮气压力，但也可污染纯氮。V-5阀打开后，高压气体顶走这部分富氧空气，氮气纯度也随之达到设计要求。以上分析，供用户使用时借鉴。

# 2. 问：浮法玻璃生产线怎样送气？

答：浮法玻璃生产中需将氮氢混合气，送入锡槽。其中，氢气还原氧化锡里的锡，用氮气防止空气进入锡槽，起保护作用。倘若车间一旦停电，氮站不产氮气，而氢站还有余气，这些氢气进入锡槽，没有氮气保护，空气冲进锡槽与氢气相混合，形成爆鸣气体，在气温很高的情况下极易引起爆炸。

因此，在送气的时候，一定要先送氮气后送氢气；停气的时候，一定要先停氢气而后停氮气。

另外，储罐里的液氨不能充灌过多，一般充灌到  $70\%$  就行。

# 3. 问：AQC-160/0.03氨分解一净化制氢（氮）装置如何倒换和分析？

答：AQC-160/0.03氨分解一净化制氢（氮)装置生产出来的氢(氮)气，净化装置里面的工作筒和再生筒需要轮流倒换，才能不断产出合格的氢(氮)气。

首先将再生好的工作筒的氮气流量进、出阀门按序关掉。再将氢(氮)气去工作筒工作的出、进阀门按序打开。氢(氮)气就能进入工作筒，筒内分子筛就投入工作了。

之后，再将另一个工作简进、出阀门按序关掉，再将氮气出、进阀门按序打开。同时控制氮气流量为  $25\mathrm{m}^3 /\mathrm{h}$  。接通电加热电源并控制中部温度在  $250^{\circ}\mathrm{C}$  ，恒温  $2\sim 4$  小时，当顶部温度到  $250^{\circ}\mathrm{C}$  时，停止加热，冷

吹待用。

AQC-160/0.03型氨分解一净化制氢（氮)装置生产出来的氢(氮)气，需及时分析气体纯度，看其是否合格，如果分析合格，就可直接供给用户。

① 分解气纯度的分析

首先用蒸馏水把试管清洗干净，然后将  $25\mathrm{ml}0.01\mathrm{NH}_{2}\mathrm{SO}_{4}$  注入试管里，把分析管道用塑料管接到试管上，打开分析管道阀门，看湿式转子流量计，指针走到  $0.1\mathrm{L}$  的时候，关掉阀门，用移液管把  $1\mathrm{ml}$  纳氏试剂滴入试管中，再将  $25\mathrm{ml}0.01\mathrm{NH}_{2}\mathrm{SO}_{4}$  注入试管，然后摇匀，看其颜色。对比标准比色阶来判断是否合格。

② 净化气纯度的分析

分解气合格以后，才能送去净化。净化气纯度的分析跟分解气纯度的分析一样，只是净化气需要  $10\mathrm{L}$  流量，分解气需要0.1L流量。试管里的颜色，应跟水一样透明，否则净化气纯度不合格。

要想知道气体纯度的具体数字，就得将试管里的颜色跟比色阶的颜色相比较，跟比色阶颜色一样或着差不多，就将比色阶的颜色数字记下，除以流量，然后乘以1000，就得出具体纯度的数字。
