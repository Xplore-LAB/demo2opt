通过吸附作用清

在实际生产中,
分子筛纯化系统的故障率较

212 出口空气中二氧化碳含量超标
出口空气中二氧化碳含量超标的最根本原因
是: 吸附剂效能限制( 吸附超时或效能降低) 。只
要能造成吸附剂效能下降甚至失效的因素都可以视
为二氧化碳含量超标的原因。通常造成吸附剂效能
降低的因素有:
( 1) 再生不彻底:
例如再生气流量或温度不
足, 加热时间不够, 再生气水含量超标。
( 2) 吸附剂数量不足:
例如吸附时间超时、空
气流量偏大等。另外,
床层不平整或局部穿透, 可
以理解为局部吸附搅数量不够。
( 3) 共吸附的影响: 例如空气水含量增大、空
气中夹带液态水。
( 4) 吸附阶段操作温度过高: 例如进口空气温
度过高,
空气水含量过多导致产生更多的吸附热,
使用前冷吹不彻底而使得床层有余热。
( 5) 吸附剂活化程度不够:
新的吸附剂在投入
使用前要进行完全活化。
( 6) 吸附剂自身老化:
吸附剂经过多次再生循
环后吸附效能会逐渐下降。

213 分子筛吸附器进水
在生产中分子筛吸附器进水的主要原因大致有
以下几种:
( 1) 空分设备停车状态下, 空冷塔有压回水阀
系统关闭不严或循环水系统阀门泄漏,
造成循环水
直接进入空冷塔,
而空冷塔又没有及时打开排水
阀;
或空分设备短期停车时没有计划排水,
造成循
环水自回水管倒流,
进入分子筛吸附器。
( 2) 空冷塔液面计指示失灵。可能原因: 一是
仪表显示误差大; 二是取样管被杂质或局部结垢堵
塞;
三是冬季气温低, 取样管内结冰冻死。以上3
个原因都会造成假信号使空冷塔出水阀关闭, 实际
液位过高而引起分子筛吸附器进水。
( 3) 空冷塔设计不合理, 气液分离空间过小。
空分设备运行时当空气流量偏大或瞬间压力突变,
如冷箱导气打开空气进冷箱阀或开车时用空气作再
生气而打开再生旁通阀过快, 分子筛吸附器升压不
彻底就切换运行,
都会造成空冷塔出口空气压力突
降, 超过允许值。
( 4) 循环水质差。菌类、藻类等悬浮杂质堵塞
空冷塔进水分配器, 造成分配器水位过高, 上升空
气带水冲过丝网除沫器。
( 5) 杀菌除藻的水质处理药剂加得太多或药剂
加错, 造成空冷塔填料层及分配器上积聚大量泡
沫, 使空气出空冷塔时夹带液体。
( 6) 水冷塔液位过高造成循环水进入污氮或氮
气管线, 而后直接进入分子筛吸附器。
( 7) 蒸汽加热器泄漏造成再生气带水。

压缩机振动值过高通常从以下几方面进行检
查、分析和处理:
( 1) 受迫振动:
可通过检查转子动平衡、电机

( 2) 共振:
可通过检查临界转速、润滑油供油

( 3) 运转: 可通过检查轴承间隙及装配情况来

( 4) 脉动: 可通过监听气流脉动、涡流及喘振

这次空压机振动值过高时,
做了以下检查与分
析:
( 1) 检查润滑油的品质、各供油点的流量、油

( 2) 全面检查设备基础、管道的紧固情况, 检
查齿轮箱,
均正常。
( 3) 压缩机体气流和转子动平衡状况。
( 4) 其他可能引起振动值过高的原因。

4 总结
( 1) 空压机振动值高的原因是多方面的, 往往
是几方面的原因共同作用引起的, 甚至还会出现这
样一个现象: 检修前运行正常,
检修过后振动值反
而升高。因此故障压缩机在检修( 包括正常压缩机
的检修) 时应从多方面仔细考虑, 尽可能地在检修
中将各类已发生的故障和潜在的故障隐患消除掉。
( 2) 空分设备运行环境空气质量普遍较差,
RIK 系列空压机使用几年后中间冷却器肯定会发
生阻力增大、翅片堵塞的问题。而多次清洗会使翅
片损坏。因此, 只要能满足正常生产需求, 应尽可
能减少空压机叶轮的清洗次数以延长其使用寿命
( 当然若找到一种对翅片无损伤或轻微损伤的清洗
方法则更好) , 此类问题有待摸索、解决。
国外配套的中间冷却器虽然换热效果好,
但价
格高、交货期长,
而国内已有厂家能生产出可满足

器翅片节距不太一致,
翅片上也没有冲压孔,
但换
热效果基本一样,
而且有价格优势,
交货期也在1
个月内。
( 3) 生产厂家说明书上要求对叶轮的清洗频率
为每周2 次,
每次总喷水量2650L。广西柳州钢铁
( 集团) 公司气体公司根据实际情况改为5 天清洗
1 次。近4 年的运行操作数据表明: 清洗后空压机
振动值的改善微乎其微, 说明短周期内清洗叶轮的
效果不大。因此,
适当地延长清洗周期可减少叶轮
及中间冷却器翅片附着物,
延长空压机运行周期。
建议将叶轮清洗周期延长至7~ 10 天( 若空气环境
质量好, 还可再延长) 。
