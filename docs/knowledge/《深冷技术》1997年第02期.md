# 湘钢再次与杭氧签订“15000”空分设备合同

杭氧集团有限公司于1990年9月与湘潭钢铁公司签订了我国首台采用常温分子筛净化一增压透平膨胀机工艺流程的  $14000\mathrm{m}^3 /\mathrm{h}$  带氩空分设备的订货合同。经双方两年多共同努力，装置调试出氧一次成功，并于1993年4月30日并网送氧，同年8月出氩，全部技术指标达到设计值。装置连续运转两年后，于1995年初在湘潭进行了产品鉴定。鉴定认为，其技术水平达到国际80年代后期水平，填补国内空白，为国家节约大量外汇，并为我国自行成套设计 $15000\mathrm{m}^3 /\mathrm{h}$  以上等级大型空分设备奠定了基础。

为了生产发展的需要，湘潭钢铁公司与杭氧集团有限公司于1997年1月25日在湘潭再次签订了采用第六代空分全新技术的  $15000\mathrm{m}^3 /\mathrm{h}$  带氩空分设备订货合同。这一合同的签订表明，杭氧  $15000\mathrm{m}^3 /\mathrm{h}$  等级空分设备的质量已取得用户的信赖。

# 世界LNG的贸易以及各国LNG的发展动向

石玉美* 顾安忠**

（上海交通大学动力机械工程系，上海市华山路1954号，200030）

摘要 液化天然气(LNG)是当今世界能源消耗中的组成部分之一。本文根据许多详实的数据资料，介绍了世界LNG贸易的现状及其LNG的进出口的情况。表3。

# 主题词：液化天然气(LNG) 贸易 发展动向

天然气是当今世界能源消耗中的重要组成部分。由于它有丰富的储量，较高的热值，洁净的燃烧，它的需求量正在持续上升。过去20年里气体供应量以平均  $5\%$  /年的速度递增[1]，这是因为：煤污染严重，公众对核能有不安全感，所以煤和核能用量将受到一定的制约，同时石油将越来越多地用作汽油燃料、柴油机燃料和化工原料，则由此带来的能源供应量短缺和能量需求量增长，就主要需要由天然气来补充[2]。

从气田开采出来的气运输给最终用户有两种方式：①用管道输送天然气；②以LNG方式输运天然气，这一输运方式在气源与用户之间增加了许多耗资巨大的设备，使其成本高于管输气。1994年世界LNG贸易量，仅为世界天然气市场的  $4\%$  [1]。但是，在某些方面，LNG有管输气无法比拟的优点：（1）LNG具有独特的运输方式，对于陆上远距离运输，用LNG运输可减少大量的风险性管道建设及站场的投资；对于远洋天然气贸易，LNG几乎是唯一的运输方式。(2)利用LNG方式是解决远海、荒漠地区气田开发以回收边远天然气的有效方法。（3)LNG的储存成本仅为气态天然气储存成本的1/6至1/7。（4）天然气在液化前要进行净化处理，清除掉酸

性气体、重烃、水、汞等不利天然气液化过程顺利进行的成分，所以它是比管输气更为洁净的燃料[3]。

正是具备以上这些优点，LNG贸易自1965年开始以来，长盛不衰。自1980年以来，世界LNG贸易量以年均  $8\%$  的速度增长[1]。

# （一）世界LNG贸易

到1995年，全球LNG贸易走过了30年的历程。世界上第一宗LNG贸易是在阿尔及利亚的Arzew LNG生产基地与英国的Canvey岛LNG接收终端之间进行的。进口LNG虽起源于英国，但它却在日本、韩国、台湾省、欧洲的中南部和美国得到了进一步的发展。

1994年，全球LNG生产基地的名义生产容量为73.5百万吨/年，它们出产于以下世界各地的LNG生产基地：（1)印度尼西亚Arun基地和Bontang基地；(2)马来西亚Bintulu基地；(3)澳大利亚西北Shelf基地；(4)文莱Lumut基地；(5)阿布扎比Das岛基地；(6)利比亚Marsa EI Brega基地；(7)阿尔及利亚Skikda基地和Arzew基地；(8)阿拉斯加Kenai基地[1]。

1994年，世界LNG的贸易量达到了65百

万吨/年，其中至少3/4的LNG出口到远东。预期到2000年，LNG贸易量可达到  $90\sim 95$  百万吨，到2010年将达到  $13\sim 16$  千万吨。导致LNG贸易持续增长的原因，是LNG的生产国和进口国都有所增加。泰国、土耳其、以色列、中国、克罗地亚、德国、印度、意大利、葡萄牙、波多黎哥、荷兰和英国有可能加入到LNG进口国行列。卡塔尔、委内瑞拉、尼日利亚、也门、阿曼、俄罗斯的萨哈林岛、挪威、新巴布亚和新几内亚有望加入到LNG出口国行列[1][4]。

亚太地区是LNG供应和需求增长最快的地区，目前在亚太地区，主要的LNG出口国是印度尼西亚、马来西亚、澳大利亚和文莱。主要进口国是日本、韩国和台湾省。亚太地区LNG贸易持续增长的原因有：(1)由亚太地区的地理条件决定的，铺设越洋管道进行气态天然气贸易在技术上是可行的，但将受到资金和政治因素的制约，所以目前用LNG方式进行天然气贸易是最合适的方法。(2)该地区天然气需求量较大的国家和地区，自身的天然气贮量很少。(3)有许多LNG供应源可保证亚太地区LNG增长的需要。该地区除了原有的供应源外，卡塔尔的北部大气田、印度尼西亚的Natuna气田、阿曼中部气田、澳大利亚的Carnavon气田、俄罗斯的萨哈林岛（气藏估计为3.5万亿立方英尺）、阿拉斯加的布里斯托尔湾、也门的Maareb和Jawf气田，这些国家和地区发现的大量天然气，可为亚太地区提供LNG。估计到2010年，亚太地区的LNG需求量将达到120百万吨/年，估计其增长部分的至少 $50\%$  将用作发电厂的燃料。这主要由以下几方面的原因：①亚太地区今后几年经济增长对电的需求量增加；②LNG是优质燃料；③新型高效燃气联合循环发电技术的成熟[6]。

欧洲西北部和美国，由于用管道输送天然气比较方便，所以LNG贸易量不会有很

大的增幅。

# （二）各国LNG的发展动向——出口国

# 一、印度尼西亚

印度尼西亚是全球船运LNG量最多的国家，从  $1959\sim 1994$  年，印度尼西亚LNG生产终端已船运出562百万立方米的LNG，其中Arun生产基地290.3百万立方米、Bontang生产基地271.8百万立方米[6]。Arun生产基地位于苏门答腊岛的北部，有6套生产机组，每套生产容量为2百万吨/年；Arun生产的LNG一般是以签订长期合同的方式销往日本、韩国和国内一些用户。Bontang生产基地位于加里曼丹岛的东部，也有6套生产机组，5套机组的设计容量均为2.3百万吨/年，第六套为2.1百万吨/年。第六套机组是1991年得到贷款开始兴建的，主要为日本的三家公司（Osaka气体公司、Tokyo气体公司、Toho气体公司）提供LNG，合同期为20年，从1994年开始输出；Bontang生产基地的LNG大多数是通过签订长期合同销售给日本、韩国、台湾省和一些国内用户。1994年初，这两个生产基地的LNG生产量为26百万吨/年[7]。

印度尼西亚的这两个LNG生产基地，是由印尼Pertamina负责和管理的[8]。

印度尼西亚也是全球最大的LNG出口国，1991年出口LNG22.8百万吨，其中18.5百万吨出口到日本、2.8百万吨出口到韩国、1.5百万吨出口到台湾省[12]。1993年，年产量为24百万吨，出口至日本、韩国和台湾省的LNG量为20.63百万吨[8]。1994年，出口LNG达26.47百万吨，占全球LNG贸易量的  $40\%$  左右，出口的LNG中有  $71\%$  出口至日本，  $20\%$  出口至韩国，  $9\%$  出口至台湾省。日本从印度尼西亚进口的LNG能满足其LNG需求量的  $45\%$  左右[9]。

1995年8月，印尼与韩国签订了一项LNG卖买合同，合同期为20年，出口量为1百万吨/年。从1999开始，印尼向韩国的LNG出口将达6.3百万吨/年。这一LNG量将从Bontang生产基地第H套生产机组输出。建设这一生产机组将耗资9亿多美元[10]。

印度尼西亚能成为头号LNG出口大国，是因为它有极其丰富的天然气资源，1991年，在加里曼丹岛东海岸Mahakam地区发现了两大气田：Tambora气田和Tunu气田。其储量超过5万亿立方英尺，可为位于加里曼丹岛东部的Bontang液化厂提供液化气源。这两大气源和在附近SiSi和N．W.Peciko发现的天然气，可使Bontang液化厂的气源供应持续到下个世纪[1]。印度尼西亚的天然气储量估计一共有100万亿立方英尺，加里曼丹岛东部约占  $25\%$  ，苏门答腊北部约占  $20\%$  ，Natuna地区约占  $50\%$  ，其余的  $5\%$  多数在Java地区[12]。

印度尼西亚的Natuna气田于1973年发现，位于Natuna岛东北偏东方向约140海里处，是个大气田，可开采的烃估计在45万亿立方英尺左右。Natuna气源组分平均  $71\%$  为二氧化碳、  $28\%$  为烃类、  $0.5\%$  为硫化氢、 $0.5\%$  为氮气。Natuna的开发需要以下三方面的工作：（1)海上气体的开采与处理；(2)陆上气体处理和液化；(3)海上废气处理。一共需要6个海上处理平台、6个钻井平台、4个喷射平台、2个居住用平台；至少565英里的管道；还要有天然气净化和淡化厂。建这一整套系统将耗资400亿美元。若能建成，Natuna将成为世界上最大的海上气体开发项目，其主要合作伙伴，是印尼的Pertamina和美国的Exxon公司[13]。该项目若能在2010年建成投产，其产量可达到15百万吨/年，最终产量可达到38百万吨/年[9]。

# 二、马来西亚

马来西亚也是亚洲一重要的LNG出口

国，从1983年起开始出口LNG至日本，合同期为20年，供应量为  $6\sim 7.5$  百万吨/年[12]。马来西亚的LNG出口基地位于沙捞越的Bintulu，有三条生产线，马来西亚Petronas对此有主控权。1992年，几家公司联合耗资16亿美元，对Bintulu生产基地进行扩建，准备增加3条生产线，使其生产总量达到15.8百万吨/年。1993年动工，计划于1996年建成，增加的产量销往日本、韩国和台湾省。

1995年底，马来西亚的Occidental LNG 有限公司、Pctonas公司、Nippon油公司、壳牌气体 BV 公司签订了一项建造一生产 LNG 工厂的协议。该厂是位于沙捞越 Bintulu 的第三座 LNG 出口工厂，是马来西亚第三次建造 LNG 的生产工厂，生产容量为 6.8 百万吨/年，预计到 2000 年投入运行。于 1992 年由 Oxy 公司发现的储量为 2.9 万亿立方英尺的 Jintan 气田，将作为其最先使用的气源，该气田位于 Luconia 盆地中央。该厂先建两条 LNG 生产线，每条产量为 3.4 百万吨/年。当这第三座工厂建成后，马来西亚的 LNG 出口量可达到 23 百万吨/年[16]。

# 三、文莱

文莱冷气公司是文莱生产LNG的合资企业，由文莱政府、RoyalDutch/Shell集团、日本Mitsubishi公司合资。该公司于1973年开始向日本三家公司(Tokyo发电厂Tokyo气体公司、Osaka气体公司)供气，年供应量为5.14百万吨，为期20年；1993年到期后，经双方同意，又签订20年的LNG卖买合同，年供应量增加  $10\%$  [12]。

# 四、澳大利亚

澳大利亚最大的LNG项目是西北Shelf LNG出口基地。该基地运营的是澳大利亚Woodside石油有限公司。此基地是在1989年8月开始向日本八家公司出口LNG的，合同期为20年，供应量为  $3.86\sim 4.86$  百万吨/年[12]。

1993年初，Woodside集团向西班牙的Enagas公司出口了少量LNG，1993年底又向韩国出口了56000吨LNG，这是韩国第一次从澳大利亚进口  $\mathrm{LNG}^{[10]}$ 。

1993年初Woodside石油有限公司又同日本用户签订了1百万吨/年的LNG卖买合同，于1995年开始实施，这将使澳大利亚LNG出口量达到7百万吨/年。

1995年，澳大利亚西北部的Carnavon盆地又发现了两大气田：(1)Ampolex发现的气田，位于Harriet油田以西20公里，在2330.33米深度测得气流量为1.1百万立方英尺/天；(2)在Undan发现一估计储量为1.5～4万亿立方英尺的大气田。这两大气田将为LNG提供丰富的气源[17]。

# 五、卡塔尔

卡塔尔在其北部波斯湾发现了一储量超过225万亿立方英尺的超大气田。对这一超大气田，卡塔尔一方面准备生产LNG，另一方面铺设管道外输。巴基斯坦和卡塔尔官员讨论，准备铺设天然气输送管道，这一管道将连接卡塔尔、阿拉伯联合酋长国、伊朗和巴基斯坦。这一管道全长1600公里，将耗资30亿美元，可使卡塔尔从北部气田出口38亿立方英尺/天的天然气，管道将由Crescent石油公司来铺设[18]。

卡塔尔的液化天然气项目有两项，一项是Ras Laffan液化天然气有限公司（简称Rasgas）；另一项是卡塔尔液化气公司（简称Qatargas)。卡塔尔通用石油公司(QGPC)对这两个项目有主控权[9]。

Qatargas建在卡塔尔东北海岸的Ras Laffan工业区，设计产量为1千万吨/年，头两条LNG生产链将于1996年晚些时候竣工，第三条生产链将于1999年完工。它的LNG买主为：(1)日本的Chubu电厂，它是Qatargas的主顾，年进口量为4百万吨，合同期为25年，从1997年开始执行；(2)日本的

另外七家公司，它们是Tohoku电厂、Tokyo电厂、Kansai电厂、Chugoku电厂、Tokyo气体公司、Osaka气体公司、Toho气体公司，它们的年进口量一共为2百万吨，合同期为24年，从1998年开始实施[9][19]。

Rasgas 也位于东北海岸处的 Ras Laf. fan 工业区，1996 年 3 月 Rasgas 宣布已经与日本的 JGC 公司、M. W. Kellogg 公司签订了一项价值 10 亿美元的 LNG 工厂的工程、采购和建造合同。Rasgas 海上处理平台的搭建和海底管道铺设工作，已于 1996 年 4 月启动。Rasgas 的液化容量为 5 百万吨/年，计划于 1999 年建成。Rasgas 生产的第一批 LNG 将运往韩国的韩国气体公司。Rasgas 的潜在用户有台湾省、日本、泰国、中国、土耳其和印度。掌握 Qatargas 和 Rasgas 两大 LNG 项目多数股权的 QGPC，计划能向台湾石油公司出口 2.1 百万吨/年 LNG，向中国出口 2.5 百万吨/年 LNG，向印度出口 2.5 百万吨/年 LNG，向土耳其出口 2 百万吨/年 LNG[20][21]。

# 六、也门

也门在Maareb和Jawf发现了两大气田，估计储量为15.75万亿立方英尺。也门准备耗资50亿美元开发利用该天然气资源，其中30亿美元用于：（1)铺设两条气体输送管道，一条用于国内用气，另一条把气体输往LNG生产基地；(2)建造位于Ras Omran的液化天然气工厂和出口终端，产量为5百万吨/年。其余20亿美元用于购制LNG的运输设备。在天然气开发项目的建设中，也门选择法国作为它的主要合作伙伴[9][22]。

# 七、阿曼

1991年，在阿曼中部发现了大量深层气，储量约在24万亿立方英尺左右，这有望使阿曼成为一LNG出口国，同时也可出口气态天然气[8]。由于经济原因，阿曼政府不能同时建造LNG项目和铺设远程管道出口天然

气，准备首先发展LNG项目。阿曼政府准备耗资60亿美元，建造一产量为6百万吨/年的LNG出口基地，建于阿曼首都马斯喀特东南200公里处的Shalilah。建造LNG工厂及相关设备的合同有望1996年达成，2000年左右出口  $\mathrm{LNG}^{[9][23]}$  。

阿曼LNG出口终端的用户将是欧洲、日本、中国和韩国[9]。1996年3月，阿曼和韩国气体公司就4百万吨/年的LNG贸易协议双方签署了备忘录[24]。

# 八、阿布扎比

阿布扎比从1977年开始向日本出口LNG，供应量为2.06百万吨/年[12]。1995年5月，阿布扎比气体液化公司(Adgas)与西班牙Enagas签订了28.5万吨的LNG买卖合同，期限为1995年6月至1996年3月，

这表明Adgas的LNG进入了欧洲市场。阿布扎比还希望与法国、比利时进行LNG贸易。这些LNG来自位于Das岛的LNG生产基地的第三条生产线，这条生产线耗资1亿美元，于1994年5月建成。该生产基地的LNG生产量为5百万吨/年[26]。

日本 Tokyo 电厂是阿布扎比的主要客户，1993年从阿布扎比进口2.7百万吨、1994年进口3.7百万吨的  $\mathrm{LNG}^{[0]}$ 。

# 十、尼日利亚

尼日利亚从1985年开始准备建造LNG生产项目。M. W. Kellogg公司和法国的Technip公司已于1990年设计完此LNG工程项目，此项工程将耗资40亿美元，产量为5.6百万吨/年。本来计划于1997年投产运行，但由于以下一些原因致使工程项目拖延：（1)资金短缺；(2)公司董事变化；(3)技术方面的问题；(4)LNG用户未得到确切的认证。对此，尼日利亚政府通过降低其在LNG工程项目中所占的股权份额来吸引外企的更深层次的参与。

目前，该项位于Bonny岛的LNG工程项目和股权分配是：尼日利亚国家石油公司 $49\%$  ，RoyalDutch/Shell集团  $24 \%$  ，法国Elfl  $5 \%$  ，意大利Agip  $10 \%$  ，国际金融公司  $2 \%$  。该项工程具体包括：(1)一条长200公里、每天输送938百万立方英尺天然气的管道；(2)建于Finima的气体液化厂；(3)LNG装载码头；（4）两个LNG贮槽，每个容量84000立方米；(4)七艘LNG运输船。建造该项目的合同最终于1995年12月签订，建造期为4年。Bonny生产基地的LNG买主是意大利、西班牙、法国和美国[7][19][31][32][33]。

# 十一、委内瑞拉

在  $1978\sim 1982$  年间，委内端拉的Pdvsa公司在委内瑞拉的Paria半岛北边的加勒比海里发现了4个气田：即RioCaribe气田、Mejillones气田、Patao气田和Dragon气田。其中RioCaribe气田的气为富气，可用作制冷剂；其余3个气田的气甲烷平均含量达  $97\%$  [34]

1993年，建造CristobalColonLNG出口项目得到委内瑞拉国会同意。该项目的产量为6百万吨/年，将耗资56亿美元，其资金的70%来自于国际金融机构，另外的30%由4个合作伙伴据股权份额按比例提供。气源需求量平均为960百万立方英尺/年。通过

两条长分别为45公里、32公里的海底管道把气源运到陆上。液化厂建于靠近Mapire海的Paria半岛的南海岸上。该LNG工程的用户主要在美国东北部和欧洲[36][30]。

# \*设计制造\*

# 水环式压缩机的设计研究

沈玉祥\*

（杭州制氧机集团有限公司活塞式压缩机厂，杭州市东新路90号，310004）

摘要 介绍用于粗氩输送的水环式压缩机的设计方法、参数的选取范围、吸气口与排气口的形状及尺寸的确定，并以设计实例说明。图2表1参4。

# 主题词：水环式压缩机 设计 参数 确定

# 一、引言

水环式压缩机是压缩气体介质的一种动力机械，具有结构简单、使用维护方便和压缩介质无外泄等优点，因此通常被用于化工、炼油及造纸工业，用来抽送有毒、有腐蚀性和有爆炸性的气体。在空分设备中，主要用于粗氩的输送。

目前，国内水环式压缩机的设计方法，还不够完善。杭氧早期的水环式压缩机，也是通过对德国西门子产品进行测绘而生产的。所以要真正设计水环式压缩机，我们碰到的最大困难是缺少系统的专门计算工具书。我们所能找到的资料，都是其他书籍中的一些片段，且只介绍一部分内容。

在设计过程中，我们曾根据不同的资料分别进行计算，结果发现计算的结果出入很大，各种资料互不一致。例如：一级叶轮在气缸中的偏心距，按资料[1]计算为  $21\mathrm{mm}$  而按资料[2]计算为  $33.17\mathrm{mm}$ ，而这个尺寸对水环式压缩机是至关重要的。由于不一致，在设计中我们只好根据各种资料都计算一遍，将计算所得的结果和测绘制造的样机实际尺寸进行对照比较，看哪些公式计算的结果比较吻合，或要作一些什么样的修正，即根据样机的实际参数对设计公式进行验证。

通过以上方法，我们设计了水环式压缩

机，并通过出厂试车验证，各种性能参数都满足设计要求。本文通过对水环式压缩机的设计实践，将设计方法作一概括性论述。

# 二、设计参数

设计水环式压缩机，须给出必要的原始数据，常给出的有：  $Q$  一气体流量；  $P_{1}$  一吸气压力；  $P_{2}$  一排气压力。

# 三、主要尺寸参数的确定

1. 转速  $n$  及叶轮宽度  $b$  、叶轮外缘半径  $r_2$  值的确定

图1是水环式压缩机平面图。

图中，  $r_1$  一叶轮轮毂半径；  $r_2$  一叶轮外缘半径；s一叶片厚度；  $\beta$  一叶片出口安装角； $a$  一叶片在工作介质中的最小淹没深度；  $R$  一壳体内表面半径；  $\delta_{x}$  一吸气口始端角；  $\varphi_{\pi}$  一吸气口末端角；  $\varphi_y$  一排气口始端角；  $\delta_y$  一排气口末端角；  $e$  一叶轮与壳体的偏心距；  $f$  一叶轮外圆与壳体间的最小间隙。

对于给定的进排气压力及工作介质，叶轮有一最小圆周速度。对最小圆周速度，各种资料都比较统一，其公式为：

$$
r _ {2} w = \frac {1}{\lambda} \sqrt {g \frac {3 P _ {2} - 2 P _ {1}}{\gamma}}
$$

式中  $\lambda$  ——叶片修正系数，取  $\lambda = 0.9 \sim 1.1$

$\gamma$  ——工作介质重度

$\pmb{w}$  ——角速度，  $\pmb{w} = 2\pi \pmb{n}$

![](images/98c7147e53ab0865a364424b720a44655c45ecde1749706db435d729ec998aaf.jpg)  
图1 水环式压缩机平面图

要满足给定的气量  $Q$  ，须要一合理的叶轮宽度  $\pmb{b}$  来保证。对于数值  $\pmb{b}$  的求取，各种资料给出的公式不一致，我们认为资料[3]采用的是实用的。其公式如下：

$$
r _ {2} b = \frac {2 Q}{\eta_ {v} \mu [ (1 - a ] ^ {2} - v ^ {2} ] r _ {2} w}
$$

式中  $\pmb{b}$  ——叶轮宽度

$\eta_{v}$  ——容积效率，通常  $\eta_{v} = 0.6 \sim 0.8$

$\mu$  ——叶片排挤系数，对于铸造叶片，可取  $\mu = 0.68 \sim 0.85$

$\alpha$  ——相对堰深，  $\alpha = \alpha / r_2$  ，通常可取  $\alpha = 0.02 \sim 0.03$

$\pmb{\nu}$  ——轮毂比，  $\pmb {\nu} = \pmb {r}_1 / \pmb{r}_2$

给定某一转速后，根据以上两式已求得的  $r_2w$  和  $br_2$  可以分别求出  $r_2$  和  $b$  ，  $b / r_2$  的比值也确定，但比值  $b / r_2$  是有范围的。  $b / r_2$  比值偏大，则叶轮进气来不及，将导致容积效率的降低。因为在直径不变的情况下，随着叶轮宽度的增加，排气量可增大，但增大的幅度并非与叶轮宽度  $\mathbf{b}$  成正比，原因是进排气孔内部都是有阻力的。

资料[1]推荐  $b / r_2 \leqslant 1.5$ ，并说明适用于双向进气，当单向进气时， $b / r_2 \leqslant 1.5 / 2$ ；资料[3]规定  $b / r_2 = 0.5 \sim 1.0$ ，未说明是单向进气还是双向进气；资料[4]建议对于双向进气  $b / r_2 = 2.5 \sim 2.8$ 。据我们对现有机器的统计，对于双向进气， $b / r_2$  比值取在  $0.8 \sim 1.2$

范围内，能保证充分吸气。

选择合适的转速  $n$ ，使比值  $b / r_2$  在允许的范围内。

# 2. 叶片设计

根据叶轮半径  $r_2$ ，选取叶片数，一般叶片数  $Z = 12 \sim 24$ ，叶片出口安装角为  $\beta = 45^{\circ} \sim 60^{\circ}$ 。

# 3. 计算偏心距  $c$  和壳体半径  $R$

偏心距的计算按不同的资料差距较大，按资料[1]公式计算显得较小，而按资料[2]则太大，资料[3]和样机比较吻合。其式为：

$$
e = \left[ (1 - \alpha) ^ {2} - v ^ {2} \right] \frac {\mu r _ {2}}{4 \lambda \zeta}
$$

通常，  $e / r_2 = 1 / 8\sim 1 / 7$

式中，  $\zeta = b / B$  ，  $B$  为壳体宽度。

壳体半径由下式确定

$$
R = \tau_ {2} + e + f
$$

式中  $f$  为径向间隙，通常取  $f = 3\sim 10\mathrm{mm}$ 。

4. 吸气口和排气口的设计

吸气口和排气口的合理设置，对水环式压缩机的成功与否是至关重要。吸排气口如何确定，许多资料都没有提及。吸排气口由始端角、末端角、内侧边框和外侧边框组成。

![](images/43c8ab18783e4929051fb01e2b5f4947484edc70b773d2042d75ca8e4528164d.jpg)  
图2 液环形状及吸排气口

吸气口的位置应保证水环式压缩机最大限度地吸入气体。为此，就应该使得叶片间的小空间容积在增大的过程中始终与吸气口相通，而当小空间的容积一旦开始变小时，就立刻与吸气口隔开。

吸气口末端角度在满足上述要求时为：

$$
1 8 0 ^ {\circ} - \frac {3 6 0 ^ {\circ}}{z} <   \varphi_ {s} <   1 8 0 ^ {\circ} - \frac {1 8 0 ^ {\circ}}{z}
$$

吸气口的始端角一般取  $\delta_{\mathrm{s}} = 15^{\circ}\sim 20^{\circ}$

吸气口的内侧边框可以用半径比  $\pmb{\tau}_{1}$  略小的同心圆弧确定，如图2中虚线所示。而吸气口的外侧边框的确定较为复杂，以下公式，经复验，比较符合样机的实际形状。

$$
\left\{ \begin{array}{l} r _ {\varphi} = r _ {2} \sqrt [ 4 ]{\frac {2 \lambda \zeta (R - \rho)}{\mu r _ {2}} + y ^ {2}} \\ \rho = r _ {2} \sqrt [ 4 ]{1 + \left(\frac {e}{r _ {2}}\right) ^ {2} + 2 \frac {e}{r _ {2}} \cos \varphi} \end{array} \right.
$$

这样，对应于某一角度  $\varphi$  ，即可求得一个  $r_{\varphi}$  值，从而吸气口的形状就确定了。

与吸气口相比，排气口形状的确定要复杂得多。排气口的形状对水环式压缩机的性能有很大的影响，排气口始端角过大，将导致过压缩现象，增加功耗；而太小，又会使气量下降。只有在等于或接近临界角时，水环式压缩机的效率才最高。

排气口始端角  $\varphi_{y}$  由下式表达：

$$
\varphi_ {y} = \varphi_ {c r} + (1 8 0 ^ {\circ} / z)
$$

其中  $\varphi_{cr}$  为临界角， $\varphi_{cr}$  可由下列方程组求出：

$$
\left\{ \begin{array}{l} \varepsilon = \frac {r _ {1} ^ {2} w ^ {2}}{2 g H _ {1}} \\ A = \frac {3}{2} \sqrt {\frac {3 e}{(\varepsilon A ^ {2} + 1) ^ {3}}} \\ A = \frac {2 (R - \rho) \zeta}{\mu [ (1 - \alpha) ^ {2} - v ^ {2} ] r _ {2}} \\ \rho = r _ {2} \sqrt {1 + \left(\frac {e}{r _ {2}}\right) ^ {2} + 2 \frac {e}{r _ {2}} \cos \varphi} \end{array} \right.
$$

其中  $H_{1} = P_{1} / \gamma$

经和现有机器比较发现，用上述方程组求得的临界角  $\varphi_{cr}$  太大，比实际的要大  $20^{\circ} \sim 30^{\circ}$  。因此，在设计中，我们按上述公式求得  $\varphi_{cr}$  后再减小  $20^{\circ} \sim 30^{\circ}$  来确定始端角。后经试车测试，机器的压缩比和气量都在设计范围内，说明由此确定的始端角是合理的。

排气口末端角可取  $\delta_{\nu}\approx 30^{\circ}$

排气口的内侧边框的确定同吸气口，而

外侧边框由以下方程组求得，且和现有机器比较吻合：

$$
\left\{ \begin{array}{l} r _ {\varphi} = r _ {2} \sqrt {y ^ {2} + \frac {2 (R - \rho) \zeta}{\mu r _ {2}}} \sqrt {\frac {1}{3} \left(\lambda^ {2} + \frac {1}{\varepsilon}\right)} \\ \rho = r _ {2} \sqrt {1 + \left(\frac {e}{r _ {2}}\right) ^ {2} + 2 \frac {e}{r _ {2}} \cos \varphi} \end{array} \right.
$$

# 离心式低温泵充气迷宫密封技术

韩澎*

（杭州制氧机集团有限公司低温泵阀厂，杭州市东新路90号，310004）

摘要充气迷宫密封是一种非接触式气体动力密封，其结构特点为直通形迷宫，结构简单，形状很象梳齿，密封有很大的直通效应。介绍了离心式低温泵两级充气迷宫密封的结构原理和设计依据。图1。

主题词：离心式低温泵 充气垫宫密封 原理 结构 设计 依据

# 一、概述

离心式低温泵的密封寿命是最为常见的问题，密封性能的好坏将直接影响到低温泵运转的安全可靠性，因此它是衡量低温泵性能是否优良的重要指标。

密封的寿命取决于操作的环境（吸入压力、排出压力、流量、工作转速、每天启动次数、总的运行时间、气蚀条件及预冷时间等）。机械密封的寿命通常是在  $100\sim 2000$  小时之间变化，对于在连续工作条件或某些密封要求苛刻的工作场合，这样的寿命期是无法满足使用要求的。而充气迷宫密封的应用将使密封寿命达到20000小时以上，从而大幅度提高低温泵运行的安全性和可靠性。

# 二、密封原理

充气迷宫密封是一种非接触式气体动力密封，其结构特点为直通形迷宫，结构简单，形状很象梳齿，密封有很大的直通效应。工作原理是在泄漏通道内由许多齿或槽组成迷宫式的间隙，当低温液体经过每一个密封齿时，流体经间隙高速进入环形空腔后，突然膨胀而产生强烈的漩涡，使低温流体的大部分能量转化为热量而汽化。该汽化的气体压力逐级下降，与外界提供的压缩密封气体在迷宫密封段的某一部位建立压力平衡，从而达到密封的目的（见图）。

![](images/01f68b857188a7316ad472bfd254d13d4902847ae5a4f4f1fb910dd36fba0b2a.jpg)  
充气迷宫密封原理示意图

充气迷宫密封的泄漏率，与转轴和机壳上静止的密封套的间隙有关，而密封有效长度与它的密封气压力的平衡有密切关系。但是，关于密封机理的提出却是假定了一个具有理想工作状态的无振动的密封，其密封与泄漏是在间隙均布的条件下，能否在有效密封长度内建立压力平衡所决定的。然而，这种假定与实际密封是很难符合的，对于密封建立的条件之所以难以阐明，是因为在密封间隙上集中的条件（成熟的数学模型、振动、加工精度、给定的间隙及介质的粘度等）过多，而且实验上有困难的缘故。因此，无法精确的进行理论上的分析。

# 三、结构设计依据

由于充气迷宫密封型低温泵输送的介质主要是液氧、液氮、液氩等深冷流体，其特点是超低温、易汽化、粘度小。所以，在结构设计上出现了两个相对矛盾的问题：如果

有效密封长度伸长，容易建立气相压力平衡，而且泵端轴承的润滑温度容易控制，其缺点是振动问题突出，特别对于卧式结构布置，悬臂轴在较高转速下的振动，使得密封间隙无法控制；如果有效密封长度缩短，机组运行的稳定性提高了，随之而来的问题是气相压力平衡点不稳。特别是在泵吸入压力波动的情况下，当密封气体压力偏高时，密封气体（常用氮气或仪表空气）将会进入流体介质内，这将直接影响输送介质的纯度，甚至在低温泵的吸入口由气体侵入而发生汽蚀。

另外，由于热桥(国外文献称为冰桥)作用的影响，低温介质将冷量沿轴截面带到轴承部位，引起轴承润滑条件的恶化，使得密封气加热器启动频繁。

为了均衡两者的利弊，一般在结构设计上采用有效密封长度与轴径之比  $\mathrm{L} / \mathrm{D}$  为1.2\~1.5，转轴与密封套的间隙控制在  $0.015\sim$ $0.045\mathrm{mm}$  范围内，轴的径向跳动量允许在 $0.02\sim 0.05\mathrm{mm}$  范围内。从产品的实际运行情况看，以上数据基本上能满足密封要求。

充气迷宫密封采用较多的密封结构布置是两级密封气体密封。第一级密封气或称主密封气，其主要作用是阻隔叶轮背部的低温液体，使其与密封气在有效密封段建立压力平衡。密封气体的压力必须高于低温泵叶轮背面液体压力，其压差值一般为  $0.02\mathrm{MPa}$  具体偏差值应根据操作情况来决定。

该密封处不仅要对密封气压力及叶轮背部的液体压力进行较精确的测量，而且可以进行自动控制其压差和联锁报警。该处所用的密封气必须是完全干燥和清洁无油的，任何微小的湿气和油蒸气都会在密封气进口处凝结，堵塞管口甚至凝结物进入迷宫密封间隙，导致磨损而增加泄漏。对于输送某种高纯度低温流体，该密封气可以考虑利用输送介质做密封气源，也就是在低温泵出口附近的高压侧引出少量液体，经汽化稳压后，作

为密封气进入迷宫密封腔进行密封。这种方法的最大优点，是输送的流体纯度不受影响，而且可以大大的简化它的控制过程。

第二级密封气或称为副密封气，其主要作用是将从低温泵迷宫密封处泄漏出来的气体和从轴承处泄漏出的被污染的空气隔离吹除；另一方面也可以由副密封气带入热量以防止泵侧轴承过冷。该密封气一般应经加热稳压后，进入低温泵密封腔。气体进入泵内的最低温度应为  $20^{\circ}\mathrm{C}$ ，副密封气进口处的压力一般应设置在  $0.01\sim 0.05\mathrm{MPa}$ ，必要时可增加到  $0.1\mathrm{MPa}$ 。为保证泵安全可靠运行，副密封气管路上应配有断气报警联锁装置。

单级充气迷宫密封结构，目前使用比较少，它一般采用浮环式充气迷宫密封。其结构原理为径向间隙密封和轴向端面密封相结合的一种流阻型非接触气体动力密封。其特点是浮环靠端面滑动的可能性来保证自由定位，结构简单，适用于高速、高压等苛刻工况条件下密封。但密封件制造精度要求高，材料选择随温度的不同对密封的影响较大。在空分设备中要满足低温泵连续运转，其使用中诸多影响因素的处理尚须实践考验。

# 四、应用

本文介绍的两级充气迷宫密封式低温泵，已广泛用于我公司空分设备的流程中。它们可满足运转可靠性和对工艺流程适应性方面的要求，而且可毫无困难地接入完全自动化的调节系统内。随着全精馏制氩工艺流程的应用，充气迷宫密封技术将会更加展现出它所特有的优势。

# 一、引言

KL-15型制氧机是航空兵场站部队的主要装备之一，它采用低温液化分离的方法，生产航空飞行和维修所必需的氧氮产品。其操作培训一直单纯依靠实装进行。由于操作复杂耗时，稳定性差，故障率高，工质流动状况不能直观显现，操作步骤不能简单地重复，运行周期能耗高，因而实习教学困难较大。为此而设计制氧机模拟练习器，以期实现训练手段与器材的现代化。

# 系统理论与应用

程华平*

（杭州制氧机集团有限公司技术服务处，杭州市东新路90号，310004）

摘要 氧气透平压缩机轴封差压系统由两部分组成：一是油封系统，另一是气封系统。文章介绍了气封系统中吸入氧气、混合气、轴封氧气、轴封氮气四部分压力的设置及控制，有实用价值。图2。

# 主题词：氧气透平压缩机 轴封差压系统 气体压力 设置 控制

氧气透平压缩机（以下简称“氧透”）鉴于其介质助燃、易爆的特点，要求其保安措施远比空气、氮气透平压缩机复杂，轴封差压系统作为技术引进的一个组成部分，就是诸多保安措施中的一种。该系统由两部分组成，一个是油封系统，另一个是气封系统。油封系统控制单一，操作简单，本文着重对气封系统进行分析，见图1。

![](images/080233096b41afc2a498dfd152cf607e38ae99a304e6282686d26ddcd6053bef.jpg)  
图1 双缸氮透气封系统示意图

该系统由四部分组成：吸入氧气、混合气、轴封氧气和轴封氮气。设其压力为  $P_{\bullet}$  产 $P_{m},P_{0}$  和  $P_{n}$  ，按引进规定，其关系如下：

$$
P _ {m} = P _ {s} + \Delta P _ {1} \tag {1}
$$

$$
P _ {0} = P _ {m} = P _ {m} + \Delta P _ {2} \tag {2}
$$

控制要求：  $\Delta P_{1} = 2.5\mathrm{kPa},$

$$
\Delta P _ {2} = 4. 0 \mathrm {k P a}
$$

控制功能：  $\Delta P_{2} < 1.0\mathrm{kPa}$  时，报警；

$\Delta P_{2} < 0.5 \mathrm{kPa}$  时，联锁。

上述表达式及控制构成气封系统的基本理论，其意义在于：防止氧气向外泄漏，同时又不能影响氧气的纯度。应用时，引进技术要求，差压  $\Delta P_{1}$  、  $\Delta P_{2}$  调至目标状态后，操纵吸入压力，使之在  $P_{\pm} \pm 3\mathrm{kPa}$  范围内波动，若  $\Delta P_{1}$  、  $\Delta P_{2}$  能够稳定跟踪，则系统处于正常工作状态。

纯理论分析式(1)、(2)，压力  $P_{0}$  、  $P_{\bullet}$  产 $P_{m}$  只是吸入压力  $P_{\bullet}$  的函数，求解方便，因此控制  $\Delta P_{1}$  、  $\Delta P_{2}$  轻而易举。实际情况远非如此，把式(1)、(2)换一种表达方式；

$$
\begin{array}{l} \Delta P _ {1} = P _ {m} - P. \\ \Delta P _ {2} = P _ {0} - P _ {m} = P _ {n} - P _ {m} \\ \end{array}
$$

可以看出，控制参数  $\Delta P_{1}$  、  $\Delta P_{2}$  是四个相关参数的函数，如将控制值分别设置为2.5kPa、4.0kPa，那么任何  $P_0$  、  $P_{n}$  、  $P_{m}$  、  $P_{s}$  及仪表、阀门开度的变化率超过一定量时，都将引起系统不稳定。系统经自动调节后若不能达到新的平衡，将遭受破坏。

实际证明，控制参数  $\Delta P_{1}$  、  $\Delta P_{2}$  的调节，有时只有通过调整仪表的积分时间、比例带甚至差压变送器的阻尼，才能维持系统的稳定。这种通过降低仪表的灵敏度以达到调节

目的的手段，严格说来是不科学的。

下面讨论一种更简单有效的手段。把式(1)改为：

$$
P _ {m} = P _ {c} \quad P _ {c} - \text {常 压} \tag {1'}
$$

即混合气压力由原来氧气吸入压力的函数设定为常压，系统中的相关参数则由原来的  $P_{0}, P_{n}, P_{m}, P_{c}$  减至  $P_{0}, P_{n}, P_{m}$ ，且三个当中有一个  $P_{m}$  是常数，实际上只剩下两个不相关参数  $P_{0}, P_{n}$ ，如此控制就简单了。而且只要  $P_{c}$  设置合理，流程不会产生任何变化，保安也不会削弱。

$P_{c}$  的设置，与吸入管网压力有关，一般可设置为  $P_{c} = P_{s} + 2.5\mathrm{kPa}$ ；若吸入压力波动很大，以至正向波幅超过  $6.5\mathrm{kPa}$ ，如仍按前方法设置，因  $P_{0} < P_{s}$ ，轴封氧无法回流，系统破坏，这种情况下， $P_{c}$  必须在现场重新设定。因此，设置  $P_{c}$  必须保证：

$$
P _ {0} > P _ {\bullet}
$$

系统才能正常工作。当然，  $P_{0}$  也不能超出 $P_{\bullet}$  太多，否则，将过多地增加轴封氧气的泄漏量与循环量，轴封氮气的泄漏量。

为了保证  $P_{0} > P_{\bullet}$  ，并最大限度地减少轴封氧的泄漏量与循环量、轴封氮的泄漏量，在控制上增加如下功能：

$$
P _ {0} - P _ {s} <   4. 0 \mathrm {k P a}
$$

或  $P_{0} - P_{s} > 11.5\mathrm{kPa}$  时，报警，调整 $P_{c}$  设定值；

$$
P _ {0} - P _ {s} <   1. 0 \mathrm {k P a} \text {时 ， 联 锁 。}
$$

$P_{0} - P_{1}$  分别为  $6.5\mathrm{kPa}$  （原理论要求）、 $11.5\mathrm{kPa}$  时的轴封氧的泄漏量与循环量、轴封氮的泄漏量对比见图2。

$P_{0} - P_{\bullet}$  分别为  $6.5\mathrm{kPa}$  和  $11.5\mathrm{kPa}$  ，可以得到：

轴封氧总循环量：

$$
\Delta Q _ {0 c} = 7 6 4. 0 / 7 7 3. 1 \mathrm {m} ^ {3} / \mathrm {h} (\text {标})
$$

轴封氧总泄漏量：

$$
\Delta Q _ {0 1} = 4 0. 3 / 4 1. 1 \mathrm {m} ^ {3} / \mathrm {h} (\text {标})
$$

轴封氮总泄漏量：

$$
\Delta Q _ {n l} = 1 8 0. 1 / 1 9 0. 0 \mathrm {m} ^ {3} / \mathrm {h} (\text {标})
$$

比较式(1)与式  $(1^{\prime})$  ，不难得出：

1. 目的相同：建立一套保安系统，使得氧气不外漏，又保证氧气的纯度；侧重点不同，由此造成应用差距。

2. 式(1)表示的是一种“跟踪理论”，其主导思想是气封系统中的其余压力必须跟踪吸入压力，以免系统遭受破坏。它也允许入口管网波动，只要在  $\pm 3\mathrm{kPa}$  范围内，差压系统能稳定跟踪。实际管网压力波动很有可能超过  $\pm 3\mathrm{kPa}$ ，“跟踪理论”应用在这种情况下，就可能导致整个系统处于一种无政府状态；如果按式(1')设置，无论入口压力波动多大，只要预先设定  $P_{c}$ ，使得  $P_{0} > P_{s,\max}$ ，气封系统就稳定工作。

3. 如果管网条件突然变化（如换网），导致入口压力骤然大幅升高或降低，应用“跟踪理论”时，唯有在管网变化之前，解除差压联锁，调整  $P_{s}$  的设定值及有关仪表，才不至于引起系统瘫痪；而采用式  $(1^{\prime})$  ，则无须如此繁杂操作，只需在管网变化后重新设定  $P_{c}$  就能满足工艺流程要求。

显然，用式  $(1^{\prime})$  代替式(1)，原有保安功能不失，但控制与操作更简单、更方便，因此应用性更强。

# 一、输液操作的时间

本设备在制订操作规程时，考虑到透平膨胀机高温高焓降的特点，确定为分阶段开车操作方式。因此，在实施输液（氧）操作时，将输液安排在第四阶段（深冷全系统一积液）内进行。同时，考虑到设备的热应力问题，输液操作一般应在主冷液氧侧冷到  $-100^{\circ}\mathrm{C}$  以下时进行，当然主冷温度若再低些则更为理想。但是空分塔主冷的冷却，是通过TG-116阀（膨胀机后进上塔空气阀），经主冷吹除阀形成冷却流路，冷却效果相对较差，所以主冷在第四阶段能冷却到  $-100^{\circ}\mathrm{C}$  就算十分理想了。

# 二、输液操作方式

根据本文中所提供的简单工艺图示，设由  $1^{\#}$  塔(运转中)向  $2^{\#}$  (启动中)输液，则应按如下方法操作：

1. 全开连通阀和  $1^{\#}$  塔主冷排液阀；  
2. 微开  $2^{\#}$  塔主冷排液阀，观察主冷温

度的下降幅度，控制在10分钟后主冷温度达 $-180^{\circ}C$  为宜，不要太快；

3. 主冷温度达到  $-180^{\circ} \mathrm{C}$  之后，主冷已建立起积液所必须的温度，此时可开大  $2^{\#}$  塔排液阀，使  $1^{\#}$  塔主冷液氧更多地进入  $2^{\#}$  塔主冷；  
4. 输液时  $1^{\#}$  塔主冷底部压力为0.055MPa，  $2^{\#}$  塔主冷压力为0.025MPa，由于压差较小，同时输液管线约有50米左右，所以输液速度较慢，一般需要  $1\sim 2$  小时；  
5. 根据我们操作发现，当  $2^{\#}$  塔主冷液面达到  $500\mathrm{mm}$  时，已输不进液氧了，此时应停止输液操作，关闭上述三个阀门。

# 三、输液操作时要注意的问题

输液操作虽不复杂，但应注意下列几个问题：

1. 一般来讲，输一次液氧，估计要  $1 \sim 1.5 \mathrm{~m}^{3}$ ，所以输液的空分塔必须积累足够的液氧。以我厂空分塔参数为例，正常时主冷液面控制在  $20000 \sim 25000 \mathrm{~mm}$  水柱，如进行输液操作，主冷液面应控制在  $30000 \mathrm{~mm}$  水柱；输液结束时，则降到  $23000 \mathrm{~mm}$  水柱，不

影响本塔工况。

2. 被输液的空分塔，应在开始输液操作时，开大TG-117阀（膨胀机后空气进污氮过冷一液化器阀），强化下塔空气液化量，使下塔有足够的液空。并及时预冷吸附器，JL-101阀(液空进上塔节流阀)投自动控制，使下塔液空形成向上塔侧的流路，以保证输液效果。  
3. 其余操作则应该按积液时必须注意的事项进行，比如：尽量降低上塔工作压力；减少进塔空气量；发挥膨胀机最大制冷能力等，以配合好输液操作。

# 四、采用输液操作的好处

我厂自采用输液操作法后，使空分塔启动时间大为缩短，增加了供氧供氮的可靠性，同时经济效益十分明显。

以缩短启动时间10小时计算，可节电20000余度，多生产氧(氮)气  $64000\mathrm{m}^3$  ，价值40000余元。

目前，我厂已将此项操作工艺，总结后正式列入设备操作规程中。

# 一、流程叙述

# 1. 生产液氮工况

自空分设备来的  $10000\mathrm{m}^3 /\mathrm{h}$  、  $15^{\circ}C$  的产品氮气，经循环氮气压缩机CNCP升压到 $2.9\mathrm{MPa}$  ，送入液化冷箱，进入  $1^{*}$  换热器与返流氮气换热，温度降到  $-10^{\circ}C$  。后进入氟里昂冷冻机蒸发器，冷却到  $-25^{\circ}C$  ；再经  $2^{*}$  换热器与返流氮气换热，温度降到  $-98^{\circ}C$  。从 $2^{*}$  换热器出来的氮气分两步走：占总量  $85\%$  的氮气去中压膨胀机，压力为  $2.9\mathrm{MPa} / 0.8$

$\mathrm{MPa} / 0.3 \mathrm{MPa}$ ，温度为  $-98^{\circ} \mathrm{C} / -142^{\circ} \mathrm{C} / -191^{\circ} \mathrm{C}$  。膨胀后的低温氮气由  $3^{\#}$  换热器冷端进入，作为  $3^{\#}, 2^{\#}, 1^{\#}$  换热器的冷源，复热后出冷箱，去CNCP的入口，循环使用； $15 \%$  的氮气进入  $3^{\#}$  换热器与返流低温氮气换热，温度降为  $-181^{\circ} \mathrm{C}$ ，并有一定的过冷度。 $2.9 \mathrm{MPa}$  的液氮再经节流阀PCV-6203节流，压力降到  $0.3 \mathrm{MPa}$ ，成为过冷液氮进入冷凝蒸发器的液氮侧，液化产品由冷凝蒸发器侧出，经HCV-6203控制后去液氮贮罐。

# 2. 生产液氧工况

自空分氧压机来的产品氧气，压力2.1MPa(流量约为  $1000\mathrm{m}^3 /\mathrm{h})$  进入冷箱经  $4^{*}$  换热器，温度降为  $-139\%$  ，过冷而全部液化。经节流阀PCV-6204进一步过冷，压力降到 $0.3\mathrm{MPa}$  ，进入冷凝蒸发器的液氧侧，成为液体氧，由调节阀HCV-6201控制后去液氧贮罐。

# 二、存在的问题

1. 从流程中我们知道，从  $2^{\#}$  换热器出来温度为  $-98\%$  的氮气产品，首先第一步进入  $1^{\#}$  换热器，温度降到  $-10\%$ ；第二步进入氟里昂蒸发器，也就是我们所说的液化预冷系统，温度由  $-10\%$  降为  $-25\%$ 。但长期以

来，由于存在着诸多问题，预冷系统一直得不到投用，使冷量大为损失。

2. 从流程中我们知道，从  $2^{\#}$  换热器出来的温度为  $-98^{\circ} \mathrm{C}$  的氮气，分两步走：占总量  $85 \%$  的氮气  $(8500 \mathrm{~m}^{3} / \mathrm{h})$  进入中压、低压膨胀机，压力为  $2.9 \mathrm{MPa} / 0.8 \mathrm{MPa} / 0.3 \mathrm{MPa}$ ，温度为  $-98^{\circ} \mathrm{C} / -142^{\circ} \mathrm{C} / -191^{\circ} \mathrm{C}$ 。 $8500 \mathrm{~m}^{3} / \mathrm{h}$  的氮气，通过膨胀机膨胀制冷后作为主要冷源，使占总量  $15 \%$  的氮气  $(1500 \mathrm{~m}^{3} / \mathrm{h})$  和  $1000 \mathrm{~m}^{3} / \mathrm{h}$  的氧气全部液化。但是长期以来，膨胀机的制冷效果一直达不到设计指标，8500  $\mathrm{m}^{3} / \mathrm{h}$  的氮气量吃不进去，最高只能达  $5500 \mathrm{~m}^{3} / \mathrm{h}$ ，由于气量不足，所以产生的冷量大为减小。  
3. 机体密封效果不好，风机出口有喷油现象。  
4. 振动大，噪声大。  
5. 密封气线、循环水线、润滑油线布置混乱，设计不合理。

由于液化系统存在着上述问题，使生产能力受到限制，一直达不到设计指标，原设计生产液氮  $1.94\mathrm{m}^3 /\mathrm{h}$  ，液氧  $1.25\mathrm{m}^3 /\mathrm{h}$  ；现只能生产液氮  $1.1\mathrm{m}^3 /\mathrm{h}$  ，液氧  $0.7\mathrm{m}^3 /\mathrm{h}$  。

# 三、改造的主要内容及目的

1. 更换膨胀机的转子，将二元流叶轮改为三元流叶轮，将键联接改为桃型轴联结。  
2. 更换风机壳体, 改变密封结构和材质 (由石墨改为玻璃钢)。

以上两项工作由哈尔滨制氧机厂和我们两家共同协商解决，具体设计、加工由哈尔滨制氧机厂完成。安装调试由我们负责。

目的有四个：

（1）增大膨胀氮气量，使膨胀机的膨胀制冷能力发挥到最大，冷量损失达到最小。  
(2) 减小振动，降低噪声。  
（3）除去冷却水。  
（4）解决风机出口喷油现象。  
3. 改变密封气线和排油烟线的布置。

4. 预冷系统除锈，酸洗。  
5. 压缩机出口增设油气分离器，蒸发器加低点排油点。

以上三项工作由我们单独完成。目的有三个：

（1）使密封气压力合理，排油烟管路畅通。  
（2）预冷系统由于长期闲置，设备内部杂质积聚很多，通过酸洗清洁干净。  
（3）预冷压缩机出口带油非常严重，使油箱油位保持不住，油压不稳定，增加油气分离器，把油分离后重新送回油箱。蒸发器加低点排油点，把氟里昂夹带的油分排出，使换热器的冷量转换效率发挥到最佳程度。

# 四、预冷系统的酸洗

# 1. 清洗液

清洗液选用了华阳恩赛有限公司生产的DEOX,为什么选用DEOX清洗液呢?主要因为系统材质是由碳钢、铜合金组成,而且附有大量污油,DEOX是由硝酸、有机酸、加缓蚀剂、分散剂等组成,具有除锈、除油等多种功能。

# 2. 酸洗方案

（1）将进、出蒸发器的管线连接好形成循环。  
（2）将酸泵出口和压缩机入口联接（注意：千万防止酸液进入压缩机）。  
（3）将压缩机出口和酸洗槽联接好（注意：千万防止酸液进入压缩机）。  
（4）加水到形成循环量的  $1.5\sim 2$  倍，检查有无泄漏、内漏等(如水太脏更换之)。  
（5）加酸洗剂DEOX，量为水的1/5左右时，开始冲洗。  
（6）每隔15分钟用试纸测一次  $\mathsf{pH}$  值，如  $\mathsf{pH}$  值大于7，补加适量的DEOX。  
（7）每隔30分钟测一次铁离子和铜离子浓度，如果离子浓度增加很快，要立即停止酸洗。

（8）连续循环  $4\sim 6$  小时后，若  $\mathsf{pH}$  值在1小时内无变化，则认为酸洗合格。  
（9）排液后，加  $\mathrm{Na_2CO_3}$  或  $\mathrm{NaHCO_3}$ ，在  $2\% \sim 3\%$  浓度下中和1小时。  
（10）然后进行水冲洗，并在各设备、管线的低点放空。  
（11）最后用干燥的仪表风或氮气吹扫，直至无杂质，无残液。

# 3. 酸洗实际操作过程

酸洗从早上9时40分开始，下午1时50分结束，按方案将清洗剂稀释5倍，每隔15分钟测一次  $\mathsf{pH}$  值，始终保持  $\mathsf{pH}$  值小于7（中间填过4次酸）。由于化验速度不能满足要求，时间太长，因此只取了三次样，见表1。

# 王守东

（黑龙江省科学院石油化学研究所，哈尔滨市南岗区中山路164号，150040）

摘要 介绍哈氧厂KDN-150/10Y型高纯制氮设备的技术改造，经过多年运行，证明技改提高了设备的性能，同时整套设备降低成本  $8\%$  。图2。

# 主题词：空气分离设备 技术改造 经济性

为了给玻璃行业及其它应用高纯氮的行业提供新的装备，哈尔滨制氧机厂于1993年研制成功KDN-150/10Y型高纯氮设备。这套设备采用返流气膨胀循环流程，先进的配套单机（如无润滑压缩机、气体轴承透平膨胀机等）及适当提高氮气输出压力等新技术。近年来，随着空分技术的不断发展，哈尔滨制氧机厂对比了德国、日本高纯氮设备的有关资料，并对一些设备进行了改造，提高了整机性能。经多年运行，证明改造是成功的。

# 一、主要改造项目

# 1. 新增氟利昂预冷机组

原流程，空气经过滤清器清除灰尘和机械杂质后，由无润滑压缩机二级压缩至0.88MPa后，直接进入分子筛纯化器组。这种流程，易造成进入分馏塔的空气随环境温度的波动而变化。新增氟利昂预冷机组后，稳定了工况，延长了纯化器的切换周期，减少了再生氮气的消耗量，同时还减小了板翅式换热器的热负荷。

# 2. 纯化器

纯化器封头、筒体壁厚，由  $8\mathrm{mm}$  改为 $6\mathrm{mm}$  。吸附器采用内外夹层结构，代替原来的单层结构。设置夹层的目的是为了绝热，减少能量损失。内筒直径  $\phi 600\mathrm{mm}$  ，外筒直径  $\phi 700\mathrm{mm}$  ，进出口处均设有过滤器。分子筛采用高效UOP13X-APG8×12目球状分

子筛，替代原5A分子筛。运行结果证明，13X-APG分子筛，对空气中的  $\mathrm{H}_2\mathrm{O}$  和  $\mathrm{CO}_{2}$  净化效果极好，使得精馏塔内各点压力、温度、产品纯度均较稳定，透平膨胀机也未发生因  $\mathrm{CO}_{2}$  增加而造成气体轴承卡机、喷嘴堵塞现象。分子筛装填量减少，使纯化器体积减小，分子筛使用寿命较5A分子筛高两倍左右，从而节省了大量分子筛更新费用。

# 4. 精馏塔

# （1）将易脱落的塔板与液流槽铆接

下塔与蒸发器、下塔与主冷焊接时，温度较高，塔内离焊缝近的几块塔板易脱落。为此，将离焊缝近的三块塔板与液流槽进行铆接。

# (2）将铜塔改为铝塔

原精馏塔、蒸发器、下塔、主冷封盖，均用铜材制成。因铜材焊接技术难度较大，造成焊缝返修次数多，而有些容器的最后一道环焊缝无法返修，这样就给生产环节造成了很大的困难。现改为铝塔，这样做，有以下几个优点：

① 压力容器类别I类， $20\%$  探伤，制造成本大大下降。  
② 哈尔滨制氧机厂有焊接铝材的成熟经验，用铝材代替铜材后制造工艺简化，原料、工艺、管理等综合成本降低了近两万元。

# （3）主冷凝蒸发器改为干式

原设备的主冷凝蒸发器(简称主冷)是湿式，比干式主冷制造工序多，容器压力类别高，见图1、图2。

# 无润滑压缩机故障分析

楼忠毅*

（杭州制氧机集团有限公司活塞式压缩机厂，杭州市东新路90号，310004）

摘要针对无润滑压缩机常见故障，例如活塞环、导向环过快磨损，阀片断裂，气缸体或气缸套过快磨损，密封器失效等，进行了理论分析和说明。

# 主题词：无润滑压缩机 活塞环 导向环 阀片 气缸 密封器 故障 分析

一般在低压压缩机中，无润滑压缩机活塞环寿命已可达到8000小时以上，对于中压压缩机也可保证在4000小时。然而，由于有相当多用户的操作人员对无润滑压缩机的设计技术不了解，造成使用维护不当，而导致机器故障频繁发生。因此，在本文中，根据本人十几年设计工作经验，着重介绍一下无润滑压缩机的常见故障及其防范措施。

# 一、活塞环、导向环过快磨损的分析

无润滑压缩机的全称应为“气缸无油润滑压缩机”，也就是说，活塞的密封元件活塞环与气缸壁面间的润滑无需外加润滑剂就能够达到润滑的目的，并能保证易损件活塞环达到能够接受的使用寿命。

这种无需外加润滑剂就能使一对摩擦件之间产生润滑的情况，我们称之为“自润”。而无润滑压缩机所使用的填充聚四氟乙烯（即TPFE)活塞环和导向环，就是一种自润滑元件。其自润机理是依靠填充聚四氟乙烯和相应的对磨金属表面初磨合时，在摩擦面粘着一层极薄的由填充聚四氟分子为主所组成的自润滑薄膜。这种自润滑薄膜一旦建立之后，填充聚四氟乙烯环的运动状况就改善了，此后填充聚四氟乙烯环的摩擦面就处于填充聚四氟乙烯分子间的相对摩擦，这时摩

擦系数大大降低，仅出现稳定的微量磨损。

由此可知，在自润滑膜层建立之后，活塞环和导向环在气缸内往复运动时，与它对磨的并不是金属气缸壁面，而是其自身的薄膜。这种薄膜一旦被破坏，或无法在气缸壁面形成这种薄膜，就会造成活塞环及导向环直接与金属气缸壁面摩擦，在这种情况下活塞环和导向环就会很快磨损失效报废，也就是通常我们所说的活塞环寿命特别短。那么是什么原因使聚四氟乙烯环无法在气缸壁面建立薄膜或者建立的薄膜很快被破坏呢？根据填充聚四氟乙烯的特点，以下几个因素都会造成填充聚四氟乙烯环过快磨损：

1. 气缸表面光洁度未达到  $\leqslant \frac{0.8}{\sqrt{}}$  
2. 气缸表面锈蚀  
3. 被压缩介质粉层含量较大  
4. 气缸内有液相水分

气缸内出现液相水分有两种原因：

第一种原因是由主机和辅机部分的漏水造成的。当气缸内发现液相水分时，应首先仔细检查压缩机的气缸和冷却器，气缸主要检查气、水腔间有无渗漏，若有渗漏，可能是结构密封失效，或者是气缸出现裂纹。而冷却器的渗漏原因主要是密封垫片的失效、管板与冷却管的胀接松动及冷却水管破损。

第二种原因是由被压缩介质含水分造成的。这也有两种情形：

一种是由于被压缩介质因工艺需要经水洗涤后再进入压缩机，这种经水洗涤的介质在经过水分离器时，由于水分离器的分离效果差而导致介质含有液相水。

另一种情况是含湿量较高的介质，如经水洗涤分离后的饱和气体，或湿度较高地区的空气。这类介质在进入气缸冷却效果较好的压缩机时，会发生温降，如果温度下降到进气压力条件下的露点温度以下，湿气体中的水分就会析出，造成缸内带水。因此，在压缩湿量较高的气体时，要求气体进入气缸后的温度不能过低，特别是对低压级更应注意。解决的办法是，调节气缸的冷却水量，保证气体进入气缸后的温度，高于其进入气缸后该压力下的露点温度。按经验，一般调节气缸冷却水排水温度高于气体进气温度  $4 \sim 5^{\circ} \mathrm{C}$  。

5. 对应于某一种被压缩介质和气缸材质，填充聚四氟乙烯环的配方选择不当

填充聚四氟乙烯磨损量可按下式计算：

$$
\Delta = K P V t \tag {1}
$$

式中，  $K$  为磨损因子，  $P$  为气体压力， $V$  为活塞头平均速度，  $\pmb{t}$  为时间。

试验表明，对于同一种对磨金属材料，不同的介质对于同一种配方的填充聚四氟乙烯环，其K值是不同的；对于同一种介质，不同的对磨材料对于同一种配方的填充聚四氟乙烯环，其K值也是不同的。

因此，压缩机制造厂家在进行产品设计时是根据被压缩的介质及气缸或气缸套的材质来选择相对应的配方的。所以用户在购买填充聚四氟乙烯环备件时，最好直接向压缩机生产厂家购买。

6. 工作温度过高。

除了设计制造因素外，气缸内的工作温度高于设计值，往往由以下三个原因造成的：

（1）是各级或某级气缸进气温度高于设计值，也可能是冷却器冷却效果差。

（2）高压级气缸气体泄漏至低压级气缸内，使低压级气缸的压缩过程出现加热而使缸内温度升高。

（3）某级气缸的进、排气阀门泄漏。进气阀门的泄漏可通过手感检查发现，即用手摸进气阀门压盖，看其是否发热。排气阀门的泄漏如泄漏量较小，则必须进行排气阀门的煤油渗漏检查。如泄漏量较大，则会引起次一级气缸排气压力的升高，通过仪表可以发现。如三级排气阀门泄漏，则二级的排气压力就会升高。

7. 活塞环或导向环与气缸壁涨死

活塞环和导向环的涨死是在活塞环和导向环工作中受热膨胀发生的，在安装和拆车的常温状态下无法发现。在活塞环和导向环的设计中，活塞环的切口安装间隙是按活塞环在工作温度下，活塞环在圆周方向的伸长量来确定的。如果活塞环的切口安装间隙小于活塞环在受热膨胀后的周向伸长量，则在受热膨胀时切口完全封闭时，就会向径向方向延伸，造成活塞环径向涨死。而导向环与气缸壁的径向间隙则是根据导向环在工作温度下的径向膨胀量来确定的。因此导向环与气缸壁的间隙不能随意改小。一般生产厂家提供给用户的技术文件和图纸中都有间隙的尺寸要求。但是由于用户购买的活塞环和导向环在配方和质量上与原配方的差异，其热胀系数也就不同。

因此，用户购买其它厂生产的活塞环和导向环时，应要求生产厂家提供该环材料的热胀系数。在安装时应重新复验和计算活塞环的切口安装间隙和导向环的径向间隙。填充聚四氟乙烯环的热膨胀量可按以下公式进行计算：

$$
\delta = \alpha l \left(t _ {2} - t _ {1}\right)
$$

式中  $\delta$  ——受热后的伸长量

$\pmb{\alpha}$  ——填充聚四氟乙烯的线膨胀系数

$l$  ——活塞环的周向长度或导向环的径向厚度

$t_2$  ——工作时的温度， $\mathrm{^\circ C}$

$t_1$  ——安装时的温度， $\mathrm{^\circ C}$

如生产厂家不提供数据，则无法通过复算确定安装间隙，建议用户不要盲目选购。

# 二、阀片易断失效分析

无油润滑压缩机一般不使用环状阀，这是因为环状阀阀片的开启和关闭需要导向凸台。由于无润滑压缩机缸内无油，因此阀片边沿与导向槽失去润滑，使阀片容易发生偏折而断裂。所以无润滑压缩机一般都使用网状阀。网状阀阀片的频繁断裂主要是以下几个因素引起的：

# 1. 阀片质量差

阀片质量差，是指阀片的材料质量和阀片的加工质量，不能在有效期间内承担阀片在工作时所受的交变疲劳应力和频繁的冲击力，在短时间内就发生断裂失效。在一般情况下，具有一定生产规模的气阀生产厂家所生产的阀片在材料质量上差异不大。阀片的质量反映在加工质量上，为提高阀片的疲劳强度，要求阀片的表面光洁达到0.8以上，阀片的厚薄应均匀。阀片的平面度要好，阀片的翘曲度允差，其间隙值应控制在0.1mm以下。阀片的所有加工部位不得留有锐角。因此，用户在订购阀片时，阀片加工质量的检查是至关重要的，否则会给自己的生产带来不应有的损失。

# 2. 弹簧质量差

除材质外，弹簧质量差主要是指生产出的弹簧不符合图纸要求，弹簧的热定型工艺不过关。这种弹簧疲劳弹度低，其同一批生产出来的弹簧弹力也有很大的差异。一般网状气阀上至少装4个弹簧，如果每个气阀几个弹簧的弹力与其它弹簧不同的话，那么阀片在这种不均匀的弹簧力作用下，阀片在工

作时的应力不均匀状况就会加重。有些部位还会产生颤振，会使阀片某些局部部位的应力值超过阀片材料的弹度值而发生断裂。因此，弹簧质量是保证气阀正常工作的关键零件，用户应到正规气阀生产厂家购买。

# 3. 维修更换不当

维修更换不当是指在更换气阀弹簧将高压级的弹簧装在低级的气阀上，或低压级的弹簧装在高压级的气阀上，以及在更换弹簧时，只将个别失效弹簧更换而不是把某一阀门组件中的全部弹簧更换。气阀弹簧的级别换错会造成阀片在工作时的颤振；而只更换个别失效弹簧，会使阀片受到不均匀的弹簧力。这是因为使用过一段时间的弹簧，其刚性系数会发生变化而使弹簧力改变。另外，在气阀中发现有个别断裂失效弹簧时，一般意味着同一组中其它剩余弹簧的寿命也快到了。同一组弹簧全部更换的方法，国外也如此，这样可保证气阀工作的状态系数稳定，提高气阀的周期寿命。

# 4.管道气柱共振

管道气柱共振主要是指与气缸进、排气口连接的管路发生气柱共振。一旦被压缩气体在管路中发生气柱共振，就会使此管段中的气体压力波动值大大增加，从理论上讲，气柱在共振时造成的压力波动增值会无限增加。这种强烈的压力波动会使气阀的阀片产生颤振，导致阀片断裂。

因此，在压缩机运行时，如发现某一级气缸的进气管或排气管振动较大，并且此级的进气或排气阀门片断裂频繁，应考虑进、排气管出现气柱共振的可能性。在进行具体检查时，可在进气管或排气管上装上压力表来观察管道内的压力波动情况。一旦确认是管道内出现气柱共振，可在气缸的气体进、出口法兰处设置管道孔板，或者改变进、出口管路的长度，通过改变气柱频率的办法来消除管道内的气柱共振。

# 三、气缸体或气缸套过快磨损分析

气缸体或气缸套(下面简称气缸)的快速磨损，其磨损速度和磨损量大大超出正常允许期限，造成气缸的失效报废。我厂生产的某些产品，气缸发生磨损失效最短时间只有3个月。气缸的磨损多发生在高压级。由于无润滑压缩机活塞环材料为填充聚四氟乙烯，其强度和硬度都大大小小于气缸材料。在这么短的时间内就发生磨损失效，许多用户首先总怀疑是气缸材料质量和铸造质量不过关，而生产制造厂则首先认为是用户使用的压缩介质是否含有某些腐蚀性较强的介质。实际上在一般情况下，以上两种情况都不太可能发生。经多次现场检查发现，气缸的过速磨损主要是由以下两个原因造成的：

# 1. 活塞环和导向环在缸内涨死

活塞环和导向环（下面简称PTFE环）在气缸内涨死的原因在前面已经介绍了。PTFE环在气缸内涨死不仅会使PTFE环迅速失效，同时也会导致气缸过速磨损。涨环造成气缸过速磨损的原因，是当PTFE环在气缸内涨死后，PTFE环与气缸壁接触部分的比压大大高于正常运转时的比压。由于比压的增大，PTFE环与缸壁间的摩擦力也大大增加。一般压缩机的活塞平均线速度都在3m/s左右，在这种线速度和大摩擦力条件下，PTFE环与气缸壁之间的接触温度会急剧上升，这种高表面接触温度会大大降低气缸壁面的金属强度，使得气缸壁面容易被磨损。

另外，由于PTFE环与气缸壁涨死，使得被磨损下来的金属粉末和PTFE环粉末无法排出气缸，并紧紧地粘附在PTFE环和气缸壁面上，又造成了磨损的进一步加剧。

上述两种情况交替出现，形成恶性循环，使气缸迅速磨损失效。因此，涨环现象应引起用户和操作人员的高度重视。否则可能会引起生产事故，给用户带来损失。

# 2. 被压缩气体过脏

被压缩气体过脏是指被压缩气体含有较多的粉层、金属粉末、非金属纤维和水等杂质。这些杂质一旦进入气缸内，有一部分就会粘附甚至嵌入PTFE环的表面，如果粘附和嵌入PTFE环表面的杂质太多，这时的PTFE环表面就象一张砂纸，使得气缸壁面不是和PTFE环摩擦，而是和这些杂质摩擦。这种摩擦就象磨削一样，使得气缸壁面极易磨损。

一般来说，如果进入气缸内的杂质颗粒较粗，在气缸壁面上会产生较明显的拉痕。如果压缩机在运行时，涨环现象和被压缩气体杂质过多情况共存，那么在短时间内造成气缸的过速磨损失效，也就可以理解了。

# 四、密封器的失效分析

密封器也称填料函，它是由密封气缸和活塞杆间隙的填料元件，也即密封圈组成的部件。填料的密封原理和活塞环相同，即利用阻塞和节流两种作用的组合。密封器的失效即指其填料元件的失效，造成压缩气体的泄漏量大大超过允许值。密封器过快失效的主要原因一般都是密封环被卡死造成的。在安装时，如果轴向间隙值  $\delta = 0$  ，或在工作时密封圈受热膨胀造成  $\delta = 0$  （这种情况我们俗称“卡死”）。在运行时，由于密封圈被卡死，在密封盒内无法浮动，也就无法通过弹簧力达到自紧，造成密封失效。因此在安装完密封器部件后，应检查密封圈与密封盒的轴向间隙，是否符合制造厂提供的图纸要求。

在组装密封器时，密封器中所有的零件都应清洗干净，特别是密封盒和密封圈的表面不得粘有任何粉层、颗粒，否则会造成密封平面的气体泄漏而直接导致密封器失效。

主题词：空气分离设备 制冷机组 故障 分析 解决办法

我厂于1992年底安装一套由开封空分设备厂生产的运行周期达一年的  $1500\mathrm{m}^3 /\mathrm{h}$  高纯氮装置。由于设计要求入纯化器气体温度为  $5\sim 10^{\circ}\mathrm{C}$  ，故配套两台制冷量为211433.4  $\mathrm{kJ / h}$  的FB-8045型制冷机组（以下简称冰机)，制冷工质为R-22。

冰机在该装置中担负预冷工作，它的工作状况将直接影响分子筛对空气的净化程度。由于净化后空气中的  $\mathrm{CO}_{2}$  含量过高，堵塞换热器，温差增加，精馏工况恶化，产品氮纯度失控，被迫对设备进行加热、吹扫。这样不仅浪费严重，同时极大地影响了全厂生产。另外，由于冰机制冷效果不好，使分子筛吸附过饱和，再生、加热时间增加，冷吹时间也相应增加，不能按时切换。设备运行中流量、质量波动较大。

针对以上情况，我们组织各方面力量进行联合攻关，以设备安装阶段的要求和指标逐项查检、整理。经过两年的探索、实践及整改，目前冰机状况已全面正常。而此间，我们也掌握了一些行之有效的办法，并取得一些实际工作经验。1995年5月份，我们又针对新空分存在的问题，走访了一些兄弟单位，发现在冰机部分普遍存在着这样那样的问题。为此我们将冰机典型故障的判断、检修和改进情况加以整理，与同行们进行进一步的研究、探讨。

# 一、FB-8045冰机故障

我们遇到最频繁的故障，是轴封器的损坏问题和压板的漏油。长时间以来过热这个问题一直困扰着我们。通过一段时间的摸索，我们发现造成频繁故障的主要原因有：

# 1. 压盖没有压均

压盖没有压均，造成漏油气和摩擦过热。解决办法：开车前对压盖要均匀受压，开车后待主机升温后，停车再对压盖螺母加以紧固，消除由于热涨冷缩而造成的封垫渗漏。

# 2. 石墨密封环损坏频繁

轴封器中石墨密封环损坏频繁，造成的原因除因石墨环研磨不佳以外，最重要的是来自外部原因，如：弹性联轴器变形、弹性橡胶环损坏及丝杆磨损严重造成的曲轴受力偏心、振动过大，石墨环在运行当中被损坏，造成大量的R-22氟里昂及油外溢。消除的办法：除加强检查外，还应随时注意冰机的声音，一旦噪声增大就及时停车处理，将故障消除在初期。另外我们曾发现两台冰机的曲轴和电机轴承不同心，这也是振动过大的重要原因。为此针对同心度做了测量和调整。

总的来说，对轴封器日常维护工作我们强调三点：①听噪声变化情况；②看渗油情况；③摸压板温度。

经过一年多的运行证明，轴封部分故障率直线下降，全年只检修2次，而其中的一

次为操作中失误造成。

# 二、油压及润滑系统故障

设备运行初期，由于冰机油泵油压不能满足  $0.15\sim 0.3\mathrm{MPa}$  压差要求，而使其压差保护器频繁起跳，对生产影响极大。

针对冰机主机8FS7的特点，我们在以下几方面进行调整：

1. 注油阀、油压调节阀及油泵调节失灵

注油阀和油压调节阀及油泵多次调节均无效，造成这种情况的原因是由于冰机  $17^{\#}$  油在设备初始运行时，因油温低，粘度大，压差尚能保持，而运行一段时间后，冰机油温上升，粘度下降，油泵产生较大的自然回油现象，压差不能保持。我们处理的方式是调节油压阀至最大压侧，同时适当降低保护器的压差为  $0.1\mathrm{MPa}$  。这样，会遇到润滑油量能否满足主机需要的问题。为保护主机，我们采用逐渐降低压差，勤观察，防止发生连锁反应并更换粘度较大的  $60^{\#}$  冷冻机油（同  $17^{\#}$  比较粘度增加42ct)，直致主机稳定工作。与此同时，我们还适当增加油箱的油位，防止由于自动回油阀工作不佳造成拉缸和过热现象，同时，微开手动回油阀，实践证明效果极佳。

2. 轴瓦及曲轴端面长期磨损

轴瓦和曲轴端面由于长期间磨损，使间隙增大，造成轴瓦套内油压得不到保证，自动停车。我们采取的措施为适当调整间隙量，同时对两个端面进行适当的研磨，防止了大量泄油（见图）。

对油泵的外盖端面进行检查和压紧量重新调整。调整时应注意，两外盖选用铅板 $(0.5\sim 0.8\mathrm{mm})$  或紫铜板  $(0.3\sim 0.5\mathrm{mm})$  及钢质垫。因为这两处对密封要求极高，如有条件可以加耐油胶水。

# 五、结语

作为  $1500\mathrm{m}^3 /\mathrm{h}$  空分高纯氮装置的重要配套设备之一，冰机的工作优劣直接影响着全套设备运行状况。但是，由于含有动力电路、仪控电路、工质回路、冷却水回路、空气线路及油路等许多回路，故而在判断故障、排除故障时，有许多困难。但是我们觉得，通过一段时间的使用，掌握它的运行规律，保障它完好运行是可以办到的。

预冷工序对空分行业既是老课题又是新课题。更新自动化程度更高的冰机，诚然是好举措，但并不是唯一的途径。在此我们真诚希望和同行们交流探讨，更希望得到同行的建议和批评，以共同完善空分净化工作。

# 板翅式换热器氧通道进液事故分析

南伟

（西宁钢厂动力部制氧车间，青海省西宁市，810005）

西宁钢厂  $1500\mathrm{m}^3 /\mathrm{h}$  空分设备是1985年安装投产的，该设备从日本神户制钢所引进，并于1989年对其进行了技术改进，改原空气膨胀流程为氮气膨胀流程，氧气产量由1500  $\mathrm{m}^3 /\mathrm{h}$  提高到  $1850\mathrm{m}^3 /\mathrm{h}$  。经三年多生产运行实践，工况一直正常。

但是，在1992年12月1日9:30，这套KOBE-22型空分设备的板翅式换热器一、二、三组氧通道同时发生进液事故，使板翅式换热器严重过冷，精馏工况急剧恶化，各部技术参数异常。由于发现及时，处理措施果断，没有造成设备损坏。经4小时精心操作调整，运行工况趋于正常状态，各部技术参数达到正常值，空分设备正常生产至今。

# 一、事故状态

在1992年12月1日9:30，上塔超压报警，发现KOBE-22型空分设备的板翅式换热器，三组环流温度同时急剧下降，温度达到 $-174^{\circ}C$  ，中部温度同时呈水平线速降。上塔压力超压，顶部安全阀开启，紧急排放，上塔底部液面下降，上塔各部阻力波动剧烈，精馏工况呈液悬症状。冷凝蒸发器液氧液面超过  $1806~\mathrm{mmH_2O}$  的定值，氧气流量指示亦超过  $1800\mathrm{m}^3 /\mathrm{h}$  （指针被打到头），氧气压缩机因吸入压力升高，被迫自动紧急停机。

# 二、事故处理

在板翅式换热器氧通道进液事故处理过程中，由于果断采取了一些合理正确的措施，通过精心调整操作，争取了时间，从而避免了带液事故的进一步扩展，使空分设备运行

工况在几个小时之内恢复到正常状态，其主要操作方法如下：

1. 采取手动迅速开大液氧液面自动调节阀LCV-102，使冷凝蒸发器液氧迅速回流到上塔，降低冷凝蒸发器液氧液面高度，使其液面维持在  $1750 \mathrm{~mmH}_{2} \mathrm{O}$  的高度。  
2. 紧急关闭氧流量调节阀 HCV-103, 开启出塔产品氧V101放空按钮, 紧急排压, 将板翅式换热器氧通道内大量汽化的气体排出放空。  
3. 迅速全开污氮出塔 N3阀，纯氮出塔 N2 阀，以便上塔排气降压。  
4. 缩短板翅式换热器通道切换时间，将切换系统的切换时间定为10秒钟，调整环流温度和中部温度及温差，避免板翅式换热器通道冻结，保证自清除。

# 三、事故原因

这次KOBE-22型空分设备的板翅式换热器严重过冷，造成主冷中大量液氧进入氧通道，主要原因分析有以下几种：

1. 板翅式换热器氧通道进液，是由于冷凝蒸发器液氧液面超过定值（ $1800\mathrm{mmH_2O}$ ）的高度，大量液氧被带入出塔氧气管线，进入板翅式换热器三组氧通道中汽化。即使在出塔流量调节阀 HCV-103 阀关闭后，氧流量指示表针仍打到头，量无法估计，使氧气压缩机吸入压力升高而自动紧急停机。  
2. 由于大量液氧在板翅式换热器氧通道内汽化，造成三组环流温度及三组中部温度急剧下降，板翅式换热器严重过冷。大量液

氧汽化，使通道内气体压力迅速升高。汽化后液氧气体又倒流回主冷及上塔下部，引起上塔超压，安全阀开启，上塔精馏工况遭到破坏，各部阻力波动，产生液悬。

3. 设备经改造，采用氮气膨胀流程，否定了环流通道带液的疑虑。空气出下塔A14阀关闭，环流气及膨胀机进气均抽下塔顶部氮气，经新加装过冷器复热后，分成两股。其中一股做环流气出来再会合去膨胀机，故环流通道及膨胀机未出现带液现象。  
4. 冷凝蒸发器液氧液面超标，关键是在液氧液面自动调节阀LCV-102阀操作失误。事故发生前，一名操作员看见液氧液面高度显示在  $1750\mathrm{mmH_2O}$ ，而定值是  $1800\mathrm{mmH_2O}$ ，故将LCV-102阀由自动打入手动操作，关小了LCV-102阀，随后又去分析产品氧纯度，却忘记将LCV-102阀打入自动操作状态，致使冷凝蒸发器液氧液面升高充满，

发生板翅式换热器通道进液事故。

# 四、事故教训

1. 首先应该加强和提高岗位操作人员的技能和责任心，熟悉工艺流程，精心尽职操作。  
2. 要求岗位操作人员严格按照《安全技术操作规程》进行操作，对关键的操作岗位，关键的技术参数，严密监测。如液氧液面自动调节阀LCV=102阀，是空分生产工艺中非常重要的调节阀门，必须严格按规程操作。  
3. 事故发生后，由于对事故原因判断准确，处理措施果断合理，故没有对空分设备造成损失。但是，从板翅式换热器氧通道进液的严重程度来看，在国内空分行业也是少见的。

以上是对这次板翅式换热器通道进液事故的分析，仅供同行们参考。

（1996年12月改稿）

# 对现用充氧软管安全的一些看法

李鸿瑞

（驻上海市塑料研究所军代表，上海杨树浦路1664号，200090）

《深冷技术》1996年第4期47页刊登了曹卫平同志的“一起金属充氧软管燃烧事故”的报道，很值得引起从事充氧工作同仁们的重视。

高压充氧导管以前都是用紫铜管的，由于使用中金属导管较硬，弯曲很费劲，用过一些时间须退火发软，而且金属管（尤其直径较细的）工作时经多次弯曲极易断裂，甚至可能会引起瓶倒管甩的危险现象。当时灌充压缩空气、氮气及惰性气体等所用的导管是外层有钢丝编织加强层的橡胶软管，但橡胶软管绝对不能用于高压氧气的灌充。这是因为橡胶是极容易燃烧的材料，在空气中用火柴一划就能点着，在高压纯氧的条件下遇到

细微火花(有如静电跳火)，便会在瞬间燃起大火，甚至危及设备和人员的安全。在军队中规定高压氧气的灌充是禁止使用橡胶软管的，只能使用金属管。充氧人员早就希望也能用上方便安全的软管，但当时无法找到可供灌充高压纯氧安全使用的软管内衬材料，也知道用任何可燃材料来生产高压充氧软管是会产生极大危险的。故一时充氧软管材料选取成了大家盼望而急待解决的问题。

随着时代的前进，含氟工业在整个世界日益发展，氟塑料在多种领域中得到广泛的应用。我国氟塑料产品也从军用领域推广到民用各行业，聚四氟乙烯制成的软管便是其中的一例。美国在60年代把此软管用在21

MPa压力的输氧系统〔见MIL-H-26635B(USAF)]，我国后来也逐渐采用了。

现在，随着社会主义市场经济的发展，充氧软管似乎成了氧气行业中热门的推销产品，原因之一是行业确实需要；二是生产充氧软管确实有利可图。但且慢，我们得先来看看所用的充氧软管是否能保证安全使用。曹卫平同志报道的发生燃烧充氧软管内衬材料是尼龙的；上钢三厂1995年底发生的一起充氧软管燃烧事故，其内衬经有关科研单位化验，证实是聚丙烯。由此看来，用作充氧软管内衬的塑料已到需重视甚至整治的时候了，否则很可能会出更大的事故。由于充氧软管内衬现都用塑料，在这儿摘录若干种塑料的燃烧特性，供大家参考。

# 一起氧气瓶爆炸事故分析

张国发

（天津伯克气体工业有限公司，天津市河西区洞庭路16号，300220）

1996年9月4日23:30分，我公司一只氧气瓶爆炸，造成汇流排架和主管道弯曲，窗户玻璃破碎，停产10小时，幸无人员伤亡。

# 一、事故经过

1996年9月4日23:30，充装三班在  $6^{\#}$  库按规定正常充装，当压力达  $14.7\mathrm{MPa}$  时，操作工去充装台准备关瓶阀，约还有8米距离时，只听一声巨响，一只气瓶爆炸，操作工当即返回总控室报告班长撤压，同时通知氧压工段紧急停车，并通知值班干部，保护现场。

# 二、事故调查

事故发生后公司领导非常重视，立即组织事故调查组进行事故调查和原因分析。现场发现，爆炸气瓶被撕展成两瓣且卷曲，瓶底掉到一处，瓶嘴断在另一处，汇流排架和主干管道被冲击波打成弯曲，瓶嘴断处无燃烧痕迹。经检查，气瓶为北京高压气瓶厂1982年10月制造，气瓶检验钢印为1995年11月至1998年，气瓶产权单位为渤海石油公司。瓶身下部1/3处，内壁被腐蚀出许多凹凸处，直径约  $5\sim 20\mathrm{mm}$  ，最大  $70~\mathrm{mm}$  。对现场多处拍照，决定暂停此库，其它库继续生产。

气瓶事故调查组汇同BOC集团总部派出的气瓶专家，一起对渤海石油公司气瓶使用现场进行调查，为验证该瓶内部有海水腐蚀，便对爆炸瓶瓶底用蒸馏水浸泡12小时后抽水分析，发现含大量氯根，定量分析为  $1500\mathrm{mg / L}$  ，而普通自来水含氯根仅为  $46\mathrm{mg / L}$  对比之下，断定气瓶被海水腐蚀。除此外，

还对该气瓶金属材料进行了化学成分分析，对渤海石油公司的气瓶抽出30个进行内部检查，并对气瓶检验站及气瓶充装夹具进行调查，对爆炸气瓶进行壁厚检测。

# 三、原因分析

经过10多天的现场调查和事故分析，首先排除氧压机润滑水的腐蚀。该气瓶并非销装和混装易燃气体，还是在钢瓶技术检验期内发生爆炸。最后确定渤海石油公司的钢瓶在偶然的情况下，倒灌入海水，在高压氧的作用下对气瓶产生了强腐蚀，导致瓶壁减薄（最小壁厚仅  $2.2\mathrm{mm}$ ），使气瓶局部承受不住高压，当充装压力达  $14.7\mathrm{MPa}$  时便发生爆炸。

# 四、预防措施

1. 将渤海石油公司的自备瓶全部停用，进行内部检查，逐只做好颜色标记，有问题的进行全面检验并测厚。  
2. 对渤海石油公司气瓶使用情况进行调查，对使用条件恶劣容易接触海水作业的气瓶做好专门标记，专用充装台专库贮存，严格按照国家气瓶技术检验规定，海水作业钢瓶检验周期为2年，以确保安全。  
3. 接触海水的专用空瓶，要求留有大于  $0.2\mathrm{MPa}$  的剩余压力。充装前应对回收空瓶进行余压检查，凡没有余压，对用户罚款，促使用户加强在使用环节的管理。  
4. 对气瓶检验站进行评价，促其提高气瓶检验质量，为气瓶的安全使用提供保证。

（1996年12月24日）

# 一、国标GB/T3863—1995《工业用氧》

国标GB/T3863—1995《工业用氧》（代替GB3863—83)，由化工部提出，西南化工研究院技术归口，北京普莱克斯实用气体有限公司、西南化工研究院负责起草，主要起草人周文慧、戴培述。国家技术监督局1995年12月20日批准并发布，1996年8月1日起实施。

# 安全要求

1. 氧气是无色、无味、无嗅、无毒、不燃的气体，是强氧化剂，能助燃。其化学性质极活泼，能同许多元素发生化学反应同时放出热量。氧气与可燃性气体按一定比例混合后容易爆炸。油脂与压缩氧气接触，温度超过燃点时可发生燃烧。  
2. 氧气在室内聚集，其体积分数超过 $23\%$ 时，有发生火灾的危险。在氧浓度有可能增加的地方，应设有通风装置，并对氧气浓度进行监测。  
3. 检修和工艺处理盛氧容器之前，必须将氧气进行置换，当氧气含量降到  $23\%$  以下，方可开始工作。  
4. 应随时保持气瓶内外清洁，不得沾染油脂和其他污物。  
5. 操作者在刚刚离开富氧环境后, 不允许立即使用明火或走近火源。  
6. 气瓶充装压力按 GB 14194 执行，充装时间不得小于  $45 \mathrm{~min}$  (40L 气瓶)。  
7. 液态氧的操作者，应有足够的人身保护措施以防冻伤。  
8. 充装氧气前应对气瓶严格检查，充装氧气的容器禁止充装其他气体。

# 制氧工问题解答（两则）

1. 问：我厂有一台  $150 \mathrm{~m}^{3} / \mathrm{h}$  制氯机，是役龄20年的老设备，最近发生氯气只有  $90 \mathrm{~m}^{3} / \mathrm{h}$  左右，产量始终开不大，总的工况是这样的：压力  $\mathrm{P}_{1} / \mathrm{P}_{2} / \mathrm{P}_{3}$  为  $2.0 / 0.48 \sim 0.50 / 0.045 \sim 0.055 \mathrm{MPa}$ ，膨胀机前压力为  $1.8 \sim 2.0 \mathrm{MPa}$ ；温度  $\mathrm{T}_{1} / \mathrm{T}_{2} / \mathrm{T}_{3}$  为  $-100 / -138 / -157^{\circ} \mathrm{C}$ ；液氯液位稳定在  $10 \mathrm{~cm} \mathrm{CCl}_{4}$  柱，偏低；阀位，节-1阀3转、节-3阀  $3 \sim 5$  转；温度，进塔空气  $32 \sim 37^{\circ} \mathrm{C}$ 、氯氮出塔  $24 \sim 25^{\circ} \mathrm{C}$ 。请问是否因空压机的毛病或故障引起的？

（浙江定海某制氧站问）

答：从所述纪录来看，该塔的病情是复杂的，总的原因是塔的冷量不足，且伴有过冷现象。请看进塔空气温度高达  $37^{\circ}C$  ，热端温差大于  $8\%$  ，高压压力2.0MPa，液氧液位只有  $10\mathrm{cmCCl_4}$  柱，显然是由产冷量不足，即冷量不平衡引起的。这时按理应当关小节-1阀，使高压压力升高，增加产冷，提高液面。但关小了节-1阀，  $\mathbf{T}_{3}$  温度还要往下降，换热器底部将出现更多液体，所以不能往下关。即使关了，高压压力也高不起来，这就是产冷量无法用一般操作得到平衡的表现。由于冷量不足，下塔热了，节-3阀只好开大，因而液面下跌到  $10\mathrm{cmCCl_4}$  柱，氧产量逐步关小到  $90\mathrm{m^3 / h}$  ，否则纯度便破坏了。

为什么正规的操作方法用不上去呢？根据中压流程的特点及该塔产冷潜力大的客观条件，不应该产生无法操作的现状，原因在哪里呢？就是因为塔被开“僵”了，换热器出现过冷而高压压力才提不高，可是下塔又过热。一方面过冷，另一方面又过热，是一对矛盾，通俗地说设备的工况被打乱了，冷热这对矛盾始终得不到解决。换热器过冷，往

往是在启动后期，收阀的时候产生的。操作人员为把工况拉牢，开大节-1阀保持  $\mathbf{T}_{3}$  温度，殊不知促使高压压力下降，导致产冷不足。下塔压力升高，又不得不把节-3阀打开，从而液位下降，产气量减少，便出现恶性循环，弄到不可收拾的地步。一般操作规程的功能，就是要平衡和调节各种互相矛盾的工况，而在这时得不到解决和应用。

遇到这种情况，问题应当如何解决呢？笔者认为需要下定决心，将所有阀门开大，按开车阶段的操作法，重新积液、收阀和调纯。只有重新来过，过冷现象才能消除。而后把液面调到略高于规定指标，高压压力随液面涨落而升降，并在冷量平衡的基础上（即液面稳定下）调整工况，促进塔的良性循环。这样，该塔是可以开得成功的。

总之，该塔产量上不去，绝不是空压机气量少所引起，恰恰是操作不当造成的。

[杭州制氧机厂（退休） 陈逸樵]

(310002，杭州惠民路十三湾巷20号401室）

# 2. 问：配  $150 \mathrm{~m}^{3} / \mathrm{h}$  空分设备的透平膨胀机停车后如何复车？

答：空分设备停车时，空分塔板的液体流到冷凝蒸发器，使液氧液面升高，液氧面满出氧气出口管就流到板翅式换热器内，使换热器的温度降低。启动时，高压空气温度下降  $(\mathrm{T}_{3} = -180\sim -190^{\circ}\mathrm{C})$  ，膨胀机机前温度也降至  $-160^{\circ}\mathrm{C}$  ，若马上启动膨胀机，则进到机内的不仅是气体，而带有液体，这就会造成膨胀机的损坏。

因此，在临时停车后恢复开车时，如果 $\mathbf{T}_{\text{总}}$ 、 $\mathbf{T}_{\text{总}}$  温度过低，可先打开节-1阀，使高压

空气进板翅式换热器，使热交换器内的液氧汽化，温度升高，并促使膨胀机机前温度回升。当  $\mathrm{T}_{1}$  温度  $\geq -150^{\circ}C$  时，再启动膨胀机。打开轴承气、密封气，并调节压力在0.5 $\sim 0.6\mathrm{MPa}$  ，然后全开膨胀机排气阀，再缓慢打开进气阀，使压力逐渐上升。控制膨胀

机转速在  $10\mathrm{r / min}$  左右，再调节好密封压力，使其高于排气压力  $0.05\mathrm{MPa}$  左右。膨胀机运转正常后，再进行空分塔的调整至设备稳定生产。

（天津第二石油化工厂化三车间张一民）

(300380, 天津市西青区)

# 氧吧好景不长 全国仅剩七家

据《扬子晚报》登载：在山东省青岛市投入几十万元建成的“梦萌氧吧”，经营不到三年，不久前关门歇业，这是青岛市最后一家停业的氧吧。

氧吧关门已成为各地的普遍现象。据一项调查表明，1995年北京、上海、天津、重庆、广州五市有氧吧300多家，到1996年底仅剩七家(不计医院内兼营的氧吧)。

氧吧源于西欧一些工业发达国家，1992年初开始进入我国深圳、广州等南方城市。在很短时间内，氧吧便成批在全国各地大城市“粉墨登场”。然而，当人们认识到氧吧不过就是吸氧间，吸过一段时间并无什么作用时，对氧吧的神秘感便消失了。

1. 由机械部和省经贸委批准“九五”投改投资额达2900万元的国家专项的可行性研究和初步设计，项目开工前的各项准备工作已基本完成。  
2. 新建的大型真空钎焊炉已成功钎焊6米等级的大截面板翅式换热器，部分产品已用于出口设备。  
3. PSAO-20型变压吸附制氧设备和KDN-50/40Y型空分设备，通过省机械厅评审，升级为一等品。  
4. 截止到10月份我厂开发的  $6000 \mathrm{~m}^{3} / \mathrm{h}$  及其以下所有规格的分子筛预净化流程空分设备，已全部投入运行。  
5. 我厂为印度尼西亚某公司提供的第四套KDON-1500/1500-9型空分设备已交户使用。  
6. 我厂投资1000多万元建成的科技大楼，已投入使用，极大的改善了科技人员的工作条件。

7. 由中国石化总公司组织的专家验收组，对我厂首台国产高压绕管式换热器通过验收。这台绕管式换热器的制造成功，结束了我国石化行业高压绕管式换热器全部依靠进口的历史。  
8. 我厂与日本酸素公司合作制造的、目前国内最大的  $1200 \mathrm{~m}^{3} / \mathrm{h}$  变压吸附制氧设备，在沈阳顺利开车。  
9. 我厂被开封市评为尊重知识、尊重人才先进单位，厂长、党委书记唐树勋被评为先进个人。  
10. 在全国召开的 CAD 应用工程工作会议上，我厂被国家科委、全国电子信息系统推广办公室及国家技术监督局，评为全国 CAD 应用工程先进单位，厂长唐树勋在会上作了典型发言。
