# 当前我国空分设备行业现状和发展前瞻

沈雄楞 陈允恺 朱燧炎

（机械部气体分离设备行业协会，杭州艮山门外东新路152号，310004）

摘要 列述了空分行业各厂空分设备、纯氮设备、变压吸附分离设备、低温液体贮运设备制造和出口外销的情况，以及外商在我国合资建立设备制造厂和工业气体厂的情况；至1994年底，整个行业累计生产各种气体分离(包括变压吸附)设备共7405套。其中，大中型空分设备463套。并指出了空分行业发展前景与存在的问题。

主题词：空气分离设备 行业现状 分析 发展 问题

# 一、行业内设备制造情况

杭氧自1978年引进林德公司技术开始生产大型空分设备后，使我国的生产技术和产品质量有了一个全面的提高，取得了用户的信任。因此，为立足国内，国家曾下达“10000 $\mathrm{m}^3/\mathrm{h}$  及以下空分设备为国家控制进口的机电产品”和“ $15000\mathrm{m}^3/\mathrm{h}$  及以下空分设备为国家统一归口，联合对外引进的设备”等限制性文件，使我国空分制造行业摆脱了困境，这是一个十分可喜的局面。

当时靠行政限止，进行了种种行业内生产产品的分工，如杭州制氧机厂生产6000 $\mathrm{m^3 / h}$  以上空分设备、开封空分设备厂生产 $1500\sim 6000\mathrm{m^3 / h}$  空分设备、四川空分设备厂生产  $1000\sim 3350\mathrm{m^3 / }$  时空分设备、哈尔滨制氧机厂生产  $1000\mathrm{m^3 / h}$  以下空分设备，其余各行业厂均生产“50”、“150”、“300”及其它小型空分设备。当时行业内借此人为的分工，来加强合作，减少重复劳动及制造设备的重复装备，表面上看是无可非议的。当国家进入了市场经济、宏观控制的年代，随着各大中型钢铁、化肥、石化企业的兴起，原有分工已远远适应不了社会需要，因此出现

了下列几种情况。

# （一）  $6000\mathrm{m}^3 /\mathrm{h}$  以上空分设备

# 1. 杭州制氧机厂

1988年9月生产出第五代带增压透平膨胀机、常温分子筛净化的6000/13000型空分设备（1989年10月12日调试出氧），自此我国大中型空分设备已普遍转入了第五代的产品生产。目前杭氧厂成套设备，最大已生产了 $15000\mathrm{m}^3 /\mathrm{h}$  空分设备，到1994年底已成套生产  $14000\mathrm{m}^3 /\mathrm{h}$  等级空分设备6套。

# 2. 开封空分设备厂

1988年开始生产  $6000\mathrm{m}^3 /\mathrm{h}$  以上空分设备，至今已生产出厂调试合格的6000/6000型空分设备4套，正在生产6000/6000型空分设备1套，在设计的  $15000\mathrm{m}^3 /\mathrm{h}$  空分设备1套。

# 3. 四川空分设备厂

# （二）小型空分及纯氮设备

1. 小型空分设备近年来发展特点，是一改过去一成不变“50”、“150”等典型产品的生产。目前已根据用户不同要求开始生产针对性设计、品种繁多的变型产品，这是一个十分可喜的局面。

2. 现在已将增压型透平膨胀机技术引用到  $300\mathrm{m}^3 /\mathrm{h}$  等级的空分设备，使小型空分设备出现了新面貌。

3. 小型空分设备已向全铝结构或多孔管高效冷凝器迈进，使主冷温差减少到  $1^{\circ} \mathrm{C}$  左右，大大降低进塔空气压力，降低能耗。

4. 有的制造厂在致力开发集装箱型小空分设备，它的开发将对边远地区及流动性工区（油田、矿区、电站），发挥灵活机动的作用。

5. 品种领域不断扩大，新型的全低压型设备已有产品。例如杭氧已生产出KDON-122Y/160Y新型液体产品，能生产122升/时液氧或160升/时液氮。

# 6. 纯氮设备的生产

由于纯氮应用领域不断扩大，纯氮设备的生产也逐步发展起来了，在1983年以前我国累计生产10台纯氮设备，到1994年我国已累计生产出208套纯氮设备。预计今后在这领域内还能得到飞速发展，当然一部分用低温精馏，但今后估计用变压吸附制氮将会得到更快的发展。

# （三）变压吸附分离设备

变压吸附分离技术，我行业率先在60年代中期就开始了制富氧技术的研究和开发，于1970年研制成功第一套小型  $\mathrm{PSA - O_2}$  设备。70年代末，  $8\mathrm{m}^3 /\mathrm{h}$  设备已批量生产。80年代，PSA设备进入低谷期，并转入试验攻关阶段。90年代初，PSA技术是发展期，成功地开发了50,150m/hPSA制氧设备，

目前已能生产  $1100\mathrm{m}^3 /\mathrm{h}$  规模的设备。80年代末引进了林德公司PSA制氮技术，从此开始生产PSA制氮设备。

目前，我行业生产过PSA设备的厂家有杭州制氧机集团公司、四川空分设备厂、开封空分设备厂、苏州制氧机厂、余杭通用机械设备公司、自贡机械一厂。至1994年底止，累计生产56套设备。其中  $\mathrm{PSA - O_2}$  设备44套，  $\mathrm{PSA = N_2}$  设备12套。

目前出口业务，特别是小型空分设备竞争激烈，互相压价，产品质量、售后服务、备件供应均出现了难以胜任的危机，应引起注意。

（1）设备流程与性能还停留在老“50”、“150”水平，比较落后。  
（2）制造质量无论是外观或内在均较粗糙，甚至于关键部位的精度与互换性有较严重的超差，个别部机材料不合格，压力容器手续不完备等等。  
（3）备件供应是外商感到最头痛的问题之一，出口设备原始履历不完整，造成备件出口的混乱。再加上这些客商操作水平低，要求能频繁提供可靠的备件，一般品种多、批量少，国内大的出口公司不愿意承担此项业务。外商不得不转向关系户直接搞备件，周转环节多，外商叫苦不迭。因此，一些外商反映中国的小空分成套价格是便宜的，但日后的备件供应困难重重。

# 三、发展前景与问题

据统计，到1994年底，整个行业共累计生产各种气体分离设备(包括PSA设备)7405套，其中大中型空分设备463套，分布在全国的氧气站（厂），估计有三、四千家。

自改革开放以来，引进国外技术，我国空分设备行业整体水平有了很大提高，大中型空分设备已参与国际竞争，已销往东南亚。

从1995年  $1\sim 4$  月统计数据来看，预测行业生产经营发展形势时还是持乐观态度。

1995年是“八五”计划的最后一年，一批新建项目已进入后期建设。为发挥效益，后期资金的投入能得到保证，同时，部分“九五”项目也将进入前期准备阶段，必然增加今年投资需求，主要是倾向能源、原材料、交通和农业方面。我国总体环境是良好的，将会带动空分行业的发展。

据专家预测，90年代中后期通用机械产品市场走势平平，增幅不大。以国内市场需求看，国家投资环境仍紧，不明显加大，则需求量不会明显增长。国家为加强农业基础地位，化肥工业看好。据化工部信息，大型化肥成套装置目前多是利用外资进行建设，国内仅承包部分设备。而重点放在大中型化肥设备改造上，改为水煤浆加压气化技术，化工部列项，推广该技术的有浙江衢化公司、黑龙江浩良河化肥厂、甘肃刘家峡化肥厂等4家，以及石化企业，如南京扬子石化公司、四川维尼龙厂等改造项目和被国家列入的煤化工发展项目。这些项目需求1.5～2万等级大型空分设备，折合制氧容量25万  $\mathrm{m}^3 /\mathrm{h}$  。在进口设备价格居高不下的同时，为降低建设投资，已明确上马国产化设备，带动空分设

备制造业的发展。

在这几年我行业高速增长中，正如春天中会有寒流一样，强劲势头中也存在着不容乐观的隐忧，值得大家思考。一个隐忧是国有企业改革滞后问题，引发出消极后果，如国有企业财务状况日益不良等；另一个隐忧是内部市场疲软，企业产品销售出现不景气，而国有企业投入少，将会失去动力和基础。

当前，我行业发展面临不少困难，在运行中还存在着一些制约发展的不利因素：

（1）企业缺乏活力，一个重要原因是企业过度负债，资产负债率高，现行体制难以遏制其发展。行业资金缓慢问题尤为突出，行业流动资产周转率为0.822次（低于全国标准值1.83次，也低于通用机械行业1.09次），这迫使企业贷款。现代企业建立资本金制度，将产品经营转到资本经营。企业围绕以财务管理和资金管理为中心来加强企业管理，可带动资金状况的转变。  
（2）企业“后劲不足”日趋明显。不尽快解决老企业的发展问题，国有资产保值也保不住。老企业要尽快更新设备，用先进制造技术，提高劳动生产率，在市场竞争中就不会被淘汰。如果企业装备落后，就竞争不过人家。  
（3）科研和开发少。我国每一代空分设备发展都是借鉴国外技术的，国外有什么，就跟着开发应用，自己无法提出设想和创新。我行业开发新产品，还非易事。如我国急待发展“三万”等级空分设备，只有借助合作生产。不重视科研和开发问题，应值得全行业注意。为了提高企业竞争能力，最好措施之一是增加科研和开发的经费投入，加快发展，带动经济持续增长，促进行业的振兴。  
（4）不断加强企业管理。历史告诉我们，对企业经营优劣，长期起作用的是企业总体素质和管理水平，特别是企业的优良传统。企业要在竞争中取得优势，提高效益，加强

企业管理尤为重要。企业能适应市场环境，要不断地调整产品结构。

（5）合资企业对行业影响。国内合资者有行业内企业，也有非行业的厂家。1994年底已形成南北争雄，或许不久将形成东南西北四方争霸局面。同时，国外几家大气体公司竟向在我国寻找合资伙伴。他们在大城市设立办事处，搞集中供气厂，远距离输送，想占领我国气体市场。我国还没有一个机构加以统筹考虑协调和统一规划，也无法宏观控制。如凭其自流，对我国空分设备制造业及气体工作带来的后果，是喜是忧还应作冷静分析。  
合资工业气体厂的建立给了我们一个启示，过去我们也曾设想建立集中供气厂（站），曾多次向国务院打报告，要求建立集中供氧站，也曾得到各级领导重视，但具体实施过程中步履艰难，虽在北京附近廊坊建设一个集中供气厂，但也因诸多原因中途夭折。  
现在合资的工业气体厂大都建在北京、天津、上海、南京、大连、广州、重庆等大城市。并还向附近城市输送液氧，就地气化供气，致使供气地区的小型氧气厂出现困境（集中供气成本要便宜  $50\%$ ）。而我国城乡上有千家小氧气厂，这些厂将逐步会被集中供气站取代，已有不少小型制氧厂，自己将小型制氧机停掉，而转向进行低温液体汽化输供；同时对空分设备制造业来说，在生产小型空分设备时，对其品种、数量等，也要有一个新的认识。  
6. 国内空分设备制造厂自己建造工业气体厂销售气体的问题。国外大的空分设备公司都是既生产设备又开办工业气体厂，而气体厂的收益远比生产销售空分设备大。我国的设备制造厂能否如此，主要难点是如何解决投入的资金问题，但优点非常明显：  
（1）建集中供气站可取代无数小型空分站，可以减少小型空分设备的生产，使我国

有限的资金用到更需要的地方去。

（2）集中供气设备大、能耗低。而节约能源是我国的基本国策。  
（3）设备制造厂自己建设中大型工业气体厂，在集中供气的同时，也可以兼作基地

进行科研、开发、试验新技术。

（4）经济效益好、收效快，使制造厂能从中获得工厂技术改造资金、科研资金及提高职工生活福利等。

这又充分说明，电子工业的需要促进高纯气体的制造技术的发展，而高纯气制造技术的改进又促进电子工业技术提高。

# 二、高纯氧制取工艺流程

工艺流程以低温精馏法制取高纯氧最为普遍，一般根据产品要求组织。现介绍几种如下：

# 1. 以液氧作原料

其原理图示于图1。

一般说来，高纯氧的制取量均较小，因此单独高纯氧设备至今尚未见到报道，而常见的是在大型空分设备上附加一个高纯氧塔，从主塔主冷中引入液氧，其氧纯度为  $99.5\% \sim 99.7\%$  ，其它为氮、氩、氪、氙及碳氢化合物，在高纯氧塔中作回流液。塔底的冷凝蒸发器，以主塔冷凝器顶部引入的中压氮作冷凝液，从主冷低压侧可制取高纯气氧或液氧，其纯度达  $99.995\%$  以上。该方法主要适用于制取少量高纯氧，它与采用主塔制取高纯氧相比，可降低下塔的操作压力，从而节约了能耗。

# 一、前言

目前工业上提供氧、氮、氩等气体的主要方式有三种：管道供气、液体供气和瓶装气体。液体供气是一种新型的供气方式，在我国起步较迟。在一定的条件下，采用低温液体充灌系统与投资一套小空分装置比较，具有投资费用小、周期短、能耗低、流程简单、操作方便等优点。80年代后期，随着我国大型气体液化装置的建立，液体供气的方式也开始了大规模的推广，区别于小空分装置的液体充灌站也纷纷建立。我们上海比欧西气体工业有限公司，最近就在上海附近建立了几个液氧充灌站。

低温液体充灌系统的核心设备是低温液体贮槽和液体泵。液体泵是用来抽送低温液体，并提高其压力的机器。为保证液体泵的正常运转，除了泵结构的可靠性外，泵的配管技术是非常关键的，一种不适当的配管，轻者会造成液体泵有夹气现象，原料消耗量大，泵的机械损耗较大；重者会造成管道汽化严重，充灌压力不能提高，液体泵无法正常工作，充灌站不能正常运行。所以，正确敷设泵的吸入管道、回气管道及排出管道，具有特别重要的意义。本文就此问题在理论上，并结合自己的实践进行一些探讨。

# 二、液体充灌系统的流程介绍

该系统的主要流程是：将液氧（或液氮·液氩）由液体槽车输入液体贮槽中，经液体泵压缩后，再经高压汽化器蒸发成气体，入汇流排充钢瓶。下面我们就液氧充灌的流程加以介绍（见图1）。

该流程分成三大系统，主要特点有：

# 1. 液氧贮槽系统

液氧贮槽一般是外购设备，贮槽的选用应根据用户的实际用量（或发展后的用量），及液体泵吸入工作压力来确定。选用汽化率低，并具有独立的可供连接液体管及回气管管口的贮槽，可便于安装管道，减少液氧的汽化率，对液体泵的进液管、回气管排温也较简便，并有利于正确地配管。

# 2. 液氧泵系统

液氧泵系统是由液氧泵、进口管、回气管、出口管路以及汽化器等部件组成。该系统合理、正确地配管，是液体泵正常运转的重要保证。液氧泵与液氮泵、液氩泵或其他惰性气体液泵有区别的是，液氧泵在使用中更应注意安全问题。为此我们在流程中采用了如下控制措施：

（1）在泵与贮槽相连的进液管和回气管路上，分别装有紧急切断阀，并与泵联锁，以防止在发生意外的事故时，可远距离地及时切断液体和气源，紧急停止液体泵的运转。

（3）高压汽化器后氧气总管上设有温度指示及低温报警装置，保证进入汇流排气体温度高于  $-10^{\circ}\mathrm{C}$  。防止液氧进入钢瓶，发生意外事故。

（1）采用新型的带防错装接头的金属软管进行充灌。

（3）汇流排的充瓶数量由泵的充灌量、充灌速度来决定。

# 三、液体泵的配管技术

液体泵的配管，要保证液体泵的正常运行，在设计配管中特别要注意的是泵的进口管，因为液体泵的吸入管易产生“汽蚀现象”。

“汽蚀现象”的产生致使泵内液体的连续性遭到破坏，泵的流量、压力及效率都降低，是影响泵正常运转的主要因素。泵的抗汽蚀能力，除了泵体结构设计中考虑外，就应考虑把吸入管道的压力降减少至最低限度。槽的选择与安装，出口管和回气管的配置，也是液泵配管技术的组成部分。

# 1. 贮槽的安装及槽内工作压力的确定

要成功地设计一个高效的低温液体泵系统，必须保证维持在泵的进口所需的净正吸入压头(NPSH)，此值应高于流动介质的蒸汽压力。NPSH是泵进口处的液体静压，与泵进口处液体开始沸腾(饱和)的压力之差，NPSH的值是随流体介质的不同、沸腾温度的不同而变化的。对压缩指定的液体，NPSH所需的值一般在液泵制造商的操作手册中有陈述，在配管中必须使实际的NPSH大于或等于所需的NPSH，否则泵的最高出率会下降。为满足所需的NPSH，贮槽中液体压力必须高于饱和压力  $(P_{SL})$  ，以克服由摩擦

引起的压力损失和液体在管道中流动至泵时的冷量损失。这些可以从图2液体泵的吸入状况中清楚地看出。

# 2. 泵的进口管

从定量上来讲，泵的进口管的最小吸入压头  $H_{\min}$  的值，是反应了各种泵进口管的配置设计效率。

对于单泵系统

$$
\begin{array}{l} H _ {\min } = N P S H + H _ {f} + H _ {q} \\ = N P S H + \left(h _ {q} \times L\right) + \left[ h _ {f} \right. \\ \times \left(L + L _ {E}\right) ] \\ \end{array}
$$

式中  $H_{\min}$  ——维持泵最高出率的最小压力换算成液体高度  $(\mathrm{m}_{\text{液}})$

NPSH——净正吸入压头  $(\mathrm{m}_{\text{液}})$

$h_{\mathbf{q}}$  ——单位长度管道的热泄漏引起的压力下降  $(\mathrm{m}$  液/m）

$$
\boldsymbol {h} _ {q} = \boldsymbol {P} _ {q} \times \boldsymbol {F}
$$

$h_{J}$  ——单位长度管道的摩擦力引起的压力下降  $(\mathrm{m} / \mathrm{m})$

$$
\boldsymbol {h} _ {f} = \boldsymbol {P} _ {f} \times \boldsymbol {F}
$$

$F$  ——将压力降换算为液体高度的系数，具体是：氧0.0893，氮0.1262，氩0.0733。

$L$  ——进口管总长(m)

$L_{E}$  ——各管配件的等效长度（m）

$P_{q}$  和  $P_{f}$  的值可分别从有关手册图表中查得。

从公式中可以看出，泵的进口最小吸入压头应克服如下几种压力降：

第一，液体的流动损失，即由于管道、阀门及弯管、过滤器等管件引起的压降，称为摩擦压降  $\mathbf{H}_f$ 。

第二，因吸入管道的冷损引起的热泄漏 $H_{q}$ 。

第三，泵正常工作时所需的最小正净吸入压头  $NPSH$ 。

# 3. 泵的回气管

液体泵在工作时由于摩擦发热，总有一部分液体被汽化，因此要完全消除“汽蚀现象”实际上是不可能的。关键在于一旦产生汽蚀，应立即脱气，在泵的吸入腔内应配置一根回气管道。

对于贮槽液体泵的充灌系统，回气管路应尽可能短，并接至贮槽顶部的气相管内。该连接形式应呈连续上升形式，不能有下弯管，这一点很重要。因为任何低点都将导致液体滞留，堵塞返流气体进入贮槽气体空间，液体泵会最终由于气体的阻塞而导致压力不上。若发生气蚀现象，应及时打开回气管上的吹除阀，消除气阻现象。回气管路也应有良好的绝热层，以防止热量进入贮槽。

# 4. 泵的出口管路

液体泵的排出管在低温、高压下工作，因此，管道要求考虑具有挠性及自动补偿应力。泵的排出管还要求尽可能地无漏泄，排出管上应装有止逆阀、截止阀、安全阀及带报警自控装置的压力表，排出管道一般不需要设置绝热层。

若液体泵后直接充瓶或送入使用点的，必须经过具有足够传热面积的汽化器，汽化器的量应考虑液体的输出量、当地的温度和湿度，并考虑在冬天严寒的气候环境下，也能保证气体进入钢瓶的温度不低于  $-10^{\circ}\mathrm{C}$ 。汽化器的选择还应考虑初次投资、运行费用、操作方便及容积大小等。

# 一、加氢除氯法流程[3]

反应机理：

GETTER 除氧法与传统的催化加氢除氧法有所不同，第一步是以铜为媒介生成氧化铜，除去粗氩中的氧；第二步在再生氮气中混合一定量的氢气，然后通入除氧器，氢气把氧化铜还原成铜和氧；第三步氢与氧反应生成水为最终反应。而氮气杂质的清除仍然用低温精馏法工艺，大量的氮气可以吸收再生中反应的热量，使其再生的温升不大。

1908年首次在一物理实验室获得了氦液化，并且以后在与工业部门的合作中进一步得到了发展。50年后工业上才开始使用氦液化器。1960年起在美国Kansas，把从贮运的天然气中分离出来的氦进行了液化，使几年内所使用的设备的液化功率上升了一个数量级。可是在大型氦液化器范围内，应用物理学仍占统治地位。以超导磁铁和高频率一气蚀为基础制造的大型粒子加速器用液氦进行冷却。这种设备的产冷量，在最大商业液化器产冷量的范围内，甚至超过它的产冷量。

# 一、概述

我厂新扩建的30t直流电弧炉，配套丁由哈尔滨制氧机厂制造的KDON-350/700型制氧机（下面简称350型制氧机），这套目前国内较先进的小型全低压空分设备，具有电耗低、构造简单、操作集中、氧氮纯度高等特点。1993年底安装完毕，1994年2月开始调试，经过3次启动调试，各项经济技术指标均达到设计要求。其中，氧气纯度为 $99.73\%$  ，氮气纯度为含氧量  $8\mathrm{ppm}$  ，氧产量为  $352~\mathrm{m^3 / h}$  ，氮产量为  $743~\mathrm{m^3 / h}$  ，单位氧电耗  $0.79\mathrm{kWh} / \mathrm{m}^{3}\mathrm{O}_{2}$  。

该套空分设备的调试并非一帆风顺，前两次因设备泄漏严重和预冷系统的冷冻机出故障而未成功。但通过前两次的启动调试，积累了一些经验。如：（1）在装置降温过程中如何解决冷量分配；（2）在调纯阶段应首先建立下塔工况（即调-1、-2、-3阀的配合）；(3)透平膨胀机最佳转速、进气量及膨胀后的压力相互关系；（4）膨胀后入上塔的进气量，即拉赫曼气阀与通-5阀配合；（5）控制

好各点  $\mathbf{T}_{4}$  、  $\mathbf{T}_{6}$  、  $\mathbf{T}_{8}$  的温度，防止偏流，尤其是  $\mathbf{T}_{4}$  的温度；(6)上塔工况的建立和纯度的调整；(7)冷凝蒸发器的热负荷及温差的控制；(8)纯化器的切换、加温及预冷系统工况等。在没有“哈氧”调试人员的情况下，我们于1994年10月11日至18日进行了第三次全面调试，成功地掌握了350型制氧机的操作。目前，该套空分设备运行正常，操作稳定。

# 五、空分系统的调试过程

# 1. 纯化器预冷系统的调试

在整个系统运行之前，首先对纯化器的预冷系统进行了调整，选用R-12代替R-22作为6L22-WL冷冻机组的制冷剂。将蒸发器蒸发温度调整在  $0\%$  以上，这样，有效地防止了蒸发器内的盘管，因结冰或结霜造成水分过多带入纯化器。

通过调整热力膨胀阀或冷凝器进水量（冬季可减少工作气缸）等措施，调整冷冻机制冷能力与蒸发器热负荷相匹配，使冷冻机长期处于空调运行状态。控制经蒸发器冷却后的空气温度在  $7\sim 9\%$  之间，以恒定状态

进入纯化器，保证入塔空气温度在工艺所要求的范围内。

# 2. 空分塔的调试

1994年10月11日9:00，350型制氧系统正式启动。在启动初期的重点是发挥两台透平膨胀机最大制冷能力，让较多的空气经过膨胀机制冷，缩短启动冷却阶段的时间。

# （1）冷却积液阶段的操作要点

此阶段的目的是使空分装置尽快冷却，将两台透平膨胀机的转速提至6万r/min左右，控制膨胀前后的压差，膨胀机后的压力为0.038MPa左右。其冷却方式采取全面和重点冷却相结合，将膨胀后空气分两路：大部分空气通过通-5阀集中冷却主换热器，使入塔空气迅速降温；另一部分空气通过调-6阀送入上塔，分别冷却上塔、液氮与液空过冷器。冷却上塔主要是冷却膨胀空气入口处以下的54块塔板，以使这些塔板在启动一

开始就得到充分冷却，缩短液氧积累时间。

此时空分塔主要阀位：调-1、-2、-3阀全开，通-7阀按80%流量的开度，通-8阀按正常流量的开度。通过通-6阀调整上、下塔压力，其压力分别为  $0.034\mathrm{MPa}$  、0.53 MPa。随着装置温度的逐步下降，空气密度增大，上下塔压力甚微小地变化。在保证上下塔压力不变的情况下，每隔10分钟微收调-1、调-2、调-3、通-6阀，并且同步进行。主换热器的热端温差，可调整通-7、通-8阀及污氮阀前的压力和流量来获得。在调整过程中，应让主换热器发挥最大的换热效率。观察膨胀后温度计，每  $7\sim 8$  秒钟降0.1℃为正常。也可根据空气T-S图检查膨胀机的效率。经七、八小时后，主换热器冷端接近饱和温度。当空气出主换热器冷端温度达  $-171.2^{\circ}C$  时，查T-S图对应的饱和压力0.51 MPa，从图中可看出此时已有  $30\% \sim 40\%$  的空气变成饱和液体流入下塔，下塔开始积累液体。

这个阶段的操作重点，是如何减少液体的汽化率，此时下塔压力应提高一点。液空转入上塔后，为了减少上塔的汽化率，将上塔压力也相应地提高，以缩短上塔积液时间。在这阶段，调  $\equiv 1$  、-2、=3阀阀位各收至 $50\%$  左右，通-6阀逐渐关闭，调-6阀全关，调整通-5阀使膨胀后压力在  $0.038\mathrm{MPa}$  左右。

由于启动一开始，上塔54块以下塔板就得到冷却，从而缩小了液体与上塔塔板的温差，减少了液空在每块塔板上的汽化率，这时上塔压力应在  $0.06\mathrm{MPa}$  左右。由于采取了以上措施，在液空出现后  $1\sim 2$  小时后，液氧也出现了。为了缩短液氧积累时间，上塔压力不变，略降低下塔压力，调整冷凝蒸发器氮侧与氧侧温度反向或减小温差。在上塔积液过程中，要保持两台透平膨胀机正常工作，转速控制在规定的范围内，经八九小时的积累，液氧而涨至  $1\mathrm{m}$  左右，此时可停一

台膨胀机，并转入下一阶段——调纯阶段。

# （2）下塔纯度的调整及操作要点

对下塔的调整是至关重要的。下塔工况决定着整个装置的精馏工况，对下塔的调整主要通过调-1、调-2阀进行。在调整过程中，应首先对液空、液氮、污液氮的纯度进行分析，根据纯度进行调整，在调整纯度的过程中下塔阻力也相应建立。通过调-1、调-2阀找出液氮、污液氮、液空三者最佳纯度配合区域，即液氮纯度为  $99.999\% \mathrm{N}_2$  以上，污液氮纯度为  $93\% \sim 94\% \mathrm{N}_2$  之间，液空纯度达  $37\% \sim 38\% \mathrm{O}_2$  （注意液空纯度不宜太高，因较低纯度的液空进上塔后有较大的蒸发量，可减少上升的蒸汽量，加大产品氧的提取）。

这时下塔回流工况可分两部分：污液氮出口处以上回流比较大，在下塔顶部得到较高纯度的液氮；污液氮出口处以下回流比较小，在下塔底部得到适当纯度的液空。通过调整调-3阀保持下塔液空液面在一定的高度 $(250\sim 300\mathrm{mmH_2O}$  柱)，同时控制上塔提馏段的回流比。三种液体纯度都达到了要求，其对应的调-1、调-2阀的阀位便为最佳阀位。调整时动作要非常缓慢，尽可能减小阻力值的变化（下塔阻力值应在  $11\sim 12\mathrm{kPa}$  之间）。

# （3）上塔的调整及操作要点

实际上在对下塔工况调整的同时，上塔精馏工况也随之建立。在调整上塔工况时必须具备三个条件：①冷凝蒸发器换热效果要充分；②膨胀后送上塔的空气量要适中；③ $\mathrm{T}_{4}$  温度（即污氮出过冷器温度）应控制在 $-168\sim -169^{\circ}C$  之间，防止污氮偏流，造成液空经液空过冷器后过冷度较大。其调整要点如下：

第一，冷凝蒸发器的液氧面高低标志着空分塔的冷量的多少，液氧面的稳定与否则是整个装置工况稳定的标志。在反复调试过程中，发现液氧面在  $900\sim 1000\mathrm{mm}$  高度时蒸

发冷凝效果最佳。如液氧面涨得过快，则应关小膨胀机进气阀，减小膨胀量，使膨胀机制冷量与空分塔的冷损平衡即可。

第二，膨胀后送上塔的空气量决定上塔提馏段的回流比。将调-6阀定在  $70^{\circ}$  左右的阀位上，用通-5阀控制压力，使膨胀后的压力略高于入塔口的压力（每块塔板阻力在25 $\sim 30\mathrm{mmH_2O}$  柱，如上塔压力为  $\mathbf{0.062MPa}$  进料口压力为  $\mathbf{0.046MPa}$  ，膨胀后压力为 $0.048\mathrm{MPa})$  。在调整过程中，调-6、通-5的配合要缓慢。在反复实践中，我们认为送多少，应由污气氮纯度决定。一般污气氮纯度比污液氮纯度低  $1\sim 2\% \mathrm{N}_{2}$  即可，使进入上塔的空气量为总进气量的  $0.15\sim 0.20_{\circ}$

第三，分馏塔主要部位的温度值是空分塔稳定运行的另一标志。在反复实践中我们掌握了控制各点温度的有效方法，在操作过程中应尽可能避免偏流现象，对各点温度须精心操作。各点温度控制如下： $\mathrm{T}_{6}$  为  $-170 \sim -171^{\circ} \mathrm{C}$  （高于饱和温度  $0.5 \sim 1^{\circ} \mathrm{C}$ ）； $\mathrm{T}_{4}$  为  $-168 \sim -169^{\circ} \mathrm{C}$  （以保证入上塔液空过冷度较小）； $\mathrm{T}_{5}$  为  $-177 \sim -178^{\circ} \mathrm{C}$  。在保证氧纯度的情况下，逐步增加氧流量的同时，应密切注意上塔阻力值的变化，使阻力值保持在  $18 \sim 19 \mathrm{kPa}$  之间。在送至  $320 \mathrm{~m}^{3} / \mathrm{hO}_{2}$  左右时，应同时增加少量空气，以保持空分设备的物料平衡（对于制冷效果较差和冷损较大的装置来说，可开大透平膨胀机进气阀，在保证入上塔空气量的情况下，略开通-5，

同时调整污气氮出冷箱的压力，控制好  $\mathbf{T}_{4}$  的温度）。

# 六、全低压流程制氧机操作特点及体会

KDON-350/700型制氧机为全低压空分设备，其设备操作原理与大型低压设备的操作原理相似，这就增加了设备的操作难度。加之所配设备仪表为Ⅱ型半，自动化程度差，这对操作工人的技术文化素质有比较高的要求。特别是350型制氧机工艺流程的特点，在操作过程中，要求操作人员要“慢、稳、勤”，就是调整动作要慢；工况要稳；观察温度变化和纯度变化要勤。在操作过程中，不但要注意产品氧的纯度，还应注意平均氮纯度。因此，全低压空分设备操作特点主要是有效地控制污气氛纯度。

操作350型全低压空分设备，在调整过程中，调整速度不能过快，更不能深度调整，因此一定要有耐心。各部位的纯度、温度、压力值、阻力值，一定要在规定的范围内，装置工况才能稳定。否则即使达到额定产量，在操作中稍有不慎，氧纯度也会下降很快。在必要时还需理论计算，要用理论和纯度分析来指导整个调试过程。

总之，350型制氧机是较先进的空分设备，要熟练地掌握其设备性能、特点，还应提高操作人员的整体素质，并且要在实践中反复摸索，进一步总结经验。

# 一、设备选型

# 1. 深冷法与吸附法之选择

① 我厂现有  $5 \mathrm{t}$  （吨）非标电炉三座，平均出钢量为13.4吨/炉，吨钢耗氧量为  $45 \sim 50$ $\mathrm{m}^{3} \mathrm{O}_{2} / \mathrm{t}$ ，年产钢锭7.5万吨。原有一套150 $\mathrm{m}^{3} / \mathrm{h}$  深冷制氧机，供一台电炉用氧。续建一套变压吸附制氧机，替代原有的深冷制氧机，经炼钢工艺测算，需要一台大于  $550 \mathrm{~m}^{3} / \mathrm{h}$  的制氧机。经询价，一套  $600 \mathrm{~m}^{3} / \mathrm{h}$  变压吸附制氧机仅为同规格的深冷制氧机的  $60 \%$  。

在设备选型时进行工程估算。由于变压吸附制氧  $(\mathrm{PSAO}_2)$  的工艺流程简单，设备的组成也明显比深冷制氧简单，除了罗茨鼓风机、真空泵、氧压机和仪表空压机以外，其余全为容器，所以土建和配电的投资比深冷制氧少得多。经估算，整个工程总造价仅为深制冷氧的  $50\%$  。

# 3. 氧气纯度

益的， $2.5\%$  的氮和  $2\%$  的氮相比只差  $0.5\%$  而且变压吸附制氧机又很容易将纯度提高到  $93.5\%$  。这样，变压吸附制氧机就完全达到了电炉炼钢的要求。

# 4. 低成本高收益

经生产厂家介绍和实地考察，变压吸附制氧机的单耗仅为  $0.45\mathrm{kWh} / \mathrm{m}^3$  （不包括氧压机）。而深冷制氧机电耗为  $0.85\mathrm{kWh} / \mathrm{m}^3$  （不包括氧压机）。此外，变压吸附制氧机还具有设备组成简单、易损件少、维修方便、自动化程度高、启动时间短等特点。

# 三、安装及试车

这套设备的安装，多数属于通用性，但是在安装过程遇到几个问题，我们的处理方法是这样的：

# 1. 蝶阀校验

切换系统的蝶阀都采用对夹式，由制造厂成套提供。对于蝶阀，在安装前应对开启度，严密性，开启灵敏性等进行严格的校验。

为此我单位制做了蝶阀校验台，一方面安装时使用，另一方面在蝶阀更换密封圈时使用。

# 2. 蝶阀旁加装膨胀节

切换系统是由制造厂供图，现场制作。对于对夹式蝶阀来讲，当更换密封圈或处理故障时，都要将蝶阀拆下，而这套设备使用的工艺管道分别是  $\mathrm{DN}325\times 10$  ，  $\mathrm{DN}273\times$  10和  $\mathrm{DN}219\times 8\mathrm{mm}$  管道，变型困难，蝶阀很难拆卸。为此，我们在每一个蝶阀旁加了一只南京晨光金属软管厂生产的膨胀节。

# 3. 吸附器内加料

由于工艺上的要求，先由活性氧化铝吸附原料空气中所含的水分，再由分子筛进行氮气吸附。在加料时，活性氧化铝装在吸附器的底层，并且将表面刮平，方可装填分子筛。装填分子筛时，要尽可能避免分子筛震碎，以减少吸附器的阻力。

# 4. 联动试车

联动试车是个相当复杂的过程，根据各地的海拔高度不同，来确定其切换周期。我单位经过几次试验之后，确定其运行周期为195秒。

# 四、结论

# 1. 经济性好

采用变压吸附制氧，不仅设备费和整个工程费较深冷制氧机相比，分别节省  $40\%$  和 $50\%$  ，而且运行费用极低。对于电炉炼钢来讲，采用变压吸附制氧，可以作为节能的一个新途径。

# 2. 生产周期长

变压吸附制氧设备，易损件极少，只有运行5个月以后方需更换一次蝶阀密封圈，而且是等周期更换，又不象深冷制氧机那样需要定期加温吹除。对于多台电炉生产的炼钢车间，一般采用换炉检修，没有将所有电炉停下检修的机会。所以深冷制氧机在一定程度上制约着炼钢的生产，不得不采用同步检修的办法来弥补这种不足。随之而来的电炉炼钢或制氧机采用同步检修，需要相当数

量的维修工，一般单位是不具备的。采用变压吸附制氧机之后，很少停机，故障率低，对炼钢来讲有很高的可靠性。

# 3. 生产灵活启动快

电炉炼钢用氧，特别是多台电炉用氧，用氧量的波峰与波谷之间的调峰，通常采用中压贮罐来调节。使用深冷制氧机时，除正常启动需用十几个小时以外，而且处理故障的复产时间也很长；而变压吸附制氧机，从启动到出氧仅为1小时。

# 4. 便于管理

变压吸附制氧机，自动化程度高，每班只需1人操作即可完成生产任务，大大简化了生产环节，降低了管理经费。

# $10000\mathrm{m}^3 /\mathrm{h}$  空分设备贫氰气回化系统改造

杨叶萍

[武汉钢铁(集团)公司氧气厂三车间，武汉市青山区，430083]

摘要 介绍了武钢从林德公司引进的  $10000 \mathrm{~m}^{3} / \mathrm{h}$  空分设备中，进行贫氮氮净化系统改造的成功经验。具体做法是，将原碱液吸收法工艺改为分子筛吸附法，清除贫氮氮中甲烷的转化物  $\mathrm{CO}_{2}$  和  $\mathrm{H}_{2} \mathrm{O}$  。不仅减少了氮氮气损失，增加了产量，而且减轻了工人劳动强度，改善了工作环境，降低了生产成本。图2表3。

主题词：大型空气分离设备 贫氟气回净化系统 碱液吸收 分子筛吸附 改造

我厂从德国林德公司引进的四套10000 $\mathrm{m^3 / h}$  制氧机中，三套配贫氮氙净化系统，均采用铂触媒清除贫氮氙中的甲烷。而甲烷的转化物之一——二氧化碳，是通过碱液吸收法清除的。采用这种流程，不仅每年每台设备用于碱的费用高达5万余元，而且在更换碱液塔  $\mathrm{LW}_1$  的碱液时和系统的开、停车时，易发生碱液向接触炉  $\mathrm{CF_1}$  或吸附器  $\mathrm{GD_7}$  、 $\mathrm{GD_8}$  内流失的事故。

碱液流失后，碱液塔  $\mathrm{LW}_{1}$  内碱液的不足将造成系统对二氧化碳净除能力的下降。而接触炉  $\mathrm{CF}_{1}$  和吸附器  $\mathrm{GD}_{7},\mathrm{GD}_{8}$  中一旦进碱，触媒和吸附剂即结块、失效，继而造成系统对甲烷和水的净除能力下降。最后的结果是当贫氮氙中的二氧化碳和水达到一定的浓度(约超过  $50~\mathrm{ppm}$  )时，就会导致二氯塔入口换热器  $\mathbf{E}_{12}$  出现堵塞现象。此时就不得不停止生产，将贫氮氙气在  $\mathrm{GD}_{7}$  或  $\mathrm{GD}_{8}$  出口处放空，并用干燥氮气返吹  $\mathbf{E}_{12}$  （如果堵塞情况严重，可能要吹两天)，造成了贫氮氙气的浪费和经济损失。

过去这类事故屡有发生，最严重的一次是A台的电加热器  $\mathbf{EH}_{\circ}$  因进碱太多，被严重

# 一、工艺系统

制氧过程中，经空气冷却塔冷却后的空气，分两路进入两组可逆式换热器的热段和冷段。在热段，空气由常温冷却至  $-110^{\circ}\mathrm{C}$  左右，而其返流气体污氮则由  $-115^{\circ}\mathrm{C}$  左右被加热至常温；在冷段，空气由  $-115^{\circ}\mathrm{C}$  继续冷却至饱和温度（约  $-172^{\circ}\mathrm{C}$ ），而返流气体污氮被加热至  $-115^{\circ}\mathrm{C}$  左右。冷段和热段串在一起构成一组。在生产过程中，要求空气和污氮按一定周期、一定规律进行切换，以确保空气中的水分和二氧化碳在一个周期被冻结，而在另一个周期被带出。同时为减少冷量损失，保证设备的正常运行，还要求两组中部温度相同。

# 二、控制系统

# 1. 中部温度检测点的选择

# 2. 调节阀及调节器的选择

考虑到空气在参加热交换的物料中比例最大，压力最高，其可控性较好等因素，因此在气动强制切换阀前的空气管道上安装了一台气动三通调节阀，通过它来分配换热器两组的空气量。因为每组空气量的增减，其中部温度都随之改变，所以改变三通阀的开度，即可实现对每组中部温度的控制。三通调节阀输出特性为当阀门开度为零时空气进入二组，一组不进空气；阀门开度为  $100\%$  时空气进入一组，二组不进空气。

# $150\mathrm{m}^3 /\mathrm{h}$  空分设备液面计堵塞分析与改进

# 张志远 殷继承 李毅

（太原化工厂空分车间，山西省太原市，030021）

# 一、堵塞现象的出现

我厂使用小制氧机已有40年的历史，在1984年和1986年先后上两套杭氧产KFON-150/600型制氧机取代原苏式制氧机。在长期的使用过程中，经常发现  $\mathrm{CCl}_4$  液氧液面计因管路填塞而失效。当被迫采用反吹等方法时，收效又甚微。

众所周知，液氧液面计是空分塔重要的安全装置之一，是空分行业操作调节的重要指标和依据。没有液面计这只“眼睛”，轻者靠高压来保持高液氧液面，维持短时间生产（而且易发生严重后果，我厂1984年就曾发生严重事故）；重者需停车重新加热吹扫。这样，不仅水、电、汽消耗猛增，经济上造成浪费，而且影响了全厂生产。

# 二、堵塞原因的分析

1992年，我们就此问题曾与兄弟单位联合探讨、攻关，最终见效甚微。是何原因造成液面计堵塞，只能解释为大量的  $\mathrm{CCl}_4$  和少量的油污所致。

后来，在进行空分塔加热吹扫过程中，我们发现由液面计下阀中吹出的加热气体中，没有油污痕迹，却有较多的  $\mathrm{CCl_4}$  气体，甚至有液体  $\mathrm{CCl_4}$  吹出。塔中  $\mathrm{CCl_4}$  唯一来源只有液面计，加之严格控制工艺指标  $250\sim$ $350\mathrm{mmCCl_4}$ ，故只靠液面差压，不可能将  $\mathrm{CCl_4}$  进入上、下阀管中。终究是什么原因造成的？针对这一问题，我们进一步做分析和观察，又发现堵塞现象夏季明显高于冬季。最终得出结论：液面计中  $\mathrm{CCl_4}$  进入了上、下阀管中，受冷后凝固，堵塞表管，造成液氧

液面计失灵。

查阅化工手册，从卤代烃蒸汽压图（1 atm 大气条件下）得出如表 1。

表 1 四氯化碳与水的性质(饱和蒸汽压)

也就是说  $\mathrm{CCl}_4$  比水易挥发（  $4\sim 6$  倍），尤其是在  $0.05\mathrm{MPa}$  ，  $\mathrm{O_2 > 99.2\%}$  的氧气中， $\mathrm{CCl}_4$  蒸汽压占总压的  $20\%$  以上。

具体机理为液体  $\mathrm{CCl}_4$  蒸发到氧气中，由气体的扩散作用移到上、下表管中，在塔中受到深冷的影响（ $-100^{\circ}\mathrm{C}$  以下）由气体凝结成固态（ $\mathrm{CCl}_4$  的冰点为  $-22.8^{\circ}\mathrm{C}$ ）。也就是说，外界温度越高， $\mathrm{CCl}_4$  在塔中表管内侧凝结速度越快，故而造成了液氧液面计的堵塞，使之失灵。

# 三、解决的办法

解决以上问题的方法有两种：

1. 选用密度与  $\mathrm{CCl}_4$  相似的介质(1.633)，同时应该具有沸点高、饱和蒸汽压极小、化学性质相当稳定、与氧气不发生反应、来源广、毒性较小等特性。  
2. 选用一种以氧作介质传递压差的装置，同时要适应范围广（压力、温度），价格合理，易维修。

围绕以上两种方法，我们做了尝试。第一种方法困难较大，主要是化学稳定性差；第二种方法通过查阅资料认为可行。

我们选用  $\mathrm{DDZ} = 3$  型差压变送器，它利用力平衡原理，可以连续测量液位压差，能将被测参数转换为  $4\sim 20\mathrm{mA}$  的直流电信号输出，已经国家鉴定取得防爆合格证书。为防止氧和仪表的膜盒因长时间接触，与膜盒中  $250^{*}$  苯甲基硅油发生危险，我们选用了材质1Gr18Ni9Ti的膜盒体。同时考虑到液空液位测量压力（  $0.45\sim 0.56\mathrm{MPa})$  ，选用耐

压较高的型号。鉴于液氧液面在全量程时为 $500\mathrm{mmCCl}_4$  ，比重  $d_{\mathrm{CCl_4}} = 1.633,g = 9.8,$  压差为  $\Delta P = h d g = 8\mathrm{kPa}$  。因此我们选用了DDZ-Ⅲ型差压变送器，其耐压6.4MPa，量程  $-14\mathrm{kPa}\sim 0$  ，电流  $4\sim 20\mathrm{mA}$  ，使用温度  $-25\sim 80^{\circ}C.$

# 液空过冷器液空通道堵塞的反吹处理

陈明

（浩良河化肥厂空分车间，黑龙江省伊春市南岔区，153103）

我厂3\*空分装置，是杭氧1989年设计制造的KDON-6000/6600-Ⅸ型空分设备。该空分装置在1990年10月投产以来，运转性能良好，氧、氮产量均超过设计指标。可1991年1月，精馏系统却出现了液空流路阻力增大而引起的下塔液面涨高现象。

# 一、故障分析

从下塔精馏情况来看，下塔阻力及馏分氮和液空纯度都很正常，可排除是调-3和调-2阀开度小所致。切换液空吸附器，吸附器后压力PI-18仍与下塔压力PI-5相接近，下塔液面无下降趋势，最后确认是液空过冷器前后或调-1阀存有阻力。

鉴于当时的生产情况，我们被迫采取了稍开调-3阀，但要保证主冷冷凝下的纯液氮纯度含氧不高于1000ppm，增开调-2阀，馏分氮纯度由  $98.5\%$  降到  $88\% \sim 90\%$  。总之，在通过减少下塔回流比来控制底部液面上涨的同时，还不能影响后部生产工序对氮气纯度的要求（  $\geqslant 99.9\% \mathrm{N}_2$  ）。

# 二、初步处理措施

1991年7月，3#空分装置进行扒珠光砂检修，并决定对液空过冷器实施吹除处理。先把调-1阀入口法兰断开，启动空压机，用0.5MPa压力空气正吹。吹除流程为：切换板翅式换热器  $\rightarrow$  下塔底部  $\rightarrow$  液空吸附器  $\rightarrow$  液空过冷器  $\rightarrow$  调-1阀入口法兰处。反吹也是利用空压机气源，通过板翅式换热器导入下塔，开大调-3和调-2阀使上塔压力在0.06MPa左右，全开调-1阀，吹除气体由调-1阀锯开的入口管道排出。

经过半个小时的正吹，在液空过冷器出口处的过滤网内聚积了一团软粒状矿渣棉。由于堵塞异物来源和吹除效果不清楚，为保证装置今后的运行，我们又在液空过冷器前后加一DN80的旁通长杆冷角阀门及管线。经检修后开车证实，当装置加工空气量及精馏工况正常时，停车前的那种靠开大调-3和调-2阀来控制下塔液面上涨已无法操作，要恢复下塔正常的阻力工况及馏分氮和液空纯度，旁通阀门需开  $4\sim 5$  圈才能控制住下塔液位，这种操作一直维持到1994年装置计划检修，表明该装置液空流路阻力仍然过大。

# 冷冻水泵汽蚀现象的处理与效果

王春友

（吉化公司化肥厂空分车间，吉林省吉林市合肥路27号，132021）

我们厂KDON-12000/18000型空分设备，在1992年11月进行空气预冷系统试车时，发现冷冻水泵汽蚀严重，无法投入正常工作，即使勉强开起来，上部冷冻水回水液位也逐渐降低，汽蚀加剧，泵不能工作。

# 一、汽蚀的原因

针对冷冻水泵汽蚀严重而不能正常工作的现象，我们将施工时所加的冷却水向空冷塔上部供水的阀全开(DN150供水阀)，关闭HV1102阀，将冷却水泵输送的水全部送至空冷塔上部(流量为  $120\mathrm{m}^3 /\mathrm{h}$  )，再进行冷冻水泵试车，冷冻水泵仍然产生汽蚀而不能工作。我们断定冷冻水回水液槽有问题。

经检查发现，该液槽底部靠空冷塔塔壁的一周均为点焊，底部钢板边缘处也有等距离的孔洞。未焊的缝隙及孔洞产生漏液，使液槽液位下降，空气窜入冷冻水回水管中，造成冷冻水泵汽蚀。

# 二、处理方法及效果

问题查出后，我们将液槽底部靠空冷塔塔壁未焊的一周缝隙，及底部钢板边缘的孔洞，进行了修复补焊。

补焊后我们重新进行了空气预冷系统的试车，试车方法按正常生产投入空气预冷系统的启动方法进行，冷冻水泵启动顺利，无汽蚀现象发生，使冷冻水泵能够投入正常工作，上部液槽液位也不再下降了。经过正常生产使用验证，空气冷却效果很好。

# 三、建议

1. 制造厂家在制做空气冷却塔时，就应将上部冷冻水回水液槽的底部焊好而不能漏液。不然将会给用户带来麻烦。  
2. 在制做空气冷却塔时，为了用户便于检查捕雾器，应在捕雾器的上部或上、下各加一  $\phi 500\mathrm{mm}$  的人孔。因有些带水事故是与捕雾器有一定关系的。捕雾器上、下各加一人孔后，就便于检查和修理。

(1994年12月）

# 空压机曲轴轴颈磨损原因及对策

孙阳 冉渝

（国营重庆造船厂动力处，四川省巴县明月沱，631339）

1990年12月，我厂制氧用5L-16/50型空压机，因使用中声音异常、转子转动时偏心等原因，拆检了空压机曲轴组件，发现已连续使用十余年的曲轴电机端3628型轴承与轴颈的配合失效。其曲轴颈的尺寸由图纸标注的 $\phi 140_{+0.004}^{+0.03}$ 已减为  $\phi 139.78$  ，轴承内圈在轴颈上可来回晃动，而轴承外圈与设备本体配合仍在可使用范围。轴承滚道内布满干油垢，滚柱转动不灵活，轴承呈缺油态。未见变色，构件也未见明显损伤。检查润滑给油管端部已砸得很扁，只留一小孔出油。考虑我厂现无曲轴备件，决定采用电刷镀技术修复曲轴、更换轴承、给油系统改造等三条措施。

# 一、电刷镀修复曲轴

刷镀设备为SD-150型。

刷镀方法是：采用专用预处理液和金属镀液，一台直流电源和相应镀笔，把曲轴清洗干净后夹持于车床上并找正，启动车床控制转速  $25\mathrm{r / min}$  ，用  $0^{\#}$  砂布将磨损的曲轴颈打磨至等，用预处理液去除刷镀部位的油膜及氧化层，镀特殊镍后再镀快速镍，镀至 $\phi 140.06$  ，用  $0^{\#}$  砂布打磨至  $\phi 140.03$  。刷镀结束时，按图纸要求对全轴进行形位公差测量，达图纸要求即可装配。

装配采用热装法，方法将纯净  $19^{\#}$  压缩机油约  $20\mathrm{kg}$  装于开口桶内，放入轴承至淹没。川明火加热至油温  $150^{\circ}C$  ，中间要把轴承上下翻一下，此温度保持20分钟，使轴承热透并注意监督油温不能超过  $240^{\circ}C$  。否则，会

使轴承退火甚至变色。此时应用长尺测量孔径，应大于轴承径，取出后迅速套入轴颈到位即可。

# 二、增大给油量改善润滑

方法是将砸扁的给油管管端锯掉，使  $\phi 6 \times 1$  紫铜给油管畅通给油；对轴端压盖进行改进，扩大回油孔，使油道内不存油。

该轴承配合失效，本人认为有以下几个因素：

1. 该空压机电机为悬挂式，转子悬挂于曲轴端，转子旋转时产生较大离心力作用于轴承处，严重影响轴承寿命。电机安装修理都是以转子为中心而调整定子，使转、定子间环向间隙均匀。压缩机长期使用后基础螺丝会产生松动，都用紧固方法处理。紧固后不检查定、转子间隙，也会造成转子偏心，偏心造成运转不稳，作用于轴承。如轴颈加工尺寸为配合尺寸下限即  $\phi 140^{+0.004}$ ，或轴承内圈尺寸不在规定范围内而过大，也会造成轴颈磨损。根据我厂经验，更换轴承时，必须先对轴承尺寸进行复查，消除是伪劣商品的可能性。

2. 该处轴承润滑不良。原设计(1978年设计)采用油泵及  $\phi 6 \times 1$  紫铜管输油管管端打扁后插入另一粗管内向3826轴承自流给油润滑。该轴承因给油管打得过于扁而少出油影响润滑，就增加了配合面之间的摩擦力。

曲轴当即修复后使用至今，运行正常。

（1995年1月）

# 一、引言

多年来，空分设备的防爆问题，已引起国内外同行的极大关注。特别是70年代末期，国内大型空分主冷爆炸迫使设备停产的事故屡见不鲜。空分行业曾多次组织过学术研究讨论，从设备的设计、制造到工艺操作的改进等方面作了大量的工作，有效地抑制了空分爆炸事故的发生。液氧中积累有爆炸危险性的物质所引起的爆炸，通常发生在主冷凝蒸发器中，并占已知爆炸总数的  $95\%$  以上，这自然将人们注意力的重点引向主冷及液氧系统，而忽视了对其它部位及液空系统的微爆预防。然而，“千里之堤溃于蚁穴”，无论空分任何部位发生局爆，所产生的后果及损失同样巨大，应足以引起我们的高度警觉。“前车之鉴，后事之师”，为给同行提供借鉴，现将我们武钢能源总厂氧气厂林德 $10000\mathrm{m}^3 /\mathrm{h}$  空分设备（B台）新增的一条液空管道局部爆炸事故的起因、经过及原因分析作一介绍。

# 二、液空盲管的来源

1990年我们武钢能源总厂氧气厂打算对B台空分设备进行改造，准备增加一套氨氩

提取装置。因此，趁空分设备检修期间，在空分塔内新增加了4根管道。其中，包括一根  $\phi 80\mathrm{mm}$  自下塔引出的低温空气管，至一氮塔底液化后，用另一根  $\phi 40\times 3\mathrm{mm}$  的铝合金管将液空送到阀120后（见图1）与下塔来的液空总管汇合后送入上塔；另外自液氧吸附器  $\mathbf{A}_2$  前引出一根  $\phi 40\mathrm{mm}$  液氧管至一氮塔，在一氮塔参与精馏之后，再从该塔顶引出一根  $\phi 100\mathrm{mm}$  气氧管道返回到上塔氧气输出总管上。后来因故氮氙装置未上马，为保证生产，这四根管道自空分冷箱引出之后均用盲板堵死。设备一直带着四根盲管运行，从此埋下了液空盲管局爆的隐患。

# 三、事故经过

1993年6月24日17:00时，在B台10000 $\mathrm{m}^3 /\mathrm{h}$  空分设备连续正常运行了一年多的情况下，空分操作工突然听到一声闷响，下塔液空液位及主冷液氧液位明显下降，17时40分发现空分冷箱壳体七处冻裂并有液体流出。当时怀疑液空吸附器  $\mathbf{A}_1$  接盘漏液，立即将 $\mathbf{A}_1$  的旁通阀205打开，同时切除  $\mathbf{A}_1$  的使用，但液位仍然持续下降，至25日4时45分，液氧液位已从正常的  $80\%$  下降至  $22\%$  ，氧、氮纯

度已无法满足用户需要。入塔空气量逐渐减少至正常值的  $80\%$  以下，生产难以为继，被迫停车，对空分进行排液大加热。扒除珠光砂，发现自一氮塔返回  $\phi 40 \times 3 \mathrm{~mm}$  、35米长液空管，被炸断了约1米左右，爆炸处靠近冷箱壳体下部转弯处（见图1的BC段）。在切除了新增的四根盲管后，设备再次按常规启动，恢复了生产。

# 四、事故原因及分析

公司能源总厂在我厂组织召开事故分析会，大家一致认为是  $\mathbf{C}_{\mathrm{n}}\mathbf{H}_{\mathrm{m}}$  积聚所引起的化学性爆炸，其主要理由如下：

1. 自 1990 年接管至事故发生的三年多时间内，B 台空分共进行了两次常规停产检修，但停产后的排液、加温操作都忽视了拆开盲管上的盲板。致使盲管内的液体自然蒸发， $\mathbf{C}_{\mathrm{n}} \mathbf{H}_{\mathrm{m}}$  被浓缩。  
2. 自1993年1月以来，液空、液氧中之 $\mathrm{C}_2\mathrm{H}_2$  含量一直超过正常控制值(0.01ppm)，最高时达到1.65ppm。经统计60多个化验数据，发现有42个数据大于0.1ppm，也就是说有  $70\%$  的数据超过了林德公司规定的报警值(0.1ppm)。虽然我们多次采用缩短液空、液氧吸附器的使用时间及排液等操作措施，但作用不大。在我们更换了液空、液氧吸附器的全部硅胶之后，这一现象基本消除了。

3. 由于盲管内的乙炔等不饱和烃在有氧存在的情况下具有很高的化学活性，当浓度达爆炸极限时，受到空分切换等工况波动的冲击而产生静电，诱发爆炸。  
4. 为了进一步证实这次爆炸的性质, 我们作了如下的测算验证工作。

# （1）爆炸强度测算

爆炸的液空管道为  $\phi 40\times 3\mathrm{mm}$  ，材质是铝镁合金，根据材料力学公式：

$$
\sigma_ {s} = \frac {1}{\mu} \left(\sigma_ {\theta} + \sigma_ {r}\right) = \frac {2 \left(P _ {1} r _ {1} ^ {2} - P _ {2} r _ {1} ^ {2}\right)}{\mu \left(r _ {2} ^ {2} - r _ {1} ^ {2}\right)}
$$

其中  $\mu = \frac{3E - 2G}{6E + 2G}$

式中  $r_1$  ——内半径

$\pmb{\tau}_{2}$  一外半径

$\pmb{r}$  ——平均半径

$\sigma_{0}$  切向应力

$\sigma_{r}$  ——径向应力

$P_{1}$  ——管内压力

$P_{2}$  ——管外压力（即大气压）

$\pmb{E}$  ——弹性模量， $10 \times 10^{6}$  磅/英寸²

$G$  ——剪切弹性模量，

$4 \times 10^{6}$  磅/英寸²

$(E, G$  基本不受温度变化影响）

温度为  $-180^{\circ}\mathrm{C}$  至  $-190^{\circ}\mathrm{C}$  时，查表知：

$$
\sigma_ {\theta} \doteq 1 1 9 \mathrm {M P a}
$$

$$
\sigma_ {r} \doteq 2 8 7 \mathrm {M P a}
$$

通过计算得  $P_{1} = 28.239\mathrm{MPa}$

即至少压力为28.239 MPa，才会造成 $\phi 40\times 3\mathrm{mm}$  的铝镁合金管炸开。而盲管的另一端与上塔相连，若是超压爆炸，势必引起安全阀起跳、空分系统压力超高，但当时并没有出现上塔超压现象，因此可排除物理爆炸。

# （2）爆炸生成物分析

检查发现炸裂口的上部管道（AB段）内壁光亮清洁，而炸裂口至堵板一段（CD段）管道内壁附着一层薄薄的黑色生成物。我们采用透射电子显微镜、扫描电子显微镜及能

谱分析仪，对管内壁表面附着的黑色生成物进行了定性分析，分析图线(见图2)表明，黑色物质为炭黑。因此，这次事故是由碳氢化合物所引起的爆炸，得到进一步确认。

# 五、预防措施

1. 为有效地预防此类事故发生，空分液体管道应尽量避免形成盲管。如工艺确实需要，在管道布置上要考虑增加倒“U”型弯管，使之形成自然气封。  
2. 液体管道的布置除了避开高温区域外，对于间断产生的高温部件（如吸附器）也应该注意间隔一定距离。  
3. 在加强对液氧进行  $\mathbf{C}_{\mathbf{a}}\mathbf{H}_{\mathbf{m}}$  分析的同时，应注意定期对液空进行  $\mathbf{C}_{\mathbf{a}}\mathbf{H}_{\mathbf{m}}$  含量分析。  
4. 所有空分排液管实际上相当于盲管，因此在空分设备运行期间，应对所有排液管进行定期少量排液。最好每三个月排一次。  
5. 重视对厂区空气中烃类物质的监测，以尽早发现并控制潜在的爆炸因素。

(1995年2月改稿)

# 三、事故原因及处理

1995年1月8日，我们扒完分馏塔全部珠光砂，对塔内设备及管道作了全面检查。发现吹蓄-13管有  $\phi 10\mathrm{mm}$ ，及PdI/1一1热端阻力仪表管(均属铝材)有  $\phi 3\mathrm{mm}$  两个小孔（见图）。通过认真分析，事故就在于此。开始时，此处缝隙较小，但随着时间延长，必然导致缝隙的进一步扩大。由于该处（压力0.5MPa）泄漏空气吹动珠光砂，致使相对处仪表管路长期受到珠光砂的冲刷，导致仪表管变薄(约有  $400\mathrm{mm}$  长）、泄漏。两处孔隙不断扩大，所以漏冷越来越多，无法补充，空气“吃”不进。吹除时发现珠光砂，也是由于此孔隙。未开吹除阀时，由于塔内设备管道振动，砂从洞入，流入吹蓄-13管道；开启吹蓄-13，砂即随气体流出塔外。

后我们将两泄漏管切除各  $1\mathrm{m}$  （沿孔上下各  $0.5\mathrm{m}$  ），用套管进行氩弧焊接，并对塔壁进行补焊。开车运行至今，无任何异常现象，一切参数均达到设计水准。

# AOD用于高质量钢铸造

AOD是原美国联碳公司、现美国普莱克斯实用气体公司的专利技术“氩氧脱碳制钢工艺”的简称。它是双炼工艺，先将废钢或原料在电弧炉或感应炉内熔化，然后在专门的AOD炉内脱碳和精炼。受控的氧、氩经下吹管吹入到熔融金属中进行脱碳，并使金属氧化物减至最少。在AOD炉中脱氧、脱硫并从渣中回收所需要的金属，在低合金钢时，还脱磷。在整个过程中，连续进行排气和均质化，从而生产出干净和均一的产品。

AOD是美国联碳于1954年发明的，经过综合开发后，1968年由位于印第安纳州Ft.Wayne的Joslyn不锈钢厂开始工业生产。世界不锈钢工业很快认识到这种工艺能用便宜低质的原料，生产出高质量产品的能力。今天世界不锈钢总产量的78%是由AOD生产，估计到本世纪末还要超过90%。

AOD工艺使用装有耳轴、开口的炉子，内附铬镁或白云石耐火材料砖。氧气和氩气经过装在炉子侧壁位于熔池内的喷管吹入。热量由熔池内组分进行可控氧化反应产生，不需任何外加热源。

起初由氧对氩比例较高的气体混合物吹入熔融金属中，随着熔池中碳含量减少，氧气比例逐步降低，以维持最佳的热动力学条件。氧气用氩气稀释是为了降低熔池内鼓泡中CO分压，这种稀释比用纯氧更有利于脱碳，使脱碳效果增加而金属氧化减少。

所有氧气都在熔池内反应完毕，通过检测和记录在吹炼中氧的消耗，达到终点碳的正确控制。

由于氧和氩高速吹入到熔池表面以下，使熔池内产生完全的混合，并使渣与金属接触良好，从而极大地提高了化学反应的速率。

工艺的主要特点，是通过下吹管喷入的氩气的脱气效应和脱碳中产生的CO结合起来，从而不需要附加的处理和昂贵的脱气设备，就可使氧、氮、氢的残留量达到很低。

# AOD在铸造工艺中采用后的优点

(1)提高金属产量；(2)在所有使用温度下可以增加刚度和可延性；(3)减少疲劳破坏；(4)改善可焊性；(5)减少铸件的次品率；(6)化学组分可精确控制，这意味着热处理反应更加一致；(7)金属产品很干净，杂质和气体含量低。

我国AOD精炼工艺技术的开发，是从1973年开始的，1983年9月自行设计的第一台AOD精炼炉在太原钢铁公司正式投产至今，国内已有好几家冶炼厂、钢铁公司建立了AOD炉，但仍然不普遍，与国外相比差距很大。因此应迅速推广这项先进技术。

随着AOD精炼工艺这项先进技术不仅仅在锻造不锈钢、合金中，而且在铸造不锈钢、合金中进一步推广，此必将带来对氧气、氩气的大量需求(99.5%O₂, 99.99%Ar)，也必将带来我国工业气体市场的繁荣。

（北普气体公司 濮春于1995年1月）

# 对一起氧压机着火燃烧事故的分析

谢琼

（涟源钢铁股份公司制氧车间，湖南省娄底市，417009）

1995年4月15日，涟钢制氧车间一台 $1800\mathrm{m}^3 /\mathrm{h}$  的3Z5.5-33/30型氧压机二级缸着火燃烧。该机是开封空分设备厂产品，采用立式三级三列双作用无油润滑。1994年仅投产半年左右，1995年1月进行了一次中修，此后一直闲置。1995年4月15日上午  $11:40$  正式开机，12：45就发生燃爆事故。结果，活塞杆被烧断，连杆折断，密封结构烧坏，下部机壳严重破坏。幸亏当班人员及时关闭氧气进口阀门，赶紧灭火，才未使事故扩大。

根据运行时间与事故现场分析，燃爆的原因大概有以下几个方面。

# 一、活塞杆上挡油环油密封性不佳

在现场发现活塞杆从活塞下  $4\mathrm{cm}$  处烧断，并且存在积炭现象，说明气缸中有油存在。据了解，中修以后，活塞杆上一直带油，维修人员去修过几次，情形虽有所好转，但没有彻底根除。由于挡油环油密封性不佳，油随活塞杆直接带入气缸，或者呈毛细管现象从挡油环表面进入气缸。进入气缸的有机油与压力氧接触，成为燃爆的原因之一。

# 二、设备检修装配中可能存在的隐患

设备中修以后，虽然进行了几次试车，但都没有按照要求进行。每次试车时间都只有一个多小时，并且使用空气作介质，没有达到试运转4小时的试车要求。现在，事故现场损坏严重，已经看不出是否在检修中有掉渣现象或气体泄漏现象。

# 三、空气试车可能产生铁锈

试车记录表明，冷却系统一切正常。因此，进入气缸的有机油，很难达到着火点。由此分析，必然在气缸中存在局部高温。而产生局部高温的原因，就是气缸中有铁锈或细渣。因为现场已经无法分析是否有掉渣现象，并且维修人员也一致否认，所以只能从

铁锈角度来分析。该氧压机曾三次试车，用的都是湿空气，并且从试车完毕到投入使用，中间有三个月的间隙。因空气湿度大，闲置时间又长，存留于气缸中的空气必然对气缸壁产生严重锈蚀。设备启动后，铁锈被活塞环刮落，在压力氧中产生局部高温，从而引燃有机油。

# 四、事后思考

由上述分析，我们必须吸取以下教训：

1. 氧气压缩机不同于其它压缩机

它的可靠性、密封性要比其它种类的压缩机要求高得多。因此，一方面，制造厂家要对此高度负责，要使密封性、安全性绝对可靠；另一方面，运行单位要在设备维修方面严格把关，维修后的试车工作一定要有组织有章程地进行。

2. 改变试车介质

因为湿空气的水分含量大，所以不妨改湿空气为污氮气或氮气，或者改为经过纯化器后的干空气。在我们车间，现有的三套空分设备都是采用自清除形式。返流污氮气含大量水分，而氮气又得保证炼钢需要，所以一般试车都是用湿空气进行。我车间新10000 $\mathrm{m}^3 /\mathrm{h}$  空分设备即将投产，这套空分设备是杭氧产品，采用分子筛纯化。因此，今后改用污氮气或干空气给氧压机试车将成为现实。

3. 注意检查隐患

设备闲置较长时间后再启用时，开车之前一定要仔细检查，确保里面没有锈蚀现象。

（1995年5月3日）

# 制氧工问题解答（3则）

1. 问：分馏塔启动后，为什么高压压力升不高（黑龙江省火电四公司读者问）？

答：分馏塔启动后，高压压力升不到设计值的可能原因主要有：（1）加工空气量不足；（2）膨胀机进、排气阀门严重泄漏；（3）高压漏低压。究竟是什么具体原因，应进行检查后才能判断，现作粗略介绍如下。

首先，测定空压机空气排气量是否达到设计要求。从氧、氮、馏分的排出量来判断，保持低压压力在  $0.055\mathrm{MPa}$  时，一般氧气流量是  $40\mathrm{cmCCl_4}$  柱以上，氮流量是  $35\mathrm{cm}$ $\mathrm{CCl_4}$  柱以上，馏分在  $12\mathrm{cmCCl_4}$  柱以上。说明空气量是达到设计要求的。

否则，检查各管路、纯化器进出口阀门、空压机各级吹除阀、油水分离器吹除阀是否漏气？检查空气过滤器是否堵塞？空压机阀门是否损坏？气缸、活塞环是否磨损？若空气量充足的话，看分馏塔中、低压是否过高？如果中压压力明显大于正常值，则应检查膨胀机进排气阀门密封情况，检查膨胀机阀门密封的方法。关闭节-1阀和通-6阀，放尽中压，将膨胀机进气阀门处于关闭状态。缓慢打开通-6阀，当高压压力达到2MPa时，打开膨胀机出口压力表接头，是否漏气。若进气阀门良好，关闭通-6阀，打开膨胀机吹除阀，放掉存气，再将膨胀机进气阀门处于升启状态，排气阀门处于关闭状态。缓慢打开通-6阀，将高压压力升到2MPa，检查膨胀机出口压力表接头是否漏气。若进排气阀漏，应拆开检修。

若膨胀机进排气阀门均良好，就要检查热交换器管子是否裂开。检查方法是，将分馏塔所有阀门关闭，打开高压空气进口阀，

将高压空气压力升到  $4\mathrm{MPa}$  ，看低压系统有否气体，若有气体则意味着高压漏低压的可能性存在，进一步确定漏气部位和修理。

（杭氧集团公司空分一厂 倪盛利质 监部时必辅）

2. 液空节流阀、液氮节流阀已达到“规定”的开度，为什么液空、液氮纯度达不到设计要求？

答：液空、液氮节流阀所谓的“规定”开度，是指产品说明书或一般资料上推荐的开度，液空节流阀开4圈左右、液氮节流阀开2圈左右；另一种含义是上次调纯后达到的最佳开度。

实际操作中，液空、液氮节流阀不存在“规定”开度的概念。同一型号空分设备所采用的节流阀在制造上存在差异，如节流阀孔径的公差或节流阀杆顶部斜度公差都有允许公差值。同时，操作中，在阀门校零位时松紧不同，就会引起阀门最终的开度不一样。说明书和资料推荐的数值，指多数设备在这个范围，本人曾对抗氧生产的50套  $150\mathrm{m}^3 /\mathrm{h}$  空分设备试车记录作过统计分析，约  $90\%$  的液空节流阀最佳开度在4圈左右，液氮节流阀最佳开度在2圈左右；但液空节流阀最佳开度最小的是3圈150度，最大的是5圈180度；液氮节流阀的最佳开度最小的是1圈90度，最大的是2圈210度。从而易见节流阀的最佳开度是不同的，要从实践中去摸索。

对同一套设备节流阀的开度也是有变化的，一年四季气温在变化，空压机吸入量有差异；空压机活塞在不停的运动，有运动摩擦必然有磨损，机件间隙在扩大；各种吹除

阀、切换阀的状态在改变。空气量的改变直接影响到节流阀的开度。

因此，一套空分设备液空、液氮节流阀的最佳开度，是因地因时摸索出来的。正确的方法，是与液空节流阀控制液空液面在  $10\sim 15\mathrm{cmH_2O}$  柱，然后用液氮节流阀控制液氮、液空纯度到设计要求。

# 3. 液空、液氮节流阀已达到最佳开度，有时候为什么纯度会变化？

答：在调纯期，液空、液氮节流阀已达到最佳开度，但随着运转时间的推移，带入塔内微量的二氧化碳逐渐增多，逐步堵塞节流阀阀孔，会使液空液面自动升高，或使液氮纯度自动升高。若不及时采取措施，液空

进上塔减少而在下塔积累，淹没塔板使下精馏完全破坏，从而使产品氧氮纯度破坏。

有时液空液面并没有上升，会发现氮气纯度自动下降，这是因为液氮节流阀被干冰轻微堵塞所致。这相当于液氮节流阀多关小几度，使下塔回流液增加，上塔顶部回流液减少，上塔上部上升蒸汽中氧分子不能得到充分冷凝，从氮气出口逸出。从现象上反映出来的是液氮纯度升高，氮气纯度下降，氧气产量下降。

因此，在正常操作中要及时吹除油水和切换纯化器，确保进塔空气的二氧化碳含量不大于  $2\mathrm{ppm}$  。每个班(8小时)要对液空、液氮节流阀刮一次霜，确保节流阀不堵塞。

（杭氧集团公司质监部 叶必楠）

# 3LE-10/8型压缩机气缸冻裂的处理方法

郑华东\*

（福建炼油厂供气车间，福建惠安，362100）

1990年1月我车间发生一起3LE-10/8型氮压机二级气缸冻裂事故，现简单介绍如下。

# 一、事情经过

1月31日下午4时，我们检查发现车间 $1^{*}$  氮压机（南京压缩机厂生产的3LE-10/8型无油滑阀压缩机）二级气缸冻裂。裂缝长37mm，最深处约  $7.5\mathrm{mm}$  。裂缝处渗出水珠，表面油漆剥落。经维修工卸下缸盖，发现气缸水套内有积水，且水套四周结有约  $20\mathrm{mm}$  厚的环状冰块。

# 二、事故原因

经分析造成气缸冻裂的主要原因是备用氮压机未放尽气缸套内冷却水，在严冬气候，造成气缸套内积水结冰，而冻裂气缸。其次要原因是1月30日的气温，骤降到摄氏零下 $10\%$  。这是当地少见的低温天气，故造成厂房内的氮压机气缸冻裂。

# 三、处理方法

由于缝隙小，裂缝长不易焊好。故我们采用环氧树酯粘结。具体如下：首先拆下气缸，用钢锯条、小锉刀在裂缝处进行清洁和打磨处理；其次用丙酮进行化学处理，采用浸泡法除锈；然后用环氧树酯进行粘结；采用多次渗透的方法进行粘结，一共进行5次。环氧树脂厚约  $15\mathrm{mm}$ ，最后进行干燥处理。处理完毕后于2月10日装机试车，先空负荷运行2小时，无异常；后负荷试车，二级压力达  $0.78\mathrm{MPa}$ ，二级排气温度达  $130^{\circ}\mathrm{C}$ ，试运4小时，运行正常。

该机通过以上处理后至今运行良好。

（1995年1月改稿）

*本稿所述事故，系作者当时还在武汉石油化工厂空分车间时发生的事。
