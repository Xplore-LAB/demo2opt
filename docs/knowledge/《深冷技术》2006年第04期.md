312 氧压机仪表密封气减压阀不匹配
在调试期间多次出现氧压机因仪表密封气压力
低低值报警而无法启动的情况。经过分析, 发现是
氧压机仪表密封气减压阀PVC1551、PVC1552 的选
型有问题,
阀的进口口径偏小,
导致启动时仪表密
封气流量不够,
仪表密封气压力低低值报警而
停车。
在与外方联系且新的减压阀未到的情况下, 经
过计算和实际分析, 并在外方同意下, 临时将该减
压阀进口加工扩大一定尺寸, 以保证氧压机运行时
需要的密封气流量。
313 膨胀机仪表气压力过低
在调试过程中, 还发现两台膨胀机在运转过程

大。对该阀在无负荷时检查, 行程完好,
而此时该
阀所使用的仪表气是密封氮气,
压力过低, 当时只

311 精氩塔的设计对操作的要求非常高

5 250mm,

3/ h,

3/ h。如果60000m
3/ h 空分设备的精氩塔

3 影响我国空分设备市场结构的原因分析
造成我国空分设备极高寡占型市场结构的原因
是多方面的, 其中规模经济水平、市场进入和退出
壁垒、产品差异化程度、产业政策和法律法规是影

[ 10] 。根据我国空分设备产
业的特点,
从进入壁垒和产品差别化两个方面来分

311 进入壁垒
( 1) 技术壁垒是造成我国空分设备极高寡占型
市场结构的主要原因。空分设备技术含量极高, 并
非投入少量资金、购置几台设备、配置一些人员就

30m

聚了政府、国内外专家和几代杭氧人的大量心血。
在此过程中,
技术进步始终贯穿于整个发展过程,
没有技术进步就没有杭氧的今天。不少小企业想进
入空分设备产业,
大多由于没有技术而得不到发
展。大中型空分设备的市场集中度明显比小型空分
设备的市场集中度高,
就很好地说明了这一点: 小

新企业根本不可能进入竞争领域。

4 改善我国空分设备产业市场结构的建议
我国空分设备产业市场结构存在的问题严重影
响了产业的健康、持续和快速发展,
规范和改善市

境中,
政府已不能完全对企业进行支配,
尤其对空
分设备这样一个规模较小的产业, 政府不会采取积

要从产业自身
出发,
从市场出发, 从企业出发。
411 积极发展中小型空分设备制造商
通过发展中小型空分设备制造商, 使他们在较
短的时间内形成较强的竞争能力, 是改变目前极高
寡占型市场结构,
提高产业绩效的有效途径。政府
部门对地方中小型空分设备制造商应给予一定的政
策支持, 特别在技术创新方面要给予大力支持, 只
有不断有新的中小型空分设备制造商加入到空分设
备市场,
才能不断促进我国空分设备技术的进步,
才能形成与跨国公司抗衡的产业群。

3 类企业中,

大部分属于国有企业或尚处改制阶段的国有企业。

营化和股份制道路, 使企业真正成为产权清晰、管

显, 建立一套自主技术创新机制是这类企业的当务

111 能耗计算方法
能耗与空分设备的大小、流程和液体的提取量
等均有关, 目前国内外空分设备能耗的统计与计算
方法不大统一
[ 2] 。
其中一种比较简单的计算方法是以总电耗除以
氧气产量,
得出所谓的/ 制氧单耗0;
另一种方法
是考虑产品所具有的3 种可逆功,
即最小分离功、
最小液化功和最小压缩功, 将几种不同的最小功直
接相加减。
实际上这两种方法都有一些问题。第一种方法
未考虑生产低温液体和带压气体产品需要提供的外
功耗, 这种方法对于以生产低压气体产品为主的空
分设备可适用,
但当应用于液体产品量较大或出冷
箱气体产品压力较高的空分设备时, 就很容易出现
计算混乱。第二种方法则忽略了实际过程中获得各
种可逆功的流程效率并不相等、而且相差很大的事
实,
当用这种方法比较两套液体产品量相差较大或
出冷箱气体产品压力相差较大的空分设备时, 决定
能耗指标的往往是液体产品量或是出冷箱气体压
力,
而不是空分设备本身的性能指标以及所体现的
技术发展水平,
因而也不能较真实地反映出设备的
能耗指标大小
[ 3] 。
空分设备经济运行软件系统是根据各个空分设

( 2) 在同样的气量、管径条件下,

低压流程
空气中油气含量较高。更换高效油分器,
但效果并
