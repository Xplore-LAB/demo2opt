# 我国气体分离与液化设备的发展

陈允恺 赵小莹

（气体分离设备行业协会，杭州艮山门外东新路152号，310004）

摘要 介绍我国气体分离与液化设备行业的主导产品、发展情况与水平，以及发展前景。并附录了我国气体分离与液化设备技术发展与行业活动大事记。表1。

# 主题词：气体分离与液化设备行业发展 主导产品 前景 大事记

我国的气体分离与液化设备，是新中国成立后才建立和发展起来的。1951年开始，试制小型制氧机  $(30\mathrm{m}^3 /\mathrm{h})$  ，1952年国家决定生产制氧机，并从1956年起批量生产制氧机（空气分离设备）。经过40多年的建设和发展，我国气体分离与液化设备工业生产规模逐步扩大，生产技术水平和管理水平不断提高，目前在我国已形成一个独立的设计和制造体系。

据统计，到1995年底为止，行业共生产7643套气体分离与液化设备，产品遍及全国各地，并已出口到世界20多个国家和地区。其中，空气分离设备的总制氧能力，相当于 $206.25\times 10^{4}\mathrm{m}^{3}/\mathrm{h}$ ，每年单是分离氧所消耗的电能大约100亿度，相当于一座功率1200MW的发电站。全国有八个气体分离和液化设备的专业制造厂和两个研究所，及一个成套公司；此外三个高等院校先后设置了低温技术和制冷专业。

# 一、行业的主导产品

1. 空气分离设备类：制氧容量为  $20\sim 30000\mathrm{m}^3 /\mathrm{h}$  系列的空气分离设备，其中有： $20\sim 300\mathrm{m}^3 /\mathrm{h}$  系列中压流程的小型空气分离设备；  $300\sim 800\mathrm{m}^3 /\mathrm{h}$  系列全低压分子筛净化增压膨胀流程的小型空气分离设备； $1000\sim 30000\mathrm{m}^3 /\mathrm{h}$  系列全低压分子筛净化的

（低压膨胀或增压膨胀)的中大型空气分离设备；  $50\sim 6000\mathrm{m}^3 /\mathrm{h}$  全低压分子筛净化流程（低压膨胀或增压膨胀)的制氮设备。

2. 低温液体设备类：包括制取液氧、液氮、液氩、液氖、液氩、液氦设备等。

3. 稀有气体提取设备类：容量  $3 \sim 1080$ $\mathrm{m}^{3} / \mathrm{h}$  氩提取设备以及氮、氦、氪、氙设备。

4. 变压吸附分离设备类：包括变压吸附制氧设备、制氮设备及制氢设备。

5. 多组分气体分离设备类：包括  $3 \times 10^{4} \mathrm{~m}^{3} / \mathrm{d} \sim 50 \times 10^{4} \mathrm{~m}^{3} / \mathrm{d}$  天然气、油田气分离液化设备、天然气提氨设备、合成氨尾气分离设备等。

6. 低温液体贮运设备类：包括  $1.2 \sim 1000 \mathrm{~m}^{3}$  低温液体贮槽及槽车、液态二氧化碳和液化石油气贮槽（贮罐）、槽车、气化器等。

7. 单元设备类：包括板翅式换热器、膨胀机、低温液体泵、低温粉碎机、低温阀门及各种活塞式和离心式压缩机等。

8. 低温冷藏设备类：包括低温冷藏车、保温车、气调库、冷藏集装箱等冷链设备。

# 二、发展情况及水平

1. 空气分离设备的发展与水平

60年代末，小型空气分离设备已采用分子筛净化流程。流程压力，从高中压向中低

压发展；制冷方式，由采用活塞式膨胀机向高效的透平膨胀机发展。90年代初， $300\mathrm{m}^3/\mathrm{h}$  等级的小型空气分离设备引用大型全低压空分设备的技术，采用了全低压的增压膨胀流程，这是小型空分设备的一项重大技术进步。

中大型空分设备经历了从全低压切换管式和板翅式流程、常温分子筛净化流程，直至目前带增压膨胀的分子筛净化流程的发展过程，而且分子筛净化一增压膨胀流程这一新技术已普遍得到推广。目前，行业内已成功地开发了高效规整填料精馏塔、全精馏制

取精氩等新技术，并在逐步推广。

1988年我国自行设计制造成功了采用计算机集散型控制系统(DCS)，同时提取五种稀有气体的  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备（设备性能见表1），1989年设计制造成功了采用DCS控制的分子筛净化带增压膨胀流程的6000 $\mathrm{m}^3 /\mathrm{h}$  空分设备，以及1993年设计制造的 $15000\mathrm{m}^3 /\mathrm{h}$  空分设备，都按期投入了正常运转；同时完成了提取五种稀有气体的30000 $\mathrm{m}^3 /\mathrm{h}$  空分设备的设计。1993年向印度国出口的  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备，也已投入运转。

铝合金板翅式换热器是气体分离设备行业及化学工业、石油化工中很重要的设备。60年代初，杭州制氧机厂和开封空分设备厂先后生产盐浴炉钎接的铝制板翅式换热器。90年代，杭州制氧机厂、四川空分设备厂、开封空分设备厂等三家工厂先后增添了大型真空钎接炉。目前，生产铝板翅换热器最大

尺寸为  $1000 \times 1200 \times 5900 \mathrm{~mm}$ , 最高工作压力达  $8 \mathrm{MPa}$  的  $3810 \times 628 \times 918 \mathrm{~mm}$  的产品已向美国出口。

1989年颁布了空分设备国家专业标准《大中型空气分离设备技术条件》、《小型空气分离设备技术条件》，至今气体分离与液化设备行业已发布国家标准与行业标准101项，使我国气体分离与液化设备行业能按标准进行设计、制造，促进了行业产品质量的提高，推动了行业的技术进步。

# 2. 稀有气体提取设备的发展与水平

我国稀有气体制取技术始于50年代中期。1958年开始设计、试制在  $3350\mathrm{m}^3 /\mathrm{h}$  （氧）

空分设备上的  $125\mathrm{m}^3 /\mathrm{h}$  制氩设备和0.042 $\mathbf{m}^{3} / \mathbf{h}$  氦氩提取设备，实际运转可获得  $60\sim$ $70\mathrm{m}^3 /\mathrm{h}$  、纯度为  $99.99\%$  的精氩，和纯度 $99.99\%$  的氮  $79.9\mathrm{m}^3 /\mathrm{a}$  及纯度  $99.99\%$  的氩 $6.5\mathrm{m}^3 /\mathrm{a}_{\circ}$

60年代初继续研制成功配于  $3350\mathrm{m}^3 /\mathrm{h}$  （氧）空分设备、制取纯度  $99.5\%$  的氮气135 $\mathbf{L} / \mathbf{h}$  和纯度  $99.5\%$  的氦气  $45\mathrm{L} / \mathrm{h}$  的提取设备，实现了在  $3350\mathrm{m}^3 /\mathrm{h}$  空分设备上制取五种稀有气体的任务。同时在  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备上制取纯度  $99.99\%$  的精氩  $45\mathrm{m}^3 /\mathrm{h}$  也获成功。氦气从天然气  $2\times 10^{4}\mathrm{m}^{3} / \mathrm{a}$  的设备上制取纯度为  $99.995\%$  的纯氮，形成生产能力达 $7\times 10^{4}\mathrm{m}^{3} / \mathrm{a}_{\circ}$  另外，从10万t/a合成氨设备尾气中提取纯度  $99.99\%$  精氩  $20\mathrm{m}^{3} / \mathrm{h}$  ；在30万t/a合成氨装置的尾气中回收氢气  $250\sim$ $300\text{万}\mathrm{m}^{3} / \mathrm{a}$  及纯度  $99.999\%$  精氩  $420\mathrm{m}^{3} / \mathrm{h},$  回收氢一氮混合气，增加合成氨  $50\sim 60\mathrm{t / d},$  也获成功。

70年代实现了在小型空分设备上用分子筛  $(5\mathrm{A} + 4\mathrm{A})$  ，在90K下用吸附法一次净化粗氩，获得纯度为  $99.99\%$  以上的精氩。

随着提氩技术的完善与发展，分别在 $6000\mathrm{m}^3 /\mathrm{h}$  和  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备上配置4XFS=45型和XKAr-120型提氩设备，并投入批量生产。这两种设备都采用加氢催化法除去粗氩中的氧来制取精氩。

80年代在消化引进技术的基础上，自行设计了XKAr-190型提氙设备，在技术上比XKAr-120型有了较大的提高。

80年代，成功地用大型色谱法分离  $\mathbf{O}_2-$ $\mathrm{N}_{2}-\mathrm{Kr}-\mathrm{Xe}$  混合气以制取氮和氩。从精氮 $[50\% (\mathrm{Kr} + \mathrm{Xe})]$  和粗氩（  $98\% \sim 99\%$  Xe）两种原料中分别制得纯度  $99.995\%$  的氮和氩的产品，提取率大于  $80\%$  。

90年代初以高提取率实现了  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备上制取五种稀有气体(见表1)。同期完成了  $30000\mathrm{m}^3 /\mathrm{h}$  大型空分设备上提取五

种稀有气体(见表1)的设计任务。从产品提取率来看，比80年代末设计的有较大的提高。1996年1月，在  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备上试验无氢全精馏制氩试车成功。这些都说明我国稀有气体提取设备的设计制造技术已达到世界先进水平。

目前，我国从事稀有气体生产和研究的单位有六家。生产的稀有气体基本满足国内科研和工业等需要，还有一定数量出口。

# 3. 天然气分离液化设备发展与水平

中国天然气分离液化设备起步迟，生产的装备水平相对较低，用户主要依靠进口。我国使用的天然气分离液化设备（轻烃回收设备）共有90套，国内目前能生产日处理量 $3\sim 50\times 10^{4}\mathrm{m}^{3}$  的天然气分离和液化设备。气体分离设备行业于80年代初研制成功  $7\times$ $10^{4}\mathrm{m}^{3} / \mathrm{d}$  天然气分离液化设备，80年代中期自行设计和制造的  $50\times 10^{4}\mathrm{m}^{3} / \mathrm{d}$  天然气分离液化设备运转成功，并形成  $3\sim 50\times 10^{4}\mathrm{m}^{3} / \mathrm{d}$  系列设备。设备主要制冷方式都采用膨胀机制冷，中大型设备采用带液透平膨胀机；小型设备采用撬装式气体轴承透平膨胀机。膨胀机设计和制造已达到一定的水平。设备与国外相比，显然是规模小些，应向大型化发展，并进一步提高产品回收率。

# 4. 非低温分离设备的发展与水平

近十年来，空气分离技术发生了巨变。目前吸附技术和膜分离技术在中小型制氧、制氮方面正在占据一席之地。

60年代中期开始研究变压吸附(PSA)分离技术。随着吸附剂性能提高，切换阀门寿命延长，加之设备造价低、操作方便等优势，80年代末我行业的变压吸附分离技术有了较大发展，成为小型低温空气分离设备的主要竞争对手。目前，我国已能设计和制造  $2\sim 1200\mathrm{m}^3 /\mathrm{h}$ $\mathrm{PSAO_2}$  系列制氧设备，  $5\sim 700$ $\mathrm{m^3 / hPSAN_2}$  系列制氮设备，  $100\sim 50000$ $\mathrm{m^3 / hPSAH_2}$  系列制氢设备。设备流程选用

真空解吸流程和超常压解吸流程。氧产品的回收率达  $52\%$  ，设备的单位电耗  $0.41 \sim 0.43 \mathrm{~kWh} / \mathrm{m}^{3} \mathrm{O}_{2}$  。

我国70年代开始研究膜分离技术，并推广于工业应用。用F46膜从合成氨尾气中提取氮气。80年代用三级并联分离器从合成氨尾气中分离氢气，纯度  $85\%$  以上，回收率达 $86\%$  。1992年中国建立了一条多功能、应变能力强的中空纤维膜分离器制造生产线。膜分离技术开始用于工业生产。膜分离装置操作简便、设备费用低、能耗省。当前正在研制各种选择性高的膜，开拓气体渗透的应用领域。

目前，国内已进行了膜分离法与低温分离法结合进行空气分离的研究，也正在进行变压吸附与低温精馏组合进行空气分离方面的研究。

# 三、气体分离与液化设备的发展前景

# 1. 重点发展重大技术装备实现国产化

根据冶金、化肥、乙烯、城市煤气与整体联合循环发电等工业部门发展的要求，给气体分离与液化设备行业发展带来了新的机遇，到2000年需要增加制氧能力144万  $\mathrm{m}^3 /$  h。随着工业装置的规模大型化和城市集中供氧的需要，促使空分设备的等级也不断增大，重点发展“  $1\sim 3$  万”等级，尤以“3万”等级空分设备需求量较多，还应着手研制“  $4\sim 6$  万”等级大型空分设备，实现重大技术装备的国产化；同时中小型空分设备仍有广阔的市场，要积极发展集装箱式小型空分设备；大力开发中小型液氧、液氮设备；重点开拓国际市场，扩大产品出口。

到2000年天然气产量达  $300\sim 400$  亿  $\mathbf{m}^3$  （目前159亿  $\mathbf{m}^3$  )，要大力发展天然气分离液化设备。根据大油气田需要，急待开发日处理量为  $120\sim 200$  万  $\mathrm{m}^3 /\mathrm{d}$  的天然气分离液化设备，预测需40多套；研制  $50\sim 150t / d$  天然气液化设备，以及配套的液化天然气贮槽和槽

车，每年可回收几十亿  $\mathbf{m}^3$  天然气。

# 2. 推广新工艺新技术

要把节能产品放在首位。空分设备是耗能大的设备之一，如果将空分设备的设计效率提高  $2\%$  ，年节电量就相当于5000万度；如现有空分设备的运行效率提高  $3\% \sim 5\%$  至少可节电3亿度。所以，在技术上要采用分子筛吸附净化带增压透平膨胀机流程、计算机集散型控制系统(DCS)、高智能负荷跟踪和调节控制系统的调峰型流程、全精馏制氩流程、规整填料精馏技术，提高透平压缩机的等温效率等来降低设备电耗，特别是实现大型空分设备分离氧的单位电耗降到0.42 $\mathrm{kWh / m^2O_2}$  以下，氧提取率达  $97\%$  以上和无故障运行周期达1.5～2年的目标。

# 3. 积极开展综合利用

空气分离氧和氮的同时得到的五种稀有气体（氩、氖、氮、氪、氙），在国民经济中都有很重要用途。开展30万t/a级合成氨尾气中回收氩、氢、氪、氙气体；从天然气处理中回收轻烃；回收放空烧掉的天然气等都能提高经济效益。研制用于油气田压注和增产的大型液氮设备，要求配套  $70\sim 105\mathrm{MPa}$  高压液氮泵。

# 4. 加快变压吸附和膜分离技术的研究，以及与低温法组合的研究，积极开拓市场应用领域

近期要开发  $2000\mathrm{m}^3 /\mathrm{h}$  以上规模的变压吸附制氧设备和  $1600\mathrm{m}^3 /\mathrm{h}$  以上规模的制氮设备。加速膜分离设备在空气分离中使用。

# 一、前言

当前在空分精馏设备中板式塔仍占重要的地位，但它的效率一般比较低，因此设法提高它的效率与分离能力是空分设备行业研究的一个重要课题。普通的精馏塔中，相邻塔板上的液体流动方向相反，为逆流操作；若各层塔板上的液体流动方向相同，则为并流操作。

早在1936年Lewis[1]就通过简化计算证明并流操作的塔板效率高于逆流操作。Haselden[2]进一步指出改善板式塔操作，大幅度地提高塔板效率的一个最有效的措施是将逆流操作改为并流操作。近年来出现了许多采用并流操作的精馏装置，如直流式塔板[3]、U形流塔板[4]等，但由于它们的结构比较复杂和不易操作，都没能够推广应用。1981年Jenkins[5]提出了一种新的并行精馏(Parastillation，简称并馏)过程，图1分别为单溢流塔板与双溢流塔板的并馏塔示意图。在并馏塔中不但能够实现并流操作，而且塔的结构也不复杂，操作也比较容易，所以表现出广泛的应用前景。

并馏过程与精馏过程的差别，在很大程度上是由于塔内两相流体的流动与混合状况具有不同特点所造成的。若利用小型试验塔进行研究，无法准确地反映这些差别，因为在直径很小、板数很少的试验塔中不可能真实地再现大型工业塔中流体的非理想流动与混合的情况，而制造大型的工业并馏塔用于进行可行性研究，从经济性考虑一般也不允许。所以研究并馏过程的适宜方法，是首先对其做准确的模拟计算，待充分和确切地了解了它的效益大小与影响因素之后，再建造工业规模的并馏塔作进一步的验证和推广应用的研究。

# 三、并馏过程的模拟计算

Jenkins[7]利用戊烷/己烷二元物系对具有6块圆形塔板的精馏塔和具有12块半圆形塔板的并馏塔分别进行了平衡级计算，表明并馏过程可以使精馏分离指数提高  $123\%$  ，或者使再沸器热负荷降低  $61.67\%$  。Canfield[8]设气液相恒摩尔流动，并取相对挥发度  $\alpha$  和塔板效率  $E_{MV}$  为不同的数值，对精馏与并馏过程进行了一系列对比模拟计算得到关联式(1)，表示在完成相同分离任务的条件下，并

馏过程所需要的半圆形塔板数  $N_{P}$  与精馏过程所需要的圆形塔板数  $N_{D}$  之比  $\zeta$  与塔板效率之间的关系。

$$
\zeta = \frac {N _ {P}}{N _ {D}} \approx 2 - \frac {E _ {M V}}{2} \tag {1}
$$

进而可以计算并馏过程所产生的塔板数减少率  $\pmb{\eta}$  为

$$
\eta = \frac {2 N _ {D} - N _ {P}}{2 N _ {D}} = 1 - \frac {\zeta}{2} \approx \frac {E _ {M V}}{4} \tag {2}
$$

由式(2)可知对于理论塔板  $(E_{MV} = 1)$  与普通精馏过程相比，并馏过程产生的塔板数减少率  $\eta = 25\%$  。Guerreri也对并馏过程的效益进行了一般化的模拟计算，他取点效率为一系列不同的常数，并利用Lewis[1]的关系式将点效率  $E_{OG}$  折算成精馏（逆流操作）和并馏(并流操作)过程的  $E_{MV}$  。计算结果 $\eta = 5\% \sim 23.6\%$  ，大体上符合关联式(2)。

这些计算结果表明：采用并馏过程能够在一定程度上提高精馏分离能力，降低能量消耗。但是这些计算都仅仅是对二元物系进行的，而且都基于一些过于简化的假设，既不能真实地模拟实际的工业精馏塔，也不能充分反映并馏过程区别于普通精馏过程的特点，所以由这些计算得出的并馏过程效益的数据不能直接应用于实际的工业生产。

如前所述并馏塔中气液两相流体的流动与混合具有独特的特点。所以若要对并馏过程的效益进行模拟计算，所使用的模型必须能够全面地考虑实际塔内存在的气液两相的不规则流动与不均匀混合的性质，以及在两相间发生的非理想传质的性质。能够完全满足这些要求的精馏模拟计算方法，是三维非平衡混合池模型[10]。与平衡级模型和非平衡级模型所作的塔板上的液体完全均匀混合的假设不同，三维非平衡混合池模型将塔板上的液体分割成二维分布的混合池。每个混合池的液体都按照一定的规律作纵横两个方向的流动，在各个混合池之间液体还按照一

定的规律进行涡流混合运动，因而造成不同程度的混合。同时在混合池内气液两相混合物之间，也发生着非平衡的质量与热量传递。

因此利用三维非平衡混合池模型，可以真实地模拟实际塔板上不同形式的液体流动与不同程度的液体混合给精馏或并馏分离过程带来的影响，从而能够可靠地评价并馏过程所产生的效益。

# 五、结论

# 一、前言

在天然气的利用过程中，常常需要进行长距离运输和大量贮存，液化天然气(LNG)因所占体积小而倍受青睐。但是，由于天然气的组成和温度随产地和运输时间长短而各异，从而其密度差别较大，会在低温贮槽中形成分层现象。对分层处理不当会引起涡旋(Rollover)，严重威胁贮槽的安全运行。由于分层是形成涡旋的前提条件，因此对该现象进行研究具有重要意义。本文主要对分层形成原因、分层的稳定性和破坏机理，以及分层时各个液层之间的传热和传质特性等加

# 二、分层形成原因

分层的形成主要有两种原因，其机理是不相同的，下面分别加以介绍：

1. 冲注引起的分层(Stratification)

一般，为了使LNG贮槽维持低温和避免冲注时的冷却损失，在LNG贮槽中总会留有一部分LNG。如果冲注的LNG与留下来的LNG，因成分和温度上不同而密度有较大的差异，则若冲注方式不恰当，并且在冲注后不搅拌使两者充分混合，就会发生分层现象，进一步还会造成涡旋现象的发生，对LNG贮槽的安全产生威胁。1971年发生在意大利La Spezia SNAM LNG终端站S=1贮槽的涡旋事故就是因为上述原因造成。

2. 自动分层(Autostratification)

对于初始时混合得较好的LNG，如果其中的含氮量较高(大于  $1\%$  )，也会发生一种自动分层现象。其机理是：在贮槽壁周围的LNG，由于受壁面漏热的加热作用而温度升高，密度减小，沿壁面上升到LNG的气一液表面，在表面发生蒸发。由于  $\mathrm{LN_2}$  的沸点低于LNG中的其他组分的沸点，故优先蒸发，从而使蒸发后的LNG的密度减小，无

法向下流动形成稳定的自然对流循环，经过一段时间后，就会形成一个轻质顶层，造成了液体的分层。

# 三、分层的稳定性和破坏机理

通过前面的分析，我们可以知道分层的形成是由于两层液体之间存在密度差异而造成的。对于LNG这样的多组分液体来说，其密度主要由液体温度和组分决定，因而密度的变化可以被表示为：

$$
d \rho = \frac {\partial \rho}{\partial T} d T + \frac {\partial \rho}{\partial S} d S \tag {1}
$$

上式中  $\partial \rho / \partial T$  和  $\partial \rho / \partial S$  是温度和组分的函数。但是两者在温度和组分变化不大的情况下变化较小，可以看作不变的定值。因此密度可以表示为：

$$
\rho = \rho_ {0} (1 - \alpha \Delta T + \beta \Delta S) \tag {2}
$$

式中  $\alpha = -\partial \rho /\partial T,\beta = \partial \rho /\partial S$

由式(2)可以看出温度和组分对密度的影响效果相反，因此密度的变化速率决定于两者变化引起密度变化的比，这样就可以得到一个决定分层稳定性的无量纲参数：  $R = \left|\beta \Delta S / \alpha \Delta T\right|$ ，当  $R < 1$  时，分层是不稳定的，会自动不断发生变化；当  $R = 1$  时，分层处于临界稳定状态，但很小的扰动就会破坏这种稳定，在实际情况下扰动是不可避免的，因而这种分层实际上是不能存在的；当  $R > 1$  时，分层的稳定性随  $\mathbb{R}$  的增大而增强，即抗扰动和抗破坏的能力愈来愈强。

分层的破坏，一般认为有两种原因：一种是边界层穿透引起破坏，即分层的破坏首先发生在贮槽周壁的流动边界层，下层液体边界层里流动的液体在浮力的作用下穿透两层液体间的界面，进入上层液体层，造成分层的破坏。另一种是中心射流卷携引起的破坏，即上层液体在中心处向下的射流和下层液体在中心处向下的卷携，引起了中心处的液一液界面的破坏，并且这种破坏随流动向四周扩散，造成了整个界面的破坏。实验证

明，在大型LNG贮槽中，因为贮槽的直径很大，而壁面附近的边界层的厚度很薄，该处界面的破坏很难扩散到其他部分，而中心射流卷携的影响很大，并以液体波的形式向周围传播，很快会破坏整个界面。

# 四、分层时相邻液层间的传热传质特性

在形成稳定的分层时，相邻两层的液体之间会产生一个非常薄的液一液界面。该界面的存在，对分层液体间的传热传质产生了很大的影响。有很多学者对这种双向扩散界面作了研究工作，通过分析和总结，可以得出以下结论：

$$
\frac {H _ {S}}{H} = F \left(\frac {\beta \Delta S}{\alpha \Delta T}\right) = 3. 8 \left(\frac {\beta \Delta S}{\alpha \Delta T}\right) \tag {3}
$$

$$
\begin{array}{l} \frac {\beta F _ {S}}{\alpha H _ {S}} = G \left(\frac {\beta \Delta S}{\alpha \Delta T}\right) \\ = \left\{ \begin{array}{l l} 1. 8 5 - 0. 8 5 \frac {\beta \Delta S}{\alpha \Delta T} & \frac {\beta \Delta S}{\alpha \Delta T} <   2 \\ 0. 1 5 & \frac {\beta \Delta S}{\alpha \Delta T} \geqslant 2 \end{array} \right. \tag {4} \\ \end{array}
$$

式中  $H = 0.085(g\alpha / k^2 v)^{1/3}(\Delta T)^{4/3}$

$H_{s}$  通过界面的热流率

$F_{s}$  ——通过界面的质流率

以上结果是对盐一水双组分系统的理论分析和实验验证而得到的，对LNG而言，目前尚没有类似的关联式。但是由于LNG分层和盐一水系统的分层具有物理上的相似性，可以认为对于LNG分层也存在类似的关联式，但需对LNG物性参数进行一些修正。这里的修正主要涉及Pr数和Le数，因为只有这两个参数对传热和传质有影响。

首先，考虑  $\mathbf{Pr}$  数对式(1)和式(2）的影响： $\mathbf{Pr}$  数是考虑粘性力和热传导对流动的影响，在式(1)分子和分母中都作了考虑，故可以不对式(1)作  $\mathbf{Pr}$  数的修正。同样，对(2)式也可以不作修正。

其次，考虑Le数对式(1)和式(2）的影响：对式(1)而言，主要涉及到传热问题，

LNG和盐一水的Le数都很大，这意味着传质对热流率的影响很小，可以忽略。因而虽然LNG和盐在水中的Le数相差约一倍，但不对式（1）作Le数的修正是可以的。对式(2)而言，因为传质是要考虑的因素之一，需作修正。为简单起见，假设Le数的影响是线性的，得到：

$$
\begin{array}{l} \left(\frac {\beta F _ {S}}{\alpha H _ {S}}\right) _ {L N G} = G _ {L N G} \left(\frac {\beta \Delta S}{\alpha \Delta T}\right) = \frac {L e _ {S A L T}}{L e _ {L N G}} G \\ = \left\{ \begin{array}{l l} 3. 7 - 1. 7 \frac {\beta \Delta S}{\alpha \Delta T} & \quad \frac {\beta \Delta S}{\alpha \Delta T} <   2 \\ 0. 3 & \quad \frac {\beta \Delta S}{\alpha \Delta T} \geqslant 2 \end{array} \right. \\ \end{array}
$$

从上式可以发现： $(\beta F_{S} / \alpha H_{S})_{LNG}$  的值开始随  $\beta \Delta S / \alpha \Delta T$  的增大而减小，温度对热流率和质流率的影响越来越小；当  $(\beta \Delta S / \alpha \Delta T)\geq 2$  时， $(\beta F_{S} / \alpha H_{S})_{LNG}$  的值不在变化，这在物理上可以解释为液体因温度差异而具有的潜能完全用于提升质量流，而不在对热流率有贡献，但尚没有比较精确的理论解释。

# 空压机防喘振调节阀的创新设计

韦文华

（杭州制氧机集团有限公司工程成套公司，杭州市东新路90号，310004）

摘要 介绍了防喘振调节阀的作用，原防喘振调节阀结构及使用情况的说明，新一代防喘振调节阀的结构方案确定，调节阀通径的详细计算方法等内容。经实践表明，新型防喘振调节阀性能稳定，安装方便，使用寿命长。图1参4。

主题词：空气透平压缩机组 防喘振调节阀 恒定调节 气动执行机构 软密封 调节性能

空气透平压缩机组(以下简称空压机)是空分设备中重要的部机，空压机运行的好坏直接影响成套空分设备的性能。防喘振调节阀对于空压机的正常运转来讲，是个十分重要的安全防护装置。

# 一、防喘振调节阀的作用

1. 空压机出口压力的恒定调节是防喘振调节阀的主要功能，在出口压力设定值的  $\pm 0.01\mathrm{MPa}$  范围内实现自动调节。  
2. 在空压机出口压力一旦快要进入喘振区时，实现快速自动全量放空，使空压机不能进入喘振工况，达到保护空压机的功能。  
3. 在空压机启动时，为减少空压机的启动负荷，要实现全量放空。  
4. 由于空压机总体结构上的原因，防喘振调节阀的阀体部分安装在一楼，手动操作部分安装在二楼，以实现紧急状态或事故状态下的手动操作。  
5. 其具有要求反应快、动作无误、运转周期长等特点。

由于防喘振调节阀在空压机系统中的重要性，如何实现上述功能，如何实现调节阀长周期可靠的运转，是改型设计的主要内容。

# 二、原防喘振调节阀结构及使用情况

1. 原调节阀结构松散, 安装时需解体安

装，且安装精度很难保证。由于阀体部分在一楼，执行机构在二楼，这两部分的中心同心度很难在安装时保证，因而直接影响了阀门的调节性能及密封性，并使阀门的寿命大大缩短。

2. 执行机构的作用力直接作用于空压机放空管道，在正常操作时，特别是在紧急放空时对管道的冲击力很大，使管道变形，影响密封面的结合，易出现大量泄漏。  
3. 由于空分行业日趋激烈的竞争，对空分设备的能耗、运转周期及可靠性要求越来越高，特别是空分设备DCS计算机集散控制系统的使用，对配套的调节阀要求更高。因此，原结构防喘振调节阀无论在性能上、结构上及运转的可靠性方面，都不能适应现在的要求，故有必要开发设计新一代的空压机防喘振调节系统。

# 三、新一代防喘振调节阀结构方案的确定

随着科技的不断进步，调节阀技术近年来在国际上有了突飞猛进的发展，如美国的Fisher公司、Masoneilan公司、德国的Gulde公司、日本的Koso、Yamatak-Honeywell公司等的调节阀技术，代表了当今世界的先进水平。

对于改型结构设计的方案，在调研用户的同时，参考了多家国内外厂家的结构型式。上海金山石化二厂进口空压机的防喘振调节阀，采用日本Yamatak-Hoeywell公司生产的HCB笼式双座调节阀，由于进口空压机结构与我公司的空气透平压缩机整体结构有很大区别，进口空压机为整体单层布置，因此调节阀的结构采用整体气动薄膜式双座结构。其结构简单，调节性能好，且安装方便，但这种结构无法实现阀体一楼安装、二楼手动操作的要求。针对调研情况，结合我公司实际情况，改型设计方案从以下几个方面进行：

1. 考虑到我国生产厂家生产的橡胶膜片的寿命，如采用薄膜式气动执行机构要达到连续运转两年的要求，有相当大的风险。因此气动执行机构部分采用气动活塞式执行机构，加蜗轮蜗杆手动操作部件，在无气源、无信号状态下，全部可用手动来操作。  
2. 取消原防喘振调节阀的中间加长杆部分，把气动执行机构部分与阀体部分直接连接，形成一个整体结构，方便安装。  
3. 阀体部分采用双座软密封调节阀型式，这种结构流通能力大，流体阻力小，采用软密封型式后其密封性能保证。

# 主题词：空气分离设备 钢制容器 铜制容器 焊接技术 新焊接工艺

焊接技术是空分设备制造中的一门关键的热加工技术，根据空分设备的结构特点和要求，选择合理的焊接方法，制定正确的焊接工艺，是非常重要的。

与国外先进水平相比，目前国内的焊接还很落后，无论是焊接设备，还是焊接工艺，都还有很大差距。目前国内空分设备厂所采用的焊接技术，与德国Linde公司、法国液空公司等相比较，差距也是很明显的。笔者已在“空分设备的焊接制造技术(一)”中较详细地介绍了空分设备用铝制管道、铝制容器的焊接制造技术，本文将介绍钢制容器及铜制容器的焊接制造技术。

# 一、钢制容器的焊接制造技术

空分设备中钢制产品很多，焊接产品有结构件、管道和容器等，容器多为I类压力容器，要求高，常采用全焊透方式以保证焊接质量。焊后检验通常包括外观检查、内部无损探伤、力学性能试验等。

空分设备用焊接压力容器，其材质通常为16MnR一类的低合金结构钢和1Cr18Ni-9Ti类的奥氏体不锈钢，材料本身的焊接性比较好，并无太大的困难。但是，各容器有

其结构特点，所以在焊接方法的选定和焊接工艺的制定方面，我们还必须充分考虑到产品的结构焊接性。有些产品的材料焊接性是很好的，无论是焊接缺陷的控制，还是保证焊接接头的力学性能，都没问题。但由于产品结构的限制，实际焊接问题很多，如缺陷超标、力学性能达不到要求等等，这就是产品的结构焊接性的具体体现。

# 二、钢制容器的焊接制造技术

$150\mathrm{m}^{3}/\mathrm{h}$  及  $150\mathrm{m}^{3}/\mathrm{h}$  以下规格的小型空分设备用焊接产品很多，分馏塔、氩塔等皆是由纯铜T2或黄铜H62制成的焊接容器，要求比较高，焊接相对来说，也比较困难。特别是黄铜H62的熔化焊焊接性很差，实际生产中常常出现质量问题。火焰锡钎焊在T2或H62的产品焊接上应用较为广泛，由于工艺比较简单，此处不作介绍，这里主要介绍T2、H62制容器的熔化焊技术。

铜及铜合金的导热系数大，焊接时可能导致母材与填充金属难以熔合，即产生焊不透。另外，铜及铜合金线胀系数和收缩率比较大，自由状态下焊接时变形大，若工件刚度大，则会导致很大的焊接应力，增大裂纹倾向。空分设备用T2制成的容器由于板较薄，现多用TIG焊焊接。为减少钨极烧损，保证电弧稳定和有足够的熔深，通常采用直流正接。由于T2对氧、氢很敏感，为抑止气孔，焊前应将工件和焊丝的表面清理干净。另外，常选用含有脱氧剂的焊丝，如HSCuSn，可降低焊缝气孔倾向，从而获得致密的焊缝。常见的焊接坡口见图1所示，一般来说，T2单面焊双面成形是比较困难的，象空分设备所用的T2制容器直径较小，难以双面焊，这就要求在单面焊时要控制规范，正确操作。

# 三、新的焊接工艺简介

# 1. H62等离子弧焊接工艺

氧乙炔气焊焊接H62，虽然具有设备简单、操作较方便的特点，在一定程度上可抑止锌的挥发，但气焊热源热量不集中，焊后变形大。焊缝和近缝区容易产生粗晶组织和其它缺陷，生产效率低，质量不稳定，焊工劳动条件差（锌的挥发会在空气中形成白色烟雾的氧化锌，能引起焊工的慢性中毒）。所以，H62的氧乙炔气焊并不是一种理想的焊接方法(工艺)。

自动的等离子弧焊与气焊相比，具有一系列优点。通常，板厚  $6\mathrm{mm}$  以下的H62对接接头可不预热，不留间隙，不开坡口，一次焊接成型(单面焊双面成形)，效率高。且由于等离子弧焊热源热量集中，热影响区窄，快速焊，所以锌的挥发并不严重，焊接变形

小，焊接应力不大，热裂倾向小。焊接时，若不加焊丝，正面焊缝稍有下凹；若配以自动送丝机构，焊缝正面下凹问题即可得以解决。笔者曾进行  $4\mathrm{mm}$  厚H62平板对接焊接试验，不填焊丝，接头拉伸试验时断裂在热影响区上，抗拉强度值与母材相近；冷弯试验时冷弯角达  $180^{\circ}$ ，而未出现裂纹。

容器的纵缝等离子弧焊易实现，对于环缝来说，相对困难一些，但由于等离子弧对坡口参数波动即装配质量并不太敏感，所以只要配以合适的工装夹具即可实现容器纵、环缝的高质量焊接，若再配以熔透自适应控制装置，即能进行焊接质量的自动控制。总之，H62的等离子弧焊是一种比较成熟、实用的焊接工艺。该工艺同样适合于T2制容器的焊接。

# 2. 奥氏体不锈钢等离子弧焊工艺

前文已述，空分设备用不锈钢制容器对接焊缝，一般采用埋弧自动焊（当板厚  $\delta \geqslant 8$  mm)，或手工电弧焊(当板厚  $\delta < 10\mathrm{mm}$  )，双面焊。实际上对于板厚  $\delta \leqslant 8\mathrm{mm}$  的不锈钢可采用等离子弧焊单面焊双面成形工艺，不开坡口，不留间隙，一次焊接成型，与手工电弧焊相比，效率高、质量好(特别是焊缝外观成形美观)。等离子弧焊与埋弧自动焊相配合，可实现空分设备用不锈钢制焊接容器纵、环缝的自动焊接，淘汰手工电弧焊这一较落后工艺。等离子弧焊也很适合于不锈钢管道的焊接。

视产品具体要求，可配自动送丝机构、熔透自适应控制装置等。一般来讲，容器纵焊缝可配工装夹具（当板较厚时也可不配夹具）进行焊接，环缝只要简单的工装和滚轮架即可进行焊接。自动等离子弧焊与下文所介绍的自动TIG焊相比，效率更高。

目前，这一成熟工艺在许多行业已得到成功应用，笔者也曾为湖南邵阳某纺织机械厂开发应用带熔透适应控制的等离子弧自动

焊接，焊接  $3\sim 8\mathrm{mm}$  厚工类不锈钢压力容器，应用效果良好。

# 3. 奥氏体不锈钢自动TIG焊工艺

钨极氩弧焊电弧燃烧十分稳定，且氩气是惰性气体，与熔化金属不起反应，不会造成合金元素的氧化烧损，氩气也不溶于液态金属而生成气孔。故氩气的保护性能十分优良，焊缝质量很高，配有送丝机构的自动钨极氩弧焊，对于焊接奥氏体不锈钢容器纵、环缝是十分有利的。德国Linde公司在焊接低温液体贮槽内旦时，就是采用此工艺。由于所得焊缝金属杂质很低，故此对接头韧度的提高是有利的。另外，由于是自动焊，所以焊缝外观成形非常好。

自动TIG焊焊容器纵、环缝时，一般都采用单面焊双面成形，为保证背面焊缝成形良好，通常背面也要加氩保护。由于该工艺一般适合于较薄产品的焊接，所以纵缝要配以合适的夹具、环缝配以合适的辅助工装予以施焊，同时对焊接装配质量要求也较高。

# 4. 富氯混合气体保护自动焊工艺

纯氩熔化极氩弧焊焊接不锈钢及低合金钢时，存在所谓“阴极飘移”现象，会引起焊缝熔深及焊缝成形的不规则，同时焊缝易产生气孔，所以纯氩的熔化极氩弧焊焊不锈钢及低合金钢并不合适。为改善焊缝成形，促进金属熔滴过渡，提高电弧稳定性及接头质量，现多采用氩一二氧化碳一氧三元混合气熔化极焊，来焊接板厚  $10\mathrm{mm}$  以下不锈钢或16MnR一类的容器纵、环缝。前文介绍的是半自动的混合气体保护焊，这里简要介绍自动焊。

富氩三元气保护焊单面焊双面成型很容易，背面不通保护气，成形十分优良。与TIG焊相比，焊不锈钢要方便。但由于焊接时，焊丝必须左右摆动，所以自动富氩三元气保护焊时必须要配备焊炬自动摆动机构，才能得到优良的焊接质量。

# 5. 铝合金交流方波等离子弧焊工艺

通常板厚  $10\mathrm{mm}$  以下的铝合金对接焊缝，自动MIG焊与交流手工TIG焊相比，并没多大的优势。单面焊双面成型需加垫板，配工装夹具；双面焊时为避免烧穿和保证接头性能，要选较细的焊丝，虽焊缝外观成形好，但高效率的特点并没体现出来。所以，在板厚  $10\mathrm{mm}$  以下的铝合金容器焊接时，很少采用自动MIG焊，而多采用交流手工TIG双人双面同时向上立焊工艺。为解决板厚  $10\mathrm{mm}$  以下铝合金容器纵、环焊缝自动焊问题，交流方波等离子弧焊就是一种很好的方法，可实现单面焊双面成形，不开坡口一次焊接成型，接头质量高，生产效率也很高，焊缝成形美观。

# 四、结束语

对空分设备用焊接产品，如何根据空分设备的特点和要求，选取比较合理的焊接方法和制定合理的焊接工艺是非常重要的。目前国内空分设备焊接制造还是以传统的手工焊（包括涂料焊条手弧焊、手工钨极氩弧焊和氧一乙炔气焊)为主，手工焊工作量大，约占整个焊接工作量的  $70\% \sim 80\%$  ，与国外先进工业国的自动、半自动焊接比例相当。所以国内空分设备制造厂家在如何提高焊接质量，提高焊接半自动、自动化水平上，尚有许多工作要做，我们应当不断应用一些成熟的新工艺、新技术，以缩小与国外先进焊接水平的差距。

# 缩短  $1500 \mathrm{~m}^{3} / \mathrm{h}$  制氧机启动时间的操作

顾德广

（南京第二钢铁厂制氧车间，江苏南京中华门外西善桥，210041)

摘要 介绍作者缩短  $1500\mathrm{m}^3 /\mathrm{h}$  制氧机启动时间的亲身体会，主要是开车前大加温一定要彻底、中部温度不得低于  $-110^{\circ}C$  、在积液时要尽量降低机后温度、积液操作不宜心急、主冷开始工作后要适时调整精馏工况。

主题词：中型空气分离设备 大加温 膨胀机 中部温度 机后温度 积液调纯

我车间有一套开封空分设备厂制造的KDON-1500/1500-5型制氧机，经过一年多的生产运行，该机性能稳定，氧气纯度  $\geqslant$ $99.6\%$  ，产量可达  $1650~\mathrm{m^3 / h}$  ；氮气纯度  $\geqslant$ $99.99\%$  ，产量  $\geqslant 1500\mathrm{m^3 / h_o}$

该制氧机经过几次原始开机，就如何缩短启动时间，我认为需要注意以下几点。

# 一、开机前的大加温一定要彻底

大加温结果的好坏，直接影响到下面的开机是否顺利。加温一定要保证各个单元干燥，任一个吹除阀都得畅通无阻，出口没有水分。有一次，我们在污氮进板式前的吹除阀还没有通的情况下就急于开机，最后导致上塔压力偏高(达0.066MPa)、污氮气量少、氧气纯度只有98.5%，最终反而影响了生产。

# 二、缩短启动时间需要注意的问题

该机使用说明书上规定，从启动第一台膨胀机到主冷液面达到规定值需40小时。一般情况下，主要是第四阶段需要的时间长，前面三个阶段只要充分发挥膨胀机的制冷效率，12个小时可以结束，其中第二阶段要尽量提高机前温度，充分发挥膨胀机潜力，做到高温高焓降。为此，机前温度调节阀TG-104最好不开，就靠开环流与关启动短路旁

通阀TG-102来操作，在保证冷端温度不回升的前提下，尽量提高机前温度，但切忌操作阀门太猛，避免温度回升太快。

缩短启动时间，实际上就是缩短第四阶段的时间，该阶段的任务是继续深冷设备，积累液体，调纯送气。这一阶段操作好坏，直接影响到送氧时间，所以这一阶段，要着重注意以下几个问题：

1. 中部温度在任何情况下都不得低于  $-110^{\circ} \mathrm{C}$

因为中部温度过低，也就是板式过冷，膨胀机产生的冷量转移到板式内，没能在塔内积累起来，这属于冷量分配不合理的问题。其原因是由于开两台膨胀机，膨胀量必然较大，这样环流量相对来说也较大，如果关小环流量，势必造成旁通量偏大，导致机前温度偏低，膨胀机过冷，不得已采用机前节流等方法，同时进塔气量又偏少。例如，我们在1996年5月13日开机的情况，机前温度 $-155^{\circ}C$ ，机后温度 $-180^{\circ}C$ ，中部温度为 $-145^{\circ}C$ ，环流温度为 $-145^{\circ}C$ ，这样无疑影响了积液时间，延长了开机时间。

根据这种现象，我们采取了停一台膨胀机，增加进塔气量的措施。为此，全开JL

-101～103阀，开大氧、氮气阀门，让更多的空气通过板翅式换热器，同时开足一台膨胀机。处理后，机前、中部、环流这三个温度回升显著，3个小时左右的时间，温度就正常了。这里还要特别提醒一点，开两台膨胀机的制冷量，未必比开一台的多，这要从膨胀量、前后压力、机前温度等多方面去考虑。待温度正常后，笔者曾试图再启动另一台膨胀机，但中部温度马上就下降，工况不易调整，只能开一台膨胀机，并设法使它满负荷运转。这是该型制氧机的特点，我们要全力为膨胀机创造产冷条件，以充分发挥膨胀机的效率。

2. 在积累液体过程中，要尽量降低机后温度。

主要目的是提高液化器中两流体的温差，强化换热，增加液化量，在保证机后不出现液体的前提下，最大限度地降低膨胀机机后温度，我们一般控制在  $-185\sim -190^{\circ}C$  之间(这与机后压力有关)，以达到缩短积液时间。

3. 主冷积液时要正确认识液面上涨慢的原因，不宜心急

在积液时，尤其是当主冷已经开始工作

时，这时候液面上升很慢，有时甚至好几个小时液面都不涨，这是什么原因呢？膨胀机产生的冷量到哪里去了呢？

主冷开始工作，说明精馏工况已经慢慢地建立，实际上膨胀机产生的冷量已经转移到塔板上了。这点可以从阻力表上反映出来，上、下塔各部分的阻力在渐渐地上升，主冷液面虽然没涨，但塔板上的液体已越来越多，精馏工况正在逐步建立，氧、氮气纯度正在逐渐变好。

4. 主冷开始工作, 要适时调整, 积累液体

当主冷液面超过  $550\mathrm{mm}$  时，主冷已经开始工作，实际上这时可以开一台膨胀机，并使它满负荷工作，同时将膨胀空气直接送入上塔，设法提高膨胀机前温度，充分发挥膨胀机的制冷能力。并且逐步地调整精馏工况，经过这样处理，主冷液面虽然没达到规定值，但氧气纯度已经很好了。我们在主冷液面为 $800\mathrm{mm}$  时，氧气纯度已经在  $99\%$  以上了，就开始向用户送氧。

在整个开机过程中，只要注意以上几点，30几个小时就可以出氧，这是我们  $1500\mathrm{m}^3 /\mathrm{h}$  制氧机冷启动的实践体会。

（1996年6月5日收稿）

# 膨胀机飞渡  $\mathrm{CO}_{2}$  冻结阶段工艺跳车简析

陈明

（浩良河化肥厂空分车间，黑龙江伊春市南岔区，153103）

摘要指出了透平膨胀机在可逆式换热器飞渡  $\mathrm{CO}_{2}$  冻结阶段末期工艺跳车的原因，是膨胀机负荷(流量)增大，间隙压差达到仅控联锁值；详述了避免膨胀机跳车的4条具体操作措施。

# 主题词：透平膨胀机 二氧化碳 冻结阶段 跳车 分析

我车间3*空分设备系杭氧KDON=6000/6600=Ⅱ型产品，其配套膨胀机组为PLPK-83.3/4.7-0.4型可调喷嘴、电机制动的透平膨胀机。自1990年9月安装投运至1994年的5次冷开车过程中，两台膨胀机在可逆式换热器飞渡  $\mathrm{CO}_{2}$  冻结阶段的末期，均出现次数不同的跳车。尤其是1994年的几次跳车，已严重地影响到开车的顺利进行。

为此，我们对膨胀机仪控设计的4个联锁停车项目，仔细查找原因，在逐一排除了油压过低、轴承温度过高和超转速的三个因素后，确认是膨胀机运行中间隙压差过大所致。1995年我们针对这一原因，对工艺操作制定了一系列的防范措施，从而确保了两台膨胀机的稳定运行。

# 一、膨胀机跳车原因的简单分析

我们知道，膨胀机的调节是通过转动叶片倾斜角度，来改变喷嘴总的流通面积，从而改变膨胀量。装置启动初期，膨胀机进气温度远远高于设计工况，启动初、末期的膨胀工质密度相差很大。因此，装置开车初期，膨胀机质量流量小于正常工况，并影响装置的开车时间。所以，在操作中应根据进气温度不同，通过改变喷嘴的通流面积，来逐步达到设计膨胀量。

具体的讲，在开车第二阶段的末期，为

使机后温度不低于  $-135\%$  ，机前温度一般控制在  $-85\%$  左右。开车转入第三阶段后，由于操作流程的改变，机前温度由  $-85\%$  一下降到  $-130\%$  ，此阶段结束时，机前温度得降到  $-160\%$  。自始至终机前温度与第二阶段末期相比，分别相差  $50\%$  和  $80\%$  。该阶段如在操作中不逐渐缩小喷嘴开度，那么随着进气温度的不断降低，膨胀空气的密度也在不断增大，其间隙压力必然升高，膨胀机负荷即流量也就随之增大。当间隙压差达到仪控联锁值时，膨胀机也就随即跳车。

总之，设置膨胀机间隙压差，不仅是保证推力轴承不受超负荷的轴向力影响，同时也是膨胀机负荷与工作性能的标志。

通过开车实践验证及有关资料介绍，可得出以下结论：膨胀机的负荷即质量流量，与喷嘴流通面积成正比。随着进气温度的降低，质量流量随之增大，相应的间隙压力亦会增加。为控制间隙压力数值，此时应逐步减小喷嘴开度，维持质量流量不变。

# 二、避免膨胀机跳车的措施

1. 在开车的全过程中，要始终保持两台膨胀机的间隙压力在  $0.19 \sim 0.21 \mathrm{MPa}$  之间操作（间隙压力与出口压力的压差联锁停车值为  $0.20 \mathrm{MPa}$ ），把这个工艺参数作为重点监控对象。这样，既能发挥膨胀机的最大制

冷量，又可加快开车进度。

2. 在开车的第一、二阶段，膨胀机机前温度不恒定时，要每小时检查一次间隙压力。  
3. 在开车进入第三阶段时，要指派专人到机房仪表盘处，密切注视间隙压力，膨胀机进气温度大幅度降低，要通过逐步缩小喷嘴开度来控制间隙压力的上涨。  
4. 在开车中, 如认为可以增加膨胀量,

操作人员必须到机房仪表盘处，根据膨胀机间隙压力值去增加，决不可盲目增加。

通过以上4条措施，我们在操作当中，根据开车各阶段膨胀机进气温度的不同，及时合理地调节喷嘴的开度，使膨胀机的间隙压力在较高负荷区间工作的同时，又保证了膨胀机的安全稳定运行。

(1996年8月13日收稿)

# 厂厂协作改造 一台氧压机顶两台用

刘昌驿 王康正

（云南维尼纶股份有限公司水气分厂空分空压站，云南省曲靖市花山镇，655038）

摘要 介绍云南维尼纶股份有限公司水气分厂，在浙江舟山定海气体压缩机制造厂的协作下，对0328氧压机运用0345A氧压机技术，进行了高压改低压的改造，结果使氧气流量由原来的  $100\mathrm{m}^3 /\mathrm{h}$  增加到  $190~\mathrm{m}^3 /\mathrm{h}$  ，降低了运行成本。

主题词：氧气压缩机 技术改造 增加流量 降低成本

我们云维股份有限公司空分空压站有两套杭氧产  $150\mathrm{m}^3 /\mathrm{h}$  制氧机，过去每月向有机分厂管道输送氧气  $5\sim 7$  天，进行聚乙烯醇生产工艺中的乙醛氧化，回收部分醋酸，其余氧气充瓶或放空。为此配置了一台2-2.833/150型（杭氧工厂代号为0328M）、两台2-1.67/150型（杭氧工厂代号0328N）氧气压缩机。随着生产的发展，1992年我公司投资建成了一条醋酸生产线，氧气需要量大大增加，每天要管道输送  $250\sim 300\mathrm{m}^3 /\mathrm{h}$  氧气。为此，1993年我们又新增了两台  $100\mathrm{m}^3 /\mathrm{h}$  氧气压缩机。

建厂时，氧压机选型为  $15\mathrm{MPa}$  充瓶用高压氧压机，后为供醋酸生产用，每次管道输送供氧时，氧气都是经过三级压缩至  $10\mathrm{MPa}$  左右送入管道，再降压至  $0.35\mathrm{MPa}$  供原料分厂使用。为减少运行费用，避免“增压一降压”使用的浪费现象，我们从1993年开始，将原来的三台0328氧压机的三级活塞及活塞杆抽除，两级压缩后直接送入管道，改为供原料分厂专用的中压氧压机。这样，虽增加了氧气管输流量，降低了醋酸生产成本，但还是不能很好满足醋酸生产的需要。

1994年，在云南省空分气体行业协会召开的年会上，我们了解到浙江舟山定海气体

压缩机制造厂是杭氧集团的成员厂，专业生产2-1.67/150型氧压机，于是在会议期间我们与定海气体压缩机制造厂的赴会代表，共同探讨了氧压机的技改设想，根据我们的生产工艺要求，提出用定海气体压缩机制造厂目前生产的工厂代号为0345A氧压机（也为1.67型，系在0328N上再改进)技术，将我厂使用的0328氧压机改造为有两个一级吸入和排出的氧压机，双方达成初步的合作意向。

1995年8月，我们利用厂部大检修的机会，对分馏塔至氧压机的管道进行了改造，利用原管架上的废弃管道DN200钢管，作为进入贮气囊的输送管，采用  $\mathrm{DN}300\times 3$  不锈钢管作为进入五台氧压机的主管道，改后效果理想：改造前开三台氧压机输氧量只能达到  $210\mathrm{m^3 / h}$  ，改造后两台氧压机输氧量就能达到  $230\mathrm{m^3 / h}$  ，彻底解决了因管径小吸气受阻和常有锈块吸入氧压机缸体内的现象，同时也为下一步对高压氧压机改造成大流量低压氧压机作好了前期准备工作。

1995年11月，我们与定海气体压缩机制造厂正式签订了改造协议，1996年3月，定海厂就按我们的要求，为我厂提供了0328氧压机中间座以上的全套技改配件，在原二级气缸的位置上，改造为  $\phi 285\mathrm{mm}$  内径的一级

气缸，两个一级缸有各自独立的进气口与出气口。

# 氟利昂蒸发器的技术改进

苏志伟 孙运超 张德清

（哈尔滨制氧机厂设计处，哈尔滨市南岗区哈西大街109号 150080）

摘要针对  $350\mathrm{m}^3 /\mathrm{h}$  制氧机、  $800\mathrm{m}^3 /\mathrm{h}$  制氮设备配套的氟利昂蒸发器换热不良、空气出口温度过高、冷冻机频繁开停等问题进行了分析，制订了相应的技术改进措施。通过改进及运行实践证明，改造是成功的，并取得了可观的经济效益。图2。

主题词：小型空气分离设备 氟里昂 蒸发器 换热效果 改进

# 一、问题的出现

我厂生产的  $\mathrm{KDON} = 350 / 700$  型及KDN  $= 800 / 40\mathrm{Y}$  型空分设备，都采用空气预冷纯化系统。该系统中氟利昂蒸发器的设计参数要求：空气入口温度为  $40^{\circ}\mathrm{C}$ ，压力为  $0.56\mathrm{MPa}$ ，水分含量为饱和状态，出口温度为  $8^{\circ}\mathrm{C}$ 。在夏季时，空气出蒸发器的温度，一般在  $12\sim -20^{\circ}\mathrm{C}$ ，有时甚至达到  $25^{\circ}\mathrm{C}$  以上，未达到设计要求。我们认为，氟利昂制冷工质在蒸发器内蒸发不完全，导致空气出口温度高，含水量大，使纯化器负荷加大，空气经过分子筛吸附后的温升也增大。这样，影响了成套空分设备的运转周期和经济指标。

# 二、原因分析

# 1. 氟利昂侧换热不充分

换热不充分的主要原因，是氟利昂工质在蒸发器入口管板上液体分配不均匀。由于蒸发器内氟利昂进口管径较小，管子数量又较多，氟利昂液体只靠一个弓形孔板来分配（如图1），使管板中心几根换热管与边缘换热管内氟利昂流量差别较大，偏流严重，造成换热器内氟利昂工质换热不充分。

# 2. 空气流向设计不合理

由于考虑回油问题，在立式氟利昂蒸发器设计上一般均采用氟利昂液体上进下出方式。若按传统的逆流换热方式，则空气采用

![](images/8c3c600d54c8c980c2599ba5c4ba4dd5e6204f38694f186c1466d6efe556f40a.jpg)  
图1 原蒸发器入口弓形孔板示意

下进上出。在夏季空气相对湿度较大地区，空气经冷却后会析出大量水分，形成液膜。同时，空气与水的流动方向相反，将阻碍水向下流动，使液膜增厚。这样，水极易在换热管壁上冻结成冰，冷量不易传递，热阻急剧增加，严重影响传热效果。

所以，空气出口温度急剧增高，使进冷冻机的氟利昂工质温度偏低，冷冻机被迫停车。每当冰融化后，才能恢复原工况。如此周而复始，冷冻机频繁开停。但若将空气流向改为上进下出（蒸发器采用顺流换热），则上述现象将得到很大改善。

# 三、改进措施

根据上述分析，我们采取如下改进措施：

1. 将原管箱法兰中心设计成  $90^{\circ}$  锥体

（如图2），取代弓形孔板，以解决氟利昂分配不均问题。当氟利昂进入管箱内，直接注入锥体顶部，然后顺锥体均匀地流入各换热管内，使得空气与氟利昂进行充分的热交换。

![](images/d47e744a08a7f6fc6467befbbf7a6ca2e174e4737ed5c2b6d993bc82d2986202.jpg)  
图2 将管辖法兰中心改成  $90^{\circ}$  锥体示意图

2. 改变空气流动方向，将进出口管方向对换。

四、改后效果

在对出口巴基斯坦 KDN=800/40Y型制氮设备进行厂内试车时，我们进行了上述整改。由于当时气温、水温偏低，大气相对湿度低，无法直接验证改进效果。因此我们模拟夏季条件操作，方法是减少空压机末级冷却器冷却水量，使出冷却器空气温度在  $38\sim 42^{\circ}C$  ，同时观察水分离器能放出水，证明空气含水量处于饱和状态。

空气出蒸发器温度为  $7\sim 10^{\circ}\mathrm{C}$  ，大大减轻了纯化器负荷，降低了空气入主换热器温度。两年多的运行表明，成套设备运转周期由原来的  $5\sim 7$  个月延长至  $10\sim 12$  个月。通过氟利昂蒸发器的技术改进，取得了可观的经济效益。

（1996年11月7日收稿）

（3）随着气量减少，塔内操作压力有所

下降，操作上要注意调整，氧气产量会相应减少，此时要尽量提高氮平均纯度，以提高氧的提取率。

（4）在空压机负荷不增加的情况下，精心调整操作，在保证氧纯度的前提下，尽量提高氧气产量，提高氧的提取率。  
（5）加强生产调度及公司总调，并与用户单位联系，合理调整生产结构。

采取上述方案，在不影响冶炼用氧的前提下， $2^{*}$  空分机组的产量降低了，单耗有所上升，但总的电耗降低，同时通过产量的调整，氧气利用率提高，达到了预期节能降耗的目的。

本方案1996年5月份实行，效果如下： $1\sim 4$  月平均氧气利用率为  $76.58\%$  ，平均电耗为1597140.5kWh/月。1996年5月实施本方案后，氧气利用率达  $90.24\%$  ，提高 $13.66\%$  ；电耗为1559127kWh/月，降低 $38013.5\mathrm{kWh}/$  月。节支15205.4元/月，全年可节支182464.80元。

# 高压充氧橡胶软管三次燃爆事故分析

叶必楠

（杭州制氧机集团有限公司工程成套公司，杭州艮山门外东新路90号，310004）

上海某制氧厂于1996年8月25日、9月15日、11月9日发生三次充氧台充氧橡皮软管燃烧爆炸事故。其中11月9日4时15分发生的燃烧爆炸事故，笔者正在该厂，目睹燃烧现场，并参加了事故分析会。

# 一、三次事故发生的经过

据厂方有关负责人介绍，8月25日、9月15日这两次事故情况相似，都是发生在充氧排上轧上“空瓶”，在打开第二只氧气瓶阀时，突然橡胶软管燃烧爆炸。此时充氧排总阀是关闭的，其余十只氧气瓶的瓶阀也是关闭的。

11月9日下午4时15分，突然听到“啪”一声巨响，笔者即意识到充氧台出事了，连忙奔向事故现场，只见充氧台灰尘弥雾，六根充氧橡胶软管尚在燃烧，余气还在喷出，先切断氧气瓶气源，用灭火机扑灭燃烧物。几分钟后两辆救火车已赶至现场。问充氧台操作工，充氧总阀处于关闭状态，充氧排管上装有11只医用氧气瓶，是在同时打开第10、11只氧气瓶阀时，发生燃烧爆炸。

# 二、现场调查情况

该厂的充氧台是由杭氧1995年提供的，充氧管配用北方某厂的高压橡胶充氧软管，1996年初安装，安装前的管道阀门都经严格的四氯化碳去油，新设备1996年1月投产使用。该厂的氧气是  $1000\mathrm{m}^3 /\mathrm{h}$  空分设备提取的，氧压机采用杭氧提供的2Z2-3/165-1型无润滑氧压机，整个氧气生产工艺过程都是无油的。

检查充氧阀门，管道无任何油迹。氧瓶阀门上也无油脂迹痕。

检查第11只氧气瓶内气体，都是氧气，没有发现可燃气体的成分。

拆下燃烧过的充氧橡胶管，锯开内管，取下橡胶，用打火机点着就燃烧。

# 三、事故原因分析

燃烧必须具备可燃物、助燃物和引燃能量三个要素，并在同时存在的条件下才能产生。充氧过程中氧气始终存在，就具备一个助燃物的条件；充氧橡胶软管内层是橡胶，用明火点着就能燃烧，那么就具备燃烧物条件；只具备这两个条件是不会燃烧的，那么引燃能量是哪里来的呢？

三次燃烧都发生在均压时，很有可能用户把满瓶当空瓶来灌充，若是  $10\mathrm{MPa}$  以上的氧瓶，瓶阀迅速打开，就高速流向空瓶。高流速的氧气与橡胶管激烈摩擦会产生摩擦热或静电，当达到引燃能量时就会产生燃烧。燃烧的瞬间压力剧烈上升，就引起爆炸。

# 四、事故教训及措施

三次爆炸幸亏没有造成严重的人伤事故，工人轻微受伤，但对工人的心里压力很大，经济损失不小，就仅第三次出动了两辆消防车，就得罚款2万元。若是爆炸管子打倒人体上，不死就是重伤，有关单位应引以为戒。防止的措施如下：

1. 充氧前应严格检查每只空瓶，若发现满瓶应不得与空瓶一起充灌，应作特殊处理，即放空后再充，或充氧压力达接近瓶内

压力再充。  
2. 充氧排空瓶装好后均压时，开启瓶阀要缓慢，听到气流声音过快时，应关小瓶阀，减少流速或拆下瓶子检查。  
3. 充氧橡胶软管不宜充氧，要更换为

金属管。对配套厂选用橡胶软管时要慎重；对橡胶充氧管制造厂推销产品时要注意灌充介质；对用橡胶软管灌充氧气的单位应引以为戒。

（1996年11月28日收稿）

# 分馏塔加温系统爆管事故分析

刘文玲 赵云海

（淄博化纤总厂动力分厂，山东省淄博市，255040）

# 一、事故经过

1996年12月10日，我厂  $\mathrm{KDN} = 3000$  型分馏塔Ⅱ号液空吸附器进口阀支撑架，因阀漏冷而被冻断裂。检修人员连忙进行焊接修复，在焊接快要结束时，突然一声巨响，只见保温材料满天飞，大量的液空从塔侧喷出。跑去一看，发现分馏塔加温管路进口总阀(DN125)前后弯头处整个断裂掉下，液空正从加温管断裂处喷出，操作人员立即关闭热空-21阀、液空吸附器进口阀通空-21，开大吹空阀。此时，主控室显示液空吸附器（Ⅱ）压力为  $0.5\mathrm{MPa}$ ，塔上其余各参数指示正常。从断裂部位看，焊接处没有焊透，坡口未打。

# 二、原因分析

1. 液空吸附器切换后，液空吸附器-Ⅱ已加温16小时，此时通知检修人员对液空吸附器进口阀支撑架断裂处进行焊接。因阀支撑架断裂，阀门关不严；同时，由于阀门处于关闭状态，支撑架断裂处开口较大，故无法进行焊接。因此，检修人员将液空吸附器的进口阀通空-21稍打开一点，开始焊接。这样，因液空吸附器进口阀已开，液空到了液空吸附器Ⅱ内，经过16小时的加温，液空吸附器Ⅱ已在  $0\%$  以上，且仍在进行加温，液空骤遇加温气，迅速汽化膨胀，加温系统内压力急骤上升，而加温管为普通无缝钢管，

遇  $-170\%$  的液空后，耐压强度迅速减弱，高压下发生爆管。这是造成这次事故的主要原因。

2. 液空吸附器吹空阀开度过小, 液空吸附器内的硅胶粉末及干冰等将吹空阀堵塞, 使急骤膨胀的气体无法及时排泄出去, 这是造成这次爆管的另一原因。

3. 从设备方面讲，通空-21阀漏冷造成阀支撑架断裂，是造成这次爆管事故的主导原因。

# 三、预防措施

综合以上各原因，提出以下预防措施：

1. 进一步加强对维修人员、运行人员的安全技术教育，认真学习操作规程和安全规程，增强职工的安全意识。  
2. 加强维修与运行人员的协调工作，互相监督，消除事故隐患，把事故消灭在萌芽中。  
3. 加强对有关人员的技术培训，提高职工的业务水平和事故预防能力。  
4. 定期更换阀门填料，杜绝阀门漏冷，消除事故隐患。同时，定期对硅胶进行筛选处理。

通过此次事故，对职工进行现场教育，杜绝此类事故再次发生。

（1996年12月25日）

# $4500\mathrm{m}^3 /\mathrm{h}$  空分设备氧产量 剧降的原因分析

李江

（西北铅锌冶炼厂制氧车间，甘肃省白银市，730900）

摘要 介绍配富氧炼铅的  $4500\mathrm{m}^3 /\mathrm{h}$  空分设备，开车四个月即氧产量剧降至1200  $\mathrm{m}^3 /\mathrm{h}$ ，被迫停车。通过故障现象及分析，排除了空气量不足及氧气管路局部堵塞之原因，从主冷换热面积减少，找出原因为主冷被  $\mathrm{CO}_{2}$  和硅胶粉末覆盖，大开车后问题解决。

主题词：中型空气分离设备 产量 剧降 原因 分析 二氧化碳 硅胶粉末

# 一、概述

$4500\mathrm{m}^3 /\mathrm{h}$  (KDON-4500/2000型)全低压大型空分设备，是开封空分设备厂专为我厂从德国鲁奇公司引进的世界上最先进的富氧炼铅工艺的核心设备QSL炉配套设计的，于1990年投产。1995年2月，由于氧气产量下降至  $1200\mathrm{m}^3 /\mathrm{h}$  （标态，下同)，无法满足需求，被迫停车。停车前，我们实际上已经触及到了故障的症结，却没有充分的理由对其进行论证，只好采取间接方法逼近问题的实质，最后通过大开车才查出故障的真实原因。

# 二、故障现象

1995年2月，设备运转仅四个月，2月22日氧气产量即从  $3500\mathrm{m}^3 /\mathrm{h}$  开始下降，到3月4日被迫停车时仅为  $1200\mathrm{m}^3 /\mathrm{h}_{\circ}$

2月22日氧产量下降到  $3000\mathrm{m}^3 /\mathrm{h}$  ，尚能满足要求，我们观察了空分工况，发现上塔底压力、下塔压力、精馏塔阻力、液氧面都很正常，且氧气纯度为  $99\%$  （本空分设备设计氧气纯度为  $95\%$  ，氧产量  $4500\mathrm{m}^3 /\mathrm{h})$  ，因此，我们没有过多担心。我们认真地分析检查了仪控系统，证实其没有误指示，尽管如

此，我们还是采取了一些措施，比如：排放不凝气；缩短切换时间；严格控制中部温度；提前倒换液空吸附器等。

然从，从23日一直到25日氧流量从2800、2500降到  $2300\mathrm{m}^3 /\mathrm{h}$  ，已经不能维持两台氧压机正常送气，只好开氧气旁通阀勉强维持供氧。由于采取措施后氧产量仍继续下降，至26日只达  $2000\mathrm{m}^3 /\mathrm{h}_{\circ}$  我们初步分析原因，是主冷冷凝蒸发器蒸发面积，被诸如  $\mathbf{CO}_2$  和硅胶之类的杂质覆盖了。

# 三、故障分析

我们根据掌握的情况，查阅了有关资料，认真进行了总结，归纳出了可能引起氧气产量下降的三个方面原因：（1)空气量不足，这里不考虑主冷蒸发面积减少的影响；（2）氧气通路局部堵塞；(3)主冷蒸发面积减少。

我们的思路是：试图通过对(1)，（2)两个原因的排除，间接地对第(3)个原因即主冷蒸发面积减少作出判断。

# 1. 对空气量不足的排除

我们确认空压机工作正常，氮水预冷系统工作正常，切换阀无泄漏后，自然地把焦点

集中在板翅式换热器上，认为板翅式换热器空气通道存在堵塞，决定停车对其进行局部加温吹除。2月28日10:30开始加温，到22:30向下塔导气，除氧流量仍为  $2000\mathrm{m}^3 /\mathrm{h}$  外，其余各种工艺参数均恢复正常。根据这些事实，可以说明空气通道基本没有问题，空气量不足的理由可排除。

# 2. 对氧气管路局部堵塞的排除

其步骤有：①停车前，我们打开了氧气蝶阀，感觉气量很大，到底有多大，没法测量；②3月1日，我们停车打开氧流量调节蝶阀DX106，此阀动作灵活；③加温试压与裸冷查漏，检查氧流量表指示情况。

3月4日，氧流量降到  $1200\mathrm{m}^3 /\mathrm{h}$  ，氧压机吸气压力随时可能到零，为避免板翅式换热器被抽真空而损坏的危险，被迫请求停车。停车后，对空分系统进行了认真细致的大加温。加温后，我们将空压机空气引入下塔，通过三只节流阀打入上塔，进行试压，检查氧气流量表指示情况，结果上塔塔底压力为 $0.01,0.02\mathrm{MPa}$  时，氧气流量指示分别为 $1500,2500\sim 3000\mathrm{m}^3 /\mathrm{h}$  。由于上塔底压力不能继续升高，遂停止试压。

3月6日，厂里决定扒珠光砂检修，利用检修，继续对问题进行取证。我们检修了自动阀箱，更换了液空吸附器硅胶和启动干燥器硅胶。3月30日晚上7:30开始裸冷，进入第二阶段后，氧流量能达到  $4500\mathrm{m}^3 /\mathrm{h}$  ，氧气出塔压力表指示为  $200\mathrm{mmH_2O}$  柱。根据 $①$  、  $②$  、  $③$  ，我们认为氧气通路基本没有堵塞，原因(2)可以排除。

综上所述，氧气流量剧降只能是由第(3)个原因造成的，即主冷被  $\mathrm{CO}_{2}$  和硅胶粉末覆盖使蒸发面积不足，从而造成氧气产量下降。这就是我们的结论。

# 四、大开车检验

由于下次QSL炉二次试车供氧关系重大，必须万无一失。因此，我们感到对氧流

# 五、应吸取的教训与操作注意事项

这次氧流量下降剧烈虽然发生在运转时间仅四个月的时间内，但实际上这次事故的症状极似接近设计运转周期时才有的症状。为避免再次发生类似情况，应吸取如下教训：

（1）严格控制中部温度，充分保证  $\mathbf{CO}_2$  自清除。

(2) 大开车前加温吹除一定要彻底，要尽量缩短大开车时间。

(3)液氧泵要定期运转以清除主冷杂质。

（4）坚持巡回检查制度，消灭事故隐患，减少不必要的停车。

（5）按时倒换液空吸附器，再生要彻底，吸附器硅胶一定要按期更换。

（6）严格控制主冷液面在允许范围内。本次事故发生前，我车间主冷液面计经常指示不准（仪表原因），实际控制液面不准确， $\mathrm{CO}_{2}$  随液面涨落、不停地溶解析出，加剧了  $\mathrm{CO}_{2}$  覆盖主冷的速度。

以上是从  $\mathrm{CO}_{2}$  的危害角度提出的教训。

在这里，我们觉得有必要强调一下硅胶粉末的危害。硅胶粉末的来源有三个：一是液空吸附器；二是液氧吸附器；三是启动干燥器。其中液空、液氧吸附器硅胶粉末是随液空、液氧进入主冷的。启动干燥器硅胶粉末进塔的时间是：启动第一阶段随空气进入板翅式换热器中，加温吹除时随加温气进入整个分馏塔系统。硅胶粉末进塔是不可避免的，关键是如何减少其进塔的量，因此在具体操作中应注意：

（1）液空、液氧吸附器及启动干燥器硅胶，一定要按期更换，装填时要筛干净。经查，至事故发生时止，吸附器与干燥器硅胶，分别差三个月和六个月就到期了。  
（2）严格把好液空、液氧吸附器及启动干燥器再生关。一定要严格控制加温气体压力、温度以及再生所需的时间。特别要多吹除，以便把硅胶粉末吹出，减少其进塔的机会。加温压力不能过高，以免冲击硅胶产生粉末。冷开车前加温吹除时尤要注意正确操作启动干燥器，一定要及时再生。  
（3）严防空气带水进入启动干燥器。因干燥器硅胶系粗孔硅胶，吸水易破裂，产生大量粉末。  
（4）对液空、液氧吸附器再生，应使用污气氮或纯氮气，尽量不用启动干燥器干燥过的空气。我车间吸附器再生使用经干燥器干燥的空气，这在客观上造成了干燥器硅胶粉末进吸附器，随液空、液氧进入主冷；此外，再生效果也不如用氮气，影响吸附器硅胶的使用寿命。  
（5）正常生产时，仪表气一定要从板翅式换热器取用，不要从启动干燥器取用。我车间仪表气，因板翅式换热器阀门生锈，不能开启，于是从干燥器取仪表气，致使干燥器长期处于工作状态，其间又没有再生，势必影响硅胶使用寿命。  
（6）对于进塔的硅胶粉末，可以用排液的方法排出塔外。冷开车产生液空、液氧时不要吝惜冷量，一定要将最初产生的液空、液氧排掉，且要排放清洁为止。正常生产时，也要定期排放部分液空、液氧。  
以上所述都是平时容易忽视的几点，应特别予以重视。

我们认为，这次事故的消除，得益于我们的正确判断和大开车前检修时更换了吸附器、干燥器硅胶，因而加温气干净，干燥器、吸附器内硅胶粉末极少。经查，我们换下来的硅胶有大量粉末，颜色发黑，显然已经失效。在加温吹除时，主冷和下塔吹除阀排出很多硅胶灰色粉末和  $\mathrm{CO}_{2}$  白色泡状物，排液时，倒掉桶中的液空、液氧后桶底有  $2\mathrm{mm}$  左右的硅胶灰色粉末一层。这充分证明了主冷覆盖物是  $\mathrm{CO}_{2}$  与硅胶粉末的混和物。

总之，这次氧产量下降的原因最终是找到了，但其中经历了不少曲折，我们把它记述下来是为了给同行提供一个借鉴。不妥之处，请批评指正。

摘要 压力低温液体贮槽在使用过程中，所贮存的液体损耗率经常偏大，其主要原因为外界热量进入或气体和液体的泄漏。本文就以上两方面原因的现象及判断进行了详细分析。图1。

# 主题词：贮槽 低温 液体 损耗率

扬子石化公司烯烃厂空分设备配有压力液氧贮槽和压力液氮贮槽，当空分设备或产品气压缩机因故突然停车时，压力贮槽可以在仪控系统作用下自动将贮存的液体送出，经汽化器汽化后到达供气总管，以保证事故状态下供气量稳定或不中断，为启动液体泵并输送液体提供时间。压力液氧贮槽和压力液氮贮槽流程相同，见图。

由于压力低温液体贮槽具有低温和带压的特点，常会因各种原因使得贮存的液体损耗率过大。归纳起来，主要由外界热量进入或气体和液体泄漏引起。

# 一、外界热量进入

# 1. 特征分析

外界热量进入贮槽内部，低温液体吸收

热量后温度升高，当达到贮存压力下的饱和温度后就会汽化；而压力调节系统将通过打开放空阀排放这部分气体，使贮槽内压力保持稳定。此外，低温液体温度升高后密度减小，比容增大（2.9MPa贮存压力下液氧温度由98K升高至138K时比容增大  $30\%$  以上），而贮槽内胆材质(1Cr18Ni9Ti不锈钢)的线膨胀系数很小，因而在这种情况下贮槽内液体实际液位有可能升高，液体甚至可能进入放空管和差压式液位计的负压管。

液体进入放空管内逐渐汽化，然后从放空口排出，而液体进入液位计负压管有可能使得液位指示出现较大幅度的抖动。无论是低温液体在贮槽内大量汽化还是满溢进放空管道，其重要特征是放空口冒出温度较低的

气体，有时放空管道会挂霜或结冰。因而如果发现放空管道结冰冒冷气，应考虑到有外界热量大量进入贮槽的可能性。

2. 贮槽内不同部位液体吸收热量后的不同影响

贮槽内上部液体和下部液体吸收热量后影响有所不同。如果底下层的液体温度升高，则底下层液体密度下降，当其小于上层的液体密度时就会

液氧贮槽和液氮贮槽流程图

3. 压力贮槽 4. 汽化器

产生对流作用而使得整个贮槽内液体温度趋于一致。在这种情况下，液体容积将逐渐增大。如果原先贮槽内贮存的液体较多，则在吸收较多热量后很容易从放空管等处溢出来从而造成损失。但如果上层液体温度升高，则不容易产生对流作用，液氧的导热系数较小，此时就会产生所谓的“分层现象”，即顶上层液体温度高而密度小，底下层液体温度低而密度大。

出现“分层现象”时，因顶上层较少的液体吸收了较多的热量而温度升高较多，因而顶上层液体很容易达到汽化温度而汽化。可通过卸压试验判断贮槽内液体的温度及分层情况。打开放空阀，如果压力下降较快，则可以认为只有顶上层液体温度较高，而底下层液体温度较低。如果压力下降很慢，则可以认为贮槽内液体上下部温度较为接近且均较高。这是因为在压力下降过程中前一种情况下只有顶上层一部分液体先达到饱和温度而汽化，而后一种情况下有较多的液体达到饱和温度而汽化。

# 3. 贮槽热量进入原因及各原因的判断

压力低温贮槽采用真空粉末绝热，如果绝热层真空度下降，则绝热层空间气体的对流传热作用将加强；如果珠光砂粉末被压实下沉，则顶部空隙处辐射传热作用将大大加强。相对而言，前者贮槽内液体温度升高上下要均匀一些，而后者主要是上部液体温度易升高。这可根据贮槽外壁“冒汗”部位来判断。

如果“冒汗”主要发生在顶上部，则多半是粉末下沉所致；如果贮槽外壁“冒汗”上下较均匀，则很可能是绝热层真空度下降了。观察贮槽外壁“冒汗”情况，最好选择相对湿度较大的清晨，而且最好是放空管结冰冒冷气时。因为此时汽冷屏温度较低而汽冷屏离外壁较近，因而此时贮槽外壁更容易“冒汗”或结霜。

# 二、气体或液体的泄漏

泄漏一般发生在贮槽连接管道上的阀门、法兰等处，可分为内泄漏和外泄漏两种。通过密封不严的阀门，物料由高压侧管道漏入低压侧管道的泄漏，一般称为内泄漏；由于法兰、阀杆密封不严或管道穿孔等引起物料漏到管道外的泄漏，称为外泄漏。

# 1. 内泄漏

贮槽内气体泄漏后，压力调节系统将通过打开增压阀使一部分液体进入汽化器汽化以补偿泄漏量而使压力保持稳定。如果增压阀泄漏，部分液体进入汽化器汽化，压力调节系统将打开放空阀排放掉这部分气体。因而，贮槽内气体泄漏或增压阀泄漏，其重要特征是贮槽出口管道和进汽化器前增压管道温度较低。而泄漏或放空的气体由于主要来自于汽化器，温度较高。

如果贮槽出口管道上的阀门出现内泄漏，由于产品气总管压力略大于贮槽压力，总管气体将倒漏入贮槽内。这种情况有点相似于热量从贮槽底部进入，因而将使得贮存的低温液体温度较均匀地升高，液体容积增大。这可根据贮槽外壁“冒汗”情况和贮槽出口管道温度或改变贮存压力、总管压力等方法与绝热性能变差情况加以区别。

# 2. 外泄漏

贮槽出口管道或进汽化器前增压管道如果出现外泄漏，则在泄漏量较大或贮槽出口至泄漏点之间的管道保温性能较好的情况下可能发展成液体泄漏。在这种情况下，由于泄漏点温度降低，泄漏间隙可能增大。

笔者认为，在低温贮槽系统密封性能较好的情况下，由于贮槽出口管道带有“气封”结构，槽内低温液体不会进入出口管道中，因而贮存过程中出口管道温度接近常温，出口管道实际上不必加以绝热。

在贮槽系统密封性能差或槽内液体送出等情况下，虽然出口管道温度低，但采用裸

管并不会引起附加损失。至于增压管道当然更不必绝热了，而充液管道由于是间断输液，管道又较短，也可以不加绝热（液体泵液体回大槽管路仍应加绝热层）。

总的说来，这些管道采用裸管并不会使得贮存的液体损耗率增大，而对于故障判断和处理是很有利的。

# 一、离心式压缩机的喷振及其危害

简单地说，喘振就是当压缩机在运转过程中，压力不断增加，而流量又不断减小，到  $Q_{\min}$  时，就会在压缩机流道中出现严重的旋转脱离，流动严重恶化，使压缩机出口压力突然大大下降，如图1所示。

由于压缩机总是和管网系统联合工作的，这时管网中的压力并不马上减低，于是管网中的气体压力就反大于压缩机出口处的压力，因而管网中的气体就倒流向压缩机，一直到管网中的压力下降至低于压缩机出口压力为止，这时倒流停止，压缩机又开始向管网供气。经过压缩机的流量又增大，压缩机又恢复正常工作。但当管网中的压力也恢复到原来的压力时，压缩机的流量又减小，

系统中气体又产生倒流。如此周而复始，就在整个系统中产生了周期性的气流振荡现象，这就是所谓的“喘振”

喘振不但和压缩机中严重的旋转脱离有关，还和管网系统密切有关。管网的容量越大，则喘振的振幅越大，频率越低；管网的容量越小，则喘振的振幅越小，频率越高。

喘振现象对压缩机十分有害，喘振时由于气流强烈的脉动和周期性振荡，会使叶片强烈振动，叶轮动应力大大增加，噪声加剧，使整个机组发生强烈振动，并可能损坏轴承、密封和仪表等。如果压送介质为氧气或易燃气体，还可能引起燃烧或爆炸，从而造成严重的事故。

为防止喘振对压缩机可能造成的损坏，离心式压缩机一般均有防喘振装置，特别是对于象安全性要求高的离心式氧压机，更是配有完善的防喘振系统，而且还在出厂试车时，测出真实的喘振线以便准确设定防喘振参数。尽管如此，引起喘振的原因很多，特别是在压缩机的调试试车阶段，如不能及时发现并排除这类隐患，而在正常的氧气运行中发生喘振，则可能产生严重后果。

根据该机组的试车规范要求，第一次氮气试车只升压到  $1.5\mathrm{MPa}$  ，运行4小时，此时各级压力参数见表。第二次氮气试车要求在设计压力下考核其运行情况(功率、振动等)，因氮气比重小于氧气，所以要达到相同的出

口压力，必须提高进口压力以增加流量。第二次试车的升压速度比第一次试车快，升压过程中发现高压缸的轴振动值比第一次试车时有所增加，升压过程中的两组压力参数见表。当出口压力  $\mathbf{P}_5$  升至  $\sim 1.7\mathrm{MPa}$  时（参数见表）喘振发生，因现场机组管网大，所以尽管及时打开了回流和放空阀，但喘振还是持续了近2分钟，而且造成进口过滤器差压计损坏。

根据对所测数据及运行情况的分析，发现测点2、3、4中的  $\mathbf{P}_4$  均大于  $\mathbf{P}_5$  ，这是不正常的。只有测点1中的  $\mathbf{P}_4$  小于  $\mathbf{P}_5$  ，且各段压力值与出厂性能试车时的数据基本相符，说明第一次试车时情况是正常的。第二次试车升压后情况就不正常，  $\mathbf{P}_4, \mathbf{P}_5$  明显倒挂，若根据该机正常的压比分配估算，当  $\mathbf{P}_4$  为  $2.2\mathrm{MPa}$  时，  $\mathbf{P}_5$  应大于  $3.3\mathrm{MPa}$ ，而实测仅为  $1.7\mathrm{MPa}$  。这说明此处有问题，究其原因不外于两种可能：

（1）测试仪表有问题（仪表管接错，取压管堵塞或泄漏）；  
(2)  $\mathbf{P}_{4}$  测点与  $\mathbf{P}_{3}$  测点间有异物，堵塞了流道。

经过对仪表及其取压管的检查，排除了第一种可能性。对第二种可能性，经分析在这段流路中9、10级叶轮和末端冷却器进口处堵塞的可能性最大。当时考虑到工作量的关系先拆冷却器进行检查，未发现异物，然后揭开压缩机高压缸缸盖进行检查，在9级叶轮进口处发现一块布绕在轴上，堵塞了叶轮进口流道(这块布面积约  $0.3 \times 0.3 \mathrm{~m}^2$  ，是脱脂去油用的)。这一发现，解释了喘振的发生及两次试车时振动值变化的原因，即这块脱脂棉布遗留在9级进口管下部的冷却器封头内，第一次试车时由于升压较慢，棉布没有随气流移动，所以一切压力指示正常。而第二次试车时升压较快，流量变化大，棉布被吸至9级叶轮入口处缠绕在轴上，所以

引起轴振动值的变化。随着压力的升高，9级叶轮入口处相当于起了一只阀门的作用，当  $\mathbf{P}_4$  为  $2.2\mathrm{MPa}$  时，如没有这块布，  $\mathbf{P}_{5}$  应该接近喘振值3.3MPa了。而由于布的存在，  $\mathbf{P}_{6}$  只有1.7MPa，远远低于与  $\mathbf{P}_{6}$  相连的旁通阀和放空阀的防喘振设定值。随着压力的继续增加，在9级叶轮之前的某一级叶轮中就发生了喘振，而这时  $\mathbf{P}_{6}$  的值低于防喘振设定值，防喘振系统不起作用，由此就引发了整个机组的喘振。

# 三、喘振现象的判断和预防

脱脂棉布固然是造成喘振的直接原因，但试车过程中，如能在监测出口压力  $\mathbf{P}_{6}$  的同时认真读取其它各段压力并加以分析，是能够避免这次喘振的。这充分说明在离心压缩机的运转，特别是试车过程中应对所有参数进行监测和分析，而不能只盯住其中的主要几项参数，否则就容易留下隐患，给设备造成损害。

# 可逆式换热器氧气通道泄漏的判断方法

关 明 显

（哈尔滨气化厂空分分厂，黑龙江省依兰县达连河镇，154854）

摘要 介绍了  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备可逆板翅式换热器的产品氧通道出现泄漏的情况。讲述了泄漏部位的判断、泄漏通道的处理和焊接、以及维护防范措施。

主题词：大型空气分离设备 可逆板翅式换热器 氧气通道 泄漏 判断 防范

我分厂  $1^{*}$  KDON=10000/10000V型空分装置，是由杭州制氧机厂设计制造、1993年5月正式投产、运行的可逆板翅式空分设备。1996年7月，第I组切换板式第二单元中的产品氧通道出现泄漏，氧气总管纯度由 $99.5\%$  降到  $93\% \mathrm{O}_2$  ，装置维持运行一个月后被迫停车加热进行检修。

# 一、判断氧通道的泄漏部位

该板翅式换热器，分为两组6个板式单元。换热器冷箱扒完珠光砂后，我们进行第一步查找，先把第2板式单元的冷、热段间的产品氧集气封头接管锯开，用空切-2阀导气，结果先判断出是热段泄漏。第二步，我们又把第2板式单元热段的产品氧出口管锯开，继续用空切-2阀导气试压，查出是正对集气封头出入口管的产品氧通道泄漏，确定了集气封头所需要的开割部位。

由于我们没有制造厂家的通道排列图纸，当集气封头锯开后，我们才确认该板式单元共有14个产品氧通道，泄漏通道是在左数第8道。至于泄漏通道的两侧通道走何工艺介质，我们根据多股流换热器的通道排列原则，排出了泄漏氧通道两侧通道不可能是返流产品氮或仪表气通道，只能是空气或污氮通道。

# 二、泄漏通道的处理与焊接

确定了泄漏产品氧通道两侧通道的工艺介质后，我们考虑到如果直接堵焊泄漏氧通道，那么该通道就与两侧相邻的空气或污氮通道联通并形成死气，在装置运行或加热中，很容易在堵焊氧通道底部积存水，留下进一步冻裂通道隔板的隐患，最终会使堵焊氧通道两侧的空气与污氮通道相连通，造成换热器局部热负荷不平衡，加工空气量损失，以及返流阻力增加。

为此，在堵焊氧通道以前，需要把泄漏的氧通道与空气或污氮通道相连通，也就是再确认氧通道是与左侧还是右侧通道发生泄漏。于是，我们又在泄漏通道的左侧通道封条上钻孔，开空切-1阀试压，结果钻孔有气而泄漏氧通道无气。这就证实了是右侧通道与泄漏氧通道相同。接着我们在泄漏氧通道热端的出口导流板上斜钻5个孔后，用LF11焊条进行了堵焊。在试压查漏后，又补焊了封头和接管。仅用两天就完成了修复任务，经开车后验证，恢复了设计氧纯度。

# 三、维护防范措施

空气分离设备中的可逆式换热器，是用来实现空气与污氮及氧氮等产品气体之间的热交换，并使空气中的水分、二氧化碳冻结

清除，同时各通道之间存在着交变应力。因此，为保证可逆式换热器能长周期安全使用，在操作维护上应注意以下几点：

1. 正常操作中要坚持吹切-1、2、3、4的定时排水，尽量减少正流空气的带入水量。  
2. 空冷塔液位工艺指标，要作为重要的监控项目和巡检内容来抓。进塔空气压力与空冷塔水泵联锁要投用完好，杜绝板式带水操作。  
3. 对于空冷塔循环冷却水中有菌藻类生长，需定期投放杀菌灭藻剂的单位，在投放过程中会产生大量泡沫，此时的安全办法就是停止循环水使用（适用于空压机有末段冷却器的装置）。  
4. 对于切换系统各切换阀，要定期检修

或更换，平衡阀不均压故障，不能长时间的坚持运行。

5. 对备用空分装置在停车后，要用干燥气体对可逆式换热器内的空气和污氮通道进行置换。有条件的单位要对全装置进行氮封处理。

我厂电炉用料主要以废钢为主，而废钢的整理全靠火焰切割，每月整理废钢（即把废钢切割成  $400 \times 400\mathrm{mm}$  以下的入炉料）7000余吨，每月用瓶氧3000余瓶。废钢料场摆放大量的氧气瓶、乙炔瓶，单从运输角度讲，每天就要两辆拖拉机往返于氧气厂和废钢库之间，大量的气瓶搬运、贮存，给生产带来很大的不便。且每年要花费30余万元外购瓶氧，很不经济。

我厂现有  $150\mathrm{m}^3 /\mathrm{h}$  深冷制氧机组一套， $600\mathrm{m}^2 /\mathrm{h}$  变压吸附(PSA)机组一套，无充填设施。由于  $150\mathrm{m}^3 /\mathrm{h}$  深冷制氧机组运行费用高，而  $600\mathrm{m}^3 /\mathrm{h}$  PSA机组的供氧量基本能满足炼钢的使用，所以  $150\mathrm{m}^3 /\mathrm{h}$  机组停运。为此，我厂决定将  $600\mathrm{m}^3 /\mathrm{h}$  机组生产的纯度为  $93\%$  的氧气代替瓶装  $99.2\%$  的氧气，并用管道输送至废钢料场。

# 一、确定管网压力

气割所用的氧气压力，一般为  $0.5\sim 0.8$  MPa，电炉吹氧所用的氧气压力为  $0.6\sim 0.8$  MPa。由于吹氧压力与气割所用压力基本一致，制氧站与炼钢车间相距  $1600\mathrm{m}$ ，所以不再单设管道，利用向炼钢输氧的管道作主管道，另向废钢库接支管道。

# 二、用氯点的布置

每个废钢料场的都是由合格区、切割区、码垛区组成，切割区的每一个天车柱子，布置一个分气包，每个分气包上有两个用氧接口，供氧嘴头、阀门设在非燃烧体的保护箱内。主管道  $\mathrm{DN}57\times 4\mathrm{mm}$  ，支管道DN32 $\times 3.5\mathrm{mm}$  ，由制氧通往炼钢的主干线管道为

DN  $76\times 4.5\mathrm{mm}_{\circ}$

# 三、切割厚料试验

使用管道氧气切割废火车头的车轮、车轴、车体主要部位等大型厚料，必须用0.8MPa的氧气，切割轻薄料时用0.35MPa即可。由于主干线管道的压力不稳定，当炼钢用氧量较大时，主干线管道的压力有时低于0.8MPa，所以废钢料场的大块厚料，须在氧气压力足时切割。试验证明，  $93\%$  的PSA氧气完全可以代替  $99.2\%$  的瓶装氧气。

# 四、经济与管理

针对现场的实际情况，我厂制订了《切割用管道氧气安全规程》，并将现场摆放的乙炔瓶实行定置管理。对现场的管理人员、切割操作人员，轮流进行安全教育，使得每人熟悉《氧气安全规程》。

1994年8月我厂把变压吸附生产的氧气用在电炉炼钢上，1996年又将该氧气用于废钢切割。实践证明，纯度  $90\% \sim 93\%$  压力0.8MPa的氧气完全能满足废钢切割。变压吸附制氧具有故障率低、成本低、启动快的优点，在我厂都得到了充分体现。在我们地区，瓶氧价格11元/瓶，每立方氧1.83元；而变压吸附制氧的成本，只有0.58元/ $\mathfrak{m}^3$  。这一措施，使我厂每月降低切割废钢所用氧气的成本费、运费、装卸费共约2万元，并且减少了气瓶运输过程中的不安全因素。

# 制氧工问题解答（三则）

1. 问： $1000\mathrm{m}^3/\mathrm{h}$  空分下塔液空是如何产生和积液的，能否在主换热器冷端产生？另外，加温气体为来自纯化器后约  $14^{\circ}\mathrm{C}$  的干燥空气，这对加温解冻是否有不利影响？还有，加温时纯化器再生气压力及流量如何控制？（江苏读者问）

答：空分设备启动过程中，液空的产生在可逆式流程和分子筛流程是各不相同的。

在可逆式流程的启动过程中，液空首先在污氮液化器，其次是氧氮液化器（如果有氧氮液化器的话）产生。但在分子筛流程的启动过程中，液空的产生首先是液空液氮过冷器，其次是主换热器的冷端，最后是主冷。

判断液空是在哪里先产生？主要看冷箱内的温度场和分布情况。加工空气在  $\mathbf{P} = 0.6$  MPa、 $\mathrm{T} = 101\mathrm{K}$  饱和状态下液化，返流气体一般在  $98\sim 94\mathrm{K}$ ，因此在那里先冷却，就在那里先出现液空。

正常运转时，有小部分空气液化是在主换热器冷端产生。

至于加温气体温度为  $14^{\circ} \mathrm{C}$ , 加温分馏塔能否彻底这个问题, 首先要明白的是, 理论上没有问题。国内外的实践都表明, 这样的加温方法和结果是完全能成功的。为什么说呢? ① 加温气体很干燥, 没有问题。② 加温气量大, 一般加温气量为空压机加工气量的  $\sim 20\%$ , 这比一般可逆式流程的加工气量大得多。

关于纯化器再生时，空气的压力和流量如何控制？精馏塔加温时，再生气的压力及流量在各出口状态一定时，主要是靠空压机的放空阀开大或关小。运行调节空压机的出口阀有三种形式：①手动放空；②自动恒压

放空；③遥控放空。

（杭氧集团有限公司技开处 施金）

2. 问:  $150 \mathrm{~m}^{3} / \mathrm{h}$  空分设备活塞式膨胀机改为透平膨胀机,  $\mathbf{T}_{1}, \mathbf{T}_{2}, \mathbf{T}_{3}$  温度如何控制?

答：随着科学技术的发展， $150\mathrm{m}^3/\mathrm{h}$  空分设备不断更新换代，90年代的 $150\mathrm{m}^3/\mathrm{h}$ 空分设备配透平膨胀机，有的单位老设备改造也采用透平膨胀机来替代活塞式膨胀机。因此，对分馏塔的温度控制也有些不同。

$\mathbf{T}_{1}$  温度是指热交换器中部，即膨胀机进口温度；  $\mathbf{T}_{2}$  温度是指膨胀机出口进下塔前的空气温度；  $\mathbf{T}_{3}$  是指热交换器出口管、高压空气节流阀  $(V_{1})$  前的温度。

$\mathbf{T}_{1}$  的作用是显示膨胀机进气温度，并控制进膨胀机前空气温度不得接近该压力下的液化温度。正常启动阶级是预测  $\mathbf{T}_{3}, \mathbf{T}_{5}$  温度的变化，从而便于  $\mathbf{T}_{2}, \mathbf{T}_{3}$  控制在理想的范围内。 $\mathbf{T}_{1}$  温度高低的变化取决于膨胀量多少的变化，膨胀量越大，进入热交换器下部的高压空气量越少，而返流气体冷量吸收减少，进入热交换器上部的冷量增多， $\mathbf{T}_{1}$  温度降低；反之，膨胀量减少， $\mathbf{T}_{1}$  温度就升高。

$\mathbf{T}_{2}$  温度在设备启动初期和  $\mathbf{T}_{3}$  温度达到  $-155^{\circ} \mathrm{C}$  以下是不必要控制的，因为开车初期，设备温度较高，为了充分发挥膨胀机的制冷潜力，两台膨胀机全负荷运转， $\mathbf{T}_{2}$  温度逐渐下降。当  $\mathbf{T}_{2}$  温度下降到  $-150^{\circ} \mathrm{C}$  时，此时打开  $\mathbf{V}_{1}$  阀，使  $\mathbf{T}_{3}$  温度逐步下降。若  $\mathbf{V}_{1}$  开启过大，通过热交换器下部的空气量过多，返流气体冷量吸收越充分，进入热交换器上部的返流气体温度升高， $\mathbf{T}_{1}$ 、 $\mathbf{T}_{2}$  温度随之升高；若  $\mathbf{V}_{1}$  阀开启过小，则反之。

因此，这段时间用  $\mathbf{V}_{1}$  阀的开度来控制 $\mathbf{T}_{2}$  温度。活塞式膨胀机的  $\mathbf{T}_{2}$  温度一般控制在  $-140^{\circ}C\sim -155^{\circ}C$  ，而透平膨胀机  $\mathbf{T}_{2}$  温度可控制在  $-155\sim -165^{\circ}C$  。应该指出的是 $\mathbf{T}_{2}$  温度水平高低与热交换器结构和热端温差有关，一般老  $150~\mathrm{m}^3 /\mathrm{h}$  是盘管式换热器， $\mathbf{T}_{2}$  温度过低，热端温差会扩大，冷损增加，这是不允许的，新型  $150~\mathrm{m}^3 /\mathrm{h}$  空分设备的热交换器铝制板翅式换热器，传热面积比盘管式要大，一般热端温差不易扩大。因此，  $\mathbf{T}_{2}$  温度水平可控制低一些，但是最低不得低于该压力的液化温度，一般应高于该压力液化温度  $3\sim 5^{\circ}C$  为安全。

$\mathbf{T}_{3}$  温度随设备温度的下降而不断下降，当达到  $-160^{\circ}C$  以下时应注意控制，液化阶段应控制在  $-160\sim -170^{\circ}C$  为宜，有利液体的产生和积聚，此时  $\mathbf{T}_{2}$  温度不必控制，因为只有  $\mathbf{T}_{3}$  温度控制好了，  $\mathbf{T}_{2}$  温度也就稳定在安全运转的范围内了。

当液氧液面达到正常值， $\mathbf{V}_2, \mathbf{V}_3$  阀基本到正常工作位置，液氧液面继续上升，可将膨胀机进口阀、 $\mathbf{V}_1$  阀逐渐开大，当膨胀机进口阀全开后，可以不必顾及  $\mathbf{T}_3$  温度的高低，开大  $\mathbf{V}_1$  阀来维持液氧液面稳定。因此，在正常运转下， $\mathbf{T}_1$ 、 $\mathbf{T}_2$ 、 $\mathbf{T}_3$  温度是用不着控制的。只有在设备启动阶段、变工况运行、不正常运转时或故障时，根据  $\mathbf{T}_1$ 、 $\mathbf{T}_2$ 、 $\mathbf{T}_3$  温度的变化来判断问题，排除故障。

# 3. 问：KDON-150/550型空分设备启动时，压力如何控制？

答：KDON-150/550型空分设备启动时加工空气最高压力控制为  $1.96\mathrm{MPa}$  ，否则，空压机会超压。启动初期，为了充分发挥两台膨胀机的制冷潜力，使膨胀机全负荷运转，用膨胀机进口阀控制两台膨胀机转速在  $10.5\sim 11$  万  $\mathbf{r / min}$  。  $\mathbf{V}_{1}$  阀可以不开，用

空压机放空阀控制加工空气压力在1.8MPa左右。

膨胀机制冷量多少，与进入膨胀机的气量多少、膨胀机前后压力大小、膨胀机进气温度和膨胀机效率等因素有关。进入膨胀机的气量越多、进气温度越高、前后压差越大、膨胀机效率越高，则制冷量越多；反之则制冷量越少。进入膨胀机的气量、膨胀机前的压力、温度、膨胀机的效率受到转速的限制，不能随意调节。而膨胀机后的压力降低，可以增大膨胀前后的压差。因此，启动初期，应设法降低下塔压力，降低下塔压力的办法是把  $\mathbf{V}_2$  、  $\mathbf{V}_3$  阀全开，打开除热交换吹除阀以外的所有吹除阀、分析阀，同时在确保分子筛纯化器再生气量的前提下，尽量降低上塔压力。

随着启动时间的延长，塔内温度逐渐下降，高压压力自动降低，因此要及时关小空压机放空阀，以确保膨胀机转速；另一方面当吹除阀、分析阀出口结霜时，应及时关闭。

当  $\mathbf{T}_{3}$  温度达到  $-150^{\circ}C$  时，打开  $\mathbf{V}_{1}$  阀，高压压力下降时，应及时关小空压机放空阀来保持高压压力  $1.96\mathrm{MPa}$ ，当空压机放空阀全关后，高压压力和膨胀机转速下降时，应保持一台膨胀机满负荷运转，把另一台膨胀机减量运转。当运转的膨胀机进口压力低于  $0.6\mathrm{MPa}$  时，可以停掉一台膨胀机来保持高压  $1.96\mathrm{MPa}$ 。

当关阀基本结束，液氧液面高于  $0.35\mathrm{m}$  时，应逐渐开大  $\mathbf{V}_{1}$  或膨胀机进口阀来降压，当膨胀机进口阀全开后，液氧液面还是上升，可开大  $\mathbf{V}_{1}$  阀来保持液氧液面。

KDON150/550型空分设备正常运转压力：高压压力  $1\sim 1.4\mathrm{MPa}$  ，下塔压力0.45 $\sim 0.55\mathrm{MPa}$  上塔压力  $0.045\sim 0.055\mathrm{MPa}$

(2、3题 杭氧工程成套公司 叶必楠)

1996年12月15日凌晨，北京大部分地区出现大雾天气，早晨7时，首都机场能见度为  $200\sim 300$  米，温度为零下  $1.9^{\circ}C$  ，为探索利用人工催化的方法，减少大雾给航空港和公路交通带来的不利影响，北京市人工影响天气办公室在有关单位协助和支持下，利用地面移动罐车泼洒液态氮，在机场环路和18公里高速路段进行了1996年冬季首次人工消除过冷雾（即低于  $0^{\circ}C$  的雾）试验。

早晨8点15分，两辆分别装满10吨和2吨液氮的北京普莱克斯罐车徐徐驶入机场环路，压力约0.4MPa的液氮均匀的呈扇状喷向空中，迅速在浓雾中扩散、延伸，随即大量冰晶在雾气中闪烁，进而变成米雪降落在地面，视线可见有效范围内地面上出

现明显积雪，有雾状生成，作业1小时后能见度大于600米。9点31分，第一架离港飞机放飞。

在机场试验作业过程中，其中一辆液氮罐车又在机场高速公路同步进行了试验作业，也取得了较好效果，能见度由作业前的  $200\sim 300$  米明显改善到500米以上。

此次试验作业的成功，说明利用液氮消除过冷雾是可行的，对建设现代化航空港和保证公路交通畅通也是有重要意义的。

# 中国工业气体工业协会1996年年会在西宁召开

中国工业气体工业协会第六次会员代表大会暨1996年年会，于1996年9月9日至14日在西宁举行。化工部行业指导司何兴湖副处长、西宁市市委书记吕兴国等领导参加了会议并讲了话。气体行业及相关行业63个单位的104位代表参加了会议。

本次会议东道主青海氧气厂潘颖厂长致欢迎词。孙国民秘书长作了中国气协1996年度工作总结报告、财务收支情况通报，并宣读了理事会《关于加强技术经济指标交流和统计工作的决定》。中科院大连化物所膜工程研究开发中心赵勇高工和冶金部武汉钢铁设计院上海分院总设计师杨涌源教授级高

工，分别作了《膜分离技术在工业制气行业中的应用》和《宝钢  $72000\mathrm{m}^2 /\mathrm{h}$  空分设备的最新技术情况》两个专题报告。会议还进行了论文发布和经验交流，以及信息发布。会议研究了气体行业企业改革和集约化经营，以及气体行业今后的发展方向题。会议确定成立制氢、二氧化碳专业委员会，成立协会科技专家网，以及1997年工作计划。最后，理事长翟国才作了总结报告。
