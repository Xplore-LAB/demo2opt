# 气分总网与制冷二专召开第十次全网大会暨技术交流会

机械部气体分离设备科技信息网、中国制冷学会第二专业委员会(深冷)联合召开的第十次全网大会暨技术交流大会，在东道主宁波化工机械集团公司的鼎力支持下，于1996年10月22日至25日在宁波奉化溪口召开，有112个单位174位代表赴会，盛况空前。机械部重大装备司原副总工程师、现部科技委员、气体分离设备行业协会秘书长朱燧炎；中国制冷学会第二专业委员会主任、机械部重大装备司科技处处长赵玉侠；中国工业气体工业协会秘书长孙国民；宁波市机械局局长桑贤路；宁波市化工局技术处处长林书灯；杭州制氧机集团有限公司副总经理张祖立等领导到会，并讲了话。

开幕式上，宁波化工机械集团公司总经理王益良介绍了公司概况，并致欢迎词；公司总工程师全从干介绍了公司产品；网理事长、杭州制氧机研究所所长张培元作了网工作报告；网秘书长林金良作了网财务报告。在闭幕式上，宁波化工机械集团公司副总工程师、低温容器厂厂长王竞雄作了“开会交友、服务行业”的讲话。会议由网常务理事长、杭州制氧机研究所行业办公室主任顾福民主持。

这次联合大会主要是气体分离行业的技术交流与信息发布。网秘书处提供了包括66篇论文与资料、16开232页的《论文集》；有22篇国外气体分离技术开发与低温应用、16开76页的《译文集》，另有自带论文资料5篇，及信息发布单位提供的各类样本。会议进行了技术交流，有20位代表在大会上宣讲了论文，内容涉及当代气体分离与液化的最新技术，如填料塔、球形低温液体贮槽、变压吸附、膜分离、液化天然气、液化石油气、空分设备技术改造等。会上，上海UOP分子筛有限公司、美国普莱克斯（上海）有限公司、美国APCI中国有限公司、德国赫司特（中国)投资有限公司MG公司等作了公司及产品与技术介绍，另外有14家空分配套单位作了信息发布。

# 规整填料塔与全精馏提氩

江楚标

（中国空分设备公司，杭州市艮山门外东新路152号，310004）

摘要 介绍规整填料在空分设备中运用的现况、规整填料塔的优点、全精馏提氩及优点，并提出了上塔用组合塔或填料塔是今后的必然趋势。文章进行了常规制氩和全精馏提氩、筛板上塔与规整填料上塔的比较，用数据阐明了规整填料塔与全精馏提氩在空分应用的优点与趋势。图3表2。

主题词：空气分离设备 筛板塔 规整填料塔 全精馏提氩 优点 趋势

规整填料塔与全精馏提氩成了我国空分行业1996年的热门话题，这不是偶然的。它至少说明了两件事：第一，规整填料塔及全精馏提氩，作为空分工艺的一次革命性的变化，它使空分产品的提取率及产品的单位电耗都达到了一个新的水平，因而具有强大的生命力；第二，我国空分设备行业已经开始掌握全精馏提氩工艺和规整填料塔的设计制造技术，已有能力提供这种新一代的产品。

随着这一新技术的推广和对它的认识的普及与深化，预计从1998年起，我国大型空分设备的用户将会出现非全精馏提氩工艺和非填料上塔(或组合式上塔)不要的状况。届时，我国空分行业在流程组织和设计方面与国外的差距，将大大缩小。

# 一、规整填料在空分设备中应用的历史

规整填料在空分设备中的应用历程，见《深冷技术》1996年第2期本人撰写的“规整填料在空分设备中的应用”一文。

到现在止，根据我们了解的情况，规整填料塔和全精馏提氩，德国Linde公司1990年已用于  $66000\mathrm{m}^3 /\mathrm{h}$  空分设备，美国APCI公司已用于  $72000\mathrm{m}^3 /\mathrm{h}$  空分设备（供宝钢），Praxair公司已用于  $80000\mathrm{m}^3 /\mathrm{h}$  空分设备

（供印度）。

# 二、规整填料塔的优点

规整填料塔的优点，来源于它的两个基本特性，即每米填料的理论板数及每米填料的阻力。苏尔寿麦勒派克填料的这两个基本特性见图1和图2。这两个图来自苏尔寿公司的产品样本。

规整填料的一些特性见《深冷技术》1996年第2期“规整填料在空分设备中的应用”。

2. 阻力小。只有筛孔塔板的  $1 / 5 \sim 1 / 7$  甚至更低。空分全精馏提氩所以能够实现，完全有赖于规整填料的这一特点和优点。  
3. 可使空分设备的单位电耗降低  $8\%$  。规整填料用于上塔后，使上塔阻力减小，空压机排压降低，达到节能降耗的目的。  
4. 负荷调节范围大, 即弹性大。精馏塔

的负荷调节范围可从  $30\%$  到  $105\%$  ，甚至更大。对空分设备来说，透平空压机的气量调节范围是有限的，约为  $70\% \sim 100\%$  ，精馏塔更大的负荷调节范围就无意义。当然，这也意味着对塔径的设计没有很严格的要求。

5. 操作稳定、负荷调节方便快捷。筛板塔每块板上都有  $12 \sim 25 \mathrm{~mm}$  一层液体，全塔的液体滞留量很大，调纯及变工况时每块板

上的液体组分都要变化。而填料塔内的滞液量要少得多。因此操作调整方便，变工况快，且工况稳定。

任何事物都是一分为二的。填料塔也有它的缺点。由于空分上塔的进出料口多，每个进出料口都要有一个液体分配器，且每个液体分配器占用的塔高要比筛板塔的进出料口占用的塔高得多。因此，空分设备的上塔使用规整填料塔后，高度要增加一些。由于规整填料塔消耗的材料要比筛板塔多，上塔的造价较之筛孔板塔也要增加一些。但一个上塔的投资在空分设备总投资中所占的比例毕竟是很小的，较之它长期运转所带来的好处，是完全可以接受的。至于塔高，在使用全精馏提氩工艺后，冷箱的高度已主要不是取决于上塔，而是取决于氩塔，因此已影响不大。

根据国外的一些介绍及我们的计算分析，我们认为上塔用填料加筛板的组合塔是合理和可取的，可以充分发挥两者的长处，

而避开两者的短处。在后面我们还要讨论这个问题。

# 三、全精馏提氯

《深冷技术》1993年第1期本人译的“规整填料对空分精馏提氩的影响”一文，是美国APCI公司两位专家在1991年第18届国际制冷年会上发表的论文。这篇文章对传统的筛板塔为什么不能实现全精馏提氩、规整填料塔为什么能实现全精馏提氩、氩中氧含量与理论塔板数的关系等作了清楚的阐述。这篇文章为我们打开了一个新的窗口，使我们看到了空分工艺的一方新天地。这篇文章也促使我们去不断地了解和探求填料塔和全精馏提氩的有关知识。在以后的一段时间里，我们通过一些计算，使文中介绍的知识变成了我们的知识，并有不少新的发现。

若粗氩塔不截断，像常规那样从氩馏分抽口一直往上布置，那冷箱将高达  $80\sim 90$  米。“粗氩塔”不截断，当然就不需要泵来联结，但这时液空却无力靠自己的压差进入氩塔顶端的冷凝蒸发器，只能靠泵加压来达到这个目的。因此，泵总是少不了的。所以，一般采用将粗氩塔截成两段这样的布置方法，以降低冷箱的高度。

还可以不要精氩塔，只要一台“粗氩塔”（应该叫氩塔）就可以得到氧、氮含量均符合要求的产品氩。这样，提氩流程就更加简单。我们知道，无论是筛板塔还是填料塔，都不可能在一只塔里同时除去氧和氮，使其符合纯氩产品要求。所以要取消精氩塔，首先要在主塔上下功夫，同时氩塔的结构也要有所变化。通过大量的计算分析，我们认为这也是可能的。预计到1998年，国产空分设备将开始采用这个新技术，因为这需要有一个工业性试验的机会和时间。当然，这个工业性试验是没有风险的，因为在这个试验中，仍将设置一台可能不用的精氩塔来起“保驾”的作用。

全精馏提氩有以下优点：

1. 简化了流程。取消了塔外加氢脱氧的所有热区设备，包括氢站。  
2. 制氩部分的投资，全精馏提氩比常规提氩要省一点。这里所说的投资，包括设备费、征地土建费和安装费等。

3. 操作管理方便。全精馏提氩由冷箱操作工统一操作。由于取消了原塔外加氢脱氧工序的管理岗位，也不需要制氢或购买氢气了，这样，每班至少可减少2名操作工。由于没有氩压机、干燥器及加氢系统，经常性的管理维修工作量也大大减少。  
4. 氩提取率可提高  $10\%$  以上。国外公司的氩提取率已达  $92\%$  。但这样高的氩提取率只有膨胀空气不进上塔的工艺流程（如新型液氧内压缩流程）才能够达到，常规的冶金型流程是达不到的。拉赫曼气量的多少对氩的提取率有很大的影响，常规冶金型流程的液体产量和氩提取率不能兼得。

现在可使氯提取率比以前提高  $10\% \sim 20\%$  。生产液体少的大型空分，氯提取率可达  $70\% \sim 75\%$  。

5. 制氩成本降低。制氩部分投资减少，动力消耗大大降低，减少了操作工，减少了维修工作和备品配件的消耗，提高了氩气产量，所有这些，都使氩气成本有较大幅度的下降。

# 四、上塔用组合塔或填料塔是今后的必然趋势

国外氧的提取率已高达  $99.8\%$  。冶金型流程空分设备氩的提取率已高达  $80\%$  以上，且可以取消精氩塔，只用一个氩塔（分成两段），就可以得到氧、氮含量均符合要求的产品氩。而到现在为止，国产空分设备氧的提取率约为  $91\% \sim 93\%$  ，氩的提取率在  $40\% \sim 50\%$  ，差距是明显的。

1. 提高氧、氩提取率和采用全精馏提氩工艺后，主塔的塔板数将会有明显的增加，它迫使我们必须考虑将上塔改成填料塔，不然的话，空透排压会比现在的分子筛吸附一增压膨胀流程还要高，甚至可能会超过0.7MPa(A)。

选择有两个：一个是仍然用筛板塔，那么空透排压就要升高。暂不去理会空透的造价会如何变化，但空透的轴功率肯定会增加。这将部分抵消甚至完全抵消提取率提高和采用全精馏提氩工艺后所带来的好处，当然也是我们大家都不希望的；另一个选择就是采用填料上塔，或者用组合上塔（这是我们起的名字，以区别于纯粹的填料塔和筛板塔）。

由于填料的阻力比筛板小得多，即使上塔塔板数增加一倍，上塔的阻力还能减少一半以上，空透排压仍能降低到与以前切换板式流程相当的压力水平，在0.62MPa(A)左右。对于海拔高、大气压力低的地方，这个压力还可以降低。这样，我们就不仅得到了采

用全精馏提氩工艺后提取率提高所带来的好处，还得到了空透排压降低所带来的好处。空分设备是长期连续运转的设备，是钢厂、化工厂的用电大户，这种多管齐下、互相配合的增产节能措施，必然会有强大的生命力，会得到社会各业的认可，从而推动其发展。

组合上塔是个什么意思呢？产生这种设想是源于膨胀空气进口以下的塔板数要占上塔板数的一半以上这样一个事实。在氩馏分抽口上下两段用规整填料塔以降低上塔的阻力，在其余几段仍用筛板塔以降低塔的高度和减少三个液体分配器，从而使塔高降低，塔的制造成本也能减少一些。这样的上塔我们称它为“组合上塔”。我们认为这样的组合上塔是会有生命力的。

2. 采用填料上塔或组合上塔后，上塔平均工作压力降低， $\mathrm{O}_2 / \mathrm{N}_2, \mathrm{O}_2 / \mathrm{Ar}$  的相对挥发度有所提高，塔板数会减少，这是个正效应。相反，全部用筛板数，塔板数增加后上塔的平均工作压力升高， $\mathrm{O}_2 / \mathrm{N}_2, \mathrm{O}_2 / \mathrm{Ar}$  的相对挥发度减小，塔板数会增加，这是个负效应。  
3. 采用全精馏提氩工艺后, 粗氩塔的理论塔板数达到约180块, 粗氩塔的高度超过50米。冷箱高度已主要根据粗氩塔来决定, 主塔高一些影响已不太大。  
4. 可能会有一种耽心, 使用规整填料后氩馏分抽口是否能正确地确定? 会不会出现因馏分口位置不当而使提氩困难或者根本出不了氩? 馏分口位置的确定确实是很重要的。但是, 我们认为氩馏分抽口位置的正确与否与用筛板塔还是填料塔关系不大。筛板塔也有因抽口位置不正确而影响提氩及氩产量的经历。关键是设计人员在确定馏分抽口位置时必须作详细计算, 作必要的变工况计算, 并在此基础上进行慎密的分析, 最后确定抽口位置。我们经过一系列计算后发现, 氩馏分抽口是容易确定的。在采用全精馏提氩工艺后, 氩馏分抽口位置有较大的自由度。

# 关于流程计算中绝热损失的一点讨论

王安平

（开封空分设备厂设计处，河南开封市公园路，475002）

摘要 介绍了空分流程计算中绝热损失的三种计算方法、三种方法计算结果比较、三种计算方法的选择与讨论。指出合理、准确地确定绝热损失大小，对膨胀机参数的确定和精馏程序的计算，有重要意义，认为第三种计算方法比较适宜。

# 主题词：空气分离设备 流程 计算 绝热损失 确定

在空分流程计算中，绝热损失的大小直接影响到膨胀量的大小。在空气膨胀的流程中，膨胀量又与精馏工况密切相关。因此准确和合理地确定绝热损失的大小，对膨胀机参数的确定，对精馏程序的计算，具有重要的意义。

# 一、绝热损失的三种计算方法

目前绝热损失的确定，大体上有三种方法：①按每立方米加工空气若干千焦进行计算；②按保冷箱单位面积若干千焦进行计算；③按与加工空气成指数关系的经验公式进行计算。

在《全低压制氧机原理及计算》（1974年编，1976年版）一书中，列出了第①种方法的参考数据。其中，小型空分设备： $q_{3}$  为  $8.37 \sim 12.56 \mathrm{~kJ} / \mathrm{m}^{3}$  加工空气；1500、3200  $\mathrm{m}^{3} / \mathrm{h}$  空分设备： $q_{3}$  为  $6.28 \mathrm{~kJ} / \mathrm{m}^{3}$  加工空气左右； $10000 \mathrm{~m}^{3} / \mathrm{h}$  以上空分设备： $q_{3}$  为  $3.35 \sim 5.02 \mathrm{~kJ} / \mathrm{m}^{3}$  加工空气。

在《制冷技术与低温装置》(1981年版)一书中，介绍了第②种方法的具体数据，即每平方米保冷箱面积绝热损失为  $167.47 \mathrm{~kJ}$ 。

按与加工空气成指数关系的第 ③ 种方法，其具体公式为：

$$
\begin{array}{l} Q _ {3 - 1} = \left(K _ {1} + K _ {2}\right) \times 2 1. 5 V _ {K} ^ {0. 6 7 1} \\ \times 4. 1 8 7 \mathrm {k J} / \mathrm {h} \tag {1} \\ \end{array}
$$

式中  $V_{K}$  ——参加精馏的空气量，不包括仪表气和切换损失量

$K_{1}$  ——系数，可逆式流程为0.75

$K_{2}$  ——系数，考虑提取稀有气体时需要增加的冷损，具体数值为0.5

公式(1)算出的绝热损失，在意义上是否等同于前两种方法算出的绝热损失呢？与我们传统的计算方法比较一下，可以看出有以下区别：

1）可逆式换热器蒸发热损失。空气带入可逆式换热器的水分，有一定数量在热端冷凝，该处冷凝水切换时被污氮冲走，未能恢复原状，故造成蒸发热损失。一般取冷凝水为进装置空气含水量的  $5\%$  ，其计算公式为：

$$
Q _ {3 - 2} = 0. 0 5 \times \Delta X \times r _ {\mathrm {H} 2 \mathrm {O}} \mathrm {k J / h} \tag {2}
$$

式中  $\Delta X$  ——空气  $(V_{K})$  中总的含水量

$\pmb{r}_{\mathrm{H2O}}$  ——水的潜热，为  $2512.08 \mathrm{~kJ} / \mathrm{kg}$

由于水的潜热值很大，该项算起来数值颇可观。

2）低温吸附器冷损。对液空、液氧吸附器均按以下公式计算：

$$
Q _ {3 - 3} = 0. 0 1 V _ {K} \times 4. 1 8 7 \mathrm {k J / h} \tag {3}
$$

此项冷损所占比例不大。

3）液氧泵冷损。按以下公式计算：

$$

# 三、三种计算方法的选择与讨论

究竟哪一种方法较准确较切合实际呢？

从第①种方法来看，主要特点是依据装置容量的大小选择不同的数据。但是数据选择的范围弹性较大，且空气量一般都在5位数以上，容易造成大的误差，而且这些数据的提出在时间上也距今较远。

第 ② 种方法仅按保冷箱面积的大小考虑，然而保冷箱的大小并非同装置容量的大小成等比例变化。

第①种方法和第②种方法在以上计算中数值比较接近。过去我们按第一种方法计算的绝热损失及最后确定的膨胀量，由于缺乏实测数据，不知与实际值究竟偏差多少。但是，从现场情况看，实际膨胀量一般小于设计膨胀量这一点是可以肯定的。如  $xx$  钢管厂  $3200\mathrm{m}^3 /\mathrm{h}$  （代号4020）、  $\times x$  钢厂和  $xx$  冶炼厂等处  $3200\mathrm{m}^3 /\mathrm{h}$  （代号4005B）透平膨胀机导流叶片开度，冬季一般仅  $20\% \sim 25\%$  夏季为  $40\% \sim 50\%$  。这说明偏差的确比较大。

第③种方法考虑的因素比较全面，比较

细致。其经验公式是在实际测试几十套空分设备之后总结而得。特别是公式(1)，用指数关系进行描述，较好地反映了“容量大，单位冷损小；容量小，单位冷损大”的数量关系。因此，选择第③种方法作为绝热损失计算的基本方法是比较适宜的。但是选择该方法还需要与我们的具体情况结合起来。譬如：我国保温材料的性能；我们设计的冷箱一般比实际要求偏大一些，以及与实际密封情况有关等等。

如果我们能够组织力量进行实际测定，确定出具体的系数当然是最理想的，但这是很难做到的。

在目前情况下，对可逆式流程，推荐(1)式中的  $K_{1} + K_{2}$  取值范围在  $0.8\sim 1.0$  之间。一般来说。对于带稀有气体提取装置的空分设备来说，推荐  $K_{1}$  取0.95，取  $K_{1} + K_{2} = 1$  。

# 地球上的氧气在减少

前些时候，美国耶鲁大学的科学家们，从自然界得到一块8000万年前的玛瑙，发现其中密封着一个气泡，经检测后得知，在这个8000万年前的气泡中，氧气的含量高达  $30\%$  。由此推断，当时地球大气中氧气含量为  $30\%$  ，但现代大气中氧气含量仅为  $21\%$  。因此，在已消逝的8000万年中，大气中的氧气已经减少了  $30\%$  ，这无疑是自然界给人类发出的一个紧急报告！要知道，氧气与地球的生命息息相关，对于人来说，只要5分钟没有氧气，就会窒息而死。

据考证，地球大气生成于40亿～4亿年前这一漫长的岁月。大气中的氧气是数十亿年来绿色植物光合作用积累下来的。远古时代，森林曾覆盖地球陆地面积的2/3，达76亿公顷。到1862年，已减少到55亿公顷。目前仅剩下28亿公顷，而且还以每年1100万公顷的速度减少。素有地球“肺叶”之称的森林，光合作用时吸收二氧化碳，放出氧气。据测定，1公顷森林每天吸收二氧化碳1吨，放出氧气0.74

吨，一年就能吸收二氧化碳365吨，放出氧气270吨。一年减少1100万公顷森林，就少吸收二氧化碳40亿吨，少放出氧气30亿吨。同时，据估算，全球每年向大气中排放的一氧化碳约50亿吨，须知每生成1吨二氧化碳，至少要消耗0.73吨氧气，排放50亿吨二氧化碳需消耗氧气达36.5亿吨。

由于全球性的毁林开荒，大量使用煤、石油等化工燃料，以及耗氧性化工产品合成等原因，使得地球上氧气的消耗率超过再生率。据环境科学家们测算，近100年来，大气中的氧气消耗了约5000亿吨。按此速率，地球上的氧气尚可维持180万年，但实际上当氧气消耗到现在有浓度一半时，人类就无法生存下去了。因此，保护森林，大力植树造林；控制煤和石油的用量，开发风能、水能、太阳能等，加速氧气的再生和减少二氧化碳的排放，是刻不容缓的事情。

# 一、氮的应用

自从1917年第一次提取氦以来，氦的应用就得到不断发展。

# 1. 气球的充填

由于氦的密度比空气低许多，可以使气球高高地升在空中，同时用氦气取代氢气充填气球，可降低气球的危险性（1994年在温州地区就发生氢气球着火爆炸并烧伤学生的事故）。故氦气用于大型宣传气球的充填、玩具气球的充填，也可用于天气预测的大气球的充填。我公司就曾向一些晚会赞助氦气，用于晚会现场所有气球的充填。

# 2.焊接

由于氮具有很强的惰性和良好的热传导性能，使之适合用作焊接不锈钢、铝、铜、镍铜、钛和其它特殊的合金钢时必需的保护气。

# 3. 气体色谱

由于氮的惰性和良好的热传导性能，它常用作载气去分析化学物质的纯度和组成，如各种止咳糖浆和天然气。

# 4. 深海潜水

它可用作2000英尺潜水的呼吸混合气。这不仅可进行海下科学探测，而且可进行海洋石油和气体的探索和生产。

# 5. 检漏

由于氮的分子尺寸小，使之成为在检测压力和真空系统泄漏时最理想的物质。它的泄漏检测能力已被大量事实证明是非常珍贵的。用于航空和航天、半导体、低温真空、制冷和食物包装工业，也用于许多科学实验室；在卫星、航天飞机送入轨道前，它们每个电子仪器的、燃料的、生命支持系统都必须用氮进行泄漏检测并仔细检查，以确保它们在空间正常运行。

# 6. 宇宙航行

由于氦的惰性、溶解度小和沸点低（4.2K），它是仅有的在液氢燃料温度下，仍保持气态的惰性气体，故可用作发射卫星和航天飞机的火箭内液氢燃料系统的预冷和压缩，并可作为火箭燃料的推进剂。

# 7. 高能物理和核物理研究

在高能粒子的研究中，由于液氦温度比液氢的温度低，因此在液氦气泡室内高能粒子间产生干涉机会比在液氢气泡室内要少，另外需要液氦冷源的数量也较液氦少，如同等效率的气泡室，用液氢需  $1500\mathrm{L}$  ，用液氦只需  $200\mathrm{L}$  。

# 8. 超导核磁共振成像设备(MRI)

在医学的放射性诊断领域，液氮用来冷却MRI的磁体，从而使它们产生超导效应

和非常强磁场，把病人放在MRI装置前，射线穿过人体组织获得非常清晰的病理照片。

# 9. 超导技术

在液氮温度下，许多金属电阻变为零。因此，电流通过时不发热、不损耗，形成“超导电性”，因此可作体积小、重量轻的超导电机，如  $6000\mathrm{kW},50\mathrm{r / min}$  的常规电机重量达370t，效率  $94\%$  ，同样功率的超导电机重量只有40t、效率为  $97\%$  。利用金属的超导性还可制作超导贮能器、超导高速悬浮列车、超导电缆、超导电磁阀、超导强磁选矿等等。

# 二、一般的氮提取方法

氦的提取主要来源于含氮的天然气，从空分设备提取氖气时，也可以回收少量氦，一套  $40000\mathrm{m^3 / h}$  的空分设备每天只能生产13 $\mathbf{m}^3$  的氦气，而且成本高，比天然气提氦要贵  $150\sim 300$  倍。因此，目前国外从空分设备提取氮仅占氦生产能力的1/500，只有少数缺乏天然气的国家才从空分设备中提取。此外，氦还可从石油伴生气和合成氨尾气中回收。

在石油伴生气中，平均含氮  $0.06\%$  ，处理  $1000000\mathrm{m}^3$  石油气可获得  $540\mathrm{m}^3$  纯氮。以含氮的天然气为原料制造合成氨时，在合成氨尾气中的氮被浓缩  $4\sim 8$  倍。

# 二、空分设备的焊接特点

空分设备中的一般焊接结构件，如冷箱等，相对来说要求较低，需注意的问题是焊接变形问题。这一类，一般结构件焊后，通常只进行外观检查，检查焊缝的外观形状、尺寸以及焊缝表面缺陷、产品的形状与尺寸，无需进行焊缝内部探伤、焊接接头的强度试验等。焊接时，一般采用手工电弧焊、 $\mathrm{CO}_{2}$

气体保护焊，控制焊接线能量，选择正确的焊接顺序和方向，配以适当的工装夹具，焊接变形即可得以控制，满足相应标准要求。

空分设备中的焊接产品，除上述的一般焊接结构件以外，还有许多焊接容器。有一些焊接容器，如低温液体贮槽内胆容器、分馏塔的下塔等，属压力容器，其制造、检验和验收必须遵循相应的国标、部标和行业标准，如GB150—89《钢制压力容器》、JB4730—94《压力容器无损检测》、JB/TQ601—88《空分设备用有色金属焊接压力容器制造规定》等。另外的焊接容器，如分馏塔的上塔、粗氩塔等，虽然其最大工作压力低于0.1MPa，但由于其工作温度低于  $-100^{\circ}\mathrm{C}$ ，从空分设备行业特点出发，为保证产品的制造质量，生产厂家往往也把这一类容器作为厂内质量控制容器，即内控容器，参照一类压力容器的相关要求制造、检验和验收。

压力容器的焊接检验，通常包括焊缝外观检查（检查焊缝外观形状、尺寸以及表面缺陷等）、内部无损探伤（检查焊缝内部缺陷）、焊接接头力学性能试验（拉伸试验、弯曲试验、冲击试验等）、焊缝强度试验和气密性试验等。为保证焊接压力容器的焊接质量，在

选择焊接方法和制定相应的焊接工艺时，必须解决焊接性问题。

焊接性包括材料的焊接性和结构的焊接性两方面的内容。材料的焊接性是不考虑产品结构形式，仅仅只针对材料本身，选择合适的焊接方法和焊接工艺使其形成具有一定使用性能的焊接接头的能力；而结构的焊接性则是要充分考虑产品结构形式，选择合适的焊接方法和焊接工艺使该焊接结构具有一定使用性能的能力。从材料的焊接性来分析，空分设备中的焊接容器用材料除铜及铜合金的焊接性较差外，其它材料的焊接性都较好。

从结构的焊接性来分析，由于焊接容器多采用全焊透方式，所以产品的具体结构对其焊接性影响很大。有的产品结构焊接性较好，可以采取双面焊工艺，容易得到高质量的焊接结构；而有的产品由于焊接空间的制约，只能采用单面焊双面成形工艺，或由于结构的刚性大而导致较大的焊接应力等。这就容易引起焊缝背面成形不良、未焊透、裂纹等焊接缺陷，使得产品的结构焊接性变差。

# 三、铝制产品的焊接制造技术

空分设备中铝制容器很多，分馏塔、冷凝蒸发器、氩塔等，都是由铝合金制成的，材质多为Al-Mg系的LF2、LF4（相当于国外的5083、AlMg4.5Mn)，也有少许的AlMn系的LF21。另外，空分设备中大量的铝管道也是由LF2或LF4制成的。

# 1. 焊接性简要分析

LF2、LF4和LF21都是不能热处理强化的铝合金，其材料焊接性较好，在实际的焊接生产中易出现的问题是：焊缝气孔、焊缝夹杂、焊接接头抗拉强度偏低和塑性不足。

就焊缝气孔、夹杂这两种最常见缺陷而言，它们都与焊丝及母材的表面氧化膜有很大关系。铝合金化学活泼性很强，表面极易形成高熔点的氧化膜，焊接时容易造成不熔合现象，也容易成为焊缝金属的夹杂物。同

时氧化膜不很致密，吸附较多水分，导致焊缝气孔。因此必须严格焊前清理，彻底去除焊丝及母材表面氧化膜。采购进厂的焊丝，除非已经清洗、干燥且密封包装，否则使用前须经化学清洗、干燥；工件坡口及附近区域的表面氧化膜，最好采用化学清洗和机械清理加以清除。化学清洗后及时施焊，施焊前再用刮刀或铣刀刮、铣坡口端面。另外，为防止焊缝气孔，还要控制焊接工艺，选择合适的焊接规范，加强焊接保护。氩弧焊时利用“阴极破碎”作用去除氧化膜，既要限制氢向熔池的溶入，又要促进氢自熔池向外逸出。

LF2、LF4和LF21在焊接时必须严格控制焊接工艺和焊接规范，以保证焊接接头的力学性能。上述的三种不能热处理强化铝合金，若在冷作硬化状态下焊接，接头有软化现象。筒体、管道即便选用了退火状态的板材，其在剪切下料、铣切焊接坡口、卷圆时，坡口及其附近区域已进行了部分冷作硬化，焊接时接头必然会有一定程度的软化。就焊缝而言，其性能主要取决于所选用的焊丝，焊接工艺规范也有一定程度的影响；对于熔合区，晶粒粗化必定降低塑性；对于热影响区，有一定程度的软化。一般而言，焊接线能量大，则焊接接头组织过热严重，性能下降，抗拉强度偏低，塑性也不足，达不到相应标准的要求。故此，为保证焊接接头力学性能，一要正确选用焊丝，二要严格控制焊接线能量。表1为氩弧焊焊接LF2、LF4及LF21时常见的焊丝选用表。

熔化极氩弧焊可用在板厚  $3\mathrm{mm}$  以上的产品焊接上，直流反接；具有良好的“阴极破碎”作用。与TIG焊相比，生产率高，焊接变形也小，熟练掌握之后，焊接质量也较稳定。另外，现在所用的MIG焊丝表面已经专门的化学或电化学抛光处理，表面形成一层极薄而均匀的氧化膜，含氢低，所以焊接气孔较少，也无需在焊前进行清理。

# 4. 铝制产品的组合焊工艺及应用

前面已分别介绍了空分设备中铝制产品焊接生产中常用的手工钨极氩弧焊(交流)、自动熔化极氩弧焊和半自动熔化极氩弧焊的工艺，各种焊接工艺都有其特点，既可单独使用，同时也可视产品的具体结构和要求，组合使用。

手工TIG双面双人同时向上立焊工艺很成熟，主要用于  $10\mathrm{mm}$  以下板厚的铝制容器和管道的对接焊缝焊接。  $10\mathrm{mm}$  以上时，可采用双人双面同时立焊打底加手工TIG平焊盖面或半自动MIG焊盖面；或双人双面同时向上立焊打底加半自动MIG焊中间过渡，自动MIG焊盖面。无法进行双人双面同时向上立焊时，也可采用加永久性垫板的单面焊(环缝、手工TIG)加半自动MIG焊盖面等组合工艺。对于角焊缝，常见的组合工艺有：双人双面同时立焊打底加手工TIG焊平焊盖面或半自动MIG焊盖面。

5. 铝制产品焊接生产中应注意的几个问题

# （1）加强焊接材料的质量管理

前面已述，焊接接头的质量好坏，与焊丝、氩气有很大关系，氩气纯度不合要求会导致焊缝气孔；焊丝含Mg量偏低对于LF2、LF4焊接将造成焊缝抗拉强度偏低，达不到相应标准要求，所以最好应向质量高的生产厂家定点采购焊接材料。

# (2）合理选用焊接设备

对于手工交流TIG焊，普通焊机即可满足要求，而对于MIG焊，则应选用高性能的焊机为宜。特别是对于细丝  $(\phi 1.6\mathrm{mm})$  自动或半自动MIG焊，由于送丝比较困难，而送丝的可靠稳定与否又决定着焊接质量的高低，所以要注重送丝机构的性能，最好选用规范一元化调节的高性能焊机。

（3）施焊时正确操作，严格控制焊接线能量

前面“焊接性”分析中已述，焊接线能量对接头力学性能有很大影响，过热组织将大大降低接头性能，使接头强度、塑性达不到相应标准要求，所以必须控制焊接线能量。一般认为，对自动MIG焊，焊丝不宜太粗。

（4）采取合适的工艺措施抑止微小焊接缺陷，保证接头力学性能

从一些力学性能不合格试样（试样从经X射线探伤合格的产品试板上截取）断口分析可知，除了组织过热之外，一些微小的焊接缺陷如微气孔、微小夹杂往往也是造成焊缝抗拉强度偏低、塑性偏低的主要原因，所以应视具体的产品结构，采取预热、控制线能量、严格焊接坡口清理等工艺措施，来减小气孔倾向，避免夹杂，减小微裂，以保证焊缝力学性能。

# 四、本装置的主要特点

1. 本套制氧机一改过去加氢除氧制氩的传统工艺，采用了全精馏制氩新工艺。下塔、上塔、粗氩塔和纯氩塔全部采用填料塔。比起老工艺节约设备和投资，并且不使用氢气，提高了安全可靠性。目前传统的加氢除氧制取精氩的工艺流程见图2。由于填料塔采用规整填料，故压降低，本装置粗氩塔(粗氩I塔+Ⅱ塔)的阻力为80mbar,纯氩

塔的阻力为12mbar。

本装置中，由于要制取五个九的高纯氩和高纯氮，因而理论计算上塔和粗氩塔的高度很高。此设计中，把上塔和粗氩塔都分别截成两节，减少了塔和冷箱的高度，现冷箱的高度为  $30\mathrm{m}$  ，上塔I塔和Ⅱ塔之间用液氧泵P3566来连接，粗氩I塔和Ⅱ塔之间用液氩泵P4566来连接。

2. 采用分子筛流程净化空气中的水分、二氧化碳、乙炔和其他碳氢化合物。分子筛吸附器的填充直径为  $1884 \mathrm{~mm}$ ，填充总高度为  $1000 \mathrm{~mm}$ ，其中下层为活性氧化铝，高度  $200 \mathrm{~mm}$ ；上层为  $13 \mathrm{X}$  分子筛，高度为  $800 \mathrm{~mm}$ 。分A、B两筒，每4.3小时自动切换一次。  
3. 主控室使用PLC装置，型号为西门子S5—135U。PLC的主要功能是：监控工艺过程的参数（温度、产品流量、压力、液面等）；工艺流程各系统画面显示；控制器数据的显示和修定；极限值的显示和修定；警告显示和停机报警；设备运行条件的显示；联接记录打印机；分子筛吸附器的自动切换程序。  
4. 采用氨冷冻装置, 使空压机出来的加工空气冷却至  $8 \sim 10^{\circ} \mathrm{C}$ , 满足进入分子筛吸附器的条件。用氨作为冷冻装置的制冷剂, 在冷冻机的压缩机、冷凝器、膨胀阀、蒸发器中完成制冷循环, 同时将冷量在蒸发器中传给冷冻水, 然后冷冻水进入冷却器使加工空气冷却。  
5. 自动分析仪。本装置在线自动分析仪共配备7台，分别对产品氧、产品氮、氩、污氮、粗氩、氩馏分进行分析，并对分子筛吸附器出口的加工空气中的二氧化碳含量进行分析，以监视吸附器的工作效果，保证设备长期正常运行。

8. 配有油制动透平膨胀机。膨胀机组由膨胀机本体和供油装置两部分组成。膨胀机两轴承间的转轴部分兼作油制动器的转子，在油制动器的转子和定子之间通入一定压力的润滑油，制动器的转子被三个制动室包围，其中内制动室的油压可调，而外制动室的油压由供油装置的油压决定。通过油量和油压的调节，就可以改变膨胀机的制动负荷和转速，功率通过摩擦转换成热量，由润滑油带到冷却器中冷却。可调喷嘴由微型电机驱动，调整喷嘴流通截面进行进气量的调节。供油装置由装有油泵与电加热器的油箱、油冷却器、油过滤器、压力油箱、油压调节装置等组成。  
9. 本装置未配备仪表空压机，仪表空气需用量为  $50\mathrm{m}^3 /\mathrm{h}$  左右，全系统启动时仪表空气须另外供给，我单位由老制氧机提供。

# 五、安装、调试及改进

制氧机组投产后能否长期安全连续的运转，除与该机组各单机设备本身的质量和装备水平有关外，还与设备安装和调试的质量直接相关。因此，在该设备安装、调试过程中，我们自始至终坚持质量第一的方针。

1. 选好设备安装的施工单位是保证安装质量的前提

林德  $1500\mathrm{m}^3 /\mathrm{h}$  制氧机工艺流程先进，设备装备水平高，仪表控制系统复杂。另外，在订购该设备时，我公司为节约投资，冷箱部分未订整件，只订散件，即冷箱内各容器和管道阀门等全部发单件到厂，进行现场接管组装冷箱，节约了设备投资。但另一方面，大大增加了装置安装的难度。冷箱内各部件的焊接全部是氩弧焊，而冷箱质量是整套空分设备安全连续运行的关键。为了确保冷箱的安装质量，我们对于冷箱的组装和安装工作采取在国内各制氧机制造厂招标的办法，最后四川空分设备厂中标，负责冷箱的组装和安装工作。仪控部分，由我公司计控部负责。机械、电气、管道等部分，由我公司设备建装安装公司承担。

2. 坚持抓好焊接质量不放松

对于空分设备安装工作中焊接质量的要求，德方技术人员和我方管理人员一致坚持高标准严要求的方针。除冷箱内焊接必须使用氩弧焊外，不锈钢也必须使用氩弧焊，其他碳钢管道也必须先用氩弧焊打底。

# 3. 全精馏制氯部分流程的修改

由于林德公司首次将全精馏制氙新工艺推广应用到  $1500\mathrm{m}^3 /\mathrm{h}$  空分设备上，在冷箱内部安装完成后，德方先后三次对冷箱内制氙部分作了以下改造：

（1）纯氩塔顶部与纯氩塔冷凝器之间连接管的改造

原设计纯氩塔顶部与冷凝器底部用一根 $\phi 40\mathrm{mm}$  管道连接，此管道靠近纯氩塔顶部的这一段既作为纯氩塔的通道，又作为冷凝液回流到纯氩塔的通道。此  $\phi 40\mathrm{mm}$  管中部开孔接一根  $\phi 25\mathrm{mm}$  的支管，支管接至冷凝器顶部，作为蒸汽进入冷凝器的通道。改造后，去掉了支管，原  $\phi 40\mathrm{mm}$  管路只作为冷凝液的回流通道。在纯氩塔顶部两侧新开一孔各接一根管道，这两根管道作为纯氩塔顶部蒸汽的通道，合为一路后以  $\phi 40\mathrm{mm}$  管道接至冷凝器顶部。

（2）粗氩塔冷凝蒸发器的液空管路作了较大改造：

① 冷凝蒸发器的液空流程，原设计分为底部、上部两路进，然后从冷凝蒸发器中上部抽出。现液空管路改为上部进、中上部出冷凝蒸发器。  
② 原设计冷凝蒸发器液空侧底部进口管路现改为排液管路，此管路上增设一个带孔截止阀4111。此阀门在停车时打开，有利于排尽冷凝蒸发器内液空，在正常运行时，此阀在关闭状态下，仍有一小孔可供液空流动，防止杂质积聚。  
③ 冷凝蒸发器 液空进料管路末端与液空出口管连接，连接处为一带  $\phi 4\mathrm{mm}$  小孔的半圆形封头，此改造也是为加强液空的流动

不致出现杂质积聚。

④ 同样也是从安全角度出发，在冷凝蒸发器中部有一  $\phi 5\mathrm{mm}$  小孔，从此小孔中抽出少量液空，原设计这部分液空排入进上塔液空管路，现改为排入液空蒸汽管路进上塔。  
⑤ 原设计粗氩塔冷凝蒸发器向纯氧塔的冷凝器供给液空，现改为液空直接分一部分进入纯氩塔冷凝器。  
⑥ 增设微量氧分析仪吹扫气。从液氮罐进料管上引出少量液体，经汽化器汽化复热后，供给微量氧分析仪作为吹扫气，即在试样气含氧高时，可将此气体吹扫分析仪，防止微量氧分析仪内部气路被含氧高的气体所污染。  
⑦ 在主塔冷凝蒸发器至下塔的液氮回流管路上新开一孔，加装一根  $\phi 25\mathrm{mm}$  的管道，与冷凝蒸发器氖氦气排放管路直接连通。

# 4. 透平膨胀机控制系统的改进

本装置配套的透平膨胀机采用油制动。1996年4月15日试车过程中，启动空压机准备吹扫空分塔，当将空分塔之空气入口阀F2615打开时，几分钟后，就听到膨胀机发出异常响声，并发生冒烟，此时，立即全系统紧急停车。停车后，膨胀机温度很高，外壳烫手不能触摸。经检查，发现膨胀机手动进口阀、出口阀均未关闭，膨胀机进口蝶阀（即紧急关闭阀)也未关。转速表、可调喷嘴电机的电源线也未接，油泵未启动。可见，膨胀机是在无油的情况下启动的，并且肯定发生了膨胀机“飞车”现象。结果此台膨胀机损坏，德方重新更换一台，损失较大。后经双方协商，对膨胀机系统作了如下改进和操作调整：

控制系统中，增加膨胀机的油泵启动后，才能打开空分塔空气入口阀FV2615控制程序，杜绝膨胀机无油启动，造成事故。

膨胀机的密封气原设计采用压力氮，必须在FV2615打开后，膨胀机才能得到密封

气，现改为采用分子筛净化后的空气。这样，只要分子筛装置开始工作就能为膨胀机提供密封气。

膨胀机启动时，应先供给密封气。如果膨胀机启动时，无密封气，油系统的油就有可能进入前后管路。

在调试过程中，多次发生空分装置临时停车。再启动时，膨胀机的机后温度急剧下降至  $-185^{\circ}C$  左右，稍有不慎，就会发生膨胀机液击现象。为了确保膨胀机的安全，我方要求德方在膨胀机的机前利用原温度指示，增加一个温度过低报警，有利于防止膨胀机温度过低出现液体损坏设备。

# 5. 关于低温液体泵的停车密封

本装置冷箱部分的工艺流程中，配备了两台离心式低温液体泵P3566和P4566。P3566为液氧泵，作为上塔I和上塔Ⅱ之间的液氧循环用；P4566为液氩泵，作为粗氩I塔和Ⅱ塔之间的液氩循环用。这两台低温液体泵原设计中配有停车密封装置，防止低温气泄漏，即在泵正常运行时，使用密封气起低温密封作用，而在停泵时，不再使用密封气，而由专门设计的机械密封自动起作用，防止泵轴处泄漏低温液体。在设备调试中，由于控制停车密封的电磁铁，其控制线路接触不良等原因，使两台泵的停车密封都被损坏，于是在泵停车时，泵轴处泄漏严重。现采取停泵时立即关闭泵的进出口阀的临时方法来解决泄漏问题。德方同意提供两套机械密封件来更换。

# 6. 设备调试时空压机空气量越来越小

1996年6月11日，调试开始时，空压机入塔加工空气量可达  $8350\mathrm{m}^3 /\mathrm{h}$  压力为4.85bar；7月6日，空气量只有  $7600\mathrm{m}^3 /\mathrm{h}$  ，压力4.5bar；7月10日，空气量继续下降到只有  $6600\mathrm{m}^3 /\mathrm{h}$  ，而此时，空压机进口导向叶片开度已达  $100\%$  ，无法再增量。多次检查导向叶片和空气过滤器都只能解决暂时问题，

时间一久，空气量又减少了，无法运行下去。经反复查找问题，发现空压机导向叶片前的进气管道上还配有一个锥形过滤器。由于进气管道中还有一段碳钢管，此管铁锈较多，空气量减少原因是由于铁锈杂质堵塞锥形过滤器所造成。后取消锥形过滤器，空气量立即恢复正常，这说明中间的一段碳钢管应作防锈处理。空压机生产厂在说明书中作出规定，此锥形过滤器只在试车时使用，试车结束投入正常生产时，应取掉它。

# 7. 关于冷冻机问题

本装置所配冷冻机为林德公司制造。配有两台氨压缩机，制冷量可自动调节，每台有  $100\%$  、  $66\%$  、  $33\%$  三档制冷量。我公司在调试过程中多次发生冷冻机故障自动停车。其中一部分原因是由于冷却水未能将冷凝器中的氨冷凝下来，但大部分原因是由于其中一台氨压缩机油位低自动停车，这时另一台全量  $100\%$  运行冷量仍然不够所致。我们认为林德冷冻机选型偏小，冷量不足，在夏季不能做到两台氨压缩机一开一备，必须两台同开。

经过观察和分析，发现油位下降的氨压缩机总是制冷量小的一台，这是因为两台氨压缩机从同一个低压贮槽中吸入氨，它们的进气压力应为一致。但是，当它们的制冷量不一致时，特别是在一台  $100\%$  ，另一台  $33\%$  状态运行时，由于  $100\%$  制冷量的氨压缩机的所有气缸都在工作，而  $33\%$  制冷量的氨压缩机只有三分之一气缸在工作，这样就会使制冷量小的氨压缩机的曲轴箱内氨蒸气压力大于制冷量大的氨压缩机曲轴箱内的氨蒸气压力。两台压缩机的油是用管道连通的，曲轴箱上部也用管道连通，可能上部连通管道不够通畅，致使压力未能平衡，因此油被压往制冷量大的一边，即曲轴箱内压力低的压缩机内，制冷量小的压缩机内油位就降低。

德方现将两台压缩机的运行方式作了改

变。由原来一台  $100\%$  不够时则另一台增加 $33\%$  的方式修改为两台制冷能力接近，最多只差一档的方式，即两台  $66\%$  不够时则增加一台到  $100\%$  ，如能力有余则一台减为  $33\%$  的方式，也就是说，两台制冷能力只能相差一档，目前运行正常。

8. 临时停车后再启动时，膨胀机的温度控制

在调试过程中，由于各种原因曾多次发生临时停车，在临时停车的再启动时，膨胀机的机后温度急剧下降至  $-185^{\circ}C$  左右，稍有不慎，就会发生膨胀机液击。调试中，德方均采用膨胀机减量、降速的方法，但收效不大，往往几小时后膨胀机温度仍不回升。经过观察分析认为，发生这种情况一般是由于在再启动时，主冷液面过低造成。由于主冷液面过低，空气就吃不进塔，入塔空气量仅为  $4000\mathrm{m}^3 /\mathrm{h}$  ，而且由于液面低，液空节流阀也自动关闭，过冷器失去回收冷量的能力，返流气入主换热器温度过低，致使主换热器过冷，造成膨胀机进口温度接近  $-160^{\circ}C$  ，机后温度当然也急剧下降。

我们认为，这时可不顾下塔液空液面将液空节流阀转为手动控制，适当开大，同时注意进塔气量增加情况。这样，一方面入塔加工空气量增大，另一方面过冷器发挥了回收返流气冷量的作用，主换热器工况也就会迅速恢复正常，膨胀机温度也能较快得到控制和回升。实践证明，这是十分可行的。

# 六、问题及建议

1. 本装置设计中，冷箱主换热器的返流气出口温度一般在  $10^{\circ}\mathrm{C}$  左右，而在重庆的夏天，厂房内温度高达  $38\sim 40^{\circ}\mathrm{C}$ ，因此，当设备在连续正常运行时，冷箱出来的产品氧、

产品氮、污氮等气体管道外表面生成大量的冷凝水，水滴不断下滴使厂房内地面到处潮湿，实属烦人，不利于文明生产。原设计，只对原料空气自冷冻机到分子筛吸附器，直至进入冷箱的每段管道，进行了隔热包扎处理。为此，建议对冷箱出口的氧气、氮气、污氮等管道，同样应采取隔热包扎的方式解决冷凝水问题。  
2. 为了对产品液体氩进行纯度分析，本装置中配置有气相色谱仪，对纯氩塔底部产品纯氩出口管上的纯氩进行分析。色谱仪为ORTHODYNE公司生产，型号：AL800型。但是这台色谱仪只能对液氩中的微量氮进行分析，不能分析其他的杂质成分。纯度 $99.999\%$  Ar的产品属高纯氩产品，国家标准对高纯氩中氧、氮、氢、水分和以甲烷为主的碳氢化合物的含量均作了明确规定，因此必须进行检测分析。目前我单位正在采取其他补救方法。  
3. 本装置所配套的氧气压缩机, 为瑞士 Sulzer 生产的迷宫式三级压缩机。总体情况较好, 但有一点不理想, 即氧压机的一级、二级、三级之间的连接管道 (长度都很短) 采用碳钢无缝管材料。设备的其他冷却器、阀门、管件、输送管道均采用不锈钢材料。如氧压机经常停机, 碳钢管将产生铁锈, 不利于安全, 应全部使用不锈钢材料。  
4. 对于液氩和液氮产品，设计中都配有容积为  $3\mathrm{m}^3$  液氩罐和液氮罐。实际使用中，液氩贮罐容积偏小，目前市场上常用的液体槽车多为  $4\mathrm{m}^3$  以上，因此液氩贮罐宜选用等于或大于  $5\mathrm{m}^3$  容积。

主题词：空气分离设备 寄冷器型 改造 更新 分子筛型 内容 效益

# 一、改造理由及经济效益

根据柳钢生产的发展，“八五”计划中后期，每小时必须提供  $8000\sim 9000$  立方米氧气，才能满足生产的需要。这些氧气，由三台制氧机提供。其中的  $3^{\#}$  制氧机是从日本引进的，每小时可提供氧气4000立方米(标态，下同)以上，是柳钢氧气生产的主力机组；其余的  $4000\sim 5000\mathrm{m}^{3} / \mathrm{h}$  氧气由国产  $1^{\#},2^{\#}3200$ $\mathrm{m^3 / h}$  制氧机组共同提供，它们的实际产氧量每台只有  $1800\mathrm{m^3 / h}$  ，单位制氧电耗高达1.2 $\mathrm{kWh / m^3O_2}$  ，设备故障频繁，运行很不可靠，严重地影响柳钢生产的顺利进行。因此，保证氧气供应已经成为柳钢生产的顺利进行和进一步发展的燃眉之急。解决的办法有两：一是立即建设新机组；二是改造“3200”。

经过技术论证和方案比较之后，柳钢决定用分子筛净化空气的新技术对蓄冷器流程的  $2^{*}3200$  首先进行改造，由开封空分设备厂负责改造设计及提供改造设备，于1992年7月改造成功投入运行。改造后的  $2^{*}3200$  产氧量达  $3400\mathrm{m^3 / h}$  ，单位制氧电耗为0.65  $\mathrm{kWh / m^3O_2}$  ，从而实现了用一台改造机代替两台未改造机，每年节省电力  $1753.46\times 10^{4}$  kWh，节省操作及维修费用约545万元（按

1992年电价0.25元/kWh计),所增加的氧气每年可增产钢材约18万吨。

$2^{\#}$  机组改造成功，解除了柳钢生产上氧气供应紧张的燃眉之急。随后柳钢又立即着手用全新的分子筛流程对  $1^{*}3200$  进行更新，由杭州制氧机厂协助改造，于1994年7月建成投产。更新后的  $1^{*}3200$  ，产氧量为 $3800\mathrm{m}^3 /\mathrm{h}$  ，还产氯  $70\mathrm{m}^3 /\mathrm{h},$  至此，两台改造及更新后的“3200”机组加上一台从日本引进的“4000”机组，每小时共能提供11000立方米的氧气，不但满足了柳钢“八五”中后期生产对氧气的需求，而且对“九五”前期的氧气供应也不成问题。

两台“3200”机组的更新改造总费用约为2550万元（其中  $1^{\#}$  为1900万元，  $2^{*}650$  万元），而若新建两台“3200”机组，则至少要花费7000万元的投资和大约三年的时间。

# 二、 $2^{*}3200$  的改造

$2^{\#}3200$  于1979年投产，为蓄冷器环流流程；  $1^{\#}3200$  于1972年投产，为蓄冷器中抽流程。考虑到  $2^{\#}3200$  机组投产时间比较短，设备相对比  $1^{\#}$  机组新，故柳钢决定先对 $2^{\#}$  机组进行局部改造，目的是为了省钱及争取时间，并由开封空分设备厂负责改造设计

及提供改造设备。但这种改造，保留了旧冷箱及重新利用一些旧设备，麻烦事很多。尤其是当时对旧设备未进行过细鉴定，就直接重新利用，改造中发现旧主冷严重串漏，无法修补，于是临时换一台新的；在试机过程中，又发现下塔有毛病，污液氮纯度很差 $(18\% \mathrm{O}_2)$ 。所以，考核试机时， $19300\mathrm{m}^3 /\mathrm{h}$  的原料空气（含仪表空气及切换损失，下同），仅能生产  $3440\mathrm{m}^3 /\mathrm{h}$  的氧气，氧提取率相对较低。后来利用设备检修的机会，对旧下塔进行了修理，更换了全部塔板及内筒，修理后，污液氮纯度很好（ $< 2\% \mathrm{O}_2$ ），用19300 $\mathrm{m}^3 /\mathrm{h}$  原料空气，生产出  $3602\mathrm{m}^3 /\mathrm{h}$  的氧气，氧提取率达  $90.13\%$ ，接近国内新造分子筛设备的水平。本项目改造决算投资为650万元，解决了柳钢氧气供应紧张的燃眉之急。

# 1. 主要改造内容

（1）新增的设备：二台分子筛吸附器，二台主换热器，一台过冷器，一台热虹吸蒸发器，一台蒸汽加热器，一台电加热器，二台冷水机组，二台增压透平膨胀机。  
（2）部分改造的设备：空气压缩机、上塔、冷箱及管道、阀门等。  
（3）重新利用的设备：下塔、主冷、循环吸附器、污氮消音器、水冷塔等。  
（4）取消的设备：四台蓄冷器、四台自动阀箱、二台液空吸附器、一台液化器、一台末端冷却器，三台固定喷咀膨胀机。

此外，对仪表及电气进行了相应的更新改造。

# 红外测温技术在氧压机上的应用

揭水经

（三明钢铁厂炼钢厂仪表室，福建省三明市，365000）

摘要 介绍了红外测温系统的组成及主要技术功能，二次仪表工作原理。指出用该测温系统对运动中的氧压机活塞杆温度进行自动巡检和状态监视的效果十分显著。图2。

主题词：氧气压缩机 红外测温技术 原理 应用

# 一、引言

据初步统计，自1976年以来，国内各厂的卧式气缸无油润滑活塞式氧气压缩机共发生了十余起着火事故，它给我厂制氧生产提出了一个严峻的问题：如何从技术、管理等方面，切实采取有效措施，来杜绝此类事故的发生？据分析，这些事故的发生原因是多种多样的，其中活塞杆温度过高的故障是主要的，发生事故次数也最多。

为此，一方面必须对氧气压缩机本身进行必要改造，以防止活塞杆升温过高；另方面，必须对氧气压缩机设置温度测量系统，以对其活塞杆温度进行在线跟踪监测。我厂采取了这两项措施，氧压机投产一年多来，从未发生着火事故，实现了安全生产。

本文拟就活塞杆温度监测工作进行一些总结和探讨。

# 二、测温系统组成及主要技术功能

首先，应选择合适的测温方案。我们经比较后认为，因为活塞杆在不停地作往复运动，它是活动部件，不能用通常的接触式温度计来测温，故确定采用非接触式测温法——红外线测温法来测温。我们用西北光电仪器厂开发的  $\mathrm{SWX} = 04$  型红外温度巡测仪及温度探测器，组成三个活塞杆温度监测系统。运行结果表明，情况良好，它为操作人

员提供了一个操作依据。

按工艺要求，为每个测温对象——各级压缩的活塞杆都装设一个红外温度探测器，同一台氧压机的四个温度（一~四级压缩4个测温点）探测器的信号都输入到同一台红外温度巡检仪，以组成一个完整的测温系统，共有三套。其配置方框图见图1。

![](images/d49fbcc81fdbe98cbb35b99b69672967768b6a59c9252fe6e54bb18b7ff77d30.jpg)  
图1红外线氧压机活塞杆温度监测系统图

SWX=04型红外巡检仪是测温系统的核心，它是具有80年代水平的智能仪表，采用MCS=51单片微处理机为主机，对输入的各路信号进行切换，经放大、A/D转换后，送CPU进行处理，实现四通道温度值及报警状态显示，采用软件对输入信号的非线性作校正。它可实现参数预置、超量程显示、自校等功能。采用0.8时LED数码管直接显示点数及温度值，四排LED二极管作为越限报警状态显示，清晰直观。按键式操作使参数设点、选点、校验，使用操作方便。测温范围为  $0\sim 200^{\circ}\mathrm{C}$ ，正符合工艺测温要求。

# 三、二次仪表工作原理

SWX=04型红外温度巡测仪由输入信号采样放大、CPU运算控制、显示器、功能键、开关电源等几部分组成，原理框图见图2。

![](images/387128e50ee19e9ba7b0a8faac222a2a7fe15e77630e145c0c620761d0064524.jpg)  
图2红外温度巡测仪原理框图

# 1. 采样放大模板

它接受来自现场的多路红外温度探测器电流输入信号，经输入滤波后，经由仪表总线上送出的地址译码信号，实现输入通道切换。切换开关采用IC模板开关电路，与通道切换同步提供四组  $\pm 12\mathrm{V}$  电源输出，供探测器使用。电源容量为：  $+12\mathrm{V},200\mathrm{mA};$ $-12\mathrm{V},100\mathrm{mA}$  。被放大的输入信号经仪表内总线传送给CPU控制运算模板进行。

# 2. CPU控制运算模板

CPU 控制运算模板采用 8031 单片微处理器为主控制器，扩展 4KRAM 区，程序存储容量为 8K 字节，I/O 扩展为一片 8155 和一片 8255 接口电路。

由CPU控制运算板送出的地址，使采样放大信号通道被选通，经放大后的信号传送至CPU控制运算板的A/D模数转换器，A/D输出，与8155接口。8155和8255还分别作为键盘六位LED数码管显示器、报警LED二极管驱动电路的接口，通过40线插头座与键盘及显示器模板相连接。

采用8031单片微处理器进行软件设计，实现对输入的多路信号进行数字滤波、非线性校正、自诊断、参数预置、超量程显示等多种功能。软件设计采用模块化结构，以易于改变或扩充仪表功能。

CPU控制运算板上设有保护电路，使仅

表可靠性得以提高。并用  $3\mathrm{V}$  的可充电电池作为断电时RAM区数据保护。

# 3. 键盘及显示器模件

键盘及显示器模件直接与CPU控制运算模板连接。六只功能键用以实现人一机联系，参数的设置、修改与检查，均通过按键进行。六位LED数码管显示器用作点序、功能、温度值显示。显示器采用软件译码，为动态显示。四排报警LED二极管与输入通道相对应，以灭灯或平光显示的方式表示输入通道正常或处于越限状态。仪表后端子板上设有一组继电器触点输出，与外部音响器配接。

# 四、注意事项

由于该测温系统能对运动中的氧压机活塞杆的温度进行自动巡检和状态监测，故它已成为氧压机运行的重要安全措施之一，其使用效果是良好的，业已产生很可观的经济效益和社会效益。本系统的开发应用，是制氧仪表的一次成功尝试。

相对而言，此测温系统比较重要也较复杂，为了使系统能达到规定的技术指标、正常地工作，除了严格把好元器件的筛选、整机的调试关外，在系统设计、安装、使用方面，我们从工作实践中体会到，还必须注意如下几点：

1. 探测器应安装于坚固的基础上，避免振动。探测器的防尘片应保持清洁，以便红外线能顺利照射到光学系统上。  
2. 接线方面，因探测器输出信号微弱，为了防止干扰，探测器与巡检仪之间必须用屏蔽线连接，且要将巡检仪可靠接地。  
3. 为了准确测量活塞杆的真实温度，必须调准活塞杆材质及其工作状态下的辐射系数  $\varepsilon$  ，否则会产生误差。在调整过程中，应注意不碰触信号电缆两端的插头，以免引起接触不良所导致的示值不稳现象。

# 一、问题的提出

我厂于1992年从四川空分设备厂购进 $\mathrm{KFON} = 20 / 100$  型空分设备用来生产氮气，因我厂原有的一台  $\mathrm{G3V} = 20 / 200$  型膜压机及一台  $\mathrm{G2V - 5 / 200}$  型膜压机不能与分馏塔配套，故我们从杭州制氧机厂购进一台  $2\mathrm{Z}2 = 1.7/$  165型氮压机。

我们的生产工艺是：生产纯氮时（氮气纯度低于  $99.99\%$ ），由分馏塔产出的氮气经缓冲罐进入压缩机充瓶或充大罐；生产高纯氮气时（氮气纯度高于  $99.999\%$ ），为了保证氮气纯度，减少在工艺过程中高纯氮气的污染，自分馏塔来的氮气直接进入氮压机一级入口。由于出塔氮气的纯度与氮气的产量有关，因而对于氮气压缩机来说，既要满足生产纯氮时氮气的产量要求，又要能在生产高纯氮气时保证出塔氮气的纯度。当分馏塔送至氮压机的氮气产量发生变化时，须采取相应的措施稳定分馏塔的工况，实现2Z2-1.7/165型氮气压缩机与分馏塔的匹配。

# 二、几种匹配方案比较

1. 改变氮压机电机与主机的传动比，适当降低氮压机主机的转速，同时采用旁通阀调节。在生产高纯氮气时，将压缩机压缩后的高压氮气返回一部分至氮压机一级入口，通过旁通阀调节高纯氮气的回流量，保证分馏塔工况的稳定和出塔氮气的纯度。该方案实施较容易，但因氮气的重复压缩，增加了能耗，且可能影响氮气产品的纯度。

2. 采用变频调速，通过变频器改变供给压缩机电机的电流频率，从而控制氮压机转

速。在操作中，根据生产需要，控制出塔氮气的产量和纯度。同时，调整变频器的输出频率，使氮压机的排气量适应分馏塔的产量，实现分馏塔与压缩机的最佳匹配。

以上两种方案中，我们选用了变频调速来实现FON=20/100型分馏塔与2Z2=1.7/165型氮压机的匹配。

# 二、膜片（压缩系统）的调试与维护

# 1. 膜片的调试

因为德国进口膜压机的油缸部位没有油压表，这对调节油压和判断故障带来了一定的困难，油压上升的幅度需用气压上升的状况来进行间接地测定。在第一次启动膜压机时需按下列步骤进行：①手动盘车确保传动部位无异常现象；②手摇油泵注入向油缸中一定量的润滑油，但油量不必过多，防止启动过程中油、气压差过大，损伤膜片或给驱动部分带来一定的超负荷。然后手动油压自控阀的手动油柄（见图3），将油缸中气雾状的油排入传动箱，以保证润滑油和膜片的吻合效果；③将油压自控阀的控制螺纹按反时针调回原位，然后关闭膜压机的压力总输出自控阀。

启动膜压机后，按顺时针方面调节一级压缩的油压控制螺纹，一级气压缓慢上升到给定标准  $1.3\mathrm{MPa}$  时停止调节，当其压力稳

膜压机的压缩效果与膜片的振幅有关，而振幅的大小与油压有关，油压偏低膜片的振幅自然缩小，压缩量和压力自然降低。如果油压超出膜压机的设计范围，膜片振幅的中心线会向气侧方向偏移，造成气缸的可变容积相应缩小、吸气量降低、气压下降等，更重要的是会导致多种不安全因素的产生。所以在调节油压时，不但要注意观察气压的上升状况，而且还要注意油压控制螺纹的进度，以防止膜片质量差或油压不能进行紧密吻合协振时出现的误调节。

# 2. 压缩系统的维护

在使用膜压机时对压缩系统的维护还应注意两点：①按时排放高压气体释放出的水分，以防水离器中水位过高，水分聚积于整个气压系统，影响介质的纯度和氧化气压系

统，降低吸排气阀封闭效果、压缩量及膜片的使用寿命。②在紧急停车紧急启动时必须手动排放高压气体，防止膜压机传动部件负荷过大，导致传动皮带打滑、电动机过电流的情况产生。同时还应手动油压自控阀，将一定量的高压油排回传动箱内，防止油、气压差过大而损伤膜片，使油、气压基本上同步上升。膜片在启动时破裂，往往就是这一原因所造成的。

# 三、压缩系统的故障判断

压缩效果差最明显的反映就是输出压力下降，达不到设计工况。压缩系统的运动部件都密封于整个机体内，看不见摸不到，但无论哪一部件出现故障都会影响整体的压缩功能，同时以不同的形式表现出来，如温度变化、异常响声、输油管道的油压脉冲失控等。因此应按相应的方法进行判断，如看、摸、听、闻。下面从四个方面来阐述膜压机的故障判断。

# 1. 膜片失去正常功能

膜片故障判断按德国压缩机的设计要求应由探伤装置进行，但由于设计方面的不足之处，即信号输出源的管道狭小或因长期使用管道堵塞，这一装置也只能探测膜片的破裂现象，对于膜片长期使用失去弹性的故障却无能为力，所以只能凭机体外部表现的物理现象来进行分析。

当膜压机压缩效果差时应首先考虑膜片的正常状况，主要是防止膜片整体破裂后，油缸中的润滑油经裂口处进入气压系统，给生产带来更大损失。

# (1) 表现形式

A.二级吸入压力偏高，输出压力明显下降。B.手感接近于吸气阀处的吸入管道温度有所上升，接近于排气阀处的排气管道温度明显下降。C.手感输入和输出油压管道的脉冲强度：若膜片失去弹性或部分破裂时均属正常；若整体膜片破裂，输出油压脉

冲强度下降。

# (2）原因分析

在油压正常的情况下，膜片失去正常功能时，与油面的吻合协振效果差，也就是反弹能力弱，导致气缸的可变容积小，吸入量降低，造成吸入压力偏高，输出压力明显降低。由于压力的变化导致吸入管道的温度有所上升，排出管道的温度明显下降，其下降的温度还包括排出量减少、管道摩擦系数降低时的下降温度。当整体膜片破裂后，超出额定范围的油量，一部分经膜片的裂口处进入气缸造成油压自控阀的启闭频数减少，油压脉冲强度下降。

# (3）处理方法

首先打开水分离器的排水阀，在排水和卸压过程中如发现水中含油，则进一步证明整体膜片破裂。这时除及时更换膜片外，还应对气路系统进行脱脂处理。若没有油污存在，则说明部分膜片破裂或失去弹性，更换下的膜片可按膜片鉴别一节的方法选择性地处理。

# 2. 油压偏低

造成油压偏低有四种情况：①输油管道存在空气或杂物堵阻，应排放空气或疏通管道。②油压自动控制阀失控，应予调整或修复。③油缸活塞体上导向环密封失效，应更换。④传动箱内连接件松动或导常响声，影响油缸正常工作，应检查并纠正。

# 3. 吸入量减少

在油压、膜片正常情况下造成吸入量减少，主要有三个方面的原因：吸入阀封闭不严；排气阀封闭不严；输出管道有泄漏存在。

# （1）吸入阀封闭不严

吸入压力上升，排出压力下降，接近于吸、排气阀处的吸入管道温度明显上升，排气管道温度明显下降。若阀门的导向部位或弹簧发生故障，则还伴有异常的响声。

分析其原因为：由于吸气阀封闭不严，

在压缩过程中介质气体产生双向运动，造成吸入压力增高，压缩介质减少，使二级排出压力明显下降。吸入管道的温度包括压力增高、介质双向运动及阀门产生故障时摩擦温度的总和。排出管道下降的温度包括高压介质压力下降、及排出量减少摩擦系数降低时反映温降现象的总和。

阀片封闭不严可以研磨，其它原因则必需更换。

# (2）排气阀封闭不严

吸入压力上升。因为压缩机的输出端有压力自控阀控制，所以排出压力基本不变，单位时间内的充装量减少。吸入管道温度有所上升，排出管道温度明显上升。

造成吸入压力上升的主要原因是排气阀封闭不严，使排出介质气体产生双向运动。在气缸的吸气阶段，吸入介质和排出介质的一部分同时进入气缸，这给吸入介质产生一定的阻力，造成吸气阀的吸入量减少，吸入压力上升。所以膜片负半周冲程的吸入量没有变，只是吸气阀的吸入量减少了；膜片正半周冲程的压缩量及排出量没有变，只是单位时间内的充装量减少了。

吸入管道上升的温度，只包括压力上升时的反映温度；排出管道上升的温度，包括介质双向运动和阀门产生故障时摩擦所产生的温度。

此故障处理方法同吸入阀封闭不严的处理方法相同。

# (3) 输出管道有泄漏存在

其表现形式为油压脉冲，吸入和输出管道的温度正常。因为膜压机的输出端有气压自控阀控制，所以吸入压力和输出压力也都正常。只是单位时间内的充装量减少了。

检查部位有膜片的周边密封区，安全防爆装置，接头及水分离器的排放部位，经水冷却器的压力输出管道。检查时可暂停冷却

水，观察冷却水出口处是否有气压存在，若有气压存在，则说明冷却器中的压力输出管道有破裂现象，介质气体进入水冷却器中。

在检查无水介质压缩机时（如氖氦、氖、氦压缩机），可直接打开水分离器的排放阀，若有水分存在，则说明经过水冷却器的压力输出管道有破裂现象。水分产生的主要原因是压力输出管道的裂口处产生了一定的锈灶，当压力介质通过时由于应力作用形成了负压区，水冷却器中的水分被吸入压力介质中，然后经过水分离器时分离而产生。

此时，对水冷却器必须进行整体更换，对其它部位应进行紧固或密封处理。

# 4. 冷却效果差

压缩缸及压缩缸附近的管道和水冷却系统的温度有明显上升，压缩机的总吸入压力有所上升。但压缩机的吸气、压缩、排气过程中的介质气体体积没有明显变化，也就是压力和单位时间内气体体积的充装量基本不变，只是经过冷却后的实际充装量有明显下降。情况严重时，压缩缸内发出的声音有沉闷感，接近于二级排出阀的高压输出管道发黑，并有狐臭味。

虽说在吸气、压缩、排气三个过程中介质气体的体积没有变，但这只是在冷却效果差时受温度影响后的膨胀体积。由于这一原因使压缩机的吸入量减少，导致压缩机前的总吸入压力上升，冷却后的实际充装量明显下降。

特别是在高压节流膨胀冷循环中使用的压缩机，若产生这一故障时会较严重地影响膨胀后的温降效果。因此采用一台冷却效果好的压缩机对制冷来讲至关重要。

为防止这一故障的发生，在保证冷却水量的前提下，对水路中的污垢应定期清理。

$$
V = \frac {K (\Delta P + P _ {K}) D ^ {4}}{\boldsymbol {y} \cdot \boldsymbol {b} _ {M}}
$$

式中  $K$  —系数

$\Delta P$  ——压力差

$P_{K}$  ——表面毛细管压头

$D$  ——毛细管直径，即残留在密封面之间的空隙大小

$\pmb{\nu}$  通过液体的粘度

$b_{M}$  密封面宽度

在上式中，D以4次方出现，因此是一个决定性因素。也即密封面加工精度愈高，达到密封接触所需施加的力就愈小，吻合度也越好。造成D增大的原因有三个方面：

① 加工精度不够；② 在使用中阀座和球体之间的腐蚀、冲蚀或进杂质、擦伤等；③ 自然磨损。

减小D值的方法：用手轻压阀座使之逆转、顺转  $90^{\circ}$  约六七次，转动时不晃动，将球体打开检查，看一下接合面与接合部分贴合情况。如果球部有硬伤，要相应处理；有凹坑部位用焊锡补上，然后反复拆装、刮修接合面。还有一种是自然磨损，这时需重新更换阀座，也可加垫片来暂时解决。

球阀的内漏还可能出现在阀体与副阀体之间，可先清除主、副阀体结合部分的杂质，然后更换新密封圈。若主、副阀体六角螺母松弛，应紧固六角螺母。

# (2）阀门的外漏

外漏包括在阀杆、阀盖连接处和阀体连接处的泄漏。

当阀门处于实际工作状态时，影响填料

密封性能的因素很多，最突出的是介质压力，介质压力对填料密封影响很大。当阀门在管道上工作时，由于温度、压力、摩擦的作用，使填料老化，轴向预紧压力随时间衰减，甚至达到松弛状态，从而发生泄漏。因此须拧紧压盖螺丝。当设备检修时，更换填料。

另有一法是在填料压盖和填料间放上两只与阀杆相配的蝶形弹簧，达到自动压紧填料之功能。球阀填料压盖用螺栓压紧，为防止压力增加，压盖偏斜而影响阀杆运动，拧紧两个螺栓时用力一定要均匀，否则会压偏而卡住阀杆。

2. 阀门动作不稳、迟钝，球体半开卡住造成以上情况的原因及排除方法：

第一，是管线漏气、气缸内漏、活塞杆套漏气、阀门摩擦力大引起的。检查后，应立即更换连接处法兰密封垫片，更换气缸活塞和活塞杆套的O型密封圈。

第二，是阀门摩擦力引起的。这是因为为了保证更好的密封性能，使阀座压在球体上的压力加大，造成球体和阀座摩擦力增大，球体动作迟钝，或出现卡住现象。另外，由于压力加大，造成阀座磨损增加，减少使用寿命。因此，在调整阀座和球体之间的压力时，用力要得当，并使密封副的缝隙一致。

填料密封压得太紧，也会增大摩擦力。正确的方法是每放进一个密封圈，就压紧一个，然后转动球体。气动拨叉或小轴磨损必须重新修整拨叉，小轴磨损必须更换新件，安装时要保证润滑。

# 扒珠光砂引起空分冷箱严重受损

孙全海

（扬子石化公司烯烃厂空分车间，南京市大厂区，210048）

# 一、事故经过

1991年6月7日晚，扬子石化公司A\* $10000\mathrm{m^3 / h}$  空分设备按检修计划停车，排液后加温8小时，于8日晨6:30开始扒珠光砂。先打开了南面膨胀机保温箱上部人孔，接着打开冷箱东面底部人孔，砂开始从北面人孔流出，并越流越快。突然冷箱内发出轰隆隆声，而砂从人孔向外喷出十多米远，接着冷箱内又传出几声沉闷的撞击声，冷箱摇晃了几下，四、五楼平台处也有珠光砂吹出，周围空气中弥漫着珠光砂粉尘。幸未造成人员伤亡。

# 二、事故后果

1. 冷箱四周均鼓肚变形，最大处向外突出  $300\mathrm{mm}$  左右，并几处裂开；固定上塔的四根拉筋中两根断裂，上塔与主冷连接处变形，上塔倾斜；冷箱内部分管道被压扁。  
2.检修计划被打乱，检修期延长大约半个月，不仅修补工作多花费100多万元，而且更主要是A\*空分没能按原计划供出氧气、氮气，使整个公司的生产受到了很大影响。

# 三、事故原因

经分析，事故的主要原因是加温时间过短。设备操作规程规定加温时间36小时，而此前实际加温时间一般也满24小时。由于未见过加温时间过短就扒珠光砂可能引起严重

后果的报道，这次一下子将加温时间缩短为8小时，结果引发严重事故。具体破坏机理可能是：

1. 珠光砂含水量较大，部分温度较低的珠光砂成团结块，流动性差，扒砂时先在底下部形成较大空隙，大块珠光砂掉落下来形成很大的冲击力，从而造成破坏。  
2. 冷箱内富氧管道轻微泄漏或外界空气进入产生冷凝作用，使得部分珠光砂带有低温液体，扒砂时与高温珠光砂混合迅速汽化产生高压从而产生破坏作用。事实上，停车前曾分析冷箱内气体含氧  $25\%$  上下，而扒砂时发现部分珠光砂的温度很低，翻动时有“噼啪”声。

# 四、防范措施

1. 严格执行设备操作规程，加温过程要达到36小时。  
2. 防止珠光砂在装入、扒出或仓库贮存时受潮。  
3. 加强对冷箱密封污氮气压力的监控，防止外界空气侵入。  
4. 增加冷箱内温度测量点，增加对冷箱内气体成分分析次数。  
5. 先在冷箱中部开人孔扒砂，冷箱上部珠光砂基本扒完后再打开底部人孔放砂。

# 带压拧螺栓 损失四十万

刘彦堂

（河北崇利制钢有限公司制氧车间，河北涉县井店，056404）

河北涉县崇利制钢有限公司制氧车间，配开封空分设备厂  $3200\mathrm{m}^3 /\mathrm{h}$  空分设备之DA400E61型空压机，1996年10月18日维修工在清洗完油过滤器后，由于操作不当，拧断转换阀门螺栓，造成全公司停产、半停产达12天之久，直接经济损失8万元，间接经济损失37.8万元，共计45.8万元。

# 一、事故经过

1996年10月16日、17日，车间主任两次发现主空压机油压不稳，于是先后派仪表班长、维修班长进行检查，但均未查出不稳原因。于是18日上午主任和有关人员商议决定检修清洗油过滤器。

修理过程中，由于过滤器阀杆很紧，副班长将阀杆螺母稍松动，用小锤轻轻沿阀杆轴向敲击，使阀杆能转动，小心地将油过滤器切换过去。油过滤器检修清洗完以后，又小心地将油过滤器切换回来。此时，阀杆漏油，按常规，将阀杆螺母拧紧即可。不料在拧紧螺母时，由于用力过猛，将M36×3阀杆突然拧断，阀芯被油压出，透平油随即喷

出，油压急剧下降，引起空压机油压过低联锁报警，负荷停车。

# 四、经验教训

（1）带压松紧螺母致使空压机发生事故负荷停车。因此，必须严格按规范操作、维修。油过滤器在检修后、切换前应作严格检查，确认无误后再切换；绝对禁止带压检修。  
（2）生产厂家是否考虑，对油过滤器转换阀门加以改进，在手柄侧加上压盖，或者考虑螺栓材质改用铸钢。

# 制氧工问题解答（五则）

1. 问：我们厂1994年底购进一套KZON=50/120型返流膨胀空分设备，开车后勉强运行两个月就要加温吹除，过后情况越来越不好，三个月要加温两次，纯化器工作不正常，设备不稳定，请代为分析一下是什么原因造成的？（四川峨嵋读者问）

答：这套设备的分馏塔纯化器工作不正常，通常可称之为“制氧综合症”，因此三个月要停车加温两次。最初的印象是纯化器加热不彻底，以致  $\mathrm{CO}_{2}$  或水分进塔，影响工况破坏，工作失调。曾用信件四次查问情况，才找到病根所在，消除之后，情况改观，病状消失，近年没有给我来信了。

第一次我建议对电加热炉进行检查，三组电阻丝是否断了一相。断一相，总功率会减少  $1 / 3$  ，所以加热出口温度提不高。其次是从调节污氮量着手，使电炉出口温度提高，从流程组织看来，上塔压力为  $0.3\sim 0.59\mathrm{MPa}$  氮量不是少而是多了，氮阀可以关小，事实上这个措施是没有多大意思的。恰恰相反，它可能会影响塔中产品纯度。我得到的答复是“加热管经电工测试正常，没坏”，又说：“不论污氮通入加热炉气量多少，加热炉出口温度不动”（猜想是纯化器出口温度上升很慢之误）。来信又提出污氮出口管冒汗到结霜严重，温度只有  $3\sim 4^{\circ}C$  左右，产品纯度不高，调不好。

第二次我建议从分子筛纯化器着手去找原因，提出分子筛是不是活化过，叫他们再次检查一下，但所得的答复是“厂家说分子筛活化过”。2F=6.5制冷机工况正常，运转理想，进纯化器空气温度为  $5\%$  无误。不过大

家一致认为毛病出在电加热炉上。

第三次我从加热炉的计算公式出发，分析电路及污氮的工作情况。（1)电路断相或电炉功率不足，电炉出口氮温度会下降，希望算算看；(2)污氮气量过多或污氮出塔温度过低，电炉出口气温也会相应下降的。究竟那一点符合当时条件，要求他们继续查找。另外要求提供当时的记录表。又从纯化器出口温度曲线来判断，还可能是纯化器进了水。

第四次来信说，经过大家的观察，讨论计算，果然发现电热管功率只有  $\sim 14\mathrm{kW}$  而原设计为  $21\mathrm{kW}$  。查其原因是大修时曾将9根电热管，更换过6根。这样一来原因便找到了，一切不正常怪现象也就容易解释了。

该设备由于电炉功率不足，导致纯化器再生入口温度低，出口温度上升慢。而切换时间已到，不得不强制切换，再生不彻底，有  $\mathrm{CO}_{2}$  入塔，主换热器换热性能下降，污氮出口管结霜严重，高压上升，下塔压力下降，液面突然上升，关小节-3阀仍无济于事，产品纯度因而下降，长期调不好，直至停车加温。虽数次加温解冻分馏塔，也只能暂时解决主换热器冻结问题，纯化器工况实际上没有因塔的加温而改善，而且每况愈下。

第五次来信寄来了记录，得知加热管刚刚换过，电炉功率  $22\mathrm{kW}$ ，达到加热2.5小时纯化器出口温度为  $125^{\circ}\mathrm{C}$  的水平。虽然污氮出口管仍在“冒汗”，塔的工况仍不稳定，但相信不久会得到解决。写这则问答已半年多，没有新情况发生，可见问题已经解决了。

该设备故障由产生到解决，历时三个多月，给我们的印象是深刻的。那就是故障分

析不光要靠对现象观察分析，还要靠理论；制氧工不仅要有实践经验，还要用理论作简易的计算，把理论和实践紧密结合起来。故障分析首先要抓住重点，要围绕影响加热器温度升不上、纯化器加热出口温度升不高的前因后果进行系统的分析，要透过现象看本质，去伪存真。此外还要注意工作质量，谁能料到这次故障的原因，是出自电阻丝的出厂或销售环节的质量上？找原因时要把那些次要因素暂时放在一边，譬如氮管“冒汗”，就是非主要因素，而是拌生现象，况且空气进塔为  $8\%$  ，热端温差按  $8\%$  设计，那么污氮管温度不应当是  $3\sim 4\%$  ，而应该是零度，第四次来信曾提及“问题不仅是污氮温度低，加热管功率不足也很重要”，这说明他们的思路已经深透到问题的本质了。所以说故障分析，也要注意方法。方法不对将会徒劳无功。

2. 问：我厂购进一套  $150 \mathrm{~m}^{3} / \mathrm{h}$  空分设备，型号是 KZON-150/600-Ⅲ型，现在关键问题是配套空压机的最高压力与分馏塔的启动压力不相符，低于成套启动时的压力要求。请问这样的配套方式是否合理？具体操作应该怎么样？（四川万县学员问）

答：制氧机是一个大系统，由几个子系统组成(许多机组)。总系统的功能要求能满足正常制氧工况和启动工况的冷量需要，所以设备的产冷量必须有一定的后备。各个机组的设计，其参数和结构也要符合上述的总体要求，这是系统总体设计的原则之一。如果疏忽了产冷量的上述后备，整个系统设计将以失败或严重失调而告终。譬如国产150 $\mathrm{m^3 / h}$  制氧机，正常工况下压力为  $2\mathrm{MPa}$ ，为了启动的需要，把(包括空压机)系统压力定为  $4\sim 4.5\mathrm{MPa}$  。这一压力的选定，是根据分馏塔热端温差为  $5\%$  、主冷温差为  $3\%$  、满足全国夏季气候条件、启动时间为8小时而计算出来的。但我国生产的这一类制氧机，最

早是以  $5\mathrm{MPa}$  为系统压力，经过实践认可才把最高压力改为  $4\sim 4.5\mathrm{MPa}$  。这样的设计，经历20年实践都没有改变；随着制氧技术水平的进步，主换热器由光管改为板翅式，热端温差从  $5^{\circ}C$  下降到  $3^{\circ}C$  ，主冷温差因推广多孔管而降低到  $2^{\circ}C$  以下。同时活塞式膨胀机改用透平膨胀机，效率提高到  $70\%$  ，正常工况的压力，由原来的  $2\mathrm{MPa}$  降低到  $1.0\sim 1.3\mathrm{MPa}$  。最高的系统压力也经过多次修改，先是定为  $2.5\mathrm{MPa}$  ，继而改为  $2\mathrm{MPa}$  ，最后改为  $2.84\mathrm{MPa}$  。因为当空压机最高压力为 $2\mathrm{MPa}$  ，启动时间增加到  $12\sim 14$  小时，与总体设计8小时的要求不符。这一过程，说明了系统的最高压力是按制氧机启动的要求而定，不能随意确定。

现在回过头来看看，你厂购置的分馏塔为KZON-150/600型空分设备，原设计的系统压力为  $4\sim 4.5\mathrm{MPa}$  ，现改配2.84 MPa的空压机，这样的配套显然是不恰当的。

这样的配套方式，将会产生哪些影响？(1)冷量后备不足，启动时间延长；(2)原来互相匹配的活塞式膨胀机，变成不适合不协调的设计了。启动时，空气不能全量通过膨胀，无法发挥膨胀机制冷潜力，因为进排气阀门和缸径结构都显得太小，不能满足启动要求(请参阅《深冷技术》1994年第1、2、3期)；(3)膨胀机结构偏小，造成部分空气放空，这也是膨胀机不相匹配引起的；(4)设备应变能力差，如遇夏季启动，塔内有漏点，最起码的影响是设备工况不稳定，调节困难甚至车子开不出来，所以操作要求高，维修工作也相应要求严格。在大型空分设备中，我们也碰到过同样的情况，启动时两台透平膨胀机仍然不能全量吸入空气，把大量空气放空，这不仅造成启动时间长，排空气体噪声大，都可以归纳为膨胀机设计结构仅仅考虑正常工况而不能满足启动工况的结果。这样的设计，从系统分析看来是不当的。

面对你们既成事实的条件，应当如何操作呢？当气体压力到达  $2.84\mathrm{MPa}$ ，空气不能全部进膨胀机时，为了保持高压，可有两种操作选择：一是暂时把多余的空气放空；另一种是打开节-1阀，将多余空气参加制冷，两种办法哪一种好，要权衡冷量得失而定。遇到夏季车子不开出来时，只好把积液时间安排到夜间气温最低时进行。

# 3. 问：为什么充氧台充氧时充填瓶数不能过少？

答：使用2-1.67/150型氧气压缩机的充氧台，一般有10套充氧卡具，每组5瓶轮换工作。有时因个别瓶有故障不进气或不能继续充氧而关闭，这样实际充填数就会少于5只。这在充填工作中是经常遇到的，甚至也有个别充氧工人故意单独充  $1\sim 2$  个气瓶的情况。这样做，在实际上是危险的。因为在氧压机排气量不变的情况下，充瓶数越少，氧气瓶阀和充氧蛇管及系统内的流速就会越高。高速气流能产生高压静电，高速气流对管壁、尤其是弯头和阀孔处的冲击摩擦力都很大。因此，发热甚至出现火花的危险性就增加了。在高压氧气中，温度太高和火花都会诱发强烈氧化，也就是燃烧。对新换过零件的充氧台和氧气瓶阀口不是很干净的情况下，尤其危险。

有的充氧工人也许会提出疑问，他可能会想：我也曾单独充过一只氧气瓶，也没发生过问题。这可能也是事实。因为制造厂选用的充氧台材料，都是难以氧化的物质，对高逸氧气有一定的耐受力。但对刚修换的零件和氧气瓶阀来说，情况就不容乐观，因为刚修换过的零件和用户使用过的氧气瓶阀口，有可能存在某些容易氧化的物质或硬质颗粒。这样，危险程度就难以预料，希望不

要冒这个危险。

在邯郸制氧站，70年代就发生过一起这类事故——充氧在正常进行中，当压力升至12.5MPa,其中有一只氧气瓶因系降压瓶，此时关闭。这只瓶本应在其余4只氧气瓶充满后一块卸下，但操作工因某种原因单独将此瓶卸下又换上一只空氧气瓶，然后先打开氧气瓶阀，再打开充氧台上分路小阀。此时立刻发现充氧紫铜管温度明显升高并变色，此时再关阀门已无济于事。火已从充氧卡具和氧气瓶阀接触处喷了出来，喷口在下面，火正好喷在气瓶上，气瓶肩部部分瞬时熔化并反吹向上，这种情况和气割钢板类似，火花高约2米。幸好这是一只空瓶，幸好车间屋顶较高，幸好此瓶没有歪倒，幸好火花没有横吹，因此没有危及其他气瓶也没造成更大损失。切断氧气源后火渐渐熄灭，仅一工人上肢烧伤，一只充氧卡具和一只氧气瓶及气瓶阀烧熔报废。不然后果不可设想。

上面的实例充分说明了高压氧气在高速流动下的危险性。当然，实例中，事故气瓶阀口和充氧蛇管内的气流速度要远高于单独充一只氧气瓶，但高速氧气流的危险和所能造成的危害从中可以充分说明。

如果在充氧中，当压力已达  $10\mathrm{MPa}$  以上，此时如在被充的氧气瓶中发现有一只氧气瓶没有进气，这时也不要再去处理，因为处理后使其进气的情况和实例相同，其危险性是不难想象的。

检查气瓶是否进气，应在  $5\mathrm{MPa}$  以上、10 MPa以下的时候进行，在压力低时发现瓶阀未开可先微开，压力平衡后再全部打开，以减小危险性。

# 4. 问：充氧台上的氧气大阀，开关时快点好还是慢点好？

答：充氧台上一般有三只大的高压氧气截止阀，其中一只为总进口阀，一般情况下

常开不关；另外两只为充氧台左右两组汇流排的截止阀。充氧时，左右两只分别开关，也就是每充完一组要切换一次。这两只阀在开或关的操作过程中，不能过快，要稍慢一点。因为在开或关的过程中，这两只阀都处于高压纯氧的条件下。操作过快时，相对运动的零件会因局部摩擦速度快、摩擦力大而发热，严重时或因局部有易燃物而燃烧。阀门在开关时产生的强烈撞击也会发热，这正像用手锤击打火柴头能使火柴头着火一样。

出于相同的原因，氧气瓶瓶阀在开或关时，也不应过快。

这两只大阀和氧气瓶阀发生燃烧事故的例子很多，不胜枚举，麻痹不得。

有些操作者也许会有疑问，邯郸制氧机厂生产的灌充器，其中有两只气动高压氧气截止阀，开关很快，不是也很安全吗？实际上，这两只大阀，运动部分的阀杆系轴向直线运动，受气压室内压缩空气的充入和放出速度的控制，并不是瞬间完成操作，运动速度并不太快。设计时，受摩擦件是不锈钢与聚四氟乙烯，摩擦力很小，所以比较安全。并且它远离操作者，这就显得更为安全。但对这种阀，安装或修理时，也要特别注意填料压盖不能压斜，否则阀杆和压盖会直接相磨，运动时也要产生摩擦热，也是不利的。

# 5. 问：能不能“倒瓶”？倒瓶时应注意哪些事项？

答：所谓“倒瓶”，就是将已充满氧气的钢瓶内的氧气，通过联接管道输入另一只空

氧气瓶内或其他容器内。

倒瓶分两种情况：一种是将一只或多只充满氧气的钢瓶通过充氧台上的管道直接利用本瓶压力将氧气充入另一只或多只空瓶，压力平衡时结束；另一种情况是直接将氧气瓶内的剩余氧气，不管压力多少通过回收管道送入贮气囊，再通过氧压机压入钢瓶。

在第一种情况下，被充氧气瓶的工作状态，和利用氧压机充氧类似，一般情况下是可以的。但需要注意的仍是管道内氧气的流速，如管道上的阀门和两边气瓶阀全开，压力高时流速也是相当大的，其危险性和笔者第一题所分析的情况一样。这种情况的事故例子也不少，我就曾看见过一次这类事故后的现场——联接两气瓶的充氧蛇管，一只烧去  $400 \mathrm{~mm}$ ，一只烧去  $1000 \mathrm{~mm}$ ，充氧卡具烧坏，充氧截止阀内的阀芯和阀体密封面均都烧流，密封件聚四氟乙烯垫全气化，操作工人的手被烧伤，幸好没造成大火。

为防止上述事故发生，应尽力避免这种操作。必要时，其操作要点是降低管道内氧气流速，也就是将压力高的气瓶阀稍开，待压力接近平衡或压力已降低后再渐渐开大。

在第二种情况下，管道内的流速和气瓶内压力以及阀门开度成正比，操作要点仍是慢慢开启气瓶阀，先微开，后全开，降低流速。另一方面，回收管也应为渐粗管，这样既可降流速保安全，还可减缓气流对贮气囊或安全水封的冲击。
