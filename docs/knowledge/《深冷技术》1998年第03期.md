# 管理办法实施细则（试行）

第一条 为贯彻实施机械工业部关于《机电产品型号管理办法》，以适应社会主义市场经济的需要，改进并完善气体分离与液化设备产品型号的行业管理，结合本专业产品型号编制和注册管理的实际情况，特制定本细则。

第二条 产品型号是识别产品品种与规格的基本标识，有利于推进产品型号的标准化，对产品的生产、使用、配套协作和维修都具有重要作用，也是维护产供需各方利益的基础性措施。

第三条 凡正式生产和新设计开发及改进设计后需变更型号的产品，除有关法律、法规另有规定者或一次性产品和特殊专用产品外，均可按本细则实行本专业产品型号注册。

第四条 产品型号注册一般应以产品型号编制方法的现行国家标准或机械行业标准依据。

第五条 本专业产品范围（按产品种类分类）

1. 空气分离设备；  
2. 稀有气体提取设备；  
3. 溶解乙炔设备；  
4. 水电解制氢设备；  
5. 天然气分离设备；  
6. 气体液化设备；  
7. 配套单机和单元设备；  
8. 本专业归口的其它产品。

第六条 受机械工业部委托，机械工业部气体分离与液化设备标准化技术委员会秘书处（以下简称秘书处）承担本专业产品型号注册管理业务，其主要职能如下：

1. 组织制订本专业范围内的产品型号编制方法标准，适时修订或修改产品型号编制方法标准。  
2. 受理本专业范围内的产品型号的注册申请及办理审核、登记、发布。  
3. 根据需要或受上级主管部门委托，承担编制有关产品目录、样本和统计信息。  
4. 为企业提供有关产品型号编制的技术咨询服务。  
5. 与相关专业协调产品型号的有关事宜。

第七条 秘书处对收到的产品型号申请表及有

关文件指定专人保管，不得扩散，以切实维护申请单位应有的权益。

第八条 秘书处在收到产品型号申请表后，一般在10天内完成审核，并填发产品型号注册证书寄送申请单位。

第九条 在产品方案确定、技术设计完成后，企业即可申请产品型号注册；新开发设计的产品，在产品试制完成并通过技术鉴定或运行考核后，可申请产品型号注册。

第十条 企业申请产品型号注册，应填写产品型号注册申请表一式二份，加盖公章，并附有关文件资料（产品基本性能参数、试制产品技术鉴定书或运行考核报告），一并送秘书处办理注册。产品型号注册申请表及附有关文件资料一份留秘书处存档。

第十一条 申请单位应按经注册的产品型号申请表确定的产品型号，设计图样、编写技术文件和制作产品标牌。

第十二条 申请单位对秘书处注册的产品型号有异议或疑问时，应及时向秘书处提出，以便尽快作出处理。

第十三条 技术引进产品和三资企业，在我国境内生产并销售的产品，一般应按我国标准确定产品型号，申请产品型号注册。

第十四条 产品型号注册，应合理收取产品型号注册管理和技术服务费，用于注册管理机构从事产品型号的业务管理，包括审核费、工本费、宣传费、邮资费等。

第十五条 秘书处定期在本专业的刊物上，报道已注册产品型号的产品及其生产企业。

第十六条 本细则经本专业标委会审核通过，报上级主管部门备案。

第十七条 本细则实施日期一九九八年一月一日。

# 低温液体贮运设备 使用安全规则

# 前言

本标准对JB6898—93《低温液体贮存设备使用安全规则》进行修订，并为强制性标准。

本标准主要修改内容：

——标准适用范围中增加了对于二氧化碳液体贮运设备可参照使用的规定；

——对低温液体贮运设备使用安全要求和措施的有关内容作了修改；

——补充规定了容器和槽车容器内的液氧中乙炔含量的检测要求及在通风不良有发生窒息危险场所的空气中检测含氧量的规定。

本标准从生效之日起，同时代替JB6898—93。

本标准由机械工业部气体分离与液化设备标准化技术委员会提出并归口。

本标准起草单位：杭州制氧机集团有限公司、杭州制氧机研究所。

本标准主要起草人：范光汉、陈永良。

# 1 范围

本标准规定了低温液体贮运设备（以下简称设备）的安全要求和措施、事故处理。

本标准适用于氧、氮、氩低温液体容器（以下简称容器）及槽车的低温液体贮运设备。

本标准对于二氧化碳液体贮运设备可参照使用。

# 3 液氧、液氮、液氩的特性及其危险

3.1 液氧、液氮、液氩均为低温液化气体，在  $101.325\mathrm{kPa}$  压力下，液氧沸点为  $-182.83^{\circ}C$  ；液氮沸点为  $-195.65^{\circ}C$  ；液氩沸点为  $-185.71^{\circ}C$  。当与人体皮肤、眼睛接触会引起冻伤（冷烧灼）。  
3.2 低温液体汽化为气体时，体积会迅速膨胀，在  $0^{\circ}\mathrm{C}$  、  $101.325\mathrm{kPa}$  状态下，1L液体汽化为气体：氧为800L；氮为647L；氩为780L。在密闭容器内，因液体汽化使压力升高，易引起容器超压危险。  
3.3 液氧和气氧是一种强助燃剂。  
3.3.1 液氧与可燃物接近时，遇明火极易引起燃烧危险。  
3.3.2 液氧与可燃物接触时，因撞击易产

生爆震危险；液氧与可燃物混合时，潜在爆炸危险。

3.3.3 液氧蒸发成气氧时，能被衣服等织物吸附，遇火源易引起闪烁燃烧危险。  
3.4 氮气、氩气是一种无色、无味、无毒、不可燃的惰性气体，有很强的室息性，会引起窒息危险。

# 4 安全要求和措施

# 4.1 设备安装场所

4.1.1 低温液体汽化站的设计应符合GB50030的规定。  
4.1.2 属压力容器的低温液体贮存容器，它的安置、使用除符合本标准的规定外，还应符合《压力容器安全技术监察规程》、《在用压力容器检验规程》及《压力容器使用登记管理规则》的有关规定。  
4.1.3 安装场所必须有良好的通风条件或设有换气通风装置，并能安全排放液体、气体。  
4.1.4 安装场所必须设有安全出口，周围应设置安全标志，安全标志的要求应符合GB2894的有关规定。  
4.1.5 安装容器的基础必须坚实牢固，并应防火耐热；安装液氧设备的基础必须无油脂及其他可燃物，严禁使用沥青地面。  
4.1.6 安装场所附近必须有充足的水源，场所必须有灭火器材，场所周围  $5\mathrm{m}$  内不得有易燃易爆物，保持场地清洁干净。  
4.1.7 安装场所应有槽车或消防车出入通道，并有足够宽度，便于槽车或消防车通行。  
4.1.8 安装液氧容器的场所内的隔墙、屋顶建筑，不得低于GBJ16中的二级防火、耐热的规定；建筑物的防雷要求，应符合GB50057的规定。  
4.1.9 液氧的贮存、汽化、充装、使用场所宜设围墙或栅栏；安全出口必须布置适当，一般需有分别安置在两侧的出入口，一旦发生危险时能使人员迅速撤离；汽化器的场所允许设一个出入口。门窗必须向外开。  
4.1.10 液氧容器间的安装间距，应不小于

相邻两容器中较大容器者的半径，且最小间距不小于  $1\mathrm{m}$ 。

4.1.11 液氧容器与其他建筑物、贮罐、堆场的建筑防火间距必须符合GBJ16的有关规定。当防火间距不能达到时，应建筑高于容器及防火物  $0.5\mathrm{m}$  的防火隔墙，可减少防火间距到上述规定的1/2。  
4.1.12 液氧的贮存、汽化、充装、使用场所的周围  $5\mathrm{m}$  内严禁明火，杜绝一切火源，并应有明显的禁火标志。  
4.1.13 液氧的贮存、汽化、充装、使用场所的周围至少在  $5\mathrm{m}$  内不准有通向低处场所（如地下室、坑穴、地井、沟渠）的开口；地沟入口处必须有挡液堰。

# 4.2 设备安置

4.2.1 容器不准安置在出入口、通道、楼梯间或它们的贴近处。  
4.2.2 液氧容器一般应安置在室外，当液氧总贮存量不超过  $10\mathrm{m}^3$  时，允许安置在一个防火耐热、耐火极限不低于  $1.5\mathrm{h}$  非燃烧材料建筑的室内，且必须是具有良好通风条件、人员流动少的单独房间；明火间距不小于  $5\mathrm{m}$ 。  
4.2.3 液氮、液氩容器宜安置在室外。若安置在室内，其安装场所应符合4.1.3的规定，且气体紧急放空口必须引出室外安全处。放空口离地面高度不得低于  $2\mathrm{m}$  
4.2.4 液氧容器不得安置在经常有人逗留的房间上下层。  
4.2.5 液氧容器安置在室外，必须设有导除静电的接地装置及防雷击装置。防止静电的接地电阻不应大于  $10\Omega$ ；防止雷击装置的最大冲击电阻  $30\Omega$ 。

# 4.3 槽车运输及充灌

4.3.1 槽车的使用、运输和检修必须按《液化气体汽车罐车安全监察规程》的有关规定；槽车容器的使用、管理和检修必须按《压力容器安全技术监察规程》的有关规定。  
4.3.2 装有低温液体的槽车应限速行驶，行驶速度一般为：一级公路最高速度为60

$\mathrm{km / h}$  ；二、三级公路为  $30\sim 50~\mathrm{km / h}$  ；在高速公路上宜低速行驶，应避免紧急制动，严防撞击。  
4.3.3 槽车行驶应避开闹市区和人口稠密区。必须通过时，应限速行驶，不得随意停靠。  
4.3.4 槽车输送液体时，要经常监视压力表的读数，严禁超过压力规定值。当压力表读数有异常升高时，槽车应开到人稀、空旷处，打开放空阀，排气卸压。排放氧气时，排放波及区内严禁明火。  
4.3.5 液氧槽车必须有导静电接地装置。  
4.3.6 槽车只有得到有关人员同意后，方可进入充灌场所进行充灌；充灌时，操作人员必须在现场。充灌操作应按操作规程进行，防止低温液体外溢。充灌液氧时， $5\mathrm{m}$  内不得有明火。  
4.3.7 槽车在连接充灌输液管前，必须处于制动状态，防止移动；在斜坡处应设置防滑块。  
4.3.8 槽车充灌结束后应及时离开，如果充灌工作中断时间过长（过夜或假日中断），应解脱输液管连接。  
4.3.9 槽车在充灌装卸作业时，汽车发动机必须关闭。  
4.4 设备的安全使用  
4.4.1 容器和槽车容器的充满率不得大于0.95，严禁过量充装。  
4.4.2 容器投入使用前，应按《压力容器安全技术监察规程》的规定检查各种阀门、仪表、安全装置是否齐全有效、灵敏可靠，以保证安全使用。  
所用压力表必须是禁油压力表；安全阀、防爆装置的材质应选用不锈钢、铜或铝，并必须脱脂去油。  
4.4.3 容器、汽化器及管路系统在使用前，应用无油干燥空气或氮气，吹除水分或潮湿气。  
4.4.4 低温液体汽化器的气体出口温度不低于  $-20^{\circ}\mathrm{C}$  。

4.4.5 容器在初次充灌时，开始应缓慢充灌（阀门应缓慢打开），然后逐渐加快，以减少对容器的热冲击破坏和减少液体蒸发。  
4.4.6 当设备上的阀门和仪表、管道连接接头等处被冻结时，严禁用铁锤敲打或明火加热。宜用  $70 \sim 80^{\circ} \mathrm{C}$  干净无油的热空气、热氮气或温水进行融化解冻。  
4.4.7 设备须按铭牌上标明的介质专用；需改换允许的介质时，必须彻底清洗吹除，并经测试分析合格，符合适合充灌另一种介质，并改变相应色标。  
4.4.8 液氧容器和液氧槽车内的液氧应定期进行乙炔含量分析，至少每月分析一次，其乙炔含量不得超过  $0.1 \times 10^{-6}$ ，否则应排放液氧。

乙炔含量的测定方法按“比色法”化学分析或色谱进行测定。

4.4.9 使用单位必须制定安全使用操作规程。操作人员独立上岗操作前，必须进行安全教育培训，经考核合格，持安全操作证上岗。  
4.4.10 操作人员应熟悉《压力容器安全技术监察规程》的有关规定和低温液体的特性及其危险性。  
4.4.11 操作人员应熟悉产品使用说明书、设备工艺流程、设备上各种阀门、仪表及其作用和操作程序；在发生故障及意外事故时必须能独立采取紧急安全措施。  
4.4.12 容器和槽车上的其他故障排除应按使用说明书的有关规定来处理。

# 4.5 设备检修

4.5.1 容器和槽车容器以及压力表、液面指示计、安全装置应按《压力容器安全技术监察规程》、《在用压力容器检验规程》及《液化气体汽车罐车安全监察规程》的规定进行定期检查和校验。  
4.5.2 液氧容器上防雷、静电接地装置应定期检测接地电阻，至少每年检测一次。  
4.5.3 液氧槽车贮有液氧时，不得动火修理。

4.5.4 容器与槽车容器的外壳上大面积结霜、结露或日蒸发量异常大时，应及时进行检修。设备大修理应在产品制造厂或专业修理单位进行。  
4.5.5 设备修理前，必须用无油干燥空气吹除置换，并使设备温度升至常温；动火必须严格执行动火制度的规定。  
4.5.6 在设备检修前排放液体或气体时，应将排放物排放到通风良好的大气中或专用排放处，必须有专人监护；排放处应设有明显的标志和警告牌，以保证排放安全。排放液氧时，排放波及区内严禁明火。  
4.5.7 设备上的阀门、仪表应有专业人员修理；用于氧的阀门、仪表修理后应严格脱脂去油，并用无油干燥空气或氮气吹洗。  
4.6 安全防护  
4.6.1 操作人员在充灌或处理低温液体时，应戴上干净易脱的皮革、凡布或棉手套。若有产生液体喷射或飞溅可能，应戴上护目镜或面罩。处理大量低温液体或低温液体严重泄漏时，应穿上无钉皮靴，裤脚套在皮靴外面。  
4.6.2 操作人员在充灌或处理液氧时，不得穿戴被油脂沾污的工作服和个人防护装备，凡被油脂沾污过的衣服和防护用品必须更换；不得穿着有静电效应的化纤服装；不得穿有钉鞋。

操作人员的服装若已渗透了氧，则不得进入有明火的场所。必要时，必须更换衣服或经过充分的吹除，在大气中至少吹除 $15\mathrm{min}$ 。

4.6.3 在进入通风不良有发生窒息危险场所处理液氮、液氩及其气体时，必须分析大气含氧量，当含氧量低于  $18\%$  时，操作员必须戴上自供式防护面具，并需在有专人监护下进行操作处理。空气中含氧检测方法可用“化学分析法”来测定。  
4.6.4 操作人员的皮肤因接触低温液体或低温气体而被冻伤时，应及时将受伤部位放入温水中浸泡或冲洗，切勿干加热。严重的冻伤应迅速到医院治疗。

# 5 事故处理

5.1 槽车容器管路系统有微小泄漏时，应及时检修处理；有严重泄漏时，必须将槽车开到人稀、空旷安全处，逐渐排放，并应严格监护；排放时，人、车应处在上风向。排放液氧时，必须关闭汽车发动机；液氧波及区内严禁明火。  
5.2 当槽车在途中发生重大事故，须及时报告当地有关部门进行处理，同时须做应急措施。  
5.3 容器附近发生火灾，若环境温度有可能加速液体汽化时，可使用冷却水喷射到容器外壳上进行降温。

# 主题词：空气分离设备 主换热器 分体式 集合式 特点 设计 效果

主换热器在分子筛吸附预净化空分流程中，起着回收冷量的重要作用，其性能好坏，直接影响着空分设备能耗的高低。以往主换热器，大多采用集合式结构（即多股返流气在同一面主换热器换热）。1996年3月，我公司为鄂城钢铁集团有限责任公司设计  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备，根据多年的设计经验，在主换热器的设计中，作了结构的改进，将以往  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备中的集合式主换热器改为分体式主换热器，分别设计成氧换热器、氮换热器和污氮换热器。本文就分体式主换热器结构特点和要解决的问题予以阐述。

# 一、分体式主换热器结构特征及优点

# 1. 热力计算更准确、可靠

设计参数中氧气、氮气、污氮气从主换热器冷端进入的温度是不同的，氧气是  $95\mathrm{K}$ $(-178^{\circ}C)$ ，氮气、污氮气是  $98\mathrm{K}$ $(-175^{\circ}C)$ 。实际上由于氮气、污氮气进入过冷器的温度不同（氮气是  $79.5\mathrm{K}$ ，污氮气  $80.5\mathrm{K}$ ），它们出过冷器的温度也是不同的。因此，集合式主换热器在热力计算时，返流气在冷端、中

部的温度，只能是一个平均值，这给换热器的温差计算带来误差。又由于截面上温度交叉，形成冷流体横向传热。最后反映在主换热器的换热面积富裕系数的选取，对各返流气来说也是不一样的。实际运行中，主换热器热端各返流气与正流空气的温差不同就是证明。而分体式主换热器就克服了以上问题，使热力计算准确、可靠。

# 2. 热端温差得到保证

（1）由于膨胀空气是从主换热器中部抽出，膨胀机机前温度靠原主换热器冷端抽出膨胀空气调节。因此，膨胀空气出主换热器中部的温度不能低于膨胀机机前温度，也就是说，主换热器的热段换热面积富裕量受到限制。  
（2）设备在启动阶段，膨胀空气需要从主换热器冷段抽出，且需要的膨胀空气是正常工况的两倍左右，应设置较多的膨胀空气通道，否则就会使膨胀空气的阻力增大，膨胀机前压力降低，单位焓降减少，增压侧的压力也降低；另一方面，因返流气流量在主换热器的截面上是均布的，造成膨胀空气通

道两侧冷量不足，使膨胀空气温降速度变缓。这两方面的作用，造成空分设备启动时间延长。如果设置较多的膨胀空气通道，空分设备在正常运行时，大部分膨胀空气是从主换热器中部抽出的，使膨胀空气通道的冷段换热面积几乎闲置，这样其相邻的返流通道的冷量将大部分上移到热段，热段冷量过剩，势必造成主换热器热端温差增大。

（3）膨胀空气设置两个抽口，将正流空气按比例分配，回收返流气的高品位冷量，这样就改善了换热工况。一方面避免了冷量上移，保证了热端温差；另一方面，可设置理想的膨胀空气通道数，兼顾启动工况，缩短开车时间。

因分体式主换热器中只走一种返流气体，使流程中配管简单，调节手段简化。不象集合式主换热器，各返流气都要向每只主换热器分配气量，各种管线复杂、阀门增多。当各主换热器阻力不同时易形成偏流现象。另外，由于分体式主换热器流体种类少，相应配置单元封头减少，配管简单，便于布置，利于减少流体局部阻力。给制造、管理带来方便，有较好的经济性。

# 二、分体式主换热器设计中需解决的几个问题

# 1.阻力匹配

在结构设计及热力计算中，从理论上解决了各主换热器之间的阻力匹配问题，运行中，主换热器中各介质的流量不用调节，就能实现热负荷平衡。

# 2. 合理分段计算

在热力计算时，主换热器根据流程要求进行分段计算（见图3）。

正确的方法是确定主换热器的冷端温差，建立冷段热平衡，求出返流气在中抽口的温度。由热段热平衡求出返流气出热端的温度，校核热端温差是否合理。冷端温差确定的合理与否，一方面直接影响各段的传热温差，即影响热力计算的正确性；另一方面影响设备的经济性，如果直接将返流气进主换热器冷端的温度做为冷端温差计算，主换热器计算出来就比较短，造成运行中因换热面积不足，热端温差大，启动困难，能耗增加，这已为不少事实所证实。如果冷端温差选得太小，势必造成主换热器过长，制造成

本上升。

正确地选定膨胀空气下抽口的过热度，使主换热器中抽口到下抽口的热力计算更可靠，保证膨胀机的安全工作。这一温度，随膨胀空气的饱和温度不同而异。

从鄂钢  $6000\mathrm{m}^3 /\mathrm{h}$  空分运行结果来看，主换热器各状态点的数据，都在设计范围之内，证明设计方案是合理的，热力计算是正确的。今后要根据空分设备流程的新发展，在主换热器的设计上不断改进，开发出更先进的设备。

# 一、前言

在本世纪50年代，国外小型空分设备已采用分子筛清除空气中的  $\mathrm{H}_2\mathrm{O}$  及  $\mathrm{CO}_{2}$ ，成功地用分子筛吸附器替代碱洗塔。70年代后期LINDE、L'AIR、APCI等相继在全低压空分设备上采用分子筛净化空气技术；不久，法液空又研制成功了立式同心圆径向流吸附器，在小型空分设备投入运行后，又不断将其推广应用到大中型空分设备中。

分子筛吸附器是为清除原料空气中的水分、二氧化碳及部分碳氢化合物而设置的，其吸附器再生效果的好坏，直接关系到空分设备的运转周期和运行安全，吸附器持久的、良好的清除效果与解吸过程的彻底性是息息相关的。现国外在空分设备中，空气预净化的分子筛技术有两种：一是TSA即变温吸附，二是PSA即变压吸附。

目前，国内的PSA技术已用于中小型制氮机、二氧化碳吸附，以及焦炉煤气制氢等工艺中。在制氮工艺中，PSA制氮装置吸附塔内通常使用4A（碳）分子筛，依靠氮、氧分子在分子筛微小孔穴内扩散率的不同，而分离氮和氧，其产品氮的纯度可达

$98\% \sim 99.9\%$

下面将我国从国外某公司引进的72000 $\mathrm{m}^3 /\mathrm{h}$  制氧机组中变压吸附技术的几个内容进行探讨。

# 二、 $72000 \mathrm{~m}^{3} / \mathrm{h}$  制氧机组 PSA 的工艺过程

国内某钢厂引进的、目前国内最大的 $72000\mathrm{m}^3 /\mathrm{h}$  制氧机组，其空气预纯化就是采用的PSA技术，处理的空气量为357000 $\mathrm{m}^3 /\mathrm{h}$  ，从空压机后冷却器分离器的顶部出来的空气［压力为  $0.53\mathrm{MPa}$  （表压）、温度为 $35^{\circ}C$  进入PSA（变压吸附）容器。吸附器系统有六个容器，其容器的直径  $\phi 3354\mathrm{mm}$  高为  $23\mathrm{m}$  ，每个容器都装有适量的吸附剂，以清除空气流中水分、二氧化碳和大部分碳氢化合物，六个容器交错循环，即当某几个容器在吸附杂质时，另几个容器则在进行活化（即再生）。吸附阶段是9～14分钟，再生阶段是18分钟。

# 1. PSA 工艺过程的 6 个步骤

（1）运行：空气进入容器的床层中进行吸附（9~14分钟）。  
(2）平衡：该床在完成吸附后与正要完

成冲洗的床层进行压力平衡，这一阶段可减少切换损失(30~60秒)。

（3）减压：床层通过减压放空消音器放空，床层减压到接近大气压力(30~60秒)。

（4）冲洗：床层用干燥的废氮冲洗，以除去吸附的杂质(12分钟)。

（5）平衡：通过回收刚完成吸附阶段床层的空气来提高压力(30~60秒)。

（6）充压：用主空气流的一部分来升高床层压力到吸附压力(4~5分钟)。在整个操作期间，DCS将监视吸附床的压力和阀位。如果DCS在某一时间检测到压力不正确，这说明该床的阀门有故障，须断开该床，并进行检修，此时则要求由6个床体转5个床体运行（阀的极限开关一般不作为阀门的故障检测，仅将压力作为故障检测，因为极限开关可靠性不高，它只作为判断阀是否故障的一个附加因素）。

# 2.6 床转 5 床的操作顺序

（1）减少氮和氧的产品流量到现有值的  $50\% \sim 80\%$  ，这需6~30秒的时间。  
（2）通过主空压机放空阀减少进入冷箱的空气量，减到最大时的  $80\% \sim 100\%$  。在去冷箱管道上有一个流量元件，以帮助进行流量校正，这将需要5~10秒的时间。  
（3）在这期间，液氮回流将从液氮回流罐进行补充，该罐大小可以保持附加的10\~20分钟回流量，以保持氮气纯度不变。  
（4）在循环的合适阶段切除有故障的床层，通常可在床层切换到下一步骤时切换，这个床层将进行9分钟的吸附循环和9分钟的再生循环。  
（5）如果需要，可缓慢关闭导叶，以减少主空压机电耗，期待的氧、氮产量是设计产量的  $70\% \sim 80\%$  ，此时氙的产量将减少。

从图1可以看出，无论是6床循环还是5床循环，总是同时保持两个床层作吸附运行。就是说国外在设计中，一个床体的最大吸附量能满足总气量的  $40\%$  的净化要求。

（4）在未采用分子筛以前，是用板式自清除流程。自清除在冷箱内进行，切换频繁，且切换损失大，进塔的空气压力波动大，导致增压一膨胀流程的实现有一定难度。在采用TSA分子筛吸附技术后，由于吸附周期的延长及再生后的均压，使空气压力波动因素大大减少，故实现了增压一膨胀流程。而该公司的PSA技术，切换频繁，周期与自清除相似，但由于有均压及充压过程，虽切换频繁，仍然可使PSA后进入冷箱的空气压力保持稳定。

# 三、PSA与TSA的比较

TSA的工艺过程一般为：吸附一泄压一热再生一热吹一冷吹一均压一吸附，作周期性循环，每个周期时间较长，约4~6小时。

来自 TSA 的空气，温度通常要求在 13 ~ 17°C 之间，所以空气由空压机加压后需进空冷塔用机械冷冻水和来自废氮塔的冷却水进行预冷，机械冷冻水由氟里昂冷冻机提供。TSA 的再生气源为氮气，但污氮需经蒸

汽或电加热器加热。

# 1. PSA比TSA的电耗小

以从国外引进的  $30000\mathrm{m}^3 /\mathrm{h}$  制氧机组为例，其空气量为  $150000\mathrm{m}^3 /\mathrm{h}$  ，氟里昂冷冻机的电机功率为  $250\mathrm{kW}$  ，水泵的电机功率  $100\mathrm{kW}$  ，经计算每立方空气每小时消耗电  $0.0023\mathrm{kW}$  （国内的机组每立方空气每小时消耗的电要大一些约  $0.005\mathrm{kW}$  ）；再考虑加热污氮的蒸汽平均耗量  $900\mathrm{kg / h}$  （每小时消耗约700度电）。综合计算，  $72000\mathrm{m}^3 /\mathrm{h}$  制氧机组的空气量为  $357000\mathrm{m}^3 /\mathrm{h}$  ，按每年8400小时计，则每年可节省电费约500多万元。

$72000\mathrm{m}^3 /\mathrm{h}$  制氧机组的纯化系统采用PSA，吸附容器的数量相对与TSA流程可能要多  $2\sim 3$  个。但采用PSA可取消空冷塔、废氮塔、分子筛的加热器、冷冻机及为空冷塔和废氮塔提供冷却水的水泵等设备，这不但使流程大为简化、生产安全可靠，还取消了空气预冷和污氮加热用地，故综合结果，空压机后至冷箱前的区域占地可减少。

# 3. PSA切换频繁、阀门质量要求高

从PSA工艺过程和TSA工艺过程的比较来看，PSA没有热再生、热吹、冷吹三个过程，另PSA变压吸附的容量比TSA变温吸附的容量要小，故每个周期的时间较短，切换频繁，气体的损失较TSA大一些。正因为如此，该公司在其工艺流程中有一均压过程，以尽量的减少切换损失。

由于PSA切换频繁，所以对阀门的质量要求很高，阀门动作执行机构的气源质量要高，其含尘量及露点要低。阀门动作的气源一般用氮气或洁净的压缩空气，压缩空气通常需冷冻、干燥。目前，国内生产的蝶阀寿命，仅能动作几万次，而  $72000\mathrm{m}^3 /\mathrm{h}$  制氧机组PSA变压吸附的气动阀门，一年要动作十几万次。现韶关钢铁厂1997年从美国引进的  $750~\mathrm{m}^3 /\mathrm{h}$  制氧机组PSA变压吸附的气动阀门，一年要动作30几万次，其寿命可保证80多万次。故现PSA变压吸附的气动阀门及其气源质量是该流程的关键因素之一，所以大型机组的切换阀门都是从国外引进。

# 四、运行情况

$72000\mathrm{m}^3 /\mathrm{h}$  制氧机组自1997年9月投运以来，PSA变压吸附系统的运行一直较佳，现PSA的空气出口  $\mathrm{CO}_{2}$  含量为  $0.1~\mathrm{ppm}$  比TSA的空气出口  $\mathrm{CO}_{2}$  含量(1ppm)还要低。

PSA与TSA相比，最大的优点是能耗小、占地面积少，缺点在于切换频繁，有一定的气量损失，且阀门的质量要高。由于今后工厂的发展对土地利用价值的要求会越来越高，所以在制氧工艺中若有高效率的吸附剂及高质量的切换阀门，PSA技术会得到更加广泛的利用。

# 一、问题的提出

目前，我国生产的空分设备基本上都处于设计工况状态下运行。这是因为空分设备操作比较复杂，要求操作工有丰富的实践经验。为了保证产品的产量、纯度和工况的稳定，一般希望在设计工况下进行生产。但在实际的生产过程中，由于用气量的负荷，随着后续生产过程的需要量而变化，一般情况下，适应这一变化的途径，只能在装置生产能力不变的情况下，进行大量产品气的放空，这样就势必造成能量的很大的浪费。如何有效地解决这一难题，一直是空分行业长期以来所面临的课题。

因此，一套先进的空分设备，除了要有先进的工艺流程、先进的设备制造工艺、先进的装置安装水平及严格的质量管理外，一套可靠先进的过程控制系统也是必不可少的。如果我们的控制系统能适应生产过程中用产品气量的变化，自动进行负荷变化跟踪，通过改变产量设定值这一手段，全自动的对相关调节回路参数进行计算、整定，就可使整个空分设备快速稳定在新工况下运行，从而满足生产过程变化的需要，达到节

能目的。我们把这一适合生产过程负荷变化的控制称为变负荷控制技术。

由于制氧过程是一个强耦合、纯滞后、非线性严重、多输入多输出的缓变复杂系统，如何使空分设备在负荷变化生产中，进一步发掘潜力，降耗节能，这就转化成控制系统将综合这一系列变化过程，建立最优控制思想，实现最优操作，提高过程控制水平，建立向控制要效益的观点。这一切若用传统的常规仪表控制是很难实现这一高级控制方案的，而计算机控制系统的应用，给变负荷控制的实现带来了可能。实现这一可能的前提是，必须深入分析工艺特点及内在关系，构成合理、灵活的多变量复杂调节系统。

# 二、自动变负荷控制

# 1. 自动变负荷控制模式

变负荷操作方式通常有两种：一种是空气选择方式；另一种是氧气选择方式，图1是模式切换图。当模式方式选定后，确定  $\mathbf{y} = \mathbf{f}(\mathbf{x})$  对应的函数关系，而模式的切换点并非在任何工况上都能进行。从图1得知，当A点工况连续变化到B点工况，即x1增加到x2，而y1也相应增加到y3，这时空分

设备处于产气工况。当B点工况变化到C点工况（或反之），即y3变化到y2，这时空分设备处于产液工况。而模式的切换必须经过两种模式的结合点才能进行，这样充分考虑系统的特性和要求，这时设备实现无扰动平稳切换。

# 2. 变负荷控制的参数

料平衡及组分平衡的基础上的。冷量收支不平衡就会导致物料、组分平衡的破坏，同样物料组分不平衡也会影响冷量的平衡。图2是自动变荷控制框图。

由图2可知，无论是空气模式或氧气模式，若改变主参数值，其余一切参数将按着各自对应的函数关系进行自动跟踪设定。这样，将被控的一切工艺参数调整到所期望的目标值。但工艺参数受非线性和滞后性等因素影响，输入量的改变，不可避免地要引起瞬态过程中的误差。此误差必须控制在允许的范围内，否则联锁系统自动中止变负荷指令执行。因此改善动态特性，提高系统准确度，这将是变负荷控制中的一个重要环节。这并非简单的外给定控制，而是一个综合的复杂高级控制，要运用一系列的判断、比较、计算处理，根据参数趋势变化进行动态补偿。

# 三、计算机控制系统

# 1. 工艺分析和控制方案

根据变负荷工艺过程多变量的输入输出复合的复杂关系，设计过程中与设计相关数据的不确定性和系统明显的滞后性，以及众多相关调节回路动态物料组分无法准确了解、机理不清、难于建模等特点，对于这样的复杂控制问题，仅用现有传统调节理论是难以解决的。其原因是整个空分设备在变负荷过程中，综合性设计目标函数的确定与过程函数的不确定之间的差异，及对各单元设备工艺参数动态变化的约束影响。这一切最终表现在反映过程终点结束标志，即变负荷控制的时间指标。从图1得知，从A操作点调整到B操作点所需要的时间，也就是最优控制所寻找的过程变化的终点标志。当然前提是保证空分设备正常运行。

由于对被控对象尚无完全清楚的了解，应用精确的控制理论实现全过程优化比较困难，但可以根据对象的具体状况进行局部优化，以优化的工艺操作点为基础，以节能为目标，协调相关参数工作点的内在关系，或对某个具体参数进行优选，从而达到最佳的变负荷控制时间指标。

因此，在经过对工艺深入认识、机理分析、数据计算、非线性回归，以及对目标函数有着密切相关的多变量输入函数的充分研究比较后，制定了变负荷控制方案。

主要的设计步骤是：

1）模式转换相互间的约束跟踪控制；

2）变负荷增减指令自动控制；  
3）相关参数调节回路目标值跟踪控制；  
4）调节器回路状态自动跟踪切换控制；  
5）计算现实的偏差和实际变化的斜率与目标函数斜率偏差的跟踪控制；  
6）偏离模式切换点操作无效的记忆跟踪控制；  
7）函数  $\mathbf{y} = \mathbf{f}(\mathbf{x})$  变化量与回路状态相关制约的跟踪控制；  
8）各函数斜率变化率与产品纯度的相关制约的控制；  
9）寻找最佳机理变化时间的综合控制。

# 3. 仿真试验

在完成设计金川自动变负荷软件后，进行了仿真实验。这是基于两个原因：①静态系统连动是整个设计的一个重要组成部分；

② 空分行业首套变负荷软件的自行设计，需找大量实验数据，从而完善整个控制方案。

根据系统的特点及对重要回路和工艺参数的深入研究，认为利用计算机丰富强大的功能，采集大量的数据是优化过程的最好资料来源，由DCS仿真采集测量的众多数据中必定隐含着变负荷过程的信息，从而为建模、寻找优化的操作条件和补偿策略、探索反应机理，创造了有利条件。经过系统特性分析，仿真输入信号为阶跃函数，其输入函数产生方法可以利用DCS内部仪表来实现。

通过仿真试验，根据趋势图大量信息显示，理论值与实际值的逼近误差程度直接反映到所有  $y = f(x)$  这一函数的特性曲线，由于多变量系统，对于所有传递函数所引起的误差迭加，对一个实际系统来讲，必须避免，否则将直接影响整个控制系统的成败。

试验中进一步了解了对于变负荷控制模式，切换开始下达增减指令的延迟功能纯滞后时间的数值，与空分设备的相对内在关系。

# 2 制氩系统流程简介及主要技术参数

# 2.1 流程简介

从上塔提馏段氩富集区抽取  $8\% \sim 12\%$  Ar的氩馏分气，送入粗氩塔进行精馏分离，在塔顶获得纯度为  $96\% \mathrm{Ar}$  、  $2.5\% \mathrm{O}_2$  、  $15\%$ $\mathbf{N}_2$  的粗氩，经复热后送至氩净化系统加氢除氧后，经冷冻机和干燥器除去水分后获得工艺氩，然后经氩换热器冷却至-165～ $-170^{\circ}C$  ，进入精氩塔，经进一步精馏后获得纯度为  $99.99\% \mathrm{Ar}$  的纯氩。

# 2.2 主要技术参数

粗氩产量  $25\mathrm{m}^3 /\mathrm{h}$  ，纯度  $96\%$  Ar、 $2.5\% \mathrm{O}_2$  、  $1.5\% \mathrm{N}_2$  。

精氙产量  $20\mathrm{m}^3 /\mathrm{h}$  ，纯度  $\geqslant 99.99\% \mathrm{Ar}_{\circ}$

# 3 调氩经过

1995年6月首次投入氩生产，在主塔工况稳定的情况下，粗氩塔投运较快，且运行稳定，生产出合格的粗氩。在粗氩塔稳定运行数天后，即加氢除氧，试投精氩塔，经艰苦调整，虽生产出了合格的精氩，但工况极不稳定。在当时有限的条件下，我们分析了原因，经多次反复整改，终于获得成功。在调试过程中，暴露了一些问题：

# 3.1 精氩塔工况不稳

3.1.1 精氩塔工况稳不住的症状

主要表现在精氩塔阻力以及冷凝器液氮面波动很大，操作工不得不频繁地调节精氩塔遥控阀JL-105（主液氮）、JL-106（补充液氮），但却很难找到工况平衡点，最终导致精氩塔被迫停产。

# 3.1.2 原因分析

针对以上症状，我们分析了原因，认为精氩塔冷凝器及本身塔体容积小，对物料平衡要求高，JL-106阀设计流量偏大，阀微小的动作，即引起平衡破坏，导致精氩塔工况波动，带来操作困难。另外，经试验，JL-105、JL-106遥控阀泄漏严重，根本无法关死，使操作工束手无策，精氩塔被迫停产。

# 3.1.3 改进措施

1995年底主塔大修，我们趁机检修了JL-105和JL-106两阀，并分别在阀前加装截止阀和冷角阀各一只，同时改小管径，以减少补充液氮流通量，使手动控制和遥控相互配合使用，稳定了精氩塔的操作。另外，由于加装了手动阀，在必要的时候，可实现精氩塔的单独加温吹除。

# 3.2 粗氩塔冷凝器堵塞

1996年3月，我们再次投入氩生产，精氩塔经上述改造后，工况相当稳定。但意外的事发生了：公司因故突然停电，导致空分紧急停车。当恢复送电后，在主塔稳定后再次投粗氩塔时，发现粗氩塔阻力无法提高，经多次努力，始终无法恢复正常。当时分析有两种可能：（1）粗氩塔冷凝器内部泄漏；（2）冷凝器被  $\mathrm{CO}_{2}$  或其他杂质堵塞。由于制造厂设计时粗氩塔与主塔相通，两者之间无切断阀门，粗氩塔无法单独加温。因此，对第二种判断也无证明手段。鉴于当时

公司形势，主塔不能停车，因此氩生产只能暂停。

1996年7月，我们利用空分主塔大修的机会，在制造厂的支持配合下，在粗氩塔和主塔之间加装了截止阀，使主塔和氩塔之间能完全隔开，必要时可实现单独加温。同时，对主塔主冷和粗氩塔冷凝器分别试压查漏，排除了泄漏的可能性。

那么粗氩塔运行不正常就很可能是冷凝器堵塞了。经过大加温后，调试粗、精氩塔运行，氩气生产恢复了正常。后由于主塔故障频繁，氩气生产时开时断。直到1997年2月，主塔运行正常后，我们再次投氩，至今已连续稳定运行八个多月，已扭亏为盈，为公司创造了可观的经济效益。

# 4 存在问题

（1）氩净化系统冷干机设计的制冷量，较实际需用冷量相对偏小。按照设计，经冷干机后，工艺氩温度为  $5\%$  左右，但实际运行时达  $10\sim 15\%$  ，特别在夏季还要高，加重了干燥器的负担，迫使我们缩短切换时间。  
(2）加氢小流量阀调节与控制严重滞后，现为手动操作，无法实现自动摇控。

# 5 结语

（1）从流程上看，开空厂  $1500\mathrm{m}^3 /\mathrm{h}$  板翅式带氩空分流程是合理的，各厂只要根据各自情况，作一定的改造，完全可以出氩。  
（2）目前国内氩气需求量缺口较大，如能将闲置空分设备“起死回生”，产生的经济效益是显著的。

# 提高XKAr-3-Ⅱ型制氙装置产量的优化操作

叶文福

（福建炼油化工有限公司供气车间，福建惠安，362100）

摘要 分析影响氩气产量的主要原因是粗氩纯度和纯化器使用周期。从操作上对提高粗氩纯度进行优化；针对粗氩纯度较高、而纯化器使用周期较短，分析原因，对纯化器5A、4A分子筛进行重新活化，活化后投入使用，效果很好，取得了较好的经济效益。

主题词：制氩装置 粗氩 纯度 氩气 产量 优化 操作

我公司XKAr-3-Ⅱ型制氩装置配套于KZON-150/600-3型空气分离设备，是杭氧集团公司1990年生产的产品。粗氩的精制采用分子筛低温吸附净化的工艺过程。1991年5月，KZON-150/600-3型空分设备进行试车一次成功，生产出合格的氧气、氮气。1991年9月对制氩装置进行试车，在试车过程中对制氩装置的操作摸索了一段时间，至1992年11月长周期平稳地生产出精氩。

在试车中，我们认为影响氙气产量最主要的因素是粗氩纯度和纯化器使用周期。现将如何优化操作提高粗氩纯度、纯化器正确使用的一些体会总结如下，供同行参考。

# 一、粗氩纯度控制在  $99\%$  以上，延长纯化器使用周期，提高氩气产量

粗氩纯度越高，精氩纯化器使用周期越长。因为粗氩纯度越高，氧、氮含量越少，纯化器吸附量达到饱和的时间就越长。北京电子管是国内第一家在  $150\mathrm{m}^3 /\mathrm{h}$  空分设备上用分子筛制氩，吸附温度为-  $183\%$  ，从实践摸索到粗氩纯度为  $92\%$  时，纯化器使用周期为12小时，当粗氩纯度提高到 $99.4\%$  ，纯化器使用周期为96小时。

粗氩纯度要提高到  $99\%$  以上，根据我们摸索，必须掌握以下几点：

1. 吹除油水，空气纯化器切换必须缓

慢进行

因为配带氩塔之后，主塔的灵敏度很高，所以在正常生产时，空压机和空气纯化器油水分离器的吹除，必须缓慢地进行，尤其是空气纯化器的切换更应缓慢。在试车初期，一到空气纯化器切换时，粗氩纯度波动就很大，从  $98\%$  下降至  $80\%$  。分析其原因有两点：① 空气纯化器切换速度过快；② 切换完毕出口温度过高。

针对上述现象我们采取两项措施：①规定空气纯化器切换时间，控制在1小时左右；②在空气纯化器后增设冷却器（ $\phi 600 \mathrm{~mm} \times 1590 \mathrm{~mm}$ ）。采取上述措施后取得较好的效果，空气纯化器切换对粗氩纯度没有影响。

# 2. 确保主塔稳定，即确保主塔液氧液面的稳定

由于配带氩塔设备，对主塔操作带来复杂性和敏感性，因此根本的操作应放在主塔，对主塔液氧液面的波动，应有预见性地调整，确保液氧液面的稳定。因为主塔液氧液面波动，要引起上塔提馏段富氙区的波动，粗氩纯度必然波动。

# 3. 把氩馏分蒸汽中氩含量控制在  $10\%$  左右

氙馏分组分的变化，对粗氙塔的工况影

响是极大的。如果氩馏分中氩含量过低，则氩馏分中氧含量必然过高，粗氩中氧含量也会过高，粗氩纯度大大下降，粗氩产量降低，粗氩提取率下降。如果氩馏分中氩含量控制为最大，则氩馏分中氮含量也多。氮是低沸点的不冷凝气，聚集在粗氩塔的顶部，使粗氩中的含氮量比氩馏分的含氮量大25倍，粗氩纯度也降低，严重的会导致粗氩冷凝器的温差减小，甚至达到零，此时粗氩塔便停止工作。所以氩馏分抽口应该设在含氮尽量少的地方，一般含氮不超过  $0.02\%$  。根据我们的实践，氩馏分中氩含量控制在 $10\%$  ，粗氩产量最大（氮含量  $<  0.5\%$  ）。

氩馏分含量的控制，通过调节主塔氧、氮产量和纯度来实现。关氮阀、开氧阀使主塔提馏段富氩区下移，则氩馏分中氩含量提高，含氧量降低，同时含氮量也相应增多。关氧阀、开氮阀则会出现相反的情况。调整这两个阀要有预见性，同时动作要缓慢。

调节的主要依据是氩馏分含氧量的多少，其次是粗氩塔的阻力、液空气量和主塔氧、氮产品的纯度。如果粗氩塔阻力下降，则说明氩馏分含氮量增加，应关氧阀、开氮阀。若液空气量减少，粗氩塔冷凝器温差减少，氩馏分含氮量增加，此时应关氧阀、开氮阀。

为了提高氩的提取率，在保证氩馏分含氮不增多的情况下（即氩塔阻力计不显著下降的情况下），应当尽量提高其含氩量。经验证明氧纯度变化  $0.1\%$  ，氩馏分中氧含量就要变化  $0.6\% \sim 0.8\%$  。因此带氩塔的空分设备，只要控制好氩馏分中的氧含量，氧气纯度也就稳定了。

# 4. 把液空纯度控制在  $32\%$  左右

液空纯度如果过低，会使上塔精馏效果变差，氧、氮纯度提不高。液空纯度过高，对氩塔不利，这是因为氩沸点在氧、氮之间，若液空纯度过高，必然会使液氮中氩增多，并带入上塔逸出。另一方面液空含氧量过高，当液空送入粗氩塔冷凝器发器后，其蒸发温度和粗氩冷凝温度之差减少，又会使

粗氩塔的工况受到影响，不利于粗氩纯度的提高。

# 5. 把液氮纯度控制在  $99.95\%$  以上

制氩设备对液氮纯度要求较高，在保证液空纯度  $32\%$  基础上，应尽量提高液氮纯度（一般保证在  $0.05\% \mathrm{O}_2$  以下）。这是因为液氮中含有一定数量的氩进入上塔顶部，虽氧、氮的逸入值相差很大，但总是有相当数量的氩被氦带出，影响氩气提取率。

# 6. 粗氩中氧、氮含量同时增加时的处理方法

先驱氮后驱氧。即在保证主塔液氧液面稳定和氩馏分含氧量在  $90\%$  左右的基础上，开大粗氩出口约10分钟，当粗氩中含氮减少（反映在阻力增加、液空蒸汽增大），再将粗氩产量控制在  $2\mathrm{m}^3 /\mathrm{h}$  左右，待粗氩含氧量小于  $2\%$  ，粗氩纯度  $95\%$  以上时，逐步开大粗氩产量到正常。

# 二、确保氩塔纯化器分子筛彻底活化，增强分子筛吸附能力，延长纯化器使用周期

氩塔纯化器的使用周期除了与粗氩纯度有关外，还与分子筛活化程度有关。试生产过程中，我们将粗氩纯度始终控制在  $99\%$  以上，氩吸附器经抽空、置换、冷冻之后投入使用，左组吸附器最长使用周期28小时，右组吸附器最长使用周期是56小时。根据上述现象，我们分析是氩纯化器分子筛没有活化彻底，必须进行重新活化。

民币6700元，使用周期超过了同行较好水平（96小时）。

1992年11月13日～16日也对右组吸附器分子筛进行活化。右组5A分子筛出口活化温度  $+330^{\circ}\mathrm{C}$ ，恒温16小时；4A分子筛出口活化温度  $+330^{\circ}\mathrm{C}$ ，恒温20小时，马上装入吸附器，并通氮气吹冷。接着对右组吸附器进行抽空、置换、冷冻，于1992年12月17日12:30投入使用，一直使用到12月22日7:30，分子筛才饱和。其使用周期为115小时，共充氩气瓶72瓶，折人民币7200元，其使用周期也超过了同行较好水平。

系统的报警功能，是当某些检测参数超过了设计的规定值或某机组非法起停或阀门非法开关时，提醒操作人员及时处理。联锁功能是当一些参数超越规定的限值，以至会影响工况或机组的安全时，自动打开或关闭阀门，或开、停运转机组，以保护系统的安全。应用了PLC控制系统，这些功能都自动检测、报警及联锁，对操作人员能够及时提醒，对设备起安全保护作用。

直观的流程图画面显示，能够动态地监视过程参数和过程变量，操作人员只要调出有关的流程图画面，就能看到有关检测参数的值、观看诸如液位等参数的动态变化，以及阀门的开启情况等，这是直观、动态的显示画面，替代了半模拟盘的静态流程画面。

# 二、DW型低温胶粘剂在修复空分设备主冷凝蒸发器上的应用

南京化学工业（集团）公司氮肥厂 $6000\mathrm{m}^3 /\mathrm{h}$  空分设备主冷凝蒸发器板式单元大面积泄漏，广东佛山铸钢厂氧气车间150 $\mathrm{m}^3 /\mathrm{h}$  空分设备分馏塔冷凝蒸发器外筒体大面积泄漏，以及江苏仪征化纤公司动力厂 $2000\mathrm{m}^3 /\mathrm{h}$  制氮设备深冷管线裂纹的修复，均采用DW型低温胶粘剂。根据不同的泄漏部位及情况，采用不同的粘补工艺，均取得了良好的使用效果。现将南京化学工业（集团）公司氮肥厂  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备主冷凝器泄漏的修复情况做一详细介绍。

# 1. “6000” 空分设备主冷凝器泄漏情况

该主冷凝蒸发器外形尺寸为  $\phi 2116\mathrm{mm}$ $\times 8\mathrm{mm}$  ，内装有5个规格为  $750~\mathrm{mm}\times 620$ $\mathrm{mm}\times 2100\mathrm{mm}$  的板翅式换热单元，每单元有41层液氧蒸发通道，通道隔板厚0.5 $\mathrm{mm}$  ，每层通道的尺寸为  $750\mathrm{mm}\times 6.5\mathrm{mm}\times$ $2100\mathrm{mm}$  ，5个单元共有205个通道，通道上方尺寸为  $6\mathrm{mm}\times 1\mathrm{mm}\times 2100\mathrm{mm}$  。经裸冷试车以后发现：空分设备主冷凝蒸发器板式

单元严重泄漏，在该主冷凝器5个板翅式单元的44个通道上有80多处泄漏。

板翅式换热器，一旦在通道上出现泄漏，便来不及象管式换热器那样用堵头堵管，而要采用氩弧焊将整个通道焊死才能堵漏。而且，此法施工困难，还易发生事故。

# 2. 选用胶种，确定粘接工艺

针对以上特点，修复主冷凝蒸发器泄漏的低温胶不仅要满足  $-183\sim -196^{\circ}\mathrm{C}$  深冷状态下运行的工艺条件，还必须满足常温条件下加压查漏，以及频繁地从深冷状态到 $40^{\circ}C$  左右气温中开停车的考验。其次，低温胶本身一定要流动性好，固化容易。因为要从  $6\mathrm{mm}\times 1\mathrm{mm}$  的通道口注胶，并使2100mm的整个通道底部注满胶，则胶的流动性一定要很好；在如此多的细长的通道内凝固，则胶的固化一定要很好。

用  $\phi 5\mathrm{mm}$  的玻璃管测定在连续注胶的情况下，DW-3型胶的流速为  $200~\mathrm{mm / s}$  。DW-3型胶适用温度范围  $-269\sim 60^{\circ}C$  ，流动性较好，适用于膨胀系数不同材料之间的粘接，在室温条件下可以预固化。DW-4型胶起始粘接力高、固化快、韧性和弹性特别好。针对以上特性，选用流动性好的DW-3型胶作为通道灌注，选用固化快的DW-4型胶作表面铝板贴衬用。

1984年，上海第五钢铁厂曾用整条通道贴封的方法来注塞一个泄漏点。但我厂主冷凝器中有44个通道上的80余处泄漏点，如采用上钢五厂的堵漏方法，将损失 $21.5\%$  以上的热交换面积，显然不宜采用。而在施工技术上，应根据泄漏的部位以及泄漏程度的大小分别处理。

我们在泄漏严重的4个通道上，采用全通道封闭粘堵的方法。在通道上下各用DW-4型胶浸润  $750\mathrm{mm}\times 8\mathrm{mm}$  的玻璃布两层粘贴，再将铝质槽型条压入通道口，并用重物压贴牢固。见施工图图4。

# 延长  $150 \mathrm{~m}^{3} / \mathrm{h}$  制氧机运行周期至一年的措施

# 陈政忠

（福建化纤化工厂动力分厂，福建省永安市汶川，366016）

我厂是化工联合企业，共有三套150 $\mathrm{m}^3 /\mathrm{h}$  制氧机，为二开一备，主导产品为氮气，纯度大于  $99.5\%$  的氮气需连续不断地供给乙炔站、电石破碎、电石储仑以及有机合成等工序作保护气。为确保设备安全运行，我们一方面抓好工艺生产和职工业务培训工作；另一方面抓好技改项目，进一步降低能耗。这样，使分馏塔运行周期从原来的2~3个月延长到12个月，与总厂安排的一年大修相协调。我厂  $150\mathrm{m}^3 /\mathrm{h}$  制氧机长周期运行所采取的措施是——

1. 用PLK-8.33/20-6型空气轴承透平膨胀机代替PZK-14.3/45-6型活塞式膨胀机，减少了由活塞环、导向环及阀杆密封填料漏气等引起的活塞式膨胀机较多的故障。  
2. 加强职工技术业务的培训，提高责任心，按时排放5L-16/50型空气压缩机各级油水，并控制气缸润滑油量（各注油点滴油量为  $8\sim 10$  滴/分）。  
3.增设GYL-16/50型高效除油过滤器（西安压缩机厂净化设备厂产品），以减少润滑油进入分子筛吸附器，避免了分子筛中毒。  
4. 将5A分子筛更换为上海环球分子筛有限公司生产的UOP13X-APG分子筛。自从1992年更换为该分子筛后，纯化器使用周期从8小时延长到12小时，再生温度

（加热炉出口温度）从  $300\%$  左右降至 $220\%$  ，再生后出口温度为  $80^{\circ}C$  时即告加热结束。5A分子筛使用寿命短，我厂一般为两年更换一次；而UOP13X-APG分子筛已使用五年，还未更换。对吸附后的高压空气进行取样分析，吸附器使用12小时后  $\mathrm{CO}_{2}$  的含量均在  $12\mathrm{ppm}$  以下。

5. 在使用透平膨胀机组后，我们果断地采用江西九江交通电器厂生产的DTZ-B-202型透平膨胀机专用智能转速表，确保了透平膨胀机的安全运行。  
6. 分馏塔运行  $8 \sim 9$  个月，若发现液氧混浊（可用容器排放一部分，让其自然蒸发后观察），可采取以下措施：每班（8小时）排放  $5 \sim 6 \mathrm{~cmCCl}_{4}$  柱液氧，约  $4 \sim 5$  天，即可消除液氧浑浊现象。  
7. 为确保分馏塔正常运行，我厂将三套制氧机的空压机三段出口管联通、分子筛吸附器后管道联通（再生用氮气管联通）、保留原活塞式膨胀机并与透平膨胀机联通（在活塞式膨胀机进出口管道各加装弯阻阀，以备互用），以达到连续运行的目的。

由于采取以上措施，使我们的  $150\mathrm{m}^3 /\mathrm{h}$  制氧机能够安全的稳定的长周期运行，并实现了节能降耗。

# KL-15型制氧制氮车振动故障分析

# 黄发新

（空军驻杭州地区军事代表室，杭州市东新路90号杭氧内，310004）

摘要 详细分析了KL-15型制氧制氮车发生振动故障的原因及危害，最后从设计、制造和使用三方面介绍了此类故障的预防措施。

主题词：车载式 制氧制氮车 空压机 制冷机 振动 故障 分析 预防

振动故障是以空压机、制冷机为主要配套部机的空分设备试车中遇到的常见故障。对于车载式KL-15型制氧制氮车来说，这种故障更为常见。本文试就KL-15型制氧制氮车产生振动的原因、危害以及预防措施等问题作一探讨。

# 一、KL-15型制氧制氮车产生振动的原因

KL-15型制氧制氮车产生振动的原因是多方面的，归纳起来主要有以下六个方面。

1. 千斤顶固定不好，开车时汽车轮胎接地产生振动

这种问题在制造厂内和部队试车中均有发生。相对于安装在地面的空分设备，车载式制氧制氮设备，由于装有机器设备的汽车是通过固定在底盘上的千斤顶与地面接触的，接触面积小，稳定性差，容易产生振动。如果千斤顶固定不好，更容易使振动增大。尤其是当千斤顶升得不够高或固定不牢时，汽车轮胎就会接地，开车中车上机器设备产生的振动，通过车身使汽车轮胎与地面（特别是当汽车轮胎与地面似离非离时）产生撞击，就会引起很强烈的振动。

2. 机器与底座安装连接中减振措施不力、固定不好产生振动

空压机与底座、四缸制冷机与底座安装中减振措施不力、固定不好产生的振动是主

要的，电机与底座及联轴器次之。按照规定，机器与底座、机器与电机的连接均应装有足够的橡胶垫等吸振材料，用以阻尼、减振。底脚螺栓的固定应牢靠，受力应均匀。由于橡胶垫太薄或老化以及底脚螺栓受力不均，致使个别螺栓应力集中而断裂，都可能降低减振效果，产生很强烈的振动。

3. 电机与机器的连接不同轴产生振动

KL-15型制氧制氮车在制造和维修中，有可能产生因四缸制冷机与电机、水泵与电机的连接安装不同轴问题，包括上下不水平、左右不同心，从而增加了运转中的四缸制冷机等机器的扭矩，产生偏转，引起振动。

4. 配合间隙超差引起振动

空压机、制冷机内部活塞与气缸、曲轴与连杆等运动部件之间的配合间隙都是有允许范围的，当选配不当或使用磨损超过规定时，对机身的振动会产生两个方面的负面影响：一是影响振动加速度谱的变化，会使功率谱中高频成分明显增加；二是影响振动加速度总振级的变化。

5. 旋转运动的部件动不平衡引起振动

空压机、制冷机的飞轮、曲轴、电机的转子等旋转运动的部件，由于材料缺陷、制造误差、重力下垂、安装误差、受热变形等原因，存在质量不平衡，升速时由于离心现

象产生较大振动。这样的旋转部件尽管平衡试验也是合格的，但由于试验台的转速与实际转速的误差，部件上残留着较多的不平衡质量，由此引起振动。

6. 由于空压机进排气导管太短、固定太紧而产生振动

按照产品图样规定，空压机进排气导管是安装时单配的，长度视每套机器安装位置而定，过长过短均不合适。过长装不上，而过短则会由于强迫装配，增加装配拉应力。从而破坏了空压机的垂直度、稳定性，导致振动。

除此以外，我们还发现过两套制氧车由于空压机安装的底座槽钢架焊接强度不足而产生振动。该产品的底盘车由JN151型改为JN1171/017型后，由于空压机安装底座比原车型升高了  $50~\mathrm{mm}$  多，增加了空压机的不稳定性，再加上其底座槽钢架只作单面焊接，强度不足，开车时产生强烈振动。

# 二、KL-15型制氧制氮车产生振动故障的危害

从流程分析及制造、维修实践可知，振动故障对KL-15型制氧制氮车的正常使用构成很大危害。

# 1. 影响工况

首先，空压机或四缸制冷机的强烈振动会引起分馏塔的晃动，破坏塔板水平，从而影响氧和氮的纯度。其次，振动会影响分馏塔内液氧、液氮流动的均匀性、连续性和流动速度，妨碍正常分馏，因而影响氧和氮的产量。

# 2. 危及安全

此危害包括两个方面：一方面，长时间强烈的振动容易引起一些导管的连接松动、裂纹，焊接处脱焊，导致漏气、漏液，损坏设备，甚至危及人身安全。特别是当氢气因导管振裂泄漏时还会发生燃爆事故。分馏塔内导管振裂或振脱（焊接处）时会因漏液而

使导管跑冷；另一方面，强烈的振动特别是长时间强烈的振动，会引起操作人员目眩、疲劳、头痛、头晕等振动性疾病，影响操作，严重时能致人晕倒，引起工伤事故。

# 3. 影响机器设备的使用寿命

振动会使空压机、制冷机的气缸与活塞、连杆与曲轴、轴承与轴承座的磨损加剧，严重时导致这些零部件报废。振动还会引起空压机皮带受力不均，导致皮带变形、松脱或断裂，不能工作。凡此种种，都会影响KL-15型制氧制氮车的使用寿命。

# 三、预防KL-15型制氧制氮车振动故障的措施

鉴于振动的危害性，采取针对性预防措施，将KL-15型制氧制氮车的振动降低到最低程度，将振幅和频率控制在规定范围内，显得十分必要。

# 1. 掌握判断振动故障的标准及方法

振动是KL-15型制氧制氮车试车中出现的普遍现象，正常的振动是不可避免的，而振动故障则是应当竭力避免的。首要的问题是如何判断正常的振动与振动故障，判断不当，会造成混乱。如将正常的振动误判为振动故障，会因所谓的排故花费不必要的人力、物力与财力；反之，则会因排故不及时而影响制氧制氮设备的正常使用，甚至会因为延误排放时间而造成设备损伤以致报废。

判断振动正常与否的方法，除了用专用的振动仪测量各部机及整车试车时的振幅和频率外，根据我们的实践，通常还可以用以下简便方法判断：一是听觉检查，用一把解刀，金属一头贴往被测部位，一般为空压机、四缸制冷机的壳体及配套电机的壳体，木把贴近检测者的耳朵，倾听机器运转声音是否均匀，有无杂音，特别是有无剧烈的跳动声，如有剧烈的跳动声（有时用手也能感觉到），就可能是振动故障。二是目测检查，这对于立式空压机、分馏塔更为适合，当振

动强烈即发生明显的振动故障时，这些机器、设备会出现强烈的晃动，有目眩之感。

2. 要从设计、制造和使用三个方面采取针对性的预防措施

# （1）设计上要不断改进

包括连接固定方式和安装材料的改进，以提高机器设备的稳定性和使用可靠性。机器设备的连接固定要牢固可靠，安装材料的减振性能要好。如从1996年第五套制氧制氮车开始，我们督促湖州客车厂在空压机底座的槽钢架上增加支撑斜梁（即加强筋），并要求双面焊接，提高了减振性能和强度。

# (2）制造时要严格监控

高速旋转的飞轮、叶轮等部件的材质要均匀、无缺陷且平衡性能要符合要求，气缸与活塞、连杆与轴瓦的配合间隙要严格控制在允许范围内。连杆、曲轴、飞轮的装配、

各电机与机器的安装联接要保持同心和水平，静止状态用手转动应灵活，不应有卡滞感。空压机和制冷机及电机的底座连接螺栓的固定要牢靠，紧度要适中。

# (3）使用中要加强检查

在KL-15型制氧制氮车使用中出现振动故障，一般不是突然发生的，都是有征兆、端倪可寻的。因此，只要加强检查，是完全可以及时发现、及时预防的，部队制氧人员在使用维护中，一定要加强对KL-15型制氧制氮车空压机、制冷机等机器运动部件的检查，通过“听、看、摸”查找振动故障隐患。发现固定松动，应及时予以紧固；听到异常声音，要查明原因，采取措施，以避免振动进一步发展成为故障。

主题词：低温容器 受力 破坏性变形 实例 计算 分析

# 一、引言

常压低温液体容器具有容积大、承压低等特点，多采用圆柱形钢质平底立式结构，内胆底部用高强度保温砖绝热，四周采用粉末堆积绝热。为提高容器的稳定性，在内胆底面圆周上，用均匀布置的多根拉杆固定于基础平台。拉杆一般较长，有利于加长热桥，减少导热。在由容器筒体、容器底面以及拉带构成的静不定系统中，各部分受力各不相同，容器内压力升高，各部分应力随之增大。此外，容器在盛液状态下，内胆与暴露于大气中的外壳存在较大温差，不可避免地存在因温差引起的附加热应力，倘若在不绝热的情况下进液（如容器裸冷），则温度应力对变形的影响，又不同于绝热情况下的盛液。

本文试图对容器在绝热与不绝热两种情况下盛液时，各部分的受力与变形进行定量分析，推导出容器发生破坏性变形的条件，即容器能承受的最大压力，并将推导结果应用于一例破坏性变形的实例分析，指出在使

用中应如何正确把握，避免容器破坏性变形的发生。

# 三、容器在绝热情况下受力与变形分析

容器在绝热情况下盛装一定量的液体，液面上气体压力记为  $\mathbf{P}$  (MPa)，液体高度记为  $\mathrm{h}_0$  (m)，液体静压力记为  $\mathbf{P}_0$  (MPa)，则

$$
\mathbf {P} _ {0} = \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6} \tag {1}
$$

式中  $\rho$  ——液体密度

g- 重力加速度

# 1. 筒体的受力与变形分析

① 受内压力  $\Sigma \mathbf{P}$  作用产生的应力与变形

假定筒体直径为D，壁厚为t，高度为h，在内压力  $\Sigma \mathbb{P}$  作用下，筒体产生两个方向的应力：与筒体轴线垂直的横截面上的正应力  $\sigma^{\prime}$  和通过筒体轴线的纵向截面上的正应力  $\sigma''$  （如图2），因此筒体受到纵向和横向两个方向的拉伸。假定筒体受拉为正，受压为负，材料的弹性模量为E、线应变为ε，则应力和变形为：

$$
\sigma^ {\prime} = \frac {\sum \mathrm {P} \cdot (\frac {\pi \mathrm {D} ^ {2}}{4})}{\pi \mathrm {D t}} = \frac {\sum \mathrm {P} \cdot \mathrm {D}}{4 \mathrm {t}} \tag {2}
$$

$$
\sigma^ {\prime \prime} = \left[ \int_ {0} ^ {\pi} \Sigma P \cdot h \cdot (D / 2) \sin \psi d \psi \right] / 2 t h =
$$

$$
\Sigma \mathrm {P h D} / 2 \mathrm {t h} = (\Sigma \mathrm {P} \cdot \mathrm {D}) / 2 \mathrm {t} \tag {3}
$$

纵向变形量为：

$$
\begin{array}{l} \Delta h _ {t} = \varepsilon \cdot h = (\sigma^ {\prime} / E) h = \\ (\Sigma \mathrm {P} \cdot \mathrm {D}) / (4 \mathrm {E t}) \mathrm {h} \tag {4} \\ \end{array}
$$

对于液面以上的筒体，内压力  $\Sigma \mathrm{P} = \mathrm{P}$ ；对于液面以下的筒体，内压力  $\Sigma \mathrm{P} = \mathrm{P} + \mathrm{P}_0'$ $= \mathrm{P} + \rho \mathrm{g}(\mathrm{h}_0 - \mathrm{h}') \times 10^{-6}$

式中  $\mathbf{h}^{\prime}$  ——任意点处的筒壁高度

显然当  $\mathbf{h}^{\prime}\rightarrow 0$  ，即在接近于底面的筒壁上，  $\Sigma \mathbb{P}$  达到最大，此时

$$
\Sigma \mathrm {P} = \mathrm {P} + \mathrm {P} _ {0} \tag {5}
$$

将式（1）、（5）代入式（2）、（3），得筒壁最大应力：

$$
\sigma_ {\max } ^ {\prime} = \left(\mathrm {D} / 4 \mathrm {t}\right) \left(\mathrm {P} + \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6}\right) \tag {6}
$$

$$
\sigma_ {\max } ^ {\prime \prime} = (D / 2 t) \left(P + \rho g h _ {0} \times 1 0 ^ {- 6}\right) \tag {7}
$$

将  $\overline{\Sigma P} = P + (\rho gh_0 \times 10^{-6}) / 2$  代入式(4)得筒体的纵向变形量：

$$
\Delta h _ {1} = \frac {\mathrm {D h}}{4 \mathrm {E t}} \left(\mathrm {P} + \frac {1}{2} \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6}\right) \tag {8}
$$

② 温度应力和变形量

在绝热情况下，盛液后的筒体内，外壁面均由常温  $\mathfrak{t}_{\mathrm{a}}$  变成液体温度  $\mathbf{t}_{\mathrm{f}}$  。由于筒体上端是自由端，筒体可以自由伸缩，所以不产生温度应力。假定材质的线膨胀系数为  $\alpha$  且不受温度变化的影响，则筒体的纵向变形量为：

$$
\Delta \mathrm {h} _ {2} = - \alpha h \Delta \mathrm {t} _ {1} = - \alpha h \left(\mathrm {t} _ {\mathrm {a}} - \mathrm {t} _ {1}\right) \tag {9}
$$

筒体在纵向总的变形量为：

$$
\begin{array}{l} \Delta h = \Delta h _ {1} + \Delta h _ {2} = \\ \frac {\mathrm {D h}}{4 \mathrm {E t}} \left(\mathrm {P} + \frac {1}{2} \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6}\right) - \\ \alpha h \left(t _ {a} - t _ {1}\right) \tag {10} \\ \end{array}
$$

③ 简体发生破坏性变形的条件

对于筒体而言，只要满足强度要求，就不会发生破坏性变形。筒体壁面上的三个主应力分别是：

$$
\sigma_ {1} = \sigma_ {\max } ^ {\prime \prime} = \left(\mathrm {D} / 2 \mathrm {t}\right) \left(\mathrm {P} + \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6}\right) \tag {11}
$$

$$
\sigma_ {2} = \sigma^ {\prime} _ {\max } = (D / 4 t) \left(P + \rho g h _ {0} \times 1 0 ^ {- 6}\right) \tag {12}
$$

$$
\sigma_ {3} = 0 \tag {13}
$$

设材料的许用应力为  $\sigma_{s}$  ，由材料力学的强度理论，可知筒体的强度条件是  $\sigma_{1} - \sigma_{3} \leqslant \sigma_{s}$  ，故筒体发生破坏性变形的条件应是  $\sigma_{1} - \sigma_{3} > \sigma_{s}$  ，将式（11）、（13）代入，得

$$
\left(\mathrm {D} / 2 \mathrm {t}\right) \left(\mathrm {P} + \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6}\right) - 0 > \sigma_ {\mathrm {s}}
$$

整理得  $\mathrm{P} > (2\mathrm{t / D})\sigma_{\mathrm{s}} - \rho \mathrm{gh}_{0}\times 10^{-6}$  (14)

式（14）即是在绝热情况下筒体发生破坏性变形的条件。

# 3. 拉带的受力与变形分析

忽略拉带随内胆底面径向收缩而发生的位移，视筒体对拉带的力沿拉带轴线方向。设拉带厚度为  $\mathbf{a}_2$  ，宽度为  $\mathbf{b}_2$  ，长度为  $\mathbf{L}$  。假定拉带材质与内胆相同。

筒体对拉带的力为：

$$
\begin{array}{l} F = \frac {\sigma_ {2} \pi D t - \rho_ {1} g h \pi D t \times 1 0 ^ {- 6}}{n} = \\ \frac {\pi \mathrm {D} ^ {2}}{4 \mathrm {n}} \left(\mathrm {P} + \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6}\right) - \\ \frac {\rho_ {1} \mathrm {g h} \pi \mathrm {D t} \times 1 0 ^ {- 6}}{\mathrm {n}} \\ \end{array}
$$

拉带的拉应力为：

$$
\sigma_ {6} = \frac {\mathrm {F}}{\mathrm {a} _ {2} \mathrm {b} _ {2}} =
$$

$$
\begin{array}{l} \frac {\pi \mathrm {D} ^ {2}}{4 \mathrm {n a} _ {2} \mathrm {b} _ {2}} \left(\mathrm {P} + \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6}\right) - \\ \frac {\rho_ {1} \mathrm {g h} \pi \mathrm {D t} \times 1 0 ^ {- 6}}{\mathrm {n a} _ {2} \mathrm {b} _ {2}} \tag {18} \\ \end{array}
$$

拉带的温度应力为：

$$
\begin{array}{l} \sigma_ {7} = - \alpha \mathrm {E} \Delta t _ {2} = \\ - \alpha E \left[ t _ {a} - \left(t _ {a} + t _ {1}\right) / 2 \right] = \\ - \alpha E \left(t _ {a} - t _ {1}\right) / 2 \tag {19} \\ \end{array}
$$

拉带发生塑性变形的条件是  $\sigma_{6} + \sigma_{7} > \sigma_{s}$ ，将式（18）、（19）代入并整理得

$$
\begin{array}{l} \mathrm {P} > \frac {4 \mathrm {n a} _ {2} \mathrm {b} _ {2}}{\pi \mathrm {D} ^ {2}} \left[ \sigma_ {\mathrm {a}} + \frac {\alpha \mathrm {E} \left(\mathrm {t} _ {\alpha} - \mathrm {t} _ {1}\right)}{2} \right] + \\ \frac {4 \rho_ {1} \mathrm {g h t} \times 1 0 ^ {- 6}}{\mathrm {D}} - \rho \mathrm {g h} _ {0} \times 1 0 ^ {- 6} \tag {20} \\ \end{array}
$$

因此，底面发生破坏性变形的条件，即是式（17）、（20）两式求得的较大者。

# 四、容器在不绝热情况下的受力与变形分析

若容器在不绝热的情况下进液，一般液体是存不住的，所以  $\mathbf{P}_0 = 0$  ，内压力  $\Sigma \mathrm{P} = \mathrm{P}$  。

# 1. 筒体的受力与变形分析

与绝热情况相比，在不绝热情况下，筒体附加了沿轴线方向的温度应力。因为筒体外壁面始终保持常温  $t_{\mathrm{a}}$  ，由外向里温度逐渐降低，直至内壁面接近液体温度。由于外壁面不发生线应变，因此筒体上端已不再是自由端，而变成了固定端。由于存在温差，就必然存在温度应力  $\sigma_{\mathrm{t}}$  ，其大小由外向里呈线性递增（如图4)。在外壁面上最小，  $\sigma_{\mathrm{t}} = 0$  在内壁面上最大，  $\sigma_{\mathrm{t}} = \alpha \mathrm{E}\Delta t_{1}$  。

温度应力是使筒体收缩的力，应为负值，所以在外壁面上筒体的应力最大，其大小为：

$$
\sigma_ {2} = \left(\mathrm {D} / 4 \mathrm {t}\right) \mathrm {P}
$$

壁面上的其它两个主应力为：

$$
\begin{array}{l} \sigma_ {1} = \left(\mathrm {D} / 2 \mathrm {t}\right) \mathrm {P} \\ \sigma_ {3} = 0 \\ \end{array}
$$

简体发生破坏性变形的条件是

$$
\mathrm {P} > (2 \mathrm {t} / \mathrm {D}) \sigma_ {\mathrm {s}} \tag {21}
$$

# 2. 内胆底面的受力与变形分析

在不绝热情况下，底面外壁始终保持常温  $t_{\mathrm{a}}$  ，内壁则为液体温度  $\mathfrak{t}_{\mathrm{l}}$  ，与筒体一样，底面也存在附加温度应力，其分布与筒体相同，即由外向里温度应力的大小呈线性递增，在内壁面上温度应力达到最大：

$$
\sigma_ {5} = - \alpha \mathrm {E} \Delta t _ {1} = - \alpha \mathrm {E} (t _ {\mathrm {a}} - t _ {1}) \tag {22}
$$

弯曲应力在内壁面上是使底面受压的力：

$$
\begin{array}{l} \sigma_ {4} = - \frac {\mathrm {M y}}{\mathrm {I}} = \\ - \frac {3 \mathrm {D} ^ {2}}{4 a _ {1} {} ^ {2}} \mathrm {P} + \frac {3 \mathrm {D}}{a _ {1} {} ^ {2}} \rho_ {1} \mathrm {g h t} \times 1 0 ^ {- 6} \tag {23} \\ \end{array}
$$

显然在内壁面上压应力达到最大，假定材料的许用压应力等于  $\sigma_{\mathrm{s}}$  ，则底面发生破坏性变形的条件是：

$$
\left| \sigma_ {4} + \sigma_ {5} \right| > \sigma_ {s}
$$

即

$$
\left| \frac {- 3 D ^ {2}}{4 a _ {1} ^ {2}} P + \frac {3 D}{a _ {1} ^ {2}} \rho_ {1} g h t \times 1 0 ^ {- 6} - \right.
$$

$$
\alpha \mathrm {E} \left(\mathrm {t} _ {\mathrm {a}} - \mathrm {t} _ {1}\right) | > \sigma_ {\mathrm {s}}
$$

$$
\frac {3 \mathrm {D} ^ {2}}{4 \mathrm {a} _ {1} ^ {2}} \mathrm {P} - \frac {3 \mathrm {D}}{\mathrm {a} _ {1} ^ {2}} \rho_ {1} \mathrm {g h t} \times 1 0 ^ {- 6} +
$$

$$
\alpha \mathrm {E} \left(\mathrm {t} _ {\mathrm {a}} - \mathrm {t} _ {1}\right) > \sigma_ {\mathrm {s}}
$$

整理得

$$
\begin{array}{l} P > \frac {4 a _ {1} {} ^ {2}}{3 D ^ {2}} \sigma_ {s} + \frac {4 t}{D} \rho_ {1} g h \times 1 0 ^ {- 6} - \\ \frac {4 \mathrm {a} _ {1} ^ {2}}{3 \mathrm {D} ^ {2}} \alpha \mathrm {E} \left(\mathrm {t} _ {\mathrm {s}} - \mathrm {t} _ {1}\right) \tag {24} \\ \end{array}
$$

如果式（24）计算结果为负值，则说明温度应力超过了材料的许用应力，即假设在

没有拉带固定的情况下，底面仅在温度应力的作用下就会发生破坏性变形。

3. 拉带的受力与变形分析

在不绝热情况下，拉带始终保持常温 $t_{\mathrm{a}}$  ，拉故带不存在温度应力，即  $\sigma_7 = 0$  。

拉带的拉应力为：

$$
\sigma_ {6} = \frac {\pi D ^ {2}}{4 \mathrm {n a} _ {2} \mathrm {b} _ {2}} \mathrm {P} - \frac {\rho_ {1} \mathrm {g h} \pi \mathrm {D t}}{\mathrm {n a} _ {2} \mathrm {b} _ {2}} \times 1 0 ^ {- 6} \tag {25}
$$

拉带发生塑性变形的条件是  $\sigma_{6} > \sigma_{s}$  ，将式（25）代入并整理得

$$
\mathrm {P} > \frac {4 \mathrm {n a} _ {2} \mathrm {b} _ {2}}{\pi \mathrm {D} ^ {2}} \sigma_ {\mathrm {a}} + \frac {4 \rho_ {1} \mathrm {g h t} \times 1 0 ^ {- 6}}{\mathrm {D}} \tag {26}
$$

因此，底面发生破坏性变形的条件，即是式（24）、（26）两式求得的较大者。

# 五、实例计算与分析

# 3. 裸冷变形的原因

计算表明，在不绝热情况下进液，当贮槽内压力达到  $33\mathrm{kPa}$  时，即发生破坏性变形。贮槽的正常控制压力  $< 9.8\mathrm{kPa}$ ，显然在裸冷过程中如果把握不好，是很容易达到这个破坏压力的。导致当时压力失控的原因主要有以下两点：

①1996年3月11日中控室DCS出现故障，空分装置被迫停车，此时因塔内液体较多，加大了贮槽的进液量，使贮槽内气体压力升高。但因计算机画面无显示，中控室不能监控贮槽压力，而现场一次压力表的灵敏度不高，操作工巡检时也没有及时发现较小的压力变化。

② 按设计要求，当内槽压力达到9.8kPa时，DN100的呼吸阀应该开启。但因工程进度要求太急，贮槽进液前未对呼吸阀进行调试。事后检查时发现，呼吸阀装有一防止运输中碰撞的顶杆，阀片被顶死，实际上呼吸阀未起作用，仅靠DN50的放空阀控制内槽压力。

# 六、结语

本文对常压低温容器的破坏性变形进行了分析，推导出了破坏性变形的条件，其分析方法同样适用于压力低温容器。实例计算结果表明，容器在不绝热情况下进液，其破坏压力远小于绝热情况下进液（约为1/3），所以低温容器不宜进行裸冷。常压低温容器的内胆在  $100\%$  焊缝探伤后，再做盛水耐压试验即可，其“冷试”应在绝热情况下进行。

常压低温容器即使在绝热情况下进液，也应十分小心，安全附件必须调试好用，应制定具体措施控制压力，防止容器破坏性变形的发生。

# 一起中心氧房汇流排高压软管燃爆分析

熊奕明

（江西省人民医院设备科，南昌市爱国路92号，330006）

1997年9月14日17时40分，我院中心氧房高压软管发生燃爆，造成双灵充装卡具及氧气瓶嘴  $2\sim 5\mathrm{mm}$  厚铜件烧残，操作工右手烧伤。

# 一、事故经过

1997年9月14日17时40分，操作工按正常操作换氧瓶，换完左边5瓶，打开氧气阀，压力为  $12.5\mathrm{MPa}$  。当开到第3瓶时，突然从氧气瓶嘴与双灵充装卡具处喷出火焰。经检查，氧气瓶嘴、瓶阀烧成残缺，双灵充装卡具横梁残缺2/3，高压软管及嘴断裂，操作工右手烧伤，当时整个氧房烟尘弥雾，余气还在喷出。立即切断氧气瓶气源，并用灭火机扑灭燃火。

# 二、事故调查分析

通过对有关人员的了解，和对燃爆现场高压软管、阀门及氧气瓶进行检查和综合分析，得出：

① 检查控制阀，外表完好无损，但关闭不严，漏气；拆下阀门内部后发现阀芯已经烧坏，通过四氯化碳彻底脱脂，更换阀芯后，使用功能恢复正常。  
② 检查氧气瓶，瓶阀嘴烧成残缺，瓶阀无任何油脂痕迹，规定使用期为1995年11月至1998年11月，钢瓶符合国家使用标准。经检查供氧厂方充装氧气记录为9月11日4班第二组，充装氧气纯度为  $99.8\%$  ，符合使用氧气规范要求，经检验员检验合格出厂。  
③对燃爆断裂及软管进行分析，发现软管为橡胶金属网结构，爆裂两处，一处半裂开，一处完全断裂，内外层均为可燃橡胶。

锯开内层，取下橡胶，用打火机一点即燃，问题就出在此。因为燃烧必须具备可燃物、助燃物和引燃能量三要素，并在同时存在的条件下，才能产生，输氧过程中，氧气始终存在，就具备一个助燃物的条件；充氧橡胶软管内层是橡胶，用明火一点就能立即燃烧，那么就具备可燃物条件；再加上开氧瓶时，高流速的氧气与不断老化的不平整橡胶层摩擦，产生摩擦热和静电，当达到引燃能量时，就会产生燃烧，燃烧的瞬间，压力剧烈上升，就会引起爆炸。

# 三、事故教训及措施

这次燃爆还造成负重  $100\mathrm{kg}$  的弹簧向外冲断，氧瓶向外冲出1米多远，幸亏人站在侧位操作，只烧伤右手。若站在正位操作，氧瓶冲向人体就有可能造成伤亡恶果，望同行引以为戒。但是事后对工人心里压力很大。

综上调查分析，基本可以肯定此次软管燃爆事故，是由于橡胶软管不适宜充氧，加上使用年限较长，是引发事故的根源。我院采取了更换橡胶软管，使用高压回形铜管，确保了医院正常供氧和汇流排安全使用。

通过这次燃爆事故，进一步加深了遵循操作规程和安全制度的重要性，再一次要求做到手、衣服、工具不准带油污；工具供氧房专用；规定操作工一定要戴帆布手套进行操作；开启瓶阀要慢，听到气流声过快，应关小瓶阀，以减轻气流的摩擦冲击，做到安全生产。

（1998年3月13日初稿，

# 低温粉碎的工业应用

盛予非

（茂名石油化工公司设计院，广东茂名市红旗北路1号，525011）

摘要 简介了低温粉碎工艺及其主要设备真空升华干燥装置、低温中粒粉碎机、低温细磨机的主要技术参数，侧重介绍了低温粉碎在生物原料、制药工业、废轮胎、化工及其他工业领域的应用。

主题词：低温粉碎 液氮 应用 脆化温度 粉体 无废料加工

由于富氧冶炼的推广和大型石化企业的消防需求，大型空气分离装置越来越多，这为开发以液氮应用为中心的低温粉碎工艺创造了极为有利条件。

把物质冷却到脆化点温度，在高速冲击和磨擦下破碎成粉体的过程，称为“低温粉碎”。中草药、香料、西药、新鲜蔬菜、水果都可以使用低温粉碎工艺。美、日、英、德诸国均有各种低温粉碎设备成套供应，俄罗斯、乌克兰也在积极试制。

在低温加工中多选用液氮作为制冷剂。因为它的沸点低达  $-196\%$  ，而且化学性质及生物性质均不活泼，对操作人员较为安全。

液氮在低温粉碎的下列环节中发挥作用：对被粉碎物料作预冷；在粉碎过程中携带释放出来的热量；又由于液氮极易气化，使整个过程处于惰性、干燥、无毒、灭菌的环境之中，故不对被加工物料构成污染。

低温粉碎的工艺过程可简述如下：通常先把被磨物料用液氮浸渍或处理，使之迅速达到脆化点温度，再在粉碎机内于液氮掩护下进行一次或多次粉碎。也可在液氮浸渍下利用振动磨或球磨。过程中由于物料不受热和氧的作用，故能最大程度保存物料的色、

香、味、养，还可以根除过程中可能出现的粉尘爆炸问题。全套设备由液氮贮存系统、物料粉碎系统、液氮控制系统、操作控制系统和保冷系统组成。物料粉碎系统是在现有通用机械的基础上，改用耐低温材质制成。主要设备及特性如下：

# (1) 真空升华干燥装置

新鲜蔬菜水果在作低温粉碎之前，需先经真空升华干燥装置脱除水分。由于物料被液氮浸渍，组织内的水分形成  $1\sim 3\mu$  的微晶，既不破坏其生物组织，又能迅速升华脱水干燥。装置用液氮作反升华介质，故构造可比原氟利昂型式的小得多，能耗也大幅度降低。

应当指出，一个完善的低温粉碎系统可以应用于各个方面的低温粉碎。

# 一、生物原料的低温粉碎

新鲜蔬菜、水果、食用植物最宜采用低温粉碎工艺。产品在采收后必须迅速脱水干燥、制粉、包装。全过程温度不宜超过

$30\%$  ，在烘干与粉碎中不宜同氧气相接触。用此工艺可以将苹果、梨、草莓等制成无酒精冲饮料；用糖萝卜、荨麻植株顶梢  $20~\mathrm{cm}$  的嫩枝叶、金盏花、紫葡萄皮、胡萝卜可生产粉末型天然食用色素；用胡椒、大茴香、大蒜、生葱、生姜、花椒等可生产调味品；用鲜食用菌的废弃伞柄可生产汤料粉末，其营养和滋味一点也不亚于鲜菇。

苹果、梨、草莓等原料，经洗净、卫生检验、初碎后置于托盘中，经液氮浸渍后送入真空升华干燥装置脱水和干燥，产品在液氮掩护下进行粗破、细破、碾成粒径小于 $20\mu$ 的粉末。在回收液氮后，脱除可能有的杂质，加上事先磨碎的糖粉和口感辅料粉末，封包成30克的铝箔塑料小包，即为无酒精冲饮料干粉，冲成  $200~\mathrm{ml}$  果汁饮用，可做到原汁原味，营养不变。每杯冲饮料所含维生素C及胡萝卜素约相当于人每日需要量的一半。

由于相当一部分合成食用色素已被证明对人体有致癌作用和其他毒副作用，所以各国对其控制越来越严，某些国家甚至完全禁用。而天然食用色素则倍受青睐，它们大多数具有很高的安全性，有悦人的清香味和丰富的营养价值。过去的加工提取工艺复杂，成本不菲。现在采用低温粉碎技术，直接制成  $20\mu$  的粉体，既实用，又方便，而且无废弃料生成。

例如，我国北方的甜菜，过去只用来制糖。如经低温粉碎到  $20\mu$  以下，它就是一种粉末型天然食用色素和食品添加剂。它具有很好的着色效应，产品具有粉嫩的玫瑰红色。它有很高的吸水性，每克干粉可吸收3 $\sim 4\mathrm{g}$  的水，故在制作糕点时，易于加工，改善了面料的弹性和塑性。

金盏花制成的细粉，也是很好的粉末型天然黄食用色素。荨麻的  $20\mathrm{cm}$  嫩枝叶则是绿色素，南瓜粉及胡萝卜粉是红色素，常用

作面包和通心粉的染色。紫葡萄皮是紫色素原料，它们还可作糖果的维生素添加剂。

# 二、低温粉碎在制药工业中的应用

人们发现提高药物的细度可以影响分子间的作用，从而影响药物的溶解度、安定性。药物磨细了，可以改变制剂的形态。

氨苯磺胺，原粒径为  $150 \sim 400 \mu$  ，经低温粉碎，可加工到  $0.01 \sim 0.1 \mu$  ，在  $20^{\circ} \mathrm{C}$  水中的溶解速度和溶解度比传统产品分别高出363倍和15倍，生物吸收速度增大2倍。原来用药后要等  $2 \sim 3$  小时才能在血液中达到最高浓度，而现在只需  $30 \sim 60$  分钟。

西洋参的有效成分是洋皂甙，口服后，经胃酸、肠道细菌和肝脏酶三大关口的破坏，其生物利用度仅为  $0.5\% \sim 2\%$  。如用低温粉碎技术将其磨至  $1\mu$  左右，压成口含片剂，可通过口腔粘膜直接吸收。

止血敷料经低温粉碎后，可得到细而洁白的粉末，具有无菌吸湿的特点，可制成喷雾剂以气溶胶的形态直接向创伤表面喷敷。

如将人体胎盘低温粉碎到  $3\sim 5\mu$  ，可制成胶囊产品，还可将其溶于生理溶液中，制成肌肉注射针剂。

肉联厂的猪、牛胰脏是提取胰岛素的原料。但因酶的分解，即使激冷至  $-30\%$  也不能制止，必需用液氮浸渍，低温粉碎，可使胰岛素的产率提高  $30\% \sim 40\%$  ，产品质量改善。

应当指出，许多药物在常温下是无法磨碎的，因为它们是含有糖分或油类的胶性体，或是一种韧性体，硬磨时会引起药物氧化，甚至变色、焦化，降低疗效，所含挥发成分则大量损失。如用液氮将其冻至脆化点以下，便易于加工了。大多数草药的脆化点为  $-50 \sim -80^{\circ} \mathrm{C}$ ，杏仁、芦荟为  $-120 \sim -150^{\circ} \mathrm{C}$ ，羚羊角为  $-125 \sim -140^{\circ} \mathrm{C}$ 。每千克中草药制成粉体约需  $1 \mathrm{~L} \sim (4-5) \mathrm{~L}$  液氮，视药材的硬脆度和产品细度的不同而不同。

# 三、废轮胎的低温粉碎

将废轮胎置于  $12\mathrm{m}$  长的管道里，被液氮冷冻至  $-80^{\circ}C$  ，取出压碎，使橡胶与纤维丝、金属丝分开，然后再把橡胶破碎到商品粒度。废胶胶粒在新胶制品中最多可掺用 $30\%$  ，也可做橡胶屋面材料、牲畜圈栏用料、铺装沥青路面及运动场跑道，作为各种金属或非金属表面的防潮、隔热、吸音涂料成分，也可作建筑结构的涂料。所得纤维可作建筑物中的垫子、席子、水泥制件的填料、隔音绝热板。所回收的金属丝，则是冶金工业的珍贵原料。

一座年回收1.2万吨废轮胎的加工厂，可生产粒径  $1\sim 0.2\mathrm{mm}$  的各种胶粒8200t（每吨市价  $200\sim 400$  美元），纤维丝  $2300\mathrm{t}$  金属丝  $1500\mathrm{t}$  。生产总值340万美元/年。

工厂年消耗：电287万  $\mathrm{kWh}$  ，液氮1.23万t，压缩风3080万  $\mathrm{m}^3$  ，工业水2.5万  $\mathrm{m}^3$  基建投资205万美元（其中，设备130万美元)，当均衡生产率为  $30\%$  时的年利润总额为75万美元。

上面所列单耗，为加工废载重汽车轮胎，且胶粒细度为  $0.1\mathrm{mm}$  。如转而处理小汽车轮胎，生产  $5\sim 0.5\mathrm{mm}$  的胶粒，则液氮单耗可降为  $0.6\mathrm{t / t}$  产品以下。

# 四、低温粉碎在化工领域中的应用

由于低温粉碎工艺所生产的粉体可以完全保存其原来的结构性质，因此可将原本无法研细的物料加以研细，赋予其新的用途。

将高压聚乙烯、聚氨酯粉碎至  $0.1\sim 0.3$ $\mathrm{mm}$  的颗粒，可用专用喷雾器将其喷涂于一定温度的钢管或制件的内外表面，可对材料形成抗刮削、抗冲击和抗腐蚀的保护膜。也可将一定温度的制件置于该粉体的沸腾床中成膜。

将二硫化钼经低温粉碎至  $2\mu$  以下组分占  $70\%$  时，它便能长时期悬浮于油中，因此可作为润滑油的抗磨添加剂。

如把水泥熟料经低温破碎至微米级，生产出来的水泥具有极高的早期强度。它只需养生一昼夜便可达到该水泥的标号强度。这对水坝、石油钻井的抢险、机场建筑、港口工程、军事交通线的建造等有重要意义。

# 五、低温粉碎在其它工业领域的用途

利用低温粉碎可以保证对无线电电子工业的许多材料作无杂质的粉磨加工。在生产陶瓷集成微电路及集成块、绝缘材料、焊药、配制能透过射线的陶瓷炉料时，都可用到低温粉碎技术。

无用的人造钻石可在液氮浸泡下磨成至小于  $5\mu$  的粉末，用于光学仪器的研磨与抛光。

对含铂、金、银、射线陶瓷的废旧料作低温粉碎，可很方便地回收其中的贵金属。

在碳化硅的生产中，由于采用低温粉碎技术，纯化了陶瓷原料，使生产微米级陶瓷有了可能。与普通陶瓷相比，其抗弯、抗拉伸和抗压强度都有很大的提高，可以用来制作高热负荷零部件。如内燃机的活塞和汽缸、加热炉吊挂件，等温变形的应变片，坩锅、喷嘴器件、燃气轮机的燃烧室等。制件的最大尺寸可达  $500~\mathrm{mm}$

金属复合陶瓷用膏也用到低温粉碎技术。将金属钼在液氮浸渍下研磨，可制成极细粉体，且粉体表面无氧化膜生成，这是活性钼粉。以它为主成分制成的粘膏可将刚玉陶瓷同金属粘合在一起，使用这种粘膏制成的复合陶瓷其强度和耐热性较原产品提高 $30\%$  ，同时改善了陶瓷的操作性能，适用于电子、真空加速技术领域。新型粘膏的烧结线性收缩率为  $6\% \sim 11\%$  ，平均耐热作用次数为135次，机械强度为  $190\sim 195~\mathrm{kg / m^2}$  。

# 六 结语

综上所述，以利用液氮为中心的低温粉碎技术，是个亦军亦民、亦科亦普的技术，对促进国民经济的发展，开拓新的生产领域具有很重要的意义。希望低温物理技术的同行们同冶金石油化工、食品工业、交通旅游单位共同努力，推动我国低温物理技术的普及和发展。

# 大雾危害和液氮消雾

秦长学

（北京市人工影响天气办公室，北京市海淀区紫竹院路44号，100081）

雾是贴地层水汽凝结产生的一种天气现象。由于雾天能见度极差，使交通堵塞，车祸增加，航空港、海港、高速公路关闭，造成大量旅客滞留，影响到人们的正常工作、生活和各种公务活动。电线雾淞和空气高湿、污染物增加，可使高压输电线路由于绝缘值下降出现污闪事故，造成断电和设备损坏。如1994年2月一次大雾天气，使首都机场被迫关闭，影响380多架次航班的正常运行，造成巨大的经济损失和社会影响。雾天在近地层一般都有较强逆温层存在，很不利污染物的扩散，使大气中污染浓度增加而超标，直接影响城市居民的身体健康。

为了抗御这一自然灾害，国内外进行了多方面的试验研究工作。常用的一些消雾方法：对  $0\%$  以上的暖雾，采用加热法以减少雾的浓度；播撒氯化钠、盐粉、尿素等吸湿剂使雾滴下沉；利用直升机或喷汽发动机搅动气流破坏逆温层结，使上下干湿空气交换，以改变雾的浓度。对于低于  $0\%$  的冷雾，多采用喷撒液态二氮化碳、丙烷、液氮等致冷剂产生冰晶或引入人工冰晶碘化银，使雾中水汽冻结下落以达到消雾的目的。

北京市人工影响天气办公室在多次云室、野外小型试验的基础上，于1994年2月在有关部门支持下，抓住一次大雾过程，进行了用液氮消除冷雾的试验，使沙河机场10平方公里范围能见度由30多米改善到1000多米，效果持续时间两小时。播撒前浓雾弥漫天顶不可辨，播撒后冰晶纷纷下落如降雪，测得降水量  $0.2\mathrm{mm}$  。随后天顶打开，阳光明媚，照物有影，可见一群喜鹊在

空中飞翔。而此时效果区外，仍是大雾笼罩。

1996年12月15日，首都机场受大雾影响能见度只有200米，到8:30已有23架航班延误。北京市人工影响天气办公室在机场航管、气象中心等有关部门配合下，8:30开始用液氮进行作业，在液氮作用下雾中水汽凝结成冰晶，变成小雪花纷纷下落，在树丛草地形成雾淞，地面也出现薄薄一层积雪，能见度逐渐好转，达到500米。由于首都机场有二类盲降设备，9:29第一架飞机起飞，至此机场开放，运营恢复正常。

从1997年12月17日起，北京地区出现大雾天气，首都机场因大雾影响傍晚关闭。18日晨7:00开始，北京市人影办在有关部门协助下，设两道防线进行消雾作业。第一防线用37个系留气球荷载液氮进行低空作业，同时在第二防线用北京普莱克斯实用气体有限公司  $10\mathrm{m}^3$  液氮槽车一辆和计生委  $2\mathrm{m}^3$  液氮槽车一辆，使用人影办研制的专用喷撒设备进行地面流动作业。共用液氮12吨，作业中冰晶雪花立即纷纷降落，地面积雪  $2\mathrm{cm}$ ，但在作业上风方却没有一点积雪。作业前能见度仅100多米，在催化作用下改善到600米。11:45第一架飞机起飞，机场开放。

此三次效果明显的试验引起了有关部门领导和专家的重视，也引起了社会各界的关心，认为人工消雾大有可为。今后应结合航空、交通、旅游活动的实际需要，继续开展人工消雾试验研究，逐步完善消雾实用技术。

# 制氧工问题解答（四则）

1. 问：我厂一台FL-50空分馏塔是邯郸制氧机厂1973年制造的，产量及产品纯度多年来一直稳定。1997年3月加热吹除后，该塔产量突然下降至  $20\mathrm{m}^3 /\mathrm{h}$  ，之后我们清洗了塔板，更换了分子筛，但产量直至12月仍不见提高，原因何在？（辽宁  $\times x$  厂问）

答：此分馏塔产量下降至  $20\mathrm{m}^3 /\mathrm{h}$  历时九月尚未解决，损失不菲！故障是加温引起，你们采取了洗塔、更换分子筛，显然不是对症下药，因为多年稳定的塔，如果需要清洗塔板，必定有渐变过程，纯度逐渐变坏，精馏工况无法调整好。目前，停车前后均无这些迹象，也不符合洗塔的条件，而是突然产量下降，氧纯度仍能达到  $99.4\% \sim 99.5\%$ ；更换分子筛也是“病急乱投医”之举。

目前，该塔稳定工况是：节-1阀、节-2阀、节-3阀的开度分别为  $9 / 12\sim 10 / 12$  周、9/12周、7/12周；高、中、低压力分别为 $3.0\mathrm{MPa}$  、  $0.48\mathrm{MPa}$  、  $0.05\mathrm{MPa}$  ；  $\mathfrak{t}_2$  、  $\mathfrak{t}_3$  分别为  $-135^{\circ}C$  、  $-150^{\circ}C$  ；液氧液面为  $450~\mathrm{mm}$  水柱；氧纯度  $99.4\%$  ；氧产量约  $20\mathrm{m}^3 /\mathrm{h}_{\circ}$  根据上述工况，有几个数据不正常，即节-3阀无法关到位、液氧液面偏低（正常为  $480$ $\sim 520~\mathrm{mm}$  ）、节-1阀前温度偏高（正常为 $-150\sim -155^{\circ}C)$  产量大大不足。很明显，此分馏塔有过热现象，冷量不平衡，甚至内部还有漏点。这是我对该塔故障的初步分析。

至于故障消除办法，应当采取先易后难，由浅而深，辩证施治。因此建议在洗塔、检漏的基础上，分两步进行。

首先要从冷量不平衡着手去解决，让分馏稳定起来，操作要领为：微关节-1阀，使  $\mathfrak{t}_3$  温度下降到  $-155^{\circ}\mathrm{C}$  ，稍开节-3阀，使

主冷液面上升至  $520\mathrm{mm}$  水柱，然后再把节-3阀逐渐关到位，先积液后收阀；同时增加膨胀机产冷量，当液面稳定在  $480~\mathrm{mm}$  水柱时，重新调纯。

倘若产量仍无大幅度提高，第二步请停车检漏，若无外漏，必有内漏。这时大概可以断定是过冷器有液体漏出，漏点在套管式热交热器处的可能性最大。因为停车后加热前，塔的静置自升温时间欠长，加热时残留器内的液体受热气化，接管处由于结构热补偿性能欠佳，焊缝产生热应力，遂被拉漏，这种泄漏因系内漏故不易被发现。

那么为什么我不说是液空过冷器漏呢？因为两只过冷器结构对比之下，液空过冷器设计结构的热补偿较佳，所以不容易被拉漏。故检漏工作应从套管式热交换器开始拆开为宜。另外还要注意到，工作超过20余载的铜管，有冷脆性，一敲即断，故拆卸及复焊工作宜小心进行。

那么为什么过冷器漏液会影响氧气产量呢？因为进入上塔的液体半路泄漏掉了，上塔回流比减少，气氮中氧含量增加导致了氧产量的降低。

注：据来电告知，调整后的工况为：高、中、低压力为  $3.4\mathrm{MPa}$  、0.52 MPa、0.045 MPa；  $\mathfrak{t}_2$  、  $\mathfrak{t}_3$  为-135℃、-155℃；节-1阀、节-2阀、节-3阀的开度为10/12周、1周、3/12周；液氧液位  $>500~\mathrm{mm}$  水柱；气氧纯度为  $99.5\%$  ，氧产量为  $22\sim 24\mathrm{m}^3 /\mathrm{h}_{\circ}$

这说明工况合理，冷量平衡了，但产量仍无大的变化。该厂决定停车检修，终于找到液空套管式过冷器上部漏液，1998年2月15日起已正常运转了。

2. 问：（接上题）从现象观察，液氮节流阀无法关到位（只能关至7/12周，正常时～3/12周），再关小则高压下降（从3.5MPa降至2.5MPa），低压上升（从0.05MPa升至0.08MPa），而且纯度也渐下降，为什么？

答：这种现象的产生，必须用精馏换热的基本理论才能作出圆满解答。

液氮节流阀从7/12周关至3/12周后，如果过冷器没有漏点，那么高压压力下降就成不可思议而无法解释的现象，为什么呢？因为再关小节-3阀，下塔液空量增加，节-2阀必须开大以保持液空液面，那么液空入上塔量增加，液氮进上塔量减少，分馏塔仍处于冷量平衡、物料平衡状态，高压压力是不会下降的。

对过热的塔而言，下塔压力将微升，液氧蒸发加剧，或氧气纯度降低，才会导致上塔压力升高，但高压压力不会下降；只有一旦液氮半路泄漏，因不会被氮气过冷，液氮将随气氮进入第二热交换器冷端，使高压空气在冷端被部分液化，高压压力才会下降。漏量不多时，高压空气压力下降幅度不大（如从  $3.5\mathrm{MPa}$  下降至  $2.5\mathrm{MPa}$ ），随着漏液量的增加，压力还会进一步下降，甚至下降到零。这种现象在液氧面过高时常常碰到，而你们所谈的高压压力下降，在笔者从事制氧40多年的经历中则是第一次遇到，原因是塔内液滴抛向主热交换器冷端造成的。今后这个现象也可为我们判定过冷器漏液的又一个旁证。

3. 问：在双塔变压吸附制氧机运行几小时后，停下一段时间再开起来，往往发现产品氧纯度，怎么也达不到停车以前的状态，不知何故？（北京  $\times x$  读者问）

答：变压吸附制氧是无热再生的一种吸附和脱附过程。分子筛吸附和脱附过程，与分子筛品种、晶体均匀度、质量稳定性、介

质的工作压力、分压、温度、湿度、置换气体容量、置换时间、切换周期、吸附介质与分子筛容量、切换损失等都密切相关，这就是为什么变压吸附制氧要经历数十年的研究实践才能成为商品的原由。

变压吸附制氧机运行几小时后，在条件不变的情况下，停下一段时间再开起来，出现纯度有所下降的原因是什么？首先，我认为分子筛等温吸附，它的饱和吸附有静态吸附和动态吸附区别，而且属于无热再生范畴。记得陕西彩色显像管厂于二十年前进口一套无热再生的吸附器，用以吸附空气中水蒸气和  $\mathrm{CO}_{2}$ ，其性能指标要运转16小时以后，晶格中水含量才降到适合空分使用的要求。国产13X分子筛用于变压吸附净化空气，8小时后产品气中  $\mathrm{CO}_{2}$  可从  $330~\mathrm{ppm}$  降至  $2\mathrm{ppm}$ ，水分从  $2140~\mathrm{ppm}$  降至  $8\mathrm{ppm}$ 。吸附制氧要把氮大比例吸附掉，则更具难点，时间更长，双塔微型制氧机Roomate-Ⅲ型进口说明书规定运行72小时后测试氧气纯度，这说明分子筛晶格中平衡含氧量或含氧量要有一定时间才能适应生产之需要。所问提及停车一段时间后再开起来，氧纯度有所下降，我认为道理相同，只要你再开72小时之后，就会发现纯度不会低下来，会有一个稳定期，而且越开越稳。

4. 问：精馏塔内冷凝蒸发器的热负荷是怎样调节的？

答：精馏塔的冷凝蒸发器是联结上下塔的纽带，它的工作情况直接影响着上下塔的精馏工况。无论是双级精馏塔还是单级精馏塔的冷凝蒸发器，其工作原理是相同的。冷凝侧的温度必须高于蒸发(沸腾)侧的温度，利用两侧的温差，使气体冷凝、液体蒸发。

根据溶液热力学原理，介质的冷疑与蒸发温度仅取决于它的压力与组分。如氧氮二

元混合物的冷凝与蒸发温度，可根据T-I-P-X-Y图查得（见《深冷简报》1970年第3期附图图20)。由图可知：以氮气冷凝而言，当冷凝压力不变，氮气纯度越高，则它的温度越低；当氮气纯度不变，它的压力越高，则温度越高。而液氧蒸发时，当压力不变，液氧纯度越高，它的温度越高；若液氧纯度不变，它的压力越高，温度越高。

例如在  $150\mathrm{m}^3 /\mathrm{h}$  空分设备中，氮气冷凝压力为  $0.5\mathrm{MPa}$  （表），液氧蒸发压力0.05MPa（表），氮气纯度为  $99.9\%$  ，液氧纯度 $99.6\%$  ，则两者的温度分别为  $96.7\mathrm{K}$  及 $94.4\mathrm{K}$  ,其温差为  $2.3\mathrm{K}$  。若氮气压力降到 $0.4\mathrm{MPa}$  （表）、冷凝温度降为  $94.3\mathrm{K}$  ，若蒸发温度不变，则两者的温差为  $-0.1\mathrm{K}$  。大家知道负温差是不能工作的，此时只有把液氧纯度下降，使传热温差为正，才能恢复工作。

实际上，不是氮气的冷凝压力一出现正温差就能工作的，而是一定要高于某个冷凝压力，即大于极限温差的那个压力才能使冷凝蒸发器正常工作。这个极限温差与它的结构有关，因为蒸发侧的液氧沸腾工况，在临界温差以内还没有正常沸腾起来，只有在临界温差以上工作，才能正常建立起来。温差越大，沸腾工况越剧烈。

冷凝蒸发器的工作温差与其冷凝蒸发的传热机理有关。比如是管内还是管外冷凝或是槽内冷凝；是管内还是管外沸腾或是槽内沸腾；是立式还是卧式的形式有关。

空分设备实际应用中的冷凝蒸发器结构形式有三种：即短立管、长立管及板翅式。一般短立管用于小型空分设备，采用管内冷凝、管外沸腾。长立管用于中大型空分设备，传热形式则相反。板翅式各种空分设备

都用，主要用于中大型空分设备，分别在相邻的槽内进行冷凝与蒸发。对它们工作时进行热负荷调节，要根据其不同的工作原理来进行。对于第一种形式，通常采用蒸发侧液面的高低，即用传热面积的大小来调节，这样，工况比较稳定，上下塔压力的变化较小；也有用下塔压力的变化，即利用冷凝温度的变化导致工作温差的变化来调节。

对第二种形式，因管内的液氧（当达到一定液面约  $30\%$  示观液面）已建立翻腾工况，传热面积已全部工作，故不能用蒸发侧的液面来调节，常采用使改变液氮面来调节相应的传热面积。该类型常用于全低压流程上，下塔压力不能随便改变（受空压机压力所限），只能随着入塔空气量的变化而变化。对第三种形式，基本上与第二种形式传热类似，只有板翅式很短的，沸腾的液体才不翻腾（从顶部流出），类似第一种形式传热。

综上所述，若不希望氧氮产品的纯度有所改变，那么根据传热公式  $\mathrm{Q} = \mathrm{KF}\Delta \mathrm{T}$  的关系，一般只能采用蒸发侧或冷凝侧的液面及冷凝压力的变化，从而使传热面积（F）及传热温差（ $\Delta \mathrm{T}$ ）改变来调节热负荷（Q）。传热系数（K）的变化，实际上随工况（工作压力、液面及温度）的改变而变化。但是，因为内在关系很复杂，故可以近似当作不变考虑。

总之，对一定的精馏工况，冷凝蒸发器的热负荷是一定的，也是不需要调节的。当环境条件及用户的需求及塔内工况有变动时，它的热负荷相应要改变。操作工要适应其工况变化，否则会影响精馏工况。
