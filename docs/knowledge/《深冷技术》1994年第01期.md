# 引言

合成燃料(油)工业可能在2000年以前成为美国最大的氧气用户。在最近通过的能源保护法中，政府和国会提出了各种等级的合成燃料发展计划，一些工业公司和公共事业单位正在对几十个工程项目进行不同阶段的研究。根据所用的工艺及原料，国家合成燃料目标在90年代要求为目前美国氧气生产总量约70000 T/d (65000 t/d) 的  $3 \sim 5$  倍。

这些合成燃料的项目，可望具有相当于 $30000\sim 160000\mathrm{m}^3 /\mathrm{d}$  燃料油的生产能力。制

氧装置将需要为这些项目提供  $3000\sim 40000$ $\mathrm{T / d}(2700\sim 36000\mathrm{t / d})$  氧气。

根据所使用汽化器型式的不同，需供的氧气压力为  $0.2\sim 10\mathrm{MPa}$  。预计早期项目趋向于采用  $3.5\sim 6.2\mathrm{MPa}$  的氧气压力，随着更有效的第二第三代工艺流程的进展，将要求更高的压力。氧气纯度通常为  $98.0\sim 99.5\%$  ，有些可使用  $95\%$  的氧气。

以前已对几种大型高纯氧装置流程作过比较，采用可逆式换热器及氧气压缩机的普通全低压流程（图1）被证明是最经济的。

此流程在本文中作为讨论的基础。

![](images/bb7d480a0090e7db71149116b57a5825ea1e731f1dc71c72a6bc9c00f05bacaf.jpg)  
图1 带可逆式板翅式换热器并含氧压机的全低压流程

在常压下制氧所需能耗随着氧气纯度的提高而增加，见图2；氧气纯度的提高使得氧提取率下降。反之，低纯氧需要较低的能耗。然而，因低纯氧中的惰性稀释气增加了合成燃料装置中氧压机和其后设备的投资和运行成本，最佳氧气纯度很少低于  $95\%$  。

![](images/eb10275c4eb7a3b93f870ba108c5141403de35551c9a9bb58dd0b0e86e18f484.jpg)  
图2 氧气纯度与能耗要求的关系

各个合成燃料工程的最佳氧气纯度会因其所用原料煤的性质、汽化器的种类及最终产品的不同而有所不同。一般说来，用于发电的中热值煤气/联合循环工艺流程，选用  $95\%$

$\mathrm{O_2}$  时最为经济。高热值煤气和间接液化流程，选用  $98\sim 99.5\% \mathrm{O_2}$  最为经济。而用作化学原料的直接液化和汽化流程，选用  $99.5\%$ $\mathrm{O_2}$  最经济。

生产  $3000\sim 40000\mathrm{T / d}(2700\sim 36000\mathrm{t / d})$  的制氧厂几乎肯定要由几套机组并联组成。单个机组的生产能力应制成足够大，大到足以适当利用其规模。但这种趋势一定要与多套机组的运行效益相适应。运行的灵活性是随着机组套数数量俱增的。这样，在装置开工不足情况下，可以取得较低的能耗率。对于较小的合成燃料装置，最理想的是使制氧装置的规模，与汽化装置的规模相匹配以简化控制。然而，对于超大型装置来说，装置规模通常取决于其中主要设备的极限尺寸。

为对超大型装置确定一个近似合理的规模，我们发现分别考察四个主要部分是有帮助的：空气压缩、主热交换器、精馏系统及氧气压缩。每一部分都按照：（a）设备极限尺寸；（b）运行效率；（c）投资费用来研究。

# 设备极限尺寸

# A. 空气压缩

空压机能力在6000 T/d (5400 t/d) 等量氧及排气压力在  $550 \mathrm{kPa}$ , 目前被认为属当代技术发展水平*, 已被提供用于高炉供气。而轴流 + 离心式或全轴流式压缩机对这些超大型装置来说, 一般是最经济的。大型轴流式压缩机的运行效率和投资费用将在后面作更详细的讨论。

驱动这些压缩机可以用电动机、蒸汽透平或燃气轮机，这取决于合成燃料工厂其它部机的综合经济效益，和对生产灵活性的要求。一台6000T/d(5400t/d)空压机需要一台  $60\sim 65\mathrm{MW}$  的驱动机，这已接近现在运行的最大电动机规格。建造大电动机本身并没有什么特别的问题，但启动它们却很困难，

因为启动设备开始接近电动机本身的规格和费用。

6000 T/d (5400 t/d) 空压机的最佳转速约为  $3000 \sim 3600 \mathrm{r} / \mathrm{min}$ , 很适合用于为发电机设计的汽轮机。蒸汽可取自合成燃料的废热回收锅炉。必须考虑蒸汽透平驱动的制氧装置启动问题, 一台启动锅炉是需要的。或者, 在多套制氧装置中选择一套用电动机驱动或燃气轮机拖动。

大型的燃气轮机主要用于发电，单机超过75MW。对于生产大量低商业价值副产品燃气的合成液体工艺流程来说，这种驱动类型具有特殊吸引力。当用燃气轮机驱动空压机时，其废热可以回收并有效地拖动氧气压缩机。从一台末端蒸汽透平可得到的功率，通常是燃气轮机输出功率的一半。而当氧气输出压力在  $3.5\sim 7\mathrm{MPa}$  之间时，氧压机所需的功率亦约为空压机的一半。

# B. 主要热交换器

所有进入制氧装置的空气，都要被冷却 $200\sim 220^{\circ}C$  ，即从大气温度降到近于露点的 $-170^{\circ}C.$  。这一冷却任务由铝制板翅式换热器来完成最为经济。最大的板式尺寸因制造厂稍异，典型的一个为断面  $1.4\mathrm{m}^2$  ，长度  $6\mathrm{m}$  可有效冷却  $65\sim 100\mathrm{T / d}(60\sim 90\mathrm{t / d})$  等量氧的空气。以  $2000\mathrm{T / d}(1800\mathrm{t / d})$  制氧装置为例，需并联使用  $20\sim 30$  个单元。

另一种可供选择的技术是使用分子筛吸附器净化空气。最近的工艺及机械设计已使投资费用及运行能耗仅增加  $1\sim 3\%$  。但比可逆板翅式热交换器更易操作和较少缺点。

非常大的制氧装置可能需要几组吸附器并联，但对吸附器尺寸的限制不会影响制氧装置规模的选择。

# C. 精馏系统

全低压制氧装置的精馏系统约能回收进入空气含氧的  $96\%$  ，这种分离装置的塔盘力求达到高精馏效率，低压力损失和相对较小的塔盘间距**。

由单列精馏系统组成的最大空分装置规模取决于可运输的最大精馏塔尺寸。大件运输的限制可能发生在接近制造厂或靠近合成燃料装置场地。初步调查指出，目前美国大多数考虑建合成燃料工程的工厂现场，可通过直径  $5.5\mathrm{m}$  的精馏塔体。这个直径相当于塔的生产能力为  $2500\sim 3000\mathrm{T / d}(2250\sim 2700$ $\mathfrak{t} / \mathfrak{d})$  氧，这是根据目前实践的一个极少或不存在技术风险的合理推断。我们后来注意到，就氧气成本来说，这一容量极限的实际意义其实是很小的。

# D. 氧气压缩

氧压机的尺寸限制，与氧气压缩的安全有关。

目前正在运行的最大氧压机，是在Sasol的  $2500\mathrm{T / d}$  (2250t/d)的离心式压缩机。制造厂预期可以做成超过6000T/d(5400t/d)的氧压机。更为经济的设计，例如利用几个轴流段的计划亦已提出。

目前正在运行的离心式氧压机的最高压力稍低于  $6.9\mathrm{MPa}$  。根据目前的技术发展水平，要求高于此压力，则需用离心/往复组合型压缩机。10 MPa 的离心氧压机的设计正在发展，一些相应的试验正在进行**。

# E. 装置极限尺寸概括

按当前的技术发展水平，制造  $6000\mathrm{T / d}$  (5400t/d)氧的空气压缩机是可能的，并能运输到大多数现场。只要搞好气流均衡，用多单元板翅式换热器组成的主换热器没有限

制。精馏塔可放大到货运最大限制达3000T/d（2700t/d）。根据现在的经验，氧压机的放大设计似无什么问题。

# 生产效率

# 氯气压缩

高压氧气可通过氧气压缩获得，亦可用泵将液氧压成高压，再汽化这种冷的超临界状态的流体得到。虽然本文着重在氧气压缩流程，但在本节中将对这两种技术，都加以评述。

A. 氧气压缩

氧气压缩机已在空分行业中广泛使用了许多年。如前面指出的，氧压机做到6000T/d(5400t/d)规模不成问题，但以往的经验表明，离心氧压机的排气压力限制在6.9MPa以下，要设计较高的压力，必须给予特别的注意。

氧压机要求压至  $7\sim 10\mathrm{MPa}$  时，目前需采用离心/往复增压组合式系统。最近交工投产的休斯顿3000T/d制氧装置，使用了这种组合，将大多数氧气产品压至8.62 MPa。这种方法能保持原有可靠的机械经验，对氧气成本影响最少。一些发展  $10\mathrm{MPa}$  氧气离心式压缩机的规划，已在着手实行。一个全尺寸的新的高压离心式氧压机的实验，已近于被证明是正确的，所以第一座合成燃料工程项目，采用这种压缩机不会冒太大的技术风险。

氧气压缩，和其他一些现代工艺一样，有一些潜在的危险必须加以认识与控制。氧压机着火起因在于：（a）轴承或其它机械磨损，或转子失稳，导致振动过大，以及压缩机内部丧失运转间隙；（b）油穿过密封系统进入机内；（c）外界异物进入机器。

针对工业上的一些事故，近年来使用单位和设备制造厂已做了大量的工作，以便对影响氧压机安全可靠运转的因素有一个较好的了解。特别有价值的是欧洲一个有关管理委员会制定的《氧气透平压缩机实施法规》。这个文件以事例研究为基础，对一些问题进行了综合研究。它提供了保证安全可靠运行的指南，这指南详细包括了设计、结构、操作与维护各方面的内容。

美国空气制品公司(APCI)已经制定了详细全面的购置说明及使用和维护规程，以保证可以有效地监控整个安全规程。

# B. 液氧泵输送

这种方法需要一个高压循环以回收超临界状态氧的冷量。低压循环采用泵送液氧型

式是低效的，还应预计到能耗有所增加。效率低的原因有三：

1. 液氧泵输送的是低温流体，其能量输入必须由低温冷量来补偿。  
2. 必须提供附加冷量，以克服高压氧的焦耳一汤普逊效应。  
3. 由于超临界状态液氧的比热是变化的，因此很难从液氧流体系统中有效地回收冷量。

当产品压力要求增高时，压送能就要增加，焦耳一汤普逊效应亦随之增大，但比热的影响却变得较小。将所有影响加以综合考虑，液氧泵循环与氧气压缩相比，能量消耗在输出压力为7MPa时增加  $8\%$  ，而当10MPa时减为  $5\%$  。

有些制氧装置操作者愿意采用液氧泵系统，以减少生产高压氧的危险性。然而我们的意见是，这种方法很少被证明是合算的，因为氧气压缩设备现在在设计和操作方面都有高度的安全可靠性。只有当压力要求非常高时(10MPa以上)，采用液氧泵循环才是有利的。\*

# 系统的可靠性

从整个合成燃料工厂来看，制氧装置是比较小的一部分，且采用很成熟的技术。因此，既适当又实际的是，氧气供应的工作系数要比后面的汽化及气体处理系统高得多。本文提供了我们的使用维护经验数据，它对合成燃料工厂作全面的可靠性分析将是有用的，特别是对确定机组的数量、设备的冗余要求、备件方针，以及备用系统是有帮助的。

# 结论

合成燃料工程将需要容量为  $40000\mathrm{T / d}$  （36000t/d）的制氧装置。超大型压缩机的经济性对采用大至6000T/d(5400t/d)的设备规模起了很大的促进作用，但是由于货运限制，可能需要使用两个并联的蒸馏系统。大型氧气工厂可采用近于设备极限尺寸的制氧装置，而小型工厂为了取得较好的生产灵活性，可以选用多套较小的制氧装置。

能源成本决定着高压氧的成本。因此，氧气纯度、流程及设备的设计都必须最佳化以减少能耗。过程计算机控制可大大节省能耗。仔细综合规划合成燃料工厂内部能源及制氧装置能源的需求，可以减少能源成本。

安全性、可靠性、投资和能耗都是氧气压缩系统设计的重要因素。6000 T/d(5400 t/d)和7 MPa的离心式氧压机已可获得。7~10 MPa的压力范围可采用离心式+往复式压缩机组合，直到高压离心式氧压机面世。对于安全运行的设计和操作准则，已有很好的了解。

以往制氧装置的操作记录以及液氧备用系统的采用，表明氧气供应应是合成燃料工

厂最可靠的部分之一。超大型制氧装置技术的发展，不应含有太大的技术风险。但是超大型工程现场施工的不稳定性可能含有较大的财政风险。

制氧装置的供应厂商，将提供具有前所未有的安全可靠的设备，以满足合成燃料工业的需要。

# 返流膨胀高纯氮空分设备参数选择

# 郭鸿业

（开封空分设备厂，河南省开封市宋门外公园路，475002）

摘要 本文提出了返流膨胀高纯氮空分设备两个主要参数，液空纯度与液空蒸发压力的确定及制约因素；认为在这种流程中，不管是否采用自清除，一律将回流比取得偏大，是不合适的，将会影响提取率和单位能耗。图1。

主题词：返流膨胀 高纯氮气 空气分离设备 液空纯度 液空蒸发压力 回流比

返流膨胀高纯氮空分设备，是制取高纯 $(99.999\% \mathrm{N}_2)$  、压力  $(0.5\sim 1.0\mathrm{MPa})$  气体氮和气体产量的  $10\%$  左右的压力液氮较为经济与简化的一种设备。该设备在石化、化纤、玻璃工业中得到了广泛应用，在我国已成系列并有几十套产品在运行。

返流膨胀高纯氮空分设备工艺流程比较简单，影响设备性能的工艺参数也较少，主要有两个参数：一是塔釜液体空气（或称废气)纯度，  $x_{L}(L$  表示液体空气），  $\% \mathrm{N}_{2}$  ；二是冷凝蒸发器液空蒸发压力，  $P_{Z}$  。

# 一、液空纯度  $\mathbf{x}_L$  的确定

工艺流程中精馏塔物料平衡如图，操作压力假设为  $0.6\mathrm{MPa}$  （绝压）。

$$
A = L \div N
$$

$$
A \cdot y _ {A} = L \cdot x _ {L} + N \cdot y _ {N}
$$

在  $L$  和  $A$  气液两相量相平衡时  $x_{L} = 60\% \mathrm{N}_{2}$  （含氧  $40\%$  ），  $y_{N} = 99.999\% \mathrm{N}_{2}$

$$
\begin{array}{l} \frac {L}{A} = \frac {y _ {N} - y _ {A}}{y _ {N} - x _ {L}} = \frac {9 9 . 9 9 9 - 7 9 . 1}{9 9 . 9 9 9 - 6 0} \\ = 0. 5 2 \\ \end{array}
$$

这是  $I - I$  截面的最小回流比。这时，氮产量为0.48，氮提取率为  $60\%$  。该数值是在操作压力  $\geqslant 0.6\mathrm{MPa}$  （绝压）时，这种工艺流程的最大提取率，实际上是达不到的。

可用两种方法来确定实际  $x_{L}$  值：

1. 取实际回流比为最小回流比的 1.1 倍

$$
\begin{array}{l} L _ {\text {实}} / A = 1. 1 \times \left(L _ {\min } / A\right) \\ = 1. 1 \times 0. 5 2 = 0. 5 7 2 \\ \end{array}
$$

这时氮产量为0.428，氮气提取率为  $54\%$  ，液空纯度为  $x_{L} = 63.5\% \mathrm{N}_{2}(36.5\% \mathrm{O}_{2})$  。

2. 取不平衡度为  $2\%$  ，即  $x_{L} = 62\% \mathrm{N}_{3}$

此时  $\frac{L}{A} = \frac{y_N - y_A}{y_N - x_L} = \frac{99.999 - 79.1}{99.999 - 62}$

$$
= 0. 5 5
$$

即氮产量为0.45，氮提取率为  $56.9\%$  。利用上述方法可以确定制取一定数量纯氮时所需的实际加工空气量、液空纯度、氮气提取率和回流比。

# 二、冷凝蒸发器液空蒸发压力  $\mathbf{P}_{\mathbb{Z}}$

冷凝蒸发器中液空蒸发压力  $P_{Z}$  是  $\pmb{x}_{L}$  确定

精塔物料平衡图

后就确定了的，也就是说  $P_{Z}$  仅与  $x_{L}$  有关。因为在精馏塔中上升的氮气被冷凝，在操作压力0.6MPa、纯度99.999%条件下，热负荷  $Q_{\text{冷}} = N \cdot r_{N}$  是可以计算出来的。 $r_{N}$  为氮冷

凝潜热，根据热量可得  $Q_{\text{冷}} = Q_{\text{蒸}} = L \cdot r_{Lm}$ 。 $r_{Lm}$  为液空蒸发过程平均潜热，利用上面公式，就可以计算出  $r_{Lm}$ 。

在  $x_{L}, r_{L_{m}}$  一定时，可以确定液空蒸发压力  $P_{Z}$ ，同时可确定蒸发温度  $T_{Z}$  和冷凝蒸发器温差。

这里要特别说明一下，液空蒸发时不是相平衡蒸发，而是等浓度蒸发，所以  $\tau_{Lm}$  是  $x_{L}$  浓度下，饱和气体和饱和液体的焓差，即  $\tau_{Lm} = i''_{xL}$  饱一  $\dot{\iota}_{xL}$  饱。

# 三、液空纯度的制约因素

由冷凝蒸发器中液空蒸发压力的确定可以看出，  $x_{L}$  是一个很重要的参数。液空中含氧量越高，氮产量就越高，提取率也越高，加工空气量越少，能耗越低。但液空中含氧量是受到一定限制的，它只能小于相平衡浓度，即小于  $40\% \mathrm{O}_2$  （在操作压力  $\geqslant 0.6 \mathrm{MPa}$  时）。

另一个因素是膨胀机前压力，也就是膨胀机制冷量。液空中含氧量越高，蒸发压力  $P_{Z}$  越低，膨胀机前压力也就越低。低到一定程度，就无法满足整个装置冷量需求了。

按照上述方法计算出的液空纯度  $x_{L}$  ，再导出的  $P_{Z}$  一般是能满足整个装置冷量需求

的。如还满足不了，就必须调整  $x_{L}$  。调整的方向是使液空中含氮量增加（即  $x_{L}$  增加）、含氧量减少。

还要说明一点，如在整个装置实际运行过程中出现冷量不够，表现出的现象是冷凝蒸发器中液空液面下降，这时液空蒸发就不是等浓度蒸发了，液空中的液氧含量就会增加；蒸发压力  $P_{z}$  跟着下降，膨胀机的进口压力、制冷量也随之下降，形成恶性循环，以致无法操作。这时必须停机排液，找出冷量不足原因，重新积累液体。

限制液空纯度  $x_{L}$  还有一个因素，空分流程如果采用分子筛清除水分和二氧化碳，液空中含氧量高些是可以的。但如果采用切换式换热器自清除，氮气产量要受到限制，一般不能超过加工空气量的  $40\%$  ，提取率为 $50\%$  。

# 四、结语

因此，在这种空分设备工艺流程计算中，不管空分设备是否采取自清除，一律将回流比取得偏大，是不合适的，它会影响提取率和单位能耗。

# 杭氧新型150m³/h空分设备简介

# 杭州制氧机集团公司 杭州制氧机研究所

杭州制氧机集团公司  $150\mathrm{m}^3 /\mathrm{h}$  空分设备新产品（工厂代号11081B），是为代替能耗高、机组落后的老“150”产品而开发研制的，目前已开始批量生产。为使广大用户对此设备有一个全面的了解，我们特作以下介绍。

# 一、新型立式空气压缩机

新“150”空压机（工厂代号为0231M）为立式三级压缩，最高排气压力1.96 MPa（正常运转时  $1.0 \sim 1.4 \mathrm{MPa}$  ；电机功率为185 kW，实际功耗  $160 \sim 165 \mathrm{kW}$  ；占地面积约  $40 \mathrm{~m}^2$  。该空压机运行平稳，可靠性高，相对原“150”配套的空压机(0262B)来讲，易损件少，运行压力低，占地面积约少1/2，且维修方便。由于空气量足，实际氧气产量可达到  $180 \sim 190 \mathrm{~m}^3 / \mathrm{h}$  。

# 二、空气预冷与长周期纯化

由于新增了空气预冷系统，纯化器切换周期由8小时延长为24小时。即使预冷系统不运行，纯化器切换周期也达16小时。由于使用周期延长，操作者劳动强度减少，能耗降低。吸附剂为上海UOP13X分子筛。

# 三、板翅式换热器与多孔管冷凝蒸发器热交换

空分塔各换热器为全铝高效板翅式换热器，热端温差减小，冷损下降。冷凝蒸发器所有换热管皆为多孔管，强化换热，降低了空分塔操作压力，减少能耗。

# 四、透平膨胀机制冷

配套的膨胀机为高效气体轴承透平膨胀机，运转平稳，噪声降低，特别是等熵效率大大提高，从而降低了高压循环压力，减少能耗。

# 方震

杭州宏达新技术开发公司

（杭州天目山路15号，310007）

# 樊光荣

杭州制氧机研究所

（杭州东新路152号，310004）

摘要 介绍了包钢氧气厂引进神钢  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备所配透平膨胀机叶轮的情况，以及将其国产化变型设计的基本方法：采用整体铣制、改进流道成型、基本保证叶轮进出口几何参数不变、保持配合尺寸不变等。投运一次成功，效率比日本原机组高  $1\sim 2\%$  。

# 主题词：透平膨胀机 叶轮 变型设计 国产化 性能 效果

包头钢铁公司氧气厂1973年从日本神钢引进的  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备（简称“万立”），原配23D型透平膨胀机的随机备件已用完，如果向国外订货，价格高且难以及时供货。为此，杭州宏达新技术开发公司、杭州制氧机研究所、杭州制氧机厂和包头钢铁公司共同合作，对转子进行了国产化设计和研制。叶轮是该项工作的核心和难点，本文介绍叶轮变型设计的简要情况和基本思路。

# 一、原配叶轮的情况

1. 为径一轴流半开式长短叶片叶轮, 长短叶片各11片, 叶轮由轮盘和导风轮两部分复合而成, 外径  $\phi 230 \mathrm{~mm}$  。轮盘上有22片径向直叶片, 导风轮上有11片扭曲叶片。扭曲模式为: 在任意半径的圆柱剖面展开图上, 叶片中心线由一大一小两个圆弧组成, 近似于抛物线。  
2. 从叶轮流动特性分析, 该叶轮的优点是采用长短叶片结构, 使得进口和出口均可有较恰当的叶片数, 对于流动是有利的。但该叶轮也有以下缺点:

① 由于采用了轮盘和导风轮分体式结构，流道相接处产生缝隙和错边，对流动不利；同时短叶片的出口边对叶间通道内的流

动也有不利影响。

② 叶片形状差不多一半是径向直叶片平面，一半是扭曲叶片（近似二次抛物线），这样的叶型没有很好的流动规律。以往曾对类似的叶轮进行过三元流动分析计算，其结果也说明了这一点。  
3. 从制造工艺的经济性来分析, 该叶轮的轮盘为锻件铣制而成, 导风轮为精密铸造铸件, 用定位销定位复合, 复合时还要尽量减少错边等。因此, 整个叶轮工艺复杂、制造工作量大、成本高。

# 二、变型设计概况

为了使本变型设计的叶轮配用于膨胀机后，能获得较好的性能，我们采取了以下几方面的措施。

# 1. 采用整体铣制叶轮

为了避免长短叶片结构带来制造上的复杂和困难，我们将短叶片取消，将长叶片增加为15片。由于叶轮制造数量只有2件，如采用精密浇铸则成本太高，因此采用锻件数控铣制的方法。采取这种结构，虽然因未采用长短叶片给流动带来不利影响，但其它方面的有利影响却很多，总的来说是利大于弊，而且制造成本也要低得多。

# 2. 改进流通成型

叶片从进口到出口为连续扭曲的曲面，其基圆柱剖面展开图上叶片凹面和凸面均为高次抛物线；同时修改子午面叶顶和叶根轮廓线，修改流道的判别标准是采用三元流方法，来验算速度沿流道的变化是否平稳，最重要的是不要产生减速流动。为此还要先进行膨胀机的热力验算，以取得叶轮进出口参数作为三元流计算的边界条件。为获得较好的使用效果，热力验算的原始参数是根据现场多年使用的常用值来确定的。

# 3. 基本保证叶轮进出口几何参数不变

对于进口，其叶片高度、外径及堵塞系数，力求保持与原叶轮一致。而对于出口，经过热力验算之后，发现原叶轮的出口角偏小，因此变型设计时略加大出口角，而保持出口通道面积基本不变。

# 4. 保持叶轮对外配合尺寸不变

因为叶顶曲线形状要适应三元流验算的需要，所以进行了修改，另外配置了扩压室。其它对外配合及相关部位均不需要修改。

# 5. 保证叶轮重量及其重量矩不大于原来叶轮

为使转子临界转速不下降，保证运行安

全，我们对叶轮的结构尺寸和材质进行了精心的考虑和精确计算，同时对原叶轮进行了精确的测量。

# 三、使用效果

经过一年多的设计和制造，膨胀机转子（包括扩压室）于1993年2月在包钢氧气厂装机验证，一次成功，性能良好，工作稳定可靠，各项指标均达到原进口机的性能。在常用工况下，效率比日本原机组高  $1\sim 2\%$

# 四、结束语

我们在进口备件国产化的工作中，总的掌握了以下几点：

1. 对研制的配件, 其外形尺寸能达到与原机合理的公差配合和互换, 其工作性能达到原配件的功能标准。

2. 对不能解体或技术已经落后的零部件，则采用我国已经掌握的国内外先进技术进行设计研制，并尽量利用研制单位现有的工装模具。

3. 供需双方紧密合作，是进口备件国产化成功的关键。不仅设计、试制、现场验证的全过程需要互相配合、互相支持，而且研制的成果也应共同享受。

主题词：油田注氮 制注氮空气分离设备 流程 特点 选择

# 一、氮气在油田开发中的应用及机理

随着油田的不断开发，油田利用天然能量(弹性驱动、天然水驱、天然气驱)和人工能量(人工水驱、人工气驱)开采的阶段完成后，将进入提高油田采收率的三次采油阶段。三次采油的方法主要有热力驱、气驱和化学驱等，这些方法可较大幅度地提高驱油效率，达到比较高的最终采收率。在这些方法中，热力驱和气驱是目前世界上应用较广、实际见效的提高采收率方法，其中热力驱宜用于油藏埋藏深度浅和原油比重大的油田。就多数油田来说，气驱应用较多，是国内外采收率研究的发展趋势。

气驱提高采收率方法的发展趋势是用非烃气代替烃类气，由于烃气价格比较昂贵，需求量越来越大，难以满足气驱提高采收率的要求。自60年代、特别是70年代以来，人们逐渐把注意力集中到非烃气上来，其中应用最多、效果最好的是二氧化碳。但由于二氧化碳气源有限，且容易产生腐蚀等问题，所以氮气的应用越来越为人们所重视。

注氮气提高采收率工艺对油田所在地理位置及所处环境无特殊要求，潮湿、干旱的气候，平原、丘陵或山区等不同地形，陆上海上的环境都可采用。适用的油藏也很广

泛，砂岩油藏或灰岩油藏，凝析油藏或较重质的油藏，油藏的大小或深浅都不受限制，对那些开发时间较长已接近衰竭的油藏以及还处于开发早期的油藏也均适用。

氮气的气源比其它气体广泛，价格也便宜得多，因此，注氮提高采收率的方法已得到很大的发展。

经过大量的室内试验和研究，对于大多数注氮气不能混相驱替的油田来说，注氮气提高油田采收率的可能机理有：

1. 在一定条件下，靠油田重力分异作用使注入气进入注入水所无法进入的油湿裂缝中，驱替出其中的残余油。  
2. 靠油田重力分离作用, 排替出被重力捕集在缝洞中的残余油。  
3. 靠注入的氮气溶解于原油，使原油体积膨胀，以其排油作用降低地面分离条件下的地层残余油饱和度。  
4. 改变流体流动方向，驱替裂缝流道中的残余油。  
5. 靠油气重力分异作用, 回采构造上部注入水未能波及到的剩余油。

此外，氮气在单井注氮诱喷采油等方面也得到较为广泛的应用。

# 二、制氮设备用于油田注氮时的特点

1. 制氮设备应配有氮气压缩装置（包括氮气压缩机、注氮管线、注氮井口及计量控制装置），以使所制取的氮气能被注入地下油藏，注入氮气的压力，取决于所注入地层的压力。注氮油藏的压力范围，一般为  $10 \sim 40$  MPa。  
2. 注入氮气的纯度一般都较高（氧气含量为5vpm），以防止油藏内的沥清质沉淀，堵塞油层。但最近的研究表明，只要氮气纯度在  $99\%$  以上，注入的氮气对地层的破坏很小，因而对氮气纯度的要求将有所降低。  
3. 制氮设备的气量，应根据所注氮油田的大小而定，油田大则所需氮气量大，反之则小。  
4. 制氮设备应具有移动性，因为某一油田注入氮气提高采收率后，该油田的进一步开发可能要用其它新的开采技术来代替。因此，制氮设备应能在使用数年后较容易地搬迁到其它油田注氮或挪作它用，这对装置的设计、制造和安装具有更高的技术要求。  
5. 制注氮装置大都需要安装在离所注氮油田较近的地方，离市区较远。因此，制注氮装置的厂址的选择除考虑供电、供水、供暖等因素外，还应特别考虑周围环境的影响，一般应和油气集输站的距离不少于500米，并考虑当地常年刮风的方向以选择空气压缩机的空气过滤器的方位，使油田中的烃

类气尽量减少对吸入空气的污染。

6. 由于油田大都处于环境比较恶劣的地区，因而制注氮装置也应能够适应野外工作，特别是冷却水系统和空气过滤器等，在设计时就应考虑防止昆虫、灰尘、花絮等的侵入。  
7. 由于装置处于野外运行，对装置的运行维护及管理提出了更高的要求，人员配备也具有一定的特殊性。

# 三、110N-TWO型制注氮装置简介及特点

雁翎油田开发后期进入高含水阶段后，综合含水已达  $99\%$  。为了提高该油田的最终采收率，华北石油管理局从1985年开始对雁翎油田进行注氮三次采油试验，并于1988年从英国氧气公司（BOC）引进了一套110N-TWO型制注氮空分设备，1989年10月安装完毕并进行了试车。从该套空分设备的试运情况来看，其主要技术生产指标基本达到了设计要求。现将该套设备的流程、配套机组及特点简述如下。

空气经缓冲罐后通过切换阀组进入可逆式换热器。切换阀组由两只空气碟阀和两只废气碟阀组成，分两组切换操作，正常运行时每12分钟切换一次。空气在可逆式换热器中被冷却和净化后，经单向阀进入气相硅胶吸附器，硅胶吸附器清除可能从可逆式换热器带出的所有的残余二氧化碳和碳氢化合物杂质。

空气经硅胶吸附器后被分为两股气流，一股气流，通过可逆式换热器冷端环流通道后，进入液化器中液化并进入精馏塔；大部分气流，则通过透平膨胀机膨胀制冷后进入精馏塔。

进塔空气经过精馏塔的精馏（约40块塔板），塔顶获得高纯度氮气，其中部分氮气在再沸器中被蒸发的塔釜液流冷凝，成为精馏塔的回流液，大部分氮气则经液化器和可逆式换热器预热后，以  $0.23\mathrm{MPa}$  表压作为产品引出。

塔釜富氧液体经节流后进入再沸器中蒸发成气体，并继续在液化器中得到预热，这股气流作为返流气进入可逆式换热器，冷却正流空气，并带走所冻析出的水分和二氧化碳，最后经切换阀后由废气消音器排放到大气中。

产品氮气进入高压氮气压缩机后，经过四级压缩，压缩至大约  $30\mathrm{MPa}$  压力后，通过2.3公里长的注氮管线、和注氮井口计量及控制装置，被注到大约  $3000\mathrm{m}$  以下的油藏中，以提高油田最终采收率。

# 一、传统的冷开车工艺

按照设计要求，空分设备自启动膨胀机直到产品合格的冷开车阶段大约需36小时。一般来讲，空分设备冷开车依次大致可分为冷却积液（简称积液）和纯度调节（简称调纯）两个阶段。空分设备的积液和调纯的时间较长，它不仅增加装置能耗，而且直接制约其它生产装置的开车和生产。

在空分设备的冷却积液阶段，需启动两台透平膨胀机制冷，使各精馏塔、管线等逐渐冷却，温度均匀下降。当空气温度下降到其液化温度时，精馏塔的下塔和上塔先后出现液体，随着时间的延长，过剩冷量不断积聚增多，气体的液化量也相对增多。当下塔

液位达到工艺要求时，上塔的液体液面缓慢上升，这个过程特别缓慢，我们通常称之为“爬坡”过程。积液阶段的实质，就是冷量积累过程。

空分设备的纯度调节阶段是指在精馏塔的下塔和上塔的液位达到工艺操作要求时，逐渐停一台膨胀机，通过调节和控制产品的产量、回流比等使装置建立起稳定的精馏工况，以保证氧气氮气产品的产量和纯度满足要求。此过程装置的制冷量和冷量损失基本平衡。

# 二、改进的冷开车工艺

# 1. 主冷反充液氧工艺

在冷开车过程中，特别是在冷却积累液体过程中，由于装置的最大制冷量是一定的，而装置的冷量损失又必然存在（只能通过防止泄漏、提高换热器效率和保温等减小冷量损失）。这样，对装置自身工况调节如采用提高压力的方法来增加积累液体速度，其效果亦不明显，并且压力的提高受空压机的影响有一定限度，操作难度大，易发生喘振现象。因此我们在内部冷量无法挖掘的情况下，利用外部冷源(液氧)反充主冷凝蒸发器来加快积液速度。

装置的最直接的外部冷源包括液氧和液

氮，其分别贮存在液体低压罐和中压罐中，考虑精馏塔的正常工况主冷凝器为液氧，更接近正常精馏工况，和下塔顶部冷凝器的气氮换热而汽化，气氧通过塔板筛孔向上完成传质传热。因此，我们利用液氧中压罐升压后向主冷反充液氧的工艺，来减少积液时间。

# 2.下塔提前调纯工艺

调整下塔纯度是调整上塔产品纯度的基

础，空气首先进入下塔并在下塔分离成富氧液空和纯液氮，然后再通过管线导到上塔进一步分离成纯氧和纯氮产品。如果下塔提供的中间产品不合格，上塔就很难生产出纯度和数量都合乎要求的氧氮产品。由于实施了主冷反充液氧技术，使主冷凝蒸发器的蒸发侧的冷量过剩，这样冷凝侧冷凝下来的液氮量较常规开车时多，用于下塔调纯的回流液来源有了充足的保证。通常在主冷液位达到正常时下塔回流阀V301渐开，下塔进入调纯阶段。由于下塔的纯液氮回流液有了保证，这样为下塔提前调纯创造有力的条件。

下塔提前调纯，是指在主冷积液过程中使下塔先达到精馏工况，即边积液边调纯。V301始终处于全开位置，主要是防止因开阀过快造成下塔的液悬（以前开车曾发生此类事故），以便有回流液氮时下塔便开始精馏，使液空和液氮纯度及早达到要求，为上塔调纯提供前提。下塔液空液位调节保持在自动状态，在主冷液位达到  $20\%$  时，逐渐关小V170和V171，下塔回液氮流量加大，于是逐渐建立起稳定的精馏工况，加快调纯进度。见图2。

摘要 介绍三明钢铁厂投产的开封空分设备厂制造的  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备，经大量增加空分塔内管道阀门支架、4M8氧压机三四级填料改造、玻璃钢水冷却塔喷头处理、用氮气代替空气进行膨胀机和空分加温气源等措施，以及经20几项小改小革后，一次开车成功。

# 主题词：大型空气分离设备 管道 阀门 支架 氮气压缩机 填料 玻璃钢水冷却塔 喷头 氮气 膨胀机试车

我厂一套开封空分设备厂制造的KDON-6000/6000-Ⅱ型空分设备，其配套的DH63-8型空压机是沈阳鼓风机厂产品，4M8氧气压缩机则是沈阳气体压缩机厂产品。此套空分设备从1992年5月开始安装，经过单机试车、管道容器吹扫、系统气密性试压保压、空分塔的全面加温和仪控、电控的调试，终于在1993年5月1日顺利出氧，获得一次裸冷、一次开车出氧成功，氧气产量达  $6000\sim$ $6200\mathrm{m}^3 /\mathrm{h}$  ，氧气纯度在  $99.6\%$  以上。

成功来之不易，是各级领导关心支持和各有关方面人员共同努力奋斗的结果。其中兄弟单位的经验和教训，给我们提供宝贵的借鉴，在一定意义上说，是我们成功之母。下面将我们采取的几个主要措施分述如下。

一、大量增加空分塔内的管道阀门支架“6000”空分塔高达33m，塔内管道纵横

交错，有的管道上下高差10多米，横向也有 $4\sim 6m$  。管道上还安装有阀门，正常运行中最低工作温度达  $-170\sim -190^{\circ}C$  ，加上流体流动冲刷振动，冷缩变形应力大，容易引起管道拉裂和阀门变形而无法开关。有个兄弟单位曾一个多月内因管道拉裂被迫三次停产扒珠光砂进行抢修，给该厂的炼钢生产带来严重的影响。

为了避免此种情况在我厂发生，我们采取领导、技术人员和检修工人三结合，在现场根据实际情况增设管架、阀架，共增加管道支架、阀架70多个，增加检修平台10多个。对解决管道冷缩变形拉裂和阀门变形无法开关，发挥了很好的作用，经受了实践考验。

# 二、4M8氯压机三、四级填料的改造

4M8氧压机由于设计制造上的原因，曾在一些单位发生着火燃烧爆炸事故，连Φ70

mm 的不锈钢活塞杆都烧熔断。我们分析其原因主要是活塞杆温度高，加上漏气漏油。为此，我们采取以下几项措施。

1. 采纳应用凌源钢铁厂氧气分厂的专项技术，对三级、四级填料组件进行改造。在凌源钢铁厂帮助指导下，经过改造，使活塞杆温度由原来  $140\sim 160^{\circ}C$  降到  $90^{\circ}C$  以下，解决了活塞杆温度高这一主要矛盾。  
2.增设一套固定式的远红外测温装置，对活塞杆的温度变化，进行跟踪监测和超温报警。  
3. 将填料原有压回水改为无压回水，以保证冷却水量。  
4. 提高氮封装置的氮气压力，对防止油漏到填料组件上和改善活塞杆的冷却效果，都有好处。  
5. 换用 扬州市长江制氧机配件厂生产的不锈钢网状气阀，对提高压缩机工作过程的可靠性，也发挥了很好的作用。

经过采取以上的几项措施，和对压缩机缸腔进行认真细致的清理后，使4M8氧压机自1993年5月份投用以来，运行正常。

# 三、玻璃钢水冷却塔喷头的处理

由于厂家粗制滥造，三个玻璃钢水冷却塔的405只喷头，可以说没有一只完全固定好，造成在单机试车期间，冷却塔填料被冲碎，碎填料流入水池又被水泵吸入，使水泵叶轮、冷却器、喷头完全堵塞，水泵压不出水。在一天一夜的空分裸冷期间，三台水泵轮翻抢修，还满足不了用水需要。

这个问题不解决，“6000”空分设备就无法投产。为此，车间钳工想出办法，用高碳钢自加工几把土丝攻，对405只接头现场重新攻螺纹后，逐只将喷头安装牢固。之后，

更换新填料，并在集水盘下水口装上自加工的过滤筛。这样才完满地解决问题，为空分设备的正常供水，提供了保证。

# 四、用氮气代替空气进行膨胀机试车和空分加温气源

“6000”空分系统的加温和膨胀机的单机试车，原设计要启动DH63空压机，用空气作加温和试车气源。由于加温用气和膨胀机试车用气量不多，若启动DH63大空压机会造成空气大量放空和能量大量浪费。又因电机制动的膨胀机单机试车比较复杂，电控、仪控调试较困难，加上有些电气、仪表质量差，试车中故障多，耽误了试车的进程。为此，我们经过商讨，从实际出发，增设一条 $60\mathrm{m}$  、  $\phi 108\times 4.5\mathrm{mm}$  的氮气管道，把两套 $3200\mathrm{m}^3 /\mathrm{h}$  空分设备的多余氮气，经氮压机加压引入，作为空分系统全面加温和膨胀机单机试车的气源。

此项改造，不仅节约大量能耗，而且气源干燥洁净，还简化流程。这次膨胀机单机试车和空分加温共用11天时间，每天可节约电费、水费约3万元，共可节约30多万元，经济效益极为可观。

除此之外，我们还进行20多项小改小革。如空气滤清器的塑料压栏，改为  $\phi 12\mathrm{mm}$  圆钢做成的栅栏，并增设一道钢丝网；增加一只空压机的油冷却器，大大降低了冷却后的润滑油温；空气冷却塔上部加高  $1\textrm{m}$ ，大大改善除水效果；膨胀机油冷却器，原压力回水改为无压回水；液氧泵原用空气密封，改用氮气密封等。另外，电气、仪控方面，也进行了不少的革新改造，这样确保了“6000”空分设备的顺利投产。

# 空分设备在线调优初探

# 潘立登 杨秋霜 魏寿彭

（北京化工学院自动化系，北京和平街北口北三环东路15号，100029）

摘要 本文对空气深冷分离，这样一个非线性多变量系统，采用数理统计、系统辨识和最优化等方法，进行了在线数学模拟和生产调优。在IBM-PC-XT机上建立了生产操作数据库，由此开发了空分设备的数学模型并将其在线化。该模型能较准确地描述和跟踪空分生产过程，实现了对设备的在线数学模拟。在模拟的基础上，应用优化算法求取了最佳工艺参数，并且根据模拟和优化的结果，制定了在线调优方案，最后实现了开环在线调优，调优结果使氧产量得到了提高。表2参2。

主题词：空分系统 操作优化 离线模型 在线模型 在线优化

# 一、在线调优的意义及其基本思想

生产调优是对系统特性（结构、机理、工参数等）进行研究，并以此为依据，通过整工艺参数来实现系统间的协调联系，发系统最大效用，以更好地达到节能降耗的。

近年来，采用生产调优寻求工业装置的最佳操作参数的方法，因其投资少见效快而越来越受到国内外各大企业的重视。到目前为止，采用的方法大都是离线的。由于一般的生产过程是时变的，已经建立的模型经过一段时间有时就会变得不再准确（由于催化剂老化、分子筛活性降低、老化或工况发生变化等）。因此对于这样的过程，进行离线调优只能维持短期效益。而在线调优则可对模型进行在线修正，工况变化时模型有一定的适应能力和跟踪能力，并在此基础上进行寻优，就可使装置保持在最优操作条件下运行，这是在线调优的基本思想。我们就是基于这种思想，对吉林化学工业公司化肥厂二空分车间4*空分塔进行在线调优的。

# 二、空分流程特点

（1）流程复杂，各个单元之间的关系极

为密切，整个系统中的物料流和能量流相互影响，其中一部分的变化不仅影响自身，而且将影响整个系统。

（2）精馏系统中上塔和下塔之间以冷凝蒸发器相联结，使两塔不仅在物流上互相联结，而且在能量平衡的角度上联立为一体，相互制约，不能单独进行调节。  
（3）对轻组分和重组分的产量和纯度要求都很高。

虽然从指标上看，设备运行情况较好，但由于空分系统在任何企业都是耗能大户，如何进一步挖掘设备潜力，实现系统的稳定最优操作，从而达到节能降耗的目的，这是厂方希望我们协助解决的问题。同时，也是对能否在运行状况较好的装置上进一步调优做一个探索。为此，首先对4*空分设备存在的调优潜力进行分析。

空分设备主要产品是纯氧、纯氮和污气氮。参照上述指标，对装置进行系统分析，发现该装置在以下三个方面存在一定的调优潜力：

（1）找到入上塔膨胀空气量的最佳值。由于全低压空分设备精馏系统正常生产时需要有一部分膨胀空气直接进入上塔，以便合理利用精馏塔的潜力。但膨胀空气入上塔会相应带来上塔精馏段回流比的减少，膨胀量过多则使精馏工况恶化；膨胀量过少则会导致氧提取率的下降。从这个角度出发，膨胀空气量存在一个最佳值。  
（2）进一步提高污气氮纯度。从前面所列各项指标中可看出，实际污气氮纯度已达到  $96.4\%$  左右，比设计值高  $0.3\%$  。虽然产品污气氮纯度已超过设计值，但仍有部分氧气被气氮带出。因此，如果污气氮纯度能进一步提高，氧气产量也可随之增加。但是，在实际精馏塔中，污气氮纯度也不可能无限制地提高，这是因为在上塔，提馏段主要进行氧和氮的分离，而精馏段则是进行氮、氩、氧三种组分的分离。由于氩的沸点介于氧和氮中间，且接近于氧（在  $760\mathrm{mmHg}$  下，氮的沸点是  $77.36\mathrm{K}$ ，氩的沸点是  $87.29\mathrm{K}$ ，氧的沸点是  $90.08\mathrm{K}$ ），所排污气氮中氧和氩的含量存在一定的比例关系，若采取极端措施使污气氮中含氧量降低，则势必有部分氩随同氧一起被洗涤下来，这样，产品氧中的含氩量增加，会使氧纯度降低。因此，如何在不影响氧纯度的前提下，最大限度地提高污气氮纯

度，是使设备在最佳状况下运行的另一重要方面。

（3）保证产品氧纯度达到  $99.6\%$  的前提下卡边操作。前述指标表明，产品氧纯度已达  $99.67\%$  ，超设计值  $0.07\%$  ，而这必定是以少取一定的纯氧为代价，且产品氧纯度达到 $99.6\%$  即可满足要求。因此，如何在保证氧纯度不低于  $99.6\%$  的条件下最高限取出纯氧，是设备保持最优操作的又一重要方面。

# 四、  $4^{*}$  空分设备数学模型的建立及在线调优方法简介

经过对设备进行系统分析，选择与纯氧产量、入上塔膨胀空气量及污氮气纯度最为相关的11个参数进行建模，由于4*空分设备采用YEWPACK机进行控制，自投产以来积累了大量准确丰富的操作数据。数据有一定的波动范围，为建立机理分析基础上的统计模型提供了准确的信息。由于空分设备工况随气温的高低会发生变化，因此我们采集数据时兼顾了自投产以来到1992年2月为止，春夏秋冬四个季节的生产操作数据（剔除异常数据后保留了822组），用非线性逐步回归法建立了以纯氧产量为目标函数的离线数学模型。

通过机理分析，用非线性回归把逐步回归剔除掉的对目标函数影响较大的参数再引入模型，经过多次非线性关联和改变函数形式，最后我们在得到的4*空分设备数学模型结构中，保留了以下十个较为显著的变量：

$P1$  ——空压机出口压力  
P2——下塔下部压力  
$T1$  ——进分子筛的空气温度  
$T2$  ——纯氧气温度  
$\Delta P1$  ——下塔下部阻力  
$\Delta P2$  ——主换热器阻力  
$F1$  ——进塔空气量  
V1——污液氮去上塔的流量  
$A1$  一上塔污氮纯度

$A2$  ——下塔液空纯度

Y——纯氧产量

模型结构为：

$$
\begin{array}{l} Y = F (P 1, P 2, T 1, T 2, \Delta P 1, \\ \Delta P 2, F 1, V 1, A 1, A 2) \\ Y = a _ {0} + a _ {1} \times P 1 + a _ {2} \times P 2 + a _ {3} \\ \because T 2 + a _ {4} \times \Delta P 1 + a _ {5} \times \Delta P 2 \\ + a _ {5} \times F 1 + a _ {7} \times A 1 + a _ {8} \times A 2 \\ + a _ {9} \times P 1 ^ {3} + a _ {1 0} \times \sqrt [ n ]{T 1} + a _ {1 1} \\ \times F 1 ^ {c} + a _ {1 2} / 1 n ^ {3} V 1 + a _ {1 3} / 1 n ^ {2} A 2 \\ \end{array}
$$

其中：

$$
\begin{array}{l} a _ {0} = 1 0 1 8 9 8. 6 0 \quad a _ {7} = 1 2 3 7 1. 7 3 \\ a _ {1} = - 8 5 3 7. 0 8 7 \quad a _ {8} = 6 8 6 6. 5 3 \\ c _ {2} = 1 9 2. 0 4 4 \quad a _ {9} = 9 6. 5 4 5 6 \\ a _ {3} = 1. 3 7 5 0 0 8 e ^ {1 2} a _ {1 0} = - 4 0. 8 3 5 9 \\ a _ {4} = 2 9. 7 5 2 \quad a _ {1 1} = 0. 0 0 0 1 2 7 \\ a _ {6} = 3 6. 9 7 6 \quad a _ {1 2} = 2 9 1. 3 8 6 1 \\ \alpha_ {6} = - 0. 7 7 8 6 \quad a _ {1 3} = 2 5 2 3 2. 0 4 \\ \end{array}
$$

用所建模型对新数据进行拟合，其平均相对误差在  $0.5\%$  以内，我们认为该模型能较为准确地反映出目标函数纯氧产量随各自变量变化趋势。

离线模型建立以后，采用系统辨识技术中的遗忘因子递推法对模型进行仿真在线更新，递推公式为：

$$
\begin{array}{l} \theta (N + 1) = \theta (N) + \gamma (N + 1) \\ \times P (N + 1) \times X (N + 1) \\ \times [ Y (N \div 1) - X (N \div 1) \\ \left. \times \theta (N + 1) \right] \\ \end{array}
$$

$$
\begin{array}{l} P (N + 1) = 1 / \lambda \times [ P (N) \\ - \gamma (N + 1) \times P (N) \\ \times X (N + 1) \times X ^ {t} (N + 1) \\ \times P (N) ] \\ \end{array}
$$

$$
\begin{array}{l} \gamma (N + 1) = 1 / [ 1 \div X ^ {\prime} (N + 1) \\ \times P (N) \times X (N + 1) ] \\ \end{array}
$$

$$
\begin{array}{l} X (N + 1) = \left[ 1 x _ {1} (N + 1) \right. \\ x _ {2} (N + 1) \dots x _ {1 3} (N + 1) ] ^ {\prime} \\ \end{array}
$$

其中  $\lambda = 0.99$ ， $X(N + 1)$  表示第  $N + 1$

次现场采样数据，  $(N + 1)$  表示第  $N + 1$  次采样。

根据上述公式，用新的数据信息不断刷新模型（模型结构不变而部分参数的系数在不断更新），使模型能够及时跟踪实际生产过程的变化，这样就实现了模型在线化。再在此基础上用复合形优化算法进行寻优，最后可找到随过程不断变化的最佳工作点。

以上工作在PC-286(386)微机上完成，用FORTRAN语言进行编程。

（2）提高上塔污气氮纯度方法，是通过调节打入上塔的污液氮流量和打入上塔纯液氮流量（分别由HV2和HV3阀来控制），具体调节方法是在保证下塔浓空纯度不低于  $36\%$  的情况下尽量关小HV2和HV3阀。  
（3）产品氧纯度由取出阀来控制，原则是在保证氧纯度不低于  $99.6\%$  的前提下尽量开大取出阀。

# 主题词：大型空气分离设备 运行 局部改造 效果

# 一、空气分离系统技术改造前的状况

本钢国产  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备是杭州制氧机厂1971年的产品，1974年9月27日设备安装完毕，试运转。虽经多次调试但产品质量、产量、可靠性均达不到设计要求。1978年先后进行了四次技术改造，其中较大项目有更换可逆式换热器，由  $1.8\mathrm{m}$  改换为  $2.4\mathrm{m}$  等几项。虽几经改造，但仍未达到理想状态。在以后数年中，当做“备用机”。产量为8000 $\sim 8500\mathrm{m}^3 /\mathrm{h}$  。1992年  $3\sim 6$  月，我们对该设备又进行了一次局部更新改造，经调试，其氧产量及纯度达到了较理想的水平，氧产量为  $10000\mathrm{m}^3 /\mathrm{h}$  ，纯度为  $99.5\% \mathrm{O}_2$

# 二、空气分离系统局部更新改造的主要项目

# 1. 喷淋冷却塔部分

更换上、下给排出管道，清洗并增添喷淋冷却塔填料拉西环，清洗上水喷头，更换喷淋冷却塔装拉西环的筛网底等。

# 2. 空分切换系统部分

原空分切换系统为两大组，四只空气切换阀，四只污氮切换阀，两只均压阀，都是70年代初制造的强制阀。均压管和空气管径为  $\phi 500\mathrm{mm}$  ，污氮管径为  $\phi 800\mathrm{mm}$  。

该系统强制阀经常发生故障，其密封为“O”型胶圈，经常脱落或不严，造成漏气严重，再加上电磁阀动作可靠性差，经常卡住，使强制阀打不开关不上。造成空气压缩机超压或空分系统压力突然下降，破坏精馏工况，直接影响产品产量和纯度。

此次局部改造将切换系统少量管道和阀门拆除，空气管道改为  $\phi 300\mathrm{mm}$ ，污氮管道改为  $\phi 400\mathrm{mm}$ ，均压管道改为  $\phi 200\mathrm{mm}$ ，原强制阀全部改为切换蝶阀。更新后使用，效果很好，防止了漏气现象，为提高氧产量打下了良好的基础。

另外，在空压机出口至切换系统前加了一个  $\phi 600\mathrm{mm}$  的电动蝶阀，来截断进塔空气，使用起来方便可靠，减少了操作工的劳动强度。

# 3. 空分塔部分

首先，扒出冷箱内珠光砂，拆掉全部箱壁上的阀门，经严格清洗、研磨、试压，合格后安装，减少了因阀门泄漏而出现的跑冷现象。更换了44个新阀门，其中液空、液氧、液氧泵等重要的液体阀门全部更换，上塔、下塔排液阀及部分加热、吹除阀也进行了更换，并把16个长杆排液阀改为短杆冷箱壁外安装

的阀门，便于该阀故障时不用扒砂即可检修。

其次，经过长期运行，觉得液空调节阀口径偏小，原设计为  $\phi 60\mathrm{mm}$  节流阀，后在1978年大修中更换为  $\phi 80\mathrm{mm}$ ，还是偏小。此次更换为  $\phi 100\mathrm{mm}$  的节流阀，效果很好，能使下塔液空全部送入上塔，稳定上塔精馏工况，为提高氧产量起到了关键作用。

另外，还发现塔内计器管腐蚀严重，此次做了局部更换。

最后，在拆除空分塔液空过滤器内壳时，检查发现，两台内壳均有大面积凹凸现象。经分析认为是液空吸附器在正常运行或解冻加热过程中，有硅胶粉碎和灰尘积聚，随后积存在液空过滤器内。此次改造在液空过滤器上加了吹除阀。当在解冻加热时，将此阀打开，把硅胶粉末和灰尘排掉；在预冷液空吸附器时也将此阀打开，防止气流冲击。经运行考察，效果很好。

# 4. 加温吹除系统部分

因该设备安装20余年，加温管道腐蚀严重。此次旧管道全部拆除，重新制作更新。

# 5. 其它部分

此次局部更新改造，液氧泵系统全部拆换，更换为杭州制氧机厂仿林德公司技术生产的液氧泵一台，运行效果较好。原仪表空压机损坏，此次更换为一台沈阳气体压缩机厂产的仪表空压机。

# 三、局部更新改造后的效果

为使设备发挥更大潜力，我们进行了细致的调试，根据纯液氮的纯度高于  $99.99\%$  及其它参数，将纯液氮节流阀由  $40\%$  开度开

至  $60\%$  ，污液氮节流阀由  $50\%$  开至  $80\%$  ，下塔纯液氮回流阀由  $10\%$  开至  $25\%$  。这样，空气量由  $56500 \mathrm{~m}^{3} / \mathrm{h}$  加至  $58500 \mathrm{~m}^{3} / \mathrm{h}$  左右，下塔压力稳定在  $0.52 \mathrm{MPa}$ ，上塔压力为  $0.045 \mathrm{MPa}$ ，基本符合设计参数。氧产量由  $9375 \mathrm{~m}^{3} / \mathrm{h}$  升至  $10000 \mathrm{~m}^{3} / \mathrm{h}$ ，纯度  $99.5\% \mathrm{O}_{2}$  。氧气透平压缩机排出压力也达  $0.5 \mathrm{MPa}$  左右，基本达到设计值。

英国威斯康星州的Ansul消防公司，计划在美国北部推销新型的INERGEN灭火剂，用它来取代目前使用的Halon1301系统和高压二氧化碳系统。

# 二、改进方法

# 1. 方法的出发点

为加快解冻速度，缩短解冻时间，在长期摸索和经验的基础上，我们在1993年装置大检修中进行了大胆尝试，并取得极好的效果。其基本做法归纳起来就是增加入冷箱空气量，但此法完全不同于日立设计方法。

# 2. 操作程序

在下达停车命令后，首先停止膨胀机的运行而其它程序一概不变，这一作法的目的是截断冷源，但仍维持可逆式换热器自清除的正常进行。然后打开上塔及有关排放阀，对塔内液体进行排放，保证设备不超压。当塔内液体排放完毕之后，再对各吸附器内液体进行排放。在排液过程中最重要最关键的一点，就是控制可逆式换热器的中部温度，应密切注视并及时调整，保证其自清除的正常进行。

此法操作的要点就是在确保自清除的前提下保证装置解冻。若可逆式换热器自清除不能保证，则水分等杂质入塔，随时可能造成管道、阀门等的堵塞，而给解冻带来更大困难。可逆式换热器冷端温度升到  $-115^{\circ}\mathrm{C}$  及 $-60^{\circ}\mathrm{C}$  时应特别注意，此两温度分别是保证二氧化碳及水分自清除的最高温度。在解冻后期为防止水分入塔堵塞管道及阀门，应缩短切换时间，保证自清除的顺利进行。等冷箱内各温度达到常温吹除的温度时，再按原来流程进行设备的加热。

# 3. 使用效果

我们在解冻过程中采用了上述方法，由于操作得当取得了极好的效果，使解冻时间大大缩短，且管道、设备未发生堵、冻现象。实践证明此方法是极为可取的。

# 4. 理论分析

进入冷箱的空气，必须保证不带有水分及二氧化碳等杂质，以防止其在低温环境中凝固并堵塞管道及阀门，进而破坏装置的运行，因此必须保证可逆式换热器的自清除。但是冷源已被切断，可逆式换热器冷端温度逐渐升高，保证自清除的条件越来越难以满足。所以必须采取措施，以保证自清除。这就要将正返流气体的传热温差，控制在自清除允许的最大温差范围内。图2是可逆式换热器实际温差曲线。

综上所述，在解冻过程中要使大量原料空气入塔，必须保证可逆式换热器自清除。只有在此前提下才能保证有大量干燥空气，才会加快解冻速度。

# $300 \mathrm{~m}^{3} / \mathrm{h}$  空分设备几项工艺改进

冯湘波

（株洲化工厂氯分厂，湖南省株洲市清水塘，412005）

我厂空分车间新建  $300\mathrm{m}^3 /\mathrm{h}$  空分设备，在安装过程中我们对冷却水系统、纯化器、透平膨胀机和再生气加热电炉等进行了工艺改进。改进方案实施后，取得了令人满意的效果，现小结如下。

# 一、加大I、Ⅱ、Ⅲ段冷却器冷却水入口压力

我厂空分车间新建  $300\mathrm{m}^3 /\mathrm{h}$  空分设备的冷却水为循环冷却水，凉水塔下水总管离设备冷却水入口高度差不到  $10\mathrm{m}$  ，除去管道阻力，冷却水进入设备前的压力不到0.08MPa，I、Ⅱ、Ⅲ段冷却器是邯郸制氧机厂的成套设备，为列管式换热器，其进出水管口通径为DN32。根据计算，在最高气温的7～8月份，冷却水流量不应少于  $40\mathrm{m}^3 /\mathrm{h}$  （凉水塔上下水温差  $3\%$  )，冷却器过小的进出口管径无法满足该流量，所以冷却器出口空气温度偏高，油水夹带十分严重。考虑到另外两套  $150\mathrm{m}^3 /\mathrm{h}$  空分设备的冷却器为敞口浸没式盘管冷却器，循环冷却水不可能先进设备再进凉水塔。为此，我们提出了另外一种改进方案。

在凉水塔下水总管另接一支管，在该支管上安装一台80SG30型管道泵，专供300 $\mathrm{m^3 / h}$  空分设备冷却用水，即先满足该装置冷却用水，其它装置冷却水量不足时，可补充少量工业水。投入使用时，只要控制管道中水压在  $0.3\mathrm{MPa}$  左右，就可满足I、Ⅱ、Ⅲ段冷却水用量，使其空气出口温度降下来。

# 二、降低空气温度，提高再生温度，延长纯化器使用时间

纯化器使用时间过短，一方面是因为I、Ⅱ、Ⅲ段冷却器出口空气温度偏高，空气严重夹带油和水分；另一方面是因为再生温度过低。我厂原规定分子筛的再生温度为  $120\sim$ $150^{\circ}\mathrm{C}$  ，通过静态实验，当再生温度提高到 $180^{\circ}\mathrm{C}$  时，其吸附量增加2倍。第一方面的问题在前述措施中得到了解决，第二方面的问题由加大电炉功率，提高再生气流温度得到解决。

# 三、增加电炉炉心温度控制点，防止电炉烧坏

原先电炉只有一个再生气出口温度控制点，在电炉投入使用时，若出口阀门关闭，电炉炉心温度就会急剧上升而烧坏电阻丝。当增加炉心温度控制点后，一旦炉心温度超过  $650^{\circ}\mathrm{C}$ ，电源即自动切断。

# 四、改油轴承透平膨胀机为空气轴承透平膨胀机，清除事故隐患

油润滑轴承透平膨胀机的最大缺陷是当操作不慎时，润滑油易入分馏系统，带来事故隐患。空气轴承透平膨胀机是近几年来发展起来的一种新型透平膨胀机，我厂另外两套  $150\mathrm{m}^3 /\mathrm{h}$  空分设备从1986年开始改用空气轴承透平膨胀机至今，使用效果良好，根除了上述事故隐患。从空分制氧行业发展趋势来看，改用空气轴承透平膨胀机势在必行。

# 小结

目前，我空分车间已全部完成上述工艺改造，从使用的情况来看，改造是成功的，以最小的投资获得了预期的效果。

（1993年12月20日）

# 主题词：医用氧气 药品管理 药典 国标 执行

医用氧气开发生产已逐渐成为全国空分行业的热门话题，并已为医疗卫生管理部门和药政管理部门所重视与接受。青海、贵州、浙江、广西、云南等省，沈阳、抚顺、蚌埠、镇江、无锡、徐州等市卫生主管部门先后发文，停止在医院内使用工业氧气，推广应用医用氧气。

在医用氧气开发生产工作中，目前还存在一些问题。这些问题不解决，将会影响医用氧气的推广，甚至回到老路上去。现就几个问题提出一些解决方法和建议。

# 一、医用氯气应纳入药品生产管理范畴

医用氧气主要用于抢救病员和缺氧病人治疗，属于药品。因而医用氧气生产应纳入药品生产管理轨道，应符合和达到国家卫生药政部门关于药品生产管理的各项要求。

在过去的几十年中，卫生药政部门一直未对医院用氧气进行管理。经过多方面的工作，部分省、市卫生药政主管部门已正式发文，对医院用氧气按药品要求进行管理，把医用氧气生产企业列入药品生产企业中，要求按《药品生产企业验收细则》进行验收，验收合格后发给药品生产企业合格证和许可证，审批药品批准文号。药品检验所也按要求定期抽查、检验医用氧气质量。这对医用氧气生产企业的管理是一个很大的促进，对

提高和保证医用氧气质量有着积极的作用。

对于氧气生产厂家来说，要符合药品生产管理的有关规定，就必须从管理人员的设置和配备、管理规章制度、生产环境条件、生产批量和批号、质量管理以及生产原始记录和台帐等各方面，向药品生产管理转变。特别是生产企业的各级管理干部，都要认真学习药品管理法规、药品质量管理规范，把管理思想、管理方法转变到药品生产和管理上来。

但也有一些气体生产企业，未经过设备工艺改造，医用氧气生产灌装与工业氧气生产灌装未分开，没有医用氧气专用灌装车间和专用生产管线、专用瓶库；不具备医用氧气检测分析仪器、仪表；氧气瓶未按要求专管专用、专用标志。也就是未按药品生产企业管理要求办，而只是在原来工业用氧气生产设备和工艺的基础上，用单纯提高质量指标或等级（如达到工业I类无水氧气）的工业氧气作为医用氧气。这是一种极不负责和极不道德的做法。这种工业氧气估计也能达到《药典》中氧的指标，甚至达到国标医用氧气的技术指标和技术要求，但经不起推敲和过细检查。

我们希望各地区卫生、药政管理部门和行政管理部门，做好医用氧气生产厂家的布

点工作，要优先选择安全条件好、厂区环境状况好、企业管理基础较好、设备状况较佳的单位开发生产，并重视按药品生产企业管理要求进行审查验收，保证工业氧气不再进入医疗卫生行业。

# 2. 国家标准

在此国标颁布以前，医院所用的氧气，均是各氧气生产厂按《工业用气态氧》标准进行生产的工业氧气。工业氧气先后有过两个标准：

# 3. 医疗用氧气标准的执行

虽然从1953年开始我国《药典》中就收有“氧”的标准，但医疗卫生部门、药品检验部门未按《药典》，对医疗用氧气进行管理和检验，一直将工业用气态氧用于医疗卫生、用

于抢救病员。

自1988年医用氧气国家标准颁发后，经部分地区氧气生产单位与卫生部门、药政管理部门宣传、解释，医疗用氧气的生产、检验、管理工作逐步得到重视，医用氧气在全国各地迅速发展，医疗卫生行业使用医用氧气取代工业氧气已成为必然趋势。

医用氧气标准在具体执行中也碰到了一些问题。国标是产品生产企业、经济主管部门和产品质量检验监督部门在检验产品质量过程中执行的标准，而《药典》是药品生产企业、卫生药政管理部门和药品检验部门在检验药品质量过程中执行的标准。国标和《药典》两种标准，都是国家级的标准，在医用氧气生产、管理、质量检验工作中到底应执行哪一种标准？

我们认为：这两种标准都是医疗用氧气的国家标准，在生产中和产品(药品)质量管理中都必须严格按标准执行才能保证医用氧气质量。具体讲，“国标”医用氧气中的技术要求较中国《药典》中氧的质量要求高。医用氧气要达到国标技术要求，要做一系列设备改造工作，气瓶必须预处理，管理制度和方法要相应加强，达到了国标技术要求，就能保证医用氧气符合中国药典中氧的技术要求。反过来，医用氧气达到和符合中国《药典》中氧的要求，不一定能符合国标医用氧气技术要求，中国《药典》中氧含量和水分两项指标，应该说低于工业用气态氧指标。

因此，我们对医疗用氧气标准执行提出如下建议。

1. 各气体生产企业在开发生产医疗用氧气产品(药品)时，一定要同时按中国《药典》和“国标”两个标准中的要求组织生产和检验，才能保证医疗用氧气的质量。  
2.卫生药政管理部门、药品检验部门在审批、管理、监督、检验医疗用氧气生产工作中，要同时根据两个标准（《药典》、“国

标”)去要求，去审查办理，确确实实提高医疗用氧气质量，保证和促进医疗卫生事业的发展。

# 三、对医疗用氯气标准修改建议

中国《药典》和“国标”中对医用氧气的技术指标和检测方法的要求，在执行过程中还有一些需完善之处，所以，我们对两种标准也提出一些修改建议。

1. 《药典》中氧与“国标”医用氧气中的检查项目和技术指标有相同之处，特别是我国1953年版药典“氧”和国标GB8982—88《医用氧气》的技术指标，检查项目有多项是一致的，这可能与参考国际标准来源（前苏联)有很大关系。但总的来说，1990年版《药典》中氧的标准较低。建议国家卫生部、化工部、国家标准总局协调，统一制订、颁布一个医用氧气标准，以利于在组织生产、审查管理、质量监督、产品检测中执行。

2. 国标GB8982—88中3.1条“氧压机内的活塞密封材料，经有关部门检验应无毒性。”中国《药典》中有“卤素”的检查项目。这一条内容较含糊，在执行中也比较难做到，由谁来执行（有关部门是谁）？目前，我国制造的氧压机活塞密封材料绝大多数为填充聚四氟乙烯，大家都有一个共同观点：聚四氟乙烯是毒性材料，尤其在高温下会分解（据试验在  $200^{\circ}\mathrm{C}$  以上就会分解），产生出氟的气体化合物（属于卤素类），是剧毒气体，前苏联医用氧气国标也禁止使用四氟材料充装医用氧气。现在，采用无毒性活塞密封材料取代填充聚四氟乙烯还需要试验研究，或者规定氧压机采用高压钢纸碗和铬革皮碗（70年代采用过）作为活塞密封件，这两种材料是无毒性的材料。

如果继续用填充聚四氟乙烯材料作为活塞密封材料，在生产过程中，因摩擦、高温气化分解出一些有害物质，但由于分解速度慢、分解量较小，采用《药典》和国标GB8986

—88中的检测方法是很难测定出来的，而且一般的化工气体检测部门或药品检测部门，也不具备检测这类有害气体成分的条件。

因而我们建议标准中应具体规定活塞密封材料的检测方法和控制指标，或规定氧气中有害物质元素的检测方法和指标，以便于具体执行。

3. 国标医用氧气标准中，规定“适用于深冷法分离空气而制得的氧气”，是不是唯一的生产方法？目前，一些化学法制得的氧气也用于病员急救和人体呼吸，如市场上有“氧立得”、“瞬得益”等便携式小化学氧盒和氧包，是不是符合医疗用氧标准？还有采用变压吸附法制取的氧气，可不可用于医疗保健？我们建议应在具体考察现有几种生产方法所生产的氧气质量指标能否适合人体呼吸的基础上，在医用氧气标准中作出制取方法的规定。

4. 医疗用氧气水分含量控制，在中国《药典》中无具体要求，“国标”规定水分含量  $\leqslant 0.07\mathrm{g / m^3}$  （相当于露点  $-43\sim -45^{\circ}C$ ）。我们认为这一指标偏高。医疗用氧中只要保证无游离水分即可，一是防止游离水分在钢瓶内锈蚀，产生气味污染氧气；二是降低对钢瓶的腐蚀，保证钢瓶安全使用。

# 管内壁喷塑防腐技术的应用

李思福

（太原平板玻璃厂保护气体车间，山西太原市和平南路光华街8号，030024）

摘要 简述了管道内壁生锈对无油润滑空压机的危害，介绍了喷塑技术在管内壁涂层防腐的应用情况，对无油空压机管内壁防腐新途径进行了探索。

# 主题词：无油空气压缩机 管内壁喷塑 流平固化 防腐

无油润滑空压机，由于具有特殊的优点——压缩后的气体不带油，在制氧行业得到了广泛应用。它延长了分子筛的使用寿命，增加了空分机组的安全性。但也带来了不少问题：各级管道、冷却器、水分离器锈蚀，使用寿命缩短。更为严重的是生成的铁锈，随空气进入空压机气阀和气缸内，破坏气阀密封面，拉伤气缸镜面，破坏活塞环与气缸间的自润滑膜，将活塞环卡死在活塞环座里，使活塞环失去密封作用，从而导致导向环、活塞环寿命大幅度下降，漏气增大，检修周期缩短，从而严重影响空分机组的正常运行。

分析生锈的原因有两个：一是高温、高压氧化；二是由冷凝水形成的电化学腐蚀。从实际情况来看，主要还是电化学腐蚀。为了不使管内壁直接与冷凝水接触，一开始我们采用了在空压机管道内壁刷漆的办法，但使用不久就全部起皮脱落。1992年5月，我们采用了静电喷塑技术，将空压机管道内壁进行了喷塑处理，其效果比刷漆有很大的提高。现简单介绍如下。

# 一、静电喷塑工艺

静电喷塑技术是近年来发展较快的新型防腐技术，它是利用高压静电形成电场，喷枪接高压正极，被喷涂工件接地成负极，构成回路。塑料粉末带正电荷由喷枪喷出，按

正负静电相吸原理，工件吸附着塑料粉末。由于静电吸附，涂层较均匀。

我们选择的涂料是上海生产的热固性环氧粉末防腐涂料，它是管道防腐的新材料，其金属附着力好。涂层具有优异的耐化学性能、良好的机械强度、较高的电绝缘性及较好的韧性。

静电喷塑工艺过程

工件  $\rightarrow$  喷砂或除油酸洗  $\rightarrow$  中和  $\rightarrow$  磷化  $\rightarrow$  静电喷塑  $\rightarrow$  流平固化  $\rightarrow$  产品检验

工件前处理 就是对工件进行除油除锈磷化，将管内壁的铁锈等杂质清除干净，形成多孔性磷化膜，以利于涂层和金属表面牢固结合。

喷塑 就是将磷化后的管道，用静电喷涂设备的喷枪伸入管内，均匀地在管内壁喷上一层  $0.05 \sim 0.08 \mathrm{~mm}$  左右的涂层。

流平固化 在热风烤箱内进行，将管道加热到一定温度，保持  $30\sim 50$  分钟，然后拿出，冷却到室温即完成。

注意事项

1. 为了使喷枪能伸进管道, 要用增加法兰的办法, 将管道加工成直线或只有一个弯曲的形状。  
2. 根据热风烤箱的大小, 来定加工管道的长度。否则, 管道将不能放到热风烤箱内流平固化。

3. 所增加的法兰，一定要预先焊好，如果喷塑后再焊，会损坏涂层。

# 二、效果和费用

我厂是1992年5月对2号空压机各级管道进行喷塑的，到现在已使用了一年零四个月。在喷塑后一年内效果很好，每次拆开气缸都很干净，没有发现铁锈；活塞环使用寿命由原来的一个月左右，延长到  $3\sim 4$  个月（指三、四级）；产气量也有所增加。当我们检查各级管道内壁涂层情况时，发现绝大部分涂层良好，只有个别地方出现小鼓泡。但一年以后，发现四级进气管涂层开始大面积脱落，其它管道也有少量脱落。

究其原因，可能有三个方面：一是当时除锈不彻底，因为我们是在旧管道上喷涂的，管内壁锈层很难彻底除尽，尤其是管径较小的四级管道  $(\phi 89\mathrm{mm})$  ，又细又长，不易

除尽；二是由于管径较小，喷枪伸到里面进行喷塑时，涂层均匀性不好掌握，有些地方涂层薄、强度低、易崩裂；三是涂料本身性能还需进一步提高。

我厂管内壁喷塑是由山西省榆次市喷塑厂进行的，当时价格为20元/米²。一台2D8空压机一级到四级管道内壁总面积大约为25米²，再加上其它费用，共投资约1000元。

# 三、展望

从上可以看出，喷塑技术应用在管道内壁涂层防腐，投资少，短期效果良好，据了解一般防锈寿命可达  $5\sim 8$  年。但我们的喷塑，长期使用效果还不尽人意，管道内表面除锈手段、喷塑工艺及涂料性能有待进一步提高。随着科学技术的不断进步，我们相信这一技术不久将有突破性进展。

# 一、运行维护中应注意的几个问题

# 1.低压大流量逆向反吹

薄壳结构抗拉不抗压。在正常操作中，压缩空气以设计比速由里向外穿过滤芯，各层骨架仅承受微弱压差产生的拉应力，即使出现短时的低压大流量正向冲击，骨架仍有能力承受高速气流产生的额外拉应力。但反向流动则不然，特别是在额外的压应力下，骨架会发生失稳而变形，导致密封失效。

# 2. 滤芯更换周期

除滤芯外，高效除油器几乎没有什么易损件，国外同类型产品一般规定其滤芯的更换周期为  $6\sim 12$  个月(需设预过滤器)。我厂产品因采用中速滤材，过滤面积比国外同规格产品大  $3\sim 5$  倍，容尘量也相应提高，所以将滤芯的预期寿命规定为  $8000\sim 12000$  小时，相当于全日制的一年至一年半。超过规定时间(包括非运行时间如停车检修等)对滤芯阻力和除油效率均有较大影响，阻力增加导致能耗上升，效率降低则危及分子筛乃至分馏塔的安全运行。

引起除油效率降低的原因，可能有以下几种：

# （1）材料老化

投入运行的滤芯，长期处于油水浸润状态，滤材中的各种树脂、增强剂以及滤芯两端的粘合剂、橡胶垫逐渐老化，镀锌处理的骨架、封头逐渐发生锈蚀等，都能导致滤芯性能下降。

# (2）阻力过大

超细玻璃纤维滤纸，属于固定的无规则孔过滤介质，当固体尘埃过度堵塞纤维空隙

时，滤芯两侧阻力损失上升，气流将在高压差下夺路而行，使原无序但固定的纤维构架松动，即发生“介质迁移”现象，造成部分孔隙增大甚至短路。

# （3）大气中含有腐蚀介质

若氧气站位于污染严重的大气环境下，则滤芯寿命必然会缩短。

# （4）没有按时排放油水

本产品有三处排污点，均设有一定量的储水容积，每班（8小时）排放两次即可。但也有用户数天不排放（多发生在动力型除油器），使滤芯处于油水浸泡中，不仅影响气体质量，也缩短了滤芯寿命。

从目前我厂800余台除油器（包括动力型）的使用情况看，大多数用户的使用时间超过一年半而未更换滤芯，有的甚至长达三年以上。根据我们的经验，滤芯的更换周期应控制在一年至一年半内，超过此期限，由于阻力上升引起能耗增加，过滤效率下降，将会对系统的安全性、产品质量以及运行费用产生不利影响。至于两年以上，则可能会在滤芯失效状态下工作。

# 二、技术改进与发展方向

国外超滤技术在80年代已得到广泛应用，如超滤膜和反渗透膜的孔径范围已相互衔接，并有较大区域的重叠，在过滤除油方面已达  $0.01\mathrm{mg / m^3}$  。若流程中有吸附剂、催化剂存在，还要在其前加一级活性炭吸附过滤器，使含油量降至  $0.003\mathrm{mg / m^3}$ ，几乎达到百分之百无油。此外国外几家著名公司均采用了高速滤材，使其产品小巧到能直接悬挂在管线上。

今后国内的发展方向应以高速、超高效为目标，采取引进技术与自主开发相结合，尽快缩短与国际先进水平的差距，为各行业用户提供更多、更好的产品。

我们气体车间空分站，现有两台沈阳产5L-16/50-Ⅲ型无油润滑空气压缩机，该机与杭氧产的KZON-150/600-Ⅲ型空分设备相配套。虽然该机解决了工质被污染的问题，对空分设备的安全、稳定运行起到了良好作用。但继之而来的问题是，该机在停车备用时，由于外界湿空气的侵入，气缸壁容易产生锈蚀现象。

1990年，我们车间安装投运了两台5L-16/50-Ⅲ型压缩机。由于玻壳生产的特殊需要，必须开一备一。停车检修期间，我们经常发现二级气缸和三级气缸有锈蚀现象（夏季尤甚），容易造成缸壁拉毛，活塞环“抱缸”。不得已只好常对气缸进行专门处理，造成不应有的损失。

# 具体措施如下：

1. 在压缩机平衡管上设一截止阀或闸板阀，从该阀前开孔将氮气引入二级气缸。  
2. 干燥氮气气源从  $36\mathrm{kW}$  加热器前开孔引出，连通压缩机平衡管。

3. 开机时将平衡管上的阀门打开, 关闭通氮气的阀门, 即可投入正常运行。  
4. 停机后，关闭平衡管上的阀门，防止氮气向外泄漏。然后打开通氮气的阀门，引入氮气。氮气自平衡管进入二级气缸、二级冷却器及三级气缸，由此来达到充氮防锈的目的。

1992年9月英国伯明翰举办全国娱乐展览，把“人工造雪”引入英国市场。这项新技术采用液氮进行人工造雪，这就使室内滑雪的娱乐活动成为现实。

# 制氧工问题解答（四则）

1. 问：我厂有一套  $50\mathrm{m}^3/\mathrm{h}$  制氧机，三级套管式冷却器移到室外的一个水槽中，其出口压力与纯化器压力相差高达  $0.7\mathrm{MPa}$ 。为什么阻力这么大？

答：据说两只压力表相差  $0.7\mathrm{MPa}$  已有好几年，总以为表的精度有问题，因平时并不影响制氧机的正常运转，就没有加以重视。直到1993年夏天，启动时分馏塔压力升不高，甚至液氧液面升不上去，才开始重视。经检查压力表尚在校验期内。进一步用同一只压力表，把压缩机三级压力表装到纯化器上进行对比检查，这样做是为了排除表的精度误差。发现纯化器压力表阀没装垫片，接头泄漏；装上新的填垫并用肥皂水检查不漏后，前后两表的压力差只有  $0.4\mathrm{MPa}$ ，这才是真实的压力差。按常识判断，这两点阻力至多也不应超过  $0.2\mathrm{MPa}$ ，所以还是有问题存在。接着把注意力集中到冷却器管路有无问题上。

在研究冷却器以前，总认为它是借鉴外单位的经验，将套管冷却器的气管取消，改用大口径的蛇管，原以为是不会产生问题的。可是经过进一步观察，发现高压空气在水槽中冷却析出的水分，会积聚在冷却器的最下端。由于冷却器出口管上提很高  $(2\sim 2.5\mathrm{m})$  ，积水有可能排不出去，出现液堵现象，使阻力增加。就把管径逐渐堵小，最后在某一小口径下达到平衡。

因为纯化器已有四年没有更换过5A分子筛，又在拆开压缩机进气阀时发现阀腔机体上有少量水分积聚，这显然是油水分离器吹除操作不彻底引起的。所以问题的另一个焦点在于，可能有过量的水进入纯化器。而

该纯化器的进气是自上而下的，压力表是从纯化器进口端筒体上引出，一旦有水滴进入分子筛，破碎后被压实(扁)，这又是一个阻力增大的原因。

原因找出后，就可以采取措施：（1)在冷却器下端增加一只小口径液体吹除阀或疏水阀，小管接头要从上部插入，如图，否则水吹不出来；(2)更换粉碎压扁（压实）的分子筛；(3)加强油水的吹除管理。

2. 问：中压带膨胀机循环的小型空分设备缩短启动时间的操作要领是什么？

答：中压带膨胀机循环的小型空分设备缩短启动时间的操作要领是：

冷却阶段：(1)保持高压压力在设备允许的最高工作压力；(2)使空气尽量通过膨胀机制冷；(3)降低中、低压压力；(4)控制 $\mathrm{T}_{2}$  （膨胀机后）温度在  $-140\sim -155^{\circ}C$  之间。

积液阶段：(1)保持高压压力；(2)控制 $\mathrm{T}_{3}$  （节一1阀前)温度在  $-155\sim -165^{\circ}C$  (3)关小氧气流量至正常流量的  $1 / 3$  左右；(4)控制低压压力在  $0.05\sim 0.055\mathrm{MPa}$  之间；(5)控制热端温差。

调纯阶段；(1)保持高压压力并及时降

压；(2)保持  $\mathrm{T}_{3}$  温度液氧液面在  $30\sim 35\mathrm{cm}$ $\mathrm{CCl}_4$  柱之间；(3)缓慢关阀并合理控制液空、液氮节流开度；(4)合理调整返流气体出口流量及温度。

# 3. 问：中压带膨胀机循环的小型空分设备缩短启动时间的操作要领的主要原理是什么？

答：其主要原理是：空分塔内温度距正常工作温度  $(-172\sim -194^{\circ}C)$  较大，精馏需要大量液体，因此，启动阶段需大量的冷量。生产冷量的多少，取决于膨胀机的制冷量和高压压力。膨胀制冷量多少决定于膨胀机效率、前后的压差、通过膨胀机的气量。当膨胀机效率一定时，扩大膨胀机前、后的压差和增大进气量就可增加制冷量。扩大膨胀机前后压差的办法是将高压压力控制在设备允许的最高压力，中压压力尽可能降低，要打开所有吹除阀、分析阀。低压压力对中压有影响，也应尽可能降低，只要能满足纯化器再生就可以。

在冷却阶段  $\mathbf{T}_{2}$  达  $-140^{\circ}C$  时打开节-1阀，过早过迟都不利于缩短启动周期。过早打开节一1阀会减少膨胀机制冷量；过迟则会使热端冷损扩大，并影响节一1阀前温度的下降。从  $-140^{\circ}C$  开始至  $\mathbf{T}_{3}$  温度达正常温度  $(-155\sim -165^{\circ}C)$  之间，约有1小时左右的气量分配转换过程，即冷却阶段向积液阶段过渡。

由于通过膨胀机的气量是不能产生液体的，而液体只能在第二热交换器内进一步冷却后通过节一1阀产生。严格地说，空气达 $3.65\mathrm{MPa}$  （绝压）、  $-140.68^{\circ}C$  开始液化，节流阀后达  $0.6\mathrm{MPa}$  、  $-173^{\circ}C$  才能产生液体。为了减少节流汽化，有一定过冷度，要把节一1阀前温度控制在  $-155\sim -165^{\circ}C$  之间。这一段过渡期间，为了确保膨胀机的制冷量，又要使  $T_{3}$  温度迅速下降，合理分配气量，

就必须控制  $\mathbf{T}_{2}$  温度在  $-140 \sim -155^{\circ} \mathrm{C}$  之间。

在积液阶段，为了尽多地产生液体并尽快地积聚起来，一方面要保持高压压力和  $\mathbf{T}_{3}$  温度；另一方面，关小氧流量和控制低压及出口温度差。关小氧流量，增加上塔底部蒸发量，有利于氧纯度的提高；减小冷凝蒸发器温差，有利液氧的积聚。从这个意义上讲应将氧气出口阀全关，但这样会使热交换器传热面积减少，热端温差扩大。所以，氧气流量控制在正常流量的1/3为佳。提高低压压力的目的，是缩小上塔与下塔之间的压差，从而使冷凝蒸发器温差减少，有利于液氧的积聚。

调纯阶段初期需要大量冷量，当液空、液氮节流阀关小时，下塔、上塔上升蒸汽量增加，阻止小孔漏液，液体在塔板上积聚并开始精馏。因此，关阀必须缓慢，确保液氧面稳定。该两阀开度是否合理的标志，是液氧液面是否保持在  $30\sim 35\mathrm{cmCCl_4}$  柱范围，关阀结束时液空、液氧纯度是否在设计范围内。

当关阀基本结束，塔内精馏工况已建立，冷量仅仅用来弥补绝热层的跑冷损失和热端温差的冷损，以及气体分离消耗的功。因此必须降压，减少冷量的生产。冷量多少的标志是液氧液面。在保持液氧液面的前提下，高压压力越低越好。

生产的是获得成品，成品的产量和纯度成反比例，必须合理调节。由于流量改变，热端温差随之改变。为此，应及时调节入塔空气量的分配。

（2、3题 杭州制氧机厂

# 4. 问：为什么小型制氧机采用分子筛常温纯化空气流程，仍会爆炸？

答：我国制氧行业早期设计的分子筛纯化器（亦称分子筛吸附器），都没有配带冷冻

机，进分子筛纯化器的空气温度偏高，而分子筛吸附  $\mathrm{H}_2\mathrm{O}$  、  $\mathrm{C}_2\mathrm{H}_2$  、  $\mathrm{CO}_{2}$  的性能，同进入的空气温度关系很大。空气中的水分随温度的升高而增加，随压力的提高而下降。水分在共吸附中先被吸附，若带入的水量过多，则势必影响后吸附  $\mathrm{C}_2\mathrm{H}_2$  、  $\mathrm{CO}_{2}$  、  $\mathrm{CnHm}$  的能力，更何况分子筛吸附  $\mathrm{C}_2\mathrm{H}_2$  、  $\mathrm{CO}_{2}$  等的能力，随吸附温度提高而下降。

大型空分设备采用分子筛常温纯化空分流程，空气进纯化器的温度一般设为  $8\%$  个别的为  $15\%$  ，就是考虑了气温对分子筛纯化空气性能的影响。当流量和流程压力一定时，分子筛的动吸附值随进入空气温度的升高而减少。

小型制氧机（一般指“20”、“50”、“150”、“300”、“800”等）是量大面广的产品，尤其是 $50$ 、 $150\mathrm{m}^3/\mathrm{h}$ 制氧机。而有的地区大气污染日益严重，碳氢化合物含量剧增，实际进入的 $\mathrm{C}_2\mathrm{H}_2$ 、 $\mathrm{CnHm}$ 含量，远远超过设计设定值；又未设置 $\mathrm{C}_2\mathrm{H}_2$ 、 $\mathrm{CO}_2$ 自动分析跟踪，但仍按原定吸附时间切换；对液氧中的 $\mathrm{C}_2\mathrm{H}_2$ 和 $\mathrm{CnHm}$ 又不定期分析。这样，就严重威胁着分馏塔的安全正常工作。

还有一些单位的分子筛纯化器不是正规

产品，而是参照设备制造厂的图纸制造的，有缩小和走样的现象。如金昌市某公司机械厂制氧车间的分子筛纯化器就是缩小的，但切换时间并没有缩短，于是在1993年7月25日晚8时25分发生爆炸，死亡一女操作工，设备全毁。在爆炸前该制氧车间已安全运行20余年。

促使分馏塔爆炸的原因，除  $\mathrm{C}_2\mathrm{H}_2$  、  $\mathrm{CnHm}$  外尚有油、氧化氮、静电感应及操作中上、下塔压力剧烈波动等因素。

还有一个不被人们完全认识和注意的原因：分子筛多次反复再生，其颗粒会被磨损，吸附性能下降，如再生2000次，吸附能力下降  $30\%$  ；再加不定期高温活化。这些都影响着分子筛的吸附能力。

引起空分设备爆炸的主要原因是纯化后 $\mathrm{C}_2\mathrm{H}_2$  、  $\mathrm{CnHm}$  含量超过规定，而引起其超过规定的原因除进纯化器空气温度高、空气污染严重、装的吸附剂量偏少及使用时间延长，导致分子筛失效外，还有空压机带油、吹除不彻底或不及时、进塔空气量增多、再生不彻底等。

[杭州制氧机研究所（退休） 陈志新]

# 液氮在饮料行业中的应用

摘要 介绍液氨用于非充气饮料装罐的优点；整个加注系统——液氮的贮存，输送和液氮注入器，以及决定贮存、输送方式和注入器的主要因素。

# 主题词：液氮加注系统 贮存 输送 液氮注入器 热漏率

最近，在饮料工业出现了新一代的产品——非充气饮品。非充气饮料市场的增长率估计要比传统的充气软饮料市场的增长率高 $50\%$ 。由于这种新产品的市场销路越来越好，生产非充气饮料的罐头厂将会越来越多。

要使这些非充气饮品在市场上具有较强的竞争力，必须解决两个主要问题：一个是必须改进装罐方法，将装罐的速率提高到700～1200罐/分。这样才能提高它的经济效益，使它与传统的软饮料罐头相比更具竞争力。第二个问题是必须在罐头最终密封之前加压加到一定的压力范围，以便在不用  $\mathrm{CO}_{2}$  的情况下采用传统的包装、堆放及运输方法。

通过使用一个特殊设计的液氮贮存、输送与加注系统，可在每只罐头内形成一定的压力，并使非充气装罐工业达到预定的加工速率。在非充气饮料罐头中加注液氮的另一个好处是，所注入的少量液氮可排除每只罐头顶部空间中的氧气，从而延长了易腐品的贮存期限。

# 一、全承包系统

用于非充气饮料装罐的整个液氮加注系统，包括三个主要部件——液氮的贮存、输送以及液氮注入器。

值得重视的是，整个液氮加注系统所有部件，都必须具有非常低的热漏率，以确保注入每只罐头的液氮是最低限度的饱和液体。

一般来说，氮的应用有两种型式——液

态或气态。在应用气态氮时，正常热漏所产生的过量气体，可通过回收管路重新加以利用。然而，非充气饮料的装罐采用的是液氮。在应用液态氮时，热漏是一个绝对值，它会造成闪蒸损失，或由于在注入点的液体中存在着过量气体而造成质量不稳定。

因此，最好是采用由同一家低温设备制造厂设计，制造并进行安装的全承包系统，以便将热漏变量控制在最小值。

# 二、液氮贮存容器

液氮的贮存方式取决于预计的液氮使用量，而液氮使用量又取决于所生产的非充气饮料罐头的数量。

决定正确贮存方式的另一个因素是节省劳力，采用大容量贮存系统可以实现这一点。使用瓶装液氮系统需要经常进行调换，需要花费较多的劳力。总之，必须根据各个工作现场的具体情况，对贮存容量和运输劳力加以适当的考虑。

一般来说，使用大容量贮存系统的费用最低(包括劳力)。选择大容量贮存容器的关键问题是它的蒸发率。目前，美国明尼苏达谷工程公司提供一种低压、超绝热的大容量贮存系统，其热漏率只有其它工业用标准珠光砂绝热贮槽的1/3（蒸发率为  $0.35\%$  /天比 $1\% /$  天）。这种低蒸发率的贮存容器能确保使最恒定的液氮输送到液氮注入器。

# 三、液氮的输送

液氮的输送可采用泡沫塑料绝热管道或真空绝热管道。泡沫塑料绝热的热漏率较高，因此不适合于这种应用。然而，在暂时使用的情况下，泡沫塑料绝热却是一种最为经济的选择方案。

绝大多数罐头厂，需要的是真空绝热管道，它包括用因瓦钢或不锈钢制的内套，不锈钢制的外套，超绝热材料，以及由不同金属制的卡口式管接头。

这种特殊的卡口式管接头是美国明尼苏达谷工程公司的专利品，它的热漏率非常低。这种管接头是由不同的金属材料制成的，内接件为因瓦钢，外接件为304C不锈钢。当它与液氮接触时，外接件的收缩量比内接件大，因此不会泄漏，在高达  $0.414\mathrm{MPa}$  (60磅/平方英寸)的试验压力下也没有产生泄漏。

由于采用这种特殊的管接头以及低蒸发率的输送管道，确保使恒定的液氨在尽可能低的饱和压力下通过液氮加注系统。其结果是，使每一只非充气饮料罐头都具有恒定的压力。

# 四、液氮注入器

目前所能获得的液氮注入器，与低热漏率的液氮贮存容器和低热漏率的真空绝热管道相辅相成。这种注入器通过一个连续流动结构引入饱和压力较低的液氮，并在罐头经过注入器时，以  $700\sim 1200$  罐/分的速率将一定数量的液氮注入每只罐头。

每只罐头的额定压力为  $0.172 \pm 0.014 \mathrm{MPa}$  （ $25 \pm 2$  磅/平方英寸）（表压），这个要求是很高的，然而采用这种低蒸发率的贮存容器、真空绝热管道以及有效的液氮注入器，就能够容易地持续地获得这种额定压力。

# 五、结束语

这种液氮加注系统与其它许多低温液体的应用一样，低热漏率的贮存和输送系统是生产恒定质量产品的重要因素。

整个系统的具体结构可根据具体工厂的布置或运行情况而定。预计的市场增长率也是确定具体设备技术要求的一个重要因素。所有的可变因素都必须通过仔细的现场勘察和分析加以考虑。

1993年2月1日，扬州制药厂特非那丁车间，在打开氢气瓶瓶阀时，气瓶发生了爆炸，死亡1人（详见1993年《深冷技术》第3期）。1993年11月26日，扬州市卫生防疫站，在对喂药器聚氯乙烯单体残留量送检样品进行检验，当打开氢气瓶瓶阀的瞬间，再次发生了氢气瓶爆炸事故，瓶阀操作者——理化检验科副科长、主管检验师蒋庆安同志当场被炸死，身体的大部分被气浪抛至  $18\mathrm{m}$  外，死者脸部飞落在距爆炸点  $40\mathrm{m}$  左右处，右臂断挂在试验楼三楼楼梯东侧阳台的铁栏杆上；气瓶至少被炸成7块碎片，其中搜集到的6块碎片总重比气瓶原重量少  $2.75\mathrm{kg}$ 。飞离最远的一块碎片重  $2.25\mathrm{kg}$ ，距爆炸中心约  $250\mathrm{m}$ 。距爆炸点  $40\mathrm{m}$  半径内的部分房屋、门窗及其玻璃损坏。

本人对该次气瓶爆炸的性质、原因等进行分析、鉴定。鉴于该爆炸气瓶的爆炸威力极大，该爆炸气瓶使用的减压阀高压侧压力表指针爆炸后停留在  $8\mathrm{MPa}$  而不能回零。又据现场破坏情况，以及结合本人以往对事故调查所积累的经验，估计该气瓶爆炸前瓶内瞬间压力高达  $100\mathrm{MPa}(1000\mathrm{kgf/cm^2})$  左右，从而该气瓶爆炸性质完全可确定是化学性爆炸。另外，又据该爆炸气瓶的购气发票及扬州晶体管厂的氢气瓶发放清单，都完全证实了该气瓶与扬州制药厂1993年2月1日爆炸的那只氢气瓶，均系扬州晶体管厂1月28日所充装。据对2月1日扬州制药厂氢气瓶爆炸的原因分析结论，也完全可推断该爆炸氢气瓶内氢中含氧高达  $11\sim 18\%$  。

# 机械工业部气体分离设备科技情报网第二届理事长会议纪要

机械工业部气体分离设备科技情报网第二届理事长会议于1993年12月16日至18日在杭州召开。12个正副理事长单位和2个网员单位26名代表参加了会议。情报网理事长、杭州制氧机研究所副所长孙德盛出席并主持会议。

会议通过讨论，纪要如下。

1. 会议听取了理事长、杭氧所副所长孙德盛高工所作的1993年情报网工作小结和1992年11月至1993年11月情报网经费开支情况汇报。会议还听取了小空分网、制氢分网的工作情况汇报和开展工作的设想。  
2. 会议确定1994年情报网成员大会在哈尔滨、西安、镇江、玉环4个地点中选一个较为合适、条件成熟的地点召开，具体事宜由总网负责联系决定。会议内容为：情报网两年来工作总结、评选表彰先进及讨论今后网工作计划。本次大会技术交流专题内容为：低温液体设备研究、设计、制造；低温液体生产、贮存、操作控制、分析、运输方法；低温液体应用领域的开发等方面的论文、经验和技术。其他如液氮低温粉碎、医用氧、高纯氮、氮气保鲜，以及空分新技术、新工艺等方面也可应征交流。欢迎全网成员单位和广大空分设备用户单位能积极参与。本次全网大会还将扩大到非网员单位，欢迎有志参与技术交流的制氧厂（站）或低温部门参加。  
3. 会议讨论了上海宝山钢铁公司能源部、攀枝花钢铁公司氧气厂、北京永丰冶炼厂等七个单位的入网申请。经讨论一致同意，批准上海宝山钢铁公司能源部、攀枝花钢铁公司氧气厂、北京永丰冶炼厂为本网新成员单位。对其余单位，理事长会议认为目前暂缓发展。

# 第4届中日（低温、超导)双边学术交流会议在京召开

第4届中日(低温、超导)双边学术交流会(The Fourth Joint Sino-Japanese Seminar on Cryocoolers and Concerned Topics-JSJS-W)于1993年10月19~23日在北京召开。

中日双边在低温科学和技术方面的专题学术交流会议的目的，是为了推动和促进双方在这领域的研究和技术工作的发展。以前已经举行过三届，由于会议的内容很集中，双方对口领域的专家出席，能进行深入的学术交流，对于促进相互了解、学习借鉴、结交同行、增进友谊是非常有益的。

# 二、APCI和BOC向俄罗斯购买氨气

APCI公司和BOC集团公司已分别与俄罗斯国家能源公司的分公司S.E.Orenburggazprom以及莫斯科的一家私营公司CRYOR签订了长期协议，购买0.0283亿米/年（1亿立方英尺/年）的氦气，主要用于供应西欧市场。

# 四、工业气体价格上涨

美国Praxair公司从1993年7月1日起提高其氧气、氮气、氩气和氢气的价格。氧气、氮气和氩气的价格上调  $2\% \sim 4\%$  ，氢气的价格上调  $10\%$  。
