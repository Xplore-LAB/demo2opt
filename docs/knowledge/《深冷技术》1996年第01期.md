# 一、制氧机的选型

根据杭钢的发展规模及制氧车间现状，扩建新设备势在必行。经多方论证及综合比较，认为上两套“6000”优于建一套“10000”。继而在选定杭氧第五代“6000”制氧机的基础上，对主空分流程及配套设施作了如下考虑和确认：

# 1. 空分塔是否需要采出液态产品

空分塔是否生产液态产品，对膨胀量、主换热器温差以及装置能耗等均有影响。就装置本身而言，主冷连续排放  $1\%$  的液氧已能满足防爆要求。鉴于我厂现有制氧设备状况不甚理想，加上氧气球罐贮存能力相对不

足，为有效稳定供氧，并兼作局部调峰操作，故在装置生产氧气  $6000\mathrm{m}^3 /\mathrm{h}$  、纯度为  $99.6\%$ $\mathrm{O}_2$  ，生产氮气  $13000\mathrm{m}^3 /\mathrm{h}$  、纯度为  $99.999\%$ $\mathbf{N}_2$  的基础上，再向杭氧提出生产  $2 \%$  的液氧（折  $120\mathrm{m}^3 /\mathrm{h}$  标态气氧）和  $0.5\%$  的液氮(折65 $\mathrm{m}^3 /\mathrm{h}$  标态气氛)的工艺要求。同时，根据总图用地条件和总体布局，选用两台  $50\mathrm{m}^3$  液氧贮槽，并配  $6000\mathrm{m}^3 /\mathrm{h}$  低压汽化装置一套。据此设计，可在主设备事故情况下，连续按 $6000\mathrm{m}^3 /\mathrm{h}$  流量供氧12小时。至于同步配置 $15\mathrm{m}^3$  液氮贮槽和相匹配的液氮泵及高压汽化装置，则是为了增加生产车间的外销品种。

# 2. 空分提氮配“120”还是配大于“120”

根据杭氧现有的设计制造水平，常温分子筛纯化流程加增压膨胀机的“6000”制氧机，能提取大于  $120\mathrm{m}^3 /\mathrm{h}$  的精氩，而按杭钢的用氩平衡表，精氩  $120\mathrm{m}^3 /\mathrm{h}$  已有富裕，但考虑到新项目尽可能接受或采用新工艺新技术，同时也为今后外销氩创造增产条件，故在互惠互利的基础上（即对杭钢的制氙装置，杭氧仍按“120”的设备价结算；而杭氧如需调试，则可在杭钢的设备上进行），选定带大于  $120\mathrm{m}^3 /\mathrm{h}$  的提氩设备，但合同值仍定为 $120\mathrm{m}^3 /\mathrm{h}$  。氩系统的主要技术参数见表1。

# 3. 杭钢“6000”空分塔的主要特点

（1）由于氧、氮、氩液态产品数量较多，空分流程计算所需的膨胀量为  $5050\mathrm{m}^3 /\mathrm{h}$  （标态）。为使氮平均纯度和氧提取率达到较先进的水平，流程设定约为  $1000\mathrm{m}^3 /\mathrm{h}$  的膨胀量不进上塔而旁通至污氮管。  
（2）增压膨胀机的进气压力  $\mathbf{P} = 0.909$  MPa(绝)；增压机的增压比  $\varepsilon = 1.438$ ；机组绝热效率  $\eta_{\mathrm{s}} = 80\%$ ；功转换率为  $\sim 60\%$ 。  
（3）下塔不抽污液氮，取消污液氮节流阀及配管设施，从而简化主塔流程。  
（4）不设增压膨胀空气换热器，而是将膨胀空气安排在主换热器的通道中，进一步简化流程并减少冷损。具体流程为：从主换热器增压空气通道中部抽气去增压膨胀机，另一路不含湿的空气则由主换热器的下部（而不是底部)抽出，用来调节膨胀机的进出口温度。  
（5）粗氩塔采用导向塔板，提高了塔板效率；精氩塔采用液相进料，既能回收  $95\%$  以上的过量氢，又使工艺氩在液氩的冷却下产生分凝提纯效果。

# 二、配套设备的选型

根据用户的需求，杭氧可以设计制造针对性很强的专用设备。这一经营举措，不仅使设备选配更趋合理化，而且也有助于拓宽工程设计的思路。

# 1. 空气过滤器选反吹式还是用脉冲式

根据空压机对吸入阻力的要求，空气过滤器有多种形式可以选择。经综合比较，设计倾向于干袋式。从清灰方式看，又有反吹

式和脉冲式之分。前者为鼓风机反吹；后者则由电磁阀、脉冲阀通过文氏管对气流的诱导作用，使过滤袋能瞬间膨胀、振动，从而达到清灰目的。脉冲过程按顺序自动进行，且无需另设反吹装置，较有特色。故选定杭氧产的脉冲式空气过滤器，其处理空气量为 $36000\sim 48000\mathrm{m}^3 /\mathrm{h}$  （标态）；净过滤面积为 $400\mathrm{m}^2$  ；过滤袋(文氏管)数量为384只；电耗  $\leqslant 6\mathrm{kW}_{\circ}$

# 2. 空压机选DH型还是配DA型

DH型双轴机组的转速，容易设计成最佳值，但噪声相对较大；而DA型单轴机组的优点是：振动小、泄漏少、噪声低、价格相对较低。考虑到制氧车间现有空压机全为单轴透平，在操作和维护上较有经验，最后选定杭氧制造的等温型单轴机组。根据杭钢所提的环境温度、湿度、大气压力等设计条件，杭氧设计制造出1TY-547/5.7型单轴五级四次中间冷却的离心式空压机，其标态流量为  $32800\mathrm{m}^3 /\mathrm{h}$  ；排气压力为0.67MPa(绝)；排气温度为  $\leqslant 90~\mathrm{^\circ C}$  ；空压机转速为8268r/min；根据杭钢的实际情况，选配  $3600\mathrm{kW}$  异步电机。

# 3. 空气预冷系统带不带水冷塔

由物料平衡可知，空分塔尚有  $2800\sim$ $4700\mathrm{m^3 / h}$  的污氮量旁通放散，利用该部分冷量，最大可使水温从  $40\%$  冷却至  $30\%$  （水量为 $\sim 10\mathrm{m}^3 /\mathrm{h})$  。但由于污氮量较少，且分子筛纯化系统所耗污氮量有一定的波动范围，会对预冷系统的工况产生不利影响；同时，为了简化流程、方便操作、减少设备占地面积，

最终，同意杭氧在设备初步设计中提出的不带水冷塔的技术方案。

# 4.冷水机组选进口机还是配国产机

对第五代空分装置来说，冷水机组的作用十分关键。它不仅要向空冷塔提供符合压力、温度、流量要求的冷冻水，而且还应同时冷却膨胀机的增压气体和除氧气后的粗氩。尤其是对空冷塔上段空气的冷却，直接影响到分子筛纯化器的工作负荷、纯化效果乃至主塔的运行工况和周期。为此，决定引进一台法国Carrier(开利)机组作为主设备。考虑到运行的可靠性、合理性以及节省投资，再另要求杭氧配两台烟台产冷冻机作调节和备用。两种机组的主要技术参数列于表2。

机组制冷量的调节范围是：开利机组为6缸2回路；烟台机组的三列工作缸可作上下载调节。当选配两种不同规格的机组时，其主要运行方式可作如下搭配：①开利机组投运，烟台机组作备用（或两种机组开备互换）；②开利机组半载（即3缸工作，3缸备用），烟台机组一开一备；③当系统需要增加制冷量时（如出现空压机排温升高或需开两台膨胀机等情况），可在开利机组满负荷运转的基础上另加一台烟台机组工作，并调节其制冷量与系统运行工况相适应。

# 5. 分子筛选用电加热还是蒸汽加热

杭氧设定分子筛的解吸温度为  $170\%$  。根据杭钢的能源情况，电加热或蒸汽加热均可采用。两种热源各有优点：电加热器作业环境清洁，再生气体无水分污染源；而蒸汽加热在供汽正常的条件下，加热工况较为稳定，

且启动操作可一步到位。据此，为了扬长避短，切实提高分子筛纯化系统的可靠性，设计选定的加热系统，既有一台电加热器及其配套的蓄热器，又并联一台蒸汽加热器。电加热器使用功率为  $150\mathrm{kW}$  （设计为  $171\mathrm{kW}$ ），出口气体温度为  $205^{\circ}\mathrm{C}$ 。当分子筛纯化器冷吹时，蓄热器为正流蓄热；而当纯化器进行加热时，则通过阀门的自动切换，蓄热器改为返流放热。如果使用蒸汽加热器，只需通入压力为  $0.9\mathrm{MPa}$ 、温度为  $250^{\circ}\mathrm{C}$  的过热蒸汽，加热系统即可投入工作。

# 6. 采用常规仪表还是选用计算机控制

自从空分装置计算机应用出现集中管理又分散控制后，其使用上的安全性和合理性均前进了一大步，因而集散系统在空分操作中的优越性也随之增加；尽管采用常规仪表在技术上早已成熟，操作上又比较习惯，但为了充分发挥第五代制氧机的技术优势，在考察调研和分析比较后，认为无论是装备水平还是人员素质，杭钢“6000”空分装置采用集散系统控制的条件已经成熟。故选定上海Foxboro（福克斯波罗）公司生产的I/A系列计算机控制系统，其硬件设施主要有：控制、通讯、应用及操作台处理机；打印机和两台TRC操作台等。

# 8. 外购瓶氢还是自建制氢站

早在1986年杭钢制氧设备技改设计时，为新建  $2^{\#}3200$  空分提氩而设置了供氢汇流排。但采购瓶氢却受到供氢单位的生产状况以及运输安全等诸多因素的制约，致使采购供应部门产生“买氢不如买氩”的想法。有鉴于此，故本次“6000”制氧工程设计决定自建一座制氢站，选用两台产量为  $12\mathrm{m}^3 /\mathrm{h}$  、最高工作压力为  $3.2\mathrm{MPa}$  的水电解制氢装置。这样配置，既能满足“6000”提氩所需氢气，并有相应备机，又可为  $2^{\#}3200$  制氩生产并网供氢。

# 三、工程设计原则

为了缓解制氧车间生产与技改的矛盾，提高新辟场地的利用率，“6000”制氧工程的设计原则定为“单元式”。即把“6000”空分装置所属的工艺设备集中布置成一个独立的单元体（4M16氧压机提前单独上马，厂房设在车间的压氧区附近），并预留  $2^{\#}6000$  主空分及辅助设施的场地；变电所、制氢站等在满足与  $2^{\#}6000$  制氧机安全距离的前提下，围绕主单元就近布置；仅把相关的工艺管线与老系统联网或沟通。

# 四、主厂房及设备布置

主厂房采用主、辅跨结构。工艺设备分为全封闭、半封闭和露天三种布置方式。厂房长  $48\mathrm{m}$ ，主跨  $18\mathrm{m}$ ，可装三台透平空压机；辅跨  $9\mathrm{m}$ ，内装膨胀机、冷冻机、空冷塔、多级水泵以及氩纯化系统等设备，辅跨

二层设主控室、分析室、变送器室和空压机操作室。工艺设备的平面布置见图1。

在设备布置中所考虑的几个问题及具体处理如下：

1. 主厂房三台空压机的设备基础由土建一次完成，但一期“6000”工程先上一台空压机。为解决  $1^{\#}6000$  空分塔的空压机备机问题，特将制氧车间原有配“3200”空分的三台DA400风机用总管并联，该总管再与“6000”空压机的送气管沟通，使得任选两台DA400机组，均可并联运行，对“6000”空分塔供气。

2. 杭州地区的严冬季节，往往使空冷塔的液位计、调节阀及水管等出现冻结堵塞现象。为此，对空冷塔采用半封闭式布置，即塔体下部以及液位计、调节阀等均设在辅跨厂房内，而让塔身的中、上部处在露天。

3. 由于氩纯化系统的钯触媒炉、空气冷却器、水冷却器和水分离器等组成一体（杭氧组装出厂），故不宜单独设钯炉室，但设计的保安措施有：① 氙净化间区段设置防爆风机两台；② 把氢控制柜单独移至氩净化操作值班室内（有防爆设施），从而使净化间内不存在常压氢气管；③ 在氩净化间和值班室设置气体检测报警仪，两室各装四只探头，八路巡回检测。一旦出现氢气或含氢工艺氩泄漏，在未达到爆炸下限值  $4\% \mathrm{H}_{2}$  时（一般设在下限值的  $50\%$  左右），即能发出报警，防止事故的发生。

4. 配建制氢站后，对车间内空分塔的液氧排放汽化放散提出了更为严格的安全要求。为此，特设计制作容积为  $52\mathrm{m}^3$  的液氧（包括液空、液氮、液氩）排放槽以及配套的长距离放散管，并将原制氧车间“3200”的排液槽与“6000”的槽体用铝管沟通，将汽化液体汇集排至“6000”主厂房东北角的僻静处，使排放口远离制氢间  $120\mathrm{m}$  以上。

# 六、工艺管线的局部修改

在对抗氧设备初步设计的会审时，经双方协商，将下列部位作局部修改：

1. 为了提高并有效调节空压机油冷却器的冷却效果，将其冷却水改为无压回水方式；另增一路新水，以改善盛夏季节的冷却条件。  
2. 空压机所属的各冷却器增设出水栓水管。这样做，一方面比较符合操作习惯，另一方面也为气体冷却器出现泄漏时（气压高于水压情况下）提供判断依据。  
3. 空压机增设DN200手动放散阀一只。该阀的操纵机构，安装在空压机操作室内，旨在紧急情况下增加人为保险因素。  
4. 两台粗氩压缩机增加试车专用进排气阀。原流程设计中，氩压机的进排气管分别与粗氩进出口总管相连；机组一开一备。因此，当其中一台机组检修后需要试车时，

其工作介质只能是氩和工艺氩的混合流。一旦试车机组启动运转，整个制氙系统的工况将出现大幅度波动：在粗氩产量不变的条件下，来自水分离器后的工艺氩回流量迅速增加，其增加值等于试车机组的排气量；此时，钯触媒炉、空气和水冷却器以及水分离器的气量负荷猛增，而经分子筛干燥器输出的工艺氩流量却锐减，导致精氩系统无法正常运行，甚至被迫停产，这是设计所不允许的。为此，在氩压机进出口管道上并联安装DN50试车专用阀及相应的配管，目的是将试车介质改为空气并使试车机组彻底与系统脱离。但是，为防止阀门泄漏或误操作对系统工况产生不良影响，故该空气进口阀在机组试车结束后应用盲板封死。

# 七、存在问题

全低压常温分子筛净化空分流程的一个显著特点是：改塔内“自清除”为塔外“他清

除”，操作重点也由塔内转向塔外。从试车情况看，杭钢“6000”制氧工程的主要问题就来自塔外系统的“冷”与“热”两个方面：

# 1. 冷却水与冷冻机方面

引进的“开利”冷冻机组性能良好，运转可靠，但其冷凝器对循环冷却水的质量显得十分敏感。机组要求冷却水的污垢系数为： $(5 \times 10^{-4}) \times 4.1868 \mathrm{~m}^{2} \cdot \mathrm{h} \cdot {}^{\circ} \mathrm{C} / \mathrm{kJ}$  。当冷凝器的冷却管壁上稍有污垢，其传热工况便满足不了设计要求，使R22蒸发压力升高，最终导致制冷压缩机过载跳闸，从而迫使空分塔临时停车。因此，有效地改善制氧车间的循环水质量，应引起足够重视并列为工程设计的重要项目。作为近期应急措施，除了及时清洗冷凝器列管上的污垢外，还可让冷冻机单独使用新水，以延长其结垢时间。

至于烟台冷冻机，因其质量与空分装置的配套要求尚有差距，有必要在今后的改造中逐步更新。

# 2. 加热器与蓄热器方面

分子筛纯化系统的电加热器，因其电控

回路中的元件质量不过关，往往使加热温度达不到设计要求，影响再生工况。更值得注意的是，为减小电加热器容量而配置的蓄热器，在运行中发现其蓄热能力不足（填料采用杭氧推荐的KXZ-A/10型蓄热瓷球）。按说明书要求的流量、温度进行加热操作时，对分子筛纯化器仅加热  $10\sim 12$  分钟（设计加热时间为26分钟），纯化器加热进口温度  $\mathbf{t}_{\text{进}}$  即开始明显下降，到加热结束时，  $\mathbf{t}_{\text{进}}$  降至  $104\sim 111^{\circ}\mathrm{C}$  ；纯化器加热出口温度  $\mathbf{t}_{\text{出}}$  一般为  $5^{\circ}\mathrm{C}$  左右。而使用蒸汽加热器时，  $\mathbf{t}_{\text{进}}$  通常能始终保持在  $158\sim 168^{\circ}\mathrm{C}$  水平，  $\mathbf{t}_{\text{出}}$  相应为  $12\sim 20^{\circ}\mathrm{C}$  范围。由此可见，若长期使用电加热器系统，对分子筛的吸附性能以及主塔工况均将产生不利影响。

至于蒸汽加热器，虽工况稳定、运行良好，但其气侧的含水量指标，始终达不到零位；采用进口露点仪检测，同样不能排除微漏的可能。这表明，制造厂应进一步提高加热设备的产品质量。

# 主题词：低温机械 动压气体轴承 止推轴承 螺旋槽

# 一、前言

螺旋槽气体止推轴承是气体动压轴承的一种基本型式，它除具有动压气体轴承的一般优点外，与其它型式的动压气体止推轴承（如阶梯式、可倾瓦式、箔带式）相比，具有结构形式简单、承载能力高、稳定性较好等优良的综合指标。因此，在大多数用气体轴承做止推元件的高速低温机械中都得到了广泛应用。

基于螺旋槽气体止推轴承的这些优点，自本世纪40年代末，国外就开始对该轴承进行系统的分析，如今已有了较成熟的理论分析及成功的应用。然而在我国只是静压气体轴承在空分等工业部门有了较大的发展，动压气体轴承的工业应用还是空白。为使人们更好地了解螺旋槽气体止推轴承，本文将从理论分析、加工方法以及应用实例等方面对该轴承作系统介绍。

# 二、理论分析的发展

螺旋槽轴承的工作机理，可简单地描述为：开槽的轴承表面相对于光滑的表面旋转时带动粘性流体，使流体流向旋转中心或由中心向外流动，最后通过密封面完成对槽内流体的压缩，从而产生压力，承受载荷。然而简单的机理却包含着复杂的理论，自本世纪40年代末英国原子能中心的惠普尔创建了

最初的螺旋槽理论以来，螺旋槽理论就一直处在发展完善阶段，其中较经典的就有惠普尔不可压缩理论、马德曼不可压缩理论和准不可压缩窄槽理论等。

惠普尔不可压缩理论做了许多假设，如流体不可压、压力线性分布、不考虑槽的曲率等。根据以上假设，将用于计算无限长平板上的、沿运动方向等距离排列的平行槽压力分布理论，应用于螺旋槽的近似计算。由于假设所带来的粗略性，使理论值与实验值有很大出入。马德曼不可压缩理论则运用了保角映射的方法，将复平面内直线形槽线映射到另一复平面内变换为对数螺旋线；并且考虑了末端效应，修正了由惠普尔理论计算出的螺旋槽进出口处的值。该理论的结果较接近于实际，但由于仍未计及压缩性，故只适用于低压缩性的场合。准不可压缩窄槽理论，是潘宏道于60年代末提出的。他假定，在一槽台副中流体是不可压缩的，而就全局而言是可压的；槽数无限多，槽宽无限窄。该理论适宜各类几何形状的轴承，曾作为一种主要的分析方法被推荐。但假设仍不尽合理，理论值仍有偏离。由以上各理论得出的最佳参数如表1所列。

螺旋槽气体止推轴承在理论上的优越性在实践中却往往不能完全地体现出来。究其原故，一方面是气体轴承的润滑膜很薄（仅 $10\mu \mathrm{m}$  左右），对几何形状非常敏感；另一方面针对螺旋槽轴承而言，复杂的表面形状给制造上带来的困难是实际轴承性能低于理想时的主要原因。实践表明，槽数越多轴承表面的变形就会越严重。因此，热效应、形位公差、滑移现象等成为螺旋槽止推轴承乃至所有气体轴承所特有的影响因素。

# 三、加工方法概述

如前所述，螺旋槽轴承表面具有复杂的几何形状，而且槽深是微米级的。因此螺旋槽轴承的成形过程是非常精密的而且是具有相当难度的，这就要求用特殊的加工方法来实现。

# 1. 光蚀刻加工法

该方法的主要步骤是先在加工表面上涂敷光致蚀刻剂，然后加盖制作好的印有螺旋槽形线的底片，放在强光下曝光；将曝光好的工件放入显影液中洗去未曝光的部分（即槽部），使其露出工件表面。接着将工件放入腐蚀液中腐蚀，腐蚀深度(即槽深)由腐蚀时间控制。最后去除台阶上的抗蚀剂，这样就

完成制做。由于腐蚀后的槽深不十分均匀，这种方法不适于有精确尺寸要求的场合。

西安交通大学低温研究所近些年来通过大量试验，已基本掌握了该加工方法，所制做的螺旋槽气体止推轴承，已成功地用于空气透平膨胀机实验台，现正积极推广应用于各种低温透平膨胀机。

# 2. 机加工法

使用机加工方法的主要优点是可使槽深均匀，但一般需要专用设备。如文献[14]介绍的带导向环、夹具、铣刀的那种机床。切削用具还可用砂轮，但这样就会固定槽宽，使槽宽比不再是常数，造成较大的误差。

# 3. 喷镀法

喷镀法是指在轴承表面上喷镀出槽脊。其步骤是先用不锈钢制成铲除槽脊的样板，然后将样板放在精加工后的轴承表面上，喷镀氧化铬。

这种方法对于硬质材料制成的轴承表面是可行的，但给最后的精加工带来困难。另外喷镀膜在低温下容易产生细微的裂纹。

# 4. 焊接法

当腐蚀工装到不了现场或被腐蚀的轴承太大而进不了腐蚀槽时(如大的曲轴)，可将已腐蚀好的金属薄片焊接在止推盘上，需注意的是金属薄片应稍厚一些，以便焊后的不规则面可被磨削掉。

# 5. 高分子材料浇铸法

由于气体轴承制做时要求精密，应用时条件苛刻，因此制造成本高，在工业领域中应用较少。为降低制造成本，国外曾发明了先用环氧树脂浇铸，然后进行冷处理的方法来制作塑料轴承，大大降低了成本。后来为进一步提高生产效率，又改进为将热处理后的环氧树脂粉末压缩成型的方法。这种环氧树脂制作的止推轴承，采用的就是螺旋槽轴承，该轴承除降低成本外，还具有良好的启停性能，即使卡死，经简单处理之后也可重新使用。

除上述介绍的几种加工方法外，还有电火花法、喷砂法、选择性腐蚀法等，这里不再详述。

以上介绍了螺旋槽形线的成形方法，此外还有两个问题要引起注意。一个是为使经常的启停不致损坏轴承表面，需要采取一些必要的措施，这些措施包括：启停时输入预压支承气、利用辅助的磁力轴承、对轴承表面进行处理等；另一个是静盘与动盘材料的选择要充分考虑到一旦发生咬合，应保证哪一方不受损坏的问题。

# 四、实际应用举例

随着大量理论研究的深入进展，与动压气体径向轴承配合使用的螺旋槽气体止推轴承，在众多领域中得到了广泛的尝试，在美、日、德等主要西方国家成功的用例不胜枚举。

目前从温度极高的反应堆循环泵到极低温的氦透平膨胀机、超低温排气泵，从飞机制导所用的陀螺仪到磁鼓主轴、激光反射镜主轴，螺旋槽轴承都以其优良的特性，充分满足着尖端技术所要求的各种苛刻的工作条件。下面就介绍几个该轴承在高速低温机械中的应用实例。

# 5. 其它领域中的应用

由前所述，螺旋槽气体止推轴承的使用是非常广泛的，以下再简要地介绍它应用于其它领域中的一些实例。

# （1）在陀螺仪中的应用

在航空器、导弹、人造卫星的惯性诱导的超精密陀螺仪中，气体轴承已得到广泛应用。在此之前，陀螺仪使用的是滚动轴承。这种陀螺仪不但价格高寿命短，而且精度也较低。采用气体轴承后从根本上解决了这些问题。为了提高轴承的刚度和稳定性，采用了螺旋槽止推轴承。所用陀螺仪的转子直径为  $\phi 75\mathrm{mm}$ ，宽度为  $50~\mathrm{mm}$ ，转速为12000r/min。轴的材料为氮化钢，轴承的材料为3号钨钴硬质合金。

# （2）在磁记忆装置中的应用

气体轴承在磁记忆装置中应用于磁鼓。在磁鼓中为防止转子振动，确保转子与壳体之间的相对位置，没有采用弹性的轴承。另外由于采用了防尘的全密封结构，也没有采用静压轴承。动压止推轴承采用的是螺旋槽轴承。该装置的径向轴承内径为  $28\mathrm{mm}$ ，在

轴承半径间隙  $\mathrm{Cr} = 4\sim 7\mu \mathrm{m}$  的条件下，转速为  $30000\mathrm{r / min}_{\circ}$

以上介绍的几个应用实例，基本上显示了螺旋槽气体止推轴承在广泛的领域，特别是在高速低温机械领域的成功应用。为确切掌握气体轴承的应用状况，1972年和1982年日本气体轴承研究会曾进行了两次调查，调查表明动压气体轴承（其中止推轴承大部分为螺旋槽轴承）在为越来越多的人们所采用。西安交通大学近几年来已在螺旋槽气体止推轴承方面进行了较多的研究工作，可以展望在不久的将来，我国一定会将自己研制的螺旋槽气体止推轴承应用于生产实践。

# $3200 \mathrm{~m}^{3} / \mathrm{h}$  制氧机改造及运行小结

方保喜 黎源瑰 廖嘉锋(执笔)

（广西柳州钢铁公司制氧厂，柳州市北雀路117号，545002）

摘要 介绍柳钢  $3200\mathrm{m}^3 /\mathrm{h}$  制氧机的改造方案、指标、试车、改造后装置特点、考核情况、存在问题及改造后经济效益。改造是成功的，氧产量由原  $1800\mathrm{m}^3 /\mathrm{h}$  增加到  $3600\mathrm{m}^3 /\mathrm{h}$ ，并新增加氩气  $70\mathrm{m}^3 /\mathrm{h}$ ，电耗由原1.3下降至  $0.85\mathrm{kWh / m^3O_2}$ 。

# 主题词：制氧机 改造‘试车 特点 效益

柳钢现有三台制氧机，容量分别为一台 $4000\mathrm{m}^3 /\mathrm{h}$  、两台  $3200\mathrm{m}^3 /\mathrm{h}$  。其中  $1^{\#}3200$ $\mathrm{m^3 / h}$  制氧机是1972年安装投产的，为蓄冷器流程，至今已运行了20多年。由于种种原因，至改造前，实际产量只有  $1500\sim 1800\mathrm{m}^3 /\mathrm{h}$ $\mathrm{O_2}$  ，远远达不到要求，能耗高，设备故障频繁，运行可靠性很差。鉴于1992年  $2^{\#}$  制氧机改造成功的经验，我们决定对  $1^{\#}$  制氧机也进行改造，改造工程的设计工作委托杭州制氧机厂负责。改造后的工艺流程为分子筛常温吸附带增压膨胀机流程(带氩）。

# 一、改造方案

将原蓄冷器流程改造为分子筛常温吸附带增压膨胀机流程(带氩)；除空压机外，空分系统原有设备全部拆除(包括冷箱)，空分系统所有设备由杭氧负责重新设计、制造；将原固定滤料式的空气过滤器改为带吹风环的袋式过滤器；空压机取消了末级冷却器，空压机本身也作了一些必要的维修工作；加热系统采用蒸汽加热器与电加热器并存。

# 二、改造主要指标

空气 17500  $\mathrm{m}^3 /\mathrm{h}$

氧气  $3200\mathrm{m}^3 /\mathrm{h}$  99.6%  $\mathbf{O}_2$

氮气  $3500\mathrm{m}^3 /\mathrm{h}$ $\leqslant 100\mathrm{ppmO_2}$

氙气  $70\mathrm{m}^3 /\mathrm{h}$  （折气态）  $99.99\% \mathrm{Ar}$

空气出空压机后压力为  $0.65\mathrm{MPa}$  （绝）

# 三、安装、调试、裸冷、试车

旧设备拆除、新设备安装，主要由柳钢修建部、计控所负责。1994年4月下旬，所有设备安装完毕，各系统经初步调试正常，进行裸冷。

由于裸冷前期工作准备充分，裸冷工作进行得比较顺利，一次成功。1994年7月3日开始试车生产，试车也是一次成功。但启动时间比较长，从启动膨胀机至开氧压机压氧，共历时82小时。一方面是由于空压机两次停机影响；另一方面，由于大部分的调节阀性能不好，不稳定，工况的调整需要多次重复，甚至不得不放弃自动及遥控操作；另外，主冷液氮至下塔回流氮-1阀及返流污氮至膨胀空气换热器调-9阀关、开刻度与实际相反，导致越调节工况越恶化，改正后，逐渐恢复正常。由于氧气放空阀调氧-102关不严，泄漏量较大，氧纯度上升较慢，也延长了启动时间。

# 四、改造后装置特点

1. 启动方便, 操作简单。空气中有害物质在塔外清除, 不需要再象原流程那样闯过水分及  $\mathrm{CO}_{2}$  关, 大大简化了操作; 主塔工况

稳定，操作调整少。

3. 增压透平膨胀机焓降大，效率高，达  $86.6\%$  ，温降  $\Delta t$  达  $50^{\circ}\mathrm{C}$  以上，氧、氮、氩全量生产，膨胀量只需  $2500\mathrm{m}^{3}/\mathrm{h}$  左右，提取率也得到提高。  
4. 各系统运行情况，通过计算机显示观察，大大优化了操作条件。  
5. 临时停车后再启动恢复供氧快。曾有数次因电力故障造成停车  $3 \sim 4$  小时，但重新启动后恢复送氧均不超过2小时。

# $350\mathrm{m}^3 /\mathrm{h}$  空分设备调试中的两个问题

李照林

（江苏靖江孤山制氧厂，江苏靖江煤矿南路88号，214522）

1994年10月底，笔者应邀调试泰州钢厂新上马的  $350\mathrm{m}^3 /\mathrm{h}$  空分设备。设备是哈尔滨制氧机厂1994年2月的产品。调试中发现了在初次使用透平膨胀机和气动薄膜调节阀时经常发生的两个问题。

# 一、透平膨胀机未运转就有毛病

笔者首先接通膨胀机轴承气，然后对两台透平膨胀机的转子与机体间的电阻值分别进行了测量。结果发现两台膨胀机都有毛病。通过测量轴与机体间电阻值来判断透平机是否正常的方法，是我们在哈尔诺制氧机厂透平机总装车间学来的。

通过解体检查，发现两台机轴承外圆体与端面都有黄色锈斑和黑色粉末，轴承内孔及转子都有摩擦的痕迹。尤其是  $2^{\#}$  机轴承内孔严重磨损，且有4个孔被堵塞，用医用针都通不开。

针对轴承外圆体黄色锈斑、黑色粉末等，我们用粒度1200目金相砂纸打磨，并用通针疏通轴承气孔，对轴承内孔及转子摩擦处，用同样金相纸稍磨一下使其恢复光亮，再用干净气源吹扫，并疏通小孔组装好，后再用测量阻值方法检查合格，投入运转，转为正常。但  $2^{*}$  号机轴承的使用寿命可能会缩短。

为什么出厂没多久的新膨胀机，未投入运行就不正常呢？

该透平膨胀机出厂前，制造厂用专门的铁皮箱包装，箱内充有  $0.2\mathrm{MPa}$  的氮气保护气，并用药棉将轴承气、密封气的进出口堵塞。一般来说，运输、贮存都不会给膨胀机带来损害。制造厂还要求贮存6个月后，要进行检查，重新充氮保护。

由于该单位初次接触透平膨胀机，在安装时，从包装箱里取出透平膨胀机后，便就位配管；在没有通入轴承气的情况下，有人拨动转子，造成轴与轴承磨损。配管结束后，因不懂透平膨胀机的保护，保管不善，轴承气、密封气进出口都未用药棉堵塞；机芯随便放在铁皮箱内，连包装箱底座螺丝都未拧紧，也未充入氮气保护，使空气、杂质等进入，造成机体锈蚀，管道堵塞。

# 二、调-1阀装配不当，设备工作时气量严重不足，无法开机

液氮调节阀（调三1阀），为气动薄膜调节阀。通过阀芯上下移动来改变阀芯与阀座间流体的流通面积，从而改变流量。它通过阀杆与薄膜执行机构连为一体。这就要求在安装此阀时做到：在薄膜处于零信号压力时，将阀芯放到底(气开式为关闭状态)，再与推杆下端相连。而该单位在安装时，疏忽了阀芯位置，结果造成调三1阀关闭时，实际上仍开着的现象。因此启动后，无法掌握流体的流量，辅塔阻力明显高于正常阻力。尤其是液氧面产生后，气量严重不足，两台膨胀机不能满负荷运转，塔内工况反复波动。

针对以上问题，根据调-1阀要求进行如下调整：在不给薄膜接通气源之下，松开推杆之拼紧螺母，将阀芯放到底（即关闭状态），再与推杆下端相连，接着重新将螺母拼紧即可。经上面方法调整后，顺利产出了氧气。

上述两个问题均发生于调试之中，自解决好后，操作中引起了重视，至今再未出现过问题，一直运转正常。

林美珍

摘要 本文在分析加热气瓶干燥法的基础上，介绍了直接将分馏塔放空氮气引入气瓶进行吹排的“无加热气瓶干燥法”及其效果，并通过理论计算说明此法的可行性。图1。

# 主题词：气瓶干燥 氮气 无加热 操作步骤 效果

随着高纯度气体的开发应用，气瓶干燥的要求也相应提高。目前各气体厂（气瓶检验站）采用的气瓶干燥方法各不相同，但不外乎两种：一是直接加热瓶体，如煤气红外线加热、电加热、蒸汽加热。通过瓶体升温至  $110^{\circ}\mathrm{C}$  左右，让瓶内水分自动蒸发来达到干燥的目的，也可称为“烘干”。这种方法投资大，耗能高，工效低，劳动强度大，现已较少采用。

二是在气瓶中通入加热了的干燥氮气或干净空气，进行吹排。一般要求把氮气或干净空气加热至  $60\sim 90^{\circ}C$  。其干燥原理是：一方面加热的气体向瓶体传递热量，使瓶体升温，加速水分蒸发；另一方面(这是主要的)利用加热气体具有的强大干燥推动力，把瓶内水分带走。这种方法比第一种方法节能和简便，但不足之处是仍然需消耗大量热能，而且还得投入一笔资金，建造气体加热设备，因此，也不是最佳干燥方法。

# 一、无加热气瓶干燥法工艺流程及理论计算

我厂及有些厂采用“无须加热气瓶干燥法”，也可称为“常温干燥法”。笔者近年来对气瓶干燥进行了动力学分析和实际测试，表明这种无加热气瓶干燥法是可行的。这种

方法简要地说，是直接将分馏塔中放空的氮气引来对气瓶进行吹排，如图1所示。所用气体没有加热就用来吹排，同样能达到高纯度气体充装的要求，而且节能。这是因为从分馏塔出来的氮气含水量极低，其露点温度达  $-70^{\circ}\mathrm{C}$  左右，因而具有强大的干燥推动力，足以把瓶内水分带干净。由于直接用来自主塔的干燥氮气，无须加热设备，也就降低了工厂的设备投资与能源消耗。

为说明分馏塔出来的氮气干燥能力，可作如下理论计算：

$$
W = K _ {P} F \left(P _ {W} - P\right)
$$

式中  $W$  ——水分气化速率

$K_{P}$  ——以蒸汽分压差为推动力的给

质系数

$F$  ——水气化表面积

$P_{W}$  ——水分表面蒸汽压

$P$  —干燥气体中的水蒸气分压

式中数据取值：

$$
K _ {P} = 0. 0 7 4 5 (\omega \gamma) ^ {0. 8}

# 二、无加热气瓶干燥法操作步骤

1. 排除瓶内积水及杂质。将气瓶阀卸掉，垂直倒立，排净瓶内积水。注意，不能图省事仅打开瓶阀而不卸下，因为瓶阀不卸掉会有死角，极易积存水分，而且此水分比内壁上的水分难以干燥，将会严重影响干燥效果。

2. 翻正气瓶，用小灯泡插入瓶内，逐只

检查有无积水及观察内壁腐蚀情况。若尚有积水须再次倒立，排净为止。根据气瓶内壁腐蚀程度，将瓶分为A、B两类；A类为无腐蚀，通体光洁；B类为有轻度腐蚀。

3. 将干燥氮气通入气瓶进行吹排。吹排时间视氮气气源大小决定。A类瓶可少吹，B类瓶应多吹，条件允许则多吹些时间，我厂有时长达8小时。

4. 吹排结束后，再次用小灯泡插入瓶内检查，若气瓶已干燥，则内壁光亮；若内壁发暗，说明还潮湿，需继续吹排。

5. 装上瓶阀，抽真空。两台真空泵串联，抽至  $-760\mathrm{mmHg}$ ，停真空泵，马上回充需充装的高纯气体至正压，并关闭瓶阀。若充装的气体纯度要求很高，抽真空置换应在2次以上。

6. 正式充装气体前，再次置换。将气瓶充装到  $0.2\mathrm{MPa}$ ，停充，打开瓶阀放空，然后再正式充装。此操作目的有二：一是进一步置换瓶内气体；二是用高速气流冲击瓶阀，把瓶阀中的残留潮气冲走。

7. 充气后存放  $24\mathrm{h}$ ，抽样检测，达标后，可以向用户送货。

我们用此法干燥的气瓶，经严格检测，露点温度为  $-65\sim -70^{\circ}C_{\bullet}$

# 氧压机油压报警联锁系统改进

陈守亚 侯志勇(执笔)

(河南省安阳钢铁公司动力厂制氧三车间，安阳市铁西区梅园庄，455004)

4M16-93.5/30型氧压机，是杭氧产KDON-6000/6000-Ⅲ型空分设备的配套压氧部机，其运动机构由单独设置的供油装置对各润滑点进行压力油润滑，因此对油压力有一定要求。润滑系统由两台2CY8/33-1型齿轮油泵加压，A、B两台泵，互为备用。正常供油压力保持在  $0.4\sim 0.45\mathrm{MPa}$  ，当运转的油泵油压太低，低到一定值时，另一台油泵应通过联锁装置自行启动。

机组进油管路设有油压报警联锁PIAS-1209，当输出油压低于  $0.35\mathrm{MPa}$  时，报警并启动备用油泵；低于  $0.3\mathrm{MPa}$  时，联锁停车。

PIAS-1209压力变送器，原设计安装在中控室，由油管从取样点将油接到中控室的油压变送器，然后转变成电信号再送到联锁报警部分。

改进前，在油压低到报警联锁值后，夏天迟滞  $5\sim 6$  分钟，冬天则长达10分钟以上，报警联锁才动作，而工艺要求这应在瞬时完

成。这样PIAS-1209形同虚设，起不到应有的保护功能，尤其是该齿轮油泵故障较多，油压波动较大，使氧压机连续稳定供油得不到保证，安全运行受到严重威胁。

经分析，找出报警联锁动作时间过长的原因，是由于油压取样点至压力变送器间的取压油管过长，油压变送器安装在中控室位置不合理。由于取压油管长达10米多，压力损失大，压力变化信号传递时间长，造成联锁报警迟滞性过大。为此，我们将油压变送器从中控室移到就地安装，固定在油泵房的楼板上，将取压油管由10米缩短至不足0.5米，从而缩短了油介质传递压力信号的距离，缩短的距离改为电信号传递。

经上面的改进后，PIAS-1209报警联锁装置对油压敏感程度大为增强，动作时间由原来长达10分钟，缩短至不足30秒，使氧压机的压力油润滑得到了有力的保证。

（1995年8月改稿）

（吉林化学工业股份有限公司化肥厂基建分部合成氨装置组，吉林市合肥路27号，132021）

摘要 介绍吉化公司化肥厂“12000”空分启动时纯化器及空分塔等严重进水，分子筛被水浸泡，未作报废处理，采用先脱水干燥后过筛，再经严格活化，重新使用，效果较佳。文章详细叙述了事故经过、进水原因、处理方法、使用效果及教训与防范。图1。

主题词：大型空气分离设备 纯化器分子筛 进水 脱水干燥 过筛 活化使用

# 一、事故经过

吉化化肥厂1992年12月末建成投产一套KDON-12000/18000型分子筛净化带增压透平膨胀机流程的空分设备。试车成功后，因工厂需氧量所限，即刻停车备用。

1993年2月2日根据我厂其它设备检修需要，该塔准备开车投产，当日启动空压机后，实施设备开车前的分子筛活化与装置启动前的加热工作。2月3日凌晨，一班接班以后，即转入上塔的加热操作，由于操作不当、经验不足等原因，造成系统压力严重下降，发生了分子筛纯化器和空分塔等严重进水事故。致使工作的一只(2#)纯化器中的分子筛和氧化铝全部被水浸泡；加热再生的一只(1#)纯化器由于事故中处于均压状态，也进了大量的水，该纯化器中的分子筛有三分之一过水，氧化铝全部被水浸泡；蓄热器、电加热炉、增压机空气入口和空分塔下塔、上塔均大量进水。

进水经过是这样的。2月3日凌晨1点一班接班后，对加热情况检查后，确认可以向上塔导气吹除干燥土塔及氧、氮系统各流

路，并于1:30时左右向上塔导气，由氧、氨系统放空。向上塔导气增加了排放流路，但没及时关小下塔各排放阀，由于气量的增加，致使空压机排气压力由  $517\mathrm{kPa}$  降至  $< 400$  kPa，操作人员发现压力下降后，就加大空压机吸入蝶阀的开度，将空压机吸入蝶阀由 $33\%$  开度增加到  $40\%$  。之后在1:41时，空压机防喘振阀自动开大，操作人员将吸入蝶阀由 $40\%$  开度关小至  $35\%$  ，后又改为  $30\%$  ，防喘振阀仍在开大，又将吸入蝶阀关小到  $25\%$  。此种现象，没能引起操作人员的重视。当发现加热电炉断电后，去现场查看时，发现入下塔前空气吹除阀V316大量喷水，才发现装置已经进水。做紧急停车处理。

# 二、进水原因

# 1. 操作不当

造成严重带水的事故原因之一是操作不当，使系统压力下降过大所致。

该套空分所配空气冷却塔为串流式筛板塔，内设7块串流式塔板。正常工作时，由于压力和气体流量处于稳定状态下工作，各块塔板上均有一定的持液量，当塔内气压下

降时，上升气体不能使塔板上原有液位保持在原来的高度，塔板上的水大量流入到塔底，致使空冷塔液位急剧上升，这是其一；其二是由于塔内气压的降低，空冷塔的排水量相应减少，也使空冷塔的水位上升。这两种因素致使空冷塔液位急剧上升超过液位测量负压管以上，使得液位测量压差变小，空冷塔液位调节阀接此指令后，该阀即刻关小，使水位急剧上升，超过空气进口的高度，由于冲击夹带作用，发生带水事故。

# 2. 报警、联锁装置未发挥作用

事故中操作人员没听到报警铃响声，光字牌也未亮。事后验证空冷塔压力过低、液位过高，不报警，铃不响，经查系报警器保险断所致，这是其一；二是加热过程中，空冷塔气体压力过低、冷却水液位过高使水泵跳车，以及纯化器压差过大停止供水关HV-101阀的联锁未投入正常工作。当空冷塔气体压力下降、液位过高、纯化器压差过大时，冷却、冷冻水泵没联锁跳车，在空冷塔自动排水阀关小后，仍在向空冷内供水，从而造成严重进水。

# 3. 操作人员经验不足

操作中操作员发现空压机防喘振阀逐渐

开大后，没去查空分系统为什么吃不进气，而是关小空压机的进口导叶。因为先是系统压力下降，而后才发生空压机排气压力上升。防喘振阀自动开大，系后部空分系统进气量下降所致，此时已发生了进水。

如果有经验，防喘振阀刚开始自动开大，就检查后部加热系统为什么吃不进气而采取相应措施，就可将事故程度缩小，不致于造成空分系统严重进水。

# 三、事故处理方法

# 1. 排放系统进入的水

拆开进水设备最低处法兰和在最低管道下方开口排放。并将增压机入口过滤网的手孔拆下排放。

# 2. 接临时管作为吹除干燥导气管

事故后因两只纯化器中的分子筛和氧化铝均已被水浸泡，纯化器不能正常工作，因此不能从纯化器中通过吹除、干燥用气。笔者建议车间将纯化器进、出口空气总管的V1233和V1234阀之间加一  $\phi 219\mathrm{mm}$  管，并安装一DN200阀门，作为吹除干燥的临时导气管，如图所示（见图的左边）。

事故处理结果表明，此建议切实可行，且起到较好的效果。

# 3. 具体处理方法

# （1）干燥电加热器

为了确认电加热器是否还能继续使用，排净水后将其上盖打开，用干燥的仪表空气吹除干燥电炉，测绝缘还能使用，未形成开路，电炉丝未烧坏。但电炉入口气源管内仍有水，为使水不继续进入电炉内，将电炉入口气源管法兰拆开排放掉积水。

# （2）吹除电炉、蓄热器、增压机入口管和入塔空气管路

临时管接好后，开一台  $2800\mathrm{kW}$  空压机为吹除干燥提供气源。首先将纯化器系统（如图所示）各阀全部关闭，将空气入保冷箱处的法兰拆开，打开HV-101阀；增压机出口处加盲板防止气体倒流造成膨胀机倒转；打开电炉气体入口控制阀TCV-1220阀和蓄热器排放阀V1236；关闭HV107A·B阀和PCV1231阀。然后确认空气冷却塔压力在  $0.4\mathrm{MPa}$  以上，打开新接的导气阀开始吹除进塔空气管路和增压机入口管。待上述管路吹完后，稍开V1208阀向电炉入口管导气吹除，同时关闭HV101阀，联接好入塔空气管路法兰。待入电炉前管路无水后，稍开HV101阀向下塔导气吹除。关闭TCV1220阀，联接电炉入口法兰，之后使气体经电炉去蓄热器吹除干燥蓄热器。测电炉绝缘情况无问题后，将电炉通电用高温气体干燥蓄热器，并蓄热。待蓄热器排气温度达  $50^{\circ}C$  以上时，结束蓄热器的吹除与干燥。

# (3) 水浸后的分子筛的脱水与干燥

先将空气冷却塔的气体压力升至0.45MPa以上，将冷却、冷冻水泵投入工作，将出塔空气温度降至小于  $15^{\circ}C$

首先进行  $2^{\#}$  纯化器中分子筛脱水干燥。如图所示：将  $2^{\#}$  纯化器的V1212、V1214、V1216、V1206阀打开，开V1217、关V1236、V1218、开V1219，加大V1208阀开度，气量控制在  $6000\sim 8000\mathrm{m}^3 /\mathrm{h}$  ，待入纯

化器的气体温度小于  $130\%$  时，切除蓄热器，单独用电炉来的气体继续干燥2#纯化器中的分子筛，当排气温度达  $100\%$  以上时，结束  $2^{\#}$  纯化器中分子筛的干燥，进行冷吹。冷却结束后，用同样方法进行  $1^{\#}$  纯化器中分子筛的脱水干燥。

在干燥纯化器中分子筛的同时，剩余气量经HV101阀导入塔内吹除下塔、上塔。

待分子筛脱水干燥工作结束后，转入蓄热器的蓄热。

# （4）上、下塔的吹除与干燥

事故中上、下塔已大量进水，从主冷排液管可排出水，因此塔内的吹除干燥必须慎重进行。方法是，将两只纯化器各阀全部关闭，由新接导气管导入吹除气，经HV101阀送入塔内，按正常装置加热的流路进行吹除干燥。直至各流路无水时为止。

# （5）扒、筛干燥后的分子筛和补装新分子筛

分子筛被水浸泡后，有相当部分分子筛粉末，为此我们进行了过筛(氧化铝未动)。筛完后先将经水浸泡过的分子筛均匀装入两只纯化器中，然后将新分子筛补装在上方，装填高度比原筛床高出  $5\mathrm{cm}$  。

# （6）活化分子筛与装置的加热干燥

因装入的分子筛有五分之三是过水的，必须进行严格的活化才能使用。活化方法是将切换系统手动操作，使两只纯化器一只处于工作状态、一只处于活化再生状态，活化气量控制在  $6000\mathrm{m}^3 /\mathrm{h}$  ，并投入已蓄热好的蓄热器工作，切换时间不定，当活化排气温度高于  $150^{\circ}C$  时，改为冷却吹除。然后按同样方法进行另一只的活化。

待每只纯化器均加热再生两个周期后，将切换系统投入自动工作，之后每只纯化器各活化一次。并测量工作着的一只纯化器出口空气中的露点，当露点  $< -70^{\circ}C$  时，活化工作结束。

分子筛活化好后，按装置的正常加热方法实施装置的设备及各流路的加热干燥，并检测各排放口的排气露点达到  $-60^{\circ}\mathrm{C}$  时，结束加热工作。

# 四、被水严重浸泡过的分子筛处理后的使用效果

我厂该套空分设备使用的分子筛型号为UOP13X-APG分子筛，系中美上海环球分子筛有限公司的产品。

本次事故后，我们没采取报废分子筛的做法，而是采取先脱水干燥后过筛，再经严格活化，结果其强度与新的无多大差别，经用于生产，使用效果较佳，出纯化器后净化空气中的  $\mathrm{CO}_{2}$  含量一直小于  $2\mathrm{ppm}$ ，露点一直小于  $-70^{\circ}C$ 。分子筛和氧化铝也没发生破碎现象，从1993年2月发生事故至今仍在使用中。其吸附性能仍然良好，主换热器阻力不随时间的延长而上升，运转一年也无多大变化，完全满足了生产的需要。

经验证明，一旦不慎分子筛纯化器发生进水后，首要的问题应尽快使之脱水，脱水的气体温度应达到  $200\%$  左右，这样可以使水迅速蒸发。缩短浸泡时间，是防止分子筛粉化的首要问题。脱水后应进行严格的活化，进水后不要翻动已进水的分子筛，脱水时气

量不能过大，以减少气体的冲刷，处理得好是可以重新使用的。

# 五、事故教训与防范措施

这次严重进水事故，虽然分子筛没有报废，空分设备的生产无严重后果。但是，由于进行事故处理及加热、干燥设备，浪费了大量的电能及人力，造成了一定的经济损失。如果本次事故是正常生产中发生的，后果不堪设想。一是空分造成波动影响正常的生产，二是将会使设备冻坏。

为防止类似事故的发生，在生产中必须做到：

1. 空分设备开车前，必须对各种报警联锁装置进行确认，无任何问题时，方可进行开车。

2. 操作中加减量要缓慢进行, 防止系统压力波动过大; 每进行一项操作要进行确认, 无问题后, 方可实施下步操作。

3. 装置加热中就应将各联锁投入工作。

4. 设备启动操作中要增加巡检次数，并做到随时观察各参数变化情况，发现异常及时查明原因处理。

5. 加强技术培训，提高操作技能，增强对异常现象的应变能力。

（1995年10月完稿，11月2日收稿）

# 液氧充装站爆炸，死亡1人伤4人

1995年12月7日21:05，宁波市某气体公司刚试充装一个月的液氧充装站发生燃烧爆炸，现场5人，死亡1人，伤4人，其中2人重伤。充装间、瓶库全部倒塌。

# $\mathrm{CO}_{2}$  影响主冷换热的处理及其特性分析

# 常兴禄

（北京燕山石化公司化纤地毯厂空分车间，北京燕山双泉路1号，102502）

摘要 本文介绍了  $\mathrm{CO}_{2}$  富集于主冷而产生的工况变化。分析了因切换换热器泄漏，造成大量  $\mathrm{CO}_{2}$  带入装置内的过程。指出了主冷局部加温，解决  $\mathrm{CO}_{2}$  影响主冷换热效率应注意的问题。对空分装置内  $\mathrm{CO}_{2}$  深冷霜的特性及其危害谈了几点看法。表1参2。

主题词：中型空气分离设备 切换换热器 泄漏 二氧化碳 深冷霜 处理分析

# 一、情况介绍

我厂KDON-3200/3200-4型空分设备运行至1993年5月22日21:00时，大量的珠光砂从污氮消音坑放空口处飞出，装置被迫停车，扒珠光砂后检查发现  $4^{\#}$  切换换热器的空气入口管道上，有一个  $250 \times 150 \mathrm{~mm}$  椭圆形的大洞，  $3^{\#}$  、  $4^{\#}$  切换换热器的热端封头靠边缘处均被珠光砂摩擦破损，污氮消音坑内充满了珠光砂。

切换换热器修复后，装置于1993年7月20日开车，22日出合格氧、氮气，27日投入氩气生产。运行到8月初时，觉察到上下塔阻力都有所下降，但氧、氮、氩均在生产，只是粗氩纯度不十分稳定。到九月份时，上下塔阻力进一步下降(见表)，上塔压力比正常时明显下降，上塔底部压力由正常时的0.049MPa下降至0.040MPa，氩气不能维持正常生产，氧气产量逐日下降，纯度不易调节，主冷液氧需要经常排放（以下称上述现象为“现象一”）。

当时经过对系统工况分析后认为，“现象一”是主冷换热不好，其原因有：(1)切换换热器冷端温度偏高  $(-162\sim -167^{\circ}C)$  ，  $\mathrm{CO}_{2}$  自清除不足，带入系统的  $\mathrm{CO}_{2}$  最后富集于主

冷。(2)空压机入口过滤器（卷帘式）过滤效果差。(3)由于5月22日  $3^{*}$  、  $4^{*}$  切换换热器泄漏，珠光砂带入系统。

鉴于此种分析，我们采取了缩短切换时间、适当提高下塔压力、更换空压机入口过滤布、一次加大液氧排放量等措施，但效果不佳。不得已装置于1993年10月25日停车，排放掉主冷内的全部液体，对主冷进行了局部加温。

加温方法是：打开主冷氮侧排放阀PF-114，关闭下塔回流液阀DX-110，打开下塔顶部加温阀SW-107，通入氮气加温；加温至主冷指示温度  $-70^{\circ}\mathrm{C}$  时，又决定对主冷氧侧加温。加温路线为：打开循环吸附器加温阀SW-105，打开循环吸附器出口阀、关入口阀，加温气从主冷底部排放阀PF-113处排出，通氮气后，主冷排放阀PF-113不通，而加温管入吸附器管线已热，这说明加温气已进入到主冷顶部，考虑到加温的目的是主冷本体，于是暂时关闭吸附器加温阀，反吹！主冷排放阀PF-113。在反吹过程中，曾几次突然跑出白色冰状碎块，但瞬间汽化，这说明碎块是  $\mathrm{CO}_{2}$  结晶体。主冷排放阀PF-113反吹通后，对主冷氧侧进行加温。

当主冷温度指示到  $20^{\circ} \mathrm{C}$  时，停止主冷氧、氮侧加温，随即启动膨胀机开车，当主冷液面上到  $2700 \mathrm{~mmH}_{2} \mathrm{O}$ ，逐渐将下塔回流液阀DX-110开启时，上塔超压。由于污氮阀DX-107已全开，用氧、氮气取出阀控制压力，发现无效，只好又将DX-110关闭，同时叫来仪表工检查氧、氮气取出阀。经检查氮气取出阀不动作，修复后可正常调节；而氧气取出阀DX-106阀杆动作，但取出量很小。开始怀疑是阀板不动作，后在氧气出切换换热器四个蝶阀DX-101～DX-104前的四个小分析取样阀处感觉气量也很小，此时才注意到是冷箱内部的氧气系统堵塞。

从氧液化器前的排放阀PF-102感觉气量很大，说明主冷至氧液化器前的管路无问题。从实际情况来看，8个换热器氧通道同时堵塞的可能性不大，而管道  $\phi 206\times 3$  堵塞的可能性很小，最后认定是氧液化器堵塞。

这时我们又将装置停下来，将压力为0.6MPa的氮气导入到氧气常压球罐（此罐可承受2.5MPa），用氧气取出阀及放空三通阀(QD-111、-112)，反复多次反吹氧液化器。

氧液化器反吹通后，重新启动膨胀机开车，半小时后氮气纯度合格，但氧气纯度几

小时后仍只有  $97\% \mathrm{O}_2$  左右。

当时工况是：下塔压力  $0.49\mathrm{MPa}$ ，上塔底部压力  $0.055\mathrm{MPa}$ ，下塔回流液阀DX-110只有  $50\%$  开度，当开度增大时，上塔底部超压  $(>0.06\mathrm{MPa})$ ，而上塔顶部压力不高，一直在  $0.018\sim 0.020\mathrm{MPa}$ ，上塔中部阻力偏高（见表），大约30小时后氧气纯度才合格，氩气也恢复生产。

装置运行到11月中旬，前面提到的“现象一”再次发生，由于氧气量逐日减少，氩气无法生产，装置于12月1日全面停车，进行全面系统的加温。

当系统加温完毕，在对10个切换阀进行试漏时，发现  $4^{\#}$  切换换热器有内漏问题（判断过程略）。由于此换热器已堵塞十几个通道，因此直接将此换热器更换掉，更换后，装置开车至今运行正常。

区别之一是：主冷泄漏使得上塔各部阻力、底部与顶部压力均比正常时高，而  $\mathrm{CO}_{2}$  深冷霜仅使堵塞塔板以下的阻力、压力升高，上塔顶部的阻力及压力比正常时反而偏低。

区别之二是：主冷泄漏使得氧气纯度降低，但不随氧气取出量而有明显变化。这是由于泄漏量一定，氧气取出量大时，泄漏成分在氧气取出量中所占比例，有所减小的原故。而  $\mathrm{CO}_{2}$  深冷霜造成的氧气纯度不合格，是由于精馏不正常而引起的，因此氧气纯度随氧气取出量有明显的变化。

# 三、几点看法

（1）在找出（消除）带入主冷  $\mathrm{CO}_{2}$  的原因后，采取主冷局部加温是可行的。但要注意在加温前关闭氧气、氮气、粗氩、污氮取出阀，采取主冷氧氮侧同时加温，以便使升华的  $\mathrm{CO}_{2}$  、有机物等能及时带出装置外，避免 $\mathrm{CO}_{2}$  重新冻结在液化器及塔板上。  
（2）在换热器上新形成的  $\mathrm{CO}_{2}$  深冷霜的附着力是有限的，当  $\mathrm{CO}_{2}$  深冷霜所造成的阻力大到一定程度时， $\mathrm{CO}_{2}$  深冷霜将被正流压力空气冲刷掉；而不是传统上所讲的  $\mathrm{CO}_{2}$  深冷霜越积越厚，最终将通道堵塞[1]。这一点可通过  $4^{\#}$  切换换热器泄漏后出现的“现象一”，以及主冷局部加温后开车出现的上塔底部压力高、中部阻力高、氧气纯度不稳定，在经过24小时后又自然恢复正常来证实。  
（3）富集于主冷内的  $\mathrm{CO}_{2}$  深冷霜，不仅是悬浮于主冷液氧液面上，或是溶浸于液氧内，而是有一部分  $\mathrm{CO}_{2}$  深冷霜附着于主冷换热通道表面上。这一点可从主冷在短期内换热效率大大降低，反复多次大量排液氧工况不见好转可以为证。  
（4）经验表明， $\mathrm{CO}_{2}$  深冷霜在通过排放阀、节流阀或换热器通道时，如果不是连续进行，一旦停留一段时间，则往往会造成阀门、通道或U形管处的堵塞。如果堆积的  $\mathrm{CO}_{2}$  深冷霜溶化后再形成的大的规整结晶，则结晶强度比原来大得多。上面主冷局部加温时，所产生的主冷氧侧排放阀堵塞、氧液化器通道堵塞，就属此类问题。  
（5） $\mathrm{CO}_{2}$  对空分装置的危害不亚于碳氢化合物。碳氢化合物对空分装置的危害引起了众多人的注意，特别是对乙炔更需防范。 $\mathrm{CO}_{2}$  对空分装置的危害可归纳为两点：一是由于  $\mathrm{CO}_{2}$  深冷霜附着于器壁，造成换热效果大幅度下降，阻力增加，堵塞管道阀门，缩短装置的运行周期；二是由于  $\mathrm{CO}_{2}$  形成的雪花状态深冷霜，具有极强的吸附作用[2]。当

$\mathrm{CO}_{2}$  形成的雪花状态深冷霜富集成较大的颗粒时，对碳氢化合物的吸附浓缩作用是极强的。这种现象有助于碳氢化合物在主冷内爆炸机理的研究，这一点应引起同行们的注意。

主题词：送氯方法 氧氮置换 均压送氧 适用范围 操作要点 送氧程序安全可靠

大型冶金企业的初始投运及生产运行、年修期间停送氧气的安全问题至关重要。

国内外大型冶金联合企业在送氧过程中，因燃烧而造成重大人身伤亡、设备及管路烧损的恶性事故，不胜枚举。冶金部和劳动部曾多次对这类事故发生通报。笔者因工作需要，曾多次主持指挥易燃、易爆或助燃介质的大停大送，对氧气的输送与投运，总结出“氧、空气置换，均压送氧”、“氧氮置换，均压送氧”的安全可靠送氧法。

本文仅对笔者总结的“氧氮置换、均压送氧”法做详细阐述。

# 一、“氮氮置换，均压送氧”法的适用范围

1. 输氧系统的初始投运阶段送氧。工程安装完毕，制氧机组按照施工验收规范及调试规程完成单机空载试车、负荷试车及联动负荷试车正常后第一次向用户厂输送氧气。  
2. 年修（或大修）期间，无论因氧气站（厂）全停机、用户厂因技术改造或检修、对用氧设施或管道改动等而引起的氧气放散，致

使管网内压力回零，再次投运送氧时。

3. 按规程对运行氧气管网定期（一般为管网内充氧18个月至24个月）吹扫管道后，再次向用户厂送氧时。

如果因企业定修期间停供氧气，或因工艺原因停用氧气、用户厂的用氧设施及管道没有改造，此时可关闭各用户点阀门，保证管网内压力为正压(不回零)，这种情况不必采取“氧氮置换，均压送氧”法，可直接开机升压送氧。

# 二、送氧前的必要技术准备

1. 必须确认系统及设施的施工记录、脱油脱脂记录、管网吹除记录的可靠性。  
2. 管道焊口应采用氩弧焊打底。  
3. 管道及设施与氧接触部分，应严格按施工规范脱油脱脂。  
4. 检查并确认所有法兰联接的静电跨接线完备与牢固；管网有可靠的接地。  
5. 检查并确认管网的吹除打靶记录，依据工程实际情况可以分段吹扫。如果氧气管道安装完毕，吹扫后经过半年还未投运，那

么应在送氧前重新吹扫打靶认证。

上述的检查均应有质检部门或质检人员参加并确认。

6. 检查确认输氧管网的材质符合设计与防火要求。  
7. 绘制输氧系统简图，现场逐一确认设备、管网的完好状况。应对每路氧气管路的设计工作流量、工作压力确认，并确定用户厂的放散点。  
8. 编制送氧方案。

# 三、“氟氮置换，均压送氧”法要点

此法与传统送氧方法不同。传统送氧法是采用用户厂各用户点阀门关闭；放散阀打开；入用户厂的阀门打开，由氧气站开启快速切断阀或手动阀送氧。其最大的弊端是：送氧流速掌握不住，出现超速送氧；管线上各阀门出现严重节流、静电超压，严重时阀体出现节流引起的降温；管线内氧气压力瞬间急骤变化。上述三点是静电起火燃烧的诱导条件，如果在管道内存有异物，如金属焊渣或安装后残留的焊条头及棉纱团或锈蚀的粉末，这将满足燃烧的充分且必要的条件，导致事故瞬间发生，不可收拾。因此，传统送氧方法是不科学、不安全的。

“氯氮置换，均压送氯”法的要点如下：

1. 用氧压机向氧气管道内压送氮气，并使网压稳定恒压

送氮气的过程可结合扫线进行。使网压不超过  $0.8\mathrm{MPa}$ ，一般可控制在  $0.4\sim 0.8$  MPa之间的某一压力值恒定维持。此时将先打开用户厂的此管线上的放散阀。

恒定压力值的确定取决于氧压机的氮气压出量及管线上用户厂放散点的放散量。在编制送氧方案时，应依据上述两量初定恒压值，在实施过程中依据实际情况进行调整（建议取  $0.7\mathrm{MPa}$ ）。如图1所示。

2. 管线上各阀门的开闭状态

送氧过程中，管线沿线不存在开启阀门

![](images/c422724f6139929e3080a23ad13a76c61f7d11afff8a55dc3ee063204b1efa58.jpg)  
图1 用氧压机向氧气管道压送氮气

的操作。

在实施压送氮气时，各用户点阀门应关闭，送氧全线各阀门全开，用户厂放散点打开。包括氧气站内调压间的气动调压阀及出站前的快速切断阀及沿管线的所有阀门、入用户厂房的总控制阀，都一一打开，所以全线是畅通的。唯有用户点阀门关闭（如氧枪前的阀台关闭、切割嘴前阀台关闭），只靠放散阀放散。

因此，送氧过程中不存在开启阀门的操作。

3. 氮气置换，均压送氧

当压送氮气实施完成后，即氧气管网内氮气压力恒定在  $0.4\sim 0.8\mathrm{MPa}$  某一值时（例如稳压0.7MPa)，复查用户厂各放散点氮气放散情况，每个放散点应设专人值守，注意周围不得有火源，在放散氮气时应注意避免离近直接吸入而窒息。

送氧全过程中设指挥1人（地点在氧气站）、副指挥1人（地点在用户厂）。

副指挥认定用户厂氮气放散情况及各放散点专人值守情况完备后，通过有效的不间断的通讯(步话机或专线调度电话)，通知指挥准备完毕并要求送氧。

指挥复查氧气站机组运行状态及站内管网阀门状态确认后，并同时落实用户厂各阀门开闭状态无误，向氧气厂及用户厂下达送氧令。

此时由1名（或2名）送氧操作工进行氧

压机吸入口的氧氮阀门恒压倒气操作，氧气站主控室监视压力恒定及变化情况。

操作要点是：

# （1）恒压稳定送出

监视并控制氧压机送出压力，始终恒定 $(0.4\sim 0.8\mathrm{MPa}$  之间，如  $0.7\mathrm{MPa})$

# （2）同步倒阀，慢开慢关

同时关闭氮气入口阀；开启氧气入口阀，操作应稳而慢，一边慢慢关闭，一边同时慢慢开启，掌握在10分钟倒阀完毕。操作速度为每分钟  $3\sim 4$  圈扣。

如1人操作，因该两阀门距离很近，可先关氮阀2圈，再开氧阀2圈，循序进行，按上述时间完成吸入口氧氮阀门的恒压倒阀操作。

在倒阀过程中，监视氧压机出口压力及管网压力恒压稳定。

在倒阀过程中，如发现送出网压突然急骤下降时，应立即遥控关闭氧气站内快速切断阀，停止送氧。站内氧气管线上的自动放散阀应投自动。

倒阀过程中，指挥与副指挥应用有效通讯工具保持不间断的联系。

倒阀完成后，立即通知在用户厂的副指挥“倒阀完毕”。

倒阀开始及过程中，副指挥应随时密切掌握各放散点的放散情况，并向指挥汇报。

放散点的专人值守者，应离放散点  $2.5\sim$  3米，不应穿钉鞋，外衣应为防静电服。

放散点周围30米内，不得有明火及闲杂人等。

倒阀完毕后，注意仍应恒压送氧，并立即由化验人员（应已在该区域待命）现场逐一化验各主要放散点的气体纯度，当纯度为 $99.6\% \sim 99.8\%$  时，认为合格。

# (3) 关闭用户厂的放散阀

氧气纯度合格后，总指挥下令关闭放散阀，要领是：

各放散点值守操作者应慢慢地关闭。依据放散阀口径不同而时间不同，一般应每关  $2\sim 3$  圈稍停  $5\sim 10$  秒，每个放散阀在3分钟左右完成关闭过程。

在关闭放散阀的过程中，网压出现缓慢上升是正常的。

放散阀全部关闭后，指挥与副指挥认定各自区域无误，可将网压提升，因此时是封闭管网，一般在  $30\sim 40$  分钟内即可升压至设计工作压力。一般工艺用氧网压为1.5MPa；切焊用氧网压为1.0MPa。至此完成送氧过程，指挥确认氧气站、外网、用户厂管线及用氧设施无异常后，即可宣布“送氧完成”。

# 对江阴商检局氧气瓶爆炸事故的分析

庄胜强

（上海梅山冶金公司炼钢厂，南京中华门外，210039）

1995年8月28日3:25，江阴商检局三楼金属实验室内正在使用的氧气瓶发生了激烈的爆炸。这次爆炸使20多万美元的进口仪器完全毁坏，三间  $100\mathrm{m}^2$  的实验室墙体遭到严重破坏，三楼西半层的门窗玻璃全部震碎，仪器、设备等直接经济损失约350万元左右。该气瓶至少被炸成19块，拣回18块，总重 $56.422\mathrm{kg}$  ，比该气瓶原重  $58.2\mathrm{kg}$  少1.778kg。18块碎片中，最重的为  $19.5\mathrm{kg}$  ，最轻的仅  $0.03\mathrm{kg}$  ，小于  $1\mathrm{kg}$  的碎片至少有10块，其爆炸威力极大；在实验室的3名工作人员2人重伤（其中1名为美国Leco公司职员）、1人轻伤。

# 一、事故经过

1995年8月28日下午3:05，美国Leco（仪器）公司职员丁  $\times x$  及江阴商检局职工朱 $\times x$  、徐  $\times x$  等三人进入三楼金属实验室，对从美国Leco公司引进的CS-444型碳硫测定仪，进行调试前的准备。在准备工作时，打开高纯氧气（气体生产厂称该氧气纯度为 $99.99\%$  )气瓶瓶阀及减压调节阀后，于3:25左右开CS-444碳硫测定仪电磁阀向仪器内送氧，其后  $3\sim 4$  秒钟，瞬即在用的氧气瓶发生了激烈的爆炸。爆炸现场一片狼籍，倾刻间  $100\mathrm{m}^2$  的金属实验室及其试验仪器遭到了严重损坏或完全毁坏。

# 二、事故性质的确定

# 现场发现：

① 该气瓶爆炸威力特别巨大，造成了实验室墙体倒塌，仪器设备严重毁坏，其爆炸威力远大于气瓶的物理性爆炸威力。

② 该气瓶至少被炸成19块，其中小于  $1\mathrm{kg}$  的碎片至少10块，拣回最小的碎片重量仅  $0.03\mathrm{kg}$ ，爆炸所释放出来的能量，远比多数化学性爆炸气瓶所释放的能量还要大。  
③ 气瓶部分碎片剖口金属尖角被烧熔，部分碎片有被高温火焰烧过的痕迹。  
④ 气瓶瓶阀爆破片熔尽，Leco公司随机带来的减压调节阀的安全阀滤网烧熔了一个洞，且其低压压力表内有炭黑。  
⑤ 爆炸是发生在接通CS-444碳硫测定仪电磁阀  $3\sim 4$  秒后发生的，按下电磁阀按钮后“氧气”立即进入温度达  $600^{\circ}\mathrm{C}$  的氧气净化室，存在点燃爆鸣性气体的客观条件。

鉴于以上事实，可十分肯定爆炸前甚至爆炸时，气瓶内发生过激烈燃烧，产生了高温、高压而导致该气瓶爆炸，即该气瓶化学性爆炸的性质是无可置疑的。

# 三、爆炸原因分析

有人曾认为：这次爆炸是由于油脂引起的。其中有人认为减压调节阀内有油脂，也有人认为瓶内有油脂。甚至也有人提出高纯氧会发生燃烧爆炸。本人作为特邀专家，到场后即否定了上述观点，其理由如下：

1. 众所周知, 氧气最重要的性质之一是助燃而不能自燃, 所谓高纯氧会自燃的说法是毫无根据的。  
2. 关于该气瓶系因油脂而引起爆炸，也是违背氧气的基本性质的。高压氧气（ $\geqslant 3$  MPa）与油脂接触即发生激烈的氧化放热反应而导致油脂激烈燃烧。由氧气性质可知，如氧气瓶内、阀内有油脂，其燃烧爆炸只能

发生在氧气瓶切换充装的瞬间，或充装过程中，它既不可能到充装结束，更不可能在气瓶充装一年半后发生燃烧、爆炸。由此可以排除该气瓶系由油脂燃烧而引起爆炸的可能性。那么，减压调节阀低压表内存在少许炭黑又该怎样认识呢？应该说减压调节阀低压压力表内脱脂是不够严格的，因而在高温燃烧时留下了炭黑。但它不是引起这次燃烧爆炸的可燃物。又鉴于气瓶碎片内无炭黑，因此可燃物也不可能是乙炔等碳氢化合物。因为如是乙炔，其量也不可能很多，故没有这么大的爆炸威力。

3. 该爆炸气瓶质量，与这次爆炸无直接关系。对气瓶碎片检查未发现该爆炸气瓶存在明显的质量问题。根据该气瓶爆炸形成碎片的情况及爆炸现场破坏情况，参照以往大量事故实例估算，该气瓶爆炸威力相当于瓶内氧气中含氢量  $\geqslant 20\%$  。因此，气瓶质量再好，在该条件下气瓶爆炸也是不可避免的。

# 4. 氢、氧混合是这次爆炸的根本原因

该爆炸气瓶系无锡（现锡山市）科学气体产品公司于1994年6月3日送货上门的“高纯氧”气瓶，标明氧气纯度  $\geqslant 99.99\%$  。

如该气瓶内的氧纯度确实  $\geqslant 99.99\%$  ，则该气瓶发生化学性爆炸是不可能的。然而爆炸威力等表明，该气瓶确系化学性爆炸，瓶内的氧纯度远未达到  $99.99\%$  。又据对无锡科学气体产品公司的调查，生产厂气瓶管理极不严格，有许多环节对产品纯度、气体质量失控。充装前仅检查是否过期（事实上过期瓶也充气）、有无余压。有余压的则不抽真空，即充“高纯氧”。氧纯度既无在线检测，充装后也不逐瓶检测。据该厂负责人介绍，每批高纯氧气瓶抽检  $2\sim 5$  瓶瓶内的水分指标。我们到该厂调查时，该项检测也未进行，所以这些高纯氧也只能是所谓的  $99.99\%$  了，事实上谁也说不清它的纯度到底是多少。

该厂生产的“高纯氧”，是由水电解的普

氧，经膜压机压缩至  $14\mathrm{MPa}$  后，进入纯化器，由钯触媒的作用，使  $\mathrm{H}_{2}$  、  $\mathrm{C}_{\mathrm{n}}\mathrm{H}_{\mathrm{m}}$  与  $\mathrm{O}_{2}$  反应生成水等，然后进入干燥器吸附水分、二氧化碳等，接着进入膜压机二级压缩到14.0MPa充瓶。由此可见，如果水电解产生的氧中含氢量过高，则工艺流程中的转化器、催化剂等均有可能烧坏。调查该厂未发生过上述情况，普氧充装也未发现问题，故基本上可排除原料普氧含氢过高的隐患。

鉴于上述分析，哪所谓“高纯氧气”瓶内的氢气是怎样混入的呢？按该厂的生产工艺和膜压机容量  $(5\mathrm{m}^3 /\mathrm{h})$  ，该气瓶内的氢气只可能是该气瓶先充过氢气，然后又来充高纯氧，从而氢氧混合形成爆鸣性气体。又因每次只充一只气瓶，所以不存在同批其他气瓶内形成爆鸣性气体的问题。

# 5. 燃烧爆炸的引燃引爆能量

在美方Leco公司试验人员打开CS-444碳硫测定仪上的电磁阀后，该“氧气”即以3.2升/分的速度通过电磁阀，进入温度为  $600\%$  的氧气净化器。当该“氧气”（实质为氢氧爆鸣性气体）进入氧气净化器后，爆鸣性气体中的氢气即被点燃（氢气在常压大气中的着火温度为  $585\%$ ）。鉴于可燃物燃点随其环境氧气压力的提高而降低的性质，故在  $0.28\mathrm{MPa}$  （该净化器中的氧气压力）氧气气氛中，氢气的着火温度肯定  $< 585\%$  。当氢气在氧气中的燃烧传播速度大于爆鸣性气体从气瓶中向外流动的速度时，点燃的爆鸣性气体就会越过仪器上的电磁阀、气瓶上的减压调节阀以及气瓶瓶阀而点燃瓶内的爆鸣性气体，从而导致了该气瓶的爆炸。

综上所述，该气瓶爆炸前的瞬间，瓶内同时满足了燃烧爆炸的三要素，即同时具有足够的可燃物、足够的助燃物及足够的引燃引爆能量，因而该气瓶的化学性爆炸就变得不可避免了。

（1995年11月）

# 一、目的意义

空气分离设备产品由小到大，结构复杂，其性能不一，产品品种、形式、规格繁多。成套空气分离设备中，包括了一系列的配套机组（单机），如气化器和气化设备、贮气柜、加热器、预冷器、干燥器、灌充器等。为此，编制空气分离设备产品型号代号，使成套空气分离设备与其配套机组(单机)都有一个相对应的型号代号，能有效地为设计、生产、用户订货选型、企业科学管理提供方便，并对于社会技术交流，广泛传播信息媒介都必不可少。尤其是，对于产品图样、产品技术文件、产品样本，以及有关的教材编写等，都有普遍的使用价值。

# 二、编制原则

空气分离设备产品型号、代号的编制原则，力求实用、简洁、直观，不求多、求全，便于编制，好懂易记。考虑生产实际情况，型号着重反映产品主要结构特征、参数等。

# 三、型号编制

1. 型号中大写汉语拼音字母的第一个（或第一、二个）字母，表示产品类别。  
2. 型号中大写汉语拼音字母的第二个字母，表示结构特点、流程特点等。  
3. 型号中大写汉语拼音字母的最后一

个字母，在复杂产品上表示介质化学元素符号（系采用国际化学元素符号）。

4. 空气分离设备产品复杂, 其变型设计范围, 按结构特点和参数与原产品型号有差异时, 采用变型设计顺序号来区别, 即用罗马数字 I、II、III……。  
5. 产品型号中的气体产量，均为标准状态（ $0^{\circ}\mathrm{C}$ ， $101.325\mathrm{kPa}$ ）下的产量。产品型号中的压力，均为表压力。

# 一、矿井防灭火的必要性

矿井火灾是煤矿生产中的重大灾害之一，严重威胁着煤矿的安全和矿工的生命。矿井火灾根据引火热源不同，分为外源火灾和内源火灾两类。据国外有关资料统计，矿井火灾有  $80\%$  以上为自然火灾。在生产过程中残留在采空区的煤炭，接触和吸收了大气中的氧后，自身氧化不断加速，当其氧化速度加快到氧化生成的热量不能及时散发而集聚使温度达到燃点时，就导致了自燃。根据煤的炭化程度的物化性能，煤的自燃倾向不尽相同，我国近  $50\%$  的煤矿有自然发火的危险性，每年发火达二三百次。尤其是从70年代后期，我国煤矿机械化程度越来越高，综采工作面大量增加，工作面的单产急剧增加，矿井产量集中在少数工作面，而工作面的采煤设备资产又高达百万元至数千万元，这种采区若遇火灾，特别是特厚煤层和近距离煤层的开采，往往会出现煤自燃火灾，造成极大的损失。目前我国这类煤矿的产量已达全国重点煤矿产量的  $30\%$  以上，因此防灭火就成为煤矿安全生产的重大课题。

# 二、煤矿氮气防灭火

煤矿的防灭火有许多方法和措施，其中

氮气防灭火是一项行之有效的技术。氮气之所以能防灭火，是由于煤炭自燃是剧烈的氧化现象，必须有足够氧气维持燃烧。实验证明，在其氧气浓度低于  $3\%$  的空气环境里，煤炭燃烧就不能维持。如果向采空区注入氮气，就改变了采空区的供氧条件，随着氮气含量的增加，会使氧气浓度相应降低，能够延缓煤炭氧化自燃过程，可抑制采空区自然发火；对于已经自燃的采空区，可以起到惰化火区和阻爆的作用，这适应于高瓦斯矿井的防灭火。

这项技术，由于氮源丰富可靠，工艺技术简便，不污染矿井环境，既可扑明火，也可控制自然发火，且灭火速度快、效率好，在世界上一些主要产煤国家早已应用。如欧洲共同体产煤国在70年代已成功地将液氮用于煤矿防灭火，解救了大量综采工作面免受火灾危险。仅前联邦德国用这项技术就扑灭了近40起矿井火灾。80年代，澳大利亚的一些煤矿也成功地应用了这项技术。

液氮汽化法防火，一般用在这项技术开发的初期，它设备庞大，最难解决的是储运。液氮运输需要液体槽车，从较大的空分厂运至发火区，往往因路途遥远和时间拖延而不

能满足需要；另一个不利因素是气体装置的运输和安全问题。虽然一些国家在技术上已解决了液氮的输送、气化及灌注，但在实际应用中，用液氮防灭火仍有许多不利推广的因素。最重要的一条就是它费用高，因而一些主要产煤国家在80年代初就将氮源由液态氮转向气态氮。如前联邦德国开发的GAN-800、GAN-1200型等移动式制氮装置，其产气量分别为  $800,1200\mathrm{m}^3 /\mathrm{h}$  ，纯度为  $97\%$  。利用这种设备可快速行动到任何地点的矿井，并直接产出气态氮输入井下高度机械化采区防治火灾，从而确保矿井安全。目前，德国为防灭火氮气量已消耗  $5\times 10^{6}\mathrm{m}^{3}$  以上。

我国煤矿用氮气防灭火已有数年，在原能源部编制的《煤矿安全规程（1992年版）执行说明》第64条“综合机械化放顶煤采煤的安全措施”的第一项就是“氮气防火技术”，其中详细说明了氮气防火的机理及注氮的管理；《煤矿安全规程》第220条是“关于采空区注入氮气的要求”；第224条是“关于放顶煤综采工作面采空区注入惰性气体防火的要求”。我国煤矿用氮防灭火的经历与国际上相仿，即氮源由液态氮转向气态氮，供氮方式由固定式转向移动式。目前，在许多用氮防灭火的矿区中，主要是利用深冷空分设备制取液态氮或气态氮，为就近矿井服务，这种方法有天府矿务局(包括重庆地区)、抚顺矿务局、新疆乌鲁木齐矿务局等单位，在急倾斜、特厚煤层中应用，取得了可喜的效益。

然而，利用液氮防灭火，除成本高外，还有我国的矿区不具备大量集中供给液氮源的条件。若自产液氮，又因厂小液氮产量低，不能满足需要。目前国内外常用的制氮法，有深冷空分法和PSA两大类。深冷空分法制氮需贮存罐和气化器，工艺复杂，通俗地说一套制氮装置实际上就是一小型的空分制氮厂，虽然解决了储存、运输等问题，但弊病不少。首先是投资大，包括大量土建工程及

重型设备；再是，启动慢，操作复杂，一般需要  $12\sim 24$  小时才能提供成品氮气，需要经专门培训的人才能操作；还有一点就是由操作复杂带来的安全与可靠系数低。在国内煤炭系统，现已趋向用变压吸附(PSA)法代替深冷空分法。

# 三、PSA制氮技术

PSA制氮技术，作为新兴的空气分离技术而引人注目，国内外新兴的空气分离方法——非低温空分工艺正在异军突起，其中之一就是变压吸附法，即PSA法。它是以压力为循环变量对气体混合物进行分离，在过程中气体混合物的压力作周期性变化，吸附剂在压力作用下吸附某种气体，卸压后进行解析，同时吸附剂得到再生。

由于PSA法具有工艺简单、设备少、投资省、操作方便和产品气的纯度容易调节等优点，很受用户的欢迎。尤其是随着PSA法技术的不断成熟，必将对传统的深冷法制氨提出挑战，乃至在某些范围内取而代之。

在PSA制氮装置吸附塔内，通常使用4A碳分子筛。碳分子筛分离空气制氮的机理，不是吸附剂固有的选择吸附性或对分子的选择排斥性，而是利用空气中氧氮在吸附剂颗粒内部的扩散性差别，即依靠氮、氧分子在碳分子筛微小孔穴内扩散率的不同而分离氮和氧的。氧的吸附要比氮快得多，由于氧分子的气体动力直径(2.8G)，小于氮分子的气体动力直径(3.0G)，所以，当空气通过碳分子筛时，氧分子很快被吸附在碳分子筛的微孔表面，而氮分子则通过碳分子筛，利用切换阀门对吸附塔进行反复切换，即可连续制取氮气。

# 四、PSA制氮装置配套于煤矿防灭火系统

在我国，把PSA制氮装置应用于煤矿防灭火系统已有先例。1992年在西山矿务局杜儿坪矿，就利用杭州制氧机厂制造的PSA制

氮设备进行过模拟火灾注氮灭火试验，其试验结论是：“用注氮方法可以极快地抑制或控制住火灾的发展，达到快速扑灭火灾的目的”，“用氮快速抑制火势，为缩小火区范围提供了可能性，为采取进一步直接灭火创造了条件”。在杜儿坪矿19110综采工作面进行了注氮防火试验，其结论为：“向开采工作面本层采空区注入氮气，能防止上层老火区复燃和本层采空区的自燃，能够保障工作面的安全回采和顺利撤架。氮气防火，是近距离复合煤层行之有效的防火方法”。

虽然PSA制氮装置用于煤矿防灭火系统，在我国前几年就有厂家进行过工作，但由于种种原因没有普遍得到应用。1995年我公司在这方面作了深入细致的工作，并得到煤炭科学研究总院重庆分院及煤炭部综采放顶煤中心的大力支持，于1995年3月同甘肃省靖远矿务局魏家地煤矿(简称魏矿)签定了PSA-500型制氮设备技术合作合同，并于该月正式执行。合同规定，该套装置是我公司在魏矿现有空气压缩机基础上的技术配套产品，该设备要求达到的合同技术指标为：

产氮量：  $500\mathrm{m}^3 /\mathrm{h}$

氮气纯度：  $97\%$

出口压力：0.6 MPa

魏家地煤矿与我公司自1995年3月签订PSA-500型制氮装置技术合作合同并正式执行后，经过双方努力合作，该套设备已于1995年7月完成安装调试工作。在调试阶段，煤炭工业部、省煤炭工业管理局、矿务局等单位的领导在现场进行了检查、指导，并给予了较高的评价。

PSA-500型制氮装置先后进行了近200小时的运行，其中连续三次每次连续工作10个小时以上，设备状态保持良好，各项技术指标均已达到或超过合同规定的技术要求：

平均流量

纯度

$530\sim 540\mathrm{m}^3 /\mathrm{h}$

98%以上

平均出口压力 0.6 MPa 以上

经过调试运行，整套设备已具备了综合放顶煤技措项目工业试验运行的能力。

PSA-500型制氮装置，采用的是目前技术比较成熟的变压吸附制氮工艺，该项技术已获国家专利(专利号：92236007.3)。设备中选用了碳分子筛作为吸附介质，利用空气中氧、氮分子在碳分子筛微孔内扩散率的不同而将氮气分离出来。该设备配置了两个吸附塔轮流工作，依靠阀门系统有序的切换得到连续、平稳的氮气气流。

PSA-500型制氮装置的特点：

# 第一，启动快

采用传统的深冷制氮方法，设备启动时间最少都在8小时以上，要得到合格产品的时间更长，而该套装置在25分钟之内即可输出稳定合格的氮气，能应急煤矿系统防灭火工作，特别适于突发性火灾事故的防治。

# 第二，可以随时停机并再次启动

深冷制氮设备不宜随时停机，停机时间过长，则恢复运行困难，再次启动的时间大约需要24小时左右；而PSA-500型装置，不但可以象深冷机组一样长时间连续运行，而且可以随时停机，并在任意时刻再次启动，再次启动所花费的时间，比首次启动时间还要短。

# 第三，运行稳定，操作人员少

PSA-500型制氮装置，经操作人员启动运行后，便能稳定可靠的工作，无需再进行专门操作，全套设备每班只需1人，或者指派人员定时巡视即可。而深冷机组，每班至少要有6个以上熟练的专职操作人员。

# 第四, 安装配套灵活方便, 对场地限制少

PSA-500型制氮装置，在设备造型、整体设计、整体布局上已是定型产品，由于魏矿方面提出要利用原有空压机（L5.5-40/8）进行配置，经过我公司技术人员的考察，对原系统压缩空气前期处理的流程进行了改进

——通过使用后冷器  $(\mathrm{AA} = 40 / 8)$  、高效除油器  $(\mathrm{QZC} = 40 / 8)$  及T级过滤器（T1700）等设备进行空气净化处理。这样既满足了工艺要求，为矿方节约了资金，又将闲置设备进行了合理利用。

在整套设备运至现场以后，根据矿方要求，针对厂房实际情况，我公司的技术人员对系统的整体布置又作了相应调整。利用现有空闲场地（实际占地面积不到  $30\mathrm{m}^2$ ），安装了制氮装置，保持了厂房内原有设备的布局，为矿上节省了设备拆除费用和基础费用，保证了其它工作的正常进行。

# 第五，对原料空气要求较低

膜分离技术作为新兴的制氮方法，有它独到的一面，但对原料空气的要求条件较高，却也是不容忽视的事实，这无形当中就增加了净化设备的投资及工艺技术的复杂程度。而PSA-500制氮装置的原料空气，是经过净化的压缩空气，目前所达到的指标已完全能够满足需要。

第六，维修、维护方便，使用寿命长

PSA-500型制氮装置的维护非常简单，只要按照设备使用说明书定时进行排污处理，定期检查阀门密封情况即可。该设备易损部件非常少，一般的检修人员即可完成维修工作。吸附过程是一个物理过程，制氮设备主要依靠的吸附介质，除有微量的损耗外，没有任何质的变化，不存在失效、过期等问题，可以长期使用。

第七，控制可靠，管理自动化

PSA-500型制氮装置，阀门切换装置电气控制系统，采用日本OMRON公司的工业PLC可编程序控制器，满足日本JIS质量标准。该控制器广泛应用于煤矿系统的洗煤工艺控制，并配备氧气传感器、压力传感器、流量传感器，可以测量氮气纯度、压力及流量的实时数据并进行控制，具有很强的抗干扰能力，平均无故障工作时间可达十几万小时，

准确有效地保证了切换装置的正常工作。

另外，PSA=500型制氮装置，配置了计算机数据管理、动态图形显示、打印系统等，计算机可对设备运行期间的参数，如压力、纯度、流量、累计流量等数据进行统计管理，并可打印多种形式的报表，还可以同矿上微机系统联网，并入魏矿整体管理系统。

# 第八，机械设备、材料国产化

PSA-500型制氮装置，加工空气净化处理、吸附组设备及切换阀全部实现国产化，其中吸附剂采用的是浙江省长兴化工厂生产的4A碳分子筛。值得一提的是，这套设备的切换阀采用了温州瑞气机电有限公司的国家专利产品(专利号：89.2.13676.6)ZSGP型管道式气动阀，我公司根据实际情况进行了相应的调整，满足了工艺要求。

# 五、存在问题

我公司生产的PSA-500型制氮装置仍为固定式机组，受距离和敷设管路长度限制，一般只能就近服务于一、二个采区或一、二个邻近的井口，不能满足防灭火地点的变迁，不能满足各矿偏远地点突发火灾的需要。鉴于这种情况，我公司在原有设备的基础上与重庆分院合作，不久我公司将推出适用于井下工作的移动式PSA制氮装置，以适应煤炭系统安全生产的需要。

# 六、结束语

PSA-500型制氮装置是我公司为煤矿防灭火系统提供的第一套设备，随着氮气在煤矿防灭火系统中的逐渐普及，我公司将不断改进技术，继续为煤矿系统服务，提供更多的优质设备。

# 工业氮、纯氮、高纯氮国家标准及六种标准气的研制通过审查

1995年10月26日至29日，全国气体标准化技术委员会、国际半导体设备和材料标准化委员会中国标准化委员会化学品分会一届二次联合会议，在四川省峨眉山市召开。此次会议审查并通过了修订的工业氮、纯氮、高纯氮国家标准及6种气体标准样品。全国各地前来参加此次会议的有关科研院所、高等院校和企事业单位的代表为气标委委员、通讯委员、化学品分会委员（含已上报待批的增补委员）、两标委秘书处人员、标准起草人、气体标样研制人员。峨眉山市技术监督局领导也应邀到会并作了讲话。会议代表共60人。

在会议进程中，与会代表听取了标准起草人对工业氮、纯氮、高纯氮3个国家标准修订的说明。即这三个标准的修订是根据化学工业部文件(1994)化督发第87号下达的标准制、修订计划任务和现行国家标准清理整顿的要求，对GB/T3864—83《工业用气态氮》、GB/T8979—88《纯氮》、GB/T8980—88《高纯氮》国家标准进行修订和合并。由于工业氮、纯氮、高纯氮是不同种产品，而它们又不是氮气的不同级别产品，所以，其生产工艺、产品适用范围、分析方法、产品包装要求等都差异甚大，因此，合并后的氮气系列标准，将其分为同一标准的三个独立部分：GB/T3864.1— $x\times x\times x$  工业氮、GB/T3864.2— $x\times x\times$  纯氮、GB/T3864.3-- $x\times x\times$  高纯氮。根据我国有关标准化文件和标准编写的规定，将原标准中的计量单位改为法定计量单位。如将压力单位  $\mathrm{kgf/cm^2}$  改为  $\mathrm{MPa}$ ，浓度ppm改为负指数  $10^{-6}$  等。再有，原工业用气态氮改名为工业氮，其适用范围中增加液态氮。……。

会议认为：经修订后的三个系列标准的标准指标分级合理，检验方法适用可靠，既向国际接轨，又兼顾了国内的生产实际，并达到了国际水平。其中工业氮的优等品还达到了国际先进水平。从而使

标准引入了竞争，有利于促进我国氮气产品质量的提高。

# 制氧工问题解答（二则）

# 1. 问：分馏塔的操作和调节的含义是什么？两者之间的区别何在？

答：空分设备的操作和调节，以分馏塔的操作和调节最为复杂，讨论它的真正含义及其区别所在，对于氧气站行政领导和操作工人来说，均具有普遍的意义。

操作工的首要任务就是能够按分馏塔基本原理及操作规程，把设备开动起来，使分馏塔进入连续运转并达到一定的要求，例如启动操作、停车操作、加温吹除、停车后再启动操作等等。操作是许多工步的集合，是有序的工步集合体，当然设备还要具备方便操作的条件，即可操作性，这是大家所知道的。分馏塔启动操作是冷却、液化、精馏等工步的综合，必须有秩序地进行，前后次序不能颠倒。而且有不同的操作要求，如气体压力的高低，液化速度的快慢，阀门开关的大小，先后、快慢，都有一定规定；缺少某一只阀门，会导致操作的失调，例如一个小型分馏塔流程中，规定热端温差必须控制在  $5\%$  以内，氧、氮两夹层之间的温差不得大于  $2\%$  可是空气进入两夹层的支管没有分别设置空气进口调节阀，那就是说主热交换器两夹层的热端温差  $5\%$  不能调节，那这个分馏塔在这一点的操作性就不佳，即使对热端温差有规定要求，其实是无法控制的。

操作工的另一个主要任务，是对分馏塔工况具有调节的本领。所谓调节，在某种意义来说也属于操作范畴，是深入的操作，如冷量平衡的调节，产品纯度和产量的调节，液空、液氮纯度及上下塔回流比调节等等。它没有一个固定的操作模式，是有序的既定

的工步的集合，不是单纯的建立液体或精馏过程，而是在保持冷量平衡的基础上，获得物料平衡和组分平衡，是通过破坏旧平衡建立新平衡的过程，说调节是操作的深入并不过分。

有不少分馏塔的操作工都说“我会操作分馏塔了”，可是拿他值班的记录本一看，温度、液面、纯度或产量等工况或个别工况很不稳定，很不理想，一眼便可看出这一班的工况没有调节好。因为人们说他会“操作”了，不等于说便有本领把产量和高压压力调节到最佳状态或遇到工况波动就能够把它扭转过来。同一个分馏塔，有的工人能够把高压压力调到最低(如1.7MPa)，而有的工人就办不到；设备的运转周期，有的分馏塔可以运转一年以上，有的只能运转2～3个月，甚至更短，除设备本身有毛病外，调节好坏是主要原因。如同人人都会炒菜，但不是每人都能把菜炒成色香味俱佳的道理基本一样。

自然科学与社会科学也有相通之处，我们说调节好分馏塔工况与现时常常谈的调控——宏观调控和微观调控的道理相类似，社会主义市场制度的建立过程，需要靠“宏观调控、微观搞活”来调节，这是时常见之报端的原则；分馏塔的调节也是如此，分馏塔启动后，初步达到液面稳定，开始送氧，但分馏系统干扰因素还很多，工况不够理想，有些指标达不到要求(未达到说明书要求)，故需要进行调节。

以上表明，调节方法是通过宏观和微观两方面来调控的，不采取宏观措施，听任自流，当然不行；采取一定的宏观调节办法，

才有可能得到预期的效果。我们说宏观调节是通过微观调控来实现的，宏观措施得当，工况将越来越好，这叫做“良性循环”；如果措施不当，即便精馏系统有自调能力，也会依微观自调机理，使工况越开越差，这叫做“恶性循环”。所以调节之前，必须对工况进行仔细调查分析，然后采取措施加以调节，若措施得当，则工况向人们预期目标前进；若措施不力或措施不当，则工况将向相反方向转化，甚至破坏工况，需要重新进行。这些道理，对有多年工作经历的操作工，必有体会。这又说明，分馏塔的调节比启动、停车、加温、吹除等操作复杂得多，也是两者区别之所在。

如果用简单公式表示出来，一般启动操作是冷量、物料、组分三者从不平衡到平衡的稳定有序过程；调节工作是破坏稳定平衡之后达到新的平衡点，是无固定模式的变化过程。

# 2. 问：怎么样进行分馏塔精馏工况的调节呢？可否举例加以说明呢？

答：这个问题提得很好也很重要，分馏塔的操作工可以用毕生精力去研究它，以掌握分馏塔调节技能。

有人说：“要把物料平衡掌握好，物料平衡掌握不好，将使整个塔的操作达不到预期目的”，这是很巧妙而含糊不清的讲法。应该看到，只要液氧液面稳定，冷量平衡了，那么塔的物料总是平衡的。说它巧妙，因为物料平衡了，冷量总是平衡的，符合了要在冷量平衡基础上进行调节的规范，否则工况肯定不会稳定；说它含糊不清，是如何掌握好，道理没有讲清楚。

一个分馏塔的稳定平衡，是冷量平衡、物料平衡和组分平衡，该掌握的首先是冷量平衡。问题在于平衡在哪一个水平上，即液面稳定在什么位置上，产量、纯度稳定在哪

个指标上。例如  $\times x$  塔稳定在  $35\mathrm{cmCCl}_4$  液位上， $99.2\% \mathrm{O}_2$ 、 $135\mathrm{m}^3 /\mathrm{h}\mathrm{O}_2$  的工况点上，只要液面稳定，产量不合格可物料却是平衡的，纯度偏低可组分却是平衡的。那如何把产量、纯度调上去呢？把不理想工况变成理想工况（合格点上），就是靠调节，这是我们的努力目标。如不采取措施，通俗说法就是开“太平车”，除非系统受到外界大的影响，那么这个塔就始终只会稳定在这个不佳的工况点上，有的操作工害怕，不敢去碰它，那也不对；应当在新的措施出台之前，先对塔的工况进行周密调研，就是对影响分馏塔工况，如上下塔压力、纯度、温度、液位、流量等等了解一遍，看看毛病出在那里。华北氧气厂的一位老工程师曾告诉我，他平时想解决分馏塔的重大问题时，常常站在分馏塔前，聚精会神地观察，一站就是  $1\sim 2$  个小时，甚至一整班，从仪表的瞬时微量变化中，连关塔内物流的变化情况。这叫做由表及里，透过现象看本质，从而悟出调节的方法和手段，这是工况调控前最重要的过程。只有这样，才能对症下药。

以前说过，调节含义就是打破旧平衡，建立新平衡，以上述纯度偏低而勉强合格、氧产量未达到的分馏塔为例，通常调节办法先要提高氧气纯度，后再开大氧产量。具体地说，如液氮浓度高于气氮，第一步是开节一4阀(液氮去上塔量的阀)，这叫做“调量”。这样一来，组分及物料平衡马上被破坏了。为什么呢？开节一4阀后，上塔精馏段回流比  $(L / V)$  增加，根据精馏的原理，气氮浓度 $\left(\% \mathrm{N}_{2}\right)$  提高了（即  $\% \mathrm{O}_2$  下降了），按物料及组分平衡式  $V_{N} = V_{K} - V_{0},V_{N}y^{N} = V_{K}y^{K} -$ $V_{0}y^{0}$  （式1），开节一4阀因  $y^{N}\% \mathrm{O}_{2}$  下降，即变成  $V_{N}y^{N} <   V_{K}y^{K} - V_{0}y^{0}$  ，不平衡了；接着开大  $V_{N}$  ，  $L / V$  下降，公式变成  $V_{N}y^{N} > V_{K}y^{K}-$ $V_{0}y^{0}$  ，氧浓度  $y^{\circ}(\% \mathrm{O}_{2})$  提高了，获得新的平衡  $V_{N}y^{N} = V_{K}y^{K} - V_{0}y^{0}$  （式2）。式2是在新

基础上的平衡，例如氧浓度从  $99.2\% \mathrm{O}_2$  变成  $99.6\% \mathrm{O}_2$  ，这是第一回合；由于氧浓度的提高，就有条件开大氧流量，式2又被破坏，因  $V_0$  上升，提馏段回流比  $L / V$  降低， $y^0$  下降，又获得新的平衡点，例如  $V_0 = 145\mathrm{m}^3 /\mathrm{h},y^0 = 99.3\% \mathrm{O}_2$  ，这是第二个回合。当人们看到通过这样的方法调节，工况从  $99.2\%$ $\mathrm{O}_2$  、  $135\mathrm{m}^3 /\mathrm{h}$  变到  $99.3\% \mathrm{O}_2$  、  $145\mathrm{m}^3 /\mathrm{h}$  ，好转了，说明调节正确。如果节一4阀仍可开大，就可利用同样办法，先将旧平衡破坏，精馏塔会自动地建立起新的平衡。就这样反复地破坏平衡达到新的平衡，再破坏又平衡，会最终达到预期目的。变动气体排出量是宏观调节，因回流比促使精馏的改变，属微观调控。建立新的平衡是目的，宏观量变是通过微观变化来达到的，这就是调节过程的精髓所在。

以上慢镜头的描述，告诉人们：(1)好的工况是靠调节来达到的；（2）调节之前必须对外部及内部条件进行调查分析；（3）调量和调纯的关系，调纯是调量引起的，说明书告诉我们调节即调量；（4）浓度变化是因量变(回流比改变)的作用而来的，两者关系在时间上有先后，是瞬时的有先后，所以说明书告诉我们一个操作之后(约5分钟)，待工况(浓度)产生变化后才采取另一操作，如果系统向预期的效果前进，可以继续前进；如

果产生负面反应，就得采取逆向操作，或采取另一措施；(5)上塔的氧氮浓度与下塔有关，所以说明书规定先调好下塔再调节上塔，否则互相干扰不易调好；（6）操作工调节是通过破坏旧平衡达到新平衡的过程，之所以如此，是因为改变塔的回流比是精馏的规律性所在，操作工必须好好学习理会精馏这个原理才能运用自如；（7）调回流比必须在冷量平衡稳定的基础上进行，调节回流比过程中液面也会出现短暂的变化，这不过是微量变化，冷量基本是稳定的，不必害怕，会很快得到新的平衡，只是液面稳定点位置略微有点改变而已，所以说明书规定液氧液面可在  $35\sim 40\mathrm{cmCCl_4}$  柱变化，即是此道理；（8）除非很有经验的操作工（具有计算机控制的分馏塔除外），同时采取几个措施，开关几个阀门是不希望的。

上面是分馏塔调节最简单的例子，它按流程不同、塔板结构不同，及质量好坏而情况各异。如双高流程，单高氧及高纯氮，正流膨胀与返流膨胀流程都有关系，虽然道理相同，却有千变万化之感。虽然如此，分馏工况是由操作工来调节的，它在微观上是有其内在规律的，只要大家提高自我的技术素质，良性循环的操作法不难被我们所认识。

# 《真空》杂志简介

《真空》杂志创刊于1964年，至今已有32年的历史，是中国真空界主要杂志之一。由机械工业部沈阳真空技术研究所主办，沈阳真空杂志社编辑和出版。自1994年真空杂志转换经营机制以来，由我国真空企业界实力雄厚的厂家与杂志社联办成立董事会，并由真空学术界著名学者组成编委会，社会效益与经济效益与日俱增，在国内真空界享有盛誉；在国外，自1989年公开发行以来，被美国、英国、德国、日本、意大利、韩国以及港澳地区越来越多的海外学者所关注，著名外商厂家争先在《真空》杂志上发布产品广告。

《真空》杂志是直属于机械工业部主管的中央级刊物，并列入《中文核心期刊要目总览》物理学类核心期刊。30多年来以理论为基础，以实用为特色，独具一格，密切结合国情，立足刊物之林，是其经久不衰的生命力所在。

《真空》杂志主要报导专业范围：真空获得、测量与检漏、真空热处理、辉光离子氮化、真空冶金与铸造、镀膜与刻蚀、真空干燥与食品保鲜、真空压力浸渍、树脂浇注、真空焊接与铝钎焊、真空包装与输送、加速器与核聚变、各种真空材料、表面科学与仪器、纳米科学与技术等等。

《真空》杂志主要栏目设置：学术论文、新产品开发、实验技术、综述、系列讲座、经验介绍、设计计算、工艺研究、考察报告、设备维修、各种信息及产品广告等。《真空》杂志为双月刊，16开本，每期64页。激光照排，胶版印刷，质量精美。

# 上海杨园压力容器制造厂低温液体贮槽通过市级鉴定

1995年12月16日由上海市郊区工业局主持，上海凯普登工业气体设备有限公司、上海市杨园压力容器制造厂  $2000\sim 15000$  升真空粉末绝热低温液体贮槽，通过了市级鉴定。参加鉴定会的有34个单位62位代表。鉴定委员会由上海交通大学、中国空分设备公司、杭州制氧机研究所、法国液空上海有限公司、武汉钢铁设计院上海分院、上海梅山冶金公司、上海市劳动局、上海郊县工业科技情报站的7名代表组成，主任委员为上海交通大学顾安忠教授。

# 技术委员会召开二届四次会议

机械工业部气体分离与液化设备标准化技术委员会二届四次会议，于1995年11月13日至15日在长沙召开。会议上，标委会秘书长陈永良受标委会主任委员张培元委托，向与会代表汇报了机械工业部气体分离与液化设备标准化技术委员会二届三次会议以来的行业标准化工作总结，其主要内容是1995年标准制、修订项目计划执行情况；关于标委会工作；标准制、修订经费问题；行业有关标准发布、出版和发行信息通报；行业标准化技术服务以及1996年标准化工作项目计划(草案)等。标委会委员和代表们听取了汇报后，进行了认真的讨论、审议，认为本标委会在执行机械工业部下达的1995年标准制、修订项目计划和标委会二届三次会议提出的行业标准化计划任务，能按期完成表示满意。并同意标委会秘书处提出的1996年行业标准化项目计划(草案)。

会议还审查了由杭州制氧机集团有限公司负责起草的《空气分离设备用活性氧化铝验收技术条件》；哈尔滨环保制氢设备工业公司、杭州制氧机研究所、中国船舶工业总公司七一八研究所和苏州水电解制氢设备公司等四个单位共同起草的《水电解制氢设备》；哈尔滨制氧机厂负责起草的《槽粕造粒机》和《槽舶造粒机产品质量分等》等四项机械行业标准，并获得了通过。

会议得到中国船舶工业总公司七一八研究所的资助，并在标委会各位委员和与会代表们的积极支持下，取得圆满成功。

# 江苏省第五届制氧技术交流会召开

江苏省制冷学会第五届制氧技术交流会，1995年10月29日至31日在无锡召开。来自省内及兄弟省市的制氧单位、制氧机厂及配套配件制造厂、研究所和高等院校等60个单位共92位代表参加了会议。机械工业部气体分离设备科技情报网常务理事长、《深冷技术》主编顾福民高工，上海市制冷学会常务理事、第一专业委员会主任顾安忠教授，浙江省气体工业协会副秘书长朱文忠工程师等，也应邀出席了会议。

大会在第二专业委员会领导下进行。省制冷学会理事曹卫平工程师主持了开幕式，省制冷学会副理事长、第二专业委员会主任杨思文教授致开幕词，无锡市蠡园乡副乡长、无锡蠡园医用氧厂蒋燮民董事长代表承办单位致欢迎词，省制冷学会理事、第二专业委员会副主任庄胜强高工介绍了大会筹备工作及会议安排情况。机械工业部小空分网理事长李春堂厂长在闭幕式上致词。

本次会议收到交流论文20余篇，经专家评审筛选，有18篇论文被正式录用并汇编铅印成论文集，其内容涉及制氧行业的新技术、新工艺、技术总结、

管理经验、节能措施、安全技术等各方面。论文印刷清晰，于开会前发给代表，大家感到满意。会议交流中，顾福民高工作了国内外制氧技术发展趋势及动态的专题报告；庄胜强高工作了最近考察美国空分气体行业的情况报告；顾安忠教授作了关于液化天然气技术报告。有20个单位发布了最近产品生产和销售开发信息。全体代表在会议内外进行了广泛的交流和讨论，达到了相互学习、增进了解、广交朋友、开拓思路、传播信息的目的。使大家得到了较大的启发和收获。
