# 一、引言

目前，在大型全低压空分流程中，大部分采用空气膨胀的卡彼查低压制冷循环，设备所需的冷量，绝大部分是由透平膨胀机来提供的。为提高产品提取率，膨胀后的空气一般作为原料气进入上塔参加精馏，以充分利用上塔的精馏潜力。因此，透平膨胀机参数对整体流程有很大的影响。在大部分流程中膨胀机的进出口压力，取决于大气压力及其本身的增压比，其变化范围不大。而膨胀机的进出口温度及膨胀量这些参数，变化较大。有的空分流程，其膨胀前温度很低，而膨胀量过大，在某些情况下，空气不能全部进入上塔参加精馏；而另外一些空分流程，膨胀前的温度又很高，虽然膨胀机焓降很大，膨胀量较小，但造成膨胀机的出口温度也很高，过热度很大。如果过热的空气进入上塔，可能会对精馏过程产生不利的影响。那么膨胀机的参数取决于流程中哪些因素？其影响规律如何？又如何加以控制和调节？关于这些问题，本文针对目前常用的分子筛增压空分流程作一全面分析和计算。

# 二、膨胀机参数的影响因素及其规律

空分流程是精馏系统、换热系统、制冷系统的有机组合，这几个系统相互依赖、相互联系，流程内部的各个参数也是相互影响、相互牵制。膨胀机参数的确定，既要满足整个设备的冷量要求，又要保证精馏系统、换热系统的正常运行。根据大量的分析和计算可知，主换热器的温差、设备冷量需求的大小及膨胀机本身的性能，这几个参数对膨胀机的膨胀量、进出口温度有较大的影响。

# 2. 主换热器冷热端温差

图2（b）、图2（c）分别表示了冷热端温差[2]变化对膨胀机参数的影响情况，从图中可以看出热端温差的变化对膨胀机的参数有很大影响。因为热端温差的变化不但会引起主换热器环流热负荷的变化，而且还引起复热不足冷损的改变，从而直接影响空分设备所需冷量的大小。热端温差的增大，使环流热负荷增大；另一方面热端温差的增大直接导致膨胀机应提供的冷量增大。前两者最终导致膨胀量的增加，焓降减低，膨胀机进出口温度下降。

主换热器冷端温差增大，同样导致膨胀机膨胀量增加，膨胀机焓降减低，膨胀机进出口温度下降。这是因为增大冷端温差，虽然对设备的冷量需求没有变化，对膨胀机制冷量没有直接影响，但使主换热器的环流热负荷下降，因而导致上述结果。从图2（b）、2（c）的相互比较可以看出，对于膨胀量，热端温差对其的影响比冷端温差要显著得多；而对于焓降和出口温度的影响，两者相差不大。因此，在某些设计中，可以通过适当改变热端温差，来达到改变膨胀量的目的；或者通过调节冷端温差，来达到改变膨胀量进出口温度，同时使膨胀量变化不大的目的。

# 3. 绝热冷损

图2(d)给出了绝热冷损不同的条件下，膨胀机参数的变化情况。横座标表示单位加工空气的冷损值，从图2(d)中可以看出，如果冷损增大，则导致膨胀量增大，膨胀后温度降低，膨胀机焓降减低。绝热冷损的改变，实际上相当于模拟了空分设备的大小，在目前采用常规绝热方法的条件下，空分设备单位加工空气量的冷损随设备容量的减少而增大。因而也可以说，在采用这种全低压空分流程时，随着设备容量的减少，在其它参数

相同的条件下，膨胀机膨胀量所占的比例增加，而膨胀机出口温度及焓降都减少。因此，设备容量的大小，对流程中膨胀机参数的影响也不容勿视。

# 4. 膨胀机的绝热效率

膨胀机的绝热效率对膨胀机参数的影响也很重要。改变不同的膨胀机绝热效率，并使其它参数与基本工况参数相同，计算后结果示于图2(e)中。可以看出，随着膨胀机效率的提高，膨胀机膨胀量显著下降；而膨胀机焓降迅速增大，膨胀后温度相应提高。这是因为，随着膨胀机绝热效率的提高，膨胀量下降，而膨胀前温度增高，使绝热焓降提高。根据这一点可以看出，对于需要制冷量比较大或产品液体较多的空分设备，应该设法提高膨胀机效率，以改善整个流程工况。

以上分析了主换热器温差、产液情况、绝热冷损的不同及膨胀机本身的绝热效率对膨胀量、膨胀机焓降及进出口温度的影响情况。除了这些因素外，还有一些因素对膨胀机参数也有影响，如设备的氧提取率、环境压力及温度等。以上分析结果对不同的流程及产品参数，其绝对数值可能不全相同，但是这种规律和趋势是相同的，分析的这些规律具有普遍的意义。

# 三、膨胀机参数的确定及调节

从流程设计中的几个主要参数对膨胀机参数影响的分析中可以看出，膨胀机的参数是受到流程中其它参数制约的，并不是可以单独改变的，这些参数的确定，一定要保证换热系统及精馏系统的正常工作。因此，膨胀机参数的确定和调节也必须保证整个系统的正常工作。

膨胀机参数的影响因素，可以归纳为两类：一类是在设计中可调节的，如主换热器的冷热端温差；另一类是受客观条件影响而不能改变或难以改变的，如产品液体量的比

例、绝热冷损、膨胀机效率等。现根据第二节的结果，分析如何适当地确定和调节膨胀机的参数。

# 1. 整个设备所需求的冷量较小

如设备容量大，液体产品需求少，也就是说空分设备的产冷能相对较大。在这种情况下，就会使膨胀量较少，而膨胀机进出口温度偏高，在某些情况下，会使膨胀后过热度很大，以致于不适合进入上塔。众所周知，膨胀量的减少对提高整体流程的氧提取率是有利的；但是进塔的膨胀空气温度过高，对精馏不利，因此应该采取适当的措施加以调节。最简单的办法就是增加一个膨胀后换热器这种方法可以减少膨胀空气的过热度，从而增加进下塔的空气焓值，增加主冷的热负荷，相当于把一部分冷量从下塔传递到上塔，改善了精馏工况。膨胀后换热器的冷源可以视具体情况采用氧气、氮气或其它返流气。

上述情况，也可以采用适当增加主换热器冷端温差的办法，来降低膨胀后空气的过热度，当然膨胀量会随之增加。这在某些场合下，可以减少换热器的面积，从而降低成本，同时充分利用上塔的精馏潜力，保持适当的氧提取率。

# 2. 设备所需的冷量大，液体产品较多

目前很多用户要求空分设备带的液体产品较多，这与第一种情况相反。此时，空分设备的冷量需求相对较大，膨胀量较大，膨胀机进出口温度较低，空分设备相当制冷能力不足，在某些情况下，会导致膨胀机出口温度接近饱和温度。此时要尽量降低主换热器的冷端温差，以减少复热不足冷损，同时尽量选用性能较好的膨胀机，以减少膨胀量，当然这些措施是有限的。若空分设备的冷量

需求过大，则应考虑别的制冷循环。

# 3. 其它情况

另外一些原因也要求调节膨胀机参数，如设计工况与操作工况偏离较远、变工况操作等。膨胀量的调节，可以通过膨胀机本身的调节来实现。对于膨胀前温度的调节可以通过二股中抽混合来达到所需求的温度，如图1所示，改变这两股量的比例，可以达到不同的膨胀机前温度。两股中抽的位置可以根据具体的工况来决定，在极限情况下，一股可以从换热器冷端抽出，或者从热端抽出。膨胀前温度的调节还可以通过设置膨胀前换热器来调节，通过改变换热器内冷气流的量或温度来达到调节膨胀机前温度的目的。

# 四、结束语

空分流程是一个有机整体，任一参数的改变都会引起膨胀机参数的改变，掌握流程中的一些主要因素对膨胀机参数的影响规律就能在设计中更合理确定这些参数。如在设计中，若发现膨胀后温度过高、膨胀量很小，则可以考虑设置膨胀后换热器或者扩大主换热器的温差，以减少换热器面积，降低一次投资成本。如在流程设计中，发现膨胀后温度过低，则应该尽可能增大换热面积，缩小换热器换热温差，并在可能的条件下，尽量压缩设备的各项冷损，或者降低氧提取率，增加加工空气量。

了解和掌握流程中各参数对膨胀机参数的影响规律，对于空分流程的操作也是很重要的。一方面，能正确地在设计中设置合理的调节控制手段，保证在必要时能实现适当的调节。另一方面，在运行过程中工况发生变化时，能指导操作者进行正确的调节。

# 一、重要意义

随着国民经济的飞速发展，社会对气体制造和应用技术的要求越来越高。为稳定工艺，确保产品气体质量，不仅需对整个工艺过程的热力学参数，如温度、压力、流量等进行测量和控制；而且对工艺过程的每个关键部位需要进行成分量的测定与控制。成分量的测定数据，对工艺条件变化的反映及时准确、直观性强这一特点，足以使它成为操作人员的得力助手和重要参谋。因此，近十多年来，在气体制造及应用中，以成分分析仪器作为必要的配套设备已逐渐成为全面质量管理的一个重要发展趋势。其作用具体表现在以下几个方面：

（1）监测工艺过程参数，为设计人员和生产操作人员及时提供其所需的成分含量数据，达到调整生产、控制操作、稳定工艺，以及改进工艺和设备的目的。  
（2）监测产品质量，严格把关，保证产品质量符合气体标准及客户的需要。  
（3）监督安全生产。如对空分生产提供液氧中乙炔及碳氢化合物等数据，及时反映生产中的问题，监督超标运行，督促生产人员及时采取安全措施。

# 二、现状和发展趋势

# 1. 现状

（1）分析仪器制造技术发展后，国内不少仪器厂家先后引进了国外先进的仪器制造技术，把国产仪器的制造水平提高到一个新阶段。另一方面，一些研究院(所)也相继研制成功不少新的仪器设备。这为提高国内气体制造及应用技术水平创造了有利条件。  
（2）气体制造及应用技术的不断发展，对分析仪器的需要不断增加，对分析仪器的质量和性能的要求也在不断提高。广大客户的需要和众多分析课题的迫切提出，大大促进了国内气体分析仪器的进一步发展。  
（3）随着气体应用技术的不断提高，各气体制造及应用部门，为了加强企业全面质量管理以及参与市场竞争，逐渐重视成分量的监测，注意安排适量资金选购及配套分析仪器，并培训人员，建设新型化验室等，这就使气体分析工作提高到一个新的阶段。

# 2. 发展趋势

国内气体分析仪器在应用方面的发展趋势表现在：

（1）加强配套监测工艺操作的在线分析仪器，可保证及时发现并及时调整操作参数，

以稳定工艺。这是自80年代以来仪器应用的趋势。目前，国内各生产部门一般都配套一定数量的在线分析仪器。这种应用正在逐渐被推广。

（2）加强气体产品（及原材料气体）的质量监测。对气体制造厂来说，加强产品质量监测可确保出厂产品质量合格。而对气体应用部门来说，加强购入气体原材料的质量监测，可确保企业在应用中不因原材料气体不合格而影响本企业生产工艺及产品质量。  
（3）配备安全分析仪器，进行经常性的安全生产分析，积累数据，研究制定实施安全措施，确保安全生产。尤其是对易燃、易爆、高压、低温液化气体，保证安全生产及使用是非常重要的。  
（4）采用国外先进分析仪器配套工艺，弥补国内仪器水平差距，有效地保障重要部门的生产及质量监控。目前，由于国产仪器与国外先进的仪器设备在性能、质量等方面尚有不小差距，因此，国内部分生产高纯气体的厂家为稳定工艺，提高产品质量，加强市场竞争，开始引进国外先进仪器设备。另外，部分生产工业气体的厂家也选购部分可靠性较高的进口仪器，作为在线监测工艺使用。还有一些企业，引进先进的分析仪器是为建立标准传递、监督与保证一线分析仪器运行完好率，从而建立并完善本企业分析检测系统和质量监控系统。

④ 配有氦纯化器，将氦气从  $99.99\% \mathrm{He}$  纯化到  $99.99999\% \mathrm{He}$ ，从而使DID灵敏度得到有效提高。

2. 气体中微量水的测定仪器——氧化铝湿度计

在工业生产过程中和实验室内，常常需要对气体中的含水量进行测量，因此各种各样的水分含量测量方法和仪器应运而生。虽然各种方法都有其特点，但总的来说，工业生产过程中能够快速、准确地连续测量水分含量的方法仍然是极少的。众多类型的湿度计都存在着因采样引起的污染问题，采样管路和器具的干湿度常会使原样品中的含水量

改变，湿度的不同将促进产生新的气液平衡，这些都会造成含水量的变化。

美国PN公司研制的、被国外许多厂家广泛采用的氧化铝湿度计，可以克服上述困难，完成在较广湿含量范围内的在线连续监测，并可进行遥远距离的测量，第一次实现了不用人工操作而对生产过程中的含水量进行监测。

# （1）测量

氧化铝湿度计测量水蒸气含量的范围，从露点/霜点  $+20^{\circ}\mathrm{C}$  到  $-110^{\circ}\mathrm{C}$  。相当于从20000微克水/升到0.001微克水/升。此仪器测量湿度的下限灵敏度比其他湿度计至少高两个数量级。

# (2）应用

在线测量：因仪器敏感元件尺寸很小，且不需要管路连接，也不需要调压器等，可将湿度计直接放在容器及工艺管路内进行无滞后测量，使用非常方便。信号还可以通过电缆远传，进行多点集中操作测量。敏感元件能够在真空或高压下进行湿度测量，总压范围可由几个微米汞柱到500磅/时²。对于静止气流或速度达1000厘米/秒的气流，可进行正常的湿度测量。

此外，氧化铝湿度计还可以做成小型便携式露点仪，作非连续取样分析用。这种仪器体积小，体重轻，快速测定，自动干燥，使用非常方便。因其价格较便宜，所以国内不少厂家选用这种仪器作为瓶装气及管道气体中水含量的快速检测用仪器。

# 3. 高灵敏度宽测量范围的氧分析仪

在空分及其他气体产品中，氧含量的检测向来都是作为主要检测项目来完成的。通常它包括常量、微量及痕量这样一个浓度范围很宽的检测范围。正因为氧含量的检测具有如此重要的地位，所以国内外不少厂家研制了各种不同种类的仪器产品。但是，总的说来，在国外应用最广的测氧方法还仍是燃

燃料电池法。这种方法做的仪器从  $\%$  至1ppb氧浓度有一整系列产品。

# 仪器性能

① 高灵敏度。除常量档外，在微量及痕量范围，不同型号的仪器有不同的最低测量范围，如316型为  $0 \sim 10\mathrm{ppmO}_2$  ，3160型为  $0 \sim 1\mathrm{ppmO}_2$  ，306型为  $0 \sim 0.5\mathrm{ppmO}_2$  ，396型为  $0 \sim 0.1\mathrm{ppmO}_2$  ，3060型为  $0 \sim 50\mathrm{ppbO}_2$  。

② 线性范围宽。在微量及痕量各不同测量范围，仪器均可有  $4 \sim 6$  档调节。

③ 传感器保养容易。微型燃料池传感管是一个密封的电化学装置，无需电解液，也不要经常清洗电极。并且它有一个绝对零点，校验时不需要零点气。

④ 用空气校验。微型燃料池传感器对于氧浓度产生一个线性输出，故可以利用空气（含氧  $20.95\%$ ）校验微量范围的氧分析仪。这就省掉了使用量程气校验，同时也排除了使用的量程气受空气污染的可能。

⑤ 氧分析仪可配套一个气体进样系统，尤其是当用于在线分析时更需如此。对不同的使用条件可有不同的进样系统。

⑥ 响应迅速。在  $0 \sim 1 \mathrm{ppm}$  内达  $90\%$  时，其响应时间少于60秒；在  $0 \sim 1000 \mathrm{ppm}$  内达  $90\%$  时，其响应时间少于10秒。

⑦ 对痕量氧分析仪，采用特殊的湿式传感器，且在仪器内部可根据需要加装校验器，不必使用外部标准气校验仪器。另外，仪器还包括一个焊制不锈钢进样系统，保证无泄漏操作，且防止被腐蚀而损坏。并装有自动断路保护洗涤器，防止过量氧损坏仪器。仪器还有零点气和量程气发生装置，可实现自身校验。

仪器性能

5. 净化空气中二氧化碳含量分析仪

大型空分设备的原料是空气，空气中对空分生产有危险的杂质很多，空分设备的安全技术是一门很重要的课题。现仅就工作中的一些实践与体会阐述如下，供大家参考。

# 一、有害杂质的种类和来源

空气中有害杂质的种类很多，主要是  $\mathrm{C}_2\mathrm{H}_2$  、  $\mathrm{CH}_4$  、  $\mathrm{C}_2\mathrm{H}_6$  和其它烃类化合物，如  $\mathrm{C}_2\mathrm{H}_4$  、  $\mathrm{C}_3\mathrm{H}_8$  、  $\mathrm{C}_3\mathrm{H}_8$  、  $\mathrm{C}_4^+$  等，  $\mathrm{NO}_{\mathrm{x}}$  和  $\mathrm{O}_3$  在空气中也有一定的含量。其中难以清除的是  $\mathrm{CH}_4$  和  $\mathrm{C}_2\mathrm{H}_6$  ，危险性最大的是  $\mathrm{C}_2\mathrm{H}_2$  。这些气体杂质的来源，在石化厂主要是电石排放渣堆，炼油、煤化工、石油化工等装置的废气排放、跑冒滴漏等。

氮的氧化物，主要来自炼焦炉、燃油废气炉、硝酸尾气、硝酸释放气等。

臭氧主要来自高压变电所、高压电线及大型电机。

据了解，吉林某厂区上空易燃易爆及腐蚀性气体多达几十种，释放量每小时约几十万立方米，在30米高空以上分析查明，无一处符合规范要求，大气污染比较严重。

# 二、有害杂质对空分设备的影响和危害

原料空气中有害杂质与空分设备安全运行关系极大。这些烃类在液氧中的含量超过溶解度即呈固体析出，溶解在液体中的  $\mathrm{C}_2\mathrm{H}_2$  等烃类化合物是没有爆炸危险性的，只有呈现固体状态时才有很大的危险性，在机械冲击作用、化学能量变化、静电放电等影响下，就会发生爆炸。  $\mathrm{O}_3$  能直接增加并强化液氧中  $\mathrm{C}_2\mathrm{H}_2$  或其它烃类化合物发生爆炸的敏感性。其它不饱和烃类的存在，当  $\mathrm{C}_2\mathrm{H}_2$  发生爆炸时会促使爆炸更加猛烈。

固体的烃类化合物与液氧液空相互作用，在一定的条件下也能发生爆炸。在操作条件下  $\mathrm{C}_2\mathrm{H}_2$  的蒸汽压很小，几乎全部冷凝在冷凝蒸发器的液氧中，多则析出。固体  $\mathrm{C}_2\mathrm{H}_2$  等烃类化合物发生爆炸的部位，主要是液体处于沸腾、不断蒸发、易积聚浓缩物的地方，如冷凝蒸发器及塔内有关死角等部位，都潜伏着固体  $\mathrm{C}_2\mathrm{H}_2$  等烃类化合物发生爆炸的危险性。

前一机部某厂曾发生过三次爆炸事故，伤亡很大，损失严重，装置几乎报废。上海某钢厂空分设备爆炸后，将1吨重的塔帽带出几十米远。吉林某空分车间处于污染严重的界区，空气中的  $\mathrm{C}_2\mathrm{H}_2$  含量经常超过规定指标，因而形成液氧中  $\mathrm{C}_2\mathrm{H}_2$  的含量较高，只好经常排放液氧。大连某厂为了回避界区的污染，曾设过远方吸气口，这些作法都是极不经济的。

# 一、方案选择

1. 板式主冷空分的特点及液氧净化新工艺的确立

我们曾研究过邯郸  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备的热虹吸蒸发器，其主冷液氧中乙炔含量可

小于  $0.1\mathrm{ppm}$  。我们还参考了日立空分的技术，主冷凝器本身就是一个热虹吸蒸发器，即在主冷凝器内安置一块隔板，将上塔液氧与主冷凝器液氧分开，使上塔底部液氧先通过吸附器，然后进入主冷凝器。

原来，此空分设备主冷凝器中的液氧，依靠液氧泵强制性克服吸附器系统的阻力，回到主冷凝器(见图1)。由于液氧泵的质量不过关，净化不完全，乙炔含量严重超标。

\begin{array}{l} V _ {\text {主 冷}} = Q / r _ {\text {氧}} = 6 9. 6 \times 1 0 ^ {6} / 2 9 8. 9 \\ = 2 3 3 0 0 \left(\mathrm {m} ^ {3} / \mathrm {h}\right) \\ \end{array}
$$

（8）主冷中乙炔的平衡关系

设氧气中乙炔的浓度为Yppm，液氧中的乙炔浓度是气氧中乙炔浓度的22.5倍（见《制氧机原理与操作》）。

作为上塔馏分的气氧量为  $23300 - 6000 = 17300\mathrm{m}^3 /\mathrm{h}$  。气氧由主冷返回上塔的带液量设为  $10\%$  ，则带回上塔的液体量为  $23300\times$ $10\% = 2330\mathrm{m}^3 /\mathrm{h}_{\circ}$

上塔底部液氧中乙炔量  $M$  液氧

$$
\begin{array}{l} M _ {\text {液 氟}} = M _ {\text {上 塔}} + 1 7 3 0 0 Y + 2 3 3 0 \times 2 2. 5 Y \\ = 6 2 6. 5 + 6 9 7 2 5 Y \\ \end{array}
$$

离开主冷气氧带走乙炔量

$$
\begin{array}{l} M _ {\text {主 冷 出}} = 2 3 3 0 0 Y + 2 3 3 0 \times 2 2. 5 Y \\ = 7 5 7 2 5 Y \\ \end{array}
$$

情况I：上塔液氧不经过液氧吸附器的乙炔量  $M_{\text{液氧}}$  就是主冷进口的乙炔量  $M_{\text{主冷进}}$ 。

即  $M$  主冷进  $= M$  液氧  $= 626.5 + 69725Y$

则  $M$  主冷进  $= M$  主冷出

$$
\begin{array}{l} \therefore 6 2 6. 5 + 6 9 7 2 5 Y = 7 5 7 2 5 Y \\ 6 2 6. 5 = 6 0 0 0 Y \\ Y = 0. 1 0 4 4 \mathrm {p p m} \\ \end{array}
$$

主冷液氧中的乙炔浓度  $\mathrm{X} = 22.5\mathrm{Y}$

$$
\begin{array}{l} \therefore X = 2 2. 5 \times 0. 1 0 4 4 \mathrm {p p m} \\ = 2. 3 4 5 p p m \\ = 2. 3 4 5 \left(\mathrm {c m} ^ {3} \mathrm {C} _ {2} \mathrm {H} _ {2} / \mathrm {m} ^ {3} \mathrm {O} _ {2}\right) \\ \end{array}
$$

因1厘米3乙炔  $= 1.1747$  毫克；1米3氧  $= 1.28$  公升液氧。所以，  $2.345\mathrm{ppm} = 2.345 \times (1.1747 / 1.28) = 2.152\mathrm{mgC}_2\mathrm{H}_2 / \mathrm{L}$  液氧。

如果在上塔和主冷之间无液氧吸附器，这将使主冷中液氧浓缩，达到空分设备安全规程所允许的乙炔含量为  $0.4\mathrm{mg / L}$  液氧的5.375倍，很容易发生爆炸事故。

情况Ⅱ：上塔液氧全部经过液氧吸附器，吸附器清除乙炔的效率  $\eta$  液氧  $= 0.95$ 。

则  $M$  主冷进  $= M$  液氧  $\cdot (1 - \eta$  液氧）

$$
\begin{array}{l} = (6 2 6. 5 + 6 9 7 2 5 Y) \\ (1 - 0. 9 5) \\ = 3 1. 3 2 5 + 3 4 8 6. 2 5 Y \\ \end{array}
$$

$$
\begin{array}{l} M _ {\text {主 冷 出}} = M _ {\text {主 冷 进}} \\ 7 5 7 2 5 Y = 3 1. 3 2 5 + 3 4 8 6. 2 5 Y \\ \end{array}
$$

得  $\mathbf{Y} = 0.0004336\mathrm{ppm}$

主冷液氧中乙炔浓度

$$
\begin{array}{l} X = 2 2. 5 Y = 2 2. 5 \times 0. 0 0 0 4 3 3 6 p p m \\ = 0. 0 0 9 7 5 6 \mathrm {p p m} = 0. 0 0 9 7 5 6 \\ \times (1. 1 7 4 7 / 1. 2 8) \\ = 0. 0 0 8 9 5 \mathrm {m g} / \mathrm {L} \text {液 氧} \\ \end{array}
$$

此值是安全规程中规定值的  $2.5\%$  。这对于设备的安全运行非常有利，但是要处理如此多的液氧，液氧吸附器将是很大的设备。

情况Ⅲ：上塔的液氧中使  $100\mathrm{cm}^3 /\mathrm{h}$  液氧经过液氧吸附器，清除乙炔后再与其余的上塔液氧汇合一起进入主冷。

上塔的液氧量为  $23300 + 2330 = 25630 \mathrm{~m}^{3} / \mathrm{h}$ ，这些液氧在液氧吸附器中除去的乙炔量为：

$$
\begin{array}{l} (6 2 6. 5 + 6 9 7 2. 5 Y) \times (1 0 0 0 0 / 2 5 6 3 0) \\ \times 0. 9 5 = 2 3 2. 2 1 8 + 2 5 8 4 4 Y \\ \end{array}
$$

$$
\begin{array}{l} M _ {\text {主 冷 进}} = 6 2 6. 5 + 6 9 7 2 5 Y \\ - (2 3 2. 2 1 8 + 2 5 8 4 4 Y) \\ = 3 9 4. 3 + 4 3 8 8 1 Y \\ \end{array}
$$

$$
\begin{array}{l} \therefore M _ {\text {主 冷 出}} = M _ {\text {主 冷 进}} \\ 7 5 7 2 5 Y = 3 9 4. 3 + 4 3 8 8 1 Y \\ Y = 0. 0 1 2 3 8 \mathrm {p p m} \\ \end{array}
$$

主冷中液氧的乙炔浓度

$$
\begin{array}{l} X = 2 2. 5 Y = 2 2. 5 \times 0. 0 1 2 3 8 p p m \\ = 0. 2 7 8 6 \mathrm {p p m} = 0. 2 7 8 6 \\ \times (1. 1 7 4 7 / 1. 2 8) \\ = 0. 2 5 5 7 (\mathrm {m g} / \mathrm {L} \text {液 氧}) \\ \end{array}
$$

此值是操作规程中乙炔允许含量  $0.4\mathrm{mg} / \mathrm{L}$  液氧的  $64\%$  。

我们认为这一方案最可行，即保证操作上的安全，又不使液氧吸附器太大，可以应用原有的液氧吸附器。同时吸附器的阻力不太大，使得上塔和主冷的改造工作量较小。

计算结果表明：

如果液氧不经净化，液氧中的乙炔含量高达  $2.15\mathrm{mg / L}$  ，高于安全规程规定的乙炔最高含量  $(0.4\mathrm{mg / L})$  的5.375倍，很容易发生爆炸，极不安全。

如果液氧全部经过液氧吸附器净化（即流量达  $15000\mathrm{m}^3 /\mathrm{h})$  ，液氧中的乙炔含量可降至0.00985mg/L，是安全规程规定的乙炔最高含量的2.5%。这完全没有必要，还将使液氧吸附器体积大大加大，很不经济。

如果液氧流量选为  $10000\mathrm{m}^3 /\mathrm{h}$  ，则净化后液氧中的乙炔含量可降至  $0.2557\mathrm{mg / L}$  是规程规定的液氧中乙炔最高含量的  $64\%$  。此方案可行，既保证了安全，又不致使液氧吸附器的体积太大。经比较验算，现在的液氧吸附器就可以满足要求。这个方案合理、适用、经济。本技术选用这个方案，液氧流量为  $10000\mathrm{m}^3 /\mathrm{h}$

# 二、系统阻力

管路系统设计完成后，主冷和液氧吸附器已经定位，根据管路设计图纸，上塔流向主冷的液氧量为  $32.58\mathrm{m}^3 /\mathrm{h}$  ，即  $0.00905\mathrm{m}^3 /\mathrm{s}_{\circ}$

经计算\*,系统总阻力，即管道阻力、吸附器阻力、液氧过滤器阻力与流量计阻力之和为  $0.3638\mathrm{mLO_2}$  柱。也就是主冷液面高度提高  $0.3638\mathrm{m}$ ，就可使液氧克服系统阻力实现液氧自上塔经吸附器流入主冷。

由前面文中介绍可知，上塔第一块塔板下部制作一个液氧槽，将液氧接住，使其流入管网再流经液氧吸附器，液氧液位差可达  $1150\mathrm{mm}$ 。

因此，流速会比理论计算的流速大，流量会超过  $10000\mathrm{m}^3 /\mathrm{h}$  。说明系统管路设计合

理，可以满足要求。

# 三、设备安装

设备安装从1993年5月10日开始，首先在上塔和主冷结合部开  $\phi 500$  孔，人钻入设备内，按设计安装了液氧槽。为便于液流通畅，液氧槽安装为外高内低，斜度为  $5\%$  可以保证液氧顺利引出。

管路严格按设计进行安装。除垂直管外主管一律采用向上或向下  $5\%$  坡度，保证不产生气塞和液封。各阀门用长杆冷角阀，阀杆向上倾斜  $10^{\circ}$ ，保证密封填料部位的液体能充分汽化，减少跑冷损失。各液面计和压力表管水平向上倾斜  $5^{\circ}$  引至冷箱外壳，然后沿塔皮向上弯作“Ω”形，高度1米，使液氧充分汽化，保证测量的压力或流量准确。

为使液氧能充满液氧吸附器，在安装时确保了液氧吸附器上封头最高位置低于主冷最低液面。

整个安装工作至1993年6月15日结束。

# 五、生产运行

这套空分设备从1993年7月15日开车制氧，8月15日各项试验告一段落，交付生产运行，到1994年3月31日为止已累计运行232天，共生产氧气30624万  $\mathfrak{m}^3$  。中途倒换液氧吸附器加热再生15次，生产一直正常。从未因液氧中的乙炔等碳氢化合物超标而排放液氧，到现在为止仍在正常运行。这套液氧自流净化系统运行也很正常，各个阀门开关灵活，各显示仪表准确、灵敏，操作简便，深受职工欢迎。

经攀钢氧气厂化验室分析，空分中的液氧未净化前其乙炔等碳氢化合物含量平均高达  $67.53\mathrm{mg / L}$  ，处于不安全运行状态。这套液氧自流净化系统投入使用后，液氧中的乙炔等碳氢化合物含量降到了  $30.22\mathrm{mg / L}$  ，较安全规格规定的  $40\mathrm{mg / L}$  低  $24.45\%$  ，处于安全运行状态。分析结果*还表明，应用本净化技术后，本空分设备液氧中的乙炔等碳氢化合物含量，低于日立  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备液氧中的碳氢化合物含量。

# 六、结论

1. 这套国产、 $3^{\#} 6000 \mathrm{~m}^{3} / \mathrm{h}$  空分设备液氧净化技术先进新颖，设计合理，理论计算正确。四川省科技成果查新结论是“国内与该成果密切相关的文献未见报道”，为国内首创。

2. 生产运行证明，其液氧流动量超过  $12000\mathrm{m}^3 /\mathrm{h}$  ，乙炔等碳氢化合物含量低于安全规程规定，确保了空分安全运行。  
3. 年经济效益达216.1万元、社会效益数千万元，同时避免了因乙炔等碳氢化合物超标而造成的人身伤亡事故。  
4. 本技术可在同类空分设备上推广应用。  
5. 本技术有待改进。为使乙炔含量更低，可以加大通过液氧吸附器的液氧流量。  
6. 从主冷引出浓缩后的液氧经净化后再去主冷，效果更佳，液氧中乙炔含量仅为0.003ppm。另外，还可以采用法国空分设备所使用的引射液氧流动法，净化效果也很满意。

# KL-15型制氧制氮车操作微机提示系统

倪其昌

（杭氧集团公司军代表室，杭州市东新路90号，310004）

摘要针对KL-15型制氧制氮车操作过程中，在规定纯度下难以达到并维持最大产量的状况，以设备制氧过程为例，对在操作上可应用的微机提示系统进行了初步探讨，此提示系统可指导操作者进行及时、准确、迅速的操作；并对微机能提示的调整方法、数据的具体处理方法进行了介绍；最后分析了此提示系统的设计思想和程序框图。图2。

主题词：移动式空气分离设备 计算机 提示系统 操作 产量 纯度

# 一、问题的提出

KL-15型制氧制氮车，是部队现役的制氧制氮设备，生产高纯度的氧、氮，满足飞行员的高空用氧及设备的氮保护。该设备构造相对复杂，控制阀门、按钮、开关较多；运行过程中随着时间的变化，零部件性能、效率以及整个系统的状态都会有不同程度的变化；且外界气候的变化也直接影响设备运行的稳定性。为了保证设备稳定运行，在设备运行过程中，操作人员按照产量、纯度受影响的程度进行调整。但由于纯度、产量受影响之初，不易被判断识别，从而延误了相应调整时间。即不易使设备达到并维持最佳运行状态。

本文就KL-15型制氧制氮车在制氧工况时，采用实时采集，检测纯度、产量以及影响纯度、产量和设备安全稳定的各项参数、状态，经过微机处理，用屏显加报警的方法(必要时，可附加打印机)，随时提供操作者一个直观的设备运行状态的文字显示，监控设备的整个运行状态，指导操作者尽可能快地使设备达到并维持最佳运行状态。

# 二、影响KL-15型制氧制氮车制氧工

# 况的因素及可控因素

影响该设备制氧工况的因素很多，但对特定的设备而言，零、部件的加工及装配质量早已确定，外界气候无法人为控制，因此只有通过精心操作，使设备运行达到最佳。

操作人员对设备的影响基本上可分为两类：一是塔内冷量；二是进塔空气量。如何根据设备运行及外界环境的变化，及时相应调整好这两者的关系，才是我们要解决的主要问题。

# 三、改变塔内冷量及进塔空气量的操作方法

通过对制氧时的KL-15型设备的流程分析，我们可以得出改变塔内冷量及进塔空气量的操作方法。

1. 增加塔内冷量及相对冷量的操作方法

（1）提高制冷机工作压力（氢气压力）。  
（2）增强压缩空气冷却效果（包括开散热器风机、打开氮水预冷器）。  
（3）适当延长切换阀的切换时间（主要根据设备运行情况及外界空气湿度，综合考虑，决定延长切换时间。实际操作中一般不

采用)。

（4）在贮液器液面高度允许的情况下，开液氧泵（增加冷量回收）。  
（5）保持冷凝蒸发器的液氧面适当的高度（实际操作中，通-4阀开，液氧面高度不变，一般为10格左右）。  
(6) 减少进塔空气量。

2. 减少塔内冷量及相对冷量的操作方法

与“1”的操作方法相反。

3. 增加进塔空气量的操作方法

（1）关小放空阀。  
(2）开大节-1阀。  
（3）开大通-1阀（实际操作中，开  $3\sim 5$  圈，基本上不调整）。

4. 减少进塔空气量的操作方法

与“3”的操作方法相反。

# 四、作为KL-15型制氧制氮车制氢操作依据所需采集、判断的数据

KL-15型设备操作依据参数，可分为四类：

1. 第一类是输出判断数据

此类数据判断设备生产产品的情况，主要有纯度和产量。

2. 第二类是限制、控制数据

它所起到的作用是保证设备运行的安全性、连续稳定性，延长运行周期；是保证产品符合要求的先决条件。其特点是其值有限定的范围，因此控制时应使其在相应的限定范围内。主要有：（1）塔压  $(0.039\sim 0.055$  MPa)；(2)空压机二级压力  $(0.39\sim 0.55$  MPa)；(3)平均工作压力(氢气压力  $<  2,60$  MPa)；(4)汽泡泵液面指示高度（一般为3格左右，大于5格即会造成汽泡泵无法提升液体)；(5)板翅式换热器中部温度  $(-90\sim$ $-110^{\circ}C)$  。

3. 第三类是状态指示参数

它显示设备中可操作附属设备的工作状

态，主要有：(1)散热器风机开关状态；(2)液氧泵开关状态；(3)氮水预冷器吹除阀开关状态；(4)放空阀开关状态；(5)液氧排出阀开关状态（用来计算产量时用）。

4. 第四类是操作(动作)判断数据

它用于判断微机提示操作后，人工是否进行了操作。此类数据涉及到每一个需要调整的阀门(开关一类，微机可直接判断)，主要有：氮水预冷器打开阀、通-3阀、通-11阀、节-1阀、通-1阀、启动箱吹除阀（吹-4阀）。

以上四类参数、数据中，应首先满足限制、控制数据，在此条件下，保证纯度，提高产量。

根据需要采集的数据，可以列出相应的需要的传感器。

（1）纯度：磁氧分析仪；  
(2）产量：贮液器、液氧面高度传感器，充氧台流量传感器（包括压差传感器、温度传感器）；

（7）板翅式换热器中部温度：（采用设备上已有的）；  
（8）散热器风机开关状态：加接判断电路；  
（9）液氧泵开关状态：加接判断电路；

# 六、设备达到最佳运行状态的调整方法

设备运行中，产品纯度和产量有以下四种情况存在：

（1）氧纯度合格，氧产量合格；  
(2) 氧纯度合格, 氧产量不合格;  
(3) 氧纯度不合格, 氧产量合格;  
（4）氧纯度不合格，氧产量不合格。

由前面的分析可知，塔压、制冷机工作压力是调整设备运行状态的主要手段。首先判断塔压、制冷机平均工作压力是否接近最大值。先设定塔压和制冷机平均工作压力接近最大值的塔压为  $0.05\mathrm{MPa}$ ，工作压力为 $2.50\mathrm{MPa}$ 。这样，塔压与平均工作压力的对应关系可分为四种情况，它反映加工空气量与塔内冷量的平衡关系。

(1)  $0.039 \mathrm{MPa} <$  塔压  $< 0.05 \mathrm{MPa}$ , 平均工作压力  $< 2.50 \mathrm{MPa}$ ; (I)  
(2）  $0.05\mathrm{MPa}\leqslant$  塔压  $\leqslant 0.055\mathrm{MPa}$  ，平均工作压力  $<  2,50\mathrm{MPa};$  （Ⅱ）  
(3)  $0.039\mathrm{MPa} < \text{塔压} < 0.05\mathrm{MPa}, 2.50\mathrm{MPa} \leqslant$  平均工作压力  $\leqslant 2.60\mathrm{MPa}$ ; (Ⅲ)  
（4）  $0.05\mathrm{MPa}\leqslant$  塔压  $\leqslant 0.055\mathrm{MPa}$ $2.50\mathrm{MPa}\leqslant$  平均工作压力  $\leqslant 2.60\mathrm{MPa}_{\circ}$  （V）

# 1. 情况I分析和调节步骤

根据纯度优先原则，无论产品纯度、产量如何：①增加进塔空气量，按第三节操作方法的“3”执行操作（以下用三、“3”表示）。这样纯度、产量都会提高，特别是对纯度的提高比较有利；②判断纯度、产量，按情况I、Ⅱ执行；③直至最佳。

# 2. 情况Ⅱ分析和调节步骤

塔压已接近最大值，工作压力还没有达到最大值，判断纯度。

纯度好：①增加进塔冷量，按三、“1”执行；②判断塔压、工作压力，按I、Ⅱ、Ⅲ、Ⅳ情况操作；③调整过程中，如发现纯度不好，按情况Ⅱ的“纯度不好”操作；④直至最佳。

纯度不好：①减少进塔空气，至塔压接近最低值，按三、“4”执行；②减少塔内冷量，按三、“2”执行，使塔压达  $0.05 \sim 0.055 \mathrm{MPa}$ ；③判断纯度，纯度不好，重复①、②操作，直至纯度合格；④按情况Ⅱ“纯度好”操作。

# 3. 情况Ⅲ分析和调节步骤

塔压未接近最大值，工作压力已接近最大值：①增加进塔空气量，按三、“3”操作，直至塔压达到  $0.05\sim 0.055\mathrm{MPa}$ ；②按Ⅳ操作。

# 4. 情况IV分析和调节步骤

塔压、工作压力都已接近最大值，判断纯度。

纯度好：继续判断还有增加冷量的方法没有？若有：①增加塔内冷量，按三、“1”执行；②判断塔压，按Ⅲ、Ⅳ执行；③重复Ⅳ，直至最佳。若没有，则已是最佳。

纯度不好：①减少进塔空气量，按三、“4”执行，至塔压接近最低值。②减小工作压力，使塔压至  $0.05 \sim 0.055 \mathrm{MPa}$ ；③判断纯度，不好，重复①、②操作，直至纯度合格。④按Ⅱ、Ⅳ操作。

四种情况的分析调节，要达到的目的是保证纯度、提高产量。每次调节后，都应由微机采集检查限制控制数据（分别延迟一定时间，以便设备响应操作），然后判断，提示操作方法，人工调节，以利于设备的安全、稳定运行，然后转入下一步操作，直至产量和纯度都合格。操作幅度可依据偏离设定值的量，采用适应模糊方法确定，最终既快又准确地使设备达到最佳运行状态。

判断最佳运行状态，主要需满足以下两

个条件之一（在纯度合格的前提下）：（1)放空阀全闭（空压机加工空气全部入塔）；（2）制冷机工作压力达到最大允许值，两只散热器风机全开，氮水预冷器打开，在贮液器液面高度允许的情况下(气氧工况)，开液氧泵。

# 七、操作微机提示系统采集数据的具体处理方法

采集数据可从设置的传感器、判断电路转换获得，产量数据需另外处理。以下介绍产量的具体处理方法。

1. 首先确定冷凝蒸发器液氧面高度与体积的关系曲线；贮液器液面高度与体积的关系曲线（采用实测法，在微机内尽可能多地输入对应值）。  
2. 实际操作中，通-4阀常开，贮液器液面有波动。为了保证检测采集的数据相对准确，采取如下措施：

（1）控制贮液器液面高度在  $3\sim 14$  格之间，因为根据设备情况，若贮液器液面高度超过14格，压力波动就比较大，液面高度就不易被测准(采取措施，放液或开液氧泵)。  
（2）程序上控制。采集贮液器液面高度值，3秒钟内采集10个点，取平均值，以便获得相对精确的值（为了简化设计，通-6阀开启时，产量计算以通-6阀开启之前时刻值为准）。  
3. 产品的输出量。现有设备主要用充瓶法或称重法测定产量，这样既不精确，又影响判断速度，延误调节时间。根据设备的情况，可在充氧台处加一气体流量传感器（其中包括压差传感器，温度传感器）。这样，在管路截面积已知的情况下， $\Delta t$  时间内通过管道的气体量，就可以用公式计算累加得出  $W_{\text{泵} \Delta t}$  （标准状态下）。  
4. 产量的确定。设  $t$  时刻：冷凝蒸发器、贮液器液面高度分别为  $h$  凝、 $h$  贮，查液面高度与内在体积曲线得对应量为  $W$  凝、 $W$  贮（标准状态下）。

$t + \Delta t$  时刻：  $h$  凝T、  $h$  贮T，对应W凝T、 $W$  贮T(标准状态下)。

$t \rightarrow t + \Delta t$  时间内：液氧泵打出的氧气为  $W_{\text{泵} \Delta t}$ 。

$$
\begin{array}{l} t \rightarrow t + \Delta t \text {时 间 内 的 产 量} \\ \left(W _ {\text {凝} T} + W _ {\text {贮} T}\right) + W _ {\text {泵} A t} - \left(W _ {\text {凝}} + W _ {\text {贮}}\right) \\ \end{array}
$$

每小时产量

$$
\begin{array}{l} \left[ \left(W _ {\text {凝} T} + W _ {\text {贮} T}\right) + W _ {\text {泵} d t} \right. \\ - (W _ {\text {凝}} + W _ {\text {贮}}) ] / \Delta t \\ \end{array}
$$

由表可知，杭氧新  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备采取了新工艺、新流程。下面对其工艺特点进行分析。

# 一、采用分子筛流程和增压透平膨胀机

# 1. 空分设备操作简便

空分设备的启动不再采用可逆式换热器流程的启动法——分步冷却法，启动操作过程大大简化，启动时间缩短。冷箱内设备得以简化，取消了自动阀箱、液空吸附器、液氧吸附器、液氧循环泵及相应的仪表、管道等，冷箱内维修量下降，空分操作更加方便。

# 2. 加工空气切换损失减少，主塔运行稳定

分子筛不需频繁切换，所以切换阀故障率大大下降，加工空气切换损失也从可逆式流程的  $4\%$  下降到  $1.5\%$  。同时，消除了频繁切换所造成的压力波动，精馏塔内压力稳定，有利于精馏，提高了氧气提取率。

# 3. 设备运转周期延长

影响空分设备运行周期的主要因素，是进塔空气中  $\mathrm{CO}_{2}$  的残留量，由于分子筛吸附器的净化效果为：  $\mathrm{CO}_{2} \leqslant 0.2 \mathrm{ppm}$ ，而可逆式换热器的净化效果为：  $\mathrm{CO}_{2} \leqslant 10 \sim 20 \mathrm{ppm}$ 。所以分子筛流程空分设备的运行周期较长，可达两年以上，而可逆式流程的空分设备的运行周期只有一年。

加工空气中的  $\mathrm{H}_2\mathrm{O}$  、  $\mathrm{C}_2\mathrm{H}_2$  和  $\mathrm{CO}_{2}$  等杂质被分子筛吸附清除，主换热器不再承担清除杂质的任务，所以主换热器的使用寿命大大延长。

# 4. 分子筛吸附系统的电耗增加

为了减少进吸附系统空气的水含量，降低分子筛的吸附负担，就必须降低进分子筛吸附器的空气温度。新  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备的空气冷却系统中增设了一套冷水机组，相应电耗增加  $220\mathrm{kW}$  ；且分子筛再生用电加热器的电耗  $265\mathrm{kW}$  。从表可以看出，分子筛流程，其冷箱外设备增多，从而维修量和电耗随之增加。

# 氩气干燥器的改进

# 王利寅冬生

（大庆石油化工总厂水气厂空分车间，黑龙江大庆市卧里屯，163714）

# 一、改进的依据

我车间  $10000\mathrm{m}^3 /\mathrm{h}$  空分设备，是从德国林德公司成套引进的分子筛流程、带氩气提取生产装置，氩气生产能力为  $275\mathrm{m}^3 /\mathrm{h}$  。自1985年开工以来，氩气生产一直不稳定，主要是脱甲烷塔塔板堵塞，塔板阻力超高，系统吃不进量，迫使停车加热。氩气生产周期最长2个月，最短18天。

分析塔板堵塞的主要原因是无氧氩气中 $\mathrm{CO}_{2}$  含量超高。干燥器工作末期分析  $\mathrm{CO}_{2}$  含量，竟达  $6.3\times 10^{-8}$  。那么粗氩气中的  $\mathrm{CO}_{2}$  是从何而来的呢？粗氩净化除氧我们采用  $\mathbf{H}_2$  。 $\mathbf{H}_2$  是化工厂裂解后的副产品的主要成分， $\mathbf{H}_2$  占  $95\%$  ， $\mathrm{CH}_4$  占  $5\%$  。若  $\mathbf{H}_2$  中  $\mathrm{CH}_4$  按  $3\%$  计算，则  $15\mathrm{m}^3 /\mathrm{h}\mathrm{H}_2$  中  $\mathrm{CH}_4$  含量为  $0.45\mathrm{m}^3 /\mathrm{h}$  。脱甲烷塔精馏脱除后，仍然有微量  $\mathrm{CH}_4$  循环进入除氧炉。在除氧炉  $450^{\circ}C$  高温下， $\mathrm{CH}_4$  与氧气发生化学反应生成  $\mathrm{CO}_{2}$  和  $\mathrm{H}_2\mathrm{O}$  。我们干燥器所采用的吸附剂是4A球型分子筛，已连续使用7年之久，它对水、  $\mathrm{CO}_{2}$  的共吸附性能较差，故有少量的  $\mathrm{CO}_{2}$  进入换热器、精馏塔并冻结。

活塞式膨胀机制取冷量的多少，除与高压P1、进气温度T1有关外，还与进气量有关。而进气量的多少直接与进、排气阀的调整有关。为了提高制冷能力，减少冷量损失，活塞式膨胀机的设计和调整都不能忽视进、排气提前角。一般需有  $8^{\circ} \sim 10^{\circ}$  进气提前角， $5^{\circ} \sim 8^{\circ}$  排气提前角。

我单位所购的活塞式膨胀机，按照其说明书，“进、排气阀门调整阀杆与顶杆之间的间隙的方法为：①调进气阀时将活塞放在上死点，调排气阀时将活塞放在下死点；②产量调节器调至零位；③放松中间顶杆上之防松螺母；④在中间顶杆与阀杆端部之间用厚薄规测出间隙，若间隙不对，可调节顶杆上的调节螺钉来保证，最后将锁紧螺母固紧即可”。

若按说明书提供的方法调整间隙，那么活塞在上、下死点时，进、排气阀不但不能提前进、排气，反而因为存在阀杆与顶杆间隙  $(0.02\sim 0.10\mathrm{mm})$  、阀杆与阀头间隙（0.30 $\sim 0.40\mathrm{mm}$  )，要待曲轴继续转动，阀杆继续上升，消除了这些间隙之和后，阀门才能开启。这时进、排气阀实则是滞后开启，于是就损失了很多冷量。因此，上面引摘的说明书中  $①$  的写法是错的。应该是：调进气阀时将活塞放在下死点，调排气阀时将活塞放在上死点。

设计者在凸轮与曲轴相对位置的配合中，已经考虑了进、排气提前角，即在活塞

处于上(下)死点时，顶杆已处于凸轮半径变化最大的斜坡上，这时，阀门已有所开启（见图）。若按说明书的调整法，此时阀杆与顶杆有间隙，需凸轮转动、顶杆上升、消除间隙，进、排气阀才能开启。显然，此时阀杆已被顶起  $0.32 \sim 0.50 \mathrm{~mm}$  。当凸轮将顶杆顶至最高点、阀门开至最大时，由于上述间隙及斜坡高度的影响，阀门的最大开度比设计的最大开度小，当然进气量也比较少。

# 氧压机一级活塞的改进

王宇雷

（营口化学纤维厂设备处，辽宁省营口市，115001）

我厂有五台3Z3.5-1.67/150型氧气压缩机，使用过程中一级活塞经常出现冲击现象，严重时造成机身振动异常，机器不能正常运行，生产中每隔两个多月就需停机检修活塞一次。使维修工作频繁，影响了生产。

# 一、存在的问题

# 1. 氧气对铝质活塞有强烈的氧化作用

氧压机压缩的是高纯度氧气（氧气纯度 $99.5\% \mathrm{O}_2$ ），经压缩后的温度达  $100^{\circ}\mathrm{C}$  以上。该状态的氧气对铝有着强烈的氧化作用，在铝活塞表面形成的氧化层阻止了继续氧化，而活塞沉孔在活塞的往复运动过程中受连续的冲击，氧化层不断遭到破坏，氧化腐蚀继续深入。这样降低了沉孔表面强度，加快了沉孔平面磨损速度。

# 2. 活塞轴向间隙过大

氧压机在设计上为了保证活塞在气缸内自动定心，活塞与活塞杆联接处留有  $0.04\sim$ $0.06\mathrm{mm}$  轴向间隙，这就使活塞在往复运动过程中不断承受冲击力，冲击力随着活塞轴向间隙增大而增加。虽然沉孔衬有钢垫，但钢垫面积小，又能在沉孔内移动，在受到活塞杆的冲击时沉孔处应力大磨损快。另外，活塞杆与活塞接触不匀，使沉孔局部应力过大，产生局部压缩变形。

# 3. 活塞杆接触面积过小

活塞杆大头下平面与调节压盘接触面积小，磨损速度快。

根据以上分析，为解决氧压机一级活塞维修频繁的问题，保证氧压机长周期运行。1992年7月，我们对氧压机一级活塞进行了

如下改进（见图）。

# 二、改进的方法

1. 把一个外圆加工出几道环槽的3Cr13衬套浇铸在活塞体内，使活塞体与衬套成为一体。这样改进不仅避免了氧气对铝活塞沉孔平面的氧化腐蚀，也避免了硬度低的活塞体沉孔平面磨损。  
活塞体沉孔直径由原  $\phi 75\mathrm{mm}$  ，改为  $\phi 140$  mm。这样改进，增加了活塞体沉孔受力面积，提高了接触强度。  
2. 活塞杆大头尺寸，由  $\phi 60\mathrm{mm}$  改为  $\phi 75$  mm。增加接触面积，降低磨损速度。

氧压机一级活塞经过改进后，效果明显。活塞维修周期由原来2个多月提高到2年以上。保证了氧压机正常运转，使维修工作量大大减少，也减少了维修费用。

（1995年2月）

# 填充聚四氟乙烯阀片的应用

万景隆

（化工部光明化工研究所，辽宁大连甘井子 116031）

压缩机的各级气阀，是压缩机的重要部件，它的质量直接关系着压缩机的工作效果和能否正常运行。目前气体压缩机的网状阀片多为钢质阀片，它具有强度高、密封性能好的优点，但在生产中也存在着不利于生产的问题。

阀片易破损。光明所空分车间1-6.66/200型空气压缩机，曾出现一只气阀一天更换三次阀片的情况。另外，阀片碎块进入气缸，会划伤气缸，轻者影响压机效率，重者造成气缸报废。光明所在1988、1990年就发生过阀门碎片划伤气缸的事故，划伤深度达 $2\mathrm{mm}$ ，造成气缸报废。

为了克服钢质阀片的缺点，确保压缩机安全连续运行，从1987年开始，光明所进行了以非金属阀片代替钢质阀片的探索。

# 一、聚四氟乙烯阀片的应用

首先选择用聚四氟乙烯阀片代替钢质阀片，主要考虑到如下两点：

（1）在工业阀门生产中，用聚四氟乙烯做阀口或阀芯的已很普遍，实践已证明它具有良好的密封性。  
（2）从资料上看，聚四氟乙烯(模压板)的使用温度为  $-180\sim 250^{\circ}\mathrm{C}$ ，气体压缩机每级气体出口温度均小于  $170~^\circ \mathrm{C}$ ，所以它能满足要求。

经半年时间，对30只阀片的试验表明它存在两个问题：其一，由于材质较软，加工后的阀片平整度不够，造成密封性能不理想；其二，阀片和阀口的密封面较窄，尽管使用温度没有超过使用上限，但在  $120\sim 165^{\circ}C$  温度下，反复与阀座撞击，阀片逐渐产生变形，造成密封不良。

# 二、填充聚四氟乙烯阀片的应用

针对聚四氟阀片存在的问题，开展了填充聚四氟阀片的试验，主攻点是强度和耐热性。无疑，强度及耐热性，与填加成分、填加量有关。

（1）从上述基本观点出发，首先试用了聚四氟乙烯、玻璃纤维、铜粉三合一阀片。经试用这种阀片强度可以满足要求，也有良好的耐热性，但阀片较脆，在阀片与阀座的撞击过程中易碎裂。这种阀片在性能上还不能完全满足生产的要求。  
（2）为了克服碎裂问题，试用了由聚四氟乙烯和玻璃纤维按一定比例配制的阀片。经近八个月的试验证明，这种阀片保持了聚四氟的柔性，并具有足够的强度，阀片有良好的密封性。试用阀片在八个月的试验中，有  $83\%$  的阀片完好如初，现仍在运行中，其余  $17\%$  阀片最短使用时间6个月。  
（3）为了提高阀片的质量，阀片的结构也做了适当修改：①加宽了密封面；②增加了筋的数量。  
（4）使用填充四氟阀片时，阀门不再用弹簧，这点在使用中应特别注意。

# 三、结论

1. 填充四氟阀片有良好的密封性，强度与耐热性，都能满足气体压缩机对气阀的要求，它可替代钢质网状阀片。  
2. 填充四氟阀片不损伤阀座，即使破碎也不会损坏气缸，所以可减少阀座维修或更新的费用。  
3. 填充四氟阀片比钢质阀片加工容易，成本低。

（1994年11月30日）

# 离心式压缩机轴瓦损坏的分析与处理

# 主题词：离心式压缩机 巴氏合金 轴瓦 主轴 材质

# 二、事故的经过与处理过程

1. 该机是石家庄炼油厂KDN-1000型高纯氮设备的配套机组，由北方安装公司安装。1994年9月2日进行安装后的首次试运转，运行约3小时，一、二级轴承温度突然升高，同时发现机器振动增大，在该级轴承处箱盖上测得振幅为  $0.02\mathrm{mm}$ ，停机检查发现Ⅰ-Ⅱ级轴瓦烧毁（Ⅲ-Ⅳ级轴瓦完好）。

2. 9月4日接到用户电话，9月5日我厂技术人员和工人师傅赶赴现场，对安装情况、润滑油质量、轴承间隙等进行了详细的检查。发现：（1)大齿轮偏载，啮合线偏向一级侧，检查Ⅰ-Ⅱ级转子的水平度，发现一级侧低；（2）Ⅰ-Ⅱ级轴承压盖的过盈量小于设计值；（3)二级进气喇叭管出现裂纹。

我们重新调整了  $\mathbf{I} \equiv \mathbf{II}$  级转子的水平度、磨了轴承压盖下的垫片，将过盈量调整在  $0.03 \sim 0.05 \mathrm{~mm}$  的设计范围内，补焊了二级

进气喇叭管，换上备用瓦块后于9月8日开始第二次试运转。空负荷运行8小时无异常情况，然后进行负荷试运转，排气压力升至 $0.9\mathrm{MPa(A)}$  ，运行4小时后发现一级轴承温度发生波动，此时检查，发现I、Ⅱ级轴承处箱盖上的振幅，已增大至  $0.025\sim 0.028$  mm。随即停车，开盖检查，发现I、Ⅱ级轴瓦巴氏合金碎裂，部分从瓦体上脱落。当时我们怀疑巴氏合金材质有问题或巴氏合金与瓦体表面结合不牢；同时担心由于两次轴瓦损坏，振动过大，可能造成转子动平衡被破坏，于是决定把I-Ⅱ级转子拉回厂里复校动平衡，同时对巴氏合金化学成分进行化验。结果发现转子残余不平衡量严重超差，而巴氏合金的化学成分符合要求。对瓦体与巴氏合金的结合面进行着色渗透检查，亦未发现异常。

3. 对转子重新做了动平衡，新加工了一组轴承瓦块到现场进行第三次试运转。此次运转约2小时，发现一级轴承温度又有波动，马上停车检查，即发现一级轴承有一瓦块巴氏合金掉块，同时发现轴封齿有磨损，且呈喇叭口形状。由于停车及时，没有造成严重损坏。

4. 这时我们怀疑是不是 I-Ⅱ级转子轴弯曲、联轴器对中值有问题。但在现场用千分表检查轴表面的跳动度，并未超出设计允许范围。我们又对联轴器的对中值进行了调整，将原来电机轴中心与压缩机大齿轮轴中心之间的高度由  $0.42\mathrm{mm}$  调整为  $0.20\mathrm{mm}$  。同时又一次检查了大齿轮的啮合情况，齿面接触情况良好。接着进行第四次试运转，此次运转4小时，仍是一级轴承温度首先出现波动，振动加大。立即停车检查，结果还是一级轴承瓦块巴氏合金损坏。

另外，在以上几次试运转过程中，该机器的噪声较以前同型号机器的噪声大，且不稳定。

5. 以上几次处理，实际上并未找到故障的真正原因。由于现场条件的限制，有些检查诊断工作难以准确进行，于是决定将整机拉回厂里作全面检查。1994年10月10日将整机拉回厂里，对主要零部件的几何尺寸进行了详细检查，复校了动平衡。主要检查结果如下：

（1）I-Ⅱ级小齿轮轴与大齿轮轴的中心距、交叉度均符合要求，涂色检查齿面接触情况良好。  
（2）I-Ⅱ级转子轴各段表面的跳动度，亦不超过允许范围。  
（3）二级叶片扩压器错装。将6069（产品代号)机器上的二级扩压器装在了本机（产品代号6065)上。这两个扩压器的安装尺寸相同，但叶片数  $z_{3}$  、扩压器宽度  $b_{3}$  、叶片进口安装角  $\alpha_{3A}$  均不相同。  
（4）I-Ⅱ级转子动平衡再一次被破坏。I级叶轮校正平面上的不平衡量为  $247\mathrm{g}\cdot$  mm,是允许最大剩余不平衡量  $(9.8\mathrm{g}\cdot \mathrm{mm})$  的25.2倍。Ⅱ级叶轮校正平面上的不平衡量为  $94.5\mathrm{g}\cdot \mathrm{mm}$  ，是允许值的9.6倍，大齿轮轴的剩余不平衡量符合要求。

鉴于这种情况，我们认为可能是二级叶片扩压器错装后，造成二级扩压器进口气流冲角过大（经计算，在设计流量下，二级扩压器进口气流平均冲角为  $+6^{\circ}$ ），造成气体与叶片之间发生冲击，扩压器流道内部发生严重的气体分离，甚至会形成脱离团，出现旋转脱离现象。

由于这种旋转脱离而引起的沿叶轮径向的压力波动，将引起I-Ⅱ级转子的振动，振动的不断加剧，造成轴承损坏，转子动平衡破坏。于是我们更换了二级扩压器，重新校正了I-Ⅱ级转子的动平衡，于1994年10月17日晚9点钟在厂内试车台上进行第五次试运转，半小时后排气压力升至  $0.9\mathrm{MPa}$ ，至晚10点40分，发现一级轴承处箱盖上振动振幅

增大到  $0.011\mathrm{mm}$  （正常值  $\leqslant 0.006\mathrm{mm}$ ），比允许值大将近一倍，立即停车。

10月18日上午开盖检查，又是一级轴承有一瓦块的巴氏合金出现裂纹，部分巴氏合金已被挤出瓦块端面。下午检查转子的动平衡情况，发现I-Ⅱ级转子的动平衡又一次被破坏，一、二级叶轮校正平面上的不平衡量分别为  $391.5\mathrm{g}\cdot \mathrm{mm}$  、  $1012.5\mathrm{g}\cdot \mathrm{mm}$  ，各为允许值的39.95倍和103.3倍。在磨床上顶住轴两端的顶锥孔检查轴的弯曲度，发现两端叶轮外缘跳动值分别为  $0.07\sim 0.08\mathrm{mm}$  中部小齿轮外圆跳动值达  $0.15\mathrm{mm}$  。显然，轴已发生弯曲，此轴已不能继续使用，于是厂里决定废弃此轴，另外赶制一套新的I-Ⅱ级转子。

新转子制成后，于1994年10月30日晚开始在厂内进行了无负荷8小时和负荷48小时试车，各项性能指标均达到设计要求。该机自1994年11月起已在用户投入生产运行。

# 三、分析

1. 这次故障持续时间长达两个月之久，前后试运转达5次之多，虽然每次试运转前对故障原因都进行了分析，并作了相应的处理，但均未能避免故障的再次发生，说明造成故障的隐患是很深的。从最终轴发生弯曲的情况看，结合前面试运转过程中转子动平衡多次遭到破坏、轴封齿磨损呈喇叭口形状的情况来分析，我们认为轴的刚性不足是本次故障的根本原因。这一点在我们对原轴取样试棒硬度检验后便得到了证实。

经检验，轴的一级端靠近叶轮处的硬度为HRC24，二级端为HRC29，一级端硬度明显低于设计值（原设计值为  $\mathrm{HRC}28\sim 32$  现已提高)。由于轴的刚性不足，在静止状态轴不弯曲（前面几次对轴弯曲度的检查即是这样)，在作低速动平衡时，剩余不平衡量亦

符合要求。但当转子处于超二阶临界转速运转时，轴已变为柔性轴，转子在不平衡惯性力的作用下，必然发生比较大的弯曲变形。由这种弯曲变形而引起的惯性力  $\mathrm{My}\omega^{2}$  要比不平衡惯性力  $\mathrm{Me}\omega^{2}$  大得多，在这种惯性力作用下，轴除了绕自身轴线转动外，同时又绕其在轴承中的平衡位置涡动，轴的刚性越差，这种涡动的振幅就越大。特别对H100压缩机来说，它的悬臂端比较长，因而由此引起的振动会更大，这不仅破坏了转子自身的动平衡，而且造成轴承的多次损坏。

之所以后面几次试运转都是一级轴瓦首先被破坏，主要是由于轴的一级端的硬度比二级端低、刚性差，加上一级叶轮质量大，在工作转速下一级端的挠曲变形也大，因而产生的惯性力也大，振幅较大。

2. 前四次试运转过程中出现的异常噪声，是由于二级扩压器装错造成的。由于二级扩压器装错，使得二级扩压器的实际宽度  $b_{3}(14.3\mathrm{mm})$ ，比设计宽度（9mm）增大了58.9%，叶片进口安装角  $\alpha_{3A} = 21.5^{\circ}$ ，比设计值(19°)大2.5°，致使在设计流量下扩压器进口处气流冲角达  $+6^{\circ}$ ，这不仅在扩压器进口处，使高速气流冲击叶片发出冲击噪声，而且也严重恶化了扩压器内部的流动，产生气体分离，发出涡流噪声。事实上，在更换了二级扩压器之后的第五次试运转过程中，这种噪声已消失。

# 四、结束语

这次故障的代价是昂贵的，教训是沉痛的。对于离心式压缩机这类高速旋转机械，设计、制造、安装、试运转过程中任一环节上的疏忽和差错，都可能造成重大事故，必须予以高度重视。

（1995年3月）

# $6000\mathrm{m}^3 /\mathrm{h}$  空分设备主冷泄漏的原因分析

贺旭光

（陕西兴平化肥厂空分车间，陕西兴平市，713100）

摘要 介绍  $6000 \mathrm{~m}^{3} / \mathrm{h}$  空分设备主冷泄漏的现象；着重分析了泄漏的原因；液空吸附器大量进水，硅胶的性能降低，吸附作用下降，从而使碳氢化合物大量进入分馏塔，并积聚在主冷凝器，引起主冷凝器发生小爆炸并产生泄漏。图2。

主题词：大型空气分离设备 主冷凝器 液空吸附器 爆炸 泄漏

# 一、现象

1987年12月31日，我厂的  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备经安装调试，加热吹除后一次开车成功。氧产量为  $6400\mathrm{m}^3 /\mathrm{h}$  （纯度高于  $99.5\% \mathrm{O}_2)$  ，氮产量为  $6700\mathrm{m}^3 /\mathrm{h}$  （纯度高于  $99.99\%$  ）。

但是几个月后，空分生产状况发生了急剧变化。上塔下部阻力的指示值在一个月内从正常一下子上升到最高点；上塔压力由原来的0.05MPa涨到0.06MPa以上；而且上塔的液氧排除阀被堵塞。主冷液氧液面计被堵塞而没有指示值。后来，只好用一个“U”形管连接两个分析小阀代替主冷液面计，有时“U”形管也被堵塞，没有液面参数作参考，只好以膨胀机的膨胀量不低于某一值作为依据，实行了“盲目”操作。最后，主冷突然泄漏，只好停车检修。整个运转周期还不足5个月，而且大部分时间是在不正常工况下运行的。

$6000\mathrm{m}^3 /\mathrm{h}$  空分设备的这种生产状况从1988年开始，屡屡发生，其中最长的运转周期也不足7个月。至今主冷凝器漏过三次，第二次泄漏后，主冷凝器已经报废，不能再修。安装一个新的主冷以后，运转了大约3个月，又漏。由于买不到新的主冷凝器，只好将其焊好后继续生产。

# 二、原因分析

主冷的泄漏，是由其多次爆炸引起的。

# 1. 爆炸原因

与  $6000\mathrm{m}^3 /\mathrm{h}$  空分设备配套的加热用蒸汽加热器，一开始就漏得很严重，蒸汽进入加热器后冷凝成水，积存在加热器内。可能是在加热某一设备或转换液空吸附器时，操作上失误；也可能是由于当换热器停用时，没有关紧蒸汽阀等原因，使冷凝水克服了管线的阻力进入加热总管，这些冷凝水就进入了液空吸附器。

液空吸附器的加热管是从加热总管的下部引出，两根管成“T”字联接，如图1。而其它设备的加热管从中部或上部引出，因此液空吸附器进水的可能性极大。再者，生产中液空吸附器频繁地倒换、加热，难免进水。

![](images/d16030c1aff7002362c7ee9cb5b63039179b914e0d7167ef588874448ef5a100.jpg)  
图1 液空吸附器加热管与加热总管的布置图

实际现象也证明了这个可能。过去倒换液空吸附器只需一个班的时间，而且预冷5

～6小时就有液体从排除阀排出，这证明液空吸附器已被预冷好。可是那时，即使预冷好几天，也无液体从排除阀排出。有人怀疑是液空吸附器有漏点，但是扒开珠光砂检查，没发现任何漏处。后经分析，认为是因为液空吸附器进水以后阻力大增，流入液空吸附器的液空量减少，又因液空吸附器的排除小阀管径很小，所以，当流入的液空量小于汽化量时，排除阀排出的总是气体。

操作工后来开大液空吸附器的进出口大阀，使流入的液空量大于汽化量，便有液体从排除阀中排出。表明液空吸附器已被预冷好，才得以倒换。

液空吸附器虽然倒换过来了，但由于大量进水，器内硅胶对碳氢化合物的吸附能力大大降低，甚至完全丧失其吸附性能。随着时间的推移，塔内的有害杂质越积越多，这些杂物不仅堵塞了塔板、管线和阀门，还粘附在主冷凝器的肋片上。主冷凝器爆炸的原因就在于此。

# 2. 由爆炸引起主冷泄漏

主冷的泄漏都是在瞬间突然发生的，且每次都有七、八个，甚至几十个通道同时泄

漏, 这显然是由爆炸引起的。而一般的泄漏, 通常由  $1 \sim 2$  个通道开始漏, 逐渐发展, 有一个时间过程。

再者，“制氧机黑匣子”即仪表纸清晰地显示出了爆炸时氧、氮流量表的流量曲线（图2）。一般泄漏，其氧、氮流量表的画线是从小到大，即流量从  $6000\mathrm{m}^3 /\mathrm{h}$  增加至8000 $\mathrm{m^3 / h}$  。而由爆炸引起的泄漏，其划线则是从大到小，即氧流量由正常的  $6000\mathrm{m^3 / h}$  瞬间增加到头，然后又逐渐减小到  $8000\mathrm{m^3 / h}$  这是因为在爆炸时，产生的高温高压使流量的指示值达到顶峰，当爆炸力消除后，主冷已经漏了，所以流量表指示为  $8000\mathrm{m^3 / h}$  。

# 全板式氨蒸发器损坏原因分析

刘智华

（陕西省兴平化肥厂空分车间，陕西省兴平市，713100）

我厂KDON-3200/3200型空分设备，系开封空分设备厂70年代产品。1992年底，我厂在开封空分设备厂中原制冷公司协助下，将原装置石头蓄冷器中抽流程，改造成氨预冷分子筛吸附工艺流程，取消原笨重的石头蓄冷器，采用板翅式换热器。1992年底改造成功，一次开车出氧，并连续运行至1994年6月。

1994年7月初，发现氨蒸发器气氛返回我厂氨加工系统时，压力逐日上升。到7月20日，气氛压力升至  $0.4\mathrm{MPa}$  以上，经分析，我们判断为氨蒸发器内漏。并从氨蒸发器气氛出口压力表接头处，取样分析气氛浓度，经多次分析气氛浓度只有  $55.2\%$  。而正常情况下，气氛浓度应在  $99\%$  以上。分析数据，进一步证明氨蒸发器泄漏严重。由于空气中氨含量在  $16\% \sim 25\%$  的界限是危险爆炸区，为了保证装置安全运行，我们果断决定，于7月21日中午紧急停车。这样，避免了一次可能发生的恶性事故。

经检查，发现氨蒸发器气氛出口板式通道一侧，颜色为深黄红色，并向另一侧颜色

逐渐变浅。根据直观现象判断，板式氨蒸发器通道发生过局部爆炸，再经打压检查，发现在氨通道侧有9个流道泄漏，在颜色深处有6个通道严重泄漏，并有齿片碎渣脱落。另一侧发现有3个通道微漏，9个泄漏通道占氨通道数的  $35\%$  。因设备损坏严重，一时无法解决设备备件，加之生产急需，经与开封空分设备厂协商，我们将泄漏的9个氨通道上下全部堵死，空气通道保留。

修复后，于1994年7月28日投运，虽然换热面积减少三分之一，但基本可以维持生产。氨蒸发器空气出口温度保持在  $\pm 5\%$  ，氧、氮产量在  $3000\mathrm{m}^3 /\mathrm{h}$  左右，解决了我厂化肥生产急需的用氧用氮。

从这次氨蒸发器损坏现象分析，我们认为氨蒸发器漏后，空气进入氨系统中，产生氢气及一氧化氮。加上我厂液氨含油，故形成爆炸条件，发生氨蒸发器微爆。由于发现及时，并采取相应措施，从而避免了一次更大事故的发生，这在用液氨作冷源的空气预冷系统中，要引起注意。

# 两起分馏塔事故的教训

李树忠

（安阳钢铁公司动力厂，河南省安阳市铁西区梅园庄，455004）

摘要 介绍  $6000 \mathrm{~m}^{3} / \mathrm{h}$  空分设备(带制氮装置)分馏塔发生的两起事故的概况、分析和处理的过程，分析了两起事故的相同及不同之处，最后总结了教训及体会。

# 主题词：空气分离设备 分馏塔 主冷凝器 漏气 局部爆炸 操作频繁 工况恶化

我厂KDON6000/6600-V型空分设备及配套的"4XFS-45-2型制氩装置，均系杭氧1982年产品，1985年10月开始安装，1986年10月18日开车，36小时送出氧气，一次试车成功。经考核，主塔全部指标符合设计要求。1987年2月份开始调试制氩，同年6月生产出了合格氩气。

# 一、两次事故概况、分析及处理

1987年4月和1990年4月，我厂空分塔先后发生了两起事故，分别影响出氧18天和13天，给公司生产带来了很大损失。

1. 1987年2月10日试投粗氩塔，11日15时粗氩产量、质量均达到设计值，12日13时左右，主塔氧纯度突然由原来的  $99.6\% \mathrm{O}_2$  下降为  $99.2\% \mathrm{O}_2$  ，粗氩纯度由  $96\% \mathrm{Ar}$  下降到  $90\% \mathrm{Ar}$ ，并很快发生“氮塞”。后虽经多次调整，粗氩塔仍无法投入使用。初步分析认为，可逆式换热器因操作不当及频繁停车导致自清除不良，且液空、液氧吸附器所装抗水硅胶吸附能力达不到要求，致使大量二氧化碳进入上塔及主冷，破坏了上塔的正常精馏工况。于4月9日停车，对空分系统全面加热。4月18日15时空分投运。20日零时分析主冷液氧和上塔气氧，发现气、液氧纯

度差过大，液氧纯度  $97\%$  ，气氧纯度仅  $84\% \sim 91\%$  。调整到23日12时，纯度仍无好转。因此，判定主冷局部爆炸，停车检修。

1987年5月4日对主冷试漏检查，发现有两个单元漏气，一个单元第15、16、17、18、19通道和另外一个单元第9通道，都泄漏严重。由此说明，检查结果和原先判断是一致的。

2. 1990年4月14日预冷粗氩塔，16日粗氩塔正式投入使用。从8时30分一直到16时55分，对主塔、粗氩塔进行了一系列工况调整，其中包括环流量、膨胀量、氧和氮产量的改变以及预冷液空吸附器等。仪表工先后检修了粗氩塔阻力表、下塔液空液面计，其中下塔液空液面计没有修复好，指示为零。16时55分，即在液空吸附器预冷结束后10分钟，下塔阻力突然满表，主冷液面波动。经调整，17时恢复正常，17时15分突然发现上塔阻力、上塔压力、空气流量、氧氮产量增加，膨胀量、环流量、下塔压力下降。17时30分，粗氩塔停运。自此以后工况急剧恶化，上塔压力、空气流量和氧、氮产量增加，下塔压力和气、液氧纯度下降。虽两次降低氧产量，氧气纯度未好转。18时30分，气氧纯度

为  $69\% \mathrm{O}_2$  ，液氧纯度  $99.5\% \mathrm{O}_2$  。初步分析，认为可能是主冷局部爆炸。空分停车。

当天20时30分左右，为了进一步判断，关闭中压系统通往低压系统的调节阀及其它阀门，进行了一次送气试验。在下塔压力为 $0.34\mathrm{MPa}$  时，上塔压力已达  $0.042\mathrm{MPa}$  。因担心发生意外，停止送气试验。当时气氧纯度为  $50\% \mathrm{O}_2$  ，液氧纯度为  $95\% \mathrm{O}_2$  。因而认为主冷局部爆炸或其它板翅式换热器击穿，造成中低压串气故障的可能性最大，决定停车检修。

1990年4月21日上午对主冷试漏检查，没发现漏点。下午又在有关部位安装盲板，将中低压系统断开，对低压系统进行全面检查，未发现中低压串气现象。经进一步检查后，加温开车，各项工艺参数均正常。由此可见，停车前作出的判断是错误的。

经仔细分析，1990年4月16日工况恶化的主要原因，是由于操作过于频繁，在粗氩塔阻力表投入不及时、液空液面计没有指示的情况下，没有把握好冷量的转移和分配。同时由于缺乏综合分析和综合判断能力，在工况恶化出现潜在威胁时，未能及时发现、及时处理，最后导致工况全面恶化。

# 一起空压机撞缸事故的分析

王增桥

（烟台冶金新材料研究所，山东烟台开发区，264006）

摘要 本文介绍了 DY8-30/15 型空压机发生撞缸事故的经过及损失情况。事故原因主要有三个：活门压套有缺陷；装配时草率行事，选用软质垫圈；操作经验不足，遇到故障处理不当。最后介绍了处理情况及从中吸取的教训。

# 主题词：空气压缩机 撞缸 气缸体 气阀 垫圈

我所现有一套KZON-300/1000-2型空分设备，是由KFZ-1800型空分设备改造而成的，配套空压机为邯郸制氧机厂1978年生产的DY8-30/15型空压机。该空压机闲置多年，保管不善，虽经检修，但仍存在一些毛病，再加上操作维护经验不足等原因，运转未满一个月就发生一起严重的撞缸事故。现介绍如下，以供借鉴。

# 一、事故经过及损害情况

1991年8月21日凌晨1时17分，空分设备运行中，突然空压机一级冷却器上安全阀起跳。现场操作工闻声后，没弄明白缘故，就急急忙忙打开一级油水吹除阀卸压。就在此同时，听到气缸中发出异响，并看见配套同步电机定子线圈处打火花，随即迅速断电停车。

当日上午，打开二级进气活门盖进行检查，发现其中一个活门压套碎成数块，进气活门翻转  $90^{\circ}$ ，部分碎片已被吸进气缸。卸下三级气缸，取出二、三级活塞体，发现活塞

粗端（即二级）一侧端面被撞了一个大窟窿，已无法修复，整个二、三级活塞体报废，而气缸端面也被撞出凹痕。

# 二、事故原因

# 1. 活门压套有缺陷

在空压机中，该活门压套件设在活门与活门盖之间，借助活门盖上紧固螺钉传来的力来固定进、排气活门，其材质为HT20-40。因考虑进出气缸空气的流通，在压套柱面上设计有4个长方孔道，面积大约占整个压套柱面的一半。这种特殊的结构和材质，决定了该压套承载能力不能很高。又经检查发现，该压套整体厚度偏薄，而且在柱面各方孔拐角部位，铸造圆角过渡粗糙，甚至个别还存在砂眼、疏松及裂纹等缺陷。因此活门压套发生破碎，首先是由其先天不足的质量因素造成。

# 2.检修装配草率行事

一个活门压套，在工作中由四只螺钉固定，这决定压套上要有4个受力点，而活门

烃含量一般都在报警极限上，频繁排放液氧的后果是极难维持正常生产。

我们的体会是：(1)说明书上的规定偏于保守；(2)说明书上只规定了  $\mathbf{C}_2\mathbf{H}_2$  和总烃

含量指标，没有规定  $\mathrm{CH}_4$  、  $\mathrm{C}_2\mathrm{H}_6$  、  $\mathrm{C}_2\mathrm{H}_4$  、  $\mathrm{C}_3\mathrm{H}_8$  等单项指标；(3)还应规定  $\mathrm{CO}_2$  等其它杂质的指标。

（1995年2月）

压套因设有4个长方孔，相应存在16个拐角。由前面分析知道，压套铸件在拐角处存有隐患，若在检修装配时将螺钉的着力点正好对准了这个薄弱处，这时紧固用力稍微偏大，便难以保证压套不出问题。另外，如果在四只螺钉加力不均或者紧固不到位的情况下，由于活门启闭及空气压差的作用，活门和压套便产生振动，从而导致相对位置的转动和构件间的轻度撞击，时间一长，同样会造成压套件脆性及疲劳破坏。

还有一种几乎被疏忽但也许是导致这次事故的最主要原因，那就是在活门与气缸体之间使用了软垫圈。经检查核实，当时在缺乏必要的备品备件的情况下急于投入生产，盲目地采用  $\phi 4\mathrm{mm}$  直径的铝丝来制作密封垫圈。认为软质垫圈可以压实而产生较好的密封效果，但忽视了该垫圈在活门因空气压差产生的周期性冲击力的作用下，由于强度太低而出现形变的事实。因而导致活门及压套发生松动和撞击，最终造成严重破坏。

以上几个原因分析说明，在保证压套有一个确定安装位置的前提下，正确选用配套硬质密封垫圈及保证紧固力大小均匀适度，是活门和活门压套安全、稳固的关键。所以说发生此类事故与有关人员检修、装配的经

验和水平有直接关系，不能麻痹大意。

# 3. 操作工缺乏经验，处理不当

由于工人大多是刚上岗的新手，技术水平低，面对紧急情况反应迟钝，判断不准，因而没能及时采取停车措施制止事故的扩大。

# 三、事故的处理与教训

1. 针对这次事故的客观原因，一方面我们加紧同制造厂家取得联系，除购进因事故损坏而急用的零部件外，进一步落实了备品备件的供给来源，同时对厂家提出了改进产品质量的建议和希望。另一方面，我们对造成这次事故的可能原因进行认真地分析，对整台空压机进行全面的检查，在活门压套备件不足的情况下，为早日恢复生产，对个别已发现出现裂纹的活门压套进行了铸铁焊修补以增强强度。淘汰软铝垫圈而配以硬质垫圈，同时落实专人，在检修过程中严格把关，杜绝类似事故再次发生。

2. 针对工人技术素质低、规章制度执行不严的现状，我们从事故中吸取教训，通过举办基本知识及技能培训班，从安全生产的角度出发，提高工人发现问题、解决问题的能力。同时严格落实岗位责任制，加强监督管理，争取从安全生产中获取实惠和效益。

（1995年3月改稿）

# 4M12-59/30型氧压机

# 燃 爆 事 故 及 预 防

章进儿

（上海吴淞化工总厂制氧分厂，上海市长江西路501号，200241）

上海吴淞化工厂制氧分厂是全国较早的制氧专业企业，主要为上海第一钢铁厂提供炼钢工业用气，输氧设备为4M12-59/30型氧气压缩机。几年来，发生多起氧压机三级气缸燃爆事故。事故发生时，火焰可窜至3～4米，活塞杆、密封函等配件严重烧毁，严重影响了生产，并危及人身安全。

# 一、原因分析

从事故现场看到，密封器冷却水管严重结垢，三级活塞杆烧断，密封器由气缸向机身方向呈放射形烧毁。据分析，原因有以下三条：

第一，由于4M12-59/30型氧压机，油密封效果较差，润滑机油或多或少渗入气缸密封函内。我们曾着重对三级气缸密封器进行检查，发现运行半年以上的机组气缸密封器内，均不同程度积有机油。在高浓度氧气氛围中，这种易燃物质是造成氧压机燃爆的主要因素之一。

第二，气缸密封函的冷却是由  $\phi 12\times 1.5$  mm紫铜管引进自来水来实现的。由于冷却水系统为封闭式供排水，流速较小，造成水管水垢堵塞，冷却水不畅，致使密封函冷却效果极差。随着活塞杆与密封函之间摩擦，温度急剧升高，增大了事故引发因素。

第三，由于上述原因，气缸密封函的密封性能恶化，造成高浓度氧气泄漏，加剧燃爆事故的发生。

# 二、事故预防

从上述原因分析中可看出，氧压机气缸燃爆事故主要由易燃机油、高浓度氧气、局部高温这三个要素形成。为此，我们针对事故采取如下措施：

1. 定期对各级气缸密封器进行脱脂。半年为一周期，安排气缸密封器全面检查，严格脱脂，清除易燃物质以及密封盒水道污垢，提高每只密封盒的冷却效果。  
2. 建立每周一次的脱脂制度，即每周定期对各级活塞杆进行脱脂。管理人员逐台检查、督促。  
3. 改进气缸密封函冷却水管。将原密封函冷却水管封闭型改成敞开式。使之流速增大，减小水垢沉积，有利于观察冷却水工作状况，便于及时排除管道堵塞。  
4. 对润滑油密封装置一旦发现渗漏，及时检修和堵漏。

# 蓄冷器产品盘管破裂事故的处理

肖伦辉

（四川威远钢铁厂制氧车间，四川威远，642469）

1994年9月25日，我厂KDON-3200/3200-I型空分设备，发生一起因蓄冷器氧产品盘管破裂引起的氧气纯度降低事故。经封堵破裂盘管的应急处理方法，使设备及时恢复正常生产。

# 一、事故经过

1994年9月25日，早班操作人员进行正常分析时，发现氧气纯度发生周期性变化，变化幅度在  $97.6\% \sim 99.6\%$  之间，引起了操作人员的注意。

# 二、分析及处理过程

经过分析，排除了分析仪漏气、氧压机进出口漏气等可能会引起纯度降低的原因，就决定对蓄冷器进行一个周期的连续分析。分析发现  $\mathbf{I}^{\#}$  、  $\mathbb{I}^{\#}$  、Ⅲ#蓄冷器走正流空气时氧气纯度为  $99.6\%$  ，而  $\mathbb{V}^{\#}$  蓄冷器走正流空气时氧气纯度为  $97.6\%$  ，两者相差达  $2\%$  。由此判断可能是  $\mathbb{V}^{\#}$  蓄冷器氧产品盘管漏气，即当其走正流空气时盘管内外压差很大，中压空气漏入产品氧中，引起产品氧气纯度下降；而走返流污氮时，因同属上塔来的气体，压差很小，且上塔底部压力高于上部，故几乎不影响产品氧气纯度。因此，蓄冷器氧产品盘管漏气对纯度的影响，只表现在1/4切换周期内，不易被及时发现。

通过分析蓄冷器结构和设备运行特点，

决定首先由 IV*蓄冷器顶部的人孔扒出部分瓷球进行检查。检查人员从蓄冷器热端人孔进入设备后发现，有八根  $\phi 19 \times 1.5 \mathrm{~mm}$  紫铜盘管严重变形破裂（其中一根已断裂），正对空气入口处的铜皮罩已被掀翻。

原因找到后，因现场不允许施焊，但为了及时恢复生产，只好采用将受损盘管两头封堵的应急措施。另制作铜皮保护罩，固定在铜管变形部位，待设备计划中修时再作进一步处理。[注]

处理后，装填满瓷球，启动运行，分析Ⅲ#、IV#蓄冷器走正流空气时产品氧气纯度，均为  $99.5\%$  ，设备运行正常。

# 三、事故原因

1. 受损盘管正对着空气进口，运行中蓄冷器瓷球不及时添加，切换时气流窜动冲击瓷球，瓷球冲击盘管和保护罩，使其变形和磨损。  
2. 蓄冷器热端上部瓷球装填不满，造成周期性切换窜动撞击。后我们制订了定期与不定期添加瓷球的制度。

注：1994年12月，设备计划中修，我们全面检查  $\mathbf{I} \sim \mathbf{II}^{*}$  蓄冷器，发现  $\mathbf{I} \sim \mathbf{II}^{*}$  蓄冷器铜管也有不同程度变形损伤，经加紫铜套管，同时换大铜皮罩的面积，全部加固修复，开车一切正常。

（1995年2月）

# 制氧工问题解答（三则）

# 1. 问：小型空分设备高压空气进口阀作用原理是什么？如何操作？

答：高压空气进口阀分别控制进入各个隔层的空气量，以调节各隔层之间的热端温差，当氧气、氮气或馏分出口之间温度不一致时，可以改变这个隔层的进入空气量。如氧气温度低于氮气出口温度时，可以开大氧气隔层高压空气进口阀，当这个阀全开后，氧气出口温度还是低于氮气出口，可以关小氮隔层高压空气进口阀。反之则亦然，隔层间的温差愈小愈好，最大不超过  $2^{\circ}\mathrm{C}$  。

膨胀机高压空气进口阀，是高压空气进膨胀机的通过阀，这个阀平时处在开启状态，当设备发生故障，紧急停电，膨胀机开车、停车前应关闭。这个阀原则上不作调节用，但是在启动后期，膨胀机凸轮已处于最小进气位置，膨胀后空气温度低于  $-140^{\circ}\mathrm{C}$  且继续下降时，如果用开大高压空气节流阀的办法来减少膨胀量，势必使高压压力下降，高压空气节流阀前温度升高，节流后液化率减少，液体的生产量减少，会使整个启动时间延长。因此，短期的采用膨胀机高压空气进口阀，来控制膨胀机后的温度，有利于塔内液体的积累。关小膨胀机高压空气进口阀则热交换器高压压力升高，机前高压下降，中压下降，温度上升，液氧液面上升。开大则与上述情况相反。降压时，应先开大膨胀机高压空气进口阀，后开凸轮，但应注意这个阀不要开得过大，只要热交换器与机前两只压力表指示相同就可，以免紧急停电时能迅速切断气源，防止膨胀机飞车。

# 2. 问：小型空分设备各冷角式弯阻阀开与关的操作要领怎样？

答：冷角式弯阻阀主要由阀座、套管、阀体、填料、压帽、阀杆、阀头、手轮等组成。150 $\mathrm{m^3 / h}$  制氧机中的角式弯阻阀如下：

液空进乙炔吸附器通过阀（称通-1阀），只有在乙炔吸附器再生时是关闭的，加温、启动、停车、正常运转时都可以全开。这个阀的口径为DN25，不能从套管中拉出。

液氧排放阀（称通-2阀），停车加温、排放液氧、启动冷却时应打开，其余时间关闭。口径为DN15，可以从套管中取出来吹除。

馏分通过阀（称通-7阀或通-8阀）。阀门口径为DN25，也是不能从套管中取出的。这两个阀是不带氩工作时，为了同时获得双高产品而设计的。如以氧气为主产品的用户应用通-7阀来抽取馏分，氮产量下降一些而氮气纯度可达到  $99.95\% \mathrm{N}_2$  。如果以氮气产量为主产品的单位应打开通-8阀抽取馏分。馏分抽出的气量由馏分出口排出阀来控制。主塔带上氩塔时，通-7、通-8阀都应关阀。如果是双高设备而实际只需单高的氧气产品时可以不抽取馏分，以利提高氧气产量。

（杭氧集团公司质监部 叶必楠）

# 3. 问：各种不同充装介质的气瓶为什么漆成不同的颜色？常用工业气体气瓶颜色标记怎样？

答：1990年7月1日起生效的《气瓶安全监察规程》（简称89《瓶规》）中规定，“钢印标记、颜色标记不符合规定及无法判定气瓶内气体的”气瓶，“严禁充装”，“不得擅自更

改气瓶的钢印和颜色标记”。那么，各种不同介质的气瓶为什么要漆成不同的颜色呢？

气瓶是一种压力容器，用于盛装各种气体，气瓶只能“专瓶专用”，严禁互相混用。因为气瓶内充装的气体，其化学危险性质各不相同，如果两种或多种气体混装，就可能由于化学反应而发生爆炸事故。如氢气瓶绝对不能拉去充装氧气，我国已发生多起因氢一氧混装、或氢气瓶混入氧气的爆炸死亡事故。1993年扬州两起氢气瓶爆炸，造成两人死亡，就是氢气瓶混入氧气所致。再如氮气瓶误充入氧气，则不仅起不到化学反应设备或管道的清洗置换的安全作用，反而有引起设备、管道爆炸的危险。其三，气瓶漆色也是保护气瓶、防止外表生锈腐蚀的一项措施。

因此，气体厂在充装气体前（特别是充装氧气、氢气），必须对气瓶进行严格检查，如发现气瓶的漆色、字样，和所装气体不符；或漆色、字样脱落，不易识别气瓶种类时，不能给予充装，甚至应扣押气瓶。而且气瓶的漆色，不允许任意涂改，必须经常保持完好，如脱落应及时补漆，或重新刷漆。

为了防止气瓶混装、混用，各种气瓶都有明显的识别标志，而且瓶帽、防护罩等的

颜色应与瓶色一致。我国对气瓶漆色也作了统一规定，确定了严格的漆色标准——GB7144—86《气瓶颜色标记》国标。现将空分行业常用的气瓶颜色标记摘录如下：

# “氧吧”保健养身法悄然兴起

世界发达国家常压纯氧保健已经十分普遍，在宾馆、医院、康复中心和高级文体活动场所都设有“氧吧”，供人们吸氧需要。

近两年内，国内“氧吧”保健养身悄然兴起。北京亚运村首先建立了第一家集娱乐与健康为一体的“氧吧”——奥克赛颐乐宫；福州铁路医院、杭州上城区人民医院在医疗治病的同时，又增设一项特殊服务——吸氧治疗；杭州庆春路“快乐城”内开办了“卡依波氧吧”；南京中山路格兰云天娱乐美食城开设了南京的首家“氧吧”；北京海淀区车公庄西路“家圣餐厅”也增设了吸氧装置；广州、深圳、丹东等地也都相继开设了集娱乐健身于一体的“氧吧”。

这些“氧吧”均采用集中供氧形式，一般有可供

$20\sim 40$  人同时吸氧的面罩。吸氧的同时，可以听音乐、看电视，也可以唱卡拉OK、品尝饮料。有些“氧吧”还将营养素、清醒剂、香料等药物雾化入气体中，随氧气吸入人体。

据了解，南京部分医院妇产病房中，安排产前孕妇每天早晨、晚上各吸一次氧气，以缓解孕妇和胎儿的氧气需求。

# 起死回生废旧制氧机组的“妙手”——绍钢制氧厂厂长陈德华

1995年2月9日《冶金报》在“技高一筹”栏目里，刊登了《冶金报》记者陈小浔、包斯文的一则通讯：“起死回生的‘妙手’——记制冷技师陈德华”，报道了浙江绍兴钢铁总厂制氧厂厂长陈德华改造报废氧压机的事迹。陈德华厂长是一位在制氧实践中自学成才、敢想敢改的制氧技师。《深冷技术》也有过他的报道。现将《冶金报》的报道，摘要如下：

1984年，干了20多年制氧操作工的陈德华，被推荐当上了绍兴钢铁总厂制氧厂厂长。当时，厂里只能生产单一品种氧气，大量的氮气被白白放掉。要灌充高纯氮气，需要高纯氮压机，陈德华厂长日夜苦思，硬“啃”书本(原只初中文化程度)，经不断探索、试验，攻克了活塞结构、活塞环材质等一道道技术上的难题，终于将一台报废氧压机改装成高纯氮压机，而只花2000多元。1991年这项改造技术获得国家专利。

1992年，陈德华厂长研制的无润滑医用氧气压缩机技术，又获得国家新型实用专利。接着，他又开发钢瓶氙气等四个新品种，产品畅销绍兴、宁波、杭州、金华、兰溪、上海等地，每年创利50多万元，大大提高了经济效益和社会效益。

陈德华厂长将“旧0328氧压机改为无润滑氧压机”简讯，《深冷技术》1992年第2期33页已作报道；“氧压机改造成无润滑高纯氮压机”，也早在《深冷技术》1988年第4期27页上报道。陈德华厂长确是个革新“妙手”，他已在“150”塔上，改装成高纯氮，早几年还配上氙塔生产氩气（现停运）。为解决压高纯氮，

自己动手改造氧压机，成为高纯氮压机。近几年瓶装高纯氮打入上海，使氮气销售收入超过氧气销售收入。在医用氧热兴起时，陈德华厂长又动脑筋，将氧压机改造成为无润滑医用氧气压缩机，而不需液氧泵，灌充外售；并向外进行技术转让，协助兄弟单位副产医用氧，成功的已有好几家，目前仍继续对外进行技术转让服务。陈德华厂长还在空压机冷却器、长周期纯化器等方面进行改进。

绍钢制氧厂现有“150”三套(两套杭氧、一套江氧)，全厂94人，书记与厂长共4人。陈德华厂长团结一班人，干群同心，向技改要效益。氧压机均已改成无润滑，相告四年多来运行的各项指标都达到技术要求，并且做到了安全生产。1995年陈德华厂长又协助别的单位建立“液氩汽化站”，不用液氩泵，而用他改造成功的无润滑高纯氮压机。当液氩用槽车从外地拉来汽化后，经压缩机充瓶。可以一边液氩放入贮槽，一边就可充瓶，只要有5只钢瓶就可以充装。如用液氩泵充装，就必须等到钢瓶多了才可充装。而液氩汽化的冷量，用于压缩机冷却器的冷却，他说：“一点勿浪费”。利用原有旧设备改造，投资省，上马快。

# 违规气瓶受扣押 全为气瓶乱穿“衣”

1995年3月10日上午10时许，笔者到宜宾天原股份有限公司供气车间执行公务，突然发现氧气瓶库房中，存放有3只瓶体为绿颜色的氢气钢瓶，不由大吃一惊！经询问，方知这3只气瓶是因违反《气瓶安全监察规程》而被扣押下来的。据了解，这3只气瓶被扣押的原因与经过是这样的：

1995年3月初，四川省建筑十一公司材料科供应员王某某，拉5只气瓶到宜宾天原股份有限公司供气车间，要求充装氧气。当供气车间赵贤英同志（气瓶检验员）发现送来的5只气瓶中，两只瓶是涂充装氧气标志的天兰色，另外3只是涂充装氢气标志的绿颜色。氢氧混装是十分危险的，极易发生爆炸，其后果是不堪设想的。气瓶检验员赵贤英女士马上意识到：这是严重违反国家劳动部颁布实施的《气瓶安全监察规程》的有关规定的。因此，当即将瓶体为绿颜色的3只氢气瓶扣押下来，并立即将情况通报单位领导。

10点40分，我们正说话间，省建十一公司材料员王某某来取氧气瓶来了。笔者随即询问了她知不知道氢气瓶不能充装氧气。她回答说：“我们拉来的气瓶中没有氢气瓶，那5只气瓶，以前一直都是充装氧气的。”于是我问：“那3只绿颜色的氢气瓶，以前你们一直都在充装氧气吗？”她解释说：“不是的，以前那5只气瓶的颜色，全部都是漆的天兰色，我见有3只气瓶的油漆脱落厉害，有些部位已严重生锈，需要刷漆保护。同时，我为公司各单位避免

相互经常乱拉错气瓶，要求油漆工在气瓶上写字，以示区别。可当时又有人提议：既然要刷漆，那就刷点颜色好看的漆呗。因此，油漆工就将库房中余下不多的绿色油漆给1只溶解乙炔气瓶与3只氧气瓶全部刷上绿颜色的油漆。等油漆干后，我就把那5只氧气瓶拉来充装氧气，结果那3只气瓶被扣押了。至于氧气瓶该刷啥子颜色的漆，我是不憧的。”对此，笔者向该材料员讲解了一些气瓶使用的安全常识，使她明白了《气瓶安全监察规程》中规定的颜色标记不符合规定的，严禁充装。涂刷气瓶外表的颜色、字样和色环，必须符合国家标准GB7144《气瓶颜色标记》的规定，以避免气瓶充错气体而发生化学爆炸的道理。随后，她深有感慨地说：“现在我明白了”。

四川省气瓶检验站 胡志雄

# 关于“氧吧”……

成年人每天约需吸500升氧气，而人体内需要氧气量最多的部位是脑细胞。人脑的重量只占体重的  $2\% \sim 3\%$  ，但其耗氧却占全身的  $20\%$  ，脑血流量占心脏输出量的  $15\%$  。若供氧不足，脑功能就会马上下降，而出现缺氧症状。吸氧，不但能满足人体新陈代谢时氧的需求，增加有氧代谢；而且具有明显的解除疲劳、养神清脑、焕发精神、增加体力，以致养颜驻春、延年益寿的作用。特别是当今，人们为应付快节奏的生活与工作，而面对生态的破坏、环境的污染，就出现了供人吸氧的“氧吧”。

“氧吧”的出现，是氧气保健进入商品化、市场化的一个标志。日本已流行，我国也在悄然兴起。今年来，问询者较多。但建立“氧吧”，也不要一哄而起，要作市场调查，并要获得公安、消防、工商等部门的许可；建了，也要宣传跟上，并能适应消费者的承受能力，还要注意经营管理。与有条件的医院联合搞，可能较好办一些。杭州1994年曾兴起两家“氧吧”，但由于种种原因，现在已经暂停了。

“缺氧为万病之源”（日本一医学权威语）。缺氧，是“看不见的危险”。相信随着人们生活质量的提高，氧气保健意识的普及与认识，吸氧会逐渐被人们所接受，“氧吧”会走进人们的生活。（顾福民）
