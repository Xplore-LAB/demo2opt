( 2) 隔声罩内表面衬以多孔和纤维材料, 平均

( 3) 为了散热而设置的通风、换气口或需要开

以

( 4) 在需要经常检查仪表和油料处应设置玻璃
镜,

( 5) 在需要打开检修的地方,

管道噪声产生的机理
( 1) 管道所连接的鼓风机、离心压缩机及阀门
等所产生的噪声通过管道中的介质或管道本身而传
递出来。
( 2) 流体在管道中由于湍流而产生流动噪声。
在离心压缩机、鼓风机的管路系统里这是最主要的
噪声源。流动噪声随流动速度的增加而增加, 当流
动速度处于临界状态时流动系统由于形成湍流而产
生噪声。管道的急拐弯和扼流区也会引起气流流向
的突变而产生湍流发出噪声。这种噪声的频谱以中
高频为主, 声功率服从六次方规律。
( 3) 管道受到某种机械或气流激振, 通过管壁
传播而发生噪声。噪声以低中频为主,
声功率服从
速度的四次方规律。
控制、治理管道噪声的方法
( 1) 在进行管道设计时应避免截面和流向的急
剧改变,
三角及直角汇流以及低速流喷入高速流体
等情况。管路弯头的半径至少是管径的5 倍, 不同
管径之间的连接应有光滑过渡, 管路弯头也可采用
吸声弯头以减小噪声。
( 2) 流速高时容易产生噪声及振动, 一般管道
流速最好限制在40m/ s 以下,
当截面及流向急剧
变化时应将流速进一步降低。
( 3) 采用管道消声器。如压缩机管路上安装入
口消声器, 在出口管道上安装出口消声器,
在发生
较大压差产生噪声的各类阀门后的管路装消声器或
阻尼器、阻尼孔板,
使压力逐级降低。
( 4) 在噪声严重的管段外壁用隔声材料包裹以
及采用在管路外壁喷涂阻尼材料等方法, 降低或阻
断噪声沿管程周围传播。
( 5) 防止因机械振动使管道发生共振,
应把配
管系统的固有频率控制在激振源振动频率的1/ 3~
3 范围以外;
在振动较大的地方加支撑或防振管
座, 或改变阀件的安装位置。
阀门噪声控制与治理
空分设备中装设各种阀门,
当流体高速流经阀
门或调节阀, 由于阀的节流作用而使压力及速度发
生急剧变化, 产生湍流而形成气体动力性噪声, 以
及由于流体引起阀门及上下游侧的配管形成机械振
动而发生噪声。
阀门噪声产生的机理
( 1) 湍流噪声是由于流体通过阀门的节流闸并
在与阀门连接的管道内形成湍流和脉动而形成。湍
流噪声以中高频为主,
脉动噪声以低频为主。
( 2) 气体动力性噪声是由于流体边界的一般干
扰, 在阀门节流部位形成喷射流, 处于临界流速下
的流体在阀门节流部位及上下游的剪切层产生强烈
紊流,
使阀门发出噪声, 噪声随阀门前后的压差增
加而增加。一旦在阀门使用过程中产生噪声,
就向
其下流的管路中传播而很少衰减。
控制、治理阀门噪声控制的方法
( 1) 对噪声源))) 阀门进行控制。应大力提倡
设计和选用低噪声阀门,
这是最基本可靠的方法。
因为低噪声阀门破坏或降低流体通过阀门时产生的
湍流紊流,
从而控制阀门噪声的发生。低噪声阀门
中一般都设有节流装置: 多段减压、小孔喷注、分
散流通、迷宫环形或其组合形式的结构,
这些结构
有利于降低阀门噪声。
( 2) 采用吸声、隔声来降低已产生的阀门噪
声。一般在阀门下游附近设置管道消声器或多段减
压阻尼孔板等来消除或降低阀门产生的空气动力性
噪声。
( 3) 防止机械振动对阀门引起振动和共振。应
避免流体和阀门的固有频率所引起的激励频率耦
合, 注意阀门操作机构的刚性以及采用适当的阀门
形式或改变阀体内的形状、杆径等方法来解决。
( 4) 增加管壁的传递损失。一般采用适当的厚
管壁或采用封闭式的包扎手段。
