# 高压气体净化器脱水效果显著

我厂有两套杭州制氧机厂70年代初生产的150型制氧机组，压氧系统配套的5台氧压机，均为气缸内注蒸馏水润滑。由于氧压机以前有过缺水烧缸的历史，所以操作工人怕出事故而人为地将润滑水量加大，从而导致大量润滑水被带入氧气瓶内。在北方地区，瓶内积水除严重腐蚀气瓶外，冬季还造成瓶阀冻结，用户常因瓶阀冻结而误以为空瓶要求退货，我们虽采取多种措施，仍不能凑效。

1996年初，我们与山东济宁氧气厂取得联系后，他们派人到我厂了解情况，根据我厂的实际，专门设计制造了一套  $350\mathrm{m}^3 /\mathrm{h}$  的高压气体净化器，于同年10月运抵我厂，并派专人安装调试。经过1年

半来的使用结果表明，脱水效果非常好，原来充装时，我们将充氧台安装了两只气水分离器，瓶卡接口处还“噼叭”直冒水。现在这种现象完全不见了，气水分离瓶内根本见不到冷凝水。经实际检测，水分含量  $< 0.07\mathrm{g} / \mathrm{m}^3$  。

为充分利用净化器的脱水能力，从根本上杜绝瓶内存水，我们设计制做了气瓶倒水架，将气瓶逐瓶进行倒水，保证了整个冬季未出现瓶阀冻结现象。高压气体净化器的应用，不但使我厂的工业氧达到了国标要求，还为我厂开发医用氧提供了条件。

# 一、概述

蒸馏设备目前主要是板式塔和填料塔两大类。70年代以前，在大型塔器中板式塔占有绝对优势，曾出现过许多新型塔板。当时的填料塔往往存在明显的放大效应，即随塔径增大而填料塔效率下降的现象。实验室内在小直径填料塔中取得的实验数据，很难推广应用到工业规模填料塔的设计中。70年代初能源危机出现后，迫使填料塔技术在近20多年来取得了长足的进步。各种高效填料和新型塔内件不断涌现，并在工业中取得了成功的应用。

60年代后，为适应生产规模大型化的要求，各种规整填料应运而生。规整填料在整个塔截面上，几何形状规则、对称、均匀。它规定了气液流路，克服了壁流和沟流现象，压力降可以很小。在相同的通量和压降下，与散装填料相比，可以安排更大的比表面积，因此效率高。由于其结构的规整性，合理的设

计可以做到几乎无放大效应。目前在大型塔器中，以板式塔为主的局面已受到了规整填料的冲击。尤其在低压、常压及减压的蒸馏操作中，规整填料塔更具有明显的优势。

瑞士Sulzer公司用规整填料改造过的塔，其塔径达  $\phi 12.4\mathrm{m}^{[1]}$  。我国天津大学已用规整填料改造乙烯装置中塔径为  $\phi 6.8\mathrm{m}$  的汽油分馏塔[2]，并取得了2385万元的年综合经济效益[3]。1998年茂名石油化工公司在塔径  $\phi 8.4\mathrm{m}$  的减压塔中采用了天津大学开发的规整填料及塔内件技术，预计也将产生明显效益。过去认为填料塔只能用于小直径塔的概念，现已彻底扭转了。

低温法空分技术经历了90多年的历程，现已发展成为一种较成熟的技术。虽然它现在开始面临诸如变压吸附法、薄膜渗透法等非低温法的挑战，但其在空分工业中的主导地位仍是不容置疑的。随着能源危机的日益加剧，强烈促进了许多领域能源的有效利用，

能耗很大的空分工业，其能耗约占空气制品价格的  $75\%$ 。在空分流程中，大部分能量用来完成分离过程，仅有一小部分用于提供带压气体产品或液体产品，其能耗分布如表1所示。

# 二、低温法空气分离流程简介

目前低温法分离空气的主要流程有两种：一是能同时分离氧、氮的双塔流程（流程图略）；另一种是能同时生产氧、氮和氩的三塔流程，如图1所示。

压力。上塔操作压力约  $140\mathrm{kPa}$ ，下塔约600 $\mathrm{kPa}$ 。这样的布置可实现双效精馏，即上塔的塔底再沸器也是下塔的塔顶冷凝器，既省设备投资，又可节能。原料空气压缩到600 $\mathrm{kPa}$ ，除去水、 $\mathrm{CO_2}$  杂质后，在主换热器中被逆流的氧氮产品气冷却后送入下塔。在下塔原料空气被分离为顶部的纯氮和底部的富氧液空，纯液氮为上塔和下塔提供塔顶回流液。富氧液空送入上塔适当部位进一步分离。在上塔顶部得到高纯氮产品，底部得到高纯氧产品。部分带压的原料空气利用膨胀功在增压机中增压，冷却后再经膨胀机膨胀，为装置提供冷量，然后也送入上塔。

在冷凝蒸发器中，将下塔顶部的氮气冷凝，同时将上塔底部的液氧蒸发。上下塔的压力差为冷凝蒸发两股流体提供了温度差。氩的浓度较高部位在上塔下段，由此抽出一股蒸气流到粗氩塔进行精馏，粗氩塔顶蒸气被来自下塔底的富氧液空冷凝，粗氩塔底部回流液仍返回上塔继续参与精馏。

# 三、规整填料对空分主精馏塔能耗的影响

空气产品的成本主要取决于能耗，能耗主要是由空气压缩机的出口压力决定的，而空压机的出口压力，取决于精馏塔中的冷凝蒸发器冷凝氮气及蒸发液氧所需的温度差而必须的压力。蒸发器液相侧的压力主要依赖于上塔的压力降，由于冷凝蒸发器的换热面积是限定的，在一定的换热温差下，当上塔底部压力降低时，必然导致下塔顶部压力也降低。计算表明[7]，当上塔底部的压力降低 $\Delta \mathrm{P}$ 时，可导致下塔顶部的压力降低 $3\Delta \mathrm{P}$ 。因此，上塔压力的降低对整个系统的降压关系重大，使上塔压力降减小也就成为空分节能的关键。

传统的筛板精馏塔，为维持正常操作，保持气体在液层内鼓泡，每块塔板必须有足够的压差，才能有稳定的两相流。否则，会漏

液造成轴向返混，使效率严重下降。为满足稳定操作，每块理论塔板所必须的压差至少为  $35\mathrm{mm}$  液柱。而规整填料的传质是在填料表面上的液膜和与其相接触的气相之间进行，而不象筛板那样受稳定性的限制。因此，采用规整填料每当量理论板的压差仅为  $5\sim$ $15\mathrm{mm}$  液柱。

对空分设备而言，为获得高纯氧产品，筛板的压差为每理论塔板  $0.3\mathrm{kPa}$  ，规整填料的压差为每当量理论板  $0.05\mathrm{kPa}$  。下塔一般操作在  $600\mathrm{kPa}$  压力，常规筛板总压降约15 $\sim 20\mathrm{kPa}$  ，即使不存在这部分压降，节能效果才  $2.5\%$  。上塔操作压力一般为  $140\mathrm{kPa}$  ，且上塔理论板数约为下塔的两倍。据APCI公司报道[8]，上塔采用规整填料可节能  $8\%$  。空分设备的原料是取之不尽的空气，产品成本多半取决于能耗，所以节能  $8\%$  是很可观的。

上塔一般分为5段或更多的填料层，每段之间必须有液体收集再分布器。因此，采用规整填料塔的初始投资会比筛板塔高。传统的空分筛板塔，板间距为  $100\sim 200\mathrm{mm}$  。对于  $\mathrm{N}_2 - \mathrm{O}_2$  系统来说，规整填料的等板高度（HETP）为  $250\sim 300\mathrm{mm}$  。上塔通常需要 $50\sim 90$  块理论塔板，采用规整填料塔，高度会增加，冷箱体积要增加，规整填料相对筛板而言要昂贵。虽然规整填料塔的经常性操作费用可降低，但设备总投资要高。

为降低投资，上塔也可采用复合塔的结构，即筛板和规整填料共存于上塔。在图2中，上塔的上半段实际进行着  $\mathrm{N}_2 / \mathrm{O}_2$  的分离，而上塔的下部约总板数的一半是起着从向下流的液相中回收氩的作用。同样填料在该两段中的效率不一致：在  $\mathrm{N}_2 / \mathrm{O}_2$  分离区填料的HETP值约为  $250\sim 300\mathrm{mm}$  ；上塔下段的效率要高，HETP值为  $175\mathrm{mm}$  左右。设计时可在空分上塔下半部采用规整填料，上半部仍采用筛板，这样布置既节能又省投资，同时上塔塔高仅略有增加。设计时也可在不同

部位采用不同品种和型号的高效填料。

# 四、规整填料对氩产品的影响

氩的沸点介于氧和氮之间，氧的沸点为 $90.18\mathrm{K}$  ，氩为  $87.29\mathrm{K}$  ，氮为  $77.36\mathrm{K}$  。氩的最大浓度点  $(12\% \sim 15\%)$  在上塔下段。通常在气相中氮浓度较低（  $25\sim 100~\mathrm{ppm})$  ，氩浓度  $9\% \sim 12\%$  处引出气相进入粗氩塔底部。粗氩塔顶的冷凝介质来自主精馏塔下塔底的富氧液空，蒸发后的富氧液空仍回上塔进一步精馏分离。粗氩塔顶冷凝液流作为塔顶回流及出料。该塔回流比较高，  $\mathbf{R} = 24\sim 49$  （或  $\mathrm{L / V = 0.96\sim 0.98)}$  。传统的粗氩塔塔顶出料中仍含  $2\% \sim 5\%$ $\mathrm{O_2}$  和  $0.05\% \sim 1\%$ $\mathbf{N}_2$  。该处氩浓度较高，  $\mathrm{Ar / O_2}$  分离的相对挥发度  $\alpha = 1,1$  。塔底液相仍返回上塔。

主精馏塔上塔下部氩浓度较低， $\mathrm{Ar} / \mathrm{O}_2$  分离的相对挥发度  $\alpha = 1.5$  ，故该段需大量分离级来降低氧产品中的氩含量。对  $99.5\%$  纯度的氧产品，提高氩提取率是很重要的。若是筛板塔，企图增加板数来提高氩提取率，有可能事与愿违。因为增加塔板数会同时增加塔压降，使上塔底压力增高，这一方面增加能耗，另一方面会降低分离系数  $\alpha$  值，从而使  $\mathrm{Ar} / \mathrm{O}_2$  分离更困难。因此，塔板增加到一定限度会得不偿失。

上塔采用规整填料可使上塔压力减小约 $20\mathrm{kPa}$  ，从而可使  $\mathrm{Ar / O_2}$  分离的  $\alpha$  值增加。即使上塔在常规的当量理论板数下，氩的提取率可增加  $10\%$  。在不增加压降的前提下，规整填料可以有更多的理论板数，可进一步提高氩的提取率。按原料空气中的氩计算，采用规整填料氩的提取率有可能高达  $95\%$  ，这在筛板塔是不可想象的。

常规筛板型粗氩塔，塔顶氩气中仍含氧 $2\% \sim 5\%$  。为获得高纯氩，还需要将粗氩催化加氢除氧，生成的水再用分子筛吸附干燥。粗氩中尚含少量氮及过剩的氢，再送入精氩塔精馏，最终得到高纯氩。加氢制氩工艺要

求有稳定的氢气源，这不仅使精氩的成本提高，而且存在安全问题。制氧站本身是高等级防火、防爆单位，存在氢气源且与氧混合反应，更要求有严格的安全措施。

上面已提到粗氩塔顶粗氩蒸气的冷凝是靠冷凝器另一侧富氧液空蒸发，而蒸发侧的压力是上塔操作压力。这就决定了粗氩塔顶的温度和压力，粗氩塔底的气相进料来自上塔，故该处压力也是确定的，从而粗氩塔的总压降也就确定了。在此压差下，对筛板可安排  $40\sim 70$  块理论塔板。因此，粗氩塔顶部氧含量无法小于  $0.2\%$  ，一般在  $2\% \sim 5\%$  而对规整填料，则可安排180块当量理论塔板。这样可以直接从粗氩塔顶获得含氧量小于  $2\mathrm{ppm}$  的粗氩，从而实现无氢制氩，可省去使用筛板塔时所必须的下游加氢除氧装置及建氢站。

主题词：板翅式冷凝蒸发器 传热系数 传热温差 循环流量 计算方法

# 一、前言

空分设备中的冷凝蒸发器的作用是使下塔顶部的氮气和上塔底部的液氧进行热交换，以实现氮气冷凝和液氧蒸发之目的。早期的冷凝蒸发器一般采用管式换热器。70年代初，随着板翅式换热器技术的发展，板翅式冷凝蒸发器在我国的空分设备中得到广泛应用。

我国早期的板翅式冷凝蒸发器设计，一直套用原苏联列管式冷凝蒸发器的计算方法，事实上仅仅只是一种估算。在80年代初我公司引进了林德公司的空分技术后，才采用林德公司提供的在特定条件下的经验方法进行设计、计算，这一经验方法虽然解决了特定条件下的实际应用问题，但有其局限性。在现代空分流程的不断改进、冷凝蒸发器使用条件改变的情况下，仍采用某一特定条件下的经验方法来进行板翅式冷凝蒸发器设计是不太合理的。对于空分设备变工况下的冷凝蒸发器的换热效率分析也缺乏必要依据。

因此，寻找一种良好的、适用于不同使用条件下的板翅式冷凝蒸发器设计计算方法，就显得十分必要。板翅式冷凝蒸发器的

传热计算是一个比较复杂的过程，现代电子计算机的发展和应用为解决复杂的计算过程创造了有利条件。下面就板翅式冷凝蒸发器的设计计算方法进行讨论，并提出计算机数学模型。

# 三、传热过程分析

在换热器的一侧为氮气全冷凝过程，氮气进换热器的压力为  $\mathrm{P}_{\mathrm{N2}}$  ，其对应的饱和温度为  $\mathrm{T}_{\mathrm{N2}}$  。经冷凝成为液氮，依靠液氮的自重形成向下的流动。因此，可以近似认为氮侧的压力不变，即有  $\mathrm{P_{DN2} = P_{BN2} = P_{N2}}$  。如果把气氮作为纯质处理，则气氮温度与液氮温度相等，即  $\mathrm{T_{DN2} = T_{BN2} = T_{N2}}$  。

在换热器另一侧，若设上塔下部的气氧压力为  $\mathrm{P_A}$  ，则其对应的饱和温度为  $\mathrm{T}_{\mathrm{A}}$  。液氧在板翅式换热器外面的筒体内，从上到下的流动过程中，虽然因液柱的影响使其压力增加到  $\mathbf{P_B}$  ，由于在这一过程中没有热量传入，液氧在B点的温度  $\mathrm{T_B}$  与A点的温度  $\mathrm{T_A}$  相等，因此进换热器的液氧为过冷状态。从换热器底部进入的过冷状态的液氧，需有一个无相变的换热过程，使其温度升高到一定值，才开始转变为蒸发换热。

设C点为液氧从无相变换热到沸腾蒸发换热的交界点（见图2)，则液氧经过B—C段后，由于液氧侧液柱静压和流动阻力的存在，使C点压力低于B点的压力，设为  $\mathrm{P_C}$  其相应的氧饱和温度为  $\mathrm{T_C}$  ，在C点液氧开始沸腾蒸发，经C一D段后变成汽液两相流，压力降低到  $\mathbf{P}_{\mathrm{D}}$  ，温度降低到与D点压力相对应的饱和温度  $\mathrm{T_D}$  。

# 四、传热计算数学模型

主冷凝蒸发器设计时，在保证其板翅式换热器外面的筒体有足够的空间盛放液氧的前提下，主要是对其换热元件进行换热性能设计。板翅式换热器的结构参数计算和物性计算在许多文献都有详细介绍，这里不再详述。下面就传热系数、传热温差和液氧循环流量的计算方法进行讨论。

# 1. 传热系数

传热系数的计算方法应根据不同的传热方式选择，这里所选择的计算方法仅作参考。

（1）从Nusselt方程[1,2,3]推导得到的氮气冷凝传热系数

$$
\begin{array}{l} \alpha_ {\mathrm {N} 2} = 1. 9 8 5 4 \cdot \lambda_ {1} \cdot [ \mathrm {A} \cdot \rho_ {1} ^ {2} / (\mathrm {W} _ {\mathrm {N} 2} \cdot \\ \left. \mu_ {1}\right) ] ^ {1 / 3} \quad \mathrm {W} / \left(\mathrm {m} ^ {2} \cdot \mathrm {K}\right) \tag {1} \\ \end{array}
$$

式中  $\lambda_{1}$  ——液氮导热系数， $\mathrm{W} / (\mathrm{m} \cdot \mathrm{K})$

$$
\begin{array}{l} \rho_ {1} \text {—— 液 氮 密 度 ，} \mathrm {k g / m ^ {3}} \\ \mu_ {1} — — \text {液 氮 动 力 粘 度}, \mathrm {P a} \cdot \mathrm {s} \\ \end{array}
$$

A——换热器氮侧单位长度上的传热面积，  $\mathrm{m}^2 /\mathrm{m}$

$$
\mathrm {W} _ {\mathrm {N 2}} — — \text {液 氮 冷 凝 量}, \mathrm {k g / s}
$$

（2）氧侧B—C段无相变传热系数

$$
\begin{array}{l} \alpha_ {\mathrm {O} 2 \mathrm {B C}} = (\mathrm {C} _ {\mathrm {P} _ {1}} \cdot \mathrm {J} _ {\mathrm {B C}} \cdot \mathrm {G}) / \Pr^ {2 / 3} \\ \mathrm {W} / \left(\mathrm {m} ^ {2} \cdot \mathrm {K}\right) \tag {2} \\ \end{array}
$$

式中  $C_{\mathrm{p_1}}$  ——液氧比热，  $\mathrm{J / (kg*K)}$

jBC 传热因子，由雷诺数  $\mathrm{Re}_1 =$  （de·G）/μi，从Re-j因子曲线上查得。

de——氧侧翅片当量直径，m

$\mu_{1}$  ——液氧动力粘度，  $\mathrm{Pa} \cdot \mathrm{s}$

G——单位流通截面积上的重力流速

$$
\mathrm {G} = \mathrm {W} _ {\mathrm {O} 2} / \mathrm {A} _ {\mathrm {C}} \quad \mathrm {k g} / (\mathrm {m} ^ {2} \cdot \mathrm {s}) \tag {3}
$$

$\mathrm{W_{O_2}}$  ——液氧流量，  $\mathrm{kg / s}$

$\mathrm{A_C}$  ——氧侧通道流通截面积，  $\mathfrak{m}^2$

$\mathbf{Pr}$  ——普郎特尽数，按下式计算

$$
\Pr - C _ {P l} \cdot \mu_ {l} / \lambda_ {l} \tag {4}
$$

$\lambda_{1}$  ——液氧导热系数，  $\mathrm{W / (m*K)}$

# （3）氧侧C—D段沸腾传热系数

液氧在板翅式换热器中的蒸发过程，虽然靠其上下部密度差进行自然循环，但事实上，液氧是在外部液氧液柱的静压作用下，在板翅式换热器的小空间内沸腾蒸发，其过程更接近强制对流沸腾换热的特点。为此，选用参考文献[2]的计算方法。

$$
\begin{array}{l} \alpha_ {\mathrm {O} 2 \mathrm {C D}} = 3. 4 a ^ {*} \cdot (1 / \mathrm {X} _ {\mathrm {T T}}) ^ {0. 4 5} \quad \mathrm {W} / (\mathrm {m} ^ {2} \cdot \mathrm {K}) (5) \\ \frac {1}{X _ {T T}} = \left(\frac {W _ {\mathrm {v m}}}{W _ {l m}}\right) ^ {0. 9} \cdot \left(\frac {\rho_ {\mathrm {V O} 2}}{\rho_ {1}}\right) ^ {0. 5} \cdot \left(\frac {\mu_ {\mathrm {V O} 2}}{\mu_ {1}}\right) ^ {0. 1} (6) \\ \end{array}
$$

$(\mathrm{W_{vm}} / \mathrm{W_{lm}})^{0.9}$  最大值取1.44

式中  $\alpha^{*}$  ——液氧在无相变时的给热系数  $\mathrm{W} / (\mathrm{m}^2 \cdot \mathrm{K})$  ，这里即为式（2）的  $\alpha_{\mathrm{O2BC}}$  。

$\rho_{\mathrm{VO_2}}$  ——饱和氧气相密度，  $\mathrm{kg / m^3}$

$\mu_{\mathrm{VO_2}}$  ——饱和氧气相动力粘度，  $\mathrm{Pa} \cdot \mathrm{s}$ $\mathbf{W}_{\mathrm{vm}}$  ——液氧蒸发段的蒸体平均量，  $\mathrm{kg / s}$

$\mathrm{W_{lm}}$  ——液氧蒸发段的液体平均量，  $\mathrm{kg / s}$

$\mathbf{W}_{\mathrm{vm}}$  、  $\mathbf{W}_{\mathrm{lm}}$  采用Gloyer法来确定，其计算方法如下：

$$
\begin{array}{r l} \text {蒸 汽 的 平 均 流 量} & \mathrm {W _ {v m} = F \bullet W _ {\triangle v}} \\ \mathrm {k g / s} & \end{array} \tag {7}
$$

$\mathbf{W}_{\triangle \mathrm{V}}$  ——液氧蒸发量，  $\mathrm{kg / s}$

F——计算蒸汽平均流量的修正系数，由温度比  $\xi = (T_{h1} - T_{C2}) / (T_{h2} - T_{C1})$  ，查 $\xi -F$  图[2]得到。这里  $T_{h1} = T_{h2} = T_{n2},T_{C1}$  为液氧侧C点的温度，  $T_{C2}$  为氧侧D点的温度。

液体平均流量  $\mathrm{W_{lm} = W_{O2} - W_{vm}\quad kg / s}$  (8)

2. U值计算[8]

$$
\frac {1}{U} = \frac {1}{\alpha_ {\mathrm {N} 2} \cdot \eta_ {\mathrm {O N} 2} \cdot A _ {\mathrm {N} 2}} + \frac {1}{\alpha_ {\mathrm {O} 2} \cdot \eta_ {\mathrm {O O} 2} \cdot A _ {\mathrm {O} 2}} \\ W (\mathrm {m} \cdot \mathrm {K}) \tag {9}
$$

式中  $\eta_{\mathrm{ON2}}$  、  $\eta_{\mathrm{OO2}}$  ——分别为氮侧和氧侧的翅片表面效率，参见文献[3]

$\mathrm{A_{N2}}$  、  $\mathrm{A}_{\mathrm{O_2}}$  ——分别为氮侧和氧侧的在单位长度上的换热面积 $(\mathrm{m}^2 /\mathrm{m})$

$\alpha_{\mathrm{N2}}$  、  $\alpha_{\mathrm{O2}}$  分别为氮侧和氧侧的给热系数，对于  $\alpha_{\mathrm{O2}}$  在B—C段为  $\alpha_{\mathrm{O2BC}}$  ，在C—B段为 $\alpha_{\mathrm{O2CD}}$

# 3. 传热平均温差

从图2可知，换热器分为液氧无相变换热段（B—C）和液氧沸腾蒸发换热段（C—D)。因此，传热平均温差也应分为两段计算，即有  $\Delta \mathrm{t}_{\mathrm{mBC}}$  和  $\Delta \mathrm{t}_{\mathrm{mCD}}$  。考虑到温度变化范围较小，这里采用对数平均温差

$$
\Delta t _ {m} = \frac {\left(T _ {h 1} - T _ {c 2}\right) - \left(T _ {h 2} - T _ {c 1}\right)}{\ln \left(T _ {h 1} - T _ {c 2}\right) / \left(T _ {h 2} - T _ {c 1}\right)} \tag {10}
$$

式中  $\mathrm{T_{bl}}$  ——热流体进口温度，K

$\mathrm{T}_{\mathrm{c1}}$  ——冷流体进口温度，K

$\mathrm{T_{h2}}$  ——热流体出口温度，K

$\mathrm{T}_{\mathrm{c1}}$  ——冷流体出口温度，K

# 4. 液氧循环流量

在图1中，A点和B点的机械能平衡为

$$
\mathrm {P} _ {\mathrm {B}} - \mathrm {P} _ {\mathrm {A}} = \rho_ {1} \cdot \mathrm {g} \cdot \Delta \mathrm {Z} _ {\mathrm {A B}} \tag {11}
$$

式中  $\mathbf{g}$  ——重力加速度

$\Delta Z_{\mathrm{AB}}$  ——A点与B点的高度差，  $\mathbf{m}$

$$
\begin{array}{l} \mathrm {P _ {B} - P _ {A} = \Delta P Z _ {B C} + \Delta P F _ {B C} + \Delta P Z _ {C D} +} \\ \Delta \mathrm {P F} _ {\mathrm {C D}} + \rho_ {1} \cdot \mathrm {g} \cdot \Delta \mathrm {Z} _ {\mathrm {D A}} \quad \mathrm {P a} \tag {12} \\ \end{array}

$$

B-C段和C-D段的理论所需长度按下式分别计算，两个理论长度几何相加即为换热器理论所需长度。

$$
\mathrm {L} = \mathrm {Q} / (\mathrm {U} \cdot \Delta t _ {\mathrm {m}}) \quad \mathrm {m} \tag {27}
$$

# 7. 计算步骤

(1)根据氮侧已知条件设定换热器结构；  
（2）按式（1）计算氮侧给热系数；  
(3) 假定液氧循环量  $\mathrm{W}_{\mathrm{O_2}}$  ，计算氧侧G；  
（4）由按式（11）确定  $\mathbf{P_B}$

（5）假定氧侧C点压力为  $\mathrm{P_C}$  （低于  $\mathbf{P_B}$  ），由  $\mathrm{P_C}$  查氧的饱和曲线得  $\mathrm{T_C}$  
（6）计算显热段的理论所需长度  $\mathrm{L_{BC}}$  ，并根据  $\mathrm{L_{BC}}$  ，由式(15)、(16)计算C点压力  $\mathbf{P}_{\mathrm{C}}$  如果计算得到的  $\mathrm{P_C}$  与步骤5假定的  $\mathbf{P}_{\mathrm{C}}$  不一致，则重新假定的  $\mathrm{P_C}$  直到两值一致为止；  
（7）根据假定的液氧循环量  $\mathbf{W}_{\mathrm{O2}}$  ，由式(26)计算液氧蒸发量  $\mathbf{W}_{\Delta V}$  ，按Gloyer法计算液氧沸腾流体的平均流量  $\mathbf{W}_{\mathrm{vm}}$  和  $\mathbf{W}_{\mathrm{lm}}$  
（8）计算液氧循环流量，如果计算得到的液氧循环流量和假定的不一致，则重新假定  $\mathbf{W}_{02}$  ，并重复以上从步骤3开始的过程，直到计算值与假定值一致为止；  
（9）计算换热器蒸发段理论所需长度；  
（10）所需总长度校核，若计算得到的换热器所需总长度小于预定的长度，且在一定的后备系数范围内，则认为预定的换热器设置是合理的。否则，重新选定结构。

师建忠

（杭州制氧机集团有限公司透平压缩机厂，杭州市东新路90号，310004）

摘要 本文简要地分析单圆弧及双圆弧型线叶轮的性能缺陷，并论述了采用二元任意曲面型线的优点，同时通过杭氧引进的美国NREC三元叶轮计算软件，对二元曲面型线叶轮进行设计介绍及流场分析。图4。

主题词：离心式压缩机 单圆弧型线 二元任意曲面型线 三元叶轮 速度分布 载荷分布

# 一、问题的提出

我们都知道离心式压缩机叶轮的设计应包括两个方面的要求：一是要满足气量及压力的要求；二是要有较高的效率。这两个方面是相互联系、相互影响的。因此在叶轮的设计上不仅要选定叶轮主要结构参数，而且要合理设计叶片型线及子午面型线。

常规的离心式压缩机叶轮，其叶片的型线通常是采用单圆弧叶片，其常规载荷分布见图1，属于卸载型分布，其缺点主要表现为以下几个方面：

![](images/3ae795471301b758cff2d1ff1b9fe560927781d73f12b13020d91b822cf4235e.jpg)  
图1 叶片常规载荷分布

# 1. 进口流动紊乱

进口段载荷过大，极易引起分离，再加上常规的一元设计未能对其冲角进行较确切的计算，因而造成其进口流场极为紊乱，这也是造成这种叶型叶轮效率较低、叶轮载荷

不易提高的主要原因之一。

# 2. 出口段载荷过小

出口段载荷过小，分离半径较低。出口段叶片未能充分发挥其承载能力，小流量、高压比的压缩机尤其明显，其出口段的相对流道的扩压度几乎不存在，甚至出现了气流膨胀压力降低现象。这时，出口段的  $\beta_{2}$  角呈减小趋势。例如，我厂某一产品的后二级叶轮就出现过这种现象。这从叶轮外径切割 $10\mathrm{mm}$  后，能量头反而有所增大可得以证明。我们从速度三角形也能看出（见图2)，由于出口段不存在扩压度，使得  $\mathbf{W}_2^{\prime} > \mathbf{W}_2$  ，造成叶轮静压在出口段有所下降（这种现象在高压比小流量压缩机中普遍存在，某一氧透设计中，曾对其10个叶轮相对流道的扩压度分布进行计算，其中7个叶轮在其出口段不存在扩压度)，虽然  $\mathrm{U}_2^{\prime} < \mathrm{U}_2$  ，但综合  $\mathbf{W}_2^{\prime}$  的变化造成  $\mathrm{C}_{2\mathrm{u}}^{\prime} > \mathrm{C}_{2\mathrm{u}}$  ，当  $\mathrm{C}_{2\mathrm{u}}^{\prime}$  增大超过  $\mathrm{U}_2^{\prime}$  减小速度时，即  $\mathrm{U}_2^{\prime} \cdot \mathrm{C}_{2\mathrm{u}}^{\prime} > \mathrm{U}_2 \cdot \mathrm{C}_{2\mathrm{u}}$  时，叶轮的总压就降低了。出现了做负功的现象，这是极其不利的。

# 3. 叶片载荷无法控制

![](images/efb212e3dbc0d245ab2ad6e47180a05d6a19c46995fe5cdc1e174c23088958a0.jpg)  
图2 某叶轮叶片速度三角形分布

叶片的载荷分布无法控制，不能充分发挥叶片的承载能力。双圆弧型线是针对单圆弧型线的改良，它的基本思路就是在叶轮进口段用一较小曲率的圆弧构成，而叶片出口段用一较大曲率的圆弧构成，且相切于相对流道的进口最窄截面处，其常规载荷见图1，它虽然改进了进口处的载荷分布，但未根本改变卸载型的载荷分布。

近年来，随着计算水平的提高，计算机的普及，使得三元叶轮广泛地应用于设计制造，三元叶轮具有效率高、性能曲线平坦的优点。但由于三元叶片是空间扭曲，使加工制造成本较常规叶轮昂贵。而且，需要高技术、高精度的机床，就最简单的可展直纹面而言，一般需要五坐标铣床才能加工制造。同时，由于在  $\mathrm{b}_2 / \mathrm{D}_2$  较小的情况下，三元叶轮效果不大明显。因此，为了降低制造成本，有必要在小流量的叶轮上采用一种能有较好的气动性能，同时也能在普通三坐标铣床上加工的二元叶轮，这就提出了二元任意曲面叶型的设计问题。

# 二、二元任意曲面型线及设计方法

二元任意曲面，顾名思义，是指在轴线方向（Z轴）不扭曲，而只在回转面可任意变化的曲面。该曲面的特点是型线没有约束条件，能最大程度迎合流场的分布规律。

一个好的二元任意曲面叶轮应具有以下优点：

1. 载荷分布均匀，能较有效地利用叶轮的承载荷能力；  
2. 流场分布较好，进口冲角小，分离半径高。总之，叶轮具有较高的效率。

就目前而言，计算二元任意曲面叶轮可

采用以下几种方法（这里是指回转面的设计）：第一，用控制相对流道的扩压度的办法来设计叶片型线，这种设计法能有效控制叶轮静压场分布，但未能控制叶轮的载荷分布；第二，用叶片表面速度的简化计算法（stanits快速计算法）计算叶片表面速度，从而控制叶片的载荷分布；第三，采用现有三元叶轮计算软件计算，在流场分布较好的情况下，使叶片沿轴线方向不扭曲。

前两种方法都是建立在一元主流线计算的基础上的，从某种意义上讲，它们本身就先天不足；而第三种方法可以说是最有效的方法，虽说计算中约束了一个条件，在计算上比三元更困难，但是，用设计的时间及难度，换来了生产与制造成本的降低，还是值得的。

# 一、工艺流程简介

这套空分设备工艺流程上采用先进的分子筛预净化，带增压透平膨胀机、无氢制氙流程，仪控部分采用美国Honeywell公司的MicroTDC3000集散控制系统，电控部分采用天津幕尼康公司的可编程控制器，主要分析仪表采用进口件，而且所有分析仪表在线分析进DCS系统。整个配套水平处于国内领先地位。

# 1. 工艺流程简介

原料空气由吸风塔吸入，经袋式过滤器处理，进H700—6.8/0.97—1型离心空压机压缩，四级出口热空气进再生预换热器预冷后，进穿流筛板空冷塔。经常温水和低温水两级喷淋冷却后，出空冷塔进分子筛纯化器预净化，再经过滤器除去粉尘，大部分经氧气、氮气、污氮气三个主换热器冷却进下塔。另有一部分空气经增压机压缩并冷却后，进三个主换热器冷却，中抽至膨胀机膨胀，膨胀后气体经液氧热虹吸蒸发器冷却后进上塔参与精馏。

进下塔空气经下塔预分离后，下塔底部液空经过冷器过冷后分三路：一路进上塔；一路作粗氩Ⅱ塔顶部冷源，经部分蒸发回上塔；一路作工艺氩液化器冷源，蒸发后回上塔。下塔顶部抽出的压力氮气，一路经氧主换热器复热后作局部加温气，另一路作热源分两支分别进粗氩Ⅱ塔底部蒸发器和精氩塔底部蒸发器冷凝后，分别节流汇合进上塔顶部。

上塔精馏的氧气直接经氧气主换热器复热送出，由ZW-53/30型氧压机压缩送炼钢。气氮经过冷器、氮气主换热器复热后，一路作产品气输出，一路去水冷塔。污氮出上塔经过冷器、污氮主换热器复热后，一路作纯化再生气，一路去水冷塔。

由上塔中部抽出的气态氩馏分，经填料型粗氩I塔、粗氩Ⅱ塔进行氩氧分离成工艺氩，经工艺氩液化器液化后，由填料型精氩塔中部进入，进行氩氮分离，含氮废气直接放空进大气，液态产品经精氩量筒送贮槽装槽车或经活塞式液氩泵加压汽化充瓶，或用管道送炼钢。粗氩Ⅱ塔底部液体经离心氩泵打回粗氩I塔作回流液后进上塔。

# 中国药典氧检定方法原理及操作事项讨论

韩柏青

（常州市药品检验所，江苏常州市麻巷64号，213003）

吴权

（常州铸造总厂制氧部，江苏常州市西门芦墅路7号，213002）

摘要 阐述了我国1995年版《药典》收载品种——氧的检定方法原理，并讨论了必要的操作注意点。

主题词：医用氧 药典 质量检验 操作事项

在卫生事业中，医用氧为抢救病人生命和减轻病人痛苦起重要作用。为确保它的质量，早在1953年我国便将氧正式列入国家药典。但是，由于受到当时生产水平的限制，医院实际使用的氧气为工业用氧气。随着社会主义经济建设的发展，我国于1988年颁发了国家标准GB8982—88《医用氧气》，并同时颁发了国家标准GB8986—88《医用及航空呼吸用氧气的检验方法》，以求统一对医用氧气的检测。这两个标准的颁发和实施，对我国医用氧气的生产起了极大的推动作用，使医用氧气的生产从无到有，并且每年都有很大的发展，从而改变了医院使用工业氧气为病人输氧的不合理状况。同时，1995年版的《药典》对“氧”的部分也作了相应的修改，取消了卤素的检测要求，增加了二氧化碳和其他气态氧化物的检测要求，其检测方法也取自国家标准GB8986—88的第4章和第7章。

为促进医用氧检验工作的开展和交流经验，我们结合工作实践将医用氧检验方法的原理及操作注意点介绍如下，以供同行们参考。

# 一、氧的鉴别

氧助燃，但本身不能燃烧。只有在一定温度下遇还原性物质如灼红的木条才能发生剧烈的发光发热的化学变化，即燃烧现象。

氧比空气略重，操作时可调节钢瓶减压阀使之成适当气流，用橡皮管导入一洁净干燥的  $250\mathrm{ml}$  三角瓶中底部约1分钟，待瓶中空气被置换，盖上玻璃片，并将灼红的小木条小心插入，即应突发燃烧。

# 二、氧的杂质检查

# 1. 酸碱度

氧气中可能含有气态酸或碱，允许的酸或碱含量，以盐酸计为：

$$
0. 0 1 \mathrm {M} \times 0. 2 / 1 0 0 0 = 2 \times 1 0 ^ {- 6} \mathrm {M}
$$

分析时，乙管的吸收液体积为  $100\mathrm{ml}$  所以吸收液的盐酸浓度为  $2\times 10^{-5}\mathrm{M}$  ，设盐酸全部离解，则  $[\mathrm{H^{+}}] = 2\times 10^{-5}\mathrm{M}$  ，得  $\mathrm{pH}$ $= 4.7$

同理在丙管中  $[\mathrm{H}^{+}] \approx 4.00 \times 10^{-5} \mathrm{M}$ , 故  $\mathrm{pH} \approx 4.4$  。甲基红的变色范围为  $\mathrm{pH} 4.2 \sim 6.3$  (红一黄), 故加酸后乙管中的红色略浅于丙管中的红色。如通入  $2000 \mathrm{ml}$  的氧后乙管中红色仍浅于丙管中的红色, 说明总酸度合格。

溴麝香草酚兰的变色范围为  $\mathrm{pH}6.0\sim$

7.6（黄一蓝）。同样，如通入  $2000\mathrm{ml}$  的氧后，乙管中的绿色色调浅于甲管中的绿色色调，又说明总碱度合格。

操作中应注意将有色溶液加热煮沸5分钟，使溶入的二氧化碳挥发尽冷却后使用，通气过程应尽量避免空气中的二氧化碳再度溶入。所用  $0.01\mathrm{M}$  盐酸液宜调整为准确浓度，加入的量  $0.20\mathrm{ml}$  和  $0.40\mathrm{ml}$  宜用  $0.5\mathrm{ml}$  的刻度吸管量取，误差不应超过  $0.01\mathrm{ml}$  。

# 2. 一氧化碳

氧气中可能含有一氧化碳，利用其还原性，当氧气通过微温的氨制硝酸银吸收液时，若含有一氧化碳，则将析出金属银而显色。

$$
\begin{array}{r l}&2 \mathrm {A g} (\mathrm {N H} _ {3}) _ {2} \mathrm {O H} + \mathrm {C O} + 3 \mathrm {H} _ {2} \mathrm {O} \rightarrow 2 \mathrm {A g} \downarrow\\&+ \mathrm {C O} _ {2} + 4 \mathrm {N H} _ {4} \mathrm {O H}\end{array}
$$

氨制硝酸银吸收液，应在临用前配制。在微温的水浴中避光通入氧气。微温指  $40\sim$ $50^{\circ}\mathrm{C}^{[2]}$

# 3. 二氧化碳

氧气中如含有二氧化碳，当通过氢氧化钡吸收液时将生成不溶性的碳酸钡而发生浑浊。

$$
\mathrm {B a} (\mathrm {O H}) _ {2} + \mathrm {C O} _ {2} \rightarrow \mathrm {B a C O} _ {3} \downarrow + \mathrm {H} _ {2} \mathrm {O}
$$

二氧化碳检出的限度（标准状况下）为

$$
\begin{aligned} & (22400\times 0.04\%) / (84\times 1000)\times 100\% \\ & \approx 0.01\% (\mathrm{V} / \mathrm{V}) \end{aligned}
$$

配制的氢氧化钡吸收液常浑浊，应先过滤，直至滤液澄清后方可使用。通氧气时也应尽量避免空气中二氧化碳的干扰。

# 4. 其它气态氧化物

氧气中如含有气态氧化物，将使碘化钾淀粉试液中的碘化钾生成碘，后遇淀粉显蓝色。

# 三、氧的含量测定

将氧气注入量气管取样  $100\mathrm{ml}$  ，通过三

通活塞，用平衡瓶压入装有铜氨溶液的吸收器中。待氧气被反应吸收后，将剩余气体再回入量气管中，读出氧气的含量。

吸收器中的铜遇氨一氯化铵溶液，即生成具还原性的氯化亚铜氨。它遇氧的反应如下[3]：

$$
\begin{array}{r l}&8 \mathrm {C u N H} _ {3} \mathrm {C l} + \mathrm {O} _ {2} + 4 \mathrm {N H} _ {4} \mathrm {C l} + 4 \mathrm {N H} _ {3} \rightarrow\\&4 \left[ \mathrm {C u C l} \cdot \mathrm {C u C l} _ {2} \cdot 4 \mathrm {N H} _ {2} \right] + 2 \mathrm {H} _ {2} \mathrm {O}\end{array}
$$

为了保证含量测定能顺利、准确地进行，应注意以下事项：

1. 将氧气注入量气管前，应注意吸收器中铜氨溶液内无气泡，并能使铜氨溶液充满至三通活塞量气管一侧液体进出口处。同时使量气管及平衡瓶中饱和氯化钠溶液，也无气泡。  
2. 用量气管准确取氧气  $100\mathrm{ml}$ ，应使平衡液瓶中液面和量气管内液面持平。  
3. 三通活塞在旋转时不得漏气，如转动不灵活时应涂活塞脂。  
4. 吸收器中的铜氨溶液使用一段时间后，玻璃内壁会附着沉积物，此时应更换新鲜铜氨液。  
5. 读取剩余气体体积，应使平衡瓶内液面与量气管内液面相平。

铜氨溶液吸收氧快而不受低温影响为其特点，但二氧化碳、一氧化碳、乙炔及烯烃等气体均能干扰氧的定量[4]。

为了追踪公司用氧变化，降低放散率，我厂两套  $30000\mathrm{m}^3 /\mathrm{h}$  空分设备均采用手动变负荷调节，氧气生产量在  $21000\sim 33000\mathrm{m}^3 /$  h范围内变化。因调节变化幅度大，且较频繁（每天每套空分设备平均变负荷5次以上），每次调节涉及到十几个参数的同步调节，易造成氩馏分中含氮上升，使粗氩中含氮也随之上升，致使过量氢大幅度波动，氩气生产中断，被迫切除精氩塔。其原因如下：

我厂每月平均外销液氩  $700\mathrm{m}^3$  ，作为运输工具的槽车，在充装液氩时有近  $10\%$  的液氩汽化放空，仅此一项每月就损失液氩量达 $70\mathrm{m}^3$  以上。针对如此巨大的浪费，我们提出了回收这部分氩气的想法。具体方法是：由于这部分氩气中含氧、氮很少，纯度基本合格，可将其送入四个  $20\mathrm{m}^3$  的气柜中，然后经一台罗茨鼓风机压缩到  $0.1\mathrm{MPa}$  后，送入氩螺杆压缩机的吸入口与空分设备生产的粗氩相混合，送氩净化间（见图“2”，增改“1”。）

在实际操作中，要注意以下几个关键问题：当启动或停止罗茨风机时，会造成粗氩量波动较大，使加氢量无法及时跟踪调节，以致过量氢含量剧烈波动，影响除氧效果，中断氩气的正常生产。

因此在操作中，要求气体开始送入与停止输送时，操作人员要精心操作，送气、撤气时须缓慢进行。同时对气柜压力须实行监控，防止气柜超压爆破或抽成负压使外界空气窜入系统带来严重后果。

# 4. 精氩塔不合格液氩的排放与回收

我厂外销产品液氩纯度要求含  $\mathrm{O}_2 < 1$  ppm，当精氩塔底部液氩中含氧超过规定值时，液氩不能送入产品贮罐，须排入大气放空。造成液氩纯度不合格的原因很多，如变负荷不当导致粗氩含氮过高、氩净化启动初期以及过量氢的波动等。一般情况下，排放的液氩中含氧约在  $30\sim 50~\mathrm{ppm}$  以内，每次排放所需时间约4小时。按每月两套空分设备平均排液8次左右计算，可知此部分液氩量较大。由于炼钢所需氩气纯度一般要求含氧相对低些（小于  $100\mathrm{ppmO_2}$ ），因此可将此部分氩气回收用于炼钢。

具体措施是：将两个闲置的  $22\mathrm{m}^3$  中间贮罐分别安装在两台空分设备的精氩塔排液阀门旁（见图2增改“2”），专门用于回收此部分液氩，当精氩塔下部液氩纯度合格后，液氩转送至产品贮罐。经多次回收后，中间贮罐液面涨到一定高度时，可用中压液氩泵P4568将其压缩至管网，供炼钢使用。

在进行此项操作时，应关闭中压液氩泵与精氩塔相连通的阀门，以免污染精氩塔。若精氩塔下部液氩中含氧大于  $100\mathrm{ppm}$  时，须将液氩及时排入大气，不应予以回收，防止造成管网用氩质量事故。

2. 作为我厂液氩外销运输工具的槽车，由于往返全国各地长途运输，加上环境差变化大，运输过程中汽化放空损失较大。所以，应每隔两个月对槽车贮罐真空绝热层进行认真抽真空处理，确保其真空度达到要求。  
3. 液氩充槽车及槽车送用户贮罐分装时，应将贮罐压力适当提高，减少充装时液

氙节流汽化损失。

# 杭氧小型空分设备新技术的应用

蔡炜

（杭州制氧机集团有限公司技术开发处，杭州东新路90号，310004）

从1991年开始，杭氧集团公司逐步在小型空分设备领域推广应用一些成熟的先进技术，同时还大胆地尝试一些在试验室试验中证明有广阔前景、但尚未经工艺性实验的保留技术，经过较长周期的实际运行试验，证明这些新技术有着良好的先进性、经济性，本文是自己在实际工作中的一些心得体会。

# 一、板翅式换热器

新技术首推板翅式换热器，以  $150\mathrm{m}^3 /\mathrm{h}$  空分设备为例，老的都是铜制绕管式主换热器、过冷器，结构复杂，原材料消耗大，热端温差高达  $5\sim 7^{\circ}C$  ，运行的经济性较差。所以我们把主换热器、过冷器以铝代铜，以板翅式结构替代绕管式结构作为新一代“150”空分设备技术革新的首要步骤。事实表明，应用真空钎焊技术焊接的换热器不仅使焊接质量上了一个档次，而且同类换热器原材料的消耗也只有铜制绕管式换热器的一半。供镇江氧气厂的首套该类“150”空分设备，经过3年多的实际运行考验，热端温差常年维持在  $2\sim 3^{\circ}C$  ，甚至更小，工况相当稳定。由于热交换器复热不足损失减少一半多，节能的优越性十分明显，创造了良好的经济效益。

其次，从1991年开始板翅式换热器在小型直冷式空气预冷系统也开始大量普及。众所周知，氟利昂蒸发器一般都采用壳管式结构，不仅体积庞大，而且换热器氟利昂侧密封性差，氟利昂容易泄漏，系统工作周期较短，氟利昂需经常补充。我们现在把氟利昂

蒸发器改成板翅式结构以后，大大缩小了蒸发器的体积，减少了氟利昂的泄漏，使得整体结构更加紧凑美观，极大地延长了系统的寿命。出口缅甸的首套该类设备已工作了七年多时间。

第三，为了推进窄流道铝制板翅式冷凝蒸发器的工艺性应用，从1995年开始，我们在“150”空分设备的变型产品  $11081\mathrm{M}$  设计试验中，大胆应用了该类冷凝蒸发器。理论上，这是一种具有强化换热机理的高性能换热器，氮气冷凝温度和液氧蒸发温度的温差可小于  $1.25^{\circ}C$  以下，从而降低了下塔冷凝压力，最终降低了空压机排压。以“150”空分设备为例，采用窄流道铝制板翅式冷凝蒸发器以后，主冷温差可从  $2.5^{\circ}C$  下降至  $1.25^{\circ}C$  ，下塔冷凝压力可下降  $0.04\mathrm{MPa}$  。首套供四川眉山机械厂的“150”空分设备已投运一年有余，工况极为稳定。针对采用中压流程的“150”空分设备来说，虽然下塔压力的降低并不能降低空压机排压，但若该项技术应用于低压流程空分设备，经初步核算，若是气体设备，空压机排压可下降  $5\% \sim 8\%$  ，节能降耗效果十分明显。

# 二、直冷式空气预冷系统

作为成套空分设备的一个有机组成部分，空气预冷系统一般串接于空气压缩机与分子筛吸附器之间，用以降低进吸附器前空气的温度和含水量。合理的使用空气预冷系统可以减轻分子筛吸附器的热负荷和吸附水

分的负担，延长空分设备的运行周期，特别是高温季节尤显重要。

空气预冷系统根据冷却方式的差异，大致可以分为直冷式和间冷式两大类。大中型空分设备由于产氮量少，有大量的污氮可供氮水预冷系统，所以大多采用间冷式空冷方式，有效利用污氮增湿效应。而以“150”空分设备为代表的小型空分设备，由于产氮量大，剩余污氮仅够供纯化器再生用，没有多余污氮进氮水预冷系统，因此，只能采用直冷式空冷方式，让冷媒和空气在换热器中直接热交换，而不通过水做冷量载体。直冷式系统由于构成简单（省略了空冷塔、水冷塔），操作方便，成本低廉，因而在小型空分设备领域得到了广泛应用。

然而杭氧公司在直冷式空冷系统设计制造中也不是一帆风顺的。起先，我们选用国产的开启式制冷压缩机，由于本身质量存在一定缺陷，加上系统成套为非专业人员进行，所以用户普遍反映使用不理想。针对这个情况，为保证成套空分的先进性，1995年开始，我们进行了广泛的调研比较，并且有针对性地进行试配套。目前，初步选定与杭州嘉联净化设备有限公司进行联合设计，生产直冷式空冷系统，制冷压缩机选配全封闭进口产品（如泰康、美优乐、比斯图等优质产品），制冷配件选用丹麦丹佛斯产品，电器选用德国西门子或法国施耐德产品。冷凝器选用香港特艺卧式壳管式系列，蒸发器选配公司自制的板翅式蒸发器或香港特艺卧式壳管式干式蒸发器，疏水器选用日本SMC或台湾省倒筒式产品，由嘉联净化设备有限公司组织制冷专业人员进行系统组装、试压、检漏、空载试车并全方位负责售后服务。

这样，从空气预冷器组的系统设计，配套零部件的选型，制冷系统的组装、调试、售后服务各个环节都本着高标准、严要求的宗旨进行努力，从试配套的情况看，出口埃及

和菲律宾的三套同类型空气预冷机组目前运行良好，用户相当满意。

# 三、长周期吸附器组

长周期吸附器组的最大一个特点就是吸附周期长，可达24小时切换一次，这是普通纯化器工作周期的  $2\sim 3$  倍。众所周知，以“150”为代表的小型空分设备，由于受成本及某些因素的限制，自动化水平比较低，一般都以人工操作为主。当设备稳定运行后，操作的主要工作量就在于一个周期倒换一次吸附器组，劳动强度相对较大，频繁切换引起的冲击损耗除对分子筛寿命有影响外，也会造成分馏塔工况不稳定，特别是在配氩塔工况下情况更加不利。

长周期吸附器的工作机理在于分子筛的静吸附值比动吸附值要大，实验数据表明，静吸附值大约是动吸附值的1.7倍。24小时切换一次，分子筛吸附过程可按静吸附考虑，由于吸附值增大，分子筛的相对数量也会大大节约，以“150”纯化器为例，8小时切换的纯化器每只吸附筒13X—APG分子筛的充装量约  $460\mathrm{kg}$ ，依次类推。若按动吸附考虑，周期延长到24小时，分子筛的充装量将为 $1380\mathrm{kg}$ ，而实际上由于有实验数据作为设计后盾，我们24小时切换的纯化器，分子筛和氧化铝总重才  $820\mathrm{kg}$ ，相对节约  $40\%$  的吸附剂量。

还有一个重要因素，纯化器的工作周期一旦延长至24小时，再生功率和再生气量都大大减少，特别是再生气量可减少约  $40\%$  。换句话说，就是可供用户实际使用的纯氮气量增加了，这对用氮量大的石化、玻璃制品等行业来说，经济效益是相当明显的。

# 四、填料上塔

我公司填料应用于空分塔的历史可以追溯到60年代。直至今日，我们生产的液氮机所配分馏塔仍然采用铜制马鞍形填料，然而最新科技成果显示，目前在化工领域已广泛

采用波纹规整填料。实验室数据表明，和拉西环马鞍型填料相比，波纹规整填料有更大的比表面积，更好的传质效果，更小的阻力，更少的材料消耗。为了推进填料在大中型空分塔上的应用，从1993年开始，我们率先在“150”空分塔上连续进行了三次填料试验。结果表明，虽然填料的精馏效率比预期的要差，大约每米填料高度相当于  $4\sim 5$  块理论块。

另外，相对于“150”填料上塔，填料高度增加相对于填料直径加大，对精馏效果的改善更有好处。然而我们希望的另一个关键参数上塔阻力则下降更多，尽管填料上塔比起筛板上塔来说高度增加了3米多，但实测阻力仅为筛板阻力的0.13倍，绝对值下降了 $0.015\mathrm{MPa}$ 。在主冷温差不变的情况下，下塔冷凝压力可减少约  $0.04\mathrm{MPa}$ 。对于采用卡皮查循环的空分设备而言，空压机排压同样下降了  $0.04\mathrm{MPa}$ ，节能效果明显。该实验的成功，对“150”空分设备本身意义并不大，但对于杭氧公司在大中型空分上塔中全面采用规整填料，却具有划时代的重要意义。

# 五、多孔管冷凝蒸发器

多孔管冷凝蒸发器实际上是一种蒸发侧为多孔表面、冷凝侧为纵向凹槽相匹配的双面强化相变传热的一种新型冷凝蒸发器，和传统的管壳式冷凝蒸发器相比具有传热效率高、温差小等显著特点。在化工领域，多孔管的应用历史可以追溯到90年代之前，而真正应用于空分设备上却从1990年才起步，杭氧公司从开发节能型“150”空分设备一开始，就大胆尝试了该项技术，并在厂内连续进行了92小时工艺性试验。结果表明，多孔管冷凝蒸发器的传热温差与光管相比从  $2.5\mathrm{C}$  减少至  $1.2\mathrm{C}$  ；传热系数由  $600\mathrm{W / m^2}\cdot \mathrm{C}$  提高到  $1500\mathrm{W / m^2}\cdot \mathrm{C}$ ，就该冷凝蒸发器而言，它和窄流道铝制板翅式冷凝蒸发器类似，取

得了成功。杭氧第一套采用该新技术的“150”空分设备，已在镇江氧气厂连续运行了整整五年，性能相当稳定。

# 六、铜铝接头

以“150”为代表的小型空分塔由于受高度限制，一般要求塔板有较高的精馏效率。实践证明，小直径环流板比对流板精馏效率要高  $10\% \sim 20\%$  ，所以小型空分塔都采用环流板，而工艺上铝制环流板目前还无法制造，因而精馏塔只能采取铜制结构，这样势必存在铜铝焊接问题。铜和铝按常规的焊接工艺是无法实现焊接的，这一难题一直是阻碍小型空分设备同时采用高效铝制板翅式换热器和铜制空分塔的拦路虎。为了解决该工艺问题，我们曾先后四次派人携带铜棒、铝棒赴北京，在北京无线电缆厂的通力合作下，用高频摩擦焊工艺成功地解决了  $\phi 50\mathrm{mm}$  至  $\phi 90\mathrm{mm}$  口径的铜铝棒焊接，这对于节能型“150”空分设备的开发起了关键作用。

# 七、空压机、膨胀机、氧压机（三机）

我们所谓的“三机”，主要是指空气压缩机组、气体轴承透平膨胀机组、氧气压缩机组。作为成套空分设备的重要组成部分，进入90年代以后，小型空分设备领域进行了全面的更新换代，几乎所有活塞式机械设备，结构形式都由L型、卧式对称型进步为立式，极大地延长了易损件的寿命，机械性能更趋优化。以ZW-17/20型立式无润滑空气压缩机、2Z2-2.83/1.65型无润滑氧气压缩机为代表的新机型已全面占领市场。除此以外，针对不同的用户，我们还把无油螺杆空气压缩机组、气体轴承透平膨胀机组等高新技术产品大胆应用于成套工艺流程，使得产品的先进性、可靠性、可维修性进一步提高，促进了产品上档次、上水平。

（1997年12月9日收稿）

# 空气压缩机的维护

# 范湘生

（铜陵市医用氧气厂，安徽省铜陵市，244022）

空分设备一经启动，各配套部机就处于联动作业状态。任何一台部机有故障，都将影响供气。为不影响生产，除合理安排检修时间外，更重要的是了解和解决好影响部机正常运转的重要技术环节，这些环节是设备管理首要监护对象，下面就此作简单的阐述，供同行参考。

# 一、空气滤清器的维护

我厂使用的是杭氧产ZW-17120型无油润滑空压机（工厂代号：0231），空气滤清器是以铁丝网夹装拉西环组成，网环间沾有粘性油，空气通过时，灰尘杂质粘附在上面从而达到过滤之目的。因此，要定期清洗粘在上面的灰尘，重新浸入粘性油，我们在维护时一般  $2\sim 3$  个月检查清洗一次，视操作情况定。

判断方法：

当二、三级进排气压力正常，气阀、活塞环、上密封器情况良好，在提高空压机压力、调整节流阀流量、增大透平机气量后，若分馏塔仍感冷量不足时，表明空压机的吸气量不够，不足的原因是滤器清器阻力过大。此时就要考虑清洗滤清器。

清洗方法：

拆下拉西环，用温度为  $60\sim 70^{\circ}C$  、浓度为  $10\%$  的苏打水溶液洗涤，并用热水冲净晾干。冬季浸入  $20^{\#}$  机械油或  $25^{\#}$  变压器油；夏季浸入  $13^{\#}$  或  $19^{\#}$  压缩机油中。浸入后，余油流尽，开车运转1小时排空后，方可送气。

值得注意的是：空压机气缸和活塞环的寿命（特别是无油润滑空压机），在某种程度上与滤清器工作的好坏密切相关。例检时，如

发现气缸中有灰尘、砂粒，说明滤清器工作效果不佳。此时，除考虑上述原因外，还要注意网孔是否太大或用油粘度是否过小。

# 二、气阀的维护

通常根据压力的变化或阀门压盖上温度的变化，来决定是否拆卸阀门。清洗剂用柴油，清洗后可用四氯化碳试漏，然后逐级吹除，逐级安装。气阀故障应根据温度、压力、声响进行综合判断，前一级进排气压力偏高应从后一级进排气阀上找原因。

# 1. 温度

如果二级气缸进气阀关闭不严，气缸内被压缩气体会漏向二级进气管道，使进气温度上升，用手摸二级进气阀罩可以感受到发烫。如果二级排气阀关不严，高压气体会倒灌回气缸内，倒灌的气体未经冷却，温度很高，从而使排气温度相应升高。从二级排气温度计上可以反映出来。

# 2. 压力

如果二级进气阀关不严，气缸内高压气体漏向吸气管道，这无疑使吸气压力上升，排气压力下降。同样，二级排气阀关不严，会使排气管内高压气体倒回气缸内，这自然使排气压力下降，吸气压力相应憋高。上述两种情形都表现在一级排气压力偏高。

另外，可用听诊器从气阀罩上听到气阀漏气时发出的嘶嘶声。

# 3. 气阀修复

气阀的修复包含阀座、阀片的修复。阀座研磨时注意用力要均，研磨膏粒度宜细些。阀片应双面微量磨削并退磁，以保证其平整不变形。

# 三、活塞环的维护

1. 活塞环是用填料聚四氟乙烯制成。由于聚四氟乙烯的热膨胀系数大，装配时务必要保证搭口处的余隙和环在槽中的轴向间隙  $(0.2\sim 0.3\mathrm{mm})$  。  
2. 活塞环搭口处由于强度弱，易损坏，气体通过损坏处从高压侧泄漏至低压侧时，影响前级排气，导致前级压力偏高，由此可判别活塞环在气缸中的完好情况。  
3. 活塞环的使用寿命与气缸内气体温度及其含湿程度密切相关。虽然，活塞环内加入的填充材料可以改善导热与耐磨性能，改善抗压能力，但由于聚四氟乙烯有机械强度差、硬度低，以及随着温度、压力的增高容易产生塑性变形的缺陷，就要求我们日常操作时，既要保证气缸内气体的温度不要太高（可通过三通，将已通过热交换的水与进气缸的水连通以达到要求的水温），以免增大塑性而降低硬度；又要保证气缸水套内与气体进行热交换的冷却，故温度不要太低，以免潮湿的气体遇冷而凝结成水。故各级气缸的进气温度应低而且干燥。这就要求冷却器的热交换效果要好，冷却水温要低，气体在分离器内作离心运动后，绝大部分冷凝水应被分离出来。

另外，在操作上，除勤放分离器的冷凝水外，还要求操作者利用气缸水套和冷却器的排水阀的开度来调节水温。各级冷却器排水阀的开度，应以温度计测得的最高排水温度为依据；气缸水套排水阀的开度，以在保证排水温度不超过某一温度的情况下，进水温度宜高不宜低来调节。由于各地气候条件不同，设备管理人员可以根据当地环境条件自行测试。

# 四、气缸的维护

整台空压机，气缸的使用寿命是设备维护的重点也是维护的难点，我厂无油润滑空压机气缸的使用寿命仅三年，气缸失效的原

因是内径磨损超差过大，特别是三级缸尤为突出（气缸磨损快的原因，可能与气缸内有冷凝水有关）。要延长其使用寿命，必须做好如下几方面的工作：

1. 定期清洗空气过滤器和进排气阀。装阀时，气腔应逐级吹除，要求粉尘和机械杂质尽量不带入气缸内。通常，空气滤清器三个月清洗一次，气阀清洗、气腔吹除也要勤。  
2. 密切注视活塞环和导向环的使用情况，一旦发现活塞环断裂(搭口处)要立即更换。维护人员应逐步准确掌握其寿命周期，更换要有计划性，一般三个月就要作好换前准备，更换时注意保持气缸内清洁干净。导向环更换较困难，装配时务必要保证其轴向间隙和搭口余隙，以防止气体温度高，环膨胀后在气缸内过量干摩擦磨损气缸。  
3. 对空压机的冷却器、分离器、管道，应每年清洗一次，一方面可以防止锈斑和积炭堵塞分离器的油水排放处，避免排不出的冷凝水进入气缸内；另一方面改善了换热效果。此外，更重要的是防止脱落的锈斑和积炭随空气进入气缸内，加剧气缸的磨损。

# 五、曲轴与连杆的维护

由于ZW-17/20型空压机采用强力润滑，润滑油在轴瓦和轴颈之间形成的油膜极大地改善了轴瓦和曲轴因磨损而降低的使用寿命，但在使用中要经常检查油质情况，保持油路的畅通。油泵过滤网两个月清洗一次，以防止使用过程中油路断油。空压机启动时必须用手柄先摇动油泵，待油压表有压力上升时方能启动空压机。

总之，影响空压机正常运转的因素很多，上述因素仅是众多原因中的突出点，操作时要密切注视空压机的各种状态参数并进行跟踪监测，发现问题及时处理，将故障消灭在萌芽状态。

（1997年10月18日收稿）

# DA 200 型空压机空气冷却器的改造

# 黄代富 朱学太 朱尚志

（达州钢铁集团公司制氧车间，四川省达川市西河路25号，635002）

# 一、改造之前

达州钢铁集团公司制氧车间1号氧气站的一套KDN  $1500 / 1500 - 2$  型空分设备是开空厂1983年出厂的产品，配套的原料空压机为DA  $200 - 61$  型三段六级两次中间冷却的单轴型离心式空气压缩机。这一类空压机是仿前苏联50年代的产品，每两级压缩后进行一次中间冷却。

由于技术陈旧，空压机的运转费用较高。虽然冷却水的用量很大，但排气温度仍然较高，气程阻力损失较大。尤其是空气冷却过程中产生的冷凝水不能有效的分离而被带入压缩机，导致设备发生电化学腐蚀损坏，使级间窜气严重，叶轮的流道表面结成难以清理的污垢，迫使设备非正常停车，严重影响生产。该压缩机自1985年投产以来，每年设备检修时都必须更换气封，必须花费一周的时间来进行叶轮流道的除垢。1992年，由于叶轮严重锈蚀，我们不得不更换新转子。

# 二、改造方案

为了延长设备的运行周期，降低能耗，我们提出了对该机的空气冷却系统进行技术改造的设想。根据分析，如果对压缩机主机也同时实施改造，效果可能会更好。但主机改造的技术风险大，费用高，企业难以承受。根据多年的运行经验和对问题的分析排队，我们认为原有的空气冷却器结构不合理，换热效率低，缺少可靠的冷凝水分离装置，是压缩机存在问题的主要病因。如果能对各段的排气进行充分冷却，并充分分离和排放冷凝水，也能在很大程度上改善空压机的运行状

况。因而认定这是一个稳妥可靠、花钱少见效快的技改方案。

改造工程的基本目标是在加大空气量的条件下，新冷却器应同时具备良好的换热和冷凝水分离性能。必须能在原有的冷却器所占的空间位置上安装。既不能改变压缩机的基础，也不能干扰其它的设备的运行与维修。改造工作的核心是采用技术先进的空气冷却器。为此，我们查阅了有关的技术资料，走访了兄弟厂家和换热器制造厂家，并吸取了兄弟厂的经验教训，得到了有价值的信息和必要的技术数据。在此基础上我们进行了空气冷却系统的设计，委托一家具备技术实力的专业生产厂家进行新冷却器的制造。

新的中间冷却器的技术参数如下：

新的空气冷却系统采用两台卧式、筒形的管壳式换热器。两台冷却器重叠放置，以充分利用压缩机基础下面原有的空间。换热元件采用矩形翅片管束单元，换热面积大、阻力小、刚性好。采用多流程布置，适当提高

# 三、改造之后的效果

1994年8月我们进行了冷却器的更换与安装，随后立即投入运行。新冷却器投产后效果十分明显：在空压机的排气量增加 $800\sim 1200\mathrm{m}^3 /\mathrm{h}$  、空气进分馏塔压力提高 $0.02\mathrm{MPa}$  的条件下，冷却器的排气温度明

显降低。主电机电流表读数略有下降。冷却水的温升由原来的  $4^{\circ}\mathrm{C}$  提高到  $8^{\circ}\mathrm{C}$  。冷却水的消耗量减少近一半。氧气产量增加了  $120\sim 150\mathrm{m}^3 /\mathrm{h}$  。年均新增效益约80万元。

自1994年9月新的冷却系统投入运行后三年来，没有发现由冷凝水引起的叶轮和气封的明显锈蚀，也未发生过由于级间气封腐蚀或叶轮流道壁面结垢引起的压缩机非正常停车。每年定期检修时，叶轮流道上的积灰很容易清除。仅减少更换级间气封一项，每年就可节省开支近万元。更重要的是确保了制氧生产的长期稳定运行。1996年7月，在四川省冶金厅的主持下通过了对这一技改成果的技术鉴定。

冯香玖

（大庆石油化工总厂水气厂，黑龙江省大庆市卧里屯，163714)

摘要 简要介绍目前分子筛流程空分设备主冷爆炸防治的基本情况，着重探讨空分主冷发生化学爆炸的机理、形成条件和碳氢化合物控制指标的制定，并针对大庆空分设备实际情况提出具体的空分设备主冷防爆的预防措施。表2。

主题词：大型空气分离设备 主冷凝器 爆炸 机理 条件 控制指标 预防措施

# 一、概述

空分设备是石化、冶金行业主要的生产装置之一。近年来，由于空分设备制造缺陷、管理欠善等问题和其它不明原因，全国已发生多起空分设备爆炸事故。据不完全统计，全国共发生小型空分设备的爆炸事故100多起，大中型空分设备的爆炸事故30多起。尤其是1997年5月16日抚顺石化公司乙烯化工有限公司空分设备，因主冷碳氢化合物超标而发生的主冷粉碎性爆炸事故，致使4人死亡，直接经济损失预计400万元。惨痛的教训再次警示人们：严格控制空分设备液氧中的乙炔及其它碳氢化合物的含量，是确保空分设备安全运行、防止恶性爆炸事故发生的重要措施。

大庆石化总厂共有两套大型空分设备，均采用分子筛吸附净化双级精馏的工艺流程。自1985年开工以来，由于大气中烃类含量超高，经常造成主冷液氧中碳氢化合物严重超标。尽管我厂采取一系列有效措施，如实行主冷全浸没操作和液氧定期排放等，在很大程度上降低碳氢化合物的含量，但乙烷单项指标仍严重超标，甚至多次达到和超过停车值（通常情况下乙烷值为  $20\sim 50~\mathrm{ppm}$  最高达  $749~\mathrm{ppm}$  。因此，加强管理，改进工艺，控制主冷碳氢化合物聚集和引爆源，是

确保空分设备安全生产的重要因素之一。

# 二、主冷爆炸机理

众所周知，形成化学性爆炸的因素有三个方面：1. 可燃物；2. 助燃物；3. 引爆源。

在空分设备主冷中，可燃物主要是乙炔、碳氢化合物或油分等爆炸危险杂质；助燃物为气氧、液氧；引爆源主要有：（1）爆炸性杂质固体微粒相互摩擦或与器壁摩擦；(2)净电放电。当液氧中含有少量冰粒、固体二氧化碳时，会产生静电荷。如果二氧化碳的含量提高到  $200\sim 300~\mathrm{ppm}$  时，所产生的静电位可达到  $3000\mathrm{V}$  ；（3）气波冲击。流体冲击或气蚀现象引起的压力脉冲，造成局部压力高而温度升高；（4）化学活性特别强的物质（臭氧、氮的化合物等）的存在，使液氧的可燃物质混合物的爆炸敏感性增大。

.不论实际是那种原因引起的爆炸，对空分设备来讲，防爆的根本措施是要清除爆炸危险杂质，控制其含量在安全允许范围内。

# 三、爆炸源形成条件

空气中除氧气、氮气外，还会有少量的水蒸气、二氧化碳、乙炔和其它碳氢化合物等气体，以及少量的灰尘等固体物质。国内大中型分子筛净化流程清除空气中水分、二氧化碳和乙炔等杂质的方法多采用吸附法。即利用硅胶或分子筛等作吸附剂把空气（液

空、液氧）中所含的水分、二氧化碳和乙炔等杂质分离出来，浓缩在吸附剂表面上（非化学反应），加温再生时把它们赶掉，从而达到净化目的。我厂使用的吸附剂为13X分子筛，正常情况下，能够将水分、乙炔和二氧化碳等有害杂质较彻底地清除。

由于我厂化工装置比较集中，如果装置泄漏量过高或烃类产品直接放空，将会造成空分设备吸入口的碳氢化合物含量超标。对我厂分子筛流程的空分设备，13X分子筛具有对孔径相近的极性分子吸附性强的特点，水分、二氧化碳和乙炔基本上可以在分子筛吸附器中脱除，其它烃如甲烷、乙烷绝大部分随空气进入空分塔中。这些物质大部分溶解在液体中，少量随液氧蒸气而带走。仅以危害性最大的乙炔为例，蒸发气化量仅为液体含量的1/24。当液体中烃的浓度不断增加，并超过其溶解度时，就会以固态形式析出并聚集，在一定条件下与氧混合形成爆炸源，当引爆因素存在时就会发生化学性爆炸。

# 四、控制指标的制定

在林德公司提供的操作手册中，仅明确了乙炔指标的分析频率和控制指标，即“每天必须至少测定一次主冷中乙炔的浓度，液氧中乙炔含量达1ppm时必须停车”，而对其它碳氢化合物的控制指标未加说明。因1986年燕山石化公司空分设备主冷爆炸事故发生后，1989年中石化总公司制定的《关于空分设备液氧中乙炔及其碳氢化合物控制指标的规定》，控制值的确定方法根据以下几点：1．首先是根据《机械工程手册》第17章安全技术1.4.1“液氧中危险杂质许可量的控制”中有关内容计算出来的某一数值，然后在根据流程及设备结构特点，结合实践经验加一修正确定。2．定标中参考林德、日本和法国有关标准。即乙炔靠近林德标准，总碳控制值靠近日本标准，单项碳氢化合物靠近法国指标。3.注意国内空分设备运行现状

和部分生产厂的大气条件。控制指标如表1。

# 五、我厂空分主冷防爆预防措施

目前我厂空分设备主冷防爆的主要隐患是乙烷含量单项指标超标严重，其最大值高达  $749~\mathrm{ppm}$  ，无法达到总公司所要求的控制标准。同时引爆源的控制也是不容忽视的问题之一。为了确保空分设备的安全生产，我厂根据自身的实际情况，应采取以下安全技

术措施。

# 1. 空分主冷碳氢化合物控制措施

# （1）保证吸气条件

空分设备吸气条件是影响主冷液氧碳氢化合物含量的直接因素，其吸气条件应满足表2。

（2）开发新型吸附剂并改进分子筛吸附器

根据我厂空分设备吸附器后乙烷的实际情况，我们正和西南化工研究设计院联合研制开发对乙烷选择吸附性强的专用分子筛或其它吸附剂，进行分层装填在13X分子筛的上部，以脱除乙烷杂质。同时，由杭氧公司对分子筛吸附器进行技术改造，将目前的卧式双筒分子筛吸附器改造成卧式单筒混合床分子筛吸附器。该吸附器有以下特点：

① 增加吸附器下层装载的13X分子筛的充填量，以确保对水分、二氧化碳和乙炔的彻底清降。  
②在吸附器上层装载脱乙烷等饱和烃的吸附剂，以确保对乙烷等杂质的清除，防止主冷液氧中该杂质超标。  
③避免我厂现用空分设备的双筒卧式纯化器容易发生床层破损、压条松动等分子筛泄漏事故，延长装置运行周期。

# (3）防止碳氢化合物析出聚集

低液面操作或液面大幅度波动会使液氧

中有害杂质析出聚集，可采取以下措施：①主冷全浸没操作；②排液时先提高主冷液面；③连续排放  $1\%$  液氧。

# 2. 控制引爆源

# （1）减少二氧化碳的进塔量

二氧化碳对分子筛流程的空分设备生产的危害较大。在主冷液氧中它不仅容易堵塞换热通道，更严重的是有害物质的吸着剂，十分危险，因此我们应采取对分子筛后空气中二氧化碳的在线监测和控制，保证其含量  $< 0.5 \mathrm{ppm}$  。

# (2）防止静电

为防止静电引爆，除空分塔有良好的防雷、防静电接地外，氧气管道（液氧、液空）上的法兰跨接电阻应  $< 0.03$  欧姆。

# （3）减少压力脉冲

在空分不排液开工时，精馏塔的升压过程必须缓慢进行。同时极力避免因操作不当而引起的液悬现象。

# 3. 增加监测措施

为能够及时监测主冷碳氢化合物的含量，以便在必要时采取防范措施，应增设碳氢化合物在线分析仪表，对碳氢化合物数值进行连续监测。

# 六、结论

综上所述，主冷防爆是空分设备安全生产的头等大事，必须予以高度重视。空分设

备的主冷防爆首先要采取技术措施，控制好液氧中碳氢化合物的含量，确保各项指标均在所要求的控制指标范围内。其次是加强引爆源的控制和增加监测措施，同时应加强管理，堵塞漏洞。只有这样，空分主冷爆炸的悲剧才不会重演。此文供烃组分杂质超标的空分设备用户和制造厂家参考与借鉴。

有害杂质气体，特别是乙炔和其它碳氢化合物在空分塔内的积聚、爆炸，与下述诸多因素有关，如制氧工艺流程、对有害杂质气体净除手段、主冷结构、管道配置及生产运行方式，以及有关对乙炔和碳氢化合物检测等等。下面就碳氢化合物在空分塔内的积

# 一、乙炔和碳氢化合物在空分塔内的积聚过程及机理

# 1. 大气中存在乙炔及碳氢化合物

3. 连续不断的排放液氧，可以防止入塔碳氢化合物在液氧中的积累

如表4所示，碳氢化合物的沸点都高于液空和液氧温度，且在  $90\mathrm{K}$  时的饱和蒸汽压均很低，这就决定了它们一旦进入塔内，碳氢化合物均会在液空、液氧中积聚。但不管是理论计算，还是实践，正常情况下，只要连续排放  $1\%$  左右的液氧，就可防止碳氢化合物在液氧中的积聚。

# 二、碳氢化合物在液氧中爆炸的机理

1. 燃烧或化学性爆炸必须同时满足三要素

众所周知，燃烧或化学性操作必须同时满足三要素：即可燃物，助燃物及引燃引爆能量。它们不仅必须在同一系统中同时存在，而且均需有足够的数量。在主冷里，只要正常生产，那么液氧和气氧是始终存在的，且其量非常充分，故对助燃物无须讨论。下面重点讨论一下可燃物和引燃引爆能量。

2. 碳氢化合物在液氧中爆炸的条件

前面已讨论过碳氢化合物在液氧中的积聚过程。碳氢化合物在液氧中是否只要有积

聚就会发生爆炸，还是要积聚到什么限度后才会发生爆炸？科学研究证明了碳氢化合物液氧中的含量必须达到其爆炸极限，且析出固体碳氢化合物后，再有足够的引燃引爆能量，才会发生爆炸。

必须指出的是：有时，甚至是大部分时间，液氧中的乙炔及其它碳氢化合物含量虽未达到它们的爆炸下限和析出固体的程度。但主冷却发生了爆炸。本人收集到的数十起爆炸前对乙炔有化验数据的主冷爆炸事故，液氧中  $\mathrm{C}_2\mathrm{H}_2$  均小于  $1\mathrm{ppm}$ ，远未达到其爆炸下限和析出固体的程度。这只能说明，由于主冷结构不合理或局部通道不畅通，造成乙炔等碳氢化合物局部浓缩析出固体，在满足其点燃能量后，即发生局部爆炸。

# 3. 爆炸威力相关因素

爆炸威力的大小，决定于参与化学反应的可燃物的数量及其高热值。参与化学反应的可燃物数量越大，其高热值越高，则爆炸威力就越大。

# 4. 引燃引爆能量

对空分塔而言，引燃引爆能量主要有：

（1）摩擦与撞击的机械作用。固体颗粒，特别是乙炔等碳氢化合物固体与器壁及主冷通道的摩擦、撞击产生的能量。

(2) 静电放电火花与干冰、分子筛粉末。固体乙炔等在液氧中沸腾时与器壁及主冷通道的摩擦、撞击，可产生很高的静电电压（因液氧电阻极大）。当液氧中干冰、分子筛粉末达  $200 \sim 300 \mathrm{ppm}$  时，静电电压即可达  $3000 \mathrm{~V}$  。静电场强度取决于固体颗粒在液氧中的运动速度、杂质的数量和性质。对同种固体微粒，其运动速度快，杂质数量多，产生的静电电压则就迅速提高。静电放电，即可引燃固体乙炔等碳氢化合物[5]。

（3）压力脉冲和气流冲击。主冷内，因液氧沸腾运动，液体的冲击波可使气泡得到瞬间压缩，从而使局部温度提高。文献指出，在液氧沸腾时，其运动速度可达  $10\mathrm{m / s}$  以上，此时顺冲击波方向的压力可达  $10\mathrm{MPa}$ ，逆冲击波方向的压力可达  $20\mathrm{MPa}$ ，从而瞬间局部会绝热压缩产生高温。可见压力脉冲、气流冲击也可能是一个引爆源[5]。

（4）具有特别反应能力物质的作用：

①臭氧  $\left(\mathrm{O}_{3}\right)$  的作用。碳氢化合物在液氧中的爆炸敏感性，随臭氧的存在而提高，如不饱和的碳氢化合物悬浮于臭氧浓度为100ppm的液氧内时，其引爆所需的冲击能则较无臭氧存在时小，在爆发率为  $100\%$  时，引爆

所需能量一般下降  $30\% \sim 45\%$

$②$  氮氧化合物的作用。当二氧化氮与臭氧在液氧中同时存在时，碳氢化合物的爆炸敏感性就有很大提高。许多主冷爆炸事故报告都指出，当氧气站周围有关装置向大气中排放大量氧化氮时，即使液氧中乙炔含量比大气中无氧化氮时的乙炔含量低很多，但主冷仍发生了爆炸，可见氮的氧化物促进了主冷的爆炸。

综上所述，当主冷或其它有氧的地方（或局部）的可燃物  $\geqslant$  其爆炸下限，并达到其点燃能量后，爆炸就变得难以避免了。

# 三、对有害杂质在空分塔内积聚、爆炸的防范

防止有害杂质在空分塔内的积聚、爆炸的基本要求如下：首先是制氧的原料空气条件要好，大气中的有害杂质不超量；第二是加工原料空气入塔前，要尽最大可能将其中的有害杂质净除掉；第三是有害杂质一旦进入塔内后，应采取有效措施阻止或不让它们在液氧中浓缩；第四是配备足够的必要的检测仪器，随时监督它们在液空、液氧中的含量，一旦超过正常运行的允许值，即采取有效措施，稀释或消除之。具体措施如下：

# 3. 防止入塔有害杂质的积聚浓缩

一是防止  $\mathrm{C}_2\mathrm{H}_2$  及  $\mathrm{C_mH_n}$  在液氧中积聚，二是防止它们在主冷中局部浓缩析出固体。

为防止  $\mathrm{C}_2\mathrm{H}_2$  及  $\mathrm{C_mH_n}$  在液氧中积聚，对只生产气氧的空分装置，在分子筛有效工作时，应连续不断的排放液氧，其量为  $0.5\% \sim 1\%$  气氧产量，即可防止乙炔及碳氢化合物在液氧中的积聚。但如果在离制氧站  $200\mathrm{m}$  范围内大量排放  $\mathrm{CO}_{2},\mathrm{C}_{2}\mathrm{H}_{2}$  或其它  $\mathrm{C_mH_n}$  时，为了防止它们在液氧中的积聚，首先是缩短分子筛的工作周期，同时增加液氧的排放量。

为防止  $\mathrm{C}_2\mathrm{H}_2$  及  $\mathrm{C_mH_n}$  在液氧中的局部

浓缩，一方面要注意改进主冷结构，另一更重要的方面是主冷凝蒸发器应采取全浸式操作，保证液氧的循环倍率。

# 4. 加强  $\mathrm{CO}_{2}$  、  $\mathrm{C}_{2} \mathrm{H}_{2}$  及  $\mathrm{C}_{\mathrm{m}} \mathrm{H}_{\mathrm{n}}$  的检测

配备足够、有效的检测仪表，随时监督 $\mathrm{CO}_{2}$  、  $\mathrm{C}_2\mathrm{H}_2$  及  $\mathrm{C_mH_n}$  总量。

① 在分子筛吸附器加工空气出口处配备在线检测仪，当  $\mathrm{CO}_{2} \geqslant 5 \mathrm{ppm}$  时，必须立即切换分子筛，保证加工空气中不含有较多的  $\mathrm{CO}_{2}$  、  $\mathrm{C}_{2} \mathrm{H}_{2}$  及  $\mathrm{C}_{\mathrm{m}} \mathrm{H}_{\mathrm{n}}$  。

② 完善监测手段，在线监测与定期监测两者并用。应尽可能配置质量可靠的在线监测仪表，随时监测液氧中的  $\mathrm{C}_2\mathrm{H}_2$  及其它碳氢化合物含量，以便及时发现问题。同时每天至少做一次离线检测，当  $\mathrm{C}_2\mathrm{H}_2$  及总碳量达到报警值时，应及时采取包括增加排放液氧量在内的各项措施，以稀释它们在液氧中的浓度。液氧中乙炔及总碳的报警值及停车值如表10。

——铝筛板优秀的安全记录是主要原因。几百套装置在30多年的使用中，从没有发生过上塔提馏段铝筛板有引燃或燃烧的事故。

——直接引燃的特征，是要有能力提供高于  $2000\mathrm{C}$  点火温度所需能量的物理机制，如摩擦、撞击、压缩等。已经查明，这些机制对填料来说是不可能发生的。Lassmann和Kirzinger在1993年已发表了详细的论述[1]。

间接引燃取决于二次材料如碳氢化合物燃烧引起的能量释放。这首先必须引燃助燃剂，然后由助燃剂的化学反应所释放的能量必须能够将铝点燃。这里的助燃剂就是氧中累积的乙炔和碳氢化合物以及填料生产过程中的润滑油。

——原料空气中的碳氢化合物累积可通过液氧的蒸发来控制。这个机制只发生在上塔的底部，而决不可能发生在其它地方。  
——以前报道过的不带分子筛吸附器的空分设备中主冷凝蒸发器的燃烧事故很可能是乙炔的自燃引起的。现在用分子筛吸附器来去除空气中的水分和二氧化碳后，也能完全将乙炔除去。所以，这一点燃（火）机制也已不复存在。  
——有几起事故是由于小凹穴内累积的碳氢化合物在加温解冻时被引燃，这多半是

由于液态碳氢化合物在引燃之前先被蒸发和压缩。对于填料来说，这个机制可以排除掉，因为既不存在氧蒸发所需的热流，也没有可以被压缩的绝热空间。

——假定表面上的含油量相同，则由填料带入塔内的润滑油量可比筛板多  $15\sim 30$  倍，因为填料的金属表面积大。因此，规定金属表面残留的润滑油不得超过  $50~\mathrm{mg / m^2}$  必须通过严格的质量管理来保证这个规定的实现。文献[2]中的数据指出：如此薄的油膜既不会引燃，也不会在储存、运输、安装和操作时使油累积，因此可排除由于油的燃烧而引起的危险。

——空分设备中上塔铝的可燃性这一问题，根据技术经验和参考文献可总结如下：

尽管早期的商用装置发生过在板翅式主冷凝蒸发器中碳氢化合物的累积和着火，但  $0.2\mathrm{mm}$  的铝翅片也没有被引燃，只是变了形。由此可以得出结论：即使在上塔底部氧浓度很高的地方，  $0.2\mathrm{mm}$  厚的规整填料也不会被引燃。

根据Benning等人著的文献[3]，铝条在压力  $140\mathrm{kPa}$  、氧浓度为  $99.8\mathrm{mol}\%$  的环境下是不可燃的，这个压力正是上塔的典型压力。

1991年Dunbobbin等人[4]发表了有关规整填料安全性的文章。文中指出，规整填料的几何形状，即填料很薄的金属、较大的比表面积和“相对的绝热性”，可能会在氧精馏塔内导致燃烧的危险。作者采用各种金属材料样品做了试验，结果是铝填料样品在以下条件下是可燃的，并与液氧（纯度  $>98.5 \mathrm{~mol} \%$  ）发生剧烈反应：

——样品必须部分浸入液氧；  
——点火必须发生在样品没有浸到液氧的部位；  
——助燃剂：  $1\mathrm{g}$  铝热剂加  $0.24\mathrm{g}$  镁。

这些反应条件在设备操作时是完全不现

实的，因为上塔的填料不可能浸入液氧之中，也不可能在上塔有助燃的铝热剂。因此，这些试验结果对填料的工作没有直接的利害关系。

尽管如此，由于空分设备的结构和安全性关系重大，我们还是对接触到液氧的铝填料的燃烧，是否真的只与填料外形有关这一问题做了试验。虽然筛板已经用了几十年了，但是其试验结果与填料的试验结果一样，都没有正式发表过。

# 三、空分设备中填料安全性的试验研究

试验方案：所看到的铝填料样品的可靠性，是否表示与筛板相比铝填料更具有燃烧的危险性，这一问题至关重要。此试验方案的基本观点就是：采用Dunbobbin等人完全相同的试验条件，对筛板和填料样品的反应进行比较。

此外，填料和筛板样品的燃烧试验，采用了碳氢化合物的燃烧来助燃，因为只有碳氢化合物才是唯一现实的点火源。另一方面，碳氢化合物燃烧时铝可能被熔化，而液氧与熔化的铝接触也可能引起燃烧。最后，我们对填料生产时残留的润滑油可能成为铝燃烧时的助燃剂这一情况进行了重新评估。

# 四、填料与筛板的可燃性比较

第一类试验中，采用了铝热剂引燃填料并观察其在氧中的燃烧方式。参考文献[4、5、6]的作者在采用了有很强助燃能力的铝热剂后才使铝填料燃烧。铜容器中填料样品（单片圆柱形填料）的一半浸入液氧中（见图1)，将铝热剂球放在没有湿的表面上，然后电子点燃铝热剂，铝热剂开始反应。

在采用单片填料或一些相互错开的填料做试验时，8次中有6次由于铝热剂燃烧的热量使铝片烧穿了孔，只有两次看到剧烈的能量释放，即铝片被扭曲并烧穿。在用圆柱形填料（铝重  $100\mathrm{g}$  ）做试验时将样品的一半浸入液氧，这时可看到铝突然燃烧。但是，如

果填料样品全部浸入液氧中的话，燃烧就不能持续下去。

气氧中的油燃烧产生的热量会使填料和筛板熔化。铝的部分燃烧是可能的，但是燃烧既不会蔓延，也不会发生用铝热剂做试验时所看到的那种剧烈能量释放。

在另一类试验中，从填料样品顶部喷淋液氧，底部再通入气氧。在这种典型的精馏塔条件下，样品上的油燃烧后填料仅有小面积的熔化。在用筛板做试验时，油在液氧喷淋下燃烧时仅在筛板上留下少量烟垢的痕迹。（图4“喷淋液氧引燃油后的填料样品”为照片，不清楚，本刊略——编者）。

所有这些试验结果都表明，油燃烧产生的热量可以引燃填料及筛板，也能使金属燃烧。在喷淋液氧即模拟精馏塔工作状况时，填料和筛板是不会燃烧的。筛板在空分设备的精馏塔中已用了35年以上，从没有发生过筛板引燃或燃烧的事故。可燃性的比较显示出筛板和填料在可燃性方面没有什么区别。因此，使用填料和使用筛板相比，不会有额外的风险。

# 六、将液态铝倒入液氧

油燃烧释放的热量有可能会使铝填料熔化。然后，液化的铝会流入液体接受器或塔釜的液氧中去。所做的试验是为了要弄清将

灼热的液态铝倒入液氧是否会引起的铝燃烧，继而引起压力的升高并导致其它的损坏。将铝加热到  $1100^{\circ}\mathrm{C}$ ，然后使热的液态铝穿过气氧流入液氧槽。当  $100\mathrm{g}$  铝流入液氧后，没有看到铝和氧的反应，也没有燃烧的迹象。

# 七、填料表面最大允许的含油量为多少？

填料表面上的油膜燃烧

填料表面上的油膜被认为是另一种引燃源。由于填料与普通筛板相比具有更大的比表面积，如等量的油粘附在填料表面上的话，上塔就会多  $15\sim 30$  倍的有机物质。

在直径  $100\mathrm{mm}$  、高度  $100\sim 200\mathrm{mm}$  的圆柱形填料表面上涂一层  $5\sim 8\mathrm{g} / \mathrm{m}^2$  的油膜，这样薄的油膜在重力的作用下是不会流动的。在进一步的试验中，圆柱形填料的表面上涂一层  $200\mathrm{mg} / \mathrm{m}^2$  的油，用灼热的钢丝点燃气氧中或喷淋液氧的样品上的油。每平方米  $200\mathrm{mg}$  的油没有燃烧，而  $5\sim 8\mathrm{g} / \mathrm{m}^2$  在氧的环境下是可以燃烧的，但油燃烧后也只看到填料上少量的烟垢和局部被熔化的区域。在液氧喷淋下的样品试验中，粘附在填料表面上的油不会燃烧。

粘附在填料上的油，燃烧后既不能引燃铝填料也不能使铝填料燃烧！因此，在上塔的工作条件下，油的燃烧是不可能的。

上塔中油的累积机理

油膜可被高流速的气体、升温蒸发或其它机械作用力从表面上剥落下来。油也有可能由于重力的作用引起的粘滞流动或者由于液氧中油化合物的溶解而流动积累起来。林德公司所用的油的流动特性和迁移方式在试验时得到了确认。

# 八、油在  $200^{\circ} \mathrm{C}$  以下的流动特性和蒸发方式

粘附在填料表面上的油在重力作用下的粘滞流动，可采用间隔称重的方法来测试。

试验表明，当油膜大于  $5\mathrm{g} / \mathrm{m}^2$  时在重力的作用下会发生粘滞流动。但是对于已安装

进塔的填料来说，决不会有这么厚的油膜。

油膜的蒸发行为是在  $25^{\circ} \mathrm{C}$  、  $100^{\circ} \mathrm{C}$  和  $200^{\circ} \mathrm{C}$  的温度下进行测试的。在试验室里，用加热器将单片填料加热到预定温度，隔一段时间测一次填料表面的含碳量。当加热到  $200^{\circ} \mathrm{C}$  时，含碳量会在几分钟内减少。而在  $100^{\circ} \mathrm{C}$  时，几天后含碳量也没有减少。林德公司所采用的油在上塔的工作条件下决不会累积，因为试验结果已经证明，油膜不会由于重力或蒸发的影响而产生粘滞流动。

# 九、油膜在低温条件下的行为

所做的试验是为了弄清楚在  $50\sim 200$ $\mathrm{mg / m^2}$  的油膜厚度范围内，油膜在低温液体中是否会固化剥落而引起油量减少。

结果指出：当油膜少于等于  $250\mathrm{mg / m^2}$  时，在低温液体中填料表面上的油没有减少。也就是说，在低温条件下油膜不会剥落，也不会流动。

# 十一、结论

我们的以上试验以及压缩气体协会任务小组的结论证实了我们的安全观点，因此，填料的安全性是以上塔不存在引燃机制为基础的。

我们所做的试验表明，铝填料用于氧的精馏与具有三十多年成功经验的筛板一样，都能符合安全性方面的高标准要求。我们作出的关于填料表面润滑油最大残余量为50 $\mathrm{mg / m^2}$  的规定也得到了压缩气体协会任务小组的确认。根据我们的文献综述和试验，如此薄的油膜既不会引燃，也不会局部累积。生

产填料时应严格控制油膜量，安装时也应特别注意保持清洁。

由我们以及其它公司所提供的文献综述和试验结果确认：铝填料与铝筛板是同样安全的。因此，铝填料完全可以用于空分设备中上塔的每一段以及其它的塔。

# 制氧工问题解答（五则）

# 1. 问：下塔（含压力精馏塔）的液空纯度（含氧量）是怎样选定的？

答：下塔（含压力精馏塔）的液空纯度是精馏塔的重要参数，与精馏塔上、下塔的精馏工况有关，直接影响上、下塔的理论塔板数。根据实践，它与三方面的因素有关：第一，下塔的操作压力；第二，工艺流程的类型（即高压、中压及低压等）；第三，所需产品的种类及数量（即气态还是液态及比例，尤其是液态产品的数量）。

为什么液空纯度与上述这三个因素有关又不能任意选定呢？因为——

第一，下塔的操作压力一般由空分设备工艺流程确定。如  $150\mathrm{m}^3 /\mathrm{h}$  空分设备属中压流程，下塔的操作压力在  $0.45\sim 0.55\mathrm{MPa}$  （表压)，空气入下塔的组分基本是不变的即含氧  $20.95\%$  。若假定空气是气态进料，那么下塔最底下的精馏塔板下落液体与入下塔的空气处于相平衡的情况下，液空纯度是  $42\% \sim 40\% \mathrm{O}_2$  。实际上最底下塔板上的下落液体与进入下塔的空气不是相平衡关系，而是精馏的操作线关系即低于相平衡的浓度，故一般不平衡度在  $1\% \sim 2\% \mathrm{O}_2$  之间选取。如果装置要提取部分液态产品（液氧、液氮及液氩)，那么入塔的空气中会部分含湿，这是因为精馏系统的物料及焓值的平衡所决定的。这部分含湿液体与下塔最底下塔板的下落液体混合后的组分，就是抽出去的液空纯度，要比上述方法确定的液空纯度要低些。实际上，只要装置的冷损增加，如生产液态产品，带上氩塔或制取高压氧（内压缩)，其液空纯度下降是必然的。

第二，不同的工艺流程液空纯度是不同的，如低压、中压及高压液体流程；内压缩

充瓶流程及附加循环液体流程等。一般装置冷损大的流程（塔釜无蒸发器）其液空纯度取得较低，如中压内压缩充瓶流程，液空纯度在  $30\% \sim 32\% \mathrm{O}_2$ ；低压空气循环液体流程在  $29\% \sim 31\% \mathrm{O}_2$ ；中压液体流程在  $34\% \sim 36\% \mathrm{O}_2$ ；附加低压氮循环的液体流程在  $36\% \sim 38\% \mathrm{O}_2$  等等。

第三，液空纯度要根据上、下塔精馏计算的结果，取上、下塔总理论塔板数最少来确定，即求得最佳液空纯度。

# 2. 问：下塔（含压力精馏塔）的液空纯度分析值，为何与装置说明书上规定的数据出入较大，原因何在？

答：下塔（含压力精馏塔）的液空纯度，若按  $150\mathrm{m}^3 /\mathrm{h}$  空分设备使用说明书上规定含氧在  $36\% \sim 38\% \mathrm{O}_2$  。可是，实际上用铜氨溶液吸附法分析时，往往出现偏高或偏低的情况。是否分析不准呢？回答是不一定。

首先从分析仪表来说，铜氨溶液是按常规方法配制的，分析时取的样品液空全部经汽化后其中的氧组分与铜氨溶液发生化学变化，产生铜的氧化物及铜盐把氧吸收掉，剩余的体积是含氮、氩等不与溶液起作用的气体，按容积测定。一般只要不是溶液失效，分析的结果液空含氧量只会偏低而不会偏高。

出现偏高的原因之一，可能液空在下塔精馏时一方面与操作工况有关，如液氮与液空的节流阀开度即液氮量与液空量的分配不当，这是由操作工决定的。如果液氮的节流阀开度偏大，则一般液氮纯度要下降，液空纯度会升高；反之则相反。另一方面，若不是操作上原因，可能与液空分析管中液空取样有关。因为液空分析管平时不开或开度很

小，分析时因怕管内压力高才开一点点取样，那么长时间积累在分析管中的液空含氧组分，可能由于绝热材料的跑冷而加热汽化，使管中的液空由于长时间的蒸发而增浓，处于相平衡的关系。由汽一液相平衡图知，在0.5MPa（表压）压力下含氧  $38\%$  的液空，其含蒸发时相平衡液体的含氧量为  $61\%$  。考虑到蒸汽的含氧量可能低于  $38\%$  ，做实际分析时液空纯度大于  $50\%$  的情况时有发生，这不能说分析仪表有什么问题，而只能说是操作工取样方法不对所致。若采取短时间把液空分析管吹除一下（气体量可开大些到结霜为止），再关小些取样分析阀，液空纯度（含氧量）会明显下降，可能在  $36\% \sim 40\% \mathrm{O}_2$  之间，这是属于正常范围。所以分析液空纯度时要特别注意取样方法的正确性。

原因之一二，有时也有分析液空纯度其含氧量偏低无法提高。这一方面要看液空节流阀的开度是否偏大，若开度偏大，会使空气未经冷凝而直接经节流阀入上塔。因为空气的含氧量为  $20.95\%$  ，比液空的含氧量（ $36\% \sim 38\%$ ）低得多，两者混合后的纯度可能低于  $35\% \mathrm{O}_2$  以下，甚至更低，这也不能说分析仪表有问题。应从调整液空节流阀入手，若遇到液空液面计损坏或液面指示管堵塞不好用的情况，这时可以凭经验听节流阀的声音，调到响声较小即可。按照这样操作之后，下塔的精馏工况会正常起来，液空的纯度也会好转。另方面也要注意液空分析取样时不能漏进空气，若仪表有漏气现象，可以在转换的旋塞上涂些凡士林油；再用橡皮筋拉紧，再不行要更换新的仪表。也要注意溶液的吸收情况，若反应迟钝应更换溶液。对于设置抽污液氮的下塔，要注意污液氮及纯液氮节流阀的开度，要按说明书的要求保证污液氮及纯液氮的纯度，否则对液空纯度也会有较大的影响。只要操作工正确的判断与操作，一般下塔的液空纯度可以分析到满意的结果。

3. 问：充氧用高压管道为什么必须用紫铜管或不锈钢管，而不能用普通钢管，而钢质气瓶为什么又可以充装高压氧气？

答：选用高压充氧管道的材质，是根据设备的安全性、使用需要和经济性来确定的。

普通钢管含有  $0.1\% \sim 0.15\%$  碳，碳属可燃性物质，在一定条件下，高压高纯的氧气遇到可燃烧物质，只要局部出现高温或火花就会爆炸；而紫铜管不含有可燃性组分，不会燃烧，安全可靠。按设计规范，不得使用普通钢管。倘若一定要用，理论上可予考虑，不过你必须创造不会燃烧或爆炸的条件，如高纯氧在钢管（含金钢管）内流速，须降低到  $6\mathrm{m / s}$  以下（见GB50030—91《氧气站设计规范》）。同时，阀门前部（气体进口上游）  $2\mathrm{m}$  内接上一段紫铜管或不锈钢管。还有，钢管弯头采取焊接，去锈疤，管道进行酸洗处理。再各道法兰的密封垫也有规定，要防止局部产生火花，等等。在大中型制氧机中，低压管道曾用过普通钢管，但因设计与操作配合失调，曾发生过爆炸，造成不少损失。

至于充氧台上用金属充氧软管，近几年开发了软管外层为不锈钢丝编织、内衬管材料为  $\phi 8\times 1$  聚氯乙烯制成的充氧管。但有的氧气厂盲目采用橡胶管、尼龙管、聚乙烯管等可燃性材料，发生爆炸，造成财产损失、人员伤亡，详见《深冷技术》：1996年第4期“一起金属充氧软管燃烧事故”、1997年第1期“上海  $\times$  厂高压充氧橡胶软管三次燃爆事故分析”、1997年第2期“对现用充氧软管安全的一些看法”等。

4. 问：膨胀机进、排气阀密封处磨偏后会引起分馏塔氧气纯度下降吗？

答：山东省荣城市某氧气厂的分馏塔，经

常发生液氧面突然下降、中压上升、氧气纯度急速下降的现象。此时停车检查膨胀机，就会发现进、排气阀严重漏气，拆检膨胀机，就会看到阀密封面已磨偏。不加温修好膨胀机进排气阀后，再开车分馏塔就会恢复正常。然而运转七天后，这种情况又会重复发生。

该厂系  $50\mathrm{m}^3 /\mathrm{h}$  制氧机，配用  $\mathrm{PZK - 5 / }$  40-6型膨胀机。在正常运转中的分馏塔，突然发生氧液面下降，中压上升。此时转动节一2阀并同时提高高压压力，液面恢复正常后，上塔不稳定，低压压力波动  $0.01\mathrm{MPa}$  中氧流量计波动  $0.5\mathrm{m}^3 /\mathrm{h}$  ，氮流量计波动  $10\sim$ $15\mathrm{m}^3 /\mathrm{h}$  ，氧气纯度下降到  $98.5\%$  。厂方说膨胀机进气阀又漏了，停车试果然进气阀漏，拆检时发现进气阀密封处磨偏。厂方说此阀七天前才修过。对膨胀机加温换阀后再开车，塔恢复正常，上塔压力不再波动，氧气纯度可达  $99.5\%$  ，氧气流量可达  $50\mathrm{m}^3 /\mathrm{h}$

查该厂纯化器前压缩空气温度达  $45\mathrm{C}$  冷却水系循环水，水温高。

根据现场分析，该厂上塔发生液泛，液泛的主要原因是冷却水温度高，以致压缩空气冷却不良，造成纯化器工作条件恶化，进入分馏塔内二氧化碳增加所致。上塔筛板原来就此较脏，小孔有堵塞也是原因之一。

为什么膨胀机修理后就会又恢复正常呢?主要原因是停车后塔内杂质沉淀下来,塔板上堵塞小孔的二氧化碳在停车过程中随塔板上液体流到冷凝蒸发器内,塔堵塞现象消除或减轻,液泛条件消失,所以塔又恢复正常运行,这并不是修膨胀机的必然结果。如果此时不修膨胀机,停车几分钟后再开,也会暂时消除液泛。随着设备的继续运转,进入上塔杂质越来越多,液泛就会再次发生。

为彻底消除塔的液泛，根本上要改善纯化器的工作条件，降低高压空气进纯化器前的温度。再是要及时更换分子筛。

5. 问：PZK-5/40-6型膨胀机的进排气

阀为什么会磨偏？怎样防止？

答：该型号的膨胀机进排气阀的密封部分为线接触，检修时经常发现接触线（或接触面）在各个方向宽窄不一，也即磨偏。这种情况很容易发生漏气，从而使膨胀机制冷功率下降，延长分馏塔的启动时间或使正常运转中的分馏塔高压降不下来。造成进排气阀接触面磨偏的原因一般有下面几种：

① 阀套或阀杆不同心。所谓不同心是指该件导向面和凡尔面不同心，对新零件来说，这意味着零件不合格；对旧零件来说，也就是导向套已磨损。② 导向套太松。也就是组装后导向部分间隙太大，这样阀杆在跳动时就难以循理想中心线运动；在进气或排气过程中，在侧向力的影响下，很容易偏向一侧。这种侧向力多是进排气流动的结果。③ 阀杆弯曲变形。④ 安装时，弹簧座偏斜或组装有误。⑤ 阀杆或阀套材质不好，硬度不均或该件热处理不良。⑥ 导向面不光滑，导向套磨损过快，使间隙很快增大。⑦ 阀套导向孔呈锥度，应注意上口小里面大，组装时间隙似乎不小，实际较大。

那么怎样防止上述磨偏现象的发生或延长被磨偏前的时间呢？

①一定要选用合格零件，对旧零件一定要修理合格后再用。  
②四氟材质的导向套间隙一定要调整合适，购来的新零件不一定合适，要进行选配后再用。  
③要定期检修，发现导向套磨损太快要查出原因并消除之。检修周期一般以一年为合适。未满一年即磨偏严重漏气说明上次修理不合格。检修周期太长容易造成正常磨损偏斜过多，不易调整，缩短阀套和阀杆使用寿命。一套修复或新安装后的阀杆正常使用时间，一般应在8000小时以上。

# 变压吸附气体分离技术

变压吸附气体分离技术(PSA)能耗小、操作简便、自动化程度高、无环境污染，是各种气体分离与回收的最理想的方法。

![](images/3ccb57a0024a320af32edd0f2d38ed1b540c09fde45074491f3c302e3126bf7f.jpg)  
广州珠江电化厂  $\mathrm{PSA - CO_2}$  、  $\mathrm{PSA - H_2}$  （两套）装置

![](images/01a54305a6069fb7e13502209b281bdce8fa3222d8448cc70960eb7e12badd80.jpg)  
供浙江镇海炼化公司5万  $\mathrm{m^3 / h}$  PSA提纯氢装置

西南化工研究设计院以其雄厚的技术力量及先进的科研装备，从70年代初在国内率先进行了变压吸附气体分离技术(简称PSA)的研究工作。于1981年实现工业化。该技术1985年获国家科技进步一等奖，1989年被列入国家重点科技推广计划。1993年被国家科委列为全国十大明星推广项目之一，1994被四川省评为省科技进步特等奖。该项技术还获得多项国家专利，其中变压吸附提纯  $\mathrm{CO}_{2}$  技术，1995年被评为中国专利优秀奖。变压吸附提纯CO技术于1996年获国家“八五”科技攻关重大科技成果奖。现已建成或正在建设的PSA工业装置有400余套，1996年在浙江镇海炼油化工股份有限公司成功投产国内最大的PSA提氢装置，生产纯氢达5万  $\mathrm{m}^3 /\mathrm{h}$  ，在大型提氢装置中采用真空变压吸附工艺，在国际上尚属首创；在上海宝山钢铁公司建成一套日产30吨的PSA提纯  $\mathrm{CO}_{2}$  装置；在山东肥城阿斯德化工有限公司建成一套  $1500\mathrm{m}^3 /\mathrm{h}$  PSA提纯CO装置。实现了大型PSA装置的国产化。装置规模和技术水平均属国际先进水平。
